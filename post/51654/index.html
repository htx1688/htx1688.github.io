在经历过去两年的喧嚣后，NFT 市场似乎进入了「寒冬」，各类蓝筹价格不断下跌。究其根源，1.0 阶段以 PFP 为主的静态 NFT 陷入增长瓶颈，2.0 阶段需要新的叙事打开增量市场，动态 NFT 成为市场新兴热点被寄予厚望。

然而，动态 NFT 对底层设施的要求较为严苛，新兴公链 Sui 的出现成功破局。Sui 使用了 Move 虚拟机（MoveVM）和 Move 编程语言来实现智能合约，降低了创作者通过动态 NFT 进行创新的界限，继而解锁出更多类型的用例和更引人入胜的应用及体验。开发人员可以实现动态 NFT，对其进行升级、捆绑和分组，并根据 NFT 的行为变化在链上实时产生反馈。

与静态 NFT 一样，动态 NFT 在初始阶段也面临着流动性困境，GiroSwap 专注于构建为 Sui 生态系统上的 NFT 提供流动性的产品，旗下产品矩阵丰富多样，包括 NFT AMM 协议、NFT 借贷、NFT 碎片化和 NFT 分期付款以及 NFT/Token Launchpad。

作为首个基于 Sui 的一站式可拓展动态 NFT 流动性应用协议，GiroSwap 致力于不断探索和引领加密市场的创新方向。通过 GiroSwap，用户和开发者可以共同探索动态 NFT 的无限潜力，并享受流动性的红利。

一、NFT2.0：动态NFT成为市场下一个风口

2023 年的 NFT 市场，似乎进入了深熊，BAYC、Azuki、Moonbirds 等多个蓝筹项目地板价不断下跌，整体购买情绪低迷。即便 3 月比特币 Ordinals 铭文曾短时引起关注，Yuga Labs 也卖力吆喝，但终究没有掀起更大的波澜，挽救颓废的 NFT 市场。

究其根源，NFT 缺少新的叙事逻辑。市场第一阶段（2021～2023年）的各类 JPEG 或 PFP 在疯狂炒作达到天花板后，价值逐渐回归，消费者（玩家）逐渐理性，很难再为没有实用价值的 NFT 付费。因此，NFT 第二阶段要想拓展增量，必须增强其实用价值，例如 NFT 作为门票、会员、身份、游戏道具、投资标的等等——第一阶段的静态 NFT 显然并不能满足需求，新兴的动态 NFT 也在近期备受关注成为市场热点。

什么是动态 NFT？动态 NFT，Dynamic NFT，简称 dNFT，是相对于静态 NFT 来说的。

事实上，我们接触到的大多数 NFT 都是“静态 NFT”。静态 NFT 是一旦铸造即不能被更改、修成便不可在链上实现更改，因为在编写代码时，其元数据（核心属性）被框定且永久不可变更，这一特性也非常适合视频、图像、GIF、3D 模型等数字艺术作品。目前，静态 NFT 大多应用于艺术项目、音乐摄影、体育竞技等赛道。

然而，静态 NFT 由于元数据的固定，无法动态升级，因此也存在一些瓶颈。一些真实世界中的通证化资产、不断更新的电子游戏等需要接入数据来持续更新，静态 NFT 无法满足需求，此时动态 NFT 可以很好地解决这一矛盾。

动态 NFT 可以根据外部条件进行变化——智能合约会触发动态 NFT 元数据发生变更。除了变更元数据之外，还可以实现动态元素，动态 NFT 可以基于某些特定条件铸造，比如当 AR 应用里发现隐藏地点时铸造 NFT。动态 NFT 还可以通过用户交互来触发“隐藏属性”，这些属性原本不存在于元数据中。

具体来看，动态 NFT 可以应用在以下几个方面：

• 链上 NFT 护照。 动态 NFT 可以无须替换数字身份文档，即可自动更新数字身份，包括居住地、婚姻状况和联系方式等详情都能自动更新。

• 游戏。 玩家使用 NFT 参与 P2E 游戏，游戏中获得成就数据（例如在线时长、胜率、MVP等等）都会实时影响所持有的 NFT 状态；后台的游戏开发者，也可以根据这些数据，对玩家的 NFT 进行升级，或者将其与其他高玩分组在一起，从而提升玩家的游戏体验。此外，还可以根据 NFT 的行为变化在链上实时产生反馈，实现开放式游戏结局（自选游戏结局）。

• 虚拟房地产。 虽然此前已有房地产 NFT，但静态 NFT 无法反映房地产价格、年限和所有权等不断变化的因素；而动态 NFT 恰好可以捕捉房地产详情，具备特定信息更新所需的灵活性。比如持有北京某地段的房产 NFT，该 NFT 会随着市场价格、市场供需等变化而改变，价格也千差万别。

• 品牌动态营销。 过往品牌 NFT 项目都是从 0 到 1 建立一个全新的 NFT 系列作品，大多由品牌单打独斗，非常费时费力。而现在可以根据动态 NFT，汇聚链上链下数据资源，为品牌方设计利益最大化的动态营销方案，最终拓宽商业链路、提升产品价值。

相较静态 NFT，动态 NFT 更智能并且能实时适应并反映外部世界的情况，还能设定限制和规则，进一步丰富了 NFT 的应用场景，更多用例值得探索。NFT 新一轮发展才刚刚起步，动态 NFT 有望为 NFT 市场注入新鲜的血液，引入更多增量不断拓宽市场规模。

二、Sui为动态NFT提供创新基石

静态 NFT 不具备内在的运行逻辑，所有赋能与 NFT 主体是分离的。其用途是脱离于 NFT 本身的故事和文化。而动态 NFT 是具备内在的运行逻辑的，所提供的赋能是内生的，可以直接展示出来的。动态 NFT 的发展前景毋庸置疑，但在落地实践中却面临极大的技术困境。

首先，动态 NFT 需要在智能合约中根据信源不断修改或更新元数据，这就对存储、交易处理速度等基础硬件要求较高。 目前静态 NFT 在以太坊上发展最好，主要是因为对这些要求较低；动辄数十美元的 Gas 费用以及低效的处理撮合（TPS 只有 15）都似得以太坊要想发展动态 NFT 的可能性大幅降低，其他公链同样不具备发展条件。Sui 的出现，补足了动态 NFT 发展的短板。

Sui 的智能合约是用 Move 语言编写的。 Move 语言是一种可编译运行在实现了 MoveVM 的区块链环境中的智能合约语言。其诞生之初，考虑到了诸多区块链和智能合约的安全性问题，并参考了一些 RUST 语言的安全设计。

与许多现有的编程语言不同，Move 语言被设计为既支持编写与不受信任的代码安全交互的程序，又支持静态验证。 Move 语言具备这样的安全特性，是因为舍弃了所有的基于灵活性考虑的非线性逻辑，不支持动态分派，也不支持递归的外部调用，而是使用了泛型、全局存储、资源等概念来实现一些具有替代性的编程模式。例如，Move 省略的动态调度和递归调用特性，这些特性在其他智能合约语言中导致了代价高昂的重入漏洞。

与其他公链相比，Sui 以无法想象的速度和规模运行——峰值吞吐量可达 29.7 万 TPS，其性能的关键在于交易并行化（transaction parallelization）。

在大多数区块链中，交易必须要排好顺序，并放入区块以循序执行；循序执行不必要地限制了这些链上的吞吐量，毕竟大多数交易互相独立。因为 Sui 要求明确说明交易的从属关系，所以它能够并行处理它们。在少数交易相互交织的情况下，Sui 仍然可以对它们进行排序并循序执行。由于独立的交易可以并行验证，Sui 可以通过给每个验证节点增加更多设备来线性增加吞吐量，以此提升规模。

不仅具有高吞吐量，Sui 也拥有低延迟。 它的共识算法专注于尽量减少验证节点之间处理交易所需的通信；与传统区块链的即发即弃广播不同，Sui 确保了请求者和批准验证者之间的双向握手，这导致简单的交易几乎立即被验证，而复杂的交易可以在 2-3 秒内执行。

高吞吐量、低延迟，Sui 上交易可以轻松地整合到需要实时完成的游戏等动态 NFT和其他设置中。

此外，大多数区块链的存储都以帐户为中心，而 Sui 的存储是围绕对象设计的。 每个对象由一个地址所拥有，默认情况下可变，也可设为不可变或在多个地址之间共享。Sui 的 Move 智能合约可以接收这些对象作为输入，对其进行操作，并将对象作为输出返回。这是一种完全不同于 Solidity 或 Rust 的智能合约编程范式，更具表现力，对于动态 NFT 和加密游戏的数字对象表达更简单。

并且，通过研究 MoveVM 的代码，我们可以清晰的看到，MoveVM 将数据的存储和调用堆栈（过程逻辑）的存储是分开的，这点是和 EVM 最大的不同的地方，例如，在 EVM 中，要实现一个 ERC20 的 Token，需要在一个合约中写好逻辑和对每个用户的状态进行记录，而在 MoveVM 中，用户状态（账户地址下的资源）是独立存储的，程序调用必须符合权限和关于资源的强制性规则，虽然牺牲了一定的灵活性，但是在安全性和执行效率（有助于实现并发执行）上获得了很大的提升。

三、GiroSwap：基于Sui的可拓展动态NFT流动性应用协议

伴随着 5 月 3 日 Sui 主网正式上线，动态 NFT 即将迎来爆发式增长，成为加密市场最引人注目的新趋势。当然，动态 NFT 只是第一步，围绕 dNFT 的各类金融产品（即 NFTFi）必不可缺。特别是对于 dNFT 来说，流动性是首要解决的问题。

Opensea 等传统的静态 NFT 市场，基本采用订单簿撮合制度，卖家挂单或者买家 bid 报价。这里有一个问题是，有时买家和卖家价格协商不统一，可能造成 NFT 有价无市或者有市无价两种极端的情况；并且，有时多个买单需要分批购买，消耗高昂的 Gas 费用。

如何破解动态 NFT 流动性难题，Sui 生态首个可拓展动态 NFT 流动性应用协议 GiroSwap 应运而生。GiroSwap 是 Sui 黑客松越南站获奖项目，目前已上线主网，主打 NFT/ Token 一站式聚合交易。

具体来说，GiroSwap 推出了多个旗舰产品满足市场刚需，包括 NFT AMM 协议、NFT 借贷、NFT 碎片化和 NFT 分期付款以及 NFT/Token Launchpad。有了 GiroSwap，用户不需要在 Sui 生态多个应用程序中辗转腾挪，一个 Dapp 即可解决所有需求，GiroSwap = Uniswap + BenDAO + Opensea + Blur+ Launchpad。

GiroSwap 推出的 NFT 去中心化交易所，采用自动做市商 (AMM) 模式。流动性提供者（LP）可以将资产存入单边或双边交易池为自己喜欢的 NFT 提供流动性，并可以选择买卖价差收取交易手续费；GiroSwap 不区分不同的 NFT ID，这意味着流动性池中的每个 NFT 都与当前价格系数绑定，愿意买卖 NFT 的用户在交易时从 NFT 池中获得相同的价格，无论 NFT ID 是否稀有。此外，GiroSwap 支持一键购买，批量购买，可以有效节省 Gas。

除了支持 NFT 交易，GiroSwap 也支持 Sui 的原生代币和桥接到 Sui 生态系统的所有代币通过 AMM 进行交易。相较于 Uniswap V2 ，GiroSwap 滑点更低、Gas 费用更低、用户体验更流畅。

需要注意的是，相比传统 NFT 市场存在局限，GiroSwap 通过采用 NFT AMM 模型解决了许多问题。传统市场中，每个 NFT Collection 的筛选功能不尽相同，而 GiroSwap 通过调用每个 Collection 合约的标签，实现了更灵活的筛选功能，这使得用户能够更便捷地找到符合自己需求的 NFT。

再者，GiroSwap 推出 NFT 借贷市场，以释放动态 NFT 资产的价值。如果 NFT 持有者不想出售自己的资产，也可以抵押 NFT 获得稳定币或 Sui 代币从而缓解现金流压力。点对池匹配借贷系统通过流动性池连接借款人和贷方，并且 GiroSwap 还实施了强大的清算系统以确保借出的 NFT 的安全并保护贷方的利益。

另外，GiroSwap 还通过其点对池交易系统提供 NFT 分期付款。此功能使用户可以通过分期付款的方式购买他们想要的 NFT，从而使 NFT 可以为更广泛的用户所接受。用户可以购买他们感兴趣的 NFT，而无需预先支付全价，从而推动 NFT 在市场上的进一步增长和采用。

另外，考虑到 NFT 发展成为蓝筹项目后价格走高，导致普通散户难以触及，GiroSwap 还引入了 NFT 碎片化功能，允许用户将他们最喜欢的 NFT 转换为代表底层 NFT 部分所有权的 MemeToken。这些 MemeTokens 可以在平台上自由交易，使用户能够接触到各种 NFT，而无需拥有整个资产。这一功能增加了整个 NFT 市场的流动性，并使对有价值的 NFT 访问更加民主化。

最后，GiroSwap 推出了极具创新的 Launchpad，可以所有项目（NFT、Token）提供自动化筹款工具和一站式流动性解决方案。GiroSwap 允许开发者筹款活动之前自定义 Token/NFT 发行、索赔和其他规则，开发者只需要设置筹款条款，可以有效降低开发和做市成本。

总得来说，GiroSwap 是第一个建立在 Sui 生态系统上的社区驱动的多协议平台，旨在通过交织各种去中心化市场和工具来创造更广泛的网络效应，为市场注入更多流动性。
