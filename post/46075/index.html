每天勤勤恳恳日更公众号有什么用，NFT 和加密爱好者的首选社交网络一定会回答我是 Twitter。

很多朋友不知道，我们不仅英文站，还有一些加密行业常用的社交媒体平台（链接放在文末了），公众号上经常让我感到很无奈，好多很不错的内容，却连个预览都发不出去，丢给我一句「可能违反相关政策」就直接显示了被删除。

虽然最近一段时间我们由于「阴阳交替」，严重精力不足，顾不上更新其他平台的内容，但今天还是准备跟大家聊一聊很多项目方在早期做运营的时候最关心的问题，该如何写出吸引收藏者注意的推文呢？如何瞄准正确的用户人群？最重要的是，如何让项目与人们建立真正的联系？

要建立一个 NFT 社区，你需要知道所有这些问题的答案。

赠品对那些想要赢得 NFT 的人很有吸引力，但太多的赠品会让普通 Twitter 用户感到厌烦，这给人感觉就像垃圾邮件。

在尝试销售你的 NFT 的过程中，你需要以娱乐、启发、教育或讲故事的形式为社区提供价值。每 6-8 条推文中只有一条应该含有销售内容。

就像 Gary Vee 关于社交媒体营销的畅销书《Jab, Jab, Jab, Jab, jab, …. Right Hook》写的一样，你不会想在每条推特上都打出致命一击。

创建内容

思考该在 twitter 上发布什么内容既紧张又耗时。你绝对不想让你的内容看起来很大众化。

如果没人回复怎么办？会不会很尴尬？你应该删除它吗？

别着急。我将向你展示一些在短时间内创建内容的简单方法。

关于推文内容并没有严格的规定，但有 4 种主要类型可以很容易地与你的项目主题相匹配。

主要的 NFT 推文类型包括呼吁行动推文、观点推文、新闻推文、教育推文和表情包。

它们都以各自的方式提供价值或娱乐。

行动呼吁 (CTA) 推文

行动呼吁推文会号召浏览者去做某事。你可以呼吁他们在 OpenSea 上购买你的 NFT。

你可以让他们加入你的 Discord。但要确保不要让他们一次做太多行动……

「当你呼吁了三个行动时，就相当于根本没有进行行动呼吁。从手机屏幕上看到这些乱七八糟的链接和短文的客户一定非常困惑。我们根本不知道应该先把注意力集中在哪里。——Gary Vee

你可能会看到一些 NFT 赠品推文要求采取许多步骤，但这些大多是由机器人参与的。

有一些 SMM 板块出售 Twitter 机器人的转发和点赞，它们甚至可以在评论中 tag 好友数千次。

与加密 / 非加密领域的所有事物一样，外观可能具有欺骗性。需要仔细观察细节，以确定什么是真正在发生的和有效的。

要实际看待你的 NFT 的价值

当你在设计你的 CTA 赠品推文时，要确保奖品值得人们付出努力。

如果没有人听说过你的 NFT，你不能要求人们付出太多努力来赢得它。

为了让你的赠品更有吸引力，你可以试试：「任何与这条推文互动的人都有资格获奖。」

此外，添加现金奖励也很有帮助，比如给赢家 200 美元的 ETH！

如果你要送出一个价值 50 万美元的 Bored Ape，那就要求人们 tag 好友、加入 Discord，并让他们付出些其他什么代价。

观点推文

这类内容是我最喜欢的，简单来说就是俩字，「输出」。

在其关于 NFT 和 Twitter 的 YouTube 视频中，Fungible 先生解释了为什么意见在 Twitter 上很重要。

「你希望那些和你有相似价值观和信仰的人关注你……因为他们更有可能参与你发布的内容。」

所以在推特上发表观点是在推特上获得正确追随者的一种有机方式。那种会被你的推文吸引的人。

我最喜欢的作家 Hunter S. Thompson 通过一遍又一遍地抄写《了不起的盖茨比》小说来学习写作。

如果你发现很难想出观点推文，可以尝试重新编写现有关于 NFT 或 Web3 的流行推文。

你将以自己的方式传播已经引人入胜的想法，并获得经验，最终创作出自己的原创观点推文。

以下是取自 @DCLBlogger 的一个病毒性观点推文的例子：

「最大的财富转移发生在你是对的，而大多数人最终都错了的时候。」

我们可以快速地把这句话用我们自己的话写成一条观点推文：

当你是对的，而大多数人是错的时候，金钱是最容易到手的。

新闻推文

新闻类内容是我最快乐也最头疼的，因为总是要第一时间赶热点，虽然扒皮讲故事很快乐，但经常会牺牲掉我宝贵的睡眠时间。

新闻推文是不言自明的，但有一件事让我很恼火，那就是当人们在推特上发布新闻链接时，推特不希望人们离开它的平台。毕竟所有的社交网络都以吸引你的注意力为目标。

因此，当你在 OpenSea 上发布一个关于 NFT 新闻的链接，甚至是你自己的 NFT 时，Twitter 会裁剪图片，使其在 feed 中不具吸引力。

如何发推文

NFT 社区的每个人都想学习新东西。在你的推文中添加 NFT 指南或教程是个好主意。

你可以使用谷歌搜索博客或 Medium 文章。它们可以简单如「如何设置 Metamask」，也可以复杂如「如何在 Polygon 上创建一个节点」。

通过囊括教程推文，你将能够了解有关 web3 的最新信息，并为你的粉丝提供价值。

创建 Twitter 日程

作为一个每天惨兮兮更内容的人，我掏心掏肺的和大家说一句切身经验，别指望自己每天都发推特，这是不可能的。

为了获得粉丝，你需要与推文保持一致，并覆盖一天中的不同时间。

你可以使用任何社交媒体软件中的日历选项，将一个月的推文安排在一个周日下午。

很多人会使用 Twitter 附带的免费软件 Tweetdeck。它对于一些基本的发布是没有问题的，但如果你想建立一个有效的 Twitter 增长系统，我推荐 Social Champ。

我就是用它来管理数百个 NFT 项目的社交媒体账户的。

在 Twitter 上保持一致是建立 NFT 社区的良好开端。

总结

现在你知道了，不要只是照搬别人在推特上的做法。因为那就像盲人给盲人引路一样。相反，找到你自己独特的方式为 NFT 社区带来价值。

你已经准备好了创作 4 种不同类型的 Twitter 内容，你知道在哪里可以找到示例和灵感。

每天都有各种各样的推文，通过创作有价值的内容，你将在推特上获得信任和新的粉丝。

当你有了自己的系统来持续发展一个有价值的推特账户时，你就可以花更多的时间与你希望与之建立个人联系的人联系。
