Pudgy Penguins 在 5 月 18 日发布 Pudgy Toys 和 Pudgy World，一步步通过 IP 衍生品和休闲游戏，获取更大的消费者群体。

Pudgy Penguins 近期做了什么？

5 月 18 日 Pudgy Penguins 宣布推出 Pudgy Toys 和 Pudgy World。

Pudgy Toys 是其第一个由社区支持的市场产品，每个玩具都可在其网站上进行数字体验。

Pudgy World 提供非常快捷的数字体验，用户只需要一个邮箱，便可以注册区块链钱包，用户可以在网站上装扮自己喜欢的企鹅。

Pudgy Toys 一共有六款，售价在几美元到二十几美元不等，通过亚马逊开展线上售卖，线下与 PMI Toys、Retail Monster 合作进行零售。

每个玩具都包含一张出生证明，出生证明是一个链接到 Pudgy World 网站的二维码，用户扫码登陆后可解锁 Pudgy World 中的独特特征，每个玩具都可解锁四到六个特征。持有者可以将这些特征装饰到其 Forever Pudgy Penguin 上，同时也可以在特征交易市场上买卖这些特征，交易市场由@OriginProtocol 提供支持。用户在装扮好自己满意的 Forever Pudgy Penguin 之后可以选择 Mint，Mint 过程是在区块链上将其铸造为灵魂绑定的 NFT（即 SBT），全程 0 gas，上链技术由 Polygon 支持。

5 月 18 日 Pudgy Toys 在亚马逊推出之后，两天时间卖出超过两万个，销售额超过 50 万美元。

过往顶尖 IP 的共同点，以及 NFT 能从它们身上学到什么？

  • 基于 IP 的衍生商品是最主要的收入来源，这些品牌的主要收入来源是商品销售。这些品牌通过销售各种相关商品（如玩具、衣服、书籍等）来实现盈利。商品通常与品牌形象和故事情节相关联，可以吸引消费者的兴趣和购买欲望。

  • 这些品牌都是以儿童为主要受众群体。这种定位有助于促进不同年龄层之间的消费行为，从而扩大了品牌的市场规模和影响力。

  • IP 的长期运行是基于用户与 IP 之间的情感效用，这些品牌的核心价值是情感效用，而不是经济驱动。这意味着这些品牌通过情感上的共鸣和认同来吸引消费者，而不是通过金融激励或其他经济驱动来促进消费行为。这种情感效用可以帮助品牌建立更深入的联系和忠诚度，从而实现更长期的盈利和发展。

今天最成功的知识产权品牌，如迪士尼、DC 漫画和游戏王等，最初都是从简单的媒介形式开始的，如动画电影、漫画和纸牌游戏。

同时每年用于 IP/娱乐的消费都达数万亿美元。人们对这类消费的需求非常高，这种需求可能源于对动画角色和故事情节的喜爱和追捧，以及对相关商品和周边产品的购买欲望。

随着区块链技术和 NFT 的发展，未来会出现新的知识产权品牌，这些品牌将会创造价值数十亿美元的市场。因此，我们有理由相信 NFT IP 时代将会创造未来的巨型知识产权品牌。

Web3 中运营 IP 的优势

  • 运营一个 Web3 中的 IP 要比传统 IP 门槛更低。故事讲述者和艺术家可以从零开始创建知识产权品牌，而不需要面临传统知识产权生态系统中的高昂成本和漫长的启动时间。与此相比，NFT IP 可以通过成本低的发行工具（如 Manifold、Fair 和 OpenSea）在几天内花费到 1000 美元的成本来发布 NFT 。这使得独立艺术家和团队有机会参与到知识产权品牌的创建中，普通人也能获得相对平等的机会。

  • 通过更精简的初始知识产权品牌创建过程，品牌可以更多地关注故事和叙事的构建。团队通过减少产品构建的时间，可以执行更多的社区建设，社区成员更有动力提供意见并吸引市场的关注，因为他们拥有 IP 所有权和相应的收益。这使得社区成员能够更好地参与和塑造知识产权品牌的方向和发展，并且可以获得更多的收益。

  • 项目成功发布之后可以期待他们重新投入资源到其他内容垂直领域（商品、玩具、电影、动画、游戏、漫画等），我们称之为最小 IP 开发。

Pudgy Penguins 的商业模式和 IP 授权的创新

  • 通过在 Tiktok 和 Instagram 等平台上，使用表情包、视频等针对年轻一代的大众推广其 NFT IP 品牌。值得注意的是，他们在宣传时并没有提到“加密”或者“NFT”，删除这些标签有效地避免了一些用户对这些技术和概念的陌生感和不信任感，从而提高了用户参与度，吸引了更广泛的用户群体。

  • 销售 IP 衍生商品确保了可持续发展的项目商业收入，扩大了消费群体，一定程度上结束了 Web3 目前争抢存量的零和游戏。

  • 与社区 Holder 统一安排授权协议。在发行的企鹅形象玩具中，有 16 个形象来源于社区中的十多位 Holder，项目方通过预付一笔签约款给这些 Holder，之后的玩具销售收入也会按照一定百分比与其对应的 NFT 编号绑定。

  • 对于多数 Holder，将其持有的企鹅形象授权用于社交媒体上的推广，如 Instagram 上的表情包，小视频，GIF 等。

  • Pudgy Penguins 或将推出一个全新的产品化的 IP 解决方案：OverPass（细节暂未公布）。

Pudgy Penguins 专注于创建基于加密货币独特属性的数字体验，并尽可能地抽象化 UI/UX 的复杂性。Pudgy Penguins希望通过提供易于使用的数字体验，让即使是那些完全不熟悉区块链技术和概念的人也能够参与区块链世界。

目前 Pudgy Penguins 在 IP 运营已经初有成效，我们期待着 Pudgy Penguins 后续更多动作，也期待着 NFT IP 成为 Web3 下一个吸引千万级用户的突破口。
