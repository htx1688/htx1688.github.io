Sui Wallet 是Sui官方推出的一款安全且易于使用的钱包，旨在为用户提供轻松访问Sui生态项目和资产的方法。本文将向您介绍如何快速安装和使用Sui钱包浏览器扩展程序。

您可以使用Sui钱包对Sui网络进行测试，目前，Sui网络仍在持续开发中，Token还没有任何价值，很多高级功能还没有包含在Sui钱包中，请参阅Sui CLI客户端文章：https://docs.sui.io/devnet/build/cli-client

概述

Sui钱包特性

您可以使用Sui钱包实现以下功能：

创建Sui账号（钱包地址）

  • 导入其他钱包的私钥来添加钱包（必须是32或64个字节的地址）

  • 质押和赚取SUI

  • 转移coins或NFTs到其他地址

  • 查看您的coins、 tokens和NFTs

  • 查看最近的交易记录

  • 自动拆分或合并coins来满足转账所需金额

  • 很容易访问到Sui浏览器查看历史记录

安装Sui钱包浏览器扩展程序

要使用Sui钱包，您必须安装Chrome浏览器扩展程序，您可以在任何支持Chrome网上应用商店的浏览器上下载并使用该钱包的扩展程序。

  1. 使用基于chromium的浏览器，打开Chrome应用商店中的Sui钱包页面

  2. 点击「Add to Chrome（添加到Chrome） 」

  3. 确认使用扩展程序的权限，然后点击「Add Extension（添加扩展） 」

创建新钱包

如果您还没有Sui钱包, 可以创建一个新的，如果需要导入已有钱包，请参阅下方导入已有Sui钱包的教程。

  1. 在浏览器中打开Sui钱包扩展程序，然后点击「Get Started（开始使用） 」

  2. 点击「Create a New Wallet（创建新钱包） 」

  3. 在「Create Password（创建密码） 」时，输入您钱包的密码，注意：这不是Sui钱包的密钥，该密码在浏览器中打开钱包时使用

  4. 在「Confirm Password（确认密码） 」时，需要再次输入相同的密码进行确认

  5. 点击复选框以接受服务条款

  6. 点击「Create Wallet（创建钱包） 」

  7. 抄写助记词并将其存储在安全位置，然后点击「I saved my recovery phrase（我已保存助记词） 」的复选框

  8. 点击「Open Sui Wallet（打开Sui钱包） 」

首次设定后，Sui钱包会在您打开时提示您输入密码。

如果您无法访问钱包，可以使用助记词来恢复它。注意：如果您丢失了助记词，将无法访问您的钱包以及存储在其中的任何coins或NFTs。

导入已有的Sui钱包

您可以在多种设备和浏览器上使用Sui钱包，创建Sui钱包后，可以使用12个助记词将您的钱包导入新的浏览器或设备中。

  1. 在浏览器中打开Sui钱包扩展程序，然后点击「Get Started（开始使用） 」

  2. 点击「Import an Existing Wallet（导入已有钱包） 」

  3. 输入要导入帐户的12个助记词，然后点击「Continue（继续） 」

  4. 在「Create Password（创建密码） 」时，输入此帐户地址的密码，注意：这不是Sui钱包的密钥，该密码在浏览器中打开钱包时使用

  5. 「Confirm Password（确认密码） 」后，再次输入相同的密码进行确认

  6. 点击「Import（导入） 」

  7. 点击「Open Sui Wallet（打开Sui钱包） 」

首次设定后，Sui钱包会在您打开时提示您输入密码。

创建新的账户地址

您可以在Sui钱包中创建和使用多个账户地址。

添加其他帐户地址的步骤如下：

  1. 点击菜单（钱包界面右上角的菜单栏），然后点击「Accounts（帐户） 」

  2. 点击「Create New Account（创建新帐户） 」

完成以上步骤，新帐户即显示在钱包中。要使用新帐户，请从「Coins 」选项卡的下拉列表中选中它，或者在钱包连接到某网站或应用程序时，选择要使用的地址。

将其他账户导入您的钱包

您可以从以前安装的Sui钱包或其他钱包处导入帐户，导入账户的地址必须为32或64个字节。

向Sui钱包中导入其他账户

  1. 点击菜单（钱包界面右上角的菜单栏），然后点击「Accounts（帐户） 」

  2. 点击「Import Private Key（导入私钥） 」

  3. 输入或粘贴要导入帐户的私钥，然后点击「Continue（继续） 」

  4. 输入钱包密码，然后点击「Import（导入） 」

在钱包显示「Accounts（帐户） 」的页面，即可看到已导入的帐户。

导出账户私钥

您可以导出一个帐户的私钥，再用私钥将该账户导入到另一个钱包中。任何人都可以使用私钥导入相应的帐户，并通过私钥访问账户及其资产。请妥善保管您的私钥，切勿将其分享于他人。

如何导出帐户的私钥

  1. 点击菜单（钱包界面右上角的菜单栏），然后点击「Accounts（帐户） 」

  2. 选中要导出私钥的帐户地址

  3. 点击「Export Private Key（导出私钥） 」

  4. 输入钱包密码，然后点击「Continue（继续） 」

  5. 点击「Copy（复制） 」将私钥复制到剪贴板

然后，您可以将此私钥粘贴至要导入钱包的相应位置。查看私钥时，选择右下角的隐藏显示图标，以防止他人窥视您的屏幕。

添加SUI tokens

当您第一次打开钱包，钱包中空空如也，您可以使用Discord中的水龙头将测试用的SUI coins添加到您的钱包。

使用钱包获取SUI测试coins

安装好钱包扩展程序后，点击「Request Devnet SUI tokens（请求Devnet SUI tokens） 」。大部分情况下，您可以点击三次来获得tokens，但是可能存在点击一次或两次后按钮会消失的情况。如果发生这种情况，您可以点击钱包界面右上角的菜单栏，在钱包设置页面上找到该按钮，该按钮只有在您第一次成功使用后才会显示在那里（最多三次请求）。点击之后，SUI测试tokens会出现在您钱包中的「Coins」选项卡上。请注意，获取测试币有频率限制，因链上活动频繁，有时可能测试tokens无法准时到账。

通过Discord获取SUI测试coins

  1. 点击「Coins 」

  2. 点击地址旁边的剪贴板图标进行复制（它靠近钱包顶部，以0x开头）

  3. 切换到您所用网络的Discord水龙头频道：

    · 开发网水龙头 ：在Discord的devnet-faucet频道

    · 测试网水龙头 ：该频道可能偶尔不可用

  4. 使用!faucet指令，同时输入您的钱包地址来请求tokens，将下方示例中的地址替换为您的钱包地址即可：

!faucet 0xe560f6778355cd6b9a2e1d87030eaf47758f7941f5225023dad9d1682cc23acb

此时，频道内机器人会显示一条消息，确认您的请求。

查看Sui钱包详情

要查看您Sui钱包的详细信息，如帐户ID、当前网络和安装的版本等，请点击Sui钱包界面右上角的菜单栏。

重置Sui钱包密码

如果您忘记了Sui钱包的密码，可以使用12个助记词重设密码。

  1. 在「Welcome Back（欢迎回来） 」页面上，点击「Forgot password?（忘记密码） 」

  2. 输入您的12个助记词，然后点击「Continue（继续） 」

  3. 输入密码，然后二次输入确认密码

  4. 点击「Reset（重置） 」

锁定您的Sui钱包

您可以锁定您的钱包以防止未经授权的访问，但必须输入密码才能解锁。

  1. 点击Sui钱包界面右上角的菜单栏

  2. 点击「Lock Wallet（锁定钱包） 」

您还可以设置一个计时器，闲置一段时间后（最多30分钟）自动锁定您的钱包。

  1. 点击Sui钱包界面右上角的菜单栏

  2. 点击「Auto-lock（自动锁定） 」

  3. 输入钱包锁定前等待的时间（最多30分钟），然后点击「Save（保存） 」

即使您在浏览器中切换选项卡，钱包也会在您指定的时间内保持解锁状态。

切换活动网络

您可以在Sui钱包中切换活动网络。目前，Sui钱包支持Sui Testnet 、Sui Devnet、Local和Custom RPC URL 四个线路， 除非您已经有用于测试的本地网络，否则请使用Devnet。

要了解如何创建本地网络，请参阅创建本地Sui网络：https://docs.sui.io/devnet/build/sui-local-network

  1. 点击Sui钱包界面右上角的菜单栏

  2. 点击「Network（网络） 」

  3. 点击要使用的网络，活动网络旁边会显示一个复选标记

查看钱包余额

要查看您的钱包余额，请点击「Coins 」，此时钱包会显示您的SUI余额，并列出您钱包中的其他coins （如果有）。

发送coins

您可以将coins从您的钱包中发送到另一个钱包地址。

  1. 在浏览器中打开Sui钱包扩展程序

  2. 点击「Coins 」然后点击「Send（发送） 」

  3. 在「Amount（数量） 」栏中，输入要发送的SUI数量，然后点击「Continue（继续） 」

  4. 输入收款人的钱包地址，然后点击「Send Coins Now（立即发送Coins） 」

质押和赚取SUI

您可以尝试质押SUI来获得奖励，但这些奖励仅用于测试，没有任何价值。当您质押SUI时，您将SUI tokens委托给验证节点进行质押，然后，验证节点会支付奖励，以感谢您将SUI委托给他们进行质押。请注意，SUI tokens在测试网络上没有任何价值。

质押SUI并获得奖励

  1. 打开您的钱包并点击「Coins 」

  2. 点击「Stake & Earn SUI（质押并赚取SUI）

  3. 选择要质押的验证节点

  4. 选择要质押的SUI数量，确保您的钱包中有足够的SUI来支付gas费

  5. 点击「Stake Now（立即质押） 」

您本次的质押将在下一个epoch开始后计算收益奖励。

查看当前质押

要查看您当前质押的详细信息，请点击钱包「Coins 」选项卡上的「Currently Staked（当前质押） 」按钮，查看您的质押金额、选择的验证节点、已赚取的金额和验证节点佣金等详细信息。

查看近期交易详情

您钱包的近期交易情况将显示在「Activity（活动） 」选项卡上，点击任何一笔交易即可查看该交易的详细信息。

在Sui浏览器上查看所有交易

您可以在Sui浏览器中查看该地址的所有交易记录：https://explorer.sui.io/

要查看您地址上的所有交易，请点击「Apps（应用程序） 」，然后点击「View account on Sui Explorer（在Sui浏览器上查看帐户） 」，Sui浏览器将打开并显示您钱包地址的详细交易信息。

查看您的NFTs

点击「NFTs 」选项卡来查看您在钱包中铸造、购买或接收的所有NFTs，包括您从连接的应用程序中获得的任何NFT。点击NFT来查看相关的详细信息、查看放大的NFT图片或将NFT发送到另一个地址。

发送NFT

您可以使用Sui钱包将NFT发送到另一个地址：

  1. 点击「NFTs 」

  2. 点击要发送的NFT，然后点击「Send NFT（发送NFT） 」

  3. 输入接收方的地址，然后点击「Send NFT Now（立即发送） 」

  4. 点击「Done（完成） 」并返回钱包

钱包中可用的应用程序

您可以在「Apps（应用程序） 」选项卡的「Playground 」中查看和试用一些已经支持Sui钱包的应用程序， 这些应用程序允许您连接Sui钱包并使用SUI tokens与其交互、交易或直接接入您钱包中的NFT。

点击一个应用程序来打开其网站，按照网站上的指导进行操作。将钱包连接到应用程序后，您可以在「Active Connections（活动连接） 」视图中查看该应用程序。

查看已连接的应用程序

要查看哪些应用程序已经与您的钱包建立有效连接，请点击「Apps（应用程序） 」按钮。默认情况下，会显示「Playground 」视图， 点击「Active Connections（活动连接） 」即可查看已连接的应用程序。

要打开与该应用连接的网站，选中该应用，然后点击查看。

断开与应用程序的连接

您可以轻松地断开您的钱包与已连接应用程序之间的连接。

  1. 点击「Apps（应用程序） 」，然后点击「Active Connections（活动连接） 」

  2. 选中该应用并断开与您钱包的连接，然后点击「Disconnect（断开连接） 」

  3. 您的钱包会立即与应用程序断开连接，并在「Apps（应用程序） 」选项卡中显示出来。
