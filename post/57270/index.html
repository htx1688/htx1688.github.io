$VELO 在我这边的交易已经结束了（在 8.9 左右结束的）。拖了几天，发出来复盘一下我在 $VELO 上的操作，大概赚了 50% 左右。

也没有秀利润啥的，自己有正确操作也有蠢 B 操作。而且我目前在一些蓝筹山寨上也套了一批，比如 $ARB 、 $LDO 。发这篇的目的主要是聊聊我如何建立观点和按照我自己的观点执行交易的。最后总结一下。

为什么会买 $VELO ？

主要原因是 Basechain 叙事——Velo 会给 veholder 空投 veAERO（Basechain 上 Velo 团队做的 ve33），结合 $OP 表现强势，我认为 $VELO 也会在这个阶段有着不俗的表现。当时给群友写了一堆逻辑和分析，并开了一个腾讯会议来具体聊这件事情。

那既然我认为逻辑成立，就用仓位来验证我的观点。于是我在 0.09-0.10 买了 $VELO ，主要是因为 OK 上 $VLEO 的流动性太差了，不好买。（以上逻辑中有一个数据错误，就是 VELO 的市值）

然后就拿着，什么也不干，看着 $VELO 上涨。

在 $VELO 上涨到 0.12-0.13 后，我发现在 OK 上的 $VELO 是 v1 token，链上是 v2 token，v1 和 v2 之间存在价差，这是因为 v1—>v2 的过程是不可逆的，因此不存在套利空间。基于未来 v2 token 才是主流的因素，我把 $VELO v1 token 换成了 v2 token（链上），在用 Velodrome 买了 v2 后，我才发现链上 ve33 的流动性要比 OK 上好太多，直接买也不会有太大的价差。最后的结果就是买到了价格在 0.1 附近的 v2。（换 v2 的另外一个原因是从长远角度来看，CEX 会把 v1 token 直接换成 v2，v2 价格和 v1 价格最终会收拢）

然后，我把 v2 $VELO 放在 Extra 的 Lending 上吃币本位收益，APY 是浮动的，不过那段时间大概就在 100% 左右浮动。

卖出是在 $BALD 出事后。在 $BALD 出事后，我意识到 Basechain 的叙事会受到极大的影响，于是卖出了 $VELO 。

蛇皮操作在于，我被 Velo 所表现出来的基本面所吸引了——在 $VELO 价格下跌之后，就在 0.12 附近抄底了 $VELO ，还好醒悟及时，在 8.10 卖掉了 $VELO ，结束了这次交易。

另外一个错误操作在于给 $VELO 过高的估值，在 v1 价格 0.15 的时候，没有卖掉 $VELO v1，直接赚到 50%，是在换 v2 之后，吃了一波价格收敛。最后勉勉强强赚到了 50%。在这种 pvp 行情，波段可能才是最优选择。

值得关注的是，在这轮上涨中，锁定 $VELO 为 ve 的操作并不是很多，因此 $VELO 更像是在情绪面的聚集，而非由基本面产生的供需关系变化而推动的上涨。在做交易类型的时候，一定要确定自己交易的真实动机，快刀斩乱麻，做叙事新闻交易就好好做，不要被基本面的假象所迷惑，尤其是要关注最根本的数据层面，而非贿赂的新高等数据。ve33 本质是在吃生态的红利，决定上涨飞轮出现的因素其实是 OP 生态的丰富度和整体行情情绪给予人的信心。

以上便是我的复盘，感谢大家的支持。
