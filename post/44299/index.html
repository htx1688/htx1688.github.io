撰文：Kyle

来源：WebX实验室

NFT+DAO 这样的想法目前对 Web3 可持续发展爱好者非常有吸引力，对于项目团队也是如此。然而，我们还有另一个问题要面对：如何教育和激励「NFT+DAO」变得更有实际意义？从我的立场来看，有三个简单的优势：

  1. 吸引更多像我这样的 NFT 爱好者；

  2. 通过 DAO 的力量表明 NFT 并不可怕，在公众和 NFT 之间架起了一座桥梁；

  3. 为简单的 NFT 项目赋予更多价值，并尽可能奖励拥有它们的人。

由于投机者的强烈参与，许多人认为 NFT 远离 Web3 行业的初衷。 我认为这部分主要由负面环境影响，例如垃圾项目和黑客事故，这些都不容易预防。NFT 还可能导致价格炒作和对真正有价值的生态系统的破坏。因此，NFT+DAO 旨在优先考虑真实度和安全性。

1 NFT、DAO 和 Layer 2

如今，NFT 是 Web3 里面最炙手可热的领域之一，并已证明其有能力协助创造就业机会，促进社会包容，保护艺术和历史资产，保护资产多样性，产生可持续的收入，并在适当规划和管理的情况下提高区块链网络福利。我鼓励每一位 Web3 参与者更加努力地改善这种情况，并创造我们将为之自豪的未来。

另外，如果我们讨论到 NFT、社交和链上生态的可持续性，那么内容将围绕工作量证明与权益证明主题展开。以前，矿工根据先前的 PoW 协议验证以太坊区块链上的交易。交易由矿工收集，并作为有序块发布到以太坊区块链。

在 PoS 权益证明中，则使用验证器代替矿工，这些服务与矿工目的相同，但他们无需将大量资金投入能源消耗中，而是将 ETH 作为针对不诚实活动的保险。通过最近的 MERGE，以太坊之前的能源消耗量减少了 99.95%，这意味着确保以太坊所需的总能量预计不到 0.01TW/ 年。

因此，NFT+DAO 选择以太坊区块链的 Layer2 方案已经是不可阻挡的潮流，因为我们认为它最适合社交的需求。同时 NFT+DAO 类项目还应该支持以太坊虚拟机，允许与主网相同的功能。它的好处是降低了成本和可扩展性。至少在一开始，DAO 需要免费向客户提供 NFT，以建立信任并建立更高的参与度。

2 无门槛的 NFT 未来

代币将改变消费者与 Web3 品牌和组织互动的方式。目前 DAO 对代币门槛的含义进行了级别设置，但说服大众放弃他们已习以为常的生活方式并鼓励他们转向 web3 是具有挑战性的。DAO 可以教育人们了解 NFT 并改善许多人对该技术的偏见，我相信 DAO 将能够激励更多的人这样做。

当涉及到 NFT 时，DAO 的主要目标是向用户展示他们的一些担忧可能是错误的，并为他们建立通往 Web3 的桥梁平台，并尽最大努力实现这一目标。为了简单起见，让我们使用一个示例进行无门槛的 NFT DAO 流程：

基本上，当用户拥有一个 DAO 代币，就可以得到一些不同的东西，类似于：

  • 进入体育场的门票；

  • 社交俱乐部的会员通行证；

  • 订阅数字内容的会员。

以上所有内容都为个人解锁了特殊或差异化的访问权限，无论是在体育场内观看比赛还是在家中观看比赛，还是阅读新闻与仅阅读开头。那么，以代币为基础构建的 NFT DAO 有什么不同呢？Web3 的核心是独特的资产标识和对象的所有权。

这种所有权是通过代币和钱包实现的。数字代币可以存储在用户的数字钱包中。现在，这个数字钱包理论上可以带着它穿越物理和数字世界。随着越来越多的品牌将产品和服务代币化，你的数字钱包将继续充满各种形式的代币，并且代币在品牌和跨品牌中的应用将会扩大。

3 区块链网络的力量

如果用户拥有 DAO 代币，则有权访问包括票务、虚拟世界名人 / 运动员见面会、物理 / 虚拟时装秀、元宇宙差异化的购物体验、独家访问产品、免费产品兑换、实物 x 数字产品、查看 / 消费数字内容、新闻订阅、博客 / 播客访问、视频流、音乐流、无广告内容、参与 / 投票 DAO 社区、独家社区聊天、投票权、DAO 或基于代币的社区管理和治理。

许多品牌已经在 Decentraland 和 The Sandbox 中建立了 NFT 品牌，以及使用代币等来访问现实生活中的事件，Shopify 使品牌能够解锁以社区为中心的商业，TIME 将数字艺术代币所有权与他们的数字内容联系起来，几乎每个 DAO 都使用代币所有权作为访问社区的一种形式。

通过代币实现可验证的数字所有权，DAO 现在可以与用户建立更好的数字关系。虽然存在风险（钱包安全，欺诈性应用程序，代币诈骗等），但 DAO 存在巨大的机会，可以为其忠诚的用户创造独特的体验和产品，与现有的基于代币的产品和公司合作和合作，并创建有意义的 DAO 社区，鼓励参与并灌输个人代理的新鲜感觉。

DAO 是不同人的分组，朝着一个共同的目标努力。以利润为目标的 DAO 开始使用 NFT 的服务，并创造更好的用户体验。Web3 产生了伟大的创新，吸引了新的人才，并推动了范式转变，这是一个很长的路要走。但是，我们都可以通过提出正确的问题来质疑现有的解决方案，从而提升协议性能。
