日本NFT圈子的独特：地域性与日本文化

不同的国家有着不同的文化，文化的差异使得不同国家的 NFT 玩家群体有着不同的特色。 这些特色差异体现在各个方面，比如项目类型上的偏好、审美上的偏好、「钻石手」的动机乃至对 NFT 的看法等等。

由于母语的不同，在各个 NFT 项目 Discord 服务器的 general（通用聊天）频道中，我们不太容易能感受到这种文化差异造成的群体特征，但只要对不同母语频道的聊天内容进行观察，就会有比较直接的感受。本次，律动 BlockBeats 与一些来自日本的 NFT 玩家进行了深入的交谈。这些接受采访的日本 NFT 玩家中既有链游公会的创办者，也有投资 Crypto 超过 8 年的 Bitcoin OG。既有经验丰富的 Discord Mod，也有普通的年轻玩家。通过与这些日本 NFT 玩家的交谈，我们得以一窥日本 NFT 玩家群体的特色。

「慎重」

投资 NFT 有着较高的学习门槛，这一直是制约新玩家进入 NFT 世界的因素之一。日本人的「慎重」让不少日本人依然对 NFT 保持观望：

「我相信有不少的日本人会说，这个 NFT 很酷，我想购买它！但是，什么是 MetaMask？什么是 OpenSea？要上手其实并不容易。同时，他们也会担心，我会不会遭遇一个可怕的骗局？」

同时，NFT 圈的主要信息流以英文为主，更对日本玩家的信息获取带来了一定的难度。因此，「本土化社区」对日本 NFT 玩家来说相当重要，这也是 NinjaDAO 成功的原因之一——日语社区既方便了日本玩家获取信息以对投资标的做出判断，也能够使他们在交流中「享受社区」。

除去影响力主要在海外的 Murakami Flowers，NinjaDAO 的 CryptoNinja Partners 就是日本本土 NFT 项目的 No.1。22222 的总量挂单率却仅有 0.8%，加上接近 3 ETH 的地板价，这是十分优异的市场表现

「享受社区」对 Discord 服务器的运营有着更高的要求——不仅是某个特定 NFT 项目的信息中心，更是玩家们交流的平台，如同一个大的 BBS，板块多且架构复杂，在「群体交流」间消弭「个体慎重」。以 NinjaDAO 为例，非日本的 NFT 玩家刚进去会觉得比较迷茫：

低价乃至 Free Mint 也是化解「慎重」的一种策略。日本 NFT 项目的发售价低于 0.005 ETH 是相当普遍的（比如上面提到的 CryptoNinja Partners，发售价格仅为 0.001 ETH），并且常以老项目的 NFT 作为筛选条件来发放新项目的白名单。部分受访者以一种比较新奇的角度来解释这种策略的有效性，他们称之为「试用」，并提到了日本独特的「宅男文化」：

「在日本，有一种文化，即购买擅长绘画的业余作者的画作，这就是所谓的宅男文化。对有此爱好的收藏者而言，购买 NFT 还是实物画作，唯一的区别可能仅仅是用日元或用 ETH 进行支付。但 NFT 毕竟比较新奇，在购买 NFT 的人群中也有许多以赚钱为目的的消费者。因此，在喜爱画作的前提下，以低成本试水，有相当的吸引力。如同在超市看到一个陌生的新品牌麦片，购买一个小份的试用装进行品尝。如果这个 NFT 之后的各方面体验都很好，玩家就会持续投资。」

此外，日本 NFT 项目对线下活动的开展以及线下影响力的展示相当注重。受访者们认为，虽然不能确认这是项目方对症下药的举措，但这的确能对「慎重」起到不错的成效：

ShikibuWorld 提及自己的 IP 在 2021 年被长崎县警方任命为「犯罪预防大使」，目前，该系列的地板价为 0.5 ETH，挂单率仅为 1%

NinjaDAO 的线下活动 CryptoNinja Festival 2022 丰富多彩，除了周边贩售与创作者见面会等常规活动，还进行了集卡大赛、任天堂明星大乱斗锦标赛以及节目丰富的现场秀。为鼓励亲子共同参与，还专门准备了儿童娱乐空间、换尿布室甚至帮忙照看小朋友的人员

「偏好」

大部分受访者称，他们开始接触 NFT 是因为 StepN。而在 StepN 的赚钱效应减弱之后，日本玩家对「XX To Earn」类型的项目依然有着很高的热情。「Play To Earn」、「Move To Earn」甚至是「Sleep To Earn」，日本玩家希望寻找到下一个如 StepN 般现象级的「XX To Earn」项目。一位受访者说，在日本玩家对「XX To Earn」项目的热情最高涨的时候，甚至有部分未成年人参与其中。

良莠不齐的新项目让一些玩家感到失望，导致这样的热情在一定程度上减退。受访者们普遍有着这样的态度：

「市场中有太多美丽但无法或根本不愿意交付的故事。我并不排斥项目初期的 Ponzi 阶段，但需要看见项目方是否对短期能且应该交付些什么、对长期想要实现什么有着清晰的规划。现在，我希望看见用心制作的精良产品，这在很多方面可以进行判断，比如 UI/UX 的美观度、使用的流畅度等等。只要我能在使用中收获乐趣，我愿意做需要等待的长期投资。」

谈及 PFP，较多受访者表示并没有太大兴趣。并且，「炒图」这个概念对他们来说比较陌生，更多还是出于收藏的兴趣，遇到自己喜欢的作品，只要价格合适，会愿意购买。虽然缺少数据支撑，但是大部分受访者都以比较肯定地认为，日本 NFT 玩家中「XX To Earn」爱好者的数量比 PFP 爱好者的数量更多，且 PFP 爱好者更愿意花钱，「XX To Earn」爱好者则倾向于「零撸」。

日本玩家对 PFP 的审美也有偏好。绝大部分受访者对 CryptoPunks 与 Bored Ape Yacht Club 的画风毫无兴趣，其中一位受访者更是直言：

「我还没遇见过喜欢 CryptoPunks 或 Bored Ape Yacht Club 的日本 NFT 玩家… 出于赚钱的目的，当然还是会有人去买的，但是画风真的不合我的胃口，或者可以说，不合日本人的胃口。日本人更喜欢日漫风格的设计。」

如果让受访者们从现有市场的蓝筹项目中挑选一个，Azuki 是绝大多数受访者的选择。从日本 NFT 项目的历史成交量排行来看，日漫风格的 NFT 的确居多：

此外，日本本土出品的 NFT 系列更受日本玩家的青睐。70% 的受访者购买的第一个 NFT 是日本出品，另有 30% 仅买过日本出品。他们表示，日本出品的 NFT 更让他们安心，也更受他们喜爱。安心的原因是管理团队当中有日本人能够给他们一种亲切而熟悉的感觉，同时也没有语言问题，能够了解得更深入。喜爱则是上面所述的画风偏好。

「佛系」

在社区中，日本玩家的「佛系」表现令我印象深刻。我曾看见过这样的发言：

「在 StepN 当中一天赚 10 美元也是相当不错的。收获健康的同时还有一些收入，挺满意的。」

我以为这样的发言可能只是个例，但在与受访者们的交谈中，发现持有如此观点的受访者并不少，但他们有着不同的前提条件。有的人认为，只要项目靠谱能够被他们视为可靠的长期投资，10 美元一天的收入的确能够令他们满意。有的人则认为，这样的观点是建立在低成本/成本已经回收的基础上的。但尽管有着这些前提条件，我还是觉得日本玩家要更「佛系」一些。

另一个「佛系」的点在于，日本社区内较少出现 FUD，更多时候展现出对项目的支持。对这一点，部分受访者认为这可能是一个「国民性问题」：

「日本玩家确实更倾向于对项目表达支持，至少很少直截了当地去批评一个项目。这可能与日本人的性格有一些关联——除非有人带头挑明了普遍的不满，否则大家还是会在批评上有所保留，大家不愿意做坏人，或担心发表不合群的意见，这种倾向在 NFT 新手与不具有影响力的社区成员身上更明显。」

那么，这是日本玩家多「钻石手」的原因吗？也不尽然。至少，从一些受访者给出的回答来看，他们的「佛系」比我想象得还要纯粹：

「您向我提及了 NinjaDAO 的低上架率现象。在他们的社区里，HOLD 是那里的共识，他们做到了令日本以外的 NFT 玩家都瞩目的水平，这对日本 NFT 将影响力扩展到海外是件好事。但是，我并不太喜欢被我必须要 HOLD 的声音围绕。随时买卖您喜欢的东西。这是您的自由——NFT 本该如此，为什么一定要强调 HOLD 呢？如果我有兴趣，我就会买入它，享受持有它的过程。如果我失去了兴趣，我会离开。厌弃出售曾经持有的 NFT，真的比较魔性。当然，这样的共识建构，与他们的 Founder Ikehaya 先生自身的影响力与受信任度密不可分，他有着忠实的追随。」

另一个有趣的点是，Discord 中的日本成员不太喜欢被频繁地 @everyone，但是关闭 Discord 通知又可能会让他们错过重要的新信息，这让他们感到苦恼。解决的办法也很简单，为他们设置一个接收通知的专属角色，会让他们感到妥帖。

结语

@zenryoku_eth 在去年的 6 月发布了一项统计，在对持有者地址进行去重以后，估算出的日本 PFP 购买者大约在 8000 人。而与日本 NFT 玩家们的交谈让我相信，日本的 NFT 市场还有着很大的开发空间。CryptoNinja Partners 的成功已经迈出了日本 NFT 出海的第一步，越来越多的人因为这个大获成功的项目而关注到了方兴未艾的日本 NFT 市场。其发行方 NinjaDAO 目前占据了 PFP 赛道的王座，但随着更多的增量资金进入，也许会有更加翻涌的后浪出现。不过，「更加翻涌的后浪」很可能还是从日本本土出产的 NFT 当中诞生。对不同国家的 NFT 市场，只有针对其文化去制定合适的「本土口味」运营策略，才能更好地被用户所喜爱和信赖。

日本 NFT 项目的社区运营策略也给到我相当大的启发。在过去，我们更多着眼于单个 NFT 项目的成功。但在发售完成以后，社区内部「凝聚力」与社区外部「注意力」的维持往往是一个很大的难题。由于长期愿景的实现不好给到用户直接而频繁的感受以增强信心，在增量资金不足、叙事吸引力降低的当下，想让用户将买入 NFT「长期投资」是困难的。如果说在牛市时是「买叙事」，那么现在则更多是「买社区」，日本的 NFT 项目正向我们展示了通过低成本、大社区规模以及「朋友关系」的构建来淡化「投机意识」的解决方案。这个解决方案里还有很多细节值得我们去揣摩。毕竟，仅凭频繁的聊天或简单与传统的社区活动，依然很难做好「朋友关系」的构建。

在文章的最后，我想对所有与我进行了交谈的受访者表达深深的谢意，也借用一位受访者在采访的最后对我说的一句话：

「感谢对 NFT 共同的热爱，让我们相识。」
