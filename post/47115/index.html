过去几周最热门的上升项目无疑是 Canto——一个与 EVM 兼容的区块链，内置 DEX、借贷市场和稳定币。

以下是你需要了解的关于 Canto 的所有信息、从哪里开始以及为什么你应该了解这个新兴的生态系统。

Canto 初学者指南

图文：Logan Craig

Canto 是一个基于 Cosmos 的新兴区块链，目前在 DeFi 和 NFT 领域都引起了广泛关注。这个 Bankless 策略将向你展示如何在 Canto 生态系统中开始。

  • 目标 ：学习如何使用 Canto

  • 技能 ：中级

  • 努力 ：不到一小时

  • 投资回报率 ：在最热门的 Alt L1 上获得早期机会

什么是 Canto？

Canto 于 2022 年 8 月推出，是一个相对较新的基于 Cosmos 的区块链，与以太坊虚拟机（EVM）兼容。 

Canto 的独特之处在于，DeFi 最重要的原语——去中心化交易所、去中心化借贷和去中心化账户单位——作为免费公共产品被原生地内置在其链上。它们分别是 Canto DEX 、Canto Lending Market 和 $ NOTE 加密货币。

Canto 提供的另一项创新是合约共享收入（CSR），通过它，Canto 智能合约的创建者可以从用户与其项目交互产生的所有交易费用中获得分成。

为什么 Canto 很重要

在许多其他 Alt L1 停滞不前或苦苦挣扎的时候，Canto 正在为 Web3 实验者提供新的肥沃土壤，例如 DeFi 构建者和 NFT 创造者，因为他们对合约级收益分享的可能性感兴趣。

Canto 的 DeFi 生态系统目前在加密经济中排名第 16 位，TVL 为 1.78 亿美元。如今 Canto 上只有少数应用程序在运行，因此 Canto 的 DeFi 场景有很大的发展空间。 

同样，Canto 的 NFT 领域目前还处于起步阶段，尽管它最近一直在加速发展——其 NFT 交易量统计数据已经开始与一些基于以太坊的 NFT 市场上看到的数据相匹配。 

尝试使用 Canto 需要什么

1.? 一个钱包 ——因为 Canto 是 EVM 兼容的，常规的以太坊钱包就可以使用

2.? 网络支持 ——使用 Chainlist 工具将 Canto 网络添加到你的钱包，网络 ID 为 7700

3.? Starter CANTO—— 就像你需要 ETH 在以太坊上购买 gas 一样，你需要 CANTO 来支付 Canto 上的交易；在这些 CANTO 不可广泛交易的初始时期，你可以通过 Canto Discord 中的“社交水龙头”频道或通过 CANTO Gas Station （或尝试 Synapse，下面会详细介绍）获得你的第一个代币。

如何过渡到 Canto

你可以使用 Alt L1 本地 Canto Bridge 分类账的“双桥”系统将 USDC、USDT 或 WETH 从以太坊转移到 Canto 区块链上。

如果你以前从未使用过 Canto，在桥接之前，你将首先被要求生成一个 Canto 公钥。请记住，在你存入代币后，需要 96 个以太坊区块或大约 20 分钟，你的代币才会出现在桥中。 

一旦你的代币确实出现在桥中，你将需要单击 Canto Bridge UI 中的第 2 步以完成你到 Canto EVM 的第二次桥接，这是可以与 Canto 的免费公共产品和第三方社区应用程序进行交互的地方。如果你在这里遇到任何困难，请使用这个出色的视觉演练来推进。 

如果你更喜欢更快、更传统的桥接体验，请注意 Synapse Bridge 支持将 USDC 和 USDT 转移/交换到 Canto。现在，Synapse 甚至为每个 Canto 桥接分配 1 CANTO，这样你就可以在另一侧为你的第一笔交易提供燃料。 

Canto DeFi 聚焦：Slingshot

Slingshot 是一个去中心化交易所，支持一系列 L1 和 L2 交易，包括 Arbitrum、BNB Chain 和 Canto。 

Slingshot 流畅的 UI + UX 使其成为根据需要交易到 CANTO的简单途径，例如，如果你已经通过 Synapse 将 USDC 转移到 Canto，然后想将该 USDC 交换到 CANTO 以换取即将到来的 NFT drop。

此外，根据 Bankless Ultimate Guide to Airdrops ，该项目也仍然是无代币的，因此是一个潜在的持续空投机会。 

Slingshot 目前正在运行 Canto 交换促销活动，该 DEX 移动应用程序的用户可以获得多达 100 个 CANTO 代币。 

Canto NFT 聚焦：Alto

Alto 是一个零费用和以爵士乐为主题的 NFT 市场，它已成为年轻的 Canto 链上 NFT 活动的中心。 

你可以使用该平台搜索热门和趋势收藏、买卖 NFT 以及铸造你自己的 NFT （目前仅支持 1/1 ）。

请注意，Canto Longnecks 是目前 Canto 上最珍贵的 NFT，因为它们是 Canto 的旗舰系列；在撰写本文时，Longneck 的底价为 4450 CANTO（约合 3000 美元）。 

最近查找新的 Canto NFT 项目的一种简单方法是定期扫描 Alto 主页的“近期销售”部分。此处你可以在铸造时找到像 Cantographs 这样的标志性收藏品。 

Canto 实验才刚刚开始

Canto 是区块链生态系统中的一个有趣实验，它带来了新的想法。

Canto 的智能合约创建者收入分享系统将值得关注，因为它可以激励高质量去中心化应用程序的开发。此外，Canto 致力于将金融基础设施作为公共产品培育，这无疑是朝着更公平的金融体系方向迈出的一步。

如果这个实验性的区块链能够继续获得巨大的吸引力，那么现在进入 Canto 的底层可能会在以后获得回报。
