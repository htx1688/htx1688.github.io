作者：JIANG

摘要

DWeb Camp 有一个独特的文化口号：JOMO (Joy Of Missing Out，错失之乐) 。跟 FOMO (Fear Of Missing Out) 文化相反，在这样一个如此丰富如此多元的盛会中，得到有趣东西的同时，难免会错失其他精彩，且将这样的错失视为乐趣的一部分，享受当下。

今年再次跨洋来到美国旧金山，参加  DWeb Camp 的活动。DWeb 就是 Decentralized Web，主要关于去中心化的网络技术，这个社群历史悠久，与开源软件社区、密码朋克浪潮、反文化运动都有交集。主办方互联网档案馆 (Internet Archive) 是位于旧金山的一个数字档案馆机构，保存了从 1996 年以来的互联网大部分公开网站的快照，以记录和存储所有人类知识为使命，在技术社区备受尊重。

几年前也曾来过  DWeb 大会，那时在旧金山市区，会议规模不大，但是聚集了 Vint Cerf (TCP/IP 协议设计者，互联网之父，图灵奖得主)，Tim Berners-Lee (万维网之父，图灵奖得主)，Whitfield Diffie (密码学大师，图灵奖得主)，Joseph Poon (Lightning Network 创始人)，Juan Benet (IPFS & Filecoin 创始人)，可谓群星闪耀。

今年的活动是露营形式，地址在加州北部的 Navarro 营地。我们置身在巨大的红杉林里，到处是十几米到几十米的高耸笔直的红杉树，很壮观，我们都沉浸在这样原生的自然环境中讨论着各种议题。我本来也关注了很多艺术、文化和社会主题，但来了之后一到技术讲座就不想走了，简直来到快乐老家。吃的东西供应也很充足，食物健康又好吃，三天时间一边聊一边吃嘴就没停过。第一次去露营，准备不太充分，结果很多要用的东西没买，买的东西大部分没用上，现场全靠朋友们接济。

（高耸的红杉）

这次 DWeb Camp 最热门的技术是 OCap，CRDT 和 Gossip。OCap (Object Capability) ，一种基于对象的权限管理模型，具有可组合性、可传递性和强大的表达能力，允许细粒度的权限控制，也是我最喜欢的编程概念之一，而常见的 RBAC/ABAC 机制是没有这样灵活性的。Deno 和 seL4 这两个著名系统都有用到OCap。

最近出现的 UCAN (User-Controlled Authorization Network) 正是基于 OCap 的，通过数字签名 Capability 表达用户对资源的授权，作为去中心化的权限管理方案，并且跟现有 Web 标准有很好的兼容性和互操作性，在这次大会中多次被提及，看到大规模应用的曙光。

跟 MetaMask 创始人 Dan 聊天，他们在钱包安全方面也采用了 OCap 的相关技术，开发了 Lava Moat ，保障软件供应链安全性。就是在引入每个 JavaScript 依赖包的时候，通过白名单声明允许这个包访问什么资源，避免引入恶意的包影响整体安全性。

CRDT 是无冲突复制数据结构 (Conflict-free Replicated Data Type) ，也是很热门的研究领域。当数据去中心化传播的时候，数据同步或取得共识的成本是很高的，依然在探索中的是，如何在不需要共识的前提下尽可能实现更多功能，表达更丰富的语意，比如实现内容发布、社交网络、协同编辑，而不依赖中心化的节点或者区块链。这次遇到 GUN DB 作者 Mark，Braid 的作者  Michael  和 vlcn 的作者  Matt，都做了很多工作，看到这个领域的高速发展非常兴奋。

Gossip 就是流言算法，在分散的、随机的、未知规模的网络结构中进行信息的传播，维持网络的  liveness，是非常重要的算法。这次也有很多的讨论，我做了一个技术分享，关于 Gossip 算法的性能瓶颈，还有 Scuttleutt 和 EBT 算法优化，类似的算法 Secure Scuttleutt 和 IPNS都在使用。跟瑞士巴塞尔大学的 Chris 教授做了交流，他在探索蓝牙通讯的信息传播，我们彼此分享了些很重要的想法。

除了网络方面，这次还有一个 OpenAI 赞助的分会场，探讨 AI 技术的安全性、社会影响，监管问题。随着  ChatGPT 的出现，AI 的影响已经是不可忽视了。AI 研究者 Aza 做了一个非常精彩的分享，关于如何用 ML 算法解析动物语言，探索跨物种的交流，NLP 技术不光让我们解析语言，也帮助我们理解语言的结构和不同语言间的共同性，这些分析工具可以延伸到动物的语言中，了解它们的交流模式，比如破译鲸鱼的语言。鲸鱼不光有自己的语言还有自己的“流行文化”，它们自己的歌声会互相传播，传递到数千公里之外，Aza 戏称“whale wide web”。这里有大量问题有待研究，AI 算法将会起到非常重要的作用。Aza 的另一个讲座是 AI Dilemma，关于 AGI 可能带来的社会问题，以及这种新的力量将如何被资本主义和超级霸权捕获。特别是我们已经经历社交媒体被平台垄断之后的剧本，而如今社交媒体和注意力经济渗透到了社会的各个方面，其影响无法隔离。在这个历史选择的关头，人类是否会重蹈覆辙呢？Aza 说，他的两个讲座，一个关于 Hope，一个关于 Fear，算是一种对冲。

这次Camp 里还有大量诗歌、音乐、手工制作、舞蹈、瑜伽、太极的活动，处处体现着人文关怀。很多人带着伴侣和孩子过来参加，变成家庭出游。也有很多中年和老年人，不少是互联网早期的参与者和开拓者，参会的年龄跨度非常大。万维网之父 Tim Berners-Lee 这次也在，跟很多人深度交流，并且很认真地听分享、提问题，老爷子还在学习。还有一个很优秀的古典乐志愿者团队，拉小提琴，每次集会都进行演奏，美妙的音乐回荡在林间，为我们带来极大的愉悦，水平不输给正式的交响乐团，更何况在如此优美的环境里，宛如天籁。有天晚上活动是去山上观星，无奈天公不作美，是个大阴天。乐团即兴演奏起来，我们躺在草坪上，闭上眼睛，变成夜间音乐会，无比的陶醉。

（欢乐的营地）

有来参会的朋友，发现这里对 blockchain 和加密货币普遍持怀疑和批评态度，其实也提供了对互联网未来的另一个视角。他随机去问一些开发者对 blockchain 怎么看，我在旁听，Solid 的开发者 Jackson 说区块链领域太多泡沫，影响人们对去中心化技术的理解；ICO 本身不道德，并且没有保护到普通人，造成了不少损失。而且很多时候 web3 并不是最好的解决方案，不是什么都需要上链，需要共识，需要金融化，它只是整个技术架构蓝图的一环。他说的这些，也引起我们的反思，如何增强 Web3 的 utility，如何解决人们真实的问题，什么要上链什么不需要。当我们正在聊天的时候，Tim Bernes-Lee 老爷子走到旁边一架钢琴前，旁若无人弹奏起来，成为我们的伴奏，独自陶醉。一个帮助全世界连接起来，并且将这个技术无偿分享给世界的人，还在为网络的未来呐喊，独自奋战着，沉浸在延续几十年的梦中。

Camp 结束后，我参加一个 CRDT 社区的线上会议，很多人也分享这次来参会的体验，说很少有一个地方可以如此轻松 say hi，看到其他人在做类似事情，觉得多了很多盟友。有人说  We see the same mountain, but climb in different ways (我们看见同样的山，只是以不同的路线攀登) 。这是 DWeb 社区令人印象深刻的地方，这是一个共同的社区，大家互相学习。聆听和分享一样重要，而不需要去“表演”或者抓取所有人注意力。即使你关注的技术很小众（比如 OCap，ActivityPub），这里也有人跟你讨论，你会发现自己并不孤独。

跟哈佛大学 Berkman Klein Center 的 Doc Searls 聊天时，我分享了自己的一个长期的小众想法， 称之为 User Side Intelligence 。在使用互联网服务的时候，那些推荐算法、应用程序虽然为我提供信息和服务，但不一定是为了我的目的，比如 YouTube 推荐视频，让我想看、想看、再想看，核心还是服务于平台，抓取我注意力，让我消费和沉迷，它的 boss 还是平台。一个产品和服务，会产生用户侧价值（User Side Value）和平台侧价值（Platform Side Value），经常是不一致的。我希望有一个软件或者智能体的形式，可能是一个浏览器插件，能够代表我的价值观，去跟不同平台、服务和数据源沟通，综合给我推荐内容，使之符合我的需要，相当于我的经纪人。这种 User Side Intelligence 必须以我的价值作为目的，我也可以为它提供个性化数据，跟随我个人生命的发展。我的想法跟 Doc 做的 Customer Commons 也有共通之处。

（与互联网先驱者们的炉边谈话，中间是 Tim Bernes-Lee）

DWeb Camp 关注于人，关注技术对人的影响。我们自己在做去中心化社会协议 Social Layer ，强调 Subjectivity，即人的主体性。这里的  Social 不是关于 Web2 意义上的社交，而是关于人和人如何去信任和协作。必须将人放在中心，人既是能动的主体，也是技术影响的承受者。一方面需要拓宽技术的边界，一方面探索技术如何为人服务。Social Layer 的探索将不局限在 Web3，也可以是 Web2、DWeb 和线下的生活，而开放、协作、以人为中心 才是不变的关键词。Social Layer 也是指社会层面，对 Web3 来说，有很多个面向，Data Layer 和 Incentive Layer 是重要的，Social Layer 也是重要的，但还远未被探索。

DWeb Camp 有一个独特的文化口号：JOMO (Joy Of Missing Out，错失之乐) 。跟 FOMO (Fear Of Missing Out) 文化相反，在这样一个如此丰富如此多元的盛会中，得到有趣东西的同时，难免会错失其他精彩，且将这样的错失视为乐趣的一部分，享受当下。议程安排也很松散，比较自在，让人不感觉被社交消耗。

这次 DWeb Camp 的核心主题是 Tomorrow，这里汇聚了互联网的先驱者、当下的建设者、未来的创新者，还有孩童，大家一起想象互联网的未来，通过技术创新，以及我们的团结和智慧，是否能修复存在的问题？闭幕时，九岁和十三岁的一对兄弟，在台上朗读诗歌，分享他们眼中的 Tomorrow，里面有社会不平等问题，有气候暖化，还有 AI 的影响。所以我们能够创造一个更好的互联网，给自己许诺一个更好的未来吗？
