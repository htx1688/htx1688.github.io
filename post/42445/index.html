新的去中心化 NFT 市场 sudoswap 最近公布了其 $SUDO 代币计划。

就像我们之前在 DeFi 和 NFT 中多次看到的那样，该协议的构建者正在通过追溯空投将其治理代币的一部分分配给其早期的流动性提供者。

如果你有兴趣从自己的 NFT 活动中获得更多这样的空投资格，一个简单的起点是尝试尚未发布治理代币的 NFT 项目，并且可以想象在未来保持这样做。

在这篇文章中，我创建了一个似乎符合这些基本定位条件的项目列表。让我们深入探讨一下，并把它们放在你的观察表上！

15 个潜在的 NFT 治理代币空投机会

管理期望

在以下所有项目中，我认为只有两个项目宣布了明确的治理代币计划。我列出的所有其他项目只是因为他们还没有治理代币，而且他们的设计元素似乎很适合未来的社区治理。

可以肯定的是，这不是最精确的定位方法！下面的一些项目可能永远不会推出治理代币，或者不会像以空投 ERC20 代币那样的形式。在过往的经历中，许多项目都会在某个时候通过 ERC20 治理代币来接受社区治理，因此假设下面列出的至少几个项目会这样做并不疯狂。

综上所述，尝试这些项目时要知道，除了获得支持开创性技术的满足感和构建 NFT 技能树的好处之外，你可能永远不会得到任何东西。如果您将来真的收到来自它们的空投，那将是锦上添花。现在让我们来看看这些候选项目，我将它们分为三个类别：市场、集体收藏和 NFTfi。

? NFT 市场

1. Zora

描述：Zora 是一种通用媒体注册协议，可让人们轻松购买、出售和管理 NFT。

资源：网站 | 推特 ‌| Discord‌

为什么选择 Zora：Zora V3 拥有协议「Fee Switch」NFT，即 ZORF，目前归 Zora DAO 所有。如果 Zora 选择向更广泛的社区开放 ZORF，它可能会使用治理代币来这样做。

潜在的空投策略：在 Zora 上创建和收集 NFT！

2. Nifty Gateway

描述：Nifty Gateway 是一个 NFT 市场。

资源：网站‌ | 推特‌ | Discord‌

为什么选择 Nifty Gateway：Nifty Gateway 刚刚推出了新的 Publishers 系统，该系统正在向社区管理开放平台。我推测该项目最终可能会采用治理代币来进一步去中心化这种管理模型。

潜在的空投策略：在 Nifty Gateway 上收集 NFT！

3. Foundation

描述：Foundation 是一个 NFT 市场。

资源：网站‌ | 推特‌

为什么选择 Foundation：Foundation 的「常见问题解答」版块说，该项目「目前没有代币」。对我来说，「目前」是一个有效的词。该措辞表明 Foundation 今后可能会推出自己的治理代币。

潜在的空投策略：在 Foundation 上创建和收集 NFT！

4. Gem

描述：Gem 是一个 NFT 市场聚合器。

资源：网站‌ | 推特‌ | Discord‌

为什么选择 Gem：当今年早些时候 OpenSea 收购这个 NFT 聚合器时，人们认为 Gem 推出自己的代币的机会破灭了。消息传出后，Gem 联合创始人 Vasa 表示，代币计划仍然「不是不可能的」，从那以后我们还没有听到任何关于代币的其他消息，所以这种可能性似乎是存在的。

潜在空投策略：扫货 Gem 上的 NFT！

5. Quix

描述：Quix 以前称为 Quixotic，是第一个基于 Optimism 第二层 (L2) 扩展解决方案的 NFT 市场。

资源：网站‌ | 推特‌ | Discord‌

为什么选择 Quix：Quix 可能希望通过治理代币为其社区注入活力，最终巩固其作为 Optimism 的先行者 NFT 市场的领先地位。

潜在的空投策略：在 Quix 上收集 NFT！

6. Stratos

描述：由构建 Quix 的同一团队创建，Stratos 是一个部署在 Arbitrum L2 上的 NFT 市场。

资源：网站‌ | 推特‌ | Discord‌

为什么选择 Stratos：Treasure 团队打造的 Trove 是当前 Arbitrum 上的主要 NFT 市场，而 Trove 已经拥有 $MAGIC 代币，因此 Stratos 最终可能也会发布自己的代币，以从其最大的竞争对手那里抢夺更多的市场份额。

潜在的空投策略：在 Stratos 上收集 NFT！

7. Protec

描述：Protecc 是一个即将推出的 NFT 流动性协议。

资源：网站‌ | 推特‌

为什么选择 Protecc：虽然 Protecc 尚未在主网上可用，但很快就会上线。由于 NFT20、NFTX 和现在的 sudoswap 等其他流动性协议都有自己的原生代币，因此有理由相信 Protecc 也会采用同样的方法。

潜在的空投策略：Protecc 推出后在该平台上买卖 NFT！

8. fx (hash)

描述：fx (hash) 是 Tezos 区块链上的一个生成艺术平台和 NFT 市场。

资源：网站‌ | 推特‌ | Discord‌

为什么选择 fx (hash)：fx (hash) 曾计划推出 DAO 和 FXH 代币，尽管这些计划目前已暂停。代币分发细节尚未公布，因此今天使用该平台可能会让您在未来获得 FXH 空投。

潜在的空投策略：在 fx (hash) 上买卖 NFT！如果您不知道从哪里开始，请查看我写的 Tezos NFT 快速指南。

? 集体收藏项目

9. Tessera

描述：Tessera 以前称为 Fractional，是一种集体拥有和管理 NFT 的协议。

资源：网站‌ | 推特‌ | Discord‌

为什么选择 Tessera：Tessera 的核心是集体所有权的概念。朝着这个想法，该项目最终可能通过治理代币使协议本身集体拥有。

潜在的空投策略：一旦项目的新 UI 启动，在 Tessera 上买卖部分 NFT！

10. PartyBid

描述：PartyBid 是一个 dapp，它可以让一群人轻松地共同竞标某个 NFT。

资源：网站‌ | 推特‌

为什么使用 PartyBid：PartyBid 由 PartyDAO 构建，通过 $PARTY 代币进行管理。如果 PartyDAO 决定进一步扩展，它可能会通过向公众发放 $PARTY 来实现。

潜在的空投策略：通过 PartyBid 收集 NFT！

NFT 金融（NFTFI）

11. Arcade

描述：Arcade 是一个 NFT 借贷协议。

资源：网站‌ | 推特‌ | Discord‌

为什么选择 Arcade：Arcade 目前不收取任何协议费用，但该项目指出未来可能会收取费用。如果发生这种情况，Arcade 可能会使用治理代币，以便其社区可以决定这些费用参数。

潜在的空投策略：在 Arcade 上借贷！

12. Astaria

说明：Astaria 是一个即将推出的 NFT 借贷协议。

资源：网站‌ | 推特‌

为什么选择 Astaria：「我们相信 web3 平台依赖于他们的社区，」Astaria 的团队上个月表示。「请务必留意我们在发布后为社区制定的计划。」

潜在的空投策略：一旦上线就在 Astaria 上进行 NFT 借贷！

13. Hook

说明：Hook 是一种新的 NFT 原生期权协议。

资源：网站‌ | 推特‌ | Discord‌

为什么 Hook：根据项目的文档，Hook 的协议「有可能在未来升级为时间锁甚至治理合约」。也就是说，如果 Hook 最终推出治理代币，请不要感到惊讶。

潜在空投策略：在流动性允许的情况下，在 Hook 上购买 NFT 看涨期权！

14. MetaStreet

描述：MetaStreet 是一个针对 NFT 的去中心化利率协议。

资源：网站‌ | 推特‌ | Discord‌

为什么选择 MetaStreet：该项目的登录页面调侃称，未来将需要治理来确定哪些新的 NFT 是否可信。

潜在的空投策略：向 MetaStreet 的任一金库存款！

15. Sodium

描述：Sodium 是一种 NFT 借贷协议。

资源：网站‌ | 推特‌ | Discord‌

为什么选择 Sodium：该项目已明确表示计划在发布后发布其 SOD 代币。

潜在的空投策略：项目上线后通过项目借入或借出！
