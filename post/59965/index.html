2023年9月26日，Outlier Ventures宣布Zero Knowledge Base Camp第二期入选项目名单：SurferMonkey、ZKON、zkHub、Absinthe Labs、Biosnarks。

据悉，Outlier Ventures的Zero Knowledge Base Camp加速器每期由五家初创公司组成，目标是使用零知识证明来推动隐私和可扩展性。

Zero Knowledge Base Camp由Outlier Ventures 与风投伙伴 ScalingX、Old Fashion Research 及生态系统合作伙伴 Mina Foundation一起为最有前途的 ZK 初创公司加速。

下面为Outlier Ventures Zero Knowledge Base Camp第二期入选项目简介：

SurferMonkey

SurferMonkey是一个通过零知识证明实现区块链隐私和合规性的即插即用工具。SurferMonkey 设想一个隐私和合规性在区块链生态系统中无缝共存的未来，为安全和私密交易设定全球标准。使命是通过提供尖端的、不可知的去中心化隐私基础设施来增强开发人员、企业和个人的能力，从而促进监管合规性并保护每笔交易中的用户隐私。

SurferMonkey内置互操作性。是DarkWeb3.0 的下一次演变， 可以跨区块链无缝、匿名交互。从 Polygon 到 Arbitrum，无需透露你的出发地或目的地即可发起操作。具有真正的匿名互操作性，由快速 2 秒 ZKP 生成、区块链路径的完全混淆以及无需后端更改的轻松 dApp 集成提供支持。

SurferMonkey的4大技术：通用zkMixer（是一种与多链的agnostic ZKP 技术）、去中心化匿名合规技术（DACT-v2）、通用插件（与 SurferMonkey 集成的 dApp，部署在所有支持的区块链上） 、前端集成（SurferMonkey 基础设施允许通过前端轻松集成）

ZKON

ZKON（Zero Knowledge Oracle Network）是一个无需信任的预言机网络。ZKON 作为安全预言机的去中心化网络运行，促进第三方应用程序的无缝连接。利用零信任架构，它可以实现帐户连接和数据聚合，而无需在单点故障中存储用户的完整身份验证方法。

ZKON可以连接和证明web3钱包和web2 帐户，无需保留任何身份验证秘密。

zkHub

zkHub是使用多方计算来快速且隐私保护的生成零知识证明。zkHub利用尖端的多方计算技术将零知识证明生成分发到我们的计算网络。证明生成是一项缓慢且计算成本高昂的任务。zkHub 无缝结合 MPC 和 zk-proofs，以闪电般的速度生成证明。与当前的证明时间相比，体验了惊人的 36 倍改进，使开发人员能够构建比以往更大的电路。

Absinthe Labs

Absinthe Labs利用零知识证明创建抗女巫攻击的智能合约。受机器人保护的 Web3 原生代币水龙头终于成为现实。无需身份验证即可提高开发人员资源的可访问性！

Biosnarks

Biosnarks是一种生物技术协议，利用 zkSNARK 来促进数据交换和信任，而不会泄露流程或 IP 敏感信息。Biosnarks使用 MINA 的 Kimchi Proof System 进行递归证明。

药物和生物技术开发是一个昂贵且缓慢的过程。单一疗法的开发可能需要长达 10 年的时间，费用在 3亿到22亿美元 之间。 罪魁祸首是基础研究的可重复性差以及数据和结果的孤立。 BioSNARKS允许参与者在不泄露敏感信息的情况下与经过验证的数据、IP、协议、目标进行交互，从而为加速和更加民主化的药物和生物技术开发开辟了新的可能性。

可把Biosnarks想象为一个堆栈，即使在联合环境中，它也能实现保护隐私的生物数据和药品市场，以实现潜在客户的生成和优化。
