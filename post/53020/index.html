近期，由波场生态全力打造的策略对战游戏WIN NFT HERO利好频出。继2023年4月份正式推出游戏v2.0版本以及全新的英雄体系之后，WIN NFT HERO在6月1日正式上线了 v2.1公测，并且官方还公告将于6月15日开启S1百万美元赛季活动。

根据官方介绍，WIN NFT HERO S1赛季的游戏场景为PvP积分赛，玩家只要拥有任意3个N及以上品质的英雄即可参与。WIN NFT HERO的S1赛季活动中将包含丰厚的$NFT、$WIN、WNH以及高品质NFT等奖励。整个赛季活动奖励价值预计将超过百万美元。

想要参与WIN NFT HERO S1赛季的新玩家，首先需要的就是拥有英雄NFT。WIN NFT HERO非免费版的英雄一共分为5种品质，从低到高分别为：N（绿色）、R（蓝色）、SR（紫色）、SSR（橙色）以及UR（红色），品质越高代表英雄的综合素质越强。

目前WIN NFT HERO游戏中共包含11个阵容的58个英雄，其中19个是在游戏v2.0推出时加入的新英雄。由于新英雄具备不同的属性、技能和羁绊效果等等，这意味着拥有新英雄的玩家不仅可以尝试更多的策略组合，同时也将在对战中占据更大的优势。

为了庆祝即将到来的S1赛季，助力玩家取得更好的成绩，WIN NFT HERO宣布将于2023年6月16日上线APENFT Launchpad并启动英雄盲盒发售。英雄盲盒数量：200个，售价为10 USDD/个。
