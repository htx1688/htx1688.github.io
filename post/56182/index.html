不靠「奥德赛」，不靠空投造势，Coinbase 所推出的 Layer 2 网络 Base，以一个千倍涨幅的 meme 神话引爆市场，吸引了无数眼球。

14 个小时左右的时间， 2000 倍涨幅，昨天下午 5 点左右，仍有接近 5600 万美元的市值，池内有 8774 枚 WETH。考虑到在上涨过程中大家仍普遍认为 Base 暂时只能将资产跨入而无法跨出（注：其实是可以跨出的，推荐阅读：教程｜手把手教你从 Base 链取回 ETH），这么大的池子与这种容易将人吓跑的「只进不出」认知形成了巨大的反差，让玩家们感到非常不可思议。大家甚至开玩笑说，$BALD 是 Coinbase 的「广告」，这个梗在 Twitter 与各个微信群中广为流传：

聪明的公链：我们有一个一天涨一千倍、一万倍的土狗

愚蠢的公链：我们即将开始一个长达 2 个月的奥德赛活动

但是仅仅过了 3 个多小时，$BALD 就露出了獠牙将一切毁灭。项目方突然在 7 分钟内撤走了总计 8660 枚 ETH 和 1.79 亿枚 $BALD 的流动性，$BALD 的价格大幅下跌，最大跌幅曾达到 80% 。目前，根据 dexscreener 上的数据，$BALD 的流动性资金池内仅剩 12 ETH，Base 网络的 TVL 也受此影响一落千丈，从 4200 万美元左右跌到了 942 万美元。

遭遇大幅亏损的玩家

「冲土狗」，是一个残酷无比的「PvP 游戏」，Crypto 玩家知道这无异于刀尖舔血，却总是在暴富故事的诱惑下前赴后继，把自己的命运交到屏幕对面可能正喝着啤酒好整以暇的匿名项目方们。我试着创造了一个「土狗」，这个过程是如此轻松加愉快，让我感到如此无奈与讽刺…

只花了 14 美元，只点了几下鼠标，一个「土狗」就诞生了

在 Thirdweb 上连接钱包并签名后，点击「Contracts」，再点击「Deploy contract」。

接着我们可以选择要部署的合约类型。这里可供选择的类型很多，无代码一键部署 ERC-20、ERC-721、ERC-1155 Token 都是支持的，甚至支持部署一个带质押功能的 ERC-721 Token、可租借的 NFT ERC-4907 乃至像 OpenSea 那样的交易市场等等。这里我只是简单的部署一个 ERC-20 Token，所以选择「Token」：

接着点击「Deploy now」进入设置页面：

设置页面如下：

- Image：Token 图案。

- Name：Token 名称。

- Symbol：Token 缩写。

- Description：Token 描述。

- Primary Sale Recipient：Token 预售的收入地址。由于我们选择的「Token」合约是不带预售价格的，这里其实也没用。

- Platform fees：Token 预售收入还可以额外加税，此外如果部署的是一个 NFT 市场之类的服务，可以对发生在该合约的所有二级交易抽税。

这些都设置完了以后，在最后选择要部署在哪个网络上，我选择了 Base：

我设置的「Name」是「ZaicheshangBuhuibaHaimeimai 111 Shufule」，「Sybmol」是 SHUFU，「Description」是「GM SHUFULE」，「Image」则是一个舒服了的贴吧表情包。花费 2.26 美元成功部署后，效果如下：

在 basescan 上已经能够看见该合约：

回到设置中心，左边的菜单栏里有「Token」选项，点进去就可以 Mint 给定数量的 Token。我 Mint 了 111111111111111 个，然后把他们和 0.186 个 ETH 放到 LeetSwap 上做了一个资金池。这样，这个 Token 就和 $BALD 一样能够在 LeetSwap 上交易了。

Mint Token 花费了我 0.42 美元，在 LeetSwap 上设置资金池花费了我 11.35 美元。我创造一个「土狗」的成本，只有 14.03 美元，外加手指上点劲来点击鼠标。

随后，我关闭了资金池，并且在 Thirdweb 设置页面的「Permission」功能中扔掉了所有权限，试验结束。

Thirdweb 没能实现买入/卖出交易税功能，于是我搜寻了一番，发现了确实有可以设置买入/卖出交易税的一键发 Token 工具，太智能了…

我无法核实该工具的安全性。不过，该工具帮助一键部署合约的原理大概是这样：这个工具本身就是一个合约，设置完确定部署后，合约会根据设置执行「Deploy Token」操作完成相应设置的部署。我随意挑选了一个用此工具部署的合约进行检测，亮灯情况比 Thirdweb 要好：

结语

我很推荐大家用 Thirdweb 去部署一个自己的 Token 玩看看，虽然只是一个简单的流程，但是亲自体会能感受到一些不一样的东西（千万不要作恶啊）。

诚然，要做起一个百万乃至千万美元市值的「土狗」，还需要做个官网、各种病毒式营销造成裂变等等工作。不过，如果您是和我一样的「Crypto 玩家」，当我们绞尽脑汁去评估一个项目方「格局够不够大」时，这一切是否会让您感到一丝讽刺？
