原文来源：AGI创新实验室

7月 24 日，由 OpenAI 创始人 Sam Altman 联合创立的 Web3 加密项目 WorldCoin 正式宣布上线。Sam Altman 与 WorldCoin CEO Alex Blania 同时发布了联合公告，正式宣布了这个消息。

三年前，我们创立了世界币，其雄心是创建一个人人拥有的新身份和金融网络； 今天开始推出。 如果成功，我们相信世界币可以极大地增加经济机会，扩展可靠的解决方案来区分在线人类和人工智能，同时保护隐私，实现全球民主进程，并最终展示一条通往人工智能资助的全民基本收入（UBI）的潜在路径。

Worldcoin由保护隐私的数字身份（World ID）和数字货币 WLD 组成（在法律允许的情况下，仅有人类可以获得）。 我们希望，在法律规则不太明确的地方，例如在美国，能够采取措施，让更多的人能够从中受益。

您现在可以下载第一个兼容协议的钱包World App，并预订您的份额。 访问 Orb（一种生物识别验证设备）后，您将收到一个 World ID。 这可以让您在网上证明您是一个真实且独一无二的人类，同时保持完全的私密性。

随着Orbs 设备的全球发行量不断增加，您可以通过 World App 和 worldcoin.org 找到最接近的一颗设备并预约时间进行验证。

世界币是全球范围内协调的一次尝试，这个过程将充满挑战，结果也不确定。 但寻找新的方式来广泛分享即将到来的技术繁荣是我们这个时代的一项严峻挑战。 我们希望您能加入我们。

Sam Altman 的又一场全球性实验

WorldCoin是 Sam Altman 最新的一场全球性实验，目前通过 WorldCoin 验证的用户数量已超过 200 万，而Sam 的目标是全人类，同时把加密货币带给全球 10 亿人。

WorldCoin已经筹集了近 2.5 亿美元资金，投资方包括 a16z、Coinbase Ventures 等。

Worldcoin是一个全球性的加密货币项目，其目标是向全球每个人提供免费的加密货币。为了实现这一目标，Worldcoin团队设计了一种名为 World ID 的系统，以确保每个人都能公平地获取他们的加密货币份额，同时防止欺诈和滥用。

World ID是一种基于生物特征的身份验证系统。它使用一种特殊的装置——眼球扫描仪 Orb，通过扫描用户的虹膜来验证其身份。由于每个人的虹膜都是独一无二的，这为构建一个公平且难以滥用的系统提供了可能。

大语言模型正在经历的繁荣，使得AI 与真实人类之间的差异正在缩小。人类证明问题、信息真假和隐私等问题正在变得越来越突出，尤其是人类证明问题。如何确定一个用户是真正的人类而不是计算机程序或机器人的问题？

那么，World ID如何解决 AI 问题呢？以下是一些主要的解决方案：

1.防止身份欺诈

AI常常需要处理身份验证问题，这对于许多应用（如在线银行、电子邮件、社交媒体等）至关重要。然而，传统的身份验证方法（如密码、短信验证码等）容易受到欺诈攻击。World ID 通过使用虹膜扫描，提供了一种更为安全、难以伪造的身份验证方式。

2.数据隐私保护

AI系统通常需要处理大量的个人数据，这可能引发隐私问题。World ID 通过不保存用户的虹膜图像，而是将其转化为一个不可逆的数字标识符，从而帮助保护用户的隐私。

3.提高公平性

AI系统在处理资源分配问题时，可能会出现不公平的情况。例如，一些用户可能会通过创建多个账户来获取更多的资源。通过使用 World ID，每个人只能创建一个账户，从而确保了每个人都能公平地获取资源。

4.防止机器人和自动程序滥用

AI可以创建机器人或自动程序，以模拟人类行为并滥用系统。但是，由于 World ID 需要用户进行虹膜扫描，这使得机器人和自动程序很难滥用系统。

需要注意的是，虽然World ID 在解决这些 AI 问题方面具有潜力，但是它也引发了一些关于隐私和道德的问题，例如，用户是否愿意为了获取加密货币而提供他们的生物特征信息，以及这种信息的使用和存储应该受到何种程度的保护等。

另外，WorldCoin还宣布开放 World ID SDK，现已可供公众使用和集成。
