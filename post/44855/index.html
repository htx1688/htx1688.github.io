作者：XiaoZ@iNFTnews.com

腾讯旗下继腾讯新闻、幻核之后，仅存的数藏平台也关停了。

11月16日，据界面新闻报道，腾讯TME旗下QQ音乐已经叫停“TME数字藏品”业务，原团队部分成员已内部活水。

一接近腾讯集团的知情人士称，腾讯方面曾对数字藏品业务进行过研讨，在综合监管因素及腾讯战略方向后，选择放弃数字藏品。

对此， iNFTnews依旧是拨打了4006016666客服热线，得到的回复与两个月以前的回复一致。

客服表示：

  • TME数藏平台目前没有收到关停的通知，若需要退款可以为其登记。

  • 入口依旧存在，入口在QQ音乐-我的-已购-数字藏品，里面的“TME数字藏品商城”和“数字藏品个人中心”两个页面可查看数字藏品。

登记这个说法很微妙，与其说暗示，不如说已经明示着平台即将关停，而两个月以来也有网友有反馈收到退款。有了退款的案例，许多TME音乐数藏平台的用户也创建了维权群，讨论如何维护自己的权益。

而平台现状处于在“停摆”的状态，banner也是一片空白，再结合之前腾讯旗下的数藏平台接连关停的情况，可以肯定的是，TME数藏平台也难逃关停的命运。

国内数藏已“死”？

腾讯作为行业的巨头，接连关停旗下的数藏平台，对于整个数藏行业来说无疑是重大的打击。

2022年8月16日，国内最大数字藏品平台之一的腾讯“幻核”发布公告称，将停止数字藏品发行，通过平台购买过数字藏品的用户可申请退款。此事在数藏圈引起轩然大波。

作为曾经国内最大的数字藏品平台之一，背靠腾讯的“幻核”，于2021年8月2日上线。因为有腾讯的背书，它受到了数字藏品爱好者们的追捧，每次藏品发售，基本都会在几分钟之内售罄。例如其推出的首款“十三邀”黑胶唱片NFT，开售不到1秒即售罄。“幻核”是腾讯最具潜力的创新业务之一，上线不到一年时间里，总销售额已超8000万元。

好景不长，一年后“幻核”宣布关停也体现出目前数藏市场的冷清。不仅买方市场遇冷，不少平台也宣告关停。随着数字藏品降温，多数台子出现了过山车式的价格暴跌，交易额较最高峰时期下跌90%，藏品价格暴跌99%，曾经一图难求，现在无人问津，难以脱手。

从百度指数来看，数字藏品5月10日的单日搜索指数就高达35万，而11月21日的搜索指数为1412，仅仅为巅峰时期的0.4%。由于数字藏品之前的火爆，从1月至今，数字藏品搜索指数整体日均值仍超1万。但自8月以来，数字藏品的搜索指数日均值还是从3000跌倒了1400。不难看出，数字藏品的正处于持续性的寒冬。

寒冬之下，百度、京东、阿里等大厂旗下的数藏平台均有减少数字藏品的发布。并且，不同于以往数藏高昂的售价，如今各个平台的数字藏品处在50元以下区间的就占了7成，在没有太多利润的情况下，许多中小型数藏平台选择了关停。

据统计，自9月以来，就有鲸起Art、华宇数藏、幻海数创、元境Meta、元狐艺术艺数链、巨象文创、敏捷文创等30多家中小型数藏平台因行业风险、低估寒潮等原因关停。

有的数藏平台会直接跑路，解散用户群，而有良心的数藏平台则是会给用户退款。

如梵核艺术，上个月还在发行数藏活动，最近则是悄无声息的解散了用户群；优特艺术则是以2.5折的价格回收数字藏品；TopHolder头号藏家数藏平台也停用了核心1群，有用户表示非常意外，TopHolder头号藏家作为天下秀旗下的数藏平台，近期还跟微博合作推出了微博六边形头像，而此前售卖的一款鱼太闲更是高达3万，如今直接停用了用户群，没有任何退款的通知，实在是令人唏嘘。

而因为QQ音乐的数藏平台停摆，许多用户加入了维权群。或许是因为大厂的停摆，更是波及到了其他大厂旗下的数藏平台，如截图就有用户问到有没有鲸探的退款群。

如此寒冬之下，生存还是死亡，或许大厂给出了答案。

海外市场仍 有潜力

在国内数藏市场临近寒冬的情况下，多家企业选择在海外开拓NFT市场，以多方位布局来打通web3的通道。

百度

据媒体报道，11月15日，百度旗下DuDu Lab发行的DuDu BearNFT已完成铸造（mint），NFT搭建于以太坊上，铸造价格为0.02ETH，发售价格为0.05ETH，共发行1万个，总价值约为455万人民币，目前已售罄。截至11月15日，DuDu Lab在NFT交易平台OpenSea上的地板价为0.09ETH，价格已翻4.5倍。

据其官方账号信息，DuDu Lab x Theirsverse DuDu Lab是百度公司孵化的结合AI能力的NFT项目。据悉，DuDu Lab通过AI给与持有者进行创作，旨在改变传统的NFT运营模式，帮助Web3转型发展意愿的企业发行与策划运营NFT等流程。DuDuLab NFT项目为DuDu的一生经历构建了一部长篇小说来搭建「DUDU Island」社区以及树立DuDu的IP形象，以小说主体为核心通过重运营模式来打造DuDu IP与DuDu Island。

此前，百度也有在国内数藏市场发力，曾先后基于旗下联盟链 " 百度超级链 " 推出 3 个数字藏品平台，百度超级链数字藏品、钱包更名为星际口袋、朝云数字藏品平台，也许是市场的影响，百度在国内数藏市场的效果微乎其乎。

如今，百度正在探索以DuDuLab为核心，以NFT进军海外原生Web3市场，有着无聊猿、加密朋克等IP的成功案例，以围绕DuDu island的故事开创Web3 NFT IP元宇宙，未来或将成为第一个Web3迪斯尼。

B站

据媒体报道，B站在4月28日宣布，将在以太坊上为海外用户发布Cheers UP NFT系列，该系列在以太坊上发布，总数1万个，铸造价格为 0.12 ETH（价值344美元）。其中233个为团队预留，其余9767个以抽奖的形式发放给用户。其具体内容为头像图片，并且每一张都不相同，且有稀有度之分。

据悉，Cheers UP NFT系列是国内主要互联网公司第一批基于以太坊的NFT。

此前，B站曾在国内发布首款数字艺术头像“鸽德”，并且至今已陆续上线寅虎、不亦乐“虎”、“干杯2022”等系列藏品，但都是由B站高能链提供上链渠道。

据NFTGO数据显示，截止至11月18日，Cheers UP 系列目前地板价为0.355ETH，24小时涨幅15.62%。

阿里巴巴

2022年3月，阿里巴巴旗下《南华早报》成立了 NFT 公司“Artifact Labs”，该项目将基于Flow区块链铸造，用户可以对某一历史事件的特定 NFT 进行竞拍，或者购买一个既定事件的头版。

2022年6月，阿里巴巴还宣布为海外用户提供 NFT 解决方案。

腾讯

2022年3月，腾讯首次参投澳大利亚 NFT 游戏公司 Immutable 高达2亿美元的融资，该公司旗下拥有知名链游《Gods Unchained》和《Guild of Guardians》，并同时兼具交易平台 Immutable X 。

字节跳动

2021年年10月，字节跳动旗下抖音海外版TikTok 宣布推出首个 NFT 系列 TikTok Top Moments，精选了来自平台上一些最受欢迎的创作者六个具有文化意义的 TikTok 视频。其中，一对一的NFT将在以太坊上提供，限量版 NFT 将由 Immutable X 提供支持。

映客

2022年6月，映客互娱集团正式更名为映宇宙（英文名称Inkeverse），业务全面向元宇宙进军，并已经在以太坊上面向海外用户发布了第一代 Pass NFT。

陌陌

2022年7月，陌陌、探探的母公司挚文集团授权 IP 予 WanderingLabs 探索 Web3 项目，在海外发行 NFT 产品 Wandering Planet。

结语

目前来看，国内数字藏品不能很好的落地应用场景，海外NFT虽然没有以前火爆，但机会仍然很多。

据NFTGo.io数据数据显示，截至11月22日，NFT总市值为206亿美元，过去24小时的交易额为1084万美元，目前NTF总持有者为367万，近24小时交易者20708，近24小时买家10770，卖家12421。

国内大厂纷纷试水海外市场，或许说明NFT的潜力不止于数字藏品，他们正在布局更广泛的web3领域。
