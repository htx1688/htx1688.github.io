Windows 11 的用户，你们准备好用上最新的 Copilot，以及 AI 支持的绘画、截图工具了吗？

随着时间来到 9 月 26 日，微软实现了上周的承诺，正式通过 Windows Insiders 推出了 Windows 11 22H2 版本，带来了 Windows、Surface、Microsoft 365、Bing 和新的 Microsoft Copilot 方面的更新。

一键总结文档、快速更换图像背景、语音接入扩展、PC 和云 PC 无缝切换……这一版本不仅使得微软 CEO 纳德拉亲自站台，并表示“通过今天发布的 Windows 11 最新更新，我们将 Copilot 的强大功能和全新的 AI 驱动体验带入 Windows PC 上的 Clipchamp、Paint、Photos 等应用程序中”，还吸引了特斯拉 CEO 马斯克带着「AI 时代正在迅速成为大众现实」的观点火速围观。

如何率先用上 Windows 11 的最新功能？

话不多说。作为 Windows 11 的用户，如何在第一时间用上这些新功能？

首先，你可以打开 Windows 11 设置中的“Windows 更新”页面，这个新页面上有一个新的按钮，上面写着 “一有最新更新，立即获取”。如果你想在今天获得这些功能，就必须在打开该切换按钮的同时在电脑上查找更新（尽管某些功能的可用性可能仍会有所不同）。

如果没有找到这个，也不要着急，预计微软会在下半月发布的十月可选非安全更新中默认启用这些功能。届时，你将不再需要启用此切换按钮，但由于这是一个可选更新，到时候你仍需手动安装。

最后，微软预计会在 11 月份带来一个最新的 Windows 11 23H2 版本，里面将覆盖所有新功能特性。

那么，接下来，看看微软这次都带来了哪些令人期待的更新吧！

Copilot 登陆 Windows 11

无论是在官方公告，还是在外媒的报道中，其都将 Windows Copilot 视为本次 Windows 11 最重大的更新，这是一个基于 AI 的数字助理，旨在帮助你完成工作。

Windows 中的 Copilot 主要基于 Bing Chat 和 GPT-4，好比将 ChatGPT 带进了 Windows 系统，它允许用户使用自然语言输入来提出复杂的问题，并得到有关旅行计划、食谱等建议。

不过，相比基础版本的 ChatGPT，Windows  Copilot 可以根据用户特定的输入直接联网搜索网络内容，并将所有内容整合在一起，从而更快地提供你需要的信息。

此外，Windows Copilot 还允许你控制 PC 上的设置、启动应用程序等等，一定程度上，也可以将它视为被微软淘汰掉的 Cortana（小娜）语音助手的拓展。

与此同时，Windows 中的 Copilot 不仅仅是 Bing Chat，它还可以总结你在 Edge 中查看到的网页内容、也能撰写电子邮件、启用蓝牙或打开电脑上的暗模式。

随着时间的推移，微软会不断填充 Copilot 的能力，未来它也会成为让 Windows 11 变成 AI 操作系统的重要驱动力。

当下要想使用该功能，可以在安装最新 Windows 11 基础上，在任务栏上查找 Copilot 图表，或者只需要按 Win+C 快捷键，就可以使用你在 Windows 上登录时使用的同一 Microsoft 账户 (MSA) 或 Microsoft Entra ID（以前称为 Azure Active Directory）账户在 Windows 中启动 Copilot 。

被 AI 升级的 Windows 11 基础功能

除了以上，微软还在 Windows 11 一些预加载的应用程序如画图（Point）、视频编辑器（Clichamp）、截图工具（Snipping Tool）和照片中融入了 AI 功能。

不用再费心下载 Photoshop 了？Windows 11 自带的画图功能能实现一切

“画图”是 Windows 中重要的一个应用，在融入 AI 功能之后，用户可以使用它轻松搞定图片背景替换、修改。同时，微软还为画图添加了图层支持，让你可以正确编辑透明图像。

这些功能有些类似 Photoshop，也无疑是“手残党”的福音，再也不用借助抠图软件，耗费几小时去改图了。

当然，在画图维度，更为值得关注的是，微软还带来了一款新工具——Paint Cocreator ，它利用 AI 通过文本提示、选择样式由此生成独特的图像，不过不知道微软是不是将 OpenAI 最新发布的 DALL-E 3 大模型融入了进来。

目前，Paint Cocreator 向 Windows Insider 成员开放，未来几周内会向所有用户全面开放。

在预览期间，首次使用 Paint Cocreator 功能的用户可以获得 50 个信用点数，每次使用会消耗掉一个信用点数。微软表示，在预览版结束后，它可能会改变这种积分系统，这表明它将来可能会对这项功能收费。

视频编辑器

视频编辑器（Clipchamp）同样也用了 AI 进行了升级，其中有一个 Auto Compose（自动合成）的功能，只需要询问你正在制作的视频类型的几个简单问题，它就可以根据你想要的风格将它们组合成一个预制视频。

以前需要几小时乃至几天的视频编辑工作，现在只需要简单点几下，就可以轻松搞定。

待视频完成之后，它可以轻松保存在 OneDrive 或 Google Drive 等云服务中，上传到 Tik Tok、Instagram 和 LinkedIn、YouTube 等社交渠道都非常方便。

截图工具

微软将 AI 应用的另一个地方就是截图工具上。最新 Windows 系统中的截图工具支持屏幕截图中的文本识别，以及快速编辑功能，这与 Android 或 iOS 上的从图片复制文本功能类似。因为使用复制的文本，用户可以执行 Web 搜索或者直接将内容复制到 Word 或者 PowerPoint 演示文稿中。

另外，还可以自动隐藏电话号码和电子邮件地址等敏感信息。

照片

照片应用也使用了 AI 来识别主题、人物和场景，你可以在搜索栏中搜索任何你想要的词来找到你要找的图片。此外，微软还在这款应用中新增了背景模糊功能。

需要注意的是，微软表示， “画图”、“照片 “和 “截图工具"这些应用更新 “现在就开始向一小部分消费者用户推出（目标是在未来几个月内全面推出）"。你可能需要等待几周或几个月才能获得 AI 增强版的功能。

全新备份体验、语音接入、安全改进….

此更新还包括一个新的 Windows 备份应用程序，现在，当用户想要迁移到新设备时，这款应用程序可以直接让你备份电脑到云端，然后自动将现有设置应用到新电脑上。

最终，包括任务栏和“开始”菜单上的应用程序放置位置、以及微软应用商店中的任何应用都会自动同步过去。

语音扩展

此外，正如上文所提及的，虽然微软淘汰了 Cortana，但是并没有放弃语音这一块的探索与应用。在 Windows 11 中，你可以使用语音轻松设置和导航想要的功能，也可以用语音在打字时轻松纠正不正确的单词。当然，也可以在锁定屏幕等位置进行语音访问。

同时，Windows 语音还支持了多种不同的语言，包括西班牙语、葡萄牙语、法语、德语、普通话、日语、韩语和英语。

Windows 11 安全性

Windows 11 的安全性也得到了一些改进，即用密钥替换密码，这是一种使用面部、指纹或者 PIN 登录账户的方法。

通过集成密钥，你可以使用 Windows Hello 凭据（而不是典型的密码）登录网站来验证自己的身份，从而降低其他人访问你账户的可能性。

微软还为带有存在传感器的设备添加了新功能，例如“接近时唤醒”和“离开时锁定”，以便你的电脑在你靠近时自动唤醒，在你离开时自动锁定，这样你就不必担心别人使用它。还有自适应调光功能，当你看向别处时，屏幕会变暗以节省电量。

其他更多功能

除了上述人工智能应用程序更新外，微软还将全面推出新的 Outlook 应用程序，以取代邮件和日历。

如果你是 PC 游戏玩家，微软应用商店还将推出新的即时游戏功能，让你无需先下载游戏，就能立即开始玩应用商店中的游戏。此外，用户很快就可以选择在哪个硬盘和位置安装游戏，这与 Xbox 应用程序的功能类似。

同时，操作系统本身也增加了内置的 RGB 灯光控制，这项 Windows 11 功能称为动态照明，将使用开放的 HID LampArray 标准控制 RGB 灯光，从而减少了第三方应用程序对定制 RGB 效果的需求。目前，宏碁、华硕、惠普、HyperX、罗技、雷蛇和 Twinkly 都与 Microsoft 合作开发了动态照明功能。

在企业用户层面，微软还改进了对拥有 Microsoft Entra ID（前身为 Azure Active Directory）账户的用户的 Windows Hello 支持，以及 Windows 365 Boot 和 Windows 365 Switch 等功能，这些功能可让你在 Windows 11 内的本地桌面和 Cloud PC 之间切换。

在开发者层面，微软还新增了 Dev Home、WinGet 配置等开发者功能，以及自动内存回收等 WSL 新功能。Dev Home，这是一种设置和使用 Windows 开发计算机的更简单的方法，Dev Home 利用微软的 Windows  包管理器 (winget) 生成配置文件来安装开发人员所需的应用程序、工具和包。它还包括使用 Microsoft Dev Box  和 GitHub Codespaces 在云中配置编码环境的能力。
