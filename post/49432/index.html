近日，新比特币 NFT 协议 Bitcoin Stamps 因「更永久的存储」而受到一些关注。不同于 Ordinals 协议的将数据存放在「见证数据」，Bitcoin Stamps 利用 2014 年就诞生、曾为「Rare Pepes」提供支持的 Counterparty 协议，将插入到 UTXO 输出中的图片数据 base 64 编码广播到比特币网络，使全节点不存在过滤图片数据的可能。

虽然比特币新 NFT 协议 Bitcoin Stamps 近日的新增 NFT 数量超过了 Ordinals 协议，不少比特币 NFT 玩家也已经「打了防身」。但是，笔者并不看好 Bitcoin Stamps 的前景。

为什么不看好 Bitcoin Stamps 呢？

贵

笔者分别在 UniSat 与 stampchain.io 上传了大小为 254 字节的一张 Punk 图片。刻录同一张图片，所需要花费的价格相差了 10 倍多。

UniSat（Ordinals）

stampchain.io（Bitcoin Stamps）

2017 年的 SegWit 升级的「见证折扣」使通过 Ordinals 协议刻录出来的比特币 NFT 能够相比于 Bitcoin Stamps 协议能够得到 75% 的优惠，但即使我们把 UniSat 的刻录费用乘以 4 ，也仅仅只有约 24 美元，而 Bitcoin Stamps 协议收取的手续费用则高达约 85 美元。多出来的那部分是啥？笔者也暂且蒙在鼓里…

当然，我们也可以不选择使用 stampchain.io 上提供的「官方代打」服务，转去选择使用 Counter Wallet 自助刻录。但是自助的过程相当繁琐，要安装配置 Counter Wallet，要把图片转换为 Base 64 字符串…此外，stampchain.io 作为「官方代打」收取如此高昂的费用也让比特币 NFT 社区觉得反感。毕竟，在没有版税机制的比特币 NFT 圈，许多玩家乃至项目方都是「用爱发电」，比特币免费和公平的精神是比特币 NFT 玩家与项目方之间建立起良好信任的基石。

自由度不足

通过 Ordinals 协议刻录的比特币 NFT 大小仅受见证数据字段的 4 MB 大小限制，可刻录的数据类型即 MIME 类型，从文本、图片、动图到音乐、视频、HTML 文件通通都支持。第 466 个被刻录出来的比特币 NFT，正是 id Software 的经典游戏「Doom」。（想试一下用比特币 NFT 玩「Doom」？点击这里）

Inscription #466 「Doom」，笔者已通关

而 Bitcoin Stamps 则排除了 MIME 类型，图像数据限定为 jpg、png、gif 与 webp 格式，且大小限制为 7 KB。大小限制与上面提到的成本问题，官方建议上传的文件为 24* 24 像素、 8 位色深的 png 与 gif 文件。

官方说这种限制很适合像素艺术。好吧，与 Ordinals 相比，确实是有种在小小的邮票上作画的感觉… 不过，既然有了 Ordinals 这块更大的「画布」，为什么还要用 Bitcoin Stamps 呢？

「永久性」上，Ordinals 真的输了吗？

Bitcoin Stamps 协议是将图像的二进制内容编码为 base 64 字符串并将此字符串插入「Description」字段并将数据分块插入到交易输出中，这样使得图像数据成为「基础块数据」，从而无法被全节点过滤。

Ordinals 协议则是将数据插入到了「见证数据」部分，全节点可以选择丢弃「见证数据」。在「比特币 NFT 开荒进行时：Ordinals 协议、FOMO 和争论」一文中，笔者曾提及比特币核心开发者 Luke Dashjr 快速开发了一款补丁供矿工筛选并过滤包含 Ordinals 协议创造的比特币 NFT 的交易广播。

这表面上是一个技术决定的问题，但是实际上是一个市场（矿工）决定的问题。目前来看，Ordinals 协议正蓬勃发展，既有 Magic Eden 这样的大 NFT 市场入场，也有布加迪这样的顶级 IRL 品牌决定通过 Ordinals 协议发行 NFT 系列。我们真的会看到比特币网络的全节点们形成共识抵制 Ordinals 协议吗？

根据 @smyyguy 的数据， 3 月份只有 0.3% 的矿工收入来自 Ordinals 协议。Ordinals 协议还很年轻，天知道会不会真的发展到影响比特币网络的运行，又哪能知道届时的影响是好是坏…

结语

更贵、更低的自由性与只存在于叙事层面的「永久性优势」，是笔者不看好 Bitcoin Stamps 协议的原因。虽然近日来 Bitcoin Stamps 的新刻录数量超过了 Ordinals，但等「防身热潮」过去后，Ordinals 协议大概率还是会重新占回上风。毕竟，Ordinals 协议的配套基建已经快速发展，众多蓝筹项目已经在 Ordinals 生态中占好坑位，协议本身甚至已经被 fork 到了莱特币与狗狗币上，这些都不是 Bitcoin Stamps 可以比拟的，也是受其自身的技术限制而难以追赶的。
