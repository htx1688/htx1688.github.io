2023 年对于探索新的 DeFi 原语潜力的建设者来说是一个非凡的一年。在这段时间里，最引人注目的发展之一是流动性质押衍生品（LSD）协议的崛起，以及随之而来的建立在 LSD 项目之上的协议，被称为 LSDfi。

这些 LSDfi 项目可以分为几个不同的部分。在本文中，我主要关注的是以 LSD 为支撑的稳定币。

一、什么是 LSD？什么是 LSDfi？

流动性质押衍生品（LSD）是一种金融工具，代表在 DeFi 协议中质押代币的所有权。这些工具使用户能够质押他们的代币，同时保留使用这些 LSD 在各种应用中的自由。一些 LSD 协议包括 Lido Finance 和 Rocket Pool。LSD 为生态系统提供了许多好处，因为它们释放了以前被锁定的资本，同时为网络提供了安全性。

LSDfi 是指利用 LSD 协议构建金融原语的项目，例如 Pendle Finance 和 Unsheth。通过提供额外的收益生成机会，LSDfi 协议允许 LSD 持有者利用他们的资产并最大化收益。

然而，作为一个子类别，还有一些以 LSD 为支撑的稳定币，例如 Raft、Gravita、Ethena、Prisma 和 Lybra，我们现在将对它们进行评估。

以 LSD 为支撑的稳定币是 CDP 模型的稳定币，需要通过流动性质押代币进行超额抵押，并存在清算风险。它们允许持有者在保留加密货币支持稳定币的关键属性的同时获得内在收益。

可以看出，以 LSD 为支撑的稳定币与已建立的加密货币支持稳定币（如LUSD、FRAX 或DAI）并没有太大的区别。以LSD为支撑的稳定币提供的主要价值主张是ETH 的质押收益，同时使用户能够继续接触 DeFi 应用。然而，新项目还提供了一些创新的功能。

为了更好地理解这个类别，让我们一一来看这些协议。

二、Prisma Finance( $mkUSD)

Prisma 是一种以 LSD 为支撑的稳定币，是 Liquity 的分叉，但有着重大改进。Prisma 使用户能够铸造由多种 LST（如wstETH、cbETH、rETH、sfrxETH 和WBETH）抵押的mkUSD。

mkUSD将在Curve和ConvexFinance上获得激励，以创建一个资本高效的飞轮，用户可以获得交易费、CRV、CVX和PRISMA，以及$ETH 的质押奖励。

我对$mkUSD 的看法如下：

  1. 有竞争力的价值主张：每个以 LSD 为支撑的稳定币都为用户提供ETH的收益，然而，由于mkUSD 池部署在 Curve 上，存入mkUSD的用户可以获得交易费、CRV、CVX和PRISMA 奖励， 这可能使$mkUSD 在竞争对手中更具竞争力。

  2. 不是交易媒介：mkUSD是一种带有收益的稳定币，该协议不优先考虑将其用作交易媒介。 大多数用户持有mkUSD 是为了获得持有$mkUSD 所提供的年利率。

  3. 带收益的资产：由于mkUSD可以为持有者产生收益，肯定会有人只是将其用作价值储存。如果用户信任其锚定稳定性，这可以是获得ETH 收益的一种好方法。

  4. 创新的代币经济模型：vePrisma 持有者将能够激励特定的池，因此，LST 提供者可能有兴趣用他们自己的 LST 激励mkUSD。 这可以为mkUSD 的需求创造一个正向循环效应。根据白皮书，投票人可以将发行量导向使用特定的抵押品进行铸造，以保持使用特定的抵押品进行活跃借贷，并向任何 LP 代币持有者发放奖励。考虑到深度流动性对于维持锚定稳定性至关重要，这将是区分 Prisma 与竞争对手的重要因素。

  5. 多种 LST 抵押品：有几种 LST 可以用作抵押品，如wstETH、cbETH、rETH、sfrxETH 和WBETH，市值不同。由于独特的代币经济模型，这些协议可以激励用户铸造mkUSD，从而增加对 Prisma 的敞口。

  6. 资本效率不足：超额抵押模型意味着$mkUSD 在资本效率方面有限，因为用户需要投入比他们获得的更多的资金。此外，由于抵押率应始终保持在 120%以上，总是存在清算风险。

  7. 强大的支持者：与竞争对手相比，尽管 Prisma Finance 进入市场较晚，但该协议得到了 Curve Finance、FRAX 和 Convex 等几个强大的支持者的支持。

三、Raft( $R)

Raft 是一种铸造 R 稳定币的协议，该稳定币由 LST 以超额抵押方式支持，并存在清算风险。用户可以通过存入储蓄利率来获得可持续的收益。

我对$R 的看法如下：

  1. 缺乏创新：Raft 是 Liquity 的分叉，只有一些小的改变，因此产品上没有太多创新，因此在 Liquity v2 推出后，可能很容易被超越，Liquity v2 将利用 LST。

  2. 不是交易媒介：R是一种带有收益的稳定币，该协议不优先考虑将其用作交易媒介。大多数用户持有R 是为了获得持有$R 所提供的年利率。

  3. 锚定稳定性：R目前价值约为0.98美元， 团队正在努力寻找恢复锚定的解决方案。团队提议实施利息费用而不是一次性费用来铸造R。通过这样做，他们旨在通过激励市场上的购买压力来恢复锚定。导致挂钩的原因可归因于铸造 R 的一次性费用、缺乏流动性以及缺乏创造有机需求的用例。

  4. 与竞争对手相比的价值主张有限：在这一点上，用户不需要支付利息费用来借入R， 因此他们可以利用他们的ETH 头寸。这是$R 的主要价值主张。然而，如果团队决定改变这个模型，Raft 将没有任何价值主张。

  5. 带收益的资产：由于R可以为持有者产生收益，肯定会有人只是将其用作价值储存。如果用户信任其锚定稳定性，这可以是获得ETH 收益的一种好方法。

  6. 资本效率不足：由于$R 是一种 CDP 模型的稳定币，需要超额抵押并存在清算风险，对于散户用户来说，它不是一种资本高效的模型。 这将限制其增长能力，因为扩展可能性有限。

四、Gravita( $GRAI)

Gravita 是 Liquity 的一个分叉项目，接受不同的 LSD 产品作为抵押品。它使用户能够无利息借款，并且不从存入的 LST 产生的收益中抽成。赎回机制在初始阶段没有启动，但在整个过程中逐渐释放。 这可能是$GRAI 从一开始就维持在 0.98 美元左右的原因，这无疑会引起用户的信任问题。

我对$GRAI 的看法如下：

  1. 缺乏创新：正如所述，Gravita 是 Liquity 的一个分叉项目，产品上没有太多创新，因此在使用 LST 的 Liquity v2 推出后，它可能很容易被超越。

  2. 与竞争对手相比的价值主张有限： 用户无需支付利息费用即可借款GRAU，因此他们可以利用自己的ETH 头寸。此外，允许 $bLUSD 用作抵押品，没有任何清算风险，并且不从质押收益中收取任何费用，这是 Gravita 提供的价值主张。

  3. 不是交换媒介：GRAI是一种带收益的稳定币，该协议并不优先考虑将其用作交换媒介。大多数用户持有GRAI 是为了持有$GRAI 提供的年利率。

  4. 锚定稳定性：自GRAI推出以来，价格一直在0.98美元左右波动。这可能是因为在推出期间不允许赎回GRAI，然后逐步释放，这可能会导致供应过剩，从而降低价格而没有套利机会。 此外，低流动性和缺乏用例来创造有机需求可能限制了对$GRAI 的需求增长，这也加剧了情况的恶化。

  5. 带收益的资产：由于GRAI可以为持有者产生收入，肯定会有需求将其用作价值储存工具。如果用户信任锚定稳定性，这可以是获得ETH 收益的好方法。

  6. 多种 LST 抵押品：有几种 LST 可以用作抵押品，例如WETH、rETH、wstETH和bLUSD。这可以作为一个优势，为用户提供了几个机会。

  7. 缺乏资本效率：超额抵押模型意味着$GRAI 在资本效率方面受到限制，因为用户需要投入比他们获得的更多的资金。此外，始终存在清算风险，这将限制增长。

五、Lybra( $eUSD)

eUSD是一种以质押的ETH 为储备的稳定币。拥有eUSD将带来稳定的收益流，年化收益率约为8LBR 的治理代币，但其实用性有限。随着 Lybra v2 的推出，发布了几个新功能，预计将改善该协议的缺点。

我对$eUSD 的看法如下：

  1. 缺乏资本效率：超额抵押模型意味着$eUSD 在资本效率方面受到限制，因为用户需要投入比他们获得的更多的资金。此外，始终存在清算风险，因为抵押率应始终高于 150%。

  2. 与竞争对手相比的价值主张有限：为了有潜力增长，新兴的以 LSD 为支持的稳定币需要具备独特的价值主张。然而，尽管$eUSD 具有早期优势，但它在抵押要求或任何重大改进方面都没有提供竞争力。

  3. 不是交换媒介：eUSD是一种带收益的稳定币，该协议并不优先考虑将其用作交换媒介。大多数用户持有eUSD 是为了持有$eUSD 提供的高年化收益。

  4. 锚定稳定性：eUSD持有者有资格获得质押的ETH 奖励。因此，大多数用户更喜欢在市场上购买eUSD，这会产生需求压力。由此导致eUSD 的需求超过供应，使其突破 1.00 美元的锚定。除非系统发生变化，$eUSD 将无法找到其锚定。 这可能会对持有者造成长期问题。

  5. 带收益的资产：由于eUSD可以为持有者产生收入，肯定会有需求将其用作价值储存工具。如果用户信任锚定稳定性，这可以是获得ETH 收益的好方法。

  6. 多种 LST 抵押品：随着 Lybra v2 的推出，可以使用新的 LST 抵押品，如rETH和WBETH。这将增加$eUSD 铸造的可能性，但我们不应高估其影响。

  7. 糟糕的代币经济学：LBR是该协议的治理代币，然而，由于几乎所有来自LSD的收益都流向eUSD 而不是LBR，该代币几乎没有实用性。 糟糕的代币经济学也使eUSD 的溢价持续存在，因此它始终超过锚定，因为用户有动力持有eUSD，因为它是一种带利息的稳定币，持有eUSD 的需求大大超过铸造$eUSD 的需求。

六、Ethena( $USDe)

Ethena Labs 是一个尚未发布的新项目。该项目通过提供一个与 CDP 模型不同的增量中性支持模型与竞争对手有所不同。通过这个模型，该项目将使用 LSD 作为抵押品，在交易所上创建一个现货多头、1 倍空头头寸，从而防止抵押品的波动性。USDe将更高效，因为它将提供1 : 1的抵押比率，除了LSD收益外，还将提供增量中性模型的资金费收益。然而，用户不会受到ETH 价格波动的影响。

我对$USDe 的看法如下：

  1. 创新：在所有现有项目中，Ethena 是唯一提供创新解决方案的项目。 我相信增量中性模型可以成功解决 LSD 支持的稳定币的一些主要问题，如资本效率、缺乏可扩展性、锚定稳定性等。

  2. 资本效率：由于增量中性模型，该协议不需要超额抵押来维持锚定，因此可以提供 1:1 的抵押比率。因此，在资本效率方面，$USDe 在竞争对手中表现最佳。

  3. 锚定稳定性：$USDe 将使用增量中性头寸来维持锚定稳定性。考虑到理论上，“在交易所上创建一个现货多头、1 倍空头头寸”将始终保护抵押品的价值。然而，重要的是要看到实践结果。

  4. 交换媒介：由于USDe提供1 : 1的抵押比率，它可以解决现有基于加密货币的稳定币的可扩展性问题。因此，USDe 可以作为具有深度流动性的平台之间的交换媒介。

  5. 与竞争对手相比的价值主张强大：USDe在市场上具有两个主要的独特优势，可以使该产品与竞争对手区别开来。首先，它可以提供1 : 1的抵押比率，这对用户更具吸引力。此外，除了LST收益外，USDe 还将提供资金费收益，这在现有项目中更具竞争力。

  6. 用户采用率：由于每个创新项目都面临相同的问题，$USDe 也将面临社区的一些怀疑，因为 Delta-Neutral 方法并不广为人知，所以 Ethena 需要一些时间来教育用户并尝试这种方法。

  7. 不受 ETH 波动的影响：由于存入的抵押品用于建立对冲头寸，用户不会面临ETH价格波动的风险。 风险规避的用户可能会将其视为一种好处，然而，ETH maxis 可能会将其视为一个缺点。

七、对 LSD 支持的稳定币的整体格局的思考

到目前为止，我分享了我对个别 LSD 支持的稳定币的看法，以便我们更好地了解这些稳定币的动态，同时分析它们的机会和限制。我相信这种分析有助于理解 LSD 支持的稳定币的竞争格局，并证明每个个别稳定币的权衡。

现在，我将分享我对 LSD 支持的稳定币格局的总体概述，以便我们可以预测这个类别可能如何发展。为此，我将实施一项 SWOT 分析：

注意：应该强调的是，对每个 LSD 支持的稳定币进行一般的 SWOT 模型分析无法提供很好的概述，因为它们每个都具有不同的价值/特点。这尤其适用于 Ethena Labs，因为他们的 Delta-Neutral 机制与 CDP 模型完全不同。例如，在弱点部分，资本效率、交换媒介和有限的用例不适用于 Ethena 的稳定币 $eUSD。

优势

  1. 价值储存：LSD 支持的稳定币是很好的价值储存工具，因为它们中的大多数已经实现了价格稳定，同时为用户提供$ETH 收益。因此，它们可以作为低风险收益机会和价值储存，在不久的将来增加市场份额。随着人们意识到 LSD 支持的稳定币通过与用户共享固有收益来赋予用户权力，采用率将会增长。

  2. 收益机会：对于散户交易者来说，稳定币的 5-8%年化收益可能并不具有吸引力，但对于大户和杠杆交易者来说，这是一个很好的机会，考虑到 DeFi 生态系统中高收益机会的有限性，尤其是在熊市持续的情况下。

  3. 释放流动性：LSD 是解锁抵押的$ETH 流动性的好方法，而且 LSDfi，尤其是 LSD 支持的稳定币进一步改善了这种情况，为 LSD 创造了新的使用案例，这肯定会进一步增加生态系统的机会。

  4. 增加ETH敞口：LSD支持的稳定币是扩大以太坊生态系统的好工具，因为它们改善了用户ETH 敞口的方式，并为其创造了新的使用案例，从而增加了更多的有机需求。

劣势

  1. 增长取决于 LSDfi 的采用：LSDfi 是一个需要进一步探索的新类别。作为该类别的先行者，LSD 支持的稳定币将高度依赖于市场的整体增长，这在一定程度上独立于它们的影响。

  2. 资本效率：由于 LSD 支持的稳定币大部分实施了 CDP 模型，它们需要超额抵押并面临清算风险。因此，资本效率成为用户面临的一个核心挑战。

  3. 交换媒介：LSD 支持的稳定币本质上用于收益机会，而且它们都依赖于 CDP 模型，因此无法将 LSD 支持的稳定币作为交换媒介处理，这将限制这些产品的可扩展性。

  4. 有限的使用案例：尽管作为可持续收益资产是一个很好的价值主张，但流动性碎片化和缺乏流动性限制了 LSD 支持的稳定币的使用案例。除了持有之外，几乎没有其他利用这些稳定币的方式。

机会

  1. ETH 质押采用：随着对以太坊生态系统的安全性和ETH质押收益的持续信任，ETH质押是我们将看到进一步增长的领域之一。随着未来可能增加的ETH 质押率，可以预测 LSD 支持的稳定币将从中受益。

  2. 对抗通胀的价值储存：由于通胀，对收益资产的强烈需求将始终存在。正如我们从构建抗通胀稳定币/平稳币的尝试中可以看到的那样，对它们存在巨大的需求。尽管从本质上讲，LSD 支持的稳定币并不存在为了抵御通胀或作为对抗通胀的价值储存的目的，但它们证明了它们可以作为对抗通胀的强大工具。

威胁

  1. 缺乏创新：我认为 LSD 支持的稳定币主要是 Liquity 的分叉，几乎没有什么不同之处。因此，它们与 Liquity 相比并没有提供太多的价值主张，除了允许使用 LST 作为抵押品之外。Liquity v2 就会实现这个目标，投资者还会继续使用它们吗？

  2. 收益可能降低：由于ETH质押收益可能随时降低，LSD支持的稳定币的收益也会降低。这可能会使用户不愿选择这些稳定币。考虑到未来将有更多的ETH 质押，这是 LSD 支持的稳定币不可避免面临的结果。

  3. 需求和流动性低：到目前为止，大多数 LSD 支持的稳定币无法维持在$1 左右的锚定。尽管这种情况有具体原因，但共同问题是这些稳定币缺乏强大的需求和流动性。

  4. 由于竞争而导致的流动性碎片化：目前，有几个团队正在尝试构建 LSD 支持的稳定币，而在这场竞赛中没有明确的赢家。这意味着流动性在竞争者之间分散，限制了增长的能力，从而阻碍了产品的更有效性或产生收入。所有这些可能对 LSD 支持的稳定币的成功产生长期影响。

  5. 熊市的结束：大多数投资者选择 LSD 支持的稳定币作为收益资产，因为在熊市中没有更好的解决方案/替代品。然而，当牛市开始时，资金可以流向更有利可图的项目，因为在牛市中 5%-8%的年化收益可能并不具有吸引力。然而，值得注意的是，熊市的结束肯定会帮助这些协议增长，因为总体市值将进一步增长。

未来影响：使 LSD 支持的稳定币更高效

很明显，随着 LSDfi 产品的兴起，对 LSD 支持的稳定币的兴趣越来越大。我相信这一趋势将继续增长。然而，我认为目前大多数 LSD 支持的稳定币模型要么不适合产品市场，要么在竞争对手面前没有竞争优势。

像R、GRAI 和eUSD这样的一些LSD支持的稳定币与crvUSD 和$LUSD 等现有项目之间没有明确的价值主张。这些协议有可能能够减少前述项目的份额。

Prisma Finance 是一个有趣的案例， 他们正在开发独特的代币经济模型，以提高稳定币持有者的收益，并为治理代币持有者创造价值。尽管该稳定币的当前 CDP 模型并不独特，也没有提供新的价值主张，但该协议可能有机会，因为它的代币经济模型为用户创造了有机需求，加深了流动性，从而更容易维持锚定。

Ethena Labs 是一个挑战现有模型的独特模型。 该协议更加高效，并且由于协议开放的无风险头寸，可以通过资金费用创造更多收入。这一点非常重要，因为这种模型在现有的 LST 收益之上创造了有机收益，并使协议更具竞争力。然而，值得注意的是，在 CDP 模型中，当抵押品价格上涨时，借款人会获得利润。然而，在 Ethena 的情况下，由于通过无风险头寸维持锚定，用户放弃了可能来自$ETH 上涨波动性的利润。总的来说，我认为 Ethena 可以解决 LSD 支持的稳定币的一些主要问题，如资本效率、缺乏可扩展性、锚定稳定性等。

总的来说，LSD 支持的稳定币的未来将取决于：

  1. 改善资本效率的新模型；

  2. 新的收益来源；

  3. ETH 质押采用规模扩大；

  4. LSDfi 的采用。

让我们拭目以待。
