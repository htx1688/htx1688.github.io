0 1  区块链有搞头吗？

这是一篇在肚子里头拖了很长时间的稿子。

时常有朋友问我，曼昆律师为什么会这么坚定的选择区块链作为深耕的行业方向，这事在国内不是不让搞吗？不让搞的行业你们来做法律服务，真的有搞头？

说实话，正是因为有很多同行甚至区块链业内的朋友持有这样的观点，对于我们来说才愈发的有搞头。

地球可能是平的，但多少还是有些丘陵。

作为一个新兴的事物，区块链技术至今不过14年，其本身的发展更是一波三折，不仅是在中国，全球都一样。自诞生以来，中国对于区块链的监管始终是在正向的进步，虽然这样正向的进步是通过不断的画出负面清单的方式来进行。

我常跟身边的朋友重复再重复区块链在中国发展的这三个负面清单：

  • 打着虚拟货币的名义进行非法集资和传销是不行的

  • 在中国或者面向中国公民开展虚拟货币交易所的业务是不行的

  • 在中国搞虚拟货币挖矿是不行的

除了上述这三点外，在中国开展区块链相关的业务完全没有问题。

很多朋友对区块链的理解都是狭隘在虚拟货币，会认为中国对虚拟货币的否定便是对区块链的否定。然而虚拟货币只是区块链技术发展初期的一个场景而已，中国对于虚拟货币在中国的限制与否定和对区块链技术的鼓励发展并行，这是理解区块链创业在中国很重要的一个前提。与区块链大概念相关有很多可以去创业的地方，和区块链在商业社会的应用场景和价值相比，中国针对虚拟货币的三个三个负面清单说是九牛一毛也不过分。

如果一个区块链创业者你活的不好，没赚到钱，反手就怪中国内地不欢迎区块链不让你发币搞募资，那可真的是有点扯淡了。

赵本山在《乡村爱情》有个经典的桥段，傻儿子跟老爸抱怨说他做事总不成是大环境不好。

本山大叔安慰他道：自己没能力就说没能力，别总说大环境不好，怎么你到哪都大环境不好，你是破坏大环境的人啊！

0 2  未来十年会怎样？

所谓看十年，干一年。

前两天和曼昆团队的毛捷豪律师喝茶的时候闲扯淡，我说最近我一直在思考一个问题：10年以后，区块链的行业会是怎样？更重要的是，中国的区块链行业会是怎样？

我说我有几个大胆的猜测，很不负责任的那种：

1、2023年行业的大机构和当红炸子鸡项目可能都挂了，包括某安、包括某太坊，正如1995年北京的瀛海威。

2、比特币要么成为另类资产配置的常规标的，价格缓慢增长到10万刀以上（不构成投资建议！），要么就彻底归零。

3、稳定币会成为跨境贸易、跨境汇款的首选，主流社交媒体和金融机构陆续推出自己的稳定币系统，Web3.0时代的全球货币大战会继续在中美两国间暗中博弈。

4、证券型代币已经成熟，只不过路径不是Token成为证券，而是证券成为Token ，包括港股、A股在内的主流证券交易所会选择使用区块链来作为底层的技术支撑，RWA对应的债权型Token层出不穷。

5、东西方都会有加密金融中心推出监管沙盒，以注册制的方式允许区块链创业团队发行Token来募资/发债，但会对募资金额和投资人进行限定，参与认购和交易的是合规基金或者合规投资人。

6、功能性代币会成为绝大多数Web3.0应用的标配，正如Web2.0时代的钱包与积分那样。功能型代币不存在募资的功能，只是用于用户激励和网络数据流通的燃烧。跨项目、跨链的功能性代币流转会成为趋势，无论是政府主导的，还是第三方商业机构发起的。

7、NFT会成员线上社交、线上游戏、电商的标配，不会再刻意的强调NFT的以虚促实，看得见摸不着的数字商品会成为日常消费品极为正常的存在。

8、更重要的是， 以上的这些猜想，大概率还是可以发生在中国。

0 3  区块链未来在中国

在和很多中国的区块链创业者交流的过程中，我发现很多初出茅庐的区块链创业者会有一种“仰望”海外项目方和创业者的坏毛病。

甚至有人还提炼出来了区块链世界的鄙视链：大致是欧美市场的区块链项目方和从业者鄙视在东南亚香港的，在东南亚香港新加坡的创业者和从业者鄙视在中国内地的。

你要问这鄙视链上游的朋友他们的优越感来自哪里？他们往往也说不出个123。这种现象确实有点令人匪夷所思，Web.后面的数值多加一个1就能鄙夷行业的前辈这操作真的不至于此。错把时代的红利错当成自己的本事的坏毛病，这跟Web几真的没关系。

一个新兴技术在发展初期，要做的不是蜗牛角上的鄙视链上窝里斗，而是要真正的想办法去解决社会问题，找到真实的商业应用场景，创造出商业价值，凭本事通过服务客户或者用户赚到钱活下去，在创造技术概念上争先后、比拼谁能半夜起来做市值管理这事属实没意义。

空谈误国，实干兴链。Web3创业得仰望星空，但更重要的是脚踏实地。不要脚踩着的是中国的土地，脑子里总惦记的是海外市场概念推概念的虚无叙事，这也是我一直不建议中国创业者过于关注RWA和DAO的原因。说实话，我觉得作为一个Web3.0区块链创业者，能够在中国市场，已经是绝对幸运的事情了。

之所以有这样的判断，是基于作为曾经Web2.0创业者的切身感受：

1、中国Web2.0时代积累了全球遥遥领先的互联网基础设施，无论是软硬件还是O2O，互联网行业上下游产业链很是完备，Web2.0的时代红利中国政府显然是尝到了甜头，互联网的全球第一梯度的领先身位中国断然不会拱手让人。

2、中国有全世界最优秀的互联网产品经理和运营人才，你说勤奋也好，内卷也罢，不可否认的是中国绝对数量领先的优秀互联网人才在产品设计、用户体验、流量获取、商业变现各维度绝对是全球遥遥领先，别看目前市面上搞钱搞项目的都是欧美面孔，背后至少一半人员是中国身影。区块链世界不能没有深圳和杭州，正好比西方不能没有耶路撒冷一样。

3、中国14.12亿人口，移动互联网的用户规模就有12亿。这样的统一大市场，多招其他国家和地区的互联网项目者的羡慕嫉妒恨，更何况这统一大市场还有一道无形的墙予以保护，别让区块链创业成围墙，中国的创业者想走出去，外头的创业者想走进来。

所以中国创业者，不用过于自愧不如，不要妄自菲薄，不用总念叨着他山之石可以攻玉，要知道，上一代的互联网的早期，是C2C（Copy2China）不假，但下半场的中国互联网出海故事更具说服力。
