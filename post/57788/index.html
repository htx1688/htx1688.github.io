作者 ： Azuma

8 月 25 日，Optimism 和 Base 联合公布了一项治理及收入共享协议，协议规定了 Optimism 和 Base 双方在短期、长期状态下不同的治理协作模式，以及双方之间的经济交互关系。

如果说在此之前，Optimism 和 Base 之间的关系仅限于后者使用了前者的技术堆栈（OP Stack），属于一种偏软性的合作，那么从今天开始，二者之间则将在治理和收入两个层面深度绑定，甚至完全可以被视作同一个生态体系之下的“一家人”。

治理层面

首先说说治理，由于 Optimism 在发行 OP 之时设计了较为完备的治理系统，所以这部分的内容主要关乎于没有治理代币的 Base（此前也说过不会发币）。

从具体的治理形式来看，短期及长期状态下双方也会有不同的协作模式。

先说短期，当前 Base 网络的升级依赖于一个 2/2 的多签合约，合约的 2 个签名权限分别被 Optimism 和 Base 所控制 ，双方会在突发紧急情况时接手控制，或是 Optimism 方面完成关于升级的治理决策后着手执行。

简单来说，这种形式虽然从结果上看来也是二者共同决议，但会较软性地依赖于双方之间的合作意愿。假设双方出现不睦（虽然可能性很低）， 2/2 的多签无法统一，Base 的升级也将陷入停滞。

而从长期来看，Base 作为采用了 OP Stack 的“超级链”之一，则将通过 Optimism 日前所提出的“Law of Chains”治理框架与 Optimism 实现更硬性的绑定。

所谓“Law of Chains”，指的是一套希望统一管理所有“超级链”（包括 OP Mainnet、Base 以及其他 OP Stack 区块链）的治理框架，该框架计划于 2024 年初落地。

根据 Optimism 和 Base 的协议内容，待“Law of Chains”落地后，双方将把多签治理权限转移至一组由分布在全球各地的独立社区成员所组成的安全理事会（Security Council），届时该理事会不但将负责 Base 的升级，也将同时负责 OP Mainnet 以及其他 OP Stack 区块链的升级。

反过来，由于 Base 的治理未来将完全依赖于 Optimism 方面，因此其也将在整个理事会的治理中拥有一部分权限。对此，Optimism 将给予 Base 一定的 OP 份额（下文收入层面会细说），从而允许 Base 参与至理事会的治理之中，但出于限制考虑，也规定了这部分投票权限不得超过总投票权限的 9% 。

收入层面

相较于偏抽象的治理层面，双方在收入层面的协作则显得更加直观。

根据协议，Base 将向 Optimism 支付排序器收入（排序器在 Layer 2 的总收入）的 2.5% 或利润（总收入减去向 Layer 1 提交数据的成本）的 15% ，二者之间取较高者；作为回报，Base 则在未来六年内可以获得“最多 1.18 亿枚 OP”（约占总供应量的 2.75% ），使其能够发挥前文在治理层面中所说的部分权限。

简单来说，二者之间将进行一次经济层面的直接交换，Optimism 需要分六年向 Base 支付最多 1.18 亿枚 OP，Base 则需永久性地向 Optimism 提供收入分成。

关于两项支出谁多谁少，这里可以简单算个账。1.18 亿枚 OP 现价值约 1.83 亿美元；而根据 Dune 的数据统计， Base 上线一个月以来（算上抢跑时间）的排序器收入约为 260 万美元，由于缺乏利润数据，采用收入的 2.5% 故略估算 Base 每年需向 Optimism 分成 78 万美元（260 * 12 * 2.5% ）。

不过需要注意的是，由于 Base 生态刚刚起步，且该网络暂时仅有团队运行的排序器，这两项因素均会对 Base 未来的收入情况造成一定影响。

从 Optimism 角度来看，虽然在绝对数字上，当前其所支付的 OP 筹码要远远高于 Base 的收入分成回馈，但基于 Base 未来仍将继续扩展的积极预期，长线来看这笔“交易”可带来的回报仍有着相当大的增幅空间。

我个人倾向于认可这笔“交易”的前瞻性。举个不太恰当的例子，在没确定这个收入共享协议的情况下，Optimism 和 Base 的关系很像此前 Cosmos 与 Terra 之间的关系，Terra 采用了 Cosmos SDK 发行（这里是 OP Stack），且在 2022 年之前靠着 UST “魔术”赚得盆满钵满，但这些价值捕获完全无法反馈至 Cosmos 以及 ATOM。

而在有了来自 Base 的收入之后，Optimism 将会获得持续性的收益流入，这对于项目基本面来说显然是有所助益的。

可能部分 OP 持仓者会顾虑 1.18 亿枚 OP 的潜在抛压，但这似乎也无需太过担心。

  • 一是 Base 已提及这部分 OP 会主要用于治理用途（但没有提到硬性约束）；

  • 二是这部分 OP 将会分六年支付，并非一次性给予；

  • 三是这部分 OP 将由 Optimism 基金会支出，用以激励 Optimism 生态的发展。 换句白话说，这些币本就是会花出去的，OP 持仓者们该担心的并不是这些钱会不会花，而是花的到底值不值。我倾向于认为，用最高 2.75% 的 OP 供应去换一个永久性的强援，这笔“交易”是值得的。

至于对 Base，这笔 OP 收入也有助于解决了 Base 运行的财务问题（包括开发者报酬、生态激励、商务拓展等等）。 此前，Base 的运行在很大程度上会依赖于 Coinbase 的隐性支持，但出于合规考虑（近来已有声音称 Coinbase 和 Base 的关系或涉嫌违规），Coinbase 未来大概率会在运行层面与 Base 作出明确切割，届时 Base 也将需要拥有自己的财务来源。

从这一角度来看，这笔“交易”或许是符合双方各自需求的双赢选择。

“超级链”蓝图，揭开面纱

整体而言，Optimism 和 Base 今日所发布的这一合作协议，意味着二者之间的合作进入了一个全新级别。这一合作关系之紧密，甚至很难在行业历史中找到类似案例。

而展望更长期的未来，通过与最重要的“超级链”之一 Base 完成深度绑定，Optimism 所规划的“超级链”蓝图也终于揭开了面纱：“Law of Chains”协动治理——Optimism 通过分发 OP 来扩充生态并推进去中心化——生态个体则将利用收入持续性地回馈母体，助益母体基本面——吸引更多个体……这一正向循环或将成为助推 Optimism 生态加速扩展的飞轮。

今日的 Base 仅仅只是个开始，Zora、Celo、PGN、Debank……未来我们或许会看到更多项目加入这一进程。
