基于 Sui 的一款链游“Sui 8192 ”日均交易数大幅飙升，链上分析提供商 Artemis 最近在推特上发布的一篇文章甚至表示，这款游戏帮助 Sui Network 在日常交易数量方面短暂超过了 Solana。

Artemis 的数据显示，Sui Network 的日交易笔数最近突破 2000 万笔，超过了 Solana 的 1600 万笔，不过该公司承认，它使用了不同的方法来计算 Sui Network 的日交易量。

链上 2048 ，爆红加密社区

与以往机制复杂、规则繁杂且需要计算各种收益成本的各类 GameFi 产品有所不同，也与各种主打区块链 3 A 大作的次世代链游产品向左。Sui 8192 的机制设计极为简洁，任何人均可立刻上手。

Sui 8192 简单、轻快，他的玩法和曾经风靡一时的 2048 完全相同，只需使用方向键移动方块，并让方块重叠即可。当两个相同的数字重叠时，可以组合成更大的数字，玩家的目标是尽可能组合出更大的数字，并获取更高得分。

尽管在机制上并无创新，但 Sui 8192 仍然引起了加密社区的关注。在加密游戏山河日下的当下， Sui 8192 是近期出现的屈指可数的值得引起关注的全链游戏（Full on-chain game）之一。

全链游戏与 GameFi 有所不同。在目前的 GameFi 范式中，往往只将资产上链，即仅有代币在链上，而游戏运行逻辑则仍然是中心化服务器处理。而全链游戏则将完整的游戏逻辑全部放入链上处理。 以 Sui 8192 为例，这款游戏的全部逻辑均被部署在了 Sui 区块链上，主要包括三个部分：

1.  Game: 用于记录游戏的整体状态和玩家的操作。

2.  Game Board: 游戏的主要逻辑。

3.  Leaderboard: 提供了排行榜功能。

在进行一句游玩时，用户需先支付 gas 费铸造一个 NFT。此后，游戏中的每次移动都会在链上发起一笔交易，并将数据记录在链上。

Odaily星球日报实测发现，第一次 mint 的费用约为 0.2 SUI，此后每次操作的 gas 消耗均在 0.0008 SUI 左右。

链上数据显示，目前玩家们已 mint 了超过 74.1 万枚游戏 NFT。而每个 NFT 都拥有大量的更新记录，以此来记录玩家的每一步操作。

从游玩流程来说，每次操作均需等待链上确认，其体验感较为阻滞。而官方也给出了另一种简易的选择，使用 Ethos 钱包。

Ethos 是什么？

“Sui 8192 ”由 Sui 钱包 Ethos Wallet 的厂商所开发。Ethos 由 Nadia Eldeib 和 Jared Cosulich 联合创立，致力于构建用户友好的终端产品及面向开发人员产品。

在本款游戏中，Ethos Wallet 为玩家提供的预授权功能让玩家可以免去每步操作均需点击确认的麻烦。通过设置最高限额，合约即有资格消耗限制金额内的代币。

除 Sui 8192 外，该厂商还开发了跳棋和国际象棋小游戏，均为与 8192 一样的全链游戏。

与此前常见的中间件厂商类似，Ethos 不仅是一款钱包，也提供了面向开发人员的钱包基础设施。

由该团队所开发的 EthosConnect 是一个开发人员友好的 API，旨在加速开发人员在 Sui 上构建应用程序的工作流程。通过 EthosConnect ，开发者可将您 Sui 的 Move 智能合约与 Ethos 钱包直接连接，并通过其自定义 React 组件和构建用户友好的界面。

Ethos Explorer 则是集成在钱包之中的加密门户。该产品可方便的发现 Sui 生态系统中的 dApp 并管理自己的加密资产。

全链游戏，下一个链游风口？

自区块链和游戏结合伊始，加密世界便在不断探索着游戏和加密技术的结合方式。

就目前的机制来看，游戏的含“链”量大抵可分为三类：完全无需区块链介入的传统游戏、仅资产上链的传统 GameFi 游戏、完全上链的全链游戏。

全链游戏与现有的常见链游，其最大的区别就在于游戏的业务逻辑也被写入链上。由智能合约完成游戏里的一系列运算与业务流程。

在传统 GameFi 中，虽然游戏内资产被放入链上，但游戏的规则、玩家的交互过程、游戏内奖励掉落等业务逻辑均由传统服务器中心化进行，尽管玩家看上去拥有了游戏内的“去中心化资产”，但由于游戏公司对游戏规则有着完整的控制权，其“去中心化”愿景更像是一种美好的海市蜃楼。

但相应的，由于游戏的核心逻辑均需进行链上计算，其成本和性能要求也远高于传统 GameFi。这也使其在现有的基础设施条件下，几乎不可能诞生出“能玩”水平的现代 3 A 游戏。

全链游戏在加密生态中是极为小众的领域。目前来看，全链游戏仍处于极为原始的探索之中，其基础设施远不足及承接系统化的商业行为。但这种早期的探索，也为我们埋下了丰富的想象空间。

8192 的收到关注让我们看到了另一种可能。虽然全链游戏尚不能实现复杂逻辑的现代大型游戏，但规则简单明了的趣味小游戏似乎是一条不错的道路。将 4399 搬到链上，会是一个不错的选择吗？
