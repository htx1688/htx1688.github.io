Aptos域名服务（ANS）

链接：https://www.aptosnames.com/

ANS 是一种将名称映射到 Aptos 地址的智能合约。申请一个域名可获得一枚特殊的NFT。

1.设置钱包，从 chrome 商店下载扩展程序 Martian Crypto Wallet ，创建一个钱包并保存私钥，点击airdrop获取测试空投；

2.铸造ANS： https://www.aptosnames.com/，点击购买即可。

Topaz Network

链接：https://www.topaz.so/

Topaz 是首屈一指的 NFT 市场，建立在 Aptos Devnet 之上，由 Aptos 团队提供支持。

链接钱包即可显示/交易域名等NFT。

Vial Protocol

链接：https://www.vial.fi/

Vial 是一种联合利率协议，允许用户在 Aptos 上借出或借入资产。

1.连接钱包并领取水龙头代币以测试网络。

2.存借

点击 Dashboard并存入自己的代币

点击Deposit，并确认交易。

借贷同理。

Pontem

Pontem正在以 Uniswap 为蓝本，在 Aptos 上开发第一个自动化生产级做市商。

1.下载 Pontem 钱包扩展并创建钱包。

2. 交易

申领代币来执行交换任务。

链接：https://liquidswap.com/#/

连接钱包并点击Request tokens。

Martian Wallet

Martian Wallet是Aptos生态系统中的顶级钱包，由 Aptos Labs 和 Toy Ventures 提供支持。

如前所述下载钱包后，可在 Devnet 上铸造NFT。

链接：https://demo-nft.martianwallet.xyz/

  • 要铸造 NFT，首先输入 NFT 的名称。尽可能发挥创造力。

  • 输入 NFT 的描述。

  • 提供 NFT 图像 URL 并检查预览。可以在 Internet 上的任何位置托管您的图像，然后输入指向图像文件而不是任何其他网站的 URL。您可以通过查看 URL 的末尾来检查这一点，它应该是.png或 . jpg或.jpeg或.gif或.svg和其他图像文件格式。如果预览显示图像，则意味着成功。

  • 现单击“创建 NFT”。（请确保您的钱包中余额>5000 测试 aptos 币。您可以空投将测试币添加到您的钱包）

  • 将弹出一个新窗口，要求您批准交易。

  • 成功交易即可。
