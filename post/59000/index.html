生成式AI热潮无边无际，“卖铲人”英伟达在产业链上的影响力也越来越大。

随着ChatGPT的爆火，全球高科技公司纷纷涌入AI领域开展“军备竞赛”。随之而来的，便是英伟达GPU需求量猛增、云服务商“算力告急”——

从中，英伟达嗅到了商机。

去年，英伟达向AWS、甲骨文、微软等云服务巨头提出了一项**“非同寻常”** 的提议：租用云服务巨头们由英伟达芯片驱动的服务器，然后把这些服务器溢价转租给AI软件开发商。

换句话说，在向云服务巨头卖卡的同时，英伟达还想要利用其硬件端“无可取代”的强势地位，向这些云服务巨头们“收税”。

01 向云服务市场「进军」！    

据媒体最新报道，微软、谷歌和甲骨文都同意了英伟达的提议，但AWS没干。

分析认为，英伟达这种意图与云服务用户建立直接关系来获取收入的做法，虽不一定会影响云提供商的现有收入，但可能严重会削弱云提供商的影响力，让这些云服务公司从“巨头”变“小弟”。

对此，AWS和谷歌云前经理Joshua Bernstein表示：

它将英伟达的品牌置于云提供商品牌之上。这表明英伟达意识到了自己在市场中的地位，以及需要采取哪些措施才能保持这一地位。

某种程度上来说，英伟达云服务DGX Cloud，便是其“野心”不再局限于当“卖铲人”、卖芯片的最新证据。

此外，英伟达最近还开始向Adobe、Getty Images和Shutterstock等客户单独出售AI软件，这些客户表示正在使用它来构建AI模型。

分析称，未来，英伟达还可以利用它与DGX Cloud客户的关系来销售更多此类AI软件。

02「3000亿美元」的机会    

8月份召开的季度财报电话会上，当被问及这个新的云服务时，英伟达首席执行官黄仁勋直言，云提供商“非常喜欢它”。

不过，接受该提议的云提供商发言人均未置评，也没有在其官网上推广该服务，只有甲骨文在3月份发布了一份新闻稿。

Bernstein分析师Stacy Rasgon表示：

我完全可以理解为什么亚马逊拒绝（英伟达云服务），因为归根结底，英伟达才是真正拥有客户关系的一方。

Rasgon表示，相比之下，同意推出DGX Cloud的云提供商规模都比亚马逊小，所以从理论上他们可以在争夺云计算市场份额方面对亚马逊形成竞争。

英伟达副总裁兼总经理Charlie Boyle在接受采访时反驳了英伟达正在与传统云提供商竞争的观点，称新的云服务“并不是英伟达抢走客户，这是英伟达与云提供商的共享之举”。

在谈到亚马逊拒绝英伟达服务时，他表示：AWS一直是“伟大的合作伙伴……但每个公司的业务都有自己的限制”。

英伟达云计算服务帮助客户开发AI模型，并为他们提供定制化预先训练模型。这些预先训练模型包括英伟达用于生成文本的Megatron 530B模型（类似于ChatGPT），以及用于识别视频片段中的人物的People Net视觉模型。

目前，英伟达DGX云服务已经获得了一些大买家的青睐 ，如IT软件巨头ServiceNow、生物制药公司Amgen和保险公司CCC Intelligence Solutions。黄仁勋上个月告诉分析师，DGX Cloud取得了“巨大的成功”，但没有透露其收入的具体细节。

同样重要的是，云服务使英伟达能够销售用于管理开发AI模型的大数据集的软件，与Databricks、Snowflake以及微软的类似产品竞争。

英伟达在8月份公布的新季度财报中告诉投资者，向开发AI或VR应用程序的公司销售软件，可能会带来3000亿美元的潜在收入机会。

首席财务官Colette Kress在8月份的季度财报电话会议上对分析师表示，软件业务每年创收数亿美元，这虽然与芯片业务相比只是九牛一毛，但仍有望实现增长。

03「关系变得复杂」    

对于传统的云服务提供商来说，DGX Cloud的崛起可能会让他们退化成“中介”。

例如，ServiceNow使用DGX Cloud开发AI软件，汇总IT请求并为机器人客服提供支持。

这家IT软件巨头的高级副总裁John Sigler表示，英伟达的服务使ServiceNow可以更轻松地在自己的数据中心以及跨多个云提供商同时运行其新的AI软件，它可以使用“单一软件平台”，并同一由英伟达来管理流程。Sigler说：

如果（英伟达的云服务）使用微软或谷歌的数据中心，那就完全没问题。

根据The Information的云数据库，ServiceNow每年在微软云服务上花费至少7500万美元。

从英伟达的角度来看，在传统云提供商的数据中心内推出这种云服务是公平的。尽管英伟达和云提供商彼此需要，但这种关系正变得更加复杂和有争议。

与此同时，AWS、微软和谷歌已经向云客户出售或正在开发自己的AI服务器芯片，希望减少对英伟达芯片的依赖，关系正变得越来越复杂……

04「DGX Cloud有多强」    

据媒体报道，英伟达的工程师利用他们对公司芯片的了解，对DGX Cloud服务器进行调优，使其性能优于其他云提供商的服务器。

Boyle表示，英伟达使用其软件来提高 DGX Cloud 服务器的性能。8月份，老黄也在电话会上表示，英伟达和云提供商的工程师“手拉手”工作，以提高DGX Cloud 的性能。

截至目前，至少有一位 DGX Cloud 客户表示已经注意到了DGX Cloud与其竞品之间的差异。AI软件公司Anyscale发现，与传统云提供商为客户提供的GPU服务器相比，DGX Cloud的性能“非常高，而且售价也非常有竞争力”，联合创始人 Ion Stoica 说。

他补充称，DGX Cloud 可能会成为一项大业务，因为GPU服务器非常难找，一些公司正使用多种云提供商，尽可能多地获取它们的访问权。因为DGX Cloud跨多个云提供商运行，它可能会成为AI开发人员的一站式解决方案。

此外，英伟达还将 DGX Cloud 定位为客户快速访问GPU服务器的一种方式，而无需向云提供商作出大量多年的承诺以确保这样的访问权限。DGX Cloud客户可以快速以每月3.7万美元的价格租用内嵌 8 个 A100 GPU的云服务器。

不过，Boyle拒绝透露 H100 GPU 的定价：

我们的定价是一个高端服务，我们肯定不是为了与任何人竞争价格而存在的。
