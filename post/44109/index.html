传统行业中最有技术含量的分析工具也有其局限性。

为什么呢？原因很简单，数据是专有的。

无论是来自华尔街季度报告的数据集，还是你从Facebook上提取的数据，这些都是公司想要披露的数据。

这就限制了分析的深度。

但在web3中，几乎所有的数据都是公开可见的。

链上侦察是一件任何人都可以做的事情。

每次有行业大事件发生时，加密推特上数十位匿名分析师都会抢先提供他们的事后分析。

而你也可以。

以太坊和其他顶级智能合约区块链充斥着数据。

如果你能很好地实时分析这些数据，就会有大量的alpha甚至工作机会等着你去解锁。

因此，Bankless这篇文章将引导你通过你可以使用的最佳资源来提高你的加密分析技能。

目标：了解顶级加密分析工具

技能：初学者

耗时：1小时即可开始

投资回报：了解“解读”区块链的最佳方式

为什么要进行加密分析？

我第一次发现加密货币是在2017年春天。

那年夏天，我尽可能多地研究了这个领域。然后在那年秋天，我在这个领域找到了第一份工作，在一家名为CryptoAnalyst的小型媒体从事加密新闻写作工作。

当时我对加密货币了解不多，区块链分析领域在当时非常早期和稀少。

从那以后，我一直在这个生态系统中工作，在过去的几年里，有一件事特别让我印象深刻，那就是加密分析无论是在数量上还是质量上都出现了激增。

这对我们所有人来说都是好消息，因为知识就是力量。

更确切地说，在加密领域，知识就是力量和金钱。

这句格言在2017年ICO ERC20代币热潮中与今天DeFi和NFT的寒武纪大爆发中一样真实。

但是，当时和现在之间最大的区别之一是当代高质量的加密分析工具的激增，这些工具使可操作的web3知识变得更加容易获得。

让我们来探索你可以用来加强你的加密研究的5个平台。

1. CoinGecko和GeckoTerminal

CoinGecko

几年前，似乎每个人都使用CoinMarketCap来查看加密货币的价格、市值、交易量等。

随后，币安于2022年4月收购了CoinMarketCap，许多人认为该平台已被“占领”。

社区开始寻找替代平台。近年来最成功的平台是CoinGecko，它在熊市期间悄悄超越了竞争对手，现在是全球最大的独立加密数据聚合商。

除了独立性，我最喜欢CoinGecko的一点是，它给人的感觉是，这个平台总是走在趋势前面。

每当我想要查看一些新的加密货币的价格走势或交易量时，CoinGecko似乎总是领先于其他平台拥有该代币的基本数据。

这种令人印象深刻的响应能力的部分原因是什么？

CoinGecko的覆盖范围非常大——该平台目前跟踪超过600个加密货币交易所的超过13000种不同加密资产的关键数据。

GeckoTerminal

因此，如果你想实时跟踪最大项目以及新币和大涨大跌等类别的加密趋势，CoinGecko是非常可靠的。

但这还不是全部——该项目最近还推出了GeckoTerminal。

GeckoTerminal是一个DEX聚合器。

还记得我说过CoinGecko的覆盖范围巨大吗？

GeckoTerminal也是如此——如果你想交易一个代币，无论它是在以太坊虚拟机(EVM)链上还是非EVM链上，GeckoTerminal都可能有你想要直接分析的图表和流动性池。

2. DeFiLlama

DeFiLlama

对于DeFi超级用户来说，使用DeFiLlama是一种享受。

为什么？

因为DeFiLlama精确而快速地跟踪+150条链和大量不同协议的总锁定价值(TVL)数据交易量，这意味着该平台是在任何特定时间“从高处看加密经济”的绝佳方式。

然而，DeFiLlama不仅仅是一个普通的DeFi统计中心。

该项目还提供实时收益指标，稳定币图表、清算活动、空投机会、黑客信息等。

换句话说，一旦你去了“Llama”，你就再也回不去了。

3.Dune Analytics

在过去的几周里，Reddit NFT爆火。

然而几个星期前，我对它们一无所知。

当我看到围绕着这些数字收藏品的活动开始激增时，我知道自己需要写一篇Metaversal文章来解释发生的基本情况，所以我写了。

在教别人之前，我去了哪里补习呢？

答案是Dune Analytics。

Dune Analytics是一个允许任何人使用SQL创建其他人可以查看的加密分析面板。

例如，我使用Dune用户petertherock创建的分析面板研究了上面提到的Reddit NFT帖子，它涵盖了我需要的所有内容，如NFT持有者的数量、销量等。

Dune最酷的地方在于，它涵盖了各种链(如以太坊、Arbitrum、Optimism、Polygon等)上的DeFi和NFT项目的数据。

我推荐将Dune作为任何链上分析信息获取的重要组成部分。

4. Nansen

如果有人给我一个挑战，让我在一年内在加密经济中赚到100万美元，并且规定我只使用一个工具，这个工具就是Nansen。

100万美元只是一个假设的任意数字，但Nansen可以帮助你实现的结果是非常真实的。

我在我2022年6月Metaversal的一篇文章Free-to-Mint NFTs中提到了这一点，我在文章中详细描述了我如何使用Nansen的Mint Master追踪器在WAGDIE NFT的地板价格最初爆炸之前进行铸造的个人经历。

当时，我说：

“我通过Mint Master找到了WAGDIE，在周五注意到有近40个‘Smart Minters’参与了该项目的铸造。在项目爆发之前，我积累了一些，现在写这篇文章时的地板价是每个1.5 ETH，所以我的认购在这时候已经翻了好几倍。

当然，并不是每一次铸造都是一场胜利，但Nansen使实时显示信号变得更简单。”

Nansen

今天依然如此。

它不会带领你不断获胜，但它会帮助你以其他项目无法做到的方式，实时呈现出“百分之一”或“千分之一”的精彩剧目。

例如，Nansen提供了以下非常棒的服务：

  • Wallet Profiler — 任何钱包地址的指标

  • Hot Contracts — 用于跟踪趋势智能合约的面板

  • DeFi Paradise — DeFi经济的完整概览

  • NFT Paradise — NFT经济的完整市场视图，包括趋势性藏品和新的mint.

  • Smart Money — 跟踪有利可图的加密货币和NFT交易员活动的面板

  • Token God Mode — 一站式的搜索引擎，用于搜索围绕代币的所有主要信息，如交易量、交易活动等。

5. Token Terminal

Token Terminal

Token Terminal是我最喜欢的加密分析平台之一，因为它可以说拥有最好的区块链、协议的费用和收入的数据。

想知道一个特定的加密项目在任何时候的盈利情况吗？

Token Terminal可以满足你的要求，无论你是研究跨某个垂直领域的项目，例如借贷市场，还整个链的项目。

成为一位更好的加密分析师

如果你学会了如何驾驭上面列出的5个加密分析项目，你将成为比现在大多数人都更好的加密分析师。

好消息是，这些平台使浏览关键区块链数据变得简单，因此不需要特别专业的知识就可以熟练使用它们。

探索和使用这些项目，你可能会开始在链上取得你以前不知道的胜利。
