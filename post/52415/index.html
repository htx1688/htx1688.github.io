就在苹果WWDC大会召开前四天，Meta在昨夜突然宣布了他们的下一款头显设备：Meta Quest 3 ，并表示该设备是“第一款具有高分辨率彩色混合现实”。此前，有消息透露出，苹果将于WWDC 2023大会上推出MR头显设备。

这一波抢跑，无疑透露出Meta和苹果头显较量的决心。前有PSVR2，后有苹果MR头显，Meta Quest3又有哪些亮点值得关注，我们来一探究竟。

Meta迄今最强大的头显设备

它是一款使用Meta Reality技术的混合现实（MR）头戴式设备，同时也支持虚拟现实（VR），因此可以用于两种类型的游戏和应用程序。

事实上，Meta公司以前称为Oculus的部分表示，将在9月27日公司的Meta Connect展示会上公布更多信息，因此我们预计Quest 3的发布日期将在那之后不久。

Meta Quest 3一览

以下是Meta Quest 3的一些技术和参数细节：

  • 128GB版本的头戴式设备起价为499.99美元。还将推出一个存储容量更高的版本，具体容量尚未确定。

  • 它由下一代高通骁龙XR芯片提供动力，相比Quest 2，图形性能提高了两倍。

  • 该头戴式设备是采用Meta Reality技术的混合现实（MR）设备，同时也支持虚拟现实（VR）。

  • 头戴式设备采用“更轻薄、更舒适”的外形设计，此外Meta表示，由于平板透镜设计，它比以前的版本薄40%。

  • 该头戴式设备提供比Quest 2更高的分辨率。它还提供高保真度的彩色透视功能，可以将真实世界与虚拟世界融合。

  • 设备配备了4MP的RGB彩色摄像头、深度传感器和比Quest 2多10倍的透视像素。

  • Quest 3使用了重新设计（没有环绕！）的Touch Plus控制器，但如果需要，也可以使用Quest Pro的Touch Pro控制器。Touch Plus控制器还使用了首次亮相于Touch Pro的TruTouch触觉技术。设备还支持直接触摸（Direct Touch）的手部追踪，因此用户不一定需要使用控制器。

  • Quest 3向后兼容Quest 2的500多款VR游戏和应用。

  • Quest 3将在目前支持Meta Quest的所有国家发售。

Meta Quest 3价格

马克·扎克伯格此前曾表示，Meta Quest 3的价格将在300美元至500美元的范围内，但我们没有预料到它会正好位于这个价格区间的顶端——499.99美元。

这个价格相对较高，必然会引起与索尼PlayStation VR 2的比较 。我们听说Quest 2的价格可能会降至299美元，这将使对虚拟现实稍感兴趣的消费者更加倾向于购买。

目前的Oculus Quest 2已经降价至299美元，这样的价格更加合理。

Quest 3 设计

Quest 3的外观肯定比其他Oculus/Meta头戴式设备更时尚——得益于使用的透镜技术（下面会详细介绍），该头戴式设备比Quest 2薄40%。

Quest 3使用了更符合人体工程学设计的Touch Plus控制器版本（如果需要，也可以使用Quest Pro的Touch Pro控制器）。值得注意的是，这些控制器上现在没有外部追踪环。Meta表示，这使得它们更像是用户手部的自然延伸。它们还占用更少的空间。

Quest 3 硬件设备

Quest 3的硬件配备了几个显著的特点，提升了用户的整体体验。以下是Quest 3硬件的一些关键方面：

1. 高分辨率显示屏 ：该头戴式设备提供了比其前代产品Quest 2更高的分辨率显示屏。这种更高的像素密度使图像更加清晰和详细，为用户提供更沉浸式的虚拟现实体验。

2. 下一代高通骁龙XR芯片 ：Quest 3采用下一代高通骁龙XR芯片驱动。该芯片的图形性能比Quest 2提升了两倍，使得游戏和应用程序的性能更流畅，图形更加逼真。

3. 更纤薄的设计 ：Quest 3采用了更纤薄、更舒适的外观设计。采用了煎饼透镜设计，使得该头戴式设备比Quest 2薄40%，使其更轻便、更符合人体工学，适合长时间佩戴。

4. 混合现实和虚拟现实支持 ：Quest 3是一款混合现实（MR）头戴式设备，采用Meta Reality技术。它结合了MR和虚拟现实（VR）的元素，使用户可以体验真实世界和虚拟内容的融合。

5. 高保真度彩色透视 ：该头戴式设备提供高保真度的彩色透视功能，可以将现实世界与虚拟环境无缝融合。这个功能增强了虚拟体验的沉浸感和逼真感。

6. 升级的跟踪和控制器 ：Quest 3配备了改进的Touch Plus控制器，提供更自然和直观的与虚拟对象的交互。此外，Quest Pro的Touch Pro控制器也与Quest 3兼容。

7. 向后兼容性 ：Quest 3与Quest 2的超过500款VR游戏和应用程序目录保持向后兼容。用户可以在新的头戴式设备上继续享受现有的VR内容。

这些Quest 3的硬件改进为用户提供了更先进、更沉浸式的虚拟现实体验，提供了改进的图形、舒适性和交互功能。

“从根本上说，我们的愿景是让用户以一种直观和愉悦的方式穿越所有现实。”这是Meta的虚拟现实副总裁Mark Rabkin的一种相对偏见的想法。他表示，Quest 3将超越“虚拟现实”和“混合现实”这两个刻板分类 ，提供真正的下一代体验，让用户轻松地将物理和虚拟世界融合在一起。

“Meta Reality既带给用户深入沉浸的虚拟现实的魔力，又赋予用户使用MR使物理世界更有趣和实用的自由和乐趣。我们对开发者和创作者在Quest平台上可以构建什么样的内容充满期待。”

Quest 3 应用和游戏

根据2023年初《The Verge》的Alex Heath的报道，Meta的VR副总裁Mark Rabkin表示，将有"41款新应用和游戏适配Quest 3平台" 。

在Meta Quest Gaming Showcase上，我们对Quest 3游戏有了很多了解。Meta谈到了16款游戏，其中包括新作、更新以及经典重制版。

该活动的最重要的揭示是《阿斯加德之怒2》和《刺客信条：Nexus VR》。还有《怪奇物语VR》、《我期待你死亡3：机器中的齿轮》和《第七位客人》。

我们还了解到Quest 3与当前的Quest 2应用和游戏库兼容，该库包含超过500个应用和游戏 。在宣布Quest 3的时候，Meta列出了以下几个例子：“现在你可以在厨房桌上玩虚拟棋盘游戏《Demeo》，通过《绘画VR》在客厅装饰虚拟艺术，或者沉浸在一个完全沉浸式的世界中去做一些其他方式不可能的事情。选择权在你手中。”

在2022年的Meta Connect活动中，该公司解释说他们与微软合作开发了一些额外的功能。其中许多功能与Office有关，但也与Xbox Game Pass集成。该功能正在逐步推出到Quest 2，因此我们有信心在Quest 3中也能看到它的出现。

Meta Quest 3值得等待吗？

Meta Quest 3被认为将成为一款巨大的消费级VR头显。虽然它可能缺乏一些更好的装备，如凹点渲染和眼球追踪，但是通过平凡透镜的提升，它将带来更高的视觉逼真度，从而实现一些最佳的无线VR体验。

Quest 3虽然价格不菲，但一款更轻薄、带有全彩透视功能、新的芯片组和独家游戏功能的头戴式设备听起来很吸引人，它也确实指引了混合现实的下一个步骤，我们认为等待Meta流行的VR头显系列绝对是值得的。
