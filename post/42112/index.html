当下互联网正在进行一场Web 3.0革命。

2022年，我们可以看到，人们可以在元宇宙内听音乐会，可以通过稳定币进行跨境转账，也可以玩网络游戏赚钱，一个接一个的创新，似乎正在昭示着以太坊联合创始人加文·伍德（Gavin Wood）最初提出的用户可以拥有产权的互联网，即Web 3.0时代（web3.0含义：一组保障人们在低壁垒市场中为自己行动的兼容协议，个人反对暴力权威的自由的基础）的加速到来。

而在过去一年，Web 3.0概念火爆出圈，开始走进大众视野。正如那句网络流行语所言——得不到的永远在骚动，当普通人也有机会接触到Web3.0后就会发现，哦，原来被吹得神乎其神的Web3.0依旧普通，依旧存在诸多短板，甚至成为了“庞氏骗局”新的温床，在吹捧与怀疑矛盾环境中，Web3.0还能是互联网的下一站吗？

反思Web 3.0

1764年，英国纺织工詹姆士·哈格里夫斯因一场事故，意外做出一台能带动八个竖直纱锭的新型纺纱机（珍妮纺纱机），让旧式纺纱机的功效提升了八倍。但是珍妮纺纱机的出现引发了手工纺纱者的恐慌，他们冲进哈格里夫斯的家里捣毁了机器，但还是没有阻挡工业革命的滚滚洪流。

从1996年刚开始使用电话拨号的Web1.0时代，到2016年已经开始锚定的Web3.0概念，这背后都是生产力提升与权益分配更加公平的表现。Web1.0首先建立了“联系”，Web2.0则开始以技术的突破为基础进行跑马圈地，积攒流量，而Web3.0则是要破除生产力提升后，权益被集中在少数寡头的局面，是每个用户可以拥有产权的互联网。

但当下真的去中心化了吗？首先从技术上来看是还未做到去中心化的，在理想的区块链生态系统中，服务提供商、DApp 和分散式系统将运行自己的节点，以完全点对点和分布式的方式验证信息和数据，但事实上，由于部署和维护相关节点的成本非常高昂，大量 DApp 应用程序正依赖于以 Infura、Alchemy 为代表的 IaaS（Infrastructure as a Service）服务商，通过使用它们提供的区块链和 IPFS 的API（API 即应用程序编程接口，是 Web 2.0 的基石，通过它可以将不同的程序软件连接起来）调用服务，解决数据请求。以热门钱包应用MetaMask为例，MetaMask 并没有部署节点以直接与以太坊区块链进行数据交互，而是通过调用 OpenSea、Etherscan 和 Infura的 API 接口来显示用户的 NFTs、最近的交易和账户余额。

其次，参与者仍然是中心化的。虽然从底层逻辑来看，Web 3.0运行在去中心化的P2P网络上，具有固有的去中心化属性，但在前端，用户却仍然需要通过交易所等应用接口/设施实现交互，这些应用接口/设施不仅能够知道用户的身份信息，还掌握着用户的资产状况，本身则拥有了类似传统网络下平台的权力。例如，在最近的俄乌冲突中，乌克兰数字化转型部向Coinbase、Binance、Huobi 等八家加密货币交易所发送信函要求其停止为俄罗斯用户提供服务；同时，受美国监管部门的制裁要求，MetaMask和OpenSea已纷纷禁止了来自俄罗斯的账户，这明显是违背了 Web 3.0 抗审查的基本原则。

最后，当下许多Web3.0项目本身也是中心化的。 Web3.0项目的底层技术都是区块链技术作为支撑，但区块链“不可能三角”理论指出，无论区块链网络采用哪种共识机制，都无法同时兼顾成本效益、准确性和去中心化，也就是说Web3.0项目本身也难以达到理论中去中心化的效果。

Web3.0的展望

通过前面简单的分析我们可以看到，Web3.0当下最核心的问题并非是一些鱼目混珠的项目，而是其本身底层技术与底层逻辑规则的不完善，如果随着基础设施和工具的持续改善，Web3.0或许将借助区块链底层协议创新周期的网络效应实现从一到 N 的增长。

当前，我们正站在从 1 到 N 的前夜。以史为鉴，当下从用户规模的角度来看，Web3.0现在与1990年代互联网刚萌芽时几乎相当（相对网民的绝对值来说，比不上1990年代的网民规模），可以说还处于一个非常早期的阶段。 但是我们也应看到Web3.0赛道中的一些新事物、新变化，根据数据研究机构 Apptopia 的报告，2022年可供下载的Web3应用程序数量的增长速度几乎是2021年的 5倍，这个行业正在真正开启尝试投机之外的实际商业应用。

其中最常见的商业应用便是数字钱包，其中MetaMask 是众多数字钱包中最为大家所熟知的一款Web3.0应用，根据 Metamask母公司ConsenSys今年三月发布的消息，MetaMask每月有超过3000万活跃用户。

再比如在浏览器领域中，目前市面上主流的Web2.0浏览器，无论是谷歌的Chrome、还是国内的360，其都或多或少存在着售卖用户信息给广告商的变现模式，而这其中自然就会存在着用户隐私泄露的问题。而当下Web3.0已经在着手改造浏览器了，以Brave浏览器为例，他推出了自家的Bat代币，如果用户想要获得激励，那么可以通过浏览器中弹出的广告获取代币，从而实现用户权益自主化；如果用户不想观看广告，那么用户完全可以屏蔽掉所有广告，这一机制则实现了用户、平台与广告主三方的共赢。

这是一个良好的开头，未来相信还有更多具备实用性的Web3.0应用涌现出来，而当这些应用慢慢开始从消费者和企业身上获得经济价值，从而产生真实、可持续的现金流，将逐步推动 Web3.0行业超越现有的逐利生态，走上真正良性、健康的发展路径。
