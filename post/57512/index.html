近期，圈内最大热点当属仍在飞升的去中心化社交平台 friend.tech。链上分析师 Spot on Chain 表示，该项目仅用 11 天就产生了 2953 枚 ETH（约合 500 万美元）的手续费收入。目前用户份额总价值达到 4435 枚 ETH（约合 740 万美元）。不过，friend.tech 也暴露出一些问题，如通过 API 泄露数据，以及用户可以在无邀请码的情况下从合约买卖份额等。

当前，friend.tech 的热潮持续，各路 KOL 或盛赞或 diss，好不热闹。Odaily星球日报曾在《Friend.tech 蹿红，个人股票是如何定价的？》一文中对其定位、玩法和定价规则做详细介绍。而今天，我们暂且搁置争议，专注吃瓜，盘点下 friend.tech 上最“值钱”的十大人物，并重点圈出其中四位做介绍。

#1. Racer：friend.tech 创始人

Racer 是目前 friend.tech 上最“贵”的用户，目前有 151 个 holder，其份额价格已经高达 3.1 ETH。

虽不贴切，但为直观，我们以蓝筹 NFT 来对标：这一“地板价”水平接近 Azuki（3.5 ETH）。

在 Paradigm 宣布投资的一则推文里，Paradigm 公开了 friend.tech 的两位创始人，这也让二人的身价迅速飙升。

遗憾的是，目前 Racer 已将自己的推特设置为隐私模式。未经批准的关注者无法查看其推文。

近日，加密 KOL nix 发现，Racer 曾发行过一款山寨 NFT，且项目已经“软 rug”。我们或许可从这些信息一窥 friend.tech 的过去。

2022 年底，NFT 项目 Kosetto 诞生。这是一款主打贴纸产品（类似于“表情包”），用户可以将自己的 NFT 贴纸复制到剪贴板，并在其他地方使用这些图片。

2023 年 1 月 29 日，Kosetto 正式上线 Chrome 插件商店，用户可在该插件上使用贴纸功能。但遗憾的是，该产品并未获得流行。仅拥有 3 个下载数量。且版本更新停滞在了 1 月 31 日。

Kosetto 官方推特的最后更新停留在 1 月底，目前拥有约 3700 名粉丝。

#6. iam4x：friend 生态开发者

iam4x 是 tuleep.trade 的创始人。tuleep.trade 是一款开源交易应用，用户可通过交易平台 API 接入本产品并使用。

在 tuleep.trade 的描述中，它将自身定义为“新闻交易应用”。与其他交易平台不同，该产品在交易的 UI 中集成了实时的新闻消息。用户交易标的的最新消息都会在交易面板中呈现。这让用户可以更快速、及时的对市场信息做出反应。

当然，这一产品并不足以使人们对其份额产生巨大的热情。除 tuleep.trade 外，iam4x 还开发了 realfriend.tech。

这是一款 friend.tech 的交易机器人，用户可在该产品上完成 friend.tech 的自动化交易。

在产品说明中，iam4x 还诚实地做出说明：他本人使用的机器人与该产品有所不同，而是与之相较更快的一款机器人。

目前，iam4x 的份额已产生 170 ETH 的交易量，他本人也赚取了 8.5 ETH 的手续费收入。

#9. FaZe Banks：一声不发，血赚 44 ETH

FaZe Banks 曾经是一位著名 Youtuber，他在 Youtube 拥有 504 万粉丝，目前已有 4 年未更新视频。

而他另一个更为人所知的身份是 FaZe 战队的创始人。FaZe Banks 真名 Richard Bengtson，于 2010 年在洛杉矶创办了电子竞技战队 FaZe Clan。FaZe 战队涉及的游戏颇为丰富，包括使命召唤、CS:GO、PUBG、彩虹六号、堡垒之夜等等。

2021 年，FaZe Banks 开始涉足Web3领域，并进行了不少的 NFT 交易。

尽管 FaZe Banks 已经是 friend.tech 最贵的人物之一。但一个有趣的事实是，我们很难将其称之为“推特 KOL”，其最新一条推特发布时间远在 4 个月之前。

（Banks 的“最新”推特）

尽管他并未在推特上活跃，但我们并不知是出于何种原因，Banks 激活了他的 friend.tech。并依靠多年来积攒的人气，在未发一声的情况下成为了 Friend.tech 最贵的人物之一。

目前，Banks 的个人份额交易量达 880 ETH，而他本人也赚取了 44 ETH 的手续费。

#10. Friend Index

Friend Index 是一款非官方的指数产品，由 Octoshi.eth 发起。该账号也是前十大“贵”人中粉丝数量最少的一个，仅不足千人。

正如其名称一样，该账号与其说是一个社交账号，更不如说是一款个人份额的 ETF 产品。Friend Index 会购入 Octoshi 所挑选的“顶级人物”。他的初始资本为 4 ETH，并用 Friend Index 后续所赚取的全部费用作为资本投入。

Octoshi.eth 的个人推特也仅仅只有一千余名粉丝。但其钱包地址却颇为富有，目前有超过 240 万枚的资产。

Octoshi 称，该产品将随市场变动而变动，Friend Index 未来将卖出不再是 Alpha 的个人份额，并买入更多的 Alpha。

需要注意的是，尽管名为“指数”，但这一产品并非被动投资，而是由 Octoshi 进行判断并主动投资。他还表示，未来或将加入由 Friend Index 持有者投票进行投资决策的机制。

#12. Christian 2022.mid

Christian 是近期热度很高的 00 后巨鲸，是加密基金 NDV （NextGenDigital Venture）联合创始人和 NFT 超级大户，此前因 Azuki 遭遇下跌一夜接盘数百枚系列 NFT 而受到关注。

NFTGO 数据显示，Christian 仅单一钱包资产就超过 150 万美元。在该产品的巨鲸综合排名中，该钱包地址也位列第 151 名。

今年 6 月底，在 Azuki 新项目崩溃之后。他曾向 Azuki 创始人 ZAGABOND 提出以 4000 万美元收购项目，Christian 也受到 NFT 社区的拥护，并在对 Azuki 创始人 Zagabond 发起诉讼的去中心化项目 AzukiDAO 社区被选为多签管理人之一。
