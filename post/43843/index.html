过去一年回顾

主要的以太坊L2启动至今已经一年多了。L2链已经发展成熟，形成了明显的生态系统和社区。以rollup为中心的未来仍然岿然不动。

差不多一年前我发了一篇帖子，现在我打算从里面提取一些内容，用以展示L2所取得的一些进展。

那时，主要ORU费用相对较高。

如今的收费情况已经大不相同，这反映了所有L2团队取得的进展。

就TVL市场份额而言，L2的排名在去年有所上升。

这是2021年11月的情况。

这是 2022 年 10 月。

在排名前10的链中，只有Arbitrum 和 Optimism 的 TVL 在过去一个月中有所增加。

在去年的过程中，我们有相当数量的用户已经桥接到 L2。这些是 Arbitrum、Optimism、zkSync 和 StarkNet 的统计数据。

然而，尽管技术取得了进步，但如果我们持批评态度并以与以太坊相比的活动来查看两个最主要的Rollups，那么两个 L2 仍有一段路要走。尽管 Arbitrum 和 Optimism 提供的费用和速度较低，但在日常活动或日常交易中并未超过以太坊。如下是 L2 目前在采用方面的高级数据快照。

关于 Optimistic Rollups

除了特定于应用程序的汇总之外，Optimistic Rollup 是第一个看到大量用户从 L1 桥接资金的通用汇总。它们是人们第一次真正有机会获得以太坊的安全保证，而无需支付高昂的gas费。

Arbitrum 已经发布了他们的旗舰升级产品：Nitro。到目前为止，他们在任何通用Rollup中获得了最多的采用。尽管出现了熊市，但他们的每日交易量已达到 4 倍，每日独立交易量翻了一番。

Arbitrum Nova是 Arbitrum 提供的产品套件的补充。一个非常适合游戏和社交应用程序的新链。Reddit 的社区积分将存在于 Arbitrum Nova 上。

他们收购了Prsym Labs。Prysmatic Labs开发并仍在运行计算机软件Prysm ，超过 43% 的以太坊节点使用该软件运行以太坊网络。

如果验证者可以在单个设备上运行 Prysm 客户端，那么他们很可能使用相同的硬件运行以太坊和 Arbitrum 序列器（并获得 Arbitrum Token）。

Optimism 代币现状？

很明显，Optimism 的原生代币 OP 已经达到了预期的效果，日常交易也在上升和向右。

Optimism每日交易

现在有超过 150,000 名 OP 持有者。OP 以许多不同的方式分发，但它现在主要用于支持生态系统的增长。

Perpetual Protocol 目前在 Optimism 上看到最多的交易，但 Uniswap 和 Velodrome 看到最独特的地址与它们交互。

Optimism 和 Arbitrum 的另一个有趣指标是他们在去年的开发者采用率。这些统计数据来自 Alchemy 最近的 web3 报告：

Arbitrum 和 Optimism 上的 dapps 生态系统自推出以来都有了相当大的增长。在 Arbitrum 的生态系统网站上列出了 280 多个项目，在 Optimism 网站上列出了 200 多个。

GMX 已成为所有加密货币中收入最高的 dapp 之一，这表明 L2 原生 dapp 可以获取价值。

更难量化的指标是两个社区的实力。我认为 Arbitrum 已经成功地发展了迄今为止所有 L2 中最大的崇拜。看看社区的规模和 Twitter 上的参与度，以及某些本地 dapp（GMX 和 Magic）有自己的铁杆粉丝的事实，Arbitrum 似乎有一个强大的基础可以发展。

我认为 Optimism 在社区倡议方面也做得很好。我认为他们最近推出的赚取 NFT 的任务执行得很好。非代币公民之家的设计决策是一项伟大的治理举措。

OP 栈和超级链

Optimism 刚刚宣布另一个重大升级即将到来：Bedrock。并且他们在几天前的一篇博文中介绍了“ OP Stack ”。

OP Stack 不仅是Rollup和Optimistic，还是一个模块化的开源蓝图，适用于各种高度可扩展、高度互操作的区块链。

OP Stack 将允许开发人员构建模块化区块链。它是一系列模块，每个组件实现堆栈的特定层：共识、执行和结算。从理论上讲，您可以将以太坊的 DA 层换成 Celestia 并运行比特币作为执行层。

一个团队已经使用 OP Stack 构建了一些很酷的东西。OPCraft 在 OP Stack 自定义Rollup上运行。使用了MUD 链上游戏引擎和@optimismFND的 OP Stack，他们构建了一个 3D 自治世界。（https://dev.optimism.io/opcraft-autonomous-world/）

如果 OP 链决定共享一个 Sequencer 集，“OP 链”将具有高度兼容性，并且能够利用原子跨链可组合性。对于最终用户来说，这一切都应该只是一个逻辑链：超级链。看比赛会很有趣。

关于 ZK Rollups

Zero-Knowledge Rollups 零知识汇总在去年发生了巨大的变化。我不会在这篇文章中解释零知识证明，但是如果你想要一个很好的资源列表来帮助你理解这项技术，请单击此处。

在今年 8 月的 ETHSeoul 期间，以太坊联合创始人 Vitalik Buterin 预测 ZK-Rollups 将赢得与 Optimistic Rollups 的第 2 层扩展战。

该图像是一个简化的生态系统，仅 ZkSync 就有超过 100 个 dapps 构建。

zkSync

zkSync 已经设法获得了大量的唯一地址。

我的猜测是，这主要是 Argent 贡献的结果。

我说的是 Argent 的移动应用程序，它允许任何人从他们的手机创建一个 zkSync 钱包，快速、廉价地使用 DeFi，同时具有以太坊的安全保证，同时使用出色的 UI/UX 导航。加密原著民习惯的所有复杂性都被抽象到后台。

帐户抽象（Account abstraction，简称：AA）将改变人们进入加密货币和交易的方式。StarkNet 和 zkSync 都支持它。下面的线程解释了 AA 的优势。

简单来说：更好的用户体验、更高的安全性、更大的灵活性，以及更多。助记词将成为过去式。

zkEVM

zkEVM 曾经被认为是几年后的事情，但现在 zkSync 正处于发布 zkSync 2.0 的风口浪尖。他们说这将是第一个完全工作的端到端的 zkEVM。

zkSync 宣布已成功完成里程碑 3：合并证明。通过集成有效性证明，zkSync 2.0 正式成为世界上第一个在公共测试网上运行的 zkEVM。

zkEVM 是第一个与以太坊完全兼容的零知识扩展解决方案。它使开发人员可以非常轻松地从与 EVM 兼容的链中迁移他们的 dApp。开发人员可以简单地部署他们现有的智能合约，而无需进行任何额外的更改。

我确实认为我们应该减少一些期望，它实际上预计将在今年年底全面推出 alpha。在此之前，任何用户都无法进行外部桥接。Arbitrum 的联合创始人 Steven Goldfeder 分享了他对所描绘的时间线的怀疑。

我不知道谁需要听到这个消息，但我要说的是，我们离主网上的第一个zkEVM没有12天的时间，没有任何意义。任何推动这种说法的人都是对社区的一种伤害。

我们在 zkEVM 比赛中还有另外两支队伍：Scroll 和 Polygon。

Scroll是我最不熟悉的项目。我认为他们没有花太多时间推销自己。

他们刚刚发布了升级版 Scroll Pre-Alpha 测试网。他们之前的版本吸引了超过 10,000 名用户来测试他们的桥接和 dapp 的演示分支，例如 Uniswap。他们新的测试网升级支持在 Scroll 上部署智能合约，允许开发人员使用他们熟悉的相同以太坊开发人员工具在 Scroll 上编写和部署自己的合约。他们的 alpha 测试网将是下一个。

Starknet

我将再次引用 Vitalik：

“Starkware 由非常聪明的密码学家组成，他们实际上是理智的”。

每次我听到 Eli 和 Uri（StarkWare 的创始人）谈话时，我都会更加相信他们的愿景很有可能成为现实。

StarkNet 一直处于 alpha 阶段。StarkNet 的 Alpha 0.10.1 刚刚在 Goerli 测试网上成功发布。他们仍然在限制可以连接到 StarkNet 的数量，同时他们专注于提高性能。

我不认为 StarkNet 指标现在很重要，因为它仍处于 Alpha 阶段。

但是，展示 StarkWare 技术可能性的指标是 StarkEx 证明者和验证者实现的规模。在 Sorare 和 ImmutableX 上已经铸造了超过 7000 万个 NFT。这比在其他 L1 和 L2 上铸造的 NFT 总和高出几个数量级。这个规模还没有任何其他系统可以匹配。上周 StarkWare（StarkEX 和 StarkNet）的总交易量是以太坊的 1.6 倍（1200 万对 750 万）。作为 L2，所有这些交易都被“汇总”成一个记录在以太坊上的证明，获得与所有以太坊交易相同的安全性和不变性。

Cairo 1.0 是一个大版本，将于 2022 年底发布。Cairo 1.0 已经“为去中心化网络重建”。它旨在支持 StarkNet 作为无许可网络的要求，使协议变得更简单、更安全。

StarkNet Alpha 现在正朝着 Regenesis 发展。Regenesis 的主要动机是利用 Cairo 1.0 带来的新可能性——即序列器 DOS 保护、抗审查和气体计量，这对于 StarkNet 作为去中心化网络至关重要。在 Regenesis 之后，StarkNet 将仅适用于基于 Cairo 1.0 的合约，并且将从一个新的创世块开始，但具有现有状态。

一些人一直怀疑开发人员是否会致力于学习 Cairo 以便在 StarkNet 上进行构建。

有趣的是，AAVE 和 Maker 目前正在 StarkNet 上构建，并且都没有将他们的 Solidity 代码转换到 Cairo。他们从零开始，这表明他们相信 Cairo 提供了他们的协议增强性能。

StarkWare 创始人的论文基本上是大多数 L2 项目将在两年内用原生 L2 代码编写。

Eli Ben-Sasson（StarkWare 创始人）

他们还宣布了STARK 代币：

  • StarkNet 的去中心化涉及一个原生代币和一个新的基础。

  • StarkNet 代币用于治理以及作为网络的支付和质押资产。

  • 已经铸造了 100 亿个代币，并且它们的分配已经开始。

希望 StarkNet 在 2023 年摆脱束缚。

关于ImmutableX的简短介绍

ImmutableX 是一个特定于应用程序的StarkEx zk-rollup，旨在追求游戏。Immutable 已经提供了令人难以置信的规模，这是支持游戏所需的（9k tps）。我长期关注的区块链游戏之一是 Illuvium，我认为 ImmutableX 决定将其作为自己游戏的家园，这是对他们的极大信任。

Aztec

OFAC 最近对隐私项目造成了很大的破坏。许多加密货币本地人对此非常关注。Aztec 带来了一线希望。

Aztec 是一个 zkRollup，允许用户私下与第 1 层以太坊上的协议进行交互。

项目不需要重构代码库以在汇总上重新部署。他们甚至不需要使用转译器来部署到 zkRollup。Aztec 基本上就像 VPN 一样，允许用户在 L1 上本地使用协议。如果你想更深入地挖掘，Jason 已经整理了这个很棒的线程。（线程：https://twitter.com/mrjasonchoi/status/1583122134334783489）

关于 Polygon Rollup

我觉得我不得不把Polygon单独拿出来讲。Polygon 已经发展成为一套自己的以太坊扩展工具。你可以看到他们有许多 L2 解决方案，大部分是被收购的独立团队/项目。

或者换一种说法。

在 L2 扩散之前，Polygon PoS 被称为 L2，但现在被定义为 EVM 侧链。

Polygon zkEVM 肯定会成为第一个投入生产的 zkEVM。

正如您对创始人所期望的那样，Mihailo 完全支持 Polygon zkEVM 作为最全面的解决方案。

Polygon 最近宣布，他们的 zkEVM 现在是第一个达到重要里程碑的 zkEVM：我们正在进入我们的测试网，其中包含一个完整的开源 ZK 证明系统。

值得注意的是，Polygon zkEVM 的测试网版本的吞吐能力有限，这意味着它距离作为优化扩展机器的最终形式还很远，它还有很长的路要走。

Polygon 令人印象深刻的一件事是他们的业务开发团队。他们确实设法让大品牌加入进来。这不是 L2 特有的，但他们最近与星巴克和 Robinhood 合作。

他们在获得品牌采用方面有着令人印象深刻的记录，因此我确信管道将有助于采用他们的 L2 解决方案。

MATIC 是我假设将在其整个扩展解决方案套件中用作费用代币的代币，这可能会给代币带来相当多的内在价值。

L2 价值捕获

有很多人预测，在未来几年，L2将是一个比ALT L1轮换交易更有利可图的游戏。

但是 L2 代币是如何获取价值的呢？

OP 的收入模式基于其发送到基础层的排序器处理交易。排序器费用由两部分组成，L2 执行费用和 L1 数据费用。

目前，他们正在将这些收入投资于公共产品，以建立生态系统，从而带来更多的用户和收入。

Optimism 或任何 L2 都可以决定将收入分配给质押者，但这在 L2 充分成熟之前是没有意义的。

使用 OP 堆栈构建的链可以决定使用 OP 代币为其链提供动力，这会产生对代币的更多需求。

STARK 的代币将用作费用代币，而不是 ETH。尽管 Starkware 表示用户也将能够以 ETH 支付“以获得良好的用户体验”。这意味着 ETH 很可能会通过 DEX 自动转换为 STARK。

ImmutableX 已经要求在 IMX 中支付每笔交易 20% 的协议费用。如果您目前没有 IMX，他们会通过在公开市场上购买 IMX 来转换费用（保持无缝的用户体验）,然后将这些费用分配给质押者。

MEV 将成为 L2 代币获取价值的另一种方式。L2s 可以看到更复杂的 MEV 和验证器和排序器的协调。MEV拍卖出售排序交易的权力，获胜者获得区块的排序权力。拍卖所得的收益随后可以发送到 DAO 的公共物品库，或通过回购或销毁的方式重新分配给代币持有者。

如果Rollup有数以百万计的用户，那么很明显他们有很多方法可以获取重要价值。

L2 的下一场游戏是什么？

所有 L2 都将奖励他们的社区，因为他们是忠诚和积极的参与者。L2 想要将他们的代币交到社区手中。

现在你可以通过使用 dapps 来赚取 OP，生态系统参与将持续进行空投，例如 OP 现在也有一个持续的任务。

至于其他 L2，你应该探索所有的生态系统、桥接、玩 dapps、使用测试网、成为Discord社区的活跃成员等等。

以 Rollup 为中心的路线图将为 ETH 带来红利，我预计最强的 L2 最终将获得价值。然而，代币的分发将需要数年时间，就像以太坊的有效分发需要数年时间一样。我们将在很长一段时间内看到低流通供应，您在评估价值时必须考虑完全稀释的估值。

进入下一个周期，L2 代币也将具有闪耀的新状态。我们都知道散户喜欢闪耀的新代币……
