编译：深潮 TechFlow

2023 年是 ZK-rollups 的一年。ZK 是以太坊扩展的未来 ，随着 zkSync、Starknet 和 Polygon zkEVM 即将推出的代币，叙事开始升温。以下是最好的已发布（和未发布）的 ZK 项目，请密切关注。

在这篇文章中，我将介绍：

  • 什么是 ZK-rollups

  • ZK 叙事（以及为什么它正在升温）

  • 最好的已发布的 ZK 项目

  • 最好的未发布的 ZK 项目（包括空投机会）。

什么是 ZK-rollups？

ZK-rollups 是第二层区块链协议，它处理交易、执行计算并在链下存储数据，同时将资产保存在链上智能合约中。

以太坊由于其设计，有固有的可扩展性问题。虽然有最终解决这些问题的路线图，但这个过程需要很长时间。

ZK-rollups 通过将交易 “汇总”到链下执行的批次中来帮助缓解拥塞。

Zk-rollups 将所有的地址和各自的余额存储在 Merkle 树中。这个 “余额树 “的根存储在链上智能合约中。

他们有 3 个主要任务：处理存款、转账和取款。 这张图片代表了一个（简化的）ZK-架构。

总而言之，ZK-rollups 是“混合扩展解决方案”（链下协议），它们独立运行但从以太坊网络中获得安全。

为什么 ZK 的叙事正在获得关注？

今年有一些主要催化剂使该板块的兴趣激增：

  • zkSync 代币即将推出。

  • StarkWareLtd 即将推出@Starknet 代币。

  • Polygon 的 zkEVM 马上到来。

这导致与 ZK 相关的协议，如$MINA、$LRC、$IMX 和 $DUSK 表现出色。

Dragonfly 的管理合伙人 Hosseeb 将 ZK-rollups 称为 2023 年最值得关注的叙事。

然而，最近的上涨很可能是由投机驱动的，因为关键的链上指标（活跃用户、交易、跨链存款）并没有随着价格上涨。

那么，这种短期的反弹是不可持续的吗？也许。

然而，持续反弹更有可能发生在 ZkSync、Polygon zkEVM 和 StarkNet 发布之前（和发布之后）。新用户的增加和新的流动性流入生态系统将有助于实现这一目标。

因此，为了做好准备，我列出了一份已发布和未发布的与 ZK 有关的加密项目清单（排名不分先后）。

已发布的 ZK 项目

$MINA

MinaProtocol 是 “世界上最轻的区块链”，它使用 zk-SNARKs，同时使用 PoS 共识机制。

Mina 极大地减少了每个用户需要下载的数据量， ZK-Rollups 允许节点存储较小的证明，而不是整个链。

$IMX

Immutable 是一个第二层的扩展解决方案，旨在为 NFT 交易和铸币提供近乎即时的确认和近乎零的 Gas 费用。

IMX 依靠 StarkWare ZK-rollups 来实现可扩展性，并计划为不同的用例支持多个 L2/L3 ZK-rollups。他们把自己描述为 “Web3 游戏的开发者平台”，Illuvium、Gods、Kongregate 等流行 Web3 游戏都建立在 IMX 上。

$LRC

Loopring 是一个为扩展和构建去中心化的加密货币交易所而设计的 Layer-2。 他们利用 ZK 证明，允许任何人建立一个高吞吐量、非托管的 DEX。

ZK 和 DEX 这两种强大的叙事相结合，使 Loopring 成为未来值得关注的有趣项目。

$SYS

Syscoin 将比特币的 POW 架构和$ETH 的 EVM 结合到一个网络中。其双层方法允许创建可扩展的智能合约，同时保持网络安全。

Syscoin 使用 EVM 的修改版本，称为 NEVM，它使用 ZK 证明来促进“每秒高达 210,000 笔交易（TPS）”。下图定义了 Syscoin 的架构。

$DUSK

Dusk 是一个以隐私为重点的第一层区块链，为交易可编程的、保密的、合规的证券而构建。 可以把它看作是金融应用的隐私区块链。

Dusk Network 使用名为“PlonKup”的 ZK 证明（ZKP）系统，以一种可负担的方式保护数据。

$MUTE

Mute 是一个基于 ZK-rollup 的去中心化交易所、质押收益平台和债券平台，建立在以太坊和 zkSync 之上， 可以将其视为一个利用了 ZK 优势的综合 DeFi 平台。

总结

值得注意的是，这种反弹（就像很多加密货币的叙事一样）大部分是基于猜测的。仅仅因为一个项目实施了 ZK 技术，并不能自动使其成为赢家。寻找那些正在使用该技术进行创新的项目，或者学会利用炒作进行交易。

对于这些项目，就像大多数山寨币一样，价格走势在很大程度上取决于 Bitcoin。 然而，如果你的市场前景是看涨的，那么最有力的叙事无疑会跑赢大盘（ZK 可能是其中之一）。

你如何处理这些项目的交易取决于你的个人策略，但要考虑以 DCA 为导向的方法，并在投资前严格定义 a）你的时间范围，和 b）财务目标。

虽然已经介绍了一些目前已经发布的利用 ZK 的项目，但我对即将到来的未发布的协议更感兴趣，这些协议是加密货币中一些支持力度最大的项目（也有空投潜力）。

未发布的 ZK 项目

zkSync

zkSync 是一个 ZKRollup，代表了扩展以太坊的 “终局”。

它是加密领域最受支持的项目之一，从 a16z、Dragonfly、Lightspeed 等公司筹集了超过 4.5 亿美元。

他们的完整主网甚至还没有启动，生态系统就已经很庞大了。它将成为 ZK 领域的一个巨无霸。

zkSync 的下一个阶段是 “Fair Onboarding Alpha”阶段，议可以在封闭环境中开始部署在 zkSync 2.0 上。

StarkWare

StarkWare 是一个二级扩展解决方案，它利用 STARK 证明来减少发送到以太坊区块链的信息量，缓解区块链拥堵。

SnarkNet 和 zkSync 一样，利用 ZK-Rollup 技术来实现这一目标。

Starkware 也得到了大量的支持，从 Paradigm、Pantera、Sequoia、Ethereum Foundation 等机构筹集了 2.85 亿美元以上。同样他们的生态系统也令人印象深刻。

Scroll

Scroll 是一个基于 zkEVM 的 zkRollup，可以实现对现有以太坊应用程序和工具的本地兼容性。

在即将推出的 L2 中，Scroll 的上市前估值排名第四。

随着代币的推出，这里可能会有空投机会。他们最近暗示要进行空投，称他们正在“创建一种鼓励参与的激励机制”。

Aztec

Aztec 是以太坊上隐私优先的 ZK-rollup，为以太坊生态系统带来保密性和成本节约。

他们最近筹集了 1 亿美元，由 a16z 领投。他们在未来可能还会有空投。

Polygon (zkEVM)

最近，Polygon 收购了 Hermez Network（一个 ZK-rollup），以致力于他们自己的 zkEVM 产品。

zkEVM 是一个虚拟机，以透明的方式执行以太坊交易，包括具有零知识验证的智能合约。zkEVM 增加了一个已经扩展的生态系统，该系统正在迅速发展。

由于 Hermez 已经与 Polygon 合并，$MATIC 代币将为该网络提供动力。

Axiom

Axiom 是以太坊的一个 ZK 协处理器。

虽然不是严格意义上的 ZK-rollup，但我想把 Axiom 列入这个名单，因为它为我们提供了如何在实际意义上利用该技术的一个很好的代表。

它允许开发者查询链上数据，并通过 ZK 获得无需信任的响应——允许开发人员以高效的方式使用 ZK 增强他们的应用程序。

总结

无论是通过空投、发布后的投资，还是投资于单个生态系统——有许多策略可以在这些即将到来的项目中定位自己。
