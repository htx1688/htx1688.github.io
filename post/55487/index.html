原文作者：深潮 TechFlow

最近，Telegram Bots 叙事异军突起，在过去七天，Unibot 上涨了 56.35% ，目前的价格为 78.53 美元，但在两个月前，它的价格仅为 2.46 美元。

巨大的涨幅带动了整个板块，从叙事角度来讲，这是一个未曾提及的赛道。从 Web3 角度来讲，新用户可以通过 Telegram Bots 很方便的进入链上购买代币，这些机器人解决了加密货币最核心的问题之一，为大众采用埋下了伏笔。那么市场上到底有哪些 Telegram Bots？本文将带你了解比较主流的几个项目。

Unibot

如前所述，当前市值排名 297 的 Unibot 是当之无愧的领导者。它具有创新的功能，它的目标是成为最快的 Telegram 交易机器人，提供无缝和极速的交易体验。它通过免费的 Telegram 机器人实现交易，并且具有全面的功能套件，旨在帮助用户取得成功。

Unibot 具有以下特点：

  1. 限价订单： 可以精确执行买低卖高的策略。

  2. 镜像狙击： 可以模仿专业交易者的成功策略进行交易。

  3. 保护卖出： 可以提前测试卖出策略，最大化卖出潜力。

  4. 私人交易： 可以保护交易隐私，智能应对三明治攻击。

此外，Unibot 还提供实时的以太坊新代币警报功能，用户可以轻松添加代币并进行狙击交易。项目还提供收入分成机制，持有者可以根据持有的代币数量获得交易费用的 40% 和总交易量的 1% 作为奖励。奖励每 2 小时计算一次， 24 小时后可领取。然而，如果在每 2 小时间隔内转移超过 200 个代币，将会失去收入分成的资格。持有 50 个$UNIBOT 代币才能有资格参与收入分成。

LootBot

Lootbot 是一个自动化空投交互的 Telegram 机器人，旨在自动化空投收割。空投一直是加密货币领域最受欢迎的方面之一，而一个更简单的界面来进行空投交互必定会吸引用户。

Lootbot 的目标是为用户提供便捷的参与空投体验。通过自动化机器人在多个 EVM 兼容链上运行，它能够筛选出最有利的空投机会，并同时实施强大的安全措施，以保护用户免受黑客和欺诈活动的侵害。

项目特点：

  1. 自动化空投收割： 生成钱包并自动在多个 EVM 兼容链上进行空投交互。

  2. 空投跟踪： 跟踪你的空投收割进度，包括参与历史、获得的奖励和即将到来的空投。

  3. 反 Sybil 检测： 通过提供不同的策划路线来预防女巫攻击，以代表真实的链上交互。

WagieBot

WagieBot 是一款多功能的 DeFi 机器人，通过在 ETH、BSC 和 Arbitrum 上运行自己的专有私有节点， 1 毫秒的响应时间使其成为市场上最快速的工具之一。

WagieBot 的目标是为用户提供在 Telegram 和 Discord 上与以太坊、币安链和 Arbitrum 上的任何流动性池进行尽职调查和交互的能力。用户只需粘贴一个合约，WagieBot 将自动获取你解析的代币的最大流动性池和所有必要信息，使其比与任何去中心化交易所交互快数十倍。

通过 WagieBot 进行的所有操作，无论是购买、卖出、抢购、复制交易等，都是即时完成的。此外，持有一定数量的$WAGIE 代币将给你带来许多优势，包括获得 WagieDAO 的访问权限以及即将推出的更多高级机器人功能。

WagieBot V4 已经上线，V4 整合了 GMX，用户现在可以通过 Telegram 在最大的去中心化永续交易所上进行做多或做空操作，此过程中 WagieBot 不会收取额外费用。

Bolt Bot

BoltBot 是一个旨在提供快速、智能交易功能的项目。通过不同的订阅层级，BoltBot 为持有者提供了多种功能和特点，以满足不同投资者的需求。通过这些功能，BoltBot 旨在提供更快速、更准确的交易执行，以及更多的交易洞察和分析工具，帮助投资者做出更明智的决策。

BoltBot 的订阅层级分为三个等级：TIER 3、TIER 2 和 TIER 1 。每个等级都提供不同的功能和特点，具体如下：

TIER 3 （所有持有者）：

  • 闪电般快速的买卖功能

  • BOLT 狙击手

  • MEV 保护/私人交易

  • 多钱包应用

  • 自动卖出

  • 下单功能

  • 0.85% 的手续费。

TIER 2 （持有超过 5, 000 $BOLT）：

  • 包含 TIER 3 中的所有功能

  • 项目可持续性监测

  • 鲸鱼观察

  • Alpha 指标

  • 可访问 BoltBot 投资者指数

  • 0.5% 的手续费。

TIER 1 （持有超过 7, 500 $BOLT ）：

  • 包含 TIER 3 和 TIER 2 中的所有功能

  • 收入分享

  • 预售抢购

  • 多钱包限制

  • AI 自动交易

  • 利润预测

  • 饱和度指标

  • 代币交易量与市场交易量指标

  • 0 手续费。

Blacksmith

Blacksmith 是一个完全可定制的、人工智能辅助的代币交易 BOT。该项目通过引入独特的代币机制，来获取平台访问权限。

其独特之处在于该项目的代币和平台的使用紧紧相关：

  1. 烧毁机制： 用户必须通过烧毁 Blacksmith Token（BS）代币来获取极其有限的 Forge Key NFTs，以获得 Blacksmith AI 平台的完全访问权限。这种机制将永久性地从流通中移除代币。

  2. 有限供应： Forge Key 将以先到先得的方式进行“制作”，每次只能制作不超过 50 个。这意味着为了制作一个 Forge Key，需要烧毁 BS 代币的相当大比例的流通供应量。每当 Forge Key 供应量增加 1 个时，BS 代币的流通供应量将减少约 1/50 。

  3. 持有 Forge Key 的特权： 持有 Forge Key 将获得以下特权：在 Beta 测试期间无限制访问 Blacksmith 平台、与其他持有者一起进入 The Forge 私人电报群组、优先考虑未来产品/奖励等。

Bridge Bot

BridgeBot 是一个去中心化金融（DeFi）协议，允许用户在 Telegram 中直接在不同区块链之间进行资金桥接。传统的跨链桥通常用户体验较差，Bridge Bot 的目标是改善这一问题，提供更简单易用的桥接功能。我可以想象新链上的项目会推动 BridgeBot 的采用。

通过 Bridge Bot，用户只需将资金存入指定地址，然后选择目标地址，即可收到其资产，而无需与原始存款地址有任何联系。这种方法赋予用户更大的对其财务隐私的控制权，提供了更高的交易机密性和安全性。

  1. 安全和匿名交易： Bridge Bot 允许用户在不透露个人信息或交易历史的情况下执行交易。这对于希望保护财务隐私的个人以及由于法律或其他原因需要匿名性的交易非常有价值。通过提供安全和隐秘的交易环境，Bridge Bot 赋予用户在财务活动中保持更高水平的隐私和机密性的能力。

  2. 可访问性和用户中心体验： 此外，该机器人提供无缝的可访问性，允许用户在无需进行 KYC 程序或经历任何批准流程的情况下使用平台。这意味着个人可以自由、轻松地使用该平台，而无需遇到身份验证或复杂的授权程序等限制。Bridge Bot 优先考虑用户的便利性和包容性，确保用户可以方便地参与匿名交易，同时完全掌控其财务隐私。

  3. 快速和安全： Bridge Bot 的功能包括资产交换、跨链桥和混合。通过先交换资产，然后进行桥接和混合，该机器人实现了无缝的过程，确保了交易的高效管理、安全转移和增强的隐私性。

0x Sniper

0x Sniper 旨在通过智能合约和以太坊的高速交易，彻底改变人们在以太坊区块链上的交易体验。0x Sniper 不仅仅是一个普通的交易工具，它旨在同时满足有经验和新手交易者的需求。凭借其直观的设计和用户友好的界面，0x Sniper 提供了一个易于导航的平台。

项目特点：

该平台提供了一系列先进功能，这些功能包括：

  • 反 MEV： 在以太坊生态系统中，矿工可重新排列或排除区块内的交易，这经常导致常规用户经历次优结果。0x Sniper 使用反 MEV 功能，为防止这种前置交易提供了额外的保护层。

  • 反 Rug： 随着“Rug”在 DeFi 领域变得突出，拥有保护措施变得至关重要。0x Sniper 机器人的反 Rug 功能有助于识别潜在的 Rug 代币，在你的交易体验中提供额外的安全层。

  • 多钱包支持： 为了增强灵活性，0x Sniper 支持多个钱包。这使你可以同时在不同账户之间执行交易，使你的交易过程更加高效。

代币持有人则享有一些特定的特权，并将在未来获得访问独家功能。0x Sniper 还计划根据社区反馈，整合更多的链并实施更多高级功能。

GenieBot

GenieBot 是一个人工智能驱动的 Telegram 机器人，旨在解决加密货币项目中的信息过载和不足的管理问题。因此，GenieBot 的目标是为特定加密货币项目的用户提供简单可靠的问题解答方式。通过收集项目的所有必要信息，机器人可以像人类一样回答各种问题，使沟通变得清晰易懂，有助于减少混淆。

这个项目的特点包括：

  1. 与用户以人类的方式互动： 机器人能够快速回答用户的问题，提供关于项目的准确信息。

  2. 回答所有与项目相关的问题： 机器人利用先进的自然语言处理技术，能够解释和回答各种问题，甚至包括复杂的措辞或不常见的术语。

  3. 知识库来源于白皮书、网站或自定义来源： 机器人从项目的白皮书和网站中获取数据，以确保对项目有全面的了解。

  4. 实时知识库扩展： 随着机器人与用户的互动，其知识库和对项目的理解将不断扩展，随着时间的推移，提供更准确和有帮助的答案。

  5. 管理员知识库扩展： 管理员通过不断向机器人提供新信息来增强 GenieBot 的智能，并及时更新最新的项目细节，从而扩展机器人的知识库。

TokenBot

TokenBot 是一个由原生代币$TKB 驱动的社交交易网络，$TKB 充当了 TokenBot 复制交易平台的代币门控会员资格。它提供实时的交易通知，并允许用户连接其交易所账户，与其他成员实时分享交易信息。通过自动复制交易功能，用户可以选择自动复制其他交易者的交易，从而提高交易效率。 此外，TokenBot 还提供了订阅付款功能，可以接受法币和加密货币支付。TokenBot 还推出了 TokenBot Card，这是一个基于以太坊区块链的 NFT 卡片，拥有 TokenBot Card 的用户可以获得 VIP 访问权，并加入到 TokenBot 社交交易社区中。

alfa.society

alfa.society 是一个旨在培养和分发“免费资金”，通过提供免费资金工具、独家 Alpha 内容、铸币和预售机会等，帮助用户扩大用户的净资产。$ALFA 是 alfa.society 的实用代币，用于控制会员资格和对所有实用程序的访问。例如，用户必须购买并持有 20 万个 ALFA 才能访问 alfa.chat 和 alfa.dapp。下面介绍一下这个 Bot 的功能：

  1. alfa.airdropbot： 这是一个一键空投变现工具，使用户能够最大化利用即将到来的免费空投。

  2. alfa.spoofer： 这个功能用于吸引关注和增加持有者数量，例如将在项目刚刚部署时。该功能将向当天交易量最活跃的 500 名 Uniswap 交易者发送极小数量的代币。由于这 500 名活跃的 Uniswap 交易者不断在 Uniswap 上进行交易，许多来自其他项目的人不可避免地会查看他们的钱包，然后他们将看到使用 alfa.spoofer 的协议的代币。

  3. wallet duster： 一个钱包清理工具，$ALFA 用户可以使用它选择并出售他们钱包中的代币（支持多钱包），从而轻松清理钱包。

SwipeBot

SwipeBot 是一个用于 ETH/ARB/BSC 的 Telegram 交易机器人，具有独特的人工智能系统。与大多数竞争对手的 Telegram 交易机器人不同，SwipeBot 的设计以易用性为首要考虑因素，在 Bot 的常用功能上，SwipeBot 都进行了一些调整：

  1. 购买代币： 用户只需输入合约地址和希望花费的 ETH 数量，SwipeBot 就能比竞争对手更快地购买用户所需的代币。

  2. 百分比购买： 它允许用户购买代币供应的一定百分比。

  3. 狙击： 当调用此功能时，SwipeBot 将在流动性添加和交易启用后立即进入用户选择的代币。

  4. 钱包创建： 该功能允许用户直接通过 SwipeBot 创建新钱包，并可用于调用所有其他功能。

  5. SwipeContractSniffer： 该功能允许用户根据 SwipeBot 提供的各种指标检测合约地址的安全性。

SwipeBOT 每两周向非持有者开放一次，所以，用户必须持有 500 $SWIPE 代币才能访问 SwipeBot，如果持有的金额低于 500 $SWIPE，将收取 0.5% 的交易费用。为了防止用户因 Swipe 的市值增长而无法支付访问费用，所需代币数量将根据 Swipe 的市值不断调整。

Scarab Tools

Scarab 是由一位全职交易员和 Degen 构建的项目，始于 2023 年 1 月。通过 Scarab，用户可以在零区块上抢购流动性，并手动快速购买现有代币，所有这些都使用多个私有 RPC。此外，Scarab 还添加了一个新代币列表和一个合约模拟器，用于检查合约状态和收费情况，下面依次介绍一下：

  1. 新代币列表： 在代币部署后的 5 秒内，即可收到每个新代币的通知。目前仅支持以太坊。

  2. 合约模拟： 通过合约模拟工具，用户可以检查代币买卖税、能否卖出等信息。

$DUNG 是 Scarab 的原生代币。当然，想要使用 Scarab，用户必须持有超过 250, 000 的$DUNG 。

根据经验来看，Bot 往往护城河不高，容易被复制，这是该叙事的弊端所在。但 Telegram Bots 叙事具有扩展性，可以与其他叙事整合，如人工智能。这些 Bots 未来也不会局限于 Telegram，有望看到此类型 Bots 整合其他聊天软件，对于 Web3 的 Mass Adoption 而言是一件好事。所以个人预计该叙事还将会继续发展。
