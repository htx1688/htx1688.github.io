行业的风向总随着巨头公司而动。

自从亚马逊公司推出 NFT 市场的消息传出后，大众的目光就锁定在了亚马逊的身上，作为科技巨头的亚马逊入局 NFT 对整个行业都有着举足轻重的意义。

受 FTX 的崩溃的影响，亚马逊 NFT 市场的推出几经波折，根据 Web3 爱好者莫里茨的说法，原定于4月24日上线的亚马逊旗下 NFT 平台“ Amazon Digital Marketplace”推迟到了5月15日。

图源：新华社

该平台将首先在美国推出，预计最终会推向全球。

早在 1 月份，Blockworks 就首次披露了亚马逊 NFT 市场的详细信息——该平台将专注于区块链游戏和相关游戏内的 NFT 收藏品。雅虎财经（YF）后来也提到亚马逊会将 NFT 与实物产品联系起来，亚马逊账户可以用作 NFT 钱包，从而绕过了对像 MetaMask 这样的专用加密钱包的需求。

这家全球最大的在线零售商一直对此事守口如瓶，未发表官方声明。

从目前未被证实的消息来看，亚马逊的 NFT 平台有哪些值得关注的地方呢？

亚马逊的 NFT 平台的特点

数字资产所有权

亚马逊计划发行与现实资产对应的 NFT，而不是像普通 NFT 平台一样局限于虚拟世界，在实体资产的拥有体验上增加了数字资产的所有权。

提升用户体验

普通 NFT 市场交易需要通过虚拟币进行交易，用户需要具备区块链或自托管钱包的技术知识。而据莫里茨承认的说法，“ Amazon Digital Marketplace” 旨在简化 NFT 购买流程并为用户提供无缝体验。它允许用户使用信用卡进行法币支付购买 NFT，只需要一个亚马逊平台账户就可以完成 NFT 交易，其 NFT 可存储在用户的亚马逊账户中。

用户群体庞大

亚马逊用户高达 1.97 亿，这些用户的群体差异性较大，如果这部分用户加入，NFT 的市场会注入更多新鲜血液，不同于常规单一化市场，用户多样化的市场活跃度会高出很多。同时由于其降低了交易门槛，让更多的非加密用户参与进来，用户拓展也会更加容易。

亚马逊的市场计划早在几年前就初露雏形，2021年，据CNBC报道，亚马逊确认了正在招聘加密货币负责人。该公司发言人曾在一份声明中表示：

“我们受到加密货币领域创新的启发，并正在探索应用于亚马逊上会是怎样的场景。我们相信未来将建立在新技术之上，这些新技术能够实现先进、快速和廉价的付款体验，并希望能尽快提供给亚马逊客户。”

其他领域

自从启动 NFT 平台项目之后，亚马逊公司就开始接触 Layer 1 区块链、区块链游戏公司等多种数字资产项目，还与 AVAX 和 Chainlink 达成了合作。

莫里茨还表示亚马逊将推出自己的私有区块链，该区块链可与以太坊、Avalanche 和 Chainlink 互操作——并计划在未来整合更多区块链。未来亚马逊可能会深度涉足区块链领域。

亚马逊在 Web3、AI 领域都有参与，在亚马逊的云服务公司 AWS 官网中，有专门的页面为区块链企业提供服务，不管是分布式的区块链网络还是集中的数据库，AWS 都能为 Web3 企业提供解决方案。据 AWS 员工透露，在2022年其区块链业务带来的营收占比预计超过 AWS 海外营收的一半。

亚马逊公司正组建一个团队研究 AI 工具，旨在为商家生成照片和视频，用于其平台上的广告活动，去年亚马逊广告收益为380亿美元。目前，亚马逊广告业务重心是在搜索结果中为商家提供助力，打造更广泛的广告业务。亚马逊发言人称，该公司正投资生成式 AI，“对客户和广告商的潜在利益感到兴奋”。

电商平台涉足 NFT 我们国内早几年前就先人一步。

2021年5月20日，中国最大的电商巨头阿里巴巴旗下淘宝阿里拍卖推出了全国首个数字艺术品专场，用户可以在其平台中直接购买数字藏品。2021年12月18日，京东正式上线灵稀数字藏品平台，开始售卖数字藏品。

不过国内平台与亚马逊还是有较大差异的，虚拟商品仅仅是借助之前的平台，并没有将实体与虚拟真正联系到一起，更多的是扮演营销的角色，多数 IP 入局也仅仅是发行独立的虚拟商品，拓展宣发渠道。

相较于以往的 NFT 市场 OpenSea、Blur 等，亚马逊展现出更多的新意，而这些创新与优势能否成为赶超这些平台的利器，还需看上线后用户的反馈。亚马逊通过修建 NFT 与实体产品的桥梁，打通 Web2 与 Web3 世界。从这个角度来看，电商与 NFT 又像是物理与数字一样相辅相成的存在。或许， NFT 的路可以让亚马逊换个起点开始。
