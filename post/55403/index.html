博链财经BroadChain获悉，7月20日，作为中国唯一符合监管要求的区块链，Conflux宣布与World Mobile展开新的合作。World Mobile是目前唯一建立在区块链技术上的全球电信服务提供商。

继2023年年初成功与中国电信开展Blockchain SIM(BSIM)卡研发后，Conflux 持续发力通信领域，与 World Mobile 确立了合作关系，双方拟在以下四个领域展开：

生态共荣

World Mobile 移动网络原生资产可通过桥接支持，在 Conflux 实现 Cardano 生态之外的首次流转。这为 Conflux 生态带来多样性的同时，也为 World Mobile 带来了丰富的流动性。

技术集成

World Mobile 和 Conflux 将在技术集成方面展开合作，将 Conflux Network 集成到 World Mobile 的侧链 AyA 中，为 Conflux Network 提供 EarthNode 以及金融结算功能。

亚洲和非洲市场的协同拓展

Conflux 和 World Mobile 在亚洲和非洲市场分别拥有强大的影响力，两家企业将携手利用这些优势，增强连接性，推动用户增长，并在关键市场进行进一步渗透。特别值得一提的是，尼日利亚地区在非洲整体的加密市场中占比近 67%，而 Conflux 在尼日利亚的加密市场中排名前三。近日，World Mobile 也在尼日利亚成功完成了混合动态网络的实地测试，此前在桑给巴尔成功推出商业化电信网络。

BSIM卡开发

World Mobile 和 Conflux 将合作探索如何在全球范围内更好地利用 BSIM 卡提高数字连接性触达的解决方案。

Conflux 联合创始人张元杰在谈到这次合作时表示：

“Conflux 的使命在于推进 Web3 普及，同时确保基础设施的低成本、去中心化以及安全性。其中一个重要策略，就是促进与全球电信服务提供商的合作。继中国电信之后，World Mobile 是 Conflux 的下一个重要合作伙伴。我们会共同将安全、易用的区块链技术桥接到亚洲与世界其他地区。”

Zachary Vann，World Mobile 的负责人补充道：

“我们很高兴与 Conflux 合作，共同构建一个更加互联互通且包容的未来。这次合作将加速我们在亚洲市场的拓展，并迈出联通全球的尝试。Conflux 作为符合监管要求以及世界上首个研发 Blockchain SIM 卡的 Web3 底层基础设施，将为我们的移动网络和 AyA 带来许多新机会。这是我们扩大我们的移动网络、引领共享经济进入亚洲市场乃至更多地区的最重要的合作机会。”
