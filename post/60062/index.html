FT 核心社区通常对该平台的潜力感到兴奋，并喜欢使用它和/或利用它来实现利润最大化。

过去几周，Friend.Tech（FT）推动了 SocialFi。许多知名的加密推特人士目前在那里都有积极参与，导致了重大的炒作。Hsaka 和 CL 的 Key 分别为 5 和 4 个 ETH。目前，自己买自己的 Vombatus 以略高于 7 个 ETH 的 Key 价格领先。

9 月初，总锁定价值（TVL）为 530 万美元。目前，它为 3900 万美元，增长近 700%。TVL 抛物线增长最近有所放缓，但仍在稳步增加。

从每日美元流入来看，可以明显看出流入总体保持健康。在 2022 年 9 月 10 日周末达到峰值，TVL 在一天内增长了 50%。

自九月初以来，FT 的交易量已经大幅增长。在过去的一周中，它们稳定在每天约 1000 万美元的范围内。值得注意的是卖出与买入量的变化，上周我们看到了明显增加的卖出压力，这预示着获利了结和用户可能退出平台。然而，自上周五实施削弱自我购买的积分以来，买家重新获得了信心，正如买入与卖出量所示的增加的买入量。

与 TVL 和交易量一致，9 月初以来，用户活动也显着增长，这是对具有产品市场匹配的健康信号。FT 目前每天约有 1 万回访用户，上周平均每天有 9 万新用户。

在看这些数字时，重要的是要意识到 FT 上有许多（交易）机器人活跃，不知道用户中有多少是机器人。

策略

FT 已经建立了一个非常活跃的核心社区。该核心社区通常对该平台的潜力感到兴奋，并喜欢使用它和/或利用它来实现利润最大化。核心社区成员利润最大化的主要目标如下：

  • 积累空投积分 ；

  • 交易 Key 以增加投资组合价值；

  • 增加自己 Key 价格。

为了实现上述目标，采用了几种主要策略。

(3,3)

这是从 OHM 开始就知道的常见定律，相互购买对方的 Key 并持有。这样可以增加彼此的 Key 价格、投资组合价值以及每个星期五分配的空投分数。分配分数的标准每周有所不同，但通常非常依赖于用户的投资组合规模。这个策略非常适合在其他社交平台上没有现有追随者的用户，因为（3，3）会查看链上数据来找到合适的（3，3）朋友，并倾向于互相跟进购买。因此，这也是引导房间活动的绝佳方式。

在过去的几周里，（3，3）的表现远远强于其他部分，使其成为目前最有利可图的一般策略之一（如果不是最有利可图的话）。

与 OHM 一样，这个策略具有很强的自反性的，对其用户有潜在的灾难后果。但由于 FT 的社交组成部分以及以其他方式增加 Key 持有人价值的潜力，例如撰写文章、封闭式工具访问和网络访问，它不一定会最终以同样的方式结束。

（3，3）社区可以说是最投入的社区，它建立了许多仪表板和工具，不仅跟踪，还激励期望的长期行为。这意味着（3，3）房间通常非常活跃，专注于 FT 的一切。看到这种行为的上演很有趣，用户们正在积极解决（潜在的）负面问题，以确保他们的社区和 FT 的繁荣。

自我购买

在过去几周中，最有效积累分数的方法是购买自己的 Key。无论是通过主账户还是通过副账户。

使用此策略，用户从一开始就购买自己的 Key。这导致他拥有最初的所有 Key，没有价格下跌风险，与如果用户从他人购买 Key 相比，投资组合价值乘数约为 2.5 倍。这是因为自买会随每次后续购买增加 Key 价格，从而也增加了投资组合价值。用户所发生的唯一费用是在购买和最终卖出 Key 时向协议支付 5%的税费。

前 50 个 Key 中目前有 15 个自买家。此外，可能还有副买家，这些是更难识别的。因此，相当大比例的积分是通过自买家获利的。

但是，在过去的几周中，许多人认为这需要削弱，因为自买对平台没有增加任何价值。此外，它稀释了有机用户的激励措施。但可能更重要的是，与现有 Key 相比，越来越多的新流动性正在流向自买家。

上周五，FT 对积分分配算法进行了剧烈变化，其中自购遭到了大幅削弱。例如，排名最高的用户“Vombatus”得到了每 ETH 3.33 分，而（3，3）平均获得了每 ETH 约 90 分。周末，这导致几个自买家卖出自己的 Key，转而采用（3，3）策略，在那里他们购买（3，3）的 Key，希望获得回报购买。

狙击

狙击使用 Bot 从新加入该平台的用户那里购买第一批 Key。初始 Key 的价格低至几美分。随后的购买会迅速提高价格，使得狙击手通常可以在狙击 Key 后数小时内获利。

对于那些技术熟练足以有效竞争的人来说，这可能曾经是、也可能仍然是最有利可图的策略。

购买 KOL

一个更简单的策略是购买专业人士、KOL，这些专业人士很可能保持在排行榜前列附近。这样做的主要原因是为了让 FT 的上行风险得到普遍曝光，同时保持事情相对容易管理，并限制下行风险。

在过去的几周里，这种策略非常有利可图，因为最高 Key 从 1-2 ETH 增加到 4-8 ETH 左右的峰值。然而，从上面的表中可以看出，它与 （3，3） 策略相比并不具有竞争力。

其他

还有许多其他策略在使用中，但这些超出了本文的范围。值得注意的其他几种策略包括：

  • 购买在 X 上拥有大量追随者但不活跃的账户；

  • NFT 社区成员互相购买对方的账户，例如 Punk 购买 Punk。这与（3，3）策略非常相似，但到目前为止没有那么极端。
