 概要

  • Karl Marx【 1 】可能会喜欢 ETH、Dogecoin 和 Worldcoin。

    Vitalik 的社会学实验 Zuzalu。Zuzalu 通过借助 ZuPass 和零知识证明技术，为登记入册的参与者提供「身份登记证明」，并通过设置市政厅来促进民主治理的实施。因此，有人将其比喻为「抽象意义上的以太坊早期阶段」。二者因为 Vitalik 建立起天然的联系，但又有所区别。Zuzalu 是形而上的，而以太坊是具象的。以太坊的可编程性能帮助开发者可以通过智能合约实现自定义的身份认证。而以太坊的货币支付是通过执行智能合约来实现的，满足符合 ERC 20 标准的货币支付，但主要以 ETH 为主。

  • Elon Musk 的火星殖民计划。Musk 的“Starlink”条款称火星是自由星球，不承认地球法律【 2 】。

    既然他的愿景是将人类带去火星，那么人类在地球上的身份证明和货币支付需要重制于火星。收购推特、将 Dogecoin 等 CryptoCurrency 纳入其旗下产品可接受支付货币的方式无疑成为 Musk 的火星殖民计划中的重要版图。

  • Sam Altman 的「世界公民」构想【 3 】。

    Altman 发现——世界上很多人，特别是以第三世界国家公民为代表的人群，很难拥有合法的身份证明，并且这些国度的金融基础设施尚处于不完善阶段，故而这些人也很难拥有基础的银行账户。Worldcoin 的搭建提供了一种全新的数字身份认证和货币体系，它通过扫描地球上每个人眼球的虹膜实现身份认证，$WLD 作为其生态内的货币。

上述三个人所构想的事业中，存在以下共同点：

  • 1）他们都关注并且探索了广义上的身份认证和货币支付课题。

  • 2）通过搭建世界级平台实现（1）的课题。

 Zuzalu: 探索无国界国家

Zuzalu 的愿景与技术需求

Zuzalu 是一个落地于真实世界的加密社区，也被称为一个关于加密、长生不老和哲学的夏令营。在 Zuzalu 中，参与者通过 ZuPass 登记并获得利用零知识证明的身份认证，以验证居民的身份。这个实验旨在探索以太坊的可编程性和智能合约，以实现自定义的身份认证和数字货币支付【 4 】。

在 Zuzalu，参与者包括研究者、学者、创始人等，在为期两个月的时间里深入探讨长生不老、公共物品、零知识证明、人工智能以及协调和网络国家等议题。Zuzalu 致力于探索如何将区块链技术应用于城市治理和运营，提高流程透明度、可信度和可验证性。长生不老是 Zuzalu 的一个重要议题之一，区块链技术被认为可以在生命科学领域提高数据管理、知识产权保护、试验透明度、资金筹集等方面的效率。另外，Zuzalu 还探讨了区块链如何解决人工智能带来的极端风险问题。人工智能的发展可能导致超级智能，而区块链技术被认为可能用于抵御超级人工智能带来的风险。

根据多个与会的参与者描述，Zuzalu 旨在建立一个无国界的数字国家，让每个数字公民拥有更加公平且透明的身份认证以及可自定义的数字货币支付系统。

总的来说，Zuzalu 是一个独特且雄心勃勃的实验，它将区块链技术、加密货币、生命科学、人工智能和哲学结合在一起。它寻求改善社会各个方面的新方法，从城市治理到科学研究和人工智能安全。参与者共同构想一个技术促进数字领域包容性、透明度和全球合作的未来。

Elon Musk 矩阵：Xeet 与数字身份、Dogecoin 与数字支付的未来展望

Musk 矩阵

Xeet（前身为 Twitter）和 Dogecoin 作为基础矩阵串联 Musk 矩阵。

Xeet 作为全球知名的社交媒体平台，拥有庞大的用户基础。Xeet 可以为 Musk 提供一个强大的数字支付和加密货币推广平台。通过整合 Xeet 与数字支付的功能，可以推动数字货币支付方式的普及，让用户在推特上实现数字支付交易。

其次，Musk 多次提及并且「喊单」Dogecoin 表现出他对加密货币的兴趣。虽然 Dogecoin 是作为一个 Meme Coin 而诞生的，但由于 Musk 的支持赢得了更多认可和关注。通过推动 Dogecoin 的发展和应用，他可以进一步探索和推广加密货币作为支付手段，并在更广泛的商业场景中实现数字货币支付的应用。

上述提及的 Xeet 和 Dogecoin 与 Musk 的其他商业版本图可以良好地串联起来一个庞大而综合的数字生态系统。诸如，已经上市的 Tesla、未上市的 SpaceX、SolarCity 和 The Boring Company 等公司可以通过 Xeet 平台推广支持加密货币支付的服务，用户可以使用数字货币购买特斯拉电动汽车、SpaceX 太空旅行等服务，实现支付应用。此外，神经脑科学公司 Neuralink 也可以在数字支付生态系统中发挥重要作用。在未来，通过脑机接口技术，用户甚至可以直接使用脑部信号进行数字支付认证，进一步提升支付的便捷性和安全性。

Musk 的数字支付热情  

可以说 Musk 对数字支付领域一直怀有梦想，并通过不断地探索，将其拟合到自己的商业矩阵中。

根据他的自述，其使命是推动科技进步，创造更好的未来，而支付领域正是他所看重的科技创新之一。从他创立 X.com（后改名为 PayPal）这家在线支付公司开始，就可以看出他对数字支付解决方案的追求和热情。马斯克创建 PayPal 的初衷是为了打造一种全球化的支付方式，让跨境交易变得容易，解决传统支付系统的限制。

虽然后来 PayPal 被 eBay 收购，但他对数字货币和支付的兴趣从未割裂。某种程度上，Twitter 改名 Xeet，图标改为「X」，也是一种对 PayPal 的致敬。

在后续的创业过程中，他探索着与数字货币、加密技术相关的项目。直到 Dogecoin 的出现，以及对推特的并购。Musk 的数字支付矩阵又回到其商业版图中，并将可能在未来发挥极其重要的作用。

此外，Musk 盛赞微信的强大功能，表示它什么都能做，而且没有垃圾信息【 6 】。他很欣赏它的功能，包括消息传递、语音和视频通话、社交媒体、食品配送、移动支付、游戏和新闻。Elon Musk 赞赏微信可能是因为他看到了这款应用在社交、支付和服务等方面的成功，并认为其多功能性和用户体验在数字支付领域中具有启示意义。

Musk 的 Web 3 野心 

Xeet 账户可以成为数字身份证明，甚至成为数字钱包；Dogecoin 可能只是其「新数字支付」矩阵的开始。

Xeet 账户扮演着关键的角色，不仅作为数字身份证明，还具备数字钱包的功能。通过利用社交图谱，Xeet 账号能够有效地定义和认证用户的身份，使得数字身份验证更加安全和便捷。

具体而言，Xeet 账号可以使用社交图谱定义/认证身份，推特头像链接 NFT PFP 与 NFT Tweet Tiles 。“Tweet Tiles”是 Twitter 官方推出的一种将可自定义格式附加到推文中的功能，支持用户与 Twitter Tiles 互动，内容覆盖文本、图像、视频或其他元素，使推文时间线更加动态和可视化。Twitter 将这一功能扩大到 NFT 市场，NFT 项目或创作者可以在自己的推文中展示 NFT，为 NFT 支持者提供展示和分享数字藏品的平台。

根据 Finnacial Time 在今年 1 月的报道，彼时的 Twitter 在 Elon Musk 的领导下，正在探索在其平台上引入支付服务，以此作为创造新收入的手段。Twitter 的核心成员 Esther Crawford 正在设计支付架构，目标是提供「点对点交易和储蓄账户」等金融服务。Musk 设想将 Twitter 打造成一个集消息、支付和商务于一体的“万能应用”。Twitter 已在美国财政部注册为支付处理商，并且正在获得必要的监管批准。

所以，马斯克的支付梦想可以通过 Xeet、支持 Dogecoin 支付以及整合他的其他商业版图来实现。这样的综合生态系统将为数字支付领域带来新的创新和可能性，推动数字货币的广泛应用，并推动支付方式的进一步发展。我们或许将见证一个由 Xeet、Dogecoin 等商业版本构建的数字化生态系统，为数字支付带来了更多的创新和可能性，改变了我们的支付方式和习惯。

Worldcoin 与第三世界

Worldcoin 如何从物理上证明我是「我」？

Worldcoin 自称其目标是通过隐私至上、自主控制和去中心化的方式来解决人的身份证明问题。

根据世界银行 ID 4 D 数据库的估计，全球有近 10 亿人缺乏任何形式的法律认可的身份证明。另有 34 亿人拥有某种类型的法律认可身份，但却难以在数字世界中使用这种身份。

这种**“数字身份”** 指的是：可以通过数字渠道进行明确的身份验证，解锁银行、政府福利、教育和许多其他关键服务的访问权限。数字身份证不仅能增强公民和社会能力，还能实现真正的、包容性的经济收益。

而 Worldcoin 的愿景便是成为世界上最大、最具包容性、属于全人类的身份和金融公共产品。

世界币协议目前包括：

  • World ID 是一种保护隐私的数字身份，旨在解决个人身份证明问题，使用生物特征证明个人的独特人格。

  • Worldcoin（WLD） 是一个在法律允许的情况下可以自由分发给独立个体的数字货币。

  • World App  则是首款与 World ID 兼容的应用程序，承载数字资产和法币支持的稳定币，可以在全球范围内进行支付、购买和转账。

根据 Wroldcoin 7 月 13 号的数据，World ID 注册人数突破了 200 万，而且只花了达到第一个 100 万一半不到的时间，每周平均有超过 40, 000 人注册经过验证的 World ID。来自世界各地的注册者，包括巴塞罗那、柏林、布宜诺斯艾利斯、纽约、 首尔、东京，通过 Tools for Humanity (TFH) 开发的 The Orb 进行虹膜和生物特征扫描注册，他们会因此获得 25 枚 WLD 代币作为奖励。Worldcoin 联合创始人兼 Tools for Humanity 首席执行官 Alex Blania 说：“世界币的目标是让世界上的任何人，无论背景、地理位置或收入如何，都能以保护隐私和去中心化的方式进入不断增长的数字和全球经济。”

Worldcoin 的技术与应用

如图所示，要获得 World ID，用户首先要下载 World App，来存储他们的 World ID。然后通过官网预约进行虹膜扫描验证身份，一旦通过 Orb 进行验证，个人就会获得一个 World ID。

这个 World ID 可以在链上通过 zk 证明来为用户向其他协议验证其身份，且可以跨链进行。同时，在法律允许的情况下，通过 World ID，用户每周可以领取一定数量的 WLD 代币。World App 允许用户在 Optimisam 链上通过 WalletConnect 使用 App，例如去 DEX 上交易。

重要的是，其他开发者可以在无需许可的情况下，基于 World ID 的 SDK 框架在其应用程序中利用 World ID，简化 Web 3 和 Web 2 应用程序的交互。

Worldcoin 的技术架构

在进行人格验证时，Orb 首先使用传感器来确保站在它前面的人是人类。然后，它会获取、处理并在默认情况下立即删除一系列虹膜图像，以创建虹膜代码，虹膜代码是虹膜纹理的数字表示。

在创建虹膜代码后，虽然 Orb 默认会立即删除虹膜图像，但个人还可以选择备份他们的图像。这将会与 Worldcoin 共享他们的虹膜图像，以训练该项目的虹膜检测和分割模型以及类似算法。

Worldcoin 宣传要基于 OP Stack 建立自己的公链，这将使得其能够继承以太坊的账户匿名和交易公开特性。如下图所示，使用 Orb 建立 World ID 后，其数据会经由 Worldcoin Protocol 在链上发布，并能够在 WorldApp 上由其他应用通过 zk 证明验证其身份。

而 SDK 内置的跨链架构可以使其在以太坊主网、其他 Layer 2 如 Optimism 和 Arbitrum 以及非 EVM 兼容链如 Solana、Sui 等之间共享身份信息。

Worldcoin 在其路线中强调了整个协议的去中心化，简单来说就是治理、开发和运营的去中心化：

  • 治理：基于 DAO 的基金会管理

  • 开发：协议开源与设备设计公开化

  • 运营：运营主体的多元化

Worldcoin 的应用

Worldcoin 希望让每个人在任何地方都可以通过 DeFi 和 DID 来参与全球数字经济，能在全球范围实现机会平等。网络越是发达，它的效果就应该越明显。换句话说，建立只属于独立个体的身份系统 World ID 和能在全球范围内进行支付、购买和转账的钱包 WorldAPP，使其成为 Web 3 入口，让所有人都有机会参与到全球经济中来，从而解决当前主要存在于第三世界国家的机会不平等问题。

这也是为什么 Worldcoin 要免费分发 WLD 给全球 10 亿人，借此快速地扩张自己的金融网络，并占据一个非常 Crypto 的叙事。根据 Worldcoin 文章，其拥有如下应用：

  • 身份

真人识别： Twitter 上的机器人、垃圾邮件和自动呼叫都是缺乏健全和顺畅的数字身份的症状。快速发展的人工智能模型加剧了这些问题，这些模型可以骗过验证码并生成令人信服的“人类”内容。因此 Worldcoin 提出的用于身份证明的解决方案是至关重要的，它可以被用作公共基础设施。如果每条消息或交易都包含“经过验证的人类”财产，则可以从数字世界中过滤掉大量噪音，特别是防御女巫攻击。

目前，Web 3 中的集体决策在很大程度上依赖基于代币的治理（一代币一票），尽管 Gitcoin 等采用了二次方投票来限制鲸鱼，但是其无法抵挡女巫攻击。当数字身份被引入现实世界时，就会面临资源分配和过度激励的问题。所以，像 World ID 这样可靠的、抗女巫攻击的人格证明为全球民主治理与资源分配机制开辟了一个新的解决方案，让每个人都参与到其治理中将变得越来越重要。

  • 基于身份的金融应用

企业激励调整： 优惠券、忠诚度计划、推荐计划以及其他与客户分享价值的系统，很容易发生女巫攻击和内部作弊，因为作恶成本极低，但回报却很高。所以，无摩擦且防欺诈的数字身份有助于调整激励措施并使消费者和公司受益。这可以让用户以股票之外的另一种形式参与公司的运营甚至部分拥有某个公司。

社会资源的平等分配： 社会补贴和社会福利在发展中经济体里面尤为重要，因为这些国家的社会福利计划不仅面临着利用虚假身份来获取超出个人公平份额的资源的问题，还有身份统计落后导致的资源份额的错配。2021 年，印度通过实施减少欺诈的生物识别系统，节省了 50 亿美元的补贴计划。去中心化的人格证明协议可以将类似的好处扩展到全球任何项目或组织。随着人工智能的进步，获得公平分配的权利以及通过全民基本收入创造的价值将能够在一定程度上抵消经济权力集中的问题。World ID 可以确保每个人只注册一次并保证公平分配。

  • 第三世界国家弯道超车的机会

Worldcoin 基于人格证明、链上信用系统和 Crypto 支付体系建立平台恰恰是一个第三世界国家当下困境的解决方案。当前的世界货币体系和经济现状，以及气候、疫情与战争等，使得第三世界国家经济环境逐步恶化，他们迫切地需要这样一个破局之道来扭转其颓势，而如今的基础设施与区块链技术的完善给了他们这样做的条件，对于数字身份和新货币体系的需求给了他们这样做的动力。以最为典型的非洲为例：

1）非洲国家当前经济状况

2023 年 4 月版 《非洲脉搏》报告显示，受全球经济不确定性、非洲最大经济体表现不佳、高通胀和投资增长急剧减速的拖累，撒哈拉以南非洲地区的经济增长依然低迷，而且不足以减少极端贫困。经济增长预计将从 2022 年的 3.6% 放缓至 2023 年的 3.1% 。

去年 12 月，加纳通货膨胀率攀升至 54.1% ，达到近 22 年来的最高水平。今年 1 月，尼日利亚通货膨胀率从前一个月的 21.34% 涨至 21.82% ，为 2005 年 9 月以来的最高水平。

  1. 政府面临着日益增加的债务支付和流动性问题，这与欧美银行暴雷导致的高昂借贷成本和美元升值有关。

  2. 非洲的自然资源财富拥有巨大的未开发经济潜力，这需要良好的部门治理、适当的税收以获取更大份额的资源租金以及区域合作和投资。

  3. 抑制通货膨胀仍然是实现宏观经济稳定的核心，非洲各国政府有权进行内部改革，以恢复宏观经济稳定，并更好地确定支出的优先顺序。

  4. 撒哈拉以南非洲各次区域的经济表现并不一致。

  5. 投资增速全面大幅下滑，这种下降存在于各次区域、资源丰富和资源稀缺国家以及投资者类型（公共、私人和外国）中。

2）基础设施

近年来非洲的互联网、移动互联网基础设施搭建迅速，根据全球移动通信系统协会发布的《 2022 年撒哈拉以南非洲移动经济》报告显示，截至 2021 年底，撒哈拉以南非洲地区已有超过 5 亿人订购移动服务，移动用户渗透率为 46% 。报告预计，到 2025 年，该地区将有 6.13 亿人订购移动服务，用户渗透率将达到 50% 。且非洲 84% 以上的互联网连接是通过智能手机进行的。尼日利亚联邦通信委员会的数据显示，截至 2022 年 6 月，尼日利亚移动互联网用户已超过 1.5 亿，互联网普及率约为 70% 。

金融科技和电商也是近几年来非洲热门应用赛道，DataSparkle 数据显示， 2022 年在经济逆风形势下，这两大赛道仍保持强劲增长。非洲财务类移动应用活跃用户规模 2022 年 12 月较同年 1 月增长了 26% ，超 7300 万；购物类应用活跃用户规模 2022 年 12 月较同年 1 月增长了 6% ，超 3600 万。

3）区块链技术

区块链技术从最初 Bitcoin 的点对点现金系统发展到了 Ethereum 的全球计算机，由于智能合约的出现，涌现了一大批新的技术与概念，这些成熟的方案可以用于改进当前的世界货币体系：

  1. 同一标准下的不同方案：Layer 2/Rollup

    各国在同一标准下根据自身政策运行各自方案，统一“度量衡”，解决多方合作下的系统摩擦。

  2. 身份与信用系统：DID

    身份与信用上链替代传统银行进行个人信用评定，并使各国身份认证系统相互兼容，优化出入境管理。

  3. 资产与数据所有权： Wallet + RWA + zk

    基于区块链的账户体系，在使用 zk 证明保证国家信息与公民数据安全的前提下，保护个人资产所有权，降低金融资产交易门槛，为每一个人提供参与经济活动的权利。

  4）国际货币体系

从由美、英、法、西德、加拿大、意大利、日本等 44 个国家建立的布雷顿森林体系到专门为发展中国家提供更大的汇率灵活性和政策空间的牙买加体系，均是一些发达国家和国际机构占据主导地位，导致发展中国家在决策制定和国际金融规则制定方面的话语权较小。同时，由于发展中国家在金融体系和技术储备方面可能面临一些挑战，限制了他们在国际货币体系中的全面参与度。因此，国际货币体系的演变进程中，几乎没有发展中国家的参与机会。

而基于 Blockchain 的身份验证和基于 Cryptocurrency 的跨境即时支付可能是发展中国家创立与深度参与国际货币新体系的良机。

5）对数字身份与新货币体系的需求：

不同于发达国家与大经济体，第三世界国家主要依赖现金货币系统，缺乏成熟的信用体系，并刚刚接触移动支付，没有建立完善的银行体系，因此：

  • 国家借贷成本高，外部投资难。

  • 经济活动范围和类别受限。

  • 在采用数字货币体系的方案过程中没有太多历史阻碍。

当前仍有大量公民没有数字身份或没有从数字身份获利，如下图，各个发展中国家都有从数字身份的使用中获得 GDP 大幅增长的潜力，而公民个体则有可能获得从数字身份产生价值的 50% 以上。以尼日利亚为例，其有 81% 的劳动力是自我雇佣的，数字身份可以帮助他们参与小微创业与投资，从而获得来自政府的支持与保障，同时也能帮助政府找回因收入欺诈丢失的税收。而对于埃塞尔比亚，数字身份可以为其带来 4100 万金融业的潜在用户，因为这能让银行进行远程的身份认证，从而为其提供相应的金融服务，这将能产生价值 $ 2.5 billion 的投资。所以，数字身份在新兴经济体能创造的经济价值远远大于成熟经济体。

 世界级平台的意义

这种世界级的平台是一个全新的叙事，也有一个巨大的潜在市场，因为它的用户是全世界人类，这是 DeSoc 想要实现的，也是中本聪想要实现的。

当前，世界级平台涵盖了来自美国、中国、欧洲以及东南亚与非洲地区的主要科技和互联网巨头公司。

在美国，谷歌、微软、亚马逊和苹果等公司在全球范围内提供搜索引擎、云计算、电子商务、智能设备等互联网服务。中国的代表性公司有百度、腾讯、阿里巴巴和华为，它们同样涵盖了搜索引擎、社交媒体、电子商务、云服务以及通信设备等领域。欧洲虽然没有像美国和中国那样的巨头公司，但也有众多知名科技企业，如 SAP、Spotify 和 ARM。

此外，东南亚与非洲地区也有一些具有重要影响力的平台，如 Grab 和传音。Grab 是东南亚最大的出行服务平台，提供打车、送餐和快递等服务。传音是一家总部位于中国、在非洲市场占据重要份额的移动通信设备制造商，其手机品牌包括 Tecno、Infinix 和 Itel。

这些世界级平台在各自地区拥有庞大的用户基础和广泛的影响力。随着新兴技术如区块链、数字身份认证和支付的崛起，未来可能会涌现更多新的全球平台，它们有望改变现有的竞争格局，推动数字经济在全球范围内的蓬勃发展。

DeSoc 以及中本聪的比特币都试图打破传统互联网产业的垄断，通过去中心化的技术为用户提供更加安全、自主和隐私保护的数字身份和支付方式。这与本文提到的 Zuzalu、Worldcoin、以及 Musk 的商业版图构想有相似之处。他们都在探索数字身份认证和货币支付的领域，并尝试构建世界级的平台，以用户驱动生态发展，让用户拥有更大的控制权和隐私保护。

这些全球级平台的潜在市场非常庞大，因为它们的用户范围是全世界的人类。这不仅涵盖了美国、中国、欧洲等已经高度发达的地区，还包括东南亚和非洲等发展中国家。在发展中国家，数字身份认证和支付系统至关重要，因为许多人缺乏合法的身份认证和金融基础设施，而这些全球级平台正试图填补这一空白。这将为这些地区的人们带来更多的便利和机遇。

然而，要在全球范围内建立这样的全新平台，还面临着诸多挑战，包括技术上的难题、法律和监管问题、隐私和数据安全等。但无论如何，这些新兴的科技和平台都在试图改变现有的互联网格局，为用户带来更多的权益和选择。

结论

Sam Altman 和 Elon Musk 都想做同一件事，即身份+支付，但是实现路径有区别。

  • 1）Altman 从判定每一个人的身份开始，发放空投，逐步引导用户使用内置钱包，从而建立生态，更像是农村包围城市，从底层向上搭建。

  • 2）Musk 则是利用成型的推特社交图谱构建互联网身份，通过集成各自 Web 3 项目与资产，将 Web 2 转化为 Web 3 ，就像是让既有生态改朝换代的君主立宪。

通过比较 Worldcoin、Twitter（现为 Xeet）和 Vitalik 的社会学实验（Zuzalu），我们可以得出以下结论：

  • Worldcoin： Worldcoin 的主要目标是通过虹膜扫描等生物特征验证实现数字身份认证，并通过免费发放 Token（$WLD）向全球 10 亿人提供数字货币体系。Worldcoin 尝试从顶层向上搭建生态系统，通过逐步引导用户利用内置钱包，建立起全球的数字身份认证和货币支付体系。它重点解决世界公民身份认证和金融基础设施不完善的问题。

  • Twitter（Xeet）： Twitter 的愿景是通过社交图谱和现有的推特社交网络实现数字身份认证和数字货币支付。Elon Musk 计划利用现有的 Xeet 用户基础和社交图谱来构建数字身份。通过集成 Web 3 项目和资产，将 Web 2 转化为 Web 3 ，让现有的生态系统升级为新的数字支付生态系统。这种方式通过对现有平台进行改革和集成，将数字身份和支付引入现有的社交网络。

  • Zuzalu（Vitalik 的社会学实验）： Zuzalu 是一个社会学实验，通过借助 ZuPass 和零知识证明技术提供身份认证，并探索以太坊的自定义性和智能合约的可拓展性，实现自定义的身份认证和数字货币支付。Zuzalu 的目标是探索如何将区块链技术监督城市治理和运营，提高流程透明度、可信度和可验证性。这种实验更注重探索数字身份认证和支付方式在城市治理和社会运营方面的应用。

综上所述，这三个项目在目标和实现路径上存在差异。Worldcoin 重点解决第三世界国家公民设施的身份认证和金融基础问题，通过从多层多维度搭建生态系统来实现。Twitter（Xeet）计划通过现有的社交网络和社交图谱引入数字身份认证和支付，将现有平台升级为新的数字支付生态系统。Zuzalu 是一个实验性质的项目，探索未来城市的数字身份认证和支付方式治理和社会运营项目中的应用。
