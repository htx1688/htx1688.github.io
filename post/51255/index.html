随着以太坊生态系统的发展，在保持其高安全性的同时，在以太坊 Layer 1 上扩展交易吞吐量和降低成本将、变得异常困难。随着时间的推移，各种提案都试图解决这个问题：支付通道、状态通道和 Plasma 等等。但这些提案中通常会以牺牲安全性来实现其目标，不过正是这些提案的出现让 Rollup 方案不断完善。

Omni Network 旨在连通不同的 Rollup 方案，通过将以太坊再质押协议 EigenLayer 和 Cosmos SDK 融合到一个统一的平台中，聚合以太坊的模块化生态系统，使开发人员能够在以太坊上的所有 Layer 2 上构建全球应用程序，同时从 ETH 本身获得安全性。通过这种方式同时兼顾以太坊的安全性、Cosmos 的速度和可编程性，从而使开发人员可以访问更多用户，并且让用户本身可以更轻松地访问新的创新应用程序。

Omni Network 的特点

目前区块链生态系统趋向于高度分散，随着越来越多的 Layer 2   项目部署到主网，可以看到以太坊的流动性网络效应随着用户资金被隔离在孤立的环境中而不断降低。除此之外，许多 Layer 2 项目计划建立自己的 Layer 3   生态系统，这会进一步分散流动性和用户。

Omni Network 和其他跨链协议不同的是，Omni Network 认为目前以太坊 Layer 2 赛道的 Rollup 解决方案已经解决了以太坊的可扩展性问题，但不同 Rollup 之间却无法实现交互，导致流动性被碎片化分割在不同的L2网络中，不仅是用户体验大打折扣，也使得以太坊比以往任何时候都更加分裂。

因此 Omni Network 旨在成为“以太坊互操作性基础设施”，为 Optimism、Arbitrum、zkSync 和 Starkware 等 Rollup 网络提供互操作性，成为以太坊上所有 Rollup 网络和模块化应用的核心基础设施，实现无缝、安全的跨 Rollup 用户体验。总体而言有以下 3 个特点：

安全性：Ethereum 再质押

Omni Network 通过以太坊再质押（restakers）协议 EigenLayer ，来实现在满足互操作性需求的同时，从以太坊本身获得安全性。Omni Network 由验证者组成，同时运行以太坊节点和集成的 Rollup 节点，通过验证者进行再质押 ETH，并证明 Rollup 的状态更新以促进全球互操作性。

Omni Network 为跨模块化区块链安全性树立了新的标准。通过利用再质押机制，Omni Network 是第一个可以实现完全可定制功能并仍从 Ethereum 获得安全性的区块链。

高效性：Tendermint 共识

通过利用 Tendermint PoS 共识，Omni Network 的验证者在达成 Rollup 的状态一致性方面可以比以太坊 Layer 1   快一个数量级。

Omni Network 将 Tendermint 与 ETH 的再质押架构来确保 Rollup 的快速结算，通过在与 EVM 兼容的可编程 / 有状态层上进行分层，开发人员可以构建从所有 Rollup 中访问状态、交易、用户和加密资产的应用程序。通过这样的方式，Omni Network 实现了在保障安全的前提下，拥有 Tendermint 提供的高速。

功能性：完全 EVM 兼容性

此外，Omni 本身就是一个可编程、状态多样的平台，因此开发人员可以利用其快速结算和安全的互操作性，构建本地跨 Rollup 的应用程序。与部署在不同网络导致彼此孤立的应用程序相比，Omni Network 的可编程互操作性将通过多 Rollup 流动性聚合等属性，提供具有经济优势的交叉 Rollup 应用程序。

Omni 的执行层与 EVM 兼容，由 Cosmos 上的 Ethermint（一个 EVM 模块）实现，因此开发人员可以使用他们熟悉的 Solidity 工具进行开发。执行层包括内置功能，可访问从集成 Rollup 中的状态、消息和应用程序，从而创建无缝访问用户和流动性的体验。通过构建可编程互操作性层，Omni 将授权开发人员创建默认为全局适用的应用程序，来减少开发人员操作的复杂性。

项目进展情况

Omni Network 在今年 4 月 26 日宣布完成了 1800 万美元融资，本轮融资由 Pantera Capital、Two Sigma Ventures、Jump Crypto、Hashed、The Spartan Group 等参投。Omni Network 表示其核心技术合作伙伴包括以太坊再质押协议 EigenLayer，以及以太坊 Layer 2 扩容网络 Arbitrum、Polygon、StarkWare、Scroll、Linea 等。

Omni Network 将在 2023 年第三季度推出 Omni Network 公共测试网，并计划在 2024 年与 Arbitrum、Polygon zkEVM、Scroll、ConsenSys 的 Linea 和 Starkware 等主要 Rollup 合作伙伴合作推出主网的第一个版本。

Omni Network 应用场景展望

虽然 Omni Network 目前尚未推出测试网，但其旨在通过模块化应用的核心基础设施，实现无缝、安全地跨 Rollup 用户体验，已经吸引了众多目光。

在 5 月 3 日，Omni Network 和 Starkware 以及 Kakarot 团队一起举办了 AMA，就 Omni Network 计划如何扩大 Starknet 流动性等问题进行了讨论。Omni Network 表示 Starknet 应用程序将能够利用 Omni Network 的互操作性基础设施，将更多的流动性从以太坊的其余部分拉入 Starknet 生态系统。这意味着在不增加最终用户体验复杂性的情况下，开发人员将能够在整个以太坊生态系统中提供他们的应用程序，同时仍然保持 Cairo 的性能优势。

除此之外，对于 Omni Network 未来的应用场景还有很多，例如：

  • Cross-rollup 保证金账户和杠杆交易：在一个域上发布保证金，并在单独的域上使用该保证金进行交易；

  • Multi-rollup NFT 铸币厂：NFT 项目的用户可以不只是被限制在一个单一的生态中了，可以使用户从任何域铸造 NFT；

  • Cross-rollup 借贷：在一个域上存入抵押品，并在另一个域上以该抵押品为抵押借款；

  • Cross-rollup 流动性挖矿：从 Omni 协调所有的流动性操作，从一个地方跨协议和跨域转移资金。

在通过从 ETH 获得安全性，Omni Network 意图重新将以太坊生态系统连接成一个整体，使得以太坊上的项目不再局限于单个 Rollup，而是能够在整个以太坊生态系统中寻找用户和流动性；这种方式对于以太坊总体的影响很值得期待。
