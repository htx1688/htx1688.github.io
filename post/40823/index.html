作者：NFTGO 分析师 0x1CC304

大多数参与者认为， NFT 就是人们聚集在一起创建强大社区的“手牌”。

从历史上来看，我们从未像今天这样能够跨国界、跨种族形成一个超强凝聚力的组织。NFT是身份和社会组织的结合，没有政治、种族参数或其他人（即政府/政治家）制定的规则，具有伟大的包容性，更没有人会根据你的互动方式，或者虚拟装扮来定义你。

每个 NFT 持有者都可以解释自己对所属社区的含义，该群体具有广泛协调的目标，这就是成功的NFT 路线图的完成方式。

NFT 是一种现实生活的解决方案，类似于以太坊和比特币，但比这两者更宽泛、更抽象。

那么，元宇宙（Metaverse）呢？ 元宇宙为 NFT 持有者带来了一个场景，以及一个与社区进行虚拟但实时交互的平台，这就是虚拟土地发挥作用的地方。

人们对产权充满热情……著名的“你必须投资房地产”（不是财务建议）就是源于这个想法，我相信拥有产权的成就感很适合大多数人。

NFT 项目的目标是确保您拥有数字资产的所有权，这同样适用于数字房地产。我的意思是，在非物质世界中，你不是他人财产的“打工仔”。

元宇宙市场正在不断增长，我们看到 Meta（也称为 Facebook）和 Roblox 等中心化公司也在挖掘这个行业的潜力。

以下是元宇宙平台龙头 Decentraland 以及用户如何从平台中获利：

Decentraland

Decentraland 是一个基于 ERC-20 网络的去中心化虚拟现实平台，一个完美的类比：Decentraland 是 NFT 的“模拟人生”。

该平台由DAO监管，因此通过沟通>社区提案>投票，安排并完成路线图，Trello 网站提供了基于 DAO 投票和提案的动态路线图，值得一看。

Decentraland 头像

代币

MANA 是平台的原生代币，它有一些很好的用途，例如：对 DAO 提案进行投票，为你的头像购买数字物品，以及购买土地。您还可以在 Celsius、Gemini 和 Crypto.com 上质押MANA。

Decentraland 的好处是资产负债表非常透明，网站上的信息非常清晰。

您可以通过 Binance、FTX、Coinbase 等 CEX（中心化交易所）获得 MANA。另一种方法是通过 DEX（去中心化交易所）获得，如 UniSwap 或 SushiSwap。

如何在 Descentraland 上赚钱?

土地

土地（Lands）是元宇宙中非常有价值的数字项目，土地也被称为虚拟房地产。 Decentraland 目前拥有 90,601 块 16 x 16 米 [52 英尺 x 52 英尺] 的土地。

通过购买 2 块以上的土地，您可以创建一个庄园，这样您就可以开始打造一个越来越大的虚拟空间。

当您拥有土地或庄园时，您可以开始出租它们。一些用户在自己的土地上制作广告，让自己接触潜在的出租人。在第一次联系并达成协议后，租户开始为房东创建环境，并提供他们为举办活动所需的东西。以下一些网站提供智能合约来出租您的房产：

  • metaverse.properties（https://metaverse.properties/）

  • dclrenting（https://dclrenting.com/）

这样做可以让您在不出售房产的情况下获得被动收入，很多用户选择为画廊或时装秀租用他们的房产。

交易

获得一些收入流的另一种方式是交易MANA，也许您应该去学习技术分析来执行此操作并等待新闻热点的到来。

但是，交易 MANA 并不是您唯一可以做的事情。由于元宇宙的物品也是 NFT，因此您也可以交易可穿戴设备。下图描述了1 年范围内的交易量和市值变化。由于新的可穿戴产品随着时间的推移而增加，市值也因此而增加。另一方面，成交量一直保持稳定。

正如持有期分布（上图）所显示的，大多数持有者持有他们的资产一年或更长时间。如果您希望快速赚钱，您可能无法获得该项目的全部利润潜力，最好的办法是长期持有。

内容创作者

Descentraland 的好处是他们为创作者提供了 2 种工具，分别是：

  • The Builder（构建器）：构建器是进入内容创建者方面的最佳“入门包”，该工具通过简单的拖放编辑器使操作尽可能简单，有很多默认项目可以使用。

  • Decentraland SDK：此工具适用于具有编码技能的高级开发人员。在这里，您需要编写代码来创建自己的场景，正因为如此，想象力才是极限。

值得一提的是，您可以使用其他第三方建模工具来创建您的环境。

一旦您选择了要使用的工具，以下是一些作为创作者赚取收入的方法：

  • 出售数字物品：衣服、头像配饰。发布费用是每个项目 500 美元的固定成本（以 MANA 支付）。

  • 用一些可穿戴物品制作迷你游戏来出售：比如纪念品。
