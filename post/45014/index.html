撰文：aididiaojp.eth，Foresight News

Binance Labs 自 2018 年成立以来，不断识别、投资并赋能优秀的区块链企业家、初创公司和社区，已为多个有助于发展区块链生态系统的行业项目提供资金。

今年 8 月份币安联合创始人何一宣布开始正式执掌风险投资部门 Binance Labs，她曾在接受采访时表示，Binance Labs 将会寻找一些能够给行业带来长期利益的早期项目，熊市是一个非常好的投资时机，将重点投资基础设施、链上应用、数据分析和安全等相关领域。

Binance Labs 通过投资构建和支持去中心化网络的技术团队来不断推动加密行业的发展，据 Binance Labs 官网显示，在近三年中，Binance Labs 已经在超过 25 个国家投资了 200 多个项目，资金规模约为 75 亿美元。其投资版图几乎涵盖了 Web3 相关的大多数赛道，包括公链、协议、基础设施、NFT、链游、元宇宙、DeFi 和 CeFi 等。

Binance Labs 投资版图涵盖了 Binance Labs 官网显示及一些公开信息中涉及的投资项目简介。

NFT

NFT 基础设施

CryptoSlam是一个多链 NFT 数据聚合器，由 Mark Cuban、Animoca Brands、Reid Hoffman、Ashton Kutcher、Polygon 等支持。

Solv Protocol是一个用于创建和交易金融 NFT 的去中心化基础设施提供商，旨在通过称为 Vouchers 的金融 NFT 来改善项目筹款。Solv Protocol 发布了 ERC-3525 代币标准，使其可用于所有未来的金融 NFT，为 DeFi 的创新解锁更多的可能性。

NFKings是一家 NFT 创意和制作工场，旨在打造可持续的数字创意空间，拥有超过 100 项知识产权 (IP)。

NFT 交易市场

Refinable是建立在币安智能链上最早的 NFT 市场之一。

Melos Studio是一个去中心化的 Web3 音乐 NFT 平台。

StarryNift是一个综合性游戏平台和 NFT 一站式商店，推出了 Starryverse 3D Gallery，为 NFT 项目提供创作、孵化、拍卖和交易等支持。

GameFi&元宇宙

游戏

Alterverse是一个基于 NFT 的区块链游戏，将分四个阶段推出。其中包括 SKY CITY、科幻射击游戏 OUTPOST、硬核生存游戏 REKT 等。

Elfin Kingdom是一个基于 NFT 的 Web3 娱乐休闲型链游。

League of Kingdoms是一款由 Kudos 研发的策略类 SLG 链游。

Nine Chronicles是一款去中心化的在线角色扮演游戏，在由玩家提供支持的区块链上运行。

StarSharks是一款基于 BSC 链的链游，以海洋生态为基础，采用 SEA-SSS 双令牌经济模型作为其基础资产。

Voxies 是一款基于 Polygon 区块链的免费 3D 回合制战术 RPG 游戏，由 AlwaysGeeky Games 在 2021 年第四季度推出。

Heroes of Mavia是一款休闲基地建设类链游。

StepN 是一款内置 GameFi 和 SocialFi 元素的 Web3 原生游戏和生活方式应用程序， 由 Solana 区块链提供支持。

Block Ape Scissors是一个 Web3 区块链游戏生态系统和元宇宙项目。

GAT Network是一个币安智能链的 NFT 游戏生态系统。

Skyark Chronicles是一款可互操作的 JRPG 链游。

Alien Worlds是 一个 WAX 链上 NFT 元宇宙项目，游戏基础包括质押、Planet DAO、交易游戏卡牌以及跨链桥。

Axie Infinity 是一款玩家通过让三个 NFT 可收藏生物相互对抗来进行决斗的在线游戏。每个 Axie 都是独一无二的，并且作为 NFT 存在。

MOBOX 是一个社区驱动的链游平台，结合了去中心化金融 (DeFi)、非同质化代币 (NFT) 和区块链游戏的优势。

基础设施

Mythical Games是一家总部位于洛杉矶的链游公司，专注于商业、大众市场用例，支持由市场领导者和创新者组成的多元化开发团队。

Cocos-BCX是一款建设多平台游戏运行环境的链游公链。

Seascape Network是一个以社区为主导的链游平台，围绕 DeFi 和 NFT 设计经济模型。

Ultiverse是一个 VR 元宇宙平台，旨在链接 Web3 与兼容 VR 的虚拟世界。

Space是一个商业元宇宙平台。

Community Gaming是一个 P2E 锦标赛平台，为锦标赛组织者和游戏玩家提供良好的游戏体验。该平台拥有快速、透明且无需区块链知识即可使用的自动支付功能，在玩家获胜时能够立即获得报酬。

Highstreet是一个元宇宙服务平台，允许其他项目、品牌和游戏在其元宇宙平台上建立他们的项目。Highstreet 的定制化侧链能够与任何链兼容，目前正在试验 BSC 和 ETH，未来也将会探索 Avalanche。

Sky mavis是一个专注于技术的链游工作室，也是 Axie Infinity 游戏的开发商，旨在构建未来使虚拟世界成为可能的基础设施。

游戏公会

Gala Games是一家链游工作室，旨在打造区块链的 Steam 平台。除了游戏之外，还涉及艺术、音乐、电影和其他数字生活。

Merit Circle是一个以 DAO 为组织模式的链游公会，通过提供有竞争力的游戏内资产、战术和策略交易，以及为潜在玩家提供有竞争力的分成。

Cosmic Guild是一家链游公会，通过将 NFT 资本与加密游戏玩家联系起来，最大限度地发挥他们的潜力。

GAMEE是一家休闲移动类链游公司，通过游戏连接玩家、品牌和加密项目。

Netmarble F&C是一家元宇宙链游平台，为韩国手游开发商 Netmarble 旗下子公司。它以区块链技术为基础，涵盖了游戏、数字人物、娱乐、内容（网络漫画、网络小说 )、商务等。

GuildFi 是一家 Web3 基础设施提供商，连接游戏、NFT、公会和社区。它创建了一个互联系统，玩家的时间和精力可以在整个游戏和元宇宙中互操作。

DeFi

交易&流动性

Pstake是一个多链流动性质押协议，帮助用户在赚取 DeFi 收益的同时质押 PoS 资产。

Woo是一个加密货币流动性平台，具有零交易费用、深度流动性和可定制 GUI 等特点。

Wombat Exchange是一个多链稳定币交易平台，旨在以最小滑点交易稳定币和以最大的收益进行质押。

Symbiosis Finance是一个去中心化的多链流动性协议，聚合所有 EVM 和部分非 EVM 网络中去中心化交易所的流动性。旨在解决不同链之间的流动性分散，以及在使用 DeFi 和 Web3 生态时用户体验不佳的问题。

Biswap是一个基于 BNB Chain 的去中心化交易所，具有三类推荐系统和最低平台交易费用等特点。

Injective Protocol是一个基于以太坊的去中心化交易协议，旨在建立去信任化、可公开验证、可解析、流通性中立和抗抢先交易的技术方案。

DODO exchange是一个去中心化交易平台，由主动做市商 (PMM) 算法支持，并为提高资本效率而构建的流动性协议。

1inch Network是一个一站式访问去中心化金融协议，汇集了各种去中心化交易所的流动资金，帮助用户以更优惠的价格进行交易。

资管&收益

Tranchess是一个代币化资管管理和衍生品交易协议，通过各种风险回报解决方案提高收益率。

Tranching Protocol是一个市场效率协议，通过收益优化器和对冲方案等提高 DeFi 收益率。

Bella Protocol是一个 DeFi 资管平台，允许用户通过复杂的套利策略进行简单地存入，并享受高收益。

借贷&衍生品

ApolloX是一家去中心化交易所平台。

Xend Finance是一个去中心化信贷协议，旨在优化、改进和增值全球信贷的核心业务，消除传统金融的障碍，提供去中心化存储、借贷和投资业务。

跨链

Band Protocol是一个可拓展的去中心化跨链预言机，将外部数据和 API 聚合，并连接到智能合约。

Push Protocol是一个 Web3 跨链通信协议，为 dapps、钱包、用户等启用跨链通知和消息传递。

Copycat Finance是一个基于 BSC 的去中心化跨链流动性平台。

基础设施

DefiDollar是一个多链 DeFi 协议实验室，其产品包括 DUSD 稳定币指数、ibBTC 和 Opeth Finance 等。

Coin98 Labs是一个多功能的 DeFi 平台，旨在创建和开发 DeFi 协议生态和多区块链应用的 DeFi 产品。

CeFi

交易平台

MX Global是一个加密货币交易平台，提供比特币交易、风险管理和区块链管理服务等。

Hg Exchange是一家数字证券交易所，旨在促进证券的跨境发行与交易。

Bitsika是一款专注于非洲市场的支付平台，可帮助人们在各个国家或地区即时汇款，支持买卖比特币和 BUSD。

Capitalise是一家交易自动化平台，通过其专有 AI、机器学习、自然语言处理（NLP）技术提供平台功能，使经纪商能够进一步扩大现有产品范围，与交易者保持互动并产生新的收入流。

Mobile Coin是一个隐私支付协议，旨在为移动设备用户提供加密货币支付、管理服务。

公链&生态

公链&生态

Aptos 是由 Meta ( 原 Facebook）原开发团队成员构建的 Layer1 公链，采用了 Diem 的编程语言 Move 和 Diem-BFT 共识技术。

Celestia是首个模块化区块链网络，主张将单一公链的共识、结算、执行等功能划分为不同层区，然后通过优化功能层区来提升性能，还允许专门的区块链托管应用在其之上部署，在可扩展性、灵活性和互操作性方面进行了优化。

Celestia 依赖 Optimistic Rollup 和数据可用性证明两种关键技术，还与 Interchain Foundation 合作，为 Cosmos SDK 带来 Optimistic Rollup 支持。

Kava Network是一个去中心化 L1 区块链，拥有 Cosmos 的互操作性，且受到以太坊开发人员支持。

Harmony 是一个去中心化高性能区块链，支持代币委托和区块奖励的复利，实现双重签名惩罚的抵押机制，以保护区块链安全性。

Polygon 是一个建立在以太坊上的 Layer2 扩展解决方案上的公链，使开发人员能够以低交易费用构建可扩展的 DApp，而无需牺牲安全性。

Figment是一家区块链基础设施提供商，为 60 多个 PoS 网络提供质押和技术支持，旨在让投资者和开发者能够轻松构建下一代互联网。

BitTorrent是一个点对点文件分享协议，旨在建设一个更好的互联网，让人们自由、安全地传递信息。

Moonbeam Network是基于波卡与以太坊兼容的智能合约平台，旨在帮助开发者轻松构建原生可互操作的应用程序。

Nym Technologies是一个基于 Mixnet 协议的隐私基础设施提供商，创建了可用于匿名加密支付、消息传输和网络导航的混合网络。

Tea是一个链上开源软件构建平台，旨在通过为以前无偿劳动的创造者和维护者提供价值来彻底改变开源开发。

Cere Network是一个基于波卡的去中心化数据云平台，为企业和消费者提供一个无需信任、可扩展的全球去中心化生态系统。

Astar Network是 Polkadot 生态系统上的智能合约平台，同时支持 Wasm、EVM 和二层解决方案。

Biconomy是一家区块链开发工具提供商，旨在帮助区块链开发成员能够更加轻松地建设其 Web3 项目。

Axelar 是一个可通过去中心化网络连接 L1 区块链的通用互操作协议。开发者可以通过 Axelar 的 SDK 进行简单的异步调用，无缝调用另一个受支持链上的合约，它的所有节点都支持 Ethereum、Avalanche、Cosmos 等其他区块链网络。

Bird.Money是一个区块链技术解决方案提供商，由机器学习提供支持的地址级预测链下分析。

Cartesi是一个区块链开发操作系统，允许开发人员使用 Linux 和标准编程环境构建去中心化逻辑，从而保持区块链的去中心化和安全性。

Celer Network 是一个跨链互操作性协议，能够帮助用户在该平台上快捷地开发、运行与使用可高度扩展的分布式应用。

MultiversX（原 Elrond）是一个高度可扩展、快速且安全的区块链平台，适用于分布式应用程序和企业用例等。

ZkSync是基于 ZK Rollups 架构的扩容方案，旨在不降低安全性和去中心化的情况下拓展以太坊技术和价值，加速加密货币的大规模采用。

Marlin Protocol是一个基于 Layer 0 的扩展解决方案。

Zecrey Protocol是一个基于 zkRollup 的 L2 扩容协议，旨在保护用户对其资产的所有权，具有隐私性和可拓展性。

Manta Network是一个 Web3 隐私保护协议，通过 zkSNARK 等领先的密码学架构提供端对端的隐私保护，旨在为区块链生态提供更加便捷的隐私保护服务。

Dusk Network是一个隐私区块链协议，由零知识密码学提供支持的 Layer1 区块链，设计了一种称为「Proof of Blind Bid」的共识，来保护网络隐私。任何人都可以使用 Dusk Network 来创建零知识 DApp。

ARPA是一个隐私计算项目，启用智能合约、数据隐私保护以及可扩展的计算分片等技术。

Oasis Labs是一个去中心化区块链隐私计算平台，旨在解决区块链在性能、安全和隐私上的痛点。

Automata Network是一个去中心化隐私服务协议，旨在增强去中心化应用的隐私保障能力。

HOPR是一个 Layer 0 元数据隐私保护协议，可用于连接设备和云端、实现链上隐私交易等。

跨链

Multichain是一个跨链路由器协议，支持以太坊、Aribiturm、Avalanche、BNB Chain、Fantom、Polygon、Moonbeam 等多个网络间的资产转移。

LayerZero是一个提供无需信任的跨链通讯协议，利用了轻节点的技术原理，将中间链的置信环节分开，从而以更低的费用换取更高的安全性。

钱包

Sender 是一个基于 NEAR 的非托管钱包。

Web3Auth是一个分布式托管 EOA 钱包，支持社交登录，并允许用户通过社区账号连接以再各种应用中验证身份。

Math Wallet是一款多链加密钱包，可实现浏览器拓展。

Web3

社交

Socios是一个基于 Chiliz 的粉丝代币应用，由 Chiliz 侧链提供支持，并允许体育团队发行粉丝代币。该应用让球迷可以在他们最喜欢的俱乐部获得代币化投票权，同时允许球迷与他们的球队互动，并提供独特的奖励，包括赛事精彩瞬间的体验、预测游戏结果和比赛日挑战。

JuventusFC是基于 Chiliz 的尤文图斯足球俱乐部世界杯代币项目，代币总量 2000 万，流通市值 1400 万美元。

Chiliz是一个基于区块链的电子竞技投票平台。

Paragraph是一个 Web3 内容发布平台，由 Colin Armstrong 于 2021 年创立，可帮助 Web3 领域作者、DAO 和 NFT 社区通过一体化的通讯服务将其内容货币化，内容创建者和社区建设者可以使用 NFT 或 ERC-20 代币通过会员机制管理其用户或粉丝。

BBS Network是一个基于区块链的论坛，所有关键数据都保存在链上，每个帖子都是 NFT，抗审查且保持网络开放性。

Audius是一个以内容创作者为主导的音乐共享和流媒体协议。

Contentos 是一个数字内容区块链平台，为各类型 DApp 产品、内容创建者、用户和广告商提供服务，目标是将权益回归创作者和内容消费者。

安全

Ancilia是一家网络安全提供商，提供自动化实时威胁检测和预防服务 SaaS 产品，由其专有的行为分析技术支持。

Certik是一家区块链网络安全公司，利用一流的形式验证和人工智能技术来保护盒监控区块链、智能合约和 Web3 应用程序。

Salus Security是一家全方位的区块链安全公司，提供自动化智能合约审计和手动漏洞检测等服务。

Keylabs 是一家专注于加密货币、区块链、硬件安全的公司，提供加密货币钱包技术、安全储存、智能合约安全、多方签名、钱包审计等服务。

基础设施

Lifeform是一家视觉 DID 解决方案提供商，开发了一个跨平台的 3D 虚拟人头像编辑器，该编辑器通过 NFT 使人们能够使用去中心化身份认证 DID 和保留多个零知识匿名数字身份。

Magic Square 是一个 Web3 App Store 解决方案，社区通过 DAO 机制对应用程序进行审查和优先级排序，并使用明确的收入指标来激励用户和验证者的参与和使用。项目开发人员可以通过 Magic Store 和 Magic Boost 轻松扩大用户群。

Numbers Protocol是一个去中心化的图片网络，旨在为数字媒体创造社区、价值和信任，帮助用户保存数据所有权。
