NFT 观「鲸」技巧和窍门。

撰文：Nansen

一.介绍

NFT 交易者应该有一个目标：找出如何找到和跟踪 NFT「巨鲸们」。 区块链技术从根本上透明、开放和可审计的特性催生了一种新的社交化复制交易实例：观鲸。社交化交易在传统金融中很流行，涉及散户观察和复制「投资专家们」或同行的交易行为。抄其它投资者的作业不需要任何技能；投资者喜欢抄袭社交媒体上大 V 的投资策略，以提高自己投资组合的收益。 因为传统金融的不透明性，抄专家的作业，不太可行。

在加密货币领域，所有人都可以使用追踪和复制「巨鲸」的能力。「巨鲸」们值得关注：最近的一份报告详细说明，不到 20% 的钱包地址控制着以太坊上 80% 以上的 NFT。与普通市场参与者相比，这些大型参与者在信息不对称的情况下运作，他们通常拥有更多的情报，使他们能够持续跑赢市场。知道如何找到这些鲸鱼的投资者可以指导他们的交易，做出更明智投资决策。 

NF 是独特的加密资产，代表存储在区块链上的数字资产的所有权。本文探讨了投资者如何找到这些 NFT 鲸鱼、跟踪他们动向并做出更好的投资决策——最终提高他们的回报，同时向更有经验和资本更雄厚的投资者学习。

二.什么是 NFT 巨鲸

「巨鲸」是传统投资者和加密投资者用来描述拥有雄厚资本的大户。「巨鲸」可直接影响市场并可能导致价格剧烈波动。 NFT「巨鲸」是拥有大量加密相关资产 (NFT) 的收藏家。 

寻求最大化回报的散户投资者应该利用他们可以使用的所有工具，因为市场本质上是对抗性的； 信息是一种武器，可以让一个投资者比另一个投资者更具优势。 链上数据是一种非常有用的资源，散户在找到「巨鲸」后可以复制「巨鲸」的交易。 因为链上数据是实时的，通过跟踪「巨鲸们」活动，投资者可以抢在价格上涨之前 Mint 铸造。

三.如何发现 NFT 巨鲸

Nansen 代表了市场巨鲸信息的最佳来源之一，提供来自数百万钱包数据的链上分析。 投资者可以浏览各种数据并实时观察可视化的链上数据反馈，使其成为寻找 NFT 巨鲸的绝佳平台。 Nansen 是保持市场领先地位和超越大多数人的重要工具。 

Nansen 的 NFT Paradise 天堂是一个功能强大的数据页，具有宏观 NFT 市场动态，包括来自 OpenSea 的地板价实时更新。 投资者无需在 Twitter 或 Discord 上浏览大量不必要的信息，就可以看到 NFT 趋势和热门 NFT 收藏品。

Nansen’s NFT Paradise Leaderboard

NFT 的 NFT 天堂，投资者可以找到他们需要的 NFT 获利排行榜，包含并标记出链上最好的 NFT 铸造者和交易者的最赚钱的钱包。 使用该工具，投资者可以定位和追踪 NFT 鲸鱼。 

通过时间框架排名，投资者可以选择跟随后起之秀或具有良好业绩记录的老牌鲸鱼。 

当用户按时间过滤时，数据页会排除只交易过一次的幸运用户，留下在 3 个 NFT 集合中至少有 10 笔 NFT 买卖的钱包地址。 从广义上讲，这是一种更通用的工具，最适合用于精准定位。 举个例子，当前排行榜第 10 名的是「Pranksy」，它是 Bored Ape 无聊猿俱乐部巨鲸，曾一度拥有超过 1,251 只 Apes NFT，但此后已售出并实现盈利。

我们可以进一步通过 Nansen Wallet Profiler 钱包分析器，每个人都可以查看 Pranksy 的在哪些 NFT 上实现了收益，了解哪些 NFT 值得持有，哪些最适合斩仓。 

如何跟踪 NFT 巨鲸？

一旦投资者通过位于 NFT 天堂下的利润排行榜或通过 NFT 上帝模式识别出 NFT 巨鲸，找到特定 NFT 集合的顶级持有者，接着他们就可以利用 Nansen Wallet Profiler 钱包地址分析器。

这提供了对任何给定链上地址的所有活动的洞察。 基础内容有，例如数字资产持有量和这个地址与哪些合约交互过。 NFT 鲸鱼追踪最相关的是 NFT 部分，它显示了这个地址当前拥有的所有 NFT 收藏品、拥有的数量以及已实现的收益。

投资者可以看到已实现收益的百分比来自哪些收藏品以及收藏品的已售出百分比。 无疑是最好的微型链上指标分析器来源之一。 最后是跟踪地址的交易数据，包括买入、卖出和铸造。

具体案例 - Shaw

Shaw 是一位获利巨大的 NFT 收藏家，于 2021 年 2 月加入 OpenSea，与 moin3au.eth 类似，他在 Crypto Punks 和 Meebits 收藏中获得了巨大收益，并且最近开始购买生成艺术 Art Blocks Curated 作品。

Shaw 花费了 1230 ETH，并产生了 13200 ETH 的收入，总利润为 12000 ETH。 然而，到目前为止，钱包只售出了 78% 的集合，并且仍然拥有这个 NFT 集合的 2 个，并且将来可能会通过卖掉这些来获利。

在不到两年的时间里，这款钱包在其原始投资倒手转卖 NFT 上的收益超过了 10 倍，而 Shaw 是 NFT 交易的典范，这使得这款鲸鱼钱包成为追随者的绝佳选择。

结论

NFT 行业继续快速增长和发展。 如果想要比大多数人更好的市场决策的投资者，就需要获得比其它人更高质量的阿尔法信息。 Nansen 是定位 NFT 鲸鱼并实时跟踪其交易的完美工具。 通过观察巨鲸操作，投资者可以辨别出正在上升的收藏品和失宠的 NFT 项目方。 

任何学习模仿专业人士的技能都会产生巨大的好处，Nansen 的 Wallet Profiler 钱包分析器 允许市场参与者跟踪顶级 NFT 交易者并复制他们的交易。 随着时间的推移，接触 NFT 巨鲸的投资者将开始了解这些巨鲸在检查 NFT 藏品时寻找的迹象，并能够自己模仿它们操作。 信息是任何交易者可以使用的最大武器。 Nansen 将大量数据提炼到一个单一的接触点并跟踪巨鲸，这样投资者就可以追踪它们的踪迹并提高他们的交易业绩和收益。
