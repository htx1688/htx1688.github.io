这是一篇关于加密货币的过去、现在和未来的文章，它会如何使世界变得更好，以及它如何正在使世界变得更糟。

介绍

2021 年 11 月，加密市场达到 3 万亿美元的市场最高点。截至本月，总市值徘徊在 1 万亿美元以下。市场崩盘当然是意料之中的。大约四年前的同一时间，加密货币的总市值低于 2000 亿美元。

价格波动确实很疯狂。但不管目前的价格如何，我认为现在是不看图表并提出更深层次问题的最好时机。在最近的牛市中，加密市场真正产生了什么？

最重要的是，能源和金钱资源的巨大浪费。

目前，我加入了批评家的行列，他们认为“加密货币”充其量只是基于虚增价值的快速致富计划。

在我看来，加密行业要么重新定位，要么变得无关紧要。要么，加密行业正在经历根本性上的转变，并将在 2024 年或 2025 年的下一轮牛市之前产生持久影响。否则，唯一真正的合并将是与现有金融系统的合并。

让我们看一下加密货币的过去、现在和未来，以正确看待这件事物。

过去

1. 密码朋克和比特币

可以说，加密运动始于 90 年代初，当时来自旧金山湾区的三位计算机科学家发起了一个邮件列表，讨论密码学、数学、政治和哲学。邮件列表的成员被称为“密码朋克”。

在互联网的早期，密码朋克已经预见到人类在数字时代将面临的隐私和安全挑战。他们建议将强大的密码学作为避免审查、政府监视和隐私侵犯的关键措施。

在The Crypto Anarchist Manifesto中，“三位原始密码朋克”之一的 Timothy May 说：

“国家当然会试图减缓或阻止这项技术的传播，理由是国家安全问题、毒贩和逃税者使用该技术，以及担心社会解体。其中许多担忧将是有效的（..）但这不会阻止加密无政府状态的蔓延。”

正如我们早些时候看到的那样，比特币的主要用例之一确实是在暗网上洗钱和交易。然而，比特币作为黑市商品（毒品、枪支、儿童色情等）货币的早期声誉并没有阻止加密运动——远非如此。

Timothy May 还认为“加密”将从根本上改变社会：

“正如印刷技术改变和削弱了中世纪行会和社会权力结构的权力一样，密码学方法也将从根本上改变公司和政府干预经济交易的性质。 “

他是否正确还有待观察。

这是“三位创始密码朋克”之一埃里克·休斯（Eric Hughes）在《密码朋克宣言》（A Cypherpunk’s Manifesto，1993）中的另一句话：

“对于电子时代的开放社会来说，隐私是必要的。隐私不是秘密。私事是不想让全世界知道的事，秘密是不想让任何人知道的事。隐私是有选择地向世界展示自己的力量。

由于我们渴望隐私，我们必须确保交易的每一方只知道该交易直接需要的知识。由于任何信息都可以谈论，我们必须确保我们尽可能少地透露。在大多数情况下，个人身份并不突出。当我在商店购买杂志并将现金交给店员时，无需知道我是谁。 (..)

因此，开放社会中的隐私需要匿名交易系统。到目前为止，现金一直是主要的此类系统。匿名交易系统不是秘密交易系统。匿名系统使个人能够在需要时且仅在需要时披露其身份；这就是隐私的本质。”

看看上面 Eric Hughes 的引用，比特币符合他对匿名交易系统的描述。

比特币并非完全“匿名”，因为所有交易都是公开的。然而，正如中本聪在比特币白皮书中所建议的那样，可以通过保持公钥匿名并为每笔交易使用新的密钥对来维护隐私。比特币被设计成一种“数字现金”，“尽可能少地透露信息”，但不是秘密交易系统。这是比特币白皮书简介中的一段话。

“这导致商家必须防备他们的客户，骚扰他们以为了得到更多的他们不再需要的信息。不可避免的，一定比例下的欺诈性交易是可以接受的。虽然使用物理货币可以避免这些成本和支付的不确定性，但是没有商家在不通过可信任的三方的沟通渠道前提下去支付。”

2. 比特币和 DeFi

密码朋克的思想体系激发并产生了大量创新。吹哨人网站 Wikileaks、匿名浏览和交流 Tor 网络的软件、点对点文件共享平台 BitTorrent，当然还有最重要的比特币。

比特币并非从虚无中出现，而是 20 年公共研究和实验创造数字货币的高潮。以下是比特币之前的 94 个数字货币项目列表：

比特币是作为一种公共物品，第一个 DAO，独立于国家控制，尊重用户隐私，同时为 DeFi（一个基于区块链技术的更广泛的金融生态系统）打开了创意之门。

比特币是一个相对较慢的网络，每秒最多只能处理 7 笔交易。由于比特币社区决定支持去中心化和安全性而不是可扩展性，因此 2022 年没有任何版本可以将 BTC 视为杂货店购物等的便捷货币。未来，Lightening Network 和其他Layer 2解决方案可能会改变这种情况。

在那之前，以太坊等其他网络上的 DeFi 负责更先进的区块链功能，可以将数字货币扩展到替代的全球金融系统。 DeFi 可以成为在不稳定的世界中提供安全保障的绝佳工具，为公益项目提供资金机制，支持全球范围内的合作和伙伴关系，为来自崩坏和腐败司法管辖区的人们提供财务自由的机会，甚至缩小代际贫富差距。

DeFi 只是一个更灵活、更透明、更公平、更开放的金融系统，继承了中本聪在创世区块中编码的比特币的理念和使命：

“泰晤士报 03/Jan/2009 财政大臣濒临第二次银行救助”

这条神秘消息是对比特币推出同一天发表的《泰晤士报》头版文章的引用。以及对银行的明确鄙视。

在过去几年中，银行和金融机构已成为加密货币的主要买家。我认为这本身并没有什么问题，相反，我认为这是一个积极的信号。令人担忧的是，寻求控制的行为加上激进的政府监管将吞噬 DeFi，直到密码朋克和比特币的价值基础和文化遗产消失。为了长寿，DeFi 协议需要在未来的每个十字路口选择去中心化而不是增长。

现在

3. 加密和传统金融

在 Instagram 上，我们看到早期的加密投资者在迪拜炫耀他们的房屋和游艇。亿万富翁名人在 Twitter 的 NFT 化身上花费了大量资金。 YouTube 上的金融服务广告——通常以浓重的美国口音讲述——不断试图通过为加密货币和 NFT 做广告来吸引新交易者。与此同时，Terra、LUNA和Celsius等庞氏骗局夺走了许多普通纳税人的生计。

总的来说，加密是一种变相的资本主义。但更糟。退出计划不受惩罚，价格波动更加极端，尽管大多数加密项目都被宣传为“去中心化”，但造就或破坏市场的是鲸鱼。更重要的是，在气候变化和地缘政治不稳定正在深刻挑战人类文明的时代，加密货币并没有产生有意义的影响。

人们总是谈论加密的潜力——这确实很棒——但这种潜力已被证明是希望的鸦片。危急关头，0.01%坐拥大半财富，掌控行业，大部分投资者“乐在其中”，大多数人对这个行业要么有点好奇，要么持怀疑态度，但从来不敢涉足。这听起来像传统金融吗？

不幸的是，密码朋克的哲学传统以及 DeFi 的潜力已经被对价格走势的狭隘的关注所掩盖。这种趋势不是特定于加密货币的，而是传统金融的产物。看看有多少合作是一心一意地把股东价值最大化放在首位。或者 GDP 如何成为衡量国家财富的最常用标准。

Michael Jensen 和 William Meckling在 1976 年的一篇论文中著名地表达了“股东至上”的教条，他们将公司的股东描述为雇佣公司董事和高管作为股东“代理人”的“负责人”。在这种观点下，对客户、员工和更广泛社区的责任不如股东价值最大化重要。无论您从法律、道德还是商业战略的角度来看，该教条都存在严重缺陷和适得其反。然而，即使到了 2022 年，一些商界学者仍顽固地声称，公司的主要目的，甚至唯一目的，是实现股东价值的最大化。

GDP 是一个国家生产的所有服务和产品的总和，但它没有考虑社区服务、志愿者工作、知识进步、不平等、犯罪率、心理健康统计数据或气候影响。根据 Rutger Bregman 的说法，理想的“GDP公民”“将是一个患有癌症的强迫性赌徒，他经历了旷日持久的离婚，他通过大量服用百忧解并在黑色星期五发疯来应对。”

4. 加密货币和资本主义

许多对现代全球经济有深入了解的有思想的人说，它无法为大多数人类提供福祉。

对短期经济开采的狭隘关注——也被称为资本主义的贪婪——对人类的自然栖息地造成了不可复制的破坏，是人为气候变化的主要驱动力。资本主义也是马太效应的原因。

“因为凡有的，还要加给他，使他有余；没有的，连他所有的也要夺去。”

——马太福音 25:29，修订标准版。

马太效应意味着越富的越富，越穷的越穷。因为这就是资本主义的规则。

加密货币现在是资本主义的工具，而不仅仅是寻求摆脱数字监控和压迫的运动。加密正在为极少数人提供物质享受，同时助长了环境破坏和不平等。如果某些事情没有很快改变，那么未来看起来很黯淡。

未来

5. Web 3.0 和 DeFi

虽然大多数加密货币都在走向火葬场，但 Web 3.0 和 DeFi 将继续存在。

Web 3.0 概念将作为一种比我们今天使用的更充满活力、更活跃的互联网版本而继续存在。 Web 3.0 由 DeFi 提供激励和经济支持。

DeFi 概念强调数字货币的功能，而不仅仅是它们的存在。比特币是“法定货币”唯一有价值的竞争对手，因为该网络可能过于分散而无法被政府监管关闭，并且可能过于安全而不会被黑客攻击破坏。

所有其他数字货币实际上都像中心化公司一样运作。甚至以太坊。想想 2016 年的 DAO 攻击，以太坊重写了其历史以“救助”黑客攻击的受害者。一种崇高的行为，它提出了关于区块链的不变性和审查阻力的问题。

数以千计的数字货币本身在现实世界中几乎没有价值，但由于它们各自的功能，它们可能在更大的 DeFi 生态系统中具有重要价值。类似于公司之所以有价值，是因为它们为社会提供了一些服务或产品。占用办公空间但没有明确使命或销售有价值的产品或服务的公司将不会长久。

6. ImpactDAO

在 DeFi 和 Web 3.0 的交汇处，我们有 ImpactDAO，这可能是加密如何为世界带来好处的最明显例子。在 Kevin Owocki 和 Alejandra Borda 关于 ImpactDAOs[8] 的书中，该概念被定义为“任何使用加密为世界创造积极外部性的 web3 项目”。

以下是非常简要总结的四个 ImpactDAO 示例：

- 人类证明：任何人都可以通过视频提交注册的“已验证人类”注册表。作为注册人，您还需要有人为您“担保”。注册的人可以向其他人证明他们是人类而不是机器人。他们还通过注册获得定期部分的通用基本收入代币（$UBI）。

- Celo：具有智能合约功能的权益证明区块链。 Celo 的部分使命是让加密支付像发送文本一样简单。部分原因是为了帮助世界上没有银行账户的人通过手机获得金融服务。 Celo 用户可以使用他们的电话号码作为钱包的公钥以低费用进行和接收几乎即时的付款。

- Gitcoin：通过众筹、赏金和黑客马拉松等不同的筹资机制，从头到尾支持开源软件和 Web 3.0 基础设施的开发。 Gitcoin 通过 GitcoinDAO 由其社区管理（请参阅我对 DAO 的介绍）。到目前为止，Gitcoin 已为开源软件贡献了 6900 万美元，并支持 Uniswap、Yearn Finance 和 1inch Exchange 等主要 DeFi 项目。

- Toucan：通过将碳信用放在公共区块链上，并附上有关特定项目的所有相关元数据，帮助标记碳信用。碳信用表示可测量和可验证的去除、减少或避免相当于一吨二氧化碳的温室气体排放。使用 Toucan，碳信用可以作为 DeFi 中的商品使用，也可以用来兑换 NFT 或游戏体验。

ImpactDAO 是一种思考价值创造的新方式。他们应用比特币和加密货币的博弈论和激励结构，以及 DeFi 的价值，即其“可组合性”元素，来支持、构建和货币化公共产品，如科学研究、开源软件或碳补偿。 ImpactDAO 在再生经济的框架下运作，而不是在资本主义的硬性法则下运作。

再生经济学

“再生资本主义”的概念是由摩根大通退休董事总经理约翰·富勒顿在 2015 年的一本小册子中定义的。在作为“华尔街兄弟”度过了 20 年之后，富勒顿决定将他的生活转向一个新的方向。根据他自己的说法：

“对华尔街的发展方向以及我所钦佩的文化的丧失感到失望，我决定在没有明确的未来计划的情况下离开。然后，在辞职后不久，我亲身经历了 9/11。我的幻灭与绝望融为一体。

接下来是多年的寻找。我一直在寻找如何理解一个我无法再向我的孩子们解释的世界。在某种程度上，我也在寻找自己的目的。

这次搜索首先让我看到了我们现在面临的深刻的、相互关联的危机——生态、经济和社会——包括我们正在破坏地球支持我们所知道的生命的能力的令人震惊的前景。然而，我最令人吃惊的发现是，现代经济和金融体系——华尔街“天才”（像我一样）实践得如此出色——形成了这些系统性危机的根本原因。这种认识发生在 2008 年金融危机之前，并且独立于它而存在。然而，这种不负责任、贪婪和欺诈的令人震惊的表现进一步证实了华尔街迷失方向的现实。 “

加密也迷失了方向……

“再生经济”基于整体的世界观，在这种世界观中，金融系统不被视为与社会其他部分或地球生物圈隔绝的实体。相反，价值和财富是根据整体的福祉来定义的，包括所有形式的资本：智力、经验、社会、文化、生活，甚至精神，以及财务和物质。简而言之，金融系统几乎被视为一个活的有机体，或者是一个具有生命成分的自然界生态系统，并且是一个更大整体的一部分。

提供关于再生经济学如何运作的更详细的总结超出了本文的范围。现在可以说，DeFi 的品质和加密货币的激励驱动、自我调节结构可以创建一个繁荣、更可持续、相互关联的金融系统，这将使大多数公民而不是少数人受益。

结论

我长期相信比特币。然而，围绕加密的公开辩论过于关注短期价格走势了。 DeFi 和 Web 3.0 具有以积极的方式改变文明的巨大潜力，其价值观植根于密码朋克运动和比特币。然而，大多数人都忙着看屏幕上的数字和图表，当价格下跌时，他们会这样做，越来越多的人会被骗（市场绝望与骗局数量之间存在相关性），各国政府开始关注。这是一种大多数网络不想要也无法处理的注意力。

在我看来，等式如下所示：

加密货币没有带来太多价值 + 价格下跌和诈骗 = 政府监管

政府监管 + 缺乏权力下放 = 传统金融

传统金融 = 资本主义

资本主义 = 富裕的少数群体、负面的气候影响和全球不平等

我真诚地希望 ImpactDAOs 取得进展，并且 DeFi 以及 TradFi 朝着再生经济框架发展。
