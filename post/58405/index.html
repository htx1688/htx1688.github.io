原文作者：Nancy，PANews

近日，Magic Book 创始人 Jake Browatzke 在社交平台曝出，区块链游戏平台 Gala Games 团队陷入内讧，创始人兼 CEO Eric Schiermeyer 和联创 Write Thurston 已相互提起诉讼，彼此称对方“盗币砸盘”及“滥用资金”等，所涉及的$GALA 资产价值高达 7.3 亿美元。

随后，Gala Games 总裁 Jason Brink 发文证实了这场内部纠纷，并表示不会对正在进行的诉讼发表评论，他本人秉持着透明度的理念，鼓励大家完整阅读两位联创的诉状，也欢迎大家进行更深入的探讨。那么，这两份诉讼书中透露了哪些信息？又有哪些值得关注的点？

Eric Schiermeyer 被指销毁价值超 6 亿美元代币并私下创办竞争公司

在文章开始前，先来了解下 Gala Games 的公司背景。Gala Games 则是一家区块链视频游戏和娱乐公司，年收入达数百万美元，拥有数百名员工。根据成立之初的约定，Eric Schiermeyer 和 Write Thurston 将以 50 ： 50 的比例分享所有商业机会，且公司的所有控制权是由他们两位董事进行授权。

8 月 31 日，Eric Schiermeyer 和 Write Thurston 在美国犹他州地方法院对彼此提起诉讼。其中，Write Thurston 以 True North 的名义对 Eric Schiermeyer 进行指控，这是家营业地点位于美国犹他州沃萨奇县的投资公司，也是 Gala Games 母公司 Blockchain Game Partners（BGP）的股东，持有 44.762% 的股份。Eric Schiermeyer 自 2009 年以来一直担任着 BGP 总裁职位。

Write Thurston 在这份起诉书中指出，在代币$GALA 初始分配之后，True North 和 Eric Schiermeyer 获得的任何剩余和所有额外的 Gala 和 NFT，并存储在后者单独控制的钱包中。但在未经董事会或股东批准或不知情的情况下，Eric Schiermeyer 销毁了属于其股东和 Gala 社区其他人的价值超 6 亿美元的 GALA 代币并浪费数百万美元的公司资产，甚至未按公开声明那样升级$GALA 代币而是转入个人控制钱包。不仅如此，Eric Schiermeyer 还从 Gala Games 中借走数百万美元资金用于个人使用，比如支付 500 万美元分期购买一架公司飞机，以及私下在瑞士和迪拜创建了本应属于 Gala Games 的离岸实体，并单方面自己设为控股股东抢夺了本该属于 BGP 的商业机会，甚至还提供严重不完整或错误的财务信息，也没有向 True North 提供必要的公司记录。

此外，此诉讼还指出，Eric Schiermeyer 已经或打算转让或使用 BGP 目前的大部分业务、技术平台、资产和知识产权。比如近期他宣布将分配 8000 个节点给 BGP 员工及关闭 True North 的节点，这些节点将会产生一定价值的数字资产。但 Eric Schiermeyer 在未向董事会或股东披露和批准情况下做此决定将损害股东利益和公司价值。而在此之前，Eric Schiermeyer 也曾私自关闭 True North 的节点。

Write Thurston 表示 Eric Schiermeyer 的渎职、管理不善和自营交易给 BGP 的声誉和股东资产造成了数亿美元的损失，他违背了自己的受托义务并拒绝了所有善意的交涉。而考虑到 BGP 需要 2/3 的多数投票表决，且只有 True North 和 Eric Schiermeyer 两位大股东，因此 Write Thurston 希望可以借此诉讼免去其董事、总裁和首席执行官职务，并给予不少于 6 亿美元的损害赔偿，以及遵守创始人协议条款禁令。

Write Thurston 被指盗取并出售价值超 1.3 亿美元 GALA，多年未参与公司运营

而 Eric Schiermeyer 则在另一份诉讼书中指出，Write Thurston 和 True North 从 Gala Game 窃取了超 86.4 亿枚$GALA（约占当时发行的 GALA 总量的 20% ），并声称为公司利益放置在某安全钱包中，但他却涉嫌转移和出售价值约 1.3 亿美元的$GALA 代币（约占被盗资产的一半）。同时，Write Thurston 还窃取了 Gala 生态系统上运营“节点”的许可证出售他人，并且操纵节点以赚取 GALA 代币，使得自己或 True North 获益，他们已被被盗的购买者起诉。值得一提的是，今年年初，美国 SEC 曾起诉 Write Thurston 和 True North 涉嫌与节能加密代币企业“Green Boxes”相关的欺诈活动。

Eric Schiermeyer 还在诉讼里指出，虽然 Write Thurston 创立了许多公司，但大多数最终都陷入诉讼、破产或被美国 SEC 起诉，Gala Games 似乎是他唯一的合法企业。在公司运营上，除了指导公司的年度纳税申报表，Write Thurston 多年来并没有参与公司的日常运营，是一名缺席董事并于最近辞职。而过去几年时间里，公司也在一直在努力解决 Write Thurston 早期埋下的问题。由于之前发现 Write Thurston 被盗行为后，认为当时向当局联系或提起诉讼可能会导致他清算被盗的 GALA，因此公司只能被迫依赖他的陈述，即将其作为 GALA 的安全保管人，但从 Write Thurston 偷偷变卖资产并阻止公司追回其被盗资产来看，他除了回应出售部分资产来购买枪支弹药外，并无其他任何回应，所谓保护公司$GALA 资产安全的说法都是虚假的。

此外，诉讼书也透露了 Gala Games 节点集中化风险问题。Write Thurston(或其控制下的公司)和 Eric Schiermeyer 在 Gala Games 成立初期分别获得 7000 个 Gala Node。不过两位联创均同意不运营这些节点，因为这些节点的集中化会违背公司去中心化的基本目标。

Eric Schiermeyer 认为 Write Thurston 的欺诈行为对 Gala Games 声誉和品牌造成了重大损害，比如 Coinbase 在 SEC 对其诉讼后拒绝支持 GALA v2升级，他请求法院对 Write Thurston 和 True North 作出判决，并请求法院下令被告返还和赔偿被盗价值数百万$GALA 的任何收益，并解除其在公司董事职务及赔偿一切诉讼和法律支出费用。

代币较高点下跌超 97% ，年底前全面迁移至 GalaChain

Gala Games 的团队内讧风波也引发了社区恐慌。CoinGeoko 数据显示，截至 9 月 4 日，Gala Games 代币$GALA 价格较最高点下跌超 97.9% ，当前总市值约为 4.3 亿美元，远低于上述诉讼书中共提及的超 7.3 亿美元 GALA 资产，较诉讼时价格下跌超 16% 。

而根据 Gala Games 最新动态，该项目将于年底前将 Gala 平台全面迁移至 GalaChain，并简化使用门槛与开发工具，以便在年底前为外部团队提供全面的自助服务。此外，基于 GalaChain 构建的社交媒体平台 Rep 已经进入从 NEAR 迁移的 QA 阶段。另外，Gala Games 旗下游戏在也未受到诉讼事件影响，多个项目均取得积极进展，根据官方披露， 3 A 游戏 Champions Arena 首周下载量突破 7 万，Web2用户向Web3的转化率为 10% ；社交扑克游戏 PokerGO 已经推出 IOS 版本；链游 Town Star 即将推出路线图和代币经济模型等。

从以上两份起诉书的内容来看，滥用盗用公司资金和节点资源、管理公司不善、不正当商业行为等都充分暴露出 Gala Games 在治理上中心化、透明度等问题，也从侧面凸显了去中心化治理的必要性。“神仙打架，投资者遭殃”，尽管 Gala Games 这场内部之争尚未有定论，但投资者却需要为项目方的治理失误买单。
