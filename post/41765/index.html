我是在2018年开始投资加密游戏的，我相信一旦开放，游戏经济和虚拟世界就不会再关闭。在内容、创作工具、开放市场和技术的共同作用下，“主权个人 “假说似乎成为了不可阻挡的现实。我确实对这一切的发展速度感到惊讶：行业正在快速发展……但方向在哪？

因为骗子、投机者和行业的低标准等因素，我曾质疑过自己的信仰。最近，我又在质疑代币是否能维持基本价值，而不是被当作非法证券。现在，最重要的是，承认失败，从错误中学习，但不要被噪音和负面情绪冲昏头脑，以至于忘记了这一切是多么的有趣。

面向长期的、规模化的Web2消费者平台，在推动采用Web3方面的作用被低估了。它们通常被忽视，因为 crypto natives 一般会高估去中心化，认为去中心化是 “终点”。但是，就像隐私一样，我不觉得去中心化要被视为目标；相反，它应该被视为实现真正目的的一种 “手段”：自我表达、社区形成和金融机会。有经验的知识产权（IP）创造者在推动消费者关心的社区和IP发展上，可以发挥重要作用。那些敢于从长远角度出发的人，通过付出来获取价值，他们会取得发展，因为对于封闭的同行来说，他们的经济结构具有指数级潜力。

在评估Web3的投资机会时，有几个急迫的实际问题是：

1、将Web3加入到消费者关系中是否真的值得？

2、下一次采用Web3的浪潮是由现在的加密人员发起，还是原生信仰者发起？

3、在这场运动中，我们是早了两步，早了一步，还是晚了一步？

4、所有这一切是否只是摧毁零售业的另一种方式？

5、Web3旨在实现和增强能力，为什么它却感觉如此不人性化？

过去几个月，我在思考这些问题的过程中得出了一些看法：

一、 Web 3的开启就是创建数字合作组织：

①透明的治理结构；②奖励创作者/开发者的生态系统；③嵌入有效的手段，通过数字收藏品交换价值。

1）透明的治理结构。建立一套链上规则，使开发者无法单方面改变系统的规则。在参与感和促进开放世界活动方面，透明的治理都起着重要作用。

2）奖励创造者/开发者的生态系统。为贡献者创造收获成长果实的潜力。开放世界游戏开发者的角色，将从闭环设计者转变为开放生态系统的创始贡献者。

3）从数字收藏品中交换价值。NFT有着为游戏环境和社会结构创造 “地位即服务 “业务的可能。

二、在创建一个有回报的社区和一个只为回报而存在的 “社区 “之间存在良好的平衡。

1）温暖的加密货币

2）社区的思考

3）社会代币的意识流

三、可持续的经济设计。

1）代币经济可以归结为一个简单的原则：流入社区金库的代币比流出的多。

2）限制质押和其他供应方的机制（尤其是在通货膨胀的情况下），不如产生需求来得重要。

3）功能性代币(1 vs. 2 vs. N)：确切的模型不如有这两点重要：一是灵活的设计；二是通过参与而不是购买获得的代币。

4）有一种创造参与/保留挂钩的好办法，即将NFT作为投机的主要渠道与 “赚取的投入"代币分开，这样也可以保持灵活性，设计出可持续的经济模型。

四、在企业层面上，亚洲进行 Web 3的实验的意愿比西方更大。

1）对于那些在团队规模或国际影响力方面处于领先地位的早期Web3采用者（如Com2us）来说，以数据科学驱动的方式可以产生有用的学习成果（如代币发行、用户体验调整等）。

2）进行代币预售的公司，只是将代币与玩家体验进行切身结合（同时在法定微交易中保留大部分经济），虽然这是对Web3的初步尝试，但他们不太可能取得成功，尤其是那些高达数十亿估值的代币 在"免费 “使用现有的IP，却没有真正建立一个以创造者为中心的内容世界。

3）一些团队在建立 “模拟生活游戏”（例如战略游戏或模拟游戏）方面有丰富的经验，尤其是中国的工作室，例如趣加，它是一家专注于Web3游戏的公司，最近完成了巨额的代币销售。类似的游戏通过以货币衡量其社会地位，很好地完成了对付费者的转化和留存。这些团队有可能会建立伟大的链游，因为对 “身份即服务 “的需求正是当今加密货币游戏所缺少的。

4）因为对赌博和虚拟商品的法币交易的限制，韩国在Web3方面的创新速度很快。另外，因Terra带来的声誉上的损失，韩国企业需要在金融技术（用于支付的超级APP）、娱乐（边娱乐边赚）和游戏（可持续的创作者驱动生态系统和大型多人网游（MMO））方面实施可持续的Web3改革。

五、从长期来看，功能单一的加密货币融资不太可能具备有利的风险/回报比。我们认为，机会在于支持经验丰富的开发团队，以此建立可持续经济模型（不需要额外考虑代币认股权）。

1）在功能单一的代币上，我们能明显看到风险投资出现了结构性过剩，大量的资本投向没有P2E游戏制作经验的开发商。

a.很多项目在产品发售前就达到数十亿的FDV，例如，比Bethesda以75亿美元卖给微软，而发售产品前的Illuvium FDV估值比75亿美元还高50%。加密货币的投资决策更多的是由FOMO和短期交易决定的，而不是依靠基本因素（例如，12-24个月的锁仓就能够证明60%的折价是合理的吗？或者说，这个代币真的值100亿美元现货中的40亿吗？）。

2）许多风险投资项目中，我们还没有看到大规模的代币解锁，这一定会对代币现货价格产生实质性的负面影响。对于许多代币而言，买方的购买需求不足以吸收解锁的大量流动性。

3）只要项目的经济系统包括基于法币和基于加密货币两部分，那些筹集了大量代币却没有股权的团队，最终会在股权持有者和代币持有者之间，产生结构性冲突。我们认为，公司进行私下销售最佳的做法是，通过代币认股权证出售股权，以协调股权和代币持有者的利益。

六、监管是否合规将成为代币项目的主要区别，并推动越来越多的企业转向以NFT为重点的Web3尝试。

1）NFT交易将成为获取Web3价值的重要领域，它的使用费和销售收益将流向股权持有者（而非代币持有者）。

2）鉴于证券法对代币的关注，以及白皮书带来的结构不灵活性，我们看到市场越来越倾向于基于股权/NFT的模式，而不是基于代币的模式。

a.由于虚拟货币的发行者需要对代币转让进行KYC/AML认证，代币对规模化游戏开发者的吸引力将越来越小。在NFT作为数字藏品而非货币的背景下，这些担忧显得不那么重要（尽管不是没有疑问）。

b.我们还认为，代币只能赚取，而不能购买的模式将成为常态。盲目的代币销售造成大量供应中断，导致设计者无法创造平衡的经济模式。

3）证券化代币将成为热点，这次是以散户为中心。将基于证券的现金流嵌入到代币的使用中，如使用费+粉丝代币，特别是在音乐、体育、电影等方面，这将推动主流接受证券化代币。像Republic这样合规的发行人会做好准备迎接这样的未来。

a.由于散户在收集这些NFT时获得的效用超过了纯粹的金融价值，因此证券型代币更有可能通过良好的社会环境和与收集社区内的零售所有权相关的交易游戏来成功实现这一目标。 

b.关键是要平衡数字藏品的设计，使其功能不再 “单一”（基础金融价值），而是功能+（收藏价值形成足够大的参与溢价，使NFTs在使用价值方面不只是 “趋向面值”）。

七、绩效营销是Web3的一个痛点，公会有可能解决这个问题。

1）在引导用户获得Web3游戏体验方面，公会发挥着重要作用，同时公会也优化与加密货币相关的若干用户体验（例如，托管、法币的兑换）。

2）“单点登录 “是Web3的一个重要概念，对公会来说，它可以将链上分析与链下玩家数据结合起来（由游戏开发者集成的SDK记录）。我们看到大型出版商在创建全球 “超级公会 “，以此搭建可靠的平台，就像Facebook为移动端APP和游戏吸引了大量消费者的注意力那样。

八、声誉和 " soulbound tokens （即SBT）“构成Web3网络社会关系的 “承诺、证书和附属关系”，他们有可能带来加密货币的0到1创新，这对个人参与去中心化经济模型至关重要。

1）虽然大众对NFT基础设施进行了大量投资，但我们认为这个市场在很大程度上已经过于饱和，而且在交换、借贷、衍生品和聚合器之外，对哪些新的金融原语可以代表数字所有权的真正创新的关注是远远不足的，例如，基于信用评分的无抵押借贷。

“部落所有权 “将让更多人重新认识Web3艺术和数字藏品。今天，NFTs所缺少的是文化和历史背景。让传统的艺术收藏家进入Web3将引领下一波浪潮。

通过硬件钱包和Metamask访问是不现实的。一个多重签名的DAO邀请 “小团体 “参与策划，是该领域的信用背书。

随着传统的艺术收藏家和博物馆进入NFT，面对庞大的收藏市场，我们相信数字艺术是最有潜力上升空间的艺术类别。

具有综合社会背景的垂直市场，NFT的交易量将显著提升，从艺术延伸到时尚、摄影、音乐、电影和各种消费者渴望的类别。统一的主题是 “社区 “是数字藏品的创造者、策划者和商业引擎，以及集体神话，让这些藏品成为理想的表达形式。

当我们消费时，我们都在寻求自我表达：代表我们的价值观和身份的东西，无论是时尚单品、艺术，还是音乐。历史上，消费是单向的，是自上而下的从卖家到买家的方式。由于NFT的稀缺性，现在消费是循环的、动态的和可投资的。

Art Blocks是自上而下策划数字收藏品沙盒的领先案例。FXHash作为 “社区策划 “假设的延伸，同样有趣。

这个概念的延伸是 “Headless Brands”。目前还没有主要的UGC驱动的品牌，我们认为融合了物理、数字、社会社区的时尚品牌，是成熟的实验温床。

“传统品牌是由企业直接规划和设计的，但网络化媒体的崛起对集中管理的品牌身份的提出了挑战。新的基于区块链的去中心化组织对此更进一步，他们通过给予用户奖励来传播他们自己的品牌叙事。”

指导原则

为了将这些想法落实到指导投资决策的实际操作手册中，我认为有必要将我的想法提炼成几个指导原则，这也提炼出我的世界观。

信任使社区团结。

金钱伴随着不忠。

没有信任的社区不能存在。

社区货币可以存在。

只要它不是以金钱构成的。

而是成为它的一部分。

→ 提示：代币会随着时间的流逝而积累大量的网络价值，但预售和追求投机利润的做法违背了这一目标。代币只能赚不能买的模式，比起过度依赖预售和投机狂热的短期模式，更有可能养成正确的 “金钱观”，同时能净化社区形成的内在动力。

生产大于承诺。

参与而不是拥有。

丰富性大于稀缺性。

创造与合作而不是游戏或折磨。

推陈出新胜过华丽的设计。

P+P=P^2胜过PvP，也胜过PvE。

→ 提示：首先建立社区，没有预售。强调通过参与赢得代币，而不是靠囤积和其稀缺性。创造和支付者的转换比机械的 “paly to earn “更持久。经济系统的灵活性是最重要的。强调游戏经济中的社会联系和用户生成的内容，可以将线性联系转化为指数经济。PvP比PvE更具可持续性（因为购买方对基于技能的游戏，和身份驱动的模拟游戏有持续的需求），但不如繁荣的UGC世界。

虚拟投入（货币、资源）的灵活性高于刚性。

对虚拟产出（地位商品）来说，刚性大于灵活性。

→ 提示：将设计者需要的灵活处理的游戏平衡工具（代币、资源、虚拟货币）与旨在累积生态系统价值的工具（NFT、NFT的金融化篮子）分开。交易费用应该回流到社区，用于长期的生态系统发展。

延迟归属而不是立即变现。

以使用费为基础的调整，而不是以预售变现。

→ 提示：实施长期的价值调整机制，如基于使用费的现金流，而不是榨干式的预售。赚取的价值应该受到延迟发放的限制，以避免吸血鬼式的榨干流动性和销售抛压。

摩擦最小化，而不是最大限度的去中心化。

委托的灵活性，而不是结构的不可改变性。

第一方排第一，UGC排第二，第三方排第三位。

中心化内容 => 民主化的修补 => 第三方生态系统。

→ 提示：在项目的早期阶段，创造经济机会比最大限度提升去中心化原则更重要。可使用性和玩家采用是至关重要的，应该信任IP创造者，以此来引导内容生态系统，为UGC提供更多机会。随着时间的推移，IP创造者可以逐步寻求将他们所创造的生态系统变得更加去中心化，可以邀请创造者和开发者来扩展IP宇宙。因为内容创造的难度，从第一天起就完全去中心化的系统，不太可能蓬勃发展。

MapleStory

最令我兴奋的项目之一是Nexon的MapleStory宇宙延伸到Web3。MapleStory有可能被视为利用开放经济和IP的 “示范标准”。Nexon团队已经研究区块链游戏有一段时间了，最初他们看到的是一个充满投机和短期暴利的市场，对此他们感到很沮丧。然而，正如Nexon韩国首席运营官Kang Daehyun所透露的那样，“我们好奇，仅仅通过在早期阶段的肤浅和不稳定的过程，就拒绝区块链技术的所有可能性，这是否是一个正确的决定。”

经过几个月的努力，Nexon团队得出结论，透明度、开放经济和价值获取的核心驱动力是强大到无法忽视的原语。如果团队要保持正确，他们就需要把一切都押上：“为了传达我们对这一愿景的诚意（而不是说我们只是因为区块链现在很热，而跟风进入），我们必须使用一个我们无法承受损失的IP来挑战自己。这能显示我们的真诚并赢得信任”。因此，Nexon决定以其传奇的MapleStory特许经营权（首次创建于2003年），为Web3重新创造IP。

到目前为止，已经确认了四项申请。

MapleStory N，基于MapleStory的RPG游戏。

MOD N，MapleStory沙盒制作平台。

N Mobile，移动RPG；以及

N SDK，生产工具，用户使用NFT创建多个APP。

MapleStory N和N Mobile：利用MapleStory宇宙的IP，通过游戏获得物品和代币，并转化为NFT。随着时间的推移，稀缺性会越来越高。有了所有权，用户就能创造自由市场经济。用户可以在NFT所有权的基础上，创造自己的自由市场经济。

经济活动产生的费用分配给生态系统的创造者，包括Nexon。

基于 “完美分享 “的原则，Nexon期望通过每个人的贡献和努力，实现更大的增长，这可以扩大整个生态系统，并发展到远远超过其现有价值。

MOD N：一个沙盒游戏创作平台，使用MapleStory NFT和外部的NFT。所有在MOD N中创建的游戏都属于创建者。包含了20多年来创建的3000万个资产。创作者根据其游戏的受欢迎程度获得奖励。

Nexon有能力为开发提供资金，而且不进行预售，同时将经济系统100%以原生代币计价（应用内没有法币购买），使它可以建立一个变革性的开放经济，而没有任何负面影响。例如，10000个 PFP降低数字稀缺性，或降低虚拟土地销售导致的非生产性地块。（这句话可能有问题，请仔细看看）首先通过建立参与感，然后通过市场机制释放经济价值，我相信Nexon将向市场展示Web3的艺术可能性。

总之，我所寻找的是数字原生的 “lollapalooza效应”，内容、市场和技术的交点中产生机会，其中，多个交叉趋势的乘法效应与基本驱动因素一样重要。正如查理芒格指出的那样：

“当几种模式结合在一起时，你会得到lollapalooza效应；这是指两种、三种或四种力量都朝着同一方向运作。而且，你得到的不是简单的加法。这往往就像物理学中的临界值，如果质量达到一定的点，就会得到核爆炸。如果没有达到这个点，就什么也得不到。有时这些力量只是简单的普通叠加，有时它们在爆发点或临界质量的基础上结合起来……真正的大效应，也就是lollapalooza效应，往往只是来自于因素的重要组合（这些因素相互加强并大大放大）。”
