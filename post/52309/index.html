摘要

本篇报告主要剖析了 2023 上半年 NFT 交易赛道的发展现状与之后的趋势发展得到以下结论：

• NFT 市场市值波动大，但侧面证明了 NFT 市场仍然有发展前景。

• 仅 Q1 季度 NFT 市场的贷款总额庞大，ParaSpace 作为备受关注的龙头项目累计贷款量超2.8 亿美元。

• NFT 市场流动性是当下首要解决的问题，NFTFi 的积极发展大大推动 NFT 应用场景。

• 数字藏品与品牌 IP 结合的 NFT 应用是第一个打破传统界限并被广大人民所熟知的 NFT 应用。

1.  2023 年发生了什么

2021 年初 NFT 项目开始在市场上崭露头角，吸引到大量的关注。尽管今年 2 月 Ordinals 协议的出现再次掀起 NFT 热潮，但四月底开始市场已经显示出疲软的迹象。

1.1 NFT 大盘回顾：跌宕起伏

2023 年开年以来，币圈的行情快速复苏后一直跌宕起伏，2023 年 1 月份整个 NFT 市场的交易总量达到了 9.41 亿美元，受 BTC Ordinals 影响 NFT市场 3 月总交易量增长到了 15 亿，但是现在 5 月目前交易量才 3.3 亿美金。

目前 NFT 的市值维持在 420 万美元，由此可见今年 NFT 市场仍然受到了大量资金的关注。

1.2 2023 NFT 重点事件

• 在第一季度，NFTFi 的借贷市场开始逐渐复苏

仅今年 1 月至3 月 NFT 市场的贷款总额达到约2500 万美元。BLUR 于 5 月 2 日上线以 NFT 作为抵押品的点对点永续借贷协议 Blend，正式进入 NFT 借贷赛道，现已收获3000 万美元存款量。其中，ParaSpace在借贷市场上拥有最大的市场份额，前三大借贷项目占整个借贷市场 65% 以上，借贷量约3500 万美元，总存款量1.2 亿美元。 NFT 借贷市场的发展潜力巨大，吸引了越来越多的用户参与其中。

• BTC Ordinals 爆发

Bitcoin Ordinals NFT 是比特币的 NFT 版本，它以 satoshi 为单位（BTC 的最小单位）刻在比特币区块链中。但由于Ordinals NFT 基于 BTC 网络，大家需要承受较慢的交易速度、高额的成本和有限的应用范围。尽管在比特币上创建 NFT 要比在以太坊上复杂，但可以利用诸如闪电网络和 RSK 等第二层协议来实现。

Ordinals 协议推出后，大量的 NFT 项目在链上进行铸造，根据 Dune 上的数据显示，Ordinals 在 3 月以来增长迅猛，掀起了一股 BTC NFT 的热潮，不少 NFT 投资者和社群成员都积极投身其中。目前，BTC Ordinals 市场的交易量和数量呈上升趋势。四个月内铸造的 NFT 总数达到 119 万个，交易量也达到了 2000 万美元。

• BLUR 发布空投

目前，BLUR 成为了链上交易量最大的 NFT 聚合交易平台，因其零手续费、代币空投等活动备受市场追捧，现已超越了此前最大的 NFT 交易平台 OpenSea。

NFT 交易量在 2023 年第一季度急剧上升，但随后下降，BLUR 的激励和空投机制是主要推动力。BLUR 发布了自己平台的代币后，各大交易所第一时间上线，将这一波NFT热潮再次推上了一个新的高度。

BLUR 的版税份额在2 月中旬超过了 OpenSea，BLUR 的可选版税和零手续费策略吸引了众多价格敏感型用户，推动了 NFT 交易量，2 月22 日最高交易量超 80 k ETH，三月后交易量逐渐下降，但是 BLUR 的增速依然没有放缓，反而扩大了其领先优势，拥有 NFT 市场上总交易量的 66% 。

• OpenSea Pro 更名 Gem.xyz ，引入“Gemesis”NFT

OpenSea Pro（原Gem V2）已经更名为 Gem.xyz，并引入了“Gemesis”NFT。在 2023 年 3 月 31 日之前曾经使用过Gem.xyz 的用户可以获得免费的“Gemesis”空投。用户可以在 5 月 4 日之前免费领取该 NFT，而“Gemesis”的底价通常保持在 0.03 ETH 到 0.05 ETH 之间。Gemesis NFT 在发行中迅速增长了交易量，但同时也出现了持有时间和购买量的明显分化，其中约 50% 的用户购买后选择继续持有资产，且资产价值相对较高。而另外 50% 的资产则在购买后不到 24 小时内就转手出售。

2.NFT市场概况及分析

2.1 NFT 市场交易现状

• NFT 市场从二月 Ordinals NFT 的火热开始受到市场关注，加上 memecoin 的刺激，市场交易热情高涨吸引资金流入，NFT 市场迅速反弹，持续至 4 月底。当前形势为 NFT 交易者创造了一个有趣且有潜力的市场。

尽管在比特币上创建 NFT 要比在以太坊上复杂，但 Ordinals 尚处于发展初期。目前Ordinals总价值达到了几千万美元，并且已经受到市场关注。比特币的市值是以太坊的 2.3 倍，如果 Ordinals 想要赶上以太坊 NFT 的市值，其增长空间还有1000 倍，所以长远来看 Ordinals NFT 还处于早期，待其生态和基础建设完善后必定还有巨大的空间。

• 借贷作为金融工具有极大的发展空间，其核心就是将 NFT 作为抵押品获取贷款，借贷市场的出现也增加了 NFT 应用场景。 当前累计 NFTFi 借贷总额 $ 430, 979, 851 ，累计借贷人数 43, 521 人。

借贷作为金融常用的手段之一在区块链市场中的运用场景也非常之多，加上NFT市场的流动性缺乏，许多用户手中的 NFT 资产并不能立即变现，这时NFT 借贷平台的出现可以让用户将自己持有的NFT放入平台作为抵押品，然后借出对应的数字货币或法币。此外，借贷平台还会根据抵押品的价值提供合适的借贷比例，随着 NFT 市场发展，借贷市场还有持续上涨的空间。

• 最近三个月中 Wrapped Cryptopunks 成交量第一占整个市场的14.50% ，MAYC 和 BAYC 分别以7.81% 、7.77% 位列二三。其中 Wrapped Cryptopunks 贡献了超 3 M ETH 的交易额，表现最为优秀。而 YugaLabs 生态中的 NFT 项目占整个市场四分之一份额。

2.2 NFT 流动性及特性

流动性问题不仅对 DeFi 的发展是致命的，对 NFT 也是如此。NFT 交易流动性不足主要源于用户数量有限、定价困难以及 NFT 价格过高等问题。 NFT 流动性与市场热度也密切相关，所以我们可以看到两种很极端的情景，排名 TOP 10 的 NFT 项目一直保持一定水平的流动性且项目之间的波动并不大，而另一部分 NFT 要以打折的方式方便出售。

NFT 市场属于新兴市场，缺乏历史数据和公认的估值分析方法，从而造成定价困难。人们对于稀有程度和主观观点不同，即使是同一系列的 NFT，价格也可能会相差甚远，导致其流动性较低和资金效率不高。加上 NFT 市场的价格通常过高，导致了很多投资者失去了兴趣。蓝筹 NFT 的进入门槛极高，平均价格区间在 11 K-120 K 美元之间，还有部分 NFT 投资者不愿意出售他们的 NFT 来换取即时的流动性，导致 NFT 流动性问题的难以解决。

从 NFT 访问量国家分布我们可以看到一个有趣的现象， 亚洲人群对 NFT 的热情普遍高于欧美人群。

NFT 中最主要和最流行的当属“PFP（Profile Picture）”项目，其总市值高达 13 亿美元，交易额高达 18 亿美元，但在流动性方面，Utility NFT 的流动性最佳。

NFT 玩家主要分布为青年男性占比超过 70% 。

3.NFT 项目发展及未来趋势

优质的 NFT 项目往往拥有强大的社区共识、团队背景、鲜明艺术风格，机构对其十分青睐。而对于普通用户来说参与蓝筹 NFT 或热门 NFT 的门槛越来越高，如今也出现了一些解决方式：NFT 碎片化、NFT 质押、NFT 流动性平台的兴起等。

3.1  NFT 交易平台

NFT 交易平台是为购买或销售 NFT 提供流量入口，目前，使用 NFT 交易平台交易的人数占比最大，BitKeep、Opensea、Looksrare、Binance有自己专门的NFT交易板块。

•  BitKeep NFT Marketplace

BitKeep NFT 市场是全网首个支持任意币购买 NFT 的市场，目前 BitKeep NFT 市场已支持 ERC 721 、ERC 1155 两大NFT 协议，Ethereum、BNB Chain、Polygon、Arbitrum、KLAY、Optimism、HECO 等 9 条公链的 NFT 生态，市场内的 NFT 项目多达 1, 000, 000+。

由于BTC参与的门槛较高，参与者们只能依靠 Google 文档来买卖 Ordinal punks，侧面展示了这个行业有多早期以及 Ordinals 多么缺乏必要的基础设施。BitKeep钱包作为 NFT 市场的基础设施，目前已上线 Ordinals 行情看板，并且即将接入比特币生态，推出比特币 NFT 交易专区，支持 Ordinals 协议 BTC NFT 的展示、铸造、转账、交易等功能。

BitKeep 钱包同时将兼容 Taproot 地址格式，为 BRC-20 代币提供资产展示、转账和交易等服务。接入完成后，BitKeep 用户可以在移动端和插件端使用 BTC Taproot 地址格式，并对 BRC-20 代币及比特币 NFT 进行充提管理、转账交易等操作。

3.2 NFT 交易聚合器

NFT 市场聚合器可能是目前 NFT 流动性解决方案中最引人注目的。通过允许用户批量购买可以为用户节省高达 40% 的 gas费用。目前排名前三的聚合器分别是Gem(已被OpenSea收购)、Genie(已被Uniswap Labs收购)和Flip，都提供了 NFT 跨平台的聚合交易，其中的“批量扫地板”功能十分有用。

•  Gem

 Gem V2 现在是 OpenSea Pro！Gem 拥有170 多个市场的最强大的NFT市场聚合器，为用户提供最优价，现已与LooksRare 的流动性整合。为大家提供更好的购买、收藏、特征筛选、排行榜列表等功能，0 费用。整合了大多数 NFT 交易市场的 NFT 上架，并为 NFT 投资者提供了丰富全面的数据和工具，Gem 的新稀有度排名和收藏分析对用户来说特别有用。

因为随着NFT变得越来越受欢迎，市场也在不断增长。NFT交易聚合器可以为大家提供更广泛的NFT市场选择，可以帮助用户获得更好的交易价格，因为它们可以在多个市场上比较价格并为用户提供最佳交易，其发展前景十分广阔。

•  Genie

Genie 主打 NFT 批量出售和购买功能，是第一个 NFT 市场聚合器，主要由 Genie Swap 和 Genie List 组成。

Genie Swap 支持的交易市场平台包括 OpenSea、NFTX、NFT 20 、Rarible 以及 Larva Labs 的 CryptoPunks，并且不收取任何平台费用。

Genie List 上线 支持用户在单次交易中从多个市场购买多个 NFT（最多 60 个NFT），并通过聚合交易降低交易成本。

3.3 NFT 项目

优质的 NFT 项目通常团队背景强大，容易获得市场的关注，比如各路明星加持的 BYAC 热度高、国产杰伦熊自带流量甚至能吸引 web2 的新资金流入，粉丝流量和名人效益在 NFT 圈子来说是非常重要的，任何一个项目的发展离不开人气的累计，而明星所带来的市场高度满足这一条件。

• Bored Ape Yacht Club

Bored Ape Yacht Club （BAYC）是典型的 PFP NFT，由10, 000 个“Bored Apes” 组成的NFT系列，于2021 年4 月30 日推出，它是著名的 Yuga Labs 上线的第一个项目。

BAYC 一直在积极的拓展其生态应用，发行其生态代币 ApeCoin（APE）打造元宇宙土地 otherside，还出售 IP 商业使用权增加版税收入为 NFT 赋能。

• Azuki

Azuki 是一个日漫风格的 PFP 型 NFT，是 2022 年最热门的 NFT 项目之一。Azuki 受到市场关注主要源于其3 种发行方式：荷兰式拍卖、白名单 mint、公开销售。Azuki 总销售额甚至一度超过了CryptoPunks 和 BAYC。

Azuki 早就开始采用碎片化 NFT，将 IP 之一#Bobu 碎片化（生成 $Bobu 代币），碎片持有者组成一个 DAO，决定 Bobu 未来的发展。Bobu 碎片化成功降低了新用户进入 Azuki 生态的门槛，也回馈了原 Holder。所以我们可以看到，尽管 Q1 NFT 市场价格波动较大，但 Azuki 在底价和市值方面成功地克服了看跌趋势。蓝筹项目普跌高达 50% ，而Azuki 却逆势而上，巨鲸持有数上涨筹码集中。

• ParaSpace

在 NFT 市场流动性缺乏的当下，需要进一步提升 NFT 应用场景的需求，通过贷出 NFT 换取现金的流动性从而达到资本利用最大化。此外，用户选择抵押 NFT 而不是出售 NFT，也有助于缓解供给过剩的问题。

ParaSpace 是 Q1 借贷市场中市场份额最大的平台，累计市场份额为 1.34 亿美元。定位为Universal NFT Liquidity Platform的ParaSpace 提供包含Lending、Staking、Trading的全方位产品。 ParaSpace 的抵押品涵盖到了多种生息代币目前出借最多的代表：WETH、cAPE、USDT、USDC和stETH。同时，作为首个支持UniSwap v3作为LP的借贷平台，ParaSpace也是首个支持Universal NFT Lending的平台。相较于其他 NFT 借贷，ParaSpace的产品更受到 DeFi 老玩家的喜爱，产品中处处都是方便用户使用的小细节。

• Fractional_art

Fractional 是一种将 NFT 进行碎片化的去中心化协议，解决 NFT 市场流动性方案之一，将一个给定的 NFT 拆分成多个可以在市场上单独交易的部分。

Fractional 是 NFT 碎片化的龙头项目，一旦用户拥有 NFT 分片代币，就有权对 NFT 底价进行投票。NFT 碎片化有利于激活 NFT 更多玩法，也降低了大家参与 NFT 市场的门槛。对于单价很高的蓝筹 NFT 项目来说，大大降低了用户的参与门槛，也可以从而提升资产流动性。随着技术发展 NFT 碎片化也会变得更加灵活更加智能化，如果被众多蓝筹 NFT 项目中充分利用起来，其应用市场特别大。

4.  NFT 下半年机遇与发展

NFT 市场还属于新兴市场处于早期，2021 年 NFT 市场的爆火主要是受到加密货币市场和明星名人代言推动，人们蜂拥进入市场。由于了解 NFT 的人还是相对较少，导致了2022 年出现长期供过于求的局面。2023 年Q1 NFT 市场的虽然波动起伏也证明了 NFT 市场仍然有活力，我们看到 Ordinals 协议引发的热潮带来了不错的数据，但是由于没有新的叙事、热点持续出现无法保持市场热度。

NFT正逐渐与艺术品、体育、娱乐以及社交等传统领域结合，从而进一步拓展其生态系统，增强其影响力吸引新的血液进入 NFT 领域。万物皆可 NFT ，现在 NFT 已经成为一切物品的潜在标的，包括元宇宙世界、收藏品、游戏资产等，这些资产都可以通过 NFT 实现经济确权，随着NFT平台和产品的迅速发展，其生态系统也在蓬勃成长，并形成了相对完整的产业链。

NFT 市场流动性问题一直被诟病，发展 NFTFi 引入衍生品去提高资金流动性，为将来NFT市场发展提供更多可组合性。为了改善NFT的流动性、拓宽应用场景以及应对门槛高的现状，NFTFi 应运而生，通过 NFT交易平台、NFT 聚合器、借贷、衍生品、碎片化等解决方案的出现和不断完善。

NFT 做为数字产品其产生都是在区块链上执行的一段标准化软件代码，高度依赖于底层区块链协议的属性。NFT 的安全保障问题也不容小觑。大部分 NFT 是在以太坊网络上铸造的，以太坊 Layer 2 发展积极旨在解决以太坊网络拓展性的问题，目前的区块链技术处于高速发展时期但其安全隐私方面并不完善，不乏有黑客份子进行盗窃、网络攻击或仿盘等诈骗行为。这些问题都提醒我们要提高安全意识，防止诈骗份子有机可乘。

结语：

NFTFi 生态正在井喷式拓展，多个细分赛道的 NFT 项目方逐渐将想法落地，共同利用衍生 NFT 产品来扩大用户群体提高市场流动性。 当下 NFT 市场发展快速，每个月都会出现新的动向，虽然5 月份的交易量有所下降，但这样的波动也证明了 NFT 市场仍然很有活力和前景，相信万物皆可 NFT 的时代也不会远了。
