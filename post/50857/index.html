自 Ordinals 协议将 BTC NFT 带火之后，BRC-20 又成为了用户、CEX 争抢布局的新赛道。这些新协议的出现让一向「古板」的比特币链又重新焕发生机。除了 mint NFT、发 Token、还有人开始创建域名、发布 BTC DeFi 等等。人们不禁期待，比特币生态 Summer 是不是要来了？

虽然相比于以太坊及其他生态，比特币生态还处于早期阶段，但自 2 月以来，比特币上的基础设施和项目也越发完善，早期上线的项目也经过多次迭代，功能逐渐多样。接下来，Odaily星球日报就带大家一起概览下目前比特币生态系统的基础设施和项目都有哪些。

钱包&市场

Ordinals Wallet

官网：ordinalswallet.com

Ordinals Wallet 是一种非托管 BTC 钱包，允许用户查看和存储 BTC 和 Ordinals NFT。Ordinals Wallet 助记词将在本地生成并存储在计算机上，且在浏览器中加密，安全性较高。目前，用户可以直接从钱包中创建转移 Ordinals NFT，并可以在 Ordinals Wallet 市场上列出 BRC-20 代币。

Hiro Wallet

官网：wallet.hiro.so

Hiro Wallet 是一家基于 Stacks 区块链开发的钱包，已经支持比特币、NFT 和 Ordinals。浏览器扩展用户可以使用相同的密钥或硬件钱包设备从他们的 Hiro 钱包发送比特币，以跨帐户管理他们的 STX 和其他资产（例如 BNS 域名和 NFT），支持所有能发送到比特币地址的资产类型。

目前，Hiro Wallet 支持 Ordinals NFT、Bitcoin Stamps，用户还可以在钱包中查看他们的 BRC-20 代币余额。

Alby Wallet

官网：getalby.com

Alby 浏览器扩展钱包可以为网站提供深度的闪电网络集成，以用于支付和身份验证流程。

该钱包的亮点是集成了 Nostr 社交协议，可以帮助用户方便地管理自己的私钥并通过网络与其他用户交互。目前，Alby Wallet 新增功能 Nostr Wallet Connect (NWC) ，它基于 Nostr 协议并利用比特币的闪电网络，允许用户使用他们首选的闪电网络钱包直接在应用程序中发送和接收付款。  

Alby 是开源的，处于 alpha 阶段。

UniSat Wallet

官网：unisat.io

UniSat Wallet 是用于比特币序号和 BRC-20 的开源 Chrome 扩展。其推出的 UniSat Marketplace 是比特币链上基于 PSBT、支持 BRC-20 资产的 Inscriptions 市场，已经成为当前主流的交易市场。目前持有 UniSat OG Pass 一定时间的用户可以先参与该市场测试，UniSat Points 数量最多的用户将在后续获得邀请。

Generative

官网：https://generative.xyz/metamask

Generative 钱包基于 Metamask 创建，用户只需使用 MetaMask 登录即可。在后台，Generative 会创建一个序号地址，该地址源自于你的以太坊地址。Generative 会为你提供密钥库、安全登录、序号地址和市场，来管理比特币 NFT。

Xverse

官网：xverse.app

Xverse 钱包是适用于 Android 和 iOS 的 Stacks 和比特币的非托管加密移动钱包。它的主要目标是成为所有由 Stacks 提供支持的比特币 Web3 应用程序的入口。目前，Xverse 可以让用户创建、发行、购买和出售各种类型的 NFT，包括艺术品、音乐、游戏、域名等 。Xverse 支持使用 STX 代币或者其他加密货币（如 BTC、ETH 等）进行交易，也支持使用 Hiro Wallet 或者其他钱包进行连接 。

Xverse 钱包由 Secret Key Labs 开发，Secret Key Labs 由原 Stacks 团队的工程师创立，致力于为 DeFi、NFT 收藏品等在内的去中心化应用程序提供支持。

目前，Xverse 钱包也提供查看 BRC-20 代币余额功能。即将推出转移、交易等功能。

Sparrow Wallet

官网：sparrowwallet.com

Sparrow Wallet 是一个基于比特币的开源钱包，支持多种交易类型和硬件设备。Sparrow Wallet 也支持 Ordinals 协议，用户可以用 Sparrow Wallet 来接收、管理和转移 BTC NFT；也可以用 Sparrow Wallet 来创造 NFT，只要有一个 Taproot（P2TR）账户，并且在一些支持 ordinals 的网站上进行操作即可。

Gamma.io

官网：gamma.io

Gamma.io 已推出无需代码的比特币 NFT 创建工具。该无代码工具允许用户以约 5 美元的固定费用（或 21, 000 sats）以及用于确认其 BTC NFT 的额外 10% 的网络费用，来创建最大为 60 kb 的图像。用户可以根据网络拥塞情况选择或自定义交易费用。较高的交易费用将帮助更快地创建 NFT。

由于 Ordinal NFT 完全是比特币原生的，不涉及额外的区块链或代币，因此人们无需连接钱包即可访问 Gamma 上的 Ordinals 功能。值得一提的是，Gamma 将关于该工具发布的公告也已作为 NFT 被记录在了比特币区块链上。

Ordswap

官网：ordswap.io

Ordswap 是一个 Ordinals 市场和钱包，可以在该市场上交易和上架比特币 NFT、集成多钱包以及具有收藏检索功能等。

Ordinals Market

官网：http://Ordinals.Market

Ordinals Market 是一个开放和去中心化的平台，任何人都可以在上面发布自己的作品或收藏别人的作品。Ordinals Market 也支持第三方应用程序和服务接入，提供更多的功能和体验。Ordinals Market 让用户可以用 ETH 来购买或出售 BTC NFT，也可以浏览和收藏各种有趣和独特的作品。

Ordinals Market 通过 Emblem Vault （一种可以让 NFT 在不同链之间轻松交易的协议）链接到经过验证的 Opensea 列表，以为用户提供所有合法的比特币 NFT。

OpenOrdex

官网：openordex.org

OpenOrdex 是一个基于部分签名的比特币交易（PSBT）的开源零费用无信任比特币 NFT 市场，由社区开发者 Oren Yomtov 创建。具体操作方式是卖方通过创建一个部分签名的比特币交易（PSBT），指定序号的价格。然后买家可以签署 PSBT 并将其广播到网络。据悉，Openordex 的中心化订单簿基于 Web3 通讯协议 Nostr 开发。

Scarce.City

官网：scarce.city

Scarce City 推出了比特币市场 SatsCrap，SatsCrap 使用比特币进行支付和抵押以促进交易，同时维护用户隐私。任何拥有互联网连接设备和比特币的人都可以出售任意合法物品并以无 KYC 的方式接收付款。不涉及注册、登录或银行账户。

Luminex

官网：luminex.io

Luminex 是一个比特币 NFT 交易市场，同时提供以太坊 NFT 收藏。

MagicEden

官网：magiceden.io/ordinals

Magic Eden 于 3 月推出比特币生态 NFT 市场，已集成对比特币钱包 Hiro 和 Xverse 等的支持，允许交易者上架、购买和出售 Ordinals NFT。此外，Magic Eden 还上线比特币 Launchpad，旨在帮助创作者直接在 Magic Eden Launchpad 上启动 Ordinals 协议制作比特币 NFT。

ORDX

官网：ordx.io

ORDX 是一个比特币 NFT 交易市场，提供即将发布的 NGFTP 列表，由于 BRC-20 的火爆，ORDX 也发布了 0 RDX BRC-20 代币。

工具

OrdinalsBot

官网：ordinalsbot.com

Ordinals bot 是最早的铭文服务之一，用户可以铭刻文件、文本和 .sats 域名。在 Ordinals bot 铸造只需几秒就能完成。

OrdinalHub

官网：ordinalhub.com

OrdinalHub 是发现、跟踪和分析比特币 NFT 的一站式平台。 OrdinalHub 提供的数据丰富，可以帮助用户在交易中获得优势，并快速寻找新的趋势。

BitCheck

官网：bitcheck.me

BitCheck 可以让你的社交帐户与你的比特币地址连接。通过 BitCheck 配置文件 ，你可以展示你所有的比特币地址，包括 NFT 和 BRC-20 代币。BIT 是 BitCheck 的代币，可用于支付和访问平台上的各种功能。

Trustless Computer

官网：https://trustless.computer/

Trustless Computer 是一种第 1 层协议，可将比特币的实用性扩展到加密货币之外。允许你创建 DAO、DEX、NFT、令牌、拍卖借贷、数据存储等等。

DeFi

btswap

官网：btswap.io

btswap 是第一个在比特币区块链 (BRC 20) 上运行的去中心化交易所。它是一个为 BRC 20 代币带来链上交换的元协议，它由 Ordinals 流动资金池提供支持。每次交换都将作为序号记录在比特币区块链上，并将永远存储在网络上。Beta 测试将很快推出。

ALEX

官网：alexlab.co

ALEX（Automated Liquidity Exchange）是第一个通过 Stacks 智能合约为比特币构建 DeFi 的平台。

ALEX 提供了一套 DeFi 机会，包括：

  • 通过启动板发现并参与 IDO。

  • 具有深度流动性的 AMM DEX。

  • 提供流动性、ALEX 代币质押和收益农业。

域名

BTCDomain

官网：btcdomains.io

BTCDomain 基于比特币构建，能够提供类似于ENS的用户体验。且 BTCDomain 现已可被完整解析。用户可以在平台上自由使用域名收发 NFT 和 BTC。除此以外，用户在官网注册或者在二级市场购买「.btc」域名后，域名不止可被解析到你的比特币地址，持有者还可以建立一个数据存储和解析完全依赖于 BTC 主网的主页，极大丰富了比特币域名的使用场景。

BTCDomain 还采用了 ZK 技术，该技术使比特币成为一个可用的数据链，创建了一个类似于智能合约的系统，完全是比特币原生的。借助 ZK 技术，用户可以通过数学证明确认系统的安全性。

names.sats

官网：docs.sats.id

所有 name.state 都基于以比特币，注册和更新名称只需要且只有比特币交易才可。比特币节点包含实现所有名称状态所需的所有信息，不需要其他数据库或受信任的服务。该项目不设 mintpass、白名单等机制，任何人都可以创建。

DAO

OrdinallDAO

官网：http://discord.gg/sJU 334 rHY 3 

OrdinallDAO 是一个面向 BTC NFT 的 DAO，其使命是加速 Ordinals NFT 的 OG 生态系统，通过利用 DAO 的能力来减少费用和风险。投资者可以通过与其一起投资来分散并减少投资失败的风险。目前，他们利用 Snapshot（用于投票）、Gnosis Safe（安全措施）和 BTC 多重签名等应用程序开展筹款活动，并在短短三天内筹集了 153515 美元。
