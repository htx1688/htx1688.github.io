1. 音乐 NFT 概述

1.1 行业概览

国内音乐行业市场在去年随着疫情过去和短视频平台兴起有所回暖，由中国传媒大学撰写的《2021 年中国音乐人报告》显示，2021 年音乐人的收入在较低收入和中等收入的占比有所降低，在高收入的占比有所提升。2021 年音乐人月收入在 5000 元以下占比 48.51%，有 28.65% 的音乐人月收入在 1 万元以上，其中月收入 5 万元以上的音乐人占比 3.41%。

报告还显示，2021 年，有 64.75% 的音乐人上传过音乐短视频，相比于 2020 年的 54.08%，同比增长 10%。其中，使用过抖音平台来传播音乐的音乐人超过半数，占比 55.21%。短视频平台的传播效果，尤其是抖音平台，得到了音乐人的肯定。在推广效果最好的数字音乐平台选择中，选择抖音的音乐人占比 33.37%，仅次于腾讯音乐，位列第二。

在流媒体播放占主要地位的海外音乐市场，流媒体的利润率对 99% 的艺术家来说很低，低到足以令人望而却步。

「…… 每月持续实现约 100 万个英国流媒体可能是某种指导，是以音乐为生的最低门槛，至少在英国流媒体得到非英国流媒体和其他收入来源补充的情况下……2020 年 10 月实现 100 万个英国流媒体或以上的艺术家数量为 1723 人…… 或 0.41% 的所有艺术家［在我们的数据库中］。」

引自《数字时代的音乐创作者的收入》，英国知识产权局（IPO）的一份 224 页的报告

经历过 Defi Summer，profile picture (PFP) 头像 NFT 赛道犹如平地惊雷式的骤增，音乐 NFT 成为了人们下一个目光汇聚的焦点。但是，生成音乐 NFT 系列所带来的社会价值和财务价值，还未达到与它们的视觉同行一样的持续成功的水平。音乐类 NFT 2021 年市值约为 12.43 亿美元，与之相比头像类 NFT 市值约为 104 亿美元。

1.2 技术背景

音乐 NFT 的交易和流通绝大多数在 OpenSea，和平台方基于 ZORA 的自有的交易市场上。ZORA 是一个 NFT 市场协议，它不会宕机，拥有可组合性、不可变更性、以及可访问性。ZORA V3 包括一些新的机制，激励建立在该协议上的平台，以及一个突破性的模块化架构设计，允许一个无权限的系统。

ZORA 内置了一种推广者激励机制，允许推广者为 NFT 寻找最终的买家。在发行一个 NFT 项目时，设置推广者激励费用的份额，用户要赚取推广者激励，只需在用户界面中提交钱包地址，平台会生成一个链接。如果该推广者找到了买家，就能赚取这一奖励。这是一个强大的激励机制，让开发者和策展人整合 ZORA 协议，直接获取他们所创造的价值。

2. 产品介绍

2.1 Euterpe

项目简介

Euterpe 是全球首个以 IP 驱动、内嵌 SocialFi 生态的版权 NFT 平台。团队致力于打造一个高质量 IP NFT 的专业 B2C 市场，免于侵权和诈骗。Euterpe 还将内嵌 SocialFi 生态系统，将被动的粉丝变成主动的利益相关者，通过他们对社区的贡献努力获得激励他们对社区的贡献。在元宇宙中，IP 就是一切。Euterpe 从音乐 IP 入手，然后扩张到包括视频、游戏、图书、音乐会、明星周边等在内的广阔 IP 领域。

产品

Euterpe 在 2021 年一季度进行了 Alpha 版测试，已与三十余家唱片公司达成合作，合作歌曲上千首。截止发稿，Euterpe 还未正式上线，根据官方披露，对于创作者和版权人而言，Euterpe 平台为他们的作品版权铸造和发行 NFT，并通过斯坦福获得诺贝尔奖的 NFT 拍卖方式大幅提升版权价值，创造即时变现的渠道，并通过 SocialFi 扩大作品的影响力；对于投资者而言，可以通过自身眼光和品味发掘优秀的艺术家及其作品，投资版权 NFT，不仅获得 NFT 增值，而且获得比高于 Spotify 和 YouTube 等传统流媒体十几倍的稳定版权收益；对于其他粉丝而言，平台提供了各类参与方式（SocialFi），在欣赏作品、宣传推广作品的同时，参与者可以获得平台丰厚奖励。平台将拥有三种 SocialFi 挖矿玩法：

播放挖矿

用户可以选择参与播放即挖矿机制，在播放作品的同时，根据自己的播放时长获得一定的通证奖励。

分享挖矿

已经下载了作品副本的用户可以获得 EUT 社区为该用户特制的链接，该用户可以在 EUT 社区或其他媒体平台，比如推特、Facebook 或 Telegram。如果其他用户通过这个链接下载或者观看了流媒体内容，那这个用户将获得一定的 Token 奖励。

投票挖矿

EUT 社区每周会发布一系列排行榜，对不同类型中最有价值的作品（如 Top 100）进行聚合，由社区里的艺术家、粉丝和其他用户共同投票，并结合使用维度来决定最终的排行榜排名，投票者投中的作品如果在排行榜上节节上升，则可以获得丰厚的 Token 奖励。

2.4 Catalog

项目简介

Catalog 是一个数字唱片店和音乐社区，供歌迷聆听、发现和直接支持他们喜爱的音乐。艺术家可以立即获得 100% 的唱片销售收入，以及二级市场转售的自选份额，而无需放弃他们的版权。歌迷们可以通过收集 Catalog 上的作品，与艺术家建立永久的联系，并可以共同分享他们的成功。

通过发行 Catalog Record，它代表了艺术家作品在网上的创世版本。它们是独特的、1/1 的唱片版本，永远存在于互联网上，不受任何一个平台的影响，并为艺术家和歌迷提供了一个可编程的画布，以建立一个新的音乐所有权世界。

资金

2021 年 6 月下旬，Catalog 由 1confirmation 领投完成了 220 万美元的融资计划，主要用于扩张团队。在融资之前，团队成员从 3 人增加到了 5 人，团队成员信息并未披露。

2.5 Royal

项目简介

是一个音乐投资平台，艺术家可以使用该平台来出售他们的歌曲，粉丝则可以从中购买歌曲的所有权，并从其购买的歌曲中获得版税收入，以此来实现双方共赢的局面。力争将传统唱片工业内只限于特定群体，如私募股权公司、对冲基金和唱片公司的投资机会更加公开地赋予艺术家和粉丝。

团队

Justin Blau AKA 3LAU (Founder & CEO)，是早期的加密采用者和区块链倡导者，他于 2018 年就开始参与 NFT。除此之外，他还是一名电子舞曲艺术家兼制作人，作为电子舞曲领域收入最高的五位艺术家之一，曾登过《财富》杂志，在其中最具影响力的创作者里排名第 8 位。2021 年 3 月，3LAU 售出了世界上第一张数字代币化专辑，创下了 1170 万美元的收入，创造了纪录，其在过去一年的 NFT 总销售额超过了 2000 万美元。

JD Ross (Co-Founder) 在这之前，JD 曾与他人共同创立了 Opendoor ($OPEN)，而且是 Atomic 的普通合伙人，该风险工作室共同创立了一些初创公司，包括 Hims & Hers、Bungalow、Butter、Homebound、Villa、OpenStore 等。

3. 权益分配

这部分将对 Euterpe 、WVRPS、AM DAO、Catalog、Royal 五款产品从版权上的进行横向对比。

选择 Euterpe 的艺术家或唱片公司等版权方将作品首发到 EUT 社区发起众筹，发行 NFT 获得早期创作资金支持同时让渡部分自己的版权给早期投资的粉丝。艺术家和粉丝共同拥有艺术作品的版权，而平台不拥有版权。NFT 持有者根据版权占比分配数字版权收益和部分挖矿奖励。

WVRPS 的持有人可以获得免费许可，将与你的 WVRP 相关的艺术作品和音乐用于个人或商业用途。WVRPS 团队不限制持有者从 WVRP 的商业化中赚取的商业利益。并且可以使用 WVRPS 的音乐作为素材进行二次创作，WVRPS 不收取费用，只需要标注采样出处。但是除了音乐之外的图片 IP，持有人在进行商业化使用时不得进行修改。

AM DAO 在版权上则更加激进地选择了和公共物品艺术家合作，AM DAO 孵化的艺术项目 采用 CC0 标准，无论是创作者还是 DAO 都放弃自己的所有权力，在公共物品的基础上进行建设。更多地通过周边衍生品，线下演出空间等方式寻求变现。

Catalog 发行的 Catalog Records 本身是 1/1 的链上唱片，通过区块链技术将音乐和智能合约相结合，艺术家根据发行时设定的分成比例享有 NFT 在二级市场转手时的版税分成。但除此之外，Catalog 作为平台方并不为艺术家或者收藏者划分明确而清晰的权益分配。所谓的「可编程的画布」实际执行和法律保护上还有待考察，详见下图。

4. 经济模型

音乐 NFT 赛道的经济模型非常明晰，艺术家让渡自己的分成权益，通过智能合约或者法律进行确权之后铸造 NFT 并发行。粉丝出于对偶像的喜爱，以及对作品未来的价值预期进行 NFT 收集。平台方简化了音乐 NFT 的发行门槛，提供版权和配套法律服务收取一定的费用作为收入。

在本文介绍的 5 款产品当中，经济模型最为亮眼的是 Euterpe。3 种 SoicalFi 的加入，在上述简单的 NFT 发行逻辑之外延长了产品链路。播放和分享两种普通消费者的消费行为被纳入了产品生态，并且通过投票挖矿的方式，在良莠不齐的 UGC 内容当中，筛选出头部的精品进行通证化。这种经济模型可以带来的飞轮效应是显而易见的：通证经济刺激了内容消费者参与版权拍卖，艺术家和粉丝评论投票增加了优秀作品的声量，粉丝分享增加流媒体播放次数，最终刺激通证经济。

传统 Web2 流媒体播放平台 Tidal 的访问量是 audius.co sound.xyz 的 10 倍以上。

5. 总结

目前在音乐和 Web3 领域主要集中在为艺术家和他们的粉丝提供小规模的解决方案，这只需要简单的文档来帮助人们浏览新的支持和互动模式。如果一个艺术家能够通过 NFT 销售赚到足够的钱养活自己和自己的艺术作品，那就是一场胜利。

然而，音乐 IP 的复杂结构并没有为简单的解决方案提供框架。艺术家所渴望的简单性与他们的艺术所处的复杂结构之间的这种紧张关系，使我们在现代音乐 NFT 领域发现了两个危险现象：

1. 简单的语言替代了 NFT 合同本应包含的对权利的准确及全面的描述。

2. 在这些合同中，对于 NFT 何时是一种投机工具，何时是一种营销工具，明显缺乏具体说明。

引自《定义音乐 NFT 的所有权，从数字世界到物理世界 - Water & Music》

音乐 NFT 赛道日益升温的背景之下，Euterpe 从版权角度出发保护创作者和通证持有人的利益，团队有实力打造有效的版权保护机制，避免版权侵权，建立合规路线，来实现平台的合规运作。AM DAO 则选择了另一个方向，和公共领域的艺术作品合作，避免版权侵权风险。将自身的重心放在孵化更多的艺术家上，通过 DAO 的赋能来解决艺术家冷启动的业务痛点。Royal 通过简化业务逻辑，将流媒体播放分成用作 NFT 背书的做法，实际上并没有改变传统的商业模式。Catalog 作为平台方是否真正有能力做到帮助和监管创作者，内容消费者能否能够兑现创作者的全部承诺还需要时间去验证。

音乐 NFT 在将来日益丰富的 Web3 世界当中应当具有更加广泛的流通性。在用户拥有了一段音乐的所有权之后，就会自发地想要在元宇宙的不同场景中使用它。比如在元宇宙当中和朋友一起聆听，在游戏胜利之后作为炫耀的方式，在个人主页当中展示作为自己铁杆粉丝象征等等。项目方在如今把音乐通过 NFT 的方式确权之外，还更加应当考虑如何让音乐广泛地传播。
