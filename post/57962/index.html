0 1  引言

0 2  常见诈骗套路

1.假冒交易所

你可能会在网页上看到一些看起来很像真的虚拟货币交易所，但其实都是假的。这些假冒的交易所会引导你输入你的个人信息和交易密码，然后他们就可以偷走你的钱或者信息。举个例子，2021年，警方就破获了一起假冒交易所的诈骗案。骗子复制了一个知名交易所的界面，创建了一个虚假的交易平台。他们诱骗用户在这个平台上充值并尝试进行虚拟货币交易。然而，这个平台实际上是由骗子控制的，他们可以随意修改数据，窃取用户的个人信息或直接转走资金。曼昆团队此前接到一个客户的咨询，ta被某诈骗团队通过虚假交易所的方式，骗走数字资产上千万。

2.山寨币、空气币

虽然从2017年开始中国禁止山寨币的发行，可还是有大量的骗子团伙以海外公司或项目方的名义发行各种虚拟货币，而这些骗子创建的“山寨币”或“空气币”通常只有骗子集团内部的共识，而没有真正的价值，只是骗子用来骗钱的工具，江湖别称“资金盘”。比如说，2022年，警方公布了一起涉及山寨币和空气币的诈骗案件。骗子利用投资者追求高回报的心理，通过社交媒体散布某虚拟货币前景广阔的虚假信息。他们声称这个币是由一个知名公司发行的，具有很高的升值潜力。在发行初期，这个虚拟货币的价格确实有所上涨，吸引了大量投资者。然而，当投资者想要卖出时，却发现这个币已经变得毫无价值，骗子也已经消失得无影无踪。

3. 高额回报诱骗拉高出货

如果某素味平生的网友突然要带你冲科太币，那他大概率是要骗你。这个相信看了电影《孤注一掷》的朋友应该都明白我在说什么。他们往往会通过所谓内幕消息的方式，让你的账户前期适当的赚一些钱，然后鼓励你投资更多的金额，并说投资越大，收益越大；并通过各种销售的收益截图朋友圈不断的刺激着你想赚钱的心脏，想方设法的诱导你进行投资。然而，绝大多数的截图和数据库都是为你量身定制的，他们的目的是诱使其他投资者高位接盘，然后自己出货，只要你敢下重注，他们就敢真跑路。

0 3  风险防范策略

1.核实信息来源

交易场所上，尽量选择Top5的虚拟货币交易所，不是说他们有多么安全，而是矮个子里头总归有个高个，市场体量那么多，有一定的良好声誉和行业节操。在投资标的方面，对于项目的真实信息可以去做下功课和了解，比如查看下网站的域名注册信息、调查团队的背景和信誉等，虽然行业中找个欧美团队在前面顶着后面都是华人团队的情况很常见，但通过网络上尽可能多的信息了解，对于过滤掉一些野鸡项目还是很有帮助的。

2.谨慎投资

在进行虚拟货币交易时，要做好充分的市场研究和风险评估。不要轻易相信所谓的“内幕消息”或“稳赚不赔”的投资机会。不要轻易相信高额回报的承诺，如果你发现虚拟货币的价格异常波动或出现大量抛售等情况，应该及时调查原因并谨慎决策，能止损的早日开始止损。此外，我个人的建议是个人散户玩家尽量不要去掺和一级市场的投资，也别去冲什么DEX上的项目，你在电报群听说的暴富传说只是项目方希望你听到的江湖故事而已。

3. 使用安全支付方式

不熟悉的网络热点和WiFi别乱连，手机流量真的花费不了几个钱，尽量的用个安全干净一点的手机来进行交易。在进行交易时，推荐使用受到广泛认可和安全保障的支付方式，如声誉良好的交易所的官方钱包等。避免使用不常见的支付方式，以防遭受诈骗。同时，要警惕陌生人的电话、短信和邮件，不要轻易点击链接或下载未知来源的文件。

4.定期备份和安全存储

定期备份重要的虚拟货币钱包和交易记录，并将其存储在安全的地方，如加密硬盘或离线的冷存储设备。当然冷钱包也可以适当的多准备几个，避免一个丢失，全部资产阵亡；如果你使用的是热钱包，请确保你的私钥助记词的保管是安全的，如果你的数字资产是存放在中心化的交易所，那么务必保管好自己的交易账户和密码安全，不要随意透露给其他人或在不安全的网络环境下使用。

04  如果上当了怎么办？

如果你不幸成为骗子的目标，首先要做的是尽快采取措施，尽量减少损失。以下是曼昆律师在实务经验中给大家积攒下来的一些经验，大家可以去抄作业。

1.立即停止交易并冻结账户： 如果你怀疑自己已经成为了诈骗的受害者，如果资产是在中心化的交易所，首先要做的是停止所有的交易活动并向平台申请冻结你的账户；如果你的虚拟货币是存放在自己的冷钱包中，且丢失的是单种虚拟货币，请在第一时间将自己的虚拟货币进行转移，确保钱包中的资产不会继续被盗，这可以防止骗子继续转移或窃取你的资金。同时，保留所有与交易相关的记录和证据。

2.收集证据： 收集所有与诈骗相关的证据，比如交易记录、邮件、短信、聊天记录等。这些证据可以帮助你追踪骗子的行为，也可以提供给相关的机构进行调查。尽可能保存好电子证据，比如交易记录、电子邮件等。

3.联系平台： 如果可能的话，联系虚拟货币交易平台，向其报告骗局并请求协助。一般主流的虚拟货币交易所都会有相关的客服团队，不过需要注意的是，联系客户千万不要在电报群或者网络检索去找，尽量通过官网或者推特的认证账号进行联系。因为有大量的骗子会冒充假客户来对你进行二次伤害，这个在行业中也是非常常见了。

4.打110报警： 如果你的损失较大，建议向当地警方报案。提供尽可能详细的信息和证据，以便警方能够调查和追踪骗子。很多朋友以为虚拟货币不受法律保护，资产丢失或者被骗后不敢报警。甚至有的朋友去报警，被公安一句你们玩虚拟货币的都是违法的给怼回去。但这显然是对虚拟货币的一种错误理解，从曼昆的实务经验来看，无论是民事法院审判，还是打击刑事犯罪，对于公民个人持有的虚拟货币都是会予以保护的。

05  结尾

天上不会掉馅饼这是一个学过基础物理学都会知道的道理。虚拟货币交易中存在多种欺诈手段并不会特别的高明，往往都是利用投资者追求高回报的心理，通过虚假的投资机会诱使其转账或购买毫无价值的虚拟货币参与资金盘里击鼓传花的游戏。有时候我们接到的一些咨询中，很多客户会说：我知道他们是骗子，可没想到我会栽进去。

《孤注一掷》里面的反诈宣传警官说了一句话很是经典，说为什么反诈宣传都已经铺天盖地了可上当受骗的人民群众还有那么多？警官的回答是：因为人有两颗心，一个是贪心，一个是不甘心。
