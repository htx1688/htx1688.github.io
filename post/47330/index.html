撰文：Alex Gedevani

NFT 领域正在迅速发展，其副产品是进一步的碎片化。这就提出了一个问题：产品如何定位自己以满足用户的需求，以及用户如何在这些产品中分配他们的时间。

答案可以归结为注意力。我们如何最好地确定 NFT 用户的注意力？最宝贵的商品在哪里？

我的心智模型很简单。想想你在过去 6 个月中使用的每一个 NFT 相关产品。有可能他们中的大多数都属于并服务于下面图形中的四个注意力板块中的至少一个。

我们都是：a）消费信息（信息层）或 b）采取行动，无论是出于财务还是非财务目的（行动层）。

让我们把这种「NFT Godmode」称为北极星，即服务于这四种注意力模式的产品充分满足所有用户需求。但我们还没有到那一步。

我们离实现 NFT Godmode 有多远，今天的产品在这些注意力块中是如何定位的？让我们在下面深入探讨。

行动层 Passive

你有没有在 Gallery 或 OnCyber 上策划过你的 NFTs？在像 Storyverse 这样的社区中基于你的 NFT 创建 UGC？在 「Passive」板块下的产品通常为用户提供在 NFT 收集过程后要做的事情，且主要是非投机性的。

Active

Active 是围绕财务意图进行的。这种注意力模式下的大多数产品侧重于改善交易体验或以富有成效的方式利用 NFTs（ NFTfi）。

注意力到哪里，流动性就流向哪里。投机仍然是加密货币最大的使用案例，具有压倒性优势。这个象限在关注度、产生的收入和团队数量方面占主导地位，这并不奇怪。看看 OpenSea，它的突破性成功导致了对 NFT 交易量蛋糕的激烈竞争。即使在熊市中，Opensea 在过去一年中以 3.91 亿美元的收入排在以太坊之后。

信息层 Push

Push，或信息层往往在行动层之后发挥作用。在这里建立的产品往往集中在两个关键支柱上：为用户节省时间和金钱。

你已经购买了资产，但现在需要围绕与你现有资产有关的信息进行一定程度的个性化。否则，你将在碎片化的信息中挣扎。我不是唯一一个在推特、DC、电报和时事通讯跟踪信息时遇到麻烦的人。人们正在忘记 Mint、白名单、提案和更多的东西。

迎合个性化的产品的一个很好的例子是 Daylight。把 Daylight 想象成一个个人助理，其唯一的工作是告诉你可以对你的 NFTs 采取哪些待定行动。

LuckyTrader 可以设置为提供与你钱包中的资产有关的个性化新闻和市场信息。它是你的消息来源，让你了解情况。

Pull

如今的另一个大问题是在这个领域缺乏可发现性和策展性。用户在以下方面经常遇到问题**：a）寻找新的产品来尝试，b）发现其他人正在接触的东西，以及 c）跟上流行的东西。**

DappRadar 和 Dapp List 等网站正试图解决应用发现的问题。像 Context 和 Soho 这样的产品正专注于钱包追踪，这可以浮现出新的想法。

这个象限有进步，但也许 Web3 原生搜索平台可以进一步推动事情发展。在一个理想的世界里，我们看到更多像 Ora Labs 这样的项目正在推动 Web3 数据的自然语言搜索，同时也倾向于基于这些数据的应用发现（「向我展示过去 7 天在以太坊上使用最多的前 5 个体育类区块链游戏」）。

与「Push」不同的是，这一层并不是围绕一个人可能使用的现有应用程序量身定做的新闻流，而是更倾向于将用户引向他们可以拉动自己的下一组产品或趋势。目前还不存在一个全面的解决方案。
