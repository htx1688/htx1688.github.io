作者：TinTinland，ODAILY星球日报

据 GlobeNewswire 最新统计数据显示，区块链技术的全球市场在 2022 年为 34 亿美元，至 2026 年将达到 199 亿美元的规模，将以 43% 的复合年增长率增长。

区块链技术如今变得越来越重要。 从概念先行到应用落地，该技术不仅为用户带来了 NFT、元宇宙等颠覆性体验，而且赋能金融、社交、体育等传统行业，极大拓展了其发展的边界。尽管今年全球经济在宏观逆风下举步维艰，加密市场也受之影响渐入寒冬，但受益于区块链及相关技术在数字身份、支付、交易等应用中的日益普及，加密领域的增长趋势有望持续。

而公链作为加密领域最重要的底层基础建设，如何未雨绸缪搭建好公链平台，未来才有机会打造更完整和丰富的 Web3 生态。 回顾加密行业上一轮牛熊转换中的公链项目，不难发现，如 Solana、Cosmos、Avalanche 一般数十倍、百倍爆发的公链项目都是自熊市逆风诞生积累，在牛市抓住机会脱颖而出，随后快速扩张，这些公链在牛市高峰期的市值超过 3000 亿美元。

在介绍新公链之前，让我们先回顾一下已有的公链发展格局。2022 年，以太坊“一骑绝尘”，而围绕以太坊可扩展性问题，产生了兼容 EVM 系公链，目前 EVM 系公链形成了一个庞大的加密生态岛群，在整个公链生态中举足轻重。而非 EVM 系公链如 DFINITY、Polkadot、Flow 等从多角度进行技术创新，各放异彩。

了解机会才能更好地把握机会，这一轮熊市中，我们该关注哪些新公链项目？又有哪些发展的机遇可以把握？本文将为大家系统盘点今年来自 L0、L1、L2 三层约 60 个新公链项目。 由于项目较多，本篇将分为上、下两篇，建议大家收藏再看。想看精简版的小伙伴可滑至文末总结。上篇将聚焦到 Layer 0 和 Layer 1 这两层公链。

2022 新公链 Highlights

关于 2022 新公链你需要知道的几点：

  • Move 系新贵强势来袭

  • 游戏、元宇宙、DeFi、隐私等专用链

  • 模块化区块链横空出世

  • 打破“孤链”，跨链万物

  • DAO 链将所有权还给社区用户

  • 第四代公链或将突破“区块链不可能三角困境”

  • 环保为上，绿色区块链受全球社区追捧

2022 年 60 个新公链（上）

从区块链逻辑架构出发，可以将公链分为 Layer 0、Layer 1 和 Layer 2 三层。具体而言：

  • Layer 0 对应的是“传输层”。 这是互联网和硬件连接的地方，它能够允许区块链之间互相交互，比如 Polkadot 和 Cosmos 都是在这一层创建了跨链可互操作的生态系统。此外，由于处于最基础底层，只要它们是使用相同的第 0 层构建，开发人员就不需要反复从头开始，很多功能可随时取用。

  • Layer 1 对应的是“数据层、网络层、共识层、激励层”。 这是区块链链上处理数据和进行交易、区块争议解决等技术运行处理的地方。而以太坊就是最为著名的一层公链，但这层公链面临最重要问题是如何克服“不可能三角困境”，即去中心化、安全性和可扩展性。尽管以太坊通过 Merge，试图通过升级以解困，但截至目前，似乎没有一层公链能很好地解决这一问题。

  • Layer 2 则对应了“应用层、合约层”。 这是与 Layer 1 结合使用的第三方集成，主要为了提高区块链的扩展性和吞吐量。目前 Layer 2 技术方案包括 State Channel、Rollups 和 Plasma 三类。其中 Rollups 方案获得以太坊社区青睐，其又分两种类型：Zero-knowledge Rollups 和 Optimistic Rollups。

此外，Layer 3 是客户端应用层，考虑到 Layer 2 的拓展性和吞吐量仍旧不能满足发展需要，因此产生了将层级间与同层间的组合操作平移至定制化 Layer 3 ，这已经不属于区块链逻辑技术范畴之中，且 Layer 3 链尚处于技术构想阶段，所以并未在分类之中。下面将介绍年度 L0、L1 这二类新公链，L2 将在下期介绍。

Layer 0 

随着 Web3 多链共存的现实不断明晰，难免会遇到信息、资产等跨链转移的问题，允许区块链相互交互 Layer 0 层成为打破“孤链”问题的门径之一。

LayerZero 跨链轻量

LayerZero 引入超轻节点（ULN），是一种全链互操作性协议，专为跨链传递轻量级消息而设计。LayerZero 通过可配置的去信任化提供真实且有保证的消息传递，其依赖两方在链上端点 ORACLE 和  RELAYER 传输消息。LayerZero 支持跨链状态共享、桥接、借贷、互换、治理等。融资： 1.35 亿美元以上。

网址：https://layerzero.network/

Wormhole 利用通用消息传递和单一 SDK 的力量，以互操作性协议使 xChain 比以往任何时候都更加丰富，进一步探索更多关于如何通过 xChain 扩大用户群。该协议的总交易量超过 $ 35 B，成功传输了数百万条消息。目前，每天有数千条消息被提交。本月，Wormhole 首次尝试连接到 Cosmos 生态系统，实现跨链联合。

网址：http://wormholenetwork.com/

Layer 1 

上一轮牛市中 Layer 1 公链之战轰轰烈烈，无论是 EVM 系公链，还是非 EVM 系公链都出现过风靡一时的佼佼者。而在今年，熊市沉淀下依旧诞生了诸多 Layer 1 公链，它们围绕着网络规模可扩展性、安全性以及“不可能三角困境”技术创新，产生了一众新兴项目。

逆风启航，新公链初心不变

随着熊市的降临，加密领域经历着波澜动荡的一年。但从新兴公链的发展来看，Web3 仍在全球范围内释放着独特魅力。公链是区块链领域中最为重要的基础组成， 2022 年以太坊一超多强的趋势没有被打破，但新公链却逆风而上，如雨后春笋一般涌现。

上文为大家介绍了年度 Layer 0 和 Layer 1 新公链的发展情况。首先，在 Layer 0 层出现了引入超轻节点的跨链项目，这意味着跨链互操的安全性将得到极大的提升 。目前对于跨链桥接和消息传递，几乎所有现有方法都属于两大类之一。第一个是形成共识，形成中间链，在链之间验证和转发消息，由于成本低，大部分项目都会采用这种方式。第二个是在链上运行一个轻节点，安全性十分高，但十分昂贵。期待该项目后期生态的发展成长，或许能够打破区块链孤岛问题。

而今年 Layer 1 公链赛道竞争依旧十分激烈。总的来看，虽然他们各自技术不同，但我们能够发现几点共性：

一是新公链的发展已跳出传统叙事。

新公链已经不满足于单纯解决区块链不可能三角困境问题，除了坚持去中心化基础上解决安全和扩展性问题，它们纷纷另辟蹊径，以帮助 Web3 行业更好地发展；

二是共识机制的转变大势所趋，公链更加注重绿色环保。

这也与今年轰轰烈烈的“绿色区块链”行动密不可分，大家希望能够扭转区块链“破坏环境”的不良印象，甚至出现了专门的环保公链，并得到了全球社区的追捧；

三是除几个公链新贵外，大部分公链项目的融资并不乐观。

我们知道公链，尤其是 Layer 1 公链的发展与成长并非一蹴而就，其发展前期是需要密集的融资、人才、时间投入的艰巨历程。受到加密寒冬的影响，在市场收紧的关键时刻，融资的支持或许有所不足，其中很多新公链会很快迎来一轮洗牌。

但优质项目能够通过时间的大浪淘沙，以技术创新走出加密寒冬，或许就能够成为下一个百倍成长的独角兽新星，让我们拭目以待。
