作者：veDAO

一个新晋 NFT 市场再次掀起加密市场的波澜。2 月 5 日，据 Dune Analytics 数据显示，Blur 近三个月成交量突破 2 亿美元，远超同类 NFT 聚合器。Gem（3818 万美元）、Reservoir（1989 万美元）位列三个月成交量排行榜第二、第三。

从 2022 年 3 月 Blur 上线，通过推荐等候名单的方式短时间内吸引了大量粉丝；Blur 公开发布后短短两周就吸引了大量用户关注，上线三天就超过 Gem，成为销售额最高的 NFT 聚合器；Blur 宣布的两个空投是当时增长背后的主要原因，Blur 的第三轮空投开启伊始，甚至导致 BAYC、Azuki 等蓝筹 NFT 交易额和地板价快速上涨。同时，在市场资金不足、“Free Mint”与“零版税”对项目可持续发展造成困扰的双重打击下，玩家们习惯了“快进快出”：一个优秀的聚合器，是现阶段活跃在市场内的玩家们的迫切需要的。Blur 是为专业交易者准备的，正是契合了当前市场的需求。

如果像以前那样直接在 NFT 交易市场内进行买入，不同市场间的底价差异与迟缓的操作响应将导致无法快速地获取价位合适的筹码，结果只会是错过或者亏损。在 OpenSea 或其他的 NFT 交易市场里，首页每天推荐着不同的新鲜玩意，寻找自己心仪的图片，选择后慢慢等待成交，如同在逛集市；而在 Blur 里，首页只有热门项目的交易数据看板，只有快节奏的买卖，仿佛是在交易所。

Blur

Blur 是一个具有很多创新功能、为专业交易者设计的 NFT 市场和聚合器。官方对自己的定义是“一个实时 NFT 市场和聚合器的结合”，支持 NFT 批量交易，并且采取零版税费用（Opensea 抽取 5% 的费用）。Blur 平台聚合了来自 OpenSea、LooksRare 和 X2Y2 等领先交易市场上的 NFT 系列，也允许用户在 Blur 上上架自己的 NFT 资产，与许多其他收取交易佣金的 NFT 市场不同，Blur 维持 0% 的 NFT 交易费率。

团队背景

2022 年 3 月，Blur 宣布获得超过 1400 万美元融资，Paradigm 领投，参投者包括 Keyboard Monkey、LedgerStatus、0xMaki、Santiago Santos、Zeneca、Deeze 等机构和天使投资人。Blur 的团队由来自 MIT、Citadel、Five Rings Capital、Twitch、Brex、Square 和 Y Combinator 等知名机构的开发人员组成。

Blur 火热的背后

针对加密老手的操作与界面

打开 Blur.io，点击链接到钱包，系统会提示用户连接他们的 Web3 钱包（目前支持 MetaMask、WalletConnect 和 Coinbase 钱包），链接成功后就开始探索网站上的各种 NFT 。

默认情况下，藏品会按照单日交易量顺序列出，并显示一些关键数据（例如地板价、持有者数量、以及各种价格和数量指标）。用户还可以通过 Blur 的投资组合选项卡查看钱包中所有基于以太坊的 NFT，继而很容易追踪列出的 NFT 及其稀有性、预估收藏价值，以及生命周期损益表。

Blur 界面是可定制的，在浏览 NFT 集合时，用户可以根据自身喜好在多个视图（例如列表和图块）之间切换。与用户需要切换选项卡才能查看销售历史或其他分析的 OpenSea 不同，Blur 在单个页面上就能显示所有内容，从而更容易访问和比较所有相关的 NFT 收藏数据。

就像前文描述的，OpenSea 像是“集市”，对于没有经验的新手也是友好的；而 Blur 则是“交易所”，更多针对加密世界老手、高级用户。总体而言，Blur 作为聚合器来说，是非常好用的。

为交易者提供高效与便捷的功能

Blur 平台的运行速度比其他 NFT 聚合器快十倍，在不到 1 秒的时间内显示 NFT 上的待处理交易，并每 4 秒更新一次列表。快如闪电的待处理交易显示，加上收藏艺术展示上近乎即时的元数据更新，为想要“快进快出”的用户提供了巨大优势。此外，Blur 的 gas 优先级预设能帮助交易用户领先于通过其他交易所或聚合器购买 NFT 的对手。最重要的是，Blur 的合约比其他市场更优化，为用户节省高达 17% 的 gas 费。

Blur 另一个广受好评的功能是可以清晰地显示不同 NFT 的稀有性和特定属性下的地板价。在 Blur 平台上，并不是通过系列清单和销售历史来评估高于地板价的 NFT ，而是根据单独特征找到价格最低的 NFT 清单，这个功能不仅可以帮助用户准确评估自己的资产，也可以让那些希望抄底的 NFT 买家受益。同时，对于那些希望抄底的用户，Blur 也提供了相应的工具，所有 NFT 系列都有一个地板价深度分析图，可以显示不同地板价上方列出了多少 NFT，让抄底用户直观了解他们购买的 NFT 将如何影响市场。

最为一个专注于 NFT 交易的平台，Blur 的优势确实领先于目前市面上的同类项目，不过目前 Blur 只支持以太坊，但其团队表示未来计划加入更多公链。

Blur 的“自定义版税”与“零费用”

一直以来，版税是 NFT 交易市场的敏感话题，“交易者希望利润最大化，收藏家希望支持创作者，创作者希望获得更多的版税收入”。因为版税无法在多个市场上强制执行，因而不同平台版税设置不一，当交易者在 Blur 上上架时，他们可以自定义其上架的版税。

此前 SudoSwap 和 X2Y2 分别的零版税和自定义版税就已经引起市场上很大的议论，市场认为自定义或零版税使项目方失去部分资金持续发展，因此 Blur 期望通过空投激励预期鼓励预期交易者使用版税。在第二轮空投中，Blur 的使用者版税设定越高将获得越高的忠诚度从而获得更多空投。但实际上，这种预期鼓励实际并没有阻止低版税甚至说零版税行为，Blur 整体平均版税率依然很低，甚至带动了整体 NFT 市场版税的下降。

Blur 并没有像 OpenSea、LooksRare 和 X2Y2 一样向使用者收取费用。交易者使用 Blur 聚合器、交易市场等功能不需要任何成本。在版税为自定义前提下，交易者可以直接获得完整收入，但目前零费用应该是作为短期吸引新用户使用的策略。

不过由于 Blur 的自定义版税，还引出了 Blur 和 OpenSea 的“黑名单之争”。

Blur 和 OpenSea 的黑名单之争

2022 年 11 月份 OpenSea 实施新政策：寻求强制版税的 NFT 项目，必须拉黑那些不完全尊重它们的市场。由于 Blur 当时并没有强制版税，所以它被拉入了黑名单，这导致在 OpenSea 上强制版税的 NFT 收藏品不能在 Blur 上进行交易。之后，Blur 尝试和 OpenSea 沟通，意图将自己从黑名单中移除，但被回绝。

2 月 1日，Blur 通过利用 OpenSea 的 Seaport 创建了一个新的 NFT 交易系统，绕过 OpenSea 的黑名单控制。由于 Seaport 不在 OpenSea 的黑名单中，导致拉黑 Blur 的 NFT 项目现在可以通过新系统进行交易，并强制执行版税。

目前 Blur 有两个系统可以在其市场上执行交易。旧系统继续处理那些没有拉黑它的 NFT 项目，而新系统则负责以前拉黑它的 NFT 项目。由于 Blur 会自动选择系统，给用户带来了流畅的体验。

以普遍理性而论，“零版税”让中小创作者/项目方变得更加艰难。但与其说是 Blur 带来的错误，不如说是 Blur 顺应了市场。2022 年加密市场的严寒尚未完全退却，目前活跃的用户主要是交易者，而正是这些交易用户所代表的当下市场造就了现在的 Blur。

写在最后

Blur 将聚合器作为市场的流量导入器固然很好，但后续面临的挑战也依然严峻。OpenSea 麾下的 Gem，将会努力缩小用户体验上的差距，夺回聚合器赛道的头把交易。从 OpenSea 推出了 Collection Offer 与 Beta 版本的数据看板，可以看出 OpenSea 也感知到了当前市场环境下用户需求的变化，将 Gem 更无缝地融入到市场中的可能是存在的，Blur 需要持续保持其差异化优势核心，但“零版税”和“无手续费”确实契合了当下的市场，DNA未来如何尚不得而知。

同时，专业用户可能会觉得 Blur 的体验尚佳，但加密新手也许会觉得 Blur 提供的功能无法与 OpenSea 等市场提供的易用性相比。随着加密和 NFT 应用的普及，Blur 也有可能扩大其用户群；但如果想要超越 OpenSea，就需要兼顾专业用户和入门新手，使其平台对更广泛受众更易于访问和友好。
