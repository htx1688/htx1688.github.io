下周多项目迎来代币解锁，其中不乏老牌项目、知名项目，包括 AVAX、LDO 等。此外，1INCH、ACA 等项目将进行常规解锁。

上一周其中明星“新公链”Avalanche 是下周解锁金额最大的项目，将解锁超 1 亿美元代币。而在新锐项目中，Space ID、Injective 解锁量也较为庞大，ID 解锁量占流通市值 6.46% ，INJ 解锁了占流通市值 3.41% ，都足以对市场产生潜在冲击，值得引起投资者的重视。

大额解锁

Avalanche

项目官网：https://www.avax.network/

官方推特：https://twitter.com/avax

本次解锁数量： 954 万枚 

本次解锁金额：（约） 1.03 亿美元

近期动态：Avalanche 基金会向 Dexalot 提供价值 300 万美元的 AVAX

相关阅读：《Avalanche 创始人证词全文：我们正站在新时代的悬崖边上》

Avalanche 在上一轮牛市的“新公链”浪潮中风头强劲。作为一款高性能的 EVM Layer 1 ，在3AC等明星资本的加持下，AVAX 代币价格一路突飞猛进。

AVAX 代币的最新解锁计划将在未来一段时间内线性释放，此次解锁为每三个月一遇的规模较大的解锁。

按当前价格计算，解锁数量将释放价值超 1 亿美元的代币，也是本周代币解锁金额最大的项目。目前 AVAX 代币的流通供应量约为 3.4 亿枚，近 1000 万枚的 AVAX 解锁量已占所有流通代币的约 3% 。作为公链项目，该代币解锁速度与同赛道项目相比也较为缓慢，目前仍有约 47% 的代币仍在锁定之中，将在未来数年后完全解锁。

Space ID

项目官网：https://space.id/

官方推特：https://twitter.com/SpaceIDProtocol

本次解锁数量： 1849 万枚

本次解锁金额：（约） 366 万美元

近期动态：SPACE ID 发布 3.0 版本，对平台和治理结构进行重大改进

相关阅读：《一文读懂 Binance IEO 项目 Space ID》

Space ID 是一个链上 DID 项目。该项目为用户提供了“.bnb”、“.eth”、“.arb”域名服务。意图将这一项目打造为用户的去中心化多链身份。此外，项目方还为用户提供了域名交易市场。

Space ID 并不是一个时间上非常新生的项目，早在 2022 年 4 月该项目就已问世。但 ID 代币却是一款较为新生的代币，直至今年该项目才完成代币发行。2023 年 3 月，Binance 官方公告宣布，ID 代币为其第 30 个 Launchpad 项目。

ID 为 SPACE ID 的治理代币，总供应量 20 亿枚。ID 主要有三种功能。

其代币经济学如下：

20% 为种子轮销售；8% 用于战略论销售；15% 归于核心团队；7% 归属于顾问；10% 用于生态系统；13% 用于市场营销；12% 归于基金会；10% 用于社区空投；5% 用于币安 Launchpad。

ID 可用于质押，以此获得市场的交易费用折扣、域名注册折扣。用作 SPACE ID 生态系统和Web3名称 SDK 集成中的支付方式。参与 SPACE ID DAO 的治理。

目前，仅 18% 的 ID 处于流通之中， 82% 的代币仍待解锁。

本次解锁的 ID 约占流通量 6.46% ，这一数字较高，值得引起投资者的注意。

从历史解锁来看，ID 代币每月均会迎来大额解锁。

不止是解锁量值得关注，从交易量来看，Coingecko 数据显示，ID 的 24 小时交易量约为 990 万美元，本次解锁量则高达 366 万美元。ID 代币是下周解锁代币中，对流通量冲击最大的币种，值得投资者关注。

Injective

项目官站：https://injective.com/

官方推特：https://twitter.com/Injective_

本次解锁数量： 286 万枚

本次解锁金额：（约） 2262 万美元

近期动态：Injective 发布代币经济学升级，大幅增加 INJ 每周燃烧量

相关阅读：《一文读懂 Injective 生态联盟》

Injective 是一款围绕 DeFi 订单簿打造的 Layer 1 网络，如近期颇受关注的 Sei 相似。Injective 针对 DeFi 优化、具备互操作性，在 Cosmos 上拥有共识层和主权。其基础设施涵盖高性能的链上去中心化交易所基础设施、去中心化桥接器、预言机和带有 CosmWasm 的可组合智能合约层。

生态内的其他协议可以利用 Injective 的 onchain-orderbook 来启动流动性和匹配服务，增加了一层组合性。这一网络围绕着订单簿所打造，可为用户提供高性能、且聚合整条链流动性的 DEX 交易体验。

本次 INJ 代币解锁数量较大，高达 286 万枚。以流通量计算，已占据流通盘的 3.41% 。

但从解锁历史来看，INJ 每隔一或二个月均会进行解锁。7 月 21 日，INJ 上次解锁之时，一次解锁价值 2900 万美元的 267 枚代币。与历史解锁相较，本次解锁金额与以往相似。

尽管本次解锁从占流通份额来看，数字颇为可观。但历史上多次解锁均与此次相似，投资者可获得充足的历史数据参考。

官方推特：https://twitter.com/AcalaNetwork

本次解锁数量： 466 万枚

本次解锁金额：（约） 22.4 万美元

Acala 是波卡生态的老牌项目，旨在打造波卡的去中心化金融网络和流动性中心。该项目为可扩展的、与以太坊兼容的智能合约平台，并针对 DeFi 进行了优化，具有内置的流动性和现成的金融应用程序。凭借其无需信任的交易所、去中心化稳定币 (aUSD)、DOT Liquid Staking (LDOT) 和 EVM+，Acala 让开发人员可以访问以太坊的核心功能和 Substrate 的全部功能。

ACA 本次解锁金额约 20 万美元，只占流通市值的 0.58% ，尚不足 1% 。本次解锁数量远低于此前数次解锁。

Lido

项目官站：https://lido.fi/

官方推特：https://twitter.com/LidoFinance

本次解锁数量： 850 万枚

本次解锁金额：（约） 1403 万美元

Lido 是一种流动性池质押解决方案。Lido 允许用户在参与链上活动（如借贷、交易）的同时，以复合回报的方式在 PoS 公链上下注，而无需最低存款或维护基础设施。目前已支持 ETH 2.0、Terra、Solana，未来有可能扩展到其他 PoS 公链。

作为老牌知名项目，Lido 的治理代币 LDO 曾在质押热潮中价格一路飙升。

相关阅读： 《Lido 的故事：价值、增长与护城河》

下周，LDO 代币将迎来久违的大额解锁。

在 2022 年 12 月至今的超过半年多的时间里，LDO 一直在进行着小额、常态化的解锁。并未出现过单笔大额解锁。

本次 LDO 解锁数量远超今年 5 月至今的累计解锁数量。但从比例来看，尽管此次解锁数量颇高，但其解锁量占流通量之比约为 0.97% 。仍不足 1% ，并不是一个颇为可观的数据。在下周即将解锁的项目之中，仍是一个较小的数据。

但 LDO 作为知名项目，其解锁事件仍值得引起投资者的一定关注。
