今年 2 月份，由 Paradigm 领投 Web3 特许经营品牌 Parallel 的首款 TCG 游戏将进入 Closed Alpha 测试阶段，而根据 Echelon 基金会的提案投票情况，Parallel 还将在 3 月 1 日解锁其链游生态核心资产 Prime 代币。

在这篇文章中，笔者会以参与者的角度解析 Parallel 链游采用的“特洛伊木马”策略，以及它如何用可持续的设计走出 Crypto 游戏的 Ponzi 怪圈，此外，Parallel 提供的可组合性游戏设计方法也值得学习与借鉴。

而在了解 Parallel 的链游生态设计之前，我们需要简单了解一下整个游戏的背景故事。

一、Parallel 的背景故事

Parallel 的故事始于不久之后的地球，由于地球上的资源正在耗尽，能源变成了一种稀缺商品，迫切寻求解决方案的科学家们，尝试了一种使用反物质的新型裂变反应实验。但这次实验失败了，并引发了一场名为 “惊世” （The Event）的全球性灾难，一种新型辐射开始在地球上传播，致使数十亿人类以及动物的死亡。

在面临生存危机的背景下，人类建造了三艘称为 “创世号” 的太空飞船，将大批幸存者带离了地球，其中一艘前往了火星，一艘前往了欧罗巴（木星二号卫星），还有一艘前往了小行星带。数千年后，人类避免了灭绝的命运，并以五个被称为 Parallel 的独立群体生存下来。此后，“惊世” 事件被那些幸存者称为 “起爆点”（The Priming）。而地球也再次变得宜居，这里还出现了一种被称为 Prime 的新能源，五大 Parallel 群体为了控制地球以及 Prime 能源，最终发生了冲突。

土灵人（The Earthen） ：在 “起爆点”（The Priming）期间，一些人类被发现对辐射具有天然的免疫力，当数十亿人死亡时，这些免疫者却存活了下来，而土灵人（The Earthen）就是这些留在地球上的幸存者的后代。土灵人以自然为导向，他们与环境有着深厚的联系。在高度辐射的地球上生活了几百年后，土灵人已经进化到只用自己的身体就能操纵能源 Prime。

马科利安人（The Marcolian） ：在大逃离事件中，名为 “红色太阳”（Soleil Rouge）的飞船前往了火星，而那里已经存在一个科学研究殖民地。来自 “红色太阳” 号的难民与火星殖民者联合了起来，形成了一个新的具有科学思想和军事导向的社会。而马科利安人（Marcolian） 就是这些人的后代。

卡塔里人（The Kathari） ：“阿尔忒弥斯.比邻星”（Artemis Proxima）号飞船的目的地是欧罗巴（木星二号卫星）。在欧罗巴，难民们在冰冷的卫星外壳下建造了一座城市。一场恐怖主义行动摧毁了该殖民地种植任何作物的能力。卡塔里人为了生存而转向了基因工程，复制食物甚至人体。卡塔里人（Kathari）是欧罗巴难民的克隆后代。

奥根库尔人（The Augencore） : 在大逃离事件中离开的第三艘飞船是 “坚韧号”（The Fortitude），“坚韧号” 的使命是永远不要将飞船的生存与一个行星体绑定，因此这艘飞船从一个小行星移动到另一个小行星以开采资源。几十年来，低重力环境对船员老化的身体造成了损害，因此他们开始尝试选择性地进行手术增强。最终，机械身体增强成为了 “坚韧号” 的一种标准生活方式，并将其更名为 “凯恩-1”（Caine-1），而奥根库尔就是 “坚韧号” 原船员的后代。

施劳得人（The Shroud） ：早在惊世事件之前，人类就有一个前往一颗名为开普勒-1649c （Kepler-1649c ）的类似地球的外行星的任务。执行这项任务的飞船是 “新曙光号” （The New Dawn），在任务期间，“新曙光号”受到了一个黑洞的威胁，它虽然逃出了黑洞，但它最终在太空中的去向，除了这艘飞船上的船员外，对所有人来说都是一个谜。几个世纪后，当 “新曙光号” 回归时，飞船上的居民们以人类无法理解的方式发生了永恒的改变，而他们就被称为施劳得人（The Shroud）。

二、Parallel 的“特洛伊木马”策略

回顾以往的链游，例如 Axie Infinity （AXS）或 STEPN ，游戏项目方往往为了强调 Web3 所有权而设置了各种门槛，最常见的便是强制要求玩家持有 NFT 才可参与游戏，这不可避免地带来了用户体验摩擦，并导致游戏 Ponzi 化，这类链游也被我们称为 GameFi。它们的特点是在短期内吸引大量投机玩家，因此呈现出爆发式增长，但随着投机的结束，日活数量、游戏市值等数据均会呈现断崖式下跌。

而一些链游为了改善 P2E 机制，选择加入了免费玩（F2P）的策略，以扩大潜在的游戏玩家群体，例如 Gods Unchained 、Skyweaver 等卡牌游戏都是采用的这种策略，它们让玩家通过玩游戏获得非 NFT 卡，并通过代币升级为 NFT 卡来参与 P2E 游戏，然而这种策略的重点依旧是以 P2E 为主，而 F2P 只是作为一种引流的手段。

而 Parallel 的首款 TCG 游戏，则采用了真正的“特洛伊木马”策略，它将自己包装成一款传统 Web2 游戏，以吸引传统免费游戏玩家和付费游戏玩家，他们可以使用信用卡等支付方式购买卡包抽卡（非 NFT），并消费皮肤等各种不影响游戏平衡性的道具，就像《炉石传说》、《MARVEL SNAP》等传统游戏一样。

在这个面向传统 Web2 游戏玩家的系统设计中，Parallel 引入了一种玩家不可转让的名为 GLINTS‌ 的游戏货币，以用于在游戏内商店进行消费，而由此产生的收入当中，有 20% 将转向 Echelon 基金会，并用于支持 Prime 生态系统，包括回购以及提供流动性等支持方式，而剩余 80 % 则用于 Parallel 生态后续游戏、漫画、电影等内容的制作。

而面向更高阶的 Web3 游戏玩家，Parallel 采用的是 win-to-earn (W2E)机制，这意味着持有 NFT 卡并在竞技比赛中获胜的玩家才能获得 Prime 代币奖励，并且每个账户每天的奖励次数存在限制，而在一副卡组（40 张）当中，NFT 卡的数量越多，获胜赢得的奖励也就越完整。

另外，玩家在游戏获胜过程当中，也会被提示使用高级卡牌（NFT）的优势，这种设计实际上激励了玩家尽可能去收集完整的 NFT 卡组，从而让传统 Web2 游戏玩家转化为潜在的 Web3 游戏玩家。

三、Crypto 游戏的可持续之路，如何走出 GameFi Ponzi 怪圈

正如上面提到的，在 Parallel 的两个系统设计当中，传统游戏玩家体验了他们所熟悉的游戏环境，并为此消费了美元，而 Web3 游戏玩家则参与了风险更大的 Crypto 游戏玩法，并从传统游戏玩家群体那获得了一部分（20 %）游戏收入。

这种做法，可以让大部分玩家避开高风险的经济游戏，并纯粹为游戏体验而买单。另一方面，高风险的 Crypto 经济游戏，也可以通过持续的外部收入增加游戏的可持续性。

除此之外，为 Crypto 经济游戏设计各种 sink 消费场景，也是增加游戏可持续性的关键，在 Parallel TCG 游戏的例子当中，这包括艺术家签名、抽卡、兑换 GLINTS （消费皮肤）、复制 NFT 卡、支付 P2P 交易服务、铸造化身和装备等，这些消费场景的目的是让经济系统保持健康，从而减轻 Prime 代币投机活动带来的负反馈影响。

（当前 Parallel TCG 已通过 2 次 sink 活动消费了 6.48 % 已认领 Prime 代币 ）

但我们需要注意的是，这些设计能够取得成功的前提是需要一款甚至多款能够吸引传统玩家参与的游戏，这将是 Parallel 等 Crypto 游戏能否走出 Ponzi 怪圈的关键所在。

四、另一种可组合性思路

我们知道，Web3 游戏除了提供所有权的好处，还可以通过链上游戏的方式提供可组合性，这是很多链上（on chain）游戏探索者所追求的一种潜在好处。

而采用 offchain 方式的 Parallel 则提供了另一种可组合性思路，其开发商为每一张 NFT 卡牌创建了 3D 资产库，以用于接下来的其他类型游戏。

例如在 MOBA 游戏设计中，玩家可以用单位/装备在 PVP 策略比赛中建立一支军队，并且装备与单位之间是可组合的，根据军队中 NFT 的比例占比，获胜能够赢得的 PRIME 比例也会不同，这和 Parallel TCG 采用了相同的系统，并利用了同一个资产库。

（注：Parallel MOBA 游戏设计图，目前尚未进入开发阶段）

而其他类型的游戏，也可以遵循相同的方式进行开发，这可以是 Parallel Studio 自己开发的游戏，也可以是外部开发商开发的游戏，只需通过 Echelon 基金会投票接入即可。

这种可组合性方式，也是对传统游戏的一种升级。

五、Parallel 项目的现状

在 2021 年 10 月份，Parallel 完成了一轮由 Paradigm 领投的 5000 万美元 A 轮股权融资，当时项目的整体估值为 5 亿美元。

在 Opensea 平台上，当前 Parallel NFT 累积的交易量为 73043 ETH，由于其游戏资产的版税高达 10%，这为 Parallel Studio 带来了大约 7300 ETH 的总手续费收入，但考虑到其中有大约 1 % 分配给了 MP 卡和 The Core 卡持有者，NFT 二级市场为 Parallel Studio 带来的收入大约为 6574 ETH。

除此之外，Parallel 还在 presale (PS15) - PD5 的 6 次卡包售卖活动中筹集超过 10000 ETH，并将在最后一次 PD6 活动中筹集大约 1500 ETH。

截至目前，Parallel Studio 已经具备了开发 1-2 款高品质游戏的资金，而从 2021 年初开发至今，Parallel TCG 游戏已被打磨了大约 2 年时间，并已定于今年 2 月份进行 Closed Alpha 测试，而正式的游戏，预计将在今年下半年推出。

而在二级市场方面，在经历 2021 年 Paradigm 投资新闻曝光后的第一次炒作期后，大部分非稀有 NFT 资产的单价已经回落，目前跌至 0.0015 ETH的地板价（大约价值 2 美元）。

而游戏的主要 NFT 资产 Prime Key 在近期的成交价大约为 13.5 ETH （当前价值 17500 美元）。

考虑到每个 Prime Key 当前对应大约 4550 Prime (注：PK 卡持有者已领取掉了一半 prime 代币)，二级市场当前对 Parallel 项目的整体 FDV （注：总量为 1.111 亿）估值大约为 4.26 亿美元，相比 21 年末牛市融资时的整体估值仅下降了 15%。

六、最后的思考

鉴于 Parallel 游戏优先的设计方式，以及独特的“特洛伊木马”策略，其具备了一些条件去吸引大量传统游戏玩家，并可能在体验游戏的过程中将部分玩家转化为 Web3 游戏玩家。另一方面，由于 Parallel 没有强制玩家采用 Crypto 游戏方式，它在市值方面可能并不会像我们熟悉的 GameFi 游戏一样实现快速增长，因此可能并不会受到投机玩家的关注。

最后，Parallel 的游戏产品能否找到市场契合度，将会决定该项目的发展前景，而通过接下来的游戏测试，我们就能找到这个问题的答案。

展望未来，我们期待能够看到更多游戏优先的 Crypto 游戏项目，无论是 offchain 链游还是 onchain 链游，从而让更多传统玩家能够认可 Web3，而不是让偏向代币经济游戏的 GameFi 摧毁这个行业。
