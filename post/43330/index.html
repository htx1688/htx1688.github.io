作者：Kimming

BNB Chain是日活跃用户最多的智能合约区块链之一（嗯，没错，虽然它出事了），最近，币安宣布了第五批27家从其Web3加速器项目MVB(Most Valuable Builder)中脱颖而出的创业公司。7个优秀项目还将获得Binance Labs的投资，Binance Labs与BSC将共同领导这个计划。从这个项目中诞生的创业公司遍及DeFi、音乐和NFTs、游戏和Metaverse、基础设施和工具、安全等领域。

DeFi

一、Antimatter DAO

Antimatter于2021年初成立时，主要目标是建立创新模型，以彻底改变去中心化金融中的衍生品。受到Uniswap的启发，Antimatter一直在与不同学术背景的人积极合作，建立Antimatter的第一个原创永续期权模型。成员包括来自美国、英国和中国的数学、经济学和金融学博士和教授。如果没有这个跨学科和国际化的团队，Antimatter就不会诞生。在这个背景下，DAO组织就成立了：

  • 学术界有兴趣探索新的衍生模型；

  • DeFi的发展需要新的创新，而不仅仅是复制其他协议；

  • DeFi新模式需要传统金融的知识基础；

  • 目前没有交流和讨论新模型的环境。

简而言之，Antimatter DAO是一个学术界和爱好者们可以一起交流意见和实验的地方，只有一个统一的目标：推动DeFi衍生品的发展。

二、Tender.fi

Tender.fi是一种创新的开源去中心化借贷协议。它将支持整个投资组合抵押、0%利息贷款和一种新的长尾资产抵押机制。

三、Aperture Finance

通过跨链的战略市场，Aperture将来自不同区块链网络的投资机会连接到一个地方，为DeFi用户提供一站式解决方案，同时弥合链与链之间的差距。

四、Velvet Capital

Velvet Capital是一个DeFi协议，可以帮助用户创建加密投资组合：以它结合了现代投资组合理论和收益耕作，构建具有额外收益的多元化投资组合。

五、IdentDeFi

IdentDeFi是一个将标准token转换为包装的AML/KYC兼容token的平台。

六、InsurAce Protocol

安全领域的保险协议InsurAce Protocol

七、Wink Financial

Wink Financial——Web3工资和财务管理解决方案

八、Deus Ex Securitas

Deus Ex Securitas可以使用机器学习自动审计智能合约，NLP模型可以根据漏洞对Solidity脚本进行分类，甚至可以指出哪些行的代码易受攻击。

基础设施和工具

九、Multi-chain event protocol (MEP)

MEP是一种互操作性协议，充当身份验证服务，验证在链之间传递的消息。

十、Nakji Network

Nakji Network是从所有Web3中快速轻松地获取区块链数据的答案。Nakji充当从区块链到互联网的数据传输器。它有效地将数百万个数据点移至链下，以便任何想要连接到Nakji的API的人自由和公开地访问。

十一、Overeality

Overeality是一个由DAO技术管理的新形式的social NFT将人们聚集在一起的社区。目标是创建一个元宇宙，这个元宇宙是由人们关注、策划、访问和驱动的——所有人都有发言权，在游戏中有自己的皮肤，对即将发生的事情拥有平等的权利。

十二、Port3 Network

Port3通过聚合Web2和Web3数据，在区块链网络上提供开放的数据服务。这为创建用户身份、保护用户隐私以及实现其他各种应用奠定了基础。

十三、Staging Labs

Staging Labs是一个基础设施层，保护Web3用户和企业免受错误、黑客攻击和骗局的伤害。

 十四、Web3Go

Web3Go是一个集开放数据分析和服务于一体的平台，每个人都可以在这里掌握区块链数据背后的价值。

十五、Zash

Zash是一家企业级的区块链分析和情报提供商。它是区块链的读中间件，已经推出了一个多链NFT数据API，将复杂的事务转换为见解。联合创始人Parit Patel(前Drover首席财务官)和Efe Surekli(前Tractable, 2x AI创始人)于2021年初走到一起，建立了一个社交网络，通过公开区块链数据分享交易想法。通过这一点，他们确定了NFT的数据障碍和提供准确数据访问的需要。由链上数据和来自社交中心(如Discord和Twitter)的链下洞察组成。Zash消除了链上冗长的地址组合和复杂的交易，并以易于理解的格式呈现碎片化的社交媒体数据，使用户能够做出合理的基于数据的决策。

十六、Space ID

最近很火的，域名类项目（.bnb)。SPACE ID正在构建一个通用名称服务网络，该网络将跨区块链的人员、信息、资产和应用程序无缝连接起来。它与链无关、去中心化、抗审查和开源。

十七、Metasset

Metasset是一个建立在区块链上的NFT社交商务流动性平台，他们的目标是通过利用社交附属网络的力量来提高NFT的流动性从而加速NFT的采用、创新和金融化。Metasset将基础电商与社交媒体相结合，希望让每个人在Web3里都可以建立自己独特的NFT社交商务品牌。

NFT

十八、Double Protocol

Double Protocol是一个完全去中心和开源的非ft租赁协议和市场，用于Metaverse和GameFi的资产、探索和非功能性测试。

十九、GiroGiroAI

GiroGiroAI的目标是为NFT建立AI基础设施，增加内在价值，并将NFT转化为链上智能生物。

二十、Rareboard

Rareboard是BSC的非金融市场聚合器。这意味着在这里，用户可以从不同市场的各种收藏品中投标、铸造和购买NFT。它还允许用户获得在它的稀有仪表板上列出的收藏品的第二个稀有和价格数据。

Gaming & Metaverse

二十一、Gameta

Gameta是一个允许用户在同一个地方玩不同休闲游戏的平台。它的目标是将尽可能多的人从Web2迁移到Web3。Gameta的目标是广泛的受众，并相信它可以将在Web2时代使用的产品，如音乐、游戏和媒体，合并到新的Web3体验中。

二十二、Meta Apes

Meta Apes是一款free-to-play,和play-and-earn的MMO策略游戏。故事发生在世界末日后的世界，在那里，人类已经终结，一个由猿人统治的新时代已经开始。下一个议程是太空统治。每个灵长类动物(是的，你)将必须与他们的帮派密切合作，成为最强的部落，并赢得最终的太空竞赛。

二十三、CryptoZombies

CryptoZombies是一所免费、开源的交互式代码学校，可以让用户学习在以太坊上创建游戏。该课程是为初学者设计的，以绝对的基础知识开始。因此，如果用户以前从未使用过solididity编写代码，不必担心，它将手把手教学。

二十四、SocialFi/DAO

Kola Labs——除了视觉效果，音频是沉浸式和交互式虚拟世界的另一个基石。音乐的全部潜力只能通过以下方式释放：1.将创造力民主化，让每个人都可以拥有和享受个性化的音乐；2.将音乐NFT与现有的Web3生态系统深度连接。在Kola，尖端的生成艺术和AI技术用于为这种体验构建可扩展的解决方案。Kola是与之相关的Web3音乐的可听识别和娱乐实用程序的先驱。

二十五、Orange Protocol

在Web2中，身份和声誉由身份提供商(如FANG)根据其在线服务生成的用户数据进行标记，并存储在其服务器中。在Web3中，身份和信誉数据应该是可移植的、自主的、由用户安全存储的，并且适用于所有在线服务。Orange是一个信誉和信任生成协议，它聚合数据和Web3信誉模型，以可验证凭证和NFT的形式生成全面的信誉证明，它支持一种自主和去中心化的方式在去中心化的组织和服务中使用、建立和增长信誉。

二十六、FusePass

Fuse Pass是一个针对招聘而优化的Web3原生专业身份网络。利用NFT徽章、链上和链下凭证，Fuse Pass为用户提供高度的可组合性来定制他们的专业身份，并在社会合法性和能力证明之间取得平衡。

二十七、AnyDAO

AnyDAO代表了DAO成熟的下一个阶段。随着Web3的实时发展，帮助分散的社区做出真实的决策。

BNB Chain的投资总监Gwendolyn Regina表示:“MVB V是最成功的团队之一，收到了来自世界各地的Web3开发者的700多份申请，涵盖了Defi、基础设施和工具、NFT、Gamefi和安全等不同领域。”

“这些项目得到了BNB Chain和Binance Labs生态系统的强有力指导和支持。这包括通用技术和产品、token经济学咨询、社区和区域市场支持。此外，我们还提供技术建议，并与BNB Chain、CoinMarketCap和Trust Wallet基础设施支持进行整合，这将帮助项目快速增长，并融入整个生态系统。”

Binance Labs投资经理Tyler Z.补充道:“Binance Labs很高兴能共同领导MVB加速器项目，并在过去三个月里通过专门的课程、指导会议和炉边聊天与项目创始人建立联系。第五季是币安实验室第一次选出表现最好的项目进行投资。我们期待看到这些项目进一步发展，并在BNB链上扩大规模。”

七剑下天山

重点关注七个获得资金支撑的项目：

Space ID 正在构建去中心化、抗审查和开源的通用标识符。它的命名服务网络允许用户使用一个人类可读的域名来代表他们在各个DApp和区块链中的身份。每个域服务都围绕以下属性构建：

  • 具有多链互操作性的通用用户名。

  • 单一事实来源 (SSOT) 系统中的去中心化身份。

  • 增强可扩展性和可用性的临时预言机链。

Overeality 正在为Web3的互操作性和可扩展性构建基础架构。它为zkBridge和ZK-rollups加速设计并实现了可扩展的零知识证明系统。Overeality的主要功能和产品包括：

  • deVirgo是一种分布式零知识证明协议，它结合了sumcheck和多项式承诺方案。它在分布式机器上运行并确保更短的证明生成时间。

  • Gemini是一种硬件友好的零知识证明协议，旨在验证区块链的历史。它有助于加快ZK-rollups。

  • 其他产品包括用于去信任跨链通信的zkBridge、可编程NFT协议和用于跨链身份验证的Meta-ID。

Multichain Event Protocol (MEP) 是一种互操作性通信协议。它可以被描述为建立在多链缓存、身份验证服务和执行环境之上的安全相互通信通道。其架构如下：

  • 跨链解决方案是通过用户自定义的子协议和消息来实现的。

  • 用于多链消息传递的所有事件都可以由具有身份验证服务的任何人公开验证。

  • MEP的执行环境提供了一个安全的内核来运行子协议，而不会出现拜占庭故障或事件混乱等问题。去中心化网络是使用委托权益证明(DPoS)和HotStuff作为共识构建的。

Meta Apes 是一款免费和赢取的移动策略游戏。这是第一款在BNB侧链上构建和发布的游戏，全球玩家超过40万。游戏设定在一个后世界末日的世界，人类已经结束，猿人统治的新时代已经开始。游戏的核心特点包括：

  • 建设：玩家将通过管理资源、训练和进行研究来发展他们的城市和军队。

  • 竞争：玩家将进行玩家对玩家（PvP）或玩家对环境（PvE）的战斗，并参与不同的挑战。

  • 社区与探索：玩家将与他们的团队成员合作并制定策略，四处旅行并发现一个充满惊喜的世界。

Velvet Capital 通过帮助用户自动投资于代币化投资组合，使DeFi更灵活、更安全。它支持跨不同项目和生态系统的资产，以确保更广泛的曝光，同时允许用户获得额外收益。其核心功能包括：

  • 简化用户投资代币化Web3指数基金、投资组合和其他具有隐私保护的金融产品的体验。

  • 一种跨链资产管理协议，支持跨多个链创建单一策略或投资组合。

  • 面向机构的“DeFi即服务”（DaaS），可以使用该协议创建自己的资金、策略和其他金融产品。

Web3Go 是一个多链开放数据分析平台，支持链上事件的推送通知。它索引区块链数据并以格式化的方式呈现它们，以便用户可以将数据转换为可视化。其核心产品包括但不限于：

  • 提供用户生成的数据仪表板和自定义仪表板API的市场。

  • 实时通知，为链上活动提供可定制的警报。

  • 跨链Web3 ID以实现额外的社交发现和参与。

Gameta 是一个平台，允许用户在一个屋檐下玩不同的休闲游戏。该团队旨在为用户构建一个流畅的Web3迁移过程。它的吸引力包括：

  • 一个基于BNB链的GameFi项目，拥有相当数量的日活跃用户。

  • 该平台提供九款休闲游戏，并有超过20个正在开发的应用程序。

结语

虽然BSC上频频暴雷，甚至在国庆假期期间出现了重大安全问题导致down机。但是不得不佩服币安的策略，没错，它有的是钱，如何让钱生钱，不断投资项目就是一个方式。但无论如何，如果自家BSC的问题不解决，那Binance Labs最好就单独成为一个系统外的VC机构好了，不然始终让人有一种不靠谱的感觉。
