再次征收NFT版税 零版税先锋X2Y2向Opensea低头

上周，NFT 市场针对版税问题的立场又迎来一个新变化——最早放弃征收 NFT 版税转向“自定义”版税的 NFT 市场 X2Y2 又更改了选择，宣布将强制执行所有藏品的版税，以获得更统一、更流畅的用户体验。

其实，早在 X2Y2第一次宣布更改为“自定义”版税时，就为自己留有后路，并没有将该决定一刀切。其明确表示零版税绝对不是前进的方向，版税政策仍在制定中且最终决定将取决于 Token 持有者，并补充说取消版税并不是 NFT 市场的完美经济模型。

这不，在头部 NFT 市场 Opensea 决定强制执行版税的一周后，X2Y2  紧随其后，“再次”对所有收藏品征收版税。

回顾 X2Y2  在版税争论上的行为，不得不说有点左右摇摆，意志不够坚定。早先，X2Y2 为了与 Sudoswap 竞争市场份额不得不放弃征收版税，而现在他们制定版税的主要原因是许多新的 NFT 系列都在使用 Opensea 的强制执行版税工具，将“自定义”版税的 NFT 市场列入黑名单。而 X2Y2 是被 Opensea 过滤器第一批列入黑名单的市场之一。鉴于 Opensea 占有 NFT 的主要市场和话语权，X2Y2 也不得不选择站队，以换取更多 NFT 系列的交易。事实证明，此次 X2Y2  讨得 Opensea 的欢心，Opensea 在第一时间回复了 X2Y2 的推文表示已经将他们从黑名单中删除。

虽然 X2Y2 被“霸主” Opensea 解禁，但此举激起了用户的一片骂声。有用户表示：“要么讨好用户将免版税进行到底，要么讨好项目方坚持征收版税，现在屈服 Opensea 不得不改回征收版税，两边都得罪，里外不是人。”

其实站在 X2Y2 的角度来看，做出这两个决定都挺困难。从 2022 年 7 月 Sudoswap 上线后，其就靠着低交易手续费和零版税迅速抢占市场，彼时 Sudoswap 的交易量在短短不到两个月的时间内就几乎逼近 X2Y2 。紧接着 Sudoswap 宣布集成至 NFT 交易聚合器 Gem，二者的集成使 Gem 获得了 Sudoswap 的零版税功能，而 Sudoswap 也可以获得 Gem 的流动性，双方的优势体现的淋漓尽致，最终结果就是 Sudoswap 迅速抢占市场份额，将 X2Y2比了下去。面对如此强劲的对手，X2Y2 只能出此下策。

正当 X2Y2以为可以喘一口气的时候，Blur 又杀了出来。自 Blur 上线以来，其凭借着空投激励以及同样的“自定义”版税功能迅速的在「聚合器」赛道站稳脚跟。根据 Dune 的数据来看， 10 月 20 日正式上线后，Blur 的交易量一直领先于 X2Y2。

客观来看，X2Y2的开局也算完美，通过给 Opensea 的用户空投代币，开启挂单挖矿等模式吸引了大批用户。但好景不长，在没有大资金涌入、玩法同质化严重以及各家 NFT 市场厮杀的背景下，X2Y2也需要重新制定竞争策略。既然靠“自定义”版税拼不过 Blur，那么投靠 Opensea 或许还能拼个前三，这也不失为一种新思路。

可是对于用户来说，X2Y2 的“自定义”版税功能一直被看做是其平台的优势之一，那么回归征收版税的制度，其平台的优势又能体现在哪里呢？

针对用户的疑问，X2Y2 的 Community Builder 表示他们正在为这方面做努力，包括聚合器和深化业务的 NFTFi 方面，例如 NFT 借贷，并将它们结合起来以产生协同效应。同时，他也表示 X2Y2 做出这个决定并不草率，因为他们充分了解“自定义”版税对它们市场份额所起到的作用。此外，他们还将加倍努力进行营销并与其他 NFT 项目合作。

X2Y2可以说是一直在做第一个吃螃蟹的人。在 Opensea 强制执行版税的背景下，Blur、LooksRare、SudoSwap 都被列入了黑名单。

近期， Blur 也作出了一点改变——为创作者创建了一个工具，创作者可以使用 Blur 的 Default Operator Filterer，这将允许他们在 Opensea 和 Blur 上强制执行并赚取版税。此外，Blur 也表明将在本月晚些时候对激励版税进行更改，以使其更加有效。鉴于目前的状况，Blur 是否会是下一个跟随  Opensea 脚步的 NFT 市场呢？我们也只能等待官方消息了。

但是不管怎么样，在这场版税大战中，Opensea 目前占据了有利地位，X2Y2 只是用自身的摇摆行为做了场社会实验。
