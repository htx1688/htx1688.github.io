原文作者：Rui, SevenX Ventures

2022 年底我共同发起了 DeSci NYC 社区，在纽约定期举办聚会，内容涵盖从 AI 驱动的蘑菇科学、蜜蜂神经研究到区块链技术。我逐渐意识到去中心化科学（DeSci）对于任何领域的人来说都是新鲜的概念 ，此文将覆盖我所理解的 DeSci 架构、国际最核心的 DeSci 项目、未来的发展趋势。

结语

传统科学研究面临的挑战

长期以来科学研究一直是创新的基石，推动着广泛领域的进步。然而，传统的研究领域一直受到许多挑战的困扰：

  • 门槛极高：学术机构以外的人可能难以参与科学研究。

  • 数据透明度和可重复性低：研究人员难以在现有工作的基础上进行构建或验证已发表研究的结果。

  • 集中资助： 2019 年前 10 名科研资助者占全部资助的 44% 。

  • 同行评审偏见：可能会受到偏见的影响，尤其是来自知名研究人员或机构的偏见。

  • 出版控制：研究人员面临很高的门槛，需要支付高额费用（通常为五位数美元）才能在顶级期刊上发表他们的作品。

随着传统科学“象牙塔”的局限性越来越明显，DeSci 解决方案的紧迫性越来越大。DeSci 使科学家能够以透明和协作的方式访问资源、共享数据、获得资金、进行同行评审和发表研究。这种对传统方法的突破性替代方法正在为建立一个更易于访问、去中心化和包容的科学社区铺平道路。

我将按照以下结构开始 DeSci 之旅：

基础设施

DeSci 项目通常以以太坊区块链为基础，允许人们开发应用程序、进行交易和利用存储。为了优化性能和最小化成本，经常利用 Optimistic Rollups (OP) 和 zkSync 等扩展解决方案，它们处理链下交易并定期将汇总结果提交到以太坊主链。此外，Arweave 等存储协议提供非结构化数据或文件存储，在安全性、可靠性和抗审查性方面都优于传统的web2存储。大多数 DeSci 项目都建立在这个强大的基础设施之上。

如果您不了解区块链基建，请访问以太坊基金会官网。

中间件

数据库：结构化数据格式

人们希望为图形或表单等结构化数据提供一种低成本、高效的去中心化存储解决方案，由于缺乏此种方案，目前开发人员被迫依赖集中式解决方案，最终将导致单点故障、可访问性和管理问题。

DBDAO 是一种创新的数据库结构，它将每一行数据都变成了一个 NFT，贡献者可以在数据库变现时分享收益。此外，DBDAO 将每个数据库都变成了一个 DAO，允许管理员有效地管理访问和审查控制。随着它的发展，我们可以看到使用 ZK（零知识）技术保护数据的有趣潜力，同时确保购买者认识到数据库的价值。

法律条款：智能合约中的协议

通过将具有法律约束力的协议纳入 NFT 的智能合约，各方可以维护自己的利益并遵守适用的法律法规。为了便于访问，这种法律协议应该尽可能让法律专业人士和区块链工程师都能理解。

OpenLaw 正在构建一个技术堆栈，以帮助使用标记的法律模板来支持智能法律协议。这种方法可能对那些喜欢更传统的合同创建过程的人有益，但它可能不具备自定义。

所有权：IP-NFT

建立和规范知识产权 (IP) 的所有权在科学领域至关重要。这可以通过 NFT 来实现，NFT 是区块链上的数字资产，通过增加资产的流动性，将创建动态的 IP 市场。这使得 IP 所有者可以与不同实体之间销售或分割知识产权许，同时确保 IP 所有者获得公平份额的二级市场版税。

Molecule 发布其标准 IP-NFT，是用来验证 IP 资产所有权和真实性的唯一标识符，可以插入法律协议，自定义智能合约功能和碎片化所有权。从技术上讲，元数据存储在 Arweave 上，而 IP 和许可证存储在 Nevermind（集中式数据存储）上，理想情况下，他们应该使用分散式数据存储层（但是现在没有太成熟的）。

隐私：零知识证明

当我们提到开源的重要性时，我们还应该强调研究的隐私控制，DeSci 应该在促进透明度和协作与保证数据机密性之间取得平衡。

零知识证明 (ZKP) 来了，它是一种加密技术，对于去中心化科学研究至关重要，它们可以保护隐私，允许研究人员在不损害数据机密性的情况下相互借鉴，并帮助研究人员遵守隐私法规。

应用层

融资

以 DAO 为结构的去中心化融资已经成为一种革命性的方式，使个人和组织能够直接向他们信任的项目提供资金。它消除了中间的中介，并提供了更高的透明度，让贡献者能够看到他们的资金是如何使用、并有权参与决策过程。

最著名的 web3 筹资平台 Gitcoin 成立于 2017 年，旨在通过赏金支持开源项目并从中获利，该平台基于二次筹资，这是一种民主分配慈善资金的新机制，允许较小的贡献者拥有更大的收益影响。迄今为止，Gitcoin Grants 已经筹集了 50 多万美元的资金。

Molecule 是一个流行的 DeSci 资助平台，专注于生物制药领域，提供了一个创建混合合法智能合约原始 IP-NFT 的协议，将研究人员和资助者之间连接起来，现已拥有 250+ 研究项目，拥有 6 个 Bio DAO。Valley DAO 也使用类似的方法。

Bio.xyz 为 DeSci DAO 提供 launchpad、资源、指导、共享网络以及一套法律和智能合约。他们的目标是使 DeSci DAO 能够利用 IP-NFT 来资助、管理和开发知识产权，从而促进数字 IP 资产的寒武纪爆发。现在生态中有 6 个 DAO。

有很多有趣的细分领域的 DAO。VitaDAO 专注于长寿研究，建立了完整的 token 激励机制。

HairDAO 用于脱发治疗研究；AthenaDAO 对女性健康的贡献；PsyDAO 为 Psychedelics 的早期研究提供小额资助；CryoDAO 资助冷冻保存领域，推动尖端医疗技术；FrontierDAO 专注于空间、聚变能源和开发空间元宇宙。

数据市场

由于数据资源有限且昂贵，许多研究人员面临挑战，这主要是由于 Web2 公司严密保护他们的数据。想象一下开放数据市场的潜力，这是一个大胆的想法：个人能够从他们的数据中获利，同时允许研究人员改进访问权限、促进协作、跨平台的交流。

正如我们上面提到的，DBDAO 是一种创新的数据库结构，其核心思想是让人们能够将自己的数据货币化。在我看来，DBDAO 不仅仅是一个数据库结构，它也是对传统数据贡献模型的重大转变，因为它可以防止研究者在盈利或者融资之前为数据预付费用，同时也用代码的方式保证了数据贡献者的预期收益。

Fleming Protocol 是一种开源数据经济，旨在通过激励患者来推动协作式生物医学，主要关注健康数据所有权。他们为患者提供了将数据存储在 IPFS 中的基础设施，确保他们保持完全的保管和控制。此外，它使患者用户能够将他们的数据货币化，能直接获得数据变现的收益，其中一部分交易费用将贡献给 FlemingDAO 国库。

研究工具

在进行研究时，经常会涉及到各种格式，例如报告、图表、代码和外部 API。我们需要一个多功能去中心化平台来无缝串联研究过程，实现动态的研究发布。

DeSci Node 是一个非常有意思的应用程序，它允许研究人员将手稿中的表格和图形与数据和代码链接起来，并创建一个动态的可互操作的版本记录，从而提升研究过程。Beta 版本于 2023 年 2 月 21 日发布，现在节点存储在 IPFS 上的开放状态存储库中，他们的未来路线图是集成计算以实现无缝发布，将资金与 Gitcoin 和 Hypercerts 集成。

LabDAO 是一个新的研究组织，不仅允许您去许可发布、获得资助而且还可以使用研究工具。这些工具从用于小分子对接、计算基础设施和湿实验室访问的 Notebook 不等。

发布平台

不再有要么发表要么消失。传统的科学出版物及其同行评审系统严重依赖少数主要行业参与者，他们控制大多数期刊、数据库和指标，同时将大多数研究者置于付费墙之后，这对公平性、质量、成本、无偿劳动、透明度无疑是巨大挑战。

The ResearchHub 由 Coinbase 的联合创始人 Brian 共同创立，是一个供研究人员自由分享作品、浏览个性化提要、进行同行评审、发布科学问题悬赏、使用研究工具并获得奖励的平台。他们有一个原生代币 RSC，用于激励内容创建、提示有价值的内容并为社区的关键决策投票。

信誉系统

为了培养一个高质量和值得信赖的社区，DeSci 平台应该建立完备的声誉系统，就像那些 Web2 论坛一样，让具有真正专业知识的个人具有更大的曝光和声量。然而，一个重大挑战是区块链技术固有的匿名性，其中代表用户的钱包地址不能直接链接到他们的真实身份。

首先，我们需要跟踪去中心化平台上的用户交互，例如点赞、评论和发布，以帮助建立可信度和专业知识。其次，利用分散的出版工具连接用户的科学工作，提高透明度和问责制。此外，将链下传统学术身份与链上记录相结合，创建一个无缝的信誉系统，承认并验证用户的证书和成就。

DeSci 社区

到目前为止，我们已经看到来自世界各地的充满活力的社区试图聚集资源和人才来建设 DeSci 的未来。向 DeSci NYC、DeSci Berlin、DeSci Tokyo 等社区致敬。

我如何看待未来的挑战和机遇

很高兴看到 Messari 研究表明，有 85 个 DeSci 项目在现在运行，其中超过 50% 是在过去一年启动的。然而，挑战也随之而来：

1. 熟练掌握代币经济并建立可持续模型

虽然一些平台已经引入代币作为激励，但他们的币价管理非常差。虽然最初可能会吸引一些为了建立声誉的用户，但最终可能会阻碍产品长期参与和增长。完善代币用途、重新评估 token 设计的合理性、建立品牌共识以及发展战略关系等行动至关重要。代币经济学可以是一把双刃剑；为了实现所承诺的自由市场未来，平台或 DAO 必须掌握 token 游戏。

2. 建立 DeSci 时代的新规则

不得不承认，传统的科学系统用中心化把关的方式保持了较高的研究质量。当我们努力创建一个开放的去中心化科学平台时，必须防止低质量和混乱的涌入。我们的重点应该是开发一个稳健的框架，确保可访问性，同时也保证具有真正才能和知识的个人得到适当的认可和奖励。实现这一目标需要实施健全的代币经济学、有效的 DAO 审查机制和基于声誉的优先系统等。

3. 让所有的科学从事者加入

打造适合资本、研究机构、研究人员、行业、期刊和公众的新科学工作流程对于 DeSci 的成功至关重要。DeSci 最初可能作为一种补充方法，在最终取代之前需要与现有框架进行集成和协作。

4. 小心去中心化的黑暗森林

我们必须注意去中心化的优势可以被恶意方利用来违反道德和法律，例如去中心化存储可以用来保存和分发被禁止的研究内容。这是道德问题而不是技术问题，为了保护去中心化的长期价值并确保其被社会接受，我们必须实施由 DAO、资助者或其他机制主导的法规。通过这样做，我们可以维护科学研究的完整性，并确保所有人都能享受权力下放的好处。

结束语

我们常听到对于区块链基建性能的抱怨，但是随着许多人致力于建设，可扩展性、安全性、性能、成本都会得到改善，直到有一天我们不需要为了更好的基础设施而牺牲任何东西。

中间件提供商必须为法律问题、知识产权所有权、隐私和数据存储等建立新的标准。为了实现大规模使用和长期价值，他们还应该着眼于建立行业共识，也许短期内会有很多竞品，但是逐渐卓越者会形成相对稳定的标准。

DApps 可以采用相同的基建标准来实现互操作性、设计好的用户体验以吸引各方、创建可持续的激励模型并根据信仰来设计界面。要知道，产品的设计可以塑造行业的重点，是注重是利润还是声誉？是鼓励大众研究人员还是激励精英？产品的制作方式都会改变游戏规则。

如今我们在世界范围内组织不同的 DeSci DAO，以提高意识、聚集人才、资金和资源进行创新。DeSci 峰会是一个很好的开始，我们还可以走进校园、机构和不同的国家进行进一步的运动。这项任务非常艰巨，需要利用去中心化技术、迁移现有的科学研究人员以及建立新的期刊等等。但随着 DeSci 的扩展，我们正处于一个转折点的风口浪尖。

我相信任何新事物的建立都会伴随着批判和挑战，这也正是我们的目的：去畅想和搭建一个可能更好的未来。
