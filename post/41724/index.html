Move 双子星之一 Sui 生态有哪些项目值得关注

近期，在新一轮公链叙事中，有着 Move 双子星之称的高性能新公链 Aptos 与 Sui，均收获了行业中众多开发者、明星机构和用户的广泛关注。 同时，他们的开发团队大部分来自于原 Facebook，且项目都获得了大额融资，因此两个项目难免会被做一些比较。

不过，两个项目之间孰优孰劣，还未有定论也属于见仁见智了。如 Aptos 势头更劲的同时，有开发者就更看好 Sui，认为它在架构上使用了基于 DAG 的内存池（Narwhal）+ Tusk 共识算法，然后在执行层利用 DAG 进行并行化，这将能有更大发展空间。

此外，在 Aptos 生态蓬勃发展的同时，Sui 在基础设施、DeFi、NFT 等方面也在加紧扩张。目前，Sui 生态中有哪些项目值得关注，BlockBeats 为读者总结如下。

钱包 / 基础设施

Sui Wallet

Sui Wallet 是一个开源的 Chrome 扩展自托管钱包，让 Sui 用户能够创建地址、查看和管理 Sui 网络上的资产以及与 dApp 交互。

随着 Sui 开始支持大规模用例、Move 驱动的所有权模型以及 Sui 独有的其他功能，Sui Wallet 存储库将成为重要的生态系统支持工具。

值得一提的是，Sui 目前上线的还只是激励测试网，用户在体验以下生态协议或应用时，可先到官方 Discord「devnet-faucet」频道输入「!faucet 钱包地址」领取 SUI 测试币。

Ethos

Ethos Wallet 是 Sui 上易于集成、自托管的 dapp 钱包，只需通过电子邮件即可注册。此外，Ethos 还支持 Sui Test Wallet 的导入。

ComingChat

ComingChat 是一个支持全链生态的移动端 Web3 应用，由 ChainX 开发，结合了加密通信、社交、钱包、NFT、去中心化数字身份、全链交换、比特币借贷等多种功能。目前，ComingChat 已支持 Aptos，并即将上线 Sui 网络。

ChainX

ChainX 的最新定位是基于 Substrate 的 BTC Layer2 网络，并同时支持 EVM 生态和包括 Aptos 与 Sui 在内的 Move 生态，旨在让比特币在这些生态系统中广泛可用。

Sui Name Service

Sui Name Service 是 Sui 生态中的去中心化数字身份服务。

Shinami

Shinami 是面向开发人员的 Sui API 和节点服务，注册可获取早期 VIP 访问权限。此外，Shinami 提供了高性能、可动态扩展的基础架构，可用于构建独特的 dapp，而不只是运行节点。

Sui Explorer

Sui Explorer 是 Sui 区块链浏览器。

DeFi

MovEx

MovEx 是 Sui 上首个 AMM + 订单簿混合流动性 DEX，旨在面向 Move 生态中下一个十亿用户。

在 MovEX 内部，有一个在 AMM 和订单簿之间分配订单的结算引擎，既保证了交易者的最小滑点，又保证了流动性提供者的公平。此外，零售流动性提供者可享受以自定义价格范围提供单面流动性的便利，并有效管理无常损失。

而专业交易者和流动性提供者则可享受订单簿提供的高级功能，如果他们的限价单被执行，交易者还可以获得佣金。这是因为限价单被视为制造商，因此可以像流动性提供者一样获得佣金，类似于中心化交易所。

NFT MarketPlace

Blue Move

BlueMove 是为 Move 语言公链而设计的 NFT 市场，目前已支持 Sui 和 Aptos。它的智能合约是在模块化系统中定制，借助明确定义执行范围的标准化签名，可以在不影响安全性的情况下随着时间的推移推出新功能。

这意味着 BlueMove 可以提供新的交互类型：

集合报价，提供涵盖所选集合中所有 NFT 的报价；

特征报价，提供购买集合中具有特定特征的任何 NFT；

多次取消，在一次交易中取消多个未结订单。

此外，BlueMove 以社区为先，奖励平台积极参与的交易者、收藏家和创作者。

获得奖励，在 BlueMove 上购买或出售任何 NFT 的用户可以获得 MOVE 代币；

质押 MOVE 可分享平台交易费用；

创作者在其 NFT 销售时获得版税。

Sui gallery

Sui gallery 是 Sui 上的 NFT 市场。用户通过连接 Sui Wallet 后，可免费 mint 一个 Sui gallery 掉落的 NFT。

NFT Collection

Sui Punks

SuiPunks PFP 是 Sui 上的一个 NFT 集合，发布数量为 2992 个，旨在用来证明该 NFT 持有者是 Sui 区块链生态的 OG 之一。

不过，由于 Sui 主网还有半年多的时间，官方表示想奖励那些更早发现该项目的用户，以及维持该项目的热度和保护它免受相关虚假 NFT 集合的攻击，因而决定先在以太坊上启动 Sui Punks Pass 集合，直到 Sui 主网上线。

此外，Sui 生态中热度较高的 NFT 集合还有：首个 meme NFT Doge SUI，以及首个鸟类 NFT Sui Birds。不过，目前还均未发布。

Gaming / Metaverse

Panzerdogs

Panzerdogs 是一个 NFT P2E 游戏，由荷兰独立游戏工作室 Lucky Kat Studios 开发，游戏已经构建到 Solana 生态。

不过，Panzerdogs 已与 Sui 合作，游戏中玩家的 NFT 可以通过 SuiEco「映射」到 Sui，并作为游戏中的化身，进而组队进行坦克对战。

Sui Monstars

Sui Monstars 是一个宠物模拟游戏，玩家可以喂养、装扮宠物，装饰农场，通过游戏和互动来提高宠物和农场等级。游戏中，宠物、农场、配件都是 NFT，不过游戏目前还未开发。

SoWork

SoWork 是一个职场元宇宙，以 Sui 作为底层链，为 SoWork 中的各种资产（如土地、物品和成就）提供动力。

目前，已有众多的公司在 SoWork 的职场元宇宙中工作，包括 Tinder、微软、谷歌、Facebook 和 Netflix 等的团队。SoWork 正在改变团队合作、协作和在线社交时间的方式。

此外，SoWork 还将在 Sui 上推出两项功能：

公司能根据可定制的标准和可解锁的津贴来奖励员工的 NFT 奖励系统，NFT 基于绩效、价值、事件驱动，可用来赠送、买卖、展示；

用户可通过 Mapmaker 创建、铸造自定义 NFT，同时把 NFT 导入职场元宇宙。此外，团队强调链的性能，铸造和导入几乎能同时完成。
