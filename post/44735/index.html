Web 3在近期得到了许多人的关注，众多企业纷纷布局。除了企业之外，城市之间也展开竞争。例如香港近期就对外宣布将会竞争成为全球虚拟数字资产的中心，此一消息引起了业界的强烈关注。 此前，01元宇宙已发布了《迪拜，沙漠中的Web 3之都》和《纽约逐渐Web 3化》，介绍了迪拜和纽约两个国际城市在Web 3方面的布局。而实际上，美国另一个城市——迈阿密在Web 3领域也是一个领先者，吸引了许多创业者的入驻。

迈阿密市长对Web 3敞开怀抱

现任迈阿密市长为Francis Suarez。Francis Suarez 本人就是比特币的积极支持者，他2021年2月时接受《福布斯》采访时透露，他的部分工资就以加密货币来支付。

2021年8月3日，迈阿密推出了全美第一个城市加密货币MiamiCoin，允许人们通过购买代币来投资城市。从正式发售开始，MiamiCoin一度从0.006美元/枚的价格暴涨至0.05美元/枚，在发布后三个月内为迈阿密赚取了超过2100万美元。不过，目前（2022年11月17日）MiamiCoin的价格已经跌到0.00055美元/枚。Francis Suarez曾公开表示，迈阿密将成为美国第一个将数字货币收益作为股息直接发放给居民的城市，并将通过数字钱包形式进行支付。

Francis Suarez也牵头成立了一个Venture Miami（迈阿密城市风险基金会），争取全美其他地方的科技公司搬到迈阿密、留在迈阿密、并雇用迈阿密当地从业者。迈阿密的初创公司在2021 年共筹集了约30 亿美元，约是 2020 年三倍，特别是其加密领域风险投资从2020年的600万美元飙升至2021年的7.45亿美元，增长了100倍之多。

据Linkedin的统计数据显示，迈阿密的科技从业者的迁入增速在2020年3月到2021年3月期间同比增长了15.4%，在全美各大科技城市中位列首位。在加密技术人才雇佣密度方面，它目前排名全美第四，仅次于旧金山、纽约和洛杉矶。

Francis Suarez曾公开表示，旧金山是为Web 2.0 服务的，而迈阿密想成为Web 3的新硅谷。

多个活动在迈阿密举行

2022年9月，多个Web 3相关活动，例如Bitcoin 2022、CryptoWorldCon、Metaverse Miami 和北美比特币大会都先后将迈阿密选为举办地。

2022年11月28 日至 30 日，Miami Web 3 Summit（首届迈阿密 Web 3 峰会）将会在迈阿密洲际酒店举行。有趣的是，此次活动，迈阿密市政府也是主办方之一。迈阿密市政府与Web 3 基础设施提供商 Atlas 及其母公司 CTH 集团、Web 3 行业专业媒体平台DeFi之道等一起举办本次的大会。

Atlas是世界领先Web 3 基础设施公司，其数据中心遍布全球，为 100 多个协议运行节点。CTH 集团是区块链基础设施、风险投资和数字资产管理领域的领先公司，其著名的投资项目包括 Binance.US、Chainlink、Coinbase 和 Polkadot 等。

此次峰会，将会有许多知名机构参与者在本次活动上分享，探索行业未来的发展方向。迈阿密Web 3 峰会将聚焦有关 Web 3 变革的趋势和热门话题，包括了基础设施和协议，新的商业模式、关键技术对环境、社会和公司治理的积极影响等。

Francis Suarez表示，Web 3 正在改变我们的日常生活，改变全球经济的未来。迈阿密正处于这场变革的前沿。迈阿密Web 3 峰会的举办将帮助迈阿密在未来几年内保持在Web 3 领域的战略前沿地位。

Atlas 创始人兼主席 Raymond Yuan 表示， Web 3 的投资正在迅速增长，生态系统中的开发者、项目方和机构方的数量也在迅速增长。作为一家专注于支撑Web 3 未来的全球公司，Atlas相信将这个生态系统的参与者聚集在一起，探讨如何以可持续和公平的方式塑造行业的发展是非常重要的。除了迅速崛起为Web 3 之都的迈阿密，没有更合适的地方。

Web 3化的迈阿密

“迈阿密现在有一种与众不同的城市气质，就是从走出机场的那一刻起，你就觉得这个城市是属于Web 3的。”一个参加了NFT大会和比特币大会的Web 3创业者如此表示。

机场里可以看到比特币ATM，走出机场则是由路边一个接一个的加密活动广告牌。就连Uber车上都会播放加密大会宣传片等等。城市也有许多墙上画着NFT涂鸦。可见，迈阿密在Web 3的路上越走越远。

迈阿密机场的第一个数字货币ATM

来源：Bitstop官网

一些迈阿密的商店、餐厅开始接受用数字货币付款。除了在机场之外，酒店、商场等很多公共场所的传统银行ATM机器旁边，也开始越来越多的出现一台数字货币ATM。

名为“比特币披萨”的快餐店目前已经在迈阿密开了三家分店。它创立的灵感来源于2000年一位名为 Laszlo Hanyecz的开发者花了 10000 个比特币购买了两张披萨，如今这两个披萨的价值大约在4亿美元。“比特币披萨”成立的目的并不是为了盈利，而是为了支持当地加密社区的发展，它取得的所有利润将100%捐赠给比特币发展基金。

迈阿密当地的很多酒吧和俱乐部也紧追潮流推出了自家NFT，并挂出招牌，凡是购买本店NFT的顾客，酒水全免等广告，希望能够刺激自家NFT的销量。

在迈阿密市中心的Park West区域，建筑开发商 Property Markets Group 和迈阿密知名的会所E11even正在共同合作开发一个地产项目，两座 65 层的建筑将包含住宅、酒店、会员俱乐部、餐厅等各种功能。而此地产项目最大的特点在于，E11EVEN Residences是全美第一个允许客户使用数字货币来购买住宅的建筑项目。数字货币交易平台FTX为这个项目的支付提供了全流程的服务，通过数字钱包的转账确认在5分钟内就能完成。如今，E11EVEN Residences已经以这种全新的形式预售出了几十套公寓。

除了购买方式的变革，客户的看房流程也被Web 3改变。开发商在Wynwood的工作室里创立了一个项目“元宇宙”，用虚拟现实游览取代传统的样板房参观。来访者戴上VR头显之后，他们被送往项目的楼顶，直升机降落在另一个塔顶，私人小屋围绕着主泳池，迈阿密的城市的全景尽收眼底。

2021年6月，全球最大加密独角兽之一的 Blockchain.com把公司总部从纽约搬到了迈阿密。Blockchain.com的首席执行官 Peter Smith表示，迈阿密在Web 3方面已经形成了一个很健康的生态系统，故此而做出此决定。

Peter Smith分享说，他曾在迈阿密的一家希腊餐厅里遇到了一个创业者。在听完了创业者讲述项目构想后觉得很有意思，就给那个创业者进行了两笔大额投资。Peter Smith表示，他也永远不会在伦敦或纽约这样做。但由于迈阿密特别的文化，在迈阿密进行Web 3方面的投资就是一个很正常的事情。

与硅谷相比，做Web 3相关的项目，可能在迈阿密还更容易找到投资。硅谷如今的创业路径很理性规范，为了说服投资人你可能需要准备大量的项目材料，你的团队成员可能必须有名校背景，你需要过五关斩六将，提供足够的证据来证明你的能力。但在迈阿密，你可能就是在某一个聚会上向身边的人分享了你正在做的项目，而他刚好对你所做的领域感兴趣，于是就当即就给你投了。迈阿密目前在Web 3方面都跟20年前的硅谷很像。

低税收的迈阿密

迈阿密的低税收也是一个不断吸引着富豪和年轻人选择到此定居的重要因素（附近的得克萨斯州也是）。相对比纽约和加州的高赋税，佛罗里达一直都是美国人的避税天堂。佛州是美国50个州里面少数几个完全没有州收入所得税的州之一，迈阿密则是美国个人所得税、营业税和销售税第三低的城市。

2021年，全球最大私募股权公司黑石集团在迈阿密开设办事处。其他的例子例如券商古根海姆投资公司首席投资官Scott Minerd从洛杉矶迁居迈阿密，著名硅谷风投Founders Found的合伙人Keith Rabois搬到迈阿密等等。

迈阿密的低税收，吸引了许多富人的同时，也增加了投融资的机会。例如在比特币大会期间，就有许多“大爷”对他们感兴趣的项目进行了投资。

也有人分享，在一个社区活动上遇到了一位大学生模样的年轻人，随意跟他分享了一下自己的创作想法。而那个人听了就立马花了30个以太坊跟他买下了自己的NFT，说是支持一下她的创作。而实际上那个人多年以前就投资了大量的比特币和以太坊，现在早已财务自由。

可见低税收不仅吸引了许多富豪，当中也实际上产生了许多机遇以及机会，让更多的Web 3的创业者前往迈阿密寻找机会。
