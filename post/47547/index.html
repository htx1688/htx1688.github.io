吴说作者：刘全凯

随着 NFT 市场走向 DeFi，围绕 NFT 定价、资金利用率和风险管理的问题日渐突出，并演变成为 NFT-FI 的三难困境，制约着 NFT 金融化市场规模的发展。特别地，很多 NFT-FI 项目在设计之初往往高估了 NFT 在熊市中流动性的糟糕状况，以风险管理为妥协，提供高额的利息率吸引借款人，最终不得不面临坏账的承担。然而，尽管 NFT 正处于寒冬，但 NFT 交易市场却仍在不断冒出，对于 NFT-FI 三难困境的破除、新的流动性解决方案和金融化探索的需求也从未减弱，NFT 一站式平台 Furion 结合了镜像通证、指数通证、AMM、点对池借贷等功能给出了自己的答案。

（利益声明：作者与项目无任何利益关联，本文仅作研究分享）

镜像通证：NFT→F-X Token & 指数通证：FFT

（1）单独的 NFT 标的流动性池

Furion 为每个 NFT 标的提供了单独的流动性池，用户可以在相对应的流动性池子中存储或锁定他们的 NFT 并铸造得到 F-X Token 以此作为该 NFT 在 ERC-20 协议中的镜像通证，其中“F-X Token”中的“X”为 NFT 标的，比如 Azuki 的流动性中得到的镜像通证即为 F-Azuki Token。

如果选择把 NFT 存储在相对应的流动性池中，用户可以获得 1000 枚相对应的 F-X Token。当把这 1000 枚 F-X Token 销毁后，用户可以赎回这个相对应的流动性池子中的任一 NFT，而不是仅能取回初始存储中的那一个。在将 F-X Token 销毁兑换为 NFT 时，将收取 100 枚平台通证 FUR 的固定费用作为平台收入的一部分。

如果选择把 NFT 锁定在相对应的流动性池中，锁定时间由用户在锁定时指定（最长锁定 30 天，但可以获得延期），用户以折扣的方式获得一半的 F-X Token，即 500 枚。锁定还将收取每月 150 枚 FUR 的锁定费用。在锁定期间，这个 NFT 不会被用于交换或者出售，仅有当用户销毁 500 枚 F-X Token 后，可以赎回。如果 NFT 未在锁定期内被赎回，那在锁定结束后，平台将不会在锁定该 NFT 的所有权，并将其释放到公共的存储池中。此外在该过程中，将收取 300 枚 F-X Token 作为惩罚，并将剩余的 200 枚 F-X Token 发送至存储用户。

（2）聚合类似但不同的 NFT 标的流动性池

区分于以 NFT 标的本身作为一个流动性池，同时作为单独池的更进一步，聚合池可以把多个 NFT 标的的镜像通证作为一个池子，以聚合不同的但类似的 NFT 标的的流动性。聚合的标准可能是项目方是共同一家的或者是某个项目的多个仿盘等，比如以 Yuga Labs 旗下的 NFT 标的聚合为池子，可能包括 F-BAYC、F-MAYC、F-BAKC 等。用户把他们的 F-X Token 存入到聚合池并铸造得到 FFT。FFT 是有一揽子 NFT 支持，并代表着所组合 NFT 标的的指数。通过销毁 FFT，用户将能重新取回其组合的 F-X Token。销毁 FFT 将收取 100 枚 FUR 的固定费用。铸造或销毁的数量由 F-X Token 的参考价格和 FFT 的公平价格共同决定。

F-X Token 的参考价格，由 Furion 的定价预言机，归集 NFT 的历史交易数据，并结合时间和空间参数所得；FFT 的公平价格等于 FFT 的总价值除以 FFT 的流通供应量，其中 FFT 的总价值等于其担保物，若 FFT 的流通供应量为 0，则 FFT 的公平价格等 0.01 ETH，这也意味着在首次铸造，1 FFT=0.01 ETH。

针对 F-X Token、FFT、FUR 作 AMM 交换

当把 NFT 变为 ERC-20 镜像通证后，针对 ERC-20 作交易的流动性显然要比 NFT 要高效许多。与一贯的 AMM 交易网格类似，Furion Swap 也以每两个 Token 组合而成一个交易对，然后遵循 x*y=k 的公式有一般的交易者和 LP 提供者。Swap 收取交易量的 0.3% 作为交易手续费，其中 99% 作为 LP 奖励，1% 作为平台收入。

根据不同的 NFT 标的所获得的 F-X Token，可以有无限种组合组成成为 Swap 的 LP Token，这为不同 NFT 对之间的直接交易提供可能。此外，LP 提供者除了交易手续费外，还能将 LP 用于流动性挖矿，以获得平台通证 FUR 奖励。

点对池（Peer-to-Pool）借贷

用户可以通过借出代表不同 NFT 或一揽子的 F-X Token、FFT 或者其他 ERC-20 通证来赚取利息收入，或者将其作为抵押品后进行借款。

与 BendDAO 局限于蓝筹资产借贷不同，Furion 首先将根据 F-X Token/FFT 的价格水平和流动性交易状况，将借贷池以风险低至高分为三层：抵押资产池、跨层资产池和隔离资产池，尽可能覆盖更多地资产。低风险池子中的通证作为抵押品可以从本池以及比其风险更高的池子中借入资产，反过来则不行。另外，根据通证的价格波动和流动性，分配不同的资产的抵押因子，抵押因子最大不超过 0.9，如 USDC 为 0.9，ETH 为 0.85，蓝筹 F-X Token 为 0.6 等。用户的最大可借金额取决于加权抵押品因素，但可以通过质押更多的平台通证 FUR 以提高抵押品价值，每 100 万 veFUR 可以提高 0.1% 的抵押品价值，最高可达 2.5%。

借贷利率取决于 NFT 标的的资产利用率。借款利率=基础利率+资产利用率×给定利率；贷款利率=资产利用率×借款利率，其中基础利率为 3%，给定利率为 20%。

当贷款金额加上所有未来利息超过抵押品的最大金额时，就会发生清算。当清算发生时，清算人最多为借款人偿还一半的借款，并从借款人处获得抵押品价值 5%-10% 的折扣作为激励。在实际清算中，按照借贷池以风险的高低，流动性越好风险越低的资产越优先清算，以降低平台的坏账率，即清算顺序为抵押资产池＞跨层资产池＞隔离资产池。

举一个例子，当 A 用户使用 1000 枚 F-BAYC Token 作为抵押品借入了一定的 F-MAYC Token。此时根据 Furion 定价预言机，若 1000 枚 F-BAYC Token 价值为 80 ETH，F-BAYC Token 抵押因子为 0.6，一定的 F-MAYC 价值为 30 ETH，每月需要支付利息为 0.8 ETH，当 1000 枚 F-BAYC Token 的价值从 80 ETH 下降至 51 ETH 时，贷款加利息（30+0.8=30.8 ETH）超过了抵押物的最大值（51*0.6=30.6 ETH），该 A 用户的 1000 枚 F-BAYC Token 将面临被清算。

此时，如果清算人 B 用户支付了 A 用户一半的贷款利息和（30.8/2=15.4 ETH），将可以获得价值（15.4*1.05=16.17 - 15.4*1.1=16.94）的 F-BAYC Token 抵押品。与此同时，A 用户的抵押品价值也恢复至（51-16.17=34.83 ETH - 51-16.94= 34.06 ETH）大于贷款利息和 30.8 ETH，债务恢复至健康状态。

此外，对于抵押资产池中的被清算资产仍有一个 24 小时保护程序。在清算后，用户仍可以在 24 小时内以清算价格的 1.2 倍对其资产进行回购。那么回到上述例子，（16.17 ETH，16.94 ETH）价值的 F-BAYC Token 仍会被暂时锁定在平台中 24 小时，只要在 24 小时内，A 用户支付（16.17*1.2=19.404 ETH - 16.94*1.2=20.328 ETH）便可以取回该部分的 F-BAYC Token。

其他

Furion 有一套自己的预言机规则，通过收集所有 NFT 市场的历史交易来构建 NFT 的价格曲线，简单而言，即是在限定时间内发生的 N 笔交易历史中，有大部分的交易价格落在某个价格区间内，那么便取该价格区间的下限作为 F-X Token 的参考价格。

此外，Furion 也引入了 veToken 模型，用户可以质押 FUR 获得 veFUR 从而享受治理、平台费用收入、提高借贷利用率等。

目前，Furion 主网预计于 2023 年第一季度上线，项目仍处于十分早期的阶段，而整个 NFT 市场陷入寒冬，市场萎缩低迷，NFT-FI 市场容量实则更小。对于蓝筹 NFT 而言，数量上限是可以预见的，需不需要那么多的借贷市场也存未知，竞争对手亦在不断发展中。此外，在 BendDAO 的大规模清算事件中，也不难发现蓝筹 NFT 在熊市中的流动性仍要比预期中的要差得多。尽管 Furion 以流动性、价格等对镜像通证划分类别，但以蓝筹资产镜像通证组成的抵押资产池也未必能在熊市中幸免，而对于资产净值小但数量更庞大的跨层资产池和隔离资产池，一旦遭遇恶性的流动性枯竭，恐将也很难避免承担连环穿仓导致的坏账。此外，如果没有更高的借贷利率，如何突破 BendDAO 的先发优势，在项目上线后吸引蓝筹用户过来质押借贷也是不小的难题。
