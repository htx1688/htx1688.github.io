在本文中，让我们来探讨正在构建的 NFT 的真实用例，以及它们将来的用途。

了解炒作周期

你听说过盖特纳炒作周期吗？它表明，颠覆性技术需要经历 5 个关键阶段：

1. 技术触发：潜在颠覆性技术的出现。

2. 期望膨胀的顶峰：早期的宣传创造了炒作。（例如：2021 年牛市泡沫）

3. 幻灭的低谷：早期项目未能兑现承诺，公众失去兴趣。我怀疑 NFT 现在处在这个幻灭的低谷附近。

4. 启蒙的斜坡：技术慢慢开始展示成功的用例。

5. 生产力的高原：主流采用已经到来，更广泛的市场应用已经证明了自己。以下是来自维基百科的 Gartner 炒作周期的更详细的视觉效果。

在 2021 年的投机性 NFT 泡沫中，@beeple 以 6900 万美元的价格出售了他的作品拼贴画。同样，@nbatopshot 也卖出了数亿美元的收藏品。这可能是人们期望膨胀的顶峰。

现在，让我们深入了解 NFT 的真实用例。

数字艺术

大多数人认为这就是 NFT 的全部 - 富人购买毫无价值的 JPEG 和 MP4 的投机泡沫。

但是，拥有一个购买，出售和拥有数字艺术品或收藏品的平台对创作者和粉丝来说是变革性的。

NFT 可用于表示唯一、具有所有者且需要所有权数字证书的任何内容。示例：域和用户名。

域和用户名

@unstoppableweb，@ensdomains、@rarible 允许用户购买 NFT 域名。NFT 域名的一大好处是它们是可转移的。Godaddy 和其他 web2 提供商上的域名不容易转移或出售。在许多情况下，用户只是租赁域名而不是购买域名。

门票

同样，演唱会门票和其他活动通行证也非常适合由 NFT 代表。它们的数量有限，访问事件需要证据。

NFT 可以消除伪造问题，并易于验证真实性和所有权。

NFT 门票也可以在二级市场上交易，这可以解锁一些东西：

具有买家和卖家保护的标准化市场（目前，门票在 FB 和 craigslist 等低效市场上交易）

免费市场定价

原始创作者的版税

4. 门票的历史所有权数据意味着表演者可以将未来的通行证，折扣，赠品等空投给粉丝。

5.NFT 通行证也可以作为徽章来展示用户的粉丝圈。

在线票务市场估计价值 300 亿美元以上。而且它正在快速增长。以下我见过的一些基于 NFT 的票务项目：

@GetProtocol

游戏资产

出于许多相同的原因，游戏内资产受益于 NFT。想象一下，有人花了 5 年的时间积累了一把罕见的游戏内收藏剑，然后长大或退出游戏。该收藏品显然对游戏玩家有价值。

游戏行业预计今年将产生 2000 亿美元的收入，其中很大一部分来自游戏内购买。

这些游戏资产的二级市场交易的版税为游戏公司创建基于 NFT 的生态系统提供了强大的动力。但数字资产仅仅是个开始。现实世界的资产可以从链上 NFT 表示中获得许多相同的价值。例如：房地产是唯一的，有所有者，并且需要所有权证明。

房地产

标记化房地产具有多种好处：

1. 可分拆，增加准入门槛以及流动性；

2. 可以抵押以提高资本效率和获得由链上资产支持的贷款；

3. 允许投资者在特定社区，城镇或城市进行多元化下注；

到目前为止，我们只讨论了 NFT 对可转让资产的好处。但是，不可转让的 NFT 呢？

这是一个人们现在称之为灵魂绑定代币或 SBT 的概念，Vitalik Buterin（以太坊的联合创始人）在他的博客中写道。就像我说的，NFT 本质上是任何人都可以验证的数字证书。

学位和文凭

这非常适合像学位和文凭这样的用例。当然，用户不希望这些是可交易的，因此它们可以成为不可转让的 SBT。在链上拥有标准化的公开文凭，学位，技能和成就证明意味着任何人都可以验证其合法性。

所有其他类型的证书或成就也是如此。例如：LinkedIn 可以为用户的学位或技能提供经过验证的复选标记。

防伪保护

同样，NFT 也可用于防伪保护。

假冒是世界上最大的犯罪企业，估计每年 2 万亿美元，而且还在不断增长。

因此，防伪技术有很多价值。

@ORIGYNTech 正在为此而努力。

身份

说到伪造，身份盗用 / 验证是另一个巨大的现实世界问题，可以使用 NFT 来解决。身份 NFT 如何解决这个问题？

NFT 是独一无二的，无法伪造。

NFT 提供了全球标准。

NFT 可以很容易地验证。

不可转让的 NFT（SBT）被锁定到特定的钱包。

如果钱包丢失或被盗，NFT 可以被撤销。

这可能是 NFT 最大的用例之一。

想象一下，一个跨国家标准化的全球身份标准，不能伪造或被盗，是数字化的，易于验证，并保护您的私人详细信息。

由于用户的身份不仅仅是单一的政府 ID，因此可能会有多个身份 NFT。

@0xPolygon 和 @civickey 正在研究链上身份等。

会员

数字和实体会员资格也可以使用 NFT 来验证访问要求。

投票

NFT 身份还可以解决投票验证等问题。

如果你还记得 2020 年美国大选，就没有必要解释为什么这是一个值得解决的问题。在线投票的简单性也可以增加选民投票率。

知识产权

NFT 也可用于保护知识产权。

这可以让创作者从他们的创作中获得版税。

NFT 在这里有 2 个关键属性：

IP 所有权→可验证性是明确定义的 / 可公开验证的。

标准化→为平台打开了大门，让创作者从他们的 IP 中赚取版税。

内容权利

没有版权的货币化 = 每个人的更多机会。

音乐是一个非常明显的用例。

创作者目前被搞砸了，从 Spotify 和苹果音乐等平台上赚的钱很少。

众筹

创作者也可以使用 NFT 进行众筹。

例如，投资者可以获得代表未来特许权使用费权利的 NFT。

这对于不在前 1% 的人无法赚钱的领域特别有用。（例如：职业运动员）

Mirror.xyz 通过博客文章实现众筹。

金融国家联络点

这就引出了金融 NFT（fNFT）的更广泛概念。有很多金融合约是独一无二的。以下例子：

个人的一篮子资产（独特的投资组合）；

与贷款人达成的债务协议，已部分偿还；

时间锁定代币；

法律合同

但这并不一定仅限于金融合同。一般来说，任何法律合同或文件都可以从 NFT 代表中受益。

电子邮件和消息

但为什么要停留在法律合同上呢？通过电子邮件和消息达成的口头协议也是独一无二的，但作为一种验证手段，它们很容易丢失并且可以伪造。

病案

医疗记录或处方是另一个对验证要求很高但缺乏适当系统的文件示例。

可以通过 NFT 表示的医疗文档示例：

疫苗接种证书

新冠病毒检测结果

处方

与身份相关的医疗状况
