原文作者：William M. Peaster，由 Odaily 星球日报译者 Katie 辜编译。

借助增长中的 DeFi 和 NFT 场景，Arbitrum 已成为领先的以太坊 L2 扩展解决方案。 而 Treasure DAO是 Arbitrum 上的“招牌”去中心化 NFT 生态系统，也是实现“Web3 任天堂”这一愿景所需的工具。

Treasure 最新官宣的的 NFT 工具——MagicSwap V2“有可能彻底改变 Web3 游戏经济”，下面，Odaily星球日报就带大家了解一下 MagicSwap V2 的亮点及其对整个 Web3 游戏生态系统的增长的影响。

MagicSwap V2：Web3游戏的改变者？

MagicSwap 是一款用于创造沉浸式游戏体验的工具，能够让用户专注于游戏而不是资产价格。 许多 Web3 的批评者声称，加密游戏已经“过度金融化”。MagicSwap 允许在不消除区块链优势的情况下减少投机性体验（例如，可转让的资产，允许熟练的玩家因他们的成就而获得奖励）。MagicSwap 在 GameFi 和 SBT 经济之间建立了最佳的中间地带。

Treasure 的新 MagicSwap 目的是为其生态系统中的玩家和游戏创造者提供更大的灵活性。Treasure 刚刚宣布计划在 2023 年第二季度推出 MagicSwap V2 NFT 自动做市商（AMM）。Treasure 设计了新改进的交易方式，使其成为加密货币中第一个通过单个路由器支持 ERC-20 和 NFT 池的 AMM，并允许游戏经济中所有项目的交易。

MagicSwap V2 的重要性

在此之前，对于同质化代币和非同质化代币，还没有共享的去中心化交易基础设施。例如，你现在可以去Uniswap，像 MATIC 一样交易 ERC-20 代币，或者像“胖企鹅”姐妹项目 Lil Pudgys 一样实现 NFT 的跨链。然而，底层基础设施目前是独立的：ERC-20 代币交易在 Uniswap 的传统 DEX 协议上，而 NFT 交易在市场聚合器上从 NFT 聚合市场 Genie 演变而来。

当然，Uniswap 已经表示，它计划在 NFT 和 ERC-20 代币之间创建“更多的互操作体验”，所以预计 DEX 巨头计划在不久的将来在某种程度上引入通用代币兼容性。

让 MagicSwap V2 引人注目的是，它是我们所知道的第一个确认的通用 DEX，并且它是为 Web3 游戏定制的，这是 Treasure 的专长。

如何解决链游开发者面临的痛点？

Treasure 正在构建一个“去中心化的任天堂”，这是一个链上游戏的集合，通过游戏内部资产的互操作性和 MAGIC（Treasure 的原生代币）作为共享货币相互连接。其中最受欢迎的游戏包括策略游戏 Bridgeworld、角色扮演游戏 The Beacon 和 Smolverse，以及资源管理和策略游戏 Realm。在这个生态系统中，许多不同的游戏都有自己独一无二的经济规则，可以促进共同发展。为此，新的 MagicSwap 被设计成一个灵活开放的游戏经济资源，允许许多项目使用。

除了我们已经讨论过的通用代币兼容性基础设施，MagicSwap 还将通过 ERC-1155 代币提供 NFT-to-NFT 的池、基于 NFT 的 LP 奖励、动态价格调整，自动版税和对 NFT 进行批量购买（Sweep）的功能。

在游戏方面，MagicSwap 将提供一系列新的可能性，包括：

  • 池许可选项——游戏创造者可以选择如何从池中交易代币，从而更严格控制的游戏经济+减少过度投机交易；

  • 基于空间和地理的交易将成为可能，因此游戏可以使用 MagicSwap 来拥有游戏内特定位置的市场；

  • MagicSwap 的自定义池选项将允许游戏创造者为玩家交易“不同代币的网络”提供便利，因此，可以调整游戏的哪些部分可以和不可以进行代币化。

有的 Web3 游戏将其游戏内资产变得过于容易交易，从而缺乏经济杠杆，面临与经济外流和过度投机作斗争的难题，这些问题可能会成为“游戏杀手”。因此，MagicSwap V2 如此引人注目的原因之一是它为 Web3 游戏开发者提供了一个开放的资源，可以根据需要灵活地管理他们的游戏经济。

Magicswap 启用 GameFi 的反向功能。金融化只存在于特定类型的游戏项目中，即有价值的点对点出售的 NFT。游戏货币用于游戏玩法和铸造/升级具有经济价值的 NFT。

今年早些时候，我对 2023 年 NFT 游戏的一个愿望是看到更多“无代币游戏”，在这些游戏中，只有 NFT，而不是 ERC-20 。我认为，使用 MagicSwap 启动 NFT-to-NFT 池的能力无疑是朝着这个方向迈出了一步。正如 Treasure 团队在其最新公告中指出的：

“例如，游戏允许玩家交换各种资源，但不能兑换成美元。如果玩家能够创造方法去升级自己的角色或制造稀有道具，游戏便能够包含一定程度的金融化内容。这些稀有道具可以在 Trove 市场上以点对点的方式出售，从而提高可收藏的交易体验的经济化，而不是只有廉价的游戏小道具。”

未来 MagicSwap V2 值得关注的三个重点

随着 MagicSwap V2 的到来和 Treasure 生态系统的持续崛起，在接下来的几个月里，这一领域有三件事情值得关注：

  1. Treasure 的原生代币 MAGIC——MAGIC 是 Treasure 和 MagicSwap 的治理和费用代币，它也是目前在 Arbitrum 上交易最多的代币之一，作为早期 Arbitrum 游戏领域的重点，今年值得重点关注。

  2. MagicSwap 的第一个游戏实验——NFT-to-NFT 池和其他自定义选项的 MagicSwap 将为更复杂和基于技术的 Web3 游戏经济铺平道路，从而提供更丰富的游戏玩法。

  3. 有哪些链值得关注？Treasure 表示，“计划先将 MagicSwap 部署到 Arbitrum One 和 Arbitrum Nova，然后再部署到其他链；例如，我也有兴趣在以太坊上看到这个基础设施，关注 MagicSwap 将在哪里推出，因为它可以在任何 NFT 交易的地方使用。”
