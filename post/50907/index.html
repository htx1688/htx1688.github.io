比特币的“BRC-20”爆炸让用户争先恐后地选择闪电网络

一位幸运的矿工在周日晚上赚取了 6.701 比特币 (BTC) 或近 200,000 美元的交易费用，超过了比特币当前的 6.25 BTC 的区块补贴——这一极为罕见的事件说明了最近与 Ordinals 协议相关的区块链活动爆发如何导致用户成本飙升。

去年年底，Ordinals 引入了“铭文”或文本或图像等任意内容，可以将其添加到按顺序编号的 satoshis 或“sats”——比特币中的最小单位——以创建独特的、不可替代的代币或 NFT。Ordinals 现在也被用于铸造 BRC-20，就所有意图和目的而言，它们都是可替代的代币。

所有这些新代币都迅速流行起来，但成本很高。他们已经让比特币网络变得拥挤，并将其推向了运营极限，为世界上第一个也是最大的区块链如何应对快速扩展的需求提供了一个真实世界的测试。

发送硬币的费用飙升，交易排队等待的时间更长，一些用户和交易所已经开始考虑替代方案。其中包括闪电网络，这是一种所谓的“第 2 层”扩展解决方案，旨在使比特币交易更快、更便宜。

“谁能解释一下我将如何用这些费用来招募员工？”比特币教育家兼公平比特币创始人 Anita Posch发推文说。“我主要是在非洲招聘员工。他们没有像您这样的特权来支付这些高额费用。他们真的需要 BTC，而你只是在玩。”

BRC-20 是 JavaScript 对象表示法 (JSON) 数据的铭文——跨不同平台移植数据结构的代码片段。由于 JSON 铭文实际上是代码，因此可以对它们进行编程以生成大量代币供应——一次一小块大小的批次——通过不可替代协议有效地创建可替代代币。

这种凭空铸造大量代币的能力是歇斯底里背后的原因——而且，对于区块链的用户来说，是焦虑。最近的区块充满了 BRC-20 交易，每笔交易的费用徘徊在 20 美元，比 2022 年大部分时间每笔交易 1-2 美元的典型费用高出 800%。

费用激增给整个比特币生态系统带来了冲击波，导致世界上最大的加密货币交易所币安因费用计算错误而暂时停止比特币提款，并在非洲和拉丁美洲的某些地区引发了沮丧和痛苦，那里的一些居民每天都依赖比特币付款。

比特币的高额费用对依赖底层支付的用户产生了不可否认的影响。这就是闪电网络的用武之地。闪电网络是一种比特币第 2 层扩展解决方案，其中一组互连的计算机将比特币支付路由到链外，从而实现更便宜、更快速的交易。

币安表示，鉴于比特币费用的突然飙升，它正在整合闪电网络。但即使是闪电网络也需要初始链上交易才能建立支付渠道，而 Posch 表示成本如此之高，以至于现在许多人都无法进行单笔交易。“不能在链上使用，不能打开通道，”Posch 发推文说。“使监管闪电成为唯一的选择。而这一切都是因为有些人认为破解比特币很有趣。”

在该链的 NFT mintfest 之后，关于打破比特币的最后一点一直是 Crypto Twitter 上反复出现的主题。 虽然 Posch 暗示喜欢迷因的黑客正在测试比特币的极限，但其他人则采取了更阴谋的解释，在没有证据的情况下暗示 BRC-20 现象实际上是对世界主导区块链的协同攻击。
