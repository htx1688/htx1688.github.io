在哥伦比亚波哥大举行的第一届 Arbitrum 黑客马拉松已经正式落下帷幕，以下是赢得 Arbitrum 赛道和合作伙伴奖励的所有团队的完整阵容。

合作伙伴赏金

Livepeer奖金得主

1. ArbiGates - 3500

扩展 Gates.wtf，以便在 Arbitrum 生态系统中根据条件轻松设置权限。

2. DeckStream - 1000

LiveStream 和课程平台。

3. Wet Papaya Club - 500

Wet Papaya Club 将通过其 NFT 集合开发一个服务系列，NFT 持有者可以在迈阿密的 WET PAPAYA 派对上提供帮助，并通过 livepeer 看到整个活动的直播。此外，回收的一部分钱将捐赠给拯救海洋，用户购买 NFT 后 Wet Papaya Club 将种植一棵带有独特二维码的木瓜树来识别它。

QuickNode 奖金得主

1. X-scanne r - 2500

Solidity 集成开发环境（IDE）的自动化 CI/CD 管道，它将检查所有推送到 GitHub 存储库的 Solidity 源代码是否有明显的可能导致漏洞利用的不安全的编码模式。

2. Arbistatements - 1500

使用 ZK 证明在 Arbitrum 上获得加密声明。

3. AA Auto Bridge - 1000

启用账户抽象的跨链桥接与 Hop 协议。用户可以在没有原生代币的情况下桥接代币，用户可以批量发送 Tx，包括批准和转移，还可以自动交换过程。

Synapse Protocol奖金得主

1. Bridges&Widgets - 1500

Bridges&Widgets 旨在帮助用户和开发者以更简单的方式进行桥接操作，用户既可以交易也可以桥接代币，同时还有分析功能。

2. bridgoooor stats - 2000

通过访问协议本身和顶级用户的数据，使用户能够更好地了解 Synapse。

Chainlink Labs奖金得主

1. ArbiGates - 1000

扩展 Gates.wtf，以便在 Arbitrum 生态系统中根据条件轻松设置权限。

2. Arbistatements - 1000

使用 ZK 证明在 Arbitrum 上获得用户的加密声明。

3. SBTn - 1000

新类型的 SBT 连接到 NFT 而不是钱包，具有多种使用案例，如游戏、元宇宙等，增加 NFT 的附加值。

Hop Protocol奖金得主

1. AA Auto Bridge - 2500

启用 Hop Protocol 支持账户抽象的跨链网桥，用户可以在没有原生代币的情况下桥接代币，用户可以批量发送 Tx，包括批准和转移，还可以自动化交易过程。

2. Hop& - 2000

在我们的协议中，你可以从 L1 桥接到 L2。我们将 Hop 合同与本地 Arbitrum 信息传递包装在一起，实现了惊人的用户体验。我们的第一个产品是 Hop&Swap，它使你能够在一次交易中桥接和交换资产。

3. Bridges&Widgets - 500

Bridges&Widgets 旨在帮助用户和开发者以更简单的方式进行桥接操作，用户既可以交易也可以桥接代币，此外还有分析。

Uniswap Foundation奖金得主

1. Toboro - 2500

为 Web3 修复图形和警报。

2. PowerUni Lottery - 2500

Poweruni 是一个协议，它允许 Arbitrum 上的小储户通过 Uniswap V3 赚钱 通过在 Poweruni 池中存入代币，储户有机会赚取 Poweruni 池从交易中获得的所有费用。

Vesta Finance奖金得主

1. Flash liquidator - 5000

Vesta protocol 的清算智能合约，通过从平衡器中闪贷 VST 代币并存入稳定池，从清算中获得更多收益。

Treasure DAO奖金得主

1. Stratagems - 1000 枚 MAGIC 代币、1 个 Bridgeworld Auxiliary Legion、2 个 Treasure NFTs、1 个 Smol Brains Pet

一个完全在链上的无限棋盘游戏，持续的、无权限的，4 种颜色的棋盘竞争的控制。

2. SBTn - 1000 枚 MAGIC 代币，1 个 Bridgeworld Auxiliary Legion、2 个 Treasure NFTs、1 个 Smol Brains Pet

新类型的 SBT 连接到 NFT 而不是钱包，具有多种使用案例，如游戏、元宇宙等，增加 NFT 的附加值。

3. ArbiGates - 1000 枚 MAGIC 代币，1 个 Bridgeworld Auxiliary Legion、2 个 Treasure NFTs、1 个 Smol Brains Pet

扩展 Gates.wtf，以便根据 Arbitrum 生态系统内的条件轻松设置权限。

Dopex奖金得主

1. d0pb0t - 5000

d0pB0t 是一个用 Python 编写的套利机器人，其目标是在 Dopex 和 Deribit，以及 Dopex 和 Lyra 金融之间寻找套利机会。

Hidden bounties

Sell-Your-Soul Wallet - 2500

Sell-Your-Soul Wallet 是一个可交易的智能合约钱包。每个钱包都是一个智能合约，其所有权表示为 ERC721 NFT，能够解释任意的合约调用。使用它，用户可以出售任何代币化的头寸，链上游戏状态、未来空投的权利，以及任何与 ETH 地址（即 SBT）相关的信息作为 NFT。

d0pb0t - 2500

d0pB0t 是一个用 Python 编写的套利机器人，其目标是在 Dopex 和 Deribit 以及 Dopex 和 Lyra 金融之间寻找套利机会。

Toboro - 2500

Hop& - 2500

在该协议中，用户可以从 L1 桥接到 L2，该协议用本地 Arbitrum 信息传递来包装 Hop 合约，其第一个产品是 Hop&Swap，它使用户能在一次交易中桥接和交易资产。

小结

此次是 Arbitrum 举办的首界黑客松，从中不仅看到了非常酷的项目，而且也更加理解开发者想从黑客马拉松中得到什么， 在下一届黑客马拉松中（日期待定），Arbitrum 计划在活动中和活动后提供更多的机会，让开发者感到他们不仅是在为奖金而参与，而是有可能在 Arbitrum 生态系统中建立自己的网络。
