关键要点：

  • 混合型应用链的兴起是公链发展的必然趋势，不仅能够提高公链的可扩展性和安全性，还能够为不同场景提供更灵活、高效的解决方案。在这个过程中，安全性将是公链发展的关键因素，而Interchain Security (ICS)等新技术和解决方案将为公链打下坚实的基础。

  • 因为在Cosmos上构建的区块链往往很难启动自己的验证器集。ICS 旨在解决这个问题，将使项目能够利用Cosmos Hub 的经济安全性。展望未来，在Cosmos上构建一个区块链将越来越便宜，同时也会像在WordPress上构建网站一样容易。

  • 这些链既能够作为Cosmos巨大生态系统之间的流动性桥梁，也可以象征性地连接不同的公链社区并将它们拉近。随着这些团队逐渐打破界限，未来开发人员将能够轻松选择他们想要构建网络的方式，而无需做出巨大妥协。

公链世界日益庞大，用户和流动性带来了交易量和使用率，也让区块空间变得稀缺和昂贵。这使得超级应用链（Super Appchains）的叙事变得具有吸引力。诸如以太坊这样的区块链正在转型为更庞大的结算层，而大多数经济活动将发生在构建在其上的许多不同的L2/L3链上。类似的情况正在Cosmos上演，Cosmos Hub 正努力成为其他应用链的规范安全层。

本期Bing Ventures研究文章将从公联发展的逻辑出发梳理Cosmos应用链的优势和生态机遇。我们认为，安全性将是Cosmos生态进一步发展的关键因素。无论是作为规范安全层还是作为应用链，公链的安全性都是不可妥协的，而Interchain Security (ICS) 的提出为安全性提供了新的解决方案，同时混合型公链的架构也为Cosmos上应用链的安全性提供了更多保障。这让Cosmos应用链未来可期。

公链发展的逻辑

随着各大公链的不断发展，目前市场上已经出现了一些大型的生态系统，比如Ethereum以及其各种L2和侧链、Cosmos上的Appchain、Avalanche及其子网、Polkadot和Kusama的平行链等。每个这类生态系统中都包含着大量的链，比如，Cosmos生态系统中就有超过50个Appchains。这些链通过区块链间通信（IBC）相互连接，构成了一个庞大的网络。在这一背景下，公链发展的新机遇也随之而来。

新机遇的出现

Cosmos是一条混合型公链，其应用链有着广泛的机遇和优势。Cosmos基础设施已经准备好通过IBC生态系统横向添加新链来吸引新的应用程序。这意味着开发者可以更加容易地创建新的应用链，通过与其他链进行互操作，形成更为复杂的生态系统。其次，通过资产发行链（如USDC）以及通过IBC连接到其他生态系统，Cosmos可以吸引大量资金流入，促进生态系统的发展。此外，在未来一年中，由于其他生态系统继续遭受桥接漏洞的困扰，Cosmos将有更高的几率增加其市场份额。

类比于人们在过去选择聚居地的方式，即人们倾向于聚集在贸易路线的交叉点上进行价值交换，我们可以预见，在Cosmos上的应用链领域，也会出现类似的机遇，即通过在各大生态系统之间构建交叉点，来实现流动性的聚合。这样，一系列新的产品也将随之涌现，如稳定币兑换、套利、MEV捕获、从两个乃至多个生态系统汇聚收益的DeFi协议等。

ICS是基石

Interchain Security，或 ICS，但是现在，团队也把它称为 Replicated Security。它是Cosmos中的一个重要进化，让应用链能够从Hub的验证者生态系统中受益。它可以让其他区块链放弃自己的验证者，并使用Cosmos Hub的安全性和验证者集合。 这样，其他区块链就可以节省成本和时间，获得Cosmos Hub提供的可靠和稳定的网络保护。

ICS 也可以让Cosmos Hub和其他区块链之间建立更紧密的合作关系，形成一个ATOM经济区（Atom Economic Zone）。 比如，使用Replicated Security的区块链（消费者链）可以向Cosmos Hub（提供者链）支付一定比例的代币或费用作为安全服务费。 这些收入可以分配给Cosmos Hub上质押ATOM的验证者和委托人，增加他们的收益。 同时，消费者链也可以从Cosmos Hub上质押ATOM的验证者和委托人那里得到更多的关注和支持。

ICS的推出对于Cosmos带来了新的机遇。ICS使得应用链从一开始就可以获得Cosmos Hub提供的安全性，而不需要经过数月的验证者测试，从而为应用链的开发和建设提供了便利。ICS能够为Cosmos带来更大规模的发展，它使得Cosmos Hub可以与消费者链达成协议，从而创造收入。这些收入可以用于生态系统的发展，或者分配给ATOM持有者。

Source: Informal Systems

ICS的推出对于应用链也带来了巨大的价值。应用链可以从Cosmos Hub提供的安全性中受益，而且在共识和数据可用性方面仍然拥有高度的主权。此外，ICS还能够为应用链带来更多的商业机会，例如交易费、MEV捕获、区块空间拍卖和收入分成等。总结来看，我们认为Cosmos应用链的机遇和优势主要在于以下三点：

  1. 基于ICS的应用链启动快速，可以更容易地启动新链；

  2. ICS允许新链租用Cosmos Hub验证者和质押者的安全性，appchains就不需要自己建立或租用验证者集合，这样就降低了新链启动的安全成本和风险；

  3. 跨链互操作：Cosmos可以利用ICS来让appchains之间实现无摩擦的资产转移、逻辑调用和事件通知。 ICS可以让appchains使用不同的共识算法、智能合约语言，而不影响彼此之间的通信和协作。 这样，appchains就可以专注于自己的核心功能，并享受其他appchains提供的服务和价值。

Cosmos应用链的优势

Appchains应用链是Cosmos公链的一个重要组成部分，其核心是限制状态的访问。与允许任何人无需许可地启动新应用程序的通用链不同，Appchains通过协调来决定它们将运行哪些应用程序。同样，跨链通信也需要通过IBC建立连接。

这里存在一个固有的权衡。Appchains失去了无需许可创新的敏捷性，但换取对其应用程序的主权。此外，它们可以优化其链以解决特定的用例，从而提供更可靠的用户体验。应用链的另一个重要优势是激励。在通用链上，收入通常泄漏到底层的gas代币中。即使是最受欢迎的dApp，也难以捕捉其产生的大部分收入。在混合型公链赛道中，Cosmos应用链具有巨大的机遇和优势。最大的推动因素之一是Cosmos是为数不多能够轻松启动新链的基础设施。

混合型公链生态

在这种趋势下，一些其他Layer1公链涌入Cosmos的新的项目也应运而生。比如Kusama上的L1网络 — — Picasso Network，由Composable Finance建造，旨在成为一个跨链枢纽，吸引来自更大的DeFi领域的流动性。Picasso通过其名为XCVM的新型跨链基础设施层来实现这一目标。它是波卡生态系统中首个通过IBC连接到Cosmos的链，也将成为这两个生态系统之间的桥梁。

另一个项目是Avalanche子网Landslide，它通过IBC的帮助，旨在将Cosmos生态系统迁移到Avalanche，并缩短Tendermint共识的最终性时间。该链将运行在基于COSM/WASM SDK的开源代码上，这使得任何基于Tendermint的DApp都可以在AVAX生态系统内本地运行。这将结合AVAX共识的高吞吐量和Rust构建的强大VM，从而在与互操作性相关方面，相较于EVM更加适用。

Sei Network的团队也意识到了这一点，他们正在开发Nitro项目，这是第一个建立在Cosmos上的Solana虚拟机链。Nitro让开发者可以无需改动现有的Solana智能合约，而用户则可以使用Solana钱包轻松地访问这些应用。Nitro实际上是Solana和Cosmos之间的网关，通过它可以实现两个公链之间的跨链交互。

跨链协作实际上是基于当前公链发展的趋势和需求。Solana作为高性能公链，其应用场景非常广泛，但是在其他方面可能存在一些瓶颈，而Cosmos作为一个跨链协作的公链则可以提供更多的可能性。Nitro的建立，为这两个公链之间的协作提供了新的途径，同时也为其他公链之间的跨链交互提供了新的思路。

与其他公链相比，Nitro的优势在于它建立在Cosmos上，这意味着它可以直接获得Cosmos生态系统的优势和应用，同时又可以继续保留Solana的性能和优势。Nitro的建立，可以帮助Solana更好地拓展生态圈，吸引更多的开发者和用户，同时也可以让Cosmos更加多元化，进一步提高其市场竞争力。

应用链的生态机遇

当我们思考应用链时，我们之所以会想到Cosmos，因为Cosmos SDK是最成熟的允许无需许可地从头开始构建新链的工具包。Appchain 如今的代表就是 Cosmos，因为 Cosmos SDK 是目前最成熟的工具包，可以无需权限从头构建一个新的链。从长期来看，我们预计随着技术的成熟，Appchain 作为 Rollup（或 Rollapp）将会得到更广泛的应用。一些值得关注的现有和即将推出的应用链包括：

  1. Neutron: PoS智能合约链

Neutron为Cosmos Hub提供了一个智能合约平台，用于尝试他们想要使用和开发的所有新/实验性事物。这包括CosmWasm合约、跨链操作的ICA和从其他链读取数据的ICQ，以及将它们组合使用以实现创新产品，例如Interchain DEX，同时保持Hub的隔离状态。

简而言之，这是一个完全无需许可的通用智能合约链，由Hub提供安全保障。任何可以在CosmWasm智能合约链上开发的内容都可以部署在Neutron上，Neutron也是Hub经济区内智能合约dApp创新的孵化场。其中一个最早推出的项目是Lido，将在Neutron上发行他们的ATOM流动质押衍生品。

2. Osmosis：Cosmos的流动性骨架

Osmosis是Cosmos中最活跃的链，也是IBC中最具有流动性的链，每月交易量超过10亿美元。同时，它还作为Cosmos链上通证的流动性骨架。Osmosis的第一个主要创新是超流动抵押（Superfluid staking），这是一种模块，允许流动性池中的底层OSMO通证同时进行流动性提供和抵押以保证Osmosis的安全。最新的Fluorine升级增加了更多的功能，如稳定币交易自动化市场制造商（stableswap AMM）、IBC速率限制和多跳路由。IBC速率限制是对跨链桥黑客攻击的一种回应措施，它在定义的时间窗口内限制了可以进出Osmosis的代币供应量的上限。

Osmosis在2023年还有更多的计划，包括集中式AMM、通过阈值解密减少MEV、与Skip合作“内部化”MEV（即让协议自己完成套利交易），推出用于借贷流动性提供通证的Mars Protocol，类似于超流动抵押的Interfluid staking（但是针对非OSMO通证的底层LP token进行抵押），网状安全等等。Osmosis的增长很大程度上归因于其高的通证发行/通胀率，但是2023年即将推出的功能和整合计划为其提供了更多的机会，有望成为整个IBC网络中的流动性骨干，并为更健壮的增长提供动力。

3. Sei：名副其实的混合型公链

作为一个基于Cosmos的应用链，Sei在混合型公链赛道中具有一定的机遇。Sei上将推出两个高度优化的Rollup方案，分别是Nitro SVM和Paddle。前者运行SolanaVM，后者则运行MoveVM。MoveVM是一个相对较新的智能合约平台，已经被Sui和Aptos等快速单体机采用，并被认为是最快和最安全的VM之一。值得一提的是，Paddle将成为第一个运行MoveVM的Rollup方案。采用优化的Rollup方案有助于提高区块链系统的性能和安全性，从而增加生态系统的吸引力。因此，Sei在DeFi领域的发展前景广阔，未来有望成为混合型公链中的佼佼者。

同时，随着更多的生态系统采用模块化架构，Cosmos和其他公链平台也将迎来新的机遇和挑战。在Sei的架构中，它允许开发者构建不同的区块链应用程序，同时也允许这些应用程序进行跨链通信。Sei的设计初衷是要解决现有公链互相隔离的问题，并且希望在不牺牲安全性和可扩展性的前提下，实现更高的性能和更低的交易成本。在这个过程中，Sei采用了Cosmos SDK的框架，并在其之上构建了一个拥有高度自由度的链体系结构。这使得开发者可以轻松地构建各种不同类型的应用，例如基于权益的应用、基于资产的应用等等。

4. dYdX：纯 DEX Appchain

dYdX的推出将与 Cosmos Hub 的“ Noble”消费链同时进行。Circle 将在这个链上发行 USDC，dYdX 是最大的第一个客户。这将极大地加速原生 USDC 在 Interchain 上的流通，这一直是 Cosmos 和 IBC 一直缺乏的。从本质上讲，dYdX 的增长对 Cosmos 上原生 USDC 的增长有好处。Cosmos 上原生 USDC 的增长对作为 Interchain 安全提供者的 Hub 的经济活动增长有好处。而作为 ICS 提供者的 Hub 的增长对其他消费链的采用同样有好处，而原生 USDC 能够使其加速启动。原生 USDC 的增长和 Hub 的经济增长对其他 Appchain 的总体流动性增长呈现正向促进作用。

5. Injective：去中心化的衍生品交易协议

Injective旨在打造一个全球化的交易市场。与传统金融衍生品交易平台相比，Injective允许任何人在其平台上自由交易各种衍生品。此外，Injective采用了一种新型的交易模型，称为”无需信任的中介”，这种模型可以保证交易的安全性和公平性。与dYdX不同的是，dYdX是专注于衍生品的纯应用链，而Sei和Injective则是为像永续合约、期权、稳定币等应用构建基础设施的基础层。

6. Duality：AMM和CLOB之间的混合体

Duality的核心功能是创建AMM池，允许以恒定价格进行交换。这与典型的集中式AMM不同，后者通过价格刻度确定价格区间。Duality允许在特定价格上放置流动性，类似于CLOB上的限价单。在Duality池上可以构建任意曲线，交易者可以利用市价或限价单来访问所有池之间共享的流动性。与其主要竞争对手Osmosis一样，Duality的目标是将套利利润从MEV内部化和重新分配给LP。

7. Stride：流动性抵押

虽然Stride是一个独立的链，但一旦ICS上线，它将成为一个消费者链，以获取来自中心化Hub的经济保障。当你考虑Cosmos流动性抵押代币时，必须确保底层资产的安全性不受市值低、验证器集合不经过充分测试的链的影响。Stride已发行了流动性抵押代币stATOM、stOSMO和stSTARS，所有这些代币在Osmosis上都有激励池。Stride的主要竞争对手将是Lido和另一种Cosmos本地流动性抵押解决方案Quicksilver。

8. Polymer：基于ZK技术的IBC协议

Polymer将成为Cosmos Hub上的第一个zk-IBC应用链。它允许在不泄露任何信息的情况下跨链传输资产。由于zk技术提供了高度的隐私和安全性，Polymer在跨链资产传输领域具有很大的潜力。

9. FairBlock：一种通过身份加密（IDE）方案来预防“坏”的MEV的方案。

总结

目前混合型公链的赛道瓶颈在于缺乏一个统一标准，许多项目都在建立自己的互操作性解决方案，导致市场进一步分化。因此，目前缺失的一环是类似于TCP/IP对于互联网的所有链之间进行通信的统一标准。然而，生活在Cosmos之外的许多Layer1公链，正在利用IBC技术进入Cosmos，这意味着这个区块链网络正在收到采纳。

在这种情况下，混合型公链的发展趋势是不可避免的。这些公链可以从各种链中选择优质的功能，组合成一个更具可扩展性和安全性的整体。在这种混合型公链中，它们的L1层可以作为结算层，而其L2/L3层可以承载更多的实际业务。这种混合型公链不仅能够解决现有链的瓶颈和问题，还能在不同场景下提供更灵活、更高效的解决方案。

网络效应是推动经济活动的引擎。每当一个新的区块链（节点）加入Cosmos网络时，它都会增加该网络的价值。最重要的是，网络的价值呈指数增长。总之，未来的加密货币将是模块化的，并且不同技术堆栈的组合将提高它们的先进性。
