Paradigm 被认为是最独特的加密 VC 基金，作为一家总部位于旧金山的 VC，成立于 2018 年，主要投资于加密货币。虽然 Paradigm 正涉足 AI 领域，但 Paradigm 的投资项目依然值得交易者关注。本文系 Paradigm 在 2023 年第一季度的投资布局。

Paradigm 被广泛认为是最独特的风险投资公司之一，因为他们只选择少数几个项目进行投资。著名的项目包括：Optimism、Blur、Synthetix 等。

资产管理规模： 132 亿美元

投资重点：

  • 加密货币（主要）

  • 人工智能（新增）

  • 新兴技术（新增）

趋势

Paradigm 最近的投资集中在区块链即服务（Blockchain as a Service）和区块链基础设施。然而，随着他们最近将注意力转移到新兴技术上，AI 是 Paradigm 希望抓住的另一个热门叙事。

Paradigm 的加密货币持仓，最大持仓：

ETH： 1.879 亿美元

LDO： 1.54 亿美元

2023 年第一季度前 5 大加密货币投资

Conduit

融资： 700 万美元，种子轮

领投方：Paradigm

其他投资者：Robert Miller、Ludwig Pettersson、Dan Romero、Guillaume Poncin、Joshua Ma 和 Lakshman Sankar。

  • Conduit 是一个 Rollup-as-a-Service 项目（旨在通过提供用户友好的工具和服务来彻底改变 Rollup 的部署），使团队能够通过提供基于 OP 构建的生产级 rollup 来快速扩展 dApp。

  • 它旨在为寻求自己的专用 rollup 的项目提供更易于访问且更具成本效益的解决方案。

Ulvetanna 

筹资： 1500 万美元，种子轮

领投方：Bain Capital Crypto and Paradigm

其他投资者：Jump Crypto

  • Ulvetanna 是一家 ZKP 硬件基础设施公司。

  • 他们构建硬件以加速生成 ZKP 的过程。

  • 专注于为L2 zk EVM 网络生成 ZKP

code 4 rena

筹资： 600 万美元，代币购买资金

领投方：Paradigm

其他投资者：无

  • Code 4 rena 为传统审计提供了一种透明且游戏化的替代方案。

  • 为以竞争形式寻找 Bug 的独立审计员提供激励。

nxyz（2022 年第四季度）

筹资： 4000 万美元

领投方：Paradigm

其他投资者：Coinbase Ventures、Greylock Partners、Sequoia、Balaji Srinivasan、Jaynti Kanani、Jing Wang、Packy McCormick、Surojit Chatterjee 和 Ryan Selkis。

  • 多链索引产品套件，通过低延迟 API 公开实时区块链数据。

  • 旨在通过提供更高效的区块链索引和数据基础架构，使建设者能够构建更好的互联网。

Exponential（2022 年第四季度）

筹资： 1400 万美元

领投方：Paradigm

其他投资者：Haun Ventures、Solana Ventures、Polygon Ventures、Circle Ventures

  • DeFi 投资发现和风险评估平台

  • 通过查看协议代码、团队经验和市场状况来分析投资风险。

  • 为用户提供 DeFi 投资评级系统（A 到 F）

近期，Paradigm 从以加密货币为重点，转变为现在可能将新兴技术/人工智能作为其投资组合的一部分，这是一种多元化的手段，并且可能会追随 A16z的脚步，加大人工智能投资。

下面看看 Paradigm 已投资的与 AI 相关的项目：

ArenaX Labs

筹资： 500 万美元， 2021 年 10 月

  • ArenaX Labs 是数字平台 AI Arena 的母公司。

  • AI Arena 是一款以太坊原生游戏，全球玩家可以在其中购买、训练和战斗由 AI 驱动的 NFT 角色。

Numerai

筹资： 1100 万美元， 2019 年 3 月

  • Numerai 是一家使用机器学习制定交易策略的对冲基金。

  • Numerai 不是在内部开发模型，而是利用全球数据科学家的预测来做出投资决策，并用 NMR 代币提供激励。
