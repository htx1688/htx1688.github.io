作者：Game Fund Partners 合伙人 Jonah；编译：Odaily 星球日报

NFT 正处于最黑暗的时刻。

随着 Azuki  事件持续发酵，社区怨声载道，许多长期持有者抛售、砸盘、出走，更带动了市场情绪的进一步踩踏触底，人们再度探讨 PFP NFT 的发展停滞和创新乏力。在 Azuki 系列作品全线下跌的同时，其它 NFT 价格也未能幸免，周一时多个蓝筹项目均触及了近期新低，尽管今日随着市场杠杆率降低而稍有反弹，但整体状况依旧不容乐观。

不过，在市场寒气如此凛冽之时，仍有一些 NFT 项目还保持着自己的开发节奏，努力向社区及市场交付其规划。虽然如今去谈 NFT 的下一轮牛市为时尚早，但或许这些努力中的某一个，有望帮助 NFT 打开通往下一阶段的道路。

Game Fund Partners 合伙人 Jonah 为我们梳理了这些为数不多的积极动态，以下为其推文内容，由 Odaily 星球日报编译（仅做编译呈现，不代表本站观点）。

1. Yuga Labs 推出了一款新游戏「NHV-MTL Forge」，尽管这并不是什么史诗级大作，但这仍然值得肯定。一些用户很喜欢他们的小游戏。

2. Azuki 展示了他们的动画制作能力，现在他们也有钱去做些真正的内容了。

  • Odaily 星球日报注：这点不敢苟同……

3. Pudgy Penguins 在版税及周边玩具方面表现良好，他们马上还要办一场漫画展。

4. OhBabyGames 做了一款非常酷炫的赛车游戏，我自己玩过，很好玩。

5. InfiniGods、Parallel 以及其它一些项目都将在夏天有大动作。

  • Odaily 星球日报注：InfiniGods 开发了以古希腊神话为背景的休闲益智游戏 InfiniMerge；Parallel 则是一个类似炉石的卡牌对战游戏，其在 2021 年曾以 5 亿美元估值获得过 Paradigm 的 5000 万美元投资。

6. Xai 将基于 Arbitrum 启动 Layer 3 ，并启动一个游戏生态系统。

7. 我了解到 Limit Break（DigiDaigaku 开发公司）正在做一些很酷的东西。

  • Odaily 星球日报注：Limit Break 最新的动态是推出了可执行的链上可编程版税新标准 ERC 721-C，但不确定 Jonah 所指的是不是这个。

8. 9 VRSE Studio 正探索 NFT 与 3D 动画、AI 软件的结合。

9. Ronin Network 生态系统上线了一些并非基于 Axie Infinity 的新游戏，我认为这些游戏相当不错。

10. The Rabbit Hole 已上线 Epic 游戏商店。

  • Odaily 星球日报注：The Rabbit Hole 是一款由独立动画技术公司 AMGI Studios 开发的 Web3 游戏，Meta CEO 扎克伯格将在其中扮演大反派。

11. 耐克旗下 NFT 平台 .SWOOSH 与《堡垒之夜》达成了合作，并在游戏内启动了名为 Airphoria 的活动岛屿，有大量玩家参与了活动。

12. Ryan Wyatt（Polygon 总裁）在华盛顿做了些很疯狂的事情（应指合规方面），Polygon 对一些新的入驻者很友好。

13. On 1 Force 以及其他一些项目也在探索与《堡垒之夜》合作。

14. MOBA 类游戏 WILDCARD 已启动了测试。

  • Odaily 星球日报注：去年 6 月，其开发商 The Wildcard Alliance 曾完成了 4600 万美元 A 轮融资，Paradigm 领投，Griffin Gaming Partners、Polygon 等参投。

15. DeGods 同一些 holders 们共同举行了一些线下活动，活动很受欢迎。

16. Doodles 收购了一家真正的动画公司，正在创作。

  • Odaily 星球日报注：今年一月，Doodles 宣布已经同意收购曾获得艾美奖提名的动画工作室 Golden Wolf，具体收购金额暂未披露。

17. Futureverse 合并了多家公司，来共建一个协作生态系统。

18. Kanpai Pandas 正在围绕着 MMA（综合格斗）做一些真正的 Web2 内容。

19. Forgotten Runes Wizard’s Cult 正在开发游戏，且已有了 demo 展示。

20. Pixel Vault 有一个新的技术产品，我偷偷看过他们的动漫项目，那真的是很酷的艺术。

21. Opepen 正变得越来越狂热。

22. Rug Radio 与 MetaMask 达成了合作。

23. BoDoggos 让 Solana NFT 再次变得有趣。

24. 多款 Web3 游戏仍在利用之前筹集的资金进行开发。

一切都会好起来的，即便不是今天……到了一定的时候也会好起来的。
