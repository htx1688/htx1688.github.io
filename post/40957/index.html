作者： Chenglin Pua

亚马逊，国际电商与云计算服务一哥。在众多国际知名科技公司例如Meta、苹果、微软和谷歌都纷纷宣布布局元宇宙相关业务时，貌似亚马逊没有什么动作。实际上亚马逊已经开始布局了元宇宙相关业务，包括推出了元宇宙云计算等相关服务、投资NFT初创公司等等。 亚马逊更是认为，元宇宙离不开云计算，未来将会从自身业务出发，来进军元宇宙。

亚马逊简介

亚马逊公司（Amazon.com, Inc.）是一家总部位于美国西雅图的跨国电子商务企业，目前是全球最大的互联网线上零售商之一，也是美国《财富》杂志2022年评选500家公司的排行榜中的第2名。

亚马逊公司在美国、加拿大、英国、法国、德国、中国 、新加坡、意大利、西班牙、巴西、日本、印度、墨西哥、澳大利亚和荷兰均开设了零售网站，而其旗下的部分商品也会通过国际航运的物流方式销售往其他国家。

2020年2月，亚马逊市值正式达到1万亿美元，次于苹果公司、微软、Alphabet旗下谷歌，为第四家市值破万亿美元的美国上市公司。

亚马逊网络服务（Amazon Web Services，AWS，亚马逊云），是由亚马逊公司所创建的云计算平台，向个人、企业和政府提供一系列包括信息技术基础架构和应用的服务，如存储、数据库、计算、机器学习等等。AWS提供的大多数服务都使用按需付费（Pay-as-you-go）的收费模式，按照用户使用资源的级别和时长收费。因此，用户可以关闭和删除未使用的AWS服务资源以节省费用，或者在用户请求突然增加时很快添加新的资源，避免在平时为峰值系统负荷付费。

AWS目前提供超过175种服务产品，其中Amazon EC2、Amazon S3、Amazon CloudFront等等是使用量最大的服务。

从自身业务出发征服元宇宙的亚马逊

和微软、Meta等巨头相比，亚马逊对于元宇宙的态度相对不温不火。这是因为亚马逊本身的主业务在电商和云计算AWS。Meta则是因为社交领域整体的瓶颈，需要寻找下一个增长点，因此对于元宇宙持有非常大的期望。亚马逊并没有像Meta那样有争夺底层入口的焦虑。

然而因为科技圈整体吹起了元宇宙的风潮，显然亚马逊也不愿错过这一巨大的机会。因此亚马逊也尝试在元宇宙里寻找一些商机。从具体的实现路径看，亚马逊更侧重于从自身现有的业务和资源出发，对元宇宙的商机进行探索。

如今，亚马逊的业务中电商、零售贡献了营收的绝大部分；云计算业务虽然在营收中所占比重不高，但却贡献了一半的利润。目前亚马逊对于元宇宙的探索，基本就是这两大业务的延伸。

亚马逊积极利用AR/VR技术在零售领域进行升级，踏出探索元宇宙的第一步。2020年8月，亚马逊推出了一款名为Room Decorator的AR购物工具。支持用户在个人空间里可视化查看家具和其他家装饰品。亚马逊平台的用户就可以在购买家装之前，提前预览装修效果。

除了零售领域之外，亚马逊也积极利用其在云计算方面的优势，推出了一系列与元宇宙相关的云服务产品。例如，在工业领域，亚马逊推出了Amazon IoT TwinMaker。开发人员可以将Amazon IoT TwinMaker连接到设备传感器、视频源和业务应用程序等数据源，在云端快速开始构建设备、装置和流程的数字孪生。毕竟在元宇宙中一个非常重要的应用就是数字孪生。数字孪生创建完毕后，开发人员就可以使用相关插件创建基于Web的应用程序。然后，工厂操作员和维护工程师们在其维修设备上就可以直接调用这个云端的数字孪生体。

亚马逊的元宇宙业务场景包括为游戏平台提供相关的基础服务。游戏平台需要不断推出不同类型的小游戏以丰富元宇宙的世界。除此之外，游戏平台还需要为玩家与开发者提供协同开发工具与代码编辑工具。

亚马逊为此推出了Amazon Lumberyard ，一个开源、免费、高扩展性的3A级游戏引擎，使客户更方便地搭建开发者生态，以便产生UGC内容。在亚马逊官网中也有实例，例如Epic Games把Unreal Engine部署在云端实现云游戏。

在元宇宙中，为NPC与玩家的互动引入人工智能的能力会极大程度提升玩家的沉浸感。除此之外，基于AI的内容审核对于海量UGC内容来说也尤为重要，有些公司会安排上千人的安全审核团队，审核平台上的所有内容。亚马逊云端上也有特定的人工智能服务（Amazon Rekognition，实现图片与视频的内容审核以极低成本检测不安全的内容），帮助公司审核内容。

为了使元宇宙玩家获得尽可能真实的体验，使用AR/VR等技术实时渲染虚拟世界是必不可少的。通过使用Amazon Sumerian，客户可以基于浏览器在几分钟之内快速构建AR/VR场景，Sumerian 编辑器提供了现成的场景模板和直观的拖放工具，使内容创建者、设计师和开发人员都可以构建交互式场景。具备现有 HTML、CSS 和JavaScript 专业知识的开发人员还可以编写自定义脚本来支持更复杂的交互。

国际知名游戏厂商例如育碧、CAPCOM和Epic Games如今都在使用亚马逊的元宇宙云服务。

元宇宙离不开云计算

2021年11月30日，2021 亚马逊云科技re:Invent全球大会召开。在大会上亚马逊云科技不仅推出了众多新服务和功能，还宣布了一系列的合作，包括Meta 公司宣布深化与亚马逊云科技的合作，将亚马逊云作为其战略云服务提供商等等。

亚马逊云科技表示，Meta 公司将会使用亚马逊云可靠的基础设施和全面的功能，补充其现有的本地基础设施，并将使用更多亚马逊云的计算、存储、数据库和安全服务，获得云端更好的隐私保护、可靠性和扩展性。除此之外，Meta 还将使用亚马逊云来加速旗下的人工智能项目研发工作。

亚马逊全球副总裁张文翊表示，元宇宙是云计算可以大量赋能的一个领域。元宇宙本身需要的就是计算、存储、机器学习等，这些都离不开云计算。例如国际知名游戏厂商 Epic Games 开发了元宇宙游戏 Fortnite。目前Fortnite在全球拥有 3.5 亿用户，Fortnite的工作负载几乎全部都跑在亚马逊云上。

亚马逊云科技大中华区产品部总经理顾凡则表示，元宇宙里面融合了大家所熟知的大量技术，而这些技术的背后就是云计算。无论是在数字世界里面建模，还是与之互动，都离不开机器学习在背后支撑。元宇宙的世界必然会产生超级大的数据量，这些数据都离不开底层平台，而云计算一定是最佳的平台。

投资NFT初创公司

2021年10月，亚马逊被报道参与了Dibbs公司高达1600万美元的A轮融资。该轮融资的投资人还有知名NBA球星Chris Paul和Kevin Love。

Dibbs是一个NFT交易平台，让用户可以在该平台上创建、交易NFT，与Opensea平台类似。Dibbs公司建立于2020年，主要目标是希望降低一般民众进入传统运动收藏品（NFT）的高门槛。目前Dibbs公司做的主要业务内容是将运动员卡牌NFT碎片化，并提供一个简单方便且价格透明的交易平台供大众使用。在交易过程中，每一笔交易Dibbs公司会收取固定2.9%的手续费。

Dibbs平台的服务建立在Wax区块链之上。Wax区块链专注于发展游戏和NFT项目。

WAX由加密货币先驱 William Quigley 和 Jonathan Yantis 创立，是向世界任何地方的任何人创建、购买、出售和交易虚拟和实物物品的安全、便捷的方式。WAX 旨在成为业内最完整、生态友好的 NFT 生态系统，面向希望利用区块链力量的领先品牌、艺术家、创作者和开发人员。
