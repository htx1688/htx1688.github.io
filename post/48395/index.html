生态大事件

AQUA 与 Polygon Labs 合作

AQUA 是一个以游戏玩家为中心的 Web3 市场，此次合作将在 Polygon 上构建一个更好的 Web3 游戏入口，帮助项目和玩家充分利用 Web3 中的机会。

Mastercard 将推出 Music Pass

Mastercard 将在四月中旬推出 Music Pass 限量 NFT，作为 Mastercard Artist Accelerator 的一部分，持有 NFT 通行证的粉丝可以解锁各种福利，包括能够访问独家节目内容以及获得独家的音乐体验，还可以与艺术家一起学习以提高他们自己的工具和空间知识。Mastercard Artist Accelerator 是 Mastercard 与 Polygon 合作推出的 Web3 音乐加速器平台。

Convex Finance 扩展到 Polygon 上

Convex Finance 是一个旨在为 CRV 利益相关者和流动性提供者等提供奖励的平台，借助其原生代币 CVX 简化 Curve 以及 CRV 锁定系统的质押。于2022年11月在 Arbitrum 上线，现在将其平台扩展到了 Polygon 上。

Fractal 与 Polygon ID 合作

Fractal 将和 Polygon 共同努力，为涵盖 Web3 身份验证中价值创造所有部分的成熟解决方案铺平道路。该合作旨在将去中心化和数字身份从概念验证阶段转移到大规模采用阶段。此次合作 Fractal 将致力于为 Polygon ID 的 KYC/AML 流程和用户入驻提供支持。

Mintify 上线 Polygon

Mintify 是 NFT 的模块化交易终端，并且已经创建了速度极快且可扩展的工具，可以轻松地与公共市场、私人市场、NFT 生态系统和协议集成。现在 Polygon 的用户将能够在一个用户界面监测数千个项目，包括交易、分析以及发现基于 Polygon 构建的许多不同的 NFT 生态系统。

新合作伙伴

Polygon 与 HyperNest 合作推出加速器计划

Polygon 与 HyperNest 将于2023年第一季度合作推出 Web3 加速器计划。该计划重点关注 DeFi 和 基础设施的初创公司，其目标是为初创公司提供为期12周的结构化计划，涵盖 Web3 空间的所有方面，并提供各种技术支持和指导。

Rebels 即将登陆 Polygon 网络

Rebels NFT 市一个社区驱动的项目，旨在构建多元化的 DeFi 生态系统和支持多链的创新 NFT 质押系统。Rebels 将与跨链的高质量 NFT 项目合作，创建一个生态系统，支持和促进加密领域内新兴的 NFT Web3 场景。

UnUniFi 与 Polygon 合作

UnUniFi 是一个带有 DeFi 工具的 NFT-Fi 平台，此次合作将帮助 UnUniFi 实现其为 NFT-Fi 提供无缝跨链支持的目标，无论 NFT 存在于哪一个区块链上。

关于 Polygon

Polygon Labs 为 Polygon 协议开发以太坊扩展解决方案。Polygon Labs 与其他生态系统的开发者合作，帮助为 Web3 提供可扩展、可负担、安全和可持续的区块链基础设施。 Polygon Labs 最初开发了一套不断增长的协议，供开发者轻松获得主要的扩展解决方案，包括 Layer 2（ZK rollups 和 Optimistic rollups）、侧链、混合链、特定应用链、企业链和数据可用性协议。Polygon 实验室最初开发的扩展解决方案已被广泛采用，有数万个去中心化的应用程序，唯一的地址超过了2.11亿个，创建了超过112万个智能合约，自成立以来共处理了23.6亿笔交易。现有的 Polygon 网络是一些最大的 Web3 项目的所在地，如 Aave、Uniswap 和 OpenSea，以及知名企业，包括 Robinhood、Stripe 和 Adobe。PolygonLabs 是碳中和的，目标是引领 Web3 达到负碳。
