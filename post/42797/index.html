9月20日，由万向区块链实验室主办的第八届区块链全球峰会在上海举行。IEEE计算机协会区块链和分布式记账技术标准委员会主席李鸣发表了“元宇宙研究与探索”的主题演讲，在演讲中，李鸣阐述了元宇宙发展的背景，并表示NFT作为数字资产凭证逐渐走向大众，极大地促进了元宇宙的发展。从其价值来看，NFT有助于内容资产化、资产确权、知识产权保护等。 此外，李鸣还表示，元宇宙内涵的初步概念是数字共识生态的集成逻辑表达。元宇宙的关键要素包括组织、身份、资产和活动四个方面，而区块链恰好能够从这四个方面支持元宇宙的发展。生态分为5层架构：技术、内容、经济、协作和治理系统；以及元宇宙对人类社会在科技、内容、经济、协作和治理等方面的影响。

以下为演讲全文：

大家好，我是李鸣。很高兴参加第八届区块链全球峰会，感谢万向区块链的邀请。今天，我想与大家分享一下我们在元宇宙领域的研究与探索。

众所周知，元宇宙已经来临。百度、英伟达、Unity、Sandbox等国内外大型的互联网公司以及相关企业都纷纷进入到元宇宙产业当中。元宇宙的出现并非偶然，它得到了政策、技术、商业、社会等相关因素的促进。

就政策而言，中国、美国、欧盟、日本、韩国都发布了数字经济、人工智能以及虚拟空间等相关领域的政策；在技术层面，新一代信息技术的发展和Web3的提出都极大地促进了元宇宙的发展；从商业角度来看，互联网的金融和数据红利基本见顶，且投资标的物减少；此外，社会领域的影视作品和小说对元宇宙的发展起到了促进的作用。

宇宙发展的背景

互联网的发展分为三个阶段，Web1、Web2和Web3。Web1是只读网络，Web2是交互网络，而Web3是价值网络。相较于Web1与Web2的迭代，Web3主要基于Web2现有的一些问题，提出了资产明确、收益公平、隐私保护以及数据主权等相应的原则。

但是从另一个视角来看，邮件是面向消息的，OA是面向事务的，ERP是面向业务的，以及现在的互联网平台是面向数据的。目前，我们已经能够在虚拟世界中进行消息、事务、业务、数据等相关的活动。如果虚拟世界能够进一步包含价值活动，那么它就会形成一个更好的生态。

人们普遍认为，Web1是通过用户名和密码登录不同网站获取信息，Web2是能够在不同的平台上进行互动，而Web3则提出希望能够用一个身份来接入所有互联网服务。事实上，Web3的本质之一是建立一个可通用的ID，把用户的文章、社交关系、身份信息、短视频等一系列相关的内容和信息与用户的个人身份进行绑定。平台需要用户信息时，可以通过商业活动或授权的方式来购买个人信息，从而真正把个人数据还给了用户，实现数据主权。

近年来，人们对区块链的技术、金融和社会属性也在逐步探索。区块链的技术属性已经发展多年，此处就不赘述了。在经济属性上，当前出现的FT和NFT推动了Defi、Fi和SocialFi的发展；在社会属性方面，最近几年DAO（分布式自组织）也对创新协作模式进行了探索。在这个过程中，不同国家有不同的监管方式。

同时，人机交互的发展直接促进了元宇宙的出现。人机交互并非AR/VR，AR/VR只是人机交互的技术之一。人机交互包括AR/VR、全息影像和脑机接口。正如我们认为耳机不是手机，手机不是互联网，AR/VR也不是元宇宙。元宇宙包含了借助人机交互技术进入数字世界，并在数字世界中完成一系列活动后，再进一步对物理世界产生反作用的完整过程。这个先进入、再输出的过程，才是真正的元宇宙的体现。

此外，NFT作为数字资产凭证逐渐走向大众，极大地促进了元宇宙的发展。从其价值来看，NFT有助于内容资产化、资产确权、知识产权保护等。但它也有很多问题，包括金融合规、炒作、投资等相关的风险。NFT未来会有以下几个趋势：

1. 内容聚合。当前的创作者在一个平台上销售数字藏品或NFT，其产品定价或利益分成比例很多是平台由规定的，然而这种方式违反了创作者经济的本质。未来可能会出现一个内容聚合平台，并以互联网平台作为销售平台来进行分销，从而完成数字藏品或者NFT的流转过程。

2. 身份聚合。当前，用户需要用不同的钱包在不同的平台上购买和保存NFT，这对用户而言操作较为复杂。相信未来，甚至当前已经有一些前沿的项目在探索身份聚合的应用，即无论用户在哪个平台上购买NFT或数字藏品都可以放到一个APP里，无需再下载一堆APP来存储不同平台上的NFT。

3. 实用性权益。当前很多NFT仅仅是销售图片，在不同国家的不同监管要求下，这些NFT难以自由流通，因而其价值就无法体现。未来，NFT应该会成为元宇宙中的通行证、道具或凭证等具有特殊意义和作用的东西，这样才能把NFT真正的价值释放出来。同时，未来还会出现一些NFT浏览器、搜索引擎等，这些工具将进一步丰富NFT的生态。

新一代信息技术的集成为元宇宙的出现提供了非常重要的支持。物联网、云计算、大数据等新一代信息技术是面向数据的技术，例如物联网作数据采集、大数据作数据处理、人工智能作数据利用、云计算作承载以及资源调度、数据传输，区块链提供可信共识防篡改的机制以及激励机制，人机交互则主要是帮助用户进入数字世界。各个单项技术有较长的发展历史，但是一直没有现象级的应用出现，只有将新一代信息技术进行集成才能支撑工业元宇宙、商业元宇宙、产业元宇宙的应用。

元宇宙到底是什么

互联网上有很多关于“元宇宙到底是什么”的文章，学术界对此有很多看法，包括Meta、微软、亚马逊等大企业也对此有一定的解释，但我们认为这些都是对元宇宙意义和作用的叙述性描述。

基于本体论的思想，我们对元宇宙的本质进行了探索。不仅如此，我们还分析了元宇宙的中英文含义的一些细微差别。英文的“meta”的主要意思是“位于……之上”，具有“超越”的含义，而“universe”它可以理解为“宇宙、太空”，是一个空间的概念；中文的“元”可以理解为“首要的、第一的、本质的”，而“宇宙”在战国时期的说法是“上下四环曰宇，古往今来曰宙”，因此宇宙可以理解为空间的长廊和时间的隧道。由此可见，元宇宙包括时间和空间两个维度。

要讨论元宇宙的内涵或者本质，就要先理清元宇宙在宏观上与新一代信息技术和数字经济有什么关系。如下图所示，区块链是信息技术之一，信息技术进行集成之后，可以推动信息技术服务，从而促进数字产业化。

元宇宙是一个庞大的数字生态，这个数字生态能吸引更多用户进行信息消费，从而促进产业数字化，并进一步通过产业数字化和数字产业化的提升来推动数字经济的发展。在这个过程中，不难发现区块链以及新一代信息技术是基础设施，而元宇宙是一个数字的消费生态，通过“一个推动、一个拉动”来共同促进数字经济的发展。

同样的，基于本体论的方法，我们也从概念化、明确性、形式化和标准化这4个方面对元宇宙的内涵进行了分析。概念化需要找到元宇宙的关键要素，明确性要找到元宇宙的关键组成部分，形式化要探索元宇宙工程化实践的过程，标准化要提出一个让全社会有共识的概念术语以及发展方向。经过一系列的分析，我们得到了一个元宇宙内涵的初步概念：元宇宙是数字共识生态的集成逻辑表达。

元宇宙的关键要素包括组织、身份、资产和活动四个方面， 而区块链恰好能够从这四个方面支持元宇宙的发展。

1. 组织

传统的平台极易产生垄断型的组织形式和商业模式，而Web3的观点是“用户和建设者共同拥有网络”。基于区块链技术的“DAO”能够为Web3提供一种在符合监管的前提下，实现共识范围最大化的模式。

2. 身份

传统的身份验证方式极易产生安全风险和数据滥用的问题。在元宇宙中，无论虚拟人或虚拟体都必须要有身份，这样才能找到对应的责任主体。区块链可以为元宇宙中的身份提供分布式身份认证体系。

3. 资产

元宇宙中必然会产生资产，且以资产为核心的场景需要形成可持续发展的生态，甚至是经济体系。区块链是当前技术中，支持资产全生命周期过程的最佳解决方案。

4. 活动

元宇宙中所有的活动都会收敛为数据，而区块链能够支持隐私保护和数据安全。

我们将整个元宇宙的生态分为5层架构：技术、内容、经济、协作和治理系统。

1. 技术系统是新一代信息技术的融合；

2. 内容系统主要为元宇宙提供一个开放的、具有创造性的、丰富多样的且可持续的的内容场景，从而让更多用户进入元宇宙；

3. 经济系统能够提供元宇宙所需的激励机制和权益保障机制；

4. 协作系统则为元宇宙一种能够实现多方协作、自组织的响应机制，从而让更多人在协作过程中产生共识；

5. 治理系统为元宇宙提供相应的社会规则和治理规则，包括版权保护、数字认证、内容监管，以及物理能量守恒、万有引力等物理规则。

今天主要介绍一下技术系统。我们将技术系统分为5层，包括网络、系统、服务、场景和空间。

元宇宙的网络层与传统的互联网架构相似，包括通讯、存储、计算和网络技术。但是以计算为例，元宇宙中实时的三维场景呈现需要大量的计算，相较于传统互联网的计算，提出了更高的要求。元系统即新一代信息技术的集成，而元服务则包括内容运营、隐私计算、身份认证、激励机制及资产管理，并基于此来支撑元场景中工业制造、社交娱乐、文化旅游等相应的应用。此外，我们元宇宙中的一系列活动需要外部空间中相应的设备来配合，包括位置感知、人机互动、感官触达等。

同时，元宇宙的未来并非全球只有一个元宇宙，可能在不同的领域会有不同的元宇宙，例如金融元宇宙、教育元宇宙等，但是每一个元宇宙之间要能够互联互通。因此，在元宇宙的整个技术体系中，需要考虑到跨平台的身份认证、内容交互、资产流通等相应的标准。当前我国很多主要城市都发布了元宇宙的相关政策，包括北上广，以及无锡、重庆、厦门、杭州等城市。

在企业方面，例如百度的希壤、网易的瑶台都是国内大企业在元宇宙领域建的平台，以此来支撑用户进入元宇宙。此外，在胡润研究院发布的《2022胡润中国元宇宙潜力企业榜》中，国移动、中国电信、腾讯、阿里巴巴、百度等企业榜上有名，他们也在逐步地进军元宇宙。

国外的元宇宙企业有多个不同的类型，包括以RobLox、SandBox、Decentral Land为代表的“经济体系+内容创建”的企业，以Unity和英伟达为代表的开发引擎工具的企业，以及Meta和微软等既做外设，也做制作工具以及具体应用的综合企业。

通过对比国内外元宇宙产业的发展情况，我们可以发现，国内元宇宙发展在制作层是有缺失的，以及产业中很多AR/VR游戏和应用都是用Unity或英伟达的Ominiverse来做的。因此，我国元宇宙产业需要加速推动元宇宙制作引擎的发展。

元宇宙对人类的影响

此外，我们也分析了元宇宙未来对人类在科技、内容、经济、协作和治理等方面的影响：

1. 技术

就硬件层面而言，当前的显示器、手机、电视等这些终端可能都会被取代。虽然现在用户进入元宇宙的形式以AR/VR眼睛为主，但是未来可能通过虚拟工作舱等设备进入元宇宙，所以当前这些终端显示设备基本可能会被颠覆。同时，硬件中的鼠标、键盘在元宇宙中基本上是用不上的，将被新型终端设备代替。

在软件层面上，当前的浏览器、输入法、车载系统或操作系统等，可能都会因为元宇宙的到来而改变，输入法和浏览器可能也将不复存在。

所以整个软件、硬件产业基本都会被元宇宙所颠覆。

2. 内容

未来，我们希望能够在元宇宙中真正实现版权回归创作者。此外，元宇宙中的创作方式也将从2D向3D转换。

3. 经济

在经济方面，数字货币的应用可能将进一步扩大，甚至可能会产生数字资产税。

4. 协作

协作方式的革新可能会产生新的职业，包括数字建筑管理员、数字藏品设计师等都可能都会是产生的新职业。

5. 治理

在治理方面，例如虚拟数字人、半机械人等一系列的身份，其主体责任由谁承担，新的诈骗行为应当如何管理等都将会成为新的治理问题。甚至在社会治理方面，可能会出现数字公安局或工商局等相应的社会治理机构。综上所述，未来元宇宙的发展将会对我们的生产、生活带来巨大的影响。

目前，国内外的标准化组织都高度重视元宇宙的标准化。在国际上，IEEE最近成立了元宇宙的研究组，而Khronos Group则联合了一些国际上著名企业，例如Meta、微软、Epic Games、英伟达等相关公司成立了一个元宇宙标准论坛，以及Sandbox等公司也成立了一个Web3的开放元宇宙联盟，旨在提出元宇宙标准，并促进Web3和其他行业协作。在国内，日前发布了三项元宇宙标准，也有各种团体组织在大力推动元宇宙团体标准研制。

就元宇宙的标准化方向而言，我们希望能够建立产业共识，推动科技向善。在标准的制定工作中，一定要以科技向善为主导原则，以推动技术发展为主要方向，通过实际应用来获取行业的最佳实践，最终达成产业的共识。

关于元宇宙的未来发展，我们也有一定的思考，核心思想是“鼓励群智探索，促进涌现创新”。一个产业的发展或者新商业场景的出现，需要通过培养大量的应用才会实现创新突破。理论引导固然有用，但仍需一定规模的应用探索，才能够出现这种涌现式的创新。在这个过程中，我们给出如下4个建议：

1. 统一产业共识

加强元宇宙技术体系、服务体系、经济体系和治理体系的研究，明确元宇宙的技术发展路径，探索元宇宙的基础理论，形成元宇宙的产业标准。

2. 探索技术集成方法

目前，单一技术都有了一定的发展，但是如果要实现技术继承，还是会有很多问题。此外，分布式身份认证、跨平台/跨生态的互联互通以及数字资产的流动和数字内容的治理等一系列问题也亟需解决。

3. 创新应用场景和商业模式

推动数字环境下的智能制造、信息消费、智慧城市等相关场景的创新，利用新技术升级传统产业的同时，建立工业元宇宙、城市元宇宙、金融元宇宙等创新型的商业生态。

4. 加强科技伦理治理研究

这方面主要是包括研判隐私保护、技术歧视、算法绑架、非法集资等相关风险，制定金融服务管理、内容和资产合规监管、数字版权保护等一系列策略，推动形成可监管、有活力、可推广的高效治理机制。
