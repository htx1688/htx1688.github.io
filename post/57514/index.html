下面让我们查看starknet目前我们认为的十大生态项目。

1. Argent（钱包）

Argent 是一个非托管加密钱包，它结合了对 Dapps 的轻松访问和安全功能，例如无需助记词的恢复、受信任的联系人和多重签名。Argent 的智能合约架构使这些功能成为可能。

融资信息：

2022-04-28 Argent 完成 4000 万美元B轮融资

2020-03-09 Argent 完成 1200 万美元A轮融资，估值为 4500 万美元

2018-11-23 Argent 完成 400 万美元种子轮融资。

目前最新估值应该到达4亿美金左右，投资机构是Fabric Ventures、Metaplanet、Paradigm、StarkWare等。大家可以留意该钱包的交互 可获取潜在的空投，这个项目是目前starknet生态中钱包赛道的龙头。

2. Braavos（插件钱包）

Braavos 是一个基于智能合约的钱包，用于管理资金和 NFT 并连接到 StarkNet 上的 dApp。

于2022-10-31 ，Braavos 完成 1000 万美元融资，预计估值1亿美金左右。

投资机构Pantera Capital、Brevan Howard Digital、Road Capital、StarkWare等，

目前这款应用使用量也很多，社交媒体关注度也比较高。

3. Jediswap （DEX）

JediSwap 是 StarkNet 上受 Uniswap V2 启发的完全无需许可且可组合的 AMM。用户可以即时交易资产，而不会失去对其资金的保管权。JediSwap 由 Mesh 社区领导。

暂列starknet链上defi应用第一名，同时该项目也是比较多人关注，算是目前starknet生态中DEX的头部项目。

4. ZKX（衍生品交易）

ZKX 是基于 StarkNet 构建的无许可衍生品协议，具有去中心化的订单簿和提供复杂金融工具作为互换的独特方式。该协议由 DAO 提供支持，并将通过游戏化的排行榜和独特的流动性治理提供更高的交易体验。

2022-07-14，ZKX 完成 450 万美元种子轮融资，投资机构StarkWare、Amber Group、HashKey Capital等

ZKX采用的去中心化订单簿，既拥有dydx使用的中心化订单簿（CLOB）速度快，流动性高，低gas的优点，也结合了GMX,GNX使用的虚拟自动做市商(vAMM)的安全性和透明性的优点。

详细分析可查看往期文章：《【ZKX】Starknet王牌生态项目!》

5.MatchboxDAO（gamefi）

MatchboxDAO是一个基于Starkware的链上游戏生态系统，帮助新的游戏应用涉足StarkNet生态系统的游戏领域。MatchboxDAO的两个核心产品是Matchbox Spark和Matchbox Studios。Matchbox Spark在StarkNet上孵化游戏产品，而Matchbox Studios则是一套加速StarkNet游戏开发的开发工具。

2022-08-17，MatchboxDAO 完成 750 万美元融资，投资机构有StarkWare、Fabric Ventures、1kx等

6. zkLend（借贷）

zkLend 是建立在 StarkNet 上的加密借贷协议，结合 zk-rollup 的可扩展性，卓越的交易速度，并通过以太坊的安全性节省成本。

2022-03-29，zkLend 完成 500 万美元种子轮融资，投资机构：Delphi Digital、StarkWare等

6. Orbiter Finance（跨链桥）

Orbiter Finance 是一个去中心化的跨 rollup 桥，用于转移以太坊的原生资产。在 Orbiter Finance 中，有 Sender 和 Maker 两个角色 。当 Sender 发起转账时，Maker 为其提供流动性。智能合约保证了这个过程的安全性。如果 Maker 有不良行为导致转账失败，Sender 可以用 Maker 的保证金向合约发起仲裁请求，然后获得超额补偿。

目前是starknet链上使用量最大的跨链桥，社交平台关注粉丝达到45万，同时据我们了解，其最新的估值已达到2亿美金。

7. zkLink（交易网格）

zkLink 是一个以交易为中心的多链 L2 网络，具有由 ZK-Rollups 保护的统一流动性。建立在 zkLink L2 网络上的 dApp 利用无缝的多链流动性，为去中心化和非托管订单簿、AMM、衍生品和 NFT 交易所提供快速部署解决方案。

2023-05-04，zkLink 完成 1000 万美元战略融资

2021-10-23，zkLink 完成 850 万美元融资

投资机构有Coinbase Ventures、Big Brain Holdings等

8. Nostra（流动性层）

Nostra是构建在Starknet上的一站式Defi解决方案，用户可以在这里借贷和交易加密货币。Nostra流动层由Nostra货币市场、UNO(Starknet原生稳定币)和Nostra Swap(Stablecoin DEX)组成。

9. Pragma （预言机）

Pragma Network 是一个 zk rollup 原生的预言机，它利用 zk 计算的增加容量来提供可验证性、可组合性和功能方面的改进。

2022-07-15，Pragma Network 完成 700 万美元种子轮融资，投资机构Variant Fund、CMT Digital、Flow Traders、StarkWare等

10. Starknet.id（域名）

Starknet.Id是Starknet的身份和命名服务，用户可以免费创建starknet身份，并将其连接到twitter, discord或github帐户。

目前社交平台粉丝达到20万，大部分是奔着空投去的，所以热度较高，但是实际上还需观察。

最后想跟大家分享的是市场是存在周期性的，这个周期你可以拆分为大的周期以年为计算，其次就是以季度为计算，不需要每天去盯着大盘的涨跌也不需要时刻去开单，把握好趋势即可，特别是一级市场，能赚到不错的收益往往都是在周期敢于下注，带来的收益才会有几何倍增长。
