作者 | Michael Hebenstreit

编译 | 白泽研究院

大约一年前，我开始涉足 NFT 并购买了人生的第一个 NFT。

我在 2022 年中向各种 NFT 项目投资了大约 4 万美元，其中包括顶级 NFT 项目之一的 Moonbirds。在本文中，我想回顾一下我投资 NFT 的经历。随着一年时间的流逝，我对 NFT 的看法发生了很大变化，我对 NFT 的未来有一些具体的想法。

现在让我们深入……

投资 NFT，割肉离场

自从深入参与 NFT 以来，我学到了很多关于 Web3、元宇宙和数字所有权的知识。我对区块链技术十分的看好。我相信随着时间的推移，这项技术有可能颠覆许多行业，并彻底改变人们拥有和交易数字资产的方式。它还将帮助越来越多的真实世界资产数字化。

在投资 NFT 之前，我是一个电报群组的成员，群组中有一些非常成功的公司高管，我们通常会在其中讨论各种话题。大多数成员都大量投资于 NFT，这也是我深入这个领域的原因。尤其是 Moonbirds NFT 项目，在这个群组中被提及的次数相当多。然而，在一开始，我只是批评一些流行的 NFT 项目的疯狂价格，并且真的无法理解为什么有人会以几十万美元的价格购买这样的 NFT。

我的怀疑引发了一些深入的讨论。

当时其他人购买 Moonbirds 时的交易价格在 20-25 ETH 左右，我预测 Moonbirds 的底价会跌到 10 ETH。

然而在讨论中，我逐渐沦陷了，我开始质疑自己的信念，因为当许多非常聪明的人都看好 NFT 或特定项目时，你根本无法视而不见。

于是，当 Moonbirds 的底价在几个月内从 40 ETH 左右跌至 10 ETH 以下后，我最终决定购买两只“鸟”，如下图所示。我以 12.75 ETH 的价格买了一只非常漂亮的“太空鸟”，以 10.71 ETH 的价格买了一只“裸鸟”。随后我又以 1.2 ETH 的价格购买了一个 Oddity，这是 Moonbirds 的姊妹项目。所以我在 Moonbirds 项目上总共投资了大约 25 ETH（约 3.5 万美元）。

我的投资论点是，Moonbirds 基本上算是前 5 名的 PFP（个人资料图片）项目。我认为像这样成熟的项目可能比其他不太成熟的项目更容易吸引流动性，更不用说即将到来的 Moonbirds Mythics 会引起的“临时炒作”。

Moonbirds Mythics 是 Moonbirds 的第二季，获取方式有两种，一种是用 Moonbirds 质押筑巢，每天随机释放 25 个；另一种是燃烧 Oddities，每天最多 25 个 Oddities 燃烧兑换成 Mythics。

所以，需求的激增可能会对 Moonbirds 的价格产生重大影响。

我猜测 Moonbird 的底价可能会上涨到 20 ETH，然后我可以卖掉一只“鸟”，基本上我就回本了。但即使这行不通，我有 2 个 Moonbirds 和 1 个 Oddity，所以我最终应该得到 3 个 Mythics 和一些未来空投的代币。我假设 Mythics 的价格会高于 Oddities，尤其是因为它们每天最多只被铸造 50 个。我认为 3 ETH 是 Mythics 的合理价格。这意味着 3 个 Mythics 大约价值 10 ETH，我可以卖出，回一些本。这就是我购买 Moonbirds 的原因，我认为风险/回报是合理的，我计划是在合理的时候取回我投资的钱。

但我越想，就越对自己的决定感到不安。几个月来我一直在批评顶级 NFT 项目的底价过高，现在我自己却也投资了。我意识到我的行为纯粹是出于投机，作为持有者，除非炒作再次升温，否则投资不太可能获得回报。

后来 FTX 危机发生了，我决定扭转错误并立即出售掉所有的 NFT，这导致我损失了大约 7 ETH，大约 20,000 美元。

而且，我不喜欢 Moonbirds 项目的开发商 PROOF 没有将持有人的底价和 NFT 的价值视为优先事项，我也不喜欢 CC0 许可变更，任何人现在都可以在不通知所有者的情况下用于商业和使用。

图片：Moonbirds 底价 / Opensea.io

其他一些 Moonbird 持有者试图说服我继续持有我的 Moonbird，但我不再听了。我能理解为什么有些人会坚定持有这些“鸟”，尤其是因为大多数持有者都蒙受了重大损失，但“hopium”（指虚假或不切实际的乐观）并不是我真正的投资策略。我太理性了。

从长远来看，我们很有可能会看到 Moonbirds 的交易价格大幅下跌。Moonbirds 没有理由不能在几个月内以 2–3 ETH 或更低的价格进行交易。如果 Mythics 因为供应量大（2 万）也下跌，甚至可能拖累 Moonbirds 的底价。但这也不是绝对的，因为 Mythics 有 DAO 投票权，可能会比 Moonbirds 更被看好。 

PROOF 还宣布他们将在某个时候推出代币。何时会有代币以及它的代币经济学如何设计还有待观察。在代币经济学方面，我认为一个项目创造可持续的生态系统是可能的，但这真的很难。这是通货膨胀和通货紧缩之间的平衡行为，大多数项目都不具备应对市场动态的经验和金融背景。

如果不是因为质押会在未来为持有人带来可观的投资回报率的“hopium”，Moonbirds 的底价现在可能已经更低。

但公平地说，大多数其他 NFT 项目也停滞不前，或者创始人完全消失了。在这方面，PROOF 肯定做得更好，但从投资回报率的角度来看，这并没有使它变得更好。团队想打造一些伟大的东西，而且他们总体上是有良好意愿的。我只是不认为这会给持有人带来多大好处（在财务方面）。

NFT 是投资还是数字商品？

在我看来，NFT 领域的一个普遍误解是，当你购买 NFT 时，你就进行了“投资”。然而，真相是，大多数 NFT 只是数字商品，持有者不是投资者，而是客户。

但是当你购买 NFT 时，除了可以欣赏之外，你并没有任何权益，也没有任何权利要求开发商。人们“希望”他们的 NFT 会变得流行，并且其他人最终为此付出更多钱，而他们可能会获得一些额外收益。

基本上就是这样。

近几个月来，NFT 领域遭受了巨大损失。许多 NFT 项目已经失去了很大一部分“价值”，知名 NFT 市场 Opensea 的交易量下降了 90% 以上。人们坚持持有他们的“投资”，因为他们将 NFT 与 BTC 或 ETH 等加密货币进行比较，认为“跌下来的东西一定会涨起来”。但事实是，这种比较在很多方面都存在缺陷。NFT 项目更像是许多小型初创公司，这些项目通常是建立在牛市之上的。而这样的项目大多数将无法在熊市生存，甚至可能会归零。

许多项目只不过是编织一个花哨的故事来进行炒作。对于项目创始人来说，这是（或曾经是）赚取资金的好方法，但对于持有人来说，尤其是对于昂贵的项目，他们必须为此付出许多努力才能获得可观的回报。尤其是现在大部分炒作已经消失，投机热潮已经结束。

大多数 NFT 持有者应该问问自己，除了投机炒作之外，从长远来看，他们将如何获得经济利益。考虑到持有者只拥有他们的 NFT，而不是股权或项目本身，你认为什么值得为 NFT 支付数千美元？仅仅是因为预测在未来的某一天有人愿意为 NFT 支付更多钱？一些项目承诺持有 NFT 有权分享项目的收入，在熊市中，它们是否仍然兑现承诺，这一点还有待观察。

Web3 领域发展迅速，竞争最终会变得更加激烈。

NFT 的炒作已经结束，整个市场现在正面临现实。

但是，总会有一批新的 NFT 项目出现，它们背后的技术也会不断发展。

NFT 炒作会卷土重来吗？

由于我们多年来处于低利率环境，大多数资产价格达到历史高位，导致各种资产泡沫。这适用于股票、债券、房地产、贵金属、奢侈品（例如劳力士手表），以及加密货币和 NFT 等另类投资。当美联储开始收紧货币环境时，大多数资产价格如预期般开始下跌。

对于加密货币和 NFT 等高风险资产尤其如此。与许多成长型股票一样，这些更具投机性的资产在利率上升的环境中跌幅最大。对于 NFT 来说，这意味着很多项目的流动性和交易量已经接近枯竭，就连热门的“蓝筹”NFT 项目也出现了大幅下滑。

我还观察到，每当 ETH 价格暂时上涨时，NFT 的底价基本上都会随之下跌，但当 ETH 价格盘整时，NFT 的底价往往不会以同样的幅度回升。这似乎是一场逐底竞争。总体而言，NFT 底价不断走低。

由于 NFT 的流动性很低，我认为高端 NFT 项目的价格发现规律有点类似于房地产市场。房地产通常不会崩盘，而是会缓慢下跌，除非黑天鹅引发恐慌性抛售。卖家以他们想要的价格出售他们的房产，直到他们意识到行不通，并且买家犹豫不决。然后他们开始逐渐降低价格。我们目前在大多数顶级 NFT 项目中都看到了类似的情况。

我认为投资的一个重要教训是寻找资产的内在价值。在此基础上，我很确定我们再也不会看到大多数 NFT 项目回到之前的高点了。但我认为迟早会有新的项目引起炒作。

NFT 炒作是由投机和“大傻瓜理论”引发的。人们根据一些奇特的故事以疯狂的价格购买 NFT，并希望其他人（更大的傻瓜）会为此支付更多钱。现在炒作已经结束了，剩下的只是一厢情愿。现在大多数持有者都是“长期投资者”，但你不能将持有 NFT 与长期持有其他实际投资进行比较，比如微软股票。

正如我之前提到的，除了欣赏你的 NFT 并按照你认为合适的方式使用它（例如在社交媒体上）之外，大多数 NFT 不会赋予你任何权利。一些小的 NFT 项目今天就可以停止，但对此你无能为力。如果你有股权，或者如果项目承诺与持有人分享利润，情况就会不同。那么你至少会得到某种股息，这也可以让你得到某种内在价值。但最终，持有者除了在区块链上拥有所有权的 JPEG 和项目以某种方式为他们创造长期价值的“希望”外，一无所有。

这就像早期投资一家初创公司，大多数初创公司都会随着时间的推移而失败。但是对于初创公司，你至少拥有公司的一部分。对于大多数 NFT 项目，你拥有的只有一个承诺，也许还有一个精美的图片。我认为未来我们不会再看到这些价格过高的 NFT，但它们将成为大多数人都可以负担得起的商品（著名艺术家的独家艺术品是例外）。在我看来，这实际上是一件好事，因为炒作也产生了很多 FUD。

还值得注意的是，有影响力的人在 NFT 热潮中发挥了重要作用。NFT 影响者基本上是跑在普通人的前面，他们与项目合作，在社交媒体上大肆宣传，然后在项目成功时抛售他们的 NFT。通过这种方式，项目也不是给每个人以平等的铸币机会，而是根据白名单将大部分供应分配给特定的人群或有影响力的人。这当然会造成利益冲突。有影响力的人通过这种方式赚了很多钱，而其他人则赔光了所有的钱。在我看来，项目与影响者的合作是一种营销方式，但影响者必须有禁售期。

现在 NFT 价格前景如何？NFT 会复苏吗？我们会看到另一个 NFT 牛市吗？我认为得看情况。像 BAYC 或 CryptoPunks 这样的顶级 NFT 项目肯定会复苏。Bored Apes 可能是最著名的 NFT 系列，由于媒体报道和参与该项目的名人众多，甚至不了解 NFT 的人也知道它。

在我看来，BAYC 是唯一一个真正可以象征身份的 PFP。但很难说当你在 BAYC NFT 上投资 100,000 至 500,000 美元时，作为持有者是否会看到投资回报率。就其价值而言，我认为这些高价值的 NFT 将随着时间的推移继续走低，然后最终趋于平稳。

NFT 将继续存在，但炒作可能已经结束。1C0 炒作并没有扼杀加密货币，NFT 炒作也不会扼杀 NFT。我们不会在短期内看到大多数 NFT 价格的历史新高，也许再也不会了。我认为 NFT 将成为一种商品，并且会有大量供应，因为基本上每个知名品牌都会推出 NFT，而它们的价格趋于平民化。高价 NFT 的时代可能已经结束（除了少数顶级项目），剩下的是一些项目现在仍然要执行当初的承诺。

NFT 的未来

如果你已经读到这里，你可能会觉得我认为 NFT 是骗局，注定要失败。但实际上，恰恰相反。我相信 NFT 会有美好的未来，他们会在很多行业发挥作用。我只是认为，大多数高价 NFT、铸造 NFT 动辄花费数千美元的日子已经结束了。

我认为 NFT 将变得更像商品（除了一些例外，例如非常著名艺术家的艺术品），并且对于大多数人来说它们将是非常便宜的。我们以堡垒之夜为例：游戏玩家不会为数字资产支付 20,000 美元，而只会支付几美元。这可能会成为游戏、音乐和元宇宙行业中 NFT 的标准。

未来，可能仍然会有昂贵的 NFT 用于解锁独家活动、体验（例如与流行歌星会面），来自奢侈品牌的 NFT 用于特殊津贴等特殊效用。但我认为基于猜测价格的炒作时代现在已经结束了。NFT 项目需要提供真正的价值，以证明人们为它们支付的价格是合理的。一张 JPEG 图片和一些精美的连帽衫还不够。至少对我来说，它不值 2 万美元。

可以公平地说，未来的大部分 NFT 活动将发生在大品牌和大公司的项目中，因为那才是用户聚集地。在这两年的 NFT 热潮中，我们有时很容易忘记，对于普通人来说，具有一些额外津贴和良好品牌体验的星巴克 NFT 在日常生活中要比 Moonbirds 有用得多。

想象一个每天都喝星巴克咖啡并且热爱这个品牌的人。那个人肯定会对提供定期折扣或有权参加特殊活动的 NFT 感兴趣。

关于品牌如何利用 NFT 技术，我有很多想法。想象一下，星巴克让人们用他们的钱包付款，并在结账过程中识别出你拥有星巴克 NFT，这样你就可以自动获得折扣或赠品（如巧克力曲奇），甚至可以让 NFT 在每次购买时在视觉上发生变化。我的意思是，如果我能在 5 分钟内想出这样的东西，我敢打赌大品牌的专家团队比我聪明得多，有更好的想法……所以，NFT 潜力无限！

这一切都与品牌互动、客户忠诚度和用户参与度有关。这些类型的 NFT 不会太贵，它们将像商品一样进行交易，我们可能会看到来自不同品牌的大量此类项目。想想会员资格、忠诚度计划、活动门票、真实世界资产的所有权等。你基本上可以将任何实物资产放在链上。NFT 是优秀的营销工具，品牌的 NFT 基本上是“营销的游戏化”。它们提高了客户参与度、品牌知名度和品牌忠诚度，同时为持有者（社区）提供好处和某些效用。

总而言之，技术将继续存在，NFT 可能会有美好的未来，尤其是现在大品牌正在进入这个领域。但未来大多数 NFT 只需要花费几美元。归根结底，NFT 是数字商品，而不是投资。
