2021年，元宇宙的概念爆火，此后万物皆可元宇宙。 不仅有运动、时尚品牌布局元宇宙，现在就连手机品牌也要来分一杯羹。

近日，欧洲奢侈手机品牌VERTU发布了一款名为METAVERTU的手机，并且号称是全球首款Web3手机，入门款的起售价23800元，顶配版本更是高达308000元，在伦敦正式发布并开启全球发售。

VERTU这款手机可谓是噱头十足，首先从手机名称METAVERTU来看，就蹭了一波元宇宙和Meta的热度。

当然，这还没完，在宣传片中，VERTU复刻了苹果30年前的经典广告，人物换成了类似于扎克伯格在horizon worlds的形象，并将手中的锤子换成了蛀虫的苹果，这操作属实把苹果狠狠地黑了一波，明显在向苹果下挑战书。

如此仗势，VERTU是什么来头？

1998年，诺基亚首席设计师弗兰克·诺佛开始构想一款奢侈手机。彼时，全球手机行业刚刚走出“大哥大”时代的设计思路，开始向“小而美”发展。

诺基亚高层对诺佛的工作很感兴趣，同年10月，VERTU正式诞生，成为诺基亚的子品牌之一。

VERTU坚持以精湛的工艺打造非凡的产品，METAVERTU手机都采用陶瓷与金属机身，同时还推出喜马拉雅鳄鱼皮版本，采用18K黄金、钻石等高端材料打造。

除了有稀有材料的加工之外，7X24小时英式私人管家服务也是VERTU的特色之一， VERTU旨在为所有热爱和过着奢华生活的人提供顶级奢华手机和优质服务。

据悉，在当时没有各种APP订购酒店、机票的年代，只要你按下服务按键表明需求，VERTU管家那边帮你把行程安排的明明白白。

可惜好景不长，随着智能手机的飞速发展，各类便利APP应运而生。

苹果、三星等手机品牌紧紧抓住了智能手机这个风向，此后迅速抢占市场。而诺基亚子公司VERTU虽然也在2011年发布了多款触摸屏智能手机，缺还是反响平平。

就这样，曾经的手机霸主诺基亚最终跌落神坛，在2012年逐渐淡出手机市场。并将子公司VERTU品牌90%的股权出售给了瑞典投资公司EQT。

之后经过数次流转，最终在2015年被香港基金公司GodinHolding联合“国际私人投资者”收购 。

如今的VERTU已今非昔比，无论是官网上的宣传图风格还是设计，无一不透露着一股浓浓的“复古风”。

而从VERTU这次发布的新品可以看出，与以往发布的机型不同，这次则是把更前沿的技术带到了手机里。

或许是在元宇宙的爆火下，VERTU似乎想抓住Web3、元宇宙这个噱头，结合自身的高端人群市场，打一波翻身仗。

VERTU Web3手机代表了未来吗？

和其他手机品牌不同，VERTU一直以来走的就是高端的“堆料”路线，喜马拉雅鳄鱼皮、钻石和红宝石等堆料，不仅让手机拥有了奢侈品的价格，更有一种身份地位的象征。

以往的机型给人的感觉就是与科技不沾边，就如经典的Signature系列，自始至终都是方正的直板机造型，功能上也仅限于打电话和发送短信，甚至连个摄像头都没有。

而此次推出的Web3手机与以往的画风不同，自带了“前沿”的科技，更是领先于众多手机品牌早早加入了这个赛道。

难道VERTU比其他手机品牌更早的掌握了Web3核心技术?

官方介绍，METAVERTU的配置采用骁龙 8 Gen1 处理器，支持5G，外加一颗A5安全芯片，针对信息隐私进行优化。以及144HZ高刷屏， 6400W主摄+5000W超广角+800W潜望式长焦和4600mAh电池+55W快充。

这么一看，似乎跟努比亚Z40 Pro有着异曲同工之处，不得不让人怀疑是同一家供应商贴牌。不过，这款手机的亮点不止于此。

VERTU团队介绍，METAVERTU手机支持「 Vtalk 功能」，相当于Web3时代的加密通话，内置「版权相机 VSHOT」，可以一键生成数字藏品（NFT），搭载LUXURY OS + WEB 3.0 OS 双系统，内置 VPASS 卡，支持一对一带入Web3世界。

并首次提出" CNCOS ——芯端链系服 " 五维整合生态概念，用手机串联芯片、智能终端、区块链、操作系统、高端服务，其目的是挑战去中心化数字霸权，并推动用户无缝流畅进入Web3世界。

尽管这款Web3手机的亮点有很多，但有业内人士认为，这款手机只是自带 NFT 交易软件+大容量网盘，在Web3时代还未到来或是普及的情况下，或许Web3手机还有很长的一段路要走。

值得注意的是，这并非最早发布的Web3手机。

早在6月24日，公链Solana的联合创始人兼执行长Anatoly Yakovenko在纽约的一场发布会上，正式宣告将推出以Web3为核心的Android手机“Saga”。

这款手机和市面上的主流智能手机没有太大差别，采用的是骁龙 8+ 处理器、6.67 英寸 120Hz OLED 屏、12 + 512GB 存储组合、4110 mAh 电池（支持 Qi 无线充电）、IP68 防尘防水、支持 microSD 扩展 512GB 存储。

其中最大的亮点的Web3功能：

  • Saga将采用高通最新的骁龙8+处理器，提供包含12GB+512GB的存储版本，正面是6.67英寸中置挖孔的Amoled屏幕；

  • Saga将包含一个Web3 DApp商店；

  • Saga集成的“Solana Pay”用于促进基于二维码的链上支付、一个移动钱包适配器和一个“seed vault”（用于存储私钥）；

  • 基于Android的开源软件Solana Mobile Stack可在Solana上开发原生Android web3应用程式。

10月21日，Solana Web3手机Saga开启了预售，前 1 万名预订客户将获得限量版 NFT。

这款手机售价为1000美元（约合 7263 元人民币），目前预订的手机需要等到2023 年初才能发货。

不难发现，VERTU推出的Web3手机与Saga手机有着异曲同工之处，只不过在市场上两者的客户群体各不相同，一个是万元以下群人市场，另一个高端富豪人群市场，或许对于高端富豪人群市场，Web3的去中心化、私密等功能特点更被他们所需要。

结语

VERTU手机为我们打开了无限的想象空间，无论是从Saga手机，还是VERTU手机来看，Web3技术似乎已成为未来手机的发展方向之一。

不过，从Web3手机高昂的定价来看，大众化的Web3时代或许还离我们有一些距离。
