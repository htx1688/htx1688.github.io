展望2023年的加密市场，无疑ZK（零知识）是最令人期待的细分赛道。一方面以太坊创始人Vitalik Buterin最看好这项技术，表示过去十年里，最强大的密码技术也许是通用简洁的零知识证明；另一方面，ZK Rollup运营商赛道规模可能将达到10至100亿美元级别（数据来自Bing Ventures发布的深度报告），其背后的Rollup TVL（总锁定价值）在2020年实现了最高75亿美元的高速增长。

Source: L2Beat Rollup TVL数据乐观的数据和期待背后，展现出ZK赛道在秘而不宣地有力发展。一些头部项目已获得了资本的密切关注，投资热情高涨。项目Starkware和ZKSync分别被推到了80亿美元和约10亿美元的估值。隐私公链项目Aleo更是获得了ZKP（零知识证明）赛道的最大融资。

不过ZK赛道的另一面是，当前ZK Rollup的现状还处在早期技术发展状态，其发展仍然面临较高的不确定性，导致其技术、价值、业绩等验证均存在滞后性，市场的接受也还需观察，其总体也尚未迎来爆发性增长（Rollup市场渗透率仅有5%，也表明提升空间巨大）。

这是当前ZK赛道的现状，一句话总结就是：ZK处在破晓的前夜，正蓄势待发。体现在进入2023年以来，ZK赛道的动作都纷纷进入更快的发展阶段，如币安的ZK技术引进、Scroll的技术更新、Way network和Nil Foundation的巨额融资，更重要的是ZK项目zkSync、Starknet和Polygon zkEVM或将推出代币消息。这都将意味着ZK赛道的叙事将在2023年开始升温。

下列项目大都走在ZK赛道前沿，我们可以重点关注。

1. Scroll

Scroll是用于以太坊的原生zkEVM Layer 2解决方案，是一个基于zkEVM的zkRollup，可实现与现有以太坊应用程序和工具的原生兼容性。其使命是解决以太坊网络面临的现有限制，改善用户与以太坊和Dapps 合作/使用的体验，并允许网络更快地扩展。即将推出的L2中，Scroll上市前估值排名第四。并且，根据众多L2项目代币的估值，Scroll的估价高于zkSync和Arbitrum。

Scroll最近暗示要进行空投，表示他们正在“创建一种鼓励参与的激励机制”，或许就像这两天Blur代币空投机制，Scroll的空投时间也不会太远。

官网：https://scroll.io/

Twitter：@Scroll_ZKPScroll

参与指南：目前，Scroll在Goerli测试网上发布了其以太坊L2网络的测试网版本，这是将zk-EVM从Pre-Alpha转移到Alpha的测试阶段。下面将介绍两个版本的交互方式：

（1）Pre-Alpha

Pre-Alpha 测试网由 Scroll L1 和 Scroll L2 测试网络组成，当前已经有一些预先部署的演示应用程序：

① Scroll L1 上的Faucet；

② 基于HopProtocol的Scroll L1和Scroll L2之间的Bridge；

③ 在Scroll L2上运行的Uniswap V2 分支；

④ 使用Scroll L1 和 Scroll L1 的区块浏览器及rollup浏览器。

用户可以通过进入网站（网址：https://scroll.io/prealpha）进行简单的交互，包括添加MetaMask钱包，进行L1和L2测试网之间的转账，作为参与并测试网络的一种形式，以实现和Scroll的互动。

（2）Alpha

用户需要一些Goerli测试币，这里介绍三种获取方式。

①https://goerlifaucet.com/这个水龙头一次只给0.2 ETH，而且需要注册Alchemy才能获得。

②https://goerli-faucet.pk910.de/这个水龙头通过PoW提供测试币，不太方便。③https://faucet.paradigm.xyz/这个水龙头需要登入Twitter，一段时间内空投0.1 测试币。有了一定数量的测试币，就可以进入官网（网址：https://scroll.io/alpha）进行交互。

转换后，可以玩一些Scoll上的DApps（https://ethdenver.scroll.systems/）。

2. zkSync

zkSync是以太坊Layer 2项目，2019年由Matter Labs开发，使用零知识（ZK rollup）技术来解决以太坊当前的可扩展性问题，被看作是以太坊扩展的“最终游戏”。zkSync从a16z、Dragonfly、Lightspeed 等公司筹集了超过 4.5 亿美元，是受资助最多的L2公链项目，被看作是加密领域最受支持的项目之一。根据Matter Labs的说法，已有超过150个项目已经部署或愿意部署在该链上。这些先行者带来了6500万美元的 TVL。

当前zkSync项目处于Baby alpha阶段，预期在2023年第一季度进入Fair Onboarding Alpha阶段。与此同时，团队将开发 SDK 和区块浏览器，同时提供漏洞赏金计划或文档改进。大众关心的代币空投或会在此开始，以吸引“矿工”发现漏洞并维护、完善网络。

zkSync直到现在也没有发布任何有关其代币和代币经济学的信息，但据团队声称，30%的代币将分配给团队和投资者，其余的将用于社区和生态系统的发展，意味着剩下的70%将分配给普通用户。作为生态体量庞大的L2 ZK项目，对它的进展关注不容错过。zkSync官网：https://zksync.io/zkSync Twitter：@zksync

3. StarkWare

StarkWare StarkWare 创建于2018年，作为一种 L2 扩展解决方案，利用ZK-STARK证明来减少发送到以太坊区块链的信息量并缓解区块链拥塞。与zkSync一样，SnarkNet也是利用ZK-Rollup技术来实现这一点。Starkware从Paradigm、Pantera、Sequoia、以太坊基金会等机构筹集了2.85亿美元以上融资，其生态系统也丰富、庞大。

去年，StarkWare 已经在链下铸造了100亿个Token，分配给不同的用户群体。由于公链去中心化的特质，Token奖励是对所有人开放的。但核心基础设施和dApp的开发人员，以及为生态系统的安全和健康做出贡献的人员更容易获得官方给予的Token奖励（贡献者获得Token的方式可通过协议自动接收）。

当前也有圈内人示分享了普通人如何通过简单的交互（交易、链接钱包等）获得代币空投机会，

官网：https://starkwaresessions.co/

Twitter：@StarkWareLtd

4. Aztec

Aztec是以太坊上隐私优先的 ZK-rollup，为以太坊生态系统带来保密性和成本节约。项目最近宣布获得了1亿美元的B轮融资，由a16z领投，并表示，Aztec的加密和隐私对于区块链获得广泛采用、人际协作、资本、社会以及下一代互联网至关重要。

Aztec生态系统当中已经有很多可以使用的产品，ZK.Money便是Aztec 项目知名的也是首个DeFi隐私基础设施。目前官方没有任何关于Token的信息，但是像A16Z等风投巨头参与其中让圈内人士猜测其代币推出时间也将不远。当前普通用户可以做的是与ZK.Money（官网：https://zk.money）项目进行交互。

第1步：进入官网连接钱包获取密钥，存入至少0.1ETH；

第2步：使用ZK.Money的DeFi 功能，转到http://zk.money/earn选项卡（当前页面功能暂停），尝试几种投资选择（即Aave，Lido或ElementFi）；

第3步：点击http://zk.money/balance上的“交易（Shield）”，粘贴您的别名，选择金额并确认（大概3h到交易进行，是批量执行的）。

第4步：在2个钱包之间汇款• 转到http://zk.money/balance，单击“发送到 L2”并添加收件人• 然后确认交易，这些资金在Aztec L2内部私下发送（就像在 zkSync 中进行交易，不会发生在以太坊主网上）

第5步：提现至ETH L1转到http://zk.money/balance ，单击“提款到 L1”，然后添加收款人

5. Polygon（zkEVM）

Polygon收购Hermez Network（ZK汇总）后，形成了自己的Polygon zkEVM，它以透明的方式执行以太坊交易，包括具有零知识证明验证的智能合约，$MATIC代币为其网络提供动力。Polygon 目前有275个兼容的 DeFi 项目，总TVL 为1.77亿美元。这使得Polygon在活跃协议当中排名第三，其TVL排名第六。

在加密领域让人兴奋的项目，Polygon一定不能缺席。它既是各大知名品牌合作较多的项目，其原生代币MATIC也在熊市期间也表现的很好。据CoinDesk消息，Polygon zkEVM测试版主网络上线时间或将选在3月27日，用户可在此之前和主网进行交互。

测试网站：https://public.zkevm-test.net/

结语

ZK赛道发展迅速，以上5个都是进入2023年以来最值得关注的项目，提前了解并参与其中或许能从中收获惊喜。

但ZK赛道早期阶段，无论其技术、市场、应用等领域，都仍有待市场证明。Web 3.0面临的掣肘包括有限的可扩展性以及高延迟。若没有足够的交易吞吐量和足够低的成本，Web 3.0很难容纳来自主流市场的容量。这既是Web3的面对的问题，也是零知识赛道项目面临的问题。

作为技术的乐观派，我们相信未来ZK赛道会有更多的发展和应用场景。2023年，对ZK赛道的展望，那就是：随着技术不断进化，更多的自由会在更低成本的信任下被释放出来。
