如何快速判断 NFT 的价值？

不同的 NFT 投资者拥有不同的方法论，大多数人会从对应 NFT 集合的交易数据层面来做出判断，也有部分人会观察社交媒体/社区中对应 NFT 话题的热度。

但两者的区别在于，交易数据在区块链上，公开透明，因此也催生了一系列 NFT 数据网站。而代表社交媒体/社区热度的数据维度极为复杂，我们对其的判断也只能局限于一个大致的区间。这里的痛点在于「如何能够总结出一个行之有效的社交媒体数据标的」。

采集 Twitter KOL/巨鲸数据的数据网站 TwitterScan 便是为此而生。

之前在社群中流传的一个段子提到：「我们可以通过 TwitterScan 来判断 NFT 的相应价值 ：17 个 KOL/巨鲸关注，价格在 0.4E 以内都算合理；当 KOL/巨鲸变为 60 个时，1E 的价格是合理的；如果到了 100 个以上，1-3E 是合理的；超过 300 个，10-20E 是可以接受的。」

这也不由得让人好奇，TwitterScan 是完成数据归集的？

起底 TwitterScan

众所周知，Twitter 是全球各行各业 KOL 发声的重要阵地，当然也包括加密行业，几乎每一个加密协议和其创始人都在 Twitter 上开设了账户，甚至 Twitter 创始人也是 BTC 的忠实拥泵。

Twitter 中的数据是繁杂但宝贵的。因此，针对 Twitter 进行数据收集的工具网站便更显得弥足珍贵。

接下来，我们不妨全方位浏览 TwitterScan 的全部功能，了解其对于我们在投资决策中的重要性。

TwitterScan 是一个由 MetaScan 推出的 NFT 数据类网站，标语是「Find NFT Gems&Trending Tokens」，网站主要分为三个板块，「Token」、「NFT」和「KOL」。

Token 板块分为四块内容：

  • Trending Tokens ：在 Twitter 中被提及最多的代币排行；

  • Crypto KOL ：KOL、Builder、VC、项目方及平台等关键意见领袖提及了哪些 Token，以及提及的次数；

  • New Projects ：新项目的链接、Twitter 和创始人 Twitter 账号；

  • Trending Topics ：Twitter 关注加密的话题热度排行；

NFT 板块分为三块内容：

  • Upcoming&Whales ：即将来临的 NFT 集合和巨鲸的持有分布，TwitterScan 将巨鲸分为了三个等级，还将 ETH 和 Solana 也做了区分。如今，TwitterScan 已经将 API 免费提供给了一些 NFT 数据网站，比如 NFTGo；

  • Chrome Plugin ：NFT 稀有度、地板价和鲸鱼的数量及持有率，支持几乎所有的 NFT 市场；

  • NFT Whale 持有信息 ：在进入 NFT 集合的页面后，我们可以看到有哪些 NFT 鲸鱼们关注了该 NFT 集合 Twitter、持有对应 NFT 集合的鲸鱼，以及鲸鱼们在推文中提及该 NFT 集合的次数；

Twitterscan 把分布在 Twitter 上的加密 KOL 发布的观点、讨论等信息进行了整合，然后结合链上数据，以结构化的方式向我们呈现，最终达成帮助普通投资者更好地投资加密货币的目的 。

而 TwitterScan，也只是 MetaScan 所构想的商业版图的一环。

Meta 的中文翻译是「元」，也就是向上跃升一层去审视现状，或者从一个高视角统筹研究底层的事物和规律。「元」是本质本身，然后所有东西再从「元」中衍生出来。

MetaScan 希望在 Twitter 和链上数据的基础上，将 Web2 和 Web3 紧密地连接起来，做一个庞大且全面的中间件生态，用来服务更多的 Web2、Web3 的品牌方和用户。

Web2 到 Web3 的跨越

对于 MetaScan 而言，TwitterScan 是一个很好的抓手，是一个不错的流量入口。在规划中，MetaScan 希望可以拓展 TwitterScan 在 NFT 领域的更多应用，将更多的流量汇集在一个池子中。

MetaScan 团队提到，「NFT 市场拥有着比 FT 市场更大的潜力，未来市场大小可能会是 FT 市场的几十倍以上。 」而这也是 MetaScan 将重心放在 NFT 领域的原因。

除了对于 Twitter KOL 数据的汇总外，TwitterScan 已经推出了免费的 NFT 白名单抽奖工具，项目方可以设定一些任务（关注转发 Twitter、持有某个 NFT、账户余额以及加入 Discord 等）来实现增长的目标。

另外，TwitterScan 还做了一款功能强大的浏览器插件。插件能够在包括 Opensea、Looksrare、Mintverse 等 NFT 交易市场的页面中为 NFT 投资者提供多维的数据 ，帮助投资者更清晰地评估该 NFT 项目。

其次，插件还支持用户对于 Twitter KOL 数据的查询 ，数据维度包括影响力分数、观点有效性、关联性和矫正性。我们也可以看到该 KOL 与其他 KOL 的互动。

未来，我们还将看到TwitterScan插件在Twitter上更多的功能，包括该博主的身份（KOL、VC和项目）以及其所发布的投票和活动、该博主最新动态和关联事件和该博主在TwitterScan上所达成的分数 。分数将会以里程碑的形式呈现，也可以用这些分数来与其他博主进行横向对比。

更重要的是，不同于同赛道的数据/功能性工具，TwitterScan 本身便具备流量入口的属性 。TwitterScan 在为 NFT 项目做抽奖的同时，还能够为其提供更高的曝光量和增长。并且，最重要的是，这种提升是零成本的。

其次，TwitterScan 所展现的信息是基于 Crypto Twitter 数据的集成，它可以轻松地统计出 KOL 们在社交媒体的数据表现，用数字来具象化每个 KOL 的影响力，并在 KOL 和项目方的沟通中起到承接作用。

而用户也可以一键低门槛地参与项目方的各类抽奖。TwitterScan 本身也会汇总在 Twitter 上对应 NFT 集合的抽奖，并提供给用户，增加最终的中奖概率。

在更远一些的未来，MetaScan 计划发行 PASS NFT，针对 C 端用户构建链上声誉系统、KOL 鲸鱼追踪及链上行为信息归集等功能。针对社区构建，MetaScan 也会推出更多工具性质的社区产品，比如 DAO 工具、投票 to Earn、社交图谱工具等。

而针对项目方和品牌方，MetaScan 将针对增长和品牌推出各类功能性的服务和产品 。我们上文提到，TwitterScan 可以为增长的需求方和 KOL 进行高效的对接。在产品层，MetaScan 正在着手准备 NFT 发行工具和 NFT 版税变革。

Web3 到 Web2 的赋能

以上内容，我们主要描述的是 MetaScan 从 Web2 社交媒体数据出发，面向 NFT 市场和加密市场推出的产品和服务，以及在其之上的衍生构建。

但他们的愿景远不止于此。

Web2 的数据应用只是一个流量入口，MetaScan 在 Web3 的应用领域大多可以归于投资决策、社区增长和品牌建设这几个方面。

除了推出 NFT 工具和推动 NFT 版税变革外，MetaScan 希望将自己在 Web3 品牌营销和运营中的丰富手段和经验，再带回到 Web2 世界的商业模式中。

对于传统 Web2 消费品牌而言，Web3 的一切都是新奇物件，而针对 Web3 世界的拓展也是必要的。一来是因为新的营销手段会为品牌获得更大的曝光量和市场影响力，二来 Web3 相关的营销手段会帮助消费品牌获得加密群体的青睐。

这样的例子也不在少数。8 月 1 日，Tiffany 宣布将推出 CryptoPunks 的衍生 NFT，售价 30 ETH。NFT 购买者也会收到与其对应的实物：定制吊坠。显然，Tiffany 此次 NFT 营销所针对的群体便是加密富豪。

之前，李宁也曾购买过 BAYC NFT，并推出 BAYC 联名衣物。这个行为更多也是在讨好加密群体，以及一些购买了 BAYC 的名人们的粉丝。

MetaScan 的优势在于它本身就是一个拥有流量优势的 Web3 基础服务设施，且在该领域拥有丰富的经验和实践。凭借这样的优势，MetaScan 完全可以帮助 Web2 各类品牌进行 Web3 营销，比如发行 NFT/链上积分/链上凭证等形式，帮助其获得新的客户群体以及更多的市场影响力。

MetaScan 的愿景

对于普通加密用户和投资者而言，MetaScan 所推出的 TwitterScan 是一个很好的数据分析标的，有助于投资决策的制定。

而作为 Web2 与 Web3 之间的中间件，MetaScan 在做的其实是通过做一套完整的工具生态来解决用户在区块链上的刚需，比如 TwitterScan。在这些工具导流的作用下，MetaScan 完全有能力构建一个庞大的链上声誉系统，从而推动区块链技术在社交领域的大规模采用，以及降低用户在链上的操作门槛。

如果我们再提升到更加宏观的视角来看，MetaScan 是一个来自 Web3 世界的布道者，通过使用 TwitterScan 等工具类产品将流量汇集在一起，在流量的加持下，适用 Web3 的新兴技术，为来自 Web2 世界的朋友们展现 Web3 世界的奇妙。

就像一本书的标语「Think Big, Start Small, Move Fast」所描述的，MetaScan 拥有一个宏大的构想，却从一个小的，但足够有趣的产品开始。TwitterScan 的功能实用且针对，能够迅速捕获流量，为以后宏图的实现打下地基。

而接下来，就是 MetaScan 的「Move Fast」。

NFT PASS 发行在即：TwitterScan 如何打通 Web2 和 Web3 世界？

TwitterScan 即将发行 PASS NFT，持有者将享受以下功能：

  • TwitterScan 高阶产品功能；

  • 作为 TwitterScan 未来的身份系统门票；

  • 优先获得在 TwitterScan 平台进行抽奖/推广的 NFT 项目的白名单，比如之前大火的 Mimic Shhans NFT 集合；

  • 优先注册 TwitterScan「.nft」DID 去中心化身份系统；

….

TwitterScan PASS NFT 将会在链上声誉系统和身份系统中发挥重要的作用。TwitterScan 将会为 PASS NFT 持有者聚合他们在 Web3 世界的声誉信息，并为其提供用以展示工作经历、学业、技能以及社区互动等内容的影响力空间 。

链上声誉系统的构建本质上是对链上用户行为的系统化汇总和记录，类似于 Web2 世界中的行为记录，但链上行为会更加透明且真实。而链上声誉系统的应用也是广泛的，例如通过链上行为，例如用户在产品使用中的痕迹，来构建对应产品的社区。这样往往能捕获产品真实的用户，由真实用户组建的社区也会更具价值。

链上声誉系统将会被应用于 MetaScan 宏大的商业版图中，用 Web3 的手段帮助 Web2 的品牌们组建社区，筛选真实用户，从而实现增长 。链上声誉系统将会是 MetaScan 在推动 Web2 和 Web3 融合的重要工具。而在这种融合的过程中，期待 MetaScan 能够为我们带来更多的惊喜。
