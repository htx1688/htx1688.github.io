头条

▌ 游戏产业成元宇宙发展突破口：上半年收入占比达94%

9月10日消息，近日，游戏产业分析机构Sensor Tower发布的《元宇宙概念下的移动游戏市场洞察》（以下简称“《报告》”）显示，2022年上半年元宇宙应用共营收6.5亿美元，其中游戏应用收入占比高达94%。

▌数据：目前有超过83%的以太坊节点准备好以太坊合并

金色财经消息，据ethernodes.org数据显示，目前有超过83%的以太坊节点准备好以太坊合并，预计以太坊合并时间为9月15日10:18。此前报道，昨日，以太坊主网完成最后一次影子分叉。

▌数据：A股共26家公司涉足数字藏品，但仅3家披露相关营收

金色财经报道，数据显示，截至目前A股中共有26家上市公司涉足数字藏品，其中有12家推出了自己的数藏平台，不过只有3家公布了数字藏品相关的营收情况，分别是曲江文旅、浙江富润、视觉中国。整体而言，目前数字藏品营收占上市公司总营收的比例微乎其微，很多上市公司的数字藏品业务是纯粹投入但很少有营收。

政策

▌俄罗斯银行Sberbank将允许用户在其区块链平台上发布NFT

9月10日消息，俄罗斯最大的银行之一的Sberbank现在打算允许用户在其区块链平台上发行NFT，该金融机构还计划与全国各地的艺术场所和画廊合作。Sberbank在今年3月获得俄罗斯中央银行发行数字金融资产的授权后创建了区块链平台。该平台目前仅对法律实体开放，但在2022年最后一个季度，个人也将获得访问权限，并允许发行、购买和出售数字金融资产（DFA）。（news.bitcoin.com）

区块链应用

▌以太坊主网第13次影子分叉已完成

9月10日消息，以太坊客户端Nethermind发推称，Nethermind所有节点已顺利完成以太坊主网第13次影子分叉（Mainnet-Shadowfork-13）的过渡，也就是合并前的最后一次影子分叉。这标志着以太坊网络已为迁移至PoS共识机制做好准备。（Cointelegraph）

▌距离在ETH合并还剩30000个区块

金色财经报道，距离ETH合并到权益证明（POS）之前，还剩30000个区块。

▌Cardano钱包Daedalus 5.0.0主网版本发布

9月8日消息，Input Output发推称，Cardano钱包Daedalus 5.0.0主网版本已经发布。该版本支持节点版本1.35.3，并且与Vasil兼容。 该版本还修复以下问题：自动更新后没有删除旧版本Daedalus的可执行文件；在授权/兑换ITN奖励过程中选择同步钱包时出错；即使用户在费用计算过程中已更改金额，系统仍执行之前发送的代币金额。此外，该版本使用cardano-launcher替换darwin-launcher.go进程映像。 Input Output还提醒称，Daedalus是全节点钱包，仅提供PC端和苹果Mac应用程序。如果用户看到“mobile Daedalus”（移动端Daedalus），这将是一个试图窃取用户种子短语和ADA的骗局。

加密货币

▌数据：过去一周Circle USDC流通量减少3亿美元

9月10日消息，据官方消息，9月2日至9月9日期间，Circle共发行21亿美元USDC，赎回24亿美元USDC，USDC流通量减少3亿美元。截至9月9日，USDC总流通量为517亿美元，储备量为519亿美元，其中现金109亿美元，短期美国国债410亿美元。

▌DeFi 借贷协议 Sentiment 完成 240 万美元种子轮融资，Archetype Ventures 领投

9月10日消息，DeFi 借贷协议 Sentiment 宣布完成 240 万美元种子轮融资，Archetype Ventures 领投，参投方包括 Castle Island Ventures、Matrixport Ventures 等。 Sentiment 允许借款人针对其资产创建杠杆债务头寸，这些头寸可用于与整个生态系统中的其他应用程序进行交互。贷方为协议提供流动性。

▌Google搜索已可显示以太坊合并倒计时

9月10日消息，据Google搜索引擎界面显示，搜索the merge、ethereum merge关键词，将直接在搜索页面显示以太坊合并倒计时、以太坊主网合并总难度（TDD）等信息。

▌CoinGecko联创：LUNC过去一周飙升或与#3568和#4159提案通过有关

9月10日消息，CoinGecko联合创始人兼首席运营官Bobby Ong表示，Luna Classic（LUNC）过去一周的飙升可能与3568和4159提案通过有关，这些提案重新启用了区块链上的授权和质押，并为每笔交易引入了1.2%的税率，预计1.2%的税收将在9月20日区块高度 9,475,200上生效。

▌重庆江北文旅委回应数藏平台涉嫌非法集资：已开展调查

金色财经报道，近日一则重庆市江北区文化和旅游发展委员会公布的《关于XX数字科技有限公司涉嫌非法集资问题的情况回复》的文件在网上流传，重庆市江北区文化旅游委关工作人员表示：“该文件是真实的，该公司曾在我们这儿做过艺术品的登记备案，目前我们针对公司经营艺术品的相关行为展开了调查，如果有违纪行为将会查处。涉及到非法集资或者网络诈骗的行为，我们已经移交给公安和市监管局。” 据爆料信息，被举报的数字藏品平台为宙藏，据天眼查APP，该平台由重庆元藏数字技术有限公司于2022年4月推出，注册资本为300万。（蓝鲸财经）

▌8 月全球区块链行业融资额环比下降21.52% ，NFT 领域逆势增长467.86%

金色财经报道，据统计，8月份，全球区块链领域融资总数为103笔，较7月份环比下降30.4%；8月份融资总额为12.91亿美元，较7月下降21.52%。其中，融资上亿美元的项目显著下降，只有两笔融得3亿美元。从区域来看，8月份美国区块链领域共有28个项目获得3.53亿美元，成为全球获得融资项目数量和金额都最多的国家，中国有2个项目获得融资4670万美元。从赛道来看，Web3.0领域融资保持稳定，区块链基础设施领域成为融资额度总额下降最多的领域，NFT领域成为融资环比增长最多的领域。（财联社）

重要经济动态

▌阿根廷今年通胀率或达95%

今年以来阿根廷的通胀水平不断攀升，今年前7个月该国累计通胀率已达46.2%。阿根廷中央银行9日发布的市场预期调查报告显示，预计今年的通胀率将达95%。报告预测，2023年和2024年阿根廷通胀率分别为84.1%和63.1%。此前，8月11日，阿根廷央行将基准利率提高9.5个百分点至69.5%，这标志着阿根廷政府对通胀采取了更强硬的立场。

▌ NFT 有哪些特点

NFT的属性使每一个NFT都是唯一的和真实的，这解决了艺术和奢侈品行业的大问题。另一个特点就是所有权属于用户。在区块链游戏中，游戏中的资产归玩家所有，即使游戏宕机或服务器关闭，玩家依旧持有NFT资产，另外也可以转换成代币在其他游戏中使用。
