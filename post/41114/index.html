熊市是积累财富的最好机会。游客（投机者）都已离开，现在正是淡季。这是建设者们埋头工作的时候，也是投资者对下一个大趋势下注的时候。

最近的牛市周期见证了 DeFi 和 NFTs 的崛起，这两个赛道都出现了大量的创新、兴趣和资本。

那么，下一轮呢？下一个牛市的关键主题和趋势将是什么？

预测未来很难，但这里有一些启发性思考。

我们知道，对于新人来说，看着价格暴跌并不容易。即使对经历过牛熊的老手来说，看着钱越来越少也绝非易事。

虽然我们是为技术而来，对吗？

熊市通常被称为是发财的契机，但这是为什么呢？

这是因为与项目的活动、发展和效用相比，以极低的价格买入项目的机会在整个加密领域中大量存在。要选择正确的项目并不容易，但机会是明确的。

这篇文章并不是致富指南，但如果你在这个熊市中花时间学习和观察，你很可能会处于有利地位。

这篇文章将概述加密市场表面正在冒着泡的东西，以及下一次大牛市中，哪些赛道将引领冲锋。

1. 采用L2

在区块链领域，有一个明确的、当前的推动力，即实现扩展和增加交易吞吐量。在上一轮牛市中，我们看到了围绕 Solana 和其他 Alt L1 等项目的大量兴趣和投机。这在很大程度上是因为以太坊还没有适当的规模来处理牛市对其区块空间的需求。

然而，这些区块链中的许多，虽然吹嘘着令人难以置信的速度和吞吐量，但持续的宕机让许多用户感到愤慨，或者在需求高的时候表现不佳。最终，这些区块链中的许多都败下阵来，因为它们是单体的，不是模块化的，它们试图在一个单一的产品中提供所有的安全、去中心化和吞吐量。

这在大规模的情况下是行不通的，而且这一点非常明显（观点有偏差，但得到了充分的支持）。那么，提供高吞吐量、高安全性和高去中心化的解决方案是什么？

进入 L2

Layer 2 解决方案提供了增加吞吐量的能力，从而提高了区块链的扩展性，同时继承了主链的安全性。像 StarkNet 和 zkSync 这样的解决方案能够实现 “批量处理”，将成千上万的交易捆绑成单一的交易，并将其发布到主链（以太坊）。这将每笔交易的成本降低了几个数量级，并提高了链上可以处理的速度和交易总量。

那么，为什么这如此神奇，为什么我们应该关心？为了让这项神奇的技术渗入主流，并作为新互联网的核心技术被采用，我们需要它快速、安全和去中心化。

L2 解决方案解决了速度问题，而主链则提供了安全性。

这将使广泛的用例能够按照我们需要的标准运作，以便实现世界性的采用。这方面的例子包括每秒处理数以千计的微交易的视频游戏，即时和低价分享的媒体，以及一眨眼就能完成的金钱交易。

如果没有 Layer 2 的解决方案，很可能在很长一段时间内都无法将区块链的潜力完全发挥出来。如果我们想获得采用，我们需要提供一个层，以便在允许大规模采用的情况下轻松进行交易 –Layer 2 提供了这个核心需求。

目前，Layer 2 解决方案，如 zkSync、Starknet、Polygon Hermez、Optimism 和 Arbitrum，都在积极开发，且有着大量的进展，很快就会达到采用所需的基本要求。

2. DeFi 和 NFT 的成熟

上一轮牛市的大部分时间是由两件事推动的：即 DeFi 和 NFT。这两个都是非常强大的工具，但两者都还处于起步阶段 –NFT 比 DeFi 更有优势。

随着市场的平静，现在是时候让这些产品完善它们的能力，并重新测试产品的市场适应性。

那么，为什么去中心化金融会促成下一个牛市的到来？

DeFi 是一个开放的金融系统，任何人、任何地方、任何数量的钱都可以参与。

这不仅为无法获得银行服务的人提供了大量的机会，而且也使各种新的金融科技得以成长和成熟。问题是，“去中心化借贷处于早期阶段。它不能提供中心化借贷的一切，但它胜过传统金融的低效，并接近被 Web3 吸引的创造性思维”。
- Bankless

现有的中心化金融系统（CeFi）是一个由银行和政府控制的带着围墙的巨大花园。他们选择谁能进入，谁能得到多少钱。即使你满足了他们的所有要求，他们也仍然可以轻易地把你拒之门外，而且经常如此。

DeFi 借贷协议和服务避开了这种许可的门槛，提供与银行目前为每个人提供的相同的金融服务。

它是 100% 透明和去信任的。每个人都可以查看对方的筹码（通过他们的地址匿名），以及根据他们自己的标准评估债权人/债务人的财务健康。这意味着你可以像以前一样层层剥离，拥有一个了解 DeFi 新时代银行内部运作的窗口。

这开辟了一个金融创新的世界，引入了新的效用、新的机会，以及围绕可以赚钱、借钱或放钱的人的明确性。

但是 NFT 呢？那些不只是 jpeg 吗？

为什么它们会引致下一个牛市的出现？

如果你这么想，那你就大错特错了。NFT 是通往真正的数字所有权世界的特洛伊木马。

在上一轮牛市中，猴子的照片和像素化的朋克统治了整个宇宙。而这只标志着一个开端，是数字所有权的基本效用的简单展示。

当我们从实体店购买一件衬衫时，它是你的，只有你拥有那件特定的衬衫，而且你有收据可以证明这一点。它在任何意义上都是“非同质化的”。

然而，在 NFT 出现之前，这一点在数字领域是不可能实现的。你当然可以在网上购买 X、Y 或 Z 的收藏品，但它们是存储在卖家公司租用的服务器上的。有了 NFT，资产就存在于你控制的钱包里。

这项技术能够并且很可能颠覆的行业包括游戏、音乐，以及房屋抵押贷款和货币贷款。其范围之大难以想象，下面概述的将引领我们进入下一个牛市的一些其他产品和解决方案，都使用了这种关键技术。

总之，关注 NFT 领域的创新和那些正在努力发挥其全部潜力的团队。

3. 采用 Web3 单点登录

单点登录 web3 认证允许在数以百万计的新网站和现有的主要新闻机构和平台上使用单一的密码和账户，且不需要附加个人详细资料。

另一个不断出现在新闻中的巨大话题是隐私和数据。像 Meta、亚马逊和谷歌这样的公司几乎拥有你的身份。他们知道你的一切 – 你的密码、出生日期、SSN，以及你在互联网上的一言一行。

他们说这是安全的，但你的数据被入侵过多少次？这基本上是《西部世界》级别的数据收集，大多数人只是允许它发生，因为目前还没有更好的解决方案。

数据收集的市场价值数十亿美元，而这就是这些公司赚钱的方式。据 Datamation 数据显示：

“2021 年全球数据收集市场的价值为 16.6 亿美元。估计到 2028 年将达到 82.1 亿美元，复合增长率为 25.6%。”

这些价值万亿的公司正在从事个人数据销售业务。这不仅仅是关于一个 25 岁、白人、住在俄亥俄州的女性的随机数据，而是你的名字、你的电子邮件、你的位置、你购买的产品，且这个清单还在继续。

Web3 单点登录解决了这个问题。它允许公司收集数据，了解你的喜好，并向你提供他们认为你感兴趣的个性化信息，同时保证你的身份安全。

他们只有在你允许的情况下才能拥有这些数据。（关于个性化数据共享的更多信息，请查看 Brave 浏览器）。

在我看来，未来会是这样的：创建一个 web3 钱包/身份，并将其链接到你设有账户的任何一百个网站上。当你访问任何一个网站时，点击一个按钮 –“用 web3 钱包登录”– 就 OK 了。这是一个可以统治所有网站的密码，同时保护你的数据和身份，使你更容易上网。

4. GameFi 的崛起

目前，在传统的游戏环境中，玩家要么预先支付游戏费用，要么按月支付，要么在免费游戏中购买数字资产。

然而，无论玩什么游戏，他们的每一个行动和他们获得的每一个项目/角色都不是他们的。它是由该游戏存储的，如果玩家停止游戏或付费，它就会死死地躺在他们无法控制的服务器中。目前的游戏工作室通常有严格的服务条款，禁止玩家进行任何“真钱交易”。

这意味着玩家在游戏中投入的所有时间和为游戏宇宙增加的所有价值，都不属于他们。目前的系统不允许玩家利用它，如果他们试图规避规则，他们会被永久封禁。

推翻这种市场的时机已经成熟。允许玩家通过可验证的 NFT 对他们投入区块链驱动的游戏的时间和金钱拥有所有权，这将为游戏和玩家创造机会。

我们可能很快就会生活在这样一个世界里，妈妈们不再说 “为什么要浪费你的时间玩游戏”，而是说 “我不明白 Jimmy 在这些游戏上做了什么，但他可以为他所有的东西买单。”

目前，游戏行业的利润比所有其他媒体行业的总和还要高，但直到现在，玩家都是被排除在这之外的。Web3 游戏并不是什么童话般的幻想，它是一个已经在形成的现实。

不幸的是，目前非常缺乏完善的可玩的 web3 游戏。但在这个熊市中，要注意那些可玩性高，且拥有强大社区的游戏。

这些游戏是你想要尝试的。

我密切关注的一些游戏有：

  • Influence：空间战略大型多人在线游戏

  • Parallel：科幻卡牌游戏

  • Loot Realms：经济学和骑士精神的大型多人链上游戏

  • Guild of Guardians：移动 RPG 游戏

5. 音乐 NFT

现如今，一个艺术家要谋生比以往任何时候都难，尽管将你的音乐传播出去的能力比以往任何时候都容易。

为什么呢？简单来说，流媒体服务和唱片公司拿走了大部分的钱，而留给艺术家的所剩无几。

作为一个艺术家，你必须是少数幸运的人，才能被唱片公司选中，或通过社交媒体的算法获得内容收入。然而，即使有了内容创作者的收入，你也需要处于排行榜的顶端，才能持续生活。

在 Web3 中，艺术家可以通过新的货币化渠道、与社区更紧密的联系以及对音乐更充分的所有权，更好地实现他们的音乐货币化。

使用 NFT 盈利可以通过几个不同的途径进行

  1. 出售独家会员制 NFT，让歌迷有机会接触艺术家，例如，演出的 VIP 通道、限量版商品、与艺术家面对面的机会。

  2. 出售收入共享的 NFT，从未来的销售中分享百分比的利润，使艺术家能够自由工作，不受制于与唱片公司和流媒体服务合作时产生的大量繁文缛节。

音乐所有权

艺术家的音乐所有权是目前市场上问题的关键所在。唱片公司拥有大部分利润，然后流媒体服务支付一小部分，然后在艺术家和唱片公司之间不公平地分配。

有了 NFT 和 web3，艺术家可以以 NFT 的形式发布音乐，无论是单曲还是整张专辑，粉丝们可以直接从艺术家那里购买音乐，而不通过唱片公司。

一个更紧密的社区

有了 web3 认证和 NFT，艺术家可以利用小规模的粉丝群来维持生计。你不需要全球观众来购买你的世界巡演 – 你所需要的只是 100 个忠实的粉丝。

一个典型的例子是艺术家 Gramatik，长期以来，他一直支持加密货币、区块链和赋予艺术家拥有自己的音乐目录和自由创作。

释放内容创造者，让他们能够以他们想要的方式，在他们想要的地方创作，同时能够拥有他们自己的观众。

在 web3 中，艺术家和内容创作者不再受他们建立受众的平台的约束，也不再仅仅依靠广告作为收入来源。

能够为他们的追随者提供一个基本的、可能是免费的 NFT，让他们访问你的私人网站、内容、讨论区和社区，将使这些内容创作者能够更接近他们的社区，将他们的追随者带到任何新的平台，并释放新的独特的 web3 盈利机会。

前进的道路上风起云涌

目前还是 web3 的早期阶段，需要大量的开发和用户体验设计来推动我们达到新的水平。

为了支持这个不断增长的生态系统，还需要一些大事情：

  • Layer 2 的解决方案要充分运作，并为基础层提供支持交易数量的能力，以支持大规模的采用和效用

  • 以太坊成功合并，关闭工作证明（PoW），转为权益证明（PoS）。

  • 将新用户纳入这些平台和产品的用户体验提高到与我们今天所享受的 web2 体验相媲美的地步。

我们不知道下一个牛市何时会来。但我们正在努力走出熊市，期待下一个牛市能够提供真正的实用性和大规模采用。
