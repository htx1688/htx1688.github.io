欧易OKX 注册链接 欧易是全球著名的数字资产交易平台之一，主要面向全球用户提供比特币、莱特币、以太币等数字资产的币币和衍生品交易服务。
HTX火币 注册链接 火币全球专业站，是火币集团旗下服务于全球专业交易用户的创新数字资产国际站，致力于发现优质的创新数字资产投资机会。

Web3 早期风投基金 Variant 第一期创始人教育计划总计收到 400 份申请，入选项目涉及 DeFi、基础设施、DAO、社交、治理、安全、NFT 等诸多领域。

近日，Web3 早期风投基金 Variant 公布了入选首批加密货币创始人教育计划「Founder Fellowship」（VFF）的 24 个项目，涉及 DeFi、基础设施、DAO、社交、治理、安全、NFT 等诸多领域。该创始人教育计划为期 3 个月，总计收到 400 份申请。

Variant 是一家早期风投基金，现已投资 52 家 Web3 公司，投资组合包括 Aptos、Aztec Network、Flashbots、Goldfinch、Maginc Eden、Mirror、Matter Labs、Scroll、Polygon、ZORA 等等。目前，VFF 第二期学员申请已开放意向表。

Capsule

Capsule（原名为 Korus）是一个 NFT 市场，面向儿童、青少年和年轻人，允许其从喜欢的角色艺术家和 KOL 处收集可定制的数字卡，创始人为 Stephaine Carroll 与 Frankie Li。

Chroma

Chroma 正在构建一个用于启动和管理 DeFi 借贷市场的全栈解决方案，其创始人为 Jack Longarzo、Tim van Scherpenzeel 与 Aaron Xie。

Clave

Clave 是一个智能合约钱包，2023 年 8 月正式推出，支持使用手机、平板电脑或笔记本等安全元件与生物识别身份验证技术实现无密钥或助记词登陆。Clave 将通过「意图」、会话密钥模块与安全模块等交易模拟器等改变用户与区块链交互方式，支持社交恢复。Clave 将首先支持 zkSync，现已开放白名单。

9 月，Clave 入选以太坊基金会 ERC-4337 账户抽象资助轮的 18 个项目。

Dawn

Dawn 是一个 iOS 上的以太坊钱包，帮助用户以最少的点击次数使用应用程序，允许用户直接从 Safari 浏览器插件连接 DApp。Dawn 基于 iPhone 的 Secure Enclave，钱包私钥会被加密并存储在钥匙串中。默认情况下，Dawn 支持 Face ID 开启钱包。Dawn 已实现 iPad 兼容，支持 Zora、Tally、Snapshot、Daylight、Gitcoin、Uniswap、Aave。目前，Dawn 已支持 10 条链，新增支持 Base Goerli、Sepolia 与 Gnosis。

Digito

Digito 总部位于拉美，允许用户使用相同的私钥签署链上交易和现实世界的文档，跟踪购买付款，实施欺诈保护。其创始人 Amhed Herrera 曾任职于 Coinbase。Digito 将在多米尼加共和国首先启动，已投入 100 万美元来建立受监管的基础设施，预计 2024 年第一季度获得多米尼加共和国第一个许可证。

Doormat

Doormat 是一个专为加密用户设计的密钥管理系统。

Forum

Forum 诞生于一个投资俱乐部，是一款与朋友共同拥有加密资产的一体化应用程序。

Future Primitive

Future Primitive 在 ERC-6551 标准（Tokenbound 账户）的基础上进行创新，将每个 NFT 变成智能合约钱包。

Hashcurve

HashCurve 是一个由社区管理的预测市场平台。

Heimdall Labs

Heimdall Labs 正在构建一个模块化与去中心化的合规层，Heimdall 是一个为加密项目提供合规智能合约解决方案的模块化协议。

Mezzanine

Mezzanine 将公司的优势与 DAO 的优点相结合，将公司治理引入链上。

Neynar

Neynar 是一个面向 Web3 社交协议的开发者平台，可让开发者读取、写入和拥有来自 Web3 社交协议的数据。

Nosh

Nosh 将餐饮店与司机和消费者直接联系起来，省去了外卖平台的中间环节。

Pathfinder

Pathfinder 是一体化的社区激活工具，利用自然语言处理技术，Pathfinder 正在构建一个以人为本的界面，用于发现链上应用并与之互动。

Ponder

Ponder 是区块链应用程序后端的开源框架，开发者可以为以太坊上的智能合约集构建 GraphQL API，具有热重载、类型安全和易于部署到生产环境的特点，解决了自行开发索引器或拼接开箱即用的 API 的后端痛点。

Poolshark

Poolshark 是一种定向 AMM，用户可以通过较少的维护从波动中获利，交易者也可以成为 LP。目前，Poolshark 已启动测试网。

Probably Something

Probably Something 正在创建一些工具，以帮助用户理解复杂的数字生态系统。目前，其产品包括 NFT 工具库 Lifeboat、AI 驱动的聊天机器人 Lifebot。

Rocko

Rocko 专注于简化 DeFi 借贷协议的借贷流程，用户可以获得最适合自己的贷款。Rocko 即将推出。

Seam

Seam 正在建立一个既怀旧又新颖的网络社交媒体应用程序，用户可以设计怀旧的自定义社交档案，并通过完成任务获得 SP。

Shield

Shield 是一家 Web3 安全公司，可简化事件响应，帮助 Web3 公司在遭受黑客攻击后迅速恢复正常。

2023 年 4 月，Shield 宣布完成 210 万美元 Pre-Seed 轮融资，投资方包括 Kraken Ventures、Eterna Capital、Alchemy、Moonpay、Bessemer Ventures DAO（Steel DAO）、Orange DAO、NGC Ventures、Wagmi Ventures、MH Ventures、Coinswitch、2Punks Capital、Lecca Ventures 和 Very Serious Ventures，参与本轮融资的还有天使投资人 Rob Behnke（加密安全审计公司 Halborn 首席执行官）、Peter Liebert（加利福尼亚州前首席信息安全官）、Vijay Maharajan（Bitscrunch 创始人）以及来自 Coinbase、Google、Microsoft 和 FireEye 的天使投资人。

Shield 表示其还得到了 a16z 的 Crypto Startup School 加速器的资助，目前正在参与其中。Shield 旨在帮助组织采取全面的安全措施来防止加密漏洞利用，并提高用户对整个生态系统的信任。Shield 的威胁检测软件可以通过 API 直接集成到任何平台上，或者作为在线社区的安全机器人集成到 Discord 中。

Studio X

Studio X 是一个音乐众筹平台，利用 NFT 为艺术家提供前期资金。用户可先收集一个 Alex Kings EP，若长期持有，未来能够获得艺术家空投。

Troops

Troops 是一款移动优先的网络应用程序，旨在将人们聚集在一起。Troops 使社区能够以一种协作方式集体拥有和定制其链上家园。Troops 的背后是一个开源「社区堆栈」，主要组件是建立在 Safe 基础上的全链权限协议。

Turbo

Turbo 希望通过制作简单易学的工具包和市场，改善独立游戏开发者的体验。
