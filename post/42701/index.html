原文作者：Mercedes Bent，Lightspeed 合伙人

原文编译：深潮 TechFlow

如果你有一个朋友通过 Bumble（约会平台），马上要进行第一次约会。她通常会说，“我已经在网上查过了，他们在大学时是足球校队的成员，在一家初创公司工作，有一只非常可爱的拉布拉多犬，名叫 Dusty，7 个月前在离我几英里的地方买了房子。以防你再也见不到我，这是他的联系方式。”

小小的线上调查可以告诉你很多关于你甚至从未见过的人的信息，但你怎么知道这些信息是真实的呢？MTV 真人秀节目 “Catfish “一次又一次地证明了伪造一个网上身份是多么容易。

那么解决方案在哪？是的，你猜得没错——区块链。

基于区块链的去中心化身份平台将发挥重要作用，使人们能够证明他们的身份。

在去中心化身份的帮助下，在几年或几个季度内，“朋友”还能查看她约会对象的数字身份档案，其中列出了他们的梦想对象属于哪些数字社区，他们参加的活动，他们支持的事业和艺术家，他们拥有的 PFP NFT，以及他们拥有的任何代币化商品。

今天，所有这些数据都被记录在区块链上。人们的加密货币钱包已经揭示了大量关于他们是谁以及他们做了什么的信息——只要你知道他们的钱包地址。而这正是数字身份平台试图解决的问题，如何以一种简便的方式显示一个人在链上的所有活动。

一个真正独特的加密货币消费产品

数字身份平台是加密货币中最令人兴奋的消费产品领域之一，原因如下：

  1. 它们直接利用了区块链技术的核心创新（对谁做了什么的可验证的、不可改变的记录）。

  2. 拥有身份层是互联网中最具战略意义的地位之一——正如苹果、Facebook和谷歌之间为成为Web2的一键式登录之王，而进行十年之久的争斗所显示的那样。（Meta/Facebook早期领先，但现在输给了苹果，苹果进入较晚，但比其他人做得更好。）

  3. 数字身份是加密货币消费类别的一个罕见的例子，它正在利用区块链来做一些真正的创新。

在加密货币社区，已经有很多关于这种新产品潜力的文章。这其实不是一个是否会发生的问题，而是谁会建立它。

许多人认为钱包供应商（如 Metamask、Phantom、Coinbase Wallet）是这个领域事实上的领导者，但我不相信钱包最终会构建我们的去中心化数字身份。

为什么？因为就像银行账户一样，人们会出于不同的目的维护不同的钱包，而且因为越来越多的主流用户在进入加密货币时，根本不想自我保管或有意识地处理钱包。

赢家要么是好玩的，要么是有用的，要么是两者都有。

当我们看一下占主导地位的 Web2 身份供应者是如何出现的，这就很明显了：

  • Google： 搜索和电子邮件的广泛作用和便利性使其成为中心枢纽。

  • Facebook/Instagram： 能够轻松地与朋友在线联系，以及分享照片。

  • AppleID： 能够使用你的 Apple ID 登录到多个设备。

  • LinkedIn： 用数字链接取代离线简历的独特作用。

简而言之，谁能创造出一个有价值的理由来聚合和更新用户资料——要么给人们一个有用的工具，要么创造娱乐价值——谁就能统治区块链身份这个赛道。

一个拥挤的领域：身份权力的游戏

有大量的初创公司在争夺创建 Web3 数字身份平台。

要使数字身份档案蓬勃发展，用户必须有动力去更新和添加内容。

成功的平台会给人们一个很好的理由来使用它们，无论是来自于网络效应（其他人都有，我也需要），现实世界的使用案例（我的资料可以帮助我找到工作或约会），还是经济激励（如果有人发现我的资料很有吸引力，他们会付钱联系我）。

在这之后，更多的实际影响开始发挥作用——比如哪个平台有最广泛的分布，谁对链上活动有最完整的看法，谁有能力将广告和营销费用与档案联系起来，谁有最引人注目的现实世界用例，谁有最大的领先优势，以及谁不占领Web3身份市场会有最大的损失……

像 Zerion 或 Zapper 这样的投资组合聚合器（在多链世界中）或像 Coinbase 或 LemonCash 这样的 CeFi 金融技术公司（在可互操作的跨链世界中），会成为我的首要选择。

以下我认为谁能在这个领域取得更好成功的案例（以人类情感的需要为视角）：

1.CeFi 金融科技公司（Coinbase，LemonCash）： 这些企业已经为新用户购买加密货币提供了主要的体验，可以说是区块链的第一个杀手级产品。Coinbase 从 2018 年或更早的时候就开始了数字身份的工作，许多 CeFi 金融技术公司同时实现了法币进出和自我托管钱包。

情感挂钩：对拥有和积累财富的需要。

2.投资组合聚合器（Zapper，Zerion）： 他们可以全面了解你在多个钱包中的持股情况，并且已经有了非常高的参与度。

情感挂钩：当持有的代币上涨或下跌时，多巴胺会发挥作用，对金融安全和保障的需求，在一个地方可以看到所有持有的代币上涨。

3.域名服务（ENS，Unstoppable Domains）： 这些已经是人们用来分享他们的加密从属关系的公开资料。

情感挂钩：完成与建立的需要。

4.身份、社交和数据协议（Lit、Ceramic、Lens、Arweave、CyberConnect）： 这些开源协议主要是整合到消费者身份档案的中间件，但如果其中任何一个决定进入应用层（Lens 已经在试验配置文件），它们可能成为一个巨大的竞争者。

情感挂钩：对安全的需要。

5.钱包（Metamask, Phantom）： 许多 dAPP 需要的现有数字身份层，使你能够登录。许多钱包还提供隐私功能（公钥和私钥）以保护其内容。

情感挂钩：对自我拥有的需要。

6.Web2 社会身份（谷歌、Facebook/Instagram、LinkedIn）： 如果一个新的数字身份标准围绕加密货币起飞，他们的损失最大。

情感挂钩：对完成、影响、联系的需要。

7.市场平台（Magic Eden，OpenSea）： 他们已经拥有涉及 NFT 持有的任何商业交易的主要数据。可以说，NFT 比可替换的代币更能分享我们的身份信息。

情感挂钩：自我表达和创造力的需要，对刺激、联系的需要。

8.证明协议（POAP，ProjectGalaxy）： 一个关于非金融加密活动的独立信息来源——如参加活动或社区成员——创造了一个新的身份层，这在Web2中是不存在的。

情感挂钩：对归属感，联系和影响的需要，对成就和建设的需要。

9.Web3 社交网络（DeSo, Bitclout, Entre）： 现有的社交资料，尚未产生巨大的娱乐价值或获得巨大的牵引力。

情感挂钩：自我表达和创造的需要，影响和联系的需要。

10.代币门槛平台（CollabLand，Mintgate）： 它们已经作为基于代币或 NFT 社区的认证工具。

情感挂钩：归属感、联系和影响力的需求。

11.票务平台（Afterparty, Tokenproof）： Tokenproof 实际上是一个所有权证明平台，但在票务用例方面已经看到了成功。这些平台有另一种形式的非金融加密货币数据，没有被其他平台捕获。

情感挂钩：归属感，联系和影响的需要。

12.支持广告的平台（Layer3, Brave）： 他们拥有强大的用户黏性，他们已经准备好与外部各方分享数据。

情感挂钩：对自己的数据进行控制并确保其安全的需要。

13.信息传递（Dialect、XMTP）： 我们如何联系其他人。

情感挂钩：对联系和归属的需要。

掌控我们的身份

一个合理的问题是：如果所有链上活动都有不可改变的记录，那为什么用户需要更新或管理他们的身份资料？这不是会自动发生吗？

答案是，隐私。

人们不会被迫分享他们在网上所做的一切；数字身份只是意味着你在网上所做的一切（以及你想分享的）都是可验证的。

因此，能够确保使用约会软件 Bumble 的朋友不会被骗。她可以相信她的约会对象是软件上写得那个人，同时还可以留下足够的神秘感来保持趣味。
