原文作者：Karen，Foresight News

在竞争激烈、爆发在即的 Layer 2 赛道中，由 Coinbase 基于 OP Stack 构建的 Layer 2 扩容方案 Base 于 7 月中旬向开发者开放主网，并在 8 月 10 日正式公开发布。

技术、爆款、生态、激励（预期）是一个 Layer 2 能否获得长期可持续性发展的必要条件，四者缺一不可，环环相扣。

技术方面，Base 依托于目前头部 Layer 2 Optimism 的开源模块化工具包 OP Stack，OP Stack 允许开发者、项目方根据自己的需求场景定制 Layer 2 网络，从而接入以太坊网络并共享安全性和资源。

而就爆款而言，Base 在经历过生态上 Meme 代币 BALD 走出一夜千倍神话后，吸足了财富效应，虽在之后又当头一棒上演几秒崩盘戏码，但从主网尚未公开发布前 Base TVL 就突破 1 亿美元便可看出热度之高。主网上线后，社交应用 friend.tech 作为 Base 链上首个病毒式社交 DApp，主打「你的社交网络就是你的净资产」，一夜之间刷屏 Twitter（X）和风靡加密领域，通过社交裂变扩大自身影响力的同时也为 Base 增色不少。

再加之潜在空投预期，引发了 Base 交互热潮。作为一个主网启动一个月、完全开放不到两周时间的 Layer 2 来说，Base TVL 数据十分亮眼。L2 BEAT 数据显示，当前，Base TVL 已达到 2.4 亿美元，在 Layer 2 中排名第五。

本文将为读者整理汇总 Base 生态。据不完全统计，当前，Base 生态项目数量已超过 100 个，规模相对完善，其中基础设施占据半壁江山，包括但不限于桥、On-Ramp、钱包、预言机、节点提供商等；Base 原生 DeFi 项目也占据了很大比重的锁仓量和成交额，也有 Uniswap、Sushi 等不少多链 DeFi 项目拓展过来；当前 OpenSea、ZORA 等主流 NFT 市场和铸造工具也已集成 Base 主网。

基础设施

跨链桥

除 Base 官方桥（仅支持在以太坊和 Base 间跨链 ETH）之外，Base 生态跨链桥包括 Orbiter Finance、Axelar、Bungee、Hop Protocol、LayerZero、Stargate、Wormhole、Debridge、跨 Rollup 桥 Owlto Finance 。Base 生态门户里列出的 Across Protocol、Celer 目前暂未支持 Base 主网。

Onchain On-Ramp

On-Ramp 被认为是法币与加密货币兑换或转移至链上钱包的重要渠道。当前 Base 生态 On-Ramp 项目包括：

Coinbase Pay： 支持 60 多种法定货币以及 100 多种加密货币。

Moso： 是一款 Web3 购物助手，以加密货币形式奖励在线购物的用户。Moso 表示，Moso 与 Base 的 Layer 2 区块链技术的集成将增强整个购物体验。通过 Moso 在受支持的在线商家进行的每笔交易都将安全地记录在 Base 区块链上。

Onboard： 允许将数字资产转换为当地法币，支持尼日利亚货币以及银行转账和手机钱包两种支付方式，未来几周将添加更多货币和付款方式。

Onramp Money： 已集成 Base，允许通过即时银行转账和本地支付方式以印度卢比、土耳其里拉、墨西哥比索、迪拉姆等购买加密货币。

Transak： 允许用户可以在应用、网站或网络插件中购买 / 出售加密货币，可通过 155 个国家 / 地区的卡、银行转账和其他支付方式在 75 多个区块链上使用 170 种加密货币。

钱包

OKX Web3 钱包 已支持 Layer 2 网络 Base，并在 Discover 板块同步支持 Base 生态的 DApp，用户无需手动配置，可一键切换至 Base 网络，通过 OKX Web3 钱包无缝交易、存储和管理 Base 生态资产，以及探索 Base 生态的 DApp 等。

Privy 为 Web3 产品带来了 Web2 级别的用户体验（例如使用电子邮件和社交登录），并允许用户链接任意数量的外部钱包，以及提供开箱即用的集成工具。社交应用 friend.tech 就使用了 Privy 的 MPC 钱包。

Uniswap Wallet 新增支持 Coinbase L2 网络 Base，允许用户在 Base 网络上兑换代币和连接到 Base DApp。

去中心化货币项目 Eco Protocol 背后团队 Eco 推出了一款名为 Beam 的加密钱包，让用户能在 Optimism 和 Base 等 Layer 2 网络上进行支付。Beam 集成账户抽象（EIP-4884）等功能，使钱包能够充当智能合约，并支持多因素身份验证和钱包社交恢复等一系列功能。

其他支持 Base 的钱包包括 Bitget Wallet、Brave Wallet、Coinbase Wallet、Safe Wallet、Trust Wallet、TokenPocket  等。

DAO

DAO 解决方案提供商 Aragon 和 XDAO 、Nouns DAO 构建的社区筹资基础设施 Prop House 、DAO 探索和支持协议 Tally 已支持 Base。

其他

WalletConnect  作为使钱包和应用能够安全连接和交互的通信协议，也已支持 Base。

预言机方面，Chainlink、Pyth Network 和 RedStone Oracles  已与 Base 集成，支持 Base 开发人员访问相关源和其他 Web3 服务。

上线 Base 主网的工具和基础设施还包括 Ankr、Blockdaemon、QuickNode 和 Blast 等节点提供商，Basescan 和 Blockscout 等区块浏览器，The Graph、Goldsky 和 Covalent 等数据索引器以及 Thirdweb 开发者工具，以及加密货币托管技术提供商 Fireblocks 。此外，区块链数据分析平台 Dune、Nansen 也已支持 Base。

DeFi

根据 DefiLlama，截止撰稿时，当前 Base 上 DeFi TVL 为 1.7 亿美元，其中，几款原生 DeFi 项目的 TVL 占据主导地位，包括 BaseSwap （5457 万美元）、Alien Base （2410 万美元）等。

 来源：DefiLlama

按交易量来看，过去 7 日，Base 生态中交易量达到 4 亿美元以上，DackieSwap 占据领先地位， 7 日交易额达到 3 亿美元，其次为 Sushi 和 Uniswap ， 7 日交易额均在 4000 万美元以上。

来源：DefiLlama

DEX

Uniswap、SushiSwap、Balancer、AMM 协议 Maverick Protocol、WOOFi、DODO 跨链聚合器 DODO X 已集成 Base。DEX 基础设施 0x 也宣布 0x Swap API 已在 Base 上线，首先聚合 Uniswap、Balancer 和 Baseswap 之间的流动性，将在额外的流动性来源上线后对其进行整合。其他上线 Base 的 DEX 包括：

DackieSwap

DackieSwap 分叉自 Uniswap V3，是 Base 生态原生 DEX，提供兑换以及 Launchpad 服务，还允许通过集中流动性做市商（CLMM）赚取更多奖励。DackieSwap 创始人为 Andrew Eddison（推特账号：@BaseOnAndrew）。

DackieSwap 已被 DefiLlama 收录，不过暂未出现在 Base 官网生态门户。根据 DefiLlama 数据，DackieSwap 近一周的交易量已达 3 亿美元，主要交易对为 WETH/USDbC（2.8 亿美元），不过锁仓额仅有 170 万美元。

考虑到 DackieSwap 已经开始进行 IDO，再加之 Base 刚上主网，提醒用户注意安全。

BaseSwap

根据 DefiLlama 数据，BaseSwap 当前是 Base 上锁仓额最高的 DEX，截止撰稿时，TVL 达到 5000 万美元以上，占据 Base 1.26 亿美元锁仓额的近 42% 。

BaseSwap 允许进行资产兑换、流动性提供、挖矿，还提供了一个 NFT 市场平台。不过，BaseSwap 尚未被 Base 官方生态门户收录，提醒用户注意安全。

Alien Base

Alien Base 目前是 Base 上 TVL 第二高的 DEX，当前 TVL 超过 2100 万美元。Alien Base 的流动性挖矿计划允许用户提供流动性和质押 LP 代币来赚取 ALB 代币。此外，质押 ALB 代币也能赚取奖励和平台收入，平台费用的最多 50% 将归 DAO 所有。下图是 Alien Base 的路线图。

Alien Base 未被 Base 官方生态门户收录。

SwapBased

SwapBased 也是 Base 上的一个原生 DEX 协议，根据 DefiLlama 数据，SwapBased 锁仓额为 1931 万美元。

除流动性提供（0.25% 的交易费用）、单边质押等服务外，SwapBased 还提供永续合约交易服务。SwapBased 也未被 Base 官方生态门户收录。

借贷

8 月中旬，借贷协议 Compound 已在 Coinbase L2 网络 Base 上线，支持抵押 ETH 和 cbETH 来借入跨链 USDC（Base 上的 USDbC）。

Aave 社区关于「在 Base 上部署 Aave V3」的提案已获得通过，目前相关激活提案将于 8 月 19 日结束。

其他已上线或者将上线 Base 的借贷协议包括 Moonbeam 上借贷协议 Moonwell 、杠杆收益策略和贷款协议 Extra 、跨链借贷协议 Pike Finance 等。

其他

Beefy.Finance 是一个多链收益优化器，目前支持以太坊、Polygon、BNB Chain、Arbitrum、Optimism、Base 等 17 个网络。Beefy 的 Vault 投资策略通过自动复利（自动认领、兑换和重新存入奖励）来帮助用户解锁更多的收益。

其他还有合成资产协议 Synthetix 、波动性协议 Volmex 、定投工具 Mean Finance 、收益聚合协议 Harvest  等 DeFi 项目也（将）上线 Base。

NFT

NFT 市场 OpenSea、ZORA、Showtime、Reservoir 、生成艺术平台 Highlight 、NFT 铸造平台和工具 Manifold 、mint.fun 已支持 Base。此外，专为会员和订阅而构建的 NFT 开源协议 Unlock Protocol 也已集成 Base 主网。

此外，可口可乐还在 Base 上推出 Masterpiece 艺术收藏品，汇集了标志性艺术品和新兴作品，所有这些都与可口可乐瓶交织在一起。

社交

friend.tech

friend.tech 作为 Base 链上首个病毒式社交 DApp，主打「你的社交网络就是你的净资产」，使用 Privy 的 MPC 钱包，通过将用户 X（原 Twitter）帐号绑定的方式，让用户可以通过以太坊购买任意 friend.tech 上用户的份额（Share），获取与其直接私密对话的机会，并且还有可能可以从中获得回报。

值得一提的是，friend.tech 将从本周五开始向应用测试人员空投奖励积分，计划分 6 个月（每周五）分发 1 亿枚积分。积分是在测试期间链下收集的，当应用正式发布时，将赋予积分特殊用途。

截止撰稿时，在 Base 上推出测试版不到 10 天后，friend.tech 独立用户数已突破 6.8 万，累计交易笔数超 110 万笔，总交易量接近 3 万枚 ETH。

CyberConnect

Web3 社交层 CyberConnect 本月宣布 CyberAccount 已在 Coinbase L2 网络 Base 上线，将 ERCC-4337 支持的智能账户引入 Base。

Warpcast

Warpcast 是去中心化社交协议 Farcaster 客户端，Warpcast 上的 NFT 已支持以太坊、Base 和 ZORA 上展示。

Blackbird

Blackbird 是一个专为酒店业构建的 Web3 平台，专注于通过忠诚度和会员服务与其客人之间建立直接联系。 

2022 年 10 月份，Blackbird 完成 1100 万美元种子轮融资，Union Square Ventures、Shine Capital 和 Multicoin Capital 共同领投，Variant、Circle Ventures 和 IAC 等参投。

其他支持 Base 的社交网络包括 Friends With Benefits（FWB ）、Web3 社区活动平台 Galxe  等。

游戏

根据 Base 官方，Animoca Brands、Game 7 DAO 、Web3 游戏解决方案提供商 ChainSafe Gaming 、区块链游戏公司 Faraway 、NFT 科幻卡牌游戏 Parallel 、冒险和竞技游戏 Pixelmon 、字谜游戏 Words 3  、Yield Guild Games 等也为其生态公司或项目。

其他

Guild.xyz  是一个管理基础设施，允许针对社区已使用的平台进行自动会员管理。Guild.xyz 正在将 Guild 模型引入到 Base 上构建的项目中。

0x Splits 模块化智能合约用于安全高效的链上支付，允许用户可以分割 NFT 销售收入、收回费用、实现收入来源多元化、预扣税款等。

其他支持 Base 的平台包括去中心化内容发布平台 Mirror  等。

需要强调的是，目前 Base 生态尚处于起步阶段，多数项目暂未披露安全报告也未公开团队信息，建议用户谨慎参与。
