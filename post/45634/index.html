Coinbase Ventures 成立于 2018 年，是加密货币交易所 Coinbase 的风险投资基金，致力于推进加密与 Web3 行业发展，对加密货币和区块链技术中所有类别的早期项目进行广泛投资，提高全球经济自由度。

Coinbase Ventures 是在 2018 年前 Linkedln 副总裁 Emilie Choi 加入 Coinbase 担任业务副总裁兼首席运营官后的产物，她帮助成立的企业发展团队意识到应在熊市期间努力建设，投资早期加密公司并建立牢固关系，从而促进加密生态系统发展。在近五年的时间里，Coinbase Ventures 投资组合已经增加到近 400 个，遍及全球 40 多个国家。

按照自己的定义，Coinbase Ventures 将其投资组合划分为五个类别：Protocols & Web3 Infrastructure（27% ）、DeFi（22% ）、CeFi（19% ）、Platform & Developer Tools（16% ）与 NFT / Metaverse / Gaming（16% ），基本囊括与加密生态相关的各个方面。值得注意的是， 2021 年加密市场的牛市效应促进了 Coinbase Ventures 投资版图的迅速扩张，它们在过去一年的总投资数量接近 150 笔，平均每 2.5 天就有一笔新的投资。而 2022 年至今，Coinbase Ventures 再度完成 150 笔左右的投资，仅一季度便完成超 70 笔投资。

尽管其覆盖面较广，但仍可从其投资组合中一窥 Coinbase Ventures 对整个加密生态发展的宏观考量。首先，Web3 基础设施、协议的投资占比最大，这反映了 Coinbase Ventures 对加密生态的基本认知与态度。其次，Coinbase Ventures 最为关注的应是市场与增长，其对 DeFi 与 NFT 的市场、借贷、流动性及资产管理投资较多；开发者工具更偏向能够简化开发人员体验的项目，以降低 Web2 开发者进入 Web3 的门槛。此外，Coinbase Ventures 还对能够带来更具爆发性增长的 CeFi 较为重视。

OpenZeppelin / Forta 是一家区块链安全公司，孵化了智能合约安全实时监控项目 Forta，通过独立节点运营商对所有交易进行扫描，以发现异常交易和对智能合约的威胁。当检测到问题时，节点运营商会通知订阅者潜在的风险，这使他们能够采取自动化的防御措施。

Sardine 是一个数字防欺诈解决方案与加密结算平台，为金融科技和加密公司提供即时、无风险的 ACH（自动清算系统）结算，使其提高资金的运转效率，减少资金延迟和支付欺诈的风险。作为连接加密资产与现实货币资产桥梁，Sardine 能为传统企业入局加密行业赋能。

数据分析 & 索引

Amberdata 是一家数字资产数据提供商，为金融机构提供有关区块链网络、加密市场和 DeFi 的数据和见解，以促进机构对数字资产的采用。

Blocknative 是一个实时 Web3 基础设施，提供 「mempool」（内存池）的数据浏览器，包括地址追踪、内部交易追踪、未成功交易的信息、被替换交易的信息等。其中，Gas 费检测器从区块链角度来查看内存池中交易的 Gas 费，保证所提供数据的及时与准确。

Coin Metrics 是一款链上和链下均可使用的加密市场数据分析工具，提供市场可视化工具、数据反馈、指数、价格参考率和指标。

Covalent 是一个区块链数据查询和索引平台，它提供一个统一的 API，允许开发者在多个区块链上重复使用查询。目前，支持 Ethereum、BNB Chain、Avalanche、Ronin、Fantom、Moonbeam、Klayth、HECO、SHIDEN 及主流 Layer 2 网络。

Dune 是一个基于社区的区块链数据分析平台，Dune 最早向用户开放自主查询，拥有最大的分析师团体和用户社区，掌握 PostgreSQL 编写能力的分析师可以自由利用链上数据写出自定义查询、分析以及可视化。现阶段，Dune 已支持 Ethereum、Polygon、BNB Chain、Solana、Optimism 和 Gnosis Chain 的数据查询。

Etherscan 是一款免费的以太坊区块链浏览器和分析工具，用于审查和分析存储在以太坊上的所有数据。它允许用户通过地址、交易哈希、区块高度、代币名称等搜索和检查历史信息，提供额外的资源，如 ETH 单位转换器、标签词云、验证合约工具等。

Flipside Crypto 是一个社区支持的 DeFi 和区块链数据分析项目，与 Dune Analytics 一样，它通过可视化的工具以及自动生成的 API 接口，让用户能够通过简单的 SQL 语句进行复杂数据的查询，也可以复制和编辑别人已经生成的 SQL 查询。

Hyperspace 是一个基于 Solana 区块链的数据分析网站，旨在构建 NFT 交易聚合器，帮助用户在主流 Solana 链上 NFT 市场购买 NFT，并使得所有交易价格信息都能在一个地方浏览访问。

Messari 是一家总部位于美国纽约的加密数据分析和研究机构，公司利用开源工具构建一套区块链数据库，其中存储的数据信息包括各个区块链项目的资金、开发历史、管理架构、以及供应计划等，希望帮助投资者、监管者和公众了解加密货币这个革命性的新资产类别，并建立数据工具推动知情决策和投资。

Nansen 是一个区块链数据分析与研究平台。在众多数据板块中，最受欢迎的两个数据面板是代币上帝模式（Token God Mode）和钱包分析器（Wallet Profiler）。此外，还有多链数据板块、DeFi 天堂和 DAO 分析等呈现多样化的加密数据。

Nomics 是一个加密货币数据分析平台，向机构加密投资者和交易所提供专业级市场数据 API。

Nxyz 是一个多链索引产品套件，通过低延迟的 API 公开实时区块链数据。目前，Nxyz 已经支持了超过 4000 个 DApp ，并为其提供了具有不同模式和路由系统的各种链接口。

Spindl 是一个链上追踪监控工具，可通过分析 Discord 帖子、Reddit 论坛、广告和其他互联网链接，将这些数据与购买、销售和交易等链上行为配对，以创建帮助 Web3 产品了解流量来源的配置文件。

The Graph 是一种用于查询以太坊和 IPFS 等网络的索引协议，任何人都可以构建和发布称为子图的开放 API，从而使数据易于访问。

社区管理 & 治理

Bonfire 是一个 Web3 社区管理工具，目前正在构建基础设施，使创作者能够构建属于自己的由社交代币和 NFT 赋能的品牌平台，释放线上社区所有权。

Samudai 是一个 DAO 管理平台，计划在 2022 年底公测，并为 DAO 开发一套完整的生产力套件，包括 Web3 原生项目管理框架、团队图和分析仪表板，以及构建一个 Web3 专业人员名单。

Utpia Labs 是一个为 DAO 提供协作式工资和费用管理的操作系统。

Backdrop Labs 是一个旨在连接代币化社区的 Web3 项目。

Commonwealth 是一个共同讨论、投票、资助项目的一体化链上社区管理平台。

Otterspace 旨在通过让 Web2 的用户清晰、直接和熟悉 DAO 的加入和贡献体验，从而帮助 DAO 扩大自己的人数，从一百个成员到一个百万成员。

Snapshot 是一个链外多治理客户端，无需手续费，它的特点是结果易于核实且难以争议，支持各种投票方法，能满足各种组织的需求。

Disco 是一个用于管理 Web3 身份和凭证的工具。

Galxe 是一个 Web3 凭证数据网络，通过凭证数据正在构建一个基于 Web3 的去中心化身份系统。

Incode 是一个由生物识别技术和计算机视觉驱动的身份验证（IDV）和认证平台。

Namebase 是域名 Handshake 开发商，Handshake 顶级域名通过链上 Vickery 拍卖进行分发，最高出价决定拍卖的获胜者，出价最高的人只支付第二高出价。

Spruce 是一家数字身份认证系统，其产品套件为用户可移植声誉所需的身份验证、凭证和存储提供支持，提供对数据的去中心化访问控制以及 Web2 API 和 Web3 之间的互操作性。

Unstoppable Domains 是一个 NFT 域名提供商，致力于为地球上的每个人创造数字身份，而这些数字身份完全归用户所有。

zCloak 是一个零知识证明隐私 DID 项目。

支付

Data Mynt 是一个全球加密货币支付处理网关，用户可以使用安全、合规的加密资产作为货币支付和金融运营基础设施。

Diagonal Finance 是一个 Web3 订阅付款应用。

Elph 是一个旨在为开发人员构建工具以在其应用程序中接受加密货币支付，并为消费者提供存储、管理和交易其加密货币的工具。

MobileCoin 是一个隐私支付协议，专注于为移动应用提供安全、隐私的加密货币钱包服务。

Coherent 是一个提供 Web3 数据的核心基础设施，允许开发者通过 API 使用其数据构建应用，未来 Coherent 将专注于链上交易历史、NFT 数据和凭证三类数据。

Diem 是一个 Meta（Facebook）的稳定币项目，原为 Libra，旨在建立一个值得信赖的创新金融网络，为世界各地的人们和企业赋能。今年 1 月，Diem 协会确认会将其知识产权和其他与 Diem 支付网络运营相关的资产出售给给加密基础设施公司 Silvergate Capital。

Espresso 是一个 Web3 隐私系统，允许资产创建者定义可定制的隐私策略，以确定在某些情况下谁可以看到有关资产持有者和交易的哪些信息。

Metawork 是一个永远在线记录代码执行各个方面的系统，特点是高速低成本，并且数据保存在个人终端上。Metawork 旨在帮助每一个人从任何环境中获取任何记录，从而更快地理解代码及其工作原理。

Nova 是一个专注于法律和合规工作流程的平台，旨在帮助企业消除文书工作，简化合规检查，允许跨行业提供商合作，帮助团队提高运营效率。

Paxos 是一个受监管的区块链基础设施平台，稳定币 USDP 和 BUSD 的发行方，其产品及服务主要包括 Paxos 加密经纪服务（Paxos Crypto Brokerage）、稳定币服务、PAXG（PAX Gold）、itBit 交易所、证券结算服务、大宗商品结算服务。

Questbook 是一个加密捐赠平台，可连接项目方和开发者，允许项目方向开发人员提供赠款，并以高效和透明的方式投资。此外，QuestBook 还致力于帮助公司在全球范围内创建小型基金，拓宽开发人员寻找工作机会的方式。

Syndicate 是一个创建投资型 DAO 的平台，旨在创建一种新的投资经济模式，降低创建投资型 DAO 的门槛与成本，它支持代币、NFT 等链上投资组合和区块链领域之外的各项资产，以及 DAO 法律实体、DAO 法律文件、银行账户等法律基础设施。

Radicle 是一个去中心化代码协作协议与主权代码基础设施，使开发者能够通过建立在 Git 上的点对点网络安全地协作开发软件。

Sunscreen 是一个隐私初创公司，旨在方便工程师构建基于全同态加密的应用。

Thresholds 是一个为 Web3 公司提供 DeFi 资产管理的 API，旨在帮助用户去除交易所和托管供应商的中间程序，开设法币账户、创建多签钱包后即可享受 DeFi 质押与收益，更快实现法币与加密货币、加密货币之间的无缝交易与兑换。

Tokensoft 是一个代币销售平台。

Worldcoin 是一个基于人格证明的隐私保护协议，其创始人是顶级硅谷孵化器 Y Combinator 的前总裁 Sam Altman，试图通过名为 Orb 的设备采集用户虹膜解决女巫攻击问题，从而创建一种新的全球数字货币。

Livepeer 是一个建立在以太坊上的去中心化视频转码网络。

Nfty Chat 是一款旨在打造「Web3 版本的 Discord」的 Web3 交流平台，包括直接发送信息，社区创建与发现社区。用户可直接与朋友聊天或创建私人 / 代币加权制社区。在 NftyChat 中，个人身份绑定钱包，无需账户密码即可免费使用。

Unlock Protocol 是一个 Web3 社交协议，使创作者能够以完全去中心化的方式通过几行代码将其内容货币化。

Rally 是一个社交代币发行平台，帮助创作者为其社区创建社交代币，用于交易、转移、激励社区成员。创作者可以设置社交代币持有者独有的权利。创作者和粉丝之间通过社交代币实现更亲密的连接。
