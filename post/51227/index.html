5 月 14 日，于葡萄牙举办的 ETHGlobal Lisbon 主题黑客松正式落幕。

本届黑客松最终筛选出了 10 家决赛入围项目，涉及了 AI、社交、NFT、DeFi、零知识证明等多个领域，值得一提的是，结合了 GPT 类的项目本次独占三席之多。这些项目将共同分享由 ETHGlobal 联合 Polygon、Worldcoin、Optimism、1inch、Uniswap 等 25 家项目提供的 275000 美元奖金。接下来，Odaily 星球日报就带大家一起来了解下这 10 家入围项目。

Credential Corgi

获奖主页：https://ethglobal.com/showcase/credential-corgi-a 7 a 0 r

Credential Corgi 是一个基于 GPT 4 和零知识证明的凭证发行及证明工具，其可作用的场景包括：支持平台创建凭证标准，支持发行与 ZKP 兼容的凭证，第三方可与凭证持有者进行证明交互。具体来说：

  • 机构平台可以利用 Credential Corgi 发布凭证的结构标准，结合 GPT-4 的自动生成能力，无需编写具体的技术规范；

  • 发行方可在机构平台上发行与零知识证明（ZKP）兼容的凭证，并可在不泄露隐私信息的前提下将其记录在链上；

  • 第三方可使用 GPT-4 来验证凭证，也无需编写技术规范；

  • 凭证持有者可使用 ZKP 完成这些验证请求，无需透露任何其他信息。

Credential Corgi 最大的特点在于可利用 GPT-4 将凭证的语言描述转换为结构化的 JSON 数据，从而简化了平台、发行者、第三方验证者的工作。

Dookies

获奖主页：https://ethglobal.com/showcase/dookies-0 g 13 u

Dookies 是一个基于 Cookie 创建的个性化、隐私型广告投放系统。

Cookie 是保存在客户机中的简单的文本文件, 这个文件与特定的 Web 文档关联在一起, 保存了该客户机访问这个 Web 文档时的信息。它可以帮助我们实现记录用户个人信息的功能，网站因此也更了解用户喜好用以推送更精准的广告信息。

Dookies 使用了 Sismo Connect，可在保护身份隐私的前提下帮助用户根据其所属的群组（比如 Aave 贷款者、Uniswap 贡献者等）来生成一个 Cookie。

Dookies 主要由 Cookie 仪表盘、Dookies 钱包及广告商仪表盘组成，从用户处获得 Cookie 后可投放更精准的广告，一方面可从广告商处获得收益，另一方面又可以为使用 Dookies 钱包的用户链上交易提供赞助。

buddy

获奖主页：https://ethglobal.com/showcase/buddy-t 7 gbs

buddy 是一个基于 Lens 协议创建的社交类 Dapp，其特点为有着与 tinder 几乎一样的 UI 设计，开发者可以在这里找到志同道合的合作伙伴。

buddy 的运转依赖于底层的社交协议 Lens，其会通过 Lens 获取来所需要社交数据，包括利用该方法找到附近也在使用 buddy 的其他 Lens 用户。

LocalPro

获奖主页：https://ethglobal.com/showcase/localpro-xckra

LocalPro 主要用于解决全球性组织与散落各地的劳动者之间的管理难题，其所提供的基础设施可利用区块链来为各地的线下工作提供链上的验证和激励。

随着世界之间的联系愈发紧密，对于异地零工的需求不断提高，但如何检验、管理这些零散的跨区域工作对于组织而言已成为了一项挑战。LocalPro 使用无需信任的身份和性能验证将全球组织与独立的本地劳动者联系起来，通过帐户抽象解决了链上协调的潜在复杂性，提供更好的用户体验，同时使用 ZK-privacy 与 PolygonID 来确保数据隐私和合规性。

在 ETH Lisbon 上，LocalPro 展示了从入职到结算的用户端潜在流程，详情可点击主页内的视频观看。

CryptoBureau

获奖主页：https://ethglobal.com/showcase/cryptobureau-u 1389 

CryptoBureau 旨在使用零知识证明来建立信用评分系统，从而在 DeFi 世界实现非足额抵押借贷。

CryptoBureau 基本的运作方式为，其会根据客户提交的证明来给出相应信用评分，进而影响所需要抵押物的多少，如果用户恶意欠款，其他协议参与者将能够检查出该行为并拒绝额外贷款。

目前，CryptoBureau 的套件部署在 Polygon Mumbai 上，因为大多数 ZK 技术在其他测试网上尚不可用。

Pinky Protocol

获奖主页：https://ethglobal.com/showcase/pinky-protocol-vsos 2 

Pinky Protocol 是一种点对点、无抵押的 NFT 借贷协议，该协议会利用集成至租借方钱包中的插件来限制 NFT 的转移，从而消除清算风险。

具体来说，NFT 的提供方可以在 Pinky Protocol 上列出其想要租赁的 NFT，并设置好租借期限和费用；租借方则可以选择他们想要租赁的 NFT，预先支付利息后即可借用。Pinky Protocol 会在租借方的钱包中添加一个插件，以防止 NFT 的转移，该 NFT 在借贷期间不能离开租借方的钱包，从而保证了 NFT 本身的的安全性和所有权。

Web3-gpt4 

获奖主页：https://ethglobal.com/showcase/web3-gpt 4-jkaja

Web3-gpt4 是一个基于 Chat GPT4 构建的智能合约管理工具，旨在简化开发者与智能合约的交互方式。用户在聊天界面中输入 Solidity 代码，选择与 EVM 兼容的区块链，该工具即可处理其他工作。

Web3-gpt4插件无需用户身份验证，可通过本地托管的开放 API 轻松访问和安全处理用户数据，其可实现的效用包括：

  • 添加合约：允许用户添加新合约，帮助跟踪各种感兴趣的合约；

  • 删除合约：允许用户从他们的列表中删除不相关或不需要的合约；

  • 查询合约：提供指定智能合约的全面概述，以便于管理。

Donation Station

获奖主页：https://ethglobal.com/showcase/donation-station-g 09 ok

Donation Station 允许用户通过其所提供的捐赠合约在 Layer 2 上为 Gitcoin 捐款，从而大幅减少所需的 gas 成本（最新一轮 Gitcoin 捐款只支持主网，许多捐赠者因 gas 过高选择了放弃）。

针对本次黑客松，Donation Station 已在 Linea、Optimism、Polygon zkEVM、Scroll 和 Mantle 上部署了简化的捐赠合约，用户可以通过浏览器扩展来打开后台脚本，进而与这些合约进行交互。

ChainGPT

获奖主页：https://ethglobal.com/showcase/chaingpt-and-accountability-for-ai-agents-doquk

ChainGPT 是一个基于区块链的去中心化、可验证的聊天 AI。

ChainGPT 使用了斯坦福的 Alpaca 大语言模型，与无法确定回复内容是否正确的 ChatGPT 不同，ChainGPT 引入了“验证器节点”的概念来校验回复信息的真实性。

ChainGPT 回复的安全性源自于其会在多个“验证器节点”上运行，这些节点可以是家用电脑，它们会一起工作从而确保每次回复的正确性。如果两个验证器节点之间的结果存在查询，ChainGPT 会在链上进行调解，如果没有差异，也会在链上提供相应的对比情况。

op-kompressor

获奖主页：https://ethglobal.com/showcase/op-kompressor-1 uboo

op-kompressor 是一组允许中继器压缩 calldata 的智能合约。

op-kompressor 的功效为，通过修改对运行长度的编码方案来在解压缩以太坊的 calldata，进而节省在 Optimism 等 Layer 2 上进行事务处理的 gas 消耗。
