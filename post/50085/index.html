巴黎时间4月20日上午10点半，法国奢侈品巨头 LVMH （路威酩轩）集团 在巴黎卢浮宫卡鲁塞尔厅（Carrousel du Louvre）召开了2023年度股东大会，集团主席/首席执行官 Bernard Arnault 、首席财务官 Jean-Jacques Guiony ，总经理 Antonio Belloni  等高管出席会议。

除了对2022年以及2023年一季度LVMH集团业绩的回顾之外，《华丽志》还在会议记录中发现了以下值得关注的要点：

  • 管理层或将发生更多变动

  • 赞助巴黎2024年奥运会事宜“正在讨论之中”

  • 重启小众制表品牌 Gérald Genta，首批100枚表已经售罄

  • 我们只会用 NFT卖真正的鞋和真正的酒

管理层或将发生更多变动

在大会上，Bernard Arnault透露不久将对集团管理层进行一次大变动。他证实了最近的传言，称LVMH 时装集团（LVMH Fashion Group）现任首席执行官 Sidney Toledano 即将调换职位。“我们将以另一种角色继续合作。 我不能再说更多了，但我们很快就会宣布一些事情。无论如何，非常感谢Sidney作为我们集团的负责人，多年来所做的工作，其中包括许多品牌，尤其是 Celine ，”

这一消息引发了人们对Sidney Toledano接班人的猜测，Michael Burke是最 热门的人选，他在担任了路易威登首席执行官十年后于今年初卸任。Arnault也对Burke也是赞赏有加：“我向Michael Burke表示祝贺。在过去的10年里，他的工作非常出色，是他把Louis Vuitton带进了奢侈品第一梯队。 ” 

目前，路易威登由Pietro Beccari领导，他此前是迪奥（Dior）旗下高级时装业务部门 Christian Dior Couture 的总裁兼首席执行官。与此同时，Bernard Arnault 的大女儿、担任了9年的路易威登产品副总裁的 Delphine Arnault 接任 Christian Dior Couture 总裁兼首席执行官；自2018年起负责 Christian Dior Couture 商务活动的执行副总裁 Charles Delapalme 晋升为总经理。“在这项非常重要的任务中，他们将继续Pietro的路线，”Bernard Arnault在股东大会上说道。

上图：Pietro Beccari（左）；Delphine Arnault（右）

此外，Bernard Arnault还在股东大会上透露，集团美容部门将“再次发生重大变化”，现任LVMH Beauty 执行总裁兼董事总经理 Claude Martinez 将转到新产品和新研究中心的研发活动中。

目前，LVMH美容部门由Stéphane Rinderknech 执掌，“我衷心祝愿他成功，在他的管理下，我们的标志性品牌在许多国家大获成功。”Bernard Arnault说道。

赞助巴黎2024年奥运会事宜“正在讨论之中”

Bernard Arnault在股东大会上表示，集团尚未签署与2024年巴黎奥运会赞助相关的合同，但谈判正在进行中。“合同……正在讨论中，就像所有的讨论一样，最终可能会签署，也可能不会，”Arnault在回答一位股东的问题时表示。

知情人士表示，与巴黎2024年奥运会和残奥会组委会的谈判只涉及LVMH旗下的几个品牌，包括香槟和集团旗下的明星品牌Louis Vuitton。

不过，这位知情人士说，LVMH旗下的手表和珠宝品牌可能不会参与其中，因为Swatch集团旗下的Omega（欧米茄）长期以来一直是奥运会的官方计时伙伴，直到2032年。

**重启小众制表品牌 Gérald Genta，首批100枚表已经售罄 **

前不久，LVMH宣布重新推出已故传奇制表师 Gérald Genta 的同名腕表品牌。此次重启并非重新发行 Gérald Genta 品牌的旧表款，而是从设计师的档案作品中汲取灵感，创造出面向21世纪的 Gérald Genta 手表，在保持较小体量的同时专注于高级制表工艺。

“这确实是一个标志性的品牌，” Bernard Arnault称赞道。Gérald Genta的重启工作目前由Bernard Arnault 24岁的小儿子、Louis Vuitton （路易威登）现任手表营销和产品开发总监 Jean Arnault 主导。

“对此我不会说太多，但一个有趣的事情是，虽然我们还没有展示产品，但已经推出了第一个版本，总共将生产100块，都已经售罄 。”Bernard Arnault在股东大会上表示。

作为世界上最好的制表工坊之一，Genta 曾开发过包括 Audemars Piguet（爱彼表）的Royal Oak（皇家橡树）、Patek Philippe（百达翡丽）的 Nautilus（鹦鹉螺）、Omega（欧米茄）的 Constellation（星座）、Vacheron Constantin（江诗丹顿）的 Overseas（纵横四海）和 IWC（万国表）的 Ingénieur（工程师）等在内的传奇表款，被誉为“钟表毕加索”、“20世纪制表业最伟大成功背后的设计师”。

“所以Gérald Genta是有史以来最具创意的制表师。我们很幸运能够买下他的品牌，连同整个品牌历史。 我们对它未来的发展非常有信心。” Bernard Arnault说道。

我们只会用 NFT卖真正的鞋和真正的酒

在股东大会上，Bernard Arnault再次强调LVMH不出售虚拟运动鞋 ，但他表示，“当我们有NFT的时候，我们会用它来卖一只真正的鞋，一瓶真正的酒。”

“你可以用NFT买一瓶酒。酒庄会把这瓶酒，无论是干邑还是别的什么酒，保存在它的酒窖里。当你决定把它卖回给酒庄，或真的享用它的时候，你的NFT虚拟物品就变成了实物。这就是它的工作原理。所以（NFT的背后）有一个真实的产品被出售 。” Bernard Arnault解释道。

这也与LVMH一直在Web 3领域所做的努力不谋而合：通过将虚拟产品与实物商品连接，重新定义物理与数字间的界限。

LVMH 旗下的许多奢侈品牌都曾深入涉足 Web 3.0领域，包括集团旗舰品牌路易威登推出的“Louis: The Game” NFT 游戏；迪奥开出的元宇宙体验店以及元宇宙展览“在路上（ON THE ROAD）”；芬迪与比特币硬件钱包制造商 Ledger 合作推出的 Web 3.0配饰；轩尼诗推出基于 NFT 的 Web3 社交俱乐部；泰格豪雅 接受加密货币支付等等。

早在2021年度业绩的电话会议上，Bernard Arnault就曾表示，“以10欧元的价格销售虚拟运动鞋，我们对此不感兴趣。”，并补充道：“我们必须警惕泡沫”，“我们必须发出警告”。

总经理Antonio Belloni 也表示，“就我们旗下品牌在全渠道的销售和营销而言，融入数字世界可以为创造实体财富做出贡献。” 他表示，数字与现实这两个维度的结合将加强品牌与客户之间的联系，反映了现实中的人际交往与信息共享，为创造独特的客户体验提供了机会。

除此之外，Antonio Belloni还提到，增强数据交互和人工智能将助力LVMH为客户提供更多个性化产品，实现无缝交易。这一举措这将使门店在创意上展开竞争，在社交媒体上分享动态并且培养品牌吸引力。“最有趣的例子就是我们将于下周开业的蒂芙尼全新旗舰店。”

近期，蒂芙尼正式宣布，品牌焕新后的纽约第五大道旗舰店将于4月28日正式对外营业 （下图：OMA 效果图）。这也是品牌自八十三年前（即1940年）入驻这一建筑以来，首次进行全面的翻新。
