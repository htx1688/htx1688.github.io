近期在大盘回调，市场缺乏热点的情况下，friend.tech 异军突起几乎占据了所有的推特版面，这个建立在 Coinbase 支持的 Layer 2 公链 Base 上的去中心化社交应用（SocialFi），受币圈顶级风投机构 Paradigm 的投资并预告了积分空投机制，让不管是散户、KOLs 甚至是行业内的机构投资者都前来共襄盛举参与其中，而跟随着这股热潮也有许多开发者针对 friend.tech 做了许多的衍生工具。

今天 Dr.DODO 就来介绍大家几款 friend.tech 的衍生工具，让你更专业更方便的玩转 friend.tech。

friend.tech 的衍生工具主要可以分成四大类：

1. 第三方前端

  • friendmex.com： 由前 Paradigm 的研究员 @_anishagnihotri 制作，专门为交易 friend.tech key 所制作的，提供用户批量购买 key、查看 key 价格走势图、实时交易看板以及新用户监控看板和价格排名表，功能非常齐全且免费使用，缺点搜寻功能稍弱。

  • risedle.com/friendtech： 由 @risedle 项目方制作的 friend.tech 数据看板，可以查看头部、最受欢迎、最高花费和最赚钱的用户排名表，界面简单明了不过唯一缺点是目前无法在他的前端上交易 key。

  • realfriend.tech： 由 @iam 4 x 制作的网页版 friend.tech 前端，提供各种交易数据包括线图、PNL、实时交易数据以及新用户看板，此外还有狙击新用户的机器人以及聊天室等功能，功能比 friendmex 更多些但缺点是需要购买创办人的 Key 通过验证才能使用。

  • friendtech.pro： 由 @NFTommo 制作，与 realfriend.tech 一样需购买创办人的 Key 才能使用，提供了更多详细数据供用户观看，包括推特追踪发文数据、平均持仓时间以及 friend.tech 每日金流的总数据等等。

  • frentech 95.com： 由 @0x submerged 制作，与前两者一样有付费门槛需购买创办人的 Key，Windows 95 复古风格的交易前端，具备交易看线图与聊天室等等基本功能，功能较阳春但界面简洁。

  • frentech.octav.fi： 由 @OctavFi 项目方为 friend.tech 用户个人制作的数据看板，可以查看个人 key 线图、市值换算、总花费、付出的总税额以及空头积分等等个人数据。

2. Telegram / Discord 交易狙击机器人

  • frenbot.tech（TG Bot)： 目前提供新用户提醒和买卖功能，跟单交易与狙击功能皆未上线。

  • palpaybot（TG Bot)： 功能简单，查询推特并买卖。

  • FriendSniper（TG Bot)： 功能较为齐全，提供新用户提醒以及用户预设狙击，当推特使用者注册 friend.tech 时便狙击买入，还有有跟单交易止盈止损等功能。

  • New Friendtech Listings / FriendTech 10 k bot（TG Bot)： 两个都是追踪新注册用户的机器人，但后者筛选出推特追踪者超过 10 K 的用户才推送。

  • meowl（DC Bot)： 本身为代币交易机器人，近期也增加了 friend.tech 交易以及狙击功能，能通过设定推特名称或是追踪者数量条件设定来狙击新注册用户。

  • FriendTech Feed（DC Bot)： 提供 friend.tech 新用户即时推送并能够通过追踪者数量筛选分类，以及提供即时交易资讯或大额交易资讯等推送内容。

3. 指数或信息聚合器

此类产品又可以分成两种，第一种是市场定价的 Key 与实际意义的 index / etf 不同概念并没有价格锚定，但透过项目方购买头部 key 并且分享资讯与未来的代币空投来让使用者享受到持有头部 key 的价值（不是价格）。目前排名较前面的有以下这几个，通常购买后就有权限进入 TG / DC 群来获得信息的分享，而之后空投的代币算法也各有不同：Herro / Herr 0x, Friend ETF, Friend Index ＆ Metakeys。

第二种则是链上合约平台追踪 friend.tech 链上数据而给出相对应的锚定价格提供用户槓杆做多/空，例如 hyperliquid。

追踪头部 20 的 key 取中位数价格作为标准，而 unidex 与 aevo 则同样的是追踪总 TVL 也就是所有 Key 的价格加总来作为标准。

Source: https://leverage.unidex.exchange/

4. 衍生品

  • perps. :  friend.tech key 永续交易，可以杠杆多空推特上的 KOL ，目前已上线主网测试有 22 个 key 可供交易。

Source: https://www.perps.tech/trade

  • friendlend.tech： friend.tech key 借贷平台，可以抵押 Key 来借出稳定币，产品尚未上线需小心是否为骗局。

  • FTSX： 与 perps. 一样目标是提供 key 永续合约杠杆多空的功能，目前产品与 friendlend.tech 一样尚未上线需注意。

笔者观点

如同笔者曾于八月中旬新公链之旅文章提及的，friend.tech 在拥有顶级风投机构背书后不只是散户会蜂拥而至体验产品，许多开发者们也会争先恐后的在 friend.tech 之上开发衍生的工具 / 项目，而间接促进整条公链的发展，如今 Base 公链随着 friend.tech 的火爆，TVL 也在不断上升。

另外需特别注意的一点是 friend.tech 官方表示空投将会在半年后发生，而目前空投积分的其中一项计算标准是持有的 Key 价值，也就是使用者在 friend.tech 上的锁仓量，因此也出现了许多玩家所谓的 3, 3 玩法，大意就是互相持有 key 不出售，大家一起赚取空投积分，也让那些持仓许多头部 Key 的 index key 具有所谓加速挖矿的功能（使用者持有 index 的 key 赚取积分同时也能透过 index 持有的 key 获得空投代币分润)，而能对 friend.tech 热潮做空的永续合约平台，则可以帮助用户适时的部分对冲风险减少实际出售所需承担的交易税。

还有一点，随着 friend.tech 赚钱效应越来越好，一些尚未加入的顶级 Kol 只要一加入 key 便会被私制机器人们疯狂购买，以至于使用公开机器人往往会挂山顶买入，更何况手动用户，因此需要谨慎使用公开的机器人，如何活用机器人跟私人狙击手们互相斗法，也是能不能获利的一大关键。
