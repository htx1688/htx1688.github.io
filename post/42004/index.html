X2Y2 关于「自定义版税 」 的推文

8 月 26 日，NFT 交易平台 X2Y2 发布推文宣布推出「自定义版税」功能，这项功能将允许买家自主选择他们希望为 NFT 项目做出贡献而支付的版税（100%、50% 或 0% 的对应比例），当选择支付 50% 和 0% 时会将部分版税返还给买家（这些版税原来是支付给项目方或创作者的）。

消息一出便激起了激烈的讨论：有人批评并反对 X2Y2 和 sudoswap 取消版税的行为，认为这是在侵犯创作者和艺术家著作权和版权，同时会严重影响 NFT 项目方资金流，后续发展可能无以为继；支持者则认为版税「抢走」了本应属于 NFT 交易者的利润，是附加的、不必要的支出，同时版税不应该支付给那些「soft rug」的项目方，「不做事」就不应持续坐享版税红利。

一时之间，原本共创双赢的「创作者经济」变为你死我活的零和博弈。细细回溯，你会发现 NFT 交易市场早已由蓝海淘金变为了红海博杀。

本文将从版税支付的差异、NFT 交易市场的变化、事件的原因、正反方观点、后续的解决方案等角度详解这次事件。

现实中和区块链中的版税分别是如何支付的？

从现实世界走进区块链的版权

根据维基百科中的定义，版税又称版权使用费，是知识产权的原创人或版权持有人对其他使用其知识产权的人所收取的金钱利益。使用拥有版权、特许权、著作权、商标、专利等知识产权的人士便可能需要付版税。一般而言，为了能够取得复制或演出作品的权利，即会支付给作品发明者或创作者款项作为版税。

佳士得拍卖行对于转售权有明确规定

在我国现行的《著作权法》中和国外的艺术家转售权/追续权的相关规定中，都有关于转售后为原作者/艺术家提供版税分享收益的条例。例如，佳士得拍卖行关于转售权的规定中明确表示了卖方及其代理 (拍卖行) 须共同及各自承担版税，佳士得会向买家收取一笔相当于转售版税的款项，并交予相关收款机构。

可见，对于艺术作品转售，支付版税一直以来都是应有之义。但是现实中的传统艺术品由于存在作品真伪难辨、各国相关法律不同、溯源难度大、信息透明度低等特点，落实版税的收取一直都是存在一定的困难。

当艺术走入区块链，加密艺术以 NFT 的形式呈现时，便很好地解决了这些问题：

区块链的不可篡改性和链上透明性，保证了 NFT 承载的加密艺术的资产确权、可溯源、可辨真伪。

「Code is Law」，智能合约以更加简单便捷的方式保证了版税能被分配到原创者/艺术家/项目方。

这是对传统艺术领域的历史颠覆。

NFT 交易市场版税的变化

交易平台作为第三方负责转移版税给创作者/项目方

一直以来，在主流 NFT 交易平台上买卖一个 NFT 通常还有有两笔「额外」费用，即交易手续费和版税，它们都由卖家支出。交易手续费由 NFT 交易平台收取，版税由 NFT 项目方/创作者收取。

本小节以一个 NFT 的交易过程为例，讲述本次「取消版税」事件主要涉及的 OpenSea、sudoswap、X2Y2 这三个 NFT 交易平台上的交易过程。

（为便于计算，假设这个 NFT 的售价为 100E，版税为 5%，三个 NFT 交易平台的交易手续费分别为 2.5%、0.5%、0.5%）

OpenSea（交易手续费 2.5%，版税由 NFT 项目方/创作者设置）

1. 卖家挂单售价为 100E，买家支付 100E

2. OpenSea 交易手续费收取 100*2.5%=2.5E，NFT 项目方收取版税 100*5%=5E

3. 卖家最终收到 92.5E，买家最终支付 100E

sudoswap（交易手续费 0.5%，绕过了 NFT 项目方/创作者收取版税的设置）

1. 卖家挂单售价为 100E，买家支付 100E

2. sudoswap 交易手续费收取 100*0.5%=0.5E，无版税

3. 卖家最终收到 99.5E，买家最终支付 100E

X2Y2（交易手续费 0.5%，版税由 NFT 项目方/创作者设置改为了由买家决定是否支付）

1. 卖家挂单售价为 100E，买家支付 100E

2. X2Y2 交易手续费收取 100*0.5%=0.5E，假设买家选择了不支付版税，则收到版税返还 100*5%=5E

3. 卖家最终收到 94.5E，买家最终支付 95E

sudoswap 和 X2Y2 先后取消版税，深层次的原因是什么？

sudoswap：身有 OTC 市场基因，尽可能减小滑点是 sudoAMM 机制顺利运行的必要条件

此前在《NFT 的 AMM 让 sudoswap 做出来了》一文中详细介绍过 sudoswap 的前 OTC 前身和现在的 sudoAMM 机制。

sudoswap 最开始是由 0xmons 于 2021 年 4 月发布的场外交易平台（OTC），它支持 ERC20、ERC721、ERC1155 等代币任意组合交易，用户可以自定义订单到期时间和指定任意或特定买家进行交易。除此以外，通过 sudoswap 交易是完全不收取交易手续费和版税的。

作为 NFT 场外交易市场它是小众的，大部分交易者是通过场外协商价格后，为了避免买卖双方有一方反悔确保交易是安全的，因而选择了 sudoswap。在正式推出 sudoAMM 之前，它的交易量和日活人数相比于 OpenSea 等交易平台几乎可以忽略不计。

sudoAMM 的出现改变了 NFT 市场的局面。sudoAMM 是一种类似于 uniswap v3 的集中流动性 AMM，流动性提供者可以自由选择一个价格范围来提供流动性，提高了资本效率。同时采用了联合曲线（线性和指数）来促进 NFT 的交易，自动调整买卖双方的报价提高了交易者和流动性提供者的灵活性。

sudoAMM 让 NFT 交易有着更高撮合效率和更好的价格发现，而实现低滑点交易却是以极低交易手续费和牺牲版税来实现的。

sudoswap 通过减小滑点来提高交易撮合效率

以上图为例，假设一个 NFT 系列的公允价值在 100E，买卖双方可忍受的滑点浮动都为 1%，版税为 5%。此时买方想以 100E 左右的价格购买，而卖方也希望能在售出后收到 100E 的全款。因此，买方会将 Offer 设置在 99E~101E，而卖方的售价则设置在 104E~106E，撮合效率很低。

当取消版税后，减小了这个滑点，买卖双方的流动性更容易在各自的上下阈值范围内达成匹配，提高了撮合效率。

总结：

sudoswap 前身是场外交易市场，后来成为 NFT 交易平台，因为它的市场体量相比于整个 NFT 交易平台是很小的，因而其取消版税的问题一直以来都没有受到重视。现在作为新晋 NFT 交易平台潜在有力竞争者，这个问题不得不搬上台面。

而对于它的 sudoAMM 的交易机制，为了最大发挥其撮合交易的效率，需要将滑点（由交易费和版税组成）尽可能达到最低，交易费已经很低了（只有 0.5%），那么把版税占比压缩或取消是这个系统能顺利运行的必要条件。

X2Y2：前有 NFT 聚合器顺水推舟，现在优势丧失殆尽无奈走上旁门左道

Genie、Gem 等 NFT 交易聚合器是作为 NFT 流动性的解决方案出现的，它们可以聚合多个 NFT 交易市场的数据，致力于解决平台之间挂单数据不互通的问题，这优化了用户的交易体验。

在这之前品牌效应是一直是 NFT 流动性的最佳解决方案，即 OpenSea 作为交易量最大的 NFT 交易平台，先发优势十分明显，即使它的手续费最高，但仍是用户交易 NFT 时的第一选择。LooksRare 和 X2Y2 等平台即使采用代币补贴和低交易手续费的方式来激励用户交易，「吸血鬼攻击」收效甚微。

目前 X2Y2 的交易用户来源（图源：nansen）

NFT 交易聚合器这个变量的出现，让 X2Y2「借来了东风」，聚合器抹平了各个交易平台的挂单的信息差，X2Y2 凭借低交易手续费获得了一定的市场份额。根据 nansen 数据显示，目前 X2Y2 的大部分用户交易来自于 NFT 聚合器 Gem，由此可见二者之间的紧密联系。

后来，OpenSea 直接收购了 Gem，最大的 NFT 交易平台和最大的 NFT 聚合器结成攻守同盟，把马太效应贯彻到了极致。但收购后并未做出其他战略调整，Gem 的运营保持着相对独立，OpenSea 并没有因为竞争关系而「加害」X2Y2 等 NFT 交易平台，各自之间保持着「和平发展」的路线。

sudoswap 交易量持续上涨

然而在 7 月，sudoswap 带着独创的 sudoAMM 横空出世，主打低交易手续费（0.5%，与 X2Y2 相同）+零版税，还有独特的针对 NFT 的自动做市商系统。

在不到两个月的时间里，表现不可谓不惊艳：据 Dune 数据显示，sudoswap 总交易额已超过了 2,700 万美元，日活用户一度超过了 1,900 人，累积交易了超过 11 万个 NFT。

Gem 上的低价挂单大部分来自于 sudoswap

然而 sudoswap 扩张的脚步并没有停下，据相关报道，8 月 26 日凌晨已经有人发现了 sudoswap 已集成至 NFT 聚合器平台 Gem。

虽然这次功能的集成 Gem 已经提前「打过预防针」，然而现实呈现出的挂单图表远比想象中的冲击更大：许多 NFT 集合的地板价全都在 sudoswap 上，Azuki、企鹅等蓝筹项目也不例外。

地板价被「攻陷」，sudoswap 的用户和交易量的增长速度还没有看到减慢的迹象；Element 等新的 NFT 交易所正在招兵买马，零交易手续费，同时集成交易平台和聚合器。它们后续都会随着时间推移和项目发展进一步扩大知名度，X2Y2 前期靠着长期的代币激励和低价激励积累的优势瞬间荡然无存，如果不做出调整或有更好的解决方案出现，原有市场份额被蚕食鲸吞应该只是时间问题。

根据 Dune 数据显示，sudoswap 的市场交易额在快速赶超 X2Y2

悬崖已现，在这种市场大环境下，X2Y2 在 8 月 26 日当天中午宣布推出了上文所说的「自定义版税」的功能。

总结：

X2Y2 曾借着 NFT 聚合器发展的东风，凭借着低交易手续费和良好的交互体验拿到了一定的市场份额。而 sudoswap 这个不按常理出牌的「新变量」打破了原有「旧格局」，它的迅猛发展直接威胁到了 X2Y2 的生存。为了能在这场红海搏杀中生存，X2Y2 选择打出这张「侵犯创作者权利」的牌。

「双赢共创」变「零和博弈」，创作者经济输了吗？

在 sudoswap、X2Y2 等 NFT 交易平台宣布取消版税的机制后，似乎存在着一个「NFT 不可能三角」：NFT 投机者的利润、NFT 交易平台盈利、创作者/项目方持续激励，三者只能同时满足其中二者。

支持「取消版税」的观点

「soft rug」一词经常被用来形容那些发售后放弃建设项目和社区的 NFT 项目方，「不做事」仍能坐享版税红利引起了社区的不满，社区与项目方互不信任，站在了对立面，支持「取消版税」的声音很多是出自这里。

然而，现在时值 NFT 市场的熊市，整个宏观的大环境也很不乐观，「地板价下跌」、「不拉盘」也被视为了「不做事」的表现。同时很多项目在致力于开发较长时间周期的项目，加密世界的节奏快，一旦项目的进度慢于持有者的预期，都会被定义为「不做事」。持有者盈利时，对版税更宽容，视版税为成本的一部分；持有者亏损时，趋利避害的心理会想把损失最小化，这时版税则同化为损失的一部分。

Yuga Labs 创始人 Wiley Aronow 曾在 Full Send 播客节目中直言：「我们从不看地板价，我不在乎，我唯一关心的是钱包持有人的数量，这也是为什么每个创始人只拥有一只无聊猿的原因」。

悲观地讲，NFT 市场里的一些玩家的主要目标并不是来支持它们喜欢的艺术作品/创作者/Web3 愿景，这些交易者希望在 NFT 的红利大潮里冲浪投机，谋一份生计来改善生活，在潮水退去时带着钱袋离开。他们想把每一笔交易的利润最大化/亏损最小化，这无可厚非，地板价对于投资者来说也是很关键的，而对于还在持续建设的项目，请多给他们一些时间。如果支持「取消版税」，选择对于所有的 NFT 项目都是「一刀切」，造成资金断流后续开发无以为继，那么双赢共创的创作者经济终将毁于一旦。

反对「取消版税」的观点

多位艺术家反对 「取消版税」

「取消版税」侵犯了创作者和艺术家的合法权利，支付版税一直以来都是应有之义。已有多位艺术家为此发声，加密 KOL Pranksy 针对 X2Y2 官宣「自定义版税」的推文直言：「你丝毫没有考虑到那些你应该感激的、赖以生存的创作者和项目。」

以加密艺术为基础搭建的 NFT 世界，创作者才是源头活水。而 NFT 交易平台为了在红海搏杀中生存下来，选择了侵犯创作者版权这一条路，「端起碗吃饭，放下筷子骂娘」，枉顾「创作者经济」大势，长远来看将会打击艺术家创作和项目运营的积极性，社区与创作者双赢共创的努力可能因此毁于一旦。

而「NFT 不可能三角」实际是可解的，只是加密世界熊市、前路的未知、煽动「零和博弈」的拦路人为它加了一些难度系数。

针对「取消版税」危机，对于创作者/NFT 项目方有什么解决方案？

以太坊网络上的 NFT 的版税支付不是在其配套的智能合约执行的，而是在交易市场层面执行的，即调用 NFT 交易平台的接口，由交易平台负责转发从买卖双方征收的版税，再分发给创作者/项目方，这也是为什么更改版税可以被 NFT 交易平台「一言堂」的原因。

针对「取消版税」危机，有这样一些方案：

1. 创作者/项目方可以禁止自己的 NFT 系列在那些零版税的交易平台上架

建立可交易 NFT 市场白名单

Azuki 的官方团队开发人员 cygaar 发布推文表示：可以更改 NFT 智能合约，建立可交易的 NFT 市场白名单，或者说，直接禁止其 NFT 集合在那些支持零版税的 NFT 交易市场上挂单交易。

2. 推出自己的 NFT 交易市场

  Truth Labs 将推出属于自己的 NFT 交易市场

goblintown.wtf 开发公司 Truth Labs 的联合创始人 Alexander Taub 在其社交平台宣布将于几周内推出 NFT 交易市场。其生态项目 goblintown.wtf、The 187、SecretSocietyJHB、grumpls、burgers 等将上线该市场进行交易。该市场所有 NFT 系列的版税定为 5%，不收取平台费用。

3. 采用类似于 EIP-2981 的能内置支付创作者版税的 NFT 智能合约标准

EIP-2981：NFT 版税标准

EIP-2981 是一个 NFT 版税智能合约标准，它由 Zach Burks, James Morgan, Blaine Malone, James Seibel 于 2020 年 9 月联合提出。EIP-2981 允许资产合约（例如支持 ERC-721 和 ERC-1155 接口的 NFT）每次出售或转售 NFT 时向 NFT 创建者或权利持有人发送需支付的特定比例的版税金额。

EIP-2981 实现了标准化的版税信息检索，它的特点可以用「全局」、「跨市场」、「链上版税」、「向后兼容性」来概括。EIP-2981 实现了无论该 NFT 在哪个市场出售或转售都可以实现准确的版税支付。但它也存在一些问题：比如目前只有部分主流的 NFT 交易市场（如 LooksRare、Coinbase NFT）支持这个合约标准，OpenSea、X2Y2 等都暂时不支持它。

4. 功能和权益制裁

  Coniun 宣布对「取消版税」反击  

8 月 27 日中午，Coniun 发布推文宣布了它对于那些支持零版税的 NFT 交易市场的制裁：删除 Coniun 平台内与零版税有关的交易平台的链接，取消在 X2Y2 上的项目发售，撤销 justaped.in 为 X2Y2 提供的服务，未支付版税购买的 NFT PASS 的平台功能将被封禁，除非补交版权费。对于取消版税的 NFT 交易平台，Coniun 用实际行动吹响了反击的号角。

结语

「取消版税」的闹剧是熊市大环境下 NFT 交易平台红海搏杀的意外结果，前有 sudoswap 这个始作俑者，当低价和补贴无法成为护城河，面临着生存危机时，X2Y2 选择祭献作为源头活水的创作者，借「惩治 soft rug」变革之名，行伤害 NFT 生态之实，其中的逻辑无法自洽。

X2Y2 可以美名为「居安思危」，但我相信相悖于 Web3 精神、不尊重创作者和艺术家、满眼都是利益的不义之师终将失败。

而在「创作者经济」的历史洪流中，有人加入内卷，为「劣币驱逐良币」火上浇油，有人则选择默默耕耘把蛋糕做大，关注长期效用和品牌的未来。这是投机客和理想家的共识硬分叉，Web3 乌托邦道阻且长。
