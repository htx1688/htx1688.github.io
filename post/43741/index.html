徽章

链上徽章是最常见的类别，即通过完成链上或链下任务而获得的代币或 NFT。 以下初创公司归为此类：

Sismo 是一个模块化的证明协议，具有去中心化、隐私性和易用性等特点。它以不可转移代币或灵魂绑定代币的形式发布证明徽章，该协议在 Polygon 上运行。

POAP 是一种将记忆保存为数字记录的协议，根据 ERC-721 标准在区块链上铸造为 NFT 代币。

Orange Protocol 是一种声誉和信任铸造协议，它聚合数据和 Web3 声誉模型，通过可验证凭证和 NFT 的形式生成系统性的声誉证明。

Noox 是一个铸造平台，可以将链上成就铸造为灵魂绑定 NFT。

链上证明

相对于链上徽章，链上证明通常需要第三方实体来验证信息的真实性。所有人可以通过 KYC 详细信息证明链上身份的真实性，例如经过认证的软件工程师。认证领域有很多值得关注的地方，每个初创公司尝试从不同的角度来解决这个问题：

Galxe 是一个 Web3 凭证数据网络，它使品牌和开发人员能够参与社区，并在 Web3 中构建强大的产品。

BrightID 是一个去中心化的开源社交身份网络，它允许人们向应用程序证明他们个人账户的真实性。

Gateway 是一个 Web3 凭证网络，可为用户的工作和能力创建证明，并构建 Web3 链上简历。

Zorro 是 Web3 公民身份系统，它可以帮助用户进行身份认证，从而获得 DAO 投票权和其他好处。

Verite 是利用身份证明与加密金融体验联系起来的数据模型来构建的协议和开源软件。

Krebit 是一个声誉评分平台，提供完全去中心化、社区验证的链上凭证。

个人简介

个人简介类别的最佳定义是一个汇总了有关你地址的所有信息，并将其显示在一个令人愉悦的用户界面的产品。此界面可能会根据配置文件在上下文中的用途而有所不同。配置文件还可以充当聚合器，并创建将所有内容集中在一个地方的元程序。Gitcoin Passport 就是一个例子。

Gitcoin Paspport 是一种使用可验证的凭证来确认来自互联网的身份认证，而不存储个人身份信息的数字身份证。

Phi 能够可视化链上身份，并基于通用 web3 设施如 ENS 域名或钱包活动来构建元宇宙数字身份系统。

社交

社交可能是最近几个月最受欢迎的类别之一，它是基于个人地址的社交媒体网络或平台。创作者的去平台化促使许多人重新考虑 Web2 社交网络。虽然目前仍然是一个小类别，但是值得期待的类别之一。

Lens Protocol 是为创作者制作的可组合的去中心化社交图谱。

Farcaster 是一个开放的去中心化社交网络，支持多种客户端。

Orbis 是一组社交数据模型，它与去中心化基础设施相结合，具有先进的 SDK 和灵活的模块，可提供完全可组合的社交体验。

Disco 是一个档案系统和数据背包，将个人资料显示在社交媒体帐户和 Web3 凭据当中。

单点登录

取代互联网登录流程如电子邮箱登录或密码注册等可能是加密行业在改变互联网基本结构方面做出的最大贡献之一。这些类别的初创公司有助于改善单点登录身份验证及其周围的基础设施的用户体验。

Spruce 是一个去中心化身份的工具包，由 DIDkit、Rabase、Keylink 和 Credible 组成，支持签名、共享和验证可信信息。

ENS 是一个基于以太坊区块链的分布式、开放和可扩展的域名系统。

PolygonID 是具有可编程隐私的区块链原生身份系统，提供链上验证和无许可证明。

链上评估和追踪

链上评估和追踪是一个在技术上较难执行的类别，但也拓展了应用程序类型的设计空间。链上评估和追踪是对大型数据集合进行连续计算，为用户评估有关其钱包的评分或属性。与静态的徽章和成就相比，此类别是动态的，并且具有更多的应用场景。链上追踪可以了解你的用户来自哪里，以及他们对你的价值。

ARCx 是基于声誉的 DeFi 去中心化信贷市场，通过使用 DeFi 信用评分，用户可以在 ETH 抵押品上借入高达 100% 的 LTV 比率。

Degen Score 是一种评分协议，根据用户的链上历史记录对其进行评分。

Spectral 是一种信用评分协议，使用链上交易数据来计算你的信用得分。

任务和用户获取

除了女巫问题，任务平台是一种加密原生用户能够获取开放渠道的机会的应用。根据用户的链上声誉如徽章、凭证或分数等某些方面获取精准用户。在这个领域已经有早期成功的迹象，但存在的最大挑战是获取精准用户后，留住他们。流动性挖矿和空投与这一类别非常相关，因为尽管存在差异，但它们的目标是一致的。

Layer3 是一种以赏金形式托管任务的协议，这些赏金是 Web3 任务，例如在 polygon 上发送代币，甚至借贷。

Rabbit Hole 是一个完成 Web3 任务获得奖励的链上成就系统，项目会在平台上发布任务，然而与 RabbitHole 合作将其代币分发给已证明价值的用户。

贡献者

尽管在不同的背景下，DAO 贡献者面临一个与传统组织相同的问题：我如何找到可以雇用的人，以及如何确保他们是合适的。在这个领域专注于身份的初创公司尝试从根本上解决问题。

Coordinape 是一个为 DAO 构建的可扩展且无需许可平台，用于去中心化奖励分发。

Source Cred 是一个 DAO 或开源社区治理的工具，用于衡量和奖励价值创造。

储存

由于可用信息的多样性，围绕数据主权和去中心化，存储是一个关键组成部分。因为它将共识问题与更大的数据集结合在一起，在技术上也非常具有挑战性。目前有一些团队正在积极研究一些解决办法。

Cyber Connect 是一种去中心化社交图谱协议，可将数据所有权还给用户，并帮助开发人员构建 web3 社交应用程序。

Spring Labs 借助其获得专利的、安全的加密和代币化解决方案来存储和共享敏感数据，确保不会泄露任何个人身份信息。

Verida 是一种建立在去中心化身份的基础上，用于可互交的数据库存储和消息传递的多链协议，其中个人数据由用户拥有和控制。

Ceramic 是一个去中心化的数据网络，可为 Web3 应用程序带来多样的数据可组合性。

小结

本文概括了链上身份和声誉的每一个类别以及代表项目，这个领域充满了活力，有许多团队和协议正在解决许多不同的问题。
