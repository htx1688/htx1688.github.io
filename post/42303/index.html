加密货币和 NFT 在现实世界中真的有用吗？现实世界的适用性对于加密技术的真正成功至关重要。如今，有人对加密货币感到困惑或厌恶。这对新的消费品来说可不是什么好事。加密货币正处在一个十字路口。该行业展现出真正的效用是至关重要的。

已经有很多消费者将这些产品付诸实际使用的例子。在本文中，我将深入探讨加密货币提供实际价值的领域，而不仅仅是投机和 Flip。

哪些 Crypto 产品是当今最可行的实例呢？

成功的消费产品往往能够满足许多核心用户需求，而不仅仅是投机性的好处。

其中一个原因是区块链应用对主流用户的吸引力还在早期。在加密货币中，用户的动机可能主要是金钱激励和所有权。

另一方面，成功的消费类产品利用了人类行为的大量核心驱动力：对意义的需求、对自我表达和创造力的需求、对联系和影响的需求、对拥有的需求、对归属的需求、对安全和保障的需求、对社会影响和与他人联系的需求、对刺激和多巴胺的需求，对完成和构建的需求。

如今，加密货币最现实的用途仍然是替代法币，尤其是用于国际支付时。像 Pancake Swap 这样的 DeFi 应用在多个国家拥有超过 200 万的月活跃用户。在阿根廷、南非和泰国等地理分散的国家，加密货币的使用率接近 20%。（全球平均为 10%；美国则略低于 13%。）

第二个最可行的类别是关于企业粘性：品牌提供访问/忠诚度奖励、实体化 NFT 和广告。

以下是现实世界中最具前景的 9 个加密实用领域。

1. 货币替代品和国际支付系统

在世界上许多国家，政府政策限制其公民获取和使用其资金。例如，在阿根廷，人们每月只能将 200 美元的比索兑换成美元。

如果你想在国外上大学，买房子，或者移民到另一个国家，这是一个巨大的挑战。另外，如果你在另一个国家给企业或个人支付过钱，你就会知道国际支付系统非常糟糕。将资金汇往国外通常会有过高的手续费。

其中一些国际支付系统参与者包括：

  • LemonCash：LemonCash 在阿根廷已经有超过 100 万用户，为其客户提供借记卡、经纪和现金返还服务，这些服务都支持加密货币。

  • Pintu：这家印尼加密交易所允许人们买卖加密货币，拥有超过 400 万注册用户。

  • Chipper Cash：这家虚拟银行在尼日利亚吸引了 500 多万注册用户，现在用户可以在非洲其他国家买卖比特币。

  • Tribal Credit：Tribal 为拉丁美洲的小企业提供公司信用卡、支持加密货币贷款和支付机制。

今年早些时候，Ukraine DAO 通过鼓励世界各地的支持者使用加密货币直接向乌克兰军事行动捐款，从而筹集了数千万美元。这使用户能够避免国际电汇费用，和政府封锁，并绕过国际银行支付系统 SWIFT。

虽然时间会告诉我们政府最终将如何监管加密货币，但与此同时，人们也在利用它获得新的机会。新兴市场的创始人有机会跨越金融技术，带领他们的用户进入新世纪。

2. 社区和品牌粘性工具

在最基本的层面上，NFT 和代币只是使用区块链验证所有权的一种方式。品牌已经开始将其作为忠诚计划的一部分，为其社区的认证成员提供独特的特权。

例子：

  • Percs：这家初创公司鼓励品牌为忠诚的客户打造 NFT，并通过独家体验来激活新的受众。用户将他们的钱包链接起来，就可以找到他们有资格获得的奖励。试用 Percs 的主要品牌包括百威、花花公子、Shopify 和 DJ Steve Aoki。

  • Cal：一个日历应用程序，只允许与你拥有相同代币的人（通过区块链验证）与你预约会议。

  • Fan Controlled Football（由球迷控制的另类职业体育联盟）：FCF 使用代币和 NFT 来为其粉丝驱动的体育联盟中的球队所有权设置访问权限。

  • POAP：出席证明协议（POAP）是 Web3 最顶级的协议之一，它可以颁发认证或凭证来验证用户是否参加了特定的活动。

此外，许多顶级品牌都将 NFT 作为提高参与度的一种方式，包括 NFL（超级碗门票作为收藏代币）、Gucci（梦幻时尚服装的数字系列）和耐克（名为 CryptoKicks 的虚拟 NFT 运动鞋）。专门从事 NFT 的机构（如 Recur、Unblocked 和 ThirdWeb）正在通过授权协议、品牌展示、忠诚计划等方式帮助其他大品牌开发他们的 NFT 和加密战略并变现。

3. NFT 实体化或现有商品的代币化

有几家公司相信，未来每个实体都会有一个 NFT 孪生品。数字标签将分享产品信息，跟踪供应链，使原品牌回收一定比例的二级销售，并允许所有者证明他们确实拥有该产品。

在这个领域中构建的参与者包括：

  • Courtyard：这家数字孪生品公司正在与 Brinks 安全公司和其他公司合作，提供“物理对象标记化”作为服务平台。

  • 元宇宙相关技术孵化器 sLABS：NFT 驱动的商品，与碧昂斯和 JayZ 等主要艺术家合作。

  • 视频库平台 Infinite Objects：通过将出售的 NFT 视频打印，将其像相框一样展示在你的家中，让虚拟变得更真实。

对于品牌和 IP 所有者来说，这似乎是最大的机会，因为他们以前只能从实体物品的主要销售中赚取版税。

如果你把冰雪奇缘里的艾莎城堡玩具转卖给另一个买家，迪士尼不会得到第二笔提成。如果它附带了一个 NFT，提供主题公园的折扣，新买家将希望在出售中附带该 NFT。但这一概念在被广泛接受之前需要大量的消费者教育。

4. 广告和联盟营销

如今，营销人员很难接触到区块链用户，因为他们无法向钱包发送消息，甚至无法知道钱包持有者的身份。

考虑到 Web3 关于数据主权和数据互操作性的理念，一些区块链应用已经选择有选择地为品牌创造广告和营销机会，向选择共享数据的用户提供代币。这明显优于 Web2，在 Web2 中，集中式实体围绕以非透明的方式监控和销售用户数据构建了免费的商业模型。

广告模式将进入加密领域，这需要一个契机。它有可能给用户带来巨大的利益。以下是一些早期的例子：

  • Brave：这个以隐私为中心的浏览器可以阻止广告商和网站跟踪你。它提供了一种名为 Brave Rewards 的订阅模式，用户可以通过一个名为 BAT（注意力币）的 token 匿名查看广告。BAT 拥有 5500 万月活跃用户，1600 万日活用户和 150 万经过认证的开发者，并声称拥有数百万用户的钱包。

  • Layer3：Layer3 用户通过参与由 Layer3 主持的 Crypto 学习教程来收集奖励。这本质上是一种联属营销，Layer3 将其企业客户提供的营销资金的一部分用于获取用户，同时向这些用户告知他们的产品。Layer3 是建立一个带有链上行动的广告网络。

  • Coinbase Earn：关于不同加密产品的教程，允许观看者在观看时赚取代币，类似于 Layer3。

  • Holder：Holder 的客户关系管理（CRM）和消息层帮助品牌开展营销活动。

另一组支持广告的参与者是加密原生消息产品，这是因为许多参与者计划向未经验证的发送者收取发送信息到钱包的费用。你可以想象一个全新的广告世界正在被创造出来，在那里营销人员决定他们愿意花多少钱来联系一个有 10 个无聊猿的钱包。

在我写这篇文章的时候，我们正在开发新的广告指标、方法和渠道。我相信在几年内我们会有一个全新的数字营销类别。

5. 个人职业融资的另一种选择

如果你是一个有抱负的音乐家或运动员，希望成为职业运动员，或者只是一个自由职业者，希望在各种演出之间保持稳定的收入，出售代币来资助你的职业生涯可能是一个可行的替代方案。代币持有者将分享你未来收入的一小部分。

例子：

  • Audius ：这种基于区块链的流媒体服务可以让音乐人绕过唱片公司的把关人，让他们对自己的工作有更多的控制权。治理代币 AUDIO 目前价值 3.98 亿美元，完全稀释。可以用来奖励艺术家和听众，以及网络节点运营商。如今，直接面向粉丝的音乐市场规模已经超过了传统音乐产业，这在一定程度上要归功于 Audius 的突破性成功（月活用户超过 5 亿）。

  • Royal：通过 Royal，粉丝们以 NFT 的形式为每首歌曲购买歌曲版权，并获得一定比例的版税。这有助于通过激励购买经过认证的歌曲来打击盗版音乐。一些 NFT 还包括一些额外的福利，比如与艺术家和数字艺术的独家接触。与 Audius 一样，Royal 也为传统唱片公司提供了另一种选择。

  • Braintrust：自由职业者集体拥有的去中心化 Boss 直聘。会员根据他们对网络的贡献获得 Braintrust 代币 BTRST，价值 5.58 亿美元 FDV（完全稀释估值），比如介绍客户或审查新会员。这些代币可以用于治理，作为质押品来防范非付费客户，并兑换专属社区福利。

  • Agrotoken：Agrotoken 使阿根廷的农民能够利用加密稳定币，将谷物转化为资产，并使用它支付其他金融服务。这消除了投资新收成和收到付款之间的长期滞后，赋予农民更好的定价权。

应用这一模式的其他机会包括具有垄断性把关人的行业。资本支出成本高（如电信和物流），为消费者带来即时价值的行业。具有较高的利润，但早期从业者的报酬很低，以及成功之路不明确但数百万人向往的工作（如写作、电影制作和其他艺术）。

6. 利用去中心化的能源、内存、存储和其他电力网络

如果你的业务需要大量的资金支出来建立网络，并且密集的覆盖对业务有利，那么利用由代币/加密货币激励的去中心化的基于区块链的网络可能是一个很好的选择。

  • Helium：一种点对点无线网络，基于个人拥有的热点，为附近的物联网设备（如滑板车或数字冰箱）供电。用户赚取 HNT 代币来支持生态系统，例如允许单个热点在网络上传输数据。Helium 对电信公司的作用就像 Airbnb 对酒店的作用一样，为一家公司在全球建立无线网络所需的昂贵资金创造了一个强有力的替代方案。

  • Flux：与 Helium 类似，Flux 是一个基于云端的网络，用于提供基础设施服务——本质上是一个去中心化的亚马逊AWS云服务。其 FLUX 代币的 FDV（完全稀释估值）市值为 4.38 亿美元。

  • Livepeer：该服务补偿用户使用闲置 GPU 对视频进行转码，提供更高质量的流媒体体验。与此同时，用户可以通过挖 LPT 代币（2.72 亿美元 FDV）来产生被动收入。截至 2022 年 6 月，去中心化视频流媒体网络已经处理了超过 9000 万分钟的视频。

  • Theta：另一个视频流媒体网络，其代币 THETA 价值 12 亿 FDV。当用户观看视频时，他们的一部分计算能力被用来将这些视频转发给其他用户。主要用户赚取代币作为奖励。

  • IFPS：这种替代 HTTP 的方法通过跨 P2P 网络分发站点数据，通过 URI（通用资源指示器）而不是 URL 访问内容。个人设备充当存储网站数据的节点。打开存储在其他网络节点上的 IFPS 站点会增加加载时间和流速度，同时也会降低网络中断的可能性。

对于任何需要对分布式硬件进行密集资本融资的行业或产品来说，这是一个机会。在这种情况下，满足客户需求的硬件是一个优势，或者创建资源密集型的产品。这为消费者创造了更低的成本。

7. 活动访问和 NFT 票务

NFT 还可以传达访问独家活动或空间的权利。你的加密钱包决定你是否可以进入，而不是身份证或钥匙卡。NFT 也是一种更好的门票形式，因为它们能让艺术家从持续的销售中获得佣金。

  • Afterparty：世界上第一个针对艺术家的售票平台在洛杉矶和拉斯维加斯举办了以代币为参与门票的活动，允许艺术家从 NFT 中获得一些二级转售价值。因为 Afterparty 直接从参与者那里收集数据，它也可以为艺术家创造更多机会，直接与他们的粉丝群体互动。

  • Tokenproof：最初推出 Tokenproof 是为了让用户在不暴露钱包的情况下证明自己拥有 NFT，现在 Tokenproof 使用该功能，支持以 NFT 为门票的活动，包括今年的 ApeFest 和 NFT NYC 活动。

  • MoviePass：这个热门品牌以 NFT 门票角度重新定义电影票房。

这里的机会比较有限。由于 LiveNation/TicketMaster 票务服务垄断了大多数赛事，新进入者需要在独立场馆（Livehouse）和独立音乐节上建立足够强大的追随者，或直接说服艺术家推动这一活动。

8. 减少碳足迹

区块链因为能源密集型而吸引了很多负面关注，但它们也带来了改善环境的解决方案。

这里有 3 个例子可以说明这条产业链是如何做到绿色环保：

  • KlimaDAO：这个去中心化的自治组织旨在通过激励碳信用来减少温室气体。它鼓励人们以折扣价用代币化的碳信用额度换取 KLIMA 代币（600 万美元的 FDV）。只有当用户证明代币化碳信用被锁在其资金库中时，代币才能被铸造。它本质上是一个由碳信用支持的代币。

  • Veritree：该项目的目标是重新造林，使用区块链来验证何时种植了新树。Cardano 是这项努力的幕后推手，今年早些时候宣布已经种植了 100 万棵树。

  • Terra0.org：当我第一次读到这个老项目时，它就吸引了我。Terra 正在建设一个自我管理的森林，使用智能合约来砍伐和出售树木。他们使用无人机来监测部分森林何时长出足够的树木来进行采伐，然后自动执行一项合约，让伐木工进入森林。

这里的机会是无限的。其他由目的驱动的领域可以从气候变化方面所做的工作中获得灵感，以激励更多长期有益于我们人类的创造。

9. 通过 X to Earn 激励健康的活动

前提很简单：付钱让人们养成健康的习惯。补偿人们坚持更健康的习惯是很有意义的。保险公司已经在鼓励健康的行为，这有助于他们在未来避免巨额赔付。为什么不选择通过加密货币达到这一目的呢？

这一类别仍处于相对初级阶段，但这里有一些有希望的候选者：

  • Stepn：用户可以通过步行赚取 GMT（41 亿美元的 FDV）。Stepn 要求人们买能记步的实体鞋。一款鼓励人们每天出门散步的优秀硬件+软件游戏显然取得了进展，在头六个月就获得了 300 万用户。

  • Sweatcoin：同样的想法——鼓励人们摆脱躺平罪恶感，行动起来。在高峰期，Sweatcoin 的日活用户超过 1000 万。该公司尚未真正实施这种加密货币，但计划很快实施。

  • Dreamland：这个新参与者（仍处于发行前模式）专注于 Sleep to Earn。是的，你没看错，目标是最终整合可穿戴技术，追踪睡眠模式，并根据人们的睡眠时间奖励他们的代币。

现实情况是，这个类别还没有找到一种可持续的经济模式。平衡这些生态系统，让用户不会立即将加密货币转换成法币一直是一项挑战。创始人的机会在于找到正确的加密商业模式。我期待看到更多的经济学博士成为这些公司的联合创始人。

总结

目前还有更多的实用程序应该存在。我希望看到一个公链跟踪主要品牌的供应链材料和来源。我还希望看到重要资产和 IP 的权利所有权的可搜索记录——一个任何人都可以上传信息的专利数据库。希望有更多为了满足消费者需求，致力于构建一个基于现实世界的区块链应用程序的创始人涌现。
