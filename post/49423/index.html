原文作者：Babywhale，Foresight News

近期，「比特币 NFT」概念受到了市场关注，并再度引发了比特币究竟应该只保留单纯的价值存储功能，还是应该添加可编程性的讨论。与此前类似的讨论不同的是，创造了比特币 NFT 的开发者认为，NFT 的出现为比特币矿工提供了额外收益，可以在未来区块奖励逐渐减少时仍然保证网络安全性。 然而也有开发者提出，比特币链上可编程性的出现，可能会为比特币带来 MEV，从而大大增加网络的复杂性。

虽然当前大量比特币的开发者和维护人员都遵循着基本的原教旨主义，但未来的发展因为 NFT 的出现再次变得悬而未决起来。

在这一轮热潮中，比特币智能合约平台 Stacks 从快要被遗忘的角落拾了起来，其代币 STX 价格在一个多月的时间内上涨了 4 倍以上。作为 2018 年推出的项目，Stacks 采用了 PoX 的共识机制，将 Stacks 区块链和比特币进行了链接，并使得 Stacks 拥有比特币网络的安全性。

具体而言，Stacks 会在比特币网络上通过拍卖的方式来选定区块的打包者，竞拍者需要通过比特币进行出价，获胜者将可以获得 Stacks 的区块奖励以及手续费奖励等。而赢家支付的比特币费用将成为 STX 质押者的奖励。本质上说，Stacks 是通过某种机制「借用了」比特币安全性（所谓的安全性其实也是区块打包权选举的透明公开，实际没有对获胜者施加过多的限制，所以其本身的安全性仍然取决于收益性价比高于作恶）的区块链网络，不是直接建立在比特币之上的智能合约平台。

但 Stacks 网络发展至今也产生了包括 DeFi 等在内的生态项目，如果未来为比特币添加可编程性被广泛接受，那么 Stacks 的发展空间仍具有不错的想象力。本文依据 Stacks 官网列出的生态项目进行介绍，请各位读者在交互项目时注意风险。

钱包

D’CENT Wallet

D’CENT Wallet是一款硬件钱包，其生物识别钱包、卡以及软件钱包均可以使用单个 App 进行管理。据官网信息显示 D’CENT Wallet 支持的代币超 3000 种。

GoSats

GoSats曾入选 Stacks 加速器，并获得了 Stacks 的投资。但据官网信息显示 GoSats 是专注于印度市场的消费返还比特币卡的公司，没有相关信息显示其和 Stacks 区块链之间的关系。

Hiro

Hiro是一个 Stacks 生态的开发者工具平台，其产品包括了 Stacks 钱包 Hiro Wallet。

Neptune Wallet

Neptune Wallet为 Stacks 开发的钱包，支持发送和接受 STX 代币、质押 STX 代币以及托管 Stacks 生态代币。

Ryder

Ryder是一款社交硬件钱包，用户可以通过钱包进行现实世界的互动。Ryder 计划为 NFT、DAO 等提供更多使用场景。

WISE

WISE是一款开源的 Stacks 生态非托管钱包，支持 Stacks 上资产的发送、接收和托管。同时 WISE 也支持.stx 域名转账。

Xverse

Xverse是一款比特币生态钱包，支持的资产包括比特币、比特币包装资产、比特币 NFT、Stacks 生态资产等，未来将支持比特币闪电网络转账。Xverse 也支持 STX 质押获取比特币奖励。

MultiSafe

MultiSafe是 Stacks 和比特币生态的多签钱包解决方案，类似于 Gnosis Safe。

DeFi

ALEX

ALEX是 Stacks 生态的 DEX，同时也包含 Launchpad 的功能。据 DefiLlama 数据显示，ALEX 上 TVL 约为 2297 万美元。

Arkadiko Protocol

Arkadiko Protocol是一个超额抵押稳定币协议，用户可以通过 STX、xBTC 以及 ALEX 进行超额抵押铸造稳定币 USDA。此外该协议也为 USDA 和其他 Stacks 生态资产的交易对建立了流动性池。据官网数据显示，Arkadiko Protocol 上 TVL 约为 867.3 万美元。

BitFlow

BitFlow是 Stacks 生态的 DEX，支持 xUSD、USDA 等稳定币的交易以及各类比特币封装资产的交易。目前其产品还未开放，用户需申请测试版本使用权限。

LNSwap

LNSwap是一个去中心化交易平台，但并非使用传统地 AMM 模式，而是参考了比特币闪电网络的技术特点推出了 submarine swap，一种可以将比特币、闪电网络上的比特币以及 Stacks 上资产进行原子交换的方案。

Stackswap

Stackswap是 Stacks 上的 DEX，除了支持 Stacks 上资产交易外，还支持比特币、以太坊和 USDC 与 Stacks 上资产的跨链交易。

Zest Protocol

Zest Protocol是一个面向机构投资者的比特币借贷平台，用户可以向该协议中存入比特币获取收益。合格的机构借款人则可以从借款池中借入比特币。

wrapped.com

wrapped.com是一个封装资产平台，支持将比特币封装为 Stacks 上的比特币封装资产 xBTC。

Lydian

Lydian是一个建立在 Stacks 上的去中心化的资产管理协议。

DLC.Link

DLC.Link支持在智能合约中使用原生比特币而无需封装以及跨链。Stacks 上的 DeFi 协议可使用该协议进行借贷或交易。

NFT

Boom

Boom是 Stacks 原生的 NFT 平台，支持 Stacks 生态代币转账，未来将支持 Stacks NFT 交易。

indexer.xyz

indexer.xyz是一个跨链 NFT 索引和 API 基础设施，目前支持 Stacks、NEAR 和 Aptos。

droplinked

droplinked是由 Flatlay 推出的 NFT 推广平台，用户可以通过分享 NFT 系列来获得奖励。droplinked 目前支持 Stacks、NEAR、Polygon、Casper、SKALE。

Gamma.io

Gamma.io是 Stacks 生态和比特币 Ordinals 的 NFT 交易平台。

LaunchMyNFT

LaunchMyNFT是基于网页的生成式 NFT Launchpad，目前支持以太坊、Solana、Aptos、BNB Chain、Avalanche、Fantom、Polygon、Stacks。

Songs To Radio

Songs To Radio是建立在 Stacks 上的音乐 NFT 平台，支持创作者通过平台创建音乐 NFT 以及 NFT 的交易。

Stacks Bridge

Stacks Bridge是一个以太坊和 Stacks 之间的 NFT 跨链桥。

Superfandom

Superfandom允许创作者、有影响力的人和名人通过 NFT 与粉丝进行链接。用户可以通过铸造创作者的灵魂绑定代币来获得独家的体验。

This Is #1 

This Is #1 是建立在 Stacks 上的 NFT 市场，支持创作者在该平台上发行 NFT，但目前不支持二级市场交易。

质押

Fast Pool

Fast Pool是由 Stacks 贡献者 Friedger 开发的 STX 质押池。

InfStones

InfStones是一个 Web3 基础设施平台，产品包括节点管理、质押、API 网关等。其质押产品支持包括 Stacks 在内的超 70 条区块链。

Planbetter

Planbetter是 Stacks 上的非托管质押平台，支持用户质押 STX 以获得比特币奖励。

DAO

Bitfari

Bitfari是建立在 Stacks 上的营销 DAO，用户可以通过安装 Bitfari 应用来投放或接收广告从而获取激励。

StackerDAO

StackerDAO是建立在 Stacks 上的 DAO 管理工具，包括资金管理、多签工具、投票工具等。

Ballot

Ballot是针对 DAO、NFT、DeFi 和 Web 3 项目的去中心化投票应用，投票将根据代币、.BTC 命名空间和 NFT 的持有量进行控制。

数据分析

BTC Leaderboard

BTC Leaderboard是一个分析推特上以.btc 域名为推特名的用户的关注人数的网站。

Stacks Data Center

Stacks Data Center是 Stacks 的链上状态可视化平台，提供的数据包括持币地址、合约、mempool、钱包等。

Stacks on Chain

Stacks on Chain是 Stacks 与比特币 Ordinals 的综合数据平台，包括了 Stacks 基本链上数据、DeFi 以及比特币 Ordinals 数据的信息。

Novum Insights

Novum Insights是专注于 Stacks 上 DeFi 协议的数据分析平台，可查看包括流动性挖矿收益、用户的投资组合等数据。

其他

CityCoins

CityCoins是建立在 Stacks 上的项目，赋予社区改善城市的力量，同时为个人贡献者和市政府等提供加密奖励。CityCoins 借鉴了 Stacks 通过质押 STX 获得比特币奖励的机制，用户可以通过质押 CityCoins 来获得 STX 奖励。目前美国迈阿密已通过 CityCoins 发行了 MiamiCoin（MIA）。

frens

frens是一款 Web3 客户管理工具，支持 Stacks 网络。提供包括 Discord 机器人、社区数据看板、NFT 忠诚度计划工具等。

HireVibes

HireVibes是一个 Web3 招聘平台，提供 STX、比特币等作为推荐奖励。

NoCodeClarity

NoCodeClarity是一个无代码开发平台，支持不精通技术的人建立 Web3 应用，目前支持 Stacks，未来将支持比特币网络和比特币闪电网络。

NoteRiot

NoteRiot是基于 Stacks 的隐私云笔记平台，不使用中心化服务器来存储用户记录的内容。

Tontine Trust

Tontine Trust是一个泛欧养老保险平台，利用 Stacks 区块链实现透明和安全性。

Sustainable Bitcoin Protocol

Sustainable Bitcoin Protocol旨在激励比特币矿工使用清洁能源挖矿，并使用 Stacks 上智能合约对使用清洁能源产出的比特币进行验证。

Sigle

Sigle是基于比特币网络和 Stacks 的 Web3 内容平台，创作者创作的内容将被存储在比特币或 Stacks 上。

BTC.US

BTC.US是 Stacks 上的域名协议，用户可以注册.btc 域名并绑定至 Stacks 地址或比特币闪电网络地址。

DESPREAD STUDIO

DESPREAD STUDIO是建立在 Stacks 上的域名系统，支持.xck.app 和 .mixmi.app 域名。
