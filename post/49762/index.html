蓝筹 NFT 进入调整期，NFT 交易市场却热闹非凡。

NFT 赛道的机会在哪？这应该是近几个月 NFT 玩家苦思冥想的事情。由于 Blur 带来的竞争压力，NFT 市场巨头 OpenSea 的市场份额正在不断的被挤压，X2Y2、LooksRare 等二线 NFT 交易市场更是没有什么机会可言，新上线的 NFT 市场也将面临重重挑战。

事实证明，NFT 市场想要留住用户是很困难的，哪里有奖励，有发币预期，他们就会去哪里。面对竞争如此激烈且白热化的 NFT 赛道，各家也都使出浑身解数，想方设法的创新产品和玩法，以抢占 NFT 市场份额。

Opensea 推出 Opensea Pro，「社区奖励」引发空投猜测

继免除市场费用和版税之后，Opensea 的又一大动作就是将 Gem v2 品牌升级为 OpenSea Pro。Opensea 自收购 Gem 之后，就没有太多有关 Gem 的动态更新，这个曾经风靡一时的 NFT 聚合器 OG 似乎逐渐躺平，淡出了 NFT 市场竞争的行列。不过，此次品牌升级，不仅重新唤醒了 Gem 彼时风光，还让其成为了 Opensea 对抗 Blur 的有力工具。

根据 OpenSea Pro 的官方介绍，此次 Gem 的品牌升级瞄准了成为「最强大的 NFT 市场聚合器」这一目标，汇总了来自 170 多个大大小小的 NFT 市场的列表，包括 Blur、LooksRare、X2Y2、OKX 等平台。此外，其他的一些新的功能包括：

  • 实时发现跨市场数据：支持实时监测功能，可以在几秒钟内更新 NFT 趋势、铸币和收藏页面的所有数据。

  • 即时销售：从整个 NFT 市场汇总 NFT 各种类型的报价，使用户能够以最优惠的价格出售。

  • 最优化的 Gas 费用：在购买 NFT 时，二者的 Gas 费用上，Pro 要明显比 Blur 便宜。

  • 0% 上币费：你可以在促销期间以 0% 的费用在 OpenSea 上列出 NFT。

  • 其他新功能包括监视列表、Live Mints 概览、批量转移等。

此外，在用户激励上，OpenSea Pro 「正在跟踪用户在 Gem 上的活动以计分」，让人与 Blur 的积分激励有所联想，虽然目前尚未公布具体的玩法细则，不过作为一直尚未发币的 NFT 市场来说，这个暗示已经让 NFT 交易家兴奋起来了。目前，OpenSea Pro 和 Gem 的日交易量和活跃的地址数都名列前茅。

Magic Eden 启动钻石奖励计划，并试水比特币市场

Solana 受 FTX 事件影响之后，Magic Eden 市场的成交量也一度下降。但这并没有阻挡 Magic Eden 的不断扩张。在玩法和产品创新上，Magic Eden 算是动作比较多的了。例如在聚焦 Solana 生态外，又将平台扩展至以太坊网络，推出 ETH 聚合器，还集成了 Polygon 网络；此外，引入折扣和奖励系统，用户可在平台上通过活动获得奖励、折扣和其他福利，且用户有机会获得 5% -45% 的交易折扣等。近期，又将目光聚焦在 Web3 游戏上，其风险投资部门 Magic Eden Ventures 已投资 11 家 Web3 游戏工作室。并在以太坊、Solana 和 Polygon 上逐步推出至少 12 款 Web3 游戏。

近日，Magic Eden 又有两项大动作，其一是推出比特币生态 NFT 市场。随着 Ordinals 协议逐渐获得大家的关注，早期上线的比特币 NFT 场外交易量激增，让越来越多的 Builder 对此感到兴奋，纷纷寻求接入比特币 NFT 生态的机会。Magic Eden 目前已集成对比特币钱包 Hiro 和 Xverse 等的支持，并允许交易者上架、购买和出售 Ordinals NFT，还决定暂时在无版税情况下支持比特币生态 NFT。

此外，Magic Eden 还推出了比特币 Launchpad，或受 Magic Eden BTC Launchpad 上线影响，Dune 数据显示 4 月 7 日 Ordinals 铭文 NFT 铸造量出现较大增长达到 36898 枚，较前一日多出超 1 万枚。Magic Eden 还宣布将在四月份上线超过 10 个基于 Ordinals 协议的比特币 NFT 项目。

Magic Eden 的第二项大动作是启动钻石奖励计划。该钻石奖励（Diamond Rewards）计划，允许用户根据不同的任务赚取钻石。根据官网介绍，钻石是一种奖励货币，每月奖励数量都会保持不变。获得钻石的方式有以下三种：在 Magic Eden 上架 NFT 并成功出售；在 Magic Eden 竞价 NFT 并被接受；完善个人资料。此外， 100% 的忠诚度得分会获得 2 倍乘数。

Magic Eden 的钻石奖励计划与 Blur 的积分有异曲同工之处。Magic Eden 也是尚未发币的 NFT 市场之一，近期不连断的新动作也让社区分析 Magic Eden 有较强的发币动机，钻石奖励也可能成为 Magic Eden 以后发币时的空投参考标准之一。

目前，Magic Eden 交易总额已经突破 20 亿美元。

LooksRare 上线 V2 版本，手续费几乎为零

由于奖励机制不断减少，LooksRare 的市场份额也逐渐降低，但其仍然是与 Opensea 竞争的有力对手，并在不断的完善自身的平台功能。LooksRare 的特色就是批量上架、购买等附加功能，以及平台的交易手续费较低，对于那些真实交易和转账的用户来说，更加友好。此次 V2 版本上线，也在这两个方面做出了更新。LooksRare 的平台费用降至 0.5% 、Gas 费比 Blur 便宜 50% ，还推出了一次签名批量挂单的功能，用户还可以一次性购买 70 多个 NFT，以及卖家可获得 ETH 而非 WETH。

熬过熊市本来就不是一件容易的事情，在「Blur 领导的积分奖励」成为 NFT 交易市场的主要引流风向时，LooksRare 并没有盲目跟风，而是坚定以更低的费用来增强 NFT 的交易体验。

不仅这些老牌 NFT 市场的新动作不断，新上线的 NFT 市场也凭借着早期、玩法创新和空投预期吸引了 NFT 玩家的关注。

Open AI CEO 领投的 mint.fun 开放通行证铸造

昨天，mint.fun 的通行证铸造在推特上的热度很高。mint.fun 是一个 NFT mint 聚合平台，由 Web3 钱包追踪应用 Context 推出，其大火的原因是 Context 去年由 OpenAI 首席执行官 Sam Altman 领投，借着 chatGPT 的热度，mint.fun 便成为了撸毛党们押注的下一个金矿。

mint.fun 目前提供的功能包括：一个界面 mint 多个 NFT；对每个项目运行自动安全检查，把可疑项目进行标记以便用户谨慎交互；汇总项目的所有信息，包括项目启动时间、已 mint 的百分比、top minters 等等；项目创建者还可以设定 mint 价格，许多项目都是免费铸造的。

此次开放的通行证铸造，会用于后续交互获取积分使用，截止发文，已有接近 25, 000 个地址成功 mint 该通行证，并获取了相应积分。该积分与 Blur 玩法相同，积分越高，获得的箱子就越稀有，并在未来追溯奖励。

Manifold 推出多 NFT 销毁机制「Multiburn」

Manifold 是一个允许创作者自行部署合约的 NFT 平台。创作者在部署完自己的 NFT 合约后， Manifold 将提供一套工具，帮助创作者自定义 Drop 机制，且无需编码能力，只需单击一下即可将其安装到你的合约中。并且 Manifold 完全免费使用，不收取任何平台费用或任何销售版权。重点是 Manifold 由 a16z 投资。

近期，Manifold 宣布推出全新 NFT 销毁机制「Multiburn」，用户可以销毁多个 ERC-721 和 ERC-1155 的 NFT 以创建全新独特的代币，多代币销毁还将提供全新的铸造费用，目前每铸造一次的费用为 0.00099 ETH，而单代币销毁的费用维持 0.00069 ETH 不变。

此次更新不仅能够减少燃烧代币所需的费用，还为创建新的 NFT 玩法提供了可能。

虽然近期的蓝筹 NFT 叙事和发展格局正在进入调整期，但是 NFT 交易市场却热闹非凡。无论 Opensea 是否能够维持其护城河，「Blur 机制」们是否会继续带动 NFT 市场流动性，高手之间的竞争一定会让 NFT 交易平台在用户体验、产品创新、奖励机制上有所突破。
