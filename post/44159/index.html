加密行业里，以太坊扩容问题的解决方案一直是个巨大的市场。 多边形战士Polygon则一直以来致力于解决以太坊的扩容问题。什么是Polygon？Polygon是以太坊的layer2扩容方案，通过提供一种通用框架来创建与以太坊相兼容的扩容解决方案，旗下拥有一套完整的以太坊扩容解决方案，今天Tokenview带你来盘点一下Polygon上那些成熟的扩容方案。

1、Polygon PoS

Polygon Proof of Stake (POS)是web3中最成熟的扩展解决方案，也是面向全球用户启动web3项目的最佳方式。

Polygon PoS是与EVM（以太坊虚拟机，Ethereum Virtual Machine）兼容的以太坊侧链。通过利用侧链进行交易处理的解决方案，实现了前所未有的交易速度和成本节约。同时，POS使用Plasma桥接框架和去中心化权益证明(POS)验证者网络来确保资产的安全性。

Polygon PoS具有以下特性：

速度

  • 直观、高效的桥接、存款和取款

  • 高达7000 tx/s的交易速度意味着比以太坊(15 tx/s)更快的性能

  • 非托管架构意味着用户可以控制他们的密钥

  • 依靠以太坊的共享安全性和Polygon的速度优势

可扩展性

  • 与主要 ERC 标准兼容

  • 由提交给以太坊的验证者和检查点（checkpoints）保护

  • 在13亿个记录交易过程中的停机时间最短

  • 一个强大的和参与的社区推动采用和用户参与

灵活性

  • 完全的 EVM兼容性 意味着直接在 Polygon 链上部署智能合约

  • 可组合的区块链意味着您可以构建各种各样的应用程序

  • 轻松往返于 Polygon 和 Ethereum

  • 在网络和移动设备上易于使用的钱包

成本

  • 由 Polygon 的权益证明架构提供支持的每笔交易的部分成本

  • 每笔交易的成本比以太坊低约 10,000 倍

  • 通过 EIP 1559 代币销毁，更可预测的Gas费和通货紧缩的代币经济学

2、Polygon zkEVM

Polygon zkEVM是第一个提供完整EVM操作码等效性和以太坊安全性的开源zkProver。最初项目名为Hermez，在2021年8月Hermez被Polygon收购后更名为Polygon zkEVM。在V神的EVM兼容性等级分类中，当前Polygon zkEVM与Scroll并列为第三级，超过其他ZK Rollup，在EVM兼容性方面处于目前的ZK Rollup里的前列。

Polygon zkEVM具有以下特性：

强大且去信任

  • zk-SNARKs 保证以太坊级别的安全性

  • 2000 TPS 的高吞吐量

  • 非托管架构

  • 资产的完全所有权和控制权（抗审查）

  • 以太坊主网上的数据可用性

唯一去中心化的zk-rollup

  • 完全开源和社区驱动

  • 与 Hermez REST API 无缝集成

  • 无许可的协调者网络

  • 带有验证奖励代币的去中心化拍卖模型

  • 治理最小化战略

简化代币转移

  • 简单的网络和移动应用程序钱包

  • 低成本代币转移

  • 原子交换（Atomic swaps）

  • 以单一货币支付的费用

  • 几分钟内快速高效的退出（提款）

3、Polygon Avail

polygon Avail是一个通用的、可扩展的、以数据可用性为中心的区块链，针对独立链和链下扩展解决方案。

Avail具有以下特性：

数据冗余

  • 每条数据都使用多项式插值进行纠删码

  • 冗余使抑制数据变得更加困难

  • 轻客户端通过执行恒定数量的查询获得约 100% 的数据可用性置信度

资源需求低

  • 无需托管全节点即可获得可用性保证

  • 应用程序客户端可以仅查询与他们感兴趣的应用程序相关的数据，而无需下载整个块

  • 系统随着用户数量的增加而扩展，增加了数据的可用性

多项式承诺

  • Avail 使用 KZG 多项式承诺避免欺诈证明

  • 每个块的数据都提交到 headers

  • 轻客户端采样数据并使用承诺开口（commitment openings）进行验证

  • 批量开口帮助我们优化验证和证明大小

4、Polygon Edge

Polygon Edge 用于构建私有或共有以太坊兼容区块链网络的模块化和可扩展框架。它以模块化架构的原则为指导，并为网络提供以太坊兼容性。

Polygon Edge具有以下特性：

随身携带以太坊智能合约

  • 用户能够通过 JSON-RPC 与行业标准钱包进行交互

  • 使用 Solidity/Vyper 开发，完整的 EVM 支持

  • 使用广泛采用的以太坊工具、库和开发工具进行构建

  • 进行跨链操作时简化 UX

自定义功能

  • 通过开发插件而不是通过分叉来构建

  • 开发人员友好的插件系统

  • 使用共识插件更改核心功能

  • 使用运行时插件超越以太坊的智能合约

跨链通信

  • 完全去信任和去中心化的内置以太坊桥解决方案

  • 在任何与 EVM 兼容的网络之间转移资产，尤其是 Polygon 和 Ethereum 主网

  • 转移 ERC-20 代币、NFT 或打包的本地货币

  • 使用桥接插件自定义桥接功能

5、Polygon Nightfall

Nightfall是Polygon与安永合作成立的一个面向企业客户的Optimistic Rollup。Nightfall 的 Optimistic-ZK rollup 可实现高效且分散的业务交易，并具有隐私性，为 Web3 上的企业开辟了新的可能性。目前已经在测试网阶段。

Polygon Nightfall 是一个 Optimistic Rollup，旨在降低私下转移 ERC20、ERC721 和 ERC1155 代币的成本。它使用 Optimistic Rollup 来降低交易费用和隐私的零知识证明。Optimistic Rollup 合约部署在以太坊（第 1 层）上。提议者将交易roll up 成块并将它们提交给Optimistic合约。挑战者向同一合约提交任何无效区块的欺诈证明。执行私人转账所需的所有信息都存在于客户端中，不依赖于任何链下第三方。

Polygon Nightfall具有以下特性：

用例

  • 供应链编排

  • 私人 NFT 市场

  • 私人 DAO 投票

交易利益

  • 效率更高，吞吐量更高，每笔交易仅需 12k gas

  • 由于 zk 技术，保密性

  • 可信度，植根于以太坊的安全性

去中心化

  • 完全拥有和控制资产

  • 非托管架构

  • 提供即时提取可替代代币的激励 LP 也可以满足最少一个诚实验证者的要求

可访问性

  • 在公共领域和社区驱动

  • 提供SDK进行开发

  • 通过 Polygon 网站提供网络钱包

6、Polygon Miden

Polygon Miden 是以太坊的第 2 层扩展解决方案。Miden 依靠零知识技术 (zk-STARKs) 将数千个第 2 层交易“roll up”为单个以太坊交易，从而提高吞吐量并降低交易费用。Polygon Miden 的核心是 Miden VM：一个图灵完备的基于 STARK 的虚拟机，它提供了一定程度的安全性并支持目前在以太坊上不可用的高级功能。

Polygon Miden具有以下特性：

安全且可扩展

  • zk-STARKs 提供的以太坊级安全性

  • 透明（无可信设置）和后量子安全

  • 分片前 1,000+ TPS 和分片后 10,000+ TPS 的高吞吐量

  • 低交易费用：比以太坊的费用低 100 倍以上

安全且可编程

  • 用 Solidity 和其他语言编写的任意智能合约

  • 以太坊上尚不可用的帐户抽象和其他功能

  • 由于安全的 VM 架构，合约的形式验证更简单

  • 清晰的隐私保护智能合约路线图

开放且无需许可

  • 完全开源和社区驱动的开发

  • 去中心化和无许可的网络

  • 非托管架构（资产的完全所有权）

  • 抗审查

7、Polygon Zero

Polygon Zero 使用 Plonky2 的速度来实现更具可扩展性和去中心化的 ZK L2。它提供汇总和验证模式，使用户能够获得更高的吞吐量和更低的费用。Polygon Zero 与其他 ZK 扩展解决方案的区别在于 Plonky2 的强大功能，它生成 ZK 证明的速度比任何其他现有技术都要快。Plonky2 支持高效的递归证明生成，允许 Polygon Zero 水平扩展，这意味着协议的吞吐量不受网络上最弱节点的限制，而仅受可用的总计算量限制。

Polygon Zero具有以下特性：

快速的

  • 由世界上最快的 zk 证明系统 Plonky2 提供支持

  • 在商用笔记本电脑上在 170 毫秒内生成 ZK 证明

  • 无需等待的以太坊安全性

轻的

  • 无状态验证器可以为每个活动帐户存储约 5 位

  • 在大小优化模式下，证明大小只有 45kb

  • 在链下执行交易，并有效地向以太坊证明

革命性的zk-EVM

  • 与以太坊虚拟机完全兼容

  • 由 Plonky2 提供支持的水平缩放

  • 一个可组合的 zkEVM，可随网络可用的总计算量进行扩展

8、Polygon Supernets

Polygon Supernets提供端到端服务，用于构建和支持专用区块链。消除了区块链开发的复杂性，而无需维护区块链基础设施。

Supernets的好处

  • Polygon Supernets 能够快速设置自定义的高性能区块链网络。

  • 可以与专业验证者合作伙伴快速启动去中心化验证者集。

  • Supernets 服务提供对高级工具和服务的访问。

Supernets特点

  • EVM 兼容

  • 可定制

  • 高性能

  • 可互操作

总结来说，Polygon是一种框架，用于创建与以太坊兼容的区块链扩展解决方案。在Polygon提供的扩展解决方案中包括零知识汇总(ZK Rollup)、乐观汇总(Optimistic Rollup)以及独立区块链。以上所述的扩展方案里，Polygon PoS是目前Polygon上最成熟和广为人知的扩容方案，而zkEVM、Zero、Miden和Polygon Nightfall4个项目组成了围绕ZK Rollup为中心的矩阵。Edge和Supernets是Polygon推出的开发者工具。未来Polygon致力于为以太坊创建一个更建康，并充满活力的Layer 2互连生态系统。
