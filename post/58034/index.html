8月29日，备受瞩目的第六期Big Demo Day在香港数码港成功举办。本次Big Demo Day 的主题是“香港数码港VS公链——香港在Web3行业的布局与规划”，是香港数码港与Central Research联合主办，Techub News、IOST、OKLink和CryptoPhd协办的大型线下活动。Big Demo Day作为系列大型活动，自今年3月举办以来，已经有超过150个项目方在活动上进行路演；大量的Web3企业在Big Demo Day中开展融资、合作等业务并获得成功，进一步推动香港Web3行业的发展。本次活动中，Techub News、金色财经，火星财经，FINMETA、Cointime、R3PO、币小白、陀螺财经、达瓴智库、世链财经作为战略合作媒体。

第六期Big Demo Day邀请了香港政府、香港本地知名企业以及Web3知名专家来进行对香港政府以及数码港对Web3领域支持的政策解读。本次Big Demo Day的演讲以及圆桌嘉宾有：Cyberport的Ian Chan先生和Rachel Lee女士、InvestHK的Terry Chan先生、 OKLink的Eason Wong先生、 Hashkey Capital的Jupiter Zheng先生和Henri Centieiro先生、万向区块链实验室的杜宇先生、IOST的Carter Lam先生、MT Capital的Eason Chan先生、TKX Capital的Chris Lee先生、O’Melveny的Edwin Kwok律师、Techub News的Alma Lee女士和Central Research的Dylan Lee女士。

而此次活动的圆桌中，Big Demo Day邀请了Web3行业的领军人物：InvestHK的Terry Chan先生、Hashkey Capital的Jupiter Zheng先生、OKLink的Eason Wong先生、IOST的Carter Lam先生和O’Melveny的Edwin Kwok先生进行了一场精彩的圆桌会议。会议主题聚焦于香港Web2企业对Web3行业的支持和布局，让众多Web3企业以及从业者更清晰的了解到香港本土企业对Web3的大力支持。

此外，活动现场还举行了一场激烈的辩论赛。来自Matr1x、Club3、Tearing Space、SEER、CredAI、Coral Finance的六位项目方代表就“在区块链游戏社区中重视社区还是产品”这一议题展开了激烈而精彩的辩论。他们的发言展现了对游戏行业发展的独到见解，还揭示了在Web3时代下游戏社区与产品之间的重要关系。

在下午的项目路演环节中，五个路演项目以精彩的演讲赢得了现场观众和资方的高度赞誉。这五个项目展示了它们的创新和潜力，同时也彰显了Web3行业的蓬勃发展和无限可能性。
