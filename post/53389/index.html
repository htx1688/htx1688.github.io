在过去的一周时间里，北美社交论坛 Reddit 上的数千个子论坛一起地变为灰色的“Dark”状态，这代表这这个论坛的管理员暂时将论坛状态设置为了受限制的访问或者私密状态。在这个状态下，普通用户无法访问这个论坛里的任何内容。截止 6 月 18 日，仍有近半（3933/8829 ，来自 reddark 网站数据）的论坛选择保持 Dark 状态。也就是说，Reddit 事实上有一半的社区管理层正处于罢工状态，而他们的抗议直接让这个北美社交巨头的一半的内容变得不可访问。

想象一下，在 2012 年左右百度贴吧最鼎盛的时期，半数贴吧因为一场抗议运动被吧主们直接关闭，普通用户完全无法访问。这会是一个什么样的场景？此时此刻，这样的一场运动就真实地发生在“大洋彼岸的百度贴吧”上。

Reddit 上 Go Dark 的社区之一（Reddit）

Reddit 上发生了什么？

从上周起，Reddit 上的数千个子论坛发起了“Go Dark”运动，暂时关闭他们的社区大门，以抗议该平台最近对 API 定价的改变。 最初计划进行 48 小时的罢工已无限期延长，许多社区选择保持私密状态。罢工管理层声称，API 定价的改变将会导致 Reddit 上的第三方应用消亡，而让更多社区内的贡献者（比如版主和内容作者）没有趁手的管理工具可用。

究其核心，这就是一个中心化的论坛公司为了提高自身收入而牺牲社区内部第三方应用和贡献者的故事 。Reddit 本身在提供了论坛平台之外，并没有对论坛内部的管理和协作提供太多资源，所以大部分的大论坛版主（有些论坛的参与人群超过 1000 万人）都依靠外部的第三方软件来管理论坛上的内容和参与者。这些第三方软件需要购买 Reddit 的 API 从而访问 Reddit 上的内容和信息。这次，Reddit 因为冲击上市的原因选择对于 API 大幅提价，而提价后的价格是绝大多数第三方软件完全无法接受的。这些软件可能会被迫关闭，从而极大地伤害了 Reddit 版主和论坛管理人员的管理效率，进而伤害了 Reddit 的普通用户们。

主要社区，包括 r/funny、r/science、r/gaming 和 r/aww，都参与了抗议活动，导致 Reddit 上最受欢迎的一大批子论坛下线。这些子论坛中包括了数百万的订阅者，占据了该平台用户基数的一大部分。一些子论坛正考虑永久关闭，除非他们的诉求得到充分解决。

根据 CNBC 的报道，Jacqueline Sheeran，在 Reddit 上以“MCHammerCurls”而闻名，是 r/Fitness 的主要版主，该论坛拥有超过 1000 万会员。她表示，志愿版主们依赖第三方应用来实施各种安全功能，以便标记关键词、短语和表达方式。而这些功能在健康相关的论坛上非常重要。

她说：“很多（来 Reddit 上寻找答案的人）有很重要的健康问题，比如饮食失调和扭伤……我们试图确保人们在进行活动时能够保持安全和健康，同时不受机器人或垃圾账户的干扰。”而这些功能都需要依靠第三方的软件和开发者才能够实现。如果这些人被迫关闭他们的软件，其带来的后果可能不止几个论坛的关闭和工具的消失。

中心化、提价和抗议

Reddit 到底把价格提高了多少，以至于这么多论坛管理人员和第三方开发者无法接受呢？根据 Reddit 上著名的第三方管理应用 Apollo 的开发者 Christian Selig 的帖子， 5000 万次请求需要 12, 000 美元，而这个数字是他远远无法接受的。作为对比，Imgur（Reddit 的主要竞争对手）上的同样数量的请求仅需要 166 美元。

Selig 说，“根据新的 API 定价，Apollo 上个月发起了 70 亿次请求，这将使每月费用达到大约 170 万美元，或者每年 2000 万美元。Apollo 用户需要支付的平均费用将达到 2.5 美元，这比每个用户能够给 Reddit 带来的收入的高出约 20 倍。”

他声称，如 Reddit 不打算改变定价计划，Apollo 会在月底关闭。

作为对比，Reddit 在 2021 年表示其季度广告收入首次达到 1 亿美元。据估计，这家公司的年收入稍稍低于 10 亿美元，但是并未实现盈利。Reddit 在 2021 年底秘密提交了上市申请，但随后的牛市结束与熊市到来极大地影响了它的上市计划。就现在来看，上市能否成功暂且不清楚。很多人把上市申请与这次的提价联系到一起，认为 Reddit 通过提高营收而仍在向上市发起冲击，而这次的规模浩大的抗议毫无疑问地打断了 Reddit 提价的野心。

Reddit Go Dark 的社区统计（Reddark）

“没有人喜欢这个情况，”版主 Croach 说道。“没有人希望关闭网站。没有人为此而欣喜若狂。没有人对此感到高兴。我们这样做是因为……我们热爱 Reddit 的方方面面，而且我们真切地感到这些决定不仅可能对网站的未来造成损害，而且对很多人（包括第三方开发者）来说，这些决定实在是太不公平了，这些人多年来都自愿为网站贡献了自己的时间……最重要的是，我们希望尽快实现积极、和平的结果，让一切都恢复正常。”
