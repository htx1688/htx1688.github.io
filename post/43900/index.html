让死亡在元宇宙中成为现实

准确点来讲，我的意思是，在加密游戏中引入永久性死亡的概念，就像在现实生活中一样。

游戏中的永久性死亡将开启新的体验和游戏方式。

当你在玩 FPS 或 RPG 游戏时，你可能会在游戏中“死亡”。此外，你将拥有一定数量的生命，要么固定，要么无限。死亡的概念存在于游戏中，但它更多的是一种反馈机制，即你在游戏玩法的某个方面表现不佳，而真正的死亡则意味着游戏彻底结束。

我认为将这一概念引入加密游戏是有机会的，它将在玩法中解锁新的体验，为玩家创造新的赚钱机会，并可能创造一些最吸引人的观赏性体验。

为什么是现在？为什么是加密游戏？

永久性死亡机制曾在各种游戏中被尝试过，但成功率极低。由于各种原因，它没有获得像其他模式那样的关注和赞赏，即固定数量的生命或无限生命。

在我看来，NFT 开辟了一个新的机会，而其中有两个关键方面：

  1. NFT 代表数字所有权

  2. NFT 可以被购买、出售、交易，或在一个游戏之外的其他情况下使用。

与之前的游戏体验相比，这些要点是非常重要和独特的，因为玩家现在不仅可以在游戏中使用角色，还可以在公开市场上进行买卖（是的，市场确实存在，但其规模和流动性还不足）。此外，我们开始看到了 NFT 在互操作性方面的可能，而我相信这将继续增长。如果这些都是真的，那就意味着游戏角色将拥有更大的价值（不仅是货币价值，还有对玩家的情感价值），因为它们不再局限于单一游戏中。这种互操作性可以表现在许多方面。能够在多款游戏中使用同一个角色就是很明显的例子。此外，我认为还会有促销合作伙伴关系、会员机会以及更多我们目前无法想象的可能性。

所有这些让我相信，NFT 角色的价值潜力比传统游戏角色大得多，因此在游戏中引入死亡的损失概念比以前更有趣，也更有吸引力。

关于新体验、行为和游戏性的理论研究

想象一下，在一个 FPS、对战游戏或其他游戏中，你的角色是一个 NFT，当你赢得战斗时，你会积累技能，但如果你战败，你的角色就会死亡，而且 NFT 会被销毁。那么，这意味着什么？

  • 如果这款游戏非常受欢迎，那么任何玩家都可以从零基础开始，纯粹依靠自己的能力去打造一个高战力角色。从理论上来讲，高战力的角色将是非常罕见的，因为许多角色会在游戏中死去。这些玩家可以用这个角色进行游戏，或者把它卖给那些想要高战力，但又不想投入精力的玩家。这就为那些几乎没有资本的优秀游戏者创造了赚钱的机会，而且这里的稀有性不是随机的（不像目前的 NFT 铸造），而是基于技能、努力和生存。

此外，我认为还有一个机会，就是皮肤或物品只有在达到一定级别或在一定数量的战斗中幸存下来才能解锁。这将为这些角色创造额外的价值，而不仅仅是游戏中的技能。

  • 由于在游戏中死亡意味着你失去了你的角色，所以玩家可能会更加专注于自己的角色，并更加重视角色的行动。而这些都可能会吸引更多的玩家参与进来。

而且，我认为，观众也有可能变得更加重视各种高战力的持久性角色。

  • 将这些战斗通过流媒体传播，可以在游戏中吸引更多的观众，更重要的是，在游戏之外，也会有更多的观众被吸引过来，就好比世界上最大的体育赛事（超级碗、世界杯、奥运会等）凭借其兴趣规模及对流行文化的影响，吸引体育迷以及许多“非体育迷”观看一样。由于游戏中的角色一旦输了就会“死去”，所以它能引起观众更多的情感，并且可能会比目前的游戏更加有趣。

要点

除了上述提到的几点之外，肯定还有无数我没有想到的其他可能性。然而，我相信游戏中永久性死亡的概念会为玩家和观众增加了许多吸引力，这在过去是行不通的，但现在可以通过 NFT 来实现。

在我看来，如果在加密游戏中采用这种方法，我们将看到许多基于较低准入门槛的经济机会，同时也会带来游戏观看的潜在革命，因为对于非游戏观众来说，这将更加真实和发自内心。就像体育迷在观看比赛时的感受一样，他们知道运动员正在将自己的身体投入到比赛中，而在某些情况下，例如赛车运动和格斗运动，他们的生命是处于危险之中的。
