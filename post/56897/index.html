2.热门项目 Popular Projects

本周( 08.07-08.11)热度最高的项目有：

OXBT（Token）： 由 KOL @BitGod 发行的 BRC 20 代币，通过多社区、kol 白单+公开免费铸造方式发行，上线后持币地址迅速达到近 2 万个；项目目前仍是 BRC 20 代币持币地址最多的项目之一，链上转账活跃，项目方在推特表示目前 BRC 20 赛道士气低落、流动性有限，需要持续建设的毅力；

XRP（Token）： 类 Bitcoin meme 币的项目，全称 HarryPotterObamaPacMan 8 Inu，疑似 KOL 发币，由 @Yourpop 8、@stekisteks、@henokcrypto 三人重点宣传，代币上线价格上涨数十倍，市值达到 840 万美金，过去 24 小时交易量 273 万美金；

LINK（Token）： Chainlink 的代币价格在试探 2 年以来的下降趋势线，自上月 Chainlink 推出跨链互操作性协议（CCIP）后，其已在拓展预言机外的基础设施能力，近期及时支持了 BASE 网络的预言机喂价，产品更新迅速，LINK 代币价格表现较强，近两个月涨幅超 60% ，维持在高位；

SHIB（Token）： 基于以太坊的 Layer 2 网络 Shibarium 跨链桥已上线公开测试，并将开启 SummerOfShibarium，SHIB 代币近一周最高上涨 20% ，热度较高；

Pandra（NFT）： Polyhedra Network 发行的 NFT， 进行 Pandra 系列 NFT 跨链，可以同时交互 zkBridge 和 LayerZero，任务截止时间为 8 月 20 日，刷空投党拉高了该 NFT 的 Transactions；

Coinbase（Dapp）： Base 正式对公众开放主网，defi 项目 Maverick 上线 Base 网络，并进行 LP 活动，提供 cbETH-ETH 等交易对的流动性可铸造 Mava IV-Base NFT，将为 Coinbase 的 LSD Token cbETH 引入到 Base 网络提供支持；

Bridegebase（Dapp）： Base L 2 宣布主网开放，并开通官方的跨链桥，支持以太坊跨链 ETH 到 Base 上，过去 7 天 Base L 2 的 TVL 资产增长迅速，达到 72% ，目前跨链资产达 1.42 亿美金；

Circle（Dapp）： USDC 稳定币发行商 Circle 推出新的可编程 Web 3 钱包平台，允许开发人员和商家集成 Circle 可编程钱包，并个性化到他们的应用中，以及在上面构建服务，让消费者发送、接收和存储加密货币，包括 Circle 的 USDC 和 NFT；
