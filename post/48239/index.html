一方面，美国政府对其大力施压，进行了“提醒流动性风险+处罚”的双重施压；另一方面，曾经关系亲密的客户纷纷宣布停止合作。而这一切，都让 Silvergate 濒于破产的边缘。就在早些时候（3月9日上午），Silvergate 在一份公告中声称，根据最新的市场和监管动弹，银行将停止运营，未来的清算计划包括全额偿还所有存款，这一公告无疑做实了破产的处境。

从行业领先到深陷困境，Silvergate 究竟经历了什么？

Silvergate 发家史：曾经炙手可热，拥有诸多知名客户。

在聊 Silvergate 的暴雷事件之前，我们先来了解一下 Silvergate 的发家史。

早些年间，Silvergate 还只是一家位于加州的社区零售银行，不过，正所谓“时势造英雄”，Silvergate 的管理团队看到了加密货币浪潮中的红利和商机，牢牢抓住机会，通过向交易所等加密货币机构提供美元和欧元结算服务，一举成为币圈银行的风向标。

事实上，Silvergate 原本是做传统金融的，它将加密货币纳入主流货币，无疑承担着巨大的风险。不过，仅仅用了5年的时间。Silvergate 就成为了广大美国加密货币创业公司的首选银行，也迅速成长为加密领域中最著名的银行之一。。包括 Coinbase，Gemini，Kraken，bitFlyer 在内的诸多交易所，都先后成为了 Silvergate 的忠实客户。

Silvergate 最重要的功能，也是最能巩固其行业地位的功能，即是引入了 Silvergate Exchange Network (SEN)，一个支持 API 的实时平台（可以随时即时和下线访问法定货币）。

Silvergate 团队曾经对外表示，Silvergate 建立了一套持续监控的流程，相关人员会使用基于区块链资产的监控工具，从而保证日常监控和日常通信的顺利进行。除此之外，团队奉行“缓慢而有选择的挑选客户”这一原则，旨在规避平台上可能出现的非法东西。

但遗憾的是，Silvergate 并没有办法完全按照自己的原则行事。

FTX 地震余波不断，Silvergate 被牵连被挤兑。

3月2日，Silvergate 对外宣布，将推迟提交及说明公司年度财务报告，并表示仍然需要两周的时间去完成2022年度的财年报告。无独有偶，Silvergate 在一份前瞻性生命中警告，它可能面临来自银行监管机构和美国司法部的询问，其在未来一年“持续经营”的能力可能会受到影响。

一石激起千层浪。消息一出，Silvergate 的股价在盘后下跌超过30%，截止消息公布当天收盘，Silvergate 大跌57.72%，报收5.72美元。更要命的是，这条消息如同多米诺骨牌一般产生了连锁反应，整个加密货币市场因此迎来大跌，比特币跌破23000美元，24小时跌幅达到4.2%；加密总市值跌至1.07万亿美元，24小时跌幅达5.2%。

大盘狂跌，很多与 Silvergate 有过交集的公司也坐不住了，纷纷发表声明与其撇清关系。

与Silvergate颇有渊源的 Coinbase 表示，“鉴于最近的事态发展和高度谨慎”，它不再接受 Silvergate 或从 Silvergate 发起付款。该交易所补充说，它对银行的“企业敞口微乎其微”，机构客户的现金交易将通过其他银行合作伙伴来促进。

加密货币金融服务公司 Galaxy Digital 表示，它已停止接受或启动向 Silvergate 的转账，并补充说它对银行“没有重大风险敞口”。

Micro Strategy 是目前最大的比特币持有机构，它的发言人表示，如果 Silvergate 面临破产，它与 Silvergate 的比特币贷款要到2025年第一季度才到期，不会提前偿还。

加密货币经纪商 Paxos 表示，将放弃 Silvergate，停止 SEN 转账和电汇到其Silvergate账户。Paxos的发言人表示，“Paxos 将继续处理付款，Paxos 与 Silvergate 没有任何实质性接触。Paxos 的首要任务始终是保护其客户的资金和资产，因此我们利用多元化的银行合作伙伴网络”。

那么，问题来了，一个实力雄厚，资源丰富的币圈银行，是如何走到破产边缘的？

Silvergate 的垮台，始于 FTX 的崩溃。

2022年11月，知名交易所FTX彻底崩盘，这次事件对 Silvergate 产生了致命一击。要知道，FTX是Silvergate最著名的客户之一，Silvergate 大约有10 %的总资产属于 FTX。FTX的倒台，意味着 Silvergate 被牢牢套住的金额超过10亿美元。

在此之后，Silvergate 立刻采取行动，紧急处理了81亿美元挤兑提款。为了满足大量提款，Silvergate 被迫紧急出售大约52亿美元的债务证券，并从美国联邦住房贷款银行（FHLB）获得了43亿美元的贷款。随后的2022年第四季度财报显示，2022年年底，Silvergate 管理的客户存款暴跌至38亿美元，远低于第三季度的119亿美元。

为了将损失降到最低，Silvergate 进行了一系列措施，包括裁员200人，暂停优先股股息支付等，但直到今天，任何补救措施都已无力回天。

美国相关部门态度强硬：不制定新加密法规，不阻止银行提供加密服务。

FTX 暴雷，资产大幅亏损，合作伙伴割席自保…Silvergate 目前的处境异常严峻，更为不幸的是，它还不得不面对来自官方的强硬态度。

FTX 的崩塌，使得美国金融监管机构对虚拟资产采取了更为严厉的立场。最近几个月，各大监管机构纷纷展开整顿行动，实施重大处罚，并向业内参与者发布了广泛警告。此外，监管机构还向质押，稳定币，NFT 项目挥起大棒。

2023年年初，美联储，美国货币总稽核办公室和美国联邦存款保险公司已经两度联合发文，要求银行体系关注与加密货币实体有关的流动性风险，当然，官方所用到的措辞是 “既不鼓励也不禁止”，但这已然说明了“政府不会偏袒加密货币银行”态度。

值得一提的是，美东时间3月7日，美联储主席鲍威尔如期出席半年一度的参议院银行委员会的听证会，就半年度货币政策报告发表证词演讲。在演讲中，鲍威尔强化了所谓“鹰派”的预期。他指出，美联储终端利率“可能会高于”先前预估的水平，更重要的是，如果有需要，美联储会做好加快加息步伐的准备。鲍威尔讲话之后，美股三大指数直线下挫，非美元货币集体遭遇重挫。

相较于应声下跌的美股，目前加密货币市场的表现还算稳定。BTC 和 ETH 始终维持着小幅震荡，并未出现大幅下跌。不过，Silvergate 突如其来的终止运营无疑让币圈陷入了多事之秋，同时多年以来的门头沟赔付事件即将尘埃落定，“3.12”纪念日的来临，多重利空因素导致当前市场氛围低迷，我们还是要慎重提醒各位小伙伴们：币圈有风险，投资需谨慎。
