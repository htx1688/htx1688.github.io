原文作者：Daniel Li，CoinVoice

Layer 1 区块链 Sui 宣布其主网将于今年第二季度推出，并开启名为「 Wave to Mainnet」的一系列教育活动和内容，旨在主网上线前，让社区了解 Sui 协议的潜力。Sui 与 Layer 1 龙头 Aptos 同属于 Meta 系公链，在项目建立时间上二者几乎同时创立，与 Aptos 相比，Sui 更注重于底层技术的创新，其目标是成为第一个互联网规模的可编程区块链平台。尽管在发展势头上 Sui 远不如 Aptos，但是以技术创新为驱动力的项目往往具备厚积薄发的潜力，Sui 在过去几年积累了丰富的技术优势，随着 Sui 的主网即将发布，相信很快 Sui 会迎来一波发展高潮。

默默耕耘的 Sui 与站在聚光灯下的 Aptos

Aptos 继承了 Die 项目近三年的开源技术和开发经验，可以快速实现底层架构的搭建和生态项目的布局。与之相比，Sui 选择从零开始构建 Layer 1 网络，导致其在开发进程上远远落后于 Aptos。所以早期在吸引资本和投资者关注方面，快速发展的 Aptos 比还在搭建底层架构的 Sui 更能吸引投资者的目光。其次，二者作为 Layer 1 公链都致力于改善区块链的安全性和可扩展性，打造一个面向数十亿人的高性能区块链基础设施网络。但在实现目标上，二者的战略或重点存在一定区别。

Aptos 注重市场营销，Aptos 被包装为来自 IT 行业最优秀开发者创造的独特产品便是其营销手段，虽然夸张，但是胜在效果好，在过去顶着 Layer 1 领导者的名号，Aptos 吸引大批的投资者。去年 Q4 启动主网并发币。虽然价格一路缩水超过 76% ，但自年初以来表现十分强劲，最近有所回调，但总市值仍达 20.8 亿美元。其次在生态项目合作方面，Aptos 注重与知名企业的合作，例如在基础设施方面与 Google Cloud 启动加速计划；在游戏方面与韩国 3 A 级游戏工作室 NPIXEL 达成合作；在支付方面与 Web3 金融科技公司 MoonPay 合作整合生态系统项目 Petra 等。与知名企业的合作为 Aptos 带来了巨大的流量和关注，也促进了其生态项目的快速发展。

与 Aptos 不同，相对于市场营销，Sui 更重视底层技术的创新应用。例如，Sui 改进版 Move 语言能够更加清晰地表明对象的所有权、可变性以及可组合性使其天然适用于搭建元宇宙设施。在生态项目方面，由于 Sui 的主网尚未上线，大部分项目还处在测试阶段，所以在关注度方面 Sui 远不如 Aptos。因此，在很多时候，Aptos 就像一个站在聚光灯下的明星，而 Sui 则更像一个在后台默默付出的耕耘者。

Sui 和 Aptos 同为 Layer 1 公链，创始团队都来自 Diem，且又同属于 Move 赛道，所以二者从一开始就存在竞争关系。但是，由于 Sui 是从零开始搭建 Layer 1 网络，开发进程相对滞后，早期市场大部分目光也都集中在 Aptos 上，因此二者之间的竞争关系在开始阶段并不明显。直到最近 Sui 的 Wave 2 测试圆满结束，并宣布主网即将上线，这才引起了 Aptos 的关注和警惕。上个月 Sui 的 CTO 就在 Twitter 上指控 Aptos 向 MOVECON（Move 生态第一场大规模论坛）的主办方 Pontem 团队施压，取消 Sui 团队的演讲，甚至把 Sui 基金会从活动组织者名单中剔除。这表明 Aptos 已经开始将正在崛起的 Sui 视为竞争对手，也从侧面肯定了 Sui 的市场地位和潜力。

与 Aptos 相比 Sui 底层技术优势明显

尽管与 Aptos 相比 Sui 还有很多不足，例如生态项目和社区规模方面 Sui 不如 Aptos，但 Sui 一直致力于底层技术的创新和应用，在过去的一年里也积累了丰富技术优势，从 Sui-Move 语言到 Narwhal-Tusk 共识机制再到存储基金代币经济学，Sui 在区块链项目的每个层面都进行了重大创新，就此而言，即使与 Aptos 等其他新的 Layer 1 公链相比，Sui 也是独一无二的。

Sui-Move

Aptos 和 Sui 都使用 Move 语言，但两者有所不同。Aptos 在设计上基本遵循了 Diem 白皮书的经典设计思路，而 Sui 则进行了升级和调整，形成了独特的 Sui- Move 语言。与 Move 最大的区别在于，Sui 采用以对象为中心的模型，而不是以地址为中心的模型。因此，在 Sui 上，几乎所有的东西，如代币、智能合约、NFT 等，都可以表示为「对象」。每个对象都有一系列属性，包括所有者地址、读写属性、可转让属性、功能属性等。Sui-Move 明确表明了对象的所有权、共享、可变或不可变等特性，而 Aptos 则没有明确表述。此外，Sui 的所有权 API 比 Aptos 的 API 更为简洁，因为它更清晰地展示了区块链设计。在功能方面，相对于 Move 语言，Sui-Move 在安全性、智能合约支持、编程模式灵活性和兼容性方面具有一定的优势。

Narwhal & Tusk

Sui 的共识机制都最大限度地减少了验证者之间处理交易所需的通信，以实现更低的延迟。Sui 的共识机制分成了两部分：Narwhal 和 Tusk。Narwhal 是一种内存池协议，用于存储未经确认的区块链交易，并在共识协议验证它们之前广播这些交易。该协议的任务是在保持数据可用性的同时达成共识，并且独立于共识协议，能够容忍异步或间歇性故障。Tusk 是一种异步共识协议，负责交易排序。该协议使用共享随机性来消除节点之间的额外通信，并允许每个节点确定交易的总顺序。当与 Narwhal 协议组合使用时，可确保在发生故障时仍能保持高性能。

理论上，Sui 这种设计的吞吐量可扩展性没有已知的上限。并且在安全性方面，相对于对同步区块链（即大多数基于工作量证明的区块链），Sui 的安全属性能够抵御恶劣的网络条件、网络分区或对验证者的 DoS 攻击，因为它不会对网络施加任何同步假设。

开发者体验

Aptos 的核心功能是无缝升级，可让开发者专注于构建，而不会干扰用户。相比之下，Sui 开发工具包（SDK）则为开发人员提供更多便利。SDK 提供了开源、通用和用户友好的工具，并允许开发人员在调试智能合约、等待审计和构建基本技术栈元素上节省大量时间。此外，SDK 还尝试连接其他生态系统和非加密用例，如：游戏 API 直接连接到通用 Layer 1 ，而不是侧链或专注于游戏的 L2；「SuiEcho」通过促进数字资产从其他生态系统到 Sui 的可移植性来引导社区；「Handshake」是一种前端工具，可作为用户向加密和非加密用户分发、领取 / 赎回 Sui 数字资产（如支付、商户优惠券）的渠道。

Sui 代币经济学

除了一系列旨在优化系统的可组合性和可扩展性的技术创新外，Sui 还实施了一种独特的代币经济学模型。与传统的权益证明链的代币经济学模型不同，Sui 的代币经济学模型增加了「存储基金」功能。由于区块链是一种只增不减的数据结构，随着时间其链上需要维护的数据会越来越多，成本也会相应提高，这就导致了后期使用 Sui 的用户需要支付比早期用户更高的费用，「存储基金」基本就是为了解决这个问题而生，让早期用户支付更多费用，使所有户可基于类似的费用水平使用该网络。

此外，Sui 的存储基金还会支付给系统中的验证者，或者实际存储区块链数据并完成数据维护工作的人。随着维护工作的增加，给验证者的支出也会增加，这也会激励更多人成为系统中的验证者，而随着越来越多的人成为验证者，可用的存储空间总量也会增加，进一步导致写入区块链的 gas 费用降低。Sui 的代币经济学模型是高度可扩展链经济模型的典型代表，使用了博弈论来确保 gas 费在区块链的整个生命周期内保持相对恒定。

Sui 生态项目盘点

混合流动性交易应用 MovEX

MovEX 是 Sui 上的第一个 AMM+ 订单簿混合流动性 DEX，它结合了 AMM 和订单簿的优点，通过创建混合流动性池为每个人提供流动性交易体验。在内部，MovEX 有一个在 AMM 和订单簿之间分配订单的结算引擎，它既保证了交易者的滑点最小，又保证了流动性提供者的公平。流动性提供者还可以自定义价格范围提供单面流动性，能够有效管理无常损失。

MovEX 支持市价交易和限价单交易两种在中心化交易平台常见的功能。普通用户可以以自定义价格范围，为流动池提供流动性；专业交易者和流动性提供者可以通过订单簿成为流动性提供者，类似于中心化交易所的做市商。目前，MovEX 产品仍处于测试阶段，用户可以通过 Sui Wallet 钱包链接应用，体验产品。

钱包 Sui Wallet

Sui Wallet 是一个浏览器扩展，充当 Web3 世界的门户，允许用户连接到 DApps 并从受信任的 DApps 签署人类可读的交易。它是 Sui 区块链的官方钱包，允许用户创建、导入和持久存储助记词和派生私钥，转移硬币和 NFT，以及查看拥有的可替代代币和 NFT。Sui 钱包最初是为 Sui 开发者测试使用而开发的，是目前测试网认可度最高的钱包。

去中心化交易应用 Suiswap

Suiswap 是 Sui 链上的自动做市商机制（AMM）的 DEX，支持 Sui 生态内原生资产的兑换和交易。目前，Suiswap 支持用户领取测试 Token，进行交易或者组建 LP 为流动性资金池提供流动性，产品的最终形态需要等 Sui 主网上线后才会见到。

超额抵押借贷应用 Leizd Protocol

Leizd Protocol 是 Sui 生态内的去中心借贷应用，采用超额抵押形式运行借贷功能。该应用以去中心化美元稳定币 USDZ 作为中间资产，像以太坊链上的 MakerDAO 一样，通过铸造稳定币来实现借贷产品的风险可控。在 LeizdProtocol，借款人可以存入加密资产作为担保物铸造 USDZ，然后借出所需的 USDZ,将之兑换成 USDT 或 USDC 等稳定币使用，到期归还所借的 USDZ 和利息即可取出抵押资产。存款人同样可以抵押加密资产铸造 USDZ 换取存款利息，或者将 USDZ 兑换为其它加密资产再存入存款池赚取利息。

据悉，Leizd Protocol 计划多链部署在 Sui 和 Aptos 两个网络上，允许用户用两条链上的资产实现借贷。不过，Leizd Protocol 的测试产品还未上线，产品还存在于概念介绍阶段。

NFT 交易平台 BlueMove

BlueMove 是一款支持 NFT 创建和交易的平台，已经在 Sui 和 Aptos 两个网络上成功部署。平台的智能合约采用了模块化系统，并通过标准化签名明确定义执行范围，可以在不影响安全性的情况下随着时间的推移推出新功能，例如集合报价、特征报价和多次取消等功能。

除此之外，BlueMove 还发行了总量为 3 亿的平台通证 MOVE，其中 5% 用于空投， 40% 用于奖励平台积极参与的交易者和收藏家， 15% 用于奖励用户质押 MOVE 代币，其余则用于融资和团队开发。MOVE 代币的主要效用为链上治理和会员权益。

目前，用户可以在 BlueMove 上购买或出售 NFT，并获得 MOVE 奖励。此外，用户还可以通过质押 MOVE 代币获取平台的交易费用分成以及额外的 MOVE 奖励。

Web3 链游：Abyss Worlds（深渊国度）

Abyss World 是一款由 Metagame Industries 开发的基于 Sui Layer 1  区块链的动作角色扮演游戏，拥有强大的 Web3 技术和游戏体验。它聚合了 Web2 和 Web3 游戏玩家，让玩家在虚拟世界中体验到身临其境的游戏乐趣。游戏中的各种角色，地图和剧情都将由 Sui 区块链技术支持，让玩家创造出真实的游戏世界。根据路线图，Abyss World 将于今年第三季度正式开启公测。

在 Web3 的玩法里，NFT 是吸引用户早期参与项目的关键选项，Abyss World 在 3 月 23 日就开启首批创世 NFT Gazer 系列的售卖活动。这些 NFT 并不是单纯的小图片，而是与游戏内的权益和功能进行了绑定：持有 NFT 的玩家可以在游戏内分享潜在收入、获得更多的稀有道具和皮肤，以及抢先得到游戏的内测资格。此外，由于有着和 Mysten Labs 的深度合作，NFT 除了绑定 Abyss World 的游戏原生代币 AWT，也有绑定 Sui 代币空投的权益，可谓是目前 Sui 生态的头部项目。

总结

Sui 是第一个从头开始设计的无许可 Layer 1 区块链，致力于为 Web3 用户提供更加友好的使用体验。然而，技术只有在解决实际问题时才能发挥其价值，并且只有在人们愿意为其付费时才能真正获得成功。Sui 一直专注于底层技术的创新和应用，这为其带来了丰富的技术优势。随着 Sui 的主网即将上线，现在是将这些技术优势转化为市场竞争力的时候了。通过「Wave to Mainnet」的系列活动，Sui 正在从幕后走向前台，向人们展示多年来积累的技术实力。未来，底层技术将如何推动整体 DX/UX，谁将能够吸引更多用户和流动性，这些问题将随着时间的推移而逐渐得到回答。
