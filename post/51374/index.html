情绪之下，一些新项目开始布局比特币 DeFi 方向，由于比特币网络的特殊性，DeFi 领域很少涉及，但居然连图片和 Token 都能炒起来，DeFi 似乎也不是没可能。BlockBeats 整理了一些比特币 DeFi 概念项目：

DeFiChain

DeFiChain 是一个创新性的 DeFi 平台，致力于提供快捷、智能、透明的 DeFi 服务。该平台旨在通过将 DeFi 功能完全集成到比特币生态系统中。DeFiChain 本身是比特币的硬分叉，其大部分编程代码与比特币相似，使用比特币共识机制来保护其网络。DeFiChain 提供各种 DeFi 服务，包括去中心化借贷、去中心化交易平台和去中心化资产管理。

据 DefiLlama 数据显示，当前 DeFiChain TVL 约为 2.21 亿美元。DFI 是 DeFiChain 平台的原生 Token，可用于支付 DeFiChain 的网络费用，赚取贷款利息，并参与治理。平台无需许可，任何人都可以参与其生态系统。

Rootstock

Rootstock 是由比特币矿工担保的智能合约平台，目前是最安全的基于工作量的智能合约网络。作为比特币侧链，它通过扩大比特币货币的使用来提升比特币生态系统的价值，还在 EVM 等方面和以太坊有很好的兼容性。去中心化应用程序可以使用 Solidity 编译器和 Web3 标准库编写，从而实现以太坊兼容性。此外，它还可以通过 RIF Lumino 支付渠道网络提供的更多链上空间和链外交易来扩展比特币支付。Rootstock 侧链的原生 Token 是 RBTC。

当前 Rootstock TVL 约为 8850 万美元，其中借贷协议 MoneyOnChain TVL 约为 4425 万美元，Sovryn TVL 约为 2250 万美元。

MoneyOnChain 是一种比特币担保的稳定币协议，允许用户通过抵押比特币来铸造与美元挂钩的稳定币 DOC。DOC 可以用来在 MoneyOnChain 平台上进行交易、借贷。

Sovryn 允许用户在不需要中心化机构的情况下借出、借入、交易比特币并赚取利息。Sovryn 是开源和无许可的，任何人都可以参与其生态系统。

2023 年 3 月 17 日，Sovryn 推出 100% 由比特币支持的稳定币 Sovryn Dollar（DLLR），DLLR 从其他比特币支持的稳定币中聚合而来的，并与美元挂钩。Sovryn Dollar 维持着大量超额抵押的 BTC 储备，以确保其始终可以被持有者兑换成 BTC。

BadgerDAO

BadgerDAO 是一个专注于在以太坊生态系统中构建比特币与 DeFi 之间桥梁的项目。BadgerDAO 旨在为比特币持有者提供更多的工具和服务，通过使比特币能够用作 DeFi 应用程序的抵押品，让用户轻松地将其资产带到 DeFi 领域，并从中获得收益。

作为一个社区驱动的项目，BadgerDAO 允许持有其治理 Token Badger 的用户参与项目的治理和决策，同时，BadgerDAO 还为开发者提供了激励计划，以鼓励他们为项目贡献代码和创新，目标是通过不断创新的产品和服务促进比特币在 DeFi 中的增长。

其核心产品是 Sett，一个基于自动化的收益优化策略的资产管理工具。Sett 支持多种基于比特币的资产，如 Wrapped Bitcoin (WBTC)、RenBTC 和 tBTC。Sett 允许用户将其资产存入智能合约，然后根据预先设定的策略在不同的 DeFi 协议中自动赚取收益。BadgerDAO 还推出了一种名为 DIGG 的弹性供应货币，其价格与比特币挂钩。DIGG 的供应量会根据市场需求进行调整，以保持与比特币的价格稳定。

Stacks

Stacks 是比特币网络的 Layer 2 ，它在不修改比特币本身的情况下为比特币带来智能合约功能。Stacks 采用金字塔方法，在底部有基础结算层（比特币），然后在其之上添加智能合约和可编程性（Stacks），然后在其之上添加层以实现可扩展性和速度（Hiro 的子网）。

通过采用这种分层方法，能够拥有与以太坊等相同的所有功能。Stacks 作为比特币的 Layer 2 具有一些独特的属性，比如拥有自己的 Token，作为一种激励机制来维护其所有交易的历史分类账，并以自己的安全预算运作。

当前 Stacks TVL 约为 3600 万美元，其中去中心化交易平台 ALEX（TVL 约为 3400 万美元）为主要 TVL 贡献方。Stacks 允许利用比特币作为基础结算层来构建去中心化和抗审查的软件。最终目标是建立一个金融系统和去中心化软件产品的网络，这些产品都使用比特币作为他们的货币。目前生态系统的最高优先级是将 Stacks 与比特币更紧密地集成，以便轻松地与 Stacks dapps 一起使用。

Mintlayer

Mintlayer 是一个建立在比特币之上的 Token 化层，支持与 BTC 的 Layer 1 、Layer 2  闪电网络进行交互。作为建立在比特币网络上的第 2 层解决方案，Mintlayer 消除了对 wrapped Token 或 Token 桥（DeFi 黑客的主要攻击媒介）的需求，而是使用原子交换进行从本机 BTC 到 Mintlayer 区块链上 Token 的 1: 1 交易。

Mintlayer 使比特币持有者能够使用其原生 BTC 访问所有类型的 DeFi 工具以及庞大的 Dapps 生态系统。通过原子互换机制，用户可以将 Mintlayer 生态系统中的任何 Token 交易为原生比特币，不再需要受信任的第三方。Mintlayer 使用比特币 UTXO 结构，而不是 Ethereum、Ripple、Stellar 或其他的帐户基础模型。这样带来的好处是更加注重隐私，并且付款可以在单笔交易中批量（汇总），从而节省大量空间。

Inscribe Finance

Inscribe Finance 是近期基于 BRC-20 构建的 DeFi 平台，旨在将生态系统的所有协议和技术汇集到该平台中，实现比特币 DeFi 民主化，并提供最佳的用户体验。目前处于超早期阶段，INSF是该平台的原生 Token，将用作治理令牌，允许与平台的未来合作者进行交互等。据官方数据显示，INSF于 5 月 13 日进行公开销售，上线 15 分钟内售罄。
