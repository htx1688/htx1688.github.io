“元宇宙的开拓者” 是我们针对元宇宙的发展而设立的专栏，主要面向那些深挖元宇宙产业或者在元宇宙进行“淘金”的从业者，分享这些企业或者创业者们的故事，以独特的视角窥见那些引领全球元宇宙发展的企业或个人，我们深信元宇宙的大幕已经拉开，引领未来20年的科技互联网已经走上了时代浪潮。以下是我们的第43期内容，Futureverse如同AI狂潮下的孤勇者，毅然选择元宇宙基建之路。

元宇宙，这个曾经被寄予厚望的概念，时至今日，似乎陷入了一种“过时”的境地。在热闹时期，我们曾看到各种元宇宙项目的涌现，但这些尝试似乎并未带来让人惊艳的实际体验，人们对元宇宙的热情也逐渐冷却。

当下资本的宠儿无疑是人工智能，大量的创业公司和创新项目都将目光投向了AI，然而，仍有一些公司坚定地选择在元宇宙领域进行探索和创新，通过提供基础设施和内容，为这个“虚无缥缈”的元宇宙增砖添瓦。

最近刚斩获5400万美元A轮融资的Futureverse就是这样一家提供元宇宙基础设施服务的公司，该轮融资由风险投资公司10T Holdings领投，XRP Ledger创建者Ripple Labs也参与其中。Futureverse恰如其名，着眼未来，其坚信元宇宙蕴藏着巨大的商机和未知的可能性 ，新的资金将会进一步推进其Metaverse目标。

纵然如此，我们仍然好奇为何其会选择在这个“过时”的时刻为元宇宙行业做基建？作为一家元宇宙初创企业，它何以获得这笔大额融资？Futureverse又会带给我们一个怎样的“未来”呢？

01.包罗万象的隐形元宇宙

不过，在正式介绍公司之前，我们先了解一个概念——什么是元宇宙？

元宇宙一词首次出现在1992年斯蒂芬森的科幻小说《雪崩》中，该书主要介绍了人类通过科技实现在虚拟世界中的各种活动，而作者将人类构建的虚拟空间称之为元宇宙。

如今，随着各个领域和技术的发展，元宇宙的概念也在不断地发生变化。虽然目前还没有一个公认的权威定义，但一般来讲，元宇宙是借助人工智能、虚拟现实、云计算、数字孪生、区块链等高科技手段，把物理世界映射到由数字、互联网组成的虚拟世界 。

和元宇宙相关的项目有很多，如Meta的地平线、英伟达的Omniverse、Roblox、Otherside、The Sandbox等，但是在Futureverse的创始人亚伦•麦克唐纳(Aaron McDonald)看来，这些项目都存在这样一个弊端：互相封闭、且只局限于元宇宙游戏 。

图：Meta的地平线

麦克唐纳指出，“大多数虚拟世界公司只是制作一款游戏，但称其为虚拟世界。我们真的相信没有人可以构建‘A’元宇宙，只有‘The’元宇宙——用户拥有的可互操作的身份、数据和内容的集合，可以在整个应用程序宇宙中使用。”

在麦克唐纳看来，元宇宙并不是一个即将到来的现象，而更应该被视为当今的互联网，重点应该放在用户体验上，其次才是技术 。元宇宙必须围绕社区和公共基础设施构建，因为“没有这些，你就无法拥有数字事物的个人所有权。”

而Futureverse的主要使命正是促进构建包罗万象的体验，让用户无需关注底层区块链或NFT的使用。其目标是让技术变得隐形，用户尽情享受它所带来的无缝体验。

因此，从一开始，Futureverse就在寻找一种能够建立真正以用户为中心的社区的技术，并在区块链与人工智能的结合中找到了它。那么，Futureverse基于这样一种理念，是怎样构建其独特的元宇宙系统呢？

02.未来数字交易平台

严格来说，Futureverse是11家不同公司的联合 ，在2022年底合并了8家公司，今年7月又收购了3家公司。该公司的主要目标是将人工智能与区块链技术相结合，为用户创造一个开放、可扩展和互操作的元宇宙世界。下文将介绍Futureverse所包含的主要公司与项目，它们构成了Futureverse的元宇宙生态系统。

AI+NFT

ASM（Altered State Machine）协议是Futureverse为构建AI协作生态系统的Web3协议，是一个囊括NFT、AI和机器学习等前沿技术的创新项目，提供了世界上首个开放的人工智能基础设施平台。ASM的愿景是宏伟的，即创造一个人人都能创建人工智能且实现价值回流的世界 。

简单来说，你可以通过该协议训练、创造、售卖独特的AI，目前主要用于驱动元宇宙或游戏世界的角色和内容。

Non-Fungible Labs同样是被合并的项目，前身为通过投资支持传统艺术家和创意社区改善数字世界的基金会，现提供了一个3D动态对象转换器NFT Asset Pipeline，可以帮助开发者快速轻松地创建和发布NFT资产。

Futureverse现旗下有众多NFT项目汇集于FLUF World。FLUF World是基于以太坊区块链的NFT项目，使用ERC-721标准来创建和交易NFT，最著名的系列莫过于Flufs。

每一个Fluf都是一个以编程方式生成的3D兔子，共有10000个头像，舞蹈、表情、场景、配乐都不相同，不仅可以收藏，命名并自定义背景故事，还能作为虚拟世界的化身。

当你购买一个Fluf后，你就能拥有完全的知识所有权，可以基于一系列2D与3D的数字文件创建自己的创意王国 。

Futureverse还与市面上的热门NFT项目展开了合作，比如Cool Cats，将帮助他们利用AI技术实现个性化资产与将2D形式转变为3D等。

Game

在AI技术的铺路下，Futureverse集成了游戏工作室Altered Phoenix与动画工作室Shadows Interactive，在去年与FIFA（国际足联）合作推出4V4女足战队经理手游《AI League》，挑选的队员行为由人工智能驱动，角色还能直接送往NFT市场售卖。

Futureverse还与著名拳击运动员穆罕默德·阿里知识产权方Muhammad Ali Enterprises共同开发了一款Web3游戏《Muhammad Ali — The Next Legends》，玩家可以训练自己的AI拳手，通过比赛朝传奇冠军进攻。

在Futureverse看来，AI大脑赋予了游戏里的拳手生命 ，会影响他们的战斗力和人格魅力。每个AI大脑都是独一无二的，与拳手组合能产生不同的基本技能，玩家可以在健身房里加强训练，增强技能。

区块链

Futureverse的组成大部分实际是与区块链相关的公司，比如多链支付项目Immersve、数字支付系统Centrapay、去中心化通信Sylo与去中心化数字身份协议Centrapass。The Root Network是Futureverse的核心区块链基础设施 ，它是一个高性能、低成本、可扩展和安全的公链，能够在开放的元宇宙中实现无缝用户体验和资产互操作性。

The Root Network可以运行EVM（以太坊虚拟机），支持执行Solidity 智能合约，使得任何以前为以太坊编写的代码能够在The Root Network 上实际运行。

而FuturePass则是Futureverse推出的智能钱包SDK，使用户能够简单、快速、安全且无需密码地体验Web3应用程序和游戏。Futureverse正是在基础设施和应用层面提供了一系列工具，不仅充当了元宇宙的金融通道，还丰富了用户的数字内容资产。

Futureverse在其白皮书中对未来构建内容也提出了规划，将从故事——角色——空间——游戏实现社区共同叙事 ，并与世界领先人气IP展开合作，并建立相应生态基金。在金融层面，Futureverse将进一步优化数字资产管理与支付系统，并构建全新的数字交易平台。

03.元宇宙的未来何去何从？

元宇宙的Future在何处？元宇宙被认为是人类社会的下一个阶段，是一个充满了无限可能性的新领域。然而，随着科技的不断发展，这个领域也面临着不少的挑战。元宇宙到底是一场短暂的狂欢，还是一场长久的变革？未来又将何去何从？

风头过后的失望与放弃

就在ChatGPT风光无限的时候，元宇宙却似乎渐渐失去了热度。

科技巨头们总是“喜新厌旧”，元宇宙的热度似乎被ChatGPT这个人工智能聊天机器人给“抢走”了。不少企业开始放弃元宇宙，转而进军人工智能领域。元宇宙真的过时了吗？

元宇宙其实是一个涵盖了多种技术和平台的生态系统，而ChatGPT只是其中的一个组成部分 。科技巨头们增加对人工智能的关注，并不意味着他们对元宇宙失去了兴趣或者信心，反倒是代表着他们在探索元宇宙中更多的可能性。

如此看来，ChatGPT之类的人工智能工具似乎更像是我们进入元宇宙的一把钥匙。

Meta对元宇宙的坚持

如果要问哪家公司对元宇宙最为坚定和投入，那么答案非Meta莫属。

Meta原名Facebook，是全球最大的社交媒体平台之一。Meta创始人扎克伯格表示，希望用五年左右的时间将Meta打造为一家元宇宙公司，而其更名的举动更是证实了Meta将All in元宇宙的野心。

根据Meta最新第二季度的财报，其负责虚拟现实和虚拟现实业务的RealityLabs部门本季度巨亏达到37.3亿美元，高于预期的36.8亿美元，也高于去年同期的28亿美元。谈及这一部分亏损时，扎克伯格坦言，“我们需要看的是长远的前景。”

他认为，新的平台在未来几十年将以一种非常深刻的方式开启新的用户体验，从商业角度来看，尽管近期现实实验室表现不佳，但公司正在对未来的基础设施做出相应尝试。Meta仍会重点布局元宇宙，尽管短期内可能会让投资者感到不安 ，但“这是一个长期的赌注”。

Meta对元宇宙的坚持，既体现了其对未来的探索和创造，也展示了其对用户的关注和责任，而我们将在未来和Meta一起见证元宇宙的奇迹。

元宇宙基础技术悄无声息融入生活

也许你还觉得元宇宙的世界离我们很遥远，实际上，元宇宙中的基础技术已经悄无声息地融入进了我们的日常生活中 。

例如，在体育领域，成都大运会首次采用VR直播技术，让观众可以沉浸式观看开幕式；在文化领域，北京故宫博物院推出了AR导览系统，让游客可以通过手机扫描展品，观看展品的3D模型和相关信息；在教育领域，AI技术被广泛应用于智能辅导、智能评测、智能推荐等方面，为学习者提供个性化和高效的学习服务。

当然，这些只是元宇宙基础技术应用案例的冰山一角，随着技术的不断进步和普及，我们可以预见，在不久的将来，元宇宙还将会给我们带来更多惊喜。

04.走下神坛的元宇宙

人人都说，2023年是AI元年，元宇宙早已过气。但Futureverse等公司对构建元宇宙的坚持与努力，以及资本的认可，足以证明元宇宙概念的炒作风声过了，但实际运作的风口还未真正达到巅峰。

Gartner的报告指出，到2026年，不少于25%的民众将沉浸在元宇宙的活动中，包括购物、工作、学习、娱乐等。而Forrester同样预测，超过一半的互联网用户将是元宇宙的早期使用者。

Futureverse的创始人曾言，元宇宙绝不是遥不可及的概念，也不是专业者卖弄知识的词汇，而是随着科技发展，必然到来的世界形态 。

元宇宙，是一个充满无限可能性和创造力的世界。Futureverse正是要打造这样一个世界，并邀请我们一起加入。短视频刚出生时同样不被看好，成千上万的公司前仆后继，最后成就了抖音，引领了短视频时代，而Futureverse能否开创这一切，我们拭目以待。
