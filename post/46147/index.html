初学者 NFT 创作者和收藏者通常通过简单的 Google 搜索开始他们的 NFT 旅程。

但是寻找最好的 NFT 市场——这在很大程度上取决于独特的数字收藏品需求——现在比以往任何时候都更需要正确的背景和尽职调查。 许多通过 FTX 抢购 NFT 的持有者在这个交易所破产后猛然惊醒。 他们对数字艺术的精心和创造性的表达现在，至少在目前，比一个 404 错误值不了多少。

没有任何指南可以完全消除投资 NFT 或任何其他加密资产的缺点，但本指南旨在分析每个交易市场如何以及为什么能够或不能满足您自己的 NFT 需求。

NFT 拍卖模式

不同的 NFT 市场对管理其交易的拍卖有不同的标准。

荷兰式拍卖， 例如，使用递减的价格形式——与更熟悉的英式模式不同。 卖方以相对较高的要价开始，然后逐渐降低要价，直到投标人决定锁定它。第一个接受当前价格的投标人获胜。 荷兰式拍卖旨在快速出售具有足够需求的 NFT。

英式 NFT 拍卖 采用 eBay 模型的加密版本。 卖家设置一个较低的底价，竞标者通过提出越来越高的报价来竞争，直到拍卖在预定时间结束——最高价者获胜时。 英式拍卖会帮助鲜为人知的艺术家以公平的价格出售 NFT。 但卖家也可以使用该格式匿名投标和操纵价格。

OpenSea 

OpenSea 于 2017 年 12 月推出，成为第一个基于以太坊的 NFT 交易平台。 随着行业的发展，该市场的知名度不断提高，现在处理了所有 NFT 交易量的 50% 以上——这个成绩对这个拥有超过一百万注册用户的最大市场来说还是不错的。

但是，由于 OpenSea 是最受欢迎的，卖家被迫相互竞争以使买家注意到他们的 NFT。 这种设置使新卖家很难脱颖而出，并可能导致销售额下降。

OpenSea 为创作者提供了一套广泛的工具来铸造和列出新的收藏品。 它还带有用于浏览和购买 NFT 的简约界面——使用一系列支付选项。 从该平台以以太坊为根基出发，OpenSea 最近扩展到支持另外五个链上的 NFT，特别是 Solana 和 Polygon。

Magic Eden

尽管 Magic Eden 于 2021 年 9 月推出，但它已经是 Solana 网络上最受欢迎的 NFT 市场。 它在 Solana NFT 市场中拥有最多的用户，并且在交易量方面是第二大 NFT 平台，仅次于 OpenSea。

Magic Eden 尤其受到寻找新鲜、崭露头角的 NFT 作品的收藏家的追捧。 Magic Eden Launchpad 定期举办 NFT 铸造活动，将创作者与收藏家联系起来。 虽然该平台最初是 Solana 原生的，但它已经扩展到支持基于以太坊的 NFT——并且还有其他网络正在筹备中。

Rarible

Rarible 于 2020 年初推出，并在当年晚些时候 NFT 行业的采用率增加时占据了整体市场份额的很大一部分。 Rarible 是为数不多的拥有原生代币的 NFT 市场之一，在平台推出大约一年后发布了其 RARI 代币。 该代币允许用户影响平台升级并访问某些独特的功能。

RARI 代币持有者可以提交有关交易费用、新功能和相关治理提案的提案或对其进行投票。 此类投票是建议性的，不具有约束力，因此 Rarible 团队仍然拥有最终决定权。

代币持有者还决定展示哪些 NFT。 排在首位的 NFT 比埋在一长串列表底部的 NFT 更有可能吸引销售。

Rarible 对创作者有着独特的吸引力。 它为买家提供了一个筛选器，以探索顶级影响者和艺术家。 此功能允许创作者与各自的收藏家社区建立更紧密的关系。 该平台还提供了一个聚合器，允许用户购买竞争对手列出的 NFT。

Nifty Gateway

Nifty Gateway 在影响力者、名人和品牌创建限量版商品的 NFT 市场列表中名列前茅。 自 2018 年推出以来，Nifty Gateway 为 Carl Cox、Kenny Scharf、Lil Yatchy 和 Trevor Jones 等名人和艺术家举办了独家发售活动。

Nifty Gateway 以 660 万美元的价格举办了一场著名的网络艺术家 Beeple 作品拍卖会。 值得一提：Nifty Gateway 与加密货币交易所 Gemini 密切相关。

币安 NFT

全球最大的加密货币交易所币安于 2021 年 6 月推出了 NFT 业务。用户需要在币安注册并验证身份才能访问该平台。 创作者和收藏者使用三种主要的加密货币进行交易：BNB、BUSD 和 ETH。

用户可以购买单个物品或包含各种精选收藏的神秘盒子。 还有一个“高级”和“标准”收藏清单。 高级收藏是受欢迎的影响者的独家掉落，而标准收藏则包含由普通用户铸造的物品。

自推出以来，币安 NFT 市场已经为知名名人和实体组织了 NFT 空投活动，包括 Russell Simmons、Paulo Del Valle、乌干达国家博物馆、Franck Muller 和 Parimatch。 币安 NFT 市场上新手的最高销售额为 280 万美元。

Coinbase NFT

Coinbase 是最近涉足 NFT 行业的另一个中心化加密货币交易所。 2022 年 5 月，该交易所开始提供一个可以购买 NFT 艺术品的场所，您可以使用以太币 (ETH) 作为主要接受的加密货币。 值得注意的是，用户需要一个自我保管的钱包——Coinbase 钱包——才能访问市场。

Coinbase NFT 平台托管来自流行艺术家和新的加密原生 NFT 项目的独家发行。 它还列出了一系列领先的 NFT 集合，包括 CryptoPunks、Bored Apes 和 Otherdeed metaverse 地块。

SuperRare

SuperRare 于 2018 年推出，将自己定位为以太坊上的数字艺术市场。 该平台为自己创造了一个利基市场，被广泛认为是从经过验证的创作者那里购买独家艺术品的最佳 NFT 市场。

SuperRare 也有一个原生资产：它的 RARE 代币。 RARE 持有者组成 RareDAO，并有机会策划市场、定义上市参数和影响平台升级。 自 2018 年推出以来，SuperRare 上的艺术家收入超过 7.6 亿美元，同期二级销售额超过 1 亿美元。

MakersPlace

MakersPlace 于 2018 年推出，主要提供一个可以购买 NFT 艺术品的平台。收藏家可以找到 David Loblaw、Mr. Longshanks、FidJay 等著名艺术家的独家艺术品。自推出以来，该平台吸引了超过 100, 000 名收藏家，仅 2021 年的交易额就达 1 亿美元。

MakersPlace NFT 交易平台非常易于浏览，即使对于没有加密经验的用户也是如此。高级搜索过滤器和对法定支付的支持也使该平台脱颖而出，成为最好的 NFT 交易平台之一。

选择 NFT 交易平台后的后续步骤

1.评估你想买什么 NFT

市场上有数百万个 NFT 集合。然而，最好的 NFT 投资通常是那些拥有强大社区或对你个人有吸引力的艺术品。它也可能是即将到来的艺术家的作品，随着艺术家的知名度上升，其价值可能会增加。根据经验，仅仅因为有影响力的人推销而购买 NFT 可能是一项糟糕的投资。

2.买它

下定决心后，下一步就是购买。购买 NFT 的步骤因平台而异。它通常涉及设置 Web3 钱包，或在 NFT 市场网站上注册并完成他们的 KYC 流程。接下来，用购买金额为您的钱包充值，或者如果平台支持，则使用您的信用卡进行购买。

3.将你的 NFT 存储在钱包中

与加密货币一样，您可以将 NFT 存储在自托管钱包中。虽然使用移动或浏览器原生钱包进行 NFT 存储是常见的做法，但硬件钱包提供了最安全的体验。这些设备通常会离线存储您的资产，并消除黑客的常见攻击媒介。理想情况下，硬件钱包是投资者将巨额资金投入 NFT 市场的必选项。

4.稍后出售 NFT 以获利

虽然大多数 NFT 具有艺术吸引力，但它们也是投资资产。如果您倾向于出售 NFT，我们建议您阅读有关如何在您选择的 NFT 交易平台上出售 NFT 的指南。这样一来，您就可以避免因错误价格或其他潜在事故列出 NFT 所带来的潜在损失。它还允许您使用最佳提款方式来节省费用并获得每次销售的全部价值。

整个 NFT 市场

本文提供了详尽的 NFT 市场列表，回顾了诸如功能、优缺点以及费用之类的内容。OpenSea 和 Magic Eden 最受欢迎，收藏品范围广，支持多个区块链。

Rarible、SuperRare、MakersPlace 和 Nifty Gateway 专注于艺术作品以及来自影响者和流行艺术家的独家发行。与此同时，Binance 和 Coinbase 的 NFT 平台也越来越突出——为投资者提供中心化交易场所。

最好的 NFT 市场最终是满足您作为创作者或收藏家的个人资料和特定需求的市场。每个平台都有自己的优缺点，可以在您的数字收藏之旅的不同阶段派上用场。
