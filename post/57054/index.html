Base 区块链是什么?

Base 是一个建立在以太坊区块链之上的，新型 Layer 2 区块链。它由 Coinbase 创建，Coinbase 是世界上最大的加密货币交易所之一。

Layer 2 是什么？

Layer 2 是建立在现有链（Layer 1）之上的独立区块链。例如，比特币和以太坊都被称为 Layer 1。Layer 1 因其安全性和去中心化而非常出色，但有时会因大量活跃用户的涌入而扩展能力不足（由于 gas 费用过高或交易等待时间长）。

Layer 2 利用了 Layer 1 区块链的安全性，同时提高了交易速度并降低 gas 费用。

Base 是 Coinbase 在以太坊之上构建的 Layer 2 区块链。Base 不是唯一的 Layer 2区块链，还有很多其他的，比如：Optimism 和 Arbitrum 。

Base 基于一组 Optimism 团队支持的开源软件“OP Stack”构建。

已经那么多 Layer 2，还需要 Base 吗?

原因是 Coinbase 需要。

Coinbase 于 2021 年底/2022 年初发布了自己的 NFT 交易市场，当时 NFT 领域的人感到无比兴奋，因为 Coinbase 交易所拥有超过 1 亿用户。当时人们天真地认为/希望 Coinbase NFT 交易市场的推出，会引来数百万人购买 NFT，但实际这种情况并未发生。

Coinbase 吸引 1 亿用户的方式，主要是将用户资产托管在 Coinbase 交易所内（即实际上不在区块链上）。Coinbase 的整个商业模式就是让用户使用他们的平台，将资金保留在他们的平台上，在他们的平台上进行交易，并从中赚取手续费。

现在他们在扩大用户规模，并鼓励用户自己管理资金，离开 Coinbase 交易所进入 Coinbase 的非托管钱包“Coinbase Wallet”。这是一家具有长期愿景的公司，相信加密货币的基本原理，并致力于在未来几年内实现去中心化的 Base 区块链。

所有这些都体现在他们对 Base 的愿景中：

“Base 的创建有一个明确的愿景：将下一批百万建设者和十亿用户带入链上 ”。

Coinbase 的 NFT 交易市场可能会陷入困境，但 Base 链可能蓬勃发展。

**Onchain 是下一个 Online **

2000 年代初，互联网第一次改变了世界。开发人员第一次能够构建将人们联系在一起的应用程序，无论他们身在何处，将他们聚集在一起读取、写入和共享信息。这解锁了令人难以置信的新产品，包括：流媒体、搜索、社交网络、智能手机等。

我们正处于下一个大规模转型的边缘：Onchain ！

Onchain 是下一代互联网，它将通过将所有权重新交还到普通人手中，来增加全球经济的自由度。

链上用户正在不断增长，全球已有超过 2.3 亿个独立的以太坊地址，以太坊 Layer 2 的每日交易量，已超过以太坊主网的交易量 。据 Electric Capital 称，这种扩张同样延伸到了开发者群体中，到 2030 年，Onchain 预计将在全球范围内创造 100 万个新的软件工程工作岗位。

Onchain 用例正在扩展，包括用于收藏品的 Zora、用于餐厅忠诚度计划的 Blackbird 、用于社交媒体的 Farcaster 、用于游戏的 Parallel，以及用于消息传递的XMTP。

写在最后

我非常看好以太坊区块链，我认为区块链将为互联网的未来提供发展动力。我相信未来是多链的世界，以太坊将成为所有应用程序用例的核心 Layer 1 区块链，而比特币将成为以“价值存储”为核心的 Layer 1 区块链 。

以太坊作为“世界计算机”的最大困难之一是，它的使用成本太高了！交易费用高得离谱，当链上交易的成本在 3 至 100 美元时，世界上 99.9% 的人永远无法相信在主网上交易是合理的。

因此需要 Layer 2，这是我们需要的扩展解决方案，它们被优雅地设计为将以太坊区块链作为 Layer 1，这就像鱼肉与熊掌兼得。享受更快的交易速度和更低廉的交易费用，同时拥有以太坊的安全性。

哪些 Layer 2 会胜出？

这并不重要，也不一定会有“一个”唯一的赢家。我们将拥有许多蓬勃发展的 Layer 2，从而构建出一个繁荣的以太坊生态。

随着时间的推移，大家会使用越来越多的 Layer 2。
