术语和基础知识  

 什么是 NFT，加密世界如何实现 NFT

Sonal : 先来快速定义一下 NFT 。

Linda:  NFT 是“非同质化代币” (Non-Fungible Token) 的缩写，用来描述一种独特的数字资产，其所有权可以在区块链上被追踪到。NFT 对资产的界定非常广泛，从数字商品（如虚拟土地和艺术品）到实物资产（如房地产或衣物）。能看到 NFT 艺术真正起飞是令人兴奋的，它涵盖了许多不同行业。

Jesse: 我更关注数字资产的部分，NFT 是一种让数字文件变得可拥有的方式 ，不同于金融资产，你现在可以在互联网上拥有数字媒体资产，比如 JPEG 和 MP3。当你创建 NFT 时，有点像是把文件上传到区块链上，任何人都可以追踪到它的出处和归属。

Linda: Dixon 在最近的一篇博客中写到，“NFT是基于区块链的记录，代表了一段唯一性的媒体”。或者用 Jesse 的话，一个文件。需要关注的是“同质化” (Fungible)，这意味着该物品是能够替换、具有统一性的。比如，1 美元是同质化的，只要我有 1 美元，这个 1 美元是什么样的就无关紧要了。同样地，1 比特币也可以说是同质化的。在这样的定义下，“非同质化”意味着它具有独特性。 在这之前，我们先聊一聊加密世界如何实现 NFT。在现实世界里，我们会引入带有公证职能的媒介，例如银行、证书等对进行信息的追踪或确权。如何在没有第三方的情况下追踪数字文件成为了关键。

Jesse: 没错，通常我们要依赖银行来维护账本，或是去政府的财产登记处办理财产所有权认证。我们总是依赖第三方来跟踪所有权的归属。比特币改变了游戏规则，实现了一个公共的、去中心化的账本，没有任何一方可以控制。持有者们都能够使用密码学验证他们的所有权，不必再依赖第三方。

Linda: 中介追踪人们不同资产的所有权并收取服务费，并且阻止我们使用这些平台。而加密世界真正强大的地方，在于你拥有所有可以相互连通的开放协议。当你拥有 NFT 时，你可以将它们放入去中心化系统，并能与世界上任何人即时交易这些 NFT。你还可以把 NFT 作为抵押品，好比从价值不菲的电子游戏物品中借出贷款。任何人不需要经过许可，就可以在区块链上拥有、发行和交易 NFT。

Sonal:  NFT 的另一个特性是能够追踪出处，它本质上有一个内置的二级市场，不仅可以跟踪出处，而且可以实际跟踪内置二级市场产生的经济利益，在数字艺术作品等情况下尤其如此。

Jesse: 二次转售通过代码构建，每当 NFT 易手时，转售价值的一部分就会返还给最初的创作者。

Sonal: 自动化和编程是加密世界的独特属性，智能合约可以实现合约的编程可操作性。

Jesse: 是的，加密世界可以实现完全自动化和完全透明，与音乐行业完全不透明、多层中间商分成的版税系统形成鲜明的对比。这是一个非常高效的架构，区块链和智能合约使之成为可能。区块链擅长追踪事物的历史，如果你给我发送一枚比特币，每个人都能看到你少了一个而我多了一个比特币，且那笔交易的历史会被永远记录在区块链上。NFT 也是如此，当它们被铸造时，创作者会使用他们的加密密钥进行签名，并且这个签名任何人都可以看到。这个消息与任何其他加密货币交易的构造方式相同。NFT 与所有其他交易一起存在于区块链上，且每个人都可以看到它。因此，如果 NFT 易手，比如 Linda 购买了我的 NFT，那么每个人都可以看到我将这个NFT的所有权转让给了 Linda。我们可以开始建立人们与互联网互动的丰富历史。

在今天，你在某个平台上看到的一张图片，都可以被截取、裁剪，然后粘贴到另一个平台。一旦你这样做，就打破了这张图片的历史轨迹和出处，人们对这张图的历史起源一无所知。而现在，加密市场可以帮我们打破这个困境。同时，通过信息的不断流动，价值也可以流动。

Sonal:  对艺术市场来说，如果某个艺术家的一幅 20 年前的作品如今大幅升值，只有该作品最后的所有者能够从中受益。而加密市场则通过内置的二级市场，使艺术家持续获得价值——作品每一次出售都能得到报酬，在此之前这是不可能的。

Jesse: 值得注意的是，当创造力变现的规则可以由互联网上的任何开发人员和创作者用代码表达时，这只是无数种可能性中的一种。

Linda: 所有权历史的确很重要，它是区块链上真正唯一可访问的东西，每个人都可以看到。因此，有些物品对某些人来说可能更具意义。例如，《万智牌》锦标赛冠军使用过的卡牌，或是你最喜爱的音乐家或创作者拥有过的某件艺术品。由于他们的所有权在区块链上可被追踪，在你眼中，这些物品可能会因为曾经的持有人而更具价值。当你可以将这些 NFT 放进所有不同的加密协议时，它会变得很强大。与传统系统不同，你可以自由使用这些 NFT 做任何你想做的事情，这是一个非常大的区别。

Jesse: 是的，我认为将 NFT 与传统网络进行对比很重要。在当下的社交媒体环境下，共享文件或一段媒体时，你要先将文件上传到平台，这实际上意味着你将文件的所有权“复制粘贴”到平台，或者说，你“签署”了服务条款，允许平台通过该内容获利。 至于你会不会从中获得分成，则完全取决于平台。而且，他们还可以决定如何使用这些内容，但大部分并没有什么创新，过去任何尝试创新的开发人员都吃了闭门羹。而现在，如果你将文件上传到区块链，然后将其变成 NFT，并让它与其他加密资产相似地运作，这意味着任何人都可以通过互联网在任何地方、无需许可地访问它们，也意味着任何第三方开发人员都可以在媒体消费方式上进行创新，比如观众如何看待它，人们如何与之互动或对其进行编程。所以，我们正在构建一个通用的、开放的媒体库，任何开发人员都可以用它构建下一个 Spotify、Instagram 或 Facebook。当更多的竞争出现时，消费者受益也会越多，而且可能对创作者也有好处，正如 Linda 提到的，所有这一切都可以在没有中间人分一杯羹的情况下发生。

什么是/不是NFT

如何区分炒作与真实，社交代币与创作者经济

Sonal: 快速理清一下什么是 NFT，什么不是，来帮助大家区分市场里的炒作和真实性。

Linda: 我认为世界上任何东西都可以成为 NFT，因为任何人都可以创造出独一无二并且可拥有的东西。实体商品的问题在于必须有人保管它，所以，你必须确保你能在现实世界中对它进行审计，而且，可能有多个人拥有这一个实体商品，这是一个非常广泛的范畴。现在我们有了代币解锁 的方式，需要有一定数量的代币才能访问这个系统。人们通过代币委托许可进行聊天，拥有一定数量的代币地人才能进入这个群体，因为这证明你对这个社区拥有某种所有权，这种所有权可以随着时间的推移而调整。比如，某一天可能会形成一个 DAO，代币持有者可以投票决定以持有多少代币作为进入门槛。这一点与 NFT 相关，当然，我并不认为社交代币 ( social token ) 本身就是NFT，但如果这个群体的创作者发行了个人徽章或独特道具，那么这就可能是一个 NFT 。

Sonal: 可以多讲讲社交代币吗？

Linda: 社交代币是指由个人或社区发行的代币的一个非常广泛的分类，有时候也会用个人代币 ( personal token )、社区代币 ( community token )、创作者代币 ( creator token ) 等术语，但是社交代币包含了所有。在这个领域也有很多不同的实验，我们看到人们代币化他们的时间，一个代币相当于一个小时，并且可以自由交易。我们也看到过像 R.A.C（格莱美奖获奖的唱片艺术家）这样的人将他的社交代币 token 化，持有者可以进入私人的 Discord 群组，且 R.A.C 会追溯并给他的支持者分发奖励，这是创作者可以与他们的早期支持者进行互动的一种方式。

Sonal: 社交代币和 NFT 相比有什么相似性吗？以及什么不是 NFT？

Linda: 当创作者直接向粉丝发行一些独特的艺术品时，社交代币可以成为 NFT。但在很多情况下，社交代币也可以只是同质化代币。就像 R.A.C 代币一样，它们都是同质化的，所以你只需要持有一定数量的代币，他们可以反复进行交易和回购，至于具体是哪个 R.A.C 代币并不重要。我认为社交代币和 NFT 之所以被混为一谈，是因为它促进了创作者经济，使创作者能够直接与粉丝进行互动，这两者通常是紧密联系在一起的。

Sonal: 如果是同质化的，则不能作为 NFT，相反，如果是非同质化的，那么就可以是 NFT。

Jesse: 我认为同质化和非同质化代币之间的界限是模糊的，因为两者之间的相互作用是巨大的。 你可以通过碎片化把一个非同质化代币变成同质化代币，然后你可以把这些同质化代币变成社交代币。有一种叫 B.20 的代币，代表了对一些 Beeple NFT 的所有权。投资者购买了 Beeple 创作的 NFT，并对其进行了所有权分割，碎片化成了同质化的 ERC-20 代币。在此基础上，第三方开发者可以重新混合并添加新的体验，被编程赋予各种各样的其他价值。除了拥有 Beeple 的一部分，拥有一定数量的 B.20 代币也可以作为进入一个 Discord 服务器的门槛。你可以轻松地将一个非同质化代币碎片化成同质化代币，再把它变成社交代币，这是一个值得探索的有趣的设计空间。

Sonal: 到目前为止，我们谈论的是现实世界中已经发生的事情，在加密世界中以数字化的方式被执行。而碎片化所有权实际上涉及到了只有加密世界可以做到的事情。你不能真正分割一个实体艺术品，但可以拥有这个艺术品的部分所有权；即使有数百万人在观看，但人们仍可以在他们最喜欢的电视节目中出价并拥有自己的时刻。

Linda: 这就是为什么我热爱加密世界，因为这些概念是以前从来没有过的，比如拥有自己的时刻、媒介、直播，想象一下 Youtube 的创作者直播他们生活中发生的事情，这里有太多的可能性。

常见问题

只是一张 JPEG？只是新形式的炫耀？只是新一轮的 ICO？

Sonal: 你们会怎么回应 “花了6900万美元就为了一张 JPEG ?” 这样的评论？Jesse，你说这其实是一份文件。但在互联网上，文件几乎毫无价值，而且可以无限复制。

Jesse ：我会说一个文件在互联网上被复制的次数，与 NFT 直接相关。换句话说，这个文件越多地在网上传播，说明它的社交价值越强。比如《蒙娜丽莎》，它可能被复刻了无数次，出现在衣服上、卢浮宫出售的明信片上，甚至可以在互联网上的任何地方看到它，但只有一幅《蒙娜丽莎》在卢浮宫博物馆里，其所有权是价值连城的。

Sonal: 你所说的东西让我想到了互联网变得更多地关注丰富而不是稀缺 ——互联网上有无限的选择。比如 Netflix 会做狂欢季和减价出售这样的活动，这在某种程度上创造了数字稀缺性，就像是限量版效应。这是对不一定稀有或稀缺的东西之间的有趣关系的另一种类比。事实上，即使是在文件和 JPEG 可以被无限复制的情况下，你也可以强制执行这种数字的所有权，甚至只是其中的一部分（如果你想要碎片化所有权），这是非常独特的。

Jesse: 你不是想成为唯一能访问指定 JPEG 或媒体片段的人，而是想拥有其他人都能看到的媒体片段。你知道，表情包在互联网上的传播速度很快，它们被无限地分享。我认为很快我们就会看到这些所有者在分布式的社交平台上得到认可。

Linda: 我们可以看看一早期 NFT 项目 CryptoPunks 的玩法。现在有两个 CryptoPunks 可以卖到单价 750 万美元，其中一个卖家是 Dylan Field，Figma 的 CEO 和 co-founder，有趣的是任何人都可以复制这张图。但是当你看看 CryptoPunks NFT 本身，你就可以知道是谁拥有它。有人这么早就拥有了它，这个事实已经成为了一种身份象征，人们可以以此证明他们早早地进入了这个领域。因此，所有权历史很重要，它能够在很早的时候发现一个艺术家或创作者，成为第一批支持者之一，并在区块链上进行追踪。未来当它被出售时，你可以证明你是早期入圈的 OG，这是非常有价值的。

Sonal: 现代艺术博物馆或一些著名博物馆的艺术品旁边会放一个标语牌，它会描述这件艺术品，包括材质、日期、艺术家，然后在最下面写着谁拥有它，或者它是由某个收藏家借出的。并不是所有人都能注意到这一点，大多数人只关心艺术品本身，但也有人关心作品的所有权。现在，你们正在互联网上做同样的事情，全世界都能看到，而不需要去现代艺术博物馆去看。

Jesse: 是的，开发者也会朝这个方向靠近。所有的资料，包括被谁拥有、从哪里来、所有权历史等，都只是一个 API 接口的调用。现在，开发人员很难在 Web2 平台上找到所有这些信息，而在 Web3，将这些信息呈现出来会容易100倍。你在博物馆里看到的那个小告示，在不久的将来会在所有的社交平台上以数字化的形式出现。

Sonal: 你们如何评价 Matt Levine 在推特上说的“NFT 是一种可交易的炫耀的新形式，而不是一种可交易的所有权的新形式。“

Jesse: 炫耀是人们收集 NFT 的潜在原因之一，但这不仅仅是投机价值，或炫耀自己是所有者。随着时间的推移，我认为NFT的所有者将开始意识到越来越多的复合效用，因为开发者为你的数字资产建造了新的空间。今天你可以购买一件数字艺术作品作为 NFT，把它带到虚拟世界（如Cryptovoxels 或 Decentraland） 里展示，这只是一个非常早期的第三方开发者的例子。与该 NFT 的创作者毫无关系的第三方开发者们能够构建一个全新的虚拟体验，而任何第三方开发者又可以在开源的、无需许可的 Cryptovoxels 上构建…你将看到这种”乐高积木式“的方法不断构建起新的体验，开发者可以用更少的钱建造更多的东西，更快地结合创新。因此，这种说法低估了作为所有者可能的实用本质。

Linda: 我不太同意 Matt Levine 的说法，它忽略了很多有用的用例。如果我们谈论的是更传统的东西，比如代表门票、金融资产或房地产的 NFT，这会是更有效的转移方式，而且不需要那么多的文书工作和中间人参与。这是对社会的净收益，而不是那些试图展示财富的人。

Sonal: 你如何看待人们把 NFT 现状与 ICO（首次代币发行，Initial Coin Offering）热潮进行比较？Nick Tomaino 认为 NFT 是一个具体的产品，是一种数字商品，而不是关于未来的承诺，我认为这是一个很好的论点。

Linda: 这让我想起了 2017 年，当时很多人进入了这个领域，ICO 这个词被不断地抛出，其中也有很多炒作宣传。但是，通过这个过程，很多令人难以置信的项目从加密货币中诞生了，而且很多加入这个领域的人最终都留下了，因为他们看到，这不仅仅能快速致富，而且社区开始发展，并且产生了真正独特的创造价值的方式。当然，有很多人进入这个领域只是想赚快钱，但也有很多真正有创造力的人加入了这个领域并继续留在这里，对正在发生的事情进行试验，构建一些我们从未见过的东西。很高兴在 NFT 的浪潮中，是那些辛勤工作的艺术家们得到了回报，这是人们所重视的东西。

Sonal: 我们该如何评估 NFT 是否有效?

Jesse: 你想问的应该哪些 NFT 是有价值的吧。这个问题有点像“什么是艺术”？任何旁观者认为是艺术的，就是艺术。同样，市场认为有价值的，就是有价值的NFT。这就是为什么你会从博客、数字艺术中看到大量的NFT形式。出于各种各样的原因，有很多小众群体想要拥有一件与他们的文化相关的物品，可能是为了在将来转售它，也可能只是想让他们的名字出现在这个 NFT 的告示上，告诉你“我支持这个创作者”。人们会因为各种不同的原因而重视 NFT，构成这个市场的也有许多不同的亚文化和价值体系，这也是它向各个方向发展的原因。

Sonal: 从 CryptoKitties 开始，NFT 早期经常出现扩容问题，而以太坊还没有准备好来应对此类问题。因此，NFT 的热潮实际上带来了良好的基础设施改善，推动了许多解决方案考虑扩容问题。在你们创立 Mediachain 之前，创作者们就一直存在追踪数字资产库的难题。

Jesse: 许多关于 NFT 的想法都是我们当时正在探索的。当时的生态系统缺少两个关键的东西，一个是轻松创建代币的能力，这是由智能合约和智能合约平台（如以太坊和 Flow）独立实现的，另一个则是这些数字资产的市场，数字资产具有价值是 NFT 等数字媒体资产具有价值的先决条件。根据 Linda 之前的观点，2017 年的市场是如今 NFT 市场的先决条件。因此，市场必须是第一位的，是先驱者。市场是不稳定的，它们驱动着投机的狂热，但也驱动着基础设施，以及坚持下来的思维模式。

NFT如何运作   

铸造与钱包，NFT 的运作原理

Sonal： 让我们快速讨论一下 NFT 是如何运作的？

Linda: 第一步是把你的作品放上区块链。铸造涉及到与智能合约的交互与提交，有不同的市场可以让你很容易做到这一点。你可以选择不同的属性，比如 NFT 的名字，二次销售的版税（如有）以及你收取的分成。最困难的部分实际上是创建钱包，进入加密货币和 NFT 的市场，有些市场可能会让你经历一段比较漫长的过程，他们可能会对艺术家进行一些尽职调查以确保这是真正的艺术品，而不是别的艺术家的复制品，并确保它是真正高质量的。还有一些市场则是任何人都可以铸造。

Sonal: 能不能解释一下钱包的部分。

Jesse: 加密钱包的概念可以归结为所谓的公私钥配对 。你在区块链上有一个公共地址，关联了你的资产和物品，比如，在以太坊上，你会有一个公共地址，你在以太坊区块链上的所有东西都与公钥相关联。而私钥是用来解锁你钱包中的资产转移的，你需要私钥来解锁你的公共地址的东西，这就是加密钱包的概念。MetaMask 是以太坊最流行的钱包，它是一个浏览器插件，你可以在任何主流浏览器上安装它。本质上，它所做的是为以太坊设置一个公共地址区块链和一个私有地址。需要注意的是加密密钥配对，它不属于MetaMask，而属于你自己。 MetaMask 看不到密钥对，他们没有任何信息，这个信息是你的。这就带来了很多责任，如果你丢失了你的私钥，你就失去了所有的东西。这就是为什么它被称为“钱包”，你的加密钱包就像一个实体钱包，如果你丢失了它，你所有的现金都没了。

Sonal: 我们现在了解了钱包是如何工作的，一些平台可以让你铸造  NFT，你可以以任何你喜欢的形式、对象、数字资产或文件创建艺术品。你可以给它命名，指定项目条款，设定版税的比例，不同的系统可能有不同的选择。我见过一个非常有创造性的设计，沿着结合曲线走得越远，作品就会逐渐显露出来，这个设计的创造性在于把静态的艺术作品变成了交互性的。那么，铸造之后会发生什么?

Linda: 你可以做不同的事情，取决于你想用你的 NFT 做些什么。一旦你在这些市场上进行了铸造，你可以把它设定好价格上架，也可以和其他人分享这个链接，让他们对这件艺术品出价，然后接受不同的报价。或者，你也可以只为自己创建一个 NFT，比如在虚拟土地上创造一个艺术画廊，在其中展示你的艺术作品。我见过不同的艺术家之间互相交换 NFT 作品，也见过有人把艺术品作为抵押品，以此获得贷款。因此，你可以对它做任何你想做的事。但最常见、最基本的情况是，人们出售他们的艺术品，然后有人购买，之后把它们存储在自己的虚拟土地上或在个人资料中显示。这里面有社交元素：人们经常谈论它，比如，他们会在推特上谈论自己刚买了这件艺术品。有一个应用程序可以把所有这些不同的市场上的购买汇总，并像 Instagram 一样显示出来，这样你就可以知道谁在购买什么，谁拥有什么，并让人们围绕它形成社区。

生态系统参与者

市场、媒体、赞助人与 DAO，全民引领的文艺复兴

Sonal: NFT 的生态中有许多现在已经出现并将不断出现的参与者，我们已经提到了几个，例如展示在线收藏的网站，在线画廊，策展画廊，还有市场以及其他管理细节的工具。你会怎么拆解分类呢？

Jesse: 有一些艺术策划或特定类型的收藏品的垂直市场，还有一些提供搜索框、可以搜索所有 NFT 的横向市场，比如 OpenSea (https://opensea.io/)。因为所有这些 NFT 都在区块链上，且是完全开放的，横向市场可以查询区块链并进行聚合，你可以找到几乎所有 NFT。

Jesse: 无论是垂直还是横向市场，通常都允许创作者在平台上铸造。简单来说，这里既有供给也有需求，你可以选择垂直形式，即专业化形式，或者是横向的、包含所有东西的形式。市场的需求方出现的另一个非常有趣的现象是，你开始看到非常有趣的收藏者。加密货币发送价值就像发送电子邮件一样简单，因此，人们以有趣的方式汇集价值以便参与这个市场。有一个非常酷的实验叫做 Flamingo DAO，其核心思想是你可以与任何拥有加密钱包的人共享资源，把钱存入这个智能合约，然后这个智能合约就可以购买 NFT，这个组织就像是一个获取 NFT 作品的基金或者”画廊“。而通过出现在收藏品中，创作者获得了最初汇集资源的受众群，这是另一个有趣的现象，也是加密货币的独特现象。

Jesse： 我提到了垂直和横向市场，也有邻近的媒体平台，就像我们在接触到的 Denis 的项目 Mirror (https://mirror.xyz/)那样，这是一个博客平台，任何人都可以在那里发布他们的博客文章作为NFT。问题是，为什么你想要创造一篇博客文章作为 NFT？举个例子来说，如果你是一名独立的调查记者，现在并没有很多有效的盈利工具，订阅可能不利于长篇新闻。类似于 Flamingo DAO 集资的想法，Mirror 允许读者以众筹的形式聚集资源。作为众筹的参与者，你不仅成为创作者或写手的赞助人，还是他们发布博客时产生的 NFT 的部分持有者。你可以分析这些赞助人的心理，但我认为这可以归结为两件事，一是赞助的概念，你是创作者的赞助人，帮助他们完成工作。但也有一个可能，在未来如果这个作品变得非常有价值，你就会在这个作品的“资本表”上，就像 Elon Musk 发表的关于特斯拉秘密总体规划的文章，或是 Twitter 创始人 Jack 以数百万美元的价格出售的第一条 NFT 推文。你可以看到这个想法走得更远，一种新的、伟大的想法以博客文章的形式进入世界，人们众筹这些伟大的想法，他们希望看到这些想法在世界上发生，并成为其部分持有者。

Sonal: Dylan Field 发布了一个很棒的 Twitter thread，讨论了一些关于 NFT 的想法的延伸，他称之为“粉丝的证明” (Proof of Fandom)，我们在加密世界里有个行话叫“权益证明” (POS)，“工作证明” (POW)。在这种情况下，谈论的是创作者获得粉丝的方式。我和 Kevin Kelly 之前的播客(https://a16z.com/2020/01/27/1000-true-fans-attention-economy-inevitable-kevin-kelly/)聊过付费模式的扭转，不是由创作者销售，而是买家和粉丝将他们的注意力货币化。Dylan 在那个例子中指出“社区是艺术”，我认为非常有趣。

Linda： 这是一个我非常期待看到的领域，还没有太多人在做这个工作，但是，让这么多 DAO 成员能够投票决定这个艺术品的外观，并让它成为集体艺术品的想法已经存在。我曾经在 Saint Fame DAO 里，我们会对服装设计的不同参数进行投票，然后这个 DAO 会生产它们，并将它们运送给购买它们的人。所以，由一群人决定设计的样子，且任何人都可以加入这些 DAO，共同创造或投资，这对我来说非常令人兴奋。

Jesse： 沿着“粉丝证明”的思路，就是我一直称之为的“赞助+”的想法。在 Web2 中，通过在 Substack 上订阅或在 Patreon 上付费，你很容易成为你欣赏的艺术家或创作者的赞助人。这样做的本质是让你能够接触到他们的作品，但同时也允许你在经济上支持他们的作品。NFT 允许你做同样的事情：在某些情况下，NFT 可以让你访问一个 Discord 或一份通讯录。但赞助+的附加部分是数字所有权所带来的全新且独特的功能，即随着创作者的知名度提高或对他们作品的需求增加，存在未来通过将所有权转售给其他人而获利的可能性。而且，我认为“+”是非常关键的，它是一开始就成为赞助人的强烈动机。所以，“赞助+”最终可能会使市场比我们在 Web2 中看到的赞助更大。

Sonal: 就像意大利文艺复兴时代的美第奇家族等一样，人们认为，最初是赞助开启了艺术的黄金时代。你所描述的是一种更民主的赞助形式，而“+”是一种随着时间的推移真正产生连锁效应的方式，是真正的投资和民主化，以一种人人都能获得的方式，不仅仅是富有的“文艺复兴家族”能提供资金。

Linda: 我一直在参与 Yield Guild Games DAO，他们是一个游戏玩家的 DAO：来自世界各地的玩家参与这些区块链游戏。有一款非常受欢迎的游戏叫做 Axie Infinity，你可以在游戏中看到像宝可梦一样的生物相互战斗，每一个生物都是NFT，你可以在游戏中战斗并赚钱。有时候这些宝可梦生物可能由于其富有价值而过于昂贵，所以我们看到这个 DAO 中的玩家将 NFT 租给其他玩家。现在，这个 DAO 在他们玩的游戏中投资虚拟土地，因为他们自己是这些游戏的专家。他们实际上是在开发这些游戏中的土地，就像在一个物理世界中开发房地产并使其变得更好。

Sonal: 为什么这些东西必须以 DAO 的形式存在？这有什么意义？

Linda: 我不认为所有东西都要成为 DAO。很多时候，公司的存在更有意义。但是，DAO 真正有趣的地方是它有更多的透明度，由 DAO 管理的资金亦然。资金的流向和来源是完全透明的，任何人可以在任何时间点查看余额。你可以想象，在传统公司，你不能随时查看资产负债表。通常情况下，它们只是每季度或每年发布一次。所以， DAO 创造了公平的竞争环境，创造了更多的透明度。在很多情况下，进入这个行业的门槛更低。你甚至不需要暴露你的身份。要加入一个没人知道你是谁的公司真的很难。这非常符合加密的理念：全球化、开放。

能源问题

 比特币的 PoW，以太坊的 PoS，NFT 与艺术的能源问题

Sonal: 最近有许多关于铸造和能源问题的热议，这显然是人们从比特币的角度思考的产物，比特币是能源密集型的。当谈到 NFT 时，能说一说能源问题吗?

Linda： 关于这一点有很多误解。PoW（工作量证明）确实需要能源，但不是每个区块链都是 PoW。PoW 涉及到实际验证这些交易发生的物理矿工，所以它确实是能量密集型的，因为你必须证明你确实在消耗一些工作量以产生输出。至于以太坊，它正在从 PoW 转移到 PoS（权益证明），这有点类似于虚拟挖矿，不需要有物理上的扩展能源。不是说在挖矿设备上花费 1,000 美元，而是将这 1,000 美元锁定在系统中，并根据你投入的资本随机选择进行验证。我们也越来越多地向 Layer 2 移动，当涉及到验证区块链的所有权时，人们确实希望减少能源消耗，我认为今后这种讨论将会越来越少。

Jesse: 即使是 PoW 的名声也不好，它消耗了大量的能源。然而，许多从事 PoW 的矿工位于有潜在产能的地区，未开发的可再生能源，如水电，存在过剩的需求，那么它就可以来挖矿。举个例子，油田里有天然气排放，这些气体本来是要进入大气的，但却可以被燃烧以产生 PoW 并赚取比特币。我不是在为这种做法辩护。但我要指出的是，PoW 采矿的能源组合很大一部分来自可再生能源。

Sonal: 很高兴你能提出这点，这也是这次访谈的重点，提出的这些差别可能存在，也可能不存在。

Jesse: 我也认为有很多关于 NFT 能源消耗的意见没有更广泛地考虑到能源消耗的相对度量。想一下巴塞尔艺术展 ( Art Basel )，每年都有很多非常富有的收藏家乘坐私人飞机到巴塞尔艺术展收藏作品。我不知道所有这些私人飞机的排放是多少，但我预计会产生大量的二氧化碳。所以，量化一个艺术家作品的具体排放量是一个非常复杂的话题，对于那些只是使用越来越高效的新工具的创作者来说，仅仅基于这种引人注目的相对价值衡量标准而停止是不公平的。

Linda： 我还看到了 Andrew Steinwold 的一条推特，他说，这些数字艺术实际上是非常环保的，因为你不需要购买任何实物，比如画布用的棉花、画架用的木头、油，还有之后把这些艺术作品运送给别人的运输费用。任何新创造出来的东西总会有多面性，你总是可以分析出它哪里不环保。

“无许可”的创新

现实世界与加密世界鱼龙混杂，无许可意味着什么

Sonal: 如何解决对无许可的事情的担忧？如果你的密钥丢了，你甚至无法找回，因为没人帮你保管。

Jesse: 我们可以用现金来类比，现金也是无许可的。如果你丢了钱包，很可能你的现金也不见了。在现金经济中，你可以买到各种各样的商品。它们可能是非法商品，也可能是完全正常的商品，而这两种东西都要用到现金。加密货币和 NFT 也是如此，这里会有坏人，而法律系统将会介入并解决这些问题。但我认为利大于弊。和只有在 Facebook 或 Twitter 工作的开发人员可以在平台上进行实验和信息创新相比，我认为如果世界上每一个开发人员都可以免费使用一种开放的方式去进行创新，而无需征得这些大平台的同意，这将会是一种更健康的状态。

Sonal: 版权侵犯、知识产权等问题现实世界中非常普遍，并且你通常不总是具有追索权。我们讨论的是你实际上已经在这个系统的问题中找到了解决方案。

Jesse: 在某种意义上，你可以认为区块链实际上使取证工作变得容易得多，因为所有的信息都是公开的，任何人都可以获得。这一切都是为了找到正确的平衡，让坏人可以被解决，同时好人可以自由创新。

企业创新和 NFT

社区的重要性、传统玩家的新思路和员工的所有权经济

Sonal: 你们能给创业公司或行业的一些实际考量吗？可以是对消费者、创作者甚至是机构参与者的。

Linda: 进行对话、把自己投入到社区中以及在公共场合进行构建，在加密领域总是非常棒的。这个领域还处于非常早期的阶段，人们都在试图弄清楚一切。没有人是完全的专家，每个人都非常愿意讨论与合作，所以永远不要害怕提问。加入不同的社区，在 Discord、Twitter 或任何他们聊天的地方。

Sonal: 大公司、大机构银行和大的 DeFi 玩家，比如银行和传统玩家，他们不是那种会进入 Discord 并尝试新事物的人，也不是那种拥有你所描述的思维模式的人。对那群人怎么看呢？

Linda: 传统机构可以考虑 NFT 如何为自己提高效率。拥有每个人都不得不追踪的金融资产，可能是低效或昂贵的。而 NFT 将使这一过程更加顺畅。这是值得研究的，不一定非得是变成 NFT 的数字艺术，也可以仅仅是必须自己管理的独特金融资产。

Jesse: 大公司和其他公司通过各种渠道参与到创意作品的市场中，许多公司与创作者和有影响力的人在营销和发行方面进行合作。NFT 为这两件事提供了一个新渠道。成为创作者作品的所有者将是获得他们关注的另一种方式，并有可能通过市场营销的视角获得分成。

Sonal ：我觉得很有吸引力的是现代对员工的新定义，员工可以在为公司工作的同时成为创作者，对自己的想法有更多的所有权。我想知道这是否会通过公司内部的 NFT 发展，甚至延伸到公司之外。
