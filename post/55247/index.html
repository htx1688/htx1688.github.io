本周，解锁项目较多，有 9 个将迎来解锁的项目值得关注，远多于此前数周。

其中，DYDX、 TOR、HFT、NYM 将面临大额解锁，LQTY、GAL、EUL、 1 INCH 则面临着小额解锁。而 SAFE 的解锁最为特殊，目前 SAFE 代币尚未开放交易。Odaily星球日报将在下文中一一详述。

X2Y2

项目官网：https://x2y2.io/

项目推特：https://twitter.com/the_x2y2

本次解锁数量： 3750 万枚

本次解锁金额：（约） 94.4 万美元

X2Y2 是一家 NFT 交易平台。在推出之初，曾以零版税和低交易费用而闻名。本次将解锁 3750 万枚代币，价值约 94 万美元。

NFTGo 数据显示，X2Y2目前交易规模在所有 NFT 交易市场中排名第三，交易量仅低于 Blur 和 Opensea。

从解锁曲线来看，同类规模的解锁每季度出现一次。X2Y2 流通市值仅 700 余万美元，本次近百万美元规模的代币解锁，已占据 X2Y2 流通量的 12.75% ，这是一次值得引起投资者注意的解锁。

Euler

项目官网：https://www.euler.finance/

项目推特：https://twitter.com/eulerfinance

本次解锁数量： 15.4 万枚

本次解锁金额：约 40.3 万美元

Euler 是一个定制的无许可借贷协议，可帮助用户借出和借入比以往更多的基于以太坊的代币。作为一个定制的无许可借贷协议，该协议具有一系列新功能，可帮助用户借出和借入比以往更多类型的代币。在借贷方面，借贷加密资产的长尾资产仍有大量未满足的需求。

EUL 流通市值仅 5000 万美元，本次解锁仅 40 万美元，不足流通量的 1% ，无需过度担忧。

1inch

项目官网：https://1inch.io/

项目推特：https://twitter.com/1inch

本次解锁数量： 36.4 万枚

本次解锁金额：（约） 13.8 万美元

1inch 是一个去中心化交易聚合器，旨在为用户提供快速、低成本和高效的交易体验。1inch 通过聚合多个 Swap 的流动性，可根据用户的需求自动选择最佳交易路径和价格。用户可在不同的 DEX 之间实现无缝交易，并获得最佳的价格和流动性。

1INCH 是 1inch 协议的原生代币，可用于支付交易费用、获得更低的手续费。1INCH 持有者可参与 1inch 生态系统的治理，还可以用于获得额外的奖励。

按照 1INCH 代币经济学，本次解锁的代币来自团队、早期投资人、VC。目前，1INCH 总供应量的 36.97% 仍在锁定之中，

不过，本次解锁并不值得引发过多的关注。仅约 15 万美元的金额，与 3.6 亿的市值相较微不足道。

SPACE ID

项目官网：https://space.id/

项目推特：https://twitter.com/SpaceIDProtocol

本次解锁数量： 1515 万枚

本次解锁金额：（约） 402 万美元

项目介绍：

Space ID 是一个较新的项目。尽管早在 2022 年 4 月该项目就已问世，但直至今年该项目才完成代币发行。2023 年 3 月，Binance 官方公告宣布，ID 代币为其第 30 个 Launchpad 项目。

Space ID 是一款去中心化域名项目，该项目为用户提供了“.bnb”、“.eth”、“.arb”域名服务。意图将这一项目打造为用户的去中心化多链身份。此外，项目方还为用户提供了域名交易市场。

解锁概况：

ID 为 SPACE ID 的治理代币，总供应量 20 亿枚。ID 主要有三种功能。

ID 可用于质押，以此获得市场的交易费用折扣、域名注册折扣。用作 SPACE ID 生态系统和Web3名称 SDK 集成中的支付方式。参与 SPACE ID DAO 的治理。

其代币经济学如下：

20% 为种子轮销售；8% 用于战略论销售；15% 归于核心团队；7% 归属于顾问；10% 用于生态系统；13% 用于市场营销；12% 归于基金会；10% 用于社区空投；5% 用于币安 Launchpad。

具体释放曲线如下：

目前，仍有高达 86% 的 ID 代币仍在锁定之中。

本次解锁代币超 1500 万枚 ID，仅为总供应的 0.7% 。但鉴于该代币绝大多数均未流通，因此其解锁量仍较为可观。以当前流通量计算，约占流通量的 5.3% 。解锁代币市值约为 402 万美元。

从流通量来看，这一解锁规模或对 ID 代币交易形成一定冲击。但此类解锁规模在以前已多次发生，自代币发行至今，每月均存在一次规模类似的解锁。

Axie Infinity

项目官网：https://axieinfinity.com/

项目推特：https://twitter.com/AxieInfinity

本次解锁数量： 343 万枚

本次解锁金额：（约） 2151 万美元

Axie Infinity 一度是最受到行业关注的 GameFi 产品。在经历了漫长的市场周期后，AXS 代币已较历史高点大幅下降。但 AXS 仍是一个颇为庞大的项目，其流通市值仍有高达 7 亿美元。

目前，大多 AXS 代币均已释放，但仍有 25% 的代币仍在锁定之中。本次解锁的 343 万枚代币价值超 2000 万美元，占流通市值的约 3% 。

尽管解锁规模颇为庞大，但代币经济学显示，本次解锁的代币归属均为质押奖励。本次解锁后，AXS 代币质押收益或将提高，大量的代币并不会第一时间流入市场。

Cartesi

项目官网：https://cartesi.io/

项目推特：https://twitter.com/cartesiproject

本次解锁数量： 154 万枚

本次解锁金额：（约） 23.5 万美元

Cartesi 为开发者提供了 Layer 2 解决方案，可以使开发人员能够使用 Python、NumPy、Rust 等主流软件堆栈构建可扩展的去中心化应用程序(DApp)。Cartesi Rollups 解决了区块链的计算限制和去中心化应用程序缺乏成熟软件堆栈的问题，它提供了模块化的特定于应用程序的 optimistic rollups 和交互式争议的解决方案。

产品的官方介绍表示，该产品提供了第一个“区块链操作系统”。这一产品允许开发者使用 Linux 和主流软件堆栈来构建可扩展和更强大的去中心化应用程序。

从解锁规模来看，本次解锁代币的价值仅为流通市值的不足 1% ，其解锁规模较小，较难对市场引发冲击。

此外，代币机经济学显示，本次解锁的代币归属均为基金会保留。
