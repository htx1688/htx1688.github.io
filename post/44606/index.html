以太坊域名服务ENS 曾被V神认为是迄今为止最成功的非金融类以太坊应用程序，区块链域名服务项目属于具备落地场景值得关注加密项目。在加密市场，除了以太坊域名服务ENS外，其他主流公链也有相应的域名服务项目，而且多数域名服务项目还没有发行代币，早期进行注册交互，未来还有望获得空投机会。本文就主流公链域名服务项目进行盘点。

以太坊域名服务(ENS)

ENS是当今最大的区块链命名标准服务，拥有超过217万个注册名称，并集成了500多个DApp上。该平台提供分级分发域名服务，可以通过TLD 来识别域名“.eth”或“.test”。用户还可以创建名称哈希，同时保留其域的属性。同时ENS还将可读的名称转换为机器可读的名称，这使得普通用户更容易与地址进行交互，同时最小化误差。此外，ENS允许用户创建去中心化的网站，并将它们上传到星际文件系统(IPFS )，允许世界各地的计算机以点对点的方式存储和共享数据。ENS还支持大多数现有的DNS(域名系统)名称，如.com、.org、.io、.app等。项目官网：https://ens.domains/

SPACE ID

SPACE ID是BNB Chain链上全链域名服务商，用户可以绑定自己多个链的身份，让用户可以在不同的链上使用同一个域名，简化操作。社区可以通过SPACE ID的网络构建自己的顶级域名服务。目前SPACE ID域名注册数量超27万。SPACE ID官网展示了52个SPACE ID集成的应用，应用类型涉及DeFi、NFT、钱包、元宇宙、SocialFi等赛道，应用涉及多条公链。与元宇宙应用的集成让SPACE ID域名可成为用户在元宇宙中的身份名片。Dune数据显示，SPACE ID平均注册成本为0.0505 BNB，最高注册成本为5 BNB。项目官网：https://space.id/

ANS

ANS 是 Algorand 网络上的去中心化域名服务。与 ENS 类似，它允许用户为钱包地址注册 “.algo” 域名。然后，用户可以分享他们的 “.algo” 名称来代替钱包地址来接收付款和其他交互。ANS 被设计为去中心化且具有成本效益，主要由注册表和姓名记录两部分组成。推特追随者12k，自 2022 年 2 月 25 日 Algorand 域名服务主网启动以来，已经有超过 13000 次注册、100 次域名转移和 150 次域名续订。项目官网：https://algonameservice.com/

Evmos domains

Evmos domains是Evmos链上域名服务商，Evmos Domains 的目标与互联网域名服务 DNS 相似，但由于 Evmos 区块链提供的功能和限制，其架构有很大不同。Evmos domains推特追随人数70.8k，到目前为止，注册名称增长至29.9k。项目官网：https://evmos.domains/

Polkadot Name System

PNS是波卡生态上的去中心化域名协议，目前主要围绕的是8~10字符域名服务，其8字符的域名注册已于7月14日开放。据了解，PNS未来将整合多链的地址和资产，并支持推特、Github等Web2平台的身份验证以及对NFT资产的域名解析。与ENS一样，PNS也采用续费模式，对于1年期的域名而言，总费用约为25美元，其中包括20美元的注册费，以及5美元的年费。截至7月28日，PNS注册量约为10361个，前100钱包地址持有的比例约为46%，其中持仓第一的地址约占整体26%。项目官网：https://www.pns.link/

StarkNet Name Service

StarkNet Name Service 是 StarkNet 黑客松获奖项目，类似以太坊上的 ENS 。注册过程和 ENS 基本类似，在网站上直接搜索域名然后注册，推特追随人数3.1k。项目官网：https://starknames.vercel.app/

Aptos Name Service

Aptos Names Services 是 Aptos 上的域名开发商。与 ENS 、SPACEID 类似，Aptos Names 允许用户使用短字符串来表示他们的钱包地址。Aptos Names 由 Aptos 基金会成员0xChucky开发。Aptos 还公开了其与该项目的关系，并承诺支持平台上的应用程序与该域名注册商的集成。Aptos Name Service（ANS）于10月19日在Aptos主网上线，支持Aptos用户为其Aptos钱包地址使用人类可读的.apt名称。ANS开放3位及以上字符名称的注册，每个域名都将按照年租模式注册，之后将启用更长的注册期限。项目官网：https://www.aptosnames.com/

Sui Name Service（未空投）

Sui Name Service 是 Sui 生态中的去中心化数字身份服务。Sui Name Service 目前处于公开测试版阶段，由于 Sui 网络每隔几天会重置一次，因此此时注册的域名每隔一段时间就会消失。但 Sui Name Service 在推文中表示计划感谢早期采用者，推特追随人数24.5k。11月3日，Sui生态域名服务Sui Name Service（SuiNS）宣布上线Sui测试网，目前已开放注册。项目官网：https://sui-names.com/
