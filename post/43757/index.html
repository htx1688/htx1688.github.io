都说熊市就好好 buidl，牛市就能一飞冲天。这里基本上是说 NFT 项目了，市场依然还是不温不火，最几天大家讨论的热点还是 aptos 的空投。而蓝筹 NFT 却也在悄悄搞事情。首先是 BAYC 宣布了下一批的衣服赋能和 ApeCoin 的生态建设，另外就是 Azuki 的一些动作。 我们一起来看看蓝筹们的熊市建设。

The BAYC x McBess x The Dudes

Bored Ape Yacht Club 在今天通过推特宣布了要搞事情，要推出三方联名的周边——The BAYC x McBess x The Dudes。在 ApeFest 上面已经有过一些信息了，但是真正达成合作就是最近的事。这一次合作专为 BAYC 和 MAYC 持有者而设，将于美国东部时间 10 月 26 日星期三中午开始，到 11 月 2 日星期三美国东部时间 11:59 结束，将在官网商城上持续一周。进入商店需要来证明 BAYC 和 MAYC 的所有权。销售细节也有进行说明，只接受 Apecoin 来进行支付，小的 McCapule 售价是相当于 175 美元的 ApeCoin，包括：

  •  骷髅和鱼 T 恤 

  • 50 厘米 x 70 厘米的艺术印刷品单独编号

  • 贴纸 

  • 珐琅别针

大型 McCapsule 相当于 500 美元的 ApeCoin，包括： 

  • 牛仔夹克

  • 补丁集 

  • 珐琅别针 

  • 船 T 恤

  • 110cm x 75cm 带单独编号的艺术印刷品

Azuki PBT

接下来是看看 Azuki，Azuki 的动作相对起来就大很多了。两天前在推特宣布推出 PBT，引起了很多人的关注。Azuki 正在通过推出 PBT（一种物理支持的 token）来扩大其在 Web3 领域的影响力。受日本动漫启发的 NFT 计划旨在将有形商品与数字 token 连接起来，从而实现有形资产的链上所有权。这种称为 PBT（物理支持 token）的开源 token 标准将物理对象连接到以太坊区块链上的虚拟 token。Physical Backed Token 为企业和创作者提供了为他们的观众开发体验和下一代故事讲述的工具。PBT 现在可在 pbt.io 进行开发。

PBT 标准是有说法的，这个新闻稿称，虽然生成数字 token 来代表真实对象并不是什么新鲜事，但在对象的整个生命周期中保持与事物的联系仍然具有挑战性。Physical Backed Token 标准是一种提供完整链式、去中心化识别和监控实物整个所有权历史的技术。这引入了无信任身份验证。每个人都可以自由地在这项技术之上进行身份验证、验证和构建交互；没有一个组织拥有对象所有权的专有权利。该存储库是开源的，目前正在进行 EIP 提交。

随附的短片提供了将金色滑板连接到豆类小吃的恰当插图。第一个获得完全链上批准的有形产品将是九款 Golden Skateboards，将会在 10 月 21 日正式发布。

根据 Azuki 的说法，bean chip 是一种真实世界的加密设备，可以独立生成非对称密钥对。是第一个 PBT 实现在芯片上的。此外，据该公司称，PBT 允许硬件提供「扫描即拥有」体验，将真实事物与链上数字 token 联系起来，并从中央服务器分散。

Azuki 声称，尽管其他 Web3 和零售企业已经开发了代表实际商品的数字 token，但两者在铸造后经常分开。通过在整个所有权沿袭链中进行分散的身份验证和跟踪，PBT 克服了这个缺点。

把 The Bean Chip「豆芯片」来出来说说，启用区块链的身份验证网络芯片（「BEAN 芯片」）是一种物理加密芯片，可自行生成非对称密钥对。PBT 使硬件能够创造一种称之为「扫描即拥有」的新体验。

一系列功能，包括去中心化身份验证和实物物品所有权溯源跟踪，已经吸引了大量持有者的注意和好奇。很多持有者对更新表示兴奋，也有其他人对团队在网络上的宣传表示赞赏。多链 DeFi 经济的缔造者 FLR Finance 就是这样一位好奇的人，他对 Twitter 上发布的更新做出了回应。

FLR Finance 表示，它最近宣布了 Ernis 协议，并希望增加对 PBT token 标准的支持。这种方法还在等待 Azuki 的回应。同时，Azuki 在其更新中进一步强调，当实物物品被转让或出售给新所有者时，后续所有者可以访问扫描到拥有功能，以实现 PBT 从其先前所有者的分割转移。补充一下 Scan-to-own，用手机扫描芯片可以铸造 PBT 或以数字方式转移到所有者的钱包中。这允许物理项目的当前所有者也拥有 PBT，PBT 可验证该项目并将该项目与他们选择的钱包中的数字 token 相关联。

目标是发现利用实物商品开发适当数字体验的方法。尽管提供了对物理投递的访问权限，但数字 token 现在已经泄露了这种方法。然而，有些人可能称之为革命，但 PBT 会带来一些剧烈的变化。老牌的玩家会感觉到潮流正在改变它的方向，然而，新一代的玩家会有什么样的反应，我们不得而知。。

PBT 带来了一种新的讲故事体验，首先，允许开发人员开始使用 PBT 进行构建。在 PBT 的官方网站上提供了审查 EIP 提交、技术文档和开源 GitHub 存储库的文档。其次，持有者将有权建立他们的收藏，以获取数字和实物贸易认证物品。品牌可以通过打开现实生活的大门来利用这个工具。更高的参与度将为用户带来兴奋和对品牌的关注。

社区成员拥有足够的权力来起草新的体验，从解锁数字体验到参与现实生活中的任务。PBT 不同于依赖中心化服务器的平台。它在新推出时坚持去中心化的意识形态。无需信任的功能使社区中的每个人都可以在团队奠定的基础上进行构建。非常欢迎公开讨论和讨论以进行改进。没有任何一个实体拥有对物品和所有权进行身份验证的权利，从而使其完全无需信任。

Azuki 宣布 PBT 具有以下初始用例及其他用例：

  • 商品的去中心化认证：只需用手机扫描即可对实物进行认证。

  • 跟踪实物商品的所有权沿袭：可验证地证明该商品的过去和现在的所有者，使品牌能够为其受众打造产品体验。

  • 使用实物产品创造数字体验：拥有数字代币可以解锁实物掉落，现在拥有实物产品可以解锁数字体验。

结语

可以明显地感觉到，NFT 项目正在摸索新的玩法和模式，越来越往「品牌」去靠近了。从 BAYC 的三方联名，到 Azuki 的 PBT，好像都有点回归线下或者说是 IRL（in real life）的体验。我对 PBT 更感兴趣，因为我身处的城市就是全球的衣服制造业中心，PBT 同时也通过区块链来解决了山寨和假货的问题（或许也称不上解决，但确实更容易地分出真伪）。唉不说了，赶紧去工厂里看看，能不能通过 NFT 来帮他们产业升级转型吧。
