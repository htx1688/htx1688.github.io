原文标题：《NFT 交易指南：如果避免成为接盘侠？》

原文作者：@RiceRiddler

交易 NFT 的有两类人：

  • 投资者

  • 流动性退出者（他们自认为是投资者）

那么如何才能尽可能避免被退出流动性呢？这篇文章是投资 NFT 的速成课……

1. DYOR——自己做研究！

对于你考虑购买的每一个 NFT，请坚持亲自进行研究。

不要仅仅因为别人的建议就买某个 NFT——这是避免被退出流动性的第一步。

我不在乎是谁告诉你或为什么告诉你要买某个 NFT——自己去找出这些问题的答案：

  • 为什么 这个 NFT 正在上涨或可能上涨

  • 谁在背后推动这一切

  • 他们的计划是什么

  • 他们什么时候会部署这些计划

你计划持有这个 NFT 的时间越长，这些答案就越重要。

2. 分类

在你做完研究之后，根据你的研究想想你的「退出点」是什么，并且要足够自律，坚持下去。

将你的 NFT 分为 4 类：

短期

中期

长期

钻石手

那么这些到底是什么意思呢？

短期

这些 NFT 的风险最高——它们会在你的钱包中花费最少的时间。

这些是快速交易游戏，交易员在短时间内看到了地板价因纯粹的炒作或预期而大幅上涨。

NFT 交易员使用包括 @curiotools、@tinyastroNFT、@Promintcc 等工具追踪那些地板价被扫荡的 NFT，并选择他们认为成交量将超过降价的 NFT。

对短期炒作要非常小心。

在一两个小时以内，成交量可能会枯竭，地板价可能会暴跌……要非常仔细地监控这些短期投资。

提示：设置一个退出价格点，以及一个备用退出价格点，这样如果事情没有按照你想要的方式进行，你的投资也不至于归零。

狙击和炒作稀有 NFT

对短期类别的一个小的附带说明。

在 NFT 揭晓后，一些稀有 NFT 会出现 -

如果你正在积极关注 @blur_io、@gemxyz、@the_x2y2 或 @opensea，在适当的时候，你可以进行狙击。

在购买和上线之前，请务必查看这些稀有 NFT 的成交量和地板价。

即便如此——考虑你的退出点。

不要沉浸在拥有稀有 NFT 的兴奋中。

策略 > 赌博。我将重复这一点。

中期

中期投资是你购买的 NFT，准备在即将到来的利好出现时卖出，这项利好将使地板价上涨。

中期投资很棘手——你需要对团队有一定的信任才能做到。

你仍需要管理你的期望值。

你认为一条消息会导致地板价暴涨，但很可能不会。

市场情绪很难预测——积极监控市场和团队，对退出点进行必要的调整。

我建议研究其他项目中类似的地板价催化剂，看看这些地板价是被什么影响的……

并定期与你圈子里有经验的交易员讨论你的中期投资。这是真正使你的交易受益的地方。

长期

这些投资你不需要每天检查。

长期投资可能需要数月、一年或更长时间才能取得成果——你全心全意地相信这个团队的远见和实现目标的能力。

但是，永远不要忘记拿到你的利润——长期投资也有一个退出点……只是需要更长的时间才能到达那里。

个人情感和依恋不能成为投资的一个因素。

个人情感是区分艺术品收藏家和投资者的关键。

将中期交易误认为长期交易，或将两者都误认为是钻石手 NFT，这是导致大多数人亏钱的原因。

你计划持有 NFT 的时间越长，你对他们团队的信任度就越高。

再次强调，不要被情绪所影响。

这个团队应该具备：

  • 在各自领域的成功模式

  • 对他们的项目和持有者的热情

  • 一个可行的计划，在 mint 资金和二次销售版税之外引入持续的价值或流动性

钻石手

在你的投资组合里，只有极少数的 NFT 可以被贴上这个标签。

这些 NFT 不需要设置退出点。

（目前我手里有两个钻石手 NFT——后面将会展开叙述。）

那么……什么样的 NFT 值得钻石手的标签呢？

这些 NFT 不仅满足了成为长期投资的所有条件，而且你花费了大量的时间在这些 NFT 社区上。

你周围的人对你的影响最大——请谨慎选择！

想一想你想从 Web3 中得到什么——什么 NFT 社区能帮助你实现这些目标，并将继续坚持下去？

和朋友们在一起很好玩？可以逐步发展你的垃圾账号？有 Web3 工作？NFT 交易？

你的钻石手 NFT 应该增强你的 Web3 经验。

我的们：

我只有两个钻石手 NFT——@AzukiOfficial Beanz（我的头像）和 @ 0 xbrotherhood pass。

这两个社区都充满了真诚善良的人，他们知道如何玩得开心，但也帮助我在 Web3 中学习和赚钱。

首先，我的 Beanz 是我的品牌。

即使我买得起 Azuki，改变我的 PFP 也是一个艰难的决定。

而且……Azuki 中有如此多的构建者、艺术家和开发者——可以向如此多的人学习和建立联系。

0 xbrotherhood pass 让我接触到了这个领域很多非常棒的人，他们已经早我一步完成了我的目标。能够与他们建立联系并不断向他们学习是无价的。

总而言之……

不要把你的投资浪漫化。

对于你买的每一个 NFT，制定一个计划来实现你的目标，并执行该计划。

不要做流动性退出者——你值得更好的身份。努力做到比别人更了解你的投资。
