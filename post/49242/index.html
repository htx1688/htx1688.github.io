原文编译：Cecilia, bfrenz DAO

加密货币仍处于初期发展阶段，NFT 更是一种全新的资产类别，也是在 2017 年 CryptoPunks 的推出后才逐渐受到广泛关注。数字货币所有权正在快速实现和推广，各种政策和法律也还在适应和消化这个新的事实，这可以理解。

上周，美国国内税收局（IRS）发布了一份文件，请求公众评论并提出了有关 NFT 税务处理的新指导意见。该声明（Notice 2023-27 ）质疑是否应将 NFT 与传统的收藏品（如邮票、艺术品和高端酒）归为同一类别。 这份文件同时发出提问，针对数字艺术是否可以被纳入收藏品类别，或者是否需要开辟一种新的类别，公开寻求大众的意见。

图片来源 Cointelegraph

TL；DR：

根据 3 月 21 日发布的提案通知，美国国税局计划将 NFT 作为收藏品进行征税。

目前处于提案发布，公开征集意见中，尚未最终确定。

美国国税局打算采用“透视分析法”来确定 NFT 是否属于收藏品。在公众征集期后，将发布最终指导方针。

如果按照提案内容，NFT 将被定义为收藏品，且按照收藏品征税，最高长期资本利得税率为 28% 。而其他资产，如股票和加密货币，一般只有最高 20% 的联邦税率。

更全面的税收措施计划，或许将有助于消除市场上的不确定性。

解读现行的 IRS 关于 NFT 的指导方针

历史上，根据 IRC 第 408 节，只有五种资产类别被归类为收藏品（ collectibles )：艺术品、地毯或古董、金属或宝石、邮票或硬币和酒精饮料。第 408 节赋予了 IRS 定义新收藏品的权力，但是特别指出这些必须是“有形个人财产”。加密货币税务公司 Tax Bit 的政府部门负责人、前 IRS 首席律师 Miles Fuller 称这是一个“法律难题”。他说，”因为事实上，IRS 在监管层面上不能说他们将所有的 NFT 都归类为收藏品，因为 NFT 是无形的。”

尽管如此，他认为 “Notice 2023-27 ” 有助于划清 NFT 持有者/收藏者的税务责任。尤其是，IRS 打算将与实物物品相关联的那些 NFT 视为收藏品，这在文件中被描述为“透视分析法（Look-through analysis）”的一种解释。基本上，它将判断根据目前税法，NFT 的相关权利或资产是否符合目前在税法中定义的收藏品标准，如果是，则 NFT 被定义也是一种收藏品。NFT 可以代表任何东西，真的是任何东西，所以 IRS 的意思是，怎么收税取决于它代表了什么。

根据 IRS 的“透视分析法”，当 NFT 作为某个实物资产的所有权证书时，其将被视为收藏品，例如一个 NFT 认证拥有一个宝石的所有权，也将作为一件收藏品。但相反，虚拟场景中使用或开发“土地“通常不属于收藏品，因此提供使用或开发该虚拟土地的 NFT 通常也不属于收藏品。 

IRS 将在未来几个月发布 NFT 指导方针之前，都使用这种“透视分析法”。

目前已经有一些使用这种“透视分析法”的具体案例：碎片化 NFT 平台 Otis 出售与实物资产相关的 NFT ，例如稀有书籍和交易卡片；或者像 BlockBar 这样的 Web3 公司专注于与连接实体高端葡萄酒和烈酒的 NFT。在这些情况下，NFT 可以像所有权证书或产权证书一样发挥作用，Fuller 解释说，“IRS 并不一定会将 NFT 作为资产本身征税，而是在于 NFT 与 实物资产的联系使其具有价值。IRS 并不是试图对技术征税，而是试图对该技术所产生的经济单位征税。税法的目的是征税于实际的经济财产。”

此外，该提案通知似乎对“透视分析法”是否适用于数字艺术文件本身存疑，以及数字艺术品是否可以像其实物艺术品一样被归类为收藏品。纽约注册会计师 Justin Macari 预测，IRS 将在确定数字资产是否具有收藏价值时，会密切关注知识产权权利。IRS 还指出，数字文件是否构成“艺术品”也不太清楚（比如写作类 NFT，一篇被收藏为 NFT 的 medium 文章）。IRS 正在就这个问题和其他与 NFT 税相关的问题征求意见，在通知中，列出了请求反馈的两个主要问题：

“我认为最根本的是涉及知识产权的使用，” Macari 告诉 CoinDesk。“我会向 IRS 提出意见，因为关于这块有很多要说的。” 不同类型的 NFT 可能属于不同类型的资本利得范围。Macari  举例 Snoop Dogg （美国说唱歌手）持有的 Bored Ape 无聊猿 #6723 。无聊猿所有者拥有与其 NFT 相关的知识产权。正如 Macari 所说，如果拥有特定的头像类 NFT (PFP)或者 1/1 NFT 所附属的 NFT ，可以使持有者拥有在这个基础上创造实物商品（卖周边）和开盈利公司的权利，则这可能是长期收藏价值的明确标识符。与之相对的是，仅代表数字资产（例如元宇宙土地）的 NFT， 这种类型的 NFT 更接近 IRS 对普通资产的定义，因此两种应被相应征税。

普通资产根据个人的收入水平，税率在  0% 到 20% 之间浮动，而收藏品资产则以 28% 的税率征税。尽管 NFT 收藏者可能会面临税率上升的风险，但 Fuller 和 Macari 都认为增加法律上的清晰度还是积极的。“从一方面来讲，这次提案通知是一件好事，因为它给 NFT 作为一个整体带来了更多的合法性，”Macari 说。

收藏品的税是怎么收的

投资者在出售资产时需要缴纳资本利得税。该税是基于卖家的利润而产生的。

短期资本利得适用于持有时间不到一年的资产。这类销售所获得的利润会按照普通收入税率缴纳，这也适用于工资等。（税率有七个边际税率，从 10% 到 37% 不等。）长期资本利得适用于持有时间超过一年的资产。这些税率通常低于普通收入税率。股票和加密货币的最高税率为 20% ，适用于高收入纳税人。（较少收入的个人则需要缴纳 0% 或 15% 的税。）

但藏品（通常被超级富豪所拥有）则受到不同的税收制度的约束。它们的最高税率为 28% 。它们的结构也不同：藏品按照普通收入税率缴税，最高可达 28% 。这与股票的三级制度（0 ％、 15 ％和 20 ％）不同。

简而言之：最高收入的美国人需要为藏品支付更高的税率。

总的来说，纳税人一般不能在个人退休账户中持有藏品，因为退休账户是受税收优惠的。一家会计所的合伙人 Lewis 提到了，最近的 IRS 提案通知继续支持这一点，意思是，NFT 被分类为藏品的情况下，不能用这些退休账户去购买 NFT，否则可能会触发所得税和罚款。

有想法？如何向 IRS 提交评论

如果你有关于这个话题的想法，可以在 2023 年 6 月 19 日或之前以书面形式提交评论。确保包括对 Notice 2023-27 参考。

最简单的方法是通过联邦 eRulemaking 网站（www.regulations.gov）电子方式提交评论（在 Regulations.gov 主页的搜索字段中键入“Notice 2023-27 ”查找此通知并提交评论）。

收藏品和 NFT 领域仍存在灰色地带

Lewis 表示， IRS 的指导“对纳税人和税务从业人员来说是一个重大进步”。这在如何利用旧的税法来处理有形收藏品并将其应用于现代数字资产方面也很有创意。

然而，对于究竟什么是“收藏品”，这个定义仍存在一些灰色地带，并不总是黑白分明的，何况对于新兴的 NFT。

比如这个场景，一个人在车库里收藏着稀有汽车，专门用于收藏。但是，如果另一个人拥有同款的车，但每天开着它去上班。这辆车是一件收藏品，还是一辆交通工具呢？同样，如果一个人将一张古董桌子用于日常生活，这是否也算一件收藏品呢？

对于 NFT 也是如此，有人放着专门用作收藏，也有人会做一些实际的用处。

了解 NFT 税收的未来

虽然 IRS 的指导计划还在等待反馈中，但已经能看出来，NFT 所有者的资产未来税收的可能性。

购买、交易加密货币和 NFT 的个人投资者们应该意识到税务的影响，而加密货币和 NFT 交易则需要开始推广相关的税务知识学习计划，以帮助 holders 避免未来的问题。

随着 28% 的资本利得税取代过去 15% 的资本利得税率，这一消息可能会给过去一年中努力维持交易量的 NFT 生态系统带来更多挑战。但是，换个视角看，这些更全面的税收措施计划，或许将有助于消除市场上的不确定性。

现在已经有人上法庭反对这些新的 IRS 措施了，但值得注意的是，目前这些文件仍是初步的指导措施，未来可能会多次修改。由于主流 NFT 市场均受 IRS 的影响，这些计划的最终决定和实施，将对整个 NFT 领域具有重要的影响。
