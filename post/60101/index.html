模块化区块链可能是开启全新加密世界的关键，在下次牛市到来之前，已经有越来越多的人去讨论这个叙事。

是什么让它如此特别，为什么你应该关心它呢？今天这篇文章统统告诉你！

从单片到模块化

相较于上面提到的模块化区块链，比特币和以太坊属于单片区块链，它们只能处理三个核心功能：

  • 共识层：提供安全性并负责处理 PoW/PoS/PoH 共识

  • 数据可用性层：保证交易数据可用

  • 执行层：负责处理事务和数据

单片区块链的问题是节点需要同时执行上面的三个操作，这不是一件容易的事。

由于上面的限制，它产生了著名的「区块链不可能三角」。

例如，以太坊 L1 为了实现去中心化，牺牲了可扩展性，带来了高昂的 Gas 费和较慢的交易确认速度。

因此，模块化区块链就有了它的用武之地。

模块化区块链关注特定功能，并将其余功能卸载到单独的层去处理，因此，它可以更好、更专业地执行特定任务。

模块化区块链由多个层组成，每个层都有自己的专门功能。

例如，一层用来处理执行，而另一层处理数据可用性，第三层处理共识。

其中，Rollups 也可以通过将多个交易捆绑到一个交易中来处理，然后数据被发布 L1 中进行验证。

这就是 Optimistic rollups (Optimism/Arbitrum) 或 zkRollups (zkSync/Starkware) 的工作原理。

为了进一步提高可扩展性，并最大化数据可用性，以太坊在未来将升级成分片架构。

简而言之：以太坊的分片设计，可以并行处理交易或充当不同区块链数据块的存储「仓库」。

这允许节点仅存储其在分片链上发布的数据，从而减少存储需求，并提高性能。

由于 Rollups 和分片技术的出现，以太坊正在成为模块化区块链。

分片正在分阶段开发，下一次的 EIP-4844 升级（即将到来的坎昆升级），将于 2023 年推出。该次升级之后，L2 Rollups 的成本将降低 10-100 倍！

Celestia

@CelestiaOrg 声称是「第一个模块化区块链网络」，它专注于共识和数据可用性，而将执行层放到了单独的链上。

Celestia 的愿景是「将 Cosmos 的主权互操作区与具有共享安全性的以 Rollup 为中心的以太坊结合起来」。

实际上，这意味着应用程序或者应用链，可以使用自己的治理规则快速启动，同时实现安全共享。

（译者注：Celestia 为模块化扩展提供了不同的解决方案，其架构目前有三种类型：

1、主权 Rollup：数据可用性层和共识层为 Celestia，结算层和执行层是自己的主权链；

2、结算 Rollup：数据可用性层和共识层为 Celestia，结算层为 Cevmos，应用链为执行层；

3、Celestium: 数据可用性层为 Celestia，共识层和结算层为以太坊，应用链为执行层。）

模块化设计的其它案例：Mantle（原 BitDao）

Mantle 近期推出了三层结构的「模块化以太坊 L2 公链」

  • 一层用于事务执行

  • 另一层用于确认最终交易

  • 第三层是数据可用性

GAS 费将通过其代币 $MNT 支付。

此外，借助模块化区块链的自身特性，用户可以轻松地创建新的区块链，这减少了部署时间，并最大限度地降低了成本。

（译者注：Mantle 近期主网上线后，目前正在举办 Mantle Jounrey 活动。）

Cosmos 的模块化之路

Cosmos SDK 已经允许用户去部署特定的应用链，并且整个过程非常简单。

目前这些 Cosmos Hub (dApp) 还没有共享安全性，但这将随着 Atom 2.0 和 Interchain Security 的推出而改变！

总结

总体而言，模块化区块链比单片区块链更有优势，例如提高可扩展性以解决区块链三难困境等。

@Delphi_Digital 称之为范式转变：我们预计它们的网络效应在未来几年内将变得越来越明显。
