撰文：0xLaughing，律动 BlockBeats  

2021 年是 NFT 开始走向大众的一年，「NFT」当选了《柯林斯词典》2021 年度词汇，2021 年也因此被很多人称为 NFT 元年；而 2022 年则是伴随着新项目和用例不断涌出，对 NFT 的未来不断探索的一年，但是这一年对于加密货币和 NFT 来说是一段艰难的旅程。加密货币世界之外更广泛的市场低迷将 NFT 市场拖入熊市，Luna 暴雷、FTX 倒闭事件本身以及造成的更深层次影响让加密市场雪上加霜，数十亿美元从生态系统中消失，监管和刑事调查也随之增加。

2022 年的 NFT 市场，从 PFP 系列的激增，到时尚、音乐、娱乐、游戏和元宇宙体验的探索性尝试，这些新想法和实验在证明它们不仅仅是区块链上的「小图片」，同时也见证了无数成功和失败，让我们一同回顾一下吧。

2022 年整体概况：至明与至暗时刻交替

市场总市值变化（图源：NFTGO）

NFT 市场总市值在 3 月曾达到峰值为 349.6 亿美元，自 5 月下跌后持续低迷，总市值持续维持在 210 至 230 亿美元之间，在 11 月跌至最低仅为 208.2 亿美元，跌幅达 40.5%。

NFT 市场交易量变化（图源：intotheblock）

NFT 市场交易量的峰值出现在 1 月，最高单日交易量超过了 9.91 亿美元。5 月 1 日也创造了一个小的峰值，当日 Otherdeed 系列 NFT 24 小时交易额超过了 3.75 亿美元。随着市场下跌，NFT 市场交易量不断萎靡，在 11 月曾有多天的日交易量不到 1,500 万美元。

NFT 的持有者人数（钱包数） （图源：NFTGO）

NFT 的持有者人数（钱包数）维持了 2021 年的增长势头，从 144.6 万增长至 377.8 万，增幅达 161.3%。

  NFT 集合的数量   （图源：intotheblock）

NFT 集合的数量相较于 2021 年的数量有大幅上升，年初时仅有 3.93 万个 NFT 系列，目前增加至 22.6 万个，增幅达 475%。

细分 NFT 板块与事件

NFT 交易平台从多链、流动性、版税等多角度竞争，OpenSea 屡屡接受挑战

随着 2021 年 NFT 市场的蓬勃发展，市场上涌现出大量 NFT 交易平台，今年年初 LooksRare 和 X2Y2 先后上线，都是上线即配套了平台 Token，主打低交易手续费和交易挖矿，以对 OpenSea 进行「吸血鬼攻击」。由于交易挖矿的利益驱使，也随后衍生出「清洗交易」等问题。在这期间，由于 OpenSea 的头部优势，各个 NFT 系列在 OpenSea 上的流动性最好，它仍是 NFT 玩家在交易时的第一选择，不显著的费用优势和补贴并不能从根本上撼动 OpenSea 长期建立的品牌优势。

随着 Gem 等 NFT 聚合器平台日渐风靡，通过聚合多个 NFT 平台的价格，为 NFT 交易者提供更好的报价，同时一键扫货可以不光高效而且还节省 Gas Fee，X2Y2 等交易平台也因此受益，逐渐可以从市场中分得一杯羹。Gem 和 Genie 两家 NFT 交易聚合器凭借着优秀的解决方案先后被 OpenSea 和 Uniswap 收购。

sudoAMM 采用联合曲线

主流的 NFT 交易所如 OpenSea、X2Y2、LooksRare 都是采用订单簿模式来交易，乍一看，这种模式与 Binance、FTX 等中心化交易所买卖加密货币时采用的限价单是十分相似的，而实际上地板价往往代表了卖家认可的最低价值，offer 代表了买家认可的最高价值，由于 NFT 不可分割且单价高（蓝筹 NFT 单价动辄 10E 起步）、数量少（总量 10k 以下最为常见）、稀有度差异（持有者可能主观认为自己的 NFT 更加与众不同），地板价与 offer 价格往往不匹配，交易深度不足，这是在 NFT 市场上的交易时撮合效率低下的主要原因。为了解决这个问题，sudoswap 在 7 月推出了零版税的 sudoAMM。

除了针对 NFT 的自动做市商机制，sudoswap 率先支持零版税进一步降低用户交易费用以提高撮合效率，打破了原有 NFT 市场的竞争格局。由于挤压了原本仅仅靠着低交易手续费优势存活的小部分 NFT 交易市场的份额，X2Y2 也推出了「可选版税」的功能希望以此维持住原有市场。然而此举引发了 NFT 领域关于「是否应该取消版税」广泛的讨论（可参考《拿版税开刀，NFT 交易市场要变天了》一文），此后 LooksRare、Magic Eden 也选择允许用户自定义版税。

10 月，NFT 市场 Blur 正式上线，且推出了相对应的空投计划，吸引了大量 NFT 交易者体验。Blur 既是直接交易市场也是聚合器，凭借着平台的高效易用、低费用（没有手续费和可选版税）、空投这一套组合拳，Blur 迅速抢占了很大一部分市场份额。

这一次，Blur 这个对手真正威胁到了 OpenSea，在 11 月初 OpenSea 推出了链上版税强制执行工具作出反制，这个工具将允许创作者将不支付版税的市场列入黑名单，他们发行的 NFT 将不会上架这些 NFT 市场。随后，为了避免被列入黑名单，X2Y2 宣布取消此前的「可选版税」功能。

而在 12 月，伴随着 Blur 第三轮空投计划，NFT 市场 Blur 交易额已突破 5 亿美元，Blur 的市场份额一度超过了 60%，在聚合器市场的份额超过了 86%。主要还是因为目前身处熊市，NFT 市场的绝大部分交易额是由此前已经发售的老项目们支撑起来的，这些项目并不支持目前 OpenSea 的强制版税工具，因而仍能够在 Blur 上能够顺畅交易。

目前这场「版税之争」尚未结束，从目前的竞争格局来看，Blur 凭借着专业的交易界面、简单快速的页面、低廉的费用俘获了一大批用户芳心，同时通过拉长空投时间以培养用户习惯。「零版税」之举伤害了创作者经济，而 OpenSea 强逼着划分阵营的行为也不是长久之计，期待未来对于版税能有更好的解决方案出现。

Free Mint 行情大火，创意项目争相整活儿

5 月，一个名为「goblintown」的项目横空出世，采用隐形发射、Free Mint 的方式发售，曾在短短几天内地板价上涨至接近 6 ETH。（参考：《一周 50 倍，哥布林 NFT 的「模仿游戏」》一文）

随后大量「没有 RoadMap、Free Mint、CC0」的博人眼球的 NFT 项目涌出，纷纷标榜「预期不足，超额支付」，由此也诞生出了很多玩法，它们都试图通过千奇百怪的创意来吸引注意力。一时之间，各个 NFT 社群都逐渐沉迷于打「图狗」，希望能够抓住下一个暴涨的「哥布林」。

Free Mint 可以让利给 NFT 的铸造者，项目方通过 Free Mint 的方式发售前后进行炒作，进而通过版税回收收入，因而它逐渐受到了 NFT 交易者和项目方们的欢迎。借着这个势头，NFT 市场迎来了一波 Free Mint 小周期，其中也诞生过 DigiDaigaku、Gossamer Seed 等人人称道的「大金狗」项目，For The Culture、Moonrunners 等 NFT 项目也通过「倒贴钱」、「可变 NFT」等新奇创意而名声大噪，更多介绍：

CC0 概念大讨论

8 月 1 日，知名加密艺术家 XCOPY 发布推文，宣布他的全部作品（不含合作款）将转为 CC0 许可；

8 月 3 日，加密机构 a16z 发文《为什么 NFT 创作者要拥抱 CC0》，进一步对 CC0 NFT 展开讨论；

8 月 5 日，Moonbirds 创始人 Kevin Rose 官宣Moonbirds 也将转为 CC0 许可，允许公共领域对 Moonbirds 进行无门槛的商业使用和创造。

时值加密熊市，NFT 的交易量接连下挫持续低迷，Moonbirds 的「中途倒戈」激发了 NFT 领域对于 CC0 概念的大讨论。

「我有 NFT 的所有权，但是我却没有它的版权，它到底属于谁？」

事实上，NFT 创作者/项目方向 NFT 持有者授予相关作品的版权是有差异的，以下三种方案最为常见：

1. 没有任何声明，让 NFT 持有者有一个界限模糊的默认许可；

2. 授予明确的限制和许可，通常是各自针对作品的商业和非商业用途；

3. 将作品版权开放给公共领域，即采用 CC0 许可。

8 月 31 日，a16z 宣布推出开源「Can’t be Evil」NFT 许可。a16z 法务总监 Miles Jennings 表示：「目前整个 NFT 生态系统存在很大的模糊性和法律风险，缺乏标准化使得 NFT 购买者很难知道他们获得了哪些权利，并且创建定制许可证的成本很高，所有这些都拖累了这个行业。CBE 许可服务的三个核心目的包括：保护创作者的知识产权、授予 NFT 持有者不可撤销、可执行且易于理解的权利基础，同时帮助创作者及其社区释放项目的经济潜力。」

而目前关于 NFT 的相关法律法规并不健全，各个 NFT 项目方也未能达成共识，CC0 作为「舶来品」，与各国的法律的适配过程中也「水土不服」，在当下是否要选择 CC0 仍需多方面考量。

（参考《CC0 分岔路，NFT 蓝筹项目的生意经》一文）

NFT 金融化迅猛发展

NFT 背后的技术释放了数字经济中一直未被开发的部分，实现了数字资产的所有权。如今，NFT 主要作为个人资料图片 (PFP）而风靡了圈内外，同时也被归类为艺术品和收藏品之列。

与 DeFi 中的资产类似，NFT 这种资产也需要类似的原语，比如借贷、流动性和资产管理，这正是目前正在被构建的领域。此外，虽然 NFT 的基本价值主张在于其唯一性 (每个 NFT 都是独一无二的)，但可互换性对于增加 NFTs 的流动性和金融化非常重要。

但是想要实现 NFT 金融化，还面临着两个问题——NFT 资产的流动性低和价格发现不足。只要解决这两个问题，NFT 的实际潜力就可以被释放，否则流动性低又将进一步导致价格发现困难。

NFT 金融化垂直领域包括：NFT 市场和聚合器、AMM、流动性池、流动性提供者、碎片化、NFT 借贷、租赁和估值等等，其中一部分领域便是用来解决上述问题的。

相比较而言，目前除了 NFT 交易市场和聚合器，其他的 NFT 金融化的领域都不太成熟，其中的 NFT 借贷协议 BendDAO 表现较好。

BendDAO 的借贷业务

BendDAO 是第一个基于去中心化点对池的 NFT 流动性协议，抵押借贷是 BendDAO 的核心业务，另外还开发了「首付买」的功能。但此前 BendDAO 也因业务不成熟爆发过流动性危机（参考《BendDAO 资不抵债，NFT 终于也要迎来次贷危机了吗？》一文），好在最后妥善解决。

不过，目前 NFT 金融化的市场格局仍处于起步阶段，解决方案仍不可避免地陷入贷款价值比低、利率高、缺乏可扩展性和流动性不足等问题。此外，缺乏可扩展、低风险和可信的价格评估机制也限制了 NFT 信贷市场，导致 NFT 支持的贷款需求超过了借贷资本的供应，并造成大量 NFT 借贷需求未能得到满足，NFT 金融化任重道远。

（可参考《NFT 金融化，面临的挑战及存在哪些机会》一文）

展望 2023 年

不仅仅是 PFP，NFT 将探索更多实际用例

PFP 是目前 NFT 市场里总市值、交易额最高的板块（图源：NFTGO）

12 月 14 日，Binance 创始人 CZ 在「CZ AMA」中就 NFT 相关问题发表看法表示：「NFT 目前仍处于早期阶段，还有很多用例未发掘，例如 NFT 世界杯门票等，我们需要有人去做这些场景用例的相关项目，Binance 很乐意去投资这些应用。」

诚然，PFP 是目前 NFT 市场里总市值、交易额最高的板块，很多人在 Twitter 等社交媒体上将 NFT 作为自己的个人资料图片，以将其帐户与他人或假的帐户区分开来。这些仅仅是开始，NFT 作为一种表示所有权证明的方式，证明数字资产的所有权比证明实物资产的所有权更加容易，这也是为什么艺术品和 PFP 成为了最开始的用例，社交价值和收藏价值驱动着 NFT 玩家更倾向于交易这类 NFT。

但是 NFT 还有许多未开发的潜力，存在更多用处。如以下一些领域中：

  • 用户名和域名

  • 品牌和市场营销

  • 门票和通行证

  • 游戏资产

  • 房地产

  • 学位和证书

  • 防伪保护

  • 会员俱乐部和忠诚度奖励

  • 投票

  • 知识产权保护

  • 众筹

  • 法律合同

  • 电子邮件和消息

  • 医疗记录

除了金融化，人们也在不断寻找 NFT 的其他创造性用例，未来通过 NFT 将链下资产和数据等大规模 Token 化，这对于整个加密货币市场的影响都会是深远的。

更多的传统知名品牌入场

今年已经有许多知名的品牌加入了 NFT 市场，包括星巴克、Nike、Adidas、Reddit、Instagram、保时捷、迈凯伦，以及 Burberry、Gucci、Louis Vuitton、Tiffany 等奢侈品牌。

其中 Adidas、李宁等品牌选择与无聊猿合作，Nike 则是更加激进地收购了 NFT 品牌 RTFTK，并发售了虚拟跑鞋。传统品牌和零售商拥抱 NFT、Web3、元宇宙（集体虚拟世界）不仅仅是为了炒作，一方面跨界联名可以收获品牌关注度，但是也有大公司似乎在押注于元宇宙，为未来人们进入虚拟生活做足准备。

NFT 正在推动品牌、零售商的销售方式和与消费者互动方式的转变，可以在收入、营销渠道、社区建设和忠诚度方面创造新的机会。这些传统品牌有一种紧迫感，想弄清楚如何利用 Web3，尤其是在许多品牌（尤其是奢侈品牌）错过了最初向电子商务扩张的机会并只能迎头赶上之后。现在，一些品牌急于避免再次犯同样的错误。

随着 Web3 技术站稳脚跟，与用户的数字互动——从销售虚拟商品到使用虚拟世界作为新的营销渠道——为时尚领域的品牌和零售商提供了巨大的品牌推广和货币化机会。

如今，传统品牌和零售商正在通过数字内容与流媒体巨头、游戏平台和社交媒体争夺消费者的注意力。随着元宇宙的兴起进一步改变消费者度过时间的方式，这一挑战只会越来越大，随着越来越多的品牌进入 Web3 和元宇宙，要想脱颖而出，需要越来越有创意的方法。随着对消费者注意力的竞争加剧，采用 Web3 技术一方面可以为品牌带来新的收入、营销和品牌相关性方面的新机会，另一方面 NFT 也能促进了主流大众更好地了解和加入加密市场。

展望 2023 年，我们可能会看到更多品牌在 NFT 世界中进行试验。
