原文标题： Web3 Social : Driving New Forms of Identity, Connection & Value Creation

原文作者： Supercrypto 和南加州大学（ USC ）区块链社区

原文编译：倩雯， ChainCatcher

Superscrypt 的使命是支持那些能为区块链领域带来下一波用户的创始人和项目。我们采用以研究为导向的方法来确定关键的 Web3 构件，这些构件将推动我们如何在全球范围内进行互动、创造和分享价值。今天，我们通过在 Mirror 和 Link 3 上发布博客来加入 Web3 社交，与世界分享我们的观点。

如何探讨 Web3 社交？

去中心化区块链技术的使用正在改变世界的运行方式。虽然区块链最流行的使用案例是金融化（代币、 DeFi 和 NFT ），但该技术驱动的社交网络，将会支持新形式身份、数据所有权和通信功能的发展。在本文中，我们将探讨 Web3 社交的核心价值主张、各种协议和平台的运作方式、目前产品与市场的契合程度，以及其成为未来用户在线互动、建立社区、交友和交易的潜力。

要了解 Web3 社交的可行性，最好的办法是首先了解过去创新浪潮是如何形成的，并评估 Web3 社交是否具有类似的特征。创新浪潮出现在技术创新和新的消费者/商业价值的交叉路口；它们改变了我们世界的运作方式，事后看来这些影响往往是显而易见。就像现实世界一样，海啸之后往往是几波巨浪。硅芯片和互联网是海啸，而个人计算、电子商务、数字金融服务和 Web2 社交平台则是紧跟其后的浪潮。在 Web3 领域，区块链是新的海啸，紧随其后的是去中心化金融、 dapp 和 NFT 等浪潮。 Web3 社交平台会成为新浪潮吗？它能比之前的革命提供更多的附加值吗？

数据开放和可迁移性

我们今天使用的社交媒体应用利用复杂的底层数据网络来实现我们的体验，这些被称为“社交图谱”；相互连接的数据结构映射出我们的数字关系和互动、我们关注和订阅的对象、喜欢的内容等等。在 Web 2.0 中，社交图谱是私有系统，所有资产、数据和行为分析都由互联网公司拥有或控制。用户对这些数据缺乏真正的所有权或控制权，因此社交图谱被“孤立”起来，与其他平台隔绝；我们的身份和体验很可能永远无法跨平台连接。要使用新平台，用户必须重建过去的社交关系和历史记录，但这带来了巨大用户体验摩擦，导致用户只能固定在某一平台。如果终端用户想从一个应用程序切换到另一个应用程序，就必须从头开始。

Web3 社交有何不同？首先，它旨在建立一个基础层，在该基础层上，用户的社交图谱数据透明、无权限、可迁移，最重要的是，用户可以控制这些数据。我们称这种模式为“开放数据”。

开放数据使用户可以轻松使用基于相同社交图谱的新应用程序，而无需处理冗余问题。任何使用 Web3 社交平台的人都可以将他们的数据用于任何他们想要的应用程序。想象一下 Joe Rogan （美国喜剧演员）决定迁出 Spotify ，目前他和其他创作者对其粉丝、订阅者和内容的访问是有限的，在某些方面，他必须在另一个平台上重新开始。以 Joe Rogan 的知名度，这应该不是问题，但对 99% 的艺术家来说，这意味着不少困难。

孤立的数据阻碍了独立开发人员利用这些社交图谱，并在此基础上为用户打造新的体验和应用。因此，社交应用领域的创新面临着巨大的冷启动问题。新的社交应用经常被开发出来，但往往因为无法获得关注而失败。在 Web3 社交的世界里，如果用户能把自己的数据和朋友带在身边，他们可能会更愿意尝试新的社交体验。新颖 Web3 社交体验的一个例子是 Link 3 ，这是一个基于 CyberConnect 协议的社交应用。 Link 3 是 LinkedIn、 Medium 和 Eventbrite 的混合体，但其中嵌入了加密原语——允许用户连接钱包、 NFT 和 ENS 名称，以建立用户的个人资料和影响力。

消除数据孤岛还能降低新开发者的准入门槛，减少社交图谱的网络效应，否则社交图谱就会垄断用户数据。开放数据允许应用程序读取和写入多个社交图谱协议的数据；用户可以将多个 Web3 社交资料中的 ID 连接到一个或多个应用程序。 Phaver 就是一个很好的例子，它通过与 Lens Protocol 和 CyberConnect 集成，在一个界面上连接了来自不同网络的用户的社交图谱。

开放数据标准提供给我们开放的内容创作边界

目前，我们在现有 Web2 社交应用程序中看到了开放数据的例子。 Facebook 和 Instagram 的所有者 META 最近推出了 Threads 。虽然 Threads 作为一个应用程序的长期价值尚待确定，但 Threads 允许用户在注册后立即关注 Instagram 好友并与他们建立联系。不过用户在 Instagram 和 Threads 上的数据仍被隔离在 META 的墙内。 Threads 宣布有可能与 ActivityPub 集成， ActivityPub 是一种标准通信协议，允许用户使用 ActivityPub 与所有平台互动，但目前集成尚未实现。

Web2 社交的封闭性也对在这些平台上运营的企业产生了巨大影响。 Reddit 最近实施了 API 定价，导致多个应用程序关闭，引起了社区的强烈不满。如果 Reddit 是建立在 Web3 社交基础架构上，这就不是问题，因为这些应用程序和社区可以将他们的粉丝和内容带到条件更有利或价值更一致的新平台上。

在内容创作方面，由于用户不需要锁定在一个平台上，创作者可以选择迁移到不同的应用程序，同时保留其社交图谱数据的所有权，从而缓解平台改变其货币化、管理或收入分享模式的情况。在 Web2 社交中，数据孤岛阻碍了创作者的这种选择自由。

可组合性：开启新的可能性之海

由于 Web3 社交协议部署在区块链上，因此通常可与其他智能合约和协议高度兼容。这种开放式数据架构允许新的加密货币创新开发者自由地与 Web3 社交协议和应用程序集成，例如连接用户的链上历史以填充他们的社交档案，将他们拥有的 NFT 显示为凭证，并使他们能够将自己的内容和帖子转化为 NFT 并进行潜在销售。传统社交媒体缺乏可组合性，即使在加密货币与 Web2 社交整合的少数案例中，也既不原生也不完整。

与开放数据相结合， web3 的可组合性使创意开发者能够将他们的社交平台与加密协议整合在一起——从 DeFi 应用程序、元宇宙游戏、 DAO、 NFT 到介于两者之间的一切。例如， Warpcast 是一个基于 Farcaster 协议的平台，它允许用户查找相同 NFT 社区的其他成员、同一 POAP 的持有者，甚至可以将 ENS 域链接到用户的个人资料。这些功能是通过将以太坊地址连接到个人资料，创建链上整体身份而实现的。其他平台也提供类似功能，其中一些平台（如 Phaver ）允许用户将 Lens 配置文件、 ENS 域和 Farcaster ID 连接到一个配置文件。 Web3 可组合性的另一个例子是 CyberConnect 的订阅模式，它允许用户通过 “Subscribe NFTs”的强大功能定义创建者与订阅者之间的关系。

另一种可能性是 SocialFi ，这是 Variant Capital 的 Mason Nystrom 最近写过的一个概念。 SocialFi 指的是人们数字生活中社交层和金融层的整合。试想一下，如果传统的社交媒体平台可以无限制地接入传统的金融交易所、银行、贷款提供商、众筹平台等，它们会做些什么。 SocialFi 的其他功能还包括在 Web3 社交应用程序上启用小额支付、与社交身份相连的链上借贷信用评分以及与社交档案相连的生物身份证明。

随着 Web3 基元的不断扩展和发展，加密货币可组合性的可能性是无限的——新的体验在等待着我们。

下一层次的社区赋能

区块链基础设施是一个强大的工具，它能让人们为了一个共同的目标走到一起。在过去几年中，我们看到了这一点：由代币驱动的社区协作起来竞标美国宪法（ ConstitutionDAO ），为 DeFi 协议治理投票（ Uniswap、Safe、Maker、Gnosis ），投资数字艺术（如 PleasrDAO ），以及围绕 NFT 藏品建立社交俱乐部。参与者成为所有者和高度参与的利益相关者。

将动态 NFT 等新的创新技术与新兴的 Web3 社交协议相结合，将使这些体验更上一层楼。将现实世界的数据与 Web3 结合，可以进一步增强这种体验；例如，认证、购买、社交关系、更多相关的社区也会立即产生，真正的关系将更容易形成，新的商业形式也将出现。

同时，可以可视化 Web3 社交图谱，该图谱由完整的身份（例如 NFT 档案或其他形式的标记）连接，可相互参照类似特征。通过开放数据和透明度，平台可以根据与社交图谱中其他配置文件共享类似特征的配置文件自动生成社区。

一些思想实验包括：

一个类似于 LinkedIn 的 Web3 平台的用户可以在其个人资料中附加一个标识符，将其与母校联系起来，比如以 NFT 属性的形式。该平台可以自动为拥有相同母校的用户创建一个校友网络社区；通过类似的方法，还可以自动创建全球性的社区，并且不受 Web2 数据护城河的影响。它不仅允许个人加入社区，还允许艺术家、作家、有影响力的人和其他创作者与所有关注他们的人建立社区。

试想一下，如果一位作者出售一本附带 NFT 的书，那么每个 NFT 持有者（也是该书的认证购买者）都可以加入一个社区，讨论这本书并与作者交流。正如我们即将看到的那样， Web3 社区功能的实际例子还包括 Warpcast 和其他 Web3 社交应用程序的用户，他们能够轻松找到同一 NFT 所有者社区中其他人的用户配置文件。这些形式的全球统一社区功能超越了 Web2 目前所能做到的阶段。

Web3 社交应用 Orb 正在将这些思想实验变为现实。 Orb 允许将任何链上活动或拥有的物品转化为一个有凝聚力的社区。例如，基于拥有特定数量的 ERC -20 代币、持有特定活动的 NFT 门票或藏品等行为都可以形成不同的社区。 Web3 社交允许以新的方式建立社区、结交朋友，而且比现在的方式更深入。

抗审查和所有权

无论信仰、信念或观点如何，言论和表达自由都是社交媒体的基石，尽管其限度一直存在争议。鉴于传统社交平台对我们的数据拥有强大的控制权，它们处于绝对的强势地位，能够对网上言论控制，根据自己的判断取消用户的平台权限。用户除了接受之外别无他法。

最近，在 Elon Musk 接管推特（现更名为“ X ”）之后，一个平台控制的实例被展示出来。推特档案（“ Twitter Files ”）是一系列被披露的推特内部文件，揭示了该公司应用程序如何根据政府要求或其认为不安全的信息，压制和控制平台上的信息。虽然 Elon 通过该文档倡导真相和透明度，但讽刺的是，在推特更名为“ X ”后，该公司却毫不客气地从一名用户手中夺走了“@x ”账号。同样的情况也发生在拥有“@music ”账号的用户身上。

这些例子凸显了在传统的 Web2 平台上，用户永远无法真正掌握自己的身份、账户，在某些情况下甚至无法发出自己的声音。 Web3 社交平台在一定程度上解决了这一问题。基于 Web3 社交协议构建的应用程序仍可对内容进行管理，但用户仍拥有完全的自主权，并可选择离开一个应用程序，转而使用另一应用程序，但仍可保留自己的数据，甚至在一切都失败的情况下，建立自己的抗审查应用程序。这里的核心原则是，用户不需要被迫锁定在特定的应用程序上——只要用户愿意，他们就可以放弃某个社交应用程序，并迁移到一个全新的应用程序，同时带走他们的所有数据。

小结

Web3 社交平台令人兴奋之处在于，它为社区、商务、可组合性、参与、所有权和身份认同带来了全新的体验，同时也解决了现有 Web2 社交平台的一些核心弊端。建立社区、交友和交易的新方式指日可待。

同样令人兴奋的是进入这一领域的企业家的素质。在 Superscrypt 的投资中，Airstack、Collab.Land、Orb、Notifi 和 Salsa 正在解决 Web3 社交在访问、体验或互操作性方面的不同挑战。

要将 Web3 社交推向大众，前路漫漫——技术和应用正在不断增长，但仍处于起步阶段。在实现真正的产品与市场的契合之前，我们期待着更多的迭代，并将很快与大家分享更多更新信息。
