原文来源及作者：ApePark

区块链行业从 DeFi Summer 到如今的加密寒冬，多年来行情起起伏伏，这段时间内新鲜概念如雨后春笋般不断出现，其中绝大部分又很快归于沉寂，但是 DAO——这个象征着区块链精神和未来 Web3 蓝本的理念从未被人遗忘。

作为专注于 GameFi 资产管理交易的数字资产交易平台，ApePark 在 DAO 治理上迈出了关键一步——创建了生态的收益型身份治理 NFT。

项目背景

在了解身份治理 NFT 之前需要简单了解 ApePark 项目本身的涉猎范围，作为一个在 GameFi 领域高度垂直的资产管理平台，ApePark 的功能非常齐全。它既是新用户了解 GameFi 领域讯息的门户，同时也是玩家体验游戏的平台，还是创作者们上传游戏和私人 Token  的启动板，更是自由交易游戏资产 NFT 的开放市场——从入门新人到自身用户，ApePark 平台可以满足所有人群的需求。

ApePark 的平台框架能够满足从入门新手到顶级玩家的全部需求，并且在游戏资产 NFT 市场开拓上进行了资源整合，极大程度地优化了用户体验。

同时在市场表现方面，ApePark 的先进理念和优质蓝图吸引了多方关注，到目前为止已获得 Daoverse Capital、Crypto Nord、Defiking 基金会等多方种子投资，Seed 轮融资总额超 500 万美金。

优秀的框架设计和充足的市场信心是 ApePark NFT 诞生的绝佳土壤！

ApePark NFT

作为一个严格践行区块链精神的项目，ApeParK 坚定地使用 DAO 治理模式对项目进行管理，项目同时拥有治理 Token  ApePark DAO 和身份治理 NFT 两种治理手段，治理 Token APD 计划发行总量 1.15 亿枚，其中用户购买和 DAO 池分别产出 5000 万枚，质押池产出 1500 万枚。

身份治理 NFT 则总计发行 10000 张，Mint 价格为 700U 或等价 BNB，身份治理 NFT 的持有者不仅是治理组织的一员，同时拥有持续的权益。

首先 10000 张 NFT 持有者将会共同平均地享有 DAO 池产出 APD Token  的 30%，即 1500 万枚，根据当前 APD Token  在 Pancake Swap 上的交易价格总价值约为 7200 万 U。当然为了项目的健康发展，这一笔 APD Token  将会线性释放。

其次在 ApePark 平台上身份 NFT 的持有者也享有特殊权益，减免一次在 Launchpad 发行和上架 Token 的服务费；在 NFT 交易市场进行游戏资产 NFT 交易时也可以减免手续费；同时享有平台 Vvip 待遇。

最后，根据 DAO 组织的治理形式，身份 NFT 的持有者们有权通过公投的形式决定平台未来的发展方向以及收益分配规则，实际地参与平台管理工作，真正做到用户助力发展，平台回馈的共赢模式。

今天看来，ApePark 或许距离成功还有很长的路要走，但它所提出的「真实的 DAO 治理」的理念，无疑将对去中心化治理的未来产生重要的影响。
