作者：YuanShan@iNFTnews.com

零知识技术及其赛道的发展随着以太坊L2 ZK Rollup的出现，受到众多关注。

2023年初始，谈论ZK（零知识）赛道过去的进程，和预测ZK赛道未来的发展，成为Web3或加密圈内一件很重要的事。

因为无论是建立在以太坊之上的L2 ZK Rollup，还是自带ZK属性从头开始建构的公链项目Aleo，ZK赛道都在呈现有目共睹的快速发展。

回顾：2022隐私计算项目的3个发展趋势

Aleo CEO Alex Pruden说：“改变 Web3 的技术之一，无疑是零知识 （ZK） 密码学。作为区块链应用生态在隐私、安全性和完整性等方面的游戏规则改变者，ZKP（零知识证明）的加速投资和发展是它正在为黄金时期做好准备的重要信号。”

可知，ZK被赋予、被视为Web3重要的变革因素，且正在迎来它的黄金时代。

加之从用户角度出发，对个人信息数据的保护变得越来越重要，而隐私计算成为数据安全与隐私保护的最优解决方案之一。

回顾2022年，隐私计算公链项目的发展趋势可以总结为三个方面：

1.技术创新：隐私计算技术在2022年进一步发展，开发出了更加高效、安全的隐私保护技术。例如，混合加密技术、私钥加密技术、聚合签名技术等都将得到更多应用。

2.市场扩张：隐私计算公链项目进一步扩张其市场份额，逐渐成为区块链领域的主流技术。例如，隐私计算项目Zcash已成为支持隐私计算公链市值最大的项目。

3.应用拓展：隐私计算公链项目逐渐扩展其应用领域，拓展到更多行业。例如，应用于医疗、DeFi行业。

具体到ZK领域，作为业内解决方案之一，在隐私保护方面掌握着区块链隐私保护应用的关键。总结而言，零知识技术既能让开发者利用公链的安全性、又能提高交易吞吐量和速度，同时可将用户个人信息储存在链下，以保护用户隐私。

2022年，在技术、市场和应用方面都得到快速发展的隐私计算项目，在2023年预测也将持续快速发展。具体而言，在ZK赛道，会有以下项目被聚集关注。

展望：2023值得关注的5大项目

ZK赛道从项目上可分成两个来源，一是从0到1自带ZK属性建构起来的ZK项目（Aleo），二是在区块链之上创建ZK协议解决方案（ZK Rollups）的ZK项目。

Rollups 是一种常见的扩展方案，其ZK Rollup的工作方式是在 L1 之外（链下）处理和执行交易，但将交易发布到 L1。这使得 Rollups 提供扩展功能的同时，又能从 L1 中获得安全，且gas 费用很低。这种模式就是以太坊L2扩展解决方案的模式。

目前，Aleo、zkSync、StarkWare和Scroll等都在积极推动ZK技术的应用，它们都将成为让（运用此算法的）应用程序面向真实用户开启大规模测试的一个重要起点。

1、zkSync

Zksync的TVL有5300万美元，根据 @Orbiter_Finance 数据，它是交易数量排名第三位的 L2；到目前为止，至少有 300 个项目加入了该生态系统。

Zksync由Matter Labs开发，团队成员包括联合创始人兼CEO Alex Gluchowski 、高级软件工程师Danil Lugovskoi。Zksync采用ZK-SNARK技术范式。

2020年6月在以太坊主网上发布ZKSync 1.0，内嵌了一个交易平台（Zigzag），为用户提供转账、mint NFT等功能。2022年2月zkEVM开放公测，是第一个和EVM在编程语言层面等效的ZK虚拟机。

2022年10月28日Zksync发布ZKSync 2.0 Baby Alpha版本，内置了zkEVM Alpha版本，将开放生态应用的部署，包括浏览器、新跨链桥、钱包等。

2022年11月下旬ZKSync 2.0进入Fair Launch Alpha阶段，开放生态开发者的项目部署权限，以便开发者能够测试并升级其合约，该阶段预计会持续几个月。

截止 2021 年 11 月份 Matter Labs 从 a16z、Union Square Ventures、Placeholder 共融资 5800 万美元。目前共计有70＋的应用，2.0 上线后，兼容EVM特性可能会导致应用数量进一步增多。

Zksync不断增长的生态系统图

预计2023年上半年ZKSync 2.0进入Full Alpha阶段，届时ZkSync 2.0上的产品将对普通用户开放。

  • 定位：加速个人主权对加密货币的大规模采用

  • 总融资：4.58亿美元

  • TVL（总锁定价值）：5300万美元

  • 生态项目数量：至少300

  • 应用程序：70+

  • 优势：使用加密有效性证明来提供去信任协议，实现以太坊上的可扩展信和低成本交易，同时拥有和以太坊相同的安全级别。

  • 不足：依赖于以太坊的安全性，主要用于Token交换或转移

  • 主要项目：用于支付和转账的zkSync 1.0支持智能合约，可供开发者编写应用的ZKSync 2.0基于ZKSwap协议建立的去中心化AMM交易所的ZKSpace提升应用在不同L2链间的跨链能力的Hyperlink（预计2023Q1上线）

2、StarkWare（StarkNet）

StarkWare公司创建于2018年，技术团队负责人之一Alessandro Chiesa是Zcash的联合创始人。

团队在ZK算法基础上提出了具有扩展性的零知识证明算法ZK-STARK，主要致力于为区块链扩容和隐私保护提供解决方案。其主要产品为混合型L2方案StarkEx，可同时兼容ZK Rollup和Validium，用户可在二者之间自由切换。

除StarkEx外，团队还开发了一个无需许可的去中心化ZK-Rollup StarkNet，是“四大”早期主流rollup之一。任何用户和开发者都可以在其中部署智能合约。

StarkNet 可能是 zkSync 最大的竞争对手，相关信息如下表：

  • 定位：StarkNet作为以太坊上的 L2 网络运行，使任何 dApp 都能实现其计算的无限规模——而不会影响以太坊的可组合性和安全性。

  • 总融资：2.73亿美元

  • TVL：460万美元

  • 生态项目数量：200多个

  • 优势：StarkNet拥有自己独有的高级编程语言和框架 Cairo，是以太坊上第一个生产级完整的冯诺依曼图灵检查者（checker）。Cairo允许开发人员使用ZKSTARK的强大功能来创建完全可扩展的应用程序。自2022年年初以来，StarkNet生态系统已达到可用性门槛：可以构建完整且令人惊叹的dApp。

  • 主要项目：StarkNet钱包：ArgentXStarkNet桥：将资金从L1（以太坊）转移到L2 Starknet的方案Zigzag：DEX，一个围绕ZKP构建的去中心化交易所，带有OrderBookZKX：永续合约交易Starkswap：DEX AMM，自动做市商的去中心化交易所zkLend：zkLend是StarkNet借贷产品，基于StarkNet的货币市场协议。ZkLend为机构客户（“Apollo”）和面向DeFi零售用户（“Artemis”）的免授权服务提供双重授权套件。StarkNet Launchpad：为所有人建立的一个风险投资平台Magnety：Magnety 是StarkNet DeFi产品，是一种资产管理协议，它允许任何人（例如投资集团、DAO或个人）通过Starknet和L1 DeFi生态系统创建和管理对冲基金。

3、Scroll

Scroll项目是什么？联合创始人张烨介绍：Scroll是一个以太坊的通用型L2扩容方案，和以太坊本身类似，开发者可以在Scroll上部署智能合约，同时，也可以和上面的各种应用进行交互。但是在上面的交易费用更加低，而且吞吐量也更高。

Scroll虽是一个扩容方案，但是在Scroll上所有的完整性都会在以太坊进行验证，要么通过ZK证明，要么通过欺诈性证明，其安全保障由于得到以太坊的支持，表现会更加强大。

Scroll是一个zkEVM L2。目标是拥有一个可扩展的网络，使现有的以太坊协议具有本地兼容性。

Scroll正在建立一个EVM等效的ZK Rollup，其设计决策遵循安全性、效率、EVM等效性和去中心化，其架构由Scroll节点、链上的智能合约、去中心化的证明器网络三部分组成。

Scroll有明确的发展计划。

第一阶段，Scroll已完成Pre-Alpha测试网。

第二阶段，邀请开发者基于其部署一些智能合约，并且开发一些额外的应用。

第三阶段将启动二层证明的外包，也就是证明生成的过程，Scroll希望能够邀请所有的社区参与，这是无需许可的，任何人都可以参与到证明网络，成为证明节点。

第四个阶段到达zkEVM主网阶段，需要经过严格的代码审计以及性能提升之后，将会部署并且上线主网。

第五个阶段，部署去中心化的排序器。

  • 定位：建立一个EVM等效的ZK Rollup

  • 总融资：3000万美元

  • 优势：可以在内部支持具有自解码级别的EVM，对于开发者来说，意味着可以支持VEM上支持的所有内容，不仅仅是Solidity这样具体的编程语言，同时也可以在字节码层面支持以太坊虚拟机，以及所有相关的开发工具。

  • 生态项目：Lens Protocol：在以太坊上进行交易和自动提供流动性的协议Uniswap Labs：一种无需许可、可组合且去中心化的社交图谱，可轻松构建 Web3 社交平台The Graph：一个用于组织和访问区块链数据的 web3 协议

4、Aztec Network

Aztec成立于2017年，是以隐私为先的架构构建的ZK Rollup，运用ZK-SNARK技术为用户提供隐私跨链桥Aztec Connect，使得用户能够私密地访问以太坊应用程序，允许用户在完全保护隐私情况下访问 Layer 1 想使用的应用程序，其产品体系为基于底层的 PLONK 证明系统，实现账户间的匿名交易，并通过网关合约的嫁接来实现和 DeFi 项目的隐私交互。

Aztec 在近期还推出了编程语言 Noir，让开发者在以太坊上更容易、更方便构建具有零知识证明的应用程序。

Aztec的主要目标是为协议提供透明度，但为用户提供隐私，其生态系统当中已经有很多可以使用的产品。

除服务Web3领域外，还和传统金融机构如JP Morgan有合作，提供ZK加密方案来解决私人结算中的隐私保护问题。

  • 定位：实现以太坊的原始承诺——一个去中心化和可验证的互联网世界，并通过使其完全加密来极大地扩展其功能。

  • 总融资：1.17亿美元

  • TVL：1100 万美元

  • 优势：B轮融资领投1亿美元的a16z表示，Aztec的加密和隐私对于区块链获得广泛采用、人际协作、资本、社会以及下一代互联网至关重要。

  • 不足：Aztec 生态处于有待开荒的起步阶段，原生项目稀少，当前以集成主网 DeFi 协议为主，其目标是实现与主网所有协议、智能合约的隐私交互。

  • 生态项目：Aztec Connect：7月7日正式上线，是一套连接以太坊 DeFi 协议和 Aztec 的 private rollup 工具Zk.Money：是Aztec Network 的第一个应用，发布于 2021 年 3 月。旧版 Zk.money 只能实现简单的转账支付，新版的 Zk.Money 则在保留原有的转账支付功能添加了隐私 DeFi 功能，用户的资金在完全保护隐私情况下可用于与主流的 DeFi 协议进行交互。ZKPay：为Aztec 隐私支付而设计的快速提款桥。在 Aztec 网络上的资产存取可能需要 6 个小时，但在 ZKPay 上只需要几秒钟。Trelis ：非托管的以太坊支付集成，现在正处于测试阶段。在测试模式下可通过使用支付链接、Trelis API 和 WooCommerce Plugin 体验 Trelis，交易上限额度 100 USDC，支持所有主流的以太坊钱包。

5、Aleo

Aleo是第一个提供完全私有应用程序的公链，利用去中心化系统和零知识密码学为用户提供具有绝对隐私的无限计算，通过构建一个默认为私有、开源且为Web构建的区块链，解决区块链隐私保护的缺点。

Aleo 是一个 L1 区块链，其开发人员将zk驱动的区块链视为第三波浪潮：首先是比特币，然后是以太坊，现在是Aleo。

Aleo的重要性在于它不像Zksync一样构建一个与EVM兼容的链，它是从头开始构建基于零知识的区块链，并创建了自己的虚拟机snarkVM和编程语言Leo。

和以太坊L2 ZK项目相比，Aleo的挖矿算法更便捷、简单。L2之上的ZK项目还需要Gas费，Aleo则完全去除了这笔费用。

目前Aleo官方公布的融资有两轮半，分别是A轮和B轮及B轮+，共计达近3亿美元。其中A轮融资金额2800万美元，B轮融资金额2亿美元，B轮+融资7000万美元。投资人有Andreessen Horowitz(a16z)、PlaceholderVC、GalaxyDigital、VariantCapital和CoinbaseVentures等，造就了Aleo成ZK赛道最大的融资项目。

  • 定位：让应用程序变得私有、安全

  • 总融资：近3亿美元

  • 优势：Aleo是第一个采用零知识证明类的POW机制高性能公链，也开启了同时采用POS和POW的应用。从零到一搭建零知识证明属性的隐私公链，是Aleo和其它L2 ZK区块链不同的地方，也是Aleo的优势，其开发了更易开发程序的Leo语言、零知识执行系统ZEXE、私密安全的计算环境 zkCloud等。

  • 不足：和以太坊L2丰富的生态项目不同，Aleo还处在开发的阶段。市场和应用生态的选择是Aleo急需关注的地方。

  • 生态项目：Leo Wallet：Aleo钱包项目，是进入Aleo网络的窗口。Forte：区块链游戏公司，在Aleo网络之上构建、开发游戏，既具备隐私保护同时又高速运行。Nucleo ：一款为组织提供开发隐私多重签名基础设施，能够进行隐私交易、隐私 DeFi、隐私筹等。Kryha：区块链游戏项目，致力于在区块链游戏行业中引入零知识技术。

结语

无论是Aleo代表的原生ZK项目，还是以太坊之上的L2 ZK项目，既在快速发展的同时稳中向好，同时它的适用性和价值累积仍有待证明，例如DeFi Dapp（去中心化金融应用程序）大规模迁移问题。

积极的一面是，如ZK Rollup，仅在今年，就有 3 个 Zk EVMs 进入了测试网阶段，其中一些已经进入主网，包括 Scroll，Polygon Hermez以及 Zksync2.0。Aleo公链之上的Dapp正在开发当中，最后一个测试阶段完成后主网便开始上线，意味着更丰富的ZK生态项目的爆发。

还需谨慎对待的是，ZK赛道早期阶段，无论其技术、市场、应用等领域，都仍有待市场证明。Web 3.0面临的掣肘包括有限的可扩展性以及高延迟。若没有足够的交易吞吐量和足够低的成本，Web 3.0很难容纳来自主流市场的容量。这既是Web3的面对的问题，也是零知识赛道项目面临的问题。

对ZK Rollup来说，ZK Rollup技术发展尚处于早期阶段（较为乐观的情况下，预计ZK Rollup运营商赛道规模将达到10至100亿美元级别），且ZK Rollup的发展仍然面临较高的不确定性，作为一项尚未迎来爆发性增长的技术，其技术、价值、业绩验证均存在滞后性。

对Aleo项目来说，它是否能和已占据市场资源的L2 ZK项目相互竞争，能否取得良好的市场效益并经受住市场验证，这是值得关注的问题。

但作为技术的乐观派，我们相信未来隐私计算技术将得到进一步完善，也将应用在如金融、医疗、政府等更多领域，同时与其它区块链技术（如模块化、集成层、智能合约等）相结合，实现更多的应用场景。

2023年，对ZK赛道的展望，那就是：随着技术不断进化，更多的自由会在更低成本的信任下被释放出来。
