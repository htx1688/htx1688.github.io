在过去十年中，金融科技（FinTech） 一直是传统金融服务公司运营方式创新的主要驱动力。 它打乱了从支付处理到预测和预算方方面面的常规秩序，催生了我们今天所熟知的金融科技领域的庞然大物，例如：Stripe、Deel 和 Brex。

作为一种新的计算原语，Web3 标志着范式的根本转变，Web2 中的传统金融科技工具无法将它复制。法定货币换成加密货币，银行账户换成数字钱包，有限责任公司换成 DAO， 这些产生新一类金融应用程序的功能已经出场，将推动金融科技的新一轮创新。

因此，我们看到历史重演，金融科技如何能够在 Web3 的新浪潮中继续创造巨大价值。这篇文章将描绘 Web3 金融工具的前景，并说明为什么它会迅速成为创业和风险投资领域下一个激动人心的前沿。

Web3 金融工具版图

法币出入通道

随着 Web3 重新构想财务管理方式的新世界，法币出入通道是许多人用来跨界的桥梁。

它们提供了一种方便的方式，可以直接将银行账户中的法币转换为钱包中的加密货币，然后探索去中心化网络上的应用程序并与之交互。

法币出入通道是基础设施的基本组成部分，已迅速成为许多人使用的 Web3 中的第一个金融科技工具。

支付网关

加密货币已经稳固地确立了自己作为一种数字货币的形式，支付网关是用户使用它的门户。它们使商家能够接受加密货币作为一种支付方式，以换取向客户提供的商品和服务。

通过为买卖双方提供无缝交易的方式，支付网关是将商业从法定货币转移到加密货币的关键组成部分。

支付协议

通过移除中心化实体，用智能合约取而代之，支付协议是最纯粹形式的加密支付网关。它们使商家和客户之间能够进行真正无需许可的商业活动。

支付协议是中心化网关的分布式替代方案，可将商业从法定货币转移到加密货币。

NFT 结账

鉴于其固有的稀缺性，NFT 已成为数字经济中任何参与者都不可或缺的所有权凭证。

NFT 结账是买家使用信用卡或借记卡无摩擦购买 NFT 的一种方式。

NFT 可以代表从凭证到社区成员的所有内容。鉴于它们的重要性，NFT 结账为用户提供了一种更简单的获取方式。

工资和费用管理

随着 Web3 作为一个行业的不断成熟，工资和费用管理成为链上劳动力管理的关键组成部分。

它通过支付贡献者和承包商以及管理以加密货币计价的费用，来帮助链上组织简化其财务运营。

Web3 改变了工作方式，加密原生工资单和费用管理解决方案是帮助组织在这个新世界中运营的关键。

资金管理

具有共享银行账户的在线社区开始以 DAO 的形式扎根，财务管理试图回答如何管理这样一个账户的问题。

组织利用资金管理通过多元化或其他方式帮助管理其链上资产。

在线社区由资产的共同所有权定义，资金管理是确保其可持续性的先决条件。

支出管理

在越来越多公司向数字化转型的同时，法定货币不再是首选货币，支出管理必须以一种新的方式支持多币种。

该工具可帮助跨越 Web2 和 Web3 的链上组织管理他们在法定货币和加密货币方面的支出。

鉴于一个组织可以在任何给定时间持有多种加密货币，支出管理发挥着越来越重要的作用。

会计与报告

组织的财务活动必须在某个时候从区块链调和到现实世界，Web3 本地会计和报告是首选工具。

链上组织使用它们来跟踪从代币交换到 NFT 铸造的所有链上交易，跨多个区块链以确保合规性。

即使在一个新的数字世界中，参与者的交易也必须被尽职地跟踪和核算，因此，会计和报告成为必要。

税务

作为生活中的两个确定性之一（死亡和税收），即使在数字世界中，税收也是不可避免的。

消费者和企业都可以使用 Web3 本地税务软件来跟踪计算资本收益、所得税等。

鉴于加密货币多达数千种，以及诸如流动性挖矿等独特机制，链上税务软件是解决这一复杂性步骤的可行方案。

商业智能

鉴于向新计算范式的转变，链上组织催生了对链上商业智能的需求。

智能平台的功能是处理区块链数据以供 Web3 组织使用，以推动更高质量的决策。

公司进入链上世界需要可操作的洞察力，商业智能是呈现这些洞察力的必要工具。

Web3 中的金融科技

这幅版图代表了公司中的新兴类别，它们在将金融技术应用于 Web3 的过程中开始产生重大价值，但这仅仅是开始。

随着现有应用程序越来越不能满足他们的需求，我们将继续看到支持链上组织的工具进一步创新，新解决方案的一些示例领域包括：

KYC ：随着 KYC 越来越多地成为了解你的钱包 (KYW)，欧盟法规给中心化交易所带来了压力，以收集有关非托管钱包的信息。监管或将扩展到去中心化应用程序，如果某些钱包是已知的恶意行为者，将被阻止与 DApps 交互。我们已经看到 Tornado.Cash 发生这种情况，它使用 Chainalysis 预言机合约来阻止 OFAC 标记的地址。由于 DApps 被迫满足更高的监管标准，开发人员需要使用 KYC API 服务来集成到他们的应用程序中以处理这种监管负担，这样他们就可以专注于他们的核心产品。

订阅管理和计费 ：随着加密货币成为越来越受消费者欢迎的商业工具，当前的计费模式仅限于一次性支付，而基于数量和循环模式等替代方案尚不可行。特定于加密货币的订阅管理和计费工具将成为商家的解决方案，不仅使他们现有的商业模式能够从 Web2 迁移到 Web3，而且还有助于管理和协调法定和加密订阅数据的差异。

采购 ：去中心化组织的无许可参与是一种超能力，但会导致协调问题，特别是在工具采购方面。DAO 采购工具通常以非标准方式发生，组织内没有明确的责任利益相关者。鉴于我期望 DAO 工具服务于特定应用程序，即服务 DAO 所需的工具与投资 DAO 所需的工具不同；用于大型 DAO 的采购管理软件将成为帮助消除噪音，以确保其获得最佳工具的必需品。

随着金融活动继续在链上迁移，Web3 中的金融科技将变得越来越重要。这是有进取心的企业家和风险资本家开始思考这一新兴类别的起点。
