作者：加密研究员 Ann；编译：白泽研究院

清晨起床，没有什么比拿出新钱包并加入 Web3 社交应用程序 friend.tech 上的炒作更好的事情了。

当前关于 friend.tech 讨论都是炒作，虽然带火了 Base 的生态，但我不想体验。然而，为了获得 Web3 新闻报道所需的应用程序屏幕截图，我最终还是妥协了，使用了它。现在，我不仅得到了我想要的截图，而且关于 friend.tech 我还有很多话要说。

什么是 friend.tech？

friend.tech 是一个运行在 Coinbase 新链 Base 上的社交 DApp。

  • 它是由一位受欢迎的加密人士 0xRacer 制作的。此人是一名加密开发人员，没有 Rug 的历史，这是一个很好的迹象。

  • 这也是为什么尽管该应用程序相当侵犯隐私，但 Crypto Twitter 仍然非常宽容。（稍后将详细讨论它。）

  • “你__朋友的市场。” 每个人的个人资料都由代币（“份额”）代表。人们可以买卖这些代币。

  • 持有某人的代币使你有权访问该人的聊天室。

  • 每周五都会有一次空投，不是代币，而是积分。

如何在 friend.tech 上赚钱？

在我成功注册并获得自己的份额（你自己的第一个份额是免费的）后，我对这个 DApp 的第一印象是，一些其他人的账户似乎自动购买了我的份额。

两分钟之内，我的份额从 0 ETH 上涨到 0.005 ETH（约 8 美元）。

有些人一直吹嘘他们通过这个 DApp 赚了很多钱。我现在明白是怎么回事了。

这与 memecoin 以及 Telegram Bot 没有什么不同。你可以寻找那些新入驻的加密 KOL 并买入他们的份额，然后在价格上涨更多后卖出，就像在 memecoin 炒作中一样。

说实话，我真的不知道对此有何感受。

如何你不热衷于“赌博”其他人的份额，那么你还可以……赚取交易费。

在这些机器人进行购买份额的那一刻，多亏了它们，我赚到了一些 ETH。

然而，提高交易费的唯一方法就是变得更加受欢迎，这样很多用户会来交易你的份额。因此，顶级排行榜上的账户正在赚取大量交易费。

除了 friend.tech 带来的“疯狂投机”之外，我还有更多的事情要吐槽。

关于 friend.tech 的缺点

friend.tech 侵犯隐私且不友好

我其实想更早的尝试一下 friend.tech，但阻碍我的一件事是他们的隐私政策，“不存在，很快就会推出”。

在注册时，friend.tech 会要求你关联 Google 帐户或 Apple 帐户。然后，要创建个人资料，它又需要你链接 Twitter 帐户。也许这样做的目的是阻止女巫攻击（确保用户的真实性），但这种方法阻碍了许多注重隐私的加密用户。

到目前为止，临时解决方案是创建一次性的电子邮箱并使用非 KYC 的 Twitter 帐户。我就是这么做的。我也建立了一个新的钱包，并通过 CEX 充值了 ETH（避免钱包关联）。在注册后，我还立马撤销了Twitter 设置中的应用程序权限。

用了不久就会感到无聊，因为无事可做

我甚至对 friend.tech 本身的“赌博”活动并不生气。我的烦恼更多地来自于这样一个事实：除了“赌博”之外，没什么其他事情可做。

你可以与其他用户聊天，但前提是你必须拥有他们的份额（或者他们拥有你的份额）。Explore 页面显示最近的交易活动，而 Airdrop 页面显示你获得的积分和邀请码。除此之外，没有更有趣的活动可做。

最重要的是，friend.tech 是一个 Web 应用程序，因此它没有原生应用程序那么流畅。

希望这种情况是暂时的。因为…

我看到的潜力

friend.tech 的核心概念从本质上来说是相当愚蠢的。如果开发人员决定将该 DApp 提升到一个新的水平，也许我会看到巨大的潜力：

  • 最好先完善作为社交平台的基本功能，例如编辑个人资料（个人资料图片、个人简介）。

  • 如果想成为一个更成熟的链上社交平台，必须有添加外部网站链接（OpenSea 个人资料、ENS）的功能。

  • 增加发帖功能，用户可以在平台上发布他们的想法，就像使用 Twitter 一样；增加 DeFi 奖励机制，就像 DeBank 在他们的 DeBank Stream 中所做的那样；向参与者（无论是否是创造者）分配价值——这正是 Web2 社交媒体所缺乏的。

  • 也有成为内容创作者平台的巨大潜力。当普通内容创作者和影响者将其受众带入其中时，friend.tech 将成为 Web3 大规模采用的门户。

  • 与其他加密社交项目合作。在我看来，这比尝试自己创建一些新功能要快。

听起来工作量很大，尤其是对于独立开发者来说。但好消息是，他们获得了 Paradigm 的投资。希望他们能够充分利用种子轮资金，雇用更多开发者并添加更多功能（请先修复用户体验！）。

目前，friend.tech 的势头相当强劲。随着该项目不断发布重大活动和进展，他们应该知道如何维持热度，进一步增长和扩张。希望 friend.tech 不会沦落到目前粉丝代币型 SocialFi 人走茶凉的窘境之中。

如何使用 friend.tech？

首先，将一些 ETH 桥接至 Base。因为我是从其他 Layer2 跨链的，所以我使用了 Orbiter Finance。你也可以使用 Base 的官方跨链桥 https://bridge.base.org/deposit。

通过手机浏览器访问 https://www.friend.tech/，将应用程序下载至手机主屏幕。

开发者选择这种安装方法是为了绕过 App Store 和 Google Play Store 规则。

然后，按照说明进行操作，直到最终提示你收到份额。根据我的经验，如果资金充足的话，在机器人狙击你的份额并抬高价格之前，立即多购买一些份额。

该应用程序的访问仅限邀请，因此需要其他用户的邀请码，可在 Twitter 上找到。

结语

friend.tech 的想法无疑很酷，尽管它在功能上有一些不太完美的选择（无论如何，该应用程序仍处于测试阶段）。我希望他们雇用更多的开发人员，甚至是一个法律团队来修复隐私政策（并通过手机应用商店发布） 。

总的来说，也许我们都可以同意的结论是，friend.tech 是一个有趣的项目，其发展值得关注。
