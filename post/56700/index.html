Layer 2 在今年已成为最重要的赛道之一，它的发展将会使得 Web 3 网络在保持新生产关系，智能合约等优势后，以更高的效率和更低的费用来服务用户，作为底层设施影响未来行业发展的速度。近来，除了专门聚焦在 L 2 赛道的项目，交易所也在 L 2 赛道动作频出。Mantle，Base，opBNB 先后推出或宣布 L 2 主网时间，引发市场关注，本文我们将盘点一下交易所 L 2 的最新情况和其背后所意为何。

一、Base

由 Coinbase 支持，Base 是一个以太坊 L 2 网络，为任何人提供安全、低成本、开发人员友好的方式来构建去中心化应用程序或链上“dapps”，旨在将下一个 10 亿用户带入 Web 3 和加密经济。

1. 技术概述

Base 是第一个与 OP Stack 团队展开合作的，并成为 OP Stack 的核心开发人员，采用 Optimistic rollup，和 OP Stack 模块化的技术堆栈，二者在 2023 的技术路线图也是保持协同的。可以说 OP Stack 的所有优势都会在 Base 被中继承，如共享以太坊安全性、EVM 的等效性、当前 L 2 中最低的费用和优秀的可拓展性、OP Stack 的模块化结构等。关于 OP Stack 进一步的技术特点，可以回看我们此前的报告《展望坎昆升级，OP 是否值得“乐观”》。

2. 当前进展

（1）时间线

2023.2.23 推出 Base 测试网，宣传与 optimism 的合作和超级链构想 — — 2023.7.13 日主网向构建者开放 — — 宣布 2023.8.9 日主网将会向全网开放，并启动为期一个月 Onchain Summer 活动。

（2） 7 月末的 meme 热潮

在 7.13 主网刚开放后，链上并没有多少资金， 7.28 日官方向早期构建者宣传铸造 NFT 的链接，铸造条件是在 Base 主网部署智能合约， 7.30 日后 Base 上的 TVL 开始快速增长，两天之内 TVL 超到 800 万美金。

来源：Base 官方

由于链上没有多少应用推出，部署的智能合约主要都是没实际应用的 meme，配合资金的操作，以 bald 为代表的 meme 币在一天之内的涨幅千倍后暴跌归零，虽然绝大多数的 meme 都在闪亮登场，快速 Rug，但夸张的数据让更多的资金涌入了 base，并因为暂时没有官方跨链桥，大量资金都保持在链上炒作，这一波让 Base 主网向公众正式上线之前就带来了极高的市场热度。

3. 未来计划

（1）构建 web 3 开放入口的愿景

从 Coinbase 成立之初拥有一个四阶段计划，目标是创建一个开放的金融体系，以提高全球经济自由度。在过去的十年中，前三个阶段已经被完成，分别是：协议（*比特币、以太坊）、建立交易所（*Coinbase Exchange）作为数字货币的桥梁，以及创建一个去中心化应用程序或“dapps”（*Coinbase 和 Coinbase 钱包）的大众市场界面。当前，进行到了第四个阶段，过去 Coinbase 的主要服务都是基于 Web 2 技术构建的，在此阶段将通过构建以智能合约编写的链，作为承载各种加密原生的 dapps 的环境，在未来将 10 亿用户带入加密经济。

来源：Base

（2）技术路线图

2023 年的目标是与 OP Labs 和 Optimism Collective 合作，将 Base 和 Op Mainnet 从第 0 阶段 Roll up（*由 Vitalik 定义）推进到第 1 阶段 Roll up，并实现高度的去中心化和去信任化。

A. 上线故障证明：为 OP 堆栈启动至少一个故障证明，通过利用一组验证器来识别和挑战故障，从而保护 Base 和 Op Mainnet 的安全。

B. 推进去中心化：将 Base 和 Op Mainnet 智能合约的升级能力去中心化，首先交给一个安全委员会，该委员会的去中心化法定人数不受任何单一实体的控制。

C. 开始落地超级链：推出超级链的初始版本，可以对 Base、Op Mainnet 和任何 OP Stack 汇总的交易进行排序，保护任何人参与这些生态系统的权利。

2024 年的目标是努力将 Base 和 Optimism 主网推进到第 2 阶段，其去中心化和安全性水平可与以太坊相媲美。

4. 项目特点

（1）暂无代币计划

Base 在官方渠道明确表示，现在没有计划发行网络代币。（*这方面可能主要考虑在美国的合规监管，未来基于 Coinbase 与 SEC 的监管博弈之下或许会有新的变化）

（2）构建一条开放的链

Base 希望搭建一座桥，不是一座岛。将提供对以太坊 L 1、其他 L 2 以及 Solana 等其他 L 1 生态系统的访问。除了使 Base 能够与其他链互操作之外，也将在 Coinbase 产品中支持尽可能多的链。

（3）Coinbase 的生态联动

Coinbase 拥有近 1 亿用户和 1300 亿美元的资产，Base 的开发和互联互通，将把这些用户通过 Coinbase 带到链上世界，也希望更多新的用户从 Base 的链上入口来到 Coinbase。

二、Mantle

由 Bitdao，Bybit 支持，Mantle 网络是使用 Optimistic rollup 的以太坊 L 2 ，有庞大的 Mantle 金库加持，产品和计划路线图由代币持有者全权治理。

1. 技术概述

（1）模块化架构与数据可用性

与 OP Stack 类似，Mantle 也使用了模块化架构，在模块化区块链，操作的核心功能（*执行、共识、结算、数据可用性）是在专门的层上进行的。这样的好处是一定程度上可以缓解区块链不可能三角问题，在可拓展性上，资源分离可以提高网络效率，因为每一层都专门负责特定任务；安全性上，所有用户都以相同的安全级别进行操作。去中心化上，在不同层上运行的欺诈证明和 zk 证明等技术减少了节点上的整体执行和验证负载，不再需要执行所有交易来检查有效性。

基于模块化架构，Mantle 拥有了单独的数据可用层 DA，意味着 Mantle 可以提高吞吐量，而不会影响安全性或减少验证节点的负担。Mantle DA 由 EigenDA 技术提供支持。

（2）门限签名方案

Optimistic rollup 有一个限制，即以挑战起的形式对提款施加较长的等待时间，为了解决挑战期间交易验证的不确定性问题，Mantle 实现了以分布式签名方案来验证区块数据。这种分布式密钥生成与数字签名相结合的范例更普遍地称为门限签名方案 TSS，它使客户端网络的每一方都能生成有效的签名，并确保分布式签名数据的可验证性（*只要有足够多诚实的签名）。由于每个 TSS 客户端仅持有私钥的一部分，因此私钥也不再是单点故障。

通过使用门限签名方案（*TSS）技术，专用节点贡献多方签名，以提高链下交易的正确性。这使得提款的挑战期限缩短。不过，当前 TSS 节点都是 Mantle 核心贡献者授权的，即中心化的。

来源：Mantle

（3）改进的欺诈证明

在目前流行的欺诈证明模型中，解决纠纷的合约 — — 链上验证者 — — 只能执行 MIPS 或 WASM 等较低级别虚拟机中的指令。这迫使以太坊虚拟机（*EVM）客户端将其欺诈证明重新编译为较低级别的语言，以便链上验证者可以解释，意味着敗诈证明的内容是在 EVM 范围之外生成的。

Mantle 网络使用交互式敗诈证明机制建立交易有效性，并通过使用 EVM 级指令编译和验证欺诈证明（*即不需要低级转译，避免出现歧义）。

来源：Mantle 整体架构

三、BNB Chain-L 2 

由 BNB Chain，Binance 支持，基于 BSC 布局了两种 L 2 的解决方案，opBNB 和 zkBNB，以此拓展 BSC 链的性能，目前 opBNB 发展更快，opBNB 也采用的 OP Stack 技术。

1. 技术概述

BNB Chain 由两条区块链组成 — — BNB 信标链（*BC）和 BNB 智能链（*BSC）。BNB 信标链是负责 BNB 链治理并管理 BNB 链上的质押和投票的区块链组件。而 BNB 智能链是与 EVM 兼容、共识层且具有多链集线器的区块链组件。

来源：BNB Chain

zkBNB 和 opBNB 是 BSC 的二层扩容解决方案，与本文介绍的其他 L 2 不同的是，zkBNB 和 opBNB 是基于 BSC 为一层进行设计的，而非以太坊，虽然 BSC 的速度远高于以太坊，但是作为承载大量游戏的链，通过 L 2 进一步提升可拓展性依然是非常重要的。

zkBNB — zkBNB 是一个基于 zk-Rollup 架构为开发人员构建的基础设施，未来可帮助他们构建具有更高吞吐量和更低甚至零交易费用的大规模基于 BSC 的应用程序，目前处于测试网阶段。BNB Chain 高级解决方案架构师 Arnaud Bauer 上半年在接受 The Block 采访中表示，zkBNB 还不兼容 EVM，所以优先推出了 opBNB。

opBNB — opBNB 网络是 BNB 智能链的第 2 层扩展解决方案，由 OP Stack 提供技术支持,能够继承 BSC 的一层安全性和 EVM 兼容性的情况下实现可扩展性。（*OP Stack 内容此处不再展开）

2. 当前进展

（1）时间线

2023.2.14 BNBChain 公布 2023 技术路线图，将可拓展性设定为长期目标 — — 2023.6.19 宣布推出 opBNB 测试网 — — 2023.08.02 宣布 8 月中旬推出 opBNB 主网，将首先向基础设施提供商开放，并于 8 月底或 9 月初向公众开放（*zkBNB 原本作为今年可拓展性的重点，但暂时没有更多进展）

（2）链上数据

当前 ETH 的 TVL 为 BSC 的七倍，BSC 的交易量是 ETH 的 3 倍，总费用是 ETH 的 7.7% ，在 L 1 上 BSC 相比于 ETH 具有更低的费用消耗，而 ETH 具有最大规模的资金量。

来源：DefiLlama

在 L 2 上，opBNB 测试网平均每日处理量在 10 万-15 万左右，平均出块时间为 1 s，日活跃账户在 6000 – 20000 名左右，每笔转账的费用约为 0.005 美元。

来源：opBNB-Scan

3. 未来计划

BNBChain 在 2023 年的技术路线图将在五个方面进行推进：

（1）提升区块链网络整体性能：主要通过提升区块 gas 限制，实行并行 EVM，升级 BSC 节点和客户端等方案

（2）提升可拓展性：通过多链策略实现 BSC 的可拓展性，rollup 方案主要为 zkBNB 和 opBNB，当前 opBNB 进展更快，zkBNB 原计划也是今年推出。

（3）搭建新的基础设施：采用更多的基础设施，重点之一是去中心化存储，今年推出了 BNB Greenfield，用户和 dApp 可以在其中创建、存储和交换具有完全所有权的数据，形成新的数据经济，同时存储也是进一步提升网络性能的关键环节。

（4）提高去中心化：在 2023 年底将活跃验证者席位从 29 个增加到 100 个，并引入更多的共识机制和链上治理。

（5）提升 BNB 安全性：引入新的安全服务，安全和风险管理框架和提升跨链组件安全性。

来源：BNB Chain

4. 项目特点

（1）建立在 BSC 上的 L 2 ：BSC 在处理速度和费用消耗方面优于 ETH，通过 opBNB 的技术，将会通过 L 2 进一步提升性能。

（2）BNB Chain 生态系统齐全且成熟：Binance 是当前最大的加密货币交易所并拥有最多的用户和资金。BNB Chain 在自己的生态中搭建了各类基础设施和应用，可以灵活高效地适配 opBNB，充分发挥生态和代币的价值。

四、结语

今年下半年，L 2 的声音会越来越多，或许 L 2 Summer 将会到来。从技术层面来看，主流的 L 2 技术有所趋同，选择更具拓展性的模块化结构，optimistic rollup 鉴于落地性上被更多采用，OP Stack 获得了更多的外部合作。

交易所纷纷布局自己的 L 2 ，主要的原因或许是在已有的 CEX 资金和用户基础上，通过建设 Web 3 原生的区块链网络，承接更多新用户和新 dapp 的进入，打造更具多样性的业务结构和增量空间。未来，谁能通过 L 2 进一步巩固自身地位或改变发展的排位，让我们拭目以待。
