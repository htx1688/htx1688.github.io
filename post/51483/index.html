作者丨元宇宙简史主理人 Fun

【元宇宙导读】 在全球范围内，有一个国家对Web3游戏的兴趣和参与度远超其他国家，那就是菲律宾。菲律宾为什么对Web3游戏如此热衷？这背后有哪些因素推动了菲律宾人对这一新兴领域的热情呢？本文将从经济、社会、政策等多个角度进行分析，揭示菲律宾在Web3游戏行业中的关键作用和领先地位。

Web3游戏是一种基于区块链技术和加密货币的新型网络游戏，它们具有去中心化、自主性和所有权的特点，让玩家可以真正拥有和控制自己的游戏资产和数据。

Web3游戏的兴起，为全球数以亿计的游戏爱好者和开发者带来了新的机会。

CoinGecko最新的调查显示，菲律宾在对Web3游戏的兴趣方面再次夺冠，这是该国连续第三年位居榜首，展现了其在不断壮大的链游行业中的关键作用。尼日利亚紧随其后，排名第二，超越了现在居于第三位的中国大陆，而巴基斯坦则从前五名之外跃升至第四名。

目前，菲律宾已经形成了一个相对完善和活跃的Web3游戏生态系统，涵盖了开发者、玩家、投资者、社区等各个方面。

那么，为什么菲律宾对Web3游戏如此热衷呢？这背后有哪些因素推动了菲律宾人对这一新兴领域的热情参与呢？

0 1、 经济因素

菲律宾是一个发展中国家，其人均GDP在2022年为3623美元，低于世界平均水平，而且菲律宾是一个贫富差距非常大的国家。

根据世界银行的数据，2020年菲律宾的贫困率为16.6%，意味着有近1700万人生活在贫困线以下。2021年受新冠疫情影响，菲律宾经济出现了9.5%的负增长，失业率也达到了10.4%。虽然菲律宾政府采取了一系列措施来应对危机，2022年经济恢复增长7.6%，但在目前全球大环境内经济下行压力仍然非常大。

在这样的背景下，Web3游戏为菲律宾人提供了一种通过玩游戏赚钱的方式，让他们可以利用自己的闲暇时间来增加收入。

例如，Axie Infinity是一款基于以太坊区块链的虚拟宠物对战游戏，玩家可以通过培育、交易和战斗自己的Axie（类似精灵宝可梦）来赚取加密货币。

据相关报道，Axie Infinity在菲律宾拥有超过100万名玩家，其中许多人每月可以赚取相当于最低工资几倍甚至十几倍的收入。这对于许多没有工作或收入减少的菲律宾人来说，是一种改善生活质量和摆脱贫困的途径。

0 2 、 博彩业发达

菲律宾是一个博彩业合法化和发达的国家，其博彩业收入占GDP比重高达7.2%。

菲律宾不仅允许经营传统的赌场、彩票等形式的博彩业务，还开放了网络博彩业务，吸引了大量来自世界各国的客户和员工。

网络博彩业务主要是通过网络平台提供在线赌博服务，包括体育博彩、电子游戏、扑克牌等。网络博彩业务为菲律宾政府带来了巨额的税收和就业机会，也为Web3游戏创造了良好的市场环境和用户基础。

由于网络博彩业务与Web3游戏有着相似的特点和操作方式，例如都需要在网络上进行交易，都需要通过网络平台进行游戏等，因此菲律宾对网络博彩业务感兴趣的人群也容易对Web3游戏产生兴趣和参与度。

0 3 、 社会因素

菲律宾是一个拥有超过1亿人口，多元文化和多种语言的国家。菲律宾人热爱社交和娱乐，也喜欢尝试新鲜事物。

Web3游戏为他们提供了一个既能满足社交需求又能体验创新技术的平台，让他们可以与来自不同国家和地区、拥有相同兴趣和目标的人建立联系和友谊。

例如，CryptoKitties是一款基于以太坊区块链的虚拟猫咪收藏和繁殖游戏，玩家可以通过购买、出售、交换和繁殖自己的CryptoKitties（每只都是独一无二且不可复制的数字资产）来赚取加密货币。

据报道，CryptoKitties在菲律宾拥有超过10万名玩家，其中许多人通过参与社区活动、加入公会、分享经验和技巧等方式来增进彼此之间的互动和信任。

0 4 、 政策因素

菲律宾政府对Web3游戏持开放和支持的态度，为其发展提供了有利的政策环境。

菲律宾央行（BSP）在2017年发布了《虚拟货币交易所规定》，规范了加密货币交易所在菲律宾境内开展业务所需遵守的规则和标准，并鼓励加密货币交易所与BSP合作以促进金融创新和普惠金融。

此外，菲律宾经济区管理局（PEZA）也在积极探索区块链技术在经济特区内应用的可能性，并与多个区块链项目合作建立试点。这些政策举措为Web3游戏在菲律宾提供了合法性、安全性和便利性，降低了玩家和开发者参与Web3游戏所面临的风险和障碍。

综上所述，我们可以看到，菲律宾对Web3游戏的高度关注度和参与度并非偶然，而是由其经济、社会和政策等多方面因素共同推动和塑造的结果。
