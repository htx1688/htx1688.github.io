Web3 Game 是什么？

Web3 游戏是建立在区块链技术的去中心化、点对点结构上的游戏。

从游戏中赚钱的概念并不新鲜，但 Web3 游戏代表了用户真正对游戏生态系统有了所有权和控制权。玩家拥有游戏内的资产，这些资产以非同质化代币（NFT）或加密货币的形式完全属于他们，并可以自由地进行交易或出售。

Web3 游戏正处于快速发展的阶段，代表着游戏行业的未来。 虽然像开放性元宇宙这样的概念仍处于初级阶段，但 Web3 游戏已经展示出了强大的的潜力。随着 Web3 游戏从新兴市场向主流市场转变，我们有充分的理由对该行业的增长前景保持乐观态度。

Web3 游戏的玩家是谁？

Web3 Game Users & Projects

即使市场深陷寒冬，但 web3 游戏每天依然有八十万的用户。

由于有许多不同的交互方式可用于不同的目的，很难对 Web3 游戏的用户进行分类。有一些平台基于唯一钱包地址的数量来做决定，但这种措施忽略了对游戏的积极参与程度和原因的评估。

Web3 游戏的用户参与到游戏之中有各种各样的理由，包括真正的玩家寻求放松、投资者希望赚钱、机器人账户操纵系统等等。为了评估 Web3 游戏的健康状况和潜力，了解不同领域用户的动机和行为很重要。

此外，Web3 游戏玩家通常比普通游戏玩家更年轻、受教育程度更高、技术上更精通。普通游戏玩家的平均年龄为 35 岁，而 Web3 游戏玩家的平均年龄在 25 至 30 岁之间。

不同类型的 Web3 游戏及其玩家

早期Web3 游戏：游戏外壳与 DeFi 的结合

早期的 Web3 游戏是从 DeFi 发展而来的，将 DeFi 的资产配置原则与游戏元素相结合。这些 Web3 游戏主要是将 DeFi 原理直接放入游戏中，基本没有真正完善的游戏系统。

Axie Infinity的用户主要关注盈利。

早期 Web3 游戏的主要目标是吸引对快速获利感兴趣的用户，而不是创造真正沉浸式的游戏体验。因此，游戏设计和图形缺乏传统游戏的深度和复杂性，从而导致用户体验较差。

Axie Infinity Daily Active Users

然而，目前据 Footprint Analytics 数据，Axie Infinity已经进入停滞期，每日活跃用户量仅约为9,000人，而在其高峰期间则有数百万活跃用户。

Axie Infinity Weekly Retention Rate

用户留存率是用户分析中的重要指标，它显示在特定阶段有多少用户依然在使用产品。 Axie Infinity 的主要用户群体关注的是赚钱，而不是游戏本身。这创造了一种 Play to Earn 的模式，玩家主要受潜在的财务收益所驱动，而不是游戏的乐趣，导致市场风口过后的留存率较低，约为10％。

开发成功的 Web3 游戏生态，需要超越 DeFi 的形态，探索将创造性地整合到游戏中的方法。开发独特而复杂的游戏经济，使用区块链技术的创新变现模型，并将用户体验作为游戏设计的优先考虑都是破局之路。

尽管如此，早期的 Web3 游戏也引入了一些创新的游戏机制，例如繁殖和战斗，尽管与传统游戏相比设计仍然相对基础，但也让人们看出了 Web3 游戏经过基础设计之后成为一个成功的生态系统的潜力。

X2E 阶段：注重叙事和现实意义

X2E 产品常常将有意义的现实生活活动或创意行为作为核心游戏机制。与早期阶段完全以盈利为驱动的游戏不同，X2E 非常重视社区文化和价值观。例如，STEPN 不仅通过体育运动促进健康生活方式，还通过 STEPN 基金会为慈善活动贡献。

STEPN整合了现实生活的叙事情节，吸引了更多用户。

STEPN 将现实生活的行为变成游戏的一部分，游戏机制更加用户友好，有更大的长期参与潜力。游戏设计鼓励用户参加体育活动，达成特定目标，并通过 STEPN 应用程序提交完成证明。游戏奖励用户的努力，激励他们继续保持健康活动的习惯。

STEPN Weekly Retention Rate

在 2022 年 3 月的活跃期间，STEPN 的周留存率能够达到 30% 以上。这是因为 STEPN 非常重视建立社区文化和价值观，吸引了有强烈社交属性和互动需求的用户。

为了增加游戏的渗透率，STEPN 需要进一步改进其游戏机制和用户界面，创造更具吸引力和互动性的体验。通过提供更有意义和具有奖励性的游戏体验，STEPN 可以培养用户的忠诚度，助力建立更强大的用户群。

STEPN Active User

然而，球鞋价格的显著上涨导致了抛售压力的激增，如果没有足够多的新用户进入生态系统，可能会让 STEPN 陷入风险。

AAA 游戏：强调游戏性和用户基础

AAA 游戏以高质量的图像、丰富的内容和卓越的玩家体验闻名，这些因素都可以显著提高 Web3 游戏的质量。 AAA 游戏工作室和资本的参与还可以为 Web3 游戏带来宝贵的资源和专业经验，进一步推动其各项指标的增长。

Illuvium 旨在通过其丰富的游戏背景和卓越的游戏机制长期运营。

Illuvium 是一种高级别的Web3 游戏项目，具有产品质量和用户体验方面固有的优势。Illuvium 花费了很大力气创建了令人惊叹的 3D 环境和模型，以迎合玩家的需求，其控制系统也非常优秀。游戏提供了各种 PVE 和 PVP 游戏玩法选项。所有这些因素结合在一起，使Illuvium成为一个高度吸引人且长期吸引用户的游戏。

Illuvium Active User Trend

尽管据 Footprint Analytics 数据，Illuvium在熊市中的表现要好于其他游戏，但该平台仍然保持着相对较低的日活跃用户水平（数百名用户）。这可能是因为该平台的总用户人数较少，而Web3 游戏中只有很少的用户长期活跃。

Illuvium Weekly Retention Analysis

此外，为了解决低留存率的问题，Illuvium 需要优化其玩家回归机制。仅凭该平台高质量的的游戏图像和强大的团队背景无法留住用户。通过增强用户参与度措施和改善用户体验， Illuvium 可以进一步提高其在市场中的知名度和影响力，吸引更多喜欢更沉浸式游戏体验的新用户。

未来开放的元宇宙：构建虚拟世界，塑造未来

根据 Report Linker 的报告，全球元宇宙游戏市场预计将在 2027 年达到 4478 亿美元，从 2020 年到 2027 年的年复合增长率（CAGR）为15.3%。这种增长将受到一些因素的推动，例如互联网普及率的提高、智能手机和移动游戏的普及以及虚拟现实和虚拟世界越来越受欢迎。

元宇宙和虚拟世界代表了区块链技术和 Web3 游戏 发展的必然趋势和方向。它将为用户创造一个持久和开放的数字空间，使他们能够体验各种娱乐和社交活动。这无疑将成为 Web3 时代用户数字生活的重要组成部分。

然而，目前，元宇宙和虚拟世界仍处于概念阶段。尽管早期的产品如 Decentraland 和 Sandbox 已经推出，但在开放性、社区活动和商业化程度方面仍有很大的改进空间。这需要社区和项目方不断地优化和丰富内容，以吸引更多的用户和资本进入。

未来我们可以期待

Web3 游戏 1.0 证明了 Web2 和 Web3 玩家属性之间的差异。项目需要通过内部循环吸收前期的投机行为，而不仅仅依靠外部循环模式。

目前，大多数 Web3 游戏的可玩性仍然不足，并且技术还没有反映出区块链的优势。为了构建一个过渡性的 Web3 游戏模型，我们需要从 Web3 用户和经济模型的角度出发。

未来，Web3 游戏需要共识和更多优质项目来共建行业生态系统。随着不同的项目发展，需要建立标准化的协议和框架，以使游戏开发人员更有效地实现跨平台交易、游戏组合和交互。

Web3 游戏的未来需要更多研究游戏内容、游戏玩法和生态系统架构。强大的背景对项目的发展有利，而维持社区共识对延长游戏生命周期更是至关重要。
