什么是 ERC-6551

ERC-6551 允许以太坊上的每个 NFT 都拥有自己的钱包，很多人不太理解这个意味着什么。

简而言之，用户现在不仅可以连接钱包，还可以连接自己的 NFT。

这个功能的解锁，有什么好处么？下面我们来举一个例子进行说明。

假设你现在拥有一个Mutant NFT，并且你通过这个 NFT，在游戏中收集一些物品，同时你还持有代币 ApeCoin。

使用 ERC-6551，所有收集到的物品和代币，现在都可以绑定到 Mutant NFT 本身，而不是与你的钱包相绑定。

虽然只是换了一个绑定目标，但是这在以前是无法实现的。

NFT 升级与去中心化消息传递

想象一下，你加入了一个 DAO 组织，作为这个组织的积极参与者，你平时收到的 POAP 或 SBT 这些奖励，统统都可以存在你的 DAO 会员卡里。

DAO 会员卡本身就是一个 NFT，你的持续参与和贡献，会使得你的 NFT 更有价值，即在不更改 NFT 元数据的情况下，去升级我们的 NFT。

你还可以将任何 NFT 或 ERC-20 代币添加到你的 PFP 帐户，然后把这个 PFP 和 AI 连接，让 AI 帮你去进行交易。

你也可以在不知道用户推特的情况下，直接向 PFP 发送消息，ERC-6551 允许项目集成去中心化的消息传递。

NFT 历史与贷款

同时，你的 PFP 现在也可以显示交易历史记录了，例如你的 Doodle 是否被用来贷款过、是否领过空投，或者是否曾经被名人持有过等等。

当 NFT 本身拥有一个钱包的时候，它的交易历史就永远不会丢失。

ERC-6551 还可以降低贷款风险，假设你以价值 10ETH 的 Mutant 作为抵押，去贷款 5 ETH。当 Mutant 地板价下跌时，你可以向 Mutant 本身发送额外的 ETH 以防止清算。

NFT 组合与不可替代性

ERC-6551 的采用，将会使得你的 NFT 更加独一无二，不可替代性大大提高。

第一个真实的例子是就是 @jeffstaple 的 Sapienz。

你可以将 Jeff 的不同 NFT 发送到 Sapienz 的钱包中，就像给已有的 NFT 进行穿衣打扮一样。

原有的 NFT 将会与新的 NFT 结合，产生不同的样式与背景，这会让你的 NFT 看起来更加独一无二，并且该功能目前已经在 Opensea 上运行。

目前的局限性

ERC-6551 仍然是一个很新的协议，在经过更多测试之前，用户可能并不是都愿意在一些昂贵的 NFT 上使用这个协议。

处于试验阶段的 ERC-6551，对于很多功能，可能仍然需要几个月的时间才能开发出来。

ERC-6551 是 NFT 的下一个创新，也是将 PFP 转变为链上身份的一大步，它将允许 NFT 单独记录自己的历史，而不是通过钱包去记录这些。

这为 NFT 的交易、在游戏上的更多应用、溯源和治理等方面的内容，开辟了更多新的可能性。
