目前DeFi最热门的领域之一是DeFi和NFT的交叉点，也被称为NFTfi。

在当前的 NFTfi 生态中，已经有五花八门的品类。

目前最大的是 NFT 市场和流动性协议，如 OpenSea、LooksRare、Zora、NFTX 和 Sudoswap。然而，还有 NFT 衍生品项目、NFT 基础设施项目、NFT 定价项目等等。人们正在探索 NFT 的前沿，因此这些领域中的每一个都在快速增长，但目前市场之外最大的 NFTfi 类别是NFT 借贷协议 。 也就是说，NFT 借贷项目有不同的形式和规模。

本文将展示如何使用该领域的顶级项目来借贷NFT。

NFTfi

NFTfi 是 NFT 抵押贷款的 P2P 市场。 

换句话说，该项目让借款人提出预期的 NFT 贷款参数，然后加密货币贷方可以选择在个人对个人的基础上接受这些条款。 

这种方法的优势在于它可以让您完全自定义和选择加入您的 NFT 贷款参数。不利的一面是，如果有的话，可能需要很长时间才能找到愿意接受您贷款另一方的人。

如何在 NFTfi 上借贷

  1. 转到app.nftfi.com/lend/assets并连接你的钱包

  2. 单击感兴趣的 NFT 贷款提议

  3. 要继续，请单击“提出要约”按钮并选择接受借款人的提议条款或自定义并请求自己的独特条款（金额、还款、时间表等）

  4. 接下来单击“Grant”按钮并使用您的钱包批准 NFTfi 使用你的资金（这是一次性交易）

  5. 然后用你的钱包敲定你的贷款报价。在提议到期之前，你的交易对手有 7 天的时间考虑你的报价。

如何在 NFTfi 上借款

  1. 转到app.nftfi.com/borrow/assets并连接你的钱包 

  2. 使用提供的过滤器在你的钱包中找到你想要作为抵押品的 NFT

  3. 使用随后的界面指定你想要的贷款金额、贷款时间表、利率等。 

  4. 然后使用“List as Collat eral”按钮完成列表

  5. 等待从接受你的条款的贷方那里得到贷款金额！ 

BendDAO

BendDAO 是一种使用点对池方法的 NFT 借贷协议。 

换句话说，BendDAO 的存款人将 ETH 提供到流动资金池中以赚取利息，然后项目的借款人可以通过这些资金池获得即时的 NFT 支持的贷款。 

如何在BendDAO上贷款

  1. 转到benddao.xyz并连接你的钱包

  2. 签署交易以验证你的地址

  3. 单击“流动性”选项卡，然后单击“存入 ETH”按钮 

  4. 输入您要存入的 ETH 数量，然后按“存入”按钮

  5. 用你的钱包完成交易，然后坐下来等待你赚取收益——目前的贷款利率是 8.5% APR 

如何在 BendDAO 上借款

  1. 转到benddao.xyz/liquidity/batch-borrow并连接钱包

  2. BendDAO 目前接受 Azuki、BAYC、CloneX、CryptoPunks、Doodles 和 MAYC NFT 的存款。如果你有其中之一并想要继续，请按“Deposit”NFT 按钮

  3. 启动两个批准交易以开始，一个批准债务代币，一个批准 NFT

  4. 输入想要的借入金额，然后按“Borrow ETH”

  5. 然后，可以根据需要通过 BendDAO 的借款仪表板偿还贷款

NFTfi才刚刚开始

当比特币和第一个加密货币出现时，你可以将它们从地址 A 发送到地址 B，依此类推。

智能合约平台以太坊的到来使数字代币具有可组合性、表现力，并且能够以各种不同的方式使用。 

我们在 NFT 中看到了从简单到高级的类似轨迹。起初，你基本上可以买卖 NFT，虽然这是一个开始，但这只是一切可能的开始。

DeFi 的 NFTfi 领域的兴起开始为各种更高级的用例铺平道路，例如 NFT 衍生品、NFT 贷款、NFT 定价协议等。 

如今，像 NFTfi 和 BendDAO 这样的 NFT 借贷项目让人们更容易用他们的 ETH 获得资金，或者通过提供 ETH 来代替 NFT 来赚钱。

这些是该领域的早期先驱，但我预计未来几年我们会看到更多的 NFT 借贷协议出现，并且它们将越来越用户友好。现在熟悉这个空间可以为未来带来红利。
