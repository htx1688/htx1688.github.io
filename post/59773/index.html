历史上，九月是加密货币领域最糟糕的一个月之一，到目前为止，这个月也没有证明会有什么不同，它正在摧毁参与这个市场的人们所剩无几的希望。越来越多的市场参与者正在离开，现在看起来只剩下了最后一个熊市中的同一批参与者以及一批关键的新参与者。散户的参与度完全消失，加密寒冬真正接近了。

现在，这是这里最好的时机，可以利用这些机会，在未来为你带来数倍的回报，但在那之前需要很多耐心。

闲话少说，让我们看看本周的头条。

市场摘要

期待已久的协议 Gammaswap 终于在 Arbitrum 主网上启动；

Daniele Sesta 推出了他的新代币 WAGMI；

治理提案通过后，ARB 激励正式上线；

Mark Cuban 的钱包被窃取 87 万美元；

Optimism 给委托人第二次空投，并在私募中售出 1.19 亿枚 OP 代币；

TON 进入加密市场市值前十名（但没人谈论它）；

随着货币危机加深，尼日利亚转向加密货币；

SEC 宣布他们计划打击更多加密平台。

跨链桥流动性

Arbitrum 明确成为这里的赢家并不令人惊讶，过去一周有 8300 万美元桥接到了该生态系统，而同样数额的 8400 万美元已从主网离开。

DEX 交易量

让我明确一点，尽管发生了板块轮动，但交易量远未回升。事实上，它继续萎缩。这里是值得采用的当前市场哲学：赚还是被套。如果你获利了，拿着你的钱走人。目前没有必要积极交易。唯一值得考虑的是那些可能已经触底的代币，以及如果你擅长这样做，可以进行链上狙击。其余的都是链上鲨鱼的食物。

交易对交易量

代币之间的交易活动日益减少，当 LINK 开始获得重大竞价并被拉升时，这通常是开始出货的提示，这次也是如此。尽管价格行动遭受重创，但 RLB 继续受到很多关注。人们正在寻找拉升过后的 Meme 币，FINE 就是最近一个例子。

由于发生了一些短线压制，上周 Optimism 出现了漂亮的反弹，现在又回归了往常，且随着基金会出售代币而出现了另一个空头催化剂。它恢复需要一段时间。现在所有目光都集中在它的直接竞争对手上。

Arbitrum 的交易量还不高，因为激励机制刚刚启动。我预计这会在未来一周内有所拉升，随着流动性挖矿奖励开始在生态系统中分配。

NFT 交易量

从今年的表现来看，Milady NFT 曾是表现最强的 NFT，但它现在是跌幅最严重影响的 NFT。

NFT 熊市继续加剧。你认为什么将是再次复苏 NFT 的催化剂？我个人认为将是游戏，但很乐意听取其他意见。

净流入

人们再次将目光投向老 DeFi，CRV 和 MKR 在资金流入方面处于领先地位。尽管风险投资公司和 Vitalik 抛售了该代币，但 MKR 的表现仍然很强，因为他们有一个最可持续的代币模型。这是一个明确的力量信号。看起来人们也愿意赌 Aave 会修复他们的 GHO 挂钩，如果是这样的话，这会给他们一个 3%的免费套利。

Optimism 上没那么多动静，尽管在其私人代币销售后收到了负面情绪，但 OP 还是收到了某种竞价。WLD 是上周受到市场欢迎的代币，Andrew Kang 是支持者，并公开认可该代币是一种值得购买的良好山寨币。

有趣的是，此表明确显示了人们在提案即将通过时在 Arbitrum 上竞相购买的内容。看到 ARB 和 GMX 领先一点也不奇怪，因为它是生态系统的原生代币和蓝筹协议，占据了最大的份额。RDNT 以第三名的成绩紧随其后，PENDLE 位居第四。这个榜单上唯一的惊讶是没有看到 MAGIC，因为所有其他大市值协议都进入了这里，而 degen 选择 WINR 作为高 beta 的低市值游戏。DPX 是一种逆势押注，相对于其规模，人们对它的出价较小，尽管催化剂可能对其有利。
