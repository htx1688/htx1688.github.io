GameStop NFT Marketplace 推出后立竿见影，但它是否符合用户的期望？

争议、起起落落、被分析师完全排除、卷土重来、重拾低迷，这些都构成了世界著名游戏配件商店 GameStop 的背景故事。

似乎是过山车般的股价暴跌，粉丝从尘埃中挑选公司的名字，以最终实现区块链技术作为游戏的奖励，让 GameStop 在 2022 年的中心舞台上占有一席之地。

GameStop 的 NFT 市场推出后立即成功，交易量超过 1,831 ETH（约 270 万美元），首日交易费用为 45,000 美元。这个数字意味着该平台在第一天就超过了 Coinbase 7.4 万美元的 NFT 市场销售额。

尽管在市场萎缩的情况下似乎立竿见影，但是，GameStop NFT Marketplace 是否辜负了用户的期望？本文是对加密领域最新 NFT 市场的深入回顾。

什么是 GameStop NFT 市场？

GameStop NFT Marketplace 是一个去中心化平台，允许用户创建、购买和出售不可替代代币 (NFT)。

GameStop 总部位于德克萨斯州格雷普韦恩，是美国领先的消费电子产品、视频游戏和游戏商品零售商。该公司经营范围广泛的游戏系统、配件和游戏机，包括 PlayStation、Xbox 和任天堂游戏，并且还提供以旧换新的服务。

随着数字视频游戏销量的增加，甚至索尼也推出了PlayStation 5，GameStop 决定进军加密领域，目前正指望它来增加收入。2021 年，该公司出现重大亏损（2021 年第四季度净亏损1.475 亿美元），甚至解雇了一些工人。

2022 年 2 月，GameStop 宣布计划开放其 NFT 市场，以进军不断增长的新兴行业，从而扩大公司的收入。市场终于在 7 月 11 日公开测试版中为NFT空间增光添彩。

GameStop NFT 市场的参与者可以购买、出售或创作数字艺术、收藏品、摄影、体育纪念品和其他创作。目前该市场上 NFT 的创建仅对选定的创建者开放。有兴趣成为平台创作者的人可以申请成为该市场上的NFT创作者。基于 NFT 的交易平台目前正在寻求扩大用户参与度，因此希望引入新的NFT创作者。

GameStop NFT 市场拥有 345 个列出的集合，其中包含超过 57,000 个 NFT。其最畅销的 NFT 系列是 Metaboy——一组总数10,000 个的动画 GIF。

成立不到一个月，GameStop NFT 市场的销售额已超过 720 万美元，每笔交易的交易费（特许权使用费）为 2.25%，收入超过 100,000 美元。GameStop NFT 市场在大约 250 个集合中拥有超过 54,000 个不同的 NFT。

GameStop NFT 市场如何运作

GameStop NFT 市场是在Loopring网络上开发的——以太坊区块链上的第 2 层 (L2) 扩展解决方案。多亏了 Loopring 协议，市场提供了高速的交易和低gas费。以太坊最近由于网络拥塞增加导致许多用户承受额外费用，交易费用已经失控。

Loopring 协议可以在 GameStop NFT 市场上每秒处理数千笔交易，同时具有成本效益的 gas 费。将 Loopring 想象成一种总线，它将市场上的所有这些交易批处理成一个大块并将其发送到以太坊。这使得交易更快、更便宜。

NFT Marketplace 仍处于早期测试阶段，目前仅支持 Loopring。它本身不支持以太坊，这意味着你不能交易以太坊上的 NFT。相反，你只能购买、出售或交易在 Loopring 上铸造的 NFT。因此，这个市场并不支持所有基于以太坊的高价值蓝筹 NFT。考虑到大多数NFT都在以太坊区块链上，这是 GameStop NFT 市场的巨大限制。

除了 Loopring，这个市场还与 Immutable X 合作，在市场上提供更多的游戏 NFT。Immutable X 是另一个与 Loopring 非常相似的以太坊层，但主要支持加密游戏 NFT，因此是公司的理想合作伙伴。带有 Immutable X 的 GameStop 目前正在为建设者和创造者提供 1 亿美元的赠款，以使用加密货币构建事物并使用市场。

GameStop NFT 市场布局

GameStop NFT Marketplace 拥有用户友好的 UI/UX 设计。布局质量高，易于浏览。主页可让你访问市场、在主要市场页面排名靠前的特色创作者。该页面以带有黑白色调背景为主。主页上有两个用于探索 NFT 市场和快速铸造（创建）NFT 的选项，但后一个选项仅适用于选定的用户。

该页面还与一个非常简单且信息丰富的集成统计页面。统计页面根据交易量对列出的 NFT 进行排名。GameStop 的市场仍处于测试阶段，因此缺少在其他 NFT 市场（如 OpenSea）中可以看到的一些功能。例如，它缺少用户可以查看 NFT 集合的交易数据和销售数据的功能。

市场功能

虽然仍处于 Beta 阶段，但 GameStop NFT 市场包含大量功能，使其成为加密领域中最好的 NFT 市场之一。该平台上的一些显着功能包括：

1.服务

GameStop NFT Marketplace 促进了来自全球社区、创作者和全球品牌的独特收藏品、艺术品等的购买、销售、创作（铸造）和收藏。这些 NFT 可以在收集者之间进行交易和转移。

用户可以使用基于类别、价格或集合的不同搜索和过滤选项浏览大量 NFT。用户还可以申请成为平台上的特色创作者，铸造自己的 NFT，设置创作者费用，并在每次出售 NFT 时赚取收入。

2.区块链

GameStop NFT 市场是在 Loopring 网络上开发的——以太坊区块链上的第 2 层（L2）扩展解决方案。市场还支持不可变 L2，以促进市场中更多游戏 NFT 的集成。该平台是一个自我托管的市场，这意味着用户是其数字资产的真正所有者。

3.支持的钱包

目前，GameStop NFT 市场仅支持三个钱包。这些是：

  • Wallet Connect

  • GameStop Wallet

  • MetaMask

由于 GameStop 钱包仍处于测试阶段，因此在使用市场时强烈建议使用 MetaMask，谨防欺诈。

4.付款方式

GameStop 目前支持加密货币支付，特别是以太坊 (ETH)。要进行支付，首先需要将 ETH 从以太坊网络转移到 GameStop NFT Loopring Layer 2。该平台不支持以美元等法定货币支付。

5.费用

GameStop NFT Marketplace 对平台上的每笔交易收取 2.5% 的费用。该费用是包含在 NFT 的售价中的，这意味着买家在购买他们最喜欢的 NFT 时无需支付任何额外费用。

市场上的创作者可以灵活地设置他们认为合适的创作者费用。然后，他们将在该特定 NFT 的每次销售中获得创作者费用。此外，他们可以随时调整费用。

6.可以在 GameStop NFT 市场上购买的 NFT

GameStop NFT Marketplace 拥有超过 250 个 NFT 集合，其中包含近 55,000 个不同的 NFT，其中一些有多个限量版。您可以在 GameStop 上购买的一些 NFT 包括：

  • 收藏品

  • 艺术

  • 电视剧

  • 音乐

  • 交易卡

  • 运动体育

  • 摄影

  • 效用

  • Sticker

  • 卡牌

NFT 集合仍在增长，未来几个月内会有更多类别。

GameStop NFT 值得吗？

GameStop 是游戏领域的领导者，拥有大量好评如潮。该公司最近通过推出加密钱包和 GameStop NFT 市场进入了加密领域。虽然市场仍处于 Beta 阶段，但尽管 NFT 销售量大幅下降，但它的推出取得了巨大成功。该平台具有以下优点和缺点：

GameStop NFT 的优点

  • 多个类别的大量 NFT

  • 强大的安全措施

  • 使用平台没有最低账户要求 – 对所有人开放。

  • 无买家费用

  • 便于使用

  • 相对快速和便宜的交易费用

  • 非托管市场

缺点

  • 它仅支持 ETH 支付，不支持法定货币

  • 由于它是基于 Loopring 构建的，因此只能购买、出售或交易在 Loopring 上铸造的 NFT。因此，无法处理大多数基于以太坊的高价值蓝筹 NFT。

  • 它不提供蓝筹 NFT 集合。

  • 有兴趣的 NFT 创作者必须申请成为精选的 NFT 创作者。

  • 低特许权使用费

结束语

GameStop NFT 市场成立不到一个月，尽管 NFT 整体销量下降，但它在销量方面取得了巨大成功。该平台的周销售额达到 720 万美元，是其竞争对手 Coinbase NFT 市场历史上 NFT 销售额的两倍。得益于 Loopring L2 网络，该市场以其直观的用户界面和相对较快的交易速度脱颖而出。

与其他 NFT 市场相比，所有销售交易的 2.25% 费用具有竞争力。但是，平台仅支持 ETH 支付，并将市场上连接的钱包数量限制为三个。总的来说，这是一个很棒的 NFT 市场，特别是如果你是一个对交易和收集各种 NFT 收藏感兴趣的狂热游戏玩家。
