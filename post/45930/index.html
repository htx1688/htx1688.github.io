原文：《从Loot仿盘一路狂奔并成功跑偏，TreasureDAO的MLP策略也许是链游的正确演进方向》

作者：北辰

从Loot仿盘一路狂奔并成功跑偏，TreasureDAO的MLP策略也许是链游的正确演进方向。

Treasure DAO 是区块链游戏领域的现象级项目，当我们直接去审视它，难免会被它丰富甚至庞杂的生态干扰判断——TreasureDAO 到底是啥？

链茶馆按照时间线梳理了 TreasureDAO 的迭代之路，我们可以清晰地发现「它从哪儿来」，以及「它要到哪儿去」，从而对「它是谁」有更准确的判断。

起步：Loot 仿盘

一切得先从 2021 年 8 月份横空出世的 Loot 说起。

Loot 是一个由 8000 个 NFT 构成的系列，NFT 是 8 个短语组成的 TXT 文件，每个短语相当于一个随机生成的属性（如武器、胸甲、头甲、项链、戒指等），具有不同的稀有度。

这个只有 8 行文字的甚至没有前端接口的 NFT 项目，让人不明觉厉，发布了几个小时之后就出现了持有者专属社群，很快从 free mint 暴涨到 20 ETH 的地板价。这种看不懂但又上头的情绪直到 2022 年 10 月份的 XEN 重新出现。

Loot 的持有者可以自由地以任何方式去解释及使用它们，完全颠覆了大家对于 NFT、游戏乃至元宇宙的想象，可以说开启了自下而上建构元宇宙的新范式。

基于 Loot 出现了一系列的衍生项目，有的是把 Loot 的文本给可视化，有的是从 8001 开始生成新的 Loot NFT，还有的是借鉴了 Loot 的思路，但另起炉灶重新搞一组短语作为 NFT，Treasure 就是这个思路。

作为 Loot 的衍生项目，Treasure 的 NFT 走的也是哥特式的神秘黑暗那一路风格，只不过短语是金币、银便士、绿宝石、珍珠、贝壳、龙尾等历史上以及传说中的货币。

Treasure 的 NFT 共 10000 枚，前 90% 是 free mint，后面的 7.5% 空投给了 Loot、xLoot、The N Project 和 AGLD 社区（总之都是 Loot 生态），2.5% 由团队控制。

此时的 Treasure 和其它 Loot 仿盘没有任何区别，除了 Loot 具有原创性，仿盘再怎么运营也运营不起来（现在地板价仅 0.049 ETH）。

Treasure 官方似乎也在极力撇清 Treasure NFT 的历史——早期的项目 Twitter 已注销，官网也没有提及 Treasure NFT，现在一切都是以 TreasureDAO 的身份在运营。

曲线救国：链游 + DeFi

仿盘注定没什么情况，但 Treasure 能走到今天，是因为在发行了 NFT 之后，团队紧接着就发行了 MAGIC 代币，总量为 3.5 亿，挖矿激励逐年减半，并为 MAGIC 量身定制了 DeFi 项目——MAGIC Farm。

这个逻辑有点跳跃，请耐心理解……

MAGIC Farm 很好理解，就是用户存入 Treasure 的 NFT，或者 Loot 的 NFT、AGLD 代币，然后赚取 MAGIC 代币。只是，MAGIC 代币是什么？

为了给 Treasure 的 NFT 系列赋能，Treasure 团队宣布要开发一款链游 Legions，就是重新推出一个 PFP 系列，并把它做成一款 PFP vs PFP 的游戏（玩法可参照 Tiny World），而 MAGIC 就是游戏代币，可以制作工具以及升级道具。

从 Treasure NFT 到 Legions

但是链游 Legions 是如何为 Treasure 的 NFT 系列赋能的？前面提到，往 MAGIC Farm 存入 Treasure 的 NFT 就可以赚取 MAGIC 代币，因此间接增加了 Treasure 的 NFT 的价值……

这个曲线救国的反射弧实在太绕，而且成败取决于链游 Legions 能否爆发，从后来 Treasure 的 NFT 基本凉了的现状来看，曲线救国的初心完全失败，但是它误打误撞地开辟了新的根据地——MAGIC 生态。

抛弃初心：MAGIC 成为核心

MAGIC Farm 推出不到一个月，Treasure 团队就充分感受到了 MAGIC 的潜力——超过 40% 的 The N Project 和 10% 的 Loot 都被锁在 MAGIC Farm 里，而且产出的 MAGIC 也有 60% 又被复投到 MAGIC Farm 里。

既然市场对 MAGIC 代币的兴趣远高于 Treasure 的 NFT，那就干脆抛弃 MAGIC 给作为 Loot 仿盘的 Treasure NFT 系列赋能的初心，从此一切都围绕着给 MAGIC 赋能。

10 月 6 日，MAGIC Farm 从以太坊迁徙到 Arbitrum 上，Treasure 团队特别推出了激励方案——每质押 10000 个 MAGIC 并拥有 1 个 Treasure 的 NFT，将获得一个普通军团，每 100 个普通军团将空投一个特殊类别的军团（刺客、战士、攻城、施法者和远程）。

此时 Treasure 社区初具规模，不仅有社区成员捐款，而且还有自发前来的开发者、设计师等作为社区志愿者协助团队执行任务。

注意，MAGIC 的价值在于它是链游 Legions 的平台币，但此时还没有影子。

赋能失利：Life NFT

一个月的时间根本开发不出链游 Legions，这意味着 MAGIC 此时还是空气币，既然 Treasure 要给 MAGIC 赋能，就要给出新的预期。

Treasure 团队联合 Jumpman 推出了另一款链游 Life NFT，这是一个简化版的《模拟人生》，赋能 MAGIC 的方式是用户只有质押 MAGIC 才能让自己的虚拟人物活着，否则 48 小时后直接去世。

但 Life NFT 后来也不了了之，Jumpman 另外搞了 NFT 数据分析平台 Origins NFT，Mirana Ventures 领投了 400 万美元的种子轮。

话说链游 Legions 直到 2022 年 5 月份才正式推出，被放在了一个更宏大的叙事里——Bridgeworld。

另立中央：Treasure DAO

MAGIC 仍在持续释放，尽管 Treasure 生态啥也没有，却有着最宝贵的东西——社区共识，尤其是对 MAGIC 的币价预期的共识。

Treasure 团队在 11 月份正式推出了 Treasure DAO，不仅向这个 DAO 移交了团队的权力（MAGIC 的分配方案、生态内孵化哪些项目等），而且还指定 DAO 成为 MAGIC 激励的唯一接收者。DAO 由质押 MAGIC 的持有者通过投票来管理。

从此 Treasure 正式围绕着给 MAGIC 赋能而展开建构，这是团队在当初推出 Loot 仿盘时所始料未及的。

但话说回来，Treasure DAO 的叙事并不新鲜，当时几乎所有的链游平台都有着相同的叙事模板——发行平台币，然后不断创建链游（早期由团队自己创建，后期直接开放），而用户玩这些链游则需要消耗平台币，并最终形成一个庞大的链游生态系统。

但目前来看，只有 Treasure DAO 成功了。

不仅是因为 Treasure DAO 后来持续推出了成功的小游戏，因为说实话，那一系列小游戏总能找到更早推出以及更优秀的竞品（仅就产品本身而言），所以其实更重要的是它的社区运营得非常好，于是只有 Treasure DAO 起飞了。

裂变：生态之上建生态

Treasure DAO 成立之后，在 2022 年正式开启了它作为链游平台的叙事——不断创建链游，以及吸纳外部链游或其它项目进来，这些场景都需要消耗平台币 MAGIC，并最终形成一个以 MAGIC 为核心的生态系统，或者说是去中心化的任天堂。

前面提到链游版《模拟人生》Life NFT 就是这个思路的产物，但它是 Treasure DAO 成立之前启动的。

Treasure DAO 成立之后，自己不断推出独立的小项目来增加 MAGIC 的使用场景。

首先是把最初计划的链游 Legions 升级成 Bridgeworld。Legions 之于 Bridgeworld，基本相当于 Tiny Kingdom 之于 Tiny World，也就是说，Bridgeworld 不是 MAGIC 生态系统里的单款游戏，它自身也是一个可以不断扩张的生态系统。

其次是推出了另外一个生态系统 Smolverse，里面有 Smol Bodies、Smol Cars、Smol Brains 等一系列 NFT 小游戏。如 Smol Brains 是猴子 NFT，但它们的智商会随着 staking 而不断增长，从而解锁更多任务。

另外，在游戏之外还推出了对标 OpenSea 的 NFT 交易平台 Trove 和 Smolville。

除了自己推出小游戏外，还积极拓展外部的链游进入 MAGIC 生态。例如和 Strider 团队合作，建立 3A 级的 NFT 和链游工作室。

Treasure DAO 仍在不断推出新的链游，后来的故事就不必再赘述，总之像滚雪球一样不断吸引外部的用户和资金进入 MAGIC 生态，或者至少维持了已经进入生态的玩家沉淀下来，而不是随着熊市而流失。

总结：社区文化 +MLP 策略

我们现在可以总结一下 Treasure DAO 的成功路径。

说实话，他们的游戏及其他产品从产品力上来说并没有很出彩（相对于竞品而言），但 Treasure DAO 成功了，而竞品的表现则远不及它，我认为有两个关键因素。

首先在于天然具有社区文化。

Treasure DAO 是从 Loot 仿盘起家，Loot 这个实验型的项目快速凝聚了 crypto 行业敢于尝新者的共识（主要是注意力以及资金），作为仿盘的 Treasure 顺势在核心社区内以激励空气币的方式来引流，从而积累了这部分核心用户成为早期贡献者。

这种社区文化近似于玄学，但真实存在，那就是只有社区内部的成员才能够敏锐地捕捉到情绪，并给出能够引起共鸣的方案（靠不靠谱则另说）。

但投资人以及创业者似乎更认可的路径是：作为外部的成功者，在看到新兴社区的潜力后，觉得这一套很简陋，于是试图进来成为主导力量。所以我们会看到链游投资有不少 3A 大作，我猜测他们大概率是带着傲慢进来，最后会因为摸不着链游真正的社区文化而走向告吹。

值得一提的是，Treasure DAO 的创始团队有 Osmosis 的增长主管以及代币经济学 / 增长策略师，可以说是 crypto 旁氏（中性词）的专家，所以最初起盘的时候并没有借助资本的力量，仅靠少量的自筹资金就起飞了。

其次在于明确的 MLP 策略。

MLP 即 Minimum Lovable Product，可以理解为最小讨喜产品，这种策略放到链游领域就是不断推出「有点可玩性但不多」的小游戏，不断吸引新用户和资金进来，Treasure DAO 做到了每月至少发布一款链游的节奏（实际上远不止）。

反观从外部进来的认为链游这一套很简陋的成功者，他们的策略往往是兜售一个梦幻的产品故事（尤其是以 3A 大作居多），代价就是需要漫长的时间才能问世，因此需要资本长期输血。

甚至即使是链游 native 的开发者，也往往倾向于推出自己的产品，这就限制了发布节奏，而 Treasure DAO 则在自研的同时，也选择与外部直接合作，从而完成经济价值累积循环。
