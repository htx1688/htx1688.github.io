近日，Hooked Protocol 宣布与 Web3 知名投资机构 Animoca Brands 战略合作，合作内容提及将通过创新的教育娱乐内容和寓教于乐的用户体验来促进 Web3 的大规模破圈 ，定位 Web2 数十亿互联网用户群体，为 Web3 行业注入更多的新鲜血液。

从与 Animoca 的合作看 Hooked 的勃勃野心

这次与 Animoca 的战略合作，不仅让 Hooked 这个项目拥有更多的想象和机会，也展示了在 Web3 教育赛道的实力地位和野心。

Animoca 更多被我们熟知的是一家 Web3 顶级投资机构，比如投出了元宇宙赛道龙头项目-The Sanbox；其实 Animoca 也是一家全球性的数字娱乐、区块链和游戏公司，开发一系列知名IP系列游戏。能够与这种行业龙头机构合作，从某种程度来说 Hooked 的实力和其所在做的事情也得到了 Animoca 的充分认可与肯定。

通过与 Animoca 的合作，Hooked有机会将自己的技术、产品和理念推向更广阔的市场，与多个知名游戏品牌与 IP 进行深度合作，进一步丰富和优化教育内容，为用户提供更多的选择。相信在不久的将来， Hooked 将与 Animoca 共同构建的合作生态系统将得到更广泛的应用。

其次，此次合作也充分展示了 Hooked 在 Web3 教育领域的布局。Hooked 在其CambridgeUpgrade 项目升级后，已经形成了 Infra-Academy-Ecosystem 三位一体的布局。Infrastructure 转化用户，Academy 教育用户，Ecosystem 沉淀用户，三者相互促进，构建一个 Web3 技术与交互相结合的大流量教育应用生态系统，这个飞轮是完完全全可以转起来的。

与 Animoca 的合作就是一个典型案例，不妨大胆想象一下，此次合作基于双方的产品生态为用户带来更多的用户场景，为项目带来更多的落地案例，并能够为生态里的合作伙伴带来可持续的用户增长。

值得一提的是，这次合作还意味着 Hooked将进一步深化其在 Web3 教育生态系统中的布局。

按照行业周期理论，目前 Web3 教育赛道处于成长期，发展到成熟期还需要一定时间。但项目护城河的建立一定是从早期开始的，目前 Hooked 所做的一切似乎都可以看作在该领域进行跑马圈地打造自己的护城河，但放大到整个行业来看，Hooked 和其他该赛道的项目所做的事情都是具备行业长期价值和影响的，正是有了这些持续 building 做事的项目，行业赛道才能不断被推动和完善。教育是一件难而正确的事，Web3 教育更是如此，任重而道远。

Hooked Protocol 是什么样的项目？

Hooked 是一个 Web3 教育项目，通过提供沉浸式、游戏化和社交学习体验帮助更多圈外用戶进入Web3。作为获得币安和红杉的种子轮投资并且是 Binance 第29个 Launchpad 的项目，Hooked 的基本面其实是非常不错的。

项目方 build 能力： 官方愿景 “帮助下一个数十亿人进入Web3”，听起来似乎有点夸大其词；但 Hooked 从去年至今交付了3款 dAppapp教育产品，所以并非画大饼，反而是名副其实。

从去年 Wild Cash 的 quiz2earn 活动，到 ToDaMoon 直播学习再到最近结合 AI Chat 的产品Hooked Academy Sensei，能看出 Hooked 不仅持续深耕 Web3 教育赛道，而且产品交付能力也非常优秀。

市场感知力 ：一个优质的项目，不仅得有艺术家实力（产品交付能力），还得有企业家实力（市场获取能力）。2023 年最火的概念无疑是 AI 方面的 ChatGPT，Hooked 与时俱进，通过结合 AI 热点快速响应，推出 AI Sensei 这款具备类似 ChatGPT 功能的 Web3 AI 学习产品，说明项目方市场感知敏锐，用市场的话来说就是项目方会搞事。

这次与行业顶级机构 Animoca 的合作，落实了此前项目方深度升级后的全新布局，预计 Hooked 将不断在与更多行业内的优质机构/项目进行积极联动合作，扩大自行业影响力，拓宽自身宽度，全方位发展。

Web3 教育赛道能爆发吗？

Web3 教育赛道是一定具备爆发潜力，原因主要可以分为 3 个方面：

1. 机构是赛道的风向标

目前，有许多顶级投资机构已经注意到了 Web3 教育赛道的潜力并开始加大对于该领域的投资力度，不难看出，Animoca 与 Hooked 的这次合作，是其在对这一赛道的强势加码，目前已经陆续在布局该赛道机构还有如币安、DCG、A16z、红杉等。

2.Web3 教育的三板斧

随着区块链技术的不断发展，Web3教育可以提供更加沉浸式、游戏化和社交化的学习体验。在 Web3 教育中，用户可以通过虚拟现实、增强现实等技术获得更加沉浸式的学习体验；同时，通过游戏化和社交化的设计，可以提高学习者的学习兴趣和积极性。另外，结合 Learn & Earn 的概念，学习者会不断通过优秀的经济模型在获取 Web3 知识的同时也能赚取一些经济上的收入。

3.Web3 教育的破圈效应

相对于数十亿用户的 Web2，Web3 的用户体量还非常小，但也意味着还有极大的发展空间，获取圈外流量已经成为市场津津乐道的话题。但 Web2 用户进入 Web3 的门槛和难度都不小，最好的解决方案就是 Web3 教育。

对于 Web2 用户来说，进入 Web3 有门槛；对于 Web3 非教育项目来说，用户心智培养教育成本也很大。如果Web3教育能够爆发并日趋成熟，对于用户对于项目甚至是对于整个 Web3行业来说都是利大于弊的，可以说 Web3 教育就是链接 Web2 和 Web3 的桥梁，这也就是Web3 教育赛道破圈效应。

Hooked 作为这个赛道的先发者，在产品、生态等布局方面已先人一步，这次与Animoca 的深度合作，Hooked 是否能够充分带动一波又一波的 Web2 流量进入到 Web3，形成大规模的破圈效应？这一点，我们拭目以待。

总之，通过解读这次合作，不难看出 Hooked在 Web3 教育市场方面的布局、实力和野心。此次合作一方面为双方打开更广阔市场和深化在 Web3 教育领域的布局提供了重要的机会，另一方面也对整个 Web3行业起到注入新鲜血液和推动行业发展的生态作用。相信在不久的将来，像 Hooked这种在 Web3 教育持续深耕持续 building 并对行业起到积极推动作用的项目，在赛道进入爆发期时成为行业内一颗颗闪耀的新星，似乎是水到渠成的事。
