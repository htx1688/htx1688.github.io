众所周知，最近一段时间比特币上的NFT Ordinals大火。很多市场人士分析认为，作为比特币智能合约L2的Stacks将是这波BTC NFT热的受益者。

现在这把大火终于烧到Stacks了。Stacks代币STX在从18日的0.32美元一路暴涨至20日的0.74美元，2天之内翻倍。虽然后来目前有所回落，截止发稿，STX价格仍达0.62美元。

STX如此涨幅甚至惊动了Stacks联合创始人muneeb.btc，他20日在推特发文亲自解释什么是Stacks。

muneeb.btc：这是一篇关于比特币L2 Stacks的文章 。什么是Stacks L2？

1、Stacks 是由一群比特币建设者发起的开源项目。它背后的开发人员在构建比特币 L1 应用程序和协议方面拥有丰富的经验。2017 年，在区块大小大战之后，很明显，在比特币上扩容交易或新用例的唯一方法是通过比特币层。闪电交易和Stacks方面的工作开始了。

2、Stacks 是用于智能合约的比特币 L2层。它有一个单独的账本来存储比特币 L1 之外的数据，开发人员可以在其上构建任何应用程序。无论你能在以太坊、Solana 上构建什么，你都可以在 Stacks L2s 上构建。Stacks最新白皮书： http://stacks.co/stacks.pdf

3、Stacks 的初始版本于2021年初上线，它启用了 (a) Clarity，一种安全的编程语言 ( http://clarity-lang.org )，(b) 读取比特币状态的能力，以及 (c) BTC 和任何L2 资产的原子交换（参见http://magic.fun）。初始版本也有一些瓶颈。之后是中本聪版本的发布。

4、今年晚些时候推出的下一个版本增加了 (a) 一个去中心化的 BTC 挂钩，可以轻松地将 BTC 资金进出 Stacks L2，(b) 安全升级，其中 100% 比特币算力决定最终确定性，以及 (c) 更快的速度（不再受限于 10 个区块比特币时间）。

5、去中心化的 BTC 挂钩工作正在由开源开发者积极构建。目标是释放 5000亿 美元的 BTC 资金。sBTC 可以为比特币 DeFi 做些什么，就像 Ordinals 为比特币 NFT 做的一样。

6、Stacks 与 Lightning 相比如何？Lightning 和 Stacks 是互相赞赏的比特币 L2。Lightning 是一个用于支付的 P2P 网络；Stacks 更像是一个 L2 链，用于智能合约完全表达。LNSwap.org可以进行LN和Stacks资产的原子交换。

7、为什么有一个单独的STX代币？STX 是第一个获得 SEC 授权发行的。该项目在主网上线之前去中心化，现在有 30 多个独立实体和其他开发者。STX 用作 (a) 对矿工的激励和 (b) 对 sBTC 挂钩签名者的激励；保持挖矿和签名去中心化与使用联盟。sBTC 推出后，用户只需使用 sBTC 即可使用大多数钱包和应用程序。

8、Stacks L2 上有 NFT 吗？Stacks L2 上有一个活跃的艺术家和创作者社区。人们已经在 Stacks L2 上铸造了 65 万个比特币 NFT。所有这些 NFT 都自动哈希到比特币 L1，并以可扩展的方式由比特币保护。此外，许多 NFT 构建者都在使用 Ordinals 的比特币 L1 上活跃，例如，trygamma、satoshibles等。

9、Ordinals与 L2 NFT？比特币 L1 上的Ordinals与 L2 上的比特币 NFT 相辅相成，例如 Stacks。Ordinals对 L1 规模有自然限制，而 L2 提供了明确的可扩展性路径。像xverseApp和hirowallet这样的钱包是最早发布或处理 Ordinals 的。

10、Stacks L2 上有 DeFi 吗？stacking合约中锁定了 2.5 亿美元，人们可以从中获得 BTC 奖励。已经有 2,200 BTC 奖励给人们。可以查看alexlab btc和Arkadiko Finance这样的应用。与比特币 NFT 类似，比特币 DeFi 还处于早期阶段。

11、这是一个非常去中心化的生态系统，我自己也难以掌握一切。要了解更多信息可以看stacks官网：http://stacks.co
