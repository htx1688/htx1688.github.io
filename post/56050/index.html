Mantle 是构建在以太坊之上的模块化 Layer 2 Rollups 解决方案，它提供 EVM 兼容性、超可扩展性和低交易费用，并具有从以太坊主网承接的安全性。

近日，Mantle 宣布正式上线主网 Alpha 版，表明了 Mantle 网络已经完成了测试网的评估测试阶段，将步入下一个阶段，让我们一起看看模块化 L2 Mantle 此次上线有什么亮点吧。

Mantle 生态

目前 Mantle Ringwood 测试网上有 83 个正在运行的项目（未来还会有更多项目），一些测试网指标包括：

-1280 万笔交易

-67 k 智能合约

-29 k 开发者

-90 万个钱包

而经过测试网测试后的主网 Alpha 能给用户和开发者带来：

-改进后的用户体验

-快速确认交易

-低 Gas 费

-可扩展性和高吞吐量

-对于构建者来说，所有必要的工具都已经可用

MNT token 实用性

MNT token 将为 Mantle 网络提供：

-原生 Gas token

-多方计算（MPC）验证器节点质押 token

-数据可用性层（DA）节点质押 token

-生态系统增长 token

-去中心化 sequencer 节点质押 token

-用于未来的 Mantle 产品

目前已上线 Mantle 主网 Alpha 的项目

DeFi

iZUMi Finance

iZUMi Finance 是多链 DeFi 协议，提供一站式流动性即服务 (LaaS）。该项目旨在使每个 token 都以高效、持久的方式获得更好的链上流动性。iZUMi 目前推出了 LiquidBox、iZiSwap 和 iUSD 三种链上流动性产品。

iZiSwap 最初于 BNB Chain 上推出，其亮点是一种创新的 DL-AMM，具体参考：「iZUMi Finance：基于 UniSwap V3 思路改进，开创 DL-AMM 新机制」；LiquidBox 是基于 DA-AMM LP NFT 和 Uniswap V3 LP NFT 的流动性挖矿方案；iUSD 是 iZUMi 推出的美元计价的可转换债券，利用智能合约和多重签名钱包管理的透明链上资金作为超额抵押品来支撑 iUSD 的价值。

FusionX

FusionX 是 Mantle 网络上的原生 DEX，目前仅支持 Mantle 主网和测试网，也是 Mantle 网络上第一个社区驱动的 AMM，该 DEX 目前支持的 token 并不多，其功能都有：Swapping、Liquidity Pools、Yield Farming 和 Initial Farm Offerings。前三个功能都比较常见，最后一个 IFO 可能是大多 DEX 没有的，该平台用户可以通过质押特定 token 获取新 token 得优惠购买资格。

Timeswap

Timeswap 是一种基于 AMM 的去中心化货币市场协议，不使用预言机或清算人。与可以实时交换资产的 Uniswap 不同，在 Timeswap 上借贷需要交换代币，直到还款完成。贷款人提供资产 A 用于借款，同时也为借款人「保管」一定数量的资产 B 作为抵押品。用户可以调整他们的风险状况以获得更高的利率和更低的抵押率，反之亦然。

Reax

Reax 是 Mantle 上无需信任的合成资产发行和交易协议，允许用户能够在链上进行无滑点的交易。也是 Mantle 网络上较新的项目，Reax 通过共享债务池提供无需信任的合成资产创建，专注于提供低风险、低费用的合成资产框架，以最大资本效率扩大 DeFi 的用例。Reax 致力于将 TradFi 和跨链资产（BTC、XRP、ADA）带入 DeFi，旨在为各类投资者提供一个不受任何限制/监管的交易和推测资产价格变动的平台。Reax 还提供广泛且多样化的资产，例如外币、股票、大宗商品、跨链加密货币等，无需依赖桥或托管。通过购买合成资产，用户可以猜测这些资产的价格走势，而无需实际拥有它们。Reax 上交易变得简单，五花点，atomic swap 不会造成无常损失，合成资产保证金交易的杠杆率可高达 125 倍。

Demex

Demex 是由 Carbon 支持的跨链衍生品 DEX，旨在支持多种类型的金融资产，从现货、期货、永续合约和货币市场到期权、债券和合成资产等加密货币衍生品和复杂金融工具的交易。其亮点是：在 Demex 上从任何 IBC、EVM 和非 EVM 链上列出任何 token 可以无需许可地创建任何市场；Demex 的出块时间为 1.9 秒，比以太坊快 400 倍；Demex 上的 Amplified pools 提供了更深度的流动性。此外，Demex 可以通过智能合约将 GLP 桥接到 Cosmos，并提供自动复利服务，允许 Cosmos 用户访问 GMX 并从 GLP 中获得收益。

Symbiosis

Symbiosis Finance 是一个去中心化的多链流动性协议。目前上线了其 V2 版本，它允许用户在不同链（EVM 与非 EVM）之间无缝地流动资产, 实现跨链流动性。Symbiosis Finance 的主要组成部分有：

Symbiosis Bridge：这是连接不同链之间的桥，实现跨链资产转移。

Liquidity Pool：流动性池子，里面有不同资产组成的流动性对。用户可以向池子里添加流动性, 赚取交易费用。

SIS token：Symbiosis Finance 的治理代币，持有者可以通过 Staking 赚取收益, 也可以用于治理协议。

Oracle：这提供了外部数据给协议，比如不同链上资产的兑换率等。

Symbiosis Finance 的优势有四点：完全去中心化、互操作性、非托管和无限制跨链流动性。

DefiLlama

DefiLlama 是大家常用的链上数据分析工具，就不过多介绍了。

Infra

Covalent

Covalent 是一种去中心化的数据基础设施，使用户能够无缝访问否则无法访问的区块链数据。它索引了整个区块链历史，并以标准化和可互操作的方式转换数据。用户可以使用统一的 API 查询给定区块链上的几乎任何数据。

RedStone Oracles

RedStone 是一种跨链数据预言机技术，可提供快速、经济高效的数据访问、完整的历史审计跟踪和保险支持的去中心化争议机制。在数据提供不准确的情况下，它为用户提供了追索权。该项目使用 Arweave 区块链来存储数据和跟踪声誉。

OpenBlock Wallet

OpenBlock Wallet 是一个无助记词、无私钥的非托管 MPC 钱包，解决了链上钱包最大的痛点，助记词和私钥的管理，降低了进入 Web 3 用户的门槛。OpenBlock 使用 MPC（安全多方计算）技术来实现阈值签名方案（TSS），可以完成分布式私钥生成和交易签名，其优点是私钥分布式、安全性高、匿名性高和更低的复杂性。

Holograph

Holograph 是 LayerZero 生态全链 NFT 协议，可以在该项目是 mint NFT 后跨到 EVM 兼容链上，也是近期 LayerZero 刷交互的重点项目之一。

Thirdweb

Thirdweb 是一个全面的开发框架，用户能够跨任何 EVM 兼容的区块链无缝构建、启动和管理 web3 应用和游戏。无论是从头开始开发 dApps 还是增强现有应用，Thirdweb 工具套件都可以加速用户跨智能合约、应用程序和基础设施层的开发工作流程。

Blast API

Blast 是 Bware Labs 的 API 提供商平台，旨在通过采用地理上分布的第三方节点来解决与可靠性和延迟相关的 Web 3 基础设施问题。Blast 提供标准化 API 服务，涵盖 Web3 开发的所有基础设施方面。对于每个可支持的链，用户能够生成一个专用端点，使他们能够获取 dAPP 开发和/或区块链交互所需的大多数 RPC 方法。

Coin 98 Super App

Coin 98 Super App 是一款内置 DEX、跨桥工具、dApp 浏览器、多链 NFT 且非托管的加密货币钱包。

LayerZero

LayerZero 就不过多介绍了，火爆已久的全链互操作协议。

Gaming

Good Games Guild

Good Games Guild 属于老牌的 NFT 游戏公会，此前曾获得 Animoca Brands 战略投资，该平台专注于 P2E，致力于最大限度地提高游戏在人类生活中的收益，Good Games 愿景是通过游戏增强并加速 Web3 行业的潜力。目前该公会在全球拥有超 300, 000 社区，也推出了 token GGG。

XBorg

Xborg 是一个开放且可组合的游戏协议，游戏、品牌和社区可以在其上构建应用并与玩家和粉丝产生互动协作凭证的基础设施，专注于电子竞技代币化，可创建由粉丝和玩家管理的去中心化电竞游戏平台，未来可以将收益代币化并在社区内分享，还推出了 X borg Launchpad，給用户提供了投资早期游戏的机会。

NFT

NFTs 2 Me

NFTs 2 Me 是一个 NFT 创建平台，可以允许用户在零编码技能的前提下免费创建自己的 NFT 合集。该平台的设计器创建 NFT 集合，定义其元数据，上传资产，设置可选价格并创建/部署合约（ERC 721 或 ERC 1155）。此外，该平台还提供自定义 mint 页面和一些 NFT 创作者所需的其它小组件。

Rarible

Rarible 是一个聚合的、以社区为中心的、多链 NFT 市场。更多参考：「全面解析开源跨链 NFT 协议 Rarible Protocol」

Mintpad

Mintpad 与 NFTs 2 Me 类似，也是一个可创建 NFT 收藏的平台。首先，用户可以描述自己的 NFT 合集并选择相应的链，确定 mint 时间和日期，还有一个选项可以设置白名单阶段（只需上传钱包列表）；然后上传相关艺术品，该平台的 NFT 生成器会自动生成相关数据，最后生成一个 mint 网站以便收藏者 mint 该 NFT 系列。

结语

Mantle 独特的去中心化 sequencer 机制，消除对单个 sequencer 的依赖，以避免单点故障或审查；Mantel 也通过 Rollups 从以太坊获得安全性，此外，通过利用 EigenLayer 的 DA，使得 Mantle 执行层的数据可用性上也保持着与以太坊相同的安全性。

作为一个模块化 L2，相对于其他 L2（Starknet、zkSync 等），Mantle 可能不被看好，但它是第一个利用 EigenLayer DA（数据可用性层）的模块化 L2，且其测试网数据并不是特别差，测试网期间构建了超 80 个项目，主网 Alpha 启动以来，不断涌入更多项目，还将在未来推出 LSD。所以，Mantle 可能会是被过于低估的 L2。
