如何唤醒亚洲 NFT 市场的潜力

NFT 市场当前处在变革期，传统的 PFP 玩法难以被用户长期接受，新型的场景在探索当中。尽管市场相比于去年下旬最低谷的时候有所回暖，但包括 Doodles、Moonbirds、CloneX 在内的多个蓝筹项目跌幅严重；依然坚挺的蓝筹项目还很多，如：BAYC、CryptoPunks、Azuki 等。

毫无疑问，为什么这些蓝筹项目依然能够保持这么强的竞争力？

数据来源：NFTGo.io

最核心的原因是，Holders 的属性。以 Azuki 举例，当时 NFT 项目没有优质的东方美学项目，Azuki 日系动漫风捕获了大量亚洲投资者的共鸣。经过当时的数据统计，10-20点的时区有大资金流入 Azuki ，也就是说相同时区的很多用户是达成了共识的。甚至很多亚洲用户的第一个nft就是 Azuki，Azuki 的价值背后也有共识在。当然，BAYC 和 CryptoPunks 的很多钻石持有者也都是亚洲用户，甚至在各个活跃的中文社群也都不乏有这些蓝筹的持有者。

当然，这也可以看出，亚洲用户在 NFT 市场是非常活跃的，亚洲市场在 NFT 市场所占比例也并不小。

对此，我们也不禁思考：是否太多项目方对亚洲市场过度有失关注？亚洲市场的潜力究竟如何？

相比于纯粹的交易来说，有哪些方式能将 NFT 的影响力扩大？纯 PFP 项目已经过时，我们可以通过哪些方式把数字资产的价值发挥到最大化？

以及，如果日漫风的 Azuki 能够被这么多人所喜爱，具有东方特色的一些 IP 是不是也会有大量受众？

我们逐个剖析这些问题。

首先，可以十分确信的是，亚洲市场的发展潜力是巨大的。从宏观角度来看，亚洲整体的发达程度和互联网普及程度不如欧美那么先进，但亚洲各国在互联网等先进技术发展到一定规模时依然会引入并加大利好，以此来引导民众去建设，有点“摸着石头过河”的意味。对于 Web3 和 NFT 来说亦是如此，亚洲的多数群体还未曾接触过真正的 Web3，一旦有机会大量的用户基数会涌进来。

而随着香港的首发性放开，也标志着亚洲 Web3 的流量入口很可能从香港开始打开，重视这块的 Web3 团队将有很大的机会抢占先机。 当前香港有非常大的战略优势（更好的资产安全、出入⾦体验等）、环境优势（诸多 Web3 Builder 聚集）以及流量优势（关注度聚焦），背靠香港，不管是做事、结算还是出海等都会更加便利。

这里我们可以聚焦一下 Ubox (ubox.io) ，⼀个基于东方文化视野的新型 NFT 交易场景。Ubox 团队成员以香港土著为主，深耕香港多年；因此毫无疑问具备香港资源、地域、文化等多方面优势。以及我们可以观察到，很少有像 Ubox 一样，愿意在 NFT 亚洲市场现在的关注度和影响力不低迷的情况下，依旧聚焦亚洲加密领域，并优先去做贴合亚洲用户的使用习惯交互和内容场景的团队。随着Web3 的普及，香港战略和嘉年华等优势下，亚洲市场会有更多新鲜用户流入到 Web3 ；而香港很有可能就是大量亚洲用户的根据地， Ubox 在内多个重视亚太区域的 Web3 应用则是大量亚洲用户的入口。

其次，我们都知道纯 PFP 时代的 NFT 已经很难被认可，这就意味着，行业需要更丰富的加密内容来扩大 NFT 的影响力，发挥 NFT 除了艺术和 PFP 之外的其他价值，发掘除去转手套利之外其他的交易可能性。Ubox 的简介里有个词很吸引人——“交易场景”。如何去理解交易场景？

NFT 以文创和数字艺术起家，但这并不会是它的最终形态——它本质是一种数字资产，更像是现在 Web3 的入口钥匙；因此 NFT 交易场景我们也可以理解为 NFT 资产交互场景。具体点哪些算交易场景，或是 Ubox 在内的应用该如何去构建更多的交易场景？

  • 优质的 GameFi 应用，用户体验游玩，在过程中希望能够将自己的道具、装备进行交易置换。

  • 去中心化、更能保护用户隐私的 SocialFi，用户在里面聊天、社交、转账（互换数字资产）等。

  • 在线上线下开展 NFT 的知识普及，引导更多 Web2ers 进入 Web3，购买 NFT。

  • 构建元宇宙，让用户有更加沉浸式的数字资产（展示、交互、映射、赠送等）体验。

能够构建出更多交易场景的，不光会对行业产生贡献，也会大量获客。而 Ubox 比较重视资产体验这块，着重提到会围绕 NFT 生态构建了更广泛的交易场景，尤其是多样化的资产交互场景，以此来提供多元化及高附加值的交易体验及交互体验。多线程做这些可能会有难度，但更关键的是如何通过自己的 IP 和优质运营留住用户，以及能够意识到“构建交易场景”的 Web3 Dapp 本身就会有在 NFT 2.0 阶段起跑线有所领先。

最重要的是，Ubox 还提到一个词——“创建你的数字绿洲”。“数字绿洲”一词会有很大的想象空间，毕竟数字资产和用户是 Web3 最核心的要素之一，而绿洲可以理解为这两块的 Web3 舒适栖息地；能够通过运营同时抓住这两部分的 Web3 应用，其发展会是一种盛况。

最后，我们回到对 Azuki 的思考，它的本质是什么？NFT？PFP？共识？或许这些都有，但更重要的是，随着发展它俨然成为 Web3 世界的一个知名 IP，围绕这个 IP 项目方能做太多的事情。很喜欢米哈游的一句话：“我们不是做游戏的，是做 IP 的。”NFT 最能吸引人的，也是它 IP 所能带来的稀缺性价值、共鸣，而不仅仅是一张图片。

当前市场着重于东方特色的 NFT 还有所空缺（Azuki 算是一种，但更偏日漫），如果能有团队在足够 Degen 的同时，创新性的设计具备东方特色的 NFT，会有大量受众。

之前 FudaoVerse 算是一种尝试，但由于时机不对（之前亚洲基本只有头部用户和韩国等少数发达国家会去关注 NFT，并无香港这种战略支持）和运营等原因，结果差强人意，已经丧失流动性。当前，Ubox 除去交易场景打造之外，也着重在考虑设计具备东方特色 NFT 系列组合，涵盖一些特色的 IP 和玩法，我们根据其当前披露的“小虾米”可以猜测出，会有武学、功法、升级合成等多种玩法。听起来似乎有点千篇一律，但要知道有了 NFT 和 NFT 交易所的加持，“随机性”和“稀有度”两个最重要的属性会公开透明；展开想象，数字世界可以与 DNFT 结合使得更像真实的元宇宙，功法等 NFT 物品也可以结合 Staking 创造更大的价值，预言机可以实体周边市场，IP 的价值远不局限于一个 NFT。以及随着诸多娱乐圈顶流入局 Web3，许多东方（武侠）影视界和漫画界的重量级人物也都跃跃欲试，未来不排除站台与合作的可能。

Ubox 首期资产发售——小虾米是拥有权益的 NFT 集合，同时具备基层会员资格。不过具体 Mint 时间还未定，大家有兴趣可以去其 Tw @ubox_io  和官网保持关注～

Web3 是全球化的，随着发展会有越来越多的用户进入 Web3；在部分赛道已经足够“内卷”的情况下，如何去占领一些有价值、但还未被驯化的山头成了许多 Web3 团队着重考虑的事情。很多被人们忽视的国家地区，如：伊朗、东南亚、阿根廷等，实际上都在不同时期有着大量用户的涌入。香港的加密市场在很长一段时间里处于灰色地带，而现在沉睡的东方金融中心已经苏醒；我们有足够多的理由，去关注 Ubox 在内这些有独特运营思路的团队～
