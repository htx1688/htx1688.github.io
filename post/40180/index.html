加密资产市场整体进入深熊阶段，独立板块 NFT 市场则新闻不断，NFT 交易赛道闯入了不少新老面孔，包括传统行业的玩家。

7 月 12 日，美国知名电子游戏零售商 GameStop（游戏驿站）正式推出其 NFT 交易平台 GameStopNFT，允许游戏玩家、创作者、收藏家和社区成员购买、出售和交易 NFT。

GameStop 入局 NFT 市场，再次显示了游戏资产 NFT 化的趋势， GameStopNFT 目前只专注于游戏领域，也说明 NFT 交易逐渐从综合性市场走向细分市场。

7 月 7 日，加密资产去中心化交易应用 Uniswap 的开发方在社交媒体上公开征询 NFT 产品的建议，这意味着这个以太坊链上最大的同质化代币交易平台开始切入 NFT 赛道。

进入 2022 年，提供 NFT 交易的市场不再是 OpenSea 一家独大了。

根据 NFTGO 数据，截至 7 月 13 日，NFT 交易平台 x2y2 近 30 日的交易量为 6.2 亿美元，成交规模已经超越 OpenSea 的 4.96 亿美元，在市场排名中居第一位，另一家平台 LooksRare 以 1.96 亿美元的成交量排在第三。

x2y2 正在动摇 OpenSea 的霸主地位，它和 LooksRare 一样，都是今年早期开始挑战 OpenSea 的新兴 NFT 交易平台，它们直奔 OpenSea 中心化、高手续费、平台收益与用户无关等痛点，以空投平台代币、低手续费、收益 100% 反哺用户等手段，开始分食被 OpenSea 垄断的 NFT 交易市场。

面对四面八方的竞争者，OpenSea 对内升级产品，对外以收购补短板，以提升自身的竞争力。

今年 4 月，OpenSea 收购了去中心化的 NFT 聚合协议 Gem；今年 5 月，OpenSea 允许用户使用除加密货币外的其它支付手段交易 NFT；6 月 15 日，该平台从旧的 Wyvern 协议迁移到新协议 Seaport 上，以降低用户的交易手续费。

NFT 交易市场逐渐进入红海，也仍有平台不惧竞争，从产品创新、细分市场等层面加入战局，企图从这个千亿级的市场中分一杯羹，比如加密艺术专业策展平台 SuperRare、多链 NFT 交易平台 Rarible、NFT 批量交易聚合平台 Gem 和 Genie、以 NFT 换 NFT 的交易平台 SudoSwap 等等，它们共同形成了当下的 NFT 市场格局。

本期 DeFi 蜂窝将梳理 NFT 交易赛道的演变，从产品的角度盘点这个市场的重要参与者。

挑战 OpenSea 之路从 Token 激励大战开始

作为 NFT 交易市场的龙头，OpenSea 在去年很长一段时间内占据了 90% 以上的市场份额，即便它的交易系统不稳定，遭遇过黑客攻击、用户 NFT 资产被盗，甚至发生过内幕交易等丑闻，但提到 NFT 交易，人们还是会选择 OpenSea。

当时，市场上能与之比肩的平台太少，优质的 NFT 项目又会首选 OpenSea 上线，虽然这家平台争议不断，但用户好像别无选择。

转折点大概发生在 2021 年最后一个月，OpenSea 计划 IPO 的传闻越来越像真的，它刚上任的首席财务官 Brian Roberts 认为，平台增长如此之快，不上市就太蠢了。这样的表态触碰了加密社区的禁忌 —— 一个从去中心化资产市场中赚钱的平台，竟然不发行平台币回馈用户，反而想走传统融资的道路，这十分「不区块链」、「不 Web3」。

OpenSea 的用户是愤怒的，这个平台日积月累的问题重新被翻出来，放在舆论的烈火上炙烤，Brian Roberts 不得不出面平息，声称媒体报道曲解了他的意思，IPO 与否还是会顾及社区意见。

然而，这一次的愤怒没有被一些有心的开发者放过。2022 年 1 月，一个建立在以太坊上、名为 LooksRare 的 NFT 交易平台横空出世，专门面向在 OpenSea 上有过交易的用户空投 LOOKS 代币，并推出了过去在加密资产交易平台中常见的「交易即挖矿」玩法，对 OpenSea 来了一次「吸血攻击」。

除了激励政策，LooksRare 的交易手续费也比 OpenSea 低，它每笔交易仅收取 2% 的手续费，比它的直接竞对低了 0.5%。上线初期，LooksRare 的交易额一度超越了 OpenSea。

但好景不长，2 月初，LooksRare 推出的激励机制引发了严重的刷量问题，用户为获得 LOOKS 奖励将 NFT 自买自卖，大量的 LOOKS 奖励也被抛向市场，造成该代币的价格从 7 美元一路下跌。后来，团队被爆出通过质押自己持有的 LOOKS 套现手续费的情况，市场不信任带来了 LOOKS 价格进一步下泄。

随着 LOOKS 的下跌，LooksRare 的热度也开始下降，尽管如此，但它的出现让外界看到了一种可能性：OpenSea 并非不可挑战。

LooksRare 的「交易挖矿」机制出现问题时，又一个新生的 NFT 交易平台 x2y2 露面。2 月，这家新平台同样通过平台代币 X2Y2 空投、手续费 100% 分配给质押 X2Y2 的用户等手段吸引流量。但在激励方面，x2y2 没有一开始就搞「挖矿交易」，它似乎更看重平台内 NFT 品类和数量，于是祭出了「上架 NFT 即挖矿」的玩法，选择奖励那些在平台上挂单 NFT 的用户。

这样的激励设计让 x2y2 解决了初期 NFT 产品数量不足的问题，且将奖励数量与用户挂售 NFT 对平台的贡献度挂钩，如此一来，相比「交易挖矿」的粗放式激励，「上架挖矿」一定程度上避免了 X2Y2 的产出速度过快的问题，即削弱了通胀和抛压。

LooksRare 的突然爆火更像是 NFT 交易用户对 OpenSea 不发行平台代币的报复，而 x2y2 则在竞争对手的产品层面做出了差异化的迭代 —— 它支持用户批量上架和购买多个 NFT，这是 OpenSea 当时还不具备的功能。

x2y2 也在发展过程中出现过问题，火爆当时，有 KOL 指出 x2y2 的智能合约存在管理权限过大的问题，引发了用户对资产在该平台的安全担忧，但很快，该平台也快速改正了问题。

今年 5 月，x2y2 的 NFT 交易量突然大幅增长。细究发现，该平台交易量的激增与其激励机制的变化有关。今年 4 月 1 日，它取消了挂单奖励，改为「交易挖矿」。有趣的是，LooksRare 也在 4 月 20 日改变了激励策略，将原先的「交易挖矿」中的部分收益分配至了「NFT 上架奖励」中。

激励机制的改变的确给 x2y2 带来了好处。根据 NFTGO 数据，截至 7 月 13 日，NFT 交易市场 x2y2 近 7 日交易量为 6.2 亿美元，成交规模已经超越 OpenSea 的 4.96 亿美元，在 NFT 交易市场中排名第一，LooksRare 以 1.96 亿美元的成交量排名第三。

x2y2 近 30 日交易量约为 1.58 亿美元

多链并存为 NFT 交易提供了新选择

Token 激励向来是去中心化平台之于中心化平台最有效的打击方式，因为链上的加密资产交易市场为 Token 提供了「转化为财富」的渠道。用这一招，LooksRare 和 x2y2 从坚守合规、谋求上市的 OpenSea 那里吸走了一波流量。

但这些竞争也仅存在于以太坊生态，即平台供给的 NFT 与服务的用户均属于以太坊生态。这个生态上的 NFT，最突出的特征是「知名的 NFT 价贵、不知名的 NFT 不值钱且流动性差」，价格贵也是 ETH 本身的市值所决定的，以太坊网络上产生的 NFT 大多用 ETH 支付，除了购买 NFT 的费用本身，用户还需要用 ETH 支付链上的 Gas 费、交易平台规定的手续费。ETH 本身的价格就比较高，用户似乎又进入了「没得选」的局面。

而区块链公链市场的多链并存状态打破了这种局面，Solana、BSC、Polygon 这些综合性公链上都出现了 NFT 交易平台应用，除此之外，还有专门支撑 NFT 发行与交易的 Flow、ImmutebleX 等。

当以太坊网络上的 NFT 越来越贵、交易成本增加时，这些公链链上 NFT 市场给用户提供了高性价比的选择，至少，链上的交易手续费低廉就足够吸引用户上去挑一挑、选一选了。

到了今年下半年，加密资产市场整体下行时，其他链上的 NFT 市场反而开始爆发。根据 Cryptoslam 显示，截至 7 月 13 日，近 30 天内，Solana 链上的 NFT 交易量为 7200 万美元，参与的用户有 12.5 万人，这个数据在在众多公链的 NFT 市场中排名第二，体量已经是以太坊的 12.5%。

Solana 链上 NFT 交易量仅次于以太坊，数据来源于 Cryptoslam

其中，Solana 链上的 Magic Eden NFT 交易市场最为显眼。

根据 SolanaFloor 的数据，截至 7 月 13 日，Magic Eden 历史总交易额为 6.9 亿美元，24 小时交易额为 208.6 万美元，是 Solana 链上最大的 NFT 交易市场。OpenSea 也在 Solana 链上部署了交易市场，该平台的 24 小时交易额仅为 1 万美元。

Magic Eden 的数据表现已经吸引了资本的注意力。6 月 22 日，该平台以 16 亿美元估值完成了 1.3 亿美元的 B 轮融资。

2021 年 9 月诞生的 Magic Eden，主要为 Solana 链上生态内的用户提供 NFT 铸造和交易服务，随着不断开发，它逐渐成为该链上的 NFT 综合服务平台，功能囊括 NFT 发行、数据分析、资产管理等。

Magic Eden 在产品体验上也不断精益，除了提供 NFT 铸造和交易这些基本功能外，还进行了产品细分，比如构建了 NFT 初次发行平台 Launchpad—— 为创作者提供技术、营销支持工具组件，使创作者启动自己的新项目变得更容易；添加了游戏 NFT 资产专用版块 EdenGame—— 为游戏开发、游戏玩家提供一站式平台，游戏内 NFT 资产可以通过此平台交易。

相比 OpenSea 之于以太坊，Magic Eden 与 Solana 链贴合的更紧密，它完全地构建在链上，从 NFT 交易起家，逐渐进化为一个一站式的 NFT 平台，集交易、发行、游戏入口、资产管理等于一身。

通过不断打磨产品从竞争激烈的 NFT 交易市场中脱颖而出，这一点在 Magic Eden 身上最为明显。

新兴竞争者从交易体验切入赛道

对于 NFT 交易平台来说，发行、细分 NFT 场景、资管都是重要功能，但交易是永远的核心。一个平台想要赢得用户，激励是获取用户流量的方式，而体验才是留住用户的根本。不少新兴的平台已经注意到了 NFT 交易市场的痛点，开始逐一针对这些痛点提供解决方案，比如实现批量交易、比如降低交易摩擦成本等。

随着 NFT 交易平台和 NFT 项目的增多，用户发现，在某个平台上想要交易多个 NFT 时，需要一个个搜索，然后查看报价，价格理想的话，则需要一个个进行交易，而每次交易都需要支付 Gas，费力又费钱，尤其是在以太坊上。

综合性 NFT 交易平台更适合有单个 NFT 交易需求的用户，并不能满足批量交易需求，于是，NFT 聚合交易平台出现了。

在 NFT 聚合器上，用户可以浏览与购买多个平台的 NFT，同时聚合器还会提供显示地板价、跨平台比价、支持多支付手段等多种功能，它们正在成为 NFT 交易平台的流量入口。

NFT 聚合交易平台能帮助用户在一笔交易中同时购买多个 NFT，将原本分散的 NFT 流动性集中起来，降低用户的交易成本，满足用户的批量购买需求，它们更像是「NFT 交易市场中的 1inch」。

在 NFT 聚合交易赛道，代表产品分别是 Genie 和 Gem。

Genie

Genie 是第一个布局 NFT 聚合器应用的平台，它于 2021 年 11 月正式上线，主要支持 NFT 资产批量购买和出售功能。今年 6 月 22 日，Uniswap Labs 宣布收购 Genie。

当前，Genie 已集成了 OpenSea、Rarible、LooksRare、x2y2 等多个 NFT 交易市场，用户在单次交易中可从多个市场购买多种 NFT，并通过 Genie Swap 聚合交易工具降低 Gas 费。

Genie 主页会显示 NFT 项目的趋势热度排行榜，Genie Swap 的整体界面简洁，左边是支付框（售出的 NFT），右边是接收框（购买的 NFT），两边可分别添加想要售出和购买的多个 NFT，差价由 ETH 弥补。

Gem

Gem 是基于以太坊开发的 NFT 聚合交易平台，它于去年 12 月 18 日上线测试版。4 月 25 日，OpenSea 宣布收购 Gem，在收购完成后，Gem 仍将作为独立品牌进行运营。

当前，Gem 聚合了 OpenSea、LooksRare、Rarbible、x2y2 等市场中的 NFT 挂单信息，它希望能整合所有 NFT 市场的挂单信息，使用户无需再一一跳转至其它 NFT 平台比价、交易。

此外，相比于 Genie 仅以 ETH 作为支付手段，Gem 在支付手段上做了更多优化升级，支持用户使用多种 Token 为买卖 NFT，甚至支持同时用 Token 组合进行支付。例如，当用户购买 NFT 时，钱包里的 ETH 余额不足，用户可以用 USDC、UNI 等其它以太坊标准的 Token 组合支付。

另外，Gem 集成了更加丰富的数据展示功能，包括每个 NFT 的多元数据（价格走势、NFT 属性特点等），用户在 Gem 上就可一站式浏览与购买各个主流交易平台上的 NFT。

除了能否批量买卖问题之外，NFT 这种非同质化代币目前面临的最大问题是流动性，相比 BTC、ETH 这类同质化加密资产，NFT 的流动性差了很多，这一点从 OpenSea 在市场下行时的日活就可以看得出来。

交易摩擦是 NFT 流动性差的原因之一，非同质化的 NFT 无法将 A 兑换为 B，必须要通过一个中间 Token 完成转换。比如，用户可以直接按照时时汇率将 BTC 兑换为 ETH，但无法直接将 CrytoPunks 系列中的一个「像素人」兑换为 Bored Ape Yacht Club 系列中的一个「无聊猿」，因为二者根本是不同的东西，每个 NFT 都独一无法，价格各不相同，彼此也没有形成市场汇率。

切中这一痛点的开发者创造出了点对点 NFT 交易平台的解决方案，开发出了「以 NFT 易 NFT」的交易模式，代表应用是 Sudoswap 和 NFT Trader。

Sudoswap

Sudoswap 支持 NFT 资产之间相互交易，即用户可以通过该平台将自己的 NFT 与其它人的 NFT 直接交换，交换双方接受交易后，NFT 资产会自动从用户钱包转移，交易过程中只需要花费链上 Gas 费。

7 月 6 日，Sudoswap 宣布推出 NFT 的 AMM（自动做市商）协议 sudoAMM，用户可以通过该协议创建 NFT 流动资产池，为流动性资金池 NFT-Token 提供流动性，获得交易手续费奖励。该协议使 NFT 交易用户将 NFT 与 Token 间进行互换时享受较低滑点。

目前 Sudoswap 提供的可交易 NFT 种类、数量都比较少，仅支持某几种 NFT 的交易。

NFT Trader

NFT Trader 同样支持 NFT 买卖双方进行点对点的置换。在双方确认交易之前，用户 NFT 资产会暂时托管到平台部署的智能合约之中。

举例来说，小明使用 NFT Trader 平台时，首先要确定自己想要接收的 NFT 类型，并确认小丽提供的 NFT 正是自己所需；然后，小明将自己的 NFT 资产转移到 NFT Trader 的智能合约上，这时智能合约会生成一个链接发送给他，这个链接包含小丽完成 NFT 交换所需的所有信息；最后，小明将链接发送给小丽，小丽确认交易，NFT 资产将会自动转给小明。

在「以 NFT 易 NFT」的过程中，NFT Trader 以智能合约充当受信任的第三方，避免用户之间的无效交易和欺诈交易。如果交易对手欺诈或者终止交易时，发起交换的用户可以在 NFT Trader 上随时撤回他们的 NFT 资产。

目前，点对点交易平台还处于发展初期，相较主流的 NFT 交易市场，产品还不够成熟。不过它们为 NFT 之间的流转提供了新的形式。

无论如何，在 NFT 交易平台这一赛道上，OpenSea 已经不再一家独大，竞争正在为这个市场带来活力，这种活力不仅仅是流量支撑的交易额，更重要的是交易产品正在竞争中一一步步迭代。
