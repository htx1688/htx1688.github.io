原文：Decentralized Science Funding

摘要

研究成果的产生具有明显的经济效益和社会效益，但也存在一些难以解决的问题。 在过去的几十年里，这些问题一直是众多公共和私人倡议的灵感来源。 随着 Web3 的兴起，已经看到相当多的去中心化科学 (DeSci) 计划获得了关注。 我们回顾研究经费中的一些关键问题，其中研究人员的时间和资助者的资金分配效率低下。 该过程受制于偏见、激励措施的错位和不良的风险回报评估，从而导致次优的投资决策和新研究想法的资金不足。 最重要的是，研究资金市场一直不透明，公众投资者难以进入。 在本文中，我们证明 Web3 不仅可以提供优于以前技术的优势，而且可以彻底改变我们资助研究的方法。

研究经费问题

在我们确定的研究资助的关键问题中，最引人注目的问题之一是研究人员在拨款申请上花费的时间过多。 联邦示范合作伙伴关系的一项调查表明，研究人员 44% 的时间花在管理上，而这些时间往往用于资助相关任务（Schneider，2018 年）。 申请资助需要撰写耗时的研究计划书，而且成功率往往有限。 这个过程相对不透明，不同的资助者的目标可能不同且不确定。 资助者往往不太愿意通过其他渠道的申请流程来获得“快速”的研究资金。 如：近年来已经建立了少量关于快速拨款的举措——DeFi 教育基金——来解决这个问题。

其次，由于资助者的目标与社会最优资助水平不一致，某些研究领域经常被忽视。 公共基金鼓励基础研究，而私人基金鼓励应用研究并立即向私人资助机构申请（Buccola、Ervin 和 Yang，2009 年）。 此外，资助者对“研究偏见”很敏感，更有可能追求“热门”（即共识）、“高净现值”或“低风险”主题。 尽管资助提案的质量很高，但资助者仍然专注于研究人员的过往成绩，倾向于资助经验丰富（或“精英”）学者的研究，而不是年轻的研究人员或声望较低的机构的研究人员。 因此，资助往往会流向同一组选定的知名研究人员、教职员工和机构。 在私人领域，资助主要用于解决特定问题的研究，并且通常与明确的预期投资回报率 (ROI) 挂钩，而不考虑更广泛的科学或社会需求。

对于“公民科学家”——不是正式研究人员但喜欢为科学做出贡献的公民——几乎不可能获得资助。 虽然大学与“局外人”协调一些研究，但个别研究人员几乎没有可用的工具、专业知识和资源来自行申请资助。 许多拨款还明确要求首席研究员 (PI) 必须隶属于大学或正规研究中心。 因此，公民科学家没有明确的资助途径，而且往往没有能力与知名大学支持的研究人员竞争。

另一个问题涉及所谓的“科学发现的终结”，学术界的几个领域近年来未能产生太多新颖的研究。 突破性进展的速度比以前慢，新的研究领域越来越少。 我们认为，资助系统在这一趋势中可以发挥重要作用，当前系统正在将资金引导至成功率最高、适合现有叙述并由“高声誉”研究人员进行的研究。 跳出框框会受到惩罚，因为无法获得同行评审，以至于几乎没有机会发表研究成果。 此外，跨多个学科的研究往往处于资助的尴尬地带。

这一趋势中两个值得注意的例外，人工智能和区块链，主要受益于它们激励私人资金的高货币化潜力。

最后，很大一部分科学是公共资助的，后来又通过私人方式货币化，很少有经济回报返回给最初的公共资助者。 由公众资助的研究往往位于收取准垄断租金的出版商的付费专区后面，学术出版已成为世界上利润率最高的企业之一（Van Noorden，2013 年）。 应该指出的是，私有化货币化不一定是一种不良影响，因为从大学到行业的知识转移可以被视为对社会有益。 然而，这可能会造成激励失调和不良副作用。在制药行业尤其如此，在新药发现方面，公共资助的研究是核心驱动力。 然而，在这个过程中，最初的研究资金与最终利润之间的联系变得模糊了。

去中心化科学资助 (DeSciFi) -研究基金的 Web3 解决方案

科学知识通常被归类为公共物品，因此，其生产得到大量公共资助。 在实践中，我们通过排他性和竞争的产生，将研究视为公共物品和私人物品。 你可以争辩说公众对知识产权研究的强烈兴趣，因为它提供了对科学投资的金钱激励。 当研究人员站在巨人的肩膀上发现新发现时，科学进步往往会逐渐下降。

Web3 协议一直在将（准）公共物品的产生与私人资金结构和开发团队结合起来。 Web3 中最流行的用例之一是去中心化金融 (DeFi)，其中机制设计已应用于各种金融应用程序的智能合约中。 这提供了一个机会，可以使用受去中心化金融启发的设计来试验私人-公共资助结构的研究。

涌现出许多 DeSci 初创公司，但是，他们中的大多数并没有真正使用 DeFi 机制，而是在推广科学的 kickstarter。 迄今为止的解决方案在设计上感觉是拟物化的，几乎没有考虑使用 Web3 原生功能。 此外，许多 DeSci 解决方案都专注于评论和出版等主题，而我们想说的是，Web3 工具最具影响力的潜力在于资金方面。 现有企业和新进入者都没有充分解决围绕经济学和适当管理研究资金的核心问题。 仅仅将研究资金转移到代币化资产轨道是不够的，它需要合理的市场设计机制来满足研究人员和投资者的需求。

我们认为，通过代币经济学激励将 DeFi 机制和基于社区的管理相结合，可以为去中心化的科学资助提供适当的解决方案。 通过进一步将研究与 NFT 联系起来，研究人员将有可能更好地评估他们的工作并从中获利。DeSciFi 可以帮助开创一种新的范式，在这种范式中，研究通过去中心化社会（Srinivasan，2022 年）和数字文明（Roscoe，2021 年）进行及资助。

不可替代代币作为构建块

目前没有简单的系统来部分资助和拥有研究。 DeSci 可以有效地改变这一点，其中数字资产的优势在于有机会融入 Web3 生态系统。 在加密货币中，已经开发了一个完整的生态系统，用于高效发行、拥有、交换数字资产，并在此基础上构建了许多其他应用程序。

更具体地说，我们认为，出于 DeSciFi 的目的，使用不可替代代币 (NFT) 很有趣。 NFT 是代表相关资产真实性的唯一数字证书的数字资产。 许多项目一直致力于将（研究）出版物放在链上——例如，ScholarOrg 开发了一种机制，可以通过 IPFS 将研究成果（论文、数据、代码）存储在不可变的链上数据库中。

这些特性导致各种组织开发与 IP 相关的 NFT。 IP 协议可以存储在去中心化的分类账上，以便链上有不可变的证据。 标准化的法律协议——在协议层面——应该到位，以确保 NFT 持有者对知识产权和版权的主张得到保障，并且可以在销售中转让。 例如，Molecule DAO 一直在为 DeSci 发行与 IP 相关的 NFT。 最后，NFT 可以帮助提供许多其他研究应用程序，例如对期刊和按使用付费系统实施访问代币。

研究市场和平台

构建一个基于智能合约的资助机制和研究市场，将研究放在 NFT 上。 研究人员可以发布由社区成员资助的提案，以换取项目的利益相关者。 发布一个初步的（与 IP 相关的）NFT，代表该项目，各个利益相关者拥有 NFT 的部分所有权。 在这种情况下，研究被 NFT 捕获，其中所有权被代币分割，项目本身（及其周围的社区）可以被视为事实上的去中心化自治组织（DAO）。

将 IP 置于 NFT 上的好处之一是它为更有效地评估这些无形资产的价值提供了机会。 除了 IP 市场更加活跃的机会，NFT 生态系统在 NFT 定价方面也一直在快速发展。 许多公司一直致力于 NFT 定价和评估，我们已经达到了大多数收藏品都可以进行可靠实时评估的地步。 此外，可以实施基于社区的评估系统。 例如，Lithium Finance 一直致力于开发一种去中心化估值协议，该协议允许市场参与者社区利用群体智慧和加密经济学进行按需价格估算。 研究定价可以基于市场的质量指标，因为价格可以反映可取性和质量。

市场体系通常可以被视为衡量价值的良好尺度，可以说是“价值发现”的有力工具。 DeSciFi 平台将允许“购买”和“出售”研究中的share，其中市场充当有效的资本分配系统。 科学创新实际上是叛逆和反现状的，而当前的制度反而会激励对共识想法或现状内的知名研究人员的资助。 基于市场的机制来鼓励资助（定价过低的）研究想法（或研究人员）。

资产NFT化，可以改善资产的流动性。当今的知识产权市场相当不透明且流动性差，寻找合适买家的搜索成本很高。 有了 NFT，可以即时验证真实性和所有权，并且实现无需信任的交易。 去中心化交易所主要利用自动做市商 (AMM) 设计来促进交易。 AMMs - 由社区资助 - 为知识产权股权提供即时流动性，从而为投资者货币化。 我们会争辩说，公众有动力为研究创造流动性市场，并且给予公平的评估价格，这可能构成公平交易。

此外，NFT 为更高效的专利使用费支付和围绕知识产权的新颖商业模式打开了大门。 通过在每笔交易中支付版税，NFT 可以为研究人员提供额外的货币化机会。 通过智能合约，每次交易 NFT 或使用其 IP 时，利益相关者都可以自动获得版税。 许可可以灵活地编码到智能合约中，创建一个永久有利于利益相关者的奖励设计设置。

资助集体和网络

如前所述，研究资助的问题之一是资助过程缓慢，这可能需要几个月的时间和大量的努力。去中心化的策展机制与（自动化的）资金集体可以加快事情的进展，实施自动化的市场决策和资金分配机制，创建公共或私人性质的资金集体。在这种情况下，资金可以根据代表的策划，基于规则的自动机制–如二次方资金–或资金集体成员的投票来分配。

该协议的公共功能和与 IP 相关的 NFT 允许一个透明和更具流动性的环境，在这个环境中，研究投资可以成为一种可行的投资策略。 平台上可以设立“研究基金”，由提供投资策划的代表积极管理。

Srinivasan (2022) 讨论了由基于共同信念的社会组成的网络状态。 你可以根据共同的信念形成科学网络，为特定的研究提供资金。 在目前的环境下，很难知道社会想要资助什么样的研究，最终成为少数有偏见的看门人的决定。 对于研究网络，您会立即知道该网络代表什么以及它相信什么样的研究。这将允许社会在这些网络之间分配资金，从而形成一个更加透明和开放的资金生态系统。

追溯资金的社区池——一种在项目成功的情况下追溯奖励项目贡献者的机制——可以进一步激励研究人员。 可以根据各种指标和/或社区成员的投票，对给定时期内最成功项目的利益相关者进行不同的分类和奖励。 追溯资助可以激励投资者资助“盈利”较低的主题的研究——例如，具有潜在有价值的知识产权的医学或工程研究将比社会科学研究更“盈利”。通过这种方式，公众将奖励对社会有贡献的研究，并且由于资助者分享成功，因此会有更多的私人进一步资助。

应用

我们相信 DeSciFi 创造了一个机会，可以扩大潜在资助者和研究人员（例如，公民研究人员）的参与者群体。 各种利益集团也可以通过使用 DAO 来整合。 这方面的一个例子可能是患者社区和其他贡献者，在投资集体资助的治疗中真正的利益相关者。 与 IP 相关的 NFT 是一种重要的催化剂，因为版税系统和许可都可以直接从核心 NFT 功能中衍生出来。 在这方面，我们看到了科学领域以外的应用，因为可以资助不同类型的研究项目。 当涉及研发阶段的企业研究项目和种子前阶段初创企业时，会出现类似的问题，因此可以使用类似的融资解决方案（如本文所讨论的）。 这些类型的项目存在一个尚未开发的资金池，目前不易通过现有渠道销售。

DeSciFi 可以根据项目的特定风险和投资者的需求，以新颖的方式促进融资。 更广泛的投资者群体可以参与企业研发资助，从而增加企业研究项目和直接激进主义的利益相关者。 投资者将能够投资于选定的项目或产品，而无需为整个公司提供资金。 这方面的例子可能是希望投资于专门针对某些治疗的研发的患者集体，或者是支持传统石油和天然气公司绿色技术研发的气候活动家。
