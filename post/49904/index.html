随着近日比特币行情站上30000美元关口，山寨MEME币又开始兴风作浪了，比如PEPE、AIDOGE、XEN等。

当然也吸引了很多smart money或者内幕者的投机，而且字面收益非常高。比如，以0x4a开头的Smart Money4月15日花费0.125枚ETH（约251美元）购买了5.9万亿枚PEPE，截至发稿，字面市值达到约170万美元。也就是说4天的字面收益超过6000倍。

也因其高热度，PEPE排在Uniswap上Top代币前五位置。

总之，无论从哪方面看，PEPE都非常像之前的那些庞氏山寨MEME币。当然，值得注意的是，MEME币爆火的时候，就是大盘岌岌可危时刻。

链上分析师Eljulien利用Nansen和Dune Analytics的Token God模式分析了具体数据，包括以下5个方面：PEPE效用和代币经济学 、社交热度 、代币数据、 Smart Money行为模式、结论及警告。

1、PEPE效用和代币经济学

代币经济学：

  • 总供应量 = 42.69万亿

  • 93.1%供给流动性池 = 391.66万亿（LP代币销毁）

  • 6.9% 给团队 = 29.03万亿

  • 效用：零 

  • Uni_V2池：7.71万亿

  • Uni_V3池：4.1万亿

  • 当前团队钱包：27.83万亿

2、社交热度

* 在dexscrener上大热

* 在dextool上大热

* 社会主导性日益增强

最近几天，人们对meme币的普遍情绪越来越高涨。

3、代币数据

持有者不断增加，目前已超2万。这对于一个上线4天的项目来说是相当多的。

乍一看，似乎还没有鲸鱼钱包（以0x92开头的地址pepecexwallet.eth是团队钱包），但事实并非如此。

4、Smart Money行为模式

由于Smart Money的数量不同，展示结果将与平时情况有所出入。

使用：

  • Nansen Token God Mode（smart-money数据尚不可用，我需要手动完成）

  • Token Dawg mode Dune仪表盘

  • Token Analyzooor Dune仪表盘

当你查看首批买入PEPE的钱包时，会发现有些奇怪。 许多钱包在20秒内以极低的价格同时买入，超70T。

首批买家里实际上有很多都是同一人，他们只是把自己的购买分成了若干投资组合。其中许多人从ByBit获得少量ETH（约150美元），并购买了数万亿的PEPE，现在价值数百万美元。

只有少数SM（Smart Money）出售了全部的PEPE。即使他们的投资组合赚了1万倍，大多数人也不没有抛售。

  • 黄色标签是押注垃圾币并且快速TP的SM。

  • 绿色标签是早早买入的SM。

  • 红色标签是无法追踪的投资组合，他们使用Bybit来隐藏自己的活动。

这里仅列出17个SM地址，但我查看的投资组合要更多。那些前24小时买入的最大持有者仍持有PEPE。

PEPE鲸鱼钱包可能还不会止盈（take-profit），因为：

  • 没有太多的流动性

  • -他们认为FOMO才刚刚开始

所以，如果你认为PEPE值得购买的话，请务必牢记，要使用如Arkham这样的工具来监控这些钱包。

5、结论及警告

PEPE完全是一个庞氏骗局。我并不是说它现在就会面临抛售，因为鲸鱼钱包仍在持有，但现在这似乎成了一个危险的游戏。

如果全球市场上涨，PEPE肯定会上涨得更快。否则，很多人很快就会被淘汰出局。

PEPE重新点燃了MEME币的狂热，众多山寨币将紧随其后。
