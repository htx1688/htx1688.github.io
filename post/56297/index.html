0 1  宣传区块链就是把学生往火坑里推？

前段时间红林律师邀请国内某知名高校老师参加一场线下交流活动，该老师此前以积极组织区块链活动而被圈内人熟知，此前红林律师还特意去学校拜访。

按照Ta的说法，Ta是希望能够让年轻的同学们了解未来新经济的发展方向在哪里，抓住时代的红利，在未来职业选择上多种可能性。但最近一段时间Ta却格外低调，面对我的邀请ta也婉拒了，并表示自己最近有点心灰意冷，甚至说自己想出国。

我问Ta为什么？

Ta说学校的其他老师跟Ta说，像你这样鼓动学生去搞区块链，是把学生往火坑里面推。很容易出事。

这样的回答让我愕然，但说实话我却不怎么意外。

作为区块链行业的法律服务机构，我们对知名高校的优秀学生的需求可以说是渴望，所以我常留意国内高校的区块链协会组织的相关活动，也添加了不少在校同学们的微信。但有时看到他们参加或者主办的活动，活动主题、邀请的项目方、讨论的内容，多少让人有一点担忧。

我不关心炒币的暴富效应可能会让祖国的大好青年们对财富的创造有错误的理解，但我稍微有点咸吃萝卜淡操心的是：区块链如影相随的无政府主义，对中国高校大学生进行的无意识渗透。

0 2  无政府主义和区块链

区块链是一种分布式的、去中心化的、不可篡改的数据存储技术，它可以实现点对点的交易、智能合约、数字身份等功能。区块链的优点是它可以保证数据的安全性、透明性和可追溯性，而不需要依赖任何第三方机构或中介。

作为一种技术工具，区块链发挥什么社会效用，取决于我们如何使用它、为了什么目的使用它、以及谁来使用它。技术人员可以用它来解决网络资源共享的问题；商业组织能够使用它来降低交易成本，提升组织发展效率；而无政府主义者则能用它来布道宣传，实现自己的政治目的。

无政府主义是一种政治哲学，它主张消除国家和政府的存在，以及所有形式的强制性权威和等级制度，实现个人的自由和平等。

无政府主义者认为国家和政府是人类社会的最大祸害，它们通过暴力、剥削、压迫和控制来维持自己的利益和统治，而不是为了人民的福祉和幸福。

无政府主义者主张通过自愿的、自组织的、合作的社会关系来替代国家和政府，建立一个没有统治者和被统治者的社会。

你看这说法，稍微改个词，是不是就常常出现在各个区块链项目方和 DAO 组织的宣传资料里？

区块链和无政府主义之间有着密切的联系，它们相互影响、相互促进、相互启发。我们可以从以下几个方面来探讨：

首先，区块链是无政府主义的技术基础。 区块链通过密码学和共识机制来保证数据的安全性和一致性，不需要依赖任何中心化的机构或个人来验证或管理。区块链上的交易是透明的、公开的、不可逆转的，任何人都可以参与和监督。区块链上的智能合约可以实现自动执行和强制履行，不需要任何第三方介入或仲裁。区块链上的数字身份可以保护个人的隐私和权利，不需要任何中介或认证。这些特性使得区块链可以为无政府主义提供一种可行的、有效的、可信的技术基础，让人们可以在没有国家和政府干预下进行自由的交流、协作和创造。 去中心化的加密货币和加密社交软件，说的可不就是这么一件事。

其次，区块链是无政府主义的社会实践。 区块链不仅是一种技术，也是一种社会运动。区块链上涌现出了各种各样的项目和DAO组织，它们都试图利用区块链技术来解决社会问题、推动社会变革、实现社会理想。例如，比特币是一种去中心化的数字货币，它旨在打破国家和银行对货币发行和管理的垄断，让人们可以自由地控制自己的财富；以太坊是一个去中心化的应用平台，它旨在提供一个开放的、创新的、协作的网络空间，让人们可以自由地开发和使用各种去中心化应用；DAO（去中心化自治组织）是一种基于区块链智能合约的组织形式，它旨在实现一种没有领导者和层级结构的民主决策和资源分配方式；IPFS（星际文件系统）是一种去中心化的存储网络，它旨在建立一个没有审查和封锁的自由信息空间；等等。这些项目和组织都体现了无政府主义的精神和价值， 它们都试图通过区块链技术来实现无政府主义的社会实践。

最后，区块链是无政府主义的思想启示。 区块链不仅是一种技术，也是一种思想。区块链的出现和发展，给我们带来了许多新的思考和启示，让我们重新审视和反思我们所处的社会和世界。例如，区块链让我们重新认识到货币的本质和作用，让我们重新思考货币的发行和管理的权力和责任；区块链让我们重新理解到契约的含义和功能，让我们重新探索契约的制定和执行的方式和标准；区块链让我们重新感受到身份的重要性和敏感性，让我们重新考虑身份的表达和保护的方法和界限；区块链让我们重新明白到信任的价值和难度，让我们重新寻找信任的建立和维持的机制和条件；等等。这些思考和启示，都可以为无政府主义提供一种新的视角和灵感，让我们可以更深入地理解和探索无政府主义的思想。

区块链不是无政府主义的唯一途径或选择，但绝对是无政府主义宣传的利器。 随着行业的日渐发展，越来越多区块链组织和社群在中国的高校中活跃起来，通过线上线下的活动宣传和组织运营，宣扬区块链的理念和价值，吸引了不少学生的关注和参与。对于新型技术的普及、推广和宣传是好事，但要谨防这好东西被歪嘴的和尚给念错了经。

因为区块链行业的跨国界属性，很多区块链项目方的发起人和管理者是海外公司或组织，因各地法律制度、监管制度、文化理念、政治观念的不同，很多海外的区块链组织和社群往往以去中心化、自由、民主、平等为口号，反对传统的权威和制度，主张用区块链来实现社会变革和创新。主张区块链可以让人们摆脱现有的政治、经济和文化的束缚，建立一个没有中央权威、没有法律规制、更加公正和开放的新世界。倡导通过区块链可以自主地选择自己的价值观、生活方式、交易伙伴，不受任何外部干涉，实现真正的民主、自由和平等。

这样的思想灌输和价值宣导我们不去评价，但过于激进的政治理念宣传和社会实验组织，很容易通过区块链进行潜移默化的思想灌输与塑造，让社会经验不足的大学生激情澎湃，陷入对政治意识形态和社会现实的偏差理解。

0 3  我们应该怎么办？

想到一段往事。1919年的五四运动当天，北大3000余名学生上街游行，学生们出发前，蔡元培校长曾前来劝阻，队伍中满是嘘声与骂声，张国焘等人将蔡元培连请带推弄了出去。当天，北大20名学生被捕。

其后，蔡与北京高校校长出动营救学生，成功后，蔡元培递交了辞呈。并给北大师生们留下了一封公开启事，其中有言：

我倦矣！杀君马者道旁儿！

骑骏马者行在路上，道旁儿鼓掌叫好，骑马者得意非常，快马加鞭，马跑得越快，鼓掌者越多，叫好声越大。跑了数百里路，收获无数掌声，马终于倒地毙命。

蔡校长不想做那骑骏马者，也不想让自己的青年学子死于道旁儿。

本篇文章必然是无力的，但该问题却值得抛出被大家关注与讨论。

作为大学生，以开放的心态学习、了解区块链在内的新兴科技，绝对是好事，但我们也要同时认识到：凡事有好有坏，要保持独立的思考、理性的判断。

区块链并不是万能的，它也有自身的局限性和缺陷。 比如，区块链的运行需要消耗大量的能源和资源，造成环境污染和浪费；区块链的安全性也不是绝对的，它可能会遭到黑客攻击或者内部分裂；区块链的规则和共识也不是固定不变的，它可能会受到利益集团或者意识形态的影响和干扰。

区块链并不能完全取代现有的社会制度和秩序，它也需要与之协调和配合。 比如，区块链需要遵守法律法规和道德规范，否则可能会引发法律纠纷或者道德争议；区块链需要与人类社会的历史、文化和价值观相适应和尊重，否则可能会引起社会分裂或者文化冲突。

要知道人性的复杂性和多样性，要理解人的社会关系需求。 比如，区块链不能完全消除人们对于信任、权威和归属感的需求、不能完全忽略人们对于隐私、安全和公平感的期待、不能完全忽视人们对于现实世界、家庭幸福和现实组织身份的追求。

最后，作为区块链行业的从业者，虽然屁股决定脑袋，但我们也得管好自己的嘴，不能只讲区块链的好，也要指出行业的坏。 促进理性和平衡，防止误解和误导，避免极端和偏激，既要鼓励大家把握区块链的优势和机遇，也要提醒区块链的风险和挑战。只有这样，才能真正地让区块链成为有益而不是有害的技术，让区块链为高校乃至社会带来更多的福祉，而不是祸害。
