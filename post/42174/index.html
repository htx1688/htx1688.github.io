从宏观角度聊一下 Web3 市场人士的思维模式。

原文标题：《营销系列 02: 如何像一个 Web3 营销专家思考》（How to Think Like a Web3 Marketer）

撰文：Michael Stelzner

编译：Siyan

正在思考对你的品牌进行 Web3 转型？却不知道该如何下手？在这篇文章中，你会了解到利用 Web3 进行市场营销的独特优势， Web3 营销模式与你的思维惯性有何差别，以及什么是项目发行的成功关键。

#01 为什么大家都开始关注 Web3 了？

Web3 和相关技术正在革新着几乎每一个行业。如果你将目光放在市场领域，你会看到 Web3 为获客、留客和用户粘性开辟了新的思路。这些增长领域被看烂的名词因为区块链与 Web3 技术拥有了新的可能。

这几年，广告人因为隐私调理和阻止广告插件已经开始被迫改变了他们的策略。用户希望可以对自己的上网体验有更多的控制权，他们不希望被迫一直被广告围绕。传统的广告模式已经衰落，我们需要也会找到新的方式商业化我们的项目与网站。

举个例子，the Gather Network 提供了一个思路—用户在网站阅读你发布的内容的时候可以选择分享自己电脑的剩余算力并因此获得 token 奖励。我们预测，在未来的三到五年内，剩余算力的价值会超过广告收入本身。

营销和广告世界已经发生了天翻地覆的变化，你必须学会应对。广告投放越来越贵也越来越难。人们厌倦了自己的信息被倒卖，厌倦了自己也成为了一个商品。创作者也厌倦了自己的创作只是为中心化平台带来了价值。Web3 正在试图改变这一切。他们在改变内容创作的商业模式。创作者开始有了更大的决定权决定他们想要什么，他们想提供什么。中间商们在这个过程中将被淘汰出历史的舞台。

仔细想想，如果你是一个漫画创作者，你现在可以直接向你的社区售卖数字创作。如果有人购买了你创作的一个漫画人物的 NFT，他可以在自己的餐馆广告里用到这个形象。在这个过程中，版权和用户留存策略的界限变得模糊，形成了一个新的态势。

Web3 为企业提供了新的机会。在确认所有权和移除中间商的过程中，会有越来越多的应用案例涌现。作为一个市场营销人员，这是令人激动的—看到新世界的涌起，并在浪潮中贡献自己的力量。

#02 了解 Web2 与 Web3 在文化上的核心差别

当我们进入 Web3 之后，会很快察觉到一些文化上的核心差异。对于新人来说，你可能听过两个跟 Web3 有关的名词: 去中心化（decentralized）和反建制（anti-establishment）。

Web2 世界正好相反—它非常中心化，也时刻处于审查之下。屈指可数的机构拥有着巨大的权力，利用平台效应从用户身上读取、收集、储存数据并借此获得巨大利益。

现在绝大多数市场营销人员都聚集在非社区驱动、非合作性的社交网站上—虽然这些平台总爱使用社区这一概念。Facebook 和 Instagram 项目方可以控制在平台上发生的所有事—这跟反建制的思想正好相反。社交平台告诉你什么能做什么不能做，如果你不愿遵从，他们就会立刻将你移除。

我们对这些控制早就习以为常。也正因如此，当我们进入一个更加开放的世界时，我们最开始甚至会觉得有点奇怪。在新世界里，所有代码和创新都在区块链上，任何人都能去使用、分享、再创造。在这里，用户、创作者、项目方共同享有所有权。

本质上来说，Web3 是一个开放的、合作的世界—所有人共创元宇宙。这大概文化创新里最突破的一个点。

#03 Web3 发行策略的核心要素

如果你的团队正在进入 Web3 领域，有一些要素是你在确定发行策略时必须要考虑到的。

确定你的目标群体

一个好的市场人员问的第一个问题通常都是：谁是我的目标群体？你必须知道你在跟谁交流，这部分人对于 Web3 又有多了解。只有这样，你才有可能策划一个社区驱动、高合作性的 Web3 营销计划。

DigitalSelf 旨在帮助用户在元宇宙中参与到线下娱乐和活动。用户通过一个采用 VR 技术的头戴设备参与到音乐会、体育比赛、或者会议中。

他们的目标群体有两个：普罗大众（比如各个球队的粉丝）和更加「加密货币原生」的群体。针对普罗大众，DigitalSelf 通过内置钱包降低用户的参与门槛。而针对加密原生用户，平台允许他们匿名参与并且使用自己的钱包。平台希望自己能够和不同群体都保持沟通并且设计出适合不同群体参与的交互系统。

再次强调，你需要知道你在和谁沟通，你又该如何跟他们沟通。虽然现在 Web3 的用户群体还不大，但很快，就会有大量用户加入到 Web3 的世界中来，你需要在这之前做好准备。这就包括开始进行用户教育。Gary Vaynerchuk 和他的 VeeFriends 在这点上做得很好。他通过自己的方式，成功地帮助了他的大批用户了解到了 Web3。与此同时，他也必须顾及到加密货币的原生用户，毕竟这群人在圈子里浸润很久，也最愿意花钱进行投资。

要在两个群体之间取得平衡并不容易。你需要结合发行项目的类别综合考虑。

发展你的社区

Web3 项目发行的最大一块就是社区建设。你需要开始思考如何赋能社区并吸引他们的兴趣。发掘一个点让你的社区成员真正参与进来，并借此让他们开始期待你的发行。

先构建一个小社区然后等着它逐渐扩大不是一个合适的想法。在项目发行前你的社区规模应该超出你的售卖规模。这个社区需要和你建立起真实的连接和关系，否则他们不会购买。

举个例子，Sylvester Stallone 的 PlanetSLY 现在有超过 26,000 个 Discord 会员并且计划售卖 9997 个 NFT。同时持有三个 NFT 的人可以获得和 Stallone 面对面的交流机会。根据现在的社区规模以及购买多个 NFT 的额外福利，这个项目很有可能取得巨大成功。

你需要亲自参与到你的社群中。你不能想着创建一个 Discord，然后它就自己开始运营。不管是通过 Discord 还是更传统的邮件，你都需要构建一个团队、制定一个计划，确保你的成员接收到重要信息。

考虑聘用一名有经验的 Discord 社区经理。虽然你也可以用 bot 回答一些常见问题，或者用 bot 和新人打招呼，但你还是需要「人」的存在。

一些 Discord 社群会采用 Discord 自带的语音房，让社区成员可以和项目创始团队直接交流。你还可以举办 Twitter space。在 Twitter 上举办活动的好处是非社区成员或者非用户也能参与进来，并且考虑要不要在二级市场上购买你的项目。
