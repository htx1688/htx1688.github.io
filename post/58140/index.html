作者：Brian Armstrong，Coinbase创始人兼CEO；翻译：金色财经0xJS

有时人们会问我，如果你今天要创办另一家加密货币公司，你会创办什么样的公司？加密领域还处于起步阶段。五年后，许多创业者将会回首，希望自己在2023年创办了一家加密货币公司。熊市正是建设的时机，所以为什么不从今天开始？

我决定分享我的十个最佳想法，希望有人能够走出去并将它们变为现实。创意很廉价，真正具有挑战性的是付出艰苦的工作和决心去建设。Coinbase拥有在加密领域中进行大量建设的资源，但我们没有时间去建设一切，希望能够帮助周围的生态系统。我们已经投资并与一些正在实施这些创意的出色团队合作。其他创意尚未深入探索。我们需要更多的人加入我们。

Coinbase Ventures将在十月举办一个创客峰会，我们希望邀请一小群顶尖创始人或项目，他们正在这些创意领域内工作或围绕这些创意工作，面对面交流。

现在先让我介绍一下我排名不分先后的十个最佳想法。

*注意：Coinbase Ventures已经投资并将继续投资在许多这些领域的初创企业，下面是其中的一些例子（不是全部）。

Flatcoin

比特币是互联网的黄金标准，这对于货币来说是一个很好的理念。但由于它还有很大的上升空间，许多人不愿意将其用作货币。它仍然需要一些增长。与法定担保的稳定币是一个很好的中间步骤，但它们受到通胀和冻结的影响，就像法定货币一样。

通过加密技术创建一种更好的货币形式会是什么样子？它可能会是完全去中心化（抵抗审查）并跟踪消费者价格（例如美国的CPI）以保持购买力，或者接近这个目标。它可能需要由一篮子资产支持以跟踪CPI，或者使用算法方法。例如，如Truflation这样的服务甚至提供了一种可能在链上跟踪通胀的方法，进一步去中心化。在加密领域帮助构建更好的货币形式的公司或协议将产生巨大影响。

参考项目：Ampleforth/Spot、Nuon、Truflation

链上声誉（Onchain Reputation）

我们现在有了去中心化的身份验证，使用ENS，但是我们怎么知道谁是可信的？我们怎么知道一个商家是有声誉的？或者我们是否在发送给真正的艺术家？

我们可以创建一个去中心化的协议，根据链上数据图谱为每个地址或ENS名称分配一个声誉分数。这与PageRank如何根据接入链接和关联元数据图谱对每个网站进行评分类似。这可以用于借贷（去中心化的FICO分数）、商家评分（Yelp）、欺诈预防（钱包在向风险地址发送前可以警告我）、向VIP提供空投等。它需要是去中心化的协议，以防止被占领或滥用。

链上广告（Onchain Ads）

在Web 2中，最好的商业模式是广告。合理的猜测是，某种形式的广告在Web 3中会很受欢迎。Web 3中的开发人员和初创公司将越来越愿意为分发付费，但是我们能否超越传统的广告模式？在Web 2中，最大的创新是从CPM（每千次观看付费）到CPC（只有当有人点击广告时才付费）。在Web 3中，也许我们可以完全过渡到CPA（只有当有人购买或在链上采取行动时才付费）。

想象一下，任何智能合约都可以公开一些有关愿意支付多少费用将用户引导到一个函数调用的元数据。该函数调用可以接收一个引荐者地址，并支付一定的佣金。索引所有支持此广告标准的智能合约将创建广告库存。现在，拥有大量用户的钱包和应用可以选择将请求路由到哪个广告或引荐链接，并在交付用户时分享利润。实时拍卖机制将有助于找到最优价格。Web 3中可能会出现广告的其他变体，但上述仅是一个示例。

参考项目：Spindl、HypeLab

链上资本形成（Onchain Capital Formation）

ICO热潮存在一些重大挑战，包括欺诈和未注册的证券。但出于某种原因，那里曾经有一阵狂热：全球的资本形成仍然存在过高的摩擦。而且分配不均。如果你是生活在布达佩斯或班加罗尔的创业者，你获得资本的机会比生活在硅谷或克利夫兰的人要低。如果能更容易地筹集资金，我们会有更多的创业公司（不仅仅是创业公司：普通人为公寓楼或其他任何东西筹集资金）。

下一代ICO可能会是什么样子，既合法又可信赖？想象一下在Web 3中将Stripe Atlas和AngelList结合起来，不仅帮助你注册你的公司或创意（在每个国家，通过提交文件，或者作为DAO，伪名），还可以帮助你为你的创意筹集资金。每个创意都可以有一个推介材料或视频、讨论和评级、参与者的声誉以及透明的链上资金流动。民主化筹款可以释放全球范围内巨大的潜在创业能量。

加密货币的工作/任务市场（Job/Task Marketplace for Crypto）

全球只有有限的人口拥有足够的可支配收入来投资加密货币。但几乎每个人都需要谋生，并在基本生活需求上花费收入。一个更加全球和自由的劳动力市场的主要障碍是向个人跨境支付的困难。为什么不通过允许发布以加密货币支付的工作和任务，并帮助员工和承包商找到它们，来创建一个更加全球和自由的劳动力市场？

参考项目：Braintrust、Earn.com

链上隐私（Privacy for Layer 2）

二层解决方案已经开始帮助加密货币扩展，这是很好的。但大多数交易仍在公链上发生。有许多情况下，透明度是一种特性，但人们不希望经济中的大多数交易是公开的。如何在二层解决方案中引入隐私，是否可以在这里建立一个业务？

参考项目：Aztec、Polygon、zkSync

完全链上的P2P交易所（P2P Exchange Fully Onchain）

加密货币的P2P交换在新兴和不发达的市场中提供了重要的入口和出口。但P2P交易所是由可以受到压力关闭的中心化实体运营的。是否有办法构建一个完全在链上运行的去中心化的P2P交易所，具有托管、声誉和调解功能？

参考项目：Onboard

链上游戏（Onchain Games）

我不是建立游戏的专家，但我感觉我们在加密货币与游戏交叉方面才刚刚开始。玩家如何真正拥有他们在游戏中收集和使用的物品？这些物品如何在持久世界（或者我敢说是元宇宙）中存在，创造更复杂的交互和经济？早期的例子，如Dark Forest，令人兴奋。

也许可以使用Loot创建更多的游戏。这些游戏不必是大型畅销游戏。即使是简单的游戏，如A Dark Room，也可以调整以使用玩家拥有的NFT虚拟物品，在一个与其他玩家共享的持久虚拟世界中。

参考项目：Azra Games、Horizon Blockchain Games、Immutable、Plai Labs、Yuga Labs

代币化现实世界资产（Tokenizing real world assets）

1:1支持的法定货币稳定币如USDC是“现实世界资产”的首批经过验证、规模化、链上实现的表现形式。我们相信随着时间的推移，许多形式的资产可以和将会在开放、无需许可的区块链网络上体现，创造新的金融原语，具备可编程性、组合性、全球流动性、信任最小化的结算等超级能力。

许多项目今天（包括我们投资组合中的一些公司）正在研究这个领域的方法，涉及债务工具、国库券、大宗商品、应收账款，甚至高端奢侈品。

参考项目：Goldfinch、Centrifuge、MoHash、Naos、Mountain Protocol

启动和管理网络国家的软件（Software To Start and Manage Network States）

《The Network State》（作者：Balaji Srinivasan）已经加速了向初创城市和网络国家的发展。更多的网络国家将被创建，并且它们需要管理投票、治理、筹款、公民权、征税、提供服务等方面的工具。经过思考，这款软件可能对许多不同类型的初创城市和社区都有用。
