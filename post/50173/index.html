近期有一部分 NFT 投资者和大户因为一些状况或 NFT 市场变化而退圈，随之而来的是一些人对 NFT 的「唱衰」， NFT 资深玩家 wale.swoosh 发文解释了 NFT 的现状，BlockBeats 编译如下：

NFT 并没有死。

Aaron 几天前发了一条病毒式推文，提到 NFT 永远不会回到 2021 年的巅峰时期，NFT 文化已死。 但这与现实相去甚远。确实，从数据上看情况不太妙。ATL 的总用户上来看，NFT 交易量的很大一部分来自空投激励，总体上价格大幅下跌。

但我们都知道，加密是周期性的，这种情况在一个周期内是正常的。巅峰时期已经过去，少数的信徒仍然忠诚。我们在加密货市场上一次又一次看到这一情况，例如，比特币在 2018/19 年看起来也不太好，不是吗？

「好吧，事实上确实也有很多 OG 退场了」，没错，但这并不是一件坏事。许多所谓的「OG」要么用付费骗局或其他见不得人的事情毁掉了他们的声誉，要么一开始只是为了炒作。

NFT 不需要这些人。相反，新一波的创始人、内容创作者和交易者将取而代之，事实已经在向好的方向发展了，新人取代旧人。

NFT 文化不仅仅是同一群人在 Twitter space 上为猴子打 call，当然，大多数 NFT 玩家不太了解。NFT 文化是关于我们将某些感受与之联系在一起的社区。它是关于过去两年在 Web 3 中发生事情的共同记忆，这是关于标志性的短语，例如「can devs do something」。

NFT 文化一直在变化。仅仅因为某些事情或人的消失并不意味着 NFT 文化已经死亡。旧人虽然退圈，满怀热情的新人还是会顶上这个重要的位置。

很多 NFT 项目今天比在当时发展得更好，走向更远，Yuga Labs 即将发布一个元宇宙，Doodles 收购了一家屡获殊荣的动画工作室，许多项目都在幕后努力。

当然，并非一切都是完美的，仍然有诈骗、贪婪和虚假营销，也仍然存在项目持有者的背叛。但身处币圈，情况不一直是如此吗。好消息是，我们的对待坏人的意识已经提升，今天对这些坏人的谴责比当时要严重得多。

那么 NFT 真的死了吗？ 不。NFT 文化正在改变，但这不意味着它会变得更糟，NFT 盛世会卷土重来，我们最初对 NFT 的感觉也终会回来，许多今天人们正在「哀悼」的 OG 也会随之归来。

但是，这是否意味着你持有的 Mutant Cat、Crypto Doodlez 将再次回到上次牛市中为它支付的价格吗？抱歉，可能不会。但真正建成的项目会卷土重来。NFT 技术本身也不会被遗弃到任何地方。

NFT 还没结束，NFT 刚刚开始！
