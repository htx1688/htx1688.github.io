撰文：Huobi Incubator 编译：angelilu，Foresight News

由 Evmos社区联合火币孵化器和Web3奖学金主办的Evmos Momentum Hackathon已经结束，此次黑客松有 865 名参与者及 172 个项目提交，第一届 Evmos Momentum Hackathon 中 1 个项目获得大奖、12 个项目获得赛道奖、2 个项目获得迁移奖、5 个项目获荣誉奖。

大奖

Evmos StayKing House —— 仅针对 Evmos 持有者的杠杆质押服务

Evmos StayKing House 是一个去中心化金融协议，它允许质押者最大化他们的收益，并允许贷方通过使用杠杆将其资产借给质押者来获得稳定的利润。其主要功能包括：

  • 杠杆质押：杠杆质押者可以从贷方借入资产并拥有杠杆敞口以获得更高的收益。他们可以通过增加抵押品或增加债务价值来调整杠杆水平，从而将利润提升到更高的水平。由于他们从池中借入资产，因此他们面临清算风险。由于 DeFi 投资者的主要目标是利用高 APY，因此该功能对于 EVMOS 持有者来说是一个有吸引力的选择，而且他们可以自动再投资他们的资产。

  • 借贷：借贷者可以借出资产以产生相对稳定和低风险的利润。贷款人可以自由存取其资产，不受 14 天的解绑期限制。我们正在想出一些不同的方法，使贷方能够获得除高贷款收益率之外的额外奖励。

  • 多链质押（即将推出）：质押者可通过此功能获得跨 IBC 协议的质押和 DeFi 收益。此功能将作为触发器创建多链资产流入 StayKing，也可作为增加用户利润的附加因素。

Evmos 团队认为 Evmos StayKing House 将多个 DeFi 原语整合在一起，并在应用程序上有很好的端到端执行的能力。

赛道奖

元宇宙 NFT 和 GameFi

近年来 NFT 和元宇宙的蓬勃发展，Evmos 生态系统必须为用户提供不仅可以购买、出售和交易 NFT 的方式，还可以让用户使用他们的 NFT 跨链和许多不同的用例。以下项目都为 NFT 创造者和用户带来平等受益的巨大价值。

——Evmos 团队

第一名：Rumble by Orbital Apes

Rumble 是一个 Evmos 链上完全去中心化且可证明的抽奖系统。Rumble 抽奖通过让用户铸造 NFT 票来作为他们参与抽奖的入口，并通过 DIA 预言机选出获胜者获得奖品，可用于出售高价值 NFT、赠品和促销、对赌等用例。Rumble 表示，在该黑客马拉松开始前，Evmos 上没有随机预言机，其主要成就之一是帮助 DIA 团队将他们的 Randomness Oracle 技术引入链中。此外，Evmos 链上 NFT 项目 Orbital Apes 的持有者将从平台产生的所有利润中获得 20%。

第二名：Orbit Market V2

Orbit Market V2 是由 Orbital Apes 运营的公共 NFT 市场 Orbit Market 的 V2 版本，该版本提高了效率、增加了新功能并更加关注社区。目前是 Evmos 链上最大的 NFT 市场，为 Evmos 用户提供免费的基础设施来创建和交易 NFT。对于 NFT 买家，它允许他们以不同的方式进行交易，例如直接上架、举行拍卖、提出收藏要约以及捆绑 NFT 销售。对于项目创建者来说，它为他们的用户提供了交易 NFT 的基础设施，并让他们能够访问最大的 NFT 社区和潜在买家的收藏。

第三名：Omnisea

Omnisea 是基于基于 LayerZero 和 Axelar 的 Omnichain NFT Launchpad。无需切换网络即可创建和铸造 NFT。部署在 9 条链上，支持跨链 + 4 条隔离等待消息传递协议。

基础设施

每一个伟大的加密项目都依赖于坚实的基础设施。以下项目都为开发人员提供了很好的工具和资源，可以集成到他们的应用程序中并发展潜在的用例。

——Evmos 团队

第一名：Eymos Tracker

Eymos Tracker Tool 是一个网络应用程序，它扩展了 Mintscan.io/Evmos 的功能，为交易者提供额外的关键缺失功能，并促进链上数据的分析。Evmos Tracker Tool 为市场背后的实时情绪提供了宝贵的见解。该工具旨在供所有人使用，包括 Evmo 持有者和非持有者。

第二名：Parachute Drop

Parachute Drop 是一个去中心化的 DAO 工具，专门用于解决 DAO 的可持续发展。Parachute Drop 旨在解决 DAO 系统在 Cosmos 上的空投和委托存在抛售空投、未来出售代币、跨链数据三个主要痛点，其通过空投归属模型、跨链代币追踪、 IBC 集成来解决这些问题。

第三名：PyEvmos

PyEvmos 是 evmos.js 的 Python 端口，让 evmos 开发适用于更广泛的开发人员类别而不是局限于 JavaScript。PyEvmos 支持创建消息和交易，使用私钥在本地签名并通过 REST API 广播结果与原始 evmos.js 库相同。

Web3

去中心化自治组织或 DAO 改变了个人为实现共同目标而进行协作的方式。在这一类别中获胜的项目在 Evmos 生态系统促成下一波 DAO 增长和创新。

——Evmos 团队

第一名：Soul Network 

Soul Network是 Cosmos 和 Ethereum/EVM 生态系统中的跨链 DID 网络，其初始产品由三个主要模块组成，包括 Badge、Stamp 和 Soul ID：

  • Badge 是用户的 OAT（链上成就代币），是包含 NFT、DeF 以及现实世界中事件和生活经历的所有成就在链上的数字记录，它是由特定发起者发行的不可转让的 NFT；

  • Stamp 是创作者为标记特定地址集合而构建的标签；

  • Soul ID 是你的通用配置文件，也是一个灵魂绑定的 NFT，它将代表你在 Web3 中的多链 / 多钱包身份。

第二名：Neokingdom DAO

Neokingdom DAO 是创建基于 EVM、合法合规的 DAO 的基础，这些 DAO 依赖于 Oblicity（一家爱沙尼亚律师事务所）使用爱沙尼亚法律体系制定的法律框架，为具有共同目标的集体服务，用虚拟领域奖励创造价值的人。核心原则是，DAO 的贡献者可以通过个人基于时间的努力或金钱贡献进行投资，以获得代表所有权、投票权和股息权的公司股份。

第三名：XDAO

XDAO 是一个创建和管理去中心化自治组织的平台，DAO 创建者无需了解代码或能够编写智能合约就能轻松创建去中心化自治组织（DAO），存放加密资产，通过投票管理它们，出售 DAO 股票并直接与 DeFi 协议交互。

去中心化金融

DeFi 是加密货币的首批有形用例之一，自从它诞生以来，很多创新不仅取代了传统金融，而且还重塑了我们对金融服务的思考方式，这令人非常兴奋。我们很高兴看到以下用例不仅为 Evmos 生态系统带来了价值，而且为整个 Cosmos 生态系统带来了价值。

——Evmos 团队

第一名：Arbiswap

Arbiswap 是具有自动套利功能的 DEX 聚合器。 通过搜索所有 DEX 并执行交换来找到当前区块的最优路径。它是 DEX Aggregator V1，对于 DEX Aggregator V2，该团队通过使用闪贷计算循环套利并在条件匹配时执行来添加 MEV 保护和滑点最小化逻辑。

第二名：Liquidation Protector

 Liquidation Protector 是一种敲入期权协议。如果标的资产价格触及障碍，期权代币的收益将自动添加为抵押品以保护清算。该团队设计了一个带有敲入期权的市场，投资者可以在其中处理其杠杆头寸的清算风险。清算保护人不仅要管理个人投资者，还要管理市场的系统性风险。

第三名：SpaceFi

SpaceFi 是 Evmos 和 zkSync 上的跨链 Web3 平台，初始产品 DEX+NFT+Starter+Spacebase，其正在构建一个完全集成的系统，支持用户进行掉期交易、质押赚取收益、铸造或出售行星 NFT、创建或加入太空基地，以及投资新项目等，所有一切通过结合 DeFi 和 NFT 的 SpaceFi 代币经济学连接起来。其终极愿景是连接 Cosmos 和以太坊 Layer2 生态，探索资产跨链和互操作解决方案。

EVM 迁移奖

虽然参加本次比赛的许多项目都是专门为 Evmos 生态系统构建的，但以下项目将他们的应用程序从其他生态系统带到 Evmos 生态系统，让用户从中受益，我们很高兴看到他们也将现有的用户也吸引到 Cosmos 生态系统。

——Evmos 团队

Cora Protocol

Cora Protocol 是没有清算的去中心化借贷协议，该协议使借款人能够在没有被清算风险的情况下获得贷款。Cora 的贷款完全由 Cora 的保护机制保障，该机制使用期权策略来管理协议的风险。

TokenQL

TokenQL 是一个跨链索引器，可查询所有代币。它作为所有 ERC-20、ERC-721 和 ERC-1155 代币的数据层，使用标准的 Web2 工具，例如 graphql 和 RESTful API。这将为所有 DeFi、GameFi、仪表板、钱包等提供基础层数据源。

荣誉奖

这次黑客马拉松共提交了 160 多个项目，除上述项目获得奖项外，还有一些项目致力于为 Evmos 生态系统构建，我们希望继续与这些构建者合作，继续迭代他们的应用程序，以便他们能够继续为开发者和用户带来价值。

——Evmos 团队

  • Cosmos IBC NFT Crosschain Bridge with NFTHashi — 与 NFTHashi 上的 Cosmos IBC 的 NFT 跨链桥，使我们能够将现有的 NFT 连接到 Evmos，实现了一个通向 Evmos 的通用桥和 Cosmos IBC 桥的 PoC。

  • Lalalend Finance — Evmos 上第一个用于借贷和 DEX 聚合的去中心化场所。

  • MEPEvmos — MEPEvmos 是使用 Evmos 的 Web3 社区创建者，支持用户轻松发行会员 NFT 并将代币和优惠券转移给用户在 Evmo 生态系统中的朋友。

  • Dexmos.me — 是 Evmos 网络的去中心化交易所浏览器，它使用表格和图形以友好的方式显示所有交换。用户还可以连接您的 MetaMask 钱包以查看交易。

  • Calamus Finance — 是一个去中心化加密流媒体平台 — 自动化实时工资、启动释放或任何具有增强安全性和透明度的交易。
