原文作者：Shih Chun Chen, CFA，Founder of InVar Finance Marlboro Zhang，Researcher from 3 WW 3 

自熊市开始以来，现实世界资产（RWAs）一词变得越来越常见。RWA 的范围很广，从实物资产的代币化到传统金融工具（TradFi），甚至是 ESG 相关资产。当与区块链技术交织时，传统的现实世界资产可以提效、增加透明度、优化流动性管理等诸多好处，最终实现普惠社会。通过将 RWA 纳入加密货币领域，中心化金融的发展将会更为多元化。

RWA 的总锁定价值（Total Value Locked）反弹至 7.5 亿美元水平 创 7 个月新高（来源: fortunafi）

通过代币化与加密货币生态系统相结合，RWA 利用区块链的优点来迭代传统金融工具（TradFi）以实现创新，包括信用贷款、基于收入的融资、企业借贷、抵押贷款和其他资产。相较于 DeFi 或是 NFT 的 2022 年市场暴跌，RWA 代币化产品的市场需求变得更为强劲。

尽管代币化并非新概念，但 RWA 代币化的发展道路却稳步增长，为人们在熟悉的场景中使用加密货币提供了渠道。最成功的 RWA 代币化案例之一是稳定币——目前市场规模为 1270 亿美元。而稳定币的创新带来了稳定性——这有利于对抗加密货币的波动，也保持了以管理资本的效率——但以数字化美元的新形式来管理资本。

 除了稳定币，房地产和传统金融工具（TradFi）的代币化也是一个巨大的市场。这个巨大的市场吸引了许多参与者，如初创公司、政府和金融机构等都在探索房地产和传统金融工具（TradFi）的代币化市场。代币化能够赋予房地产或传统金融工具（TradFi）的交易规模、交易时限和交易流程更大的灵活性。此外，诸如可编程性、不可更改性和可组合性等区块链的原生特性，都将赋予 RWA 代币化新的范式。

通过回顾 RWA 代币化的历史，我们可以发现，当 ICO 变得混乱时，就是安全代币发行（STO）入场的时刻；而当 DeFi 和 Layer 1 变得更容易被大众使用时，代币即媒介（token-as-medium）可以向更多的投资者提供部分所有权和收益。此外，当 DeFi 由于弱可扩展性和高昂的入局成本而陷入小众市场时，我们可以从代币化的短期国库券或 RWA 抵押贷款中获得传统金融收益。因此，RWA 还处于早期发展阶段，但对加密货币的价值早已显现。 

探索前沿 RWA 应用

RWA 代币化不仅仅局限于将硬资产、传统金融业务引入链上，还有更多值得探索的地方。例如，MakerDAO 正在增加 RWA 的应用，不仅使现金流更为多样化，还帮助金融机构利用链上的高资本效率优势，以此释放流动性。这是推动 RWA 代币化新场景的突破性里程碑。

另外一个具有代表性的例子是衍生品 DEX ProsperEx，它也是 RWA 领域发展早期的一个用例，该 DEX 本身保留了衍生品 DEX 的固有形态，即包含了链上永续合约、链上期权等主要衍生品交易模式，同时其引入了 RWA 资产比如股票、债券等能够作为交易标的，并能进一步用于衍生品交易。

ProsperEx 将为衍生品交易者提供独特的机会和市场效率，其完全建立在链上，基于点对池、点对点（订单簿）两种主流交易方式，以及零知识证明技术的采用，能够以非许可、隐私的方式向所有交易者提供服务，这是其优势所在。同时其也具备拓展更多交易功能的特性，比如除了交易 FT 资产外，也将支持 NFT 等更广泛资产类型的交易。ProsperEx 有望为 DEX 在 RWA 领域的发展提供早期的参照。

RWA 代币化为 DeFi 和传统金融创造了沟通和互动的方式，以满足不同的使用场景。RWA 的市值远远大于加密货币市场，这是因为 RWA 的代币化是将流动性重新引导到加密货币的强大催化剂，并且能够对抗去中心化经济的弱点。使用 RWA 作为抵押品是代币化的流行概念，资产发起人可以使用代币化的 RWA 在 DeFi 中借贷资本。不仅如此，用户还可以通过代币化转让底层资产的所有权或相关权利。在这一应用中，链上价值和释放流动性潜力是 RWA 代币化潜力的充分体现。

正如在我们在加密货币市场所看到的，RWA 具有无限潜力，可以成为释放价值和增强生态系统可持续性的必要工具。InVar Finance 迈出了第一步，开发信任最小化代币化模型，探索了最大化代币化 RWA 的灵活性以及桥接Web2的解决方案。通过在 ERC-3525 基础上建立代币化模型，InVar Finance 信任最小化代币化模型突破了 ERC-721 和 1155 解决方案的限制，同时也为用户增加了额外的投资灵活性。

ERC-1155 v.s. ERC-3525 

想象一下，世界上每个人都可以投资 100 美元在房地产上，有自由裁量权来管理 RWA NFTs 并能够在 ProsperEx 这样的 DEX 上自由交易，而不用担心安全问题。而这正是 RWA NFTs 与传统投资相比的巨大优势。较大的数据容量和高度的可视化使得 NFT 成为 RWA 链上身份的最佳媒介。因此，DeFi 和 NFT 基础设施对于 RWA NFT 的新应用扩展极为重要。 

D 3 X，一个新的去中心化 NFT 交易所，旨在通过全套的交易基础设施为 NFT 提供终极流动性。无论如何，对于 NFT 而言，提高流动性都是必不可少的任务。因此，通过支持 NFT 项目进入一体化平台的定制流动性池，包括 NFT AMM、激励分配、NFT 发布等，D 3 X 为 RWA NFT 提供了一个即时交易 RWA 的渠道，也使新用户在保持 RWA 风险敞口的同时获得相关权益，包括 D 3 X 提供的代币奖励。

RWA：战胜挑战，勇往直前

尽管 RWA 代币化潜力无限，但仍需要克服现阶段的一系列困难。

RWA 代币化的一个主要问题是链上和链下流程的执行和协调。与 DeFi 借贷协议不同的是，DeFi 借贷协议的清算是完全自动化的，由代码管理；而 RWA 抵押品的清算（至少部分）必须在链外处理，通常是通过法律系统。而这又给债务人带来了复杂性，并会给 RWA 作为抵押品的协议带来法律纠纷。

此外，RWA 代币化的估值和流动性可能受到市场动态和投资者情绪的影响，而市场动态和投资者情绪并不总是与资产的内在价值或表现直接相关。而这会带来了有别于、或者说更甚于传统资产的波动性和不确定性。

此外，管理 RWA 代币化的法律和监管框架可能无法与数字领域无缝对接。市场还需要解决与管辖权、产权和可执行性相关的问题，以保护代币化资产、保证代币化资产的合法性。

现实世界资产（RWA）的潜力在于亚洲、非洲和拉丁美洲的新兴国家。

3 WW 3 （亚非拉Web3研究院）研究员 Marlboro 认为，目前全球正处于缺乏投资目标的阶段。随着中美对抗的加剧，许多普通投资者希望能够投资新兴市场国家。然而，由于准入门槛高、缺乏有效渠道以及外汇管制等原因，这一愿景往往难以实现。

目前，越南、印尼、印度等新兴国家的股票和房地产市场越来越受到投资者的青睐。这是因为，这些地区较快的经济增长和巨大的市场潜力为许多跨国资本投资者提供了丰厚的回报。然而，由于缺乏直接投资渠道，普通投资者无法直接受益。

区块链可以实现 RWA 代币化，并将其带往全球投资市场，以此丰富加密货币世界的投资标的，让更多的非加密原生用户参与到加密货币交易市场中，从而为Web3带来更多的新增用户以及资金，减缓比特币四年周期带来的整体性牛熊波动。

总而言之，RWA 的范畴得到了前所未有的扩张。RWA 代币化从本质上来说远胜于传统的金融工具，因为其能够改善流动性、成本效益、透明度等诸多方面。尽管 RWA 代币化的发展道路崎岖而复杂，但仍有许多从业者不懈努力、以此实现 RWA 代币化普惠社会的目标。

以加密货币以外的价值为支撑，RWA 允许在 DeFi 中引入新型抵押品。用户可以通过加密货币轻松获得房地产或短期国库券等传统金融工具的链上投资，并获得区块链以外产生的收益，或者混合收益。然而，加密货币和真实货币间的差异带来了一定的挑战。而如果能够克服这些挑战，RWA 的繁荣终将到来。

总体而言，由于原生自带的多样性和坚实的价值基础，RWA 代币化可以成为加密货币行业的新希望。除了技术之外，通过用能够证明 RWA 代币化的意义、真实可靠的证据来教育市场，同时监管、技术、工具和消费者行为也需要同步进行、跟上 RWA 代币化的步伐。
