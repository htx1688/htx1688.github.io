在疫情爆发的头两年，「数字游民」的概念一度进入大众视野，凡是利用互联网远程工作，将工作和生活融为一体其从业者被称为「数字游民」。「数字游民」不算新，但在 Covid-19 暴发后，这种方式则变得更为流行。

而在 Web3 世界，这种方式已经见怪不怪，分布式工作已经成为一个全球化组织的必备素质，而在摆脱传统的坐班组织形式的背景下，收入来源由多个模块组成，而不依赖于单一全职工作，——斜杠青年也不断涌现。

Web3 世界中 DAO（一种基于区块链的、去中心化的全新组织形式）组织的出现，给了越来越多的公链项目方趋向于用此种方式促进生态发展的灵感，「斜杠」的形态可以是在业余时间成为项目的贡献者，也可以是参与黑客松赢取奖励、也可以是纯粹的兼职，从 Web3er 的角度来看，区块链，不止是分布式办公，它还有更多的工作方式。

对此，TinTinLand 采访了两位在波卡生态项目的开发者，与他们聊了聊怎么进入 Web3 世界以及开启另一种身份的故事，未来 TinTin 也将继续此系列，分享 Web3 开发者的斜杠生活。

小蔡 — Dorahacks 核心开发者

我今年研究生刚毕业，目前是去中心化全球极客社区 Dorahacks 的一名开发者，主要负责 dorafactory 平行链的开发和维护。

2018 年区块链第一次在国内大火，就读信息工程专业的我第一次认识到区块链与比特币，但研究更多的还是联盟链。但联盟链其实和传统的开发并无太大区别，甚至在慢慢演化为一种传统信息系统开发。所以在一开始我想接触整个 Web3 的时候，我在 2021 年初加入了 Dorahacks，做区块链相关的运营工作（当然这也是我读研期间的远程兼职工作，并不影响我的学习生活），这才真正进入了 Web3。

在 Dorahacks 实习的期间，是我慢慢建立起对公链生态认知框架的一个重要时期。因为当时我对公链的技术生态是比较陌生的，很多技术名词充斥眼前，让我眼花缭乱。我的工作就是去和参加各个公链黑客松的项目方进行对接，甚至筛选掉一些不靠谱的项目，自己做一些直播安排。因为 Dorahacks 平台的原因，我可以接触到很多比较前沿的公链生态，包括 ETH、Flow 、Solana 等等。在这个运营工作的过程中，我慢慢建立起对整个 Web3 行业各个生态的认知，也为后来加入到公链开发奠定了一些基础。

在研究生毕业之前我也纠结过到底是为进入互联网大厂做准备还是一毕业就 All in Web3。最终选择 Web3 的原因一方面是因为就业形势，另一方面更多还是意识到 Web3 存在很大的人才缺口。

选择成为 Substrate 的大使，其实一开始并不是为了这份 Grant，更多的是抱着学习的心态，想要进一步深入了解波卡生态。

在大使的工作是教研组，主要是担任开发者课程的助教、翻译与创作一些有关波卡生态的文章。整个工作其实不会很忙碌，没有占用到自己的本职工作时间。因为我本身的工作就是波卡开发，所以和大使需要做的内容重合度比较高，所以我觉得还是相对轻松的。当然在这个过程中收获也很多，比如我在工作中需要去研究某个功能模块的时候，经常会去看 Substrate 文档，波卡 wiki 以及 Substrate exchange，而这些中文资料都非常少，也就成为翻译或者创作的素材。

这次大使是以 DAO 的方式来进行，虽然整个模式没有完全的理想化，但也是初见成效的，Grant 的存在，会让我的责任感更强，因为工作证明是大使积分的依据，而这份依据也是我获得多少 Grant 的决定因素。同时波卡官方也会给大使一些福利和个人 ip 曝光，这对于大使和波卡整个社区来说都是双向互益的。

我对于想要在不影响自己本职工作的 Web2er 的建议就是，可以先尝试做一些 Web3 运营的一些工作，一方面压力会比较小，另一方面又能慢慢接触到 Web3 的东西。对于有一些开发经验同时时间比较充足的小伙伴来说，我觉得可以先学习 Solidity 开发，尝试接一些 Web3 的兼职工作（兼职最是自己擅长或者感兴趣的，既不会累，又能更好提升自己的开发能力），这样可以更快速地进入到这个行业中。

Bob — Biforst 核心开发者

我是 Bob, 我⽬前是⼀名全职的波卡⽣态的区块链开发者，今年年初决定出来进⾏全职做区块链开发，在这之前我是在⼀家券商从事后端核⼼业务的开发。

我个⼈是⼀个技术极客，是真正对技术有兴趣的⼀个⼈，也⾮常喜欢把技术应⽤到⾃⼰的⽇常⽣活中来。区块链⾏业是我觉得唯⼀能让程序员发挥最⼤⼒量的⾏业，有⾮常多我感兴趣的技术，像核⼼的共识协议、异步编程、虚拟机、云原⽣、Serverless 等。我⽬前主要做的⼯作有 Substrate 上的 Runtime 开发, 钱包 Runtime 适配以及后端服务中间件等开发⼯作。

之所以成为波卡大使，也是因为想把自己掌握的技术知识反哺给同样想在区块链行业、波卡生态找寻发展机会的技术爱好者们。

其主要工作是在教研组担任 Substrate 入门课的助教，大使的工作多数都是可以认领的，我会选择感兴趣的来做，我通常都是根据工作相关性比较大的，占用的精力不会很多，主要还是学习和交流，大家的灵感能够互相碰撞。 

在这之前，我也在 NEAR 生态中做过贡献，会做一些技术文档的翻译、参加 NEAR 组织的黑客松，也是以 DAO 的形式申请 Grant。最近也有在研究 SUI 和 Aptos，学习 Move 语言，目前也申请了测试网的验证节点。

Grant 并不是我选择投入的主要因素，更多的还是考虑公链的技术是否吸引我，NEAR 的分片技术、合约协议与 SDK 我都很感兴趣，也会看公链的发展前景，考虑会不会在这个生态做一个项目。

如果要给行业的开发者提一些建议的话，我觉得做区块链开发相关工作一定要有关注的重点。Web3 行业涉及到的方向还是很广的，可以从自己最擅长的方向入手，要保持一定的专注性，在这个前提之下可以去凭借自己的技术贡献去申请一下项目的 Grant / 技术大使，这样自己有收益的同时，也是对工作之外的技术方向的补充。

作为一个开发者，要对自己技术水平有一定认知，如果有一定的项目经验，那么可以参与各种 Grant 。如果是新手不是非常建议去做黑客松，因为这需要你对链的了解、开发要求都比较高，新手可以先静下心来学习, 从简单的事情做起, 比如: 文档翻译或者修复简单的 issue 等。最好关注一些热门的新项目，因为项目初期提供的工作任务一般都是较简单的，发展一段时间后其实简单的任务都已经被人领完了。

你申请 Grant 了吗？

斜杠，代表就业不止一种形式。在 Web3 世界中，至少在这两位开发者的眼中，斜杠更多的是以多重身份参与到项目建设当中去，这是出于对项目价值的信心，也是不断提升自己的证明。他们有强大的学习动力和能力，能在某一或某些领域专注且长期投入，也坚信垂直深耕的意义。

但其实，目前，已有不少公链推出 Grant 计划，是去中心化应用的创业基金之外的一种补充奖励计划，面向奖励人群更广，门槛更低。

以太坊的「生态系统支持计划」

ESP（ecosystem support program）是以太坊生态基金会面向公众的资金分配部门，支持加强以太坊基础的开源项目。ESP 有两种 Grants 形式：

  1. Small Grants，指的是 3 万美元以内的 Grants，有一个简化的申请和评估过程，在项目提交后两周左右做出决定；

  2. Project Grants，不设资金上限，但需要经过更仔细的调查和更长的决策时间（2 个月左右）。

Polkadot 的 Web3 基金会

Grant 是 Web3 基金会于 2018 年启动的资助计划，该计划用以帮助去中心化技术堆栈的团队。主要分为三部分：

  • 议案（Proposal）支出一般针对较大的项目，如软件开发和线下活动等，金额较大，申请和审批流程较为复杂，需要缴纳一定的押金；

  • 小费（Tip）一般针对较为轻量的贡献，比如翻译、设计、线上活动等，申请流程简单，一般金额较小；

  • 赏金任务（Bounty）由提出者指定任务内容，有专门的管理人跟进。

Flow 基金

Flow 基金是为下一代游戏、应用程序和数字资产提供支持的 Web3 平台 在 2022 年 5 月推出的 7.25 亿美元生态基金。生态基金划为四部分用途，其中一部分即为开发者 Grants，主要针对开源维护、开发者工具和服务、技术教育提供 Grant。

NEAR 协议的 Bounty 奖金计划

NEAR Grant 主要针对（1）开源组件服务；（2）创建教程、视频、研讨会或编写文档（3）用于地方、国家和全球的聚会和培训提供 Grant。

Chainlink 去中心化预言机网络

Chainlink 社区激励计划为开发团队提供奖金，激励他们为 Chainlink 生态开发工具或集成 Chainlink 技术。分为五个奖金池：

  1. 社区激励奖金，授予 Chainlink 生态中的开发工具和基础设施；

  2. 集成奖金，授予为 Chainlink 网络提供各种技术和服务的项目；

  3. Bug 赏金计划，授予查看 Chainlink 核心代码的开发者和安全工程师；

  4. 研究计划，授予学术研究者用以开展并发布创新研究；

  5. 社会影响力奖金，授予利用智能合约和链上数据造福新兴市场和全世界各地人民的非盈利组织和机构。创者者可以随时在官网上提交自己的项目。
