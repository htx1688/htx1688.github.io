直到 2021 年底，OpenSea 是购买和出售 NFT 的平台。 作为最初的 NFT 交易市场，它利用其先发优势成为最大的同类平台，在 2022 年 1 月 9 日达到了每天 94,000 个用户的峰值。

尽管按用户数计算，OpenSea 在 NFT 交易市场上仍然遥遥领先，但已经出现了几个新选项，它们正尝试以不同的功能、用户体验和收入结构来抢占市场份额。

截至 2022 年 9 月，如果用户想购买或出售 NFT，有 4 个平台可供选择：

  • OpenSea

  • X2Y2

  • LooksRare

  • Magic Eden

在本文中，我们将解释了解这些 NFT 交易市场的优势、劣势和战略方向的最重要特征。

NFT 交易市场的特征

  • “multichain” 是什么意思？

交易市场平台需要在用户所在的地方出现。尽管以太坊是 NFTs 最大的生态系统，但 Solana 也有一个活跃的 NFT 社区，Polygon 的社区也在增长。下图显示了平均每天大约有 12,000 名用户在 Solana 上与 Magic Eden 互动。

Magic Eden x OpenSea, Solana - Source: Footprint Analytics

随着越来越多的区块链项目走向多链化，仅限于一条链的交易市场将在竞争中失去优势。OpenSea 和 Magic Eden 正在开发多链功能，而 X2Y2 和 LooksRare 仍专注于以太坊链上。

  • 什么是白名单管理？

白名单是为即将到来的 NFT 铸造保留一个位置的过程。它通常包括几个步骤，以在实际推出期间保持社区参与和项目互动。

这项活动的管理通常涉及项目团队的大量工作。例如，它需要跟踪符合白名单条件的 Discord 频道角色。手动或在不同的平台上执行此操作是一场管理噩梦。

Whitelist section on Magic Eden, with the Discord “role” column

Magic Eden 已介入提供工具来简化此过程，提供与 Discord 的集成以使此过程自动化，并在平台上独占铸币的过程。这有助于提高用户保留率和对其服务的利用率。

  • 什么是 NFT Launchpads？

Launchpads 是交易市场的一部分，它突出了在其平台内铸造的收藏品。通常，当一个项目出现在特定平台上时，它会在管理所有步骤方面提供一些帮助，直到实际的收藏品推出为止。Magic Eden 允许创作者通过它来铸造他们的收藏品，同时为创作者提供技术和营销知识。

Magic Eden Launchpad Application page

没有其他平台开发过这样的系统。OpenSea 使创作者能够通过他们的智能合约铸造收藏品，但仅此而已。创作者仍然需要处理所有的营销以及实际的铸币、投币的管理。

Minting page at OpenSea

  • 什么是 NFT 私人销售？

一些 NFT 交易市场提供支持 NFT 私人销售的功能。这是一种只有特定地址才能购买列出的 NFT 的销售类型。这就像场外交易（OTC）销售，由于发生在市场平台上，具有额外的安全性。

Private sale on X2Y2

在 X2Y2 的案例中，它不对这项服务收取任何费用，并且在这项交易中也免除了创作者的版权费。这一功能对于习惯于直接向其他收藏家进行销售的用户，或者在买方想要协商一个不同的价格，而买方又不希望在搜索上轻易获得的情况下，是非常有用的。

  • 什么是上线条件挂单？

条件挂单允许卖家在以下条件下出售两件商品：如果其中一件已售出，则另一件将从列表中删除。 LooksRare 是唯一提供此选项的 NFT 交易市场。

Conditional Listing on LooksRare

例如，当交易者试图出售具有 NFT 系列特定特征的物品时，只想出售其中一个，这个就很有用。

  • 什么是可选的特许权使用费？

版税是每次出售收藏品时支付给 NFT 收藏创建者的费用。它们是从卖家收到的交易总额中扣除，通常在 2.5% 至 7.5% 之间。

Optional Royalty on X2Y2

现在，买家可以选择在几个 NFTs 交易市场上支付或不支付。发生这种情况是因为版税没有编码在 NFT 集合的智能合约中，而是部署在市场本身中的一项功能。所以 X2Y2 开始在他们的网站上直接购买时提供此功能。

  • NFT 市场交易 tokens

治理 token 是一些 NFT 交易市场的特点。提供它的人还为其 token 持有者制定了费用分配计划，以激励交易活动。

LooksRare rewards page

虽然策略和价值可能会有一些变化，但奖励系统基本相同：用户通过挂单、买卖 NFT 与市场互动来赚取 token，然后他可以将其兑换成不同的加密货币，或将其质押在市场上以获得更高的奖励。

  • 什么是收藏品报价？

当买家想要购买集合中的任何物品而不是特定的 NFT 时，就会出现这种类型。 当卖家接受买家的报价时，商品即被购买，订单完成。

Collection offer

交易者通常使用此功能来获取低于底价的物品。

  • 什么是批量订单？

批量订单是交易者使用的一项功能，他们希望购买许多商品并优化流程，使其更快、更便宜。

Bulk Swaps on Magic Eden

交易者可以选择每件商品的最高价格或他想购买的商品数量，然后平台就会进行批量订单。

  • 费用结构

每个 NFT 市场都有不同的费用结构。用户数较多的平台收费较高（OpenSea 为 2.5%；Magic Eden 为 2%）。这些也是不与交易者分享任何部分收入的平台。

X2Y2 和 LooksRare 与其 token 持有者分享费用收入。因此，尽管每天的交易数量较少，但由于在其上交易的物品平均价值较高，拥有 token 并分配收入给用户带来了收益。

X2Y2 Trading Fees

卖方支付这些费用（意味着他的销售程序等于销售面值减去平台费和版权费）。正如我们之前看到的，这些费用是可变的，每个平台都有特定的管理（X2Y2 现在提供 0% 的交易费用）。

顶级 NFT 交易市场概况

OpenSea

OpenSea 由 Devin Finzer 和 Alex Atallah 于 2017 年创立。它是 NFT 市场中的领导者之一，拥有超过 200 万个收藏品和超过 200 亿的交易量（仅 2022 年就超过 70 亿）。

Opensea Main Page

OpenSea 将其收藏品分为几个类别：艺术（从社交媒体账户的个人资料图片到抽象作品）、域名（像 handsome.eth 之类的名称，可以链接到区块链地址）和音乐。

OpenSea 现在已经部署在 Ethereum、Polygon、Klaytn 和 Solana 区块链上，并支持加密货币支付和信用卡购买。

  • 优势

OpenSea 的主要卖点是它是第一个也是最大的 NFT 市场，所以它依靠用户数量来激励 NFT 挂单。其想法是，推出新系列的最佳营销方式是让它在拥有最多观众的平台上可见。

因此，如果大多数收藏品都在努力成为 OpenSea 上经过验证的收藏品，那么收藏家自然会关注上线的新收藏品。

按照这一战略，Opensea 正试图通过迁移到 Solana 链上来巩固其作为最大交易市场的地位，该区块链是加密货币中第二大 NFT 社区的所在地。

  • 劣势

OpenSea 的一个大问题是与其智能合约互动的成本。在很长一段时间内，它是最大的ETH gas 费用消费者之一。

Gas consumption by protocol - Source: Gas Guzzlers

这为竞争者以更好的整体交易条件吸引用户，并引发了他们协议的更新，称为 Seaport。这套新的智能合约于 2022 年 6 月发布，其耗气量比之前的版本小38%，gas 消耗量比之前的版本减少了 38%，使得挂单和购买更便宜。

X2Y2

X2Y2 是以太坊区块链上的 NFT 交易市场。于 2022 年初推出，其座右铭是成为用户的平台，为用户服务。它是交易量的领导者，其成功部分归功于其奖励计划，该计划为用户提供了原生 token X2Y2 的一部分。

X2Y2 Rewards page

自推出以来，它已经奖励了购买和出售 NFT 的用户。它将交易费用转给决定在平台上质押平台 token 的持有者。请注意，这也造成了清洗交易的问题（下文有解释）。

X2Y2 也是整体功能最多的平台之一。

  • 优势

X2Y2 专注于用户、买家的购物体验，为他们提供不同的额外购买选项，例如设置可选的版税值。这一功能使买家能够决定是否要向收藏品的创作者支付版权税，从而有可能以更便宜的价格购买该物品。X2Y2 积极将其费用推销为市场最低，甚至开展 0% 交易费用的活动。

  • 劣势

由于在平台上进行交易的奖励，X2Y2 有很大的洗盘交易量，用户在账户和农场激励之间来回翻转 NFT。这意味着许多交易都不是真实的，这使初学者很难评估某些 NFT 系列的市场价值和需求。

X2Y2 也没有表现出任何主动性来超越 Ethereum，在一个不那么拥挤的环境中复制其模式。

LooksRare

LooksRare 于 2022 年初推出，作为社区首个 NFT 交易市场，并积极奖励交易者、收藏家和创造者的参与。带着这个想法，它向符合条件的 OpenSea 用户推出了 “7 天窗口空投” 的 LOOK token，以吸引他们在这个新平台上进行交易。

Listing Rewards on LooksRare

它实施了一项奖励计划，在 LooksRare 上购买或出售 NFT 的用户可以获得 LOOKS token。然后，来自平台的交易费（销售额的2%）由 LOOKS 质押者赚取。此外，创作者在销售时也会收到他们的版税。

LooksRare 的开发人员不依赖以前的材料（例如来自 Opensea）编写他们的智能合约。 因此，它们可以使用模块化系统进行定制，以便随着时间的推移推出新功能。

  • 优势

NFT 市场上的一些现在标准的功能在 LooksRare 上首次亮相。

  • 收藏品报价：提出涵盖所选系列中所有 NFT 的报价

  • 特征报价：购买集合中具有特定特征的任何 NFT

  • 多重取消：在一次交易中取消多个未结订单

并继续拥有专门的独特功能，如有条件的挂单，用户可以列出两个 NFT，如果其中一个被售出，另一个将自动从市场上删除，无法购买。

  • 劣势

LooksRare 在最初的启动阶段向以前的 OpenSea 用户分发 LOOKS token 后，无法得到更多的欢迎。它仍然只专注于以太坊区块链，为更复杂的交易者提供利基方法。与 X2Y2 一样，LooksRare 也面临着洗盘交易的问题，希望这对平台本身来说不是问题（因为它收取了交易费用），但它使用户对流动性和地板价数据产生了怀疑。

Magic Eden 

Magic Eden 于 2021 年 9 月推出，迅速成为 Solana 上领先的 NFT 市场。

自从在 Solana 上占据主导地位后，Magic Eden 也开始向以太坊扩张。 目前，用户可以在他们的启动板上铸造收藏品并使用 ETH 支付，即使收藏品部署在 Solana。 它还开始在以太坊上启用铸币。

Mint calendar filtered by ETH Blockchain

跨链功能的缺失部分是让 Solana 用户购买使用 SOL 支付的 ETH NFT。 这将解锁 Magic Eden 社区从 Solana 区块链本地购买这些 NFT 的可能性。

  • 优势

Magic Eden 有几个非常独特的功能。 这包括一个启动板，允许用户在几分钟内创建 NFT，而无需丰富的 NFT 知识或开发经验。 该市场还帮助营销其启动板上的特色项目。

Launchpad

另一个只有在 Magic Eden 上才能找到的独特功能是白名单管理。 Magic Eden 支持即将推出的精选列表，并为其白名单过程提供软件管理。

Whitelisted drops managed by Magic Eden

  • 劣势

Magic Eden 是 Solana 上的主导平台（占链上 NFT 交易的 80% 以上），目前还没有任何真正的竞争。然而，Magic Eden 缺乏真正的是区别于以太坊上其他平台的用户体验。

总结

随着 OpenSea 巩固其在 NFT 垂直市场的主导地位，许多活跃在该行业的人注意到了独特特点和功能的机会。

随着 NFT 行业的发展，用户可能会选择几个强大的市场选项，每个选项都有自己的优缺点。
