近期，AlienSwap 宣布推出探索者路线图，并公开其完整产品版图和空投计划，其希望通过一揽子 Game Changing 式的创新产品，与业内巨头 OpenSea 和 Blur 展开竞争，开启新一轮 NFT 牛市。

AlienSwap NFT 探索者计划路线图一览

AlienSwap NFT 探索者计划包含一系列即将发布的创新产品和空投计划，其中包括推出 NEON-ZERO NFT 探索实验，推出 NFT 市场第 1 个平台级无代码创作工具 Create + ∞，推出 NFT 创新交易区 Mirror Market，上线外星人发射器 Alien LaunchPad，发布太空竞赛空投及支持多链测试网部署。

NONE-ZERO NFT 实验

NONE-ZERO 是一项有趣的 NFT 社区实验，所有参与 NONE-ZERO 实验的用户将免费获得一个创世纪念 NFT 空投和 ALIEN LaunchPad 项目的签号奖励。

用户可以通过完成任务等方式免费获得更多签号。持有的签号越多，参与 ALIEN LaunchPad 的分配机会就越高。

据悉，NONE-ZERO 实验将在未来一周内发布。

无代码 NFT 创作工具：Create + ∞

Create + ∞ 是 AlienSwap 即将推出的 NFT 市场首个平台级无代码 NFT 创作工具，旨在释放 NFT 创作的边界。NFT 创作者和艺术家可以通过 Create + ∞ 在 AlienSwap 上自由创作 NFT 系列，无需编写代码即可快速部署 NFT 智能合约。

其功能特性包括：

●支持无代码创建 NFT 智能合约

●支持在以太坊主网和 EVM 兼容网络上部署

●一站式批量生成 NFT 系列并设置稀有度

●支持超过 10 万个 NFT 的一次生成

所有通过 Create + 创建的 NFT 未来都支持 在 AlienSwap Mirror Market 创新区低成本交易。AlienSwap 将向早期参与的创作者提供空投激励。

Mirror Market | NFT 交易创新区

Mirror Market 是 AlienSwap 即将推出的 NFT 交易创新区，旨在改造现有 NFT 交易机制。

以太坊 NFT 熊市有很多表现，大量 NFT 没有玩家交易，大量 NFT 活跃玩家离场，大量的 NFT 创作者流失。这些既是表象也是原因，但高昂的 Gas 费用成为 NFT 市场牛市复苏的重要障碍。

95% 的 NFT 单价低于 200 美金，我们以此为例，计算交易成本。在以太坊买卖单价 1-200 美金的 NFT，双向交易平均需要花费至少 50 美金的 Gas，Gas 支出占交易成本的 20% -5000% 。极高的参与成本，成为 NFT 牛市迟迟无法到来的重要原因之一。然而，因为以太坊二层交互和 token 交易的活跃，Gas 将长期居高不下。

Mirror Market 允许用户以不到 1/10 的 Gas 成本，交易以太坊 NFT，从而提高流动性并推动 NFT 市场的发展。它将支持所有通过 Create + ∞创建和发行的全部 NFT 系列进行低成本交易。

多链测试网部署

AlienSwap NFT 市场将根据社区需求部署和支持 Layer 2 链的测试网。 这一持续的过程将于 6 月开始，初步支持 Scroll、Linea、Mantle 等。Create + ∞ 平台也将部署在上述测试网络上。

外星人发射器｜ALIEN LaunchPad

除了进入成本，NFT 牛市和 NFT Summer 的到来，需要更多创新、有生命力的 NFT 社区驱动型项目，需要更多爆款 Alpha。

ALIEN LaunchPad 通过更创新、更公平的参与机制，发掘最有创造力的 NFT 创作者。并会尝试将更多全球知名 IP 带入Web3，为 NFT 社区引入更多新的 Alpha NFT。

Space Race Airdrop

NFT 太空竞赛空投将是 $ALIEN 正式发行前最受社区关注和规模空前的一轮空投活动，太空竞赛的奖励规模将是阶段 1 的 2-5 倍。

参与 NFT 太空竞赛，将获得总量为第 1 轮 2-5 倍的空投，并赚取未来的空投奖励和以太坊奖励。

行业早期，总是需要新叙事、新玩法和鲶鱼效应来打破垄断，在行情底部推动新一轮繁荣。我们将对 AlienSwap 各项计划的落地保持关注。
