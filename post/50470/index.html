美联储将于 4 日凌晨 2: 00 公布 5 月利率决议，当前加息 25 个基点概率为 87.4%

Sui Token 跨链桥现已上线

Odaily星球日报讯 Sui Token 跨链桥现已上线，链上任何 Wormhole 支持的资产都可直接跨链至 Sui 网络。

Crypto.com 已推出基于 ChatGPT 的AI用户助手 Amy，将提供有关特定代币和项目的实时信息

Odaily星球日报讯 加密交易所 Crypto.com 现已推出基于 ChatGPT 的 AI 用户助手 Amy，可以提供有关特定代币和项目的实时信息，以及价格清单和历史事件的详细信息。目前，Amy 正在通过具体案例进一步训练，以便为 Crypto.com 用户提供更准确和相关的结果。Amy 正处于 Beta 阶段，目前可供使用 Crypto.com/Price 的部分用户使用。在接下来的几周内，对 Amy 的访问将稳步扩展到更多 Crypto.com 用户。随着时间的推移，Crypto.com 还将向 Amy 介绍更多 Crypto.com 体验，包括在 Crypto.com 应用程序中。

Web3身份服务Polygon ID 已完成三项更新

Odaily星球日报讯 据官方推特，身份基础设施工具 Polygon ID 已完成三项更新。开发者可以更轻松地将DID集成到DApp中。

其一，与现有工具和项目集成，借助 SelectiveDisclosure 功能，开发者将有权共享凭证中的选定字段，实施新的用例。新的 JS SDK 使开发者能够在多个设备（Web 钱包、浏览器扩展程序等）中创建身份客户端，该 SDK 包括凭证发行功能，允许从本地设备进行无服务器发行。

其二，改进用户体验，新的发行人节点用户界面使设置、管理和使用发行人节点更加容易。在终端用户方面，现在移动端 SDK 和参考实施应用程序（Polygon ID 钱包）中支持推送通知。

其三，为开发者提供新工具，包括 JWZ Validator、部署在PoS主网上的 Polygon ID 智能合约、移动端 SDK 的 Android 和 React 版本与在 W 3 C 注册的 DID。

数据：巨鲸地址过去两天内用 1628 枚ETH买入约 6.4 万枚 RPL

Odaily星球日报讯 据链上观察员余烬数据监测，巨鲸地址“rethwhale.eth”在过去两天内以 1628 枚买入 63, 765 枚 RPL（约 310 万美元）。最近一笔交易是 13 小时前，该巨鲸将 700 枚 ETH 交易为 2.7 万枚 RPL，持有量在 RPL 持有者之中排 22 名。

此外，该巨鲸还将 5052 枚 ETH（940 万美元）存入 Rocket Pool 换成 4726 枚 ETH。

NFT 借贷协议 Blend 贷出 ETH 总量突破 1 万枚

Odaily星球日报讯 Dune 数据显示，Blur 平台昨日推出的 NFT 点对点永续借贷协议 Blend 贷出的 ETH 总量已突破 1 万枚，现为 10, 314 枚，共撮合 911 笔借贷，其中活跃贷款交易量 454 笔，活跃贷款 ETH 总量为 6, 047 枚，成为以太坊区块链上按数量和用户计算排名第一的 NFT 借贷协议。

此外，黄立成（matchbigbrother.eth）仍是目前 Blend 协议的最大放款人，目前累计完成 40 笔贷款，总贷款额为 1, 034.5 ETH。

元宇宙GameFi与 NFT

韩国游戏发行商 Neowiz 推出 1000 万美元的 Polygon 游戏加速器

Odaily星球日报讯 韩国游戏开发商 Neowiz 希望通过其加密子公司 intela X 新的 1000 万美元游戏加速器，为 Polygon 上的下一代Web3游戏提供资金。

此前报道，Intella X 是由 Polygon 与 Neowiz 合作推出的，今年早些时候，Intella X 从 Polygon Labs、Animoca Brands、Magic Eden 和 WeMix 等投资者那里筹集 1200 万美元。（Decrypt）

Merlin Protocol 在Arbitrum上发行“RWA”NFT 已售罄

Odaily星球日报讯 Merlin protocol 在 Arbitrum 上发行的第一个 RWA NFT-HashNFT 已于 5 月 3 日全部售罄。

Merlin Protocol 通过预言机和 NFT 技术实现将 RWA（现实世界资产）带入Web3的世界，提供了新的 DeFi 解决方案，主要专注于 Asset-Backed NFT 的发行以及撮合交易。

HashNFT 是其 Asset-Backed NFT 的第一个用例，通过将BTC算力上链发行成 NFT，持有者可通过加购算力包的模式获取有效期内的 BTC 算力产出收益。

Aptos基金会公布第一轮生态赠款接受者，总赠款超过 350 万美元

Odaily星球日报讯 Aptos 基金会公布第一轮生态系统赠款接受者，向 50 个新项目团队提供总额超过 350 万美元的赠款，以扩展 Aptos 区块链。

本次赠款设计四个重点领域，包括开发者工具、基础设施、教育与生态系统，包括 Mokshya、Pontem、Aries、KYCDao、Overmind、Aptos Eden、Tsunami 和 Topaz 等新项目。

此外，Aptos 生态系统赠款申请第二阶段已于 4 月开放。

加密风投公司 Credibly Neutral 完成 550 万美元融资，Solana和 Polygon 创始人等参投

Odaily星球日报讯 加密风投公司 Credibly Neutral 完成 550 万美元融资，Solana 和 Polygon 创始人等参投。Credibly Neutral 是一个专注于早期阶段的基金，将在协议、基础设施和加密软件即服务（SaaS）项目上投资 5 万至 25 万美元。投资兴趣包括 Layer 1、Layer 2、DeFi、开发者工具、安全软件、分析工具以及法币出入金解决方案。（CoinDesk）

VEGA 完成 200 万美元种子轮融资，Super Web3担任财务顾问

5 月 3 日，Web3娱乐科技平台 VEGA 宣布完成 200 万美⾦种⼦轮融资，本轮融资由 Arcane Group 领投，New Wheel Capital，行业知名投资⼈等共同参投。本轮融资资⾦将⽤于扩⼤运营和开发团队，拓展全球市场，推动技术创新，建⽴合作伙伴关系。

据悉，VEGA 核⼼团队来自 TikTok、京东⾦融创始团队，由资深加密原⽣⼯程师、爱奇艺、亚洲顶级制作⼈、媒体营销团队组成。

Super Web3创始人 Spencer Fan 表示：“Web 3.0 行业整体的投资侧重点正在逐渐由基础设施层向应用层转变。一流的创业团队，可以选择恰当的产品形态，成功构建必要的商业关系，让区块链技术发挥其应有的价值，为应用场景赋能。VEGA 团队兼具一线互联网企业及 Web 3.0 行业的丰富实操经验，有实力推动其娱乐平台产品服务尚不熟悉区块链技术的海量用户，成为娱乐科技赛道的明星企业。我们非常荣幸能长期支持 VEGA，期待共同成长。”

硬件钱包制造商 Tangem 完成 800 万美元融资，Shima Capital 领投

Odaily星球日报讯 硬件钱包解决方案提供商 Tangem 完成 800 万美元融资，Shima Capital 领投。此轮融资资金将用于扩大其产品范围。（PRNewswire）

据此前报道，加密货币智能卡生产商 Tangem 在 2019 年 1 月完成 1500 万美元种子轮融资，投资方为 SBI Holdings 全资拥有的数字资产风险投资机构 SBI Crypto Investment。

全球政策

法国参议院：或允许已注册加密交易所通过 KOL 进行付费宣传

Odaily星球日报讯 根据周三法国参议院经济委员会达成的提案，如果加密货币公司已经注册或获得监管机构颁发的许可证，则可以向 KOL 支付费用进行宣传，该修正案将在下周讨论时提交给参议院全体会议。

据悉，该法案须得到参议院和议会的同意才能通过成为法律。

彭博社：不丹计划设立 5 亿美元加密矿业基金

Odaily星球日报讯 彭博社报道，不丹投资部门计划和纳斯达克上市公司 Bitdeer Technologies Group 合作为其即将推出的 5 亿美元基金寻求投资者，面向机构投资者的筹资活动将于 5 月底开始，目标是利用不丹丰富的水力发电建立 0 碳加密挖矿基地。

不丹投资部门 Druk Holding&Investments 首席执行官表示，挖矿是不丹参与加密领域风险最小的方式，目前该国将专注于比特币挖矿。
