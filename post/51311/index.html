近日，日本通讯巨头 Line 旗下 NFT 子公司 Line Next 计划下半年在其 Web3 游戏平台发布首批五款游戏，坐拥数亿用户的 Line 又将在 Gamefi 领域掀起什么样的浪潮呢？

Gamefi 的价值模型有何不同？

  • 最传统的游戏厂商盈利模式：付费游戏模式

游戏厂商制作游戏，吸引玩家付费购买以访问游戏全部内容，通过销售游戏产品和 DLC（可下载内容，“拓展包”）来获取利润，如 GTA、GOW（战神）均采用此模式，游戏厂商的目标是卖出更多的游戏产品

  • 近十年出现的盈利模式：免费游戏模式

游戏下载和进入免费，通过出售游戏内产品进行盈利，如皮肤/装备/广告，LOL 和 CSGO 均采用此模式，吸引到的玩家人数越多，则出售的皮肤/装备就更多，游戏厂商的目标是吸引到更多玩家付费。

  • 在 Web3游戏中出现的新模式：资产所有权模式

游戏厂商制作游戏并在游戏内设计经济模型，游戏需要具备一定的可玩性和趣味性来吸引玩家和铸造游戏内资产（NFT），游戏厂商的目标是关注游戏内资产的价值。

GameFi 凉了吗？

相比于说 GameFi 凉了，不如称之为人们对现阶段市场上的 GameFi 失去了兴趣。

为什么会导致玩家失去兴趣？  

其一，游戏玩法本身很无聊。

如果一款游戏本身没有高质量的玩法，那么玩家只会专注于赚钱，然后感到无聊，最后选择退出。2021 年热度很高的 Web 3 游戏 Axie Infinity，游戏玩法设计比较简单，就玩家忠诚度而言，完全无法和 LoL 或者 Dota 相提并论，所以在其爆火一段时间之后，玩家也迅速流失。

其二，游戏内玩家缺乏自由度。

GameFi 游戏通常在自定义选项方面受到比较大的限制，极大地阻碍了玩家的创造力，反观魔兽可以自定义地图，而给玩家们带来无限想象力。此外现有的 GameFi 大多都是中心化运营，只有项目方可以更改游戏规则。

最后，游戏内还设计了一些迷你经济系统，往往被认为是 DeFi 的延续而非新的游戏类别，也加速了玩家失去兴趣。

所以一款好的 GameFi 游戏带来的不能仅仅是利润，还必须是足够有趣和吸引人的。

Line Next 计划在下半年发布 5 款新游戏

LINE 旗下致力于开发和扩展 NFT 生态系统的 LINE NEXT 准备下半年在 Web3 游戏平台 GAME DOSI 陆续推出五款新游戏，GAME DOSI 也是 Line 旗下的Web3游戏平台。这些游戏昨天在 GAME DOSI 的推广网站上正式宣布，包括：

  • 《Sweet Monster Guardians》：一款塔防游戏，玩家通过拖放独特的角色卡牌来保卫村庄，此款游戏也将率先在 GAME DOSI 提供预售。

  • 《Vestria The Last Order》：一款收藏角色扮演的游戏，玩家在游戏中的战后世界进行各种冒险与对战。

  • 《KEROZ》：一款 Rogue 类型的游戏，动作场面华丽震撼。

  • 《Drawship Kingdom Reverse》：一款通过虚拟人物、玩偶、机器赚取 NFT 的游戏。

  • 《Project GD》：一款内部开发的 IP，将随着时间推移涉及多款游戏。

据该公司称，每款游戏都有独特的游戏机制。

Line 进军 GameFi 都具有哪些优势？

  1. 一体化平台：GAME DOSI 作为 Line 旗下的 Web3 游戏平台，目前已经推出一体化平台，包括可提供 GameFi 平台的 SDK 和 API，NFT 商店（B2C、C2C），游戏代币和商品交易系统，运营方面的支持。

  2. 成熟的代币经济学设计经验：基于 Line 生态系统，GAME DOSI 支持针对不同的游戏内容进行代币经济和商品结构设计。

  3. 巨大的用户基础：Line 是一款全球知名的通讯应用，拥有数亿用户，这为 Line Next 进军 Web3 游戏领域提供了强大的用户基础。这意味着 Line Next 可以更快地吸引用户，提高用户黏性，从而更快地推广其 Web3 游戏。

  4. 强大的技术支持：如前所述，Line Next 将利用 Line 的技术优势，为游戏开发者提供 SDK 和 API，游戏开发人员可以快速轻松开发Web3游戏。

  5. 法律咨询和风险管理指导：GAME DOSI 计划为进入该平台的游戏提供持续的监管检查和风险指南，包括加密钱包、代币和 NFT，帮助游戏开发商减少加密货币行业法律法规问题，专注于游戏业务。

  6. 丰富的资源优势：作为 Line 的子公司，Line Next 可以充分利用 Line 的资源优势，例如 Line 的品牌影响力、市场渠道、人才储备等，这些资源可以为 Line Next 在 Web3 游戏领域提供强大的支持。

为什么看好 Web3 游戏？

Web3游戏领域处在非常早期阶段，仍然还有很多可能。

1. P2E 模式或将再度焕春

游戏行业去年的收入为 1750 亿美元，预测到 2025 年可以增长到 2690 亿美元，如果游戏玩家可以通过 NFT 所有权从中分得一杯羹，那么相信其可以成为 Web3 游戏崛起的一个重要引擎。

2. Web3 游戏领域建设者仍在努力

随着 Web3 游戏的发展，GameFi 也涌现出了很多优秀的建设者，也逐渐出现一些非常具有创新性、品质一流的游戏。

在此介绍一些优秀的 Web3 游戏及开发商：

  1. @Treasure_Dao，被誉为 Web3 的任天堂，其是 Web3 游戏领域中的最大的所有知名游戏组成的 DAO，虽然每个游戏都有其各自的游戏系统，但是他们都可以通过 $MAGIC 和 Trove 来交易每一个游戏内的 NFT。

  2. @ohBabyGames，被誉为 Web3 的 SEGA，其已经拥有超过 75 名员工，ohBabyGames 已经发布了很多款优秀的游戏，构建了一个完整的生态系统，如《Mario Kart》《Super Smash Bros》《Fall Guys》《Slay spire》。

  3. @The_Beacon_GG，是一款 2D 角色扮演游戏，该游戏是 Treasure_Dao 生态系统的一部分，在测试阶段已经达到 百万 $MAGIC 的交易量，这是一款具有浓郁像素艺术的奇幻动作角色扮演游戏之一，是目前 Web3 平台上最好的独立游戏之一。

3. 区块链需要活跃用户

区块链需要应用创新，以带来更多的活跃用户。区块链本身目前缺少能够保证用户持续在链上活跃的应用，而一款更有趣更复杂更沉浸的 GameFi 相信可以吸引更多人参与到区块链行业中来。因此游戏可能成为区块链行业进一步扩展的目标领域。

我们期待坐拥数亿用户的 Line 在 Web3 中的表现，希望他们可以利用 Line 的品牌影响力和市场渠道，快速推广其 Web3 游戏，吸引更多的玩家参与到区块链世界中。
