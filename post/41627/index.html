“当你在凝视深渊的时候，深渊也正在凝视着你”——这句名言出自哲学家尼采的《善恶的彼岸》。

对于那些深入研究 Web3 经济的人来说，深渊的故事听起来非常熟悉，毕竟加密圈里经常提及“掉进加密兔子洞”这句话不是没有原因的。当然，每个人都想成为爱丽丝，在仙境里展开奇幻冒险之旅；或是成为黑客帝国里的 Leo，化身解决问题的“救世主”。但在现实中，倘若你真的坠入深渊，结局是，越陷越深。

今天，我想用一种可视化、成体系地方式向大家介绍“Web3 深渊”。

通常来说，“Web3 深渊”可以被划分成不同的层，如果你想探索不同“Web3 深渊层”，那么就需要具备一定的行业知识储备 ，否则很难完全理解一层中存在的所有现象，让你知识储备越来越多，就可以继续探索下一层，不断下沉。当然，在这个过程中，亲自体验并投资于一些 Web3 项目有助于您快速掌握知识，但在尝试投资之前必须要自己做好充分研究。

需要注意的是，“Web3 深渊”里的每一层都不是静态的，偶尔会交换位置。 随着时间的推移，以前隐藏的深层可能会逐渐浮出水面。通过“凝视深渊”，你可以看到下一层发生了哪些情况，每下沉一层，深渊都会变得更加“黑暗”。

Layer 0：深渊周围的地表

Layer 0 通常聚集了一些不愿探索 Web3 的人，这些人对 Web3 内容不感兴趣，有些人甚至选择完全忽略 Web3。不过他们可能认识到城市中心已产生了大洞。但是，当你站在深渊入口边缘的地方，会发现周围也有许多窥视深渊的人，他们很可能会被吸引。

Layer 1：协议平原

这里通常是初露头角的 Web3 开发者的第一站，应该会接触到这一领域内几个规模最大的协议，包括比特币和以太坊。在 Layer 1，人们或多或少已经学会了些经验（通常是在痛苦的教训中获得的），比如如何建立自己的自托管钱包，但大多数人依然会选择将自己的 Token 放在中心化交易平台。

偶尔，一些小型加密货币也会浮出水面并引起人们的注意。例如，Tezos 在 2020 年和 2021 年初横空出世，但到了 2021 年晚些时候就被 Dogecoin 和 Shiba Inu 这些 meme Token 给取代了。

在 Layer 1，你可以找到：比特币、以太坊。

常见危险：私钥丢失/被盗、交易所被黑客入侵/破产、加密货币价格波动等。

Layer 2：NFT 草地

NFT 最近一次出现在这一层是在 2021 年。毫无疑问，这是整个 Web3 深渊中最丰富多彩的一层，充满了游戏和数字艺术。通常而言，在这一层里，你会发现加密行业里一些不那么丰富多彩的项目会尝试将 NFT 技术用于其他用例，例如，对实物资产进行代币化。

但是，Layer 2 也是第一次让你感受到堕落深渊的危险之地。如果您不能承受全部损失，请不要在这一层进行任何投资！尤其注意：并不是每朵花都像 CryptoPunk 一样甜美。

在 Layer 2，你可以找到：NFT 游戏、数字艺术、元宇宙项目、新颖的 NFT 用例。

常见危险：诈骗、山寨项目、伪造的 NFT 系列、NFT 欺诈。

Layer 3：稳定币苔原

与 NFT 草地相比，稳定币苔原里并没有那么丰富多彩，但却是 Web3 基础设施的关键组成部分，因为它们提供了一种买卖 Token 且无需将其转换为法定货币的方法，特别是在熊市环境里会成为人们关注的焦点。

此外，您在 Layer 3 还可以找到一些提供稳定币存款收益的中心化服务商，但请务必选择一个值得信赖的平台。要知道，最近 Celsius、三箭资本、以及 Voyager 几家大型 CeFi 贷方问题频发，让许多债权人追悔莫及。

在 Layer 3，你可以找到：中心化稳定币（USDT、USDC、BUSD）、超额抵押稳定币（DAI）、CeFi。

常见危险：CeFi 破产、去中心化稳定币崩盘。

Layer 4：基础设施工厂

在 Layer 4，你会发现许多规模较小的 Web3 协议，包括一些 Layer 1 智能合约替代平台、基于以太坊区块链的各种 Layer 2 协议，以及各种适用于不同区块链用例的共识机制。此外，这一层还容纳了 Web3 行业的从业者。

一般来说，在 Layer 4 探索和投资通常相对安全，但请注意，这一层同样会出现较大的波动性，甚至有时候比“Layer 1协议平原”的波动性还要大。

在 Layer 4，你可以找到：L1s、L2s、“Proof-of-X”协议。

常见危险：波动性增加。

Layer 5：DeFi 湖泊

在Layer 5 风景如画的湖泊周围，您可以找到许多去中心化金融 (DeFi) 最常见的用例，包括借贷、超额抵押的去中心化稳定币、以及 DEX。游过湖面，你会发现一个深邃而黑暗的洞穴的入口，其中包含更晦涩的 DeFi 用例，例如保险、无抵押贷款、代币化和算法稳定币。

当你越来越接近洞穴的底部，一家旅馆赫然出现，它的窗户散发出一种看似金色的光芒。在下沉到第五层后，你也许感到旅途劳顿，于是决定在旅馆中休息一下。你竟然能听懂旅客们所说的大部分内容。

你想知道：一旦你回到地表，那里的人是否也听不懂你在说什么？你还想回去吗？不，你看到了地表生活的平淡，而且已经领略到 Web3 深渊中的各种奇迹。你开始意识到，自己已经走在了一条不归路上。

在旅馆里，当您与其他人交流时，会发现他们中的大多数人至少曾经是 DeFi 黑客或欺诈活动的受害者，也许你深有体会也有自己的故事要讲。不过，你突然注意到许多人在旅馆后面的一扇僻静的门里进进出出，门上方的标志写着“IDO 平台”。

尽管你很好奇，但依然决定探索一下，突然间，你发现自己身处一个巨大的赌场。“IDO 平台”是一个每天都有新的加密货币诞生的地方，但其中大多数代币都“死得很快”。倘若你把所有积蓄都放在这些加密货币身上，结局将会非常危险。但是，那些幸存下来的人则可以通过早期早期投资获得大量财富。一旦亲身进行了几次风险，您也许会注意到输赢变得越来越无关紧要了。

在 Layer 5，你可以在这里找到：基本的 DeFi 用例、高级的 DeFi 用例、shitcoins。

常见危险：黑客攻击、欺诈、shitcoins 本身等。

Layer 6：DAO 群岛

当你沉入第六层，会发现这里有一堆紧贴在 Web3 深渊边缘的“小村庄”。从高处望去，它们就像是茫茫黑暗海洋中的小岛，下降的越深，岛屿数量就会越多，您想知道这些岛屿中可能有多少个。

当你决定探索一个岛屿，会发现岛屿上村落里的居民语言比 DeFi 旅馆里的人还要陌生，但他们都很友好和热情。你同意帮助他们完成一些任务以换取一些“当地货币”，数量不多，但足以帮助您在岛屿上生存下来。

居民们一直在努力建设他们的村庄并建造通往其他岛屿的桥梁，但总的来说，他们过着幸福的生活。于是，你决定多呆一会儿，帮助他们，你喜欢呆在那里建造村庄，但一些年长的居民会警告你，不要太拼命赚取“当地货币”。

在 Layer 6，你可以找到：去中心化的自治组织。

常见危险：心理健康问题、DAO 衰退。

Layer 7：无底洞

天知道你在一些岛屿上工作了多久，终于有一天，你决定潜入 Web3 深渊的第七层，那里的天很黑，你看不太远，也看不到底部，没人知道这一层是否是 Web3 深渊旅程的终点，或者在 Layer 7 之下是否还有更多层，你甚至不知道自己已经爬了多远。

在这里，你看到的东西也许无比宏伟、有趣，但同时也非常可怕，你很难用语言来描述这一层中的东西。你以为自己可以逃离这一层，但转念一想，还不如叫更多人下来……

你会等待，一直等待……

在 Layer 7，你可以找到？？？

常见危险：崩盘。
