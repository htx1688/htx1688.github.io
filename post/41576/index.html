去中心化的互联网目前可以理解为一个独立的国家，而不是一个统一的领土。

撰写：Witek Radomski 

编译：深潮 TechFlow  

通用编程标准在数字世界并不新鲜，毕竟互联网在很大程度上是建立在这种标准之上的。其中最具影响力的超文本传输协议（HTTP）的开发始于 1989 年欧洲核子研究中心（CERN）的 Tim Berners-Lee，其设计目的是使数据交换简单化。这种设计是经过深思熟虑的，它降低了 Berner 的另一个新创造：万维网的采用门槛，可以说他的策略在某种程度上是成功的。

在 1997 年达到第一个标准化版本 HTTP/1.1 之前，HTTP 经历了多次迭代的演变。 这个标准化的基石是网络空间扩张的关键，也是以我们认为的理所当然的方式相互交流的关键。网络上的互操作性使信息能够跨越国界、设备和基础设施的限制。

万维网的演变为去中心化的元宇宙发展以及代币标准的发展——如何促进长期采用和成功提供了重要的经验。随着行业的成熟，多链格局，而不是单链，看起来越来越可能成为行业未来。优先考虑这些不同区块链和元宇宙平台之间的互操作性的代币标准将是长期成功的关键，但这些标准究竟应该包含什么，我们如何才能在这样一个多样化的行业中取得社区之间的共识？

建立标准

标准有多种形式，它可以是实体的，如欧盟的 USB-C 指令，也可以是抽象的，如互联网。标准对区块链特别重要，因为它们使协议之间能够有效沟通。

区块链协议现在开始类似于国家，像以太坊这样的大型链拥有自己的人口、经济和代币标准，如 ERC-20、ERC-721 和 ERC-1155。在使用以太坊时，这些都可以有效地进行，并执行其预期功能。然而，在一个区块链和另一个区块链之间转移资产是非常困难的，就像城市之间没有高速公路一样。

去中心化的互联网目前可以理解为一个独立的国家，而不是一个统一的领土。例如，Decentraland 与 The Sandbox 是不同的虚拟土地，这两个不同的生态系统之间没有直接的沟通方式，更不用说其他大量的去中心化元宇宙平台。然而，如果没有一个共同的框架来实现这些不同的元宇宙生态系统之间的价值转移，它们的潜力就会是孤立的，并且从根本上受到限制。

共享相同的基础

不可替代的代币（NFT）意味着所有权的出处，但如果没有适当的方法来跟踪它在链之间的旅程，就真的无法知道是否有人干扰了你的代币。 元数据标准适用于这种情况，管理链之间的灰色区域，并为构建者提供可以发挥其创造力的护栏。

以 HTML、CSS 和 XML 为例，它们都允许开发人员在 HTTP 之上构建独特的网站。 这意味着尽管网页可以用一个数字领域的语言编写，但它与所有其他页面共享相同的基础 。HTML 上的元数据特别允许用户定义数据集的特征，如作者或关键字。

链间（元数据）标准将确保 NFT 的所有权经济遵守透明、灵活和个人自由的普遍原则。元数据将消除对信任的需求，因为它将表示监管部门的批准，编码知识产权、隐私法和其他关键属性。

技术标准

Web3.0 的精神——创造一个更民主、更公平和用户拥有的互联网版本——在制定通用代币标准方面也面临着挑战。首先，它要求 Web3 的标准设计是由社区主导的。

像 Web3 基金会这样的组织正在开创这样一种方法。Web3 基金会现在有超过 450 个不同的成员，包括非营利组织、大学、政府实体和越来越多的个人。它是万维网联盟（W3C）的自下而上的对应物，它为可互操作的 Web2 技术开发规范和指南。

重要的是，制定适当的 Web3 互操作性标准可以补充特定领域的语言（DSL），以实现跨产业链和生态系统的实用性移植。这对于 Soulbound NFT 或甚至像 POAP 这样的平台来说尤其重要，这些平台用独特的数字资产奖励用户，表示完成了游戏中的里程碑或出席了虚拟活动。将元数据嵌入这些类型的资产，并将其与去中心化的标识符（DID）配对，将是可互操作的去中心化元宇宙成功的关键，增加不同区块链及其社区之间的协同作用。

例如，以太坊域名服务（ENS）的名称可以作为 DID 使用，这是隐私转发、无需信任的身份网络的一个很好的例子。ENS 域名在 EIP-721 标准上运行，意味着 “.eth " 记录的移动与 NFT 类似，并且可以提供建设“智慧城市”所需的安全基础和充足的数据主权。然而，如果没有一个可互操作的代币标准，这样的 “智能城市 “只能在以太坊上运行，无法与在 Polkadot 上构建的城市进行交易。

这只是代币标准在元宇宙中可以实现的一个开始。另一个重要的好处在于合并了多用途互联网邮件扩展（MIME）标准。通过在链上的音频文件或 3D 模型的元数据中插入 MIME 信息，我们可以自定义游戏中角色的物品交换，如在不同的游戏和元宇宙生态系统中的声音效果或数字服装。

携手共进，共创未来

在制定一个可互操作的代币标准之前，元数据在不同区块链、元宇宙平台和游戏生态系统之间的传输将从根本上受到限制。Web3 行业有一个多样化和深厚的人才库，可以共同合作，使这样一个标准的开发获得成功。

正如互联网等其他技术的发展所证明的那样，互操作性标准对完整性和采用至关重要。在一个以社区和协作为重点的行业中，共同开发此类标准对于进步是不可缺少的。
