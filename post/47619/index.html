比特币 NFT 的兴起一直是 NFT 生态系统中最近最热的叙述之一。 然而，一开始您可能无从下手，本文将向您阐述如何以简单直接的方式铸造您自己的比特币 NFT。

  • 目标：学习铸造比特币 NFT

  • 技能：初级/中级

  • 市场：不到一小时

  • 投资回报率：OG 区块链上的 DIY 铸造

我最近写了一篇关于比特币 NFT 的兴起和繁荣的文章，这一市场在 2014 年首次出现，当时推出了基于比特币的自定义代币网络 Counterparty。然而，Counterparty 最终未能获得重大关注，并最终被以太坊智能合约的兴起所掩盖，此后催生了多次 NFT 繁荣。

近十年来，大量的“比特币 NFT 生态系统”的概念一直不被认为是可行的，或者，甚至被加密经济中的绝大多数人认为是不可行的。 随着 Ordinals 的到来，这一切都发生了变化。Ordinals 是一种新的协议，可以将元数据写入单个 satoshis—即最小的 BTC 面额。

那么，为什么这个概念再次异军突起？ 2014 年和今天有什么不同？

首先，当代加密经济现在拥有一个蓬勃发展且发展良好的 NFT 生态系统，其中充满了能够轻松驾驭多个不同链的探索者。 可以说，这次我们准备好了。

其次，尽管如此，Ordinals 式的 NFT 是一项创新突破，因为它们提供了一种新方法，可以将完全链上的 NFT 直接铸造到比特币上。 在这里，“完全在链上”是指元数据，例如 NFT 的艺术，将可以无限期地从比特币区块链中检索，而不是从私有存储服务器或 Arweave、Filecoin、IPFS 等存储网络中检索。

也就是说，就像我们迄今为止看到的所有 NFT 方法一样，Ordinals 形式的 NFT 具有许多独特的优点和缺点。

这种新的比特币 NFT 提供了强大的永久性保证——至少目前如此——以及廉价的链上存储，这通常非常昂贵。

至于缺点，这种形式缺乏智能合约提供的可组合性和表达可能性，并且比特币节点有可能最终将它们删除。

不过，目前可以明确的是，这个新的比特币场景将继续存在，并且无疑会催生新的对 Ordinals 友好的钱包和铸币服务的激增。 最近几周我们已经看到这种激增的开始，因为多个项目已经开始提供简化的比特币 NFT 工具，因为最初它需要运行你自己的比特币节点才能与这些 NFT 交互。

比特币 NFT 入门

1.一些比特币

2.兼容 Ordinals 的钱包

就像在以太坊上铸造 NFT 需要 ETH 作为 Gas 一样，你用 BTC 支付来完成比特币 NFT 铸造交易。 在撰写本文时，gamma.io 上的正常铸币费约为 50,000 聪，即 ~₿0.0005 或 13 美元。 然而，根据需求，该费用可能会大幅波动。 因此，购买价值 25 美元的 BTC，大约是₿ 0.001，现在应该可以支付你的第一笔铸币费用，但购买价值 50 美元的比特币可能是你将波动性考虑在内的最佳选择。

在设置钱包时，请不要嫌麻烦。 如果您尝试使用普通的比特币钱包继续前进，您可能会不小心花掉铸造 NFT 所需要的聪！ 因此，您现在可以使用的最简单的钱包之一是 Xverse，这是一种基于Stacks (STX)平台开发的比特币钱包，目前可作为 Chrome 扩展程序或适用于 Android 或 iOS 用户的移动应用程序使用。

下载扩展程序或应用程序后（任何一个），系统都会提示您创建一个新钱包。 系统会询问你是现在还是稍后备份你的钱包，然后提示你为你的新帐户设置密码。 完成后，您可以使用您的 Xverse BTC 地址接收您的铸币资金并支付比特币 NFT 铸币费用。

此时，在您的 Xverse 钱包收到 BTC 后，您就可以开始铸造了。 为此，前面提到的 gamma.io 网站是目前可用的最直接的 Ordinals 风格的铸币资源之一，因此我向比特币 NFT 初学者推荐它。

要继续使用 Gamma，只需在 Xverse 钱包中导入 BTC 后前往该项目的网站。 然后单击页面顶部的“Ordirals”按钮并选择“Create Inscription”。 这样做会将您带到如下所示的上传仪表板：

在这一步，首先上传您想要的文件类型。 然后选择您想要的费用级别（正常、更高或自定义）并输入您的 Ordinal 地址以铸造您的 NFT。 注意：这不是您 Xverse 应用程序中的 BTC 地址； 您需要单击收藏品选项卡并按“接收”，然后复制 Ordinals NFT 地址以继续。

此时，在 Gamma 上按“继续”，同意使用条款，然后为您的铸币交易付款，系统会提示您这样做：

现在呢？ 您只需等待。 根据交易费用和当时使用网络的需求，您的inscription可能需要数小时甚至数天才能完成。 Gamma 将提供一个等待页面，可以在其中查看铸币状态并放大交易详情。

一旦您的 mint 交易完成，您将能够在 Gamma 或您的 Xverse 钱包中查看它（在收藏品选项卡中选择“打开画廊”选项）。 如果想出售你的 NFT，可以尝试在 Gamma 上调出收藏列表，或者在其他一些可用的 Ordinals 市场上试试：

总而言之，现在在比特币上铸造 NFT 并不适合胆小的人，但与一些人现在使用的更激进的方式相比，像 Xverse 和 Gamma 这样的服务让它变得容易得多，例如不需要自己运行比特币节点。

从广义范围来看，看看比特币 NFT 能发展到何种程度将会很有趣，对于 NFT 纯粹主义者来说，中本聪inscription的永久性和链上可能性令人信服，但生态系统未来将如何形成仍有待观察。 比特币有可能成为领先的链上存储选项，让我们拭目以待这条赛道带来的新变化！
