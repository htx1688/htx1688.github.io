近日，根据相关法庭文件披露，Mysten Labs 3月22日与FTX债务人达成协议，将以9630万美元的现金回购 FTX 对Mysten Labs的股权投资和SUI代币权证。

长达123页的文件议，详细还原了2022年8月FTX 投资 Mysten Labs 的情况。 结合法庭文件和 Sui 近期的一系列动作，Sui World 进行了一些总结及分析：

1、 Mysten Labs 约20亿美元估值完成超3亿美元的B轮融资

这个融资信息是去年的公开消息了。FTX及B轮投资者们通过投资Mysten Labs 的B轮优先股权，获得了Sui Token 的代币权证。

2、 FTX 约1.02亿美元获得约8.9亿枚Sui Token

FTX在Mysten Labs 的B轮融中总共投资了1.02亿美元，获得了约57万股优先股，和获得约8.9亿枚Sui Token的代币认购权证。作为领投方的FTX 投资Sui Token的平均成本是非常低的，约0.114美元。

FTX这里有两笔获得SUI代币权证非常重要：Cottonwood Grove Limited（Alameda）和 FTX Trading 分别以500美元的价格获得了约1.73 亿和3.46亿枚共计5.19亿枚Sui Token 的代币权证。这5.19 枚Sui Token 在 Sui NetWork 主网上线和Sui Token启动后，满足一定条件，将用于在FTX 平台上做市交易。（Sui World 注：Cottonwood Grove Limited 是Alameda Research全资控股公司）

作为曾经的头部交易平台和做市商， FTX 领投Sui 超过1亿美元的的一个重要条件，就是旗下的做市商Alameda 和FTX Trading几乎免费拿到了5.19亿枚 Sui Token用于在FTX.com 平台上做市。但是随着FTX在Sui 启动之前就崩溃了，这个代币权证的协议也不存在了。

3、部分B轮投资者的Sui Token筹码成本约为0.28-0.31美元

抛开 FTX 的做市筹码部分，其他部分B轮投资者获得Sui Token代币权证的成本仍然可以参考 FTX Ventures，平均成本约在0.28-0.31美元之间。

4、Sui 将收回大量低成本的机构筹码

除了约 0.28-0.31美元之间的Sui Token 投资筹码，对于领投方FTX来说，还额外获得了5.19亿枚Sui Token的做市筹码。

如今Mysten Labs以9630万美元的现金回购 FTX 对Mysten Labs的股权投资和SUI代币权证，意味着收回了大量FTX 低成本的投资筹码和做市筹码。对于未来上线的 Sui Token 来说是个好事。

值得提的是，FTX同样也以相似的估值、投资金额领投了另一条Move 链Aptos，FTX对Aptos的投资方案或和Sui非常相似，即FTX 持有大量（超过流通量1/3）低成本的投资筹码和做市筹码。这意味着在FTX 崩溃之前上线的Aptos，部分低价筹码已经进入市场，没出售完的筹码也会等待FTX债权人的处置方案。

5、FTX债务人可以继续“向其它第三方寻求更高的报价”

文件还披露，目前FTX债务人可以继续“向其它第三方寻求更高的报价”，直到法院最终确定出售日期。Mysten Labs的股权投资和Sui 代币权证对于不少机构来说还是有吸引力的。

6、SUI 主网上线少了最大的外部障碍

处理完与 FTX 债务人的问题，也就处理了SUI 主网上线的最大的一个外部障碍。SUI 主网上线与Sui Token的发布，不用考虑来自 FTX 此前的投资筹码的处理问题。

7、9630万美元对应了1亿美元基金

9630万美元的现金回购 一旦完成，对于去年刚融资超过3亿美元的Mysten Labs来说，短期并无财务影响。 长期的影响或主要在于Mysten Labs 对Web3的一些布局，比如投资、收购 Web3项目。

这也对应上了此前的消息，Mysten Labs 首席执行官Evan Cheng 正寻求为一个 Web3 基金筹集超 1 亿美元资金，该基金将不限于 Sui 生态。Hashed 联合创始人 Alex Shin 和资深对冲基金投资者 Sandeep Ramesh 将作为普通合伙人加入新创立的 Web3 风险基金。
