原文：《Friend.tech: The Newest Canvas for Crypto Artists》

作者：William Murphy

编译：Odaily星球日报夫如何

近期，对 friend.tech 的讨论卷土重来，就其未来发展众说纷纭，有人坚定看好，更多人认为是昙花一现。

不过，根据 9 月 15 日 DefiLlama 数据：friend.tech 24 小时协议收入达 227 万美元，超过 Lido 排名第二，仅次于 Ethereum。

协议收入看似美丽，但 friend.tech 的真正作用在哪？泡沫中填充何种气体才能减速破裂？Bankless 的 William Murphy 提供一个新的思路：friend.tech 可以形成一种新颖的 NFT 销售方案。

Odaily星球日报现将其文章精华编译如下。

想象一个世界，在这个世界里，艺术家可以将他们的作品变现，建立起一支忠实的粉丝群体，甚至可以通过他们现有的 X（Twitter）粉丝来启动社群。那么 friend.tech 将是不错的选择。

friend.tech 为个人创建可交易的 Key，从而引入了以下功能：

● 内置流动性的社交代币 。每个 Key 销售的 90% 都用于内置流动性，在智能合约中锁定，确保人们可以随时出售。剩下的 10% 平均分配给 friend.tech 和正在销售 Key 的用户。 

● 带门槛的聊天室 ，您可以与您的“Key 持有者”分享文本和图片。 

● 每个“Key”都有一个自我调节的市场 ，定价按照递增曲线上涨，随着更多人购买而增加，如果更多人卖出则下降。由于这个机制，您不需要为 Key 设置单独的定价。 

● 与 X 集成 ，使用户可以轻松找到您并购买一张或多张 Key 以访问您的内容。这使得现有 X 追随者的无缝导入 friend.tech。

上述这些功能都有各自的用处，下文将进一步讨论：在一个 NFT 市场面临流动性问题时，艺术家们难以变现他们的作品，friend.tech 颠覆性的提供了一个替代的路径来铸造和销售作品。

NFT 现阶段问题

目前，NFT 市场面临着重大挑战，特别是在艺术领域。随着流动性的减少，越来越少的收藏家希望收购版本或 1/1 作品，还在购买的人群预算也逐渐减少。

加密艺术收藏家分为两大类：一类是纯粹为了欣赏特定艺术品而收藏的人，另一类是看到潜在投资机会的人。随着二级市场几乎停滞不前，后一类别的人大多已经转向其他地方寻求收益。尽管艺术家们对这些追求收益的收藏家退出市场持有不同的观点，但显然资金流入的减少正在伤害加密艺术社群。

考虑到加密艺术市场面临的这些挑战，让我们探讨一下 friend.tech 如何成为艺术家出售 NFT 的一个可行解决方案，或者至少是他们当前营销和销售作品方式的一定价值的补充。

friend.tech 的用例

friend.tech 提供了一个新颖的替代方案，可以避免直接将艺术作品作为 NFT 进行销售。其中一个主要优势是内置的流动性池汇聚每个 Key 销售的 90% 。这确保了购买 Key 的任何人通常至少可以回收一部分资金，除非 Key 的价值降为零。

相比之下， 1/1 NFT 艺术品或版本缺乏这种流动性。NFT 卖家要么必须耐心等待合适的买家，要么要大幅降低价格，有效地创造出艺术家作品的数字跳蚤市场。这种情况对艺术家和收藏家都是不利的。它削弱了人们对艺术家作品的价值认知，强迫收藏家接受较低的转售价格。

friend.tech 作为一个平台的优势在于灵活性特质 ，任何与在线观众分享内容的人都可以充分利用 friend.tech，不仅仅局限在艺术领域，比如经验丰富的交易者可以分享对市场和当前交易的看法，音乐家可以分享未发布音乐的早期链接，产品创始人可以与他们最忠实的用户分享更新。

我认为最好的理解 friend.tech 的方式是将其视为一种经过代币化的 Patreon_（Odaily星球日报注：一个帮助创作者众筹引流、支持数字内容销售的海外平台）_，用户可以买卖会员资格，并在他们希望销售时获得流动性保证。作为一位艺术家，你可以抽奖 1/1 的艺术作品，为你的 Key 持有者创建代币门控的版本，分享艺术的预览，谈论你正在进行的新项目，分享幕后内容等等。

就个人而言，我很早就加入了这个应用，并迄今为止通过人们进入和退出我的聊天室已经产生了接近 0.1 ETH 的交易费用。我的最后一次 1/1 NFT 销售的价格也在这个范围内，所以由于收入已经达到经济上的合理点，我为我的 Key 持有者抽奖了一次 1/1 作品。除此之外，我在外出拍照时分享我的生活和幕后内容，并且在一天中与 Key 持有者聊天。

虽然从 Key 销售中获得的 5% 份额可能会比传统的 NFT 1/1 和版画销售产生更少的收入，但在形势严峻的时候，多样化收入来源尤为重要。任何经验丰富的艺术家都明白当一个收入来源减少时，需要具备财务适应能力。我相信，当真诚地向 Key 持有者提供价值时，friend.tech 可以作为一个相对稳定的收入来源，而不仅仅依赖于直接的艺术销售。

friend.tech 的玩法

friend.tech 是一个渐进式网络应用（Progressive Web App，PWA）。加入的过程非常简单和直观。

以下是加入 friend.tech 的指南：

1.  添加应用 ：由于 friend.tech 是一个 PWA，与从应用商店下载常规应用的注册流程有所不同。相反，打开你的移动浏览器，导航到 friend.tech，并按照网站上的指示将该网站添加到你的主屏幕上。这将使该网站在你的移动设备上像其他应用一样显示为一个功能性的应用程序。

2.  使用手机或电子邮件注册 ：在添加并打开应用后，系统会提示你使用电子邮件或手机号注册帐号。 

3.  获取 friend.tech 邀请码 ：目前，friend.tech 在技术上处于测试阶段，仅限邀请加入，因此你需要一个访问码。在这里你有几个选择： 

● 从 X 上的朋友获取邀请码：很多人已经加入了该平台，我敢打赌，如果你发布帖子寻求邀请码，很快就会有人提供。 

● 在 X 上搜索：如果你不想发布这样的帖子，你可以使用 X 的搜索功能来寻找发布邀请码的人。我刚刚搜索了“friendtech 邀请码”这个词，找到了很多结果。 

4.  链接你的 X 帐户 ：friend.tech 的用户名系统由 X 提供支持，所以在开始使用帐户之前，需要链接你的 X 帐户。为此，你需要给 friend.tech 一些有关你的 X 帐户的权限，你可以选择之后取消这些权限。 

5.  将 ETH 转入你的 friend.Tech 钱包 ：如上所述，为了在 friend.tech 进行交易，你需要在 BASE 网络上有一些 ETH，然后将它们发送到你的新 friend.tech 钱包，即与你的新帐户关联的以太坊地址。

6.   购买你的第一个 Key ：首先，你必须购买至少一个自己的 Key；第一个 Key 只需支付很小的 Gas Fee。完成此交易后，人们将能够购买和销售你的 Key 以获得进入你的聊天室的权限。 

7.  开始体验 ：当你的 ETH 转移完毕并购买了第一个 Key 后，你就可以开始交易和结交新朋友了。你现在是 friend.tech 社区的一员！

艺术的未来具备流动性和社交属性

在加密艺术面临流动性问题、艺术家们努力应对生存挑战时，friend.tech 为铸造和直接销售作品提供了一个新颖的替代方案。friend.tech 不仅仅是一个平台，更是由社区、市场和社交网络组成的综合体。 通过内置的流动性和自我调节的 Key 市场，friend.tech 为艺术家和收藏家提供了一个优化的市场，在加密艺术这个波动性大、流动性不足的世界中提供了更大程度的稳定性和规律性的感觉。

除了收益方面，friend.tech 也是创意的舞台。无论你是艺术家、音乐家还是经验丰富的交易者，该平台都允许你与志同道合的人分享你的作品和想法。这是一个空间，不仅可以销售艺术作品，还可以以更亲密和有意义的方式与观众互动，以你认为合适的方式。

作为深入探索这个平台的人，我可以证明它作为一个稳定的收入来源的潜力。虽然来自 Key 销售的 5% 份额可能无法取代传统艺术销售的收入，但它提供了一种在波动的加密市场中难以获得的稳定性。

因此，如果你是一位艺术家，正在寻找一个展示作品、建立社区，并赚些钱的新途径，那么 friend.tech 值得进一步了解。

艺术的未来不仅仅是数字化的，也是社交的、有趣的和流动的，而这个未来可能正在 friend.tech 上展开。
