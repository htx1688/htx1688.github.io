谈及 NFT，大家的目光毫无疑问都首先聚焦在 ETH 上，其次是 Solana，今年刚兴起的比特币 NFT 也有着很高的讨论度。哪怕是 Layer 2 的 Arbitrum，都曾因为 Beacon 乃至 Magic 生态的爆火而让 NFT 玩家们对其有过较高的期待。

相比之下，Polygon 的 NFT 市场的讨论度差很多，显得寂寂无闻。昨天，DeGods 旗下的 NFT 系列 y 00 ts 还宣布要从 Polygon 迁移到 ETH 主网，并将退还 Polygon 给予的 300 万美元资金资助。这似乎显得 Polygon NFT 市场萎靡不振。

其实，Polygon NFT 市场的历史总成交量在所有区块链网络中排名第 5 。如果我们把 Ronin 这样的「Axie 游戏链」和 Flow 这样的「NFT 专链」去掉的话，Polygon 仅次于 ETH 和 Solana。

在过去的 30 天里，Polygon NFT 市场的总成交量排名第 3 ，与 Solana 的总成交量差距非常之小。考虑到比特币 NFT 的总成交量里包含了很多 BRC-20 Token 的成交量，Polygon 可以说是「稳 3 抢 2 」的水平。

Polygon NFT 市场，会复制当年 Solana NFT 市场的盛况吗？现在的 Polygon NFT 市场，又有哪些值得关注的项目呢？

Solana 后的又一个「NFT 下沉市场」？

Solana NFT 市场第一波真正意义上的「出圈」是 2022 年 4 月末 Okay Bears 的快速暴涨。由于 Okay Bears 吸引了大量的市场关注，Degods、Solana Monkey Business 等项目也一并「出圈」，Solana NFT 市场中形成了「蓝筹梯队」。

当时，ETH NFT 市场乃至整个 Crypto 市场都处于下沉趋势，但是 Solana NFT 市场却逆势上升。最重要的，「出圈」让 Solana NFT 有了「注意力」，这使大家相信无论是寻找潜在的新「蓝筹」还是狙击短期内流动性提升带来的「非蓝筹」机会都能够赚钱。手续费低、新项目的发售价格便宜，让 Solana NFT 成了一张潜在收益率更好的「彩票」——毕竟，在 ETH 上，你很难抓到下一个 CryptoPunks、BAYC 了，这些项目对于新人来说过于昂贵。而新项目卖的也不便宜，上涨的空间也能看到天花板，就像一座已经被开采了差不多的金矿。

现在让我们来看看 Polygon NFT 市场。

在 B 端，Polygon NFT 市场有很多 Web2 巨头入驻。

星巴克忠诚度奖励机制 Starbucks Rewards 的拓展「Starbucks Odyssey」建构在 Polygon 上，目前场内的总市值超过 900 万美元，二级市场成交量超过 250 万美元，累计 Mint 了超过 21 万个「旅程邮票」NFT。

Starbucks Odyssey

Nike 的 NFT 平台 .SWOOSH 建构在 Polygon 上，在其上所推出的首个数字运动鞋系列「Our Force 1 」销量超过 97000 份，销售额超过 190 万美元。今年 6 月，Nike Virtual Studios 还宣布与视频游戏巨头 EA Sports 合作，将 Nike.SWOOSH 平台上的 Polygon NFT 和数字内容引入 EA 的游戏生态系统。

「Our Force 1 」

Reddit 的 Collectible Avatar 在 Polygon 网络上已经发行到第 4 代，总市值超过 5000 万美元，销售额超过 4000 万美元，有接近 1600 万名持有者，是全球持有者数量最多的 NFT 系列。

Reddit「Collectible Avatar」

Polygon 上的游戏 NFT 表现也比较亮眼。单看链上的数据，当前 Polygon NFT 历史成交量前 2 名的项目都是游戏类 NFT。

排名第 1 的 ZED RUN 是一个赛马游戏，玩家免费借用或购买自己的马进行训练然后和其他玩家的马对战。根据该项目官网显示，其品牌赞助商包括百威啤酒、雅达利和 Netflix 等。根据 Cryptoslam 的数据，该项目总成交额超过 2 亿美元。

ZED RUN

排名第 2 的 DraftKings 7 天内成交量达到 670 万美元，约占历史总成交量的 35% ，成交量在最近的快速增长使 DraftKings 在最近被形容为「Polygon 上的 NBA Top Shot」。DraftKings Marketplace 是由 DraftKings（纳斯达克股票代码：DKNG）为「面向大众」构建的 NFT 市场，里面卖的有球星卡、Reignmakers 足球、高尔夫和 UFC 经营游戏的资产以及一些虚实结合的产品。

DraftKings Marketplace

@0x obsessed 对 Polygon 上的游戏项目做了一个分类，涵盖了超过 150 个建构在 Polygon 上的游戏，游戏真挺多。

可以说，Polygon NFT 市场是一个「不是我们想象中 Web3 那个味」的 NFT 市场，在「面向大众」产品和游戏类产品上，Polygon 相较于 ETH 和 Solana 都要走得更远一些，共有的特点是封闭的市场、为用户提供托管钱包、支持/仅支持法币支付等。在现阶段，这些 B 端的成绩还没有能够很好地将 Web2 用户转化到 Web3 以帮助 Polygon 像 Solana 那样建立起自己的 Web3 NFT 社区乃至文化。

而在 C 端，Polygon NFT 市场还没有出现 Solana 之前那样 NFT 项目引爆整个生态的情况出现。虽然既有 CyberKongz VX 这样的 ETH 一线 PFP 项目的「支线」，有 The Sandbox 这样迁移而来的成熟的元宇宙土地，也有美国前总统唐纳德·特朗普的首个 NFT 系列「Trump Digital Trading Cards」这样 meme 味挺浓、引起过较高关注的效用类 NFT 项目，但是 PFP 类型上始终没有出现过一个「出圈明星」。由于 y 00 ts 的离开，Polygon NFT 上 PFP 项目的王座也出现了真空，有理由期待这个真空在未来被填补的同时，出现「Polygon NFT 之季」的盛况。

虽然有理由期待「Polygon NFT 之季」的出现，但同时也不可否认今时不同往日。首先 Crypto 市场的整体情况比「Solana NFT 之夏」出现时更熊，其次是 NFT 玩家对 PFP 叙事的兴趣度和信任度几乎消磨殆尽。想要看到「Polygon NFT 之季」出现，我们可能首先要期待的是 Crypto 市场的整体转暖，有足够的流动性支持 Polygon NFT 上出现财富效应。

值得关注的 Polygon NFT 项目

PFP

效用

FireDrops 是一个品牌忠诚度奖励平台，用户在其上可以完成自己喜欢的品牌所给定的忠诚度任务，比如社媒参与、产品购买、用户推荐以及其它品牌任务等。通过完成忠诚度任务，用户可以获得现金折扣、免费商品、优先体验权等奖励。当然，还有 NFT。用户能够在未来的内置市场上交易这些奖励。

这听上去是和星巴克的「Starbucks Odyssey」一样，是一个 Web2.5 的忠诚度奖励产品，但是覆盖的范围会比「Starbucks Odyssey」要广很多。目前，可以在 FireDrops 官网 申请加入等候名单，据官方透露，已有超过 100 万用户进行了申请。

结语

Polygon NFT 现阶段在 Web2 巨头入驻和游戏生态的发展上有突出表现，加上之前 Polygon 2.0 （$POL）的宣布，有些 NFT 玩家对 Polygon NFT 市场的发展有着较高的乐观情绪，甚至基于这几点提出了「Polygon NFT Q4 登月理论」：

不过我们要明确两点。首先，这里的「登月」指的是 Polygon NFT 像 ETH、Solana 那样发展出坚实的社区（玩家群体）以及具有特色性的玩家文化，更直接的体现就是 Crypto 玩家们熟悉的 PFP 类型 NFT 发展出「出圈的蓝筹」，能够有大量的 Web3 原生 NFT 玩家入场淘金。其次，吸引 Web3 原生 NFT 玩家在现阶段看最需要的还是「财富效应」，而 Polygon NFT 所积累的大批 Web2 用户，在短时间内还看不见很好地将他们吸引进 Web3 世界的渠道，星巴克、Nike 等巨头的市场就像 Polygon NFT 市场内的「独立王国」，Web2 用户依然被他们封闭的市场所阻隔，后续能否成功向 Web3 用户导入增量仍需观察。

新的市场总是充满机遇，还没有真正意义上 Polygon 原生的「蓝筹」出现，给了我们足够的想象空间。Polygon 似乎和当时的 Solana 一样，万事俱备，只欠一个 Okay Bears 这样的「东风」。我们会看到「Polygon NFT 之季」的出现吗？
