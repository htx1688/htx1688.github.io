纵览过去的一个周期，GameFi 产品从无到有、从新生到强盛，这一产品类别无疑为加密世界开创新的用例和产品品类。但所有的加密投资者都深知一个事实，GameFi 仍然是强金融属性的，其“游戏”属性只能说聊胜于无。

近日，一款基于 Unity 开发的名为 Fusionist 的链游受到了加密市场和 Unity 的关注。

（Fusionist 中由玩家所控制的机甲）

与以往玩家群体对链游在游戏领域的“忽视”有所不同，这款链游已经在游戏市场引起了一定的关注，甚至 Unity 官方也对其技术表示了称赞。Odaily星球日报近日关注到了这一项目，Fusionist 是如何同时俘获游戏和加密两个领域的？

机甲、科幻、模拟经营，经典元素与加密如何融合？

Fusionist 是一款融合了模拟经营、回合制战⽃、SLG 玩法等元素的游戏，于去年 11 月正式对外公布。目前官方Discord 社群⽤户累计⾄ 94 万，Twitter 粉丝数达 84 万。

在 Fusionist 的世界里，游戏背景设置在了2800 年，由于人类的贪婪、环保情况恶化，地球已不再适宜居住，由此引发了大规模的战争。

这一游戏采用了经典的机甲、星球探索的题材，为玩家打造了一个尺度颇为宏观的宇宙。玩家将扮演机甲指挥官管理⾃⼰的星球，收集稀有资源、升级技术、制造机甲。该游戏计划于 2023 年底，正式登录 Steam 平台。

在大多数熟悉游戏的玩家群体中，加密和游戏并没有完成很好的结合。而在诸多玩家社区，也对颇多早期 GameFi 产品进行了批评或嘲讽。而 Fusionist 则凭借 3 A 级的游戏体验和卓越的技术力，为我们展现了“链游”是如何具备足以和传统游戏一较高下的实力的。

在Fusionist 中，玩家可以探索一个充满未知和惊喜的宇宙。游戏中存在多样化的任务和挑战等待玩家去完成，此外还有大量的可收集物品和稀有奖励可供玩家收集。而由于加密的特性，Fusionist 还提供了一个完整的经济系统，使得玩家可以在游戏中进行交易和投资。

与其他Web3游戏不同，Fusionist 注重玩家体验和可持续性。早期链游的挫败已经足以说明，或许只有通过提供高质量的内容、优秀的设计等特点来吸引玩家，并建立可持续性的生态系统，才可让链游在“游”这一属性上更进一步，以此来实现更长的产品寿命和更长远的成功。

强技术力得到 Unity 官方称赞，链游如何媲美传统 3 A？

在传统 GameFi 攻城伐地之时，GameFi 不好玩这一因素始终困扰着绝大多数用户。而对开发者来说，这或许更多是受限于加密产业的先天原因。在加密产业，强金融、弱用例始终是我们这一新兴产业的特点之一，而Web2开发经验的不足也让 GameFi 产品缺乏开发一个“好游戏”的技术基础。

而 Fusionist 则有所不同。目前，项目组共有 47 名全职成员，均是游戏行业的资深从业者。这也为其提供了深厚的技术积累。他们在业界早已成熟的 Unity 引擎基础之上构建了 Fusionist 的世界，并采用了大量的前沿技术。包括 DOTS、六⾯光照贴图等，这为其游戏提供了逼真的水体、云层、爆炸等精美的画面。

相信游戏玩家都对 HDR 有所耳闻。而 Fusionist 则采用了 HDRP 技术，打造一个更加物理真实的游戏世界。

所谓 HDRP 即⾼清渲染管线(High Definition Render Pipeline）技术，与传统的 HDR 有所不同，该技术采⽤了基于物理的渲染框架,可通过游戏内物体的材质、纹理、反射等数据，⽣成更加真实的画⾯效果。

在 Fusionist 中，使⽤到了 HDRP 管线的许多技术，包括 Layered Lit Material，PBR 物理渲染、SSR、SSAO、体积雾、后处理、HDR 等。

（光线模糊反射的场景对比）

而基于这一技术，Fusionist 还引入了 Path Tracing、ScreenSpaceReflection、Layered Lit Material等多项技术。这些技术均可使游戏中的重要物体——机甲——呈现出更逼真的光线、材质、渲染效果。而基于物理的渲染，也让其可以表现更多特性。例如，PBRmask 贴图被决定材质的表⾯物理属性，这可充分反映机甲的磨损等细节。

而在环境效果上，KWS water System 系统为Fusionist 提供了基于物理的逼真的模拟⽔池。Caustic 效果甚至可以实现真是的⽔下画面，可⽤于模拟光线在⽔中的折射和反射。

（更加真实的光照及环境效果）

此外，在扩展性上Fusionist 也体现了其卓越的技术实力。作为一款全球同服的太空 MMO-SLG，⾼峰时服务器需要实时动态维护⼏⼗万个 Entity。Netcode for Entities技术实现了前后端实时同步。

在内部性能评测中，⼀台 AWS c 6 a.xlarge 服务器，可⽀撑近百个 System，⼏⼗万个 Entity 规模的宇宙模拟。服务器的 Logic Ticket 始终维持在 10  FPS以上。

在 Unity 官方发布的技术分享文章中，他们盛赞到，“Fusionist 是⼀个⾮常好的例⼦，详细说明了 Unity 引擎的最新功能在实践中的技术应⽤。凭借其令⼈印象深刻的视觉效果和游戏性，Fusionist 揭开了 Unity 的在 3 A 游戏⽅⾯的⽆限可能。”

Endurance 链为 Fusionist 提供底层驱动

一款优秀的区块链游戏产品自然需要一个优秀的基础设施。团队不仅在游戏开发上可以与传统游戏一较高下，Fusionist 的 Endurance 区块链也颇为优秀。

Endurance 是一个旨在为开发人员提供一个快速、高效和可扩展的区块链基础设施，以构建去中心化应用程序，包括游戏和社交网络平台。

具体来说，Endurance 拥有多种优势。首先是其卓越的性能：Endurance 采用了一种称为“状态压缩”的技术，可以将区块链数据压缩到更小的存储空间中。这使得 Endurance 可以在不牺牲安全性和可靠性的情况下降低存储要求，并且具有更高的吞吐量和更低的延迟。

此外，其低成本的存储和更高的网络性能也为其支持高并发的数据处理提供了基础。这使得在游戏和社交网络平台上执行大量交易变得更加容易。

Endurance 为开发人员提供了完善的配套开发工具，包括 SDK、API 和文档等。其基础设施，支撑了其可以和现代化的 3 A 游戏相结合。这也是其对行业的积极贡献，或可让区块链游戏产品摆脱纯金融的简单游戏换皮的现状。

目前，Fusionist 及 Endurance 还有待更多信息披露。Odaily星球日报将对这一项目保持关注，并持续关注其后续进展。

作为一条高性能的链，Endurance 不仅可以用于游戏和社交网络平台，还可以用于其他各种应用程序。它的协议支持多种用例，包括但不限于金融、物联网、供应链管理等。因此，开发人员可以使用 Endurance 构建各种 dApp，以满足不同行业和领域的需求。

关于更多技术细节，可关注 Unity 官方公众号详细阅读。
