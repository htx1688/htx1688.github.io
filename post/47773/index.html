感受到那个蓝点了吗？在 Coinbase 发布了一个神秘的短视频‌，指定当天为 Big day 后，CT 一时间充斥着蓝点热潮。甚至 Optimism 也加入了这一行列！

昨天一整天，CToors 一直宅阅读字里行间的内容，其中一些认为他们发现了传说中的 Arbitrum 空投！

然而，这些说法是错误的，也是完全不符合事实的！那么，到底发生了什么？

? Coinbase 正在推出自己的 optimistic L2 Rollup，称为 Base

? 建立在 optimistic 的 OP Stack 之上

Coinbase 告诉我们，他们曾两次考虑推出 Binance Smart Chain 式的 L1，但决定不采取这种做法，因为他们担心这样做会使他们的链与新兴的加密经济隔离。

OP Stack--Optimism 的模块化、开源蓝图，用于创建高度可扩展、高度可互操作的各类区块链 – 被证明是 Coinbase 的游戏改变者，为他们提供了建立自己的链的机会。

在大约两个月内向主网推出，Base 将作为一个去权限的 L2 运作。尽管有相关猜测，但这不是 KYC 的区块链。我们得到的是一个直接的 optimistic L2 rollup，对所有的开发者和用户开放。不幸的是，在最开始，Coinbase 将是该链的唯一定序者，Base 被归类为一个完整的训练轮“阶段 0”rollup，与 Vitalik 的 rollup 分类法相一致。

Coinbase 对加密价值的总体忠诚度通过他们对去中心化和安全的承诺得以展示：

  • Base 将过渡到一个完全去权限的验证器集

  • Base 将在 2023 年 EOY 之前作为一个有限的训练轮“Stage 1”rollup 运行

  • Base 将在 2024 年底前升级为无训练轮的“Stage 2”

有限训练轮阶段的广泛实现和 EIP-4844 的推出将使以太坊达到 Vitalik 的以太坊路线图上的“基本 rollup 扩展里程碑”，完成 Surge ? 中两个步骤的第一步。

去中心化的快速进展，链上安全的疯狂飞跃，以及 Base 在两个月内从 2 月 1 日的测试网到发布主网的能力，都是由 OP Stack 实现的。Coinbase 能够为其 rollup 提供经过实战考验的 Optimism 代码，是 Base 开发速度的巨大推动力。

那么，他们是否只是在揩油 Optimism 团队的劳动成果？

并非如此。事实上，Coinbase 正在与 Optimism 合作，成为 OP Stack 的第二个核心贡献者。有了两个 L2 团队在同一个堆栈上构建，发展将被加速。

Base 的开发将受益于一个开源和免费提供的扩展解决方案代码库，rollup 本身建立在以太坊之上。

Base 的部分交易费用将用于 Optimism Collective，为 Optimism 上的公共产品提供资金。此外，Coinbase 还承诺启动一个 Base 原生生态系统基金！

除了提高速度和增强团队间的合作外，由于 OP Stack 链之间的标准化消息传递格式，OP Stack 的未来为共享序列器集的链提供了原子化的可组合性！

这就是 OP Stack 飞轮的威力。

Coinbase 的愿景 ?️

Coinbase 不仅仅是一个简单的交易所。

它的产品涵盖了数字资产质押解决方案、流动性质押衍生品、机构托管解决方案、非托管钱包产品、应用内集成的 DeFi 可访问性以及其他等等。Base 是这一产品线的自然延伸，可以为 Coinbase 用户提供一个直接融入 Web3 世界的机会。

虽然 Coinbase 已经建立了一个声誉良好的品牌，是加密交易和投机的可信赖的用户端口，但 DeFi 和其他链上原语的日益采用将创造出对直接和可靠的端口的需求，以满足越来越多的用户需求 –Coinbase 旨在成为用户的服务提供商和可信赖的前端。

Coinbase 认为自己是链上活动的主要网关，Base 和建立在其上的协议，包括 Coinbase 开发的 dApps，不仅对提高公司的下限有利，而且可能为更广泛的加密经济提供积极的推动。

对于 Coinbase 的广大用户来说，一个简单的、相对无风险的与 Web3 互动的解决方案是非常可取的。该公司旨在提供一个默认的、开箱即用的体验，目的是消除用户对与区块链互动的恐惧 – 这是进入 Web3 的主要摩擦点。

此外，Coinbase 的目标是引导其 1.1 亿的验证用户，以促进低成本的 DApp 用户获取。

目前的空投计划非常容易受到影响，通常只会带来短暂的活动和 TVL 的上涨。Base 将作为希望获得新用户的协议的基础设施层，同时串联起用户的选择，将他们与他们需要的服务连接起来。

信用卡供应商和银行使用财务激励措施来吸引新用户并改变行为；Coinbase 相信它可以通过类似的模式彻底改变 Web3 的消费者获取。

何时发币 ?

你是否希望你 90% 的 Coinbase 垃圾币投资组合能让你有资格获得一笔 BASE 空投？又或者你是一个正在寻找代币分配的 COIN 股权持有人？

我很痛苦地告诉大家 Coinbase 的坏消息：事实是，没有关于 BASE 代币的计划。

BNB 的竞争者？?

我们认为不是……

鉴于没有永续和其他高杠杆交易方案，Coinbase 的用户群往往不怎么 degen。而随着 Coinbase 的客户成为 Base 最可能的用户，我们也不太可能看到类似的杠杆热潮。借用杠杆有利于产生收益，从而吸引 TVL。

一个主要的 DeFi 用例不仅可能在 Base 上会获得较少的吸引力，而且由此产生的对杠杆的需求不足可能会转化为较低的收益率生成，减少 TVL，并减少交易量，削减网络活动和费用生成。

此外，Coinbase 的许多用户已经可以在成熟的 TradFi 市场上获得了替代的金融解决方案。

BNB 已经找到了一个利基，为发展中市场缺乏稳定货币或复杂金融系统的用户提供了低费用的选择。尽管加密资产的定价存在波动，但代币（特别是货币品种）为受高通胀率影响的当地货币提供了一个有价值的选择。点对点交易在链上很常见，去许可借贷市场替代了传统的生息储蓄账户。

2023 年 1 月，Coinbase 的月度现货交易量占 Binance 记录的 1.108 万亿美元的月度现货、BTC 期货和 ETH 期货交易量总和的 5.0%（553.2 亿美元）。

不幸的是，西方的金融体系还停留在过去的时代。

点对点的交易发生在 Venmo 和 CashApp 上，需要三天的时间来结算。我们已经接受了部分储备的银行系统，我们被迫参与其中，生活在一个靠塑料卡和 ApplePay 运行的社会。

在 Coinbase 渗透最深的市场上，向 Web3 的过渡对消费者来说更加困难，因为他们可以使用传统的金融解决方案，并且觉得继续保持旧的习惯比向 Web3 的未知世界转换更味舒服。

看涨 OP ?

Coinbase 的公告中，绝对无争议的赢家是 OP。Coinbase 选择在 OP Stack 上建立 Base 是 Optimism 生态的一个巨大胜利！

今天的公告基本上告知了公众一个绝对庞大的 Optimism 核心团队的收购！Optimism 不仅免费获得了这一开发者支持，而且 Base 还将交易收入的一部分让给了姊妹链！

Coinbase 的这一举动真正验证了 OP Stack 的飞轮理论：加入一个大规模的核心贡献者，为 rollup 提供额外的激励，以建立或迁移到 OP Stack！

Optimism 是建立在 OP Stack 上的，这不是什么秘密；Metis 和 Boba 也是如此。增加 Coinbase 作为第一个核心贡献者，将使联合团队能够摧毁阻断器，并开发出最佳的 Optimism rollup 解决方案。随着更多的 rollup 系统建立在 OP Stack 上，在模块化和开源代码的支持下，所提供的设计灵活性只会增加。

随着原子合成能力在未来的出现，OP Stack 可以为 DEXs、GameFi、NFT 交易或任何其他 Optimism 的愿望释放出一个高度专业化的 rollup 解决方案世界。

OP Stack 的推出降低了建立自己的 rollup 的门槛，就像流动性产生流动性一样，核心贡献者产生核心贡献者。

Rollup 的夏天即将来到你附近的 OP Stack？

天堂里的烦恼？?

真奇怪……为什么在“天堂里的烦恼”旁边有一个蓝点？

尽管推出 Coinbase 原生 L2 听起来很牛逼，但我们不太相信它（应该）会大大改变你的 COIN 投资论。Coinbase 当然有用户和资产，但他们离让用户进入 Base 和 Web3 世界还有一段距离。

然而，加密原生借贷市场提供的稳定币利率，远远优于富国银行储蓄账户每年应计的 0.0001%！我想，这将是一个巨大的挑战。

每个人都会想这么做的。

Coinbase 已经为他们的客户提供了一个应用内的托管方法，让他们可以进入 Web3 并使用 dApps。没有比这更容易的事情了！

虽然 Coinbase 声称，与空投相比，Base 将降低用户获取费用，同时将用户与他们需要的协议联系起来，一路提供指导，但这些说法似乎毫无根据。

Coinbase 的“学习奖励”标签已经为协议提供了一个与他们的 1.1 亿用户联系的平台。此外，“Web3”标签已经为用户提供了推荐的 dApps，甚至以社区星级的形式提供了社会信号。

也许 Base 成功的最大障碍是 Coinbase 作为美国股票市场上的上市公司所处的具有挑战性的监管环境。

与 Optimism 不同的是，Optimism 使用 OP 来资助公共产品，并通过经常性的空投来激励行为 – 或者其竞争对手 Arbitrum（尚未推出代币）可能会实施精心设计的空投，以吸引流动性和可持续的采用 –Coinbase 在不久之后推出代币的机会是无限小的。

让法律部门批准一个去许可的 rollup 可能是一项繁重的任务，而且美国证券交易委员会（SEC）、纽约证券交易所（NYDFS）、美国中央银行（OCC）或其他对美国金融系统有着管辖权的机构仍有可能在 Base 的问题上找上 Coinbase。

发行代币？可笑的是，根本不可能。

在美国开始为空投以及美股如何发行空投制定明确的监管框架之前，在没有潜在代币的情况下，相对于其竞争对手，Base 将处于重大不利地位。

乐观前景?

Coinbase 确实有一个有利的因素，即该公司仍然拥有坚实的品牌资产。

虽然消费者可能不愿意使用 Coinbase Web3 一栏中列出的 Aave 和 Uniswap 等协议，但随着 Coinbase 品牌 dApps 的推出，这种情况可能会改变。我当然可以看到，最无摩擦的方式是通过交易商的品牌产品，让不熟悉的非加密原住民进入 Web3。

Base 很可能是通往许可、KYC DeFi 的门户，但该产业链肯定不是只有 KYC。

毕竟，正如 Coinbase web3 负责人 Jesse Pollak 在采访中提到的那样，Coinbase 在扩展解决方案方面的尝试包括一套具有去中心化身份基元的基础设施产品。

由于监管透明度有限，机构不清楚如何与 DeFi 打交道，而好打官司的监管机构也在紧追不舍，Coinbase 品牌的、半许可的 dApps 可以作为 DeFi 这个可怕世界中的一个儿童游泳池。Coinbase 已经有了这些关系，为机构和高净值个人提供着托管和大宗经纪服务。

机构 ?

Base 是否有可能成为 coinbase 品牌 dapp 的启动平台，从监管风险和资金安全的角度为这些实体提供安慰？如果是这样的话，这可能会是一个巨大的解锁，向以前面临结构性障碍的一类不情愿的投资者打开 DeFi 的世界，让他们进入这个空间！

毫无疑问，Base 的推出对整个 Coinbase 团队来说是一个巨大的成就，代表着 OP Stack 的一个不可思议的飞跃。但现实情况是，消费者宁愿生活在 TradFi 的围墙花园内，对上链仍然犹豫不决。虽然 Coinbase 有 1.1 亿的验证用户，并在平台上持有 800 亿美元的资产，但其中只有一小部分可能会迁移到 Base 上。

不过，对 Coinbase 来说，800 亿美元仍然是一个非常大的数字，仅有 2.5% 的迁移就足以超越 Arbitrum，确保 Base 成为 TVL 的第四大链。

区块链技术仍然处于起步阶段！每月有 1000 万个活跃地址以及约 100 万个实际的个人在链上进行交易，这个领域的增长潜力是巨大的。

Coinbase 决定出击并创建自己的 L2，是对以太坊和更广泛的加密金融系统的巨大信心投票，可能会在未来某个未知的时间点获得丰厚的回报，但值得注意的是，Optimism 和 Arbitrum 在过去一年只获得了 900 万美元的收入，而 Coinbase 的顶线收入超过 30 亿美元。

在可预见的未来，Base 除了成为一个金钱陷阱之外，几乎不可能再充当任何角色。

虽然 Coinbase 已经提供了各种迹象表明它打算成为以太坊生态系统的忠实管家，通过对 OP Stack 的贡献和成为 RocketPool oDAO 的成员（以及其他行动）来促进去中心化，但让该公司履行他们的承诺仍然是至关重要的。

请记住，伙伴们：DYOR，不要 ape in。当你读到这份免责声明时，这里所反映的事实很可能已经纳入了 COIN 和 OP 的价格中！

你的优势不在于试图超越使用光纤连接的高频算法交易员，你的优势在于了解加密领域的发展，并在代币（和加密货币股票 ?）中进行具有巨大不对称上升空间的高预测性赌注！你的优势在于了解加密领域的发展。
