作者:WILLIAM M. PEASTER；编译：MetaCat

最近几天，friend.tech 在很短的时间内产生了非常令人印象深刻的活动量。有人称其为“社交代币领域的 Uniswap”，friend.tech 是建立在全新的堆栈之上。但它会很快将 NFT 添加到这个创新堆栈中吗？

从某种意义上说，friend.tech已经支持 NFT，所以这取决于你如何看待它。本文将介绍 friend.tech 的基础知识，并深入探讨其在 NFT 领域的可能性。

你可能还记得今年4月的 Stealcam ， Stealcam 是 Arbitrum 上一款令人上瘾的独立“steal to reveal”应用程序。

NFT 持有者可以 mint 隐藏的图像，其他人可以通过支付 ETH 来“窃取” NFT，从而获得并揭示该 NFT 的内容。

每当被他人“窃取”时，Stealcam NFT 的价格就会上涨 0.001 ETH 加上之前价格的 10%，即沿着联合曲线上涨。前所有者将收到他自己的之前的付款，然后剩余的差额的 45% 将分配给创作者、45% 分配给前持有者、10% 分配给 Stealcam。

我在这里提到 Stealcam 的原因有两个。首先，因为 Stealcam 团队随后将品牌更名为 friend.tech，并扩展为“你的朋友们的交易市场”的愿景。同时这也是因为，friend.tech 的 ERC20 社交代币还沿着联合曲线销售，因此请注意 friend.tech 的血统，它前身版本中直接获得灵感。

稍后会回到这一点上，现在让我们先了解一些 friend.tech 的基础知识。

friend.tech如何运作？

friend.tech 是一款部署在 Coinbase L2 解决方案 Base 链上的社交应用程序，“用于与朋友和金钱一起玩耍”。一周前，该应用程序推出了邀请制的测试版，并开始分发邀请码，用户可以在该平台上买卖关联了 Twitter 帐户的股份。

当你在 friend.tech 上购买他人帐户的股份时，你会自动获得与该帐户所有者及其其他利益相关者的私人聊天室的使用权。你还可以沿着应用程序的原生联合曲线系统对买到的股份进行销售，有时价格高于你最初买入的价格，有时则低于你最初买入的价格，具体取决于最近的交易活动。

为什么 friend.tech 很重要？

是的，我们之前见过很多社交代币。那么 friend.tech 有什么特别之处呢？

答案是堆栈！

正如你在下图中看到的那样，friend.tech 以前所未见的方式将各种有趣的东西组合在一起。L2（便宜、快速的交易）、渐进式网络应用程序PWA（绕过集中式应用程序商店）、嵌入式钱包（用于降低入门门槛）等。

来源：https://twitter.com/nnnnicholas/status/1691887931684704630

将这种创新设计与社交代币和加密货币的前沿技术相结合，你就可以理解为什么 friend.tech 如此迅速地受到欢迎。

让我们来看一些数据：在 friend.tech 发布第一周，交易量已经超过 8,374 ETH。这意味着在过去的 7 天里，friend.tech 的交易量超过了除 DeGods（约 21,779 ETH）和 Bored Apes（约 8,515 ETH）这两个 NFT 系列之外的所有 NFT 系列。

friend.tech 还处于起步阶段，但事实证明它的设计非常引人注目，并且已经吸引了极大的兴趣。有可能比 Uniswap 的流动性大 100 倍吗？这还有待观察，但至少该应用程序可以在不久的将来为 NFT 生态系统带来福音。

NFT 角度

还记得之前提到的血统和灵感吗？

从 TweetDAO 和 Stealcam 项目可见，friend.tech 团队显然很了解 NFT。该应用程序测试版是 MVP 测试版，因此随着它的发展，创作者可能选择更紧密地整合或拥抱 NFT。

当然，NFT 领域目前正处于熊市，而与顶级 NFT 平台相比，friend.tech 的活跃数据已经相当令人印象深刻，至少就目前情况而言是这样。因此，如果该应用程序继续获得关注，然后更紧密地拥抱 NFT，那么这可能会成为重新提升 NFT 活跃水平的重要渠道。

来源：https://twitter.com/_ThePinkyToe/status/1691818070317081080

我知道这基于一系列的“假设”，而且拯救 NFT 世界不是 friend.tech的工作。但是我从 friend.tech的早期测试者那里看到的最需要的功能之一是对聊天中图片消息的支持。NFT 基本上紧随其后，上传、铸造在如今的加密领域几乎是标配。

然而，就像我在介绍中提到的那样，艺术家和创意人员已经可以通过 friend.tech beta 发挥有关 NFT 的想象力。

例如前面提到的@unbanksyETH 等人注意到，已经有多种不同类别的使用friend.tech 的方式，包括艺术自我表达、营销等。因此，即使在测试版中，加密艺术家也可以在聊天中通过文本分享“死忠粉独家 alpha 和 mint 链接”。

因此，friend.tech 上的 NFT是已经可能的了。但随着时间的推移，它们可能会变得更加容易。如果它发生并且一切顺利的话，这可能会成为 NFT 活跃水平的一个巨大的积极催化剂，这就是我所要说的！
