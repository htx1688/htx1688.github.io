撰文：William M. Peaster

编译：二丫&Cecilia，bfrenz DAO

稳定币是早期建立 DeFi 生态系统最大的吸引力之一，然而并非所有的稳定币都是一样的。

这节 Bankless 实践课将带你了解两个目前最好的去中心化稳定币的基础知识，即 Liquity 的 LUSD 和 Reflexer 的 RAI。相信阅读本文是你 DeFi 学习之路上的一节干货实践课。

A comparison of Liquity, Reflexer, and MakerDAO - via Liquity

为什么稳定币很重要

稳定币是被设计为价值稳定的数字资产。

最常见的保持价格稳定的方法是与法定货币挂钩，例如与美元价值挂钩的 USDC 和 USDT。

但与法币不同的是，稳定币是无国界的，互联网上任何人都可以使用，而且转移速度快，成本低。这些优势使得稳定币近年来蓬勃发展。例如，在 2022 年，稳定币大约结算价值 7 万亿美元的交易。（没错，7 万亿！）‍‍

+$7T in stables txs in 2022, up from $6T in 2021 - via Coin Metrics

不同类型的稳定币

稳定币有各种类型的，今天我们主要讲的类型有：

  • 法币挂钩型稳定币：由现实世界机构的法币存款 1:1 支持的稳定币，例如 Circle 的 USDC、iFinex 的 USDT 和 Binance 的Paxos 发行的 BUSD。

  • 加密货币过量抵押型稳定币：将过量的加密货币抵押到 DeFi 协议中，例如 MakerDAO 的 DAI，Synthetix 的 sUSD，和 Liquity 的 LUSD。

  • 算法稳定币：依靠供需算法来维持价格稳定的稳定币，例如 Frax Finance 的 FRAX 和 Reflexer Finance 的 RAI 。

因为稳定币可以有各种形式，所以在脑子里对它们的去中心化程度形成光谱是重要的，可以参考 TokenBrice 的这张图，从左到右去中心化程度逐渐提高。

via TokenBrice

在这里，TokenBrice 对 DeFi 协议进行了排名，但同样的概念也可以普遍适用于稳定币。

在这个光谱图的最左边是完全中心化的项目，如 USDC 和 USDT，这些项目最终由其运营商进行审查。这些代币对许多人来说已经足够好了，因为运营商不会随意审查他们的余额——然而他们的基础设计会带来一定的交易对手风险（counterparty risk），如部分储备系统出错、未经授权的第三方妥协、或者来自政府的压力等等。因为这些风险几乎很难被完全排除，所以像 USDC 和 USDT 这样的稳定币并没有通过我们的「韧性」测试。

在去中心化光谱的中间，例如 MakerDAO 的美元挂钩 DAI 稳定币的项目。你可以将 ETH 等加密货币存入 Maker，并以它们为抵押借入 DAI。协议是去中心化的，但是今天存在的所有 DAI 中约有 40% 是由 USDC 产生的，而 USDC 是中心化的。

The current collateral underpinning the DAI - via daistats.com

DAI 也存在交易对手风险，不过相比之下，已经相当具有韧性。但由于目前 DAI 背后的许多资金最终是可以审查的，而且由于 Maker 可以通过治理而迭代发展，DAI 仍然不是铁定的「去中心化」。

那到底什么样的稳定币才算呢？这就把我们带到了今天文章的主题：处于「去中心化光谱」最右端的去中心化稳定币。

去中心化稳定币的重要性

有效的去中心化稳定币之所以如此强大，有两个根本原因：最少的外部依赖性，和最好的自托管性。

ps.自托管（self-custody）可以解释为将你的数字资产储存在一个只有你能访问的安全地点的方法。这意味着你决定自己处理你的私钥，而不是把它委托给第三方。它不是由银行或任何其他机构发行的，也不能用来验证交易。

符合这一条件的两个稳定币是 Liquity 的无治理 LUSD 和 Reflexer Finance 的最小化治理 RAI。

这些稳定币的设计范围很窄，以便可靠地始终如一地运行在这个充满不确定性的行业潮流中。由于没有或小到忽略不计的治理要求，这些代币消除了交易对手风险，这是一个巨大的优势。

via Liquity

因此，在没有任何外部依赖的情况下，你可以长期持有和使用这些代币。这些代币的自托管性质使得你可以通过自我保管来保持对它们的完全控制。

这种耐久性让这些稳定币在其核心用例中享有卓越的可靠性，如稳定可靠的加密货币支付、链上储蓄和汇款。

例如，假设你在接下来的几年里投资了大量的 BUSD，但是下一届美国总统政府在财政部安了一个鹰派官员，他在扩大 OFAC 的执法范围方面采取了核武器一般的方式。他们的新规则使你成为当年使用 Tornado Cash 的目标，所以 Paxos 被迫收回你的储蓄。

ps.Tornado.Cash 是一个混币协议，基于零知识证明在以太坊上实现的隐私交易中间件，能够以不可追溯的方式将 ETH 以及 ERC-20 代币 发送到任何地址。

这绝对是最坏的情况，而且不太可能发生，但也不能完全排除这样的黑天鹅事件。对比之下，去中心化稳定币在这一维度中大放异彩，因为它们相对来说是反脆弱和抗审查的。

去中心化稳定币的特点介绍完了。下面，我们来学习 LUSD 和 RAI 稳定币的具体实操知识。

LUSD 快速入门

via Liquity

受 MakerDAO 最初版本的启发，Liquity 是一个去中心化的借贷协议，用于推动 ETH 的无息贷款，它只允许用 ETH 存款产生稳定币 LUSD。由于不涉及管理密钥或治理，Liquity 被设计为以这种方式永久地工作。

而且，与其他常见 DeFi 协议的 150% 的抵押率相比，Liquity 只要求以 110% 的最低抵押率进行 LUSD 贷款 。

需要注意的是，你的 LUSD 都可以在任何时候赎回你的 ETH，并且 Liquity 没有官方的前端，因此各种不同的社区前端都可以用来与 Liquity 协议交互。

也就是说，你可以通过两种方式获得 LUSD，即从去中心化的交易所购买或从 Liquity 协议中借入。

如果你只是对分散资产配置感兴趣，那么简单地从 DEX 购买 LUSD 是最好的。为了获得现在 DeFi 中顶级的 LUSD 交易池的全景图，建议先去 GeckoTerminal 之类的数据聚合平台搜索「LUSD」了解下情况。

例如， GeckoTerminal 显示， Curve 的 LUSD/3Crv 池现在有价值 5300 万美元的流动性，所以如果你想去目前最大的 LUSD 池，直接去 Curve 交换就可以啦。

The LUSD/3Crv dashboard on GeckoTerminal

至于借入 LUSD，如果你不介意管理一个债务头寸，可以按照以下步骤进行：

1、从公共的 UI 目录中选择一个前端来使用（https://www.liquity.org/frontend#list）。

2、连接你的钱包。

3、找到并点击 “Open Trove “按钮（Trove 是 Liquity 对 MakerDAO 所说的 “Vaults “的称呼）。

4、输入你想存入的 ETH 金额和你想要的 LUSD 借款金额（注意最低 1800 LUSD 的开仓要求，较小金额的借款是不可行的）。

5、查看你的 Trove 信息：

  • 清算准备金=在你的借贷金额上增加 200 美元，以防你的头寸被清算，如果 / 当你偿还你的 Trove 时，它将被退还。

  • 借款费用 = 从你的 LUSD 借款金额中扣除的一次性费用。

  • 清算价格 = 当 ETH 下降到某一价格时，你的 Trove 成为抵押品不足的状态从而被清算。

  • 抵押率 = 你的 ETH 抵押品的美元价值与你所提取的 LUSD 债务价值之间的比率，需要一直保持在 110% 以上。

6、按下「确认」按钮，并与你的钱包完成 Trove 创建交易，之后你将拥有 LUSD，然后按你自己的时间表偿还债务就可以了。

RAI 快速入门

via commit.press

我们今天要看的另一个去中心化的稳定币是 RAI。

RAI 是独一无二的，它是当今加密世界中唯一不与任何特定资产价值挂钩的稳定币——不是美元，不是黄金，不是任何东西。正如 Bankless 的 Ryan Sean Adams 曾经解释的那样，算法稳定币 RAI 是"由 ETH 支持的 DeFi 货币，有「代码 + 激励’机制来维持稳定的价格」。

因此，RAI 的价值随着供应和需求的波动自由而稳定地浮动。在写这篇文章的时候，1 个 RAI 的价值相当于 2.78 美元。RAI 的市场价格越是偏离不断更新的 RAI 赎回目标价格，基础算法就越是调整利率，套利者会将这两个价格重新拉近。

via stats.reflexer.finance

1 RAI = 1 RAI，这种非挂钩稳定币的价值由对美元的汇率将取决于当前的算法供需条件。

像 LUSD 一样，你可以购买或借入 RAI 。

关于购买，像购买其他代币一样：找到一个具有适当流动性的池子，例如 RAI/DAI Uniswap V3 池子，并根据需要进行交换。然后确认支付。

当涉及到借入 RAI 时，这次不是将 ETH 存入 “Vault “或 “Trove”，这个稳定币的债务头寸被称为 “Safes”。现在最低的铸币金额是 2819 RAI（约 8000 美元），所以如果这是你的范围，你可以通过以下步骤借入 RAI：

1、进入 app.reflexer.finance 并连接你的钱包。

2、按 “创建账户 “按钮，并完成随后的交易（如果你已经注册过 MakerDAO 或 Balancer 代理账户，则跳过这一步）。

3、接下来按 “Create a Safe “按钮，之后你会被提示输入你想存入多少 ETH 以及你想借多少 RAI。

4、按「Review Transaction」查看所有最后的细节，如所需的 145% 的抵押率等等，然后按「确认交易」。‍

5、一旦你完成该交易，你的保险箱将被创建，你将拥有新造的 RAI，可以通过你的保险箱应用程序的用户界面，按照你自己的时间表来支付和管理你的头寸。

LUSD 和 RAI 的可靠性

像 LUSD 和 RAI 这样的去中心化稳定币对 DeFi 很重要，因为它们提供了强大的可靠保障。与中心化的稳定币不同，中心化的稳定币由中心化的组织支持，会受到监管的压力，而去中心化的稳定币不受任何单一实体的控制，因此可以抵抗审查和组织滥用。

因此，当你把这种可靠性水平与代币所能提供的可使用性、可负担性、可组合性和速度结合起来时，去中心化稳定币将是 DeFi 的最佳潜力股。
