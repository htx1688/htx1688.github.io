导读

Blur 的版税制度与大多数主流 NFT 市场不同，即交易者可以自行设置版税，意味着如果交易者不支付版税，原创作者不会从该 NFT 的二次销售中获得任何收入。

不过，Blur 也在通过激励计划尽力让交易者们自愿支付版税，从而为创作者争取更多的收入。例如，那些支付更高版税费率的交易者能够获得更多空投代币。

Blur 创始人 Pacman 近日在 Mirror 上发表了一篇文章，介绍了 Blur 平台最新的版税政策，同时也呼吁 OpenSea 取消对 Blur 的限制，让 NFT 项目可以在两平台上赚取全额版税，为当前 NFT 版税相关问题寻找更好的解决办法。

正文

目前，创作者无法同时在 Blur 和 OpenSea 上赚取版税。也就是说他们只能在 OpenSea 或 Blur 中的其中一个上面获得全额版税，但不能在两个平台上同时获得。

第三轮空投将是我们最后一轮的 Blur 空投。第一轮空投适用于在 Blur 市场推出之前 6 个月内进行过 NFT 交易的所有人；第二轮空投适用于 11 月之前在 Blur 上积极挂单的交易者；第三轮空投则适用于在 Blur 上积极出价的交易者。第三轮空投将是最大的 Blur 空投（大约是第二轮的 1-2 倍）。

Blur 是一个由社区驱动的交易市场。这就是为什么我们会在第一季空投中向创作者空投 BLUR 代币，以欢迎他们加入社区。

我们正在努力帮助创作者让他们能在任何地方赚取全额版税，在本文中，我将介绍创作者目前拥有的选项，以及我们如何在 OpenSea 和 Blur 上执行全额版税。

快速浏览一下这些选项：

1、创作者不屏蔽任何平台

不屏蔽任平台的 NFT 创作者可以实现最大程度的去中心化，同样的，创作者也无法阻止任何的零版税或可选版税的 NFT 市场。

在这个选项下，这些项目的版税在 OpenSea 是可选择的，而 Blur 则是会收取最少 0.5% 的版税（卖家也可以自行调高）。

2、创作者屏蔽 Blur

当 NFT 项目屏蔽掉 OpenSea 黑名单（包括 Blur）中的 NFT 交易平台时，OpenSea 则不会设置可选版税，而是会强制执行全额版税；而 Blur 依旧会收取 0.5% 的版税。

不过在屏蔽掉 Blur 之后，这些项目将无法使用 Blur 的 Bidding（出价）功能，但仍可在 Blur 上上架和交易。

Blur 的出价功能能够为项目提供地板价支持，增加项目的交易量，提高创作者的版税收入，禁用出价功能会伤害创作者赚取版税的能力。因此我们认为有（比屏蔽 Blur）更好的办法。

3、创作者屏蔽 OpenSea

创作者应该能够在他们列入白名单的所有市场上赚取全额版税，而不是被迫选择。（注：目前创作者如果想要在 OS 上交易，只有同时屏蔽掉其他 OS 的黑名单平台，才能赚取全额版税，否则 OS 将会执行可选版税）

为了鼓励这一点，如果 NFT 项目屏蔽 OpenSea ，Blur 会为该项目收取全额版税，不过其将无法在 OpenSea 上交易。

值得注意的是，采用这个选项的创作者将有资格获得 Blur 第二季空投的奖励。

4、OpenSea 取消对 Blur 的限制

我们认为将 OpenSea 及 Blur 都列入白名单的项目应该能在两平台上同时赚取版税。

但是，目前 OpenSea 在检测到项目 Blur 上的交易时，会自动将项目版税设置为可选版税。我们呼吁 OpenSea 取消这个政策，使创作者可以在任何平台上收取版税。

除了在 Blur 上赚取全部版税外，创作者还可以与我们合作，在我们的主页、Twitter（15 万粉丝）和 Discord（11 万会员）上展示即将推出的产品。

常问问题

为什么 Blur 不对没有设置黑名单平台的 NFT 项目强制收取全额版税？

在这种情况下征收版税时，交易者往往会转向零版税市场。这在 Sudo 于 2022 年 7 月推出时已经发生过，从那时起，采用零版税的新平台如雨后春笋般涌现。目前还没有链上的解决方案可以解决现有的版税问题。

Blur 正在努力通过提高最低版税同时保持价格竞争力来最大限度地提高创作者的版税收入。保持价格竞争力对于防止交易者转向完全零版税市场至关重要。为此，Blur 已开始对没有设置黑名单的项目强制执行最低 0.5% 的版税，目标是争取随着时间的推移提高最低版税水平。

为什么在 Blur 被列入白名单时，OpenSea 会将版税设置为可选？

让创作者屏蔽 Blur 能够让 OpenSea 的业务受益。

但是，我们认为这不是他们实施这个政策背后的主要动机。归根结底，我们相信 OpenSea 也希望创作者做正确的事，只是 OpenSea 对“合适的”解决方案有不同的看法。

因为旧的 NFT 项目没有将 Blur 屏蔽，因此 OpenSea 认为新发行的 NFT 项目都应该将 Blur 列入黑名单。然而，他们提出的解决方案存在严重缺陷（见问题一）。这就是为什么 Blur 采取了另一种更有可能彻底解决问题的方法。我们想邀请 OpenSea 与 Blur 合作，并为新系列启用全额版税，并继续探索适用于所有 NFT 项目的解决方案，无论是否采用屏蔽的方式。
