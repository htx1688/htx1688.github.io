X2Y2 是建立在以太坊上的 NFT 综合交易市场，上线采用代币空投向 Opensea 进行吸血鬼攻击，并推出一系列挂单奖励、Gas 费用返还和交易挖矿奖励来抢占市场份额。 交易挖矿仅是早期获客手段，无法形成护城河。平台收取 0.5% 交易费用，主打低价策略吸引用户，目前市场数据表现良好，但存在刷量虚假交易问题，后期关键点在于如何优化产品体验留住用户，同时持续性的吸引新用户和高质量 NFT 项目进入。

项目概要

X2Y2 的细分赛道为以太坊链上的 NFT 综合交易平台。

X2Y2 在产品设计上追求优于 Opensea 的用户体验，除了基础 NFT 买卖外，还推出了批量上架、批量购买和点对点转账交易等附加功能，平台收取 0.5% 的交易手续费，相比于 Opensea 上 2.5% 和 LooksRare 上 2% 的手续费，低价策略对用户交易更为友好。团队公布了路线图，目前正在对平台功能进行持续性完善，如拍卖功能将很快推出，未来将考虑二层的部署。

X2Y2 通过一系列挖矿奖励已占据了一定市场份额，但存在刷量虚假交易。项目上线时通过代币空投方式对 Opensea 进行吸血鬼攻击，此前推出了挂单奖励和 Gas 费用补贴，由于产生的激励效果不佳均已被取消；团队于今年五月份正式推出交易挖矿奖励，用户和交易量显著上升。交易挖矿在短期内能够形成正向循环，但该种激励模式会导致刷量虚假交易的产生，不但没有给平台带来真实流动性，还会挤压真实用户的交易奖励，一旦奖励下降，用户将很快流失，可见交易挖矿仅是早期获客手段，无法形成项目的护城河。另外，由于 X2Y2 上大部分蓝筹项目是没有设置版税费的，无法通过剔除没有版税的交易来还原真实交易量，因此仅剔除了 X2Y2 上点对点两地址间倒手买卖的交易数据，发现剩余交易量占全部交易量不到 10%，真实交易量会更低一些，平台存在大量的刷量交易。

目前 Opensea 在 NFT 综合交易市场占据着绝对优势地位，X2Y2 和 LooksRare 选择恰当时间点利用代币空投和挖矿奖励抢占了一小部分市场份额，但二者均存在刷量交易问题。在 LooksRare 分阶段缩减交易奖励的机制下，会使得每次奖励减半对平台交易量和代币价格有很大的冲击，相比之下 X2Y2 采用每天固定数量的奖励能够降低该种打击，两年的线性释放时间给予项目更长缓冲期来打磨产品。目前 X2Y2 业务数据优于 LooksRare，但如何优化产品体验留住用户，同时持续性的吸引新用户和高质量项目进入是未来发展的关键。

1. 基本概况

1.1 项目简介

X2Y2 是建立在以太坊上的 NFT 综合交易市场，除了基础 NFT 交易功能外，还推出批量上架、批量购买和点对点转账交易功能，平台收取 0.5% 的交易手续费。项目上线时采用代币空投的方式对 Opensea 进吸血鬼攻击，又先后推出了挂单奖励和 Gas 费用补贴，激励效果不佳已被取消；团队于今年五月份正式推出交易奖励活动，吸引了大量用户，交易量显著上升。

2. 项目详解

2.1 团队

X2Y2 团队匿名，仅知晓团队创始人名为 TP，是位华人，曾在 AMA 中提及到自己跟团队是 2017 年进入到的区块链行业，团队成员大部分拥有 NFT 交易所的从业经验，是个国际化的团队。目前无法进一步获取团队成员更多的履历信息，整体上无法判断团队成员实力水平。创始人的推特信息如下：

2.2 资金

截至目前，X2Y2 暂未披露任何外部融资信息。团队表示不会进行任何私募融资，只进行过基于白名单系统进行代币的公开市场预售（Initial Liquidity Offering），共筹集到 1500 ETH。ILO 详细内容见 4.1.2。

2.3 代码

X2Y2 代码目前还未开源，团队表示未来会逐渐开源的，API[3]和 SDK[4]已经公布。此前，Cobo 创始人神鱼发推文表示经技术人员查阅在 X2Y2 NFT 挂单挖矿存在风险，理论上官方存在升级合约转走用户 NFT 的可能，团队据此为了降低风险立即将合约的管理权限转移至多签钱包，同时加上了时间锁。协议智能合约已经通过了审计，官方对审计报告也已进行了披露[5]。

2.4 产品

X2Y2 是建立在以太坊上的 NFT 交易平台，产品功能上与 Opensea 类似，主要提供 NFT 交易买卖功能，还开拓了稀有度查询、批量上架和批量购买等附加功能，另外还支持点对点的交易功能，产品操作简便，用户体验较好。平台对 NFT 交易收取 0.5% 的手续费，手续费收入奖励给生态中的 X2Y2 代币持有者。目前平台支持的钱包类型多样，包括 Metamask、Coinbase Wallet、imToken 和 SafePal 等等。

用户能够在平台上搜索查询 NFT，首页会显示近期热搜关注的 NFT 系列（Trending NFTs），用户点击即可查看该系列 NFT 的详情。另外，平台还收录了 X2Y2、Opensea 和 LooksRare 上的热门 NFT 系列（Hot Collection），用户可以看到每个系列 NFT 当前的市场数据，包括交易量、交易额、交易平均价格和持有人数，更多详情可以点击 Ranking 功能模块，用户可以在该模块能够看到每隔 NFT 系列的简介信息，了解到目前市场上的 NFT 热点。

图 2-2 X2Y2 首页热门 NFT 界面[6]

用户连接钱包后点击钱包头像即可看到数据面板，该面板会披露钱包中的目前的 ETH、WETH 和 X2Y2 代币余额以及 X2Y2 代币奖励；最关键是该页面提供了目前钱包中持有的 NFT、用户收藏的 NFT、出价的 NFT 已经批量上的 NFT 等统计模块，方便用户归类查询，也是用户进行部分功能的入口端。

2.4.1 购买

用户连接完钱包后即可在平台上进行 NFT 交易，用户可以选择购买单个 NFT，也可以进行批量购买。

（1）单个购买

当用户选择购买单个 NFT 时，可以进行搜索 NFT 名称和合约地址来找到该 NFT。平台一般会显示该 NFT 的稀有度，如下图所示，编号 7245 的 Stray Dog 的稀有度排名 5476。一般情况下 NFT 特征越稀有的，价值越高。X2Y2 官网集成了团队所开发的三个特征工具：rarity rank、trait sniper 和 ratity sniper rank，用户在购买页面下方的属性列表中可以看到特征信息，也能看到该 NFT 的历史交易活动记录。用户也可以点击 NFT 旁钻石指标的稀有度排名值来查看该系列所有 NFT 的稀有度详情，如下图所示。

图 2-4 购买单个 NFT 操作页面[7]

图 2-5 稀有度属性详细页面[8]

用户点击可以选择立刻以固定价格或者出价的方式来购买 NFT。当选择以固定价格购买，首次购买的用户会弹出接受服务条款的对话框，用户点击确认继续购买即可；当选择为 NFT 出价，用户需要点击「Make Offer」进入出价操作页面，用户选择出价金额和到期时间即可，报价是以 WETH 来计价的。如下图所示，为编号 7245 的 Stray Dog 出价，金额为 0.015 WETH，到期时间为 30 天。

图 2-6 出价购买页面

用户也能够对整个 NFT 系列进行出价，平台对能够进行系列出价的 NFT 都设置了蓝色标识，如下图所示，用户点击「Make Collection offer」即可对该系列 NFT 报价，只需要输入报价金额和到期时间即可，以 WETH 计价，报价可以随时取消。用户报价后金额并不会立刻被交易划走，而是等到有卖家接受出价才会完成交易。如果卖家接受了其中一个 NFT 的报价，那么用户对系列中其他 NFT 的报价都将会取消。

图 2-7 系列报价 NFT 标识

（2）批量购买

X2Y2 的批量购买功能与淘宝购物类似，用户需要将其先添加到购物车中，再通过购物车来进行批量付款购买，支付一次性的 Gas 费，从而节省 Gas 费用。在每个项目的下方都有立即购买和添加购物车两种选项，用户点击「Add to cart」就可以将其添加到购物车中；用户点击钱包头像旁的购物车按钮就可以将添加的 NFT 进行批量购买，使用 ETH 或 WETH 进行购买。

图 2-8 购物车批量购买页面

2.4.2 出售

与购买活动一样，用户可以选择以固定价格出售 NFT，团队正在开发拍卖功能，目前还未正式推出。用户点击钱包选择想要出售的 NFT，设置好出售的价格和到期时间即可。

如果用户想要接受买方出价来出售 NFT，需要先点击钱包选择 NFT，点击「Offers」选项即可看到每个买家的出价情况，如下图所示，编号 13121 的 Zorb 目前买方出价 0.0277 WETH，用户点击接受按钮完成批准后即可完成出售交易。

图 2-9 接受报价操作页面

2.4.3 批量上架

用户点击钱包中的「Bulk Listing」即可进行 NFT 批量上架操作，批量上架分为三个部分，首先是许可授权，每个系列的 NFT 需要收取 Gas 费用，该笔费用是给矿工的；其次用户选择多个 NFT 填写上架价格；最后点击「List」上架，进行钱包签名即可。每个系列 NFT 只需要进行许可授权一次，后期再次进行上架是不需要进行许可的。

图 2-10 批量上架操作页面[9]

2.4.4 点对点交易

X2Y2 的点对点交易是通过预先指定的钱包地址实现的，即 OTC 功能，能够实现无平台手续费和版税费的交易。用户点击钱包中的 NFT，选择「Fixed Price」功能，在交易页面输入价格以及交易地址，意味着只有该交易地址可以购买次此 NFT；最后点击「Approve」即可完成。该交易是免费的，没有手续费和版税费。当对方通过钱包地址登陆 X2Y2，平台会自动出现定向购买的信息，用户点击立即购买即可。该功能对于有真实 NFT 交易转账的用户比较友好，但同时无手续费和版税费的设置可能会给平台带来虚假交易，用户利用点对点交易，将 NFT 在自己的两个钱包中频繁转换来获取交易奖励。

图 2-11 点对点交易页面

总结：X2Y2 目标是与 Opensea 争夺市场份额，在产品功能设计上追求优于 Opensea 的用户体验，除了基础 NFT 买卖交易外，平台还推出了批量上架、批量购买和点对点转账交易等附加功能；另外还集成了团队所开发的三个特征工具，方便用户对 NFT 的稀有度进行查询。平台收取 0.5% 的交易手续费，点对点转账交易不收入版税费和交易费，对于有真实 NFT 交易转账的用户较为友好，但也会给平台带来刷量交易。团队目前仍对平台功能进行持续性完善，如拍卖功能将很快推出。

3.2 现状

3.2.1 运营数据

X2Y2 于今年二月份正式上线以太坊主网，上线时通过代币空投的方式向 Opensea 进行吸血鬼攻击，同时推出了挂单奖励活动和低手续费方式来吸引第一批用户；挂单奖励持续到四月份就被取消了，后又推出了维持一个月的 Gas 费用补贴，现也已取消。团队于今年五月份正式推出了交易挖矿奖励。通过一系列的获客策略，目前平台已经积累一定的交易量，根据平台数据显示，目前平台 24 小时交易量达到 15,242 ETH[11]，以下是其详细业务数据：

（1）交易量

图 3-1 Dune Analytics 上 X2Y2 周交易量数据[12]

从上图的交易量数据可以看出，目前平台已经积累一定的交易量，交易挖矿奖励起到较为明显的激励作用。X2Y2 上线时推出了挂单奖励并没有给平台吸引来用户，四月份推出的 Gas 费补贴活动也没有带来交易量的增长；但随着交易挖矿活动的推出，平台的交易量迅速增长，最高的周交易量达到了 20,000 ETH；当交易挖矿开展到一定阶段，平台积累一定的用户和市场知名度，数据能够维持在相对稳定状态，目前根据 Dune Analytics 上的数据统计累计交易量达到 1,387,257 ETH，过去 24 小时交易量达到 15,242 ETH。

（2）用户人数

图 3-2 Dune Analytics 上 X2Y2 的活跃用户人数

从上图可知，X2Y2 的日活跃人数对交易量变化数据二者相对应，挂单奖励并没有吸引到用户，但交易挖矿活动的开启给平台带来了用户数从而促进交易量的提高，最高峰时今年六月初日活跃人数突破了 4,700 人；伴随加密货币熊市情绪以及整个 NFT 交易市场交易量的下降，平台日活跃人数有所下降，但依靠着此前系列奖励活动的激励，平台已经积累了一定的用户群体，根据 Dune Analytics 的数据显示，平台累计用户数已经达到 724,828 人，目前日活跃用户量集中在 3,000 至 4,000 人。

（3）交易手续费

图 3-3 Dune Analytics 上 X2Y2 的手续费数据

X2Y2 为吸引用户在四月份时推出了为期一个月的 Gas 费返还活动，目前在平台上交易需要收取 0.5% 的手续费，相比于 Opensea 收取 2.5% 的手续费，X2Y2 有着明显的价格优势。低费率交易也是获客策略之一，从上图可以看出，X2Y2 上的手续费收入在不断上升，特别是五月份交易挖矿启动后激增，在代币激励下用户涌入平台，促使交易手续费上升，质押收益上升，代币价格上涨，交易奖励上升，进一步吸引用户交易，手续费收入再次增加，如此形成正向循环。

（4）真实交易量

交易奖励在短期内能够促使平台用户数和交易量显著上升，X2Y2 的用户数和交易量甚至一度超过 Opensea，市场数据表现良好，但是该种激励模式会给平台带来刷量虚假交易，一种方式是用户选择没有创作者费用的 NFT，通过左手倒右手的方式进行买卖，完成交易挖矿获取奖励；另一种方式是平台推出的点对点交易功能，用户利用该功能实现 NFT 在两个地址之间频繁转换来获取奖励。两种方式的整个过程并未给市场带来流动性，还积压了真实用户的参与交易的积极性。从下图的统计数据也可以看出，目前 X2Y2 平台上存在刷量情况，前端呈现的交易数据需要剔除刷量部分才是平台真实的交易量。

图 3-4 X2Y2 上存在部分刷量交易数据[13]

图 3-5 X2Y2 上剔除刷量交易的真实数据[14]

根据 Dune 上的数据统计，剔除了两个地址之间买卖倒手的交易后，目前 X2Y2 上真实交易量为 200 万美元左右；此前包括刷量的交易数据大约是 2,500 万美元，可见上图剔除后的真实交易量占总交易数据之比不到 10%。但是需要额外说明的是，还有一种零版税 NFT 交易的情况未剔除，但考虑到 X2Y2 对于项目方未进行版税设置的 NFT 都默认为零版税，部分零版税交易也可能是真实交易。综上所述，上述数据仅剔除了 X2Y2 上左右手倒卖交易的情况，与真实交易量仍有偏差，真实交易量实际上会更低一些。

3.2.2 社交媒体规模

表 3-2 X2Y2 的社交媒体数据

社群规模能够反应项目在市场上的受欢迎程度和未来发展潜力，截至目前 X2Y2 主要开设了 Twitter 和 Discord 两个社交媒体平台，Discord 中设有专门的中文社区，日常交流较为活跃，用户讨论度较高，团队对用户的 bug 反馈处理较为及时；项目于 22 年 1 月加入 Twitter，至今累积了 8.33 万的粉丝量，消息日更，评论点赞人数较多。

3.3 未来

团队在官网公布了路线图[15]，包括短期、中期和长期计划，制定了多个细分目标。在短期计划中，目前已经实现了批量上架、批量购买和分析图表等功能，但与 Opensea 集成、拍卖、NFT 项目启动工具和支持 ERC-1155 等功能还未开发完成，其中 ERC-1155 的功能团队在社区中表示近期将会推出。团队在中期计划中协议将致力于官方 NFT 更多功能的开发以及 X2Y2 代币的赋能，在长期计划中将考虑二层的部署。

总结：X2Y2 此前推出的挂单奖励和 Gas 费补贴活动并未给平台带来交易量的增长，取消后于五月份正式推出交易挖矿奖励，产生了明显的激励效果，用户数和交易量都迅速增长，数据表现良好。交易挖矿在短期内是吸引用户提高交易量的最有效直接的方式，但是该激励模式会给平台带来刷量虚假交易，刷量交易不但没有带来真实流动性，还会挤压真实用户的交易奖励。通过剔除 X2Y2 上点对点两地址间倒手买卖这部分的交易量，平台真实交易量占全部交易量不到 10%，实际交易量会更低一些。团队路线图清晰，目前致力于平台功能的开发，未来考虑二层的部署。

4.1.2 ILO [17]

团队于 22 年 2 月 14 日启动 ILO 计划，对 1,000 名在 Beta 版上的交易用户和社群用户开放白名单[18]，以抽奖方式获取白名单，共出售了 1,500 万枚 X2Y2，占总供应量比例为 1.5%，1 枚 X2Y2 价格为 0.001 ETH（约 0.25 美元）。团队将参与 ILO 计划的 X2Y2 分成 1,000 份，每份包含 1.5 万枚的 X2Y2，价值 1.5 ETH，共筹集了 1,500 ETH，该部分资金与质押奖励中的 1% 代币（即 1,000 万枚）会被一起存入到 Uniswap 上作为初始流动性，作为初始流动性获得的 LP 代币也将会被销毁，意味着这部分流动性将会锁定在池中。

通过 ILO 出售的 X2Y2 代币将会在 360 天内按照区块线性解锁，解锁的 X2Y2 可以用于交易和质押，用户持有通过 ILO 获得的 X2Y2 能够参与获得一定比例的 WETH 市场交易费用作为奖励。另外，白名单用户有机会参与领取官方 NFT，即 Ajin Youth Rebels，简称为 AYOR，是附属于 X2Y2 的权益类 NFT 合集，持有该 NFT 能够享有一定权益，例如在交易奖励中买方能够获得双倍奖励。Discord 社区中的 OG 角色最多可以领取两个 NFT，其他白名单用户只能申请一个。团队通过 ILO 策略向普通用户募资，对于早期平台来说短时间内能够赢得用户对平台的信任，由于利益绑定，能够让用户自发的其组织宣传 X2Y2。

4.1.3 代币空投

（3）小结

交易挖矿是使用代币激励的方式来促进交易，在上线初期能够形成正向循环，用户为了获取奖励会不断在平台上交易，促使交易量显著上升，从而平台手续费用会上升，手续费分发给质押代币者，质押收益提高后吸引更多持有者质押代币，代币价格上涨，进一步有又促进了交易挖矿激励上升，从而形成一个正向循环。短期内交易挖矿能够给 X2Y2 吸引大量的用户和交易量，但从长期视角来看，并不能带来持续的发展。当巨鲸用户涌入市场时，通过设置成交价格虚高的 NFT 交易来挤压散户的交易挖矿空间，再利用左手倒右手的方式完成交易来获取交易奖励，该过程实质上并没有为平台带来流动性，并未给平台带来有效用户，同时也挤压了散户参与交易的积极性。用户在交易平台的转移成本较低，一旦交易奖励下降，将很有可能导致用户流失，质押奖励下降，进而导致币价下跌，陷入死亡螺旋。

4.2 需求

4.2.1 代币用途

X2Y2 目前未有清晰明确的用途。协议注重社区用户，社区投票参与经济模型修改，团队承诺后期会以 DAO 形式管理。

总结：X2Y2 通过空投代币对 Opensea 进行吸血鬼攻击，初期团队希望快速提升流动性推出了挂单奖励，但对交易量的激励效果并不明显而被取消。经过几次经济模型修改于今年五月团队又推出了交易挖矿奖励，短期内能够形成正向循环，快速吸引到用户流量，但长期内并不能给平台带来持续性的发展，一方面巨鲸用户进场刷单套利，挤走了散户的交易奖励，也并未给平台带来有效用户；另一方面用户转移成本低，一旦奖励下降用户很有可能快速流失，平台交易量骤减，导致交易奖励进一步下降的负向循环；可见交易奖励只是早期的获客手段，无法形成平台的护城河。

5. 竞争

5.1 行业概述

5.1.1 定义

X2Y2 的细分赛道是建立在以太坊上的 NFT 综合交易平台。

5.1.2 现状

（1）高速增长的 NFT 市场

非同质化代币（NFTs）市场在 21 年掀起一场风暴，其交易量和用户出现爆发式增长。根据 NFTGO 的统计数据显示，可以看出在过去一年以太坊上的 NFT 交易量和市值快速增长，特别是 21 年下半年市场经历了一次较大幅度的增长。今年年初，在比特币和以太等市场行情下跌的情况下，NFT 交易市场再次迎来了爆发式增长，大量资金和用户不断涌入市场，市值和交易量再次突破了历史新高。最近几个月以来，伴随着整个加密货币市场的下行，以太坊上的 NFT 市场也有所下跌，但总交易量仍然高于 21 年。目前以太坊上 NFT 市场的交易市值达到 246.3 亿美元，七天交易量达到 2.92 亿美元。

图 5-1 以太坊上 NFT 市值与交易量变化趋势图[25]

（2）NFT 交易平台竞争格局

NFT 交易平台的竞争也随着用户和资金的进场愈来愈激烈。头部 NFT 综合交易市场 Opensea 是目前最大的 NFT 交易平台，囊括了各个种类的 NFT，占据综合类 NFT 交易市场份额的 80% 以上，拥有最好的流动性，是大多是 NFT 玩家的首选平台。其在 21 年初获得了 3 亿美元的融资，市场打开了对 NFT 综合交易平台的想象力，Opensea 面临着来自新兴交易所的冲击。

从纵向维度回顾 NFT 交易平台发展，大致可以分为三个阶段：首先是每条公链上开始涌现了自己原生的 NFT 综合交易市场；其次是各个公链上开始出现不同的挑战者，包括层次不穷的仿盘，都在与该公链上的头部项目竞争；最后在各公链上竞争胜出的项目将考虑扩展进军其他公链，进一步扩大市场份额。尽管各大公链都在跟以太坊争夺流动性，部分项目目前也占据了一定的市场份额，如 Solana 链上最大的 NFT 交易市场 Magic Eden 总交易额已突破 15 亿美元，近期刚完成 1.3 亿美元的 B 轮融资；但目前 NFT 市场的主要资金和用户仍然在以太坊链上，以太坊是 NFT 交易的主战场，也是交易所们竞争的主要阵地。

目前以太坊上市场份额占比前三的交易所是 Opensea、LooksRare 和 X2Y2，但整体上还是 Opensea 一家独大的局面。LookRare 和 X2Y2 诞生于今年年初，上线均采取了吸血鬼攻击的方式来跟 Opensea 争夺市场份额，通过向早期 Opensea 的用户发送空投代币来吸引用户，并推出了质押代币分发交易手续费、交易挖矿和挂单奖励等方式来吸引用户参与到平台交易，在今年年初的 NFT 牛市中二者都快速获得了用户流量，但代币激励的方式依旧无法撼动 Opensea 的龙头地位。

5.2 竞品项目

5.2.1 Opensea

18 年就开始深耕 NFT 赛道，发展至今成为目前涵盖类别最广泛，数字商品最多的 NFT 交易市场。项目获得 a16z 等业内顶尖投资机构的融资，财政充足。项目目前尚未发行代币，有美股上市的倾向，Opensea 的首席财务官于 21 年 12 月表示会进行 IPO，在今年 1 月 4 日以 133 亿美元的投后估值获得了 3 亿美元的融资。今年 4 月 6 日，Opensea 正式上线基于 Solana 发行的 NFT，并以 SOL 计价，这是继以太坊、Polygon 和 Klaytn 之后 Opensea 支持的第四个区块链网络。近日，平台宣布正式启动 Solana Launchpad，支持创作者在 Solana 链上铸造 NFT。受加密货币不断下跌情绪的影响，团队宣布将裁员 20% 以渡过加密熊市，联合创始人 Alex Atallah 也宣布将于 7 月 30 日离职。

5.2.2 LooksRare

成立于今年年初，1 月 11 日正式上线以太坊主网，上线之初就推出了协议代币 LOOKS，并对 Opensea 的用户进行空投，随后启动交易挖矿活动来吸引用户，实际交易量在高峰时达到 Opensea 的 28% 左右。上线之前团队还通过战略销售向早期投资人募集资金，投资者包括丹华资本、知名 NFT 玩家 Dingaling 以及部分推特大 V 等，借助其社区影响力来完成最初的市场营销。团队于今年 3 月份宣布与聚合起 Gem 合作，交易量瞬间上涨；在上线四个多月后团队又推出了挂单奖励来激励交易量。近日，平台宣布与 Coinbase Wallet 集成，团队目前正持续的对平台功能进行改善。

5.3 竞品要素

5.3.1 运营策略

Opensea 在早期就与众多的 NFT 项目合作，依靠着丰富的项目和大量的用户建立了庞大的交易市场，在去年一年的交易量就达到了 140 亿美元，近期还拓宽了自己的网络部署打开了 NFT 市场，其他 NFT 交易市场都在采取各自措施争夺 Opensea 的用户。今年年初的时候，Opensea 以 133 亿美元的估值完成融资，团队 CFO 表示 Opensea 在计划上市，打击了市场上强烈空投预期的用户，LooksRare 和 X2Y2 便采取了代币空投的方式对其进行吸血鬼攻击，以及展开一系列挖矿奖励来争夺市场份额。

X2Y2 是今年二月份正式上线的，为争夺 Opensea 的市场份额开展了一系列的运营策略：1）首先在上线前期通过首次流动性发行（ILO）的方式来进行募资，以抽奖方式获得白名单，参与 ILO 获得代币的用户交易费用奖励和官方 NFT，持有官方 NFT 在交易挖矿中能够获得更多的奖励比例，该种募资方式让用户与平台绑定，利益驱使下能够让早期用户去社区中宣传。2）其次 X2Y2 上线首次推出了挂单奖励，激励 NFT 持有者来平台上上架，当平台上 NFT 种类丰富后会吸引到更多用户来交易，但由于上线初期规则未设置完善，导致激励效果并不佳。3）再者团队在四月份推出了为期一个月的 Gas 费用补贴来吸引用户前来交易，效果并不佳也被取消了。4）今年五月团队正式推出交易挖矿奖励，启动后用户涌入，平台交易量显著上升。

LooksRare 上线时间比 X2Y2 早，对代币空投设置了领取规则，用户需要挂单才能领取代币，该措施给平台带来了一批新用户。相比于 X2Y2 的 ILO，LooksRare 采用的是战略销售向早期投资人募集资金，早期投资人是 NFT 行业知名玩家和部分推特大 V，希望借助其在社区的影响力来完成早期市场营销。上线初期团队就推出了交易挖矿，交易量直线飙升，数据上曾一度超过 Opensea，但随着后期挖矿奖励减半，市场份额出现大幅度下降。团队在今年四月份改变了激励策略，推出了挂单奖励，挖矿奖励和挂单奖励相结合。

小结： X2Y2 与 LooksRare 采取的挖矿激励计划最大的区别在于：X2Y2 由挂单奖励转为交易奖励，LooksRare 在最早的交易奖励基础上又推出了挂单奖励。

挂单奖励属于单边激励，激励卖方来平台挂单 NFT，当平台上的能够交易的 NFT 种类和数量比较丰富，能够吸引更多的用户交易。从 X2Y2 上线就启动挂单奖励可以看出，其产生的激励效果并不好，一方面有真实卖出 NFT 需求的用户会更倾向于选择在 Opensea 挂单，因为 Opensea 的高流量会促使其能更快的被成交；另一方面单纯想通过挂单获取代币奖励的用户会通过设置高额的价格来保证 NFT 不被成交，这样只是提高了挂单数量，对交易的促进并不明显。

交易挖矿属于买卖双方激励，通过代币激励来吸引用户进行买卖交易。交易挖矿能够快速解决项目冷启动的问题，吸引用户体验产品促进交易量上升，在短期内是很好的获客策略，但同时也会带来刷量虚假交易的问题。当套利者发现代币奖励收益大于交易成本时，参与挖矿的方式之一是通过选择没有创作费用的 NFT，设置高额的挂单价格，再通过左手倒右手的方式进行买卖实现交易挖矿，这种方式实际上并未给平台带来真实流动性，同时还会挤压真实用户的交易奖励。当交易挖矿启动一段时间后，新用户涌入量会逐渐下降，但平台已经积累了一定的用户群体，交易量会维持一段时间相对稳定，而能够持续的时间长短则取决于交易挖矿的设计，一旦激励奖励下降，容易导致交易量迅速下降，质押收益下降的负循环中。X2Y2 和 LooksRare 都存在有刷量交易的问题，关键在于后期如何继续打磨产品，提高用户的转移成本，来预防奖励下降用户流失的情况。

5.3.2 代币机制

Opensea 没有发行代币，并且有美股上市的倾向，今年年初以 133 亿的投后估值获得了 3 亿美元融资，用户对其空投预期降低。

X2Y2 和 LooksRare 上线就空投代币，二者经济模型设计中代币分配大致相同，主要区别在于初始代币销售和代币奖励机制设计不同。1）首先 X2Y2 早期通过 ILO 分配的代币会在 360 天内线性解锁；而 LooksRare 分配给早期投资者的代币会在上线 180 天后一次性释放。2）其次在交易奖励设计中，X2Y2 设计的是每天固定数量释分发 624,902.8 枚 X2Y2，720 天内分发完，其中 604,902.8 枚是预留分发给卖方，20,000 枚留给买方，偏向于激励卖方市场；LooksRare 是将奖励分成四个阶段，360 天内分发完成，交易奖励会逐次衰减，每次衰减都会对平台交易量和代币价格有很大的冲击。因此相比较下 X2Y2 的交易挖矿的时间会更久一些，并且持续固定数量的奖励能够降低对交易量的冲击。

5.3.4 产品体验

NFT 交易平台的转移成本较低，大部分项目都是在 Opensea 的蓝本上进行产品优化，如 X2Y2 推出了 NFT 批量上架，购物车批量购买和点对点转账交易等功能；LooksRare 提供全系列 NFT 一键出价等功能。另外，手续费是影响 NFT 交易的重要因素之一，Opensea 占据龙头地位收取了 2.5% 的交易手续费，LooksRare 收取 2% 的手续费，而 X2Y2 主打低价策略，仅收取 0.5% 的手续费，对用户交易更为友好。

总结： 目前 Opensea 在 NFT 综合交易市场占据着绝对领先的优势，X2Y2 和 LooksRare 通过代币空投和一系列奖励挖矿方式抢占了小部分市场份额。二者采用交易挖矿方式在短期内能产生明显的激励效果，但同时会产生刷量虚假交易问题，挤压真实交易者的奖励，并不能给项目带来长期持续性的发展。在 LooksRare 分阶段缩减交易奖励的机制下，会使得每次奖励减半对平台交易量和代币价格有很大的冲击，相比之下 X2Y2 采用每天固定数量代币奖励能够降低该种冲击，给予了平台更长的缓冲期去打磨产品。另外，X2Y2 主打低价战略，仅收取 0.5% 的手续费，对用户较为友好。目前 X2Y2 的业务数据变现优于 LooksRare，但如何优化产品体验留住用户，同时持续性吸引新用户和新的 NFT 项目进入，捕获更多的流量是未来发展的关键。

6. 风险

1）NFT 估值泡沫

此前由于 NFT 市场的火热不少 NFT 被卖出高价甚至天价，高价格下存在着泡沫；随着行业的发展，后续将不可避免的出现大量仿盘出现，用户需谨慎投资。近日伴随加密货币行情下行，NFT 市场的交易量明显在下降。

2）代币奖励模型不具备可持续性

交易挖矿奖励在短期内能够吸引大量用户，但长期内并不能给平台带来可持续性的发展，当巨鲸用户发现有利可图，进场刷单套利会挤压散户的奖励，同时并未给平台带来真实交易量；用户转移成本低，奖励下降用户将会快速流失，交易量骤减造成奖励进一步下降的负向循环。
