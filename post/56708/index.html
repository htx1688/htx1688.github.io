GameFi 2.0 的发展与挑战

GameFi 2.0 作为游戏与金融相融合的下一个阶段，目前尚未有一个统一的定义和标准，但各家厂商之间依然存在相对一致的共识——即朝着优化和改进 GameFi 1.0 现有问题的方向努力，如改进经济模型的可持续性、可玩性以及留存率等 ，GameFi 2.0 的发展旨在解决这些问题，为游戏金融领域带来更加稳定和可持续的发展 。

在 GameFi 1.0 中，大多数项目的经济模型设计不够合理，导致了经济系统的不稳定性和不可持续性。而像以 Stepn 等代表的 GameFi 2.0 项目通过引入更加健全的经济模型（双代币、合成机制和资源消耗机制）等，来提高整个游戏金融生态系统的稳定性和可持续性，使得玩家和投资者从中获得长期的收益，进而增加其留存率。

GameFi 2.0 还致力于提升游戏的可玩性，早期的 GameFi 项目重金融性而轻游戏性，甚至在很长一段时间内完全放弃了其游戏性以促进其“左脚踩右脚”的内循环，这样做固然在短期内能够涌入用户，但一旦经济模型不再起效，投机者们就会纷纷离场。GameFi 2.0 尝试通过引入更加丰富和多样化的游戏玩法、挑战和奖励机制，来增加玩家的参与度和游戏的乐趣。这样一来，玩家将更加愿意花费时间和精力来真正游玩游戏，而非单纯因为游戏所带有的金融属性而来，进而能够增加整个生态系统的活跃度和吸引力。

然而，GameFi 2.0 的发展也面临一些挑战。不同的厂商对于 GameFi 2.0 可能有不同的理解和实践，无论是前期的“ 3A 大作”亦或后来的链改，其成果目前仍然未能够使得行业达成一致的共识，也使得整体的发展在新模式出现前趋于缓慢；其次，大部分 GameFi 2.0 项目仍然将重心押注在经济模型的鲁棒性和可循环性上，缺乏与真正用户间的交互，这种趋势在短期内依然难以改变。

探索与创新

尽管如此，许多开发者仍然通过自己的奇思妙想以及精妙的设计探索着 GameFi 2.0 的道路，最典型的例子就是 Stepn——通过向Web3游戏融入线下可扩展性和社交属性，以及将真实世界的跑步与虚拟世界的"Play-to-Earn"相结合，Stepn 为玩家在经济、社交和身心健康方面带来了全面的反馈。

Stepn 的成功在于实现了行业内期待已久的"破圈效应"，将区块链和 NFT 的概念扩展到以往 GameFi 游戏所无法触及的人群中。通过将跑步与游戏相结合，Stepn 创造了一种全新的游戏体验，鼓励玩家积极参与身体活动，并通过完成任务和达成目标获得经济回报，这种独特的玩法不仅增加了游戏的乐趣，还激发了玩家的健康意识和积极参与社区内社交互动的愿望。

Stepn 的成功经验为 GameFi 2.0 的发展提供了有益的借鉴。它证明了通过创新的玩法、线下可扩展性和社交属性的融入，可以吸引更广泛的玩家群体参与到 GameFi 游戏中 ，同时，Stepn 也为其他开发者提供了启示，鼓励他们以类似的创新思维和设计理念探索 GameFi 2.0 的未来发展方向。

而像 Block Ape Scissors 等平台则正在尝试着另外一条道路——在重视社交媒体的宣传同时，在适当增强可玩性的基础之上完善其经济系统和模型，将 DeFi、NFT 与游戏有机结合，最终达成“三条腿走路”的稳固生态系统 。

另一种尝试

Block Ape Scissors 于最近推出了其第一款游戏 Project Ape Shooter，这是一款快节奏的街机射击游戏，主要以第三人称视角展现。 它将虚幻 5 的最新技术与完善的 Web3 生态系统相结合，并采用了免费游玩的模式，玩家可以通过购买 NFT 和 Token 以获得额外的增强，其中 NFT 将存储玩家游玩过程中用于升级的统计数据和特征，这些特征将使玩家能够个性化自己的游戏风格。

在 Project Ape Shooter 中，玩家将有机会装备和打造自己的 NFT，使其具备 5 项数据（力量、敏捷、耐力、智力和运气），并结合 12 种不同的特质，从而在战斗中获得特殊能力，同时还可以解锁和调整 21 种各不相同的武器，武器又分为 3 个类别和 5 个等级，而且未来还会有更多武器加入。

不仅如此，玩家还可以通过持有 Banana 代币来增加战斗通行证的进度，以解锁特殊物品，例如武器和角色化妆品等，并且通过装备 NFT，玩家可以解锁专属的自定义可玩角色，并通过参加排名赛来证明自己，以攀登排位等级。

目前，Project Ape Shooter 的抢先体验版已经发布，并且 正在 Twitch 上进行直播展示 。 抢先体验版在 Epic Games Store 上发布，并提供多人游戏功能，若想在游戏中展示 NFT 资产，玩家只需将其钱包与 Epic Games 帐户 ID 进行链接即可。这个抢先体验版将为 Block Ape Scissors 的Web3生态系统提供初步支持和打下基础。

Block Ape Scissors 的定位

Block Ape Scissors 是由 Binance Labs 孵化并支持的 Web3 游戏平台，其目标是成为 BSC 上最活跃的游戏平台，连接开发者和玩家，让他们共同创造和推广游戏，并参与游戏决策。 Block Ape Scissors 将游戏、NFT 和 DeFi 进行无缝集成，将游戏体验与流动性挖矿连接到一起，为全新的游戏赚取奖励系统铺平道路，并将成功的游戏模式融入 DeFi 的新世界，通过这种有趣、令人兴奋且有利可图的方式，玩家可以在享受到游戏乐趣的同时达到P2E的目的，同时享受个性化体验。

作为一个去中心化的游戏平台，Block Ape Scissors 旨在通过其 DAO 治理模式，为游戏创作者和玩家提供一个新的公平、透明、非垄断的游戏生态系统。这一愿景颠覆了传统游戏行业的权力结构，使游戏创作者能够在市场上拥有更大的影响力和收入。此外，社区由大量游戏玩家和游戏开发者组成，并共同在 DAO 的框架内共同管理生态系统，促进 IP、内容、游戏、公会等的相互整合和融合。

同时 Block Ape Scissors 还为用户提供了 Banana Miner 功能。 购买 Banana Miner 的用户可以对其 NFT 进行质押或取消质押，创建矿池的用户被称为“矿工”。矿工在创建矿池后需提交申请以获得官方认可。矿池激活后，矿工将获得定期收益，并在其他用户加入矿池时获得额外收益。新用户带来的部分收入将自动奖励给矿工。矿工可随时对其 NFT 进行质押或取消质押。收益根据质押金额计算；若未质押 NFT，则不会产生收益。创建矿池的矿工的收入与矿池中 NFT 的数量有关。若矿池中无 NFT，将获得定期奖励；若矿池中有一定数量的 NFT，则采用矿池的质押计算逻辑。

技术上，Block Ape Scissors 还采用了 zk-SNARKs 技术来确保平台交易的隐私性和安全性，同时保持效率和低成本，AIGC 的引入能够使游戏开发人员可以生成更真实和多样化的游戏内容，从而为玩家带来更身临其境的游戏体验，与传统方法相比，内容创建效率更高。
