当有大公司还在考虑如何将大语言模型和产品结合时，已经有大厂开始用 AI 大模型赚钱了。最可怕的是，这家公司，根本，不缺钱！

美国时间 7 月 18 日，在 Microsoft Inspire 活动中，这家老牌巨头宣布，此前发布的 Microsoft 365 Copilot，将以每月额外 30 美元/用户的定价，为商业订阅用户提供 AI 能力 ，而这些 AI 能力将体现在公司的经典产品 Word、Excel 上。 

同时，浏览器上的人工智能助手 Bing Chat，在普通消费者用户中打开市场后，现在将推出面向商业用户版本的 Bing Chat Enterprise，聚焦在给予商业用户更好的数据保护。 

发布会后，微软股价小涨 4%，打破一个月前的纪录 ，创造股价新高——可见资本市场对于微软在 AI 创收上的积极看法。 

萨提亚·纳德拉在 Microsoft Inspire 大会上宣布与 Meta 在新一代大模型上的合作｜Microsoft 

同一天，Meta 公布了新一代大语言模型的 Llama 2，微软成为 Meta 首选的合作伙伴。 

一手 OpenAI，一手 Meta，在商业软件层面又能迅速进行商业变现，微软可能才是 AI 浪潮中的第一个赢家？ 

AI 商业化初胜

早在 4 个月前，紧接着自己深度绑定的 OpenAI 的 GPT-4 发布会，微软在名为「用人工智能重塑生产力」的发布会，官宣将旗下商业软件 Microsoft 365 全线接入 GPT-4，并推出一款新 Copilot，旨在帮助人们生成文档、电子邮件、演示文稿等。 

用自然语言生成 PPT、文档、邮件、从实时视频通话中截取摘要，这些看起来「神乎其神」的技法，让人感叹「社畜」的人生可能要从此改变。 

当然，当时的人们更多还是被 OpenAI 的 ChatGPT 吸引，认为嵌在浏览器中的 AI 助手，更有趣一些。但真相是，将 AI 加入到商业软件中，才真的能赚钱。 

在 18 日的会上，微软正式公布了 Microsoft 365 Copilot 的定价——每位用户 30 美元/月。请注意，这是在用户已经购买或者订阅了 Microsoft 365 服务的基础上，也就是额外收取的费用 。 

根据 CNBC 的报道，Copilot 这额外 30 美元每月的收费，将让微软企业客户每月订阅价格上涨 83% ——实打实的真金白银的赚钱。 

3 月份 Microsoft 365  Copilot 就已经 惊艳众人｜Microsoft 

微软官网介绍，目前能够「享受」Copilot 这额外收费服务的客户包括 Microsoft 365 E3、E5、商业标准版和商业高级版客户。 

Copilot 在 3 月份曝光后，微软已经和通用汽车、固特异等 600 家客户进行测试。可以预料，在商业软件领域根基深厚的微软，在当下这个「AI 焦虑」的时代，用 AI Copilot 再榨出客户的付费意愿，应该不难。 

如果觉得每位用户 30 美元/月的订阅价格偏高，微软还藏了一手——Bing Chat Enterprise，是的，群众喜闻乐见的微软浏览器 AI 助手，现在有了商业版。 

和普通版的 Bing Chat 相同，商业版的 Bing Chat Enterprise 也以网络数据为基础，提供完整、可验证的答案（附有引文）以及包括图形、图表和图像的可视化答案。 

Bing Chat Enterprise 瞄准企业用户｜Microsoft 

但和普通版不同的是，商业版的 Bing Chat 更强调对于商业用户数据的保护。根据微软博客，和商业版 Bing Chat 的聊天数据「不会保存、不会用于训练模型，并且微软无法进行监视和访问。 」 

目前，Bing Chat Enterprise 现已推出预览版，并且免费包含在 Microsoft 365 E3、E5、商业标准版和商业高级版中。 

未来，Bing Chat Enterprise 还将被作为独立产品推出，费用是每位用户 5 美元/每月，相比 Copilot 要亲密一些。 

对于企业用户来说，微软的两个「AI 助手」来的恰逢其时，虽然人们对 AI 助手兴趣很浓，但是作为企业，却对数据外泄格外敏感，有了微软在幕后背书，显然会让企业用户能更放心员工使用 AI 工具来提升效率，也能缓解老板们的「AI 焦虑症」，也许。 

牵手 Meta，两头下注

在 Microsoft Inspire 大会上，微软掌门萨提亚·纳德拉还公布了另一个重磅消息，即联手 Meta，成为后者新一代大语言模型 Llama 2 的「首选合作伙伴」。 

在 Meta 的报告中，团队强调了 Llama 2 在安全性和可靠性方面的突破，接近 GPT 等非开源模型的表现，让其成为更实用的开源模型。用 Meta 自己的话是，「可能是封闭模型的合适替代产品。 」 

「这将会改变 LLM（大语言模型）市场的格局」Meta AI 的大拿 LeCun 在推特上郑重宣布。 

更重要的是，Meta 在宣布 Llama 2 开源的同时，也将其开放给商用。 

你猜这个新一代的语言模型，会部署在哪里？微软 Azure 上！ 

扎克伯格在 Threads 上秀出的与纳德拉的合影｜Threads 

Meta 透露，Llama 2 可在 Azure AI 模型目录中使用，使使用 Microsoft Azure 的开发人员能够使用它进行构建，并利用其云原生工具进行内容过滤和安全功能。 

另外，Meta 还针对在 Llama 2 在 Windows 上本地运行进行了优化，为开发人员提供了无缝的工作流程，为跨不同平台的客户带来生成式 AI 体验 。 

「通过扩大合作伙伴关系，微软和 Meta 支持一种开放的方法，以增加对基础人工智能技术的访问，从而使全球企业受益。」Meta 在官方博客中写道。 

微软和 Meta 的突如其来的「秀恩爱」，多少让人有点吃惊，因为此前人们一直认为，作为投入最多精力和成本的 OpenAI，才是微软进行强绑定的对象。但是此次微软和 Meta 的合作，多少给人一点「赛马」的感觉。毕竟，微软和 OpenAI 的关系，远非人们想象中的那样和谐。 

「远交近攻」，微软这手连 Meta 制 OpenAI 的手法，可以说非常高明。因为无论两家合作伙伴谁的模型更好，收益的要么是微软 Azure 云计算，要么是 Microsoft 365 Copilot——微软稳赚不赔。 

「我们对未来感到兴奋——而我们才刚刚开始。」微软在官方博客中写道。
