投票治理大家都很熟悉，源自 Curve 推出的 veTokenomic 模型，在该模型中，用户必须将其治理 token 按照一定期限锁定为 veToken 间，以提高其获得的 APR 并通过投票来决定协议的治理 token 的释放，有助于平台控制治理 token 通胀，并将更多治理 token 分配给长期持有者。然而 veTokenomics 模型并不是很完善，仍有改进空间，诸如以下问题：

-买入和锁定一段时间的要求阻止了很多用户的参与；

-veToken 持有者的收入不高，甚至可能为零；

-高成本的投票权——用户必须购买、锁定 token 获得投票权，但没有获得被动收入的资格。

而随着 DeFi 的创新，致力于高质押收益又可以提高投票权的解决方案出现了，今天为大家介绍一个通过引入 veToken 机制来最大化用户收益的项目——Magpie。

Magpie 简介

Magpie 是通过与 Wombat 交易平台合作建立的收益聚合器和多链 DeFi 应用，该项目利用了 Curve 的 veToken 机制，通过减少供应来奖励长期持有者，同时将治理权分配给 token 持有者。

该协议由交易平台 Wombat 支持，且 Magpie 目前管理着 Wombat 约 30% 的治理权力，帮助该协议优化其多种跨链 DeFi 服务。截止发文，Magpie 锁仓量为 1916 万美元，而 Magpie 平台涉及到两类重要 token，分别是 WOM 和 MGP，WOM 是 Wombat 交易平台的 token，而 MGP 是 Magpie 的平台 token。

用户可以通过该平台将他们的稳定币、BNB、Liquid BNB、frxETH、ETH、WOM、mWOM 存入单边池中以获得高 APR，同时支持 Wombat 交易平台的治理投票，赚取被动收入。而 Magpie 将所有转换的 WOM 锁定为 Wombat 上的 veWOM，Magpie 通过这种方式积累 veWOM，veWOM 使 Magpie 有权在 Wombat 上获得更高的 WOM 奖励和治理收益。

MGP 是 Magpie 的治理和收益共享 token，用户可以以 1: 1 的比例锁定 MGP 获得 vlMGP。锁定 MGP 可以平台上赚取被动收入并共享治理收益（平台获得 MGP 质押奖励和收益分成）。veWOM 和 vlMGP 持有者的投票权是可以共享的，两类持有者都能对 Wombat 的治理进行投票。

Wombat 与 Magpie

如前文所说，谈到 Magpie，就不得不提起 Wombat 交易平台，Wombat 是一个创新性多链稳定币交易平台，致力于最小滑点 swap 稳定币并在最大收益的前提下进行质押。Wombat 也是采用的 veToken 模型，其平台币为 WOM，WOM 持有者可以通过锁定 WOM 为 veWOM，提供流动性的基础上还可以确保治理权，以此获得更高的 APR。

Magpie 用例、优势

Magpie 可以使 WOM 持有者最大化 WOM 用例，共分为四类：

提高 WOM 持有者的奖励

通过 Magpie，WOM 持有者可以将 token 转换为 mWOM 以获得更高的收益。这有利于持有者获得更多收益：

-MGP

-12% 的 WOM 收入分成

-Magpie 还积累了 WOM 治理权，遵循与 PlutusDAO 类似的模型。

增加 LP 奖励

在 Magpie 上锁仓 WOM 为 veWOM 后，用户可以在其流动池中增加 token，通过提供流动性来享受更高的奖励。

提供有效的投票权

在 Magpie 上，WOM 持有者可以锁定他们的 token 参与治理，赚取被动收入，同时通过$MGP 的所有权保持对投票权的控制，但同时也会放弃在 Wombat 上的收益。

控制 WOM 流通供应

所有在 Magpie 上的 WOM 都会被自动永久锁定，通过控制 WOM 的流通供应来提供最大化的收益。

如何玩转 Magpie

对于存入者来说

首先，选择你可用的网络（Arbitrum 或 BNB Chain），选择 Liquidity Pools，将你的可用资产存入，本文以 USDT 为例；

存入池子后，可以在界面看见自己的 LP，存入成功后你的账户就会开始收到 WOM 或 MGP 奖励；

接下来可以 1: 1 锁仓你的 MGP 为 vIMGP，通过投票和获取贿赂来最大化你的资产价值。当然此过程中你随时可以将资产提取出来。

对于 WOM 质押者来说

对于 WOM 持有者来说，需要选择 Magpie Pools，目前共有三种选项，点击将持有的 WOM convert 为 mWOM。

可以通过 Uniswap 购买 WOM，然后通过 Magpie 将 WOM 转换并进行质押，目前来看该 Magpie Pools 的 APR 还是很不错的。

有两点需要注意，一是 WOM 转换为 mWOM 时的比率是超过 1: 1 的，也就是说，你可以获得超过 WOM 总数的 mWOM；二是，WOM 转换为 mWOM 是不可逆的，你可以质押和取消质押 mWOM，但不能将其转换回 WOM，二级市场中存在以不同的市场价格将 mWOM 交换为 WOM。

对于 MGP 持有者来说

MGP 持有者需要通过 Magpie 锁仓持有的 MGP 并以 1: 1 比率获得 vlMGP，持有 vlMGP 的优势是：赚取收益分成；赚取 MGP 奖励；最大化 Wombat 的投票权；Magpie 的治理效益；收到别人没收的奖励；行贿所得。MGP 可以通过 Trader Joe/PancakeSwap 购买。

MGP 一旦进入锁定时就会处于无限锁定状态，用户点击「开始解锁」后需要 60 天的解锁期，也称冷却期，强制解锁需要支付一定的费用，而 MGP 在冷却期无法参与投票治理。

领取收益和投票

Magpie 的收益领取，在其界面的 Claim 和 Bribe 上，你可以看到自己能领到的收益份额，通过选取相应的部分进行 Claim 就可以完成领取。

而其 Bribe 界面会显示的收到的 Bribe 和投票状态，比如何时可以参与 Wombat 的 cast votes 投票治理。

与 Pendle 合作推出的 Penpie

除了与 Wombat 合作之外，Magpie 还为 Pendle 打造了 Penpie，Penpie 是 Magpie 与 Pendle 合作推出的一款产品，目前支持以太坊主网和 Arbitrum。

同样，Penpie 的玩法与 Magpie 类似但更简单，用户通过 Penpie 将 PENDLE 转换为 mPENDLE 用以赚取 PNP 收益，PNP 是 Penpie 即将发行的治理 token。现 Penpie 总存款为 241 万个 PENDLE。

而在 Penpie 上存款的用户会获得一些利益：

-将有资格分到 PNP 供应总量的 2% 。PNP 根据积分分给用户，积分评判如下：

存入 0-1000 PENDLE 并 convert，每个 PENDLE 获得 1 分；

存入 1000-5000 PENDLE 并 convert，每个 PENDLE 获 1.1 分；

存入 5000-20000 PENDLE 并 convert，每个 PENDLE 获 1.3 分；

存入 20000-100000 PENDLE 并 convert，每个 PENDLE 获 1.6 分；

存入 100000 以上 PENDLE 并 convert，每个 PENDLE 获 2 分。

-当 Penpie 全面上线时将获得 mPENDLE 空投，mPENDLE 允许在以太坊和 Arbitrum 间垮链。

注意，PENDLE 到 mPENDLE 依旧是不可逆的。且目前该界面只有质押一个选项。

在未来，Penpie 将允许 PENDLE 持有者获得更多的 PENDLE 奖励作为被动收入，为 Pendle Finance 用户提供具有成本效益的投票权，并使流动性提供者无需将 PENDLE 锁定为 vePENDLE 即可获得更高的 APR%。

通过 Penpie 为 PENDLE 持有者转换为 mPENDLE 来获得丰厚奖励的机会。用户可以在 Penpie 上抵押 mPENDLE，并获得 PENDLE 收益。当用户将 PENDLE 转换为 mPENDLE 时，Penpie 会自动在 Pendle Finance 中将转换后的 PENDLE 锁定为 vePENDLE。该机制使 PENDLE 持有者能够通过 mPENDLE 获得更大的回报，同时还授予 Penpie 治理权和更高的 APR 作为 Pendle Finance 的 LP。

Penpie 通过其持有的 vePENDLE 积累的投票权在 vlPNP 持有者之间共享，使他们能够以具有成本效益的方式参与 Pendle Finance 的治理，可以锁定为 vlPNP 来获取其收益。

投票锁定的 PNP 持有者可以从平台中获得收入分成，并有效地影响 Pendle Finance 的治理，同时最大化他们的回报。与 viGMP 类似，用户必须等待 60 天，vlPNP 才能完全解锁为 PNP。

Penpie IDO

近期 Penpie 正在进行其 token PNP 的 IDO。Penpie IDO 的 FDV 为 300 万美元，上限为 60 万美元（占总供应量的 20% ），使早期支持者能够以极具优势的价格购买 PNP。

目前 Penpie IDO 推出了 PNP 优先访问计划以供用户获得优先购买资格，包括：

-mPENDLE 持有者— 30% 

-Penpie 流动性提供者— 30% 

-vlMGP 持有人— 25% 

Galxe launch 活动（幸运抽奖）— 5% 

而 IDO 筹集的资金将用于：

-初始 PNP/ETH 流动性的 40% ：用于确保 PNP 的稳定和强大的流动性池；

-30% 用于开发和运营：用于改进 Penpie，优化其功能，并保持平稳、不间断的运营；

-20% 用于市场营销和业务发展：确保 Penpie 的持续发展，提高我们其知名度、吸引新用户并发展合作关系；

-10% 用于审计：确保平台保持安全、可靠和值得信赖。

结语

虽然 Magpie 推出的高收益机制有些复杂，但无疑是为 veToken 机制开创了一类新模式，享有治理权的同时也能带来高收益，目前看来这种将用户收益最大化和治理权 token 化的方式能给 DeFi 带来一些新东西，也期待 Magpie 可以和更多项目合作带给用户更直观的收益。
