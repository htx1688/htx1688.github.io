接下来要进入 web3 的千万和上亿的用户，他们会从哪里来？NFT 的大规模采用会因用户热爱 IP 而来：胖企鹅 CEO 分享 PFP 的商业模式设计。

上星期我们聊了「PFP 之后 NFT 市场的下一个新叙事」，分享了 Open Loyalty  相关的案例与思考。

但不得不承认，PFP/IP 仍然是 NFT 目前的最大品类， Web2 的 IP 也在尝试进入 Web3，比如我们上一期提到的宝可梦和芝麻街，其中一个重要原因就是基于 Web3 社区可以更好打造和发展 IP.

近一年，同样是 PFP 项目，在 CloneX, Doodles, Moonbirds 不断缩水的同时，Pudgy Penguins 却一直在逆市上扬，它做对了什么？

带着这些疑问，rubywang  花了几十个小时研究了其 CEO Luca Netz 的采访，并梳理了他对打造 Web3 IP 与商业模式的思考，供 buidler 们参考，欢迎阅读

TL; DR

  1. NFT 项目想要赢得成功，必须构建一个可持续的，可增长的收入来源

  2. 胖企鹅的 playbook：ToB 合作先行，ToC 提升品牌影响力，最终提升直接销售占比。

  3. 运作上的出彩点：建立「社交媒体→授权变现」通路，并 NFT 持有者分享品牌的成功和商业价值

  4. 坚信 NFT 会获得大规模采用是来自于用户对 IP 的热爱

一、背景介绍

项目介绍

Luca Netz 作为 pudgy penguins（后文统一称 “胖企鹅”）的 CEO，并非胖企鹅的创始人。22 年 4 月，他以 250 万美金（当时 750ETH）价格买下了这个 PFP 项目。一年以来，胖企鹅的交易量和地板价都有了稳定的增长。尤其最近 1 个月，胖企鹅的地板价一度 flip 了多个蓝筹项目，占据 Blur 首页推荐。

Luca 带领团队究竟做对了什么？有那些独特的思考？

出于对其项目运营的好奇，我认真研究了他近期的访谈，梳理他对打造 IP 商业模式的思考，同时也谈谈我的看法。

声明：本文作者未曾持有 Pudgy Penguins/Lil Penguins 系列 NFT

近一年来胖企鹅的地板价数据表现

关于 Luca Netz

虽然非常年轻，24 岁的 Luca 已经是个连续创业者。

成长于穷困的家庭，儿时的他和妈妈有近 10 年时间无家可归，经常需要跨国搬家并借住在朋友们的客厅。他不得已在 16 岁高中辍学后，开始在洛杉矶做生意。先是在仓库经营派对赚钱，然后加入了一家销售门铃的创业公司 Ring Doorbell，从那儿开始找到了自己 擅长的领域：通过社交媒体与网红（KOLs）销售商品。

Luca 成了典型的 drop shipper，即通过与网红们合作，处理背后的商品供应链，并直接从品牌销售到消费者的电商卖家。他曾打造过不少爆款，包括 gel blaster 的玩具枪，不仅在线取得千万美金的销售成绩还打入了全美沃尔玛（感兴趣的可以看 YouTube 上的测评视频）。

Gel Blaster 网红爆款在美国亚马逊的销售图

二、商业模式

Yuga Labs 的特殊性：连续空投在熊市行不通

Luca 认为，Yuga Labs 的成功，在牛市推出是一个不可忽视的因素。其他项目是否可以一波波地新发，给持有者持续空投？如果项目的地板价没到 20ETH，那子系列的价格可能只在 0.5E 左右。不仅没多大意义，而且还会稀释整个项目的供给。 并且在熊市，这种增发新系列的方式也行不通。

很多 web3 的持有者在今天想要的主要是交易价值（transactional utility），他们希望价格上涨，free money，拿到空投的那种喜悦。

我当然也想给我的社区这些；但是同时我们也得清醒地认识到这是凭空造物（create things out of thin air）或者我们所说的 ‘庞氏’。

NFT 项目需要有长期的可持续收入

想要赢得成功，就需要构建一个可持续的，可增长的收入来源。之前 NFT_GOD 也分享过，公司需要建立在超越 NFT 的事物上。

一方面，Luca 清醒的认识到，靠版税和新发都很难在熊市中 work。

以太坊上的版税是我无法控制的变量——上个月胖企鹅可能靠版税赚了 50 万美元，下个月赚 5 万美元。但这不是我的错，而是因为某种宏观经济条件对流动性的影响。

另一方面，PFP 的终局是 IP。 根据 2020 年全球最赚钱的 25 个 IP 的收入来源，实际上 IP 的主要收入方式为衍生品销售、游戏、 影视票房三大类，其中衍生品与游戏可以说是 IP 变现的两大引擎。

可以看到，宝可梦至 1996 年以来的 24 年收入近 1000 亿美元，说明 IP 的天花板是非常高的

The World’s 25 Most Successful Media Franchises

NFT 头部项目里，Yuga Labs 主要选择了游戏方向，同时收购的 10KTF 也会与品牌实体结合以商品做变现；Azuki 在去年推出 PBT 协议和 Ambush 的联名，也在探索实物商品与数字体验结合变现的道路。

Luca 选择了 IP 衍生品，他从接管胖企鹅开始，就着手操作 IP 授权和商品企划相关事宜，同时与用户创造情感连接。

目前的阶段，数字收藏品（digital collectibles）并不是大众用户每天参与并且有情感互动的。我们有一天会破圈到日常大众，但是目前显然还没有。从营销和消费者行为角度，品牌创造尽可能多的高频用户接触点非常重要。

我如何可以尽可能多地触达你，让你留下记忆，创造情感连接？比如你通过网上的动图，或者朋友送给你的企鹅玩具，或者零售店里的企鹅形象了解到我们的品牌，那你就更有可能成为我们的一员。

三、胖企鹅的 Playbook

所以胖企鹅打算怎么做？我总结大致的变现思路：

ToB 合作授权先行，ToC 提升品牌影响力，最终提升直接销售占比。

先利用最初的全网社交媒体影响力和曝光，用 ToB 的授权打开局面，在线下零售店与合作网站上销售胖企鹅 IP 的商品。 好处在于可以先拿到部分收入，也降低最初供应链，渠道铺设等投入成本，同时扩大了品牌商品的触点。

在 ToC 方面，团队聚焦提升品牌影响力，提升全网对 IP 的好感度。 当长期靠自有渠道与社区影响力做 DTC 销售的时候，能够有效降低降低营销成本，提升转化。

具体在运作上，胖企鹅有些不错的巧思，总结以下三个出彩点：

  1. 善于利用社交媒体和优秀的 meme 传播能力

  2. 短平快的内容先行，利用影响力做授权变现

  3. 品牌的成功和商业价值与 NFT 持有者分享

社交媒体与 Meme 传播

胖企鹅在社交媒体上的表现是可圈可点的。除了 Twitter 的 10 万粉丝，值得注意的是团队花了不少精力在 web2 的平台上。

Instagram 上有 36 万粉丝，以每日更新小企鹅可爱的内容获得了不少评论，有些小动画甚至有 40 万的赞。 像这样的互动，很容易与广泛年轻用户产生联结。

胖企鹅 Instagram 官方账号

另外在 Giphy，一个支持所有社交媒体分享 Gif 的平台，胖企鹅应该是唯一官方建立了品牌账号，并获得 2K 的 gif 动图上传与 20 亿次跨平台浏览量的 NFT 项目。我翻了下几乎没有太多 web3 项目官方在 Giphy 上，除了像无聊猿，BEANZ 等有不少是社区上传的。Nike，Starbucks 品牌也有自己的 Giphy 账号，浏览量目前和企鹅在一个数量级上。

短平快的内容先行，利用影响力做授权变现

Luca 说他很认同推特上一个帖子的评价，“胖企鹅可以看成是 NFT 届的 doge”。他思考这种相似之处是 meme 导向，而他要做的是让大家可以到处传播这些 meme。 比如选择 Giphy，好处在于它已经集成了很多大的 web2 平台尤其是 Twitter，梗图无处不在，用户 UGC 的力量可以方便地将 IP 渗透到互联网各个角落。

社交媒体的作用更像是转化漏斗最上一层的背景信息，让大家熟悉这个品牌并种草。比如在美国，使用 Giphy 的用户中，年龄在 18-29 岁之间的占比最高，达到了 37%。同时如果他是一个 web3 圈内人，现在收到妈妈或者女朋友给他发带有企鹅形象的 gif，他们就知道了（项目方在做事！）

Luca 认为，Gif 梗图这招尤其对这些人的胃口，增加了他们对项目的好感度，并可能将其转化成为企鹅的拥护者。

当然最终，社交媒体主要是服务于品牌传播而非直接变现渠道。目前这些曝光数据，年轻人活跃的互动行为是团队作为和授权商与内容制作工作室谈判的筹码。这类似国内潮玩 IP 的启动策略，靠一些可爱的形象和短平快的内容让大家先对这个形象和部分角色有好感。而不是一上来就花费巨资做长线内容，再依靠有故事的动画剧集内容延长角色的生命力做 IP 的长周期变现。

据悉，胖企鹅最近已经在推进商标注册，涉及北美的糖果，零食，漫画书，在线社交媒体等多个垂直品类，为品牌的长期发展做准备。

品牌的成功和商业价值与 NFT 持有者分享

读到这里，你可能会问：这都是 web2 传统的 IP 变现方式，那和 web3 以及 NFT 社区有什么关系呢？

Luca 认为 web3 项目应该要利用好 NFT 不能被忽视的一个独特优势——抓住 5000-10000 人的注意力，并且让他们成为你品牌的支持者。  

在最初研究各个 NFT 头部项目时，他发现 BAYC 是做得最好的，因为他们让最初的持有者富有，而这会构建社区的信任基础，并且使得社区成员成为项目的拥护者，自发推动项目的成长。5000 人不是一个小数目，可能你建设一个上亿的生意（如果有强大的 5000 个传播者）也够了。人们会支持你，因为这与持有 NFT 利益相关。

因此回到胖企鹅的业务，无论 IP 成长为什么形态：一个游戏，TV show 还是实体玩具，能够把业务累积的价值回馈给 NFT，保持核心社区的强大是非常重要的。

在具体的商务安排上，胖企鹅目前探索了两种授权模式：

“

方案一：少部分持有人的形象授权实物生产，商业收入分成

Luca 称胖企鹅是第一个从社区征集形象做实物授权的项目方，之前 Doodles 和 BAYC 也做过 figure 和潮玩，但是完全用了项目方自持形象。 他比较理想地希望形象用社区持有者的，然后收入与他们分配。但是当真的实操了才发现，原来之前没有项目这么做的主要原因是法务上真的很棘手。比如上，怎么解决持有人授权 NFT 后又交易的问题，永久性的授权（perpetuity）是很难的，因为需要锁定这个 NFT 的流动性。

后来团队换了种方式试图解决这个问题。使用 holder 的 NFT 资产以预付款+非排他性的合同条款来解决：

假设我们打造一个玩具业务线，一年能够销售 10 亿美金，我们可以用授权协议安排使得价值回到持有人手里。用预付的方式先签下一些企鹅形象，然后每款玩具的销售收入与该授权人分成，每年做结算。

在这个模式里，胖企鹅团队负责媒体宣传，商品设计和渠道合作等。唯一的问题在于这种操作比较难规模化，而且目前也没有公开信息可以查到持有人的收益与预付款信息（期待未来有链上方案）。

胖企鹅分别在去年的 9 月与今年 2 月宣布与玩具销售与授权的公司 PMI 和 Retail Monster 达成合作，将推出不同系列的玩具商品，其中一些包含数字体验。目前这些商品都还没有问世，因此价格与销量也暂时没有数据。

方案二：大部分形象授权社交媒体曝光，共同打造品牌知名度

以方案一的授权方式无法照顾到所有 8000+个 NFT 企鹅形象，至少 5 年内不可能。因此 Luca 的思考是大部分企鹅形象主要授权团队来做社交媒体曝光。 Instagram 官号每天发一个不同的企鹅的图，Giphy 上，每周发布 100 个 Gif，每周在社交媒体上就曝光 107 个企鹅，每年就是 6000+个。

不同的曝光数据是否会影响到 NFT 的交易行为？或者，最后是不是可以给每个企鹅曝光的分成呢？团队希望靠规模化运作来探索这样的形式。

四、胖企鹅的长期目标 vs 行业观察

胖企鹅的长期目标

Luca 对胖企鹅的长期目标：

创建世界上第一家伟大的 web3 IP 公司；成为第一个全球公认的品牌和 IP 之一。

他认为 NFT 成功的必要条件是有 IP 通过起源于 NFT 而发家成功，这点我深表同意：

如果不是我，也需要有人来赢得这场战斗，并征服这个垂直领域，在我看来，我们的圈子才会真正受到重视。希望到今年年底我们毫无疑问地成为 Top3 的项目。

如何看待大的制片工作室

Luca 认为，真正会塑造 web3 和 NFT 的这些人是现在 20 多岁的年轻人。

像我这样的年轻一代，成长在移动互联网时代，我们在社交媒体增长与建立用户联结上会比那些资深内容行业的人有优势。我们的核心能力并不是 storytelling 和大的世界观的架构，这是大的制片工作室擅长的地方。

我的思考是，我们做好我们擅长的，让我们创造一个巨大的差距。然后他们会来找我们做他们最擅长的事情， 我们可以将彼此擅长的事情融合，去建立合作伙伴关系。

就像 BAYC 的创始人都没有开发过游戏，他们玩游戏但并不了解制作游戏；但他们建立了如此大的品牌和如此多的影响力，以至于游戏开发商来到他们面前说，嘿，我们会为你打造你的游戏，BAYC 也可以拿到很好的条件。

同时，他也谈到，关于故事和角色开发，已经与一些大的工作室聊过。但是目前的交易条款还不太平衡，显然这些工作室没有给出满意的报价。他不希望长期在非常重要的一个营收垂直品类（影视制作）上为了现在能够达成合作而以很低的金额让步。

Luca 希望向华纳和迪士尼以及所有这些主要的电影公司展示，web3 IP 可以从他们那里夺取市场份额：

如果我们可以从他们那里夺取市场份额，他们将在这个领域派遣他们的精兵强将，因为现在只是试水，他们派的人和投入的资源显然不是最好最核心的。我期待他们投入最好的 IP 和制作能力到 web3 里来。

非常有战斗力的一个年轻 CEO

五、NFT 的下一个千万级用户从哪里来

所以，接下来要进入 web3 的千万和上亿的用户，他们会从哪里来？

Luca 不相信 web3 通过赞助某个比赛，logo 打到广告大屏幕获得大规模的应用；

认为 NFT 会获得大规模采用是来自于用户对 IP 的热爱。

因为如果这些 NFT 不是角色，那只是区块链上的生成图片，只有很少量的 NFT 能够成功。用户将通过想要支持一个他们相信其愿景的品牌并拥有与该品牌价值一致的激励机制来到 web3；

他们真的会成为品牌的拥护是因为他们喜欢一个角色，和喜欢那个角色背后所传达的信息。我坚信我们要做的是为之后「星球大战」和「哈利波特」等 IP 打开闸门。
