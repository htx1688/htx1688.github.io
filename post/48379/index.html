数据源： Use Retention; Walken; STEPN-Solana & BSC

Move-to-Earn，一种通过地理追踪跟踪用户的运动并对他们的身体活动进行奖励的协议，在去年五月获得大众的关注以及追捧，在搜索兴趣方面超过了GameFi。然而，这一类型的游戏，目前只有Solana上的两个项目：Walken和STEPN。这两款游戏有类似的概念与玩法，但在通证、游戏性和设计上有不同的做法。

GameFi在牛市中有大量的模仿项目。当一个项目做得很好时，就会有几十个复制品冒出来，希望能复制它的成功。因此，尽管M2E只有两个代表性项目，但它们相对于其他链游项目更具备原创性。该类型项目，某种程度上，也是区块链技术的创新和真正的有趣应用的出现。

通过调查Walken和STEPN内的链上活动，我们可以开始分析在该流派中什么是有效的，以及我们可以期待什么样的功能在未来出现并取得成功。

时机对链游项目来说很重要

STEPN在牛市高峰前推出，随着 ETH 4月份价格的反弹，它的市值出现了暴涨。

所有的加密货币项目最终都与宏观经济条件联系在一起。随着2022年春季加密货币的崩溃，STEPN也是无法逃避面临着价格的暴跌。由于其在中国的用户的GPS服务被拒绝，更是加剧了项目的崩盘。

STEPN Token Market Cap

虽然从最初的那几个月开始，游戏的功能和用户界面仍有在调整，但在项目的市值、活动和代币价格方面还没有恢复。

项目开发者仍然活跃，并公开露面讨论推出的问题，但价格的走向，依旧让它看起来像一个已经被大家抛弃的项目。

StepN Token Price（GST + GMT）

在区块链行业，信任就是一切，而且无一例外地与发展势头有关，而发展势头在消亡后很难再恢复。

Walken在STEPN之后七个月推出。虽然它的设计和玩法明显不同，但它也是Solana上的一个M2E游戏。随着市场开始复苏，Walken出现了用户和增长的高潮。

与STEPN相比，Walken的增长并不只是STEPN一年前的巨大数字造成的错觉——Walken乘着GameFi和Solana生态系统的乐观浪潮产生了更多的搜索兴趣，而不是停留在恢复STEPN以前价值的叙述中。

这已经转化为其NFT交易量和代币价格的一个更积极的趋势。

虽然这两个项目都是时尚的、移动优先以及将运动游戏化的应用程序，但在GameFi中叙事很重要，而社区是善变的。即使死亡螺旋在很大程度上是不可避免的，负面的叙述已被证明会阻碍项目的恢复。

数据表明，GameFi项目的代币价格和用户活动与市场不是简单的1比1的关系。时机很重要，不要让你的大部分用户抓到一把落水的刀。

M2E项目必须让用户活跃起来（恕我直言）

STEPN和Walken在第一次下载时都让新用户感到紧张。

在作者看来，STEPN的上机操作相对更直观，因为跑鞋和跑步之间的联系比Walken的卡通猫和走路之间的联系更直接。

这并不是说STEPN对一个加密货币新手来说很容易。NFT的质量、稀有性和矿工都是陌生的——开发商似乎忘记了这一点。例如，为什么一只鞋会有一个 “薄荷”？他们闻起来很臭吗？而在这一切之上，用户必须购买一个NFT才能开始游戏——这对那些想把日常慢跑游戏化而不是成为NFT交易者的人来说是一个很大的阻碍点。

Walken有一个相当复杂的游戏玩法，有许多元素——奖赏、比赛、服装和各种类型的战利品箱，此外还有双代币（像STEPN和大多数区块链游戏）。对于不熟悉区块链游戏的人来说，要跳进去的东西很多。然而，Walken的留存率比STEPN高得多。

Use Retention

有一个显著的区别可以作为解释。

WALKEN的新用户在第一次打开应用时自动获得CATlete。然后游戏开始通过用户的手机追踪步数，每日自动得到奖励。用户只需点击一个按钮即可获得GEM，并按照自己的节奏进一步进入游戏。虽然Walken的主屏幕可能看起来很吓人，但这种方法比要求用户购买NFT，然后记得手动将该应用程序纳入他们的日常运行中的阻碍要少得多。

这两种方法都有优点和缺点，但目前来看，哪种方法产生的DAU更多是显而易见的。

GameFi分析的难点

许多加密货币作家和分析师的记忆短暂得像金鱼一样，因为短期的代币价格上涨而每隔8秒就从一个项目带入另一个项目。

这也太容易陷入简化主义思维。在这种情况下，它将把上述游戏的价格和活动减少到一个或两个原因。现在试图了解GameFi的每个人都像在黑暗中摸索。没有人知道接下来会发生什么，或者哪个项目会爆发或崩溃。事后进行的分析总是很容易，而且没有风险。

这篇文章只是为了说明为什么时机和新用户入职对M2E项目很重要。此外，像Footprint Analytics这样的区块链分析工具可以发现它们与游戏的关键指标之间的关系。 

Walken和STEPN对M2E采取了不同的方法，在实现他们的愿景方面面临着根本不同的挑战。然而，这两个项目团队都是加密货币领域最强大的团队，无论在顺境还是逆境中都不断与他们的社区进行沟通。随着不断的更新和改进，他们已经准备好在可预见的未来确定M2E的方向。
