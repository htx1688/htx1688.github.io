无论牛熊，无论新旧，项目在结合新领域或者新公链后总会催生出新的叙事，获得更多关注，DeFi 研究员 Louis Cooper 近期介绍了一些 AI+crypto、新上线 L 2、和创新叙事的项目，也是他认为被低估的几个项目，BlockBeats 将其整理编译如下：

X7 Finance

X7 Finance 的应用是 Xchange，是一个提供杠杆化初始流动性的抗审查 Dex，为用户提供 10-1000 倍的杠杆，进行 swap、借贷等，可以向创业者提供大量资本，拥有复杂的多 token 生态，相比于 Uniswap，Xchange 完全依靠链上数据进行 swap。别信任何人，信代

码，DeFi 永存。

BeepERC20

BeepERC20 是一个 AI（其 AI：Jarvis）操作的全自动发布和管理 token 项目，AI 真的可以代替人类接管 token 的发布吗？BeepERC20 可以根据指令以 24 小时全天候自动化的方式监测和解决 token Taxes、回购和销毁问题问题，人类管理总是会出现这样那样的问题。

Verasity

Verasity 是一个广告优化协议，利用观看证明（Proof of View）来提高效率，同时奖励 token 持有者，PoV 在美中均获得专利，引用该机制是为了消耗广告欺诈，旨在区分广告观看对象是人类还是机器人，PoV 为拥有真实人类观众的创造者带来收益，而不是奖励虚假观看（非人类观看广告）。观看广告是不可避免的，40% 以上的广告费都损失在广告欺诈上，一个更有效的系统会让所有人受益。

yieldification

Arbitrum / ETH 上的 perp Dex，提供 NFT 作为收据，yieldification 是我去年年底热衷的一项协议，现已经进入整合阶段，其 NFT 市场刚刚上线，很想看看他们在 2023 年的表现。

Sperax

Sperax 是一个开源的金融软件，以 SperaxUSD（USDs）为核心，Sperax 通过提供这种完全抵押（1:1 支持）的稳定币（而不是像 UST 那样的算稳），为其持有者产生有机和可持续的自动收益，新 Demeter 协议允许项目创建无代码激励的 Uniswap v3 farm，其新协议可能会促进 Arbitrum 的采用。

Syrup.Finance

Syrup 是一个完全去中心化的 spot 和 perp 交易平台，允许用户以高达 100 倍的杠杆（可根据可用流动性进行扩展）做多做空，而无需注册或放弃其资产的托管，Syrup 的终极目标是成为一个由高级交易工具组成的一站式平台，适用于任何类型的交易者，无论是新手还是老手，他们在熊市中推出了令人印象深刻的产品。

MASQ Web3 Browser

MASQ Web3 Browser 是一个基于 Web 3 的浏览器，其亮点在于：

-内置 VPN 和 earn 生态系统，并刚刚发布了他们的 Dapp store；

-与 Starlink 合作；

-源于对 TradFi 的不信任而充分关注隐私。

Gamma

Gamma 是一个专注于流动性管理和做市策略的协议，近期上线了 Arbitrum，其优势是：

-将流动性转化为简单的被动收益策略；

-有像 Frax 这样的深度合作伙伴，管理大量不同协议的流动性；

-利用 LSD 实现出色的 APR。

Smolverse

Smolverse 是 Treasure 生态系统中构建的第一个概念验证 NFT 项目，并与 Treasure 的 token MAGIC 深度融合，同时也建立在 Arbitrum 上，可能意味着在下一轮牛市中 Smol 会成为 Arbitrum 蓝筹项目。

Vesta

Vesta 是 Arbitrum 上的具有超额抵押铸造稳定币的借贷协议，零息借贷，有着 2200 万且不断增长的 TVL 和无数新金库，而超额抵押使其成为更安全选择。

DEXTools

DEXT 是我用来跟踪长期和短期交易的首选图表聚合器和工具，增加了一些高级版本，鼓励人们使用 DEXT，且开发者在不断更新完善，成为更完美的工具只是时间问题。

PlutusDAO

Plutus 是一个 Arbitrum 原生治理聚合器，旨在最大化用户流动性和收益，同时聚合 PLS token 的治理，Plutus 专注于围绕 Arbitrum 上治理 token 创建优秀的产品，高 APR，高排放 ATM，涵盖所有主要协议。

Fetch.ai

Fetch.ai 是一个 P2P AI 驱动协议，作为物联网的操作系统，目标是创建一个互联的数字世界，在这个世界中，AI 代理实时交互，自动做出决策并执行协议，而无需人工监督，在 AI 和 crypto 结合的领域，Fetch.ai 是最大最有前途的项目，有着清晰的收入路线图，本质上是 SaaS，所有支付媒介都是其 token FET。

「到 2025 年，全球人 AI 软件市场的收入预计将达 1260 亿美元」，而预计 Fetch.ai 市值将超过 10 亿美元。
