日本是亚洲，乃至全球最成熟的IP大国之一，它不仅有着大量的优质IP，同时也是世界上IP商业化做的最好的国家之一。其许多IP在运营方面已经形成了较为成熟的商业模式，但即便如此，大量的传统IP仍旧无法发挥其全部的经济潜力。且在中心化模式的统筹之下，IP的变现能力总显得低效与局限。但在Web3时代这个问题或将得到解决。IP和Web3的结合拥有公认的巨大潜力，这种全新的模式将持续提升IP在实体商业以及虚拟空间的商业价值。

NFT是Web3重要的基础设施，可用作虚拟资产的确权凭证，具有不可篡改、独一无二的特性。这些特性与IP的性质不谋而合，因此，市场也看到了借助NFT重新塑造IP的机会。同时，日本也是亚洲加密市场最活跃的国家之一，其拥有百万量级的加密资产活跃交易账户。而在文化属性和宏观发展方略方面，日本近年来也有意引导加密市场的发展。因此，对于加密市场来说，日本拥有相当肥沃的土壤。

因此，AllRecode今天将对布局于日本NFT交易市场的UneMeta的创始人进行一场深度专访。

Q1: 您好，请介绍下您在加密领域（包括NFT）的经历，为什么想要做一个NFT市场以及您最喜欢什么类型的NFT社区？

我本人从13年开始接触加密行业，17-18年的时候开始试水token fund项目。但是我本职一直都是做科技类私募股权投资。

NFT和DeFi是当下Web3落地程度较高的两个应用，它们其中包含让Web3破圈的元素。NFT是元宇宙里的基础设施，虚拟世界中不存在实体物品的交换，NFT在其中就可以起到产权界定的作用。UneMeta旨在架起现实艺术IP和NFT艺术社区之间的桥梁，促进Web3社区的包容性和多样性。未来，我们将以一流的产品体验和平台属性，打造一个包容交汇的元宇宙社区。

Q2: 能和我们详细介绍下UneMeta是如何和漫画艺术家进行合作，帮助他们孵化自身IP的吗？另外，在未来这些合作艺术家会持续给作品赋能吗？

用漫画IP产业最发达的日本来举例子，我们也是优先选择经验最为丰富的日本合作方合作。日本漫画的 NFT 作品和市场有非常大的潜力。日本动画已成为全球文化的连结点，得益于拥有丰富且优质的艺术家、动漫游戏 IP 和 成熟的用户市场。而传统的内容进入web3是需要一定的“翻译”工作的。漫画艺术家是内容类IP，内容类IP优势在于有较为忠实的群体，并且可以转化的web3产品有的放矢。我们会结合内容本身，打造适合IP的故事线和NFT设计。

因为很多知名的日本漫画已经开发过其他形态的受欢迎的内容，最好借鉴的就是游戏。游戏也与NFT本身有较强的的关联性,比如现在爆火的Gamefi。初期阶段我们会用作者和用户都更加熟悉的展示形式来合作，比如日本市场喜欢的抽卡，还有收集类的。也会推进粉丝权益类的内容，比如周边和可销售的二创。帮助他们为自己的IP持续注入新鲜活力。

Q3:近期Nike、Adidas以及潮流文化圈的知名IP纷纷发行NFT，可以预见未来还会有更多知名公司和顶流IP加入进来。这些中心化商业模式的公司进入加密行业，无疑会影响未来竞争格局，您如何看待传统IP在NFT市场的未来竞争格局？

顶流IP跨入加密行业是必然的趋势，因为IP本身具有强个性和文化属性，随着行业的发展，IP传播也势必要接纳最新的趋势。早期的Web3原生内容，比如BAYC、Azuki的发展具有一定的偶然性，而目前NFT项目对内容的要求已经越来越高。纯原生项目的崛起会越来越困难。优质的IP如果能够充分理解Web3的打法，将会在NFT市场上获得强劲的影响力，甚至可以轻松破圈。

Q4:主流人群迅速进入NFT领域，未来的NFT市场很难由OpenSea一家独大，如何看待NFT市场细分领域的机会？（游戏NFT市场/艺术品NFT市场/音乐NFT平台）

作为当今世界上最大的NFT市场，OpenSea已经成为NFT市场的风向标。但它过于中心化，也缺乏社区利益共享系统，Web3用户无法享受平台增长带来的利益。而未来的NFT市场格局，必然不会再是OS一家独大。经过了初期的野蛮生长，消费者和收藏者们的品味和要求，必然会随着市场的不断成熟而提高，Opensea这种类似于ebay的模式就将难以为继。拥有优质IP和服务的新生平台将会良币驱逐劣币。

Q5:您怎么看待NFT市场的产品护城河？UneMeta未来准备在哪方面扩张自身的壁垒？

UneMeta比较看重提升用户体验和增加NFT的应用场景两个方面。NFT市场本质上是流量聚集。因此，如何吸引更多的用户，实现转化和复购是需要思考的问题。但最终的壁垒还是品牌本身，Unemeta希望通过给NFT市场带来更高质量的NFT项目和服务来建立自己的品牌影响力。

UneMeta会将用户体验大大提升，与现有产品区别开。具体分为两方面，第一是用户可以获得的真切福利，比如我们的积分机制可以将价值回馈给社区；第二是资产安全方面，虽然Web3是去中心化的用户体系，但UneMeta仍旧希望可以通过产品设计对用户的资产安全提供充分的保障。资产安全也可吸引更多优质的项目方与我们合作。

Q6:日本IP的运营已经有较为成熟的商业模式，但是我们可以看到许多传统IP进军NFT市场的发展并不是特别理想，平台未来孵化商业IP时如何在两者之间做衔接？

成熟的IP产业链其实从一定程度上也是一种背书。而关于IP产业价值回归社区日本已经有过尝试，即二创经济。日本的粉丝二创经济非常发达，如果通过DAO管理社群，就可以更好的管理二创经济，扩大收入。我们曾与具有上百亿市值的商业IP进行深入的探讨，我们认为真正将Web3的IP商业化还需要专业的引导，如Yuga Labs虽然将无聊猿授予每一个拥有者，但是如果拥有者自己没有成熟的商业化经验，其实很难把优质IP变现。甚至连Yuga labs自己的IP变现也依赖其专业的游戏策划设计能力，即最近推出的Otherside。

因此，Web3社区的商业化如果想实现长期运作，也需要社区存在专业人才。UneMeta社区拥有吸引到这样的专业人才，从而形成真正的良性商业化。我相信这样web3社区也需要这样的新生案例。

Q7:主流群体认为NFT处于投机泡沫时期，您觉得NFT现在处于什么阶段，未来什么走向？NFT如果要持续扩张，需要什么具备什么条件？或者什么类型的NFT会增加采用率？

目前应该属于泡沫初期。NFT的扩张首先需要完善基础设施，比如更低的gas费，更多更容易理解和上手使用的应用场景等。简单概括就是降低认知门槛，增加应用场景。

我个人比较看好收藏类和功能类的NFT，从这次熊市来看收藏类的NFT，没有出现很大程度的贬值，功能类的NFT也因为应用场景的存在，也没有出现巨大的价格波动。

Q8:团队未来1-3年的战略规划，侧重于哪方面的扩张？

NFT 在大多数 NFT 交易平台上大致分为九类：艺术品、收藏品、域名、音乐、照片、体育、交易卡、实用工具和虚拟世界。这些品类基本涵盖了整个 NFT 市场。但是，如果我们考虑 NFT 是如何呈现的，它们中的大多数都是以图像的形式出现的。这反映了 NFT 交易市场的同质化。

目前，那些伟大的杰作在未来的任何时候都可以转向 NFT，这是趋势，用户增长和产品体验值得一试。那些真正的艺术品收藏是其他 NFT 平台，比如 Opensea 做不到的。

Q9:如何向一个完全不了解NFT的主流用户介绍NFT的效用和优势，以及为什么选择UneMeta？

NFT目前是Web3文化属性极具代表性的一个产品，you are what nft you own。是快速融入web3社区和学习的入场券。UneMeta从第一天起就更注重社区，并且会量化突出的社区个人贡献；Unemeta重在运营优质有基础，有商业价值的IP项目，用熟悉的文化标签降低NFT项目认知门槛，用已有案例证明我们发布的nft项目在努力创造实实在在的价值商业价值。

UneMeta的中心会持续放在真正的产品体验上，并在我们强大的投资人背景和创始人团队加持下，打造出NFT领域的新生代明星产品。

寻找下一个亚马逊？

随着众多NFT市场的出现，许多人可能会问，在众多平台中，哪一个有可能成为亚马逊一般的行业标准。传统市场往往表现出赢家通吃的态势，这意味着一旦单一平台达到一定规模，竞争对手就几乎不可能超越它。因此，关注一个不会被淘汰的平台理所应当。然而，与传统市场相比，我们认为没有任何一个NFT平台有可能占据这种主导地位，有两个关键原因。

首先，NFT市场在本质上比传统市场更加开放。NFTs建立在完全公开的区块链上，大多数交易数据是公开的，这限制了平台建立导致垄断的各种数据护城河。

此外，增强型平台的成功表明，差异化对买家和卖家都很有价值。我们已经看到，市场对共存的NFT市场有强烈的需求，每个市场都专注于不同的领域，并提供专门的工具来帮助用户满足需求。例如，某个NFT艺术平台可能专注于铸造功能，而另一个可能专注于建立在艺术NFT之上的游戏体验。

虽然在精简的平台中，提供最低交易成本的市场可能会吸引大多数买家和卖家，但这些平台永远无法提供增强型平台所能提供的定制化服务和行业支持。

因此，如果你是一个创造者，就不要觉得必须等待“NFT领域的亚马逊”出现。评估目前可用的市场平台，看它们与你的产品和业务需求有多大的契合度即可。虽然NFTs仍然是一个新兴行业，但它们已经显示出高度有利可图的潜力。它们最初可能是一个主要由加密货币爱好者和热爱风险、精通技术的艺术家推动的前沿项目，但其现在正迅速进入主流。无论是像NBA这样的大品牌还是新兴的独立艺术家，与正确的平台合作是推动客户参与和确保你在新兴数字经济中地位的关键一步。
