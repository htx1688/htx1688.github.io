原文标题：《社交代币设计过程中的 10 个关键学习点，回答「社交代币悖论」问题 》（10 Key learnings in designing social tokens answering the “Social token paradox”）

撰文：Mengyao Han

编辑：Skypiea

人类历史发展过程中充斥着各种失败的经济尝试。当资本主义真实存在时，人们需要更多的机会。我们的现代生活早已成为商品、人员和信息传输的数字展示，我们迄今为止积累的社会价值在很大程度上依赖于我们的关系和联系。这些人道价值观在我们的数字经济中很难映射，而社交代币正是为了弥补这一缺口。

建立一个社交网络是很困难的。作为 Web3 社交的建设者，设计一个万能的经济模型是一种幻想。我们唯一能做的就是提出正确的问题，保持我们的眼界和思想开放，并在此过程中进行构建和迭代。

什么是「社交代币」？

有些人认为「创作者代币」或「社交代币」指的是同一件事。它可以是指代特定个人 / 影响力者的代币，但它可以远不止于此：社交代币网络有可能代表我们的数字经济。

我们团队正在设计一个相当创新但复杂的创作者经济。一路走来，我们潜入这片深水，并从杰出的同行那里收集了深刻的见解。在这篇博文中，我将通过敲响高级原理的钟声来总结所有的学习成果，同时无需任何数学知识。

社交代币悖论

Gaby Goldberg 定义了社交代币面临的基本困境：

「当代币的价值上涨时，其社区的价值也会上涨。而当代币价格更高时，社区自然会变得更渴望加入。这里的问题——我称之为社交代币悖论——是通过这种心理模型，激励代币化社区维护排他性。总的来说，如果您希望您的社区有价值，您需要社区的代币价格上涨。这是有道理的，当然。但是要使代币价格上涨，您有两个选择：您可以提高代币的价格，或者您可以减少潜在会员可用的代币数量。无论哪种方式，这都会限制对社区的访问并促进排他性，编纂和实施某种程度的稀缺性，这几乎与 Web3 的真正开放互联网的愿景背道而驰。」

社交代币旨在具有社会可扩展性以产生重大影响。然而，社区的成功不能仅仅由代币的价格来代表，因为它很快就会开始与网络的社交可扩展性产生冲突，并将人们过滤掉。

这提醒我们要从头开始重新思考代币的价值。让我们一起探索。

社交代币设计的 10 个关键学习点

在这里，我总结了前面的所有要点：

  1. 将有效的利他主义与经济结构中的自身利益相结合

  2. 工作获得（work-in）和买入获得（Buy-in）应该有一个健康的平衡

  3. 社交代币可以带来额外的网络生产力

  4. 不可转让的声誉信号更强大

  5. 在通货膨胀和用户增长之间保持健康的平衡

  6. 促进长期参与和新需求

  7. 瞄准正值应计以逃避庞氏骗局

  8. 尽早建立多元化社区

  9. 社交网络应以乐趣为先

  10. 关键逻辑要上链

我将一一探索每一个学习点。让我们开始吧！

1. 将有效的利他主义与经济结构中的自身利益相结合

有效的利他主义是一种哲学和社会运动，它主张「用证据和理由弄清楚如何尽可能地造福他人，并在此基础上采取行动」。

回到数字经济，只有当网络创造价值时，才会有可持续的个人利益。因此，经济需要激励和调整参与者，让他们从他们的行为中为整个网络做出贡献，同时忠于他们的自我动机。

换句话说，在集体增长的基本设置之前，代币不能只是某人或某个群体的孤立投机工具。将每个社区想象成一家公司，其中每个成员都是股东。如果公司没有收入，持有股权就没有任何意义。收入将用于支付在公司工作的工资，基于他们贡献的多少，而不是他们拥有多少股权。看到问题了吗？在我们当前的社交网络中，「工资和股权」被视为同一种资产。然后，这个网络不可避免地对我们拥有的东西起作用，而不是激励我们所做的事情。

所以我们首先要解决的问题是：靠别人的成功搭便车的人会越来越少。并且网络将更多地围绕我们所做的，而不是我们拥有的。

2. 工作获得（work-in）和买入获得（Buy-in）之间应该有一个健康的平衡

从第一点开始，许多代币化社区以成员拥有的东西为中心，而不是他们在社区中所做的事情。如何微调这两者之间的健康关系提出了两个基本问题：

资本在代币化生态系统中的作用是什么？以及如何公平地衡量个人的工作和贡献？

让我们再次以经营一家公司为例：

员工既拿股权也拿工资。未来收益的股权和作为劳动补偿的工资。员工根据每月的贡献获得现金，同时持有股权作为长期收益。投资者纯粹是为了长期上涨。他们不需要领薪水。他们的投资甚至用来支付人们的薪水。

该计划之所以如此完美，是因为现金和股权都为自己的目的服务。它们就像两个具有不同效用的「代币」。因此，对于社交代币，如果只有一种资产可以支付所有人的劳动和长期收益，就会出现社交代币悖论。因此，结论很简单：在这个网络中不能以同样的方式对待劳动力和投资。

在社交网络中拥有多种资产或机制来服务于这种目的差异，可能会可持续地在工作和购买权力之间进行授权和微调。在知道了「为什么」之后，我将在接下来的学习中更多地探讨「如何」。

3. 社交代币可以带来额外的网络生产力

让我们退后一步，想想人们为什么要购买社交代币。Mario Gabriele 在他的文章《社交代币：你的经济》中解决了这个问题：

「购买代币是一种赞助还是一种投资？虽然不是相互排斥的，但使用这两个框架——赞助和投资——阐明了代币背后的价值。」

赞助：「进入（access）和排他性。」代币需要具有情感价值。

投资：毕竟是「进入」——它的功能类似于股息。收藏家支付象征性的费用参与，获得潜在的更大价值。

在赞助方面，所有权成为我们身份的一部分，因此排他性成为可预见的动态。不幸的是，这揭示了社交代币悖论并阻止了社区获得大规模采用。

投资心态也有其弊端。早早到达那里有望获得先发优势，即坐在那里无所事事。对于社会经济而言，这种心态会降低提供实际工作的吸引力，最终会损害社区的长期发展。

因此，除了赞助和投资之外，社交代币还应为网络带来生产力，并使我们为网络的长期增长提供工作的自我动机保持一致。

这有助于社交网络激励其成员之间的积极参与，允许个人以自己的技能动手做贡献，并最终实现有效的利他主义，使网络的成长成为一个良性循环。

4. 不可转让的声誉信号更强大

一旦社交代币经济不再依赖于排他性（状态），资本将不再是网络的主要标志。相反，社区成员如何为网络贡献价值成为网络协调的重要指标。这些行为代表了社交网络中的声誉，这可能比纯粹的经济激励更强大。

意识到社交网络的长期繁荣在很大程度上依赖于其文化沉淀，网络声誉需要在代表和延续社区早期文化方面发挥这一作用，以促进网络文化的发展。

声誉的价值

我们都知道声誉很宝贵，但它值多少钱？这是另一个有趣的观察，可以帮助我们理解声誉社交网络中的普遍矛盾：

「这提出了一个悖论：如果代币可以轻松转移，那么没有声誉的人可以简单地购买它，这降低了代币作为声誉信号的能力。」出自 Jad Esber 和 Scott Kominers 编写的《一个用于基于信誉的系统的新型框架》。

这意味着如果声誉可以买到，它的信号就会减少。但是，如果不能交易，声誉的价值究竟是什么？用户仍然需要有一定的利益，并以某种形式量化它。

在这里，我们大声思考了一种替代方案：声誉（不可转让）有自己的衡量标准，设计用于用户在网络中的积极影响。并且可以有另一个代币（可转让）与信誉系统链接，信誉积分可以以这种代币的形式用于引导流动性价值，他们的红利将基于他们的应用工作，这将进一步培养他们的网络信誉，同时保持声誉公正和链上真相来源。

Vitalik Buterin 在他的博客中写到了灵魂绑定代币 (SBT)。他探讨了不可转让的 NFT 如何代表身份和一个人在去中心化社会 (DeSoc) 中的承诺、证书和从属关系。您可以将 SBT 视为区块链钱包中的扩展 CV。而这里的关键创新是让我们的数字身份不可转让，这样我们就可以长期培育它，并进一步增加我们自己的网络价值。

5. 在通货膨胀和用户增长之间保持健康的平衡

社交网络的通货膨胀可能是健康的。进入网络的新用户在开始时有很多激励措施，但如果通货膨胀对内部贡献者进行持续工作的回报较少，这也会对现有用户的利益产生负面影响。

固定通胀也不理想。这就像作为一家成熟的公司经营一家初创公司。在这种高度波动的环境中失去灵活性并忽视市场反馈是一个常见的创业陷阱。

因此，社交网络必须密切关注其货币政策。关键是在用户增长和代币通胀之间找到适当的平衡，以激励所需的用户行为。

让我们更深入地了解网络中新用户和现有用户之间的区别：

  • 新用户的需求：扩张阶段会产生对网络代币的过度需求。用更高的通货膨胀来满足高需求是健康的，但协议需要检查内部参与和合理的代币购买力（价格）之间的动态，以免产生增长和爆炸的债务。

  • 现有用户的网络参与：这是内部劳动力驱动。当代币价格升值时，它会进一步激励社区成员提供工作。同时，新铸造的代币（来自社区的价值生成）减少了代币价格的上行压力，而较低的代币价格可能会降低为网络做出贡献的内部动力。

因此，拥有灵活的发行计划以不断控制上述动态可以使社交网络更加反脆弱。这个决定也可以与社区治理相结合，就像许多大型协议已经做的那样。

6、促进长期参与和新需求

管理社交代币的难点在于其稀薄的流动性和在市场走势中的脆弱性。项目经常进行流动性挖矿来奖励持有者，但这并不能完全解决流动性不足的问题，因为缺乏社会需求和合适的工具来促进他们。以下是一些关于如何做大的思考：

探索新需求

社交代币有可能被整合到整个加密经济中。它可用于支付服务、出租资产、参与不同协议以赚取收益，或锁定以进一步增加我们在朋友圈中的社交信号并传播我们的信念。建设者可以不断探索促进社交代币的新实用程序，并提供支持它们的技术和基础设施。

结合 Defi 工具来扩展网络容量

社交代币相当碎片化。他们在不同的网络上，很难相互交流。我们需要更好的平台来连接这些微观经济体，以便我们的社区可以成为更大生态系统的一部分。这可以创造跨项目的价值发现和交换，形成一个更加多学科的社会利益市场，并且我们都可以利用这种联合流动性来弥补流动性缺口并扩大整体空间。如果我们在这里深入研究，DeFi 和社交代币的交叉点有可能解锁尚不存在的全新资产和机制。即使为社会治理开发一些基于锁定和质押的适当治理机制，如果他们希望与网络更深入地结合，已经可以为社会代币持有者产生额外的长期激励。

NFT 可以帮助产生网络收入

NFT 是代表我们社会身份、所有权和声誉的不同种类的动物。他们还可以为创作者带来额外收入，并在早期引导阶段将收入注入社交网络。这可以为代币持有者带来一些额外的奖励，并以有益的方式扩大网络的范围和互动。从用户的角度来看，拥有 NFT 给他们带来了所有权和额外的乐趣，交易它们建立了更广泛的社交联系，所有这些都是网络的健康动力。

7. 瞄准正值应计以逃避庞氏骗局

现在我们已经在社会代币经济的深水中。一切都设置妥当，我们获得了第一批用户，推出了代币，同时事情变得更加令人兴奋或可怕。我们如何验证我们是否在做正确的事情并继续弥补差距以实现大规模采用？

网络价值累积有助于验证社交网络的生命周期

答案很简单，网络需要及时增值。这里的这个价值是一个比收入本身更广泛的概念。网络能给人们带来价值吗？世界是否因为这些社区的存在而变得更加丰富多彩？人们是否拥有更好的服务、更好的所有权和统一的道德？网络是否有一个内部自由市场来适应人们价值交换的本质？印刷更多的代币作为呈现经济增长的唯一方式绝对不是一种可持续的方式。以下是我对如何验证网络价值累积的想法：

价值应计验证：

有意义的目的

这个网络的最终成功是什么？为公众创建更多内容？建立一个以声誉为基础的更强大的社区？如果拥有适当的所有权，信息是否更容易获得？人们的联系越来越紧密了吗？这些答案可能会设定社交网络最高潜力的上限，我们不想忽视它们。

正外部性

要区分一个项目是否是庞氏骗局，我们只需要验证它在可预见的未来是否已经或将产生「正外部性」。如果一个系统没有创造新的价值，那么里面的所有价值都有零和游戏的风险，只是利益相关者之间的再分配。但这种外部价值可能比纯粹的财务收入要广泛得多。如果我们考虑比特币的外部价值是什么，它是开放、自由、无可争议的所有权以及对其共识的绝对信任。

内部价值消费

网络内的价值消费对于网络的生命力是必不可少的。它有助于经济实现足够的外部性并增加网络收入。因此，用户进入的目的不仅仅是作为一个投机者，相反，价值是易手、支付公用事业费用、认可他人，一旦生态系统内的一切都可以用其本地货币完成，用户就不需要兑换回来。

小心增长的债务

可能有可能在早期使用庞氏模型来增加用户数量，并给项目更多时间来创造实体经济。但重要的是及时积累收入，这样网络可以抵消早期用户增长的债务，最终创造收入并成为净正数。

8. 尽早建立多元化社区

在 Web 3 中进入市场就是进入社区。拥有一个真实的社区是对抗加密货币波动性的最佳力量。了解社区的优势可以培育具有巨大力量的社交网络。

社区中也不能只有一种类型的利益相关者。利益的多样性对于适应未来的经济活动至关重要。只有当每个人都需要别人的东西时，经济才能被证明是可持续的。例如，创建者、消费者和管理者。他们每个人都可以在网络中拥有一组利益和独特的经济行为，并需要其他人的一些东西，而网络只需要管理他们的利益并以积极的方式调整利益相关者的激励。

同样，声誉可以鼓励积极参与

金钱不是社交网络的最终驱动力，声誉才是。与其创造出只有退出目的的投资者，拥有支持其活动的声誉可以让用户在网络中停留更长时间。这里的简单解决方案是，当用户对治理或社区工作产生积极影响时，使用户的声誉得分乘以最终奖励，因此即使不直接将其货币化，它也会对进一步激励善行产生影响。

9. 社交网络应以乐趣为先

永远不要忘记，社交网络是情感的、功能性的和无私的。它的优势在于它在日常生活中为用户带来的所有情感价值。在社交网络中，网络价值依赖于用户对用户而不是用户对平台。而这种情感价值应该作为更高的使命放在任何货币化目标之前。因为没有正确的「为什么」，「如何」和「什么」将不可避免地在风中摇摆。

这里的关键价值在于人。区块链提供了更好的工具来实现这种点对点关系。这不仅限于纯粹的经济回报。提供有趣且有意义的社交体验，同时提供对数据和价值的真正所有权，这就是我们这么多人爱上 Web3 社交的原因。

10. 关键逻辑要上链

可以理解的是，为了获得预期的用户体验，当前的社交网络必须利用中心化的后端或缓存层来让事情先运行起来。但需要解决的是，这种妥协可能会降低网络的透明度和信任度，并可能对项目的商业模式构成威胁。因此，网络的核心功能和经济逻辑应该是链上的。

这里的风险是在没有真正去中心化经济的情况下将 Web 2 商业模式隐藏在 Web 3 窗帘后面。例如，用户身份、声誉、内容、用户数据、价值生成和分配，如果这些元素的核心逻辑不在链上，那么我们就冒着构建一个带有代币的 Web 2 平台的风险。

当然，社交网络使用中心化服务器来运行其部分逻辑是一个折衷的决定，因为现阶段 Layer 1 的性能和成本无法充分发挥其潜力。但是事情会改变。最好的 Web3 应用程序还没有构建出来，所以让我们一起构建 Web3 技术的成长，拥有正确的基因。

现在是成为 Web3 创造者的好时机

Web3 技术为所有主权个人提供了前所未有的创造力和灵活性。历史上第一次，我们拥有公平所有权、开放性和可组合性的美好未来。我们刚刚迈出了建立社交代币网络的第一步。

作为一名建设者，成为这个新兴的令人兴奋的空间的一部分已经是一种乐趣，我渴望了解更多。对于本文，我要感谢 Gaby Goldberg、Mario Gabriele、Chris Dixon、Patrick Woods、Vitalik Buterin、Jad Esber、Scott Kominers 等，感谢他们在我探索社交代币空间时所做的出色工作。我们正在建立在您们的见解之上。
