作者：Nichanan Kesonpat，1kx；翻译：金色财经cryptonaitive

Crypto为消费者体验创造了一个全新的交互设计领域。

迄今为止，在链上展示实物商品的尝试是由加密原生品牌带头的，主要用于高端时尚商品的真实性和出处。与此同时，绝大多数链上行为，如 NFT 铸造、证明、交易和装备，仍然仅限于数字环境。

我们已经看到通过数字物理（digiphysical）消费品将链上交互带入物理环境的巨大潜力，并相信数字物理基础设施将很快为任何物理产品带来可编程效用。

加密原生数字物理产品是以物理形式存在的商品，并通过 NFC 技术维护与其数字（链上）记录的加密链接。该记录可以采用 NFT、注册表项或具有签名功能的链上ID的形式。

芯片技术与链上可编程性相结合，为任何物理对象提供了类似钱包的能力（例如，代币门控访问、签名和资产所有权），解锁了以前不可能实现的终级消费者体验。

我们对下一代数字物理产品感到兴奋，因为它们：

  • 将链上交互带入物理环境。 嵌入式可扫描硬件允许消费者以可验证的方式证明他们与某个物体的接近程度或在物理空间中的存在。该证明可以任意解锁对位置敏感的实用程序，例如现场铸币，或作为参与记录，使持有人有资格获得未来的奖励。

  • 赋予任何消费品主体性和声誉。 具有签名生成功能的芯片可以让任何实体商品建立自己的链上历史，并在其所有者在现实中与品牌和社区互动时积累价值。因为物品的链上记录与所有者分离，所以物品可以带着它的完全历史出售并得到相应定价。

  • 为发行人创建新的分销渠道。 品牌、社区和创作者可以为其产品和行业提供持续、动态的效用。在现实中互动成为一种衡量标准，用于将差异化奖励扩展到参与度最高的成员，而这些成员又会受到激励更多地参与社区活动。

  • 为消费者解锁全栈效用。 数字物理产品的所有者可以同时在物理（佩戴物品）和数字（灵活所有权、抢先体验权、独家内容）领域获取价值。

过去的数字物理产品是什么样的？

第一个数字物理产品存在于加密之前。我们可以将它们视为与其数字形式“捆绑”在一起的物理产品。 想想带有数字下载代码的黑胶唱片，或附有pdf格式的实体书。虽然这些捆绑产品允许购买者以多种方式消费内容，但数字和物理对应物彼此之间没有严格关联，并且被视为附加组件，通常未被使用。

在加密领域，早期的实验包括通过销毁兑换等链上机制销售未芯片化的“数字孪生”产品。 在这里，所有者必须销毁/更换 NFT 才能领取实物产品，或者在进行实物购买后领取 NFT 附加产品。与捆绑产品一样，早期的“数字双孪生”产品将每个对应产品视为次要产品，而不是彼此紧密结合的产品组件。

虽然这两种形式的实体-数字捆绑都为消费者提供了增量利益，但实体产品和数字产品之间的耦合是松散的。理论上，每种商品都可以单独出售，而不会失去其自身的内在价值或效用。

嵌入式芯片与链上可编程性相结合，使我们能够实现物理对象与其链上记录和产权之间的紧密耦合。 这意味着物理和数字不再是截然不同的产品，而是同一枚硬币的两个面。物理产品是持有者读取或写入其链上记录或通过现实交互行使随之而来的权利的一种方式。

非web3和web3的数字物理产品

数字物理商品的主流心智模型是物理时尚商品，其有一个 NFT 对应物可在虚拟世界中穿戴。虽然这是一个用例，但它只是触及了已打开的设计空间的表面。

更好的类比是（实体）信用卡和它授予的（数字）金融身份、购买力和访问权限。信用卡是持卡人在现实世界中进行购物的一种方式，而银行持有信用卡使用情况的数字记录以及随之而来的任何收益。发卡机构与供应商建立合作伙伴关系，为持卡人提供持续的效用，例如机场贵宾室使用权和酒店折扣。

数字记录不一定是实体卡的“镜像”，而是第三方通过现实交互读取和更新其记录的功能的重要组成部分。

另一个例子是节日腕带，与会者可以使用它进入会场、支付和领取奖励。腕带本身是一种方便的支付方式，但其具有的访问权限和可消费余额的数字记录本质上是使其有用的一部分。

加密原生数字物理产品：以加密方式链接到链上记录的实物商品

加密原生数字物理产品将这种可编程的交互性带到任何消费产品中，用于各种金融、社交和娱乐用例。它们可以是随身携带或佩戴的物品（例如衣物、POAP 卡）或固定物品（例如画廊中的艺术品、收藏品），扫描以提取其所有权历史、真实性证明、历史互动和相关效用。它的链上记录使交互性可组合——任何人都可以在没有门控 API 的任何商品分配奖励和授予权利给。

加密原生数字物理产品有哪些用例？

流行的元宇宙概念是一个类似于 Ready Player One 的 VR 环境，其中“数字孪生”的人类和物体在沉浸式虚拟世界中独立运行，但我们相信它更可能是具有链上记录及跨物理和数字环境全栈效用的行业。

创作者货币化和粉丝参与

  • 扩展商品交互性。 创作者现在可以将任何实物商品变成具有售后效用的会员卡。商品成为粉丝和粉丝彼此之间的直接渠道，使创作者能够与粉丝以及粉丝群内部建立更深层次的关系。商品附带的体验是可编程的，因此交互性是持续的。

  • 粉丝细分。 如今，由于票务和商品销售是通过第三方进行的，而第三方控制着他们与艺人分享的很多数据，因此艺人关于其粉丝群的信息存在巨大差距。通过在活动中分发 POAP 并建立点击和扫描等现实交互记录，创作者可以详细了解他们最活跃的粉丝是谁。

  • 有针对性的奖励。 在活动中使用芯片化和可扫描商品，创作者可以直接奖励粉丝。粉丝可以在社区内建立自己的声誉，因为现实和数字参与可以记录在链上。在多个活动中佩戴有芯片的商品可以为商品累积忠诚度积分，可以扫描商品以解锁未来演出的预售。

代币化社区

  • 连接证明和社交游戏。 专为个人设计的产品，例如一件衣服或一张卡片，可以成为社区与其他成员见面时联系的一种方式。它还支持围绕个人连接的新型游戏，例如9dcc Network Points。

  • 社交图谱。 “I met”POAP 转账活动创建了一个自下而上的社交图谱，可用于查看社区成员将参加哪些活动，以及“连接”最多的成员。这可以作为一个原始数据点，将奖励分配给在活动中最积极地向社区传播福音的成员。

  • 付款和共同签名。 因为物理实体现在可以生成加密签名，所以它可以用作多重签名的签名者，以作为 DAO 成员进行非接触式支付。这可以是会员卡、连帽衫、手镯或其他任何东西。

去中心化ID和所有权证明

  • 奢侈品和收藏品。 用于奢侈品、艺术品和高端收藏品的传统认证系统依赖于中心化服务器，使用芯片的链上认证标准使实体产品能够自我证明其真实性，即使它们的发行人不在了。一旦扫描，就会有一个透明的记录，记录谁制造了芯片，哪个品牌发行了产品，以及它有什么权利。

  • 全栈弹性。 使用像实物支持 NFT这样的标准，品牌可以将实体服装绑定到数字可穿戴设备上，所有者可以跨物理-虚拟堆栈使用。代币标准确保 NFT 和实物物品的所有权保持紧密联系，允许所有者在数字和物理空间之间可验证地灵活调整所有权。

  • 加密签名。 KrauseHause Metafactory 夹克，篮球运动员可以点击并进行数字签名。

数字物理交互

  • 真人角色扮演游戏。 通过芯片化物理产品和玩家在现实世界中的互动记录，链上游戏可以来添加现实任务。Loot LARP是 NFT NYC 2022 上的真人角色 larp 体验，玩家使用嵌入了 Kong 芯片的武器互相战斗。互动后，武器的分数被记录下来。

  • 现场铸造。 画廊可以设计引人入胜的互动式铸造体验，方法是让客人扫描芯片光盘以购买展示艺术品的版本或铸造生成作品。

  • 留言簿代币化** 。** 观众可以在数字作品中记录他们的“存在证明”。NFT 元数据可以随着来自与它非常接近的人的每条消息而更新。

Web3领域的数字物理实验

Web3领域数字物理实验和基础设施出现的时间线

2019——2021：早期尝试连接链上和实体

  • 使用芯片的著名实验有Kong Cash、Signed Star Trek Collectable和 MetaFactory 的Genesis Bomber（2020 年 2 月），它们使用了SiLos 芯片。

  • Saint Fame $FAME、Unisocks $SOCKS、Zora $TAPE使用 ERC-20兑换机制，该机制在代币持有者可以销毁/替换代币以换取实物产品的日期之前在公开市场上进行交易。除了出示销毁证明或赎回代币作为收据外，实体产品的所有者无法实时证明其所有权，发行人也无法继续提供福利，因为没有链上链接。

  • POAP 的推出是为了让个人在链上记录他们参与活动的情况。

2021年：新的基础设施层

  • IYK为发行人推出了在芯片上编写实用程序的工具，例如 tap-to-own 和 tap-to-mint。这后来在9dcc ITERATION-001 衬衫的VÉRITÉ Crewneck和 POAP 铸造中实现了直接到实体的消息传递。

  • 4k和Courtyard等金库解决方案也在此时推出，提供了一种在赎回之前或交易期间保护实物的解决方案。

  • 随着 NFT 和生成艺术的兴起，Bright Moments开始将链上艺术带入物理空间，并很快开始与ArtBlocks和fx(hash)等平台合作，将生成艺术铸造/展示体验带入物理环境。

2022年——2023年初：基于协议的早期应用

在硬件和可编程性层的支持下，在过去的一年里，我们看到更多市场采用了数字物理技术，特别是围绕代币化社区和创作者与粉丝的互动。

  • 像 Nouns 这样的链上社区开始发行商品，与Dopamine合作推出带有芯片的街头服饰，买家可以扫描这些服饰以获得数字奖励。DAO 还批准了启动Nouns Vision 的预算，Nouns 成员可以在铸造前使用 NFT 兑换豪华的 Nouns 太阳镜。

  • VÉRITÉ 与 IYK 合作推出了VÉRITÉ Crewneck，点击https://iyk.app/drops/verite-crewneck可以抢先体验她的下一张唱片和其他独家体验。

  • 代币门控是Collab.Land首创的一种限制访问 Discord 和 Telegram 频道的机制，开始应用于物理空间。组织可以使用tokenproof等协议让与会者验证资产所有权以进入会场。

  • 时尚以外的加密原生品牌如Azuki 等通过在其Golden Skateboards中使用 Kong 芯片并制定物理支持代币(PBT)标准，进军数字物理领域。

  • 随着越来越多的加密原生人士接触到芯片产品，社交游戏应运而生，旨在与尽可能多的佩戴该系列产品的人会面。来自9dcc ITERATION-001衬衫的 POAP 铸造可以在链上作为迈阿密巴塞尔艺术展社区的社交图谱被观察到。在 ETH Denver，Bead DAO形成了一个黑客马拉松项目，在这个项目中，串珠蜥蜴嵌入了 Kong 芯片，新成员可以在遇到携带蜥蜴的人时加入。

  • 我们看到了第一笔针对数字物理产品的贷款，gmoney 用一件 9dcc IT-001 衬衫借出一笔贷款，以 4k 为金库。2023 年 4 月，4k 发行的 PBT 让劳力士可用于 DeFi 贷款。

Web3数字物理堆栈

Web3数字物理堆栈

基础设施层

硬件。 物理接口，例如专门的 NFC 芯片、动态 QR 码或stenography，为区块链提供链接和通信方式，以更新数字物理的链上记录。重要的是要注意并非所有芯片都是平等的。例如，Kong 最新的HaLo 芯片，与其前身 SiLo 芯片和典型的 NFC 标签不同，它可以生成自己的签名，因此表现得像一个具有自己身份（链上地址）的硬件钱包，这支持支付和共同签名用例。

创作者工具、铸造和生成引擎。 硬件基础设施之上的可编程层使创作者能够设计数字物理体验。IYK 为发行人提供与芯片无关的软件模块，包括后期分发的点击式分析工具。Metalabel 是一个供创作者发现彼此并以数字物理记录包和现实体验的形式共同发布作品的平台。fxhash，ArtBlocks为生成艺术提供铸造引擎，在画廊和现实活动中实现现场铸造体验。

存储、托管和商业。 4k、Courtyard和Mynaswap 等托管协议在 NFT 赎回之前或交易期间为实物产品提供运输和存储服务。这涉及通过每个平台自己的铸币服务对产品进行 nft代币化。Americana更进一步，将其自己的芯片技术附加到运送到仓库的实物中。这些协议是代表链上实物资产所有权的关键基础设施，因为代币所有者必须确信基础资产以所代表的确切形式和数量存在。

供应链和价值链。 Permet和Crowdmuse等协议在从设计师到制造商的实体产品供应链中提供贡献跟踪服务，这有助于在未来项目复用原语时进行后续价值分配。

RWA代币化和数字化协议 用于为现有实物商品提供链上表示，通过芯片化（Mattereum的星际迷航收藏品、Sunya手镯）或通过去中心化Guardians网络 ( 4k ) 或者中心化服务 ( Courtyard )铸造“实物支持代币(PBT) ”，确保实物商品和链上数字物品的链接。然后，实物支持代币(PBT) 可以在 OpenSea 等开放市场上进行交易，或用作 DeFi 协议中的金融工具。数字化平台Altr为实体品牌提供了将其档案上传为高保真 3D 资产的工具，支持虚拟试穿、可穿戴设备和数字物理兑换选项。

NFT金融化。 实物资产曾经在链上表示为 NFT，可以接入现有的 NFT 金融基础设施。它们可以像OpenSea上的任何其他 NFT 一样进行交易，并用于在NFTfi 和Arcade等平台上抵押贷款。还有专门的市场，例如代币化奢侈品的unxd ，代币化纺织品和美术的Tectyle ，以及复古服装的mntge 。

生成式制造/数字到现实。 迄今为止，生成式引擎的采用一直在数字艺术市场上进行。随着消费者希望与他们的现实收藏互动，Trame和Artmatr等项目为创作者提供了一个制造管道，将数字文件转化为实体艺术。在艺术市场之外，这些制造管道最终可以作为任何生成式消费产品的基础设施。例如，生成式时尚物品可以首先被铸造成 NFT，然后持有者可以用它来购买相应的实物产品。

标准与注册。 实物支持代币(PBT) 是一种标准，通过该标准，芯片化的实物商品可以证明自己的真实性，并与其链上的 NFT 对应物保持紧密耦合。这是通过可以生成自己的签名的芯片实现的，并且只有在将芯片的签名提供给合约的情况下才允许 NFT 转移（保证在未经实物所有者同意的情况下不能转移代币）。以太坊现实服务(Ethereum Reality Service，ERS)，由Arx牵头（Kong 芯片），为芯片、制造商和服务提供商提出规范注册，证明芯片的功能以及将它们嵌入对象的项目/公司。它还提出了一些模式，开发人员可以借此表明针对芯片的声明和功能是可能的。

应用层

代币门控、票务和会员资格。 Collab.Land、Guild、Tokenproof和Unlock等协议允许构建基于代币的自定义逻辑，实现允许某人获得进入数字空间、参见现实事件和获得数字物理折扣（例如 Shopify 折扣）。除了收藏家的链上历史和资产所有权之外，这些工具还可用于将资格扩展到拥有实物产品的人，例如扫描你的连帽衫以提前获得门票销售。

现场铸造。 Bright Moments和VerticalCrypto Art等社区在世界各地举办以展示生成式艺术为中心的现实活动，通常与艺术家合作，为持票人和 DAO 成员设计实物展示和铸造体验。一些画廊，如imnotArt，也有虚拟地产，用户可以在其中访问和竞标策划的艺术品。

社区参与。 虽然POAP 被广泛认为是证明你参加过活动的徽章，但当个人佩戴有芯片的社区商品时，该协议会变得更加强大，因为他们自己可以成为 POAP 发行者。TYB帮助品牌利用 web3 会员资格来推动产品开发和激活，奖励店内访问，代币门控Shopify 折扣，并支持社区共创。

结论

  • 芯片将变得更便宜且经济可行，可用于高端时尚、收藏品和艺术品以外的产品

  • 消费者很快就会期望创作者和社区的商品被芯片化，并能够在现实活动中与它们互动以获得未来的奖励

  • DAO 和加密原生社区将开始使用卡片形式的数字物理产品来执行共同签名和支付等操作

  • 因为代币化和保险协议，NFT 将越来越多地用于代表实物资产的所有权，从而将交易成本和速度降低到纯数字资产的水平。我们还将看到代币化的现实世界资产，如收藏品、高端时尚商品在NFTFi协议中得到利用。
