作者：WaTell 来源：推特，@0xWatell

OpenSea 为它更换头像、修改 ID，beeple 为它进行多次二创， 一场新的文化革命正在英文 NFT 圈子里爆发翻腾，花 5 分钟时间快速了解它。

在 8 个小时前，@opensea修改了自己官推的头像，并将 ID 修改为 OpepenSea 。 作为整个 NFT 圈子里的顶流，是什么驱使着它做出这样的举动？Openpen 又是什么？

如果你是一个专业的 NFT 玩家，或者对 NFT 有一定了解的话，肯定能一眼认出这是@jackbutcher的 “Opepen Edition” NFT 系列。 这位曾靠 “Checks - VV Edition” 引燃「NFT Open Edition 热潮」的多元艺术家以「Pepe」为原型，创造了这个具有极高标识度的 Web3 原创形象。

看到这里，你的脑海中可能会浮现一个问题： “我确实认识/看过这个 NFT 形象，但它不是几个月前就出现了么，为什么最近又爆火了？而且几乎每个英文区的 NFT 玩家都在发不同风格配色的 Opepen，这是为什么？“好问题，这就是今天这条线程的重点。

@notthreadguy—— 一位专注于 NFT 内容输出的 SPACE HOST。 经常举办各类 TWITTER SPACE，讨论有趣且善意的话题，因为其独特的风格、高涨的活力得到了 NFT 玩家的一致好评。 (他已经更换了新的头像，为了激起你的记忆/让话题能够延续，我在配图中放置的是他原来的头像 —— 一只MAYC)

7月12日，一位叫做@BoredOpepen的匿名用户(真实身份为 Opepen 的第一持仓大户)发推表示：“愿意用自己价值 20 ETH 的 ‘001 Opepen’ 来交换 ThreadGuy 的 MAYC。” 交换条件是 ThreadGuy 需要持有 ‘001 Opepen’ 至少 3 个月，Bored Opepen 会持有其 MAYC 至少一年。

两天之后，7月14日，Web3.0 彩票公司@trustwayllc表示：“愿意为 ThreadGuy 的 MAYC 提供 42.69ETH 的 Offer。” 这场非正式的、甚至不是竞拍品拥有者举办的拍卖正在以一种不可言喻的热度开始逐渐爆发。

但我们的主人公 ThreadGuy 是肯定不会接受任何一笔 Offer 的，毕竟这只 MAYC 已经成为了他个人品牌的代名词，卖掉它就等于卖掉自己。 随后，他就在自己的推特上公布了 “不会以任何价格卖出这只MAYC” 的决定，看起来事情的发展要告一段落，但事实确是远不止于此 …

ThreadGuy 在宣布不会卖出的同一天，po 出了一张非官方的 Opepen 二创图，虽然只是一则普通的 GM TWITTER ，但是却引来了 Opepen 的作者 Jack Butcher 的注意。 Jack Butcher 在 ThreadGuy 的推特评论区中 po 出了自己为他创作的官方版 Opepen ，推特瞬间炸锅。

这张 Opepen 是 Jack Butcher 完全按照 ThreadGuy 的 MAYC 进行的风格化创作，相当于为他定制的 1/1 。 这当然是值得激动的 —— 试问，谁不想获得知名艺术家的专属定制呢？ 于是这场拍卖活动开始朝着另一个方向发展：复制与粘贴。

许多 Jack Butcher 的忠实粉丝开始把自己的推特头像换成这张定制化的 Opepen，因为他们觉得 ThreadGuy 是不会更换自己的头像的。 既然如此：“为什么我们不能成为 Opepen 版的 ThreadGuy 呢？” 于是，越来越多的人开始加入这场 “成为ThreadGuy” 的活动中。

这场活动的规模越来越大、传播范围越来越广、参与的人越来越多，最终： “ThreadGuy 本人也加入了这场有趣的实验。

为此，他还特意举办了一场 SPACE 来讨论这件事，这场 SPACE 共计有 3689 名收听者，几乎所有的英文区 NFT KOL 都来参与了，而听众席上更是满满当当的Opepen ThreadGuy。所有人都很开心，因为这种充满了幽默的创造力正是NFT社区所推崇的，但事情还没有结束。

7月17日，Jack Butcher 推出了一个名为 的 Open Edition NFT Collection。 据他所说，此举是为了 “将一些经济能量重新分配给该生态系统的巨大贡献者”，因此铸造的所有收入都会捐赠给 ThreadGuy 。 这是对他一直专注于 Web3 BUIDL 的奖励。

至此，我们终于迎来了本次文化运动的高潮。 在我写这行字的同时，Opepen Threadition>已经铸造超过 38,231 个，ThreadGuy 将会得到超过六万美元的奖励。

对于此次名为 “FOR THE CULTURE” 的文化运动，有人评价道：一场创作者革命正在进行中、NFT 文化的不断发展让那些真正深爱 NFT 的人能够得到他们应得的报酬。但同时，也有人评价：这是一场仅适用于那些 “Web3主角”的狂欢，与大部分的建设者无关。

结语

我在想： 不论如何，正如@nftnow所说的那样：NFT 几乎每天都在改变我们的生活，它让社区中的人们建立起别样的友谊，并能以 NFT 的方式将这种不可触摸的情感保存下来，这肯定是属于每个 NFT 玩家的独特经历。问题来了：在看完这些之后，你的想法是什么？
