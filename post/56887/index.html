在今年的 EthGlobal Waterloo 2023 会议上，Vitalik 发表演讲，表达了他对当前 web3 SocialFi 空间的担忧。他表示，现有的 web3 社交 Dapp 可能不足以取代 web2 的对应产品。

他还建议开发者尝试利用 Web3 的独特功能迭代新的社交 Dapp。我们的目标是构建一款比 Web2 同类产品好 10 倍的社交应用程序。

在 DWF Ventures，我们的主要投资重点之一是非金融消费 Dapp。这促使我们探索 SocialFi 领域，特别是社交媒体。在之前的帖子中，我们的团队讨论了 web3 社交媒体，回应了 Vitalik 的担忧，并鼓励使用区块链创新和独特功能来创建全新的 Dapp。

在本文中，我们将继续研究区块链技术的独特之处，并揭示其彻底改变社交媒体、改变每个人的用户体验的潜力。

Web2 社交媒体平台简要概述

根据最常被引用的社交媒体定义之一，

“社交媒体是一组基于互联网的应用程序，它建立在 Web 2.0 的思想和技术基础上，允许创建和交换用户生成的内容。”

_–_卡普兰 AM、Haenlein M. (2010)

该定义涵盖了 Facebook、Instagram、Twitter、YouTube、Snapchat 和 TikTok 等社交媒体平台。这些平台为我们提供了多种源，不断传递信息并让我们全天保持参与。在社交媒体应用程序组中，它可以进一步分为八个主要垂直领域：

  1. 内容社区 - Pinterest、Flickr、Youtube、TikTok、Soundcloud、Jamendo

  2. 合作项目——维基百科、Github

  3. 博客——Wordpress、Blogspot、Substack

  4. 微博——Twitter

  5. 社交新闻网站 — Reddit、Digg

  6. 虚拟世界——《魔兽世界》、《第二人生》、《Bondee》、《Clubhouse》

  7. 社交网站 — Linkedin、Facebook、Myspace

  8. 商业社区 — Amazon、eBay、iStockPhoto、Shopify

目前，web2 用户愿意向社交媒体平台泄露大量信息和社会影响力。然而，这些数据在每个应用程序中仍然是隔离的，要求用户在使用新应用程序时从头开始。算法缺乏透明度和公共代码共享的缺乏也导致了这些挑战。Twitter 文件的发布证明了这些问题是真实存在的，该文件暴露了 web2 社交媒体平台缺乏中立性和透明度。

Web3 是社交媒体的游戏规则改变者

与 web2 相比，web3 重塑了用户与社交媒体的交互方式。

在web3社交媒体中，用户账户成为与区块链交互的主要方式。社交媒体应用程序扮演着不同的角色，充当帐户身份和算法的接口。web3 社交媒体平台不依赖集中式可信第三方 (TTP) 数据库，而是利用区块链作为公共数据库。这使得开放验证成为可能，并促进各种社交媒体应用程序之间的数据无缝传输。

web3 社交媒体捆绑完全颠覆了 web2 社交媒体环境中应用程序和用户之间当前的关系动态。用户不再局限于特定的应用程序，因为他们可以访问代码、算法、自己的内容和社会资本。

Web3 社交媒体中利益相关者的双赢场景

除了用户之外，内容创建者和构建者现在可以更动态地参与社交媒体领域。区块链技术允许智能合约通过 NFT 自动支付版税，而无需与 TTP 分摊费用。由于 web3 项目通常是开源的，因此构建者可以轻松访问相关空间上的代码，以在需要时构建新的应用程序。开发者也无需担心用户的增长，因为他们可以利用社交资本轻松移植到新的应用程序。

Web3 社交媒体平台的潜在发展方式

社交媒体自 1979 年早期的“Usenet”（一个用于计算机讨论的基本网络）以来已经走过了漫长的道路。从非交互式文章到 4chan 上 6144 KB 的图像上传限制，社交媒体的定义随着三个主要驱动力的发展和调整：

  1. 技术进步

  2. 经济增长

  3. 社会文化影响

web2 社交媒体在其演变过程中遇到的局限性与 web3 当前的局限性有相似之处。链上数据存储的缺失让人想起过去的上传文件大小限制。无法处理大规模交易让人想起论坛上服务器和网关的局限性。昂贵的汽油费与早期互联网使用的高成本相呼应。

通过认识到这些相似之处，我们可以识别 web3 背后的驱动力，并概述有助于开发 web3 社交媒体理想平台的潜在因素。

Web3 社交媒体已准备就绪

催化第一波 10 亿 web3 社交媒体用户所必需的潜在因素已经在发挥作用。鉴于Twitter和Reddit API 的视图限制以及版主惨败，对 web3 社交媒体的需求正处于顶峰，这为其出现提供了一个有利时机。
