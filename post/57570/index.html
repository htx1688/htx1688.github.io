在过去几年里，加密货币行业中有两个领域经历了快速发展。一个 2020 年爆发的 DeFi 

Summer，另一个是 2021 年引发的 NFT 热潮，当 DeFi 逐渐褪去当年的热度，NFT 赛道发展遇到瓶颈时，人们开始探讨或许二者的结合能迸发出新的契机，于是 NFTFi 出现了。

关于 NFT 近期大环境的变化，无论是蓝筹 NFT 的急剧下跌还是 BTC NFT 的崛起，以及近期 ERC 6551 的讨论热度，都显示出了 NFT 赛道下一轮叙事将会有重大变化。目前市场上关于 NFT 赛道的发展主要有三个论调，一是蓝筹叙事的终结，NFT 赛道的陨落；二是 NFT 赛道创新更迭，新叙事的起航；三是 NFT 赛道重新洗牌，沉淀之后大有可为。虽然目前 NFT 市场很冷，但是 NFT 赛道绝不会一蹶不振，未来需要更多的创新性以及更强的金融属性，而以近期 Azuki 事件所带来的市场连锁反应，反而是给 NFT 赛道带来了更大的机会。

LittleMami 在 8 月 10 发起了“NFT 借贷工具是如何在熊市中坚挺的“Twitter Space，邀请到了 NFTGO、BendDAO、ParaSpace、xBank、LENA 几位在 NFT 行业及 NFT 借贷都有突出表现的项目共同进行了该主题的分享。

LittleMami：LittleMami 是一个Web3品牌，致力于通过 NFT 的文化共识和金融交互打造一个可玩性、收益性、可持续性的 Metaverse。 LittleMami 的核心是通过 Mami Protocol 打造一个元宇宙的基本金融应用设施，为 NFT 提供金融衍生品服务，并通过质押，贷款和其他功能增加流动性。

NFTGO：NFTGo 旨在成为全球领先的 NFT 数据情报提供商，在大数据、人工智能和区块链技术的驱动下，让所有参与 NFT 交易和使用的人受益。

BendDAO：BendDAO 是一个基于点对池的 NFT 流动性协议。借款人可以使用 NFT 作为抵押品通过借贷池借入 ETH。存款人可以向借贷池提供 ETH 流动性以赚取利息。

ParaSpace：ParaSpace 是一个跨保证金的 NFT 借贷协议，允许用户以 NFT 和 ERC-20 进行借贷。ParaSpace 允许用户利用未被使用的 NFT 进行进一步的投资并赚取收益。

xBank：xBank 是一个具有稳定性、效率和偿付能力的开放式货币市场。xBank 通过先买后付（BNPL）、抵押品出售和升级交易等服务，匹配货币需求和供应，为Web3的新用户提供货币杠杆。

LENA：LENA 是最新的Web3流动性提供者，为质押奖励、借贷、借出以及更多提供 ETH 流动性池。赋予用户自治权和透明度是我们设计理念的核心，我们提供的服务涵盖了所有Web3资产。

LittleMami：首先邀请到的是 NFTGO 从数据端的角度为我们分析当下 NFT 市场整体表现，为后续其他问题的展开提供数据支持。

NFTGO：

  • NFT 市场的 30 D 的 PNL， 90% 的⼈都在亏钱。依然很早期， BAYC 的 all time trader 是 17 k， MAYC 是 35 k。

  • 目前的市场表现基本在 21 年 7/8 月份的水平。 NFT 市值和交易量在 22 年 1 月和 22 年 4 月是两个最高点。交易量目前的数据为 21 年 7/8 月份的水平， 基本就是上轮牛市爆发初期了。近 30 日对 ETH 上 NFT 的 trader 数量为不到 12 万， 日均 trader 在 1 万左右， 目前的数据也是接近 21 年 7/8 月份的水平。

  • 值得注意的是， 其实 7 月比 6 月的交易量降了很多 (腰斩) ，可能主要受到由于 azuki elemental 的影响。市场以 PFP 市值占比在 21 年期间持续攀升， 到 22 年 1 月达到 56% ， 之后平稳在 55% -60% 的区间

  • 近 3 个月（截止 8 月初）， NFT 交易量还是蓝筹为主， BAYC + MAYC + BAKC = 34% ， Azuki + Azuki Elemental +beanz = 19% (但是砸盘占比高) ， degods， captainz， milady 各自能占到 3% 左右。而且最近三 个月和最近 1 个月都没有现象级的新项目出来。前⼗名以后的全部项目总占比仅 36% ，头部效应明显。

  • 近三个月， 去除 washtrade， blur 交易量已经是 opensea + opensea pro 的 2.8 倍； 但是 opensea 的 trader 数量是 blur 的 3-4 倍。

LittleMami：非常感谢 NFTGO 提供的很全面的数据分析，总体而言，虽然 NFT 市场经历了一个明显的下滑，但这并不意味着 NFT 市场失去了增长潜力，市场上依然有很多的 NFT 项目源源不断的出现，NFT 仍然是具备高潜在增长倍数的市场之一。另外刚才也有提到 7 月份市场很熊的一个原因是因为没有一匹黑马出来，那所谓的黑马是怎样定义的呢？

NFTGO： 7 月整体市场很熊的原因一个是没有黑马出现，另一个是 Azuki 事件的影响，从数据侧来看，交易量直接腰斩，几乎失去了流动性，这种情况其实是很少见的。

关于黑马的特征主要在于它的持续性，如果它是一个持续比较久的黑马，像之前的 Milady，它一定是要有一个很明确的特点的，就比如有人曾经总结过 BAYC 是一个性冷淡风，从审美侧它有一些比较特征性的一些东西，或者就是说它有一个很不一样的叙事，就是像 Milady，其实它们在各自的赛道都切中了一个，之前没有人真的去切中的一个叙事，就是 checks 在能做到在艺术赛道的一个跟这个 IP 化、跟个人影响力更结合的，然后并且能够不断地去做这种。

Milady 作为一个 Trader 社区，它会有这种自己的独立的叙事，所以如果真的想出一批像这样子的马，它一定能够要走出来一波自己的独立叙事。就是这个叙事一定要不能是很简单的，比如说模仿，或者就是元宇宙是叙事，或者就是 IP 叙事，这种已经被大家炒过的就很难吸引装的，或者像一些就做事商的一些介入了。

但如果它是一批很短暂的黑马，那其实就是 MEME 的逻辑了，那无非就是看有没有人愿意去拉盘和炒作，那这种黑马的周期可能生命周期只有在 3 周到一个月左右，那像 Milady、 checks 可能就是能稳住，对，所以我觉得这个也要看这个特点。

LittleMami：刚刚从 NFTGO 给到的一些关于市场数据分析的情况可以看出，NFT 市场萎缩、蓝筹几乎全线下跌，作为 NFT 借贷平台是如何应对这一挑战的呢？

Xbank：

xBank 对 NFT 市场数据也很关注，选择了 NFT 借贷作为核心方向，以及通过 BLPL（先买后付）和期望通过新的 NFT 发行手段来应对不同情况下的用户需求和市场机会。

团队密切关注 NFT 市场数据，特别是 NFT 市场总市值的变化。其中有一个很明显的变化是 NFT 市场总市值从之前巅峰的时候 200 多亿美金，跌到现在只有 56 亿美金左右，这也说明目前 NFT 市场还处于相对小的规模。同时我们也发现基于现实世界一个资产它的衍生品可能是其原生资产的 30-50 倍，因此我们才选择了 NFT 借贷这个赛道。

xBank 的核心产品是 BNPL（先买后付），这体现了团队在熊市下的策略。通过降低新用户的进入门槛，引入增量用户。例如，用户可以选择先买后付的模式来购买高价 NFT，使用杠杆来分期付款。我们也注意到像 Matr 1 x 这样热门的 gamefi 项目，一些用户需要金融支持。xBank 提供新的方式，让用户通过打金收益来偿还 NFT 分期付款的利息和尾款。

当下，我们计划推出 xBank Legend 计划，通过这种方式探索新的 NFT 发行方式，帮助艺术家和初级社区起步，同时吸引新的用户。该计划为项目方提供先买后付方式，无需提供 ETH 流动性，以获得新用户。

BendDAO：

我从 2020 年底开始买 NFT，将近三年的时间，我目前看到了一个市场的信心度其实还是非常不足的，持续的阴跌局面令人痛苦。市场的不断下跌和供给过剩导致 NFT 市场萎缩。

不止于此，再加上之前 FTX、LUNA、Multichain 等事件影响，BendDAO 其实也更加明确了在现阶段我们需要做的事情，一个是用户资产安全性保障，这个问题可以拉到最高的层级来讨论，所以 BendDAO 其实主打的就是安全性的方面，不追求速度，而是确保安全的前提下才推出产品。我们不是着眼于现在，而是更加关注未来，如果 NFT 市场有起色，有更多人加入以后我们在现在持续建这建设才会有价值，到那一天才会开花结果。

BendDAO 在做的另一件事就是持续建设社区。定期在 Twitter 上举办活动，赞助实体活动，加强与社群的沟通。熊市是与社群建立强联系的好时机，社群在低迷时仍坚持，将成为未来牛市时的主导力量，与社群建立关系和共同成长是 BendDAO 当前的重要任务。

ParaSpace：

ParaSpace 的长期目标是将未来金融带给十亿用户。我们是 NFTFi 领域中第一个采用 Cross Margin 的借贷协议，不仅仅局限于 NFT 借贷，我们还在努力拓展跨链借贷领域，目前还新增了 Arbitrum、Polygon 和 Zksync 等链的支持。

从数据角度来看，NFTFi 赛道的市值约为 2 亿美元，涵盖 26 个 NFTFi 项目。头部项目 BendDAO 和我们的 Paraspace 占据了整个 TVL 的 60% ，前六名项目占据了总 TVL 的 95% 。尽管近两个月市值缩水近 70% ，但我们依然坚信这只是爆发前期的一种状态，市场具备增长潜力。相较加密货币市场，NFT 和 NFTFi 市场的增量潜力可能更大。未来的 NFT 领域将有许多新机会，例如身份生态系统、效用型 NFT、Financial NFT、RWA NFT，以及衍生品市场等，这些领域都具备巨大潜力。

目前，Paraspace 正在为 2.0 升级做准备，我们旨在为用户提供更出色的Web3金融解决方案。无论是 NFT 还是 DeFi，我们希望成为通用的借贷协议。未来甚至可以让用户使用现实生活中的资产进行质押借贷，获取所需的任何资产。

此外，我们非常赞同 BendDAO 提出的观点，即在熊市中与社区站在一起非常重要。我们也在这方面做出了许多努力。我们不仅在线上举办各种 Space 活动，还会持续地开展线下活动。我们在 7 月份赞助了 BAYC Japan 活动，摒弃与 YugaLabs 的 CEO 进行了深入交流。接下来在 9 月份的新加坡 Token 2049 活动以及 10、 11 月份在香港的活动中，Paraspace 将发起一系列周边活动。

在熊市中，NFT 市场讨论最多的问题之一是流动性。像 xBank 提出的 BNPL 模式，我认为是一个很好的解决方案，尤其适用于在熊市中进行抄底操作。在 BNPL 方面，Paraspace 也非常灵活方便使用。另外，之前 Paraspace 出现清算的情况，实际上也是一个抄底的信号。

LNEA：

在熊市的背景下，我们意识到了 LENA 项目的潜力。我们的目标是为不同的 NFT 系列提供服务，包括市值较低或波动性较大的 NFT 系列等。我们注意到在熊市中，许多 NFT 项目都面临着问题，而我们希望通过这些事件能够对未来发展有更好的应对策略。

我们认为，当市场疲软时，是一个测试产品承受市场压力的好时机。例如，像 Azuki 事件，市场可能会出现巨大的波动，这考验了我们的清算协议是否能应对市场下跌。我们需要检查协议中的风险限制和风险控制是否足够，这些都可以在熊市中得到验证。在产品规模较小的时候，我们有机会进行优化，以便在未来市场扩大时能够更加自信地为客户提供服务。

我们也谈到了 BNPL，这是一个能够降低多样性门槛的工具。它允许用户通过最终的支付服务来提高自己的杠杆，从而在相对平静的市场中获取更大的利润。

在这个市场相对疲软的时期，我们认为这是一个扩展的绝佳机会。我们正在深入了解自己的产品定位，寻找改进的机会，找到更多的可能性。LENA 的愿景是有远见的，就如讨论的 ERC 6551 等协议，以及关于 RWA 的可能性，我们考虑如何将这些资产转化为 NFT，为客户提供服务。这是我们团队在熊市中努力追求的目标，即更深入了解可服务的市场。

LittleMami：许多 NFT 项目缺乏长期的内在价值，未来各位是否想引入其他 NFT 项目入驻平台？对他们有什么要求或者期待呢？

xBank：

我们讨论的主题是 NFT 项目缺乏长期内在价值。我们正试图探索一些新的玩法，比如赊销，项目方提供 NFT 供销售。当然，这些适用于开盘阶段或后期的特定阶段。同时，我们可以回顾一些比较成功的 NFT 市场项目，如 looks rare，它首创了“list to earn”的概念，用户可以在平台上列出 NFT 并赚取代币。随后，blur 推出了“bid to earn”和“lending to earn”的玩法。这些初衷都是为了提高 NFT 的流动性，因为市场需要买方和卖方来促成交易。

那我们呢？我们的代币经济尚未公布，而且我们正处于熊市中。尽管如此，我们不断地完善产品，同时也希望与更多 NFT 项目方共同成长。我们目前正在尝试设计一种新的模型，后续会逐步公布细节，正在不断地进行实验和尝试。

关于是否考虑引入其他 NFT 项目，以及对它们有何要求或期待，我认为在这个 NFT 市场能够生存下来的项目都是好项目，无论是国内的小幽灵，还是海外的各种项目，如猴子、Zuki，只要能够在这个竞争激烈的市场中生存，都是值得期待和合作的项目。

BendDAO：

虽然我之前提到在短期内看不到市场信心上升的机会，但我长期来看仍然非常看好。我之所以持有这个观点，是因为我认识了一个在 NFT 领域已经玩了好几年的朋友，大概有五六年了，他告诉我他在玩 NFT 之前并不认识很多人，但是一旦开始涉足 NFT，他认识了几百个人，而且这些人一起协作做了一些事情。他认为这种社群共识非常强大，我个人也深信不疑。

事实上，NFT 的核心就在于共识的建立。众所周知，建立共识是最困难的部分，就像比特币和以太坊之所以具有高价值就是因为有共识存在。对于 NFT 来说，拥有一致的共识是非常难得的，需要一开始就有数千人支持某个理念。共识建立后，文化才能长久传承。因此，我认为 NFT 的未来充满希望。

另外，价格只是客观数据之一，我认为还有其他很多数据值得考虑。例如，项目方在社群中的活跃度，他们在 Twitter 上的发文频率，社群推广的次数，举办活动的频率，以及与社群互动的次数。我认为这些数据更客观，因为 NFT 的价值不仅仅是某段时间的价格，而是一个长时间滚动的社群参与热情，展示出其真正的价值，这才能被称为蓝筹。

举个例子，我们都熟悉的企鹅，最初地板价甚至不到 1 个 ETH，但是他们的社群一直非常活跃。即使项目方发生变动，社群依然紧密团结，逐步将地板价提升。因此，地板价当然是一项重要的客观数据，但社群的参与度、举办活动的次数、社群凝聚力等数据同样值得考虑。未来，其他 NFT 项目可以将这些数据纳入考虑价。

然而，我认为只要社群存在，项目方积极参与，总会有回报的一天。对于其他 NFT 项目的要求或期待，我希望他们能够多与 BendDAP 合作。我们持有开放的态度，希望与更多社群、NFT 项目以及团队合作。在熊市中，前行并不容易，但我们一起努力，渡过难关。我们也会继续支持更多优秀的 NFT 项目。

ParaSpace：

ParaSpace 在引入其他 NFT 项目时有一些标准，主要包括以下几点。首先，我们会看一个项目是否符合蓝筹标准。最主要的标准之一就是项目的交易量，我们会考察过去一段时间内项目的交易量以及成交金额。此外，我们还会关注项目的价格稳定性，即过去一段时间内价格是否保持在一定的水平上。

另外，我们会评估社区的共识程度，看项目在社区中是否有强烈的共识。还有项目的持有者人数也会被列入考虑因素之一。这些要素都会被综合考虑，判断一个项目是否适合作为蓝筹项目入驻我们的平台。

在 Paraspace 上，我们已经有一些项目，比如像 Yuga 的那些虽然交易量很大，但价格并不是很高，我们会将其纳入我们的平台。此外，我们也非常欢迎其他有潜力的蓝筹项目与我们合作沟通。我们希望能够共同探讨是否存在合作的机会，以共同发展。

LENA：

我认为项目入驻这个问题实际上在我们这样的平台来看，并不是一个非黑即白的问题。因为我们可以通过多种方法来调控风险，例如维持保证金、借贷利率以及其他风险基准的调整，以提供更高的审慎性。
