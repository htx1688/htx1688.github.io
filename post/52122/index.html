作者: daniel@footprint.network

熊市之中，PlanetIX 成长为最强的 Web3 游戏，在 Polygon 网络上独占鳌头。而其开发团队深度使用了 Footprint Analtics 的零代码数据分析平台和-GameFi 的数据 API 来提升用户的游戏体验。

近日，Footprint 与 PlanetIX 的 CMO Melly 一同讨论了 GameFi 的现状和未来，以及其他项目如何从中获取宝贵经验。

您可以阅读以下总结或收听完整的 Twitter Space，以了解如何开发运营大热的区块链游戏项目。

Footprint provides Planet IX with several key metrics, such as Asset Daily Trading Volume

Web3 营销中的利器：空投

空投是区块链项目中常用的营销策略，是指通过免费分发代币来引起潜在用户的关注，激励社区参与并奖励早期采用者的过程。空投通常会针对满足特定条件的用户进行代币分发，例如持有特定代币数量或完成特定任务的用户。代币通常会直接发放到用户的钱包中，十分简单高效。

尽管空投已成为激励用户参与项目的重要营销策略，但一些人利用机器人和其他方法自动获取奖励，即使这些地址并不会为项目或者其社区做出贡献。

Footprint’s Airdrop Effect dashboard shows the key metrics for a given airdrop campaign.

Melly 非常擅长空投营销，在他看来，虽然有人会利用规则刷空投，但情况并没有想象中那么糟糕。PlanetIX 曾发放过价值约六百万美元的最高额空投。

“所有的一切都记录于链上，如果你试图单单通过空投来获利，其实也会被定义为活跃用户，对生态系统的影响可能仍然是积极的。” 但他补充道：“我们正在努力寻找更多真正的玩家。例如，我们正在通过激励玩家，让那些使用独特钱包账户的用户在游戏中获得更多收益，以此来吸引真正的玩家参与游戏。”

总的来说，相比于看空投对单个项目的影响，看行业其实更有价值。空投教会了 Web3 参与者：一切行为都是是有价值的。

“大多数 Web3 参与者明白一个道理，如果保持活跃，关注最新的协议和最新的领域，你将很可能会得到空投奖励，”Melly 说。“当你回顾 DeFi Summer，很多人得到了奖励，后来的 NFT 领域也是如此，很多用户得到了空投，领到了代币。”

AI 如何改变 Web3 游戏

AI 将改变 Web3 游戏，其中一个场景是：AI 将使链上的机器人账户大量存在。有了 AI 的加持，未来还会有更先进的机器人出现。幸运的是，我们可以通过链上和链下的数据来识别机器人。比如，机器人通常全天候运行。如果你查看钱包的热度图，可能会发现一些钱包在任何时间都保持活跃，或者在可疑的时间段内活跃，这些账户可能就是机器人账户。通过综合链上和链下的数据，能够辨别机器人并深入了解用户。

Melly 说：“当进入这个小众且在 Web2 中面临高度竞争的领域时，你需要尽可能提高效率。AI 不会消失，它只会越来越重要。”他说：“我相信未来我们会更多使用 AI。”

如何创造成功且可持续的游戏

开发游戏时，代币经济、游戏玩法的设计和用户留存的策略取决于项目所处的阶段及其长期目标。PlanetIX 是一个从 Web2 过渡到 Web3 的项目，因此在开始建设时并不需要寻找新用户。

随着 Web3 游戏的发展，许多初创公司和项目也在尝试从 Web2 吸引玩家。

Melly 表示：“与其拘泥于数据指标，你更应该考虑如何吸引用户。如果你仅仅希望从 Web3 用户群中获取用户，可能会面临很大困难，因为目前 Web3 游戏玩家的数量并不多。”

“退后一步，我认为更重要的是，我们应该思考Web3社区如何共同努力，将更多来自Web2的用户引入 Web3 的世界。这样我们才能实现共同发展，而不是每个人都争夺有限的资源。”

对于 PlanetIX 来说，吸引这些新用户的关键在于让他们在链上世界感到舒适和安全。对于新手来说，链上世界可能充满了不确定性和风险。

他说：“初入区块链世界，可能会遇到很多不愉快的情况。让这些新用户感到舒适，将会产生更好的效果。”

“我们相信：付出大量努力让新用户感到舒适，他们就会选择留下。比如，我们的增长负责人在东南亚参加了社区组织的活动，与他们建立联系……信任和链接是非常宝贵的。”

Web2 和 Web3 营销的区别

Melly 表示：“Web3 做广告的方式与 Web2 完全不同，因为 NFT 这类数字资产相关的广告内容，由于种种原因很难存在于主流广告渠道。”

“所以我们需要游击战术，需要新颖的营销方式。我认为最好的方法是提升社区的实力以及与合作伙伴建立的关系。”

“由于目前 Web3 领域用户和项目规模都相对较小，提高知名度并不特别困难。难的是给新用户留下深刻的印象。“如果你的产品不够好，很难吸引用户。”

Melly 补充说，尽管 Web3 市场人员面临限制，但这个领域的新工具也不断推出，并借鉴 Web2 营销经验。

如何避免代币经济成为庞氏经济

Footprint 数据显示，Web3 游戏中有 90% 在前30天内就会变得不活跃。

这可能是因为游戏质量不佳，设计不完善，或者代币经济不合理，或者以上几个因素同时存在。其中，不可持续的代币经济是一个重要原因。

对于 Planet IX 来说，构建一个可持续的生态系统就在于维持不同类型玩家之间的共生关系。

Melly 表示：“有一些玩家非常专注于挖矿，他们通常生活在低收入地区，有很多时间可以投入其中。他们并不需要赚取太多财富才认为有价值。还有一些资金较多的玩家，虽然时间不那么充裕，但是有更多的资金可以通过购买数字资产来跻身顶级阵营。”对于 Planet IX 来说，关键在于平衡这些不同类型的用户。
