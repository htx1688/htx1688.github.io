原文作者：Andrew Hayward

原文编译：PANews

5 月 3 日，Sui Network 主网正式上线，但在这一里程碑事件之前，一个活跃的NFT社区已经在这个新兴区块链上扎根，不少项目发行的 NFT 不仅产生了数十万美元的交易额，也吸引了不少用户的关注。

虽然以太坊和Solana区块链上仍然拥有当前最活跃的 NFT 社区，但 Sui Network 也吸引了不少 NFT 交易者。作为 Mysten Labs 公司开发的Layer 1 ，Sui Network 无疑是近期最受期待的区块链项目，甚至一些人将其称为是“Solana 杀手”，不过 Sui 究竟能否能持续支持一个繁荣的 NFT 空间还有待观察，毕竟像Aptos和NEAR等网络也都曾有过热度爆棚又快速降温的经历。

典型案例： “未能按预期运作”的 Sui NFT 市场 Clutchy

5 月 4 日，NFT 项目“Fuddies”在 Sui Network 上启动投放，该项目由 10, 000 张 PFP 组成，每张都描绘了各种姿势和风格的卡通猫头鹰，也是迄今为止 Sui Network 上发行量最大的 NFT 项目。然而就在 Fuddies NFT 完成铸造后，其托管市场 Clutchy 却出现了技术问题，引发了 NFT 交易者和Web3建设者的极大不满。

Clutchy 其实是 Mysten Labs 公司力推的 Sui 链上新贵 NFT 市场，按照计划，他们将以 95 SUI（约 125 美元）的价格向添加到发布前许可名单的用户提供 Fuddies NFT，公开发售则以荷兰式拍卖的形式启动，从 300 SUI（约合 395 美元）的初始价格逐渐下降，在售罄之前达到 150 SUI（近 200 美元）。

但问题是，Clutchy 的二级市场未能在 Sui Network 上按预期运行，导致大量希望抛售 Fuddies NFT 以快速获利的买家很快大失所望。在遇到问题之后，Clutchy 已经推迟原计划下周三（5 月 10 日）的另一轮 NFT 铸造活动，还宣布关闭 Launchpad，并向 Sui 社区发布了致歉信，坦言 Beta 发布阶段很不顺利并且向创作者和用户致歉。

不过 Clutchy 很快又重启了二级市场，虽然仍不清楚其问题是否已得到充分解决，Fuddies NFT 的交易也再次启动，据称在 24 小时内产生了 605, 000 SUI（约合 792, 500 美元）的交易额。

Fuddies 随后也在社交媒体发布数据称，恢复交易后的 24 小时交易额超过了同一时期的Polygon链上交易总额。

Sui 链上 NFT 项目销售表现两极分化明显

现阶段，Sui 链上一些比较火爆的 NFT 项目似乎都是从“衍生仿盘”入手，加密社区其实不难发现，作为迄今为止 Sui Network 上规模最大的 NFT 项目 Fuddies 其实和以太坊上同样以“猫头鹰”为主题的 NFT 项目 Moonbirds 非常相似，而在 Clutchy 上铸造并售罄的另一个 NFT 项目 Suishi，则像是 Cool Cats 和“胖企鹅”Pudgy Penguins 的结合物（如下图所示）。

事实上，Sui 链上 NFT 项目销售表现两极分化明显，比如与 Fuddies 接近 80 万美元的 24 销售额相比，Suishi 自铸造以来仅获得了价值约 10, 900 SUI（约合 1.43 万美元）的二级交易额。

另一方面，Clutchy 并不是唯一活跃的 Sui NFT 市场，去年秋天在 Aptos 区块链上推出的 NFT 市场 Souff l3也扩展到了 Sui Network，该平台上的头部 NFT 项目也都没有引发任何销售轰动，比如 Souff l3上最大的 Sui NFT 项目是“Sui Trident”，上线后 24 小时内只累积了价值约 6.5 万美元的交易额。

SUI 链上 NFT 场景能持续下去吗？

在 Sui Network 主网发布之前，人们对其链上 NFT 生态场景期望很高，Clutchy 创始人兼首席执行官 Jacob “Dizzy Lizzy” Lawless 当时曾直言不讳地称他“看到来自影响者、投资者、创作者和旁观者的大量兴趣”。

实际上，以太坊、Solana 以及最近比特币 Ordinals 上都出现过“淘金热”，这些区块链平台上的早期 NFT 价值飙升时，创作者总是渴望率先在新链上推出自己的项目。同样地，精明的数字藏家也会尝试抢购一些早期推出的 NFT 资产，因为这些 NFT 的价值很可能会在未来获得大幅提升。

举个例子， 2022 年 10 月 Aptos 首次亮相后，其链上 NFT 交易额激增，与此同时 Solana 链上 NFT 销售额则出现缩水，当时 Aptos 主网启动一周后 Aptos Monkey NFT 项目上线，短短 24 小时就产生了价值约 230 万美元的巨额交易，但热潮很快消散。现在，同样的状况现在发生在了 Sui Network 上，虽然 Sui 还没有达到 Aptos 当初那种高度。

那么在 NFT 场景中，Sui 生态是否真的会重蹈 Aptos 的覆辙呢？未必。

首先，Sui 生态创作者非常清楚一点：相比于其他区块链，Sui Network 能够实现更多、更高级的功能，比如实时更新的动态 NFT，这种用例能给区块链游戏带来许多优势，游戏内的武器或盔甲会随着时间推移而不断升级改进，而且 NFT 也可以在这种“动态”过程中变得更有用或更有价值。

从这个角度来看，游戏最终可能会成为推动 Sui NFT 场景发展势头的最大驱动力之一，据悉 Sui 生态下的一系列早期游戏：如《守望先锋》风格的英雄射击游戏 Bushi、一款仿《行尸走肉》的游戏、以及一款 Solana 移植版的坦克游戏 Panzerdogs 都将在未来几周内推出。Web3项目顾问 Dabblerer 解释说：“Sui 区块链上正在进行很多有趣的实验，最终会带来一波 NFT 和DApp创新，这些创新吸取了此前区块链 NFT 生态中不少失败案例的经验教训。”

其次，Sui NFT 场景还可以拓展其他潜在用例，比如模仿以太坊域名服务ENS的域名项目“Sui Name Service (SuiNS)”，该项目将域名应用做了进一步升级，把域名与 NFT 和智能合约相关联，继而使用户更信任、更安全地进行交互。

总体而言，Sui Network 的技术供了一个创新平台，它采用的 Move 语言降低了创作者通过动态 NFT 进行创新的界限，继而解锁出更多类型的用例和更引人入胜的应用及体验。当然，由于 Sui 主网上线时间较短，许多创新项目和应用能否长时间生存下来还有待观察，但有一点可以明确：Sui 入场后，NFT 市场的良性竞争性势必将会进一步增加，对整个生态来说其实利大于弊。
