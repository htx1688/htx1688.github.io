本周，有 6 个将迎来解锁的项目值得关注。

其中，YGG、OP、ACA、GAL 将面临大额解锁，1INCH、AGIX 则面临常规解锁。

以 1INCH 为例，其市值高达 3 亿美元，本次解锁数量仅约 2 万美元的金额，对于其市值而言，如此量级的解锁抛压可以忽略。AGIX 的解锁规模也并不大：SingularityNET 流通市值约为 2.8 亿美元，本次解锁代币价值约 237 万美元。

Yield Guild Games

项目官网：https://yieldguild.io/

官方推特：https://twitter.com/YieldGuild

本次解锁数量：16, 872, 453 

本次解锁金额：（约） 266 万

YGG 是此前曾火爆一时的游戏公会 DAO 项目，公会买入 GameFi 内的道具，并将其租借予玩家，抽取玩家的打金收入。

YGG 代币分配如下：

其解锁信息表明，本次解锁代币中超过一半（976 万枚）均归属于投资者，其余则为社区、金库、顾问。

在最近一段时间，YGG 始终在频繁解锁。YGG 也有大量的代币未曾流向市场，目前有 72% 的代币仍在锁定之中。从释放模型来看，本次解锁并不算特殊。相似规模或量级的解锁，每月均在发生。且在未来并不会发生量级与此类常规解锁不同的更大规模解锁。投资者拥有充分的历史数据可供参考。

本周，YGG 代币将解锁约 1687 万枚，约合 266 万美元。目前 YGG 流通数量仅 1.85 亿枚，本次解锁数量约为流通量的 9.1% ，占总供应量的 1.7% 。

Galxe

项目官网：https://galxe.com/

官方推特：https://twitter.com/Galxe

本次解锁数量： 586 , 666 

本次解锁金额：（约） 76 万美元

Galxe 是一个帮助Web3项目建立社区的平台。Galxe 可通过奖励计划帮助项目方获取用户，用户可通过完成指定任务获取项目方的奖励。此外，Galxe 还为用户、项目方提供了数字凭证功能。该项目旨在构建一个开放和协作的凭证数据网络，帮助Web3开发人员利用凭证数据构建更好的产品和更具吸引力的社区。Galxe 还拥有解决方案、多链身份 Galxe ID 等产品。

GAL 代币是 Galxe 生态系统的支付和治理代币。开发者使用 GAL 支付平台费用，如 Galxe Oracle Engine 和 Galxe Credential API 的查询费用等。

代币分配：按照 GAL 代币经济学，本次解锁的代币来自团队份额。

目前，GAL 总供应量的 57% 仍在锁定之中，GAL 流通市值约 6051 万美元，但 FDV 已高达 2.6 亿美元。

GAL 的解锁曲线呈现阶梯状。本次解锁并仅为一次约 40 万枚的小额解锁，但在 8 月，GAL 将迎来一次约 800 万枚代币的大额解锁。

Optimism

项目官网：https://www.optimism.io/

官方推特：https://twitter.com/optimismFND

本次解锁数量： 24, 159, 191 

本次解锁金额：（约） 3237 万美元

Optimism 是现已上线的以太坊 Layer 2 解决方案之一，也是第一个与 EVM 兼容的 Optimistic Rollup 路线的 L2。Optimism 与以太坊并行，能够在继承以太坊安全性的同时大规模处理交易。

本次释放的 2400 余万枚代币中，超 1275 万枚均归属于核心贡献者。其余 1140 万枚则归属于投资者。

OP 的解锁曲线较为特殊，在 5 月底，OP 刚刚完成其代币释放历史上最大规模的代币解锁。

从解锁数量来看，本次解锁占流通量的 3.7% ，占总量的 0.56% 。尽管其数量已经较为可观，但与上次大额解锁相较，其数量仍差距甚巨。

Acala

项目官网：https://acala.network/

官方推特：https://twitter.com/AcalaNetwork

Acala 是波卡生态的老牌项目，旨在打造波卡的去中心化金融网络和流动性中心。该项目为可扩展的、与以太坊兼容的智能合约平台，并针对 DeFi 进行了优化，具有内置的流动性和现成的金融应用程序。凭借其无需信任的交易所、去中心化稳定币 (aUSD)、DOT Liquid Staking (LDOT) 和 EVM+，Acala 让开发人员可以访问以太坊的核心功能和 Substrate 的全部功能。

ACA 本次解锁金额较大，约占流通市值的 3.9% 。目前，超过 61% 的 ACA 仍在锁定之中等待解锁。若以总量计算，本次解锁占总量的 2.7% 。

但从解锁历史来看，本次解锁与过往历次解锁并无二异。与本次解锁同类规模的解锁，每月均有一次。投资者有充足的历史数据可供参考。
