自“央数藏”在9月16日正式上线，国内央媒几乎全员入局数字藏品市场，这对于国内数藏市场的发展来说无疑是个大动作，有着风向标作用的国家媒体数藏平台是否能成为新一轮的数藏顶流，也是很多人关注的热点话题。

截至目前，共有9家央媒背景的数字藏品平台：灵境人民艺术馆、新华数藏、时藏、豹豹青春宇宙、光明艺品、人民科技数字藏品、中网艺云、光明数藏、央数藏。 （排名不分先后）

灵境人民艺术馆

上线时间：2022年1月25日

背景：人民网

上链信息：人民链（联盟链）

入口：微信公众号菜单、“人民网+”APP搜索

“灵境人民艺术馆”是人民网推出的数字藏品平台，于2022年1月份上线，上线以来共发行了14个系列的数字藏品，共60款。

“灵境人民艺术馆”定位于助力普及大众艺术作品，为艺术爱好者提供更多展示空间，为艺术品数字版权保护、数字出版、数字转化、数字认证(区块链)等提供服务，用数字技术服务大众艺术，促进大众艺术品市场的健康发展，让艺术走进寻常百姓家。

平台上线后曾发售过《人世间》梁晓声原著授权数字藏品、阎立本《步辇图》王希孟《千里江山图卷》等古代名画数字藏品以及以“妙造自然”为主题的齐白石数字水墨。在音乐方面，杰威尔音乐、薄盒Mints平台在9月份联合推出“周杰伦限定珍藏DEMO空间”音乐元宇宙活动，“灵境人民艺术馆”开设专属通道直达“周杰伦的元宇宙空间”。

（数据来源：ForeChain NFT Database）

从平台内提供的数据来看，“灵境人民艺术馆”发行的数字藏品中有86.67%的藏品发行价在100元以下。根据目前收集到的数据，平台共销售了240,975个数字藏品，共计6,840,701元，其中有20款数字藏品售罄。（数据来源：ForeChain NFT Database）

新华数藏

上线时间：2022年4月5日

背景：新华网

上链信息：新华·星火· 链网（联盟链）

入口：新华网App搜索、公众号菜单

新华数藏是由新华网推出的数字藏品平台，于今年4月份上线，目前共发行了100款数字藏品。

新华数藏的定位于用数字藏品助力于公益事业。在4月29日和5月23日，新华网发行了两期“乡村振兴”数字藏品并将销售收益都会捐赠至中国足球发展基金会用于支持“县域青少年快乐足球公益行”活动，这也是央媒首个基于数字藏品的公益助力计划的启动标志。该计划将围绕公益领域发挥独特功能优势，为乡村振兴及公益领域赋能，助推乡村振兴、应急救援、社会救助、妇女儿童保护等领域的建设和发展。

新华数藏的数字藏品多聚焦在“公益”和“乡村振兴”这两个方面，因此平台内的数字藏品多与基金会或博物馆联名。

“灵境人民艺术馆”是人民网推出的数字藏品平台，于2022年1月份上线，上线以来共发行了14个系列的数字藏品，共60款。

（数据来源：ForeChain NFT Database）

从平台内提供的数据来看，平台中有30%的数字藏品为免费赠送，其余的发行价格也在百元以下。根据目前收集到的数据，平台共销售了104,606个数字藏品，共计221,1767.9元，其中有37款数字藏品售罄。（数据来源：ForeChain NFT Database）

时藏

上线时间：2022年3月17日

背景：中国搜索（新华社、人民日报社等旗下）

上链信息：媒体融合链（联盟链）

入口：时藏APP、微信公众号菜单

时藏是新华社旗下“中国搜索”打造的数字藏品平台，于今年3月上线。据官方介绍，时藏体现跨域时空、铭记永恒的美好希冀，初衷是用区块链见证伟大时代，记录重要历史时刻，珍藏时空美好瞬间，弘扬中华优秀传统文化，增强民族文化自信，因此把“中国好故事”汉服萌娃形象作为首发藏品。

（数据来源：ForeChain NFT Database）

平台上线以来共发行70款数字藏品，发行量都在1000份以上。从平台内的数据来看，发行价都低于30元，平均发行价是15.77元。

豹豹青春宇宙

上线时间：2022年6月6日

背景：中国青年报

上链信息：树图链（公链）

入口：微信公众号菜单

豹豹·青春宇宙的官方定位是官方优质文化产品的发行平台，青年人创新、创造、创效的平台，于今年6月上线。平台授权由中国青年网具体运营，以树图链为技术底座，储备了空投、拼图、盲盒、预售、转赠等主流玩法及功能，该平台的首款数字藏品“航天青年数字徽章”由中国青年报社和国家航天局新闻宣传中心联合推出。

（数据来源：ForeChain NFT Database）

光明艺品

上线时间：2022年6月15日

背景：光明日报

上链信息：BSN（联盟链）

入口：光明艺品APP、微信公众号菜单

目前，平台共上线17款数字藏品，发行量均在2000份左右，最高发行单价是29.9元，均价为16.61元。光明艺品是光明日报旗下的数字藏品平台，于今年6月上线，首发的数字藏品是《光明日报》创刊号。从平台内发行的数字藏品可以看出平台的定位是宣传主旋律精神、时代记忆、传统文化的继承与弘扬。因为是依托于央媒光明日报的，因此光明艺品内的数字藏品除了对IP资源进行推广以外，还肩负着更多文化宣传的职责。

（数据来源：ForeChain NFT Database）

平台上线后共有48款数字藏品，目前还没有售罄的款式。截止目前，成交量为71,083个，成交额为6,682,699.3元。从单品发行价格来看，大部分位于低价位区间，发行价格为39.9——89的之间。

人民科技数字藏品

上线时间：2022年6月19日

背景：人民科技（人民网旗下）

上链信息：BSN（联盟链）

入口：“人民智作”微信公众号菜单

人民科技数字藏品平台是以《人民日报》、人民网海量内容资源为依托，基于自有区块链技术，集存证、确权、鉴权、发行、售卖等功能为一体的数字藏品业务平台。平台上每一个藏品和用户信息都在人民可信链上存证，并映射唯一标识。平台以弘扬中华优秀传统文化、革命文化、社会主义先进文化为己任，在挖掘《人民日报》珍贵历史数据资源发行自有版权IP产品的基础上，还将与优秀行业伙伴合作，推出更多兼具文化内涵和收藏价值的数字藏品，打造权威可信的数字藏品平台，服务广大数字藏品受众。

平台发布的第一份数字藏品——《人民日报》头版数字藏品·新中国第一个特等发明奖授予袁隆平团队，藏品以公益的方式，面向公众免费限量发行5000份。

（数据来源：ForeChain NFT Database）

平台上线4个月共发行了18个系列的数字藏品，其中有2个系列全部售罄，平台已卖出96,860个数字藏品，销售额为1,616,628.1元。不同于其他平台，人民科技数字藏品的发行价较高，其中7个系列为免费赠送，5个系列的数字藏品发行价高于100元，平台内的平局发行价为81.72元。

中网艺云

上线时间：2022年7月22日

背景：中国网

上链信息：百度超级链（联盟链）

入口：微信公众号菜单

中网艺云是由中国网发起并联合海内外艺术大师、IP资源方、设计力量参与共创的数字藏品平台，从对海内外受众讲好中国文化故事的角度出发，将中国优秀传统文化内容借助设计构思、数字表达、时空交织、场景转换、区块链技术等科技手段创新表达。在如何讲好中国故事，特别是讲好中国传统文化故事上进行了大胆创新。

高度写实与大胆写意的巧妙结合，使主旋律内容充满了数字艺术的创新魅力，结合数字藏品的运营玩法，使传统和创新能够有机交互。

中网艺云数藏平台以主旋律精神、时代记忆、中国传统文化类数字藏品为主要抓手，推动与促进文创载体表现形式的多样化，拓宽文创边界，助力艺术创作，从而促进文化记忆传承，促进文化市场有序发展，促进文化产业高质量成长。

平台发行的第一款数字藏品是“中国精神”系列数字藏品，共1000份，900份用于免费发行，100份用于定向空投。

光明数藏

上线时间：2022年9月9日

背景：光明网

上链信息：星火· 链网（联盟链）

入口：微信公众号菜单

光明数藏是由光明网和北文中心联合打造，于今年9月上线并首发《光明数藏上线臻藏纪念券》数字藏品。光明数藏是为各级政府文化部门、非物质文化遗产传承人打造的区块链数字藏品发行平台，也是数字藏品爱好者收藏文博、非遗类数字藏品的重要渠道。

光明数藏使用工业和信息化部、中国信通院主导研发的区块链技术的新型基础设施、国家主链“星火· 链网”作为底层基础链；以国家信息中心、中国移动、中国银联主导的BSN-DDN联盟链作为备用链，以光明云媒自主研发的光明数藏系统为依托，按照等保三级标准建立。

上线数字藏品利用区块链技术，致力于以数字艺术链接文化领域，每个藏品在区块链都有链上唯一标识，其不可复制、不可篡改和一定范围内的可流通属性，赋予了它独特的收藏价值和经济价值。

（数据来源：ForeChain NFT Database）

目前，平台内发行的藏品较少，共8款。8款数字藏品共计26,981个，都已经售罄，共计销售额为748,000元。平台内的数字藏品发行价都属于低价位，平均发行价为26.18元。

央数藏

上线时间：2022年9月16日

背景：央视网

上链信息：星火· 链网（联盟链）

入口：央视影音App搜索

央数藏是由央视网孵化建立的数字藏品发售平台，9月16日正式上线并首次发售了2022个创世数藏Y，瞬间售罄。9月19日15时，“央数藏”上线的2022个创世数藏S，不到一分钟即售罄，标价为128元。“央数藏”于9月20日发售了一款新的创世数藏C。官方介绍称，“央数藏”目前不支持二级市场交易，但是支持30天的转赠功能。

央数藏三款创世数字藏品分别对应不同的版块，创意出发点也各有不同：

“Y”对应板块：“小央艺术”，主要包含：国内著名IP及品牌、各级政府、国内地标性建筑、国家名片、国家政策、大型活动、大型赛事等。“Y”以“书籍”作为开篇，以“文字”作为结尾，通过书籍的翻阅展现历史的流转，通过文字、偏旁和部首的罗列展现中国文化的博大精深，因此该件数字藏品的创意在文字、偏旁和部首上的选择颇为考究。

“S”对应板块：“数字潮流”，主要包含：潮玩、数字文创、电影、音乐等。从北京到上海，从中央广播电视总台地标建筑到“龙”头列车，从卫星到宇航员到探月，“S”的设计，将地标建筑、中国文化融入其中，加上复古的录音机和液晶显示屏，以更加超酷的形象和生动的文化演绎，重新定义数字藏品的真实价值。

“C”对应版块：文化非遗艺术，主要包含：博物馆及文物、非遗艺术家及作品、现代艺术、艺术类展览展示活动。“C”的设计融入了圆月、古建筑、熊猫、竹子等对于中国人来讲有特殊寓意的元素。以藤蔓、绿叶和花朵来代指繁花似锦的中国文化；竹节象征央视网和民族文化的长青不败；熊猫来表达中国文化活泼而又不失灵动的一面；晶莹剔透的露珠来形容历史长河对于中国文化传承的孕育和滋养。

（数据来源：ForeChain NFT Database）

上线后平台共发行了60款数字藏品，总计共卖出118,901个，交易额为8,757,188.5元，其中有37款全部售罄，占总数的61.67%。较其他平台，央数藏的发行价格较高，除了1款之外，均超过50元，多集中在50元—100元的价格区间，平均发行价格为73.73元。

国家队的独特之处

由于不同的政策和监管的要求，NFT概念在进入国内后更多的被成为“数字藏品”，其金融属性被淡化的同时文化属性得以强化。例如，2021年支付宝联合敦煌美术研究所发布的“敦煌飞天”和“九色鹿”，都是有着强烈的文化属性。作为传统媒体的央媒拥有丰富文创资源和ip，因此能够推出更具有国家特色和传统文化的数字藏品。但又不同于互联网公司推出的平台，国家队的数藏平台要更多地担负起文化传播的责任，因此这些平台的选品更多地是侧重在弘扬主旋律、重温时代记忆、继承民族精神和创新传统文化，比如央数藏的首批YSC系列藏品就是分别侧重在“小央艺术”、“数字潮流”和文化非遗艺术，“时藏”作为新华社旗下的中国搜索推出的平台发行了《半月谈》1980年创刊号数字藏品，光明日报社旗下“光明艺品”的首发藏品也正是《光明日报》创刊号。

因为肩负着文化宣传的责任，因此这些平台的藏品价格定位都较低，大多集中在百元以下的价位，且很多藏品的发行方式都是免费投放，这样可以让更多的人参与其中。同时，因为这些平台有比较可靠的背景，也会有更多的需求，在发行数量方面也都较大，基本在千份以上，个别系列更是在万份。

在“玩法”的方面，央媒数藏平台也在努力接轨应用场景方面的潮流。“光明艺品”发行的毕加索数字艺术藏品的拥有者可获得空投对应的原版数字真迹一幅。“新华数藏”的“茶百戏”系列藏品的拥有者有机会额外获得《梦华录》周边实体产品一份。央数藏的创世藏品权益里增加了社区DAO的说明，持有者有机会成为央数藏DAO社群志愿者，配合社群管，对于有突出贡献的社群管理者和志愿者，央数藏将赠送精美礼品以及央数藏志愿者证书。

总结

目前，数字藏品还是一个比较新的行业，难免会有影响行业生态的行为出现，不仅会影响扰乱行业秩序，也会让大众产生不正确的认知和负面看法。越来越多的央媒平台选择入局无疑是对整个行业注入了一剂“强心针”，可以为这个正在发展的行业带来更多正面的影响，让更多人转变对数字藏品的负面看法，促进数字藏品行业朝着更加合规、更加健康的方向发展。
