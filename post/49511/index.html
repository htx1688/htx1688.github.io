金色财经记者Jessy

4月5日至4月8日，为期四天的以太坊黑客马拉松“ETHBeijing”在北京举行。此次以太坊黑客马拉松，由北京大学区块链协会和WTF Academy联合举办，获得了以太坊基金会ESP，Scroll，和Mask Network的赞助，总奖池达30000美元。

大赛设立了“open Research”、“Innovative Layer2 Dapp”、“Public Goods”三个赛道。在比赛结束时，共有55个项目提交。目前，在来自Scroll、币信、Gitcoin等机构评委的初步评选下，每个赛道不计排名选出了前五优秀项目。

最终，每个赛道会选出前三名，分别奖励5000、3000、2000美金。据了解，最终排名大概会在一周后公布。

比赛现场优秀国人项目Scroll在做项目介绍并和参赛团队进行交流

年轻的开发者们聚集在ETHBeijing

据北京大学区块链协会会长0xAA透露，此次以太坊黑客松，70%的团队都来自于各大高校。而在各大高校内，最活跃的区块链开发社群则是在各学校的区块链社团里。据统计，大约有四十多所高校都有现都成立了区块链社团。此次活动的举办方之一，便是北京大学区块链协会。

评委币信投资经理dongxuan印象最深的项目之一，便是一个来自北京大学深圳研究生院Blockchain DAO所提交的以太坊共识层中心化风险及优化方案。在黑客松比赛中，他最关注的是开发者对于赛道的协议层以及基础设施这些方面是否有自己的想法而这个项目让他看见了年轻的开发者对于以太坊基础协议层的存在的一些问题的思考和改变上的实践。该团队发现了目前火热的DVT技术（尤其是SSV）的重大缺陷，提出基于DVT技术的客户端共识规则协议改进方案并制作了EIP draft。

他也看到了可能会有商业前景的一些项目，比如项目GasLockR，这是一个去中心化的GasFi衍生品，能够帮助用户对冲Gas价格风险，还支持用户使用法币支付包月Gas费，项目方代为支付Gas费等场景，这能够更好地帮助用户进入web3世界，有着广泛的用户群体。

这次黑客松比赛，时间仓促，所以对于项目，评委们最看中的还是项目的创新性。短短几天内开发者团队一般只能做出一个Demo,想法是否新颖会很重要。

而在往常考察一个项目是否值得投资，dongxuan告诉记者币信会主要会参考以下几个因素：赛道是不是够大；产品是不是有创新，而不是简单的一个copy;这个问题是不是一定要用区块链技术去解决。

在产品之外，则要看团队的技术背景是否够，执行能力如何等等。平常的工作中，他时常是国内和国外的团队都看，他对比发现，国内的团队技术实力过硬，但是在叙事能力上会相对偏弱，对于市场的影响力也不如国外团队多。

和经验比较丰富的社会团队相比，大学生团队的项目还是相对青涩。但dongxuan很欣喜，因为他从项目的呈现上看到了年轻开发者们在区块链技术开发上的潜力。目前一些高校都开办了区块链相关专业，也开设了区块链相关的课程，高校老师们也会进行相关课题的研究。金色财经记者在比赛现场遇到的一队来自广州大学研二学生所组成的团队，他们在学校内的研究方向便是区块链安全。

比赛现场正在敲写代码的广州大学参赛团队

清华大学区块链协会会长Luna告诉记者，在清华大学内，就有类似“区块链和加密数字货币”课程的开办。清华大学区块链协会成立于2017年，无论是区块链协会内部还是校园内，真正在做区块链技术开发的同学都不少。平时协会不仅会举办自己的闭门会议对每周行业热点进行讨论分析、输出研报、请一些行业内人分享。对外也输出了区块链相关的课程，举办行业内活动。在北京，各大高校的区块链协会们们的联结也十分紧密， 已然成为一个北京Web3重镇。

清华大学区块链协会的会长Luna观察，大概从2022年开始，全国各地的高校开始陆陆续续成立区块链协会，相较于社会上成熟的开发者来说，高校内的技术开发团队虽然技术不太成熟，但环境对于开发者的容错率更高。同学们因为热情聚集在一起，目的也更加存粹。协会内也跑着各种项目，Luna现在深度参与的一个协会内的项目为socialFi方向。

以下为“ETHBeijing”每个赛道前五名的项目介绍：

Public Goods

SLOADS ：目标是准备给以太坊智能合约开发框架Foundry添加一个 feature，这样能够方便检索智能合约里面的所有 storage slot，特别是动态数据结构，如 Array，Map。基于此，开发者可以更加方便地深入探索链上智能合约的状态，比如查找某个 token 的所有持币地址。

PADO ：该项目致力于帮助个人用户获取他在币安交易所的账户总资产信息（包括现金账户和现货账户），并在链下完成隐私保护证明（资产金额大于一个阈值）。用户可以选择将证明结果编码成ethereum attestation提交到以太坊网络。项目主要优势之一是采用了交互式零知识证明(IZK)，相比于大家比较熟悉的zk-snarks/zk-starks等非交互式ZK技术，IZK在内存占用、处理布尔逻辑等方面具有巨大优势，非常适用于在用户终端直接部署zk-prover。

Puzzle3 ：这是一款通过智能合约交互实现的链上解谜、答题平台，让Web3 学习平台、区块链初创团队用户能够通过更直接、简洁、有趣的方式了解各自团队的产品。

Trust Bridge ：基于区块链的内容分享，内容发现和内容社交协议。总的来说，该协议提供了一个去中心化和透明的平台，用于发布和审查 NFT。无论你是内容创作者还是评论员，该协议都致力于成为分享你的作品和发现优秀内容的完美解决方案。

“NFC”（non fungible crush） ：一个完成NFT资产的同质化和同质化做市商做市逻辑的改变的协议，现在包括但局限于erc721，erc1155的非同质化可枚举资产。参考curve的token模型让NFT项目方跟DEX进行利益捆绑。

 Layer 2

GasLockR：该项目使用 Axiom，一个 ZK 协处理器，用于读取历史 gas 价格，以根据实时更新的金融模型提供可验证的正确定价的 GasFi 衍生品。GasLockR 可与其他协议互操作，可用作基础链上基础设施来构建协议和服务。从而降低Gas费。

NirVANA：一个基于ERC2535的模块化SBT发行工具，且支持合约部署后再次添加/替换/删除模块，目前已支持：DAO治理，社交恢复，ZK验证模块。

Signer：一个危险网站提醒插件。该插件基于一套名为 Truth Protocol 的链上协议，该协议致力于激励所有人参与危险网站的标记，以此降低危险网站库的维护成本及中心化共识非议。这些危险网站数据将会存储在 Scroll 链上，Signer 插件也将实时同步这些链上信息，在用户访问这些危险网站时给予最及时的提醒。除了服务于危险网站标记，Truth Protocol 还将有效应用于金融领域，构造出一种链上的去中心化的具有保险属性的 CDS（Credit Default Swap）。

Lightm：项目中文名又叫旅行笔记，利用可组合NFT技术来可视化用户在Scroll L2上的链上成就。其技术上是 基于 ERC2535 实现 RMRK.app 提出的可组合NFT标准。

DeMatch：使用智能合约作为撮合中间人，实现交易双方的自动撮合和交易执行。因为无须第三方从而大大降低了信任成本。此外，DeMatch 采用去中心化存储，确保平台数据的安全、可靠、不可篡改。并且可以指定交易人，支持链上资产与实物的撮合交易。

 Open Research

以太坊共识层中心化风险及优化方案：ETH共识层控制着出块与验证工作，共识层的中心化可能导致一些中心化攻击。而目前以太坊社区没有一个体系和指标来衡量共识层的中心化程度。本项目梳理了共识层各类中心化，以及不同层次的中心化攻击。通过这两项工作对目前ETH共识层的中心化风险问题做了较精准的描述，这可以为ETH社区提供一种标准和警戒。在整理完现有中心化风险解决方案后，该团队发现了目前火热的DVT技术（尤其是SSV）的重要缺陷，采用DVT与不采用DVT相比，宕机相同数量的operator，会导致宕机更多的验证器，从而降低了整个网络的安全性。因此我们对这一技术进行了改善，提出基于DVT技术的客户端共识规则协议改进方案并制作了EIP draft。

Trusta Labs：旨在为web3项目和团队提供在空投、市场活动、捐赠等等场景的用户分析和女巫识别。TrustScan完全基于链上数据，集合资金网络分析、行为序列等多种识别策略，通过AI算法整合输出结果。

0xdeadbeef：构建MEV基础设施Pioplat，Pioplat将提供以下能力：1）低延时的区块链网络，通过运用更好的网络算法及分布式同步协议，相比于其它区块链基础设施Pioplat将具有更低的区块（交易）接收及发送延迟；2）恶意Token检测工具，通过符号执行等技术手段，帮助MEV Bot识别出恶意Token，避免因买入恶意Token而遭受损失。目标是调研MEV现状，构建低延迟的区块链基础设施，构建准确率更高的恶意Token检测工具。

3CM from MetaBorn：将研究和实现合约和 mining 程序原型，并尽可能制作网页 dashboard 让大家可视化的看到协议的运行过程。

Deep Stack Fantasy：该项目缘起是看见EVM 栈的访问限制是 Solidity 臭名昭著的 stack too deep 问题的源头，该项目给出修改后的以太坊客户端和 Solidity 编译器；并探讨新的客户端与编译器在 zkEVM 上运行的可能性。
