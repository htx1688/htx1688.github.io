长期以来，比特币持有者的口号之一是将 BTC 作为货币的唯一用途。随着比特币NFT的兴起，这种观点似乎正在改变，本文将介绍如何在比特币网络上铸造属于你自己的 NFT。

比特币 NFT 的兴起

比特币 NFT 的兴起无疑是 NFT 生态近期最大的热点。NFT最早出现于2015年，是在 Couterparty 上出现的，这是一种区块链网络，使用比特币的 OP_RETURN 输出来制作非同质资产。

然而，Counterparty 最终未能获得成功，并被以太坊智能合约的兴起所掩盖，此后，以太坊智能合约催化了多次 NFT 繁荣。

近十年来，大量“比特币 NFT 生态系统”的概念始终不被认可，随着 Ordinals 的到来，这一切都发生了变化，Ordinals 是一种新协议，可以将元数据记录在单个 satoshis（比特币的最小单位）上。

那么，为什么 Ordinals 会在此时异军突起？2014 年和今天有什么不同？

首先，当前加密经济拥有一个蓬勃发展的 NFT 生态系统，市场环境较为友好。

其次，Ordinals 风格的 NFT 是一项创新突破，它把 NFT 的全部内容都存储在链上，不像其它 NFT 那样仅仅把一条链接放到链上，然后从私有存储服务器或 Arweave、Filecoin、IPFS 等存储网络中检索。

序数 NFT 允许用户在比特币网络上创建和交易基于数字资产的合约，这些合约可以代表任何形式的资产，包括实物资产、其他数字资产和 NFT 等。与传统的 NFT 不同，序数 NFT 具有可分性，可以根据其价值进行拆分和组合。这使得序数 NFT 更具灵活性和流动性，从而更容易在市场上进行交易。

不可否认，序数 NFT 还存在一些缺点，这些铭文数据（嵌入到区块链的数据/内容）缺乏智能合约提供的可组合性和表达的可能性，并且比特币节点有可能最终将它们删除。

不过，目前可以确定的是，序数NFT将继续存在，它们已经引起了许多比特币和NFT爱好者的兴趣，同时，不少初创公司和开发者已经开始创建和推广序数NFT。

如何铸造比特币 NFT

如果你想尝试创建一个比特币 NFT，首先需要准备一些比特币和一个兼容 Ordinals 的钱包。

就像在以太坊上铸造 NFT 需要 ETH 作为 gas费一样，你需要用 BTC 支付来完成比特币 NFT 铸造交易。在撰写本文时，gamma.io 上的正常铸币费为0.0005 BTC ，大约 13 美元。

第二步，建立一个Xverse钱包，这是一种 Stacks 和比特币钱包，目前可作为 Chrome 扩展程序或适用于 Android 或 iOS 用户的移动应用程序使用。

下载扩展程序或应用程序后，系统会提示你创建一个新钱包，然后提示为你的新帐户设置密码。完成后，你可以使用你的 Xverse BTC 地址接收铸币资金并支付比特币 NFT 铸币费用。

当你的 Xverse 钱包收到 BTC 后，你就可以开始铸造了。为此，前面提到的 gamma.io 网站是目前可用的最直接的 Ordinals 风格的铸币资源之一，因此我向比特币 NFT 初学者推荐它。

打开gamma.io 网站，然后点击页面顶部的“Ordirals”按钮并选择“Create Inscription”，接着进入如下所示的上传仪表板。

在这里，首先上传你想要的文件类型。然后选择你想要的费用级别（正常、更高或自定义）并输入你的 Ordinal 收件人地址以铸造你的 NFT。注意：这不是你在 Xverse 应用程序中的 BTC 地址；你需要点击收藏品选项并按“接收”，然后复制你明确的 Ordinals NFT 地址来执行。

根据你的交易费用和当时使用网络的需求，你的铭文可能需要几个小时甚至几天才能完成。Gamma 将提供一个等待页面，你可以在其中查看铸造状态。

一旦你的 mint 交易完成，你将能够在 Gamma 或 Xverse 钱包中查看它。如果需要出售 NFT，你可以尝试在 Gamma 上申请一个收藏列表，或者在其他一些可用的 Ordinals 市场上售出。

结语

总的来说，序数NFT是比特币和NFT结合的新兴形式，相比以太坊上的NFT具有更高的安全性和不变性，也给比特币带来了新的应用场景。不可否认的是，序数NFT不会消失，它在数字资产市场中的前景值得关注，至于后续能否带来像以太坊NFT一样的热潮，让我们拭目以待。
