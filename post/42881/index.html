外媒最新消息指出，欧盟具有里程碑意义且现已敲定的加密资产市场 (MiCA) 立法草案声称，作为大型收藏品的组成部分出售的 NFT 几乎没有或没有明显独特的品质或效用，因此，将受到与加密货币相同的监管审查。 

根据肯塔基大学法学教授 Brian Fyre 的说法，该立法的措辞相当于欧盟将Bored Ape Yacht Club (BAYC)、Cryptopunks和 Doodles 等 主要蓝筹NFT收藏品标记为证券。如果欧盟开始将蓝筹 NFT 集合归类为证券，Fyre 认为对美国监管机构的影响将是不可避免的。 

因为长期以来，政府监管机构应将加密行业的哪些组成部分视为证券的问题一直是热点问题，该问题最近有所升级。本周早些时候，美国证券交易委员会在一份联邦诉讼文件中暗示，整个以太坊网络应被视为美国监管范围内的证券交易所。 

据悉，MiCA—欧盟对全面加密监管框架的回答—已被预期多年，预计将于2024年生效。9月21日消息，欧盟已经敲定加密资产市场立法（MiCA）的全文。预计将成为该立法的最终版本。 虽然官方上，该文本仍然可以发表评论，但据了解谈判情况的消息人士表示，该文本实际上已经定稿。经核实的法案草案泄露版本，敦促欧盟执法人员对该法案采取“实质重于形式”的态度，这意味着该法案的条款甚至可能适用于某些被归类为NFT的资产。

MiCA是由欧盟委员会、欧盟立法者和成员国就最新的加密监管法规达成的一项协议。这项名为“Markets in Crypto-Assets”（MiCA）的法规是为该地区数字资产创建全面监管框架的首次尝试。欧盟议员 Stefan Berger 表示，这些规则将“整顿加密资产的狂野西部”。

该协议是欧盟将首次把加密资产、加密资产发行方和加密资产服务提供商纳入监管框架，在保护投资者并保持金融稳定的前提下允许创新。法国经济、金融、工业和数字主权部长 Bruno Le Maire 表示，（加密货币）这个迅速发展的行业的最新发展证实了推出欧盟范围内的（统一）监管的必要性。MiCA 将更好地保护投资于这些资产的民众，并防止加密资产的滥用，同时有利于更健康的创新。这项具有里程碑意义的法规将结束加密货币的狂野西部时代，并确立欧盟作为数字资产标准制定者的身份。

早些时候，MiCA 没有提到 NFT，原则上将NFT排除在框架之外，该框架更多的要求加密资产发行人发布包含技术路线图的白皮书，供平台向当局注册，并要求稳定币发行人持有资本并进行审慎管理。 

但最新的修订包括提到了碎片化 NFT。该法案规定，“应将加密资产作为大型系列或集合中的不可替代代币发行，作为其可替代性的指标。” 

根据 Fyre 的说法，这种语言直接攻击了像 Bored Ape Yacht Club 这样的主要 NFT 集合，这些集合由 10,000 个视觉上相似的 NFT 组成，但通过编号机制（例如，Bored Ape #6443）来区分，据称这些机制赋予每个 NFT 持有者完整的扼杀知识产权。

“他们的意思是，当你出售 10,000 NFT 的收藏品时，你真正出售的是整个项目的股份，”Fyre 说。“换句话说，每个 NFT在功能上只是整个项目价值的可替代份额。”从本质上讲，在 Fyre 的解释中，MiCA 的语言断言，在欧盟的眼中，每个Bored Ape NFT 持有者并不拥有一件独特的艺术品。尽管这种区别可能看起来是语义上的，但如果作为法律颁布并如此解释，它可能会产生巨大的影响。它将有效地将流行的NFT收藏品（如 BAYC、CryptoPunks 和 Doodles 等）视为证券并对其进行监管。 

而MiCA 作为欧盟为数字资产创建全面监管的第一次尝试，后续也将会对全球范围内其他国家的监管产生影响。
