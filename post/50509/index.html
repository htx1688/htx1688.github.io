在开始进入正题之前，首先需要明确一点，即：尽管市场出现暂时下跌，但 NFT 将继续存在。市场波动是正常的，加密行业也经历过类似的周期。

好了，现在让我们进入正题，解释一下为什么另一场 NFT 牛市即将到来。

就现阶段而言，我们确实看到市场出现一些负面表现，比如：

1、NFT 价格大幅下跌。

2、NFT 交易量持续走低。

3、NFT 总用户数也开始走下坡路。

但是，NFT 绝不会向某些人所说的那样消亡，因为市场周期本身是呈现自然性的，有牛市，就一定会有熊市。

L2网络的崛起将推动 NFT 市场再次崛起

还记得 CloneX、Moonbirds 和 Doodles 这些地板价曾超过 8 万美元的所谓蓝筹 NFT 项目吗？现在它们的地板价都低于 5000 美元。NFT 的核心是数字藏品，有些具有实用性，但其主要价值来自于 IP+社区，现在 NFT 项目数量越来越多，而且在下一个牛市来临之前，这个数字只会增加，流动性将会在更多项目之间分配，对于大多数 NFT 项目来说，可以分得的“蛋糕”将变得越来越小，即使整个蛋糕盘子变大了。

但需要注意的是，收集东西似乎是人类的天性，无论是实物，还是数字藏品，因此 NFT 不会消失，但以昂贵项目主导的 NFT 市场未来很可能会发生变化，大规模采用可能会通过更便宜的 NFT 实现，Reddit 就是个很好的例子，而且它们还大胆尝试通过L2（Polygon）来提升普及应用，毕竟为 50 美元的 NFT 支付 10 美元的费用毫无意义。

毫无疑问，二层解决方案在 NFT 市场将发挥至关重要的作用，因为L2将降低成本并提升交易速度，使所有人可以进行 NFT 收藏，比如 y 00 ts 也是受益于L2的另一个典型例子。

但是，随着涌入 NFT 市场的用户数量越来越多，即便大多数人会收集价格更低的 NFT，但也会在一定程度上推动部分 NFT 价格上涨，这就像不同的汽车、手表、以及其他藏品都有不同的价格区间一样，NFT 也是如此。

基于上述思路，下一个 NFT 牛市周期很可能会出现一些分散流动性的项目，这将导致出现一个更加平衡和稳定的市场，多元化的生态系统对内容创作者和 NFT 持有者更有利。

下一个 NFT 周期会是什么样子？

但随着 NFT 领域的向前发展，会有新的内容创作者和交易者进入市场，继而推动创新和增长，实际上这种转变已经开始发生，而这种不断发生的变化其实正是一个健康、成熟、潜力无限的市场标志。

对于下一次牛市，具有真正价值和长期潜力的项目将蓬勃发展，短期炒作则不会获得长期效果，所以你需要关注并长期持有哪些 NFT 项目呢？事实上，好的 NFT 项目通常三个要素：

1、具有强大基本面。

2、活跃项目创始人。

3、充满激情的社区。

NFT 的真正价值在于其创新、执行、创造独特体验、以及在经历困难时期时优先考虑社区，坦率地说，许多令人惊叹的项目都遭遇了“滑铁卢”，因此选择 NFT 项目需要睁大眼睛。另一方面，市场会更聪明，同时我们有可能再次看到疯狂的地板价，甚至可能比上次更高，但并不是每个项目都会像上次一样疯狂地抽水，或许会有新的 NFT 项目将成为下一个周期的冠军。

那么，哪些项目将成为下一轮周期中的赢家呢？拥有Web2受众现在并不重要，因为愿意花 1 万美元购买 jpeg 的人将越来越少。在这种情况下，未来同时吸引Web2和Web3受众的项目可以通过产生更高的曝光率来触发下一轮 NFT 牛市，随着这两个世界的结合，由此产生的增长将改变 NFT 领域。更重要的是，下一轮周期中的新 NFT 项目将受益于之前项目的经验教训，从而带来更好、更具创新性的举措，这种改进的正向循环将推动 NFT 市场不断前进发展，甚至吸引更多Web2巨头入场。

此外，如今日益数字化的世界清楚地表明 NFT 将成为我们未来生活中的一部分，我们可以看到 NFT 用例在以下几个领域激增，例如：

  • 游戏

  • 票务

  • 房地产

随着普及应用不断扩大，NFT 技术也将得到进一步验证和推进，不要忘记，人类喜爱收藏的天性会驱使我们持续关注这一新兴领域。围绕 NFT 的兴奋、乐观和 FOMO 也将不可避免地引发另一场牛市，继而把市场推向另一个新的高度——这就是为什么 NFT 远未消亡，我们肯定会看到另一场牛市，尽管需要一定的市场条件，但该领域仍将继续发展和成熟，随着 NFT 技术的进步和更多人入场，NFT 的未来只会变得更加光明。

五点总结

1、NFT 不会归零。

2、疯狂的价格行为很可能会再次出现，但仅限于极少数 NFT 项目。

3、如果 NFT 项目团队执行力较差，即便入场时间很早，但也不会获得成功。

4、新 NFT 项目将比老 NFT 项目更具市场优势。

5、未来将会看到市场转向更便宜的 NFT 项目，以及L2在 NFT 领域里的普及应用。
