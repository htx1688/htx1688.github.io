气候变化是人类面临的全球性问题，随着全世界范围内二氧化碳超标排放，温室气体猛增，对生命系统形成威胁，世界各国正以全球协约的方式减排温室气体，我国还提出碳达峰和碳中和目标。在此背景下，可再生金融ReFi走进了人们的视野，并提出要提供对地球有利的技术和金融系统，试图通过Web3技术扭转气候变化，引入激励，为那些积极实行碳减排，为绿色发展作出正外部性效应的行为进行奖励，从而推动更大范围的绿色发展。

什么是ReFi？

可再生金融ReFi 是一个价值数万亿美元的，激励人们做好事的机会。随着区块链等技术的发展，众多热爱环保的人士发起了众多项目，利用区块链更透明的机制，将现实中的低碳资产引入链上，引入激励机制，试图通过Web3技术扭转气候变化。他们为此不断努力，希望通过可再生金融，释放更多的市场活力，激励更多的人参与到碳减排的行动中来。ReFi是促进现实世界更绿色低碳、公平发展的金融活动。

ReFi领域项目众多，代表项目之一是Celo，我们试图通过解析Celo，更好的阐述什么是ReFi。

Celo是一个移动优先、对地球有利的区块链。Celo基金会的使命是建立一个为所有人创造繁荣条件的金融体系。作为第 1 层协议，Celo采用节能的权益证明机制，并且完全兼容 EVM。Celo 协议具有快速、超轻客户端和内置稳定币，由加密货币和自然资产担保。全球数百万人已经受益于基于Celo构建的资产和应用程序。Celo 基金会以资本支持其愿景和价值观，以推动对地球有益的技术，这将对人类对气候变化的反应产生真正的影响。

Celo的一个使命就是Building a Regenerative Economy。在诞生开始之初，它就把自己定义为一个为可再生经济服务的区块链。

Celo是一种负碳 L1 协议，拥有丰富的全球合作伙伴生态系统，构建 Web3、移动优先应用程序，以创建一个更易于访问和更具包容性的全球金融系统，旨在让世界上任何地方的任何拥有智能手机的人都可以访问和使用它。其创始人出于对环境保护的一个偏好，希望建立一个零碳区块链，在其中去做用自然资本资产支持的稳定币进行P2P支付。

Celo希望消除大规模采用加密货币作为支付手段的障碍，确定了移动优先和建立本地稳定货币的方式。本地稳定币与法定货币挂钩，有储备金支持，并希望借稳定币将自然资本进行资本化。在白皮书当中，他们提到了希望通过部分使用自然资本支持的资产，来达到保护区目的。比如由碳补偿支持的代币或受保护森林或农场碳汇的NFT。

Celo的架构从一开始就设计一个高性能的区块链，以分散的方式在世界任何地方的手机上运行，产生负面的碳影响和低于1美分的汽油费，同时保持用户体验简单流畅。Celo早就做出了一些关键选择，比如决定在它流行之前使用权益证明(PoS)共识机制，因为该机制的能源密集程度要低几个数量级，以及因此比工作量证明更环保。区块链不需要对环境有害。他们还选择通过早期投资使用BLS签名聚合技术，从一开始就面向未来。如今，由于通过Project Wren自动购买补偿，Celo实现了碳负排放。

推动代币化碳信用的采用

Celo基金会致力于扩大Celo社区通过Web3技术扭转气候变化的努力。为了继续协调高效、有效的气候响应，并使Celo区块链成为再生金融之家，创始人和开发人员转向应对气候变化，基金会正在加强与气候集体和长期集体合作伙伴Toucan的合作伙伴关系协议。

Toucan是一家基础设施提供商，计划部署在Celo上，为全球60亿智能手机用户带来透明的气候行动，它已经负责通过Polygon将世界上很大一部分自愿碳信用额供应连接到分散的 Web3金融系统中。

虽然基于Polygon的ReFi项目，如KlimaDao已经展示了区块链技术帮助应对气候变化的能力，但代币化碳信用的未来是多链的。Toucan选择利用Celo的移动优先、负碳区块链，因为它有助于实现Toucan为最佳气候危机解决方案提供资金的使命。

在Toucan将如何与Celo和气候集体合作以稳定我们的气候的高级路线图 中，概述了四个关键目标：

创建一个Celo-Polygon桥。Toucan将与区块链解决方案架构师Curve Labs合作，推出一座桥梁，使用户能够在Celo和Polygon之间移动其原生代币Base Carbon Tonne (BCT)和Nature Carbon Tonne (NCT)。

在Celo上部署Toucan协议。一旦桥梁开始运行，Toucan将在Celo上启动，为整个Celo生态系统提供完整的功能，包括访问、存入池和淘汰TCO2（代币化碳信用）的能力；将BCT和NCT兑换成池中持有的碳代币等。重要的是，开发人员将能够将Toucan的技术集成到他们的项目中。

建议将BCT和NCT添加到Celo Reserve。Toucan将在未来几周内向Celo治理社区提交一份提案，将BCT和NCT代币添加到Celo 储备库中—支持Celo原生稳定币的资产组合。将 BCT 和NCT添加到储备中将推动对Toucan代币的需求，并加深Celo上自然支持的资产市场的流动性。

启动新的生态系统赠款计划。Celo基金会和Toucan将建立一个赠款计划，旨在增加Celo生态系统中ReFi项目的数量。

几个项目已经承诺利用Toucan基础设施和Celo区块链的综合功能，包括：

Senken，即将推出的碳代币开发和交易平台。

Return Protocol，不可替代代币 (NFT) 的创建者，用于抵消链上碳排放。

Solid World是一个去中心化的自治组织 (DAO)，资助高质量的碳项目。

Eden Dao是一个旨在加速碳去除并帮助为子孙后代创造财富的 DAO。

Celo 基金会期待支持这些项目以及由于 Toucan 将其生态系统引入 Celo 而可能出现的其他项目。

Celo基金会致力于ReFi 运动

Celo基金会的使命-建立一个创造繁荣条件的全球金融体系-受到查尔斯·爱森斯坦 ( Charles Eisenstein ) 工作的启发。他的《神圣经济学》一书认为，虽然金钱可能是世界上一些弊病的根源-疏离、竞争和稀缺-但它也有可能帮助社会过渡到一种更加联系和可持续的生活方式。

这种潜力不仅被基金会的联合创始人 Rene Reinsberg、Marek Olszewski 和 Sep Kamvar 所接受，而且还被基于Celo的企业家和开发人员所接受，他们相信使用金钱和区块链/加密技术来解决各种系统性问题，包括那些由气候变化引起的。使用金钱为世界社区创造积极变化并扭转气候变化的概念已被称为再生金融（ReFI）。

2021年10月，Celo气候集体成立，以支持ReFi工作，并激励世界各地具有气候意识的贡献者在Celo的基础上构建并帮助发展Celo生态系统。作为一项社区驱动的倡议，Climate Collective通过调整对Celo稳定币的需求来应对气候变化—目前是cUSD，它跟踪美元的价值；cEUR，跟踪欧元的价值；和cREAL，它跟踪巴西雷亚尔的价值—努力保护地球。随着对这些稳定资产的需求增加，在储备中分配的自然支持资产的百分比也会增加。集体的目标是到 2025 年底将自然资产的配置多样化并增加到40%。

金钱追踪、促进和塑造经济活动。金钱的哪些特征可能会导致一个更美丽的世界？无论什么支持金钱，人们都会赚更多。当金钱有牛支持时，人们饲养的牛就更多。当它有黄金支持时，人们开采了更多的黄金。 那么，为什么不用更多我们希望在世界上看到的东西来支持资金：清洁的水体、茂密的森林、生物多样性呢？ 这是一种似乎过于理想化而无法实现的想法，直到它钻进你的脑海。

除美元外，大多数货币都需要有某种东西支持才能保持稳定。法定货币和稳定币也是如此。USDC 由其储备中以美元计价的资产支持，USDT（Tether）也是如此。甚至，算法稳定币实际上并没有以美元计价的储备或任何东西作为支持。如果人们接受支持稳定币而没有任何东西和其他组成的货币，为什么你不能用保护地球的东西来支持稳定币，并获得自身稳定的额外好处：自然资本支持的资产？Celo 的白皮书在 ReFi 出现之前写于 2018 年，其中提到允许“储备包括实物资产”： 从稳定性的角度来看，这很有帮助，并且还允许自然资本支持的支付手段货币（例如，以林地为支持的货币），对这些货币的需求增长将增加支持它们的自然资本数量。

举个例子，以一种可能支持货币的自然资本资产为例：雨林。热带雨林本质上是一种有限的宝贵资产。它可以作为一种储备资产，支持用于支持美元的黄金等货币。不同之处在于，黄金可以熔化成金条，存放在某个地方的金库中，并且很容易追踪和计算；热带雨林需要停留在它们应该有价值的地方，而且更难追踪。 

但随着更多的卫星和传感器、更好的人工智能以及像开放森林协议(OFP) 这样激励分布式监控、报告和验证 ( MRV ) 的项目，会计不再是交易破坏者。因此，像 OFP 这样的项目可以验证热带雨林的特定部分是否已得到保护或重新造林，并在 Celo 上发行代币，这些代表热带雨林的代币可以在新的 cUSD 投入流通时以编程方式购买到储备中。 如果它实现了目标，那么这一承诺可能会对防止气温上升斗争产生有意义的影响。

ReFi的未来-造福于人类的系统

ReFi是一个绝妙的想法—使用人类现在可以使用的工具重新构想金融系统，以更好地满足当前和未来所有利益相关者的需求。它为外部性定价，向创造负外部性的人收费（如Once-ler）并奖励那些创造正外部性的人（如为树木说话的群体）。

ReFi是一种新的金融模式，它将人们作为具有独特才能的个人、作为经济中心的社区以及全球生态系统的健康和生物多样性纳入并重生。其参与者希望通过更好的衡量和融资来加速新的、对地球有利的技术的开发。 

ReFi 提供了对地球有利的技术，并投射出更强大的信息素来吸引群体的注意力。利用web3 工具对人类的运作方式产生影响。对碳和其他自然资本进行代币化是 ReFi 迄今为止吸引最多注意力和资金的部分，这是有充分理由的。它代表了一个机会，可以创造一个价值数万亿美元的新市场，让地球变得更健康、更富裕。此外，这是一个特别适合加密的用例：从分散式 MRV 系统到更开放的资产登记处，再到更具流动性的市场，将加密与传感器、无人机和人工智能的进步相结合，为构建一个独特的时刻提供了一个独特的时机。再生地球自然资源的金融市场。 

尽管现在还为时过早，但一些基础部分已经到位。比如在Celo上构建的关键部分：Toucan Protocol 、Flow Carbon和Loam 。

Toucan 通过 Toucan Bridge 将自愿碳市场与 web3 连接起来，将现实世界的碳信用代币化以换取半可替代的代币化 CO2 (TCO2) 代币。通过链上的信用，他们创建了一个透明的元注册和基础碳吨（BCT）和自然碳吨（NCT）代币池，以增加流动性。正如它在其主页上所说，Toucan“将可编程碳带入 web3”。

另一个致力于碳信用代币化的项目是Flowcarbon 。它正在创建 GNT 代币，由它在链上带来的企业级碳信用支持，并提供基础设施来帮助企业和个人抵消他们的排放。 

虽然 Toucan 和 Flowcarbon 代币化器碳信用额度，但Loam的计划实际上是“将碳存回地球母亲”并在此过程中获得新的补偿。 Loam 由 Birju Shah 和 Ravi Yenduri 创立，其愿景是创建一个农业数据市场，一个快速、简单、透明的平台，以激励农民参与再生实践。正如 Ravi 所描述的那样，他们希望创造“今天的变革，并在长期内获得回报”。 

这些项目中的每一个都是引人注目的，这些更广泛的 ReFi 生态系统的力量，真正关心并希望建立造福于所有人的系统。目标是在为时已晚之前建立一个新的再生系统，让人类拥有正确的工具和技术。

（本文信息来自Celo基金会、Not Boring以及Rene、Ed、Tomer、Marek、Phil、Ravi、Mike、Anna、Sep、Jonathan和Deana 的对话。）
