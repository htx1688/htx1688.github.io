一场关于冷钱包 的争议

近日，一款冷钱包发布的新功能引起广泛关注。

据悉，该功能是一种基于 ID 的「可订阅式」密钥恢复服务。该功能将钱包助记词（密钥）分成三个部分（加密分片技术），并分发给三个托管人：该冷钱包公司、加密资产托管公司和代码托管公司。如果有人丢失了密钥，三个分片中的两个可以结合起来以重新获得对锁定资金的访问权。订阅该服务的价格是 9.99 美元/月。

然而，一些用户表示担心，若将私钥分割给第三方可能会让私钥泄露，从而破坏了冷钱包的最终目的。此外，如果用户使用该项服务，他们需要向公司提供实名认证，一些人认为这违反了数字资产的隐私价值。

批评者还提到了该冷钱包的安全性，引用了2020年的数据泄露事件，该事件暴露了近10,000名客户的电子邮件。虽然没有钱包被泄露，但这一事件也使该公司的安全性受到质疑。

此次争议还引发了不少人对数字资产钱包的好奇。那么，数字资产钱包有哪些？冷钱包是什么？市面上又有哪些冷钱包可用呢？本篇文章将带您走进冷钱包的世界。

什么是数字资产钱包？

与传统钱包不同，传统钱包大多携带现金、银行卡和其他实物，而数字资产钱包则是数字资产的存储器。

我们的数字资产都存储在区块链上，只有私钥可以帮助我们访问它，所以私钥就代表存在于数字钱包中的数字资产的所有权。因此，保证私钥的安全是非常重要的。如果私钥丢失，将导致您失去钱包内的所有资产。

目前，数字资产钱包主要有两种形式：硬件钱包，或称冷钱包；以及软件钱包，或称热钱包。

什么是冷钱包？

冷钱包，也被称为冷存储，通常是一个物理设备，保护资产免受未经授权的访问，也就是离线存储。一般认为冷钱包比热钱包更安全，因为热钱包是连接到互联网的，存在黑客盗取私钥的风险。而冷钱包通常是硬件设备，一般看起来像是一个USB驱动器或其他物理设备。

为什么一定要用数字资产钱包？

数字资产钱包被认为是数字资产持有者自我托管的安全选择。

对于数字资产安全来说，自我托管非常重要，它让资产所有者可以完全控制他们自己的数字资产，避免在Centralized交易平台上存储数字资产的风险。

一般持有者将他们的数字资产存储在一个Centralized交易平台上，是因为他们信任第三方能随时保证其资产安全。然而，过去一年的种种事故表明，Centralized交易平台并不安全——参考FTX、Celsius、Voyager等公司的破产案例。

通过自我托管拥有您的数字资产私钥，您能够存储、发送和接收数字资产，而不需要经过其他人的许可。

同时，通过高难度密码以及其他安全预防措施，如双因素认证（2FA），这些设置可以确保资产所有者是唯一能够访问这些资金的人，避免资产在交易平台上被盗。

冷钱包和热钱包的区别

热钱包是需要联网的，不需要在离线和在线模式之间切换来执行交易。一般不建议用户将大量的数字资产保存在热钱包中。

另一方面，冷钱包是专门生产的硬件设备，用于以物理形式存储各种数字资产。冷钱包显著的优点是它将私钥离线存储在一个更安全的环境中。这些钱包通常很小，携带方便。

热门的冷钱包

尽管最近发生了一些争议，但我们还是有必要深入了解目前市场上流行的一些冷钱包，以下排名不分先后。

Ballet RealBitcoin

RealBitcoin是BalletWallet的旗舰产品，是全球首个用于存储数字资产的非电子物理钱包。支持Erc-20, Trc-20, Bep-20等链上所有Token，主流Token大概支持200多种，目前还在不断增长。

RealBitcoin硬件本身没有任何电子元件，采用纯钢板的制作工艺。

RealBitcoin采用的是BIP38协议，没有传统的助记词而是采用双要素私钥，并在钱包上有防篡改设计，避免拿到二手产品或假冒产品。

RealBitcoin配备了一个应用软件——Ballet Crypto，软件本身并不会存储资产和私钥，主要用来监控资产以及进行数字资产交易。软件无需账号注册和月费。

Ledger Nano X

Ledger的第二代冷钱包被称为Nano X。通过集成的Ledger Live平台支持5500多种数字资产，包括比特币、以太坊和Cardano。应用户要求，每年都会在Ledger Live平台上增加对其他数字资产的支持。

Ledger Nano X的使用上是用一根USB线将其连接到电脑上，也支持蓝牙技术将其连接到安卓和iOS移动设备上。

Ledger Live软件直接集成到Nano X冷钱包中，它可以帮助用户通过其简单的用户界面管理他们所有的数字资产。

Trezor Model T

Trezor Model T是该品牌的第二代钱包，与Ledger类似，但与之相比增加了一项功能——它允许其用户从Trezor网站访问第三方交易平台，如Changelly和CoinSwitch。

Trezor T型有一个触摸屏，还具有一个MicroSD卡端口，可以对PIN进行加密，确保钱包免受网络攻击。

Trezor T独家安全功能包括Shamir备份，它可以建立多达16个私钥碎片，用于在您的设备出现问题时恢复资产。

Trezor Model T使用时，您也可以使用Trezor的QR按钮来查看地址的QR码，对方可以扫描它来开始向您转账。付款一经处理，就会在交易页面列出。

CoolWallet Pro

CoolWallet Pro是一个新的以DeFi为核心的冷钱包，于2021年5月发布。它具有信用卡的外形和加密的蓝牙通信，并具有访问DeFi的功能。

CoolWallet Pro配备了一个醒目的电子墨水屏幕，可以直观地查看数据，还有一个下沉式按钮，可以确认所有交易。

CoolWallet Pro是CoolWallet系列的第三款产品，可以追溯到2016年，它迎合了DeFi用户希望将其资产用于PoS（Proof-of-Stake）网络的需求。

除了现有的主流产品组合，Pro还提供对Polkadot（DOT）、Cosmos（ATOM）、Tron（TRX）和TRC20、BNB智能链的BEP-20资产、ETH、Cardano（ADA）和NFT的支持。

KeepKey

KeepKey是一个笔盘形式的冷钱包，钱包尺寸要明显大于其他知名的数字资产钱包，但是，这并不妨碍它的使用，设置它并进行数字资产交易仍然很方便。

KeepKey可以设置密码，以防止物理盗窃。

KeepKey使用离线冷存储技术来存储您的公钥和私钥，并在您的KeepKey 丢失或损坏时使用备份助记词。

KeepKey与ShapeShift资产兑换平台协同运行，可以进行多种资产的交易。

KeepKey的软件是完全开源的，您也可以在GitHub上查看，对其进行交叉验证。

Ellipal Titan

Ellipal Titan包括一个4英寸的彩色触摸屏，可以简单地设置和监控所有实时交易。

Ellipal Titan不利用USB或蓝牙来签署交易，Ellipal APP只是使用QR码。

Ellipal Titan电路板的设计禁止了所有连接，包括Wifi、蓝牙和手机。

Ellipal Titan具有防篡改自毁功能，它能保护您的设备免受在线和离线攻击。如果它感觉到有人强行入侵，它将销毁其中包含的所有信息（私钥）。

Safepal S1

SafePal S1冷钱包早在2019年就已推出。它支持54条链和主流的数字资产，包括NFTs。

SafePal在安全方面纳入了几个功能和安全要素，包括恢复种子、双因素认证（2FA）、设置密码、私钥删除池。

SafePal S1具有极简的外观，类似于一个MP3播放器，在防篡改包装上贴上两个独特的封条。它有一个相当大的彩色屏幕，一个二维码扫描仪，以及一个用于导航钱包的D-pad控制器。

SafePal还配备了一个SafePal钱包软件，您可以用它来监控您的数字资产余额和进行数字资产交易。

SecuX V20

SecuX V20是SecuX系列非常受欢迎的冷钱包之一。

SecuX V20有一个2.8英寸的彩色触摸屏显示器和一个用于离线/断开操作的6000mAH电池，可持续使用约7小时。

SecuX V20通过USB或蓝牙连接到主机，它可以在台式电脑、笔记本电脑、iPhone、iPad、Android智能手机、平板电脑或Chromebook上使用。

SecuX V20拥有一个来自Infineon的CC EAL5+认证安全元件芯片。SecuX V20可以通过低能耗蓝牙连接做到完全离线，实现数据保护。

SecuX V20钱包中的二维码功能允许它在钱包设备上进行简单的资金转移和交易。

Keystone Pro

Keystone Pro（以前称为Cobo Vault Pro），它配有一个四英寸的触摸屏，带有指纹传感器的开源固件，并支持PSBT比特币多义词，以增加恶意攻击的安全性。

Keystone是一个分层确定性（HD）的冷钱包，符合BIP 32/39/44标准。通过整合QR码来签名，Keystone降低了恶意软件入侵的风险，并增强了可视性。

Keystone Pro的钱包电池是可拆卸的，Essential和Pro版本都兼容AAA电池，同时配备了可充电电池。

究竟如何挑选钱包？

对于交互频率较高的人，热钱包简单便捷。但如果你更关注资产安全，首选冷钱包，当然，不同的冷钱包也适合不同的使用偏好。

无论是冷钱包还是热钱包，最终目的都是存储资产并保护资产安全。因此，如果仅从这个角度出发，Ballet RealBitcoin、Trezor Model T以及SafePal都是不错的选择。

总体来说，一个好的冷钱包应该简单易用，私钥存储不能太繁琐，否则安全性反而会降低。大部分冷钱包都有电子元件比如显示屏、芯片，也就面临着容易损坏和修复问题，所以制造商的售后服务支持也尤为重要。
