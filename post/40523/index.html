以太坊的未来是通过Rollup扩展。 

而一类特殊的Rollup——与以太坊虚拟机兼容的 ZK Rollup——是目前实现这一未来的最有希望的方式。  

展望未来，可以毫不夸张地说，“zkEVM”解决方案将成为一般 web3 项目（包括 NFT 项目）采用的游戏规则改变者。 

那么请注意，因为本周不是一个，不是两个，而是三个不同的团队宣布了大型 zkEVM 开发！让我们分解这些进步，并谈谈 NFT 用户在 zkEVM 领域未来的期望。 

zkEVM 和 NFT

“总的来说，我自己的观点是，在短期内， optimistic rollups可能会在通用 EVM 计算中胜出，而 ZK  rollups可能会在简单的支付、交换和其他特定于应用程序的用例中胜出，但在随着 ZK-SNARK 技术的改进，中长期 ZK  rollups将在所有用例中胜出。” — Vitalik Buterin。

什么是 zkEVM？

Rollups 是第二层 (L2) 扩展解决方案，因为它们在以太坊“第一层”之上工作，使基于以太坊的交易非常实惠且非常快速。今天最流行的两种Rollups类型是 Optimistic Rollups、零知识证明和ZK Rollups。

这两种类型都在链下执行交易，即在以太坊 L1 之外，这两个类型也继承以太坊广泛的安全保证，通过将其批处理的交易数据发布到以太坊。Optimistic 和 ZK Rollups 的不同之处在于它们如何促进发布数据，即欺诈证明或零知识有效性证明，以及它们可以为用户提供哪些优势。

一开始，Optimistic Rollups 非常适合“通用 EVM 计算”，这意味着以太坊项目（如 NFT 项目）可以轻松移植用 Solidity 编写的 L1 智能合约或在这些 L2 上创建类似 L1 的智能合约。 

相比之下，ZK Rollups非常高效和安全，并且随着时间的推移它们会变得更加如此，但迄今为止，某些设计细微差别意味着这些 Rollups 还没有提供 EVM 兼容性。 

综上所述，具有与 EVM 兼容的执行环境的ZK Rollups就是 zkEVM。到目前为止，还没有这样的项目付诸实施，这就是为什么 zkEVM 的进步代表了 zk rollups 的“圣杯”：它们最终将允许这些 L2 将通用 EVM 计算与零知识证明提供的广泛事务效率相结合。 

从长远来看，这种升级将有助于ZK Rollups“在所有用例中胜出”，正如 Vitalik Buterin 所预测的那样。而现在，“所有用例”包含了一个丰富的 NFT 生态系统。但在我们讨论 zkEVM + NFT 的影响之前，让我们简要回顾一下本周的发展，看看 zkEVM 的未来是如何在此时此地实现的。 

三个新的 zkEVM 公告

7 月 18 日星期一，Scroll 团队为其基于 zkEVM 的ZK Rollup推出了一个pre-alpha 测试网，并以 Scroll 同名：

“ Scroll 扩展以太坊并为数十亿用户和开发者服务的计划是一个需要仔细考虑和执行的长期路线图。我们坚信 zkEVM 的未来是扩展以太坊的关键，因此，我们致力于以一种使我们能够集中精力应对任何挑战并将反馈与我们的路线图结合起来的方式发布它。”

然后在 7 月 20 日星期三，发布了另外两个主要的 zkEVM 公告： 

  • zkSync建设者 Matter Labs 透露，zkSync 2.0是一个基于 zkEVM 的  ZK Rollup，“将在 100 天内上线主网上。”

  • Polygon为Polygon zkEVM引入了初始开源实现，并且即将推出公共测试网。值得注意的是，这个 zk rollup 的设计不仅是为了 EVM 兼容性，而且是为了与 EVM 完全等效，这意味着在这个 zk rollup 上编写代码和使用工具将与在以太坊上做同样事情的 UX 没有区别。这种动态对于 web3 构建者来说将是非常用户友好的。

为什么 zkEVM 对 NFT 很重要

我们正朝着基于 zkEVM 的 zk rollup 成为扩展 web3 项目的黄金标准的未来迈进。 

在密码学而非经济激励措施的支持下，zk rollup 比Optimistic rollup 更安全。对于多个用例，zk rollups 还提供了非常便宜的交易成本。此外，与Optimistic rollup所需的约 1 周等待时间相比，从 zk rollup中提取 NFT 可能需要 15 分钟。 

因此，快速且超实惠的交易 + 高加密安全保证 + 通用 EVM 计算 + 快速 NFT 提款 = zkEVM 是独特的 L2 基础设施，可支持 NFT 的广泛主流采用。 

事实上，早期 NFT 生态系统的瓶颈一直是以太坊 L1 的技术限制，以及随之而来的用户体验痛点，例如昂贵的交易。zkEVM 的出现和发展将有助于一劳永逸地打破这一瓶颈，让 NFT 项目轻松地为数百万用户（即任何人，而不仅仅是成百上千）提供基于以太坊的体验。 

还值得注意的是，这个 zkEVM 的未来有望变得更加光明。例如，以太坊备受期待的Merge 更新将在今年晚些时候进行，但这并不是领先的 L1 技术路线图的结束。The Merge 之后将出现 The Surge，它将通过分片为汇总引入大规模的可扩展性改进：

这意味着具有超低交易成本的超高性能 L2 即将到来，而这种演变将决定性地为 NFT 走向世界舞台做好准备。从长远来看，以太坊本身很可能会从 EVM 迁移到 zkEVM 格式，因此 NFT 在多种意义上的未来都将通过 zkEVM 运行。有时在加密货币中，你有一个清晰的机会来设想即将发生的事情，这肯定是其中之一。
