马斯克收购推特之后，裁员风波还未平息，上任不久的马斯克又放出“一把火”。

上周三，马斯克在一次 Space 上的公开对话中充实了其对推特的愿景。据称，他设想用户将他们的在线银行账户连接到社交媒体服务，将支付功能整合到推特中。也就是说**，推特将目标设置为一款类似微信的“一站式平台”。** 即从单纯的社交平台转变为涵盖更多功能，从游戏到购物、再到支付等，以承载用户的更多需求，而支付则是一切延展服务的入口。这一想法在近期已经付诸行动了。11 月 10 日，据《纽约时报》获得的一份 FinCEN 文件显示，推特上周提交了一份文件，已涉足支付业务 。

然而，想要在海外市场做出微信显然并不是一件容易的事情 。无论 Facebook、WhatsApp，还是曾经微信的启蒙者“Kik Messenger”，都曾试图成为类似微信这样的超级应用，但结果却无一不是以失败告终，外界也普遍认为马斯克此举恐怕也将面临困难重重。

推特做支付面临的最大难题是什么？Web3 的构想能否赋能这一巨大流量入口的支付业务？ 本文将盘点波卡中的隐私支付协议、去中心化身份聚合协议，探讨 Web2 与 Web3 结合的可能性。

 推特做支付的难点  

对于推特而言，从单一的内容发布平台转向综合类软件的痛点，或许是数据安全问题 。

推特如今显然并不具备即时通信的私密性 ，而更像是海外市场的“微博”，更像是内容发布平台，也就是说，推特身份的信息的暴露范围已经不限于数百人的好友圈，而是数亿人的全球范围。

作为一个公开内容发布平台，推特的信息安全并不乐观 。2022 年 7 月，有黑客以 3 万美元（约合 20 万人民币）兜售 540 万  Twitter 账户数据。这并非 Twitter 首次发生数据泄露事件 。2019 年 1 月 ，Twitter 披露了其修复的一个安全漏洞，而在此前四年多的时间里，该漏洞使得许多用户的私人推文被泄露。2020 年 12 月 ，因 Twitter 在此数据泄露事件中未能及时通知和充分记录违规行为，爱尔兰数据保护委员会（DPC）对其开出了 45 万欧元的罚单。

一旦推特携带金融属性，黑客对于数据安全的破坏活动显然会更加猖獗，此外，账户数据泄漏、被盗带来的后果显然也会更加严重。

除此之外，不仅是支付，推特想要成为一个超级 APP，往往需要获取众多的隐私权限 ，这是西方用户无法容忍的，也与众多欧美国家更为严厉的反垄断法和隐私保护等法案相违背。

 推特结合 Web3 协议的潜力  

对于推特来说，如何保证足够的数据安全与私密才是展开支付业务的制胜法宝。

不谋而合的是，隐私与安全也是 Web3 运动的重要组成部分 。在 Web3 时代，用户将倾向于用更彻底的方式保护个人数据隐私，从而引发数据所有权和价值的转移。随着应用的去中心化，链上数据可查的情况下，用户行为、产生的数据乃至应用协议同样需要得到隐私保护。

从技术实现的角度而言 ，隐私保护是多方面的，包括基础区块链平台隐私保护、存储数据隐私（分布式存储）、用户私钥管理、匿名协议等多方面。

从应用场景而言， 隐私保护也不仅仅限于 Web3 ，同样也有与 Web2 产品相结合的案例。加密通讯应用 Signal 就在 2021 年宣布，将在其快速增长的加密通信网络中推出用户点对点支付功能。 为此，它集成了对加密货币 MobileCoin（MOB）的支持。

而推特作为 Web3er 的最重要聚集地，Binance 在本次收购案中也占据了一席之地，加之马斯克本人一向对 BTC、DOGE 等加密货币关注有加，在外界看来，推特未来的改革方向 Web3 也不是没有发挥的空间 。毕竟 Web2 和 Web3 的结合，我们已经从现实案例窥得其发展潜力和未来趋势。

 波卡如何解困推特的 Web2 难题？  

作为跨链生态的佼佼者，Gavin Wood 于 2016 年创立的波卡一直以来的目标就是成为下一代网络的基础设施而不断前进着，百余个生态项目正在通过平行链以及其他方式源源不断的进入波卡生态，而这些项目就涉及了隐私、安全、支付以及 DID 等多个领域。

Manta Network

======================

Manta Network 是波卡上面向 DeFi 和 Web3 生态系统的以隐私为中心的解决方案。该平台使用 zkSNARK 和 Groth16 等加密技术为各种链上活动提供端对端的隐私层。

Manta Network 成立于 2020 年，是一个 layer1 扩展解决方案。与第 2 层解决方案不同，Manta 的隐私保护功能是网络核心架构的一部分，而不是基于其他区块链之上。该平台将在 DeFi 领域推出多种隐私保护协议，包括MantaSwap — 去中心化隐私交易 AMM 平台和 MantaPay — 去中心化隐私支付协议。

MantaPay  允许用户将公开资产隐私化，进行隐私资产的转账，以及将隐私资产赎回至公开地址。

MantaSwap  是 AMM DEX，允许用户在保持隐私的同时进行交易。MantaPlatform 引入了可编程的隐私资产， 开发者可以创建隐私保护应用。例如创建隐私拍卖应用，防止将策略暴露给竞拍对手。

Manta 还基于 ZKP 创立了新的资产类型 zkAssets ，支持波卡平行链资产间的隐私交易，还支持 ERC20 等同质化代币，ERC721、ERC1155 等非同质化代币，以及灵魂绑定代币（不可转让的非同质化代币）的隐私交易。

与 Monero 或 ZCash 等现有隐私解决方案相比，Manta Network 通过利用 Polkadot 提供无缝的跨链互操作性，为波卡平行链上的锚定 BTC、稳定币等主流资产提供链上隐私保护 。

Raze Network

=====================

Raze Network 是基于 Substrate 框架开发的跨链隐私中间件协议， 旨在保持跨链审计透明性的同时，通过零知识证明技术为波卡生态中整个 DeFi 领域提供端到端的匿名支付服务。

Raze Network CEO Justin Kellison 认为，当下互联网用户的隐私并未被有效保护，除了大量数据被 Facebook、Google 等大公司滥用外，粉尘攻击等各种网络攻击也让用户的身份活动变得可追溯，容易导致钓鱼诈骗的发生。其中，用户的金钱支付行为数据最为敏感，因此对其保密至关重要。

Raze Network 核心技术模块是 DeFi 生态系统的第二层匿名支付。 该模块将作为基于 Substrate 的智能合约导入，将用作波卡的通用的即插即用基础结构，允许用户在加入 DeFi 之前隐藏自己的帐户地址和财务信息。 由于 Zether 框架支持以太坊账户模型，因此 Raze 合约将支持在波卡生态中发行的代币（例如 DOT 和 Kusama）以及 ERC-20 跨链私密支付，从而显著增加波卡生态的流动性。

Raze Network 采用 zkSNARK 零知识证明方案，不需要任何受信任的设置。用户可以使用 Raze Network 通过隐藏相关方的身份、转账金额、账户余额以及隐藏整个交易历史来实现链上交易的私密性。

Litentry

=================

与推特有结合想象空间的除了隐私协议，还有去中心化身份**（Decentralized Identity，简称 DID）** 。

DID 是指一套完全去中介化并且允许个人或组织能够完全拥有对自己数字身份及其数据的所有权、管理权和控制权的身份。即你的身份信息并不是存在某个机构的服务器，被某个机构所掌控，而是分布在许多的电脑上，完全由你自己掌控。从这个方向而言，DID 或许能一定程度上消解推特用户的广场属性，既可让用户可信身份快速登录发言，又能使用户保持匿名。

Litentry 是波卡首个基于多网络的去中心化身份聚合协议，拥有去中心化身份（DID）索引机制和基于 Substrate 的信用计算网络。 它的目标是帮助去中心化身份聚合、验证和信用计算。旗下涵盖 Litentry WebApp, Litentry Graph, Drop3 等系列产品，目前已为 Phala World 提供聚合身份传送支持。

一旦有了这样从不同链上聚合的数字身份之后，可以深刻的扩展现有区块链的应用范围，比如：多元化应用场景：聚合数字身份可以实现信用的查询，身份的授权和认证，也可以实现包括单账号多链登陆和代理支付的功能，也可以与链下的应用相结合。比如领英, Twitter 中的数据，拓展聚合数字身份的维度和应用场景。

 总结  

马斯克接管推特，一定程度上给 Web3 协议进入Web2 产品带来了曙光，Web3 中为 DeFi 而生的去中心化隐私协议、去中心化聚合身份协议，与推特转型的支付业务，有着同样的金融属性。

与此同时，波卡生态的隐私赛道项目已经初见雏形 ，除了 Manta 采用 ZKP 技术创建了隐私资产和隐私支付之外，另有 Phala 将 TEE 和区块链结合，在完成复杂运算的同时保护好用户的数据隐私；Automata 将 TEE 与 Oblivious RAM 算法结合，让用户数据无法被第三方查看、更改和访问，从而防止 MEV 和进行隐私投票等等。同样，新兴赛道 DID 中的佼佼者 Litentry 也为推特用户 ID 提供了新的方向 。

相信随着这些基础设施的逐步实施，只要推特愿意向 Web3 展开怀抱， Web3 也不乏收割推特 2 亿日活流量的机会。
