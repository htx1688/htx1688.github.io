第 1 天：什么是 web3

Web1 只能阅读——web2 既可以阅读也可以创作——web3 是关于阅读和创作得所有权。 在web2中我们使用社交媒体应用程序时，所有数据都存储在公司的服务器上，而且大多数大型平台仅由少数几家公司处理。
我们对这些公司贡献我们的数据，并依赖他们。不过我们真的想这样做吗？只是因为我们没有其他选择罢了。
就像艺术家从来没有真正拥有他们制作的东西，我们也并不能“真正”控制自己的钱。不过银行可以，银行有权冻结某人的账户，而且在历史上曾发生过多次停止提款的情况。
由此可见，我们生活的世界是多么的中心化！

还有很多其他例子说明为什么我们需要去中心化和 web3 的世界，为什么这很重要。
如果有了解sound.xyz（一个音乐网站）以及它如何帮助艺术家独立创作并通过所有权赚更多的钱就能很会理解了。
中心化平台占据主导地位已久，以至于许多人都忘记了还可以有更好的方式来构建互联网服务。
而基于加密网络的Web3正是这样一种存在，它为第三方开发人员、创建者和 企业提供公平竞争的环境，是开发社区共有的网络。

第 2 天：区块链基础知识

区块链是“块”链或数据的共享分类账本 - 每个人都可以实时查看正在发生的事情和所有交易。

如果信任可以通过代码设置，那么创作者就不必依赖中间商。

他们：不必通过银行就能获得钱才；不必信任律师才能起草合同；不必依赖互联网才能谋生。

区块链的三个特点是：

去中心化： 交易存储在计算机（节点）网络上。

不可篡改： 事务一旦提交到块就不能更改。

公开透明： 任何人都可以查看交易。

比特币使用区块链，因此是去中心化、不可篡改和公开透明的。这也是由于：

硬上限：永远只有 2100 万比特币。

单一用途：许多持有者只是希望它成为一种存储价值的数字Token。

从开发人员的角度来看，这是一个永不宕机的后端，人们可以通过地址匿名看到大家在做什么。可以部署产品并使用它来创建一个去中心化的应用程序，而且每个人都拥有他们所生产的东西。

Day3：钱包、共识算法

在区块链上，你的钱包就是你的身份，对的，没有名字，只有钱包地址。在线钱包就像你口袋里的那个钱包——我们需要它来存钱。
这些钱包要么是软件（例如，Coinbase、Metamask），要么是硬件（例如，Ledger）。
最简单的学习方法就是尝试一些东西。找到可以购买一些加密资产的平台。设置一个像 metamask 这样的钱包，将钱转移到那个钱包，然后看看你是否可以交换一些Token。

每个钱包都有一个安全密钥，需要保管好以确保你不会丢失它。 在大多数情况下，大多数人会使用CEX来管理他们的资金，比如 binance 和 coinbase。
共识算法是计算机科学中用于在分布式进程或系统之间就单个数据值达成一致的过程。

工作量证明（POW）和权益证明（POS）是两种主要的共识算法。
POW： 节点（矿工）尝试解决问题——第一个解决问题的人获得奖励——其他节点检查它是否正确。

POS： 矿工用他们的份额来解决问题——其他验证者检查它并在正确的情况下获得份额——最终将块添加到链中。

第 5 天：Defi

DeFi （去中心化金融）是一项旨在建立一个对所有人开放且不需要像银行这样的信任中介的新金融系统的运动。 为了实现它，DeFi 在很大程度上依赖于密码学、区块链和智能合约。
即DeFi 是指 “使用区块链技术构建的金融应用生态系统，而无需银行介入”。
学习 Defi 的一个很棒的理由是，一些应用程序的 APY（年收益率）是 10%-20%，而银行的传统回报率为 0.01%。想想看——你的钱的回报率有一天可以扩大达到 2000 倍！

第 6 天：NFT

第 12 天：预言机，Chainlink

智能合约需要一些东西来与现实世界的数据进行交互。而且我们不能依靠单点交互来获取这些数据。 我们获得的数据仍然是集中的。如果提供该 API （应用程序编程接口） 的公司出现故障，我们在简单应用程序中使用的 API 可能会使该应用程序停机。我们使用预言机的解决方案来解决这些问题，它作为分散的方式将现实世界的数据输入智能合约。
比较领先的预言机之一是 Chainlink。Chainlink 是一个建立在以太坊上的去中心化区块链预言机网络。该网络旨在用于促进防篡改数据从链下来源到链上智能合约的传输。
如果您使用外部 API，您实际上是在将去中心化置于风险之中，因为该 API 是中心化的并且归一个实体所有，因此如果未来该 API 出于某种原因出现故障，一切报废。

第 13 天：山寨币 - Eth、Sol

比特币以外的任何coin或项目都称为山寨币， 但以太坊、solana 和其他项目的巨大成功并不能真正证明这个名字的合理性。
许多山寨币是使用比特币的基本框架构建的， 为它们中的大多数提供了“peer-to-peer”的理想对接。此外，在模仿比特币的基础上，它们旨在提供一种便宜而有效的交易方式。

另一个著名的山寨币和项目是 Solana。使用 Rust 作为底层语言来构建相关工程会更快、更便宜。如果有兴趣，可以自行查阅相关资源。

小结

Web3总的来说是一个终极、开放、无需信任、无需权限的理想网络架构，它集所有前沿信息于一身，并且消除巨头垄断，一切以用户为核心，自主掌控数据。

深究原理我们会发现，Web3并不是互联网的对立面，反而它脱胎于互联网，是互联网诞生之初就希望构建的一个“开放互联网”。 只是经过20多年发展，互联网从业者并没有非常好的实现当初的理想。相反，在存在形式和技术架构上，似乎互联网越来越成为垄断主义城墙高筑的一座座围城。不过，前途是光明的，只不过还在曲折的道路中罢了！
