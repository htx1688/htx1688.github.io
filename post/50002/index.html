原文作者：Karen，Foresight News

3 月底，由 Polygon 打造的新产品 Polygon zkEVM 正式发布主网 Beta 版本。Polygon zkEVM 是一种 Layer 2 Rollup 解决方案，三个主要组件包括可信定序器（Trusted Sequencer）、可信聚合器（Trusted Aggregator）以及部署在 Layer 1 上的共识合约（Consensus Contract）。

其中，可信定序器组件负责接收 Layer 2 交易，并进行排序和生成批次，然后以序列的方式提交至共识合约的存储插槽（storage slots）中。可信定序器执行批量交易时还会将交易广播到 Layer 2 网络节点；可信聚合器组件根据定序器的 Layer 2 交易批次计算 Layer 2 状态，并获取由可信定序器提交到 Layer 2 的批次，并生成零知识证明来证明批处理的计算完整性。共识合约的逻辑会验证零知识证明，证明交易批次的正确执行，使得 zkEVM 能够继承 Layer 1 安全性。

整理了上线 Polygon zkEVM 的 12 个 DeFi 项目。在上线主网三周多的时间，已有主要包括跨链和 DEX 的协议上线 ，除了扩展过来的项目之外，多数 DeFi 项目还处于非常早期的阶段（建议用户谨慎参与），且在 NFT、元宇宙、游戏、借贷等领域鲜有触及。考虑到 Polygon 本来的生态相对繁荣，之后有待涌入更多的生态项目来 zkEVM 构建。

桥 / 跨链

当前，Polygon 官方桥仅支持从以太坊桥接至 Polygon zkEVM 或撤回，支持的资产包括 ETH、MATIC、USDC、USDT、DAI、Aave 等多个资产。除 Polygon 官方桥之外，支持 Polygon zkEVM 的跨链 / 多链协议还包括：

Celer Network

Celer Network 于 4 月 20 日宣布其用于资产桥接的 cBridge 已集成 Polygon zkEVM，最初支持在 Polygon zkEVM 和以太坊、Arbitrum、Optimism 和 BNB Chain、Polygon、Avalanche、Fantom 之间的 ETH 跨链桥接。

Layerswap

LayerSwap 旨在帮助用户资产从 CEX 转移到 Layer 2 ，目前已支持交易所与 Polygon zkEVM 间资产的出入金，以及 Polygon zkEVM 与其他多条主流公链 / 网络之间的跨链转移。

Multichain

跨链协议 Multichain 已支持 Polygon zkEVM 与其他网络之间的跨链转移，不过截止撰文时，在 Polygon zkEVM 上的资金池规模只有 0.3585 ETH。

Meson

Meson 是一个稳定币跨链交易协议，目前已支持 16 条链 / 网络上的资产跨链转移，包括以太坊和 Polygon zkEVM 等。

2022 年 8 月份，Meson 完成 400 万美元融资，GSRV 领投，BAI Capital、Plug and Play、Mirana Ventures、 M 77 Ventures、SNZ、A&T Capital、Smrti Lab 等参投。

Orbiter Finance 

Layer 2 跨 Rollup 桥 Orbiter Finance 于 3 月底开始支持以太坊、Arbitrum、zkSync、Optimism 等网络与 Polygon zkEVM 间 ETH 的跨链。

rhino.fi

rhino.fi 原来是 DeversiFi（此前定位是 Layer 2 去中心化交易平台），在 2022 年 7 月份宣布更名为 rhino.fi，并转型为多链 DeFi 聚合器，其桥功能也已在今年 4 月上旬支持 Polygon zkEVM。

DEX/AMM

Quickswap

Quickswap 是 Polygon 上原生的 DEX 协议，目前也已上线 Polygon zkEVM。根据 DefiLlama 汇总的 8 个 Polygon zkEVM 上线的协议统计数据，Quickswap 的锁仓量目前为 116 万美元，占据 Polygon zkEVM 总锁仓量的 57% 。

Uniswap

4 月中旬，「在 Polygon zkEVM 上启动 Uniswap V3」链上投票现已通过。

Antfarm Finance

Antfarm Finance 是一个多链 DEX 池优化器，目前已上线以太坊、Arbitrum 和 Polygon zkEVM。在 Antfarm Finance 上创建和加入流动性池时提供多种费用选择，包括 1% 、 10% 、 25% 、 50% 和 100% ，费用高的话可能会产生一些套利机会，不过滑点问题也需注意。

Antfarm 的实用代币 ATF 用于支付兑换费用，在任何兑换中， 85% 由流动性提供者共享， 15% 将被销毁。Antfarm 还有一个治理代币 AGT，持有者可参与决策过程。Antfarm 官网暂未披露团队背景信息，建议谨慎参与。

D 8 X

D 8 X 是一个 Polygon zkEVM 上的机构级去中心化永续合约交易协议，目前已上线 Polygon 测试网，抵押资产为 USDC 或 MATIC，提供的订单类型包括市价和现价订单，每笔交易中 AMM 为交易对手。

费用方面，D 8 X 收取三种费用，交易费用用来维持项目运营，限价单和清算由于是第三方执行订单，所以第三方会活动一定报酬；清算费用为要清算的头寸规模的 2% ，清算人赚取总清算费的 50% ，另外 50% 的费用将支付给交易协议。

DoveSwap

DoveSwap 由去中心化策略金库管理协议 Dovish Finance（Set Protocol 分叉）推出，是 Polygon zkEVM 上的一个原生 AMM 协议，目前的总 TVL 只有 7.8 万美元。DoveSwap 会在本周宣布启动治理代币 DOV。感兴趣的用户可以查看代币经济学。DoveSwap V3 采用集中流动性方式来帮助用户提升资金效率。Dovish Finance 官网暂未披露团队背景信息，建议谨慎参与。

LeetSwap

LeetSwap 是一个构建在 Canto 和 Polygon zkEVM 上的 DEX 协议。LeetSwap V1 DEX 于 2022 年 11 月 Canto 黑客松期间构建的项目，V1 中交易对的交易费用为零。LeetSwap V2 LeetSwap V2 于 2023 年 2 月 Canto 黑客松期间构建，LeetSwap 合约从 Solidly 分叉，并进行了一些优化，比如完全支持转账收费代币、动态交易费用和可选协议费用。

借贷

0 VIX

0 VIX 是 Polygon 和 Polygon zkEVM 上的一个借贷协议，提供针对多个资产的借贷市场。此外，此外，存入的资产可以作为抵押品，允许用户借入其他资产，通过存款赚取的利息可以抵消借款的累计利率。

0 VIX 顾问包括 Sushiswap 前核心开发者 Keno、QuickSwap 核心贡献者 Sameep Singhania、Polygon DeFi 和增长负责人 Hamzah Khan、Polygon Labs 首席信息安全官 Mudit Gupta 等。

根据官方网站， 0 VIX 当前在 Polygon PoS 上的供应量为 907 万美元，总借款量为 310 万美元，在 Polygon zkEVM 上的供应量为 122 万美元，总借款量为 57 万美元。

Aave

Aave 社区关于「在 zkEVM 主网上部署 Aave V3 的 MVP 版本」的 ARFC 提案已于三月底获得社区通过，不过尚需正式提案发布。
