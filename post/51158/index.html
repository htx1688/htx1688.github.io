作者：Nancy，PANews

伴随着BRC-20概念爆火出圈，相关Token总市值一路创新高的同时，也带动了比特币NFT生态的发展。Dune数据显示，Ordinals铭文总量已突破550.8万枚，迄今产生的总费用超1148枚比特币。不久前，Galaxy Digital研究部门也曾发布调研报告称，到2025年，基于铭文和序数的比特币NFT市场规模将达到45亿美元，空间巨大。

比特币生态走红下，根据Dune数据显示，Ordinals市场的交易量超过1.03亿美元，主要交易市场包括UniSat、Ordinals Wallet、Magic Eden、Ordinals Market和Ordswap等，其中前三个占据超96.6%的市场份额。本文PANews将为大家盘点当前部分热门比特币NFT，供玩家们参考。

Pixel Pepes

Pixel Pepes是发布在Ordinals上的项目，由比特币钱包服务商Ordinals Wallet运营，总量为1563个稀有Pixel Pepes被空投给在区块777888之前在该钱包上进行过交易的用户。

Ordinals Wallet数据显示，截至5月11日，Pixel Pepes的地板价为0.26 BTC，总交易额超104.5 BTC。

Sub 100k

Sub 100k是铭文号排名前1万的NFT合集，形象包含BAYC、CyptoPunk、Cool Cart、Meebits和Quacks等类似项目。Ordinals Wallet数据显示，截至5月11日，Sub 100k的地板价为0.005 BTC，总交易额超103 BTC。

BTC DeGods

BTC DeGods是Solana上的蓝筹NFT在比特币网络上发行的项目，该NFT系列共有534枚，其中500枚将用于比特币铸造，34枚DeGods用于以DUST代币的形式进行拍卖/抽奖。Magic Eden数据显示，截至5月11日，BTC DeGods的地板价为0.969 BTC，总交易额近84.6 BTC。

Bitcoin Frogs

Bitcoin Frogs由10000个生成青蛙组成，是比特币版的蛙蛙NFT，每一层内所有特征的稀有性都是平等的。Magic Eden数据显示，截至5月11日，Bitcoin Frogs的地板价为0.027 BTC，总交易额超44.8 BTC，持有地址约3300个。

Bitcoin Punks

Bitcoin Punks是首个使用Ordinals协议，并成功将以太坊上的CryptoPunk成功上传至比特币的NFT项目，允许玩家免费铸造，所有铸造在不到24小时内完成。

Ordinals Wallet数据显示，截至5月11日，Bitcoin Punks的地板价为0.097 BTC，总交易额超29.6 BTC。

JOMOHippo

JOMOHippo是由2600个像素河马NFT合集，旨在通过鼓励个人拥抱JOMO生活方式并找到内心的平静，来对抗加密领域常见的FOMO现象。JOMOHippo于今年2月28日在以太坊上启动了第一阶段，并于3月20日在比特币上启动了第二阶段。

Magic Eden数据显示，截至5月11日，JOMOHippo的地板价为0.01 BTC，总交易额超15.8 BTC。

UniSat OG PASS

比特币铭文市场UniSat Marketplace需用户持有UniSat OG PASS或者UniSat积分才可进行交易。UniSat Marketplace于今年4月上线不到1天就遭受大量双花攻击，后重新开放。

Ordinals Wallet数据显示，截至5月11日，UniSat OG PASS的地板价为0.44 BTC，总交易额超10 BTC。

DogePunks

DogePunks灵感来自于Ordinals上的Dogecoin和CryptoPunks，总量为1万个。Magic Eden数据显示，截至5月11日，DogePunks的地板价为0.006 BTC，总交易额超8.5 BTC。

Bitcoin Pizzas

Bitcoin Pizzas是比特币原生Meme文化的NFT项目。2010年5月22日，美国软件工程师Laszlo Hanyecz支付1万枚比特币购买两个pizza，因而这一天被称为Bitcoin Pizza Day。Magic Eden数据显示，截至5月11日，Bitcoin Pizzas的地板价为0.052 BTC，总交易额超7.2 BTC。

TwelveFold  

TwelveFold是由BAYC母公司Yuga Labs基于Ordinal协议发布的NFT系列，包含300件限量版生成艺术作品，每个NFT结合3D图形和手绘特征，由Yuga Labs内部艺术团队设计和创作，TwelveFold与其任何一个以太坊链上IP均无关联，未来也不会有任何交互。

在TwelveFold拍卖中，排名前288位的投标人出价均在2 BTC以上，最高出价为7.1159 BTC。另据Yuga Labs首席创意官schmigge figge透露，TwelveFold系列分为冬季、春季、夏季和秋季四个季节，可以细分为25个独立的系列。每个系列中的所有12件作品都是相关的，按顺序排放。所有文件大小都在229kb到114kb之间。Yuga Labs将会把部分销售收益进行再投资，以继续探索比特币区块链。

Taproot Wizard

Taproot Wizard是由加密开发者Udi Wertheimer主导的比特币NFT项目，旨在纪念比特币的Taproot升级及“比特币巫师”形象。今年2月，比特币矿业公司Luxor挖出了有史以来最大的比特币区块，该区块大小为3.96MB，包含Taproot Wizards NFT。不过，据The Block报道，该事件似乎是Luxor和Udi Wertheimer之间的某种合作。

BTC Machine

BTC Machine属于铲子型NFT项目，陆续空投了BTC Operators与BTC Virus两个项目。Magic Eden数据显示，截至5月11日，BTC Machine的地板价为0.234 BTC，总交易额超2 BTC；BTC Operators的地板价为0.05 BTC，BTC Virus则接近0.003 BTC。
