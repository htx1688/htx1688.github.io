前言

“Bitcoin——用于结算金融资产的全球储备货币

Liquid—— 未来全球金融体系的支柱”——Blockstream

比特币的诞生是去中心化金融的开端，它为传统金融体系带来了一系列的革命。然而，尽管比特币本身的安全性和去中心化令人印象深刻，但它同样也面临着一系列的挑战，比如其扩展性以及处理大量链上交易的能力，而这恰恰就是Liquid Network出现的原因。Liquid Network不仅是比特币的侧链，还是一种创新性的技术解决方案，旨在改进比特币的速度、隐私性和可扩展性，同时还能够赋予用户更多的金融自主权。本篇文章将深入探究Liquid Network，包括其核心功能、技术运作和安全性、目前的发展状况以及未来的前景展望。

核心功能

Liquid Network不仅是比特币的侧链，还是一个具有多重核心功能的去中心化金融工具。这一部分将详细探讨 Liquid Network 的核心功能，包括快速结算、隐私性、数字资产发行以及与比特币锚定的关系。

1. 快速结算

Liquid Network的一个显著特点是其快速结算能力。这对金融市场中的高频交易和快速变动的市场条件至关重要。具体而言，Liquid Network 具有以下两个特点：

  • 不变的区块时间：Liquid Network的区块时间是一分钟，相对于比特币主链的较长区块时间，这意味着交易可以更快地被打包和确认。这使得Liquid Network非常适合需要迅速结算的交易场景。

  • 两个区块的交易确定性：与比特币主链的六个区块交易确定性相比，Liquid Network 仅需要两个区块来确认交易。这降低了双重支付的风险，使用户能够更快地进行交易。

Liquid Network的快速结算功能对金融市场的高频交易至关重要。交易员可以利用 Liquid Network 的一分钟区块时间来更快地执行订单，从而实现更高效的套利和交易策略。这种优势使得金融市场更具竞争力，交易员可以更迅速地响应市场波动。

2. 隐私性

隐私性是Liquid Network 的另一个关键特性。在传统比特币交易中，所有交易的数量和资产类型都是公开可见的。相比之下，Liquid Network 提供了更高的隐私保护，具体表现在以下方面：

  • 机密交易：Liquid Network 使用了机密交易技术，使交易的数量和资产类型对第三方不可见。这有助于保护用户的财务隐私。Liquid网络协议经过精心设计，确保了资产的类型和数额对于除了发送者和接收者之外的任何人都是不可见的。这一强大的隐私特性是通过Blockstream开发的先进密码学协议"机密交易"来实现的。这对于交易员来说具有极大的重要性。以比特币区块链为例，如果他们需要转移大额资金，传统的区块链分析工具可能会监测到这种活动，从而导致市场在交易员有机会执行交易之前就做出反应，影响资产价格。但由于Liquid交易的高度保密性，交易员可以放心地进行资金转移，而无需担心他们的交易意图会被泄露。这为用户提供了更大的交易自由度和安全性。

  • 资产类型的保护：Liquid Network不会公开披露交易中涉及的资产类型。这意味着用户可以在Liquid Network上进行多种数字资产的交易，而不必担心他们的资产类型被揭示。这对于金融机构、投资者和交易员来说尤为重要。Liquid Network 的隐私性大大降低了金融市场中大宗交易的前沿风险。Liquid Network 的隐私性功能可以确保大宗交易的交易金额不被第三方可见。这意味着其他市场参与者无法在交易执行前获取有关交易的信息。大宗交易方可以更自信地进行交易，而不必担心信息泄露。

  • KYC：Liquid Network与任何涉及金融的网络一样，KYC程序是一个备受用户关注的话题。在Liquid Network中，内置的KYC程序并不是强制性的。这意味着任何人都可以使用比特币核心（Bitcoin Core）和Elements全节点将比特币锁定到Liquid Network上，而无需进行KYC验证。这一政策也反映了比特币生态系统中的去中心化和匿名性原则。然而，根据 Liquid Network 上的服务提供商和使用情况，有可能会要求用户提交 KYC 信息，以便满足当地监管要求。对于 peg-out 操作，即从 Liquid Network 取回 BTC 到比特币主链上，一些平台可能要求用户提交 KYC 信息。这是出于确保交易的合法性，以及遵守国际安全法的需要。同时，这也是为了保障代币持有者的投资权益并为其提供适当的保护措施。尽管在这些特定的情况下需要 KYC，但用户仍然可以在 Liquid Network 上以匿名方式进行交易某些资产。Liquid Network为用户提供了选择参与KYC的自由，同时保护了他们的交易隐私。这种平衡使Liquid Network成为金融和数字资产领域的创新解决方案，旨在提高交易效率、维护隐私和合规性。

3. 数字资 产发行

Liquid Network允许个人、机构和公司发行各种数字资产，包括稳定币、安全代币以及其他金融工具。这一功能为BTC生态的金融创新提供了更多的可能性，使用户能够将各种资产转化为数字形式并进行交易。与 Liquid Network 上的其他资产一样，发行的数字资产也可以享受快速的交易确认和高度的隐私性。目前，Liquid Network 已经支持多种不同类型的资产发行。以下是 Liquid Network 上可以发行的不同资产类型的一些示例：

1.稳定币：Liquid Network 提供了一个理想的平台，Tether的USDT稳定币已经在LiquidNetwork网络上发行。这些稳定币的价值与相应的法定货币（例如美元）挂钩，使其成为跨境支付、跨链交易和杠杆交易的理想选择。稳定币的发行可以通过简单的资产锁定和认证过程完成。

2.安全代币：Liquid Network 支持安全代币的发行，这些代币代表着各种金融工具，如股票、债券和其他证券。例如，Blockstream Mining Note 和 El Salvador Bitcoin Bond 都是在 Liquid Network 上发行的安全代币。这些代币可以更容易地进行交易，而且具备与 Liquid Bitcoin 相同的高效结算和隐私性。

3.数字收藏品：对于游戏和数字收藏品市场，Liquid Network 提供了一种便捷的方式来代币化虚拟资产。例如，在线战斗游戏 Lightnite 和 MMO 太空战略游戏 Infinite Fleet 都选择在 Liquid Network 上代币化其游戏内物品。

4.其他资产：Liquid Network 上还支持发行各种其他资产，如奖励积分、代金券等。这些资产可以根据不同的用例和需求进行定制发行。

4. 比特币锚定关系

Liquid Bitcoin：L-BTC，是 Liquid Network 上的一种特殊资产。它的特殊之处在于，其供应量与比特币（BTC）一一对应。这意味着，每当有人将 BTC 转移到 Liquid Network 上时，相应数量的 L-BTC 将被创建。同样，当有人从 Liquid Network 中取回 BTC 时，相应数量的L-BTC将被销毁。这种一对一的关系确保了 L-BTC 的价值与比特币相等。

L-BTC 的存在是为了在 Liquid Network 上提供更高的交易速度和隐私性。交易员和用户可以将比特币移动到 Liquid Network 上，以便更快速、更隐私地进行交易。此外，L-BTC 也用于支付 Liquid Network 上的各种交易费用。

比特币的双向锚定：比特币的双向锚定是指BTC可以在比特币主链和Liquid侧链之间“转移”。将BTC从比特币主链转移到Liquid侧链称为“入金”，涉及将BTC发送到由Liquid Federation控制的多重签名钱包。一旦确认，发起入金的用户可以索取相等数量的L-BTC。任何拥有Liquid节点的人都可以完成入金。

将BTC从Liquid侧链转移到主链称为“出金”，涉及到Liquid Federation成员销毁他们持有的L-BTC。一旦确认，比特币联邦的多重签名钱包将向成员的白名单地址之一发送相等数量的BTC。为了增强Liquid网络上持有的BTC的安全性，目前只有Liquid Federation成员可以完成出金。

总的来说，Liquid Network提供了一个多样化的资产发行平台，使发行者能够将各种资产带入 Liquid Network 生态系统中，利用其高效的结算速度和隐私性功能，为金融和数字资产领域带来了新的机会和创新。不仅可以代币化传统金融工具，还可以探索数字收藏品和虚拟资产的新领域。 Liquid Network 为资产数字化提供了一个灵活、高效和安全的解决方案。

技术运作

Liquid Network的技术运作方式相当独特，其核心特征包括 Liquid Federation 和区块签名。

Liquid Federation

Liquid Network由一组称为 Liquid Federation 的成员共同管理。这些成员包括主要的交易所、交易台、钱包提供商、支付处理服务、金融机构等 Bitcoin 领域的重要公司。这些成员在全球范围内分布，使得 Liquid Network 具有去中心化的特性。然而，需要强调的是，Liquid Network 的每个成员都不具备单一的控制权。相反，他们通过多签名智能合约协作，确保 Liquid Network 的稳定性和安全性。

区块签名

Functionaries 设备

在Liquid Network 中，区块的生成不依赖于工作量证明（Proof-of-Work），而是通过 Liquid Federation 中的 15 名特殊硬件单位Functionaries轮流签名来完成。在一个循环中，这15个Functionaries轮流提议区块。只有当其中11个Functionaries验证并签署了一个区块时，该区块才会被添加到链上。这些硬件单位分布在世界各地，构成了 Liquid Network 的“心脏”，负责签署交易、生成区块以及保护 Liquid Network 中锁定的 BTC。Functionaries可以说是Liquid网络的核心，它们扮演着至关重要的角色，提供了网络连接、生成区块以及保障比特币的双向锚定。

硬件安全模块（HSM）

每个硬件设备Functionary上的HSM用于保护签署Liquid网络区块以及联邦多重签名钱包的15个密钥之一。HSM可以防止大于一个区块的链重新组织。它通过跟踪最后一个签署的区块的区块高度以及先前区块的哈希来实现。HSM只在两种情况下签署区块：

  • 如果前一个区块相同，那么区块高度必须相等。

  • 如果前一个区块不同，那么区块高度必须高于上一次签署的前一个区块的高度。

HSM绝不会签署低于上一次签署高度的区块。此外，HSM还验证并签署出金交易，但在签署之前会确保满足某些条件，例如，交易输出必须返回到联邦或白名单地址。

综上所述，Liquid Network 的安全性是由其联合管理模式（Liquid Federation）和多签名智能合约保障的。Liquid Federation 成员分布广泛提高了 Liquid Network 的抗攻击性。此外，Liquid Network 中的交易数据受到机密交易（Confidential Transactions）的保护，使其难以被监视或追踪。

Liquid Network 现状及展望

自2018年底推出，Liquid Network已经取得了显著的增长。Liquid Network 的网络容量是衡量其采用情况的一个关键指标，2022到2023这一年期间里，网络容量一直维持在3,500 BTC左右，这显示了越来越多的用户和实体正在选择将比特币锁定到 Liquid Network 上。

目前流通的L-BTC数量

BTC二层网络互通：由于比特币网络拥堵导致的高交易费用，市场对于二层解决方案需求不断增加。Blockstream green 团队正在将各种btc二层网络整合，使用户能够在同一平台上管理BTC主链、Liquid和Lighting。随着BTC生态的不断发展，Liquid作为BTC生态上核心的金融资产二层，与Lighting作为BTC生态的通用桥梁之间的无缝结合将会支持更多种类的去中心化金融服务等。这将扩大 Liquid Network 的适用范围，使其在更多领域发挥作用。
