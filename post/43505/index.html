 摘要：在我们上一篇介绍 DeSci 的文章中，我们从较为宏观的角度详细地介绍了 DeSci 乃至整个自然科学的发展历史，从文艺复兴到启蒙运动，再到现在的 Web3.0 革命，可以说虽然狭义的 DeSci 是从最近的 Web3.0 革命开始，但广义的 DeSci 运动和精神却从自然科学诞生的一刻起就深深根植于人类文明的内心深处，我们相信 DeSci 运动是一种基于区块链技术和 Web3.0 行业的伟大社会实验，它有希望改变整个自然科学的底层架构。本文将以目前数个最具代表性的 DeSci 项目的出发，从更加具象化的视角和简练的角度来阐述我们所认为的 DeSci 运动的起源和未来。

一、DeSci 所解决的问题

经费问题

传统的科研是以中心化的方式来寻求科研经费以及资源，如国家自然基金，这使得中心化的科研方式在经费申请过程中有大量的问题，如少数学阀控制和垄断整个经费申请；其次是申请经费的整个流程冗长和效率低下。以中国为例，国内的经费周期往往是从年初开始写申请书，到年末才结题和审核，大量的时间用于申请的手续和考核流程，另外在考核和审批过程中也存在着相当的问题：如对科研项目缺少全面高效的审批机制，往往一个项目申请科研经费的时候，进度已经超过一半，科研经费很难落实到科研真正最需要的地方；其次，审核机制单一，如现在中心化科研经费审核中期刊文章和专利发表是审核中心化科研经费的重要参考点，基于此，期刊文章和专利的发表成为刚需，甚至部分科研杂志明码实价形成产业链，这种单一且不合理的审核机制导致期刊文章逐渐功利化，偏离科研方向初心。再者，由于大环境的恶化如经济恶化和环保原因，也导致整体科研经费逐渐降低，科研经费问题日趋严重。

建立更加公开公平的环境，对科研成果确权

在科研圈往往存在着一定的资源和要素分配不合理的情况，对科研资源，人员存在着冗余，浪费甚至内耗严重的情况，基于区块链技术的 DeSci 由于公开透明数据不可更改的特性，在一定程度上可以优化这种资源和要素分配，并且在一定程度上帮助科研人员确权，这将有助于优化整个科研环境，促进成果诞生。

消除中心化机构垄断

如同前文所述，自古以来，科研行业的话语权往往掌握在中心化的基金或者权威期刊手中，甚至有出现「学阀」等学术垄断，学术「黑帮」现象，长期以来存在着以不透明不公正标准来衡量学术成果，乃至控制学术圈的情况，基于区块链技术的 DeSci 以一种去中心化的方式将审核权话语权下放，这不但有利于消除中心化机构的垄断，减少中心化作恶的现象，也将有利于资源优化配置，促进科研行业与各行各业的合作与发展。

促进公民科学运动

公众科学（Citizen Science），又称公民科学，描述的是公众和职业科学家之间的合作关系。通常，公众科学指的是公众成员参与收集、分类、记录或分析科学数据的项目。其中「公众」包括非职业的科学家、科学爱好者和志愿者等。公众科学项目一般由科学家发起，邀请公众参与并共同开展。也有项目由公众策划、发起，然后邀请科学家作为技术指导。这种运动旨在号召整个社会共同参与研究，合理分配社会资源，该运动和 Web3.0 精神不谋而合，基于区块链的 DeSci 技术将有为该项运动提供技术的可能。无论如何，DeSci 都将从大局上促进科学科研运动的平民化和普适化，这将促进整个科研行业的发展。

二、目前具有代表性的DeSci 概念项目

在本段中，选取了数个笔者所认为的较有代表性的 DeSci 项目，所评选的维度有：项目生态（市值、融资历史、功能性、重要程度、合作方和孵化项目数量以及整体行业地位等），项目影响力（日活、社交媒体热度等），项目更新频率，项目上线时间，团队情况等。目前 WIKI 收录了较为完整的 DeSci 生态（如下图），整个 DeSci 赛道还处于非常早期，而在目前的 DeSci 的生态版图中，以 DAO 为最繁荣，这应该是由于 DAO 是为了解决目前 DeSci 最大也是最需要直面的融资难题。其次也是专注于去中心化融资的板块，然后是为了解决 DeSci 第二大难题的同行评议的痛点的出版和论文文献审核平台概念。而其他的如 NFT 相关的组合概念，都还处于比较蛮荒早期的状态。目前的 DeSci 项目里倾向于生物赛道的最多，这可能是由于本身科研生物方向本身的数量较多导致的以及近期疫情因素普遍公众对生物科学的关注度提高有关。

本段分别从文献出版和同行评审，DAO 概念和本赛道的底层基础设施里选出了部分具有代表性的项目分析探讨。其中，DAO 概念和生态底层设施以 Molecule 生态最为完整，影响力最大，笔者建议如果有感兴趣的读者可以从该项目了解切入。

1. Ants-Review：类 Bounty 的开放性匿名科学行业同行评审系统

Ants-Review 来自于 ETHTurin 2020 黑客松上，由 MetaBounty 团队提交。Ants-Review 项目获得了当时的黑客松一等奖。并且获得了都灵大学、Gitcoin、以太坊基金会的投资。（ETHTurin 是以太坊社区的第一个意大利黑客马拉松，黑客可以在这里开发基于以太坊的区块链应用程序，以解决本地影响和可持续发展目标目标）。

同行评审是科学出版物必不可少的质量控制步骤。为了解决科学出版物同行评审昂贵，效率低并且中心化的问题，Ants-Review 提出了一个基于区块链的激励协议，奖励科学家对其他科学家工作的贡献，并建立声誉体系。AntsReview 协议允许任何作者发出请求，要求同行评审他们的科学出版物，并且在该评审的过程中将保持评审者的匿名性。值得一提的是，Ants-Review 中关于 Ant 的命名来源于创始人认为一篇科学论文背后的工作类似于一个如蚁丘复杂的有机体，是许多微小个体的总和：在该总和里面，所有的贡献（即便如果蚂蚁一般微小）对整体来说都是必不可少的，值得承认。

Ants-Review 甚至创造性的提出了 PoC 概念（Proof of Concept）: 在评审过程中，满足 Ants-Review 智能合约要求的审查由外部编辑审核，外部编辑验证内容。如果审核被作者接受，审核人员将可以获得由 Ants-Review 发行的 ANT 代币，ANT 代币为 ERC20 格式，智能合约则由 Solidity 语言编写。

目前 Ants-Review 项目尚处于早期，并且还有一些关键问题（如如何确定同行评议审核的资格；如何避免刻意作恶；如何避免作弊等）有待解决，但 Ants-Review 这一协议仍然展示了 Web3.0 中 DeSci 如何建立初始的经济激励模型。在未来，Ants-Review 会发行 ANT Token，并且在 Ants-Review 中加入零知识证明协议，以实现匿名化的同行评审，也将加入个人 DID 功能以形成个人声望并集成 DEFI，模块化协议允许扩展和改进逻辑等。

2. Molecule：去中心化的生物技术底层协议

Molecule Project 项目是一个生物技术研究的筹款平台。目前，约有 40 名科学家在该网站上为他们的项目寻找资金。任何人都可以捐赠和支持未来的重要发现。在创立之初，Molecule 受到了公民科学运动和药物开发开放化的启发，最初的想法来源于患者社区，旨在跨越理论与临床研究的死亡谷。项目最初的想法是利用区块链工具帮助推进和资助早期药物开发。为此，Molecule 公司做出了几项决定，这几项决定推动了 DeSci 领域的发展，其中包括构思知识产权 NFT、与知识产权相关的 NFT（药物或基于研究的知识产权），以及启动 VitaDAO 社区以资助长寿研究。

Molecule 建立在以太坊之上，总共包括两层系统，分别为 Molecule Discovery 和 Molecule Finance。它们可以为分散的模块化药物开发创建一个充满活力的生态系统。

Molecule Discovery：在 Molecule Discovery 上，生物制药资产的信息是允许公开的。也就是说，研究人员可以让全球范围内的用户看到他们的生物制药资产（一般包括数据和预先申请的专利），并与潜在的资助者互动。该平台允许他们列出项目的详细信息，有兴趣的投资者可以发现和关注他们感兴趣领域的项目，并与研究团队协商合作，进行进一步的交互。

Molecule Finance：通过创建虚拟资金和协作环境来实现模块化药物开发。NFT 资产可以转移到自动做市商中以筹集资金，或者插入到作为特定疾病治疗垂直领域的投资组合管理的 DAO 中。这些结构几乎可以等同于由各种全球利益相关者、资助者和研究人员组成的生物制药初创公司或控股公司，但与其不同的是，实际的治理权将完全分配给社区。

IP-NFT：

Molecule 采用知识产权 NFT (IP-NFT) 的新颖方法使任何人都可以共享研究项目的所有权以及未来成果，例如知识产权、版税、数据等等。Molecule 开发的 IP-NFT 主要由两大合约组成：法律合约和智能合约。它们之间相互交叉、互相作用。法律合同是 IP 许可证，其语言参考区块链交易、地址和签名。智能合约是一个 NFT，其代码引用 IP 许可协议，并将其与某些数据组件混合后存储在去中心化文件存储网络上。两者结合起来，就形成了 IP-NFT。IP-NFT 是一种利用分布式账本技术来管理知识产权的全新机制。IP-NFT 的目标是在一个开放和分布式的市场中实现对 IP 的融资、拥有和交易，同时保留与非注册 IP 相关的隐私和开发潜力。

IP-NFT 代表了对生物制药研究的完全合法知识产权和数据访问控制权限。但更重要的是，它们还代表了一种全新资产类别的先驱——虚拟化 IP 类 NFT。可以说在这方面，Molecule 是将产权和相关信息数据 NFT 化的先驱。

借助 IP-NFT，筹款人员，生物医学研究人员和生物技术公司可以：

  • 无需及早申请专利或创建初创公司即可进行商业筹款 

  • 直接与患者等利益相关者团体互动，与其他研究人员合作促进开放科学，并通过公共市场获得关注

  • 利用数据访问控制合作研究任务 

  • 创建并实市场的融资 / 货币化策略

Molecule 网站架构

关于 Molecule 生态：

Molecule 建立在模块化的开放堆栈上，利用各种 DeFi（Balancer、Gnosis）和治理组件（The LAO、Moloch）以及法律框架（OpenLaw）来搭建自己的生态。在上述这些功能之外，通过 Molecule Discovery 还可以进行资产采购、资产评估并且包含临床数据托管等基础设施。Molecule 并不局限于自身网站的基础功能，而是建立了完善并且庞大的生态系统，使患者集体和药物开发 DAO 能够通过 NFT 持有、治理现实世界的 IP 并与之交互，加上在 IP-NFT 方面的创新（例如通过将 NFT 集成到去中心化拍卖系统和 AMM（如 Balancer、Uniswap 或 Sushi）中的资金框架），有人认为 Molecule 更像是 DeSci 赛道的 OpenSea，但在笔者看来，宏观上 Molecule 坚持播种新的 DAO 和旗下 DeSci 项目，包括 VITA DAO（第一个生物技术 DAO）ValleyDAO AthenaDAO HairDAO 等，不仅帮助推进早期药物开发的资金，也从前中端同时帮了 DeSci 的发展，如搭建 DeSci 赛道基础设施如 bio.xyz（DAO 的 launchpad）。Molecule 的生态正在逐步完善，布局宏大，我们有理由相信 Molecule 在未来将成为 DeSci 概念的领军项目。

在今年 6 月 13 日，Molecule 完成了 1300 万美元的种子轮融资，该轮融资由 Northpond Ventures 领投，Backed VC、Shine Capital、Speedinvest 以及前 Coinbase 首席技术官 Balaji Srinivasan 参投。在今年 9 月 1 日，辉瑞表示与去中心化长寿研究组织 VitaDAO 达成战略合作，并向其投资 50 万美元用于延长寿命的新药研发项目。

3. VitaDAO

VitaDAO 是 Molecule 旗下最著名的社区管理和去中心化药物开发的 DAO 集体。同时也是世界上第一个生物技术 DAO。VitaDAO 的核心使命是加速长寿空间的研发，延长人类寿命和健康寿命。为了实现这一目标并协调激励措施并激活长寿生物制药的早期资金，VitaDAO 结合了新颖的治理框架和去中心化自治组织 (DAO)、NFT 以及算法自动化做市商 (AMM) 等金融工程工具在以太坊区块链上运行，并且以 IP NFT 的形式共同资助和数字化研究。

VitaDAO 以人类历史上最长寿人的寿命秒数做单位，发行了同等数量的代币。并承诺，若人的寿命极限可超越此记录，他们就以新记录为基准增发代币。代币 VITA 可以通过向 VitaDAO 贡献工作、资金或其他资源（如数据和 IP）来获得。VITA 的所有权允许持有者参与 VitaDAO 的民主治理，指导其研究，访问其数据存储库并从中获利，并管理其 IP 组合。

VITA 不授予知识产权所有权而是授予完全治理权，VITA 的设计遵循可持续性循环原则，即随着研发项目获得资金并开始产生数据，VitaDAO 的 IP 价值随着积极的研究成果而增长。成功的商业化需要 VitaDAO 生态系统的发展和更大的资金可用性。这反过来又吸引了更高质量的 IP，从而为更多项目提供资金并进一步发展 VitaDAO 生态系统。

公众成员可以通过贡献资金、工作或贡献有价值的研究数据或 IP 资产来加入 VitaDAO 并通过 VITA 代币管理其 IP。VITA 代币使其持有者能够参与 VitaDAO 研究的决策和治理，表示对特定计划的支持，并管理其数据存储库和 IP 组合。

VitaDAO 将收购和委托研究，以及拥有、开发和货币化由此产生的知识产权资产。VITADAO 的产品组合包括：

  • 代表治疗研究项目的知识产权、专利和许可的 NFT。

  • 围绕其研究项目和 NFT 资助研发产生的数据资产。

为了资助更多的研究并为 DAO 运营提供长期资金，有多种选择可以将拥有的数据和 IP 货币化。其中产权 IP 的最终归属是可以商议的，目前大部分归属于 VitaDAO，小部分属于作者和发起人。

目前为止，已经有超过两百个 DeSci 项目通过 Vita DAO 募集资金，募资金额超过三百万美元。

 8 月 30 日，辉瑞表示与去中心化长寿研究组织 VitaDAO 达成战略合作，并向其投资 50 万美元用于延长寿命的新药研发项目。

4.LabDAO 致力于 DeSci 基础工具与设施的 DAO 组织

LabDAO 是一个由科学家和工程师组成的社区，致力于共同分享生命科学研究的开放工具。该社区的使命是通过使其所有成员都可以使用计算和科学工具来加速生命科学的进步。

LabDAO 希望生活在一个能够释放个人开发技术潜力的世界，使得研究者与组织的研究能力不受资金、软硬件和基础设施限制的世界中。LabDAO 共同构建的工具旨在降低发明者筹集资金、开发技术并以无门槛方式分享技术的障碍。

在过去 20 年中，开源代码加速了信息技术的进步。以低成本共享代码并以高可靠性重现结果的能力使这种成功成为可能。此外，技术市场动态加速了代码的共享，在这种动态中，知识产权与上市时间和实施相关的商业秘密的相关性较低。相比之下，生物医学中的开源一直没有那么具有变革性。复制生命科学过程的高成本和生成预期结果的低可靠性一直是开源市场动态的障碍。此外，共享指令受到知识产权和商业秘密的阻碍，这是在漫长的监管审批过程中抵御竞争对手的主要方式。

为了加速这一进程，LabDAO 开发了诸如 Openlab 之类的工具，使科学家能够相互分享他们的专业工具和服务。

关于 Openlab：

Openlab 使得科学家们可以在全球市场上不受限制的交换工具和服务，而不是漫长地寻找合作者或合同研究组织。作为 DeSci 重要的底层基础设施，Openlab 使每一位生物技术科学家、研究生、博士后和教授都能成为独立的发明家或企业家。

三、对于 DeSci 未来可能发展路径的展望

基于对 DeSci 运动现状以及区块链当前应用的分析，我们得出了以下几个 DeSci 未来可以发展的方向和具体应用方案：知识产权确权：学术声誉系统；资源再分配：行为激励机制；数据储存、加密与共享：数据流转中心。

1.知识产权确权：学术声誉系统

当前学术文章及专利的确权以及后续应用的版权费用流转是十分契合区块链透明且不可篡改的特性，而且现在已经存在相对成熟的基于 NFT 的版权确权与版税流转方案。同时，基于用户交互行为而建立的链上社交图谱和声誉系统，如 CyberConnect、ProjectGalaxy，也在快速发展。因此，我们为其提供了一个应用方案：

首先所有的地址都可以通过上传链下身份证明认证自身的身份，并获得一个可更新的 Soulbound Token（SBT）新标准作为证明凭证，从而将地址与个人身份绑定，以实现链上与链下声誉的同步。基于该身份 SBT，我们可以建立一个学术声誉系统：在这个系统中，所有的学术交互行为都会获得相关的学术 SBT 凭证，该凭证也是可以更新的，一个地址的身份 SBT 会基于该地址所拥有的学术 SBT 凭证的更新而进行更新，因此生态中的其他项目只需读取该 SBT 即可了解对应作者的学术成就，而不是获取该账户所有的学术凭证。而事实上，不仅仅学术文章，任何知识类输出都可以按照这个逻辑进行。

以学术文章发表为例，用户在链上发布学术文章可以获得一个对应 NFT 以确权和一个 SBT 以记录，同时在该文章发表过程中负责的编辑、进行评议的审稿人、推荐人、出资人、甚至数据收集者等都会获得一个对应 SBT 以记录其相关性，而文章作者也可以自行设置 NFT 版税分配，或任何其他盈利分配，如获奖的奖金分红等。每篇文章都会被反应在身份 SBT 上，从而会被声誉平台识别，给予对应的分数来为该行为赋予价值。此外，每一个对该文章进行引用的行为都会反应到该文章的 SBT 和相关贡献 SBT 上，以证明持有人影响力的改变，进而建立其每个人的学术关系图谱。由于 Web3 数据个人化及区块链透明化的特性，这将改变传统学术文章从创作到发表过程繁琐且利益分配不均的现状，将每个环节去中心化，但又能明确每个参与者的作用。

当一个地址的学术声誉得以建立，用户可以通过 SBT 或某个学术 NFT 凭证进行金融化操作。例如，用户可以基于自身 SBT 进行信用借贷，以其学术声誉作为信用基础获得无抵押贷款，又或是抵押其学术 NFT，以该 NFT 自身价值和未来版税或收益分红作为背书，借出流通资产。保守一点的话，也可以根据用户声誉或学术 NFT 提升用户资金的抵押率。另外，用户还可以通过 NFT 碎片化将 NFT 权益等比例拆分，以此作为权益凭证进行募资。而知识产权的租用或买断等商业化行为也可以通过智能合约与 NFT 实现去信任化，避免知网一类严重侵害作者的霸王条款。

2. 资源再分配：行为激励机制

该方案意在解决学术研究的资源分配不均的问题，该资源不一定是资金、设备或人际关系，还可以是环境资源、气候条件、文化遗产等。借鉴当前长尾资产市场的方案，我们可以建立一个资源再分配平台，它是如 PartyBid 一般的项目募资平台为学术研究提供资金，同时换取碎片化 NFT 作为权益证明；它也如 Keep3r 一般的任务发布平台以 Token 或其他资源作为报酬寻求帮助，并以 SubDAO 形式进行一个 Topic 的分布式研究；此外，它是一个像 Abacus 的同行激励评估打分平台，通过信誉系统利用多节点评估方式保证公平和激励。因此，我们为其设计了一个方案雏形：

该平台的功能被分为三部分：募资、合作与评估。而这个平台的关键就是将三个部分与激励和声誉系统结合，实现公共物品的价值化和长周期事物的短期收益。

具体来说，任何人都可以在平台开启募资，募集资金将被锁定在平台，募资人以提案方式申请使用资金，并由投资人投票决定是否为不通过，当选择不通过的募资人数超过总人数 10% 时，将进行正式投票，否则将直接予以通过。而募资人将其产出的作品的 NFT 作为报酬，以碎片化或 ERC-1155 标准的 NFT 形式分发给投资人，赋予其部分所有权和分红权，同时可以附加如使用权、商业化等额外权力。特别的是，投资人商业化产生的收益，同样会有一定比例回流到作品 NFT，而不是广撒网就能一劳永逸利用别人产出盈利。 这样贡献者和出资人都能从中获益。

而合作模式就像是现在的维基百科一样，大家通力合作完成一个课题，只不过所有贡献者都可以拿到与自己贡献相匹配的报酬，这将定义出每一个贡献者，并为公共商品的产出定价。比如，一个新加坡的团队研究某种红树林的生长速率与水质和洋流的相关性。由于新加坡的地理因素限制，该研究需要对其他区域，如马来西亚、印度尼西亚、泰国等的红树林进行研究。如果该团队选择自己实地考察，不仅需要解决合规性等问题，还需要考虑时间和人力等。所以，该团队可以在这个平台发布合作需求，并提供对应报酬（Token、法币、该文章 NFT 碎片等），这将为当地红树林保护提供一定的资金支持，从而产生正向效应。而作为一个大型课题，其必然会被分解为数个小课题，然后由不同的博士与研究生甚至是本科生共同完成。当这样的分解不再受地理和国家地区限制，那么整个课题的可行性和完成效率将直线上升，同时还能将优质学院的研究资源共享给其他区域相对落后的研究团队，带动其学术发展。

同行评估则是借鉴了当前 NFTFi 的解决方案，利用多节点质押 Token 共评，通过激励和罚没来实现高效公允的评价。与 Abacus 类似，多个评估节点需要质押相应数量的 Token 作为作恶成本，对同一作品进行打分评估，分数会根据每个人声誉进行加权，与最后分数较为一致的评估节点将会获得由评估申请人提供的激励。而整个过程是由质疑者所监督的，一旦发现有恶意打分或女巫攻击，将可以重启评估，并获得作恶者被罚没得 Token 作为激励。此外，为了促进整个匹配流程的进行，会有一个像传统编辑一样的角色匹配各位评估节点和申请人，我们姑且叫他策展人，当作品打分成功，他们将会获得申请人激励的一部分。由于对作品的认知是实时变化的，而作品的价值也可以持续产生，因此，整个打分过程可以持续进行。每轮打分结算结束后，申请人可以用额外的激励申请额外评估，评估节点可以通过额外质押 Token 来更改其打分，这时其他节点也可以受邀来进行额外的打分。

3.数据储存、加密与共享：数据流转中心

学术数据的储存是一个一直以来都存在的问题。关键数据的丢失将会导致整个项目的流产，所以需要有一个可靠的储存设备且留有备份。但一旦机密数据泄漏又会使得的这个项目被抄袭或盗用，也会让项目止步于此。可是大多数数据是需要被及时分享并能够被验证真实性的，这就形成了一个不可能三角。而当前的区块链技术却有现成的成熟方案可以借用，即 IPFS、ZK 与 query/API。这三个技术可以打破数据孤岛、保证数据安全并实时同步数据以验证真实性。

基于 IPFS、ZK 和 API 实现的数据流转中心作为一个资源整合平台，并不实际构建服务，所以不托管用户数据，而是为用户寻找合适的去中心化数据储存服务并根据用户需求代收储存费。所有数据都会被加密以保证隐私性，并且通过 zk 技术实现数据验证。由于所有检测数据都会被实时上传，所以非机密数据都可以及时被共享给需要的用户。而数据 API/query 访问门槛则可以由数据所有人设定，以实现基于数据的盈利。更重要的，数据本身的真实性和唯一性可以通过区块链特性得以保证，同时完整的原始数据也使得数据审查与实验再现变得更加简单且可信。

DeSci 生态系统：

当三个方案被整合时，将能建立一个全面的 DeSci 生态网络。数据流转中心为该生态搭建了一个必不可少的基础设施，以解决最根本的数据问题，保证了数据安全，打破了信息不对称。当一切信息透明时，一个自由市场才能够得以建立。而声誉系统为参与者提供了动力，其记录了每一个参与者的贡献，并将其赋予价值。这不仅仅是把所有人的贡献透明化，还保障了知识产权。当所有努力都能得到见证的时候，整个生态才能够正向发展。最重要的是，行为激励机制使得整个生态的资源得以再分配，让每个人可以各取所需，将每份资源的利用率最大化。这是 DeSci 生态可以持续运转的关键，也是 Web3 的本质，每个人都能够掌控自己个人的资源，并基于其获得收益。

举例来说，研究人员跟随导师的博士生所做的毕业设计，隶属于某个日后获得诺贝尔奖的研究的子项目，或只是单纯的为之提供了部分数据。但毕业多年后，当时的研究人员仍旧可以因此获得一笔微不足道报酬，可它来自于诺贝尔奖金，而且研究人员参与诺贝尔奖研究的记录将永远存在。研究人员甚至可以因此得到银行额外的一笔低息贷款。另一个例子可以是，某个人随手根据自己领域的专业知识所撰写了维基百科词条的一部分。而日后该词条访问量激增，大量用户为之付费，这其中很大一部分会自动流进 TA 所拥有的 NFT。这都得益于数据的长期保存，声誉系统和激励机制的存在。

这样的环境，才是 DeSci 生存的土壤，才可以开出名为「大师」的花，结出能够改变世界发展的果。
