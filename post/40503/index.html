下一代 L2 有哪些？一文详解 Scroll、Aztec、Immutable X、Polygon zkEVM

以太坊的第二层（L2）生态系统继续快速增长。目前在 22 个第二层中的总锁定价值（TVL）超过43.6亿美元，比一年前增长了1194%。哪里有增长，哪里就有竞争。随着质押的增加，L2 生态系统内的竞争也在加剧。

迄今为止，围绕 L2 的大部分讨论主要集中在“四大Rollups”上，其中包括 Optimistic Rollups (ORU)和 Arbitrum，它们已上线并拥有快速增长的用户和开发人员基础。四大Rollups还包括 zk-rollups (ZKRs) 和 StarkNet。虽然表现出色，但两者都没有像 ORU 那样受到关注（稍后会详细介绍）。

然而，有一些 L2正在等待颠覆，虽然目前鲜为人知。这些网络带来了独特的设计选择和功能，而一些网络则高度专业化，针对某些特定用例，让我们一起来看看下一代L2有哪些。

Fuel

Fuel 的独特之处在于它使用自己的定制虚拟机 Fuel VM，以及用 Sway 编写的智能合约。这意味着与大多数 L2 不同，Fuel 与EVM不兼容，这可能会阻碍其采用。

Fuel 还通过使用与比特币相同的 UTXO 模型，而不是以太坊和 L2 使用的基于帐户的传统模型来区分自己。这些独特的设计选择使 Fuel 具有高性能，因为网络能够并行验证多个交易，而不是串行（一次一个）。

相对于其他 L2，Fuel 还拥有更好的用户体验。不像传统的Optimistic Rollups那样需要 7 天的提款时间，Fuel 上的用户可以使用原子交换（Atomic Swaps）在几分钟内回到以太坊主网。此外，交易者可以用任何代币支付gas费，而不仅仅是 ETH。与任何 L2 一样，网络仍必须购买以太坊区块空间来验证。

Fuel V1 于 2020 年 12 月推出，开发人员最近将重点转移到Fuel V2上。Fuel Labs 团队最近在 Devnet 上推出了去中心化交易所 SwaySwap。独特的是，SwaySwap 上的资产完全是 Fuel 的原生资产，这意味着它们继承了与以太坊上的 ETH 等原生资产相同的功能。

虽然它们缺乏 EVM 兼容性可能使其增长速度较慢，但 Fuel 相对于其他 ORU 的功能和吞吐量应该有助于它吸引开发人员并从长远来看扩大使用。

Scroll

Scroll 是一个开发 zkEVM 的 zk-rollup (ZKR)。zkEVM 有可能从根本上改变 L2 和整个区块链扩展格局。

除了在生产中实施 ZKR 的困难之外，该技术采用的一个主要障碍是它缺乏 EVM 兼容性。虽然 ZKR 在吞吐量和 gas 费用方面表现更好，但无法轻松地将 EVM 集成到 zk-tech 中，导致每个 ZKR 必须从头开始构建独特的开发人员工具和基础设施。

这就是为什么 zkEVM 被许多人认为是 L2 扩展的“终极游戏”，因为它允许 ZKR 继承 EVM 的网络效应，同时保持相同的高水平性能。

虽然 zkSync 和 Polygon 等其他竞争对手正在开发 zkEVM，但可能让 Scroll 与这些项目竞争的一个因素是，它的目标是与EVM等同。这意味着部署在以太坊主网上的合约可以移植到 Scroll上，而无需对代码库进行任何重大更改。这应该有助于促进 Scroll 的采用，因为它极大地增加了项目迁移到网络的便利性。

Scroll 目前正朝着发挥这一潜力迈出第一步，因为它最近宣布计划推出一个 pre-alpha 测试网，用户将能够在该网络上进行交易并尝试单独的应用程序。

Aztec

Aztec 是一个注重隐私的 zkRollup。

Aztec 通过其独特的交易架构为用户提供隐私保护。虽然复杂，但在高层次上，该架构的功能类似于 UTXO 模型，并使用ZK证明来转移资产所有权，同时保护任何相关方的身份。

迄今为止，Aztec 的一个主要用例是通过 Aztec Connect 实现的私有 DeFi。

Aztec Connect 使用户能够从 L2 内直接与部署在 L1 上的智能合约进行交互。这意味着用户可以继承 Aztec 的隐私，同时还可以获得以太坊上 dapp 的流动性和可组合性。此外，尽管与 L1 直接交互，Aztec用户也只需支付最低的 gas 费，因为与任何 Rollup 一样，gas 费仍然在所有交易者之间分批和摊销。根据 L2 Beat 数据，Aztec Connect 目前的总锁定价值为 312 万美元。

利用 Aztec 技术构建的最受欢迎的个人应用程序是zkMoney，这是一种交易屏蔽服务。zkMoney 使用户能够与各种著名的 L1 协议进行私下交易，例如 Uniswap、Aave、Compound、Lido。

随着越来越多的应用程序建立在网络上，而且廉价的私人交易的价值主张变得更加清晰，Aztec应该会在这一利基市场继续增长。然而，值得一提的是，与隐私项目相关的监管问题可能会在短期内影响 L2。

Immutable X

Immutable X是使用StarkEx构建的Validium。

Immutable X被认为是一种Validium，因为交易数据是通过称为数据可用性委员会的链下解决方案存储的，而不是完全在链上存储。虽然这意味着 Immutable X 相对于其他结算到以太坊的Rollups增加了信任假设，但它允许网络显著降低交易成本，网络完全为最终用户补贴交易成本。

Immutable X 由 IMX 代币管理。除了这些权利之外，IMX 持有者还可以抵押他们的代币以赚取 20% 的交易费用，以及包括在 L2 上进行的所有二级 NFT 交易的 2% 的协议费用。

Immutable X 专注于游戏，在网络上构建了许多备受瞩目的项目，例如基于 NFT 的交易卡牌游戏 Gods Unchained 和目前处于私人测试阶段的具有 3A 质量图形的自动战斗器 Illuvium。

Illuvium 最近在 Immutable X 上进行了 330 万美元的土地出售，而 L2 还与迪士尼、Gamestop 和 Turner Sports 等公司建立了合作伙伴关系。

这些合作伙伴关系，以及 Illuvium 和 Ember Sword 和 Guild of Guardians 等其他备受期待的游戏的公开发布，似乎有望促进 Immutable X 的持续增长和采用。

Polygon zkEVM

Polygon 正在开发一套 ZKRs，其中最突出的是 Polygon zkEVM。

与 Scroll 类似，Polygon 的 zkEVM 在发布前一天被披露，有可能从根本上改变 L2 的格局。

与其竞争对手一样，Polygon Rollup 将与 EVM 等效，使开发人员能够近乎无缝地将其 L1 合约部署到 L2 上。zkEVM 还将利用 Polygon Zero 开发的 Plonky2 验证系统，Polygon Zero是在以 4 亿美元收购 Mir 后被引入 Polygon 生态系统的另一个 ZKR。Plonky2 能够以极快的速度进行验证，从而减少了 L2 交易到达最终阶段所需的时间。

zkEVM 计划在 2022 年夏天的某个时候推出其测试网，并计划在 2023 年初部署主网。

Polygon 在其 PoS 链上拥有重要的开发人员和用户群，在收购其他致力于 zk-tech 的团队（如 Hermez）后拥有大量的内部人才，以及源源不断的合作伙伴关系。在可预见的未来，Polygon似乎已准备好成为ZK生态系统中的一支力量。

多 L2 世界

除了Fuel、Scroll、Aztec、Immutable X和Polygon的zkEVM，还有许多 L2 似乎有望在未来几个月内实现增长。

Fuel、Scroll 和 Polygon 的 zkEVM 都是通用平台，相对于现有平台，它们可能会带来更高的性能和更好的开发者体验，而 Aztec 和 Immutable X 则继续在各自的隐私和游戏利基市场中执行。

目前尚不清楚它们将如何发展，以及谁将成为赢家。但更确定的是，在可预见的未来，我们将生活在一个多 L2 世界。
