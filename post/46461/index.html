近日，一款基于 Cult.DAO 生态的特殊博弈游戏 The Rug Game 受到了社区关注，并带动了 CULT 价格一时间的上涨，这款名为“跑路”的游戏不仅助推了 CULT 的燃烧，也迎合了寻求刺激、激进的投机玩家需求，因为赢家只有一个，而输家真的会被 Rug。

玩法

游戏内一共有 10 个 Token，分别是：T、H、E、R、U、G、G*、A、M、E* 十个字母，这些 Token 对应的是链游本身的名字即 The Rug Game，官方表示未来将向 dCULT（Cult.DAO）、uRVLT（Cult.DAO 旗下 Revolt 2 Earn）和持有至少一个来自 The Rug Game 中的字母 Token 的持有人空投 Token。

游戏 Token 交易的手续费为 4%，其中 1% 发送到销毁钱包，1% 用于回购并销毁 Cult.DAO Token CULT，1% 用于回购并销毁即将发布并待空投的游戏治理 Token TRG，1% 以 ETH 的形式保存到合约中。

The Rug Game 将以 30-60 天作为一个游戏回合，每个回合将会以末位淘汰制的形式淘汰分数最低的 Token（在游戏 Token、CULT 和 TRG 中烧毁数量最少的 Token ），分数最低的游戏 Token LP 池子将会被 Rug；获得游戏分数最高的游戏 Token 将会收到从其他九个社区中收集的 1% ETH 奖励。由于每个游戏回合结束时间不确定，因此游戏在 30 天后，游戏 Token 价格可能会出现剧烈波动。当前提高不同 Token 分数主要有两种途径：提高该 Token 的交易量和贿赂（Bribe）。

贿赂是 The Rug Game 较有特色的玩法，玩家可以支付 TRG 或 CULT 向 DAO 行贿，以提高自己持有字母 Token 的分数。行贿提高的分数相当于玩家支付 Token 数量 x 150%，例如如果 Token A 为 90 分，Token B 为 100 分，如果有玩家支付了 20 TRG 并选择了 Token A，那么最终得分将是 A 为 120 分，B 为 100 分。玩家支付的行贿的 50% 将分配给 TRG 或 CULT 质押者，而另外 50% 则被烧毁。

（1 月 12 日 15:30）

问题

The Rug Game 尚处于早期阶段，同时作为博弈游戏也还存在一定的问题。

(1)  Token 不仅是 Token，背后还代表着一个社区，10 个 Token 即 10 个社区，选择或过多。目前 The Rug Game 并未像去年的 Cult.DAO 那般引起大范围的 FOMO，仅在小范围的圈子内传播，这点从官方 Twitter 的粉丝数与话题参与量便可看出。加之现在加密市场整体仍处于熊市，市场流动性有限，可能会造成不同 Token 之间资金分散，一些 Token 无人问津无人购买的现象。据 The Rug Game 官方提供的 24 小时交易量数据，Token T 交易量遥遥领先其他 Token，交易量最少的 Token A 仅约为 Token T 的约 3.3%。

从传播的角度来看，在用户初次接触到这些字母的时候其实都是无感的，要么就是凭借感觉选，要么就是选择在社交媒体上首先被推荐或曝光率大的。在 Twitter 线程中，一般越靠前的浏览量越大，其次就是越靠后的，而处在中间的则会被折叠，这对于处在首位的 Token T 来说，可以获得最大的曝光量，自然而然地也成为了这几个的龙头。

另外，要为仅有一个字母的 Token 创造社区文化属性也非常不容易，特别是在早期没有影响力者带动的情况下，社区大多数都是一盘散沙，行贿的玩法很难以实施。对于大多数玩家而言，从游戏直接获取的奖励可能不如对这些游戏 Token 进行炒作后的价差中来的多；后续玩家在社群宣传自己的 Token 拉拢其他玩家加入阵营，除了是为了提高分数外，也有可能是为了提前买入的出货，最终游戏的结果可能并不重要。

(2)  这不仅是博弈战，也是一场持久战，实时激励机制不足容易导致竞争性不强。玩家的统一目标是先“保级”后“争胜”，只要不被淘汰就会有可能停止进一步投入。在只有排名最后的 Token 会被淘汰的条件下，玩家会陷入想获得第一名不太出力，落后成为“副班长”后不想努力的怪圈。在强者恒强，弱者无法翻身的悲观氛围，如果后续官方推出的是回合结束分数不清零的规则，这种马太效应可能会导致最后一名的池子一直被 Rug 的恶性循环。即便分数清零，社区也不会被清零，第一之所以是第一，还在于其比其他社区形成的共识要强。

目前分数榜已俨然形成了 Token T 作为唯一的超级大国之势，新入场的玩家看到如此排名在考虑其他 Token 前，必然会地选择配置一定的 Token T 以尽可能实现对游戏结果的对冲。其他游戏 Token 就目前来看已经丧失了争夺第一的悬念，若这样的格局持续下去，唯一的保留节目可能是谁将会被淘汰的问题。如果官方后续能对落后的社区给予令人感到逆转可能性的激励手段，以及对领头羊进行“反垄断式”的资源遏制，这样的博弈可能会更加具有竞技性。

(3)  贿赂玩法存在感不足。首先，在社区没合力的情况下，小额持有者进行贿赂没有实质影响性；而大户则会陷入自我博弈中，判断出资行贿是否足以赢得足够的奖励回报（目前是获得第一），若最终不能如尝所愿，其行贿的资金将变成质押者瓜分的 APR。其次，在游戏 Token 社区内部无奖励的规则下，社区行贿的动力只有改变该社区 Token 的分数排名，但分数较高的 Token 社区在确保自己押注的 Token “保级”成功后，其内部玩家并不一定都具备动力进行贿赂，特别是目前在一方遥遥领先的情况下。最后，如果 TRG 或 CULT 价格不在合理区间更会影响玩家行贿的意愿。

目前来说，The Rug Game 在规则上仍有很大的调整空间，假如可以在保留榜首奖励和对榜尾进行处罚的前提上，增添处于中间排名的奖励（类似于英超联赛的欧冠区/争四等）以及个人行贿数据的奖励，或者根据 Token 排名进行差异化空投等改善手段，或许能吸引更多的新玩家参与博弈。
