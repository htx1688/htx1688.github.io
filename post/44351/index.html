在过去几年里，大量艺术家进入了加密经济 。

这些创作者的动力是什么？

能够从他们作品的二次销售中赚取版税。

然而，在目前的模式下，NFT 版税并非必选项。

传统的 NFT 市场通过链下基础设施兑现版税，但许多市场最近已经开始采取行动，明确地将这些支付设为可选或完全取消。

OpenSea 是最近开始探索这一选项的代表项目，这让许多人想知道接下来会发生什么？

在我看来，我们肯定会看到的一件事是新 NFT 市场的爆炸式增长。

首先，从 NFT 版税的基础知识说起

假设 Alice mint 了一个 NFT，然后以 1 ETH 的价格卖给 Bob。

随后 Bob 通过 OpenSea 的二级交易将这个 NFT 以 5 ETH 的价格转卖给了 Charlie。

因为 Alice 事先在 OpenSea 上设置了 10% 的版税，她可以从这笔交易中获得 0.5 ETH。

然而，OpenSea 是通过其自有的链下系统来跟踪并且支付这些版税的。

如果 Alice 没有设置版税参数，并且 OpenSea 没有使用这个系统，她就不会自动获得 0.5 ETH。这就需要 Bob 主动转出这笔版税，但是在大多数情况下，是不会有人主动交出版税的。

为什么会出现这样的情况呢？因为目前在智能合约层面没有能够执行 NFT 版税的很好的方式。

你可以直接在 NFT 的 transferFrom (）函数中加入版税，但这将导致发送者无论在何种交易中都必须支付费用。如果你只是把 NFT 发送到你的另一个钱包或赠送给其他人，那么这种方法就很糟糕。

因此，近年来，NFT 市场一直使用独立的链外系统来执行版税。OpenSea 有自己的技术，其他市场也有各自的基础设施，而这些系统因为没有标准而不能相互 “交流”。

这导致了在一个平台上铸造的 NFT 在另一个平台上出售时，NFT 版税不被兑现的情况。一些创新的努力，如版税登记处（The Royalty Registry），正在帮助 NFT 市场建立统一的链上版税标准，但这个系统并非是强制性的，尚未获得广泛的采用。

The Royalty Registry 的查找系统

综上所述，最近几个月，我们看到越来越多的 NFT 市场削减了 NFT 版税或使其成为可有可无的选项，如 Blur、LooksRare、X2 Y2 和 sudoswap。现在看来，OpenSea 可能是最新的多米诺骨牌……也许。

最新动态

OpenSea 最近宣布，其已经创建了一个新的链上版税执行工具，可以 “限制 NFT 销售，只在执行创作者费用的市场进行”。

此外，该公司表示将在 2022 年 12 月 8 日之前决定其对版税的最终立场，其中一种可能性是完全转向可选版税：

“我们认识到这是第一步，因此我们致力于与我们的社区就现有藏品的解决方案进行交流。考虑到对现有收藏收取链上费用有多么困难，我们至少在 2022 年 12 月 8 日之前不会对现有收藏进行任何更改。为了透明起见，对 12 月 8 日之后发生的事情的考虑是完全开放的——我们正在考虑各种选择，从继续对某些收藏强制执行链下费用，到允许可选的创作者费用，再到为创作者就其他链上执行选项进行合作。我们认识到并非所有的创作者、收藏品和社区都是相同的，我们正在寻求制定一项反映这一点的长期政策。”

可以理解的是，这一声明引发了大量的社区辩论和讨论。

一些人，比如 BAYC 团队，认为在版税上打擦边球将是对那些使 OpenSea 变得伟大的创作者的背叛。

其他人已经注意到，一旦开始深入研究最近的版税统计数据，OpenSea 链上版税名单对于 NFT 项目来说是“轻而易举”的。

无论 OpenSea 下个月的决定是什么，很明显，所有的 NFT 市场都需要明确的版税立场，以便创作者可以随时决定哪里最适合他们。

例如，Nifty Gateway 刚刚宣布它将永远尊重 NFT 版税，并公布了自己的创作者版税标准提案。展望未来，其他市场平台也会在这个问题上有类似的突破。

走向更多定制的 NFT 市场

不同类型的 NFT 以不同的方式进行交易。

例如，像 BAYC 这样的头像项目流动性强，交易量大。一个唯一的加密艺术作品，比如早期的 XCOPY NFT 流动性差，交易量低。

这些差异导致了独特的动态，如数量大的藏品在可选的版税平台上很流行，例如 Blur，而数量唯一的收藏在 Nifty Gateway 和 SuperRare 等版税友好平台上仍占主导地位。

然而，像 Art Blocks、XCOPY 的 Grifters 或 Finiliar 这样介于两者之间的项目呢？艺术性绝对是核心，但这些 NFT 的交易模式更像流动性大的藏品而不是数量唯一的藏品？这些创作者是否应该仅仅因为他们的作品在可选的或没有版税的平台上交易而放弃他们的版税？

我表示反对，要获得这些收入。当涉及到 NFT 版税时，我对所有的艺术家都是这样说的。但是，对于大大小小的创作者来说，什么才是真正的最佳出路？在我看来，答案是定制的创作者拥有的 NFT 市场。

换句话说，未来每个艺术家和项目都可能有自己的定制市场，根据他们各自的 NFT 需求（包括版税需求）进行定制。

无论底层基础设施是基于 sudoswap、Reservoir、Zora 还是其他，这些创作者拥有的市场——可以由个人或艺术家 DAO 管理——可以提供激励措施，例如 NFT 空投，以巩固交易活动并更好地保证持续的版税支付。

具体要做什么？让你的市场成为你作品的首选目的地，以便你的粉丝在必须或想要时来这里进行交易。随着围绕 NFT 的创新不断涌现，期待创建这些 DIY 版税友好的市场变得越来越容易。

为什么这很重要？

Galaxy 的研究部门最近估计，到目前为止，以太坊上已经支付了价值近 20 亿美元的 NFT 版税。在我看来，这个数字是年轻的加密经济最伟大的成就之一，因为它为许多创作者带来了真正改变生活的收入，而且它提醒我们，我们可以生活在一个创造力是非常有价值的世界里。

从某种意义上说，NFT 版税的讨论才刚刚开始，但我知道许多人将继续为帮助创作者增加版税而奋斗，我认为其中许多艺术家拥有的市场在这里可以成为一个有力的答案。我们将拭目以待！
