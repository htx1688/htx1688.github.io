以太坊和其他顶级智能合约区块链上充斥着海量数据。

如果你能很好地实时分析这些数据，就会有大量的alpha甚至还会有工作机会等着你去解锁。

因此，本篇Bankless策略文将带你通过你能使用的最佳资源来提高你的加密分析能力。

为什么要进行加密分析？

我第一次知道加密货币是在2017年的春天。

那年夏天，我尽可能地投入精力研究这个领域，于是在同年秋天，我得到了我在加密领域的第一份工作——在一家名为CryptoAnalyst的小型媒体做加密新闻写作工作。

我当时面临的问题是，我对加密货币不甚了解，区块链分析领域在当时还是早期的原始阶段，不成气候。

从那份工作开始，我一直在加密生态系统里工作。在过去的几年里，有这么一件事让我特别印象深刻，那就是加密分析在近几年无论是在数量上还是质量上都呈现激增态势。

这对我们所有人来说都是好消息，因为知识就是力量。

或者更确切地说，在加密世界里，知识就是力量和金钱。

这句格言在2017年ICO ERC20代币繁荣时期是正确的，在2022年DeFi和NFT寒武纪大爆发时期也是正确的。

但是，当时和现在最大的区别之一就是，现今高质量的加密分析工具激增，这些工具使获得可实践的web3知识变得加倍容易。

有兴趣充分利用这些成就，前所未有地升级你的加密研究技能吗？那么是时候加倍努力研究那些最好的分析工具了。

让我们来探索5个可以用来加强你加密研究能力的平台。

1、CoinGecko & GeckoTerminal

几年前，似乎每个人都使用CoinMarketCap来查看加密货币的价格、市值、交易量等。

然后，币安于2022年4月收购了CoinMarketCap，许多人认为该平台已被“占领”。

加密社区开始寻找替代方案。近年来最成功的平台是CoinGecko，它在熊市期间悄悄超越了竞争对手，现在成为了全世界最大的独立加密数据聚合器。

除了独立性，我最喜欢CoinGecko的一点是，它给人的感觉是，这个平台总是走在潮流的前端。

每当我想要查看一些鲜为人知的新的加密货币的价格走势或交易量时，CoinGecko似乎总是比其他竞争对手抢先掌握该代币的基本数据。

这种令人印象深刻的响应能力的原因是什么？

CoinGecko的覆盖范围非常大——该平台目前追踪超过600个加密交易所超过13000个不同加密资产的关键数据！

因此，如果你希望实时跟踪最大项目的加密趋势，以及新币、大赢家和输家等类别的加密趋势，CoinGecko是非常可靠的。

但这还不是全部——该项目最近还推出了GeckoTerminal。

GeckoTerminal是一个DEX（去中心化交易所）聚合器。

还记得我上面说过CoinGecko的覆盖范围很大吗？

GeckoTerminal也是如此——如果你想交易一个代币，无论它是在以太坊虚拟机（EVM）链上还是在非EVM链上，GeckoTerminal都可能有你想要直接上手分析的图表和池！

2、DeFi Llama

作为一个DeFi的高级用户，DeFiLlama提供了一场使用的盛宴。

为什么这么说？

因为DeFiLlama精确而快速地跟踪总锁仓价值（TVL）数据和交易量数据，跨越超150个链和大量不同的协议，这意味着该平台是在任何特定时间“俯瞰加密经济”的绝佳方式。

然而，DeFiLlama不仅仅是一个普通的DeFi数据中心。

该项目还提供实时收益率指标、稳定币图表、清算活动、空投机会、黑客信息等。

换句话说，一旦你“用了Llama拉玛，就再也离不开了”。

3、Dune Analytics

在过去几周里，Reddit上NFT层出不穷。

然而几个星期前，我还对这些项目一无所知！

当我看到围绕着这些数字收藏品的活动开始激增时，我便知道自己需要写一篇Metaversal文章对这些现象做些基本阐述了，所以我便这么做了。

在教别人之前，我先要去哪里补课呢？

答案是Dune Analytics。

Dune Analytics是一个允许任何人创建加密分析仪表盘的平台，任何人都可以使用SQL（结构化查询语言）进行查看。

例如，我使用Dune的user petertherock功能创建了分析仪表盘，进行了上面提到的Reddit NFT文章的研究工作，里面有我需要的所有内容，如NFT持有者的数量、销量等。

Dune最酷的地方在于，它涵盖了各种链（如以太坊、Arbitrum、Optimism、Polygon等）上的DeFi和NFT项目的统计数据。

我推荐大家把Dune作为任何链上信息分析的重要组成部分。

4、Nansen

如果有人向我发起一个挑战，让我在一年内在加密经济中赚到100万美元，并且规定我只准使用一个工具，这个工具我会选择Nansen。

100万美元只是一个假设的数字，但Nansen可以帮助你达到的成果还是非常真实的。

我在2022年6月份撰写的Metaversal的《免费铸造NFTs》一文中提到了这一点，在那篇文章里我详细描述了自己的经历——我是如何使用Nansen的Mint Master追踪器在WAGDIE NFTs价格触底开始上涨前进行铸造的。

那时，我是这么说的：

“我通过Mint Master找到了WAGDIE，因为注意到周五有近40个‘Smart Minters’参与了该项目的铸造。在项目爆发之前，我有了一些积累，现在的底价是1.5 ETH，所以足以覆盖我的订阅费好几倍。当然，并不是每一次铸造都能获胜，但Nansen能够实时显示趋势信号，让铸造更加容易。”

今天仍然如此。

它不会让你屡战屡胜，但它会以其他项目无法做到的方式，实时呈现出“百分之一”或“千分之一”的机会帮助到你。

例如，Nansen提供了以下非常棒的服务：

  • Wallet Profiler ——可提供任何钱包地址的指标。

  • Hot Contracts ——是一个用于跟踪热门智能合约的仪表盘。

  • DeFi Paradise ——提供DeFi经济的完整概览。

  • NFT Paradise ——提供NFT经济的完整市场观点，包括热门收藏和新铸造。

  • Smart Money ——一个用于跟踪盈利丰厚的加密货币和NFT交易者活动的仪表盘。 

  • Token God Mode ——一个一站式搜索引擎，可以搜索到某个代币的全部主要信息，例如交易量、交易活动等。

5、Token Terminal

Token Terminal是我最喜欢的加密分析平台之一，因为可以说它拥有区块链和协议最好的费用和收入统计。

想知道某个特定的加密项目在某个时间的盈利情况吗？

无论你是研究某个垂直领域（如借贷市场）的项目，还是研究全链项目，Token Terminal都会满足你的需求。

成为一个更好的加密分析师

如果你学会了如何操作上面列出的5个加密分析项目，你马上就会成为比大多数人都优秀的加密分析师了。

好消息是，通过这些平台获取关键区块链数据都很简单，所以并不需要特别专业的知识储备就可以熟练使用它们。

现在就去尝试并使用这些项目吧！谁知道呢？你可能会开始收获链上的胜利，那些放到以前你觉得并不可能的胜利。
