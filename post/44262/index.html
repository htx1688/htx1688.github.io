人们常说好的艺术是主观的，但对于 NFT 来说也可以这样说——数字创作无疑是艺术的，但也是产品，旨在让他们的创作者赚钱？

虽然美丽在旁观者的眼中，但顶级系列和工作室遵循一些原则来确保他们的项目蓬勃发展。

一致的设计

随着 NFT 变得越来越主流，已经出现了数十种类型、案例和样式。比如收藏品、艺术 NFT、游戏内资产、虚拟世界土地和各种实用程序等。项目的无限可能性和纯粹的输出意味着那些具有独特、一致的艺术主题的人脱颖而出。

对于 PFP 项目，具有重复主题和总体主题的一致设计是重中之重。 例如，以下是目前 3 个顶级收藏品如何解决这个问题。

  • CryptoPunks 的反判。以反乌托邦的精神作为主旨，收获的是有同样精神诉求的群体。

  • BAYC 的厌世。它的灵感来自一种对生活感到厌倦，却拥有世界上所有金钱和时间的猿类。可以通过在无聊猿的身上找到自己的形象，这种艺术形象的表达特别符合人类的认同感。

  • Doodles 的愉悦。与许多顶级收藏形成鲜明对比的是，这些 NFT 的设计看起来毫无攻击性，看起来让人心情很愉悦。尤其对于女性群体来说，在传播性上有天然的优势。

一致的设计超越了头像本身，也必须反映在项目的信息传递、任务和网站中。

网站的设计

虽然不像 NFTs 本身的设计那样重要，但它确实预示着项目背后团队的整体能力。比如 GameFi 游戏项目 Illuvium，将科幻元素与 3D 模型结合起来进行宣传，吸引了许多玩家——甚至在它有任何游戏或游戏内 NFT 推出之前就锁定了他们的股权资产。

以下通过 5 个项目的官网设计归纳其特点：

  • CryptoPunks：古老、朴素而全面

  • BAYC：俱乐部式的网站，典型，较全面

  • Azuki：最用心最充实最合理的产品设计

  • Meebits：偏像素风，较全面

  • Art Blocks：艺术品集合，特殊类型代表

链上机制

与绘画不同，NFTs 是互动的、有生命力的物体，可以被纳入二级项目，有费用和自己的生态系统机制。因此，项目所在的区块链至关重要-——NFTs 最活跃的链是以太坊。在 BNB Chain、Solana 或 Polygon（NFTs 的第二大链）上的项目，将有不同的逻辑，在买卖双方支付货币会存在不同的操盘手法。

除此之外，还有 Collection 的 NFT 数量也很关键。目前大家买 NFT 的动机就是投资，低买高卖。也就意味着，创建者得要让大家觉得它设计的 NFT 是有前途和升值的价值，否则大家也不会来 Mint、买和卖。

设计规则还包括：

  • 定价多少

  • 售卖方式

  • 顶级的二级市场挂单和以什么样的拍卖形式

  • 收费标准

  • 激励政策

NFT稀有性设计

供应和需求是经济学的核心原则。因此，人为设计的稀有性和制作数量会影响到非传统文化艺术品的价格。

虽然一件 NFT 艺术品的价值通常由各种因素决定，但流动性溢价和稀有性是可以量化的。NF T的稀有性会影响 NFT 的价格，这就是为什么同一项目的 NFT 可以卖出大不相同的价格。

社交媒体活动

虽然这看起来并不与艺术品的质量相关，但 NFT 项目是我们的生活，是社区驱动的努力，是不断迭代的。例如，像 BAKC 这样的分支系列。

活跃的社交媒体账户使团队能够验证他们的艺术和想法，以及获得关于他们社区喜欢什么的第一手信息。如团队会在 Discord 和 Twitter 等社交媒体上同步自己项目想表达的内容，与行业平台、用户互动。

小结

随着 NFT 领域的无限可能性，只有好看的项目才能茁壮成长。但根据个人喜好，什么是 “审美”？虽然没有正确或错误，但有几个原则指导顶级项目做出 “好 “的艺术。
