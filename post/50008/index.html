虽然比特币长期以来一直被称为安全和去中心化的价值存储或“数字黄金”，但其支持更高级应用程序的潜力相对尚未开发。

我们会问这样一个问题：比特币还有人在创新吗？它仍然只是数字黄金还是我可以将其用于其他用途？

答案是肯定的，你现在可以将你的 satoshis 用于无数不同的事情，尽管在相对以太坊的欠发达的生态系统中。

NFT、稳定币、DeFi——所有这些东西现在开始在整个比特币网络中涌现。

不过，在比特币上开发一个全面的生态系统并非没有挑战。L1未针对可组合性进行优化，因此对于复杂的应用程序或 dApp 来说体积庞大且效率低下。但是，在L2网络上构建并确保L1的价值为大多数用例提供了一种更实用的方法。

这与我们在以太坊网络上看到的起飞方式相同，L2网络远远超过了L1层以太坊链的交易量。

由于Stacks、Lightning 和RSK等这些第 2 层网络的出现，比特币即将开启新的可能性。这些网络支持智能合约、NFT 以及你从以太坊和其他智能合约链中了解和喜爱的所有好东西，允许用户访问构建在比特币之上的更广泛的功能。

这些2层网络为利用比特币庞大的用户群和资金池提供了巨大的机会，随着越来越多的开发者和用户被它们吸引，比特币成为创新和创造力中心的潜力只会增加。

为什么要关注比特币L2？

比特币社区在历史上一直抵制变革，通常坚持一种“极简主义”的心态，即比特币应该保持简单并且只有一个目的：去中心化的数字价值存储。

这种心态导致一些人担心和质疑比特币的长期安全性。他们认为，比特币依靠“价格上涨”作为长期安全论点是不可持续的。为了保护比特币网络，必须激励比特币矿工继续挖矿，否则整个网络就会崩溃。

每四年，分发给矿工的比特币数量就会减少一半。这一事件被称为减半。从历史上看，减半导致比特币价格飙升，这意味着矿工仍然非常乐意继续挖矿。是的，他们收到的比特币少了，但它的价值比以前高了 2 倍、3 倍，甚至 10 倍，所以生意很好！

然而，数字上升并不是一个可持续的长期安全模型。这将要求比特币的价格继续至少每四年翻一番。如果比特币的价格每减半就翻一番，它将在短短二十年内超过整个全球 M2 货币供应量。

现在，随着长期安全的挑战越来越明显，许多社区成员开始采用更灵活的方法，拥抱创造力和创新。

Ordinals（一种NFT协议）的推出和在L2 Stacks 上的比特币域名服务（BNS）的日益普及是这种文化转变的主要例子。我们将在下面更详细地探讨这些示例。

还必须考虑到比特币日历上最大的事件即将来临。从历史上看，减半一直是所有目光都集中在比特币上的时期。到目前为止，只有一种方法可以接触到这一事件——购买比特币。

然而，随着网络上新发现的创造力和创新的发生，这些项目很可能会在前期获得大量曝光。

比特币L2叙事的主要项目

有许多建立在比特币之上的项目看起来很有前途，并表明比特币社区内仍然存在活跃的创新。

如果我们考虑到这些项目都还处于相对早期的开发阶段，并且到明年减半到来时可能会有更好的用户体验，那么所有这些项目也都具有巨大的“先发优势”。

下面是一些在比特币之上进行创新的项目：

Stacks

Stacks是1层区块链解决方案，专门设计用于与比特币网络协调工作。它可以在比特币区块链上执行智能合约和创建去中心化应用程序 (dApp)，从而扩展网络的功能和实用性。

Stacks (STX) 代币用于执行智能合约、参与 Stacks 生态系统和支持 dApp 开发。此外，通过“stacking”你的代币，STX 代币持有者可以锁定他们的代币并获得比特币奖励，为在生态系统内持有和使用 STX 提供额外的激励。stacking = staking。

大多数出现在比特币之上的新创新都是在后端使用Stacks。

BNS

比特币域名系统( BNS) 正在革新人们与比特币地址交互的方式。正如域名服务 (DNS) 通过将复杂的 IP 地址转换为易于使用的文本使互联网更加用户友好一样，BNS 将复杂的钱包地址转换为人类可读的格式。BNS 使用 Stacks 将名称注册到比特币链。

我们在类似的以太坊域名服务 (ENS) 上看到，随着用户急于获得最佳地址，令人垂涎的名称价格会迅速飙升。早点收集一些可能会在未来得到回报。

Ordinals

当发现由于先前的安全升级，用户现在可以将数据写入单个Satoshi，有效地创建了NFT，开发人员争先恐后地在比特币网络上建立他们的收藏。

Ordinals充分利用了这种新的热潮，并创建了一个协议，允许用户创建、买卖比特币支持的 NFT。它们现在已成为想要在比特币网络上发布 NFT 的项目的首选协议。

虽然与以太坊 NFT 相比，Ordinals NFT 可能缺乏某些功能，但比特币 NFT 的新颖理念吸引了大量资金，并为投机者创造了一些丰厚的回报。

“铭文”是指比特币网络上的 NFT

Trustless Computer

Trustless Computer使开发人员能够在比特币上构建去中心化应用程序。利用 Trustless Computer，开发人员可以构建 dApp，例如 DAO、DEX、NFT、代币、拍卖、借贷平台、存储解决方案以及介于两者之间的一切。

通过Taproot-style交易，将任何数据嵌入比特币区块链现在是可行的，确保永久存储在比特币网络上并继承其特性，例如可访问性和不变性。Trustless Computer 正在努力实现在比特币网络上运行的图灵完备虚拟机。

你现在已经可以创建钱包，从水龙头中获取一些TC并开始体验。
