人们不关心去中心化 是我们的错

当人们谈论去中心化这个概念时，总会提到抗审查性、去信任化或无单点故障等等这类词语。不过我想告诉你的是，95% 的人都不关心去中心化，或者他们虽然关心，却并没有准确理解这些短语：他们会把抗审查性和纳粹或儿童色情影片联想到一起；当听到去信任化一词时，他们会想到俄罗斯和朝鲜的特工秘密运行的网络；而无单点故障在他们眼里只是没人会去关心的无聊备份文件。

因此，我们必须提起普通人对于去中心化的兴趣。我的想法是，我们可以少谈功能，多谈好处。不过在实际操作当中，这种方法也很难推行，因为大部分新兴技术都是借助宣传功能来推广的。

比如说，当你说自己的车型可以达到 400 马力时，你就是在谈论功能。但每个人都知道，拥有 400 马力比 200 马力更好。这是因为汽车这一事物已经存在了很长时间，人们会下意识地在脑海中将高马力联想为该车的优势：百米加速更快、车速更高、发动机声音更悦耳，等等。同样，人们也会将高性能相机和联网设备自动理解为更佳的画质或更实用的上网功能。 几十年来，数以千万的营销人员不断向用户植入这些概念，人们早于习惯于这种思维模式。不幸的是，去中心化作为一个新兴事物，并不具备该优势。不仅如此，我们生活中的大部分事物都是中心化的，比如政府、企业、运动队，等等。

这就好比森林中尚未探索的路径，如果人们没有多次穿过这些道路，自然也就不会遵循它们。

那么我们可以有什么解决办法呢？

在我看来，与其不断重复「去中心化」这几个字，我们不如用真实的例子告诉人们它的实际含义：

1. 如果你想进入 FAANG（五大科技巨头）的管理层，那你需要花一个月的时间经历 6 轮的面试，最后可能也不会对公司决策有多大影响；而如果你想进入某个 DAO 或开源 Crypto 项目，那你直接上线 Discord 或 GitHub 并开始参与就可以了——每个人都可以加入并提出想法，编写代码或促进项目的发展。去中心化意味着人们能够共同创造所使用的产品。

2. 当你在 YouTube 或 Twitter 上收获了粉丝之后，平台可以改变算法强制你付费，否则你就无法与你多年积累的粉丝产生任何交集——Facebook 的粉丝页面就有这样的设定。如果你在 Web3 平台上面建立了受众，平台所有的决策都公开透明，并且可以由投票决定。因此，即便他们最终改变了算法，你也有时间将你的粉丝转移到其他地方，比如邮件列表。去中心化意味着人们能够了解你的产品规划，并对其未来决策进行投票。

3. 你不能把你的朋友列表从 Twitter 转移到 Instagram，因为这些公司通过出售对其用户的访问权来赚钱，所以他们不会让你把这些信息移动到他们竞争对手的用户数据库当中。但你可以轻松地将你的朋友列表从一个 Web3 社交产品转移到另一个，这是因为所有包含你的朋友或追随者信息的数据库并不属于 Twitter、Facebook 或任何其他公司。相反，它们都应该是可供不同服务平台使用的公开数据。去中心化意味着我们可以迁移至更好的产品而不被旧的产品所约束。

虽然这些只是三个我印象最深的例子，但我相信，以这种方式向普通人讲述去中心化会更有效果。
