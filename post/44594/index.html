Web3 社交图谱协议能够衍生出多种多样的社交场景，具有落地价值，对于要集成这类基础设施的平台或协议而言，可以针对拥有某一属性的地址空投、开放测试、DAO 治理或提供其他个性化的服务，也可以在一开始快速获取定向用户，对于用户而言，不仅可以结交志趣相投的朋友，还能在多个集成社交图谱协议的游戏、社交以及内容发布平台上跨 DApp，不仅能够省去重复的身份注册步骤，还可以继续保持和拥有原有的社交关系。

今天给大家整理了 Web3 社交图谱项目，有兴趣收藏，不作为投资建议。

1)ULAS

ULAS Network 是第一个做社交图谱评级的社交引擎，核心产品是社交图谱评级协议，旨在多维度量化一个人的社交图谱，使社交图谱可真正为商业所用。项目方、广告主可通过社交图谱评级协议找到最适合、最高效的推广渠道。同时，用户拥有链上数据，数据直接为用户产生收益。

Ulas 建立在 Cosmos SDK、Tendermint Core 和 CosmWasm 等核心模块之上。它是 Cosmos 生态系统中最重要的网络之一。除了社交图谱评级协议，Ulas 还开发了社交应用，例如发帖、加入社群、管理社群工具等。Ulas 多链 DID 是 web3.0 身份的基础。借助 IPFS+IPNS 存储，可扩展整合 Web3.0 身份的多链数据，实现数据价值最大化。Ulas Protocol 的特点：1.社交行为通证化：用户不止 DID 是 NFT，在 Ulas 上关注他人、发帖、评论等都会被记录上链。2.允许其他应用接入 Ulas 的社交图谱评级协议，使用该协议，Ulas 的用户也将得到更多数据产生的收益。3.配套提供了可视化的社交图谱数据分析面板，方便查看各项数据，项目方和广告主可在此面板与用户成交商业推广。

3）Lens Protocol

Lens 是一个搭建在 Polygon 上，具有开源、可组合性的去中心化社交媒体底层协议（他们自称 Social Graph），具有一般的社交媒体功能，例如个人资料编辑、评论、转发帖子等。不同的是，Lens Protocol 由 NFT 提供支持，支持用户拥有和控制其所创作的所有内容。

Lens Protocol 的特点：

  1. 内容、所有权通证化：用户的主页会被 Mint 为 Profile NFT 一直跟随用户，创作的内容也可以额外发行 NFT 出售给粉丝。

  2. 组件模块化，允许开发者使用模块化组件在 Lens 上任意搭建自己的社交应用，鼓励开发者开发提升产品体验的新组件。外部其他应用也可以接入 Lens，并且共享 Lens 生态优势。

  3. 还提供了 Market 行情查看、Exchange 交易功能、Save 存款管理、Pools 添加流动池、Borrow 借贷管理等功能。更像是一个 Back-end，而非前端产品。Lens protocol 更像是一个提供给各种社交媒体开发者的后台产品而非一个已经开发好的可以直接使用的前端产品。

4）Atem Network

Atem 是一个基于 NFT 的群组社交平台，旨在通过 NFT 把用户连接起来，探索基于 NFT 的社交网络，并丰富 NFT 资产与持有者，以及持有者彼此之间的关系表达。Atem 平台上的群组基于个人拥有的 Token 或 NFT 形成，并与交易实现无缝结合。用户根据自己持有的 NFT，可以很方便的进入各种各样的社群里，并可以很方便的和群里的人讨论，或进行点对点交易。

7）Mem Protocol

Mem 的 web3 社交层构建工具。

根据 Mem Protocol 的官网展示的路线图，协议的基础功能分为四个模板：

  • 第一个模块是与各个开发商的集成 ，让用户能使用 Mem Protocol 扩展去中心化社交图谱；

  • 第二个模块是一个可视化区块链浏览器 ，这个浏览器不是用来浏览 Hashes 而是在 Web 3 的社交层浏览关注的人。目前这个功能用户可通过官网申请 Alpha 内测资格，当协议开始内测时，符合条件的用户会收到邮件；

  • 第三个模块是一个问答工具 ，用户通过提问以及回答问题可以赚取赏金奖励；

  • 第四个模块是一个去中心化的朋友圈 ，用户可在 Web 3 与朋友建立群组。

2021.11 月份完成由 a16z 领投的 310 万美元种子轮融资，前 Coinbase 首席技术官 Balaji Srinivasan、Polychain 的联合创始人 Olaf Carlson-Wee 以及投资者 Charlie Songhurst、Jude Gomila、Charlie Cheever 等参与了此轮投资。

10）5Degrees

5Degrees 是 TP 钱包于今年初推出的基于 NFT 的去中心化社交网络协议，1 月上线官网。5Degrees 相信 Web3.0 的核心是解决 Web2.0 用户数据所有权的问题，通过区块链技术将数据所有权还给用户，围绕“链上用户关系”已推出三大类型产品。

一方面，5Degrees 团队将 5Degrees 协议与 Uniswap、PancakeSwap 代码集成，用户可在具有社交功能的 DEX 版本中跟踪感兴趣账户的交易动态。

另一方面，5Degrees 还推出了“Web3.0社交桥梁”，将 Web3.0 用户与 Web2.0 用户连接起来，帮助推特等传统社交空间内的用户轻松探索 Web3.0。目前，5Degrees 支持 ETH、BNB Chain（原 BSC）、Polygon 等公链，很快就将覆盖更多公链用户。

11）RSS3

RSS 被称为 Really Simple Syndication，翻译为 Really Simple Syndication。总的来说，它是一个简单的开源程序，允许用户主动订阅内容。通过向 Web 内容开发者添加 RSS 提要，用户可以根据个人喜好订阅不同的博客、新闻媒体和其他 Web 内容。RSS 程序可以自动为用户即时聚合最新内容，无需任何平台审核。
