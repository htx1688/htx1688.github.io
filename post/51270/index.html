原文作者：雨中狂睡

去中心化社交应用一直是Web3行业的一个重要应用方向，虽然我们一直看到Web3社交应用的出现，但也见证了它们的热度消失。Lens Protocol 拥有庞大的用户群，不过大多数用户都是出于对其空投的预期。本质来看，是因为这些社交协议没有能力与Web2好用的社交产品抗衡。

而一款来自于 Sui 生态中的社交应用正在改变这一情况。如今 Suia 正在利用 Sui Network 所带来的成本优势、速度优势和强大组合性来落实社交应用在该方向的采用。

如今 Suia 正在迎来其最关键的应用发展节点之一：代币发行。5 月 15 日，Suia 将于 Sui 生态应用 Cetus 的 LaunchPad 板块进行代币发行。

在代币发行前，Suia 也取得了不少的优异表现：

  • 经过十个月的建设，Suia 已经拥有超过 32 万用户，与超过 100 个的 Sui 上的品牌达成了合作，日活跃用户突破 50000 ；

  • Suia 获得了来自 Sui 基金会授予的 Sui Grants 和 Sui Builder Heros 奖励，并赢得了 Move CTF 冠军和 Sui Demo day 冠军；

  • 作为 Sui 基金会亚洲首个合作项目，Suia 已帮助 Sui 基金会链接超过 80 个来自亚太地区生态系统中的项目；

  • Suia 与 Sui 基金会联合开发了线下 NFT 咖啡系统，并前往台北智慧城市峰会参展；

如今，Suia 已经 Sui Network 主网上最受欢迎的去中心化应用程序，拥有超过 20 万次的交互量。那么，Suia 到底有什么过人之处？

了解 Sui 生态社交应用 Suia

Suia 是一个部署在 Sui Network 上的新型社交应用，他们的理念是通过产品创造一个开放、强大以及由用户驱动的新型社区。为了实现这一目标，在 Suia 的产品设计中，我们可以看到他们设计了很多 NFT 活动勋章，比如

  • 「Seoul Sui Builder House」，参加首尔 Sui 活动的人或者对该活动感兴趣的人都可以 Claim 该 NFT 勋章；

  • 「Sui Insights =」「SeaPad」，这类产品在 Suia 的 NFT 铸造类似于一种社区的冷启动——通过 Suia 平台让更多人了解应用产品，从而推动兴趣的产生；

  • 「Suirfboard NFT」，有些 NFT 项目会直接选择 Suia 作为发行平台；

  • 「Cetus Launch on Sui Mainnet」，记录协议的某一特定时间节点，比如代币发行。

通过 Suia，我们可以探索各类链上链下的应用场景。就像是我们在上文中提到的，就像在Web2社交产品中体验的那样，我们可以铸造由社区活动组织者提供的纪念 NFT、艺术品 NFT、品牌 NFT，并基于 Sui 的强大组合性将这些 NFT 置入自己的个人资料中。同时，这些行为无许可不受限制的。

并且，这些 NFT 铸造功能往往是互惠互利的，即一些冷启动协议可以在 Suia 上获得曝光，或者通过 NFT 铸造吸引更多的用户参与社区；而 Suia 社区成员也可以通过 Suia 发行更多有趣的 NFT 来参与社区，或者在 NFT 铸造中获得未来潜在的空投等好处。

Suia 的 Explore「（探索）」功能正在帮助 Sui 生态项目进行曝光和为社区成员推送有趣的 NFT。

而 Suia 的「Feed（投喂）」功能则构建了一个以 Sui 区块链地址构建的社交网络，我们可以在其上分享见解和生活。而这些 Feed 算法推流是基于用户的链上活动数据实现的。

Suia 的另外一个功能是「Brand（品牌俱乐部）」，Brand 允许 Sui 生态系统中的品牌方们入驻，并开放特定的 NFT 铸造活动。不论是从为品牌建立社区，还是增加曝光率的角度来看，对于 Sui 生态品牌们而言，Suia 都是一个非常好用的工具。

除此之外，Suia 开放了很多有趣的功能，比如去年 12 月的世界杯预测活动，Suia 社区用户可以通过领取球队 NFT 来表达自己对球队的支持。在这场活动中，一个值得关注的地方是，借助 Sui 面向 Object 的编程模型，Suia 的世界杯 NFT 有效地与 Capy NFT 进行了组合，如下图所示。

在 Sui Network CEO Evan 和 CPO Adeniyi 看来，「这是 Sui 与其他公链最大的不同之一，这样的可组合性能催动很多新场景的诞生，（我们）很喜欢这个例子（Capy 和 Suia 的组合）。」

另外一个可组合性的例子是 Suia 推出了一个名为「SuiBox」的功能。

我们可以将 SuiBox 理解为一个 NFT 收藏夹，就像我们收集邮票并把它们都放在收藏夹中那样，我们可以把钱包所拥有的 NFT 都放到 SuiBox 中，并在 Suia 的个人 Portfolio 中展示。不管是面对文字、视频、声音等各类 NFT，SuiBox 都能很好地完成任务。原始 NFT 的所有权将会被置入 SuiBox 中，所有者可以随时取回所有权。

SuiBox 是 Sui 所带来的强大组合性的一个优秀例子，未来 Suia 还会创造出更多基于可组合性的用例。也正如 Mysten Labs 联合创始人兼 CEO Evan Cheng 所提到的，「资产的可组合性是可扩展性的关键。想象一下，在一个支持 UGC 的游戏中，杯子是一种资产。该合约允许水、酒、果汁与杯子 “合成”。如果它是建立在 Sui 上，那么之前将流动黄金倒入杯子中就成为了可能——任何东西都可以合成。」

SUIA 代币发行

SUIA 将于 5 月 15 日在 Sui 生态应用 Cetus 上发行，此次代币发行是 Suia 最重要的里程碑事件之一。

SUIA 代币将通过经济激励机制来鼓励用户参与生态系统的发展，并为他们的努力提供公平的补偿。这将创建一个互惠互利的系统，每个参与者的努力都将得到公平的回报。这也就意味着，积极参与 Suia 生态的用户能够得到公平的回报，不参与 Suia 生态或者很少参与 Suia 生态的用户则得不到经济激励。

而代币发行也是 Suia 实现去中心化和自动化运行的重要途径。去中心化可以为 Suia 创造一个真实开放且公平的社区环境，而自动化运行其实是应用方的权力下放，即将制定平台规则和运营平台的权力交给社区。这将会为社区营造出一种主人翁心态，从而推动平台的长期发展。

就像 Suia 在 Gitbook 中所提到的，「我们相信数字资产的所有权是新一代社会的一个重要组成部分。持有相同资产的人可以成为朋友、同事，甚至是家人。」

Suia 应用在以下四个维度进行了创新：

1、个性化 NFT 推荐；

2、基于链上活动数据的 Feed；

3、品牌俱乐部入驻；

4、更强大的可组合性。

从本质来看，Suia 是在利用Web3对于所有权的天然优势，将品牌方、社区和 UGC 内容有机地结合在了一起。更令人惊喜的是，对 Sui 低成本高性能和强大可组合性的利用，相较于其他社交产品，Suia 拥有更强大的竞争力和更高的产品上限。因此，这也更让我们期待 Suia 的未来发展。未来它将如何改变整个社交网络，让我们拭目以待。
