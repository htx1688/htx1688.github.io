TG bot 赛道的迭代比大部分人想象的要快得多，Maestro 在未发币的情况下，用户量超过前龙头 Unibot，现在 Banana Gun 也在起势。

根据 Dune 的最新数据显示，Banana Gun 的日活用户数量已经远超 Unibot，直逼 Maestro。

图源：Dune

和 Unibot 一样，Banana Gun 是一个帮助用户狙击合约快速获取代币的电报机器人，最出名的功能是狙击开盘 Sniper 功能，有望成为用户在以太坊上的首选狙击手和手动买家。

收入是评价 TG bot 的要素之一，值得注意的是，在 Token 发售前，Banana Gun 的 Telegram 群不到 300 人，却可以实现每日稳定的 10-25ETH 收入进账。

发售在即，Token Economics 如何？

Banana Gun 将发行 Token 视为品牌营销、实现正向飞轮的一种方式，此次预售将在他们自己的 dApp 上进行。在发售的前三周里，Banana Gun 过滤了完成任务的机器人用户，并对其发放了 NFT，虽然那时还没有公布 NFT 的用途。

在此次预售中，预计上限为 800ETH，预售发行价为 0.65 美元，为了确保均匀分配，团队将在预售期间引入分层制，预售将托管在他们自己的去中心化应用程序（dApp）上。

分发共有四轮，先到先得。第一轮不超发，对象为持有 2~5 个 Quest NFT 的用户、对项目有过帮助的用户以及 KOL；后三轮均为超发，侧重于早期忠实用户。考虑到 Banana Gun 在过去的短短三个月中就产生了 500 ETH 收入，此次发售也许会在第二轮就结束。

图源：Banana Gun

Banana Gun 为了确保稳定的发布情况，将在发布、收入分享、香蕉奖金和空投之间留出几天的时间。

此次将发布 1,000 万枚$BANANA，其中预售占比 20%，流动性池占比 3%，空投占比 1.2%，团队预留 10%，团队预留中一半锁仓 2 年后 3 年线性释放，另一半锁仓 8 年后 3 年线性释放，国库预留 65.8%，大部分锁仓，释放部分将用于用户使用 Bot 的奖励。

预售价格为每枚 $BANANA 0.65 美元，对应的初始市值为 156 万美元，初始 FDV 为 650 万美元。初始流动性将由 30 万美元 $BANANA 和 19.5 万美元流动性注入，总流动性为 39 万美元。理想状态下，$BANANA 的流动性比率将达到流通市值的 25%，可以在波动性和深度方面保持 Token 市场的健康。

届时将有 240 万枚 Token 流通，占总 Token 数的 24.2%。买卖税均为 4%，团队承诺会在「市值大幅增长时降低税收」，目前的买卖税中 2% 分配给 Token 持有人，1% 分配给团队，1% 分配给国库。

Banana Gun 生态机制

Banana Gun 在构建的生态系统中提供了使机器人用户、$BANANA 持有者、Banana Gun 的投资者三种角色相互转换的机制。使用 Banana Gun 进行的每一笔交易都会得到作为奖励的$BANANA，计算方式为手续费*X，X 在 0.05~1 之间，团队称会根据市场情况进行调整；$BANANA 持有者可以在 dApp 上获得收入份额，包括手续费收入的 40% 和 Token 交易税收的 50%；使用 Banana Gun 的手续费为 1%，手动买卖的手续费为 0.5%，Token 交易税均为 4%。

对可能面临的 Token 通货膨胀风险，Banana Gun 也做了一些考虑将通缩/通胀率控制在 30% 以内。Token 价格、机器人数量和交易量是要考虑的因素。首先，将会回购 1% 的国库税，通常这部分就可以抵消大部分通胀问题；此外，$BANANA 发行量是可调节的，其生态机制的特性也会鼓励用户积极参与交易并持有$BANANA。团队还称后续会上线可以消耗 Token 的功能。

据官方文档的内容显示，Banana Gun 为投入时间和精力的用户，准备了两轮空投计划：分别为 100,000 和 20,000 个。第一次空投将基于用户之前在 Banana Gun 上收集其 NFT 的情况，不同难度的 NFT 对应不同的积分数，经过计算后会得出可以获得的 Token 数，在 dApp 上可以直接查看。第二次空投将在后续社交媒体营销环节中释放。

Banana Gun 怎么狙击开盘？

与 Unibot 不同的是，Banana Gun 的特点和最出名的功能是狙击开盘（Sniper 功能），主打 First Block。其次就是交易功能。现在我们就来看看，这个拥有优秀的狙击开盘功能，以及明确了空投计划的 Banana Gun 怎么玩？

进入Banana Gun 的 Telegram bot 频道，点击 start 进入指引，通过 setting 进入 Sniper 的设置页面。

在参数展示区的下方，选择要修改的参数。

设置成功后点击 Wallets 增加钱包，可以导入私钥或直接生成。每个 Telegram 账号最多可以使用 5 个钱包。钱包生成后，继续设置自动钱包和手动钱包。然后设定狙击要付的小费。Banana Gun 使用的形式是统一打包贿赂，同个合约中，排队顺序由给出的小费高低决定，因此需要根据不同的项目设置不同额度的小费。

基本设置完成后，就可以输入想要「狙击」的合约和每个钱包的最大支出了。Banana Gun 可以自动抓取电报频道，以获取来自机器人或管理员的任何包含以太坊合约地址的消息，找到后将启动购买或启动自动狙击手。

Bot 赛道的龙二之争

社区也有不少人认为 Bot 还有很大的增长空间。短期里，Bot 的 Token 经济学打开了资金与币价赋能的新思路，关注如 Banana Gun 这样的新产品和新迭代或许将有机会捕捉新一轮的行情上涨。

上个月时，Unibot 的成交量就占据了 Uniswap 的 2%，参考 Uniswap 在 DEX 赛道拥有的绝对领导力，通过 Unibot 的成交量可能超过一些著名 VC 支持的 DEX。难怪社区中有人认为 Unibot 和 Banana Gun 的关系就像是 Uniswap 和 Sushi。

图源：DefiLlama

从最新的收入数据来看，Maestro 的收入已经排在第九位，数字或许可以说明 Bot 赛道并不是虚假的叙事，而拥有真实的需求。

Bot 工具为加密用户解决一些长期存在的问题。这些功能包括跟单交易、新币狙击、自动化空投交互，以及利用 AI 生成智能合约或 NFT。如果没有这些先进的 Bot 工具，个体的加密交易者在提升技术优势上将面临巨大的挑战。

在 Telegram 月活 5.5 亿的加持下，Bot 生态越来越丰富，将继续关注 TG 生态。
