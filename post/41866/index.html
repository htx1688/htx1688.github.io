1、PFP熊市现状&原因

NFT市场的现状，来看一张OpenSea的统计图。虽然有Looksrare、X2Y2兴起，ETH价格下跌等原因，但无论是交易人数，还是交易价值都远远低于高峰时。

从“蓝筹”来看，BAYC的地板价跌幅已经超过50%，而Azuki、CloneX等项目的地板价，跌幅已经达到70%-80%。

OpenSea数据

在我看来，造成这种情况，主要有两个原因：

1、熊市典型特征：赚钱效应下降，原本持有者倾向于卖出。没有新鲜血液的加入，没有资金流入。这两个方面，造成资金不断从NFT市场流出，现象便是蓝筹地板价不断降低，其余PFP社区几乎完全丧失流动性。

2、PFP叙事失效：BAYC开拓的路，对外人而言，以空投赋能为主。无论是BAYC还是药水变异为MAYC，或是Otherside土地空投，都是一种不断发行新资产给主资产赋能的行为。这种叙事是无法持续的，牛市有人买单，但熊市缺乏资金的情况下，新资产的价值从何而来？

另一方面，文化基因、精神共鸣、身份认同等叙事没有继续发展，导致PFP整体叙事失效了。

综上所述，现在PFP市场的痛点是：发展自己真正的叙事，吸引扩展新用户。

我的观点

2、PFP 是 IP 的新范式

先让我们回归本源，来看看PFP到底是什么。PFP：Profile Picture，名字已经说明，PFP反映的是社交网络中头像的价值。人们可以为自己喜爱的PFP付费，从而延伸出身份认同、新型社交网络的叙事。

我之前的文章，已经很详细的探讨了，PFP是IP的新范式。

IP是PFP真正的叙事，只要意识到这一点，我们就明白PFP的未来应该怎么走。IP的力量远超人们的想象，是侵入人们心智的最好方式。

IP一直贯穿于人类历史中，上帝佛祖都是IP，是文化符号的具象化。

首先我们来明确以下两个事实：

1、不同的时代，IP有不同的载体和形式。最早人们口口相传，书写于纸上、电影、电视、互联网。迪士尼通过电视渠道，打造了自己的IP帝国。那下个时代的IP载体是什么呢？

2、不同的时代，需要顺应时代潮流的文化符号。远一些有百家争鸣、文艺复兴，近一些有披头士、摇滚等。盗火的普罗米修斯，吃下智慧之果的亚当夏娃，削骨还肉的哪吒，与天抗争的齐天大圣，都是不同时代的文化符号。

IP一旦成为文化符号，便会有极强的凝聚力和生命力，可以穿越时间。

3、Web3 的先进性

这时候我们要问：为什么是NFT & Web3？为什么我相信Web3会出现席卷全球的大IP？

“Web3的先进性，在于所有行为都有价值，Web3解放了人的创造力。”这里引用我《PFP：IP的新范式》里的内容，不再赘述。

“Metaverse IP 和传统 IP 的不同点，还在于价值的转移。Web2.0 时代，我喜欢皮卡丘，我只能通过玩 Pokemon 游戏、看动画、买周边来表达。也就是说我只能通过消费来表达我的爱，我所有的创作都真真是为爱发电。哪怕我的同人创作，为 IP 的传播起到了很大的作用，但我还是没有一点收益。所有的收益都给 IP 所有者，也就是公司拿走了。

Metaverse 的 IP 则不同，如果 BAYC 受到了更多人关注，他们的价值会自然反映在 NFT 的价格上。那持有者就会收到 IP 扩大的收益，每个 PFP 的地板价上升，都是整体 IP 价值上升的体现。

这不就是 Web3.0 的意义吗？Web3.0 解放了人的创造力。基于此，所有 PFP 持有者，都有极大的动力来为 IP 创作。这样的行为不再只是为爱发电了，我们创造的价值是属于我们自己的。”

“谁能缔造最好的社区，谁能发挥社区的创造力，谁就最有可能抓住Web3的红利。”PFP的IP不会仅止于由项目方来建设，社区的人们已经开始用自己的方式来参与构建。

下面是@Harry_forj 整理的BAYC社区IP地图，可以看到BAYC的IP授权已经多点开花。BAYC的IP从Web3延伸到Web2市场，试图将所有人们Ape化。

4、PFP 目标：成为文化符号

再来，从叙事的角度思考。

现在的PFP正如Will所说，没有故事，缺乏灵魂。在我看来，灵魂是蕴藏在故事中的，上帝需要圣经传递福音，释迦摩尼剔肉喂鹰述说佛法。

PFP有可能成为这个时代的文化符号吗？

首先他们需要讲好故事，内容的构建是IP建设不可或缺的一环。比如BAYC选择建立元宇宙，来传递精神：畅游在Metaverse的猴子。

或许很多人觉得一个PFP，竟然扯上时代符号，是不是太过了？这张图可以代表我的想法。

在我看来，NFT迟早会深入我们所有人的生活，而现在NFT已经走到了Brands的阶段，接下来便是Culture-文化，时代精神的载体很大可能是NFT。

从讲好故事开始，构建PFP的灵魂。用IP触达更大规模的人们，让他们听到PFP的故事，让他们喜欢PFP的精神。这才能让PFP走出价格下跌，玩家流失的负循环。

我没有希望PFP成为能流传千年的神话，但我们期待PFP能出现比肩Web2 TOP IP的项目。PFP是文化和金融的结合，让文化属性发挥出最大的作用，是解决PFP痛点的必要路径。

5、IP收益应取代版权费

Sudo的兴起，让人们担心0%的版权费是否会敲响PFP的丧钟，让PFP项目方损失最大的收入来源。但NFT的特性决定了，迟早会出现0%版权费的平台，总有一部分人不希望分享自己的利益。

我们要学习Web2第一大IP：精灵宝可梦的成长路径。让IP的授权费用，品牌实体经济，逐渐取代版权费成为PFP项目的第一大收入。这才能可持续化的发展，并且让每个PFP的持有者都能获利，让社区获得极大的动力来发挥其创造性，形成正循环。

我之前的文章，探讨了CC0不太适合PFP的原因，就是会缺失了IP授权费。

同时，我们不要忘记PFP的发展才短短1年左右，BAYC是去年5月Mint的，而Azuki刚诞生7个月。他们的发展速度已经超乎我们的想象，但需要更多的时间来夯实基础。

如果你相信Web3会占领人们的生活，用户量会指数型增长，那么不要只关注价格的短期涨跌。

看得长远些，从PFP当中找到最可能席卷全球的IP吧 。
