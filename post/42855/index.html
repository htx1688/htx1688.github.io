原文标题：《走向 Web3 的 NFT 市场》

撰文：张烽，万商天勤律师事务所合伙人

目前 NFT 市场技术发展非常迅速，带动许多线上线下业务模式创新，激发了市场活力，但确实也存在包括但不限于技术协议还比较初级、应用场景还比较单一、市场规范体系还不健康、司法保障还需要进一步完善等很多问题。

产生这些问题的原因也有很多，但可能最主要是，NFT 产品应用虽然是代表 web3 发展方向 ，但目前却仍然依靠 web2 基础设施在运作，并基于这种「错位」产生了很多实践操作问题和法律规制问题。长期来看，NFT 生态必然形成特有并不依赖于传统互联网的业务模式和生态体系，但在具体发展中仍然需要自觉培育 web3 发展方向，因为这样有助于正确解决市场发展中碰到的问题。

一、基于 Web2 运行的 NFT 市场

如前所述，目前 NFT 市场出现众多问题，其根源在于 NFT 市场目前仍运行于 web2 模式之上。总体上说，就这一点而言境内与境外并无本质区别。作为一个发展尚处于初级阶段的新模式新业态，也是可以理解的，但需要市场主体积极正面应对，才能更好地解决问题。

NFT 产品所依靠的区块链，目前处于一个个「区块链孤岛」状态。目前国内主流是基于联盟链发行 NFT 产品，但各联盟链之间还没有很顺畅地实现跨链操作，也没有产生实质上的跨生态应用，各联盟链自身总体上还主要是基于其主要发起机构的信用来运行，相互之间事实上还是处于「区块链孤岛」状态，因此其本质上与传统互联网模式无异。即便境外 NFT 产品运行于公链，各公链之间基本也还处于各自为战状态，其生态也还是相对割裂的。

基于 NFT 产品的新型数字营销，仍然基于传统互联网模式。目前虽然各 NFT 产品或藏品平台百花齐放百家争鸣，但是其业务运营还总体上是依靠传统平台或载体，资产发行方式和交易方式仍然还是基于传统模式。

比如用户获取信息还主要是靠传统互联网媒体，有一些媒体或相关公众号经常会定期发布相关藏品信息，而用户搜索了解获得 NFT 产品信息，仍然基于传统媒体传统网络平台。而且这些公众号或相应媒体一定程度上成为目前阶段数字藏品的流量入口。

又如很多的 NFT 产品发行其内容权利来自线上，但其数据和权益确认、交易、实现、利用均需要通过传统互联网市场主体来操作，与线下有千丝万缕的联系。至今为止，很多数字资产交易仍然基于传统模式的中心化交易所来操作。

用户权益保障仍然基于传统互联网模式。用户基于传统互联网账户如小程序或其他身份登录，其数据保存在中心化服务器中，用户对于链上信息把握能力很弱，以至于有时候有些数字藏品甚至并非基于区块链发行这样的情况也并不鲜见，通常用户也无法掌握自己的账户操作，无法查看相应数据信息，此时要保障用户权益只能寄希望于平台，因此对行政监管机构和司法机构而言，自然而然会把更多的责任和义务施加于平台，与传统互联网模式无异。

二、走向 Web3 的 NFT 市场

当然，目前涉及 NFT 市场发展的问题有非常多，既有技术的也有应用的，既在法律法规层面的也有实际市场环境方面的。但从根本上说，解决目前 NFT 市场应用问题的钥匙，是建设区块链底层基础设施，开发更多的 NFT 技术应用协议，推出更多保障用户权益措施，努力构建基于 web3 运行的 NFT 产品。

加强底层区块链基础设施仍然非常重要。 从目前联盟链运行状况来看，其信用还是基于其主要发起机构或运营机构的信用，其业务和生态还是趋于封闭的，但这其实并不符合区块链应用和生态所应具有的开放、智能、可信的内在属性，要真正让区块链基础设施发挥出 web3 生态底层的作用，应当以公链的格局来运营联盟链，以体系开放化、运营智能化、数据可信化要求来发展联盟链生态，真正为市场提供一个打破数据孤岛实现生态互操作的底层基础设施。从这个角度来说，联盟链与公链的区别并非是根本性的，一个具有高度共识的联盟链要比作为孤岛存在的公链要好很多。

开发更多的基于 NFT 技术应用协议。 目前的协议还相对初级，应用场景还比较单一，应增强用户个性化、沉浸感、互动化需求，开发诸如音乐、戏曲、文字作品等一些基于 NFT 产品新型消费模式，并将具体社交场景、经济活动场景融入其中，激发诸入文旅、艺术、汽车、教育等行业新场景。

虽然目前各数字藏品平台仍然类似于传统互联网平台，但由于其侧重于技术平台构建和技术协议开发应用，对具体业务场景相对包容，可以容纳基于不同数字内容和业务场景的 NFT 产品发行，这有利于不断丰富 NFT 技术应用协议，也有利于形成健康多样包容性的 NFT 业务生态。

提升用户权益在 NFT 生态和市场的地位。 在应用和业务发展初期，依靠于传统生态和模式是不可避免的，但这不应当是方向。应当努力从提升用户权益角度，进一步丰富和发挥用户钱包、用户空间的功能并基于此改变用户搜集、发现、利用、使用 NFT 数字藏品习惯，把业务场景和要求融入到这种新型用户习惯中，逐渐形成 web3 基础上的新型生态，让 NFT 成为未来业务平台和流量入口，逐渐产生和发展中独立的业务模式和生态体系，焕发出 web3 业务的真正生机。

三、走向 Web3 的 NFT 市场若干问题

NFT 市场需要秉持敬畏法治、敬畏专业、敬畏市场、敬畏风险的理念，贴近市场、贴近用户，积极构建走向 web3 业务模式和生态体系。

遵守 web2 的监管规定。 由于 NFT 市场总体上还是运行于 web2 业务模式之上，因此不可避免需要遵守有关 web2 模式下的相关业务逻辑和监管要求。集中体现于需要按照传统平台治理和监管要求操作，包括信息发布、资产发行和交易、用户社区管理、隐私保护等。比如在传统网络媒体中发布信息，需要遵守其相关要求。

维护市场秩序、用户权益和体系稳定。 在走向 web3 过程中，除了一些典型的 web2 场景，可能也会出现一些 web3 业务环节，有些可能介乎于二者之间。此时，需要从业者自觉维护市场秩序、用户权益，维护整个市场体系的稳定性和有效性。比如在发布区块链智能合约时，需要考虑到技术协议的可靠性稳定性，考虑到业务逻辑的合规性，同时也要考虑到用户权益保护，以及对其他正常业务的影响。

保持服务和产品的有效性。 在走向 web3 过程中，有些市场主体热衷于发行一些始终停留于概念阶段的产品或服务，理由就是目前 web3 还处于发展早期，还无法实现更多功能和应用。这可能是技术业务环境事实，但不应成为业者固步自封的借口甚至成为其「割韭菜」的工具。比如有些市场发行主体不去想如何增加服务和产品的有效性以解决现实的业务场景问题，转而去挖空心思凭空去增加流量扩大交易，这种缺乏真实需求支撑的行为对市场和用户都不是好事。

防止误导、欺诈和操纵，防止与 NFT 应用相关非法活动。 很多从业者在主观意识上并不尊重相关法律法规和规定，认为这些规定都是基于 web2 的，自己从事的业务并不应该属于规制范围，从实际行动上也并不遵守。比如在很多市场机构在开展业务时，有意无益会夸大产品和服务的功能，甚至有意无意暗示和误导产品预期回报，并且在实际操作中甚至主动参与欺诈、欺骗、盗窃、市场操纵等。这些行为无论在境内还是境外，都是监管所不允许的，也是必然受到打击的。

积极在考虑数字人民币在 web3 场景中试点应用。 数字人民币是我国十四五规划中的重要部署，对我国发展数字经济和 web3 应用具有重要意义。首先，可以更加提升支付效率，促进数字技术应用创新，促进一带一路中的扩大数字人民币在跨境交易中的使用。其次，以基于现有的在线支付服务基础上为消费者提供更多便利选择来促进金融体系公平性和安全性。另外还可以促进防范网络风险和业务体系风险，促进数据安全和隐私保护，打击非法金融交易活动。比如基于数字人民币的智能合约应用，结合 NFT 产品业务场景，更加有利于促进技术和业务模式创新，保障用户合法权益，促进数据安全和金融体系安全。
