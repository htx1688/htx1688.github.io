加密行业迭代的产物， 0 流动性的“Shitcoin”

Web3 营销主要基于块链技术来实现品牌与用户之间的直接互动，以实现品牌与用户之间的直接互动和价值交换，品牌发行代币来直接激励用户参与营销活动中，是最为常见的 Web3 营销手段。基于此，用户则可以通过代币获得品牌的产品或服务，实现品牌和用户之间的无缝合作等。

基于加密代币的营销在早期能够起到不错的拉新促活效果，比如 Uniswap、ENS、Aptos、Optimism、Arbitrum One 等头部生态，就曾为大量符合条件的社区用户（通常都是率先使用、为生态发展贡献过的用户），发放了价值不菲的代币空投，并且在空投后都取得了相当不错的效果（用户通常都会主动的成为社区的粘性用户），而在代币奖励的驱动下我们看到，加密行业绝大多数用户都期盼成为一些早期生态的早期用户。实际上，也很少有人会拒绝这种“动动手指”，就可能获得的“免费的午餐”。

当然，这种“简单粗暴”的空投方式之所以能够持续的奏效，通常建立在强大的社区共识之上，这种共识通常能够对空投所带来的负面效果进行对冲。

而对于缺乏社区向心力的加密项目（多数都属于此类），在为用户提供空投后，往往会因价值基础薄弱而难以对冲负面效果，代币在面向市场后短期内暴跌并失去流动性。

实际上，自 2020 年后，通过代币空投方式进行营销的加密项目多达数千个，而能够具备价值且具备良好流动性的代币寥寥无几。这意味着，疯狂的空投营销行为，产生了大量几乎是 0 流动性的“Shitcoin”。

除了空投外，每一轮牛熊的交替、每一轮行业热点的更迭，都会导致大量加密项目的死亡以及 Rug Pull，它们也同样留下了大量的 0 流动性的山寨币，而随着加密行业不断地迭代，似乎不再有人记得这些代币所代表的项目，曾经所拥有过的辉煌。

几乎在每一个“加密原住民”的加密账户中，可能都存在数十个上述“Shitcoin”资产，这不仅代表了他们曾经的“战绩”，也表明了加密行业迭代的速度以及竞争的残酷。

这些 “Shitcoin”还有救吗？

在现实世界中，绝大多数物品在经过了一定的使用年限后，它们的使用价值将变得越来越低。近年来，随着各类可回收技术的发展以及各国环保意识的加强，各类物品的再回收体系不断健全，基于此，我们能够通过再回收的方式，将这些物品的残存价值进一步变现，并为可循环经济做出贡献。

在 Web3 世界，每一个“Shitcoin”背后，可能都代表了持有者曾经的努力，或者一笔失败的投资，而在这些资产彻底沦为 0 流动性的 “Shitcoin” 后，持有者们在遗憾之于也或都表示无能为力。

现在，这些曾经的 “Shitcoin” 也能像现实世界中，通过“回收站”获得提取残余价值的机会。一个叫做 Poop 的项目正在成为 DeFi 领域的“回收站”，它正在为这些“Shitcoin”建立一系列的回收机制与设施，通过代币回收来帮助这些资产重新建立流动性，并正在成为很多“Web3 原住民”们的福音。

Poop：DeFi 世界的“回收站”

Poop 正在成为 0 流动性资产回收的一个链上场所，该协议建立了一套具备创造价值动能的回收机制，该协议允许所有“Shitcoin”通过其回收系统实现“回收”，并获得 $POOP 代币的返还，以将这些资产剩余的残存价值变现。

Poop 回收体系包含了几个部分：Shitcoin Recycle、POOP Station、Referral Program

  • Shitcoin Recycle

Shitcoin Recycle 是 Poop 平台的核心功能，该板块是“Shitcoin” 回收的主要执行场所。在 Shitcoin Recycle 中，我们可以通过像 DEX 交易一样，将手中因投资失败、空投、Rug Pull 等方式产生的 0 流动性资产进行与 $POOP 代币的兑换，而在兑换完成后，“Shitcoin”持有者能够将手中的 $POOP 代币出售获得收入，“Shitcoin” 则直接被焚烧销毁。

在实现路径上，针对于不同流动性的“Shitcoin” ， Shitcoin Recycle 会做不同路径的处理。

当一枚“Shitcoin” 的流动性≥所需的最低流动性时，Shitcoin Recycle 协议将通过智能路由器将“Shitcoin” 转换为 Layer 1 代币（比如 ETH、BNB 等），然后用 Layer 1 代币再去购买 $POOP 代币。

而当一枚 “Shitcoin” 的流动性<所需的最低流动性时，用户需要支付  Layer 1 代币的费用，该费用等于所需的最低流动性。然后 Shitcoin Recycle 进行回收，而用户则得到 $POOP 作为回报。

  • POOP Station

除了 Shitcoin Recycle 协议外，POOP Station 是 Poop 协议中，$POOP 代币的铸造、销毁场所。在 POOP Station 中，用户可以通过该交易枢纽，通过 Layer 1 代币来购买或者出售 $POOP 代币。而在这个过程中，将持续的产生 $POOP 代币的通缩与赋能，以帮助 $POOP 代币构建更为坚固的价值基础。

除此之外，Poop 协议协议还允许 $POOP 持有者以 Staking、LP 的形式进一步获利，并能够持续的从通缩机制中获得分红。

  • Referral Program

Poop 协议也建立了一个邀请系统，旨在奖励帮助 Poop 扩大社区并分享产品的用户，当一个用户推荐另外一个用户使用 Poop 协议后，被邀请者所产生的收益的 10% 将归邀请者所有以作为激励，这些奖励将在交易完成时立即以 Layer 1 代币的形式发放到邀请者的账户中。通过这种直接的激励模型，正在帮助 Poop 建立强大的社区与用户基础，并为贡献者带来不菲的收入。 

Poop 协议建立了一套围绕 $POOP 代币为基础的“回收”体系，当然在这个过程中，为了让 $POOP 代币持续的获得价值捕获的能力，以抵消潜在抛压所带来的影响，其也设计了一套高度通缩赋能机制，保证 $POOP 始终能够在其循环经济体系中，保持上涨螺旋，鼓励更多 $POOP 持有者成为 Holder。

 Poop 循环经济中的上涨螺旋

Poop 协议为 $POOP 建立了一套基于抵押弹性供应机制为基础的上涨体系，并且该资产无私募以及早期巨鲸投资者、做市上，通过公平的方式早期启动，保证 $POOP 能够实现上涨螺旋。

  • 购买促进 $POOP 价值上扬

当用户通过 POOP Station 使用 Layer 1 代币购买 $POOP 代币时，新的代币将通过 Poop Finance 合约铸造新的 $POOP 。而在购买过程中， 购买者需要缴纳一定的税款（10% ），这部分税款将分配给 Backing Vault（占比税收 60% ，该池为 $POOP 代币的流动性储备池）、LP 质押者以及贡献者。

虽然 $POOP 代币的总供应量增加，但税收机制保证了购买者实际通过 Layer 1 代币购买获的 $POOP 数量变少。

比如在 BNB/POOP 汇率为 1000 时，通过 1 个 BNB 在 POOP Station 购买获得的 $POOP 数量将小于 1000 个，进一步促进 $POOP 代币价格的持续上涨（价值上涨抵消通货膨胀带来的影响）。购买行为也促进了 POOP Station 金库中， Layer 1 代币的价值储备也正在上升。

  • 出售 $POOP 加速销毁 

当用户出售 $POOP 换取 Layer 1 代币时，虽然 POOP Station 中 Layer 1 代币储备下降，但出售的 $POOP 代币将被 Poop Finance 合约销毁。同时，$POOP 的出售同样需要缴纳税款（10% ），该税款也将分配给 Backing Vault、LP 质押者和贡献者，加速紧缩。 $POOP 的销毁，也同样将加速 $POOP 代币价格的上涨。

可见，无论是购买、出售 $POOP 都将为 $POOP 的价值上扬带来新的增益。这意味着，越来越多的用户将 “Shitcoin” 通过 Poop 实现回收，Poop 生态以及 $POOP 将不断地积累新的价值，让每一个参与者都是受益者。当然， $POOP 的上涨螺旋，正在促进更多持有者的长期持有，以从上涨螺旋体系中持续获利。

Poop 协议将于 6 月 28 日 20: 00 （UTC+ 8）在 BNBChain 上线，后续将进一步向 Ehtereum、Arbitrum、Optimism、zySync、Polygon 等生态部署。

结语

从行业的角度看，绝大多数加密玩家都是 “Shitcoin”的持有者，他们也都将是 Poop 协议的潜在用户群体，尤其在 Referral Program 的激励下，将加速这些用户涌入 Poop 生态的趋势。随着越来越多用户参与到“回收”交易中，也将加速 Poop 生态螺旋飞轮的运转，该生态的价值基数也将不断被放大。
