详解 DeWi 项目 WiFi Map：1.5 亿用户的 App 搭上了 Web3 的列车

作者 : 黑米，白泽研究院

一个新兴领域正在 Web3 中兴起——DeWi，即去中心化无线。

通过建立基于区块链技术的激励机制，借助“大众的力量”或网络运营商提供热点，用户以无需信任、无需许可和可编程的方式创建和管理去中心化的无线网络，而不再是由单一的、集中式的电信公司运营。

在这其中，Helium、Karrier One、Pollen Mobile、Hexagon 等项目脱颖而出。

但是，如果我告诉你有一个 DeWi 项目拥有 400 万每月活跃用户，产品有着超过 1.5 亿次下载，并且在应用商店被超过 200 万人评为 4.6 分？ 

很多人可能难以置信，但这就是事实。

这就是在今年第一季度从 Web2 过渡到 Web3 的公共热点网络 WiFi Map。

WiFi Map 的 Past：没有营销，却走向了世界

WiFi Map 成立于 2014 年，旨在成为一个“社区驱动的无线网络和自筹资金的初创公司”。由 Denis Sklyarov（首席执行官）和 Igor Goldenberg （首席技术官）共同创立，如今已在全球范围内发展到约 30 名团队成员。

Denis 和 Igor 曾在屡获殊荣的移动应用程序软件开发公司 Kenlo Group 共事，直到他们意识到有大量的人无法在连接可靠的互联网的情况下环游世界。

在当时，人们会在 Foursquare、Facebook、Tripadvisor 和其他平台上搜寻他们旅行目的地的详细信息，以提前了解是否有 WiFi。这对精打细算的旅行者来说是必不可少的，因为当时的移动数据非常昂贵。

在看到一些平台上的用户乐于与陌生人分享这些信息而不期望任何回报时，他们萌生了一个想法：是否可以将所有这些信息集中在一个应用程序上？以便人们可以快速、简单地访问。

早期的 WiFi Map 便应运而生，作为联创，二者每月可以获得 300 美元的应用程序收入。

不过，因为 WiFi Map 是以由旅行者、网络基础设施不足的国家民众组成的社区为驱动的，当他们增设了“添加 WiFi”的按钮后，很多人开始自愿添加热点。此后，WiFi Map 陆续添加了更多的“用户服务”，例如 WiFi 全球热点地图（可离线查看）、eSIM 计划（当 WiFi 不可用时的移动数据）、VPN 服务和网络速度测试。使之在印度尼西亚、印度等低收入国家被迅速采用。

最终，这个由贡献者组成的社区成为了一支有影响力的军队。创始人会分享应用程序的幕后开发，然后社区会在 Twitter 和其他社交平台上谈论 WiFi Map。

WiFi Map 在没有任何营销的情况下走向了世界。

在接下来的 8 年多里，WiFi Map 围绕热点共享和隐私建立了强大的双向经济。他们通过应用内的“积分”系统已经激励了超过 1300 万贡献者添加了超过 1 亿个 WiFi 热点，同时“积分”可以让贡献者在消费时打折。

这种“没有代币”的 DeWi 项目的最终结果是，他们现在正在为全球 200 多个国家/地区的 1.5 亿用户提供服务。

WiFi Map 的核心依然是“连接性”，就像我们今天看到的其他 DeWi 网络一样。只不过，他们并没有激励参与者部署硬件来构建新网络，而是利用现有的基础设施（公共/私人 WiFi 热点、本地手机信号塔），并改善了用户访问它们的简易性。

这种模式让 WiFi Map 进入了一个正飞轮：

应用程序的更多下载=更多的贡献者和用户

更多的贡献者 = 更多映射的 WiFi 热点

更多的贡献者+更多的 WiFi 热点=应用程序增值

应用程序增值=更多的广告商

更多的广告收入=拓展用户服务

拓展用户服务=更多下载

WiFi Map 的 Now：通过三个渠道提升年收入

根据 Data.ai 的数据，WiFi Map 目前的每年应用内收入约为 65 万美元。

根据公开信息，他们正在或希望通过以下三个渠道增加收入：

1. 额外服务付费订阅（核心产品仍然免费）

有三个付费订阅服务，“Pro”、“VPN”和“Pro+VPN”，并且按月付费和按年付费的价格还是有区别的。

在这里我们简单计算，假设所有的付费用户都是选择了最高级别的“Pro+VPN”。

假设 65 万美元的每月收入主要归因于付费订阅，那么大概只有 18,500 名付费用户。（65 万美元/35美元订阅费=18,571 名付费用户）

PS：一些用户可能选择了更便宜的订阅服务，因此现实情况的用户总数可能会更多。

作为一个拥有 1.5 亿总用户、只有约 18,500 名付费用户的应用程序，WiFi Map 当前每年收入为 65 万美元。但随着 WiFi Map 的继续发展，在付费用户增长十倍（≈18.5 万）的情况下，每年收入将增至 650 万美元。

2. 应用内广告

餐馆、商店等广告商可以付费在 WiFi Map 的热点地图上突出显示。虽然 WiFi Map 目前还没有通过这种方式获利，但计划未来会这样做。

按收入计算，谷歌是 2022 年全球最大的以广告为主要收入的公司。谷歌在 2019 年第二季度的 15 亿次广告展示和 4200 万次浏览的 CPM (cost per thousand views) 约为 40 美元。CPM 代表了每千次浏览的成本。

那么 WiFi Map 的 400 万月活用户/1000 = 4000。4,000 * 40 美元 (CPM) = 每月广告收入 160,000 美元 = 每年广告收入 192 万美元。

即在现有用户量的情况下，WiFi Map 一年的广告收入约为 192 万美元 。 

3. 集成第三方服务

这种渠道目前没有太多公开信息，可能 WiFi Map 会在未来与其他行业的公司合作。

WiFi Map 可以集成其他公司的第三方服务，为用户提供更多的便利。例如，在 WiFi Map 内集成 Uber 打车、酒店预订、旅行预订、汽车租赁。

在这种情况下，假设 WiFi Map 使用佣金结构，只对在应用内下单第三方服务的客户收取佣金。一般酒店、汽车/摩托车租赁、旅行社等第三方服务的佣金百分比范围约为 10~17%。

那么简单计算如下：

假设 400 万月活用户中，有 10% 的人每月在第三方服务上花费 10 美元。40 万月活用户 * 每年 120 美元的支出 * 10% 的佣金=每年 480 万美元的佣金收入 。

综上所述：

  • 在没有用户增长的情况下，WiFi Map 通过上述三个渠道的预计年收入 = 737 万美元。

  • 如果用户量增长 10 倍，那么这一数字 = 7370 万美元。

  • 从 2023 年 2 月 15 日到 2023 年 3 月 15 日，该应用在 Google Play 和 Apple App Store 中获得了约 185 万次下载，环比增长约 20%。虽然并非所有新用户都会变成活跃用户，但这是 WiFi Map 走向将其用户群扩大 10 倍的坚实道路。

所以，这时你可能不禁好奇，既然 WiFi Map 有这么多收入优势，为何还要转向 Web3？不妨想想他们能够成功的因素之一——社区。

WiFi Map 的 Future：从 Web2 跳进 Web3

WiFi Map 生态系统的发展，脱离不了社区的支持。

虽然 WiFi Map 开发商为用户引入了许多便利的功能，但能够增长的关键还是在于社区成员自愿“添加热点”，换取可有可无的“积分”。

不论是继续吸引更多用户使用 App，还是激励社区成员增加热点的覆盖范围，WiFi Map 转向 Web3，成为一个可持续、自给自足的去中心化热点网络看起来都是必要的。

2022 年夏季，WiFi Map 宣布，将在 Polygon 网络上推出原生代币 $WIFI，取代“积分”作为价值存储、记账单位和交换媒介。需要注意的是，以往的“积分”不能与 $WIFI 代币 1:1 兑换。

其实代币并不是 WiFi Map 的首选，只是 PayPal 等支付方式的费用太昂贵，而且有许多国家不接受它。

Igor Goldenberg，WiFi Map CTO：

在我们有了 $WIFI 代币的想法之前，我们想激励那些做出最重要贡献的人，但我们遇到了一些技术问题。例如，我们查看了 PayPal，但费用是天文数字，而且有许多国家不接受它。加密是一种通用的轻量级技术，可以在世界任何地方访问。我们知道有些人会决定将其用作投机工具，我们对此没有问题，但对我们来说，它是一个很棒的工具，可以为我们提供所需的实用性。

$WIFI 的推出和分发是为了激励社区成员映射新的 WiFi 热点、刷新旧的热点，并可用于在 App 上换取所需的服务。

除了一般应用程序原生代币都有的质押功能之外，$WIFI 的亮点还在于： 

1. WiFi Map 允许用户在 App 内使用 $WIFI 进行消费时，会以代币的形式返现消费价值的 15–20%。更有趣的是，通过法定货币在 App 内消费会以代币形式返现 3-5%。这是一种广泛的代币分配机制，可以让更多用户获得 $WIFI。

2. 用户使用 $WIFI 购买 eSIM 移动数据套餐时，会锁定 $WIFI，在套餐到期时，会退还给用户未使用的移动数据所对应数量的 $WIFI。

3. 在 App 内打赏提供热点的贡献者

4. 用于治理。鉴于该团队在近十年内成功激励了数百万贡献者，我相信他们会倾听那些每天使用他们应用程序的人的反馈。

此外，WiFi Map 还将致力于成为一个包含游戏化的去中心化网络，可能会与 Polygon 链上的其他项目合作，在 WiFi Map 热点地图上增加收集 NFT 的功能和其他游戏化元素。

$WIFI 总量为 10 亿：

  • 31.5% 用于生态系统奖励和发展

  • 20% 存入国库

  • 18% 分配给团队和顾问

  • 13% 分配给私人、公共投资者

  • 10% 用于流动性

  • 7.5% 用于营销

今年 3 月，WiFi Map 以每个代币 0.0125 美元的价格向私人投资者出售了 8.83% 的 $WIFI，筹集了约 110 万美元的资金。

3 月 30 日，他们以每个代币 0.025 美元的价格公开出售了 4.17% 的 $WIFI，再筹集约 100 万美元。

收益将用于扩大开发团队、增加应用内功能和服务，以及“在认为最佳时进行战略收购”。

结语

WiFi Map 的使命是使互联网民主化。

这是一个由 Web2 社区构建的应用程序，凭借其 1.5 亿次应用程序下载和超过 1 亿个互联网热点，进入了 Web3 行业。

拓宽收入渠道 + 拓展更多对用户有利的服务 + Web3 激励机制 = 有助于 WiFi Map 走向巅峰 = 提升 Web3 的高度

WiFi Map 的 Web3 之旅，也可以让我们看到，即使是成功的 Web2 公司也可以深思熟虑地推出代币，尤其是在熊市中，以促进增长并建立自己的迷你经济。
