撰文：Karen，Foresight News

去中心化钱包作为 Web3 超级流量入口，不仅仅承载着数千万乃至上亿用户的真金白银，也充当着连接不胜枚举的 DeFi、元宇宙、链游以及社交应用的网关。

Meta、Twitter、星巴克等巨头涌入 Web3 浪潮以及多地监管传递积极信号等，无一不在加速释放着流量红利。在此背景下，Web3 钱包竞争态势日益激烈，并且开始在生态兼容度、功能多样化、用户体验、安全检测等方面开启精细化竞争。

上周 Foresight News 发布了五款常用 Web3 钱包的硬核对比（《钱包的流量江湖｜硬核评测五款 Web3 钱包》），今天我们继续加码，把第一篇文章发布后社区内呼声最高的另外三款 Web3 钱包（BitKeep、TokenPocket 和 Trust Wallet）加入横评。

本文将全面评测八款 Web3 钱包，包括 MetaMask、BitKeep、欧易 Web3 钱包、Trust Wallet、Coinbase Wallet、imToken、TokenPocket 以及 Uniswap 最新推出的移动钱包，并试图从易用性、功能性、安全性等方面来呈现每款钱包的核心特点和优势。

MetaMask：插件简洁，Portfolio DApp 加持

MetaMask 作为以太坊 EVM 兼容生态的「独角兽」钱包，已成立近 7 年时间，其背后的开发运营商 ConsenSys 在去年 3 月宣布其月活用户数突破 3000 万。

MetaMask 插件钱包以核心功能清爽简洁和安全坐拥 Web3 钱包第一宝座，除资产存储之外，其插件钱包的四大主要场景为访问 DApp、购买、转账和交易，其中，MetaMask 提供的交易服务聚合了多个 DEX、DEX 聚合器和做市商的数据，来帮助用户以最低网络费用获得最优价格，不过交易服务会收取 0.875% 的费用作钱包持续开发。

去年下半年，MetaMask 推出 Portfolio DApp，可以将其视为钱包网页端，在资产可视化方面有很大提升，除此之外，Portfolio DApp 也支持加密货币购买、交易，还推出了跨链桥功能，目前也在测试质押，当前集成了 Lido Staked ETH（stETH）、Rocket Pool Staked ETH（rETH）。

值得一提的是，ConsenSys 此前更新的隐私政策中引发了用户对数据隐私的担忧，对此，ConsenSys 表示，打算将 MetaMask 用户数据仅保留 7 天。此外，在安全隐私方面，MetaMask 允许用户设置显示针对 Ethereum 用户的网络钓鱼域名警告，MetaMask 在近期更新的新功能中还支持插件钱包用户在创建新钱包时可以通过单击「设置高级隐私」以更改其 RPC 提供商并切换至第三方服务以发送请求，帮助检测网络钓鱼和识别传入交易等。

使用体验： 需自行设置 RPC，切链繁琐，偶有卡顿情况发生。

BitKeep：Web3 便捷参与入口

BitKeep 是一个相对便捷的 Web3 多链钱包，在去年 5 月份完成 1500 万美元 A 轮融资，Dragonfly Capital 领投，A&T Capital、Matrixport、丹华资本、Foresight Ventures 等参投。

BitKeep 当前支持超 90 条主链和基本的 RPC 配置，聚合了 Swap、NFT、DApp 探索等板块。其中，Swap 利用多个 DEX 和桥接器聚合的流动性帮助跨链交易，此外，BitKeep 的借 Gas 费交易功能帮助用户在无主网币作为 Gas 费的情况下完成兑换，比较方便。

BitKeep NFT 市场不仅支持以太坊、Polygon、BNB Chain、Arbitrum 等网络，除交易外还支持批量挂单、FreeMint 以及 LaunchPad 等功能。2 月份，BitKeep NFT 市场启动为期 6 个月的 Arbitrum 链 0 手续费交易活动，在本月还推出 Arbitrum Argonaut 生态月，启动包括 NFT 免手续费交易及挂单补贴在内的一系列生态激励措施。

此外，BitKeep DApp 探索页面也收录了比较全的 DeFi、NFT、GameFi、社交等应用。

安全方面，BitKeep 应用 DESM 双重加密算法，保证用户的私钥及助记词不被第三方通过破解算法来窃取，也接入了授权检测和合约检测服务。此外，BitKeep 储备 100 万美元作为安全保障基金初始基金，并承诺将每月 Swap 收入的 10% 分配至该基金。

使用体验： 交互相对流畅，支持生态和接链数量较多，聚合 Swap、NFT、DApp 探索、Launchpad 等板块，功能丰富。多链 DApp 切链时会自动弹出网络切换批准页面，相比 MetaMask 更方便，另外，红包、批量转账和 DEX K 线功能也比较有特色。

OKX Wallet：异构多链生态钱包和一站式 Web3 聚合入口

OKX Wallet 也是一款上功能比较齐全的异构多链生态钱包，当前支持 EVM、UTXO、Cosmos、Tron、Solana 和 Move 六大生态，已接入 50+ 条链。除此之外，OKX Wallet 也是为数不多的支持创建多套助记词钱包和一套助记词派生多个账户的多链钱包之一。

OKX Wallet 钱包内置了 DEX、赚币、NFT 市场、DApp 探索等板块，为用户提供一站式 Web3 入口和服务。DApp 跨生态无感切链功能是欧易 Web3 钱包的一大亮点之处，在连接多链 DApp 时，无需进行切链便可自由地在 EVM 及 Solana 生态间使用该 DApp 功能。

跨链 DEX 方面，OKX Wallet 聚合了逾 100 个 DEX，综合计算价格、滑点和网络费用，并通过 X-Routing 智能拆分订单为用户提供最优交易路径；NFT 市场方面，欧易 Web3 钱包整合了以太坊和 OKC 等 11 条链，聚合 OpenSea、LooksRare、X2Y2 和 Magic Eden 等主流 NFT 市场，还支持多平台挂单和 NFT 购物车；赚币方面，欧易 Web3 钱包聚合 300 多种链上投资协议和产品，并且支持一键跨币投资，允许用户一键将任意币种存入流动性池来节省网络费用。

安全方面，欧易 Web3 钱包还接入了 OKX 风险地址库和支持授权管理以及「首尾相似地址钓鱼检测」。

使用体验 ：支持生态和接链数量多，并且支持无感切链，比较便捷，交互相对顺滑。内置的水龙头功能对测试网交互用户也比较友好。

imToken：具备先发优势，但仅支持移动端

imToken 创立于 2016 年 5 月，2018 年获得由 IDG 独家投资的 1000 万美元融资，2021 年，完成由启明创投领投的 3000 万美元 B 轮融资。

凭借着先发优势，imToken 有一定的用户基础，当前支持比特币、以太坊、EOS、Cosmos 等多个生态，不过目前仅支持移动端，用户在连接 DApp 时需要通过扫描二维码建立 WalletConnet 连接来使用，或者在移动端 DApp 浏览页面自行搜索应用。

imToken 使用一组助记词为用户生成数字身份，数字身份下对应多链钱包就是用户的身份钱包。通过一组身份助记词即可恢复身份对应的多链钱包，免去繁琐备份步骤，如果要额外创建钱包，支持创建多套助记词钱包。

理财方面，imToken 支持稳定币理财和 Cosmos、Polkadot、Kusama 和 Tezos 质押挖矿。另外，imToken 交易服务已深度集成其孵化的 DEX 协议 Tokenlon。

使用体验 ：网页端连接 DApp 时需要在 imToken 移动端通过 WalletConnet 来连接，切链或交互时也需要在移动端输入密码进行确认，不过也可以用 Face ID 设置免密支付。交易时加载相对较慢。

Trust Walllet：中规中矩，内置隐私浏览器

Trust Walllet 是币安于 2018 年收购的一款移动钱包，收购后 Trust Walllet 独立运作。Trust Walllet 目前主要有 App 端和插件端，功能体验方面比较中规中矩，支持加密货币购买、存储、NFT 展示、质押、DApp 探索等服务。

Trust Walllet 移动端还内置 DuckDuckGo 互联网搜索引擎，该引擎承诺不监控、不记录使用者的搜寻内容。

使用体验 ：功能简单，极简风格

Coinbase Wallet：添加多种安全防护机制

Coinbase Wallet 插件钱包当前仅支持以太坊 EVM 兼容和 Solana 生态，其移动端和插件钱包界面也相对简洁，DApp 探索页面有接入 Coinbase NFT 市场以及一些流行的 DApp。

安全方面，Coinbase Wallet 提供交易预览、DApp 黑名单、垃圾邮件代币管理、恶意 DApp 访问警告功能，为用户提供多种安全防护机制。

TokenPocket：功能简洁，支持多签

TokenPocket 支持比特币、以太坊、BNB Chain、TRON、Aptos、Polygon、Solana、Cosmos 等网络，除基本的存储、转账功能之外，TokenPocket 支持闪兑和 DApp 探索功能。最近更新的钱包版本支持多签钱包创建和导入。

使用体验 ：功能简洁，DApp 探索只允许用户自行搜索，不提供 DApp 列表或趋势呈现。

Uniswap Wallet：支持探索 Uniswap 交易或 /TVL 趋势

Uniswap Wallet 移动端于本月初推出，目前支持 Swap 和 Web3 探索。DApp 连接时需要依赖 WalletConnect，交互时只需要确认签名即可。Web3 探索页面按照 Uniswap 上的成交额、TVL 或者市值排名展示代币，目前还支持代币和钱包搜索功能。

使用体验 ：Uniswap Wallet 产品目前还比较粗糙，Web3 探索页面加载缓慢。

小结

MetaMask 凭借先发优势（2016 年创立）以及简洁易用、直观友好的特点在 Web3 钱包中占据主导地位，不过，仅支持 EVM 生态、需自行配置 PR 以及交易手续费较高的特点促使用户探索其他异构多链钱包。同年创立的多链钱包 imToken 虽有一定用户基础，但功能简洁，仅支持移动钱包，DApp 访问也依赖于 WalletConnect。

BitKeep 和 OKX Wallet 近期也开始发力，致力于打造 Web 聚合超级入口，内置 NFT 市场、跨链 Swap、DApp 探索等比较全面的功能，并且每个板块的细化功能也相对丰富且易用。

Trust Wallet、Coinbase Wallet 以及 TokenPocket 的界面均相对简洁且中规中矩，没有太多复杂的功能，最近推出的 Uniswap Wallet 产品目前也尚待完善。
