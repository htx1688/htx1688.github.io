期货是金融服务行业重要组成部分，也是降低市场风险的最主要形式之一，允许交易者实施更稳固的风险管理策略。Injective 正是一个基于以太坊的 Layer-2 项目，侧链基于 Cosmos SDK 和 Tendermint 共识机制，快至 1 秒的出块时间，可以极大的提高订单撮合效率并且支持以太坊跨链。为了解决抢先交易问题，通过集成可验证延迟函数（VDF）为订单添加时间戳，杜绝交易作弊，防止矿工「抢跑」用户。

本文接下来将介绍 Injective 生态联盟中的各个主流应用。

NFT

Burnt Finance 是首个基于 Solana 的完全去中心化拍卖协议。Burnt 允许用户自己进行铸造、交易和拍卖各类资产。该平台可支持的拍卖资产类型包括合成资产、NFT、数字资产等。Burnt Finance 由 Burnt Banksy 发起，他是一位匿名人物，通过烧毁著名街头艺术家班克西 (Banksy) 的作品铸造成 NFT 版本，在艺术圈引起了轰动。Burnt Banksy 表示：“我们这么做是想说明实物艺术品的价值能够转移至 NFT 作品中，NFT 作品能确保区块链技术的智能合约及作品的唯一性。”这一行为获得了广泛的报道，包括彭博社、CoinDesk、GQ、CoinTelegraph、BBC、CBS、HYPEBEAST、Art.Net 等。Burnt Finance 由 Injective孵化，也是 Injective 第一个生态子项目。

Superfarm 是一种跨链 DeFi 协议，旨在简化 NFT 代币的发布流程，并无需代码。通过市场和创新的工具套件，任何项目都可以使用自定义规则部署挖矿。SUPER 是 SuperFarm 的原生功能型代币，用于治理、支付费用、抵押和获取 NFT。

Dvision Network 是一个基于区块链的 NFT 元宇宙平台。用户可以通过电脑、移动设备或 VR 头盔访问内容，内容创作者可以在元宇宙内制作时装秀、艺术展、游戏或虚拟商店。平台有三大支柱：NFT 市场（用户创建或交易独特数字商品的 NFT 市场）, Meta Space（用户在这里获取并自定义自己的虚拟空间），Meta City （虚拟空间的公共集合）。虚拟环境支持各种社交、商业和服务相关工具的使用。这将包括聊天功能和视频会议，还有人工智能控制的非玩家角色和综合支付选项。

预言机

Chainlink 是业内顶尖的智能合约去中心化预言机网络解决方案，使智能合约能够安全地访问链外数据源、网站 API 和传统的银行支付。

UMA 是以太坊上部署和执行合成资产的开源基础架构，使开发人员能够快速轻松地构建可跟踪任何价格的合成代币。UMA 与 Synthetix 之间的主要区别之一是，抵押品和债务敞口在 UMA 中是隔离的，而它们在 Synthetix 是汇总在所有 LP 中，这降低了单个合成资产市场的流动性，不过也降低了用户风险。同时，UMA 还开发了管理和执行 UMA 合约的去中心化预言机服务，并提出了「无喂价」，指以最小化链上预言机的使用，减少预言机攻击的频率和攻击面，主要通过引入「清算者」和「争议者」这两个参与者实现，本质上是一种博弈机制。

Band Protocol 定位 Layer 2 中间件协议，为区块链提供了合约调用更快、手续费更便宜、更易于集成和可扩展的去中心化预言机。更重要的是，Band Protocol 基于多代币机制的经济激励模型和以代币经济驱动，确保生成可信、准确和安全的数据源。

DIA 是一个利用加密经济激励来收集和验证数据的开源金融信息平台。DIA 让市场参与者能够供应、分享和使用加密金融数据和传统金融数据。预言机仅仅是 DIA 功能的一部分,与 ChainLink 以及 DOS 等喂价模式不同。DIA 预言机的数据均来源于交易所的 API 以及其他公共来源，这一点上来看其保证了数据的准确性以及透明性。

API3 是一种推广「去中心化 API」（dAPI） 开发的解决方案，实际上是链上的聚合器合约。dAPI 接受 DAO 第三方服务提供商的管理，后者运营自己的预言机，可以直接将数据写入区块链。这些预言机都接受 API3 去中心化自治组织 DAO 管理。

Kylin Network 致力于建立一个跨链平台，为 Polkadot 上的数据经济提供动力，将成为 Polkadot 上 DeFi 和 Web 3.0 应用的数据基础结构，并为其提供经过验证的、安全可靠的、具有成本效益的和易于协调的数据源和数据分析。

Razor Network 是一个去中心化的预言机项目，当前在以太坊网络上运行，但并未将目标锁定在以太坊主网上的应用，而是会与多个 L1 和 L2 区块链合作，以在其平台上提供完全去中心化的预言机。希望以快速、安全的方式将智能合约与数据连接起来。目前旗舰产品去中心化预言机网络的测试网已经推出，与此同时官方还发布了用于浏览测试网信息的测试网浏览器。Razor 采用滑动计算机制阻止可能在预言机上发生的无效源以及（行贿）信号等攻击。该机制可迭代解决对预言机结果的争议，并且会保证系统安全新稳步提升。项目着眼于在不牺牲速度的大前提下提供博弈论所许可的最大安全性，也正因如此，Razor 在效率和安全性上寻找到了一个平衡点，而其完全去中心化的设定也避免了中心化预言机可能遭遇的瓶颈问题。

Lithium Finance 是私有资产数据预言机协议，为 DeFi 协议提供如 pre-IPO 股票价格、私募股权价格等无法立即兑现的资产价格喂价，通过给予报价正确者激励与报价错误者惩罚来实现喂价精准度。

数据处理

Big Data Protocol 是一个为具有商业价值的数据提供流动性市场的协议，类似于去中心化的数据交易市场，其数据主要来自该项目联合创始人在 15 年创立的数据平台 Amass Insights 所对接的 14141 家专业数据提供商，例如由 ClosingBell 提供的股票评级数据、由 Glassnode 提供的链上交易指标等、由 Wootrade 提供的币安 BTC / USDT 每小时数据。

Litentry 是一个基于 W3C 标准提出的一个跨系统的 DID 聚合协议，可以给不同系统的 DID 数据提供互操作性和流动性。并通过分布式去中心化身份（DID）索引和基于 Substrate 构建的信用计算网络，为 DID 用户和 DID 应用程序提供了提供了包括 DID 验证、信用计算及评级等身份聚合服务。Litentry 通过将多个 DID 标准集成到聚合身份模型，解决了未知 DID 机制带来的技术难题。这些服务可以在 DeFi 和链上治理中被广泛应用，用户能够聚合其任意身份数据并利用其 DID 高效、安全地参与链上治理。

Ocean Protocol 是一个基于区块链构建的生态体系，它使个人和企业都可以轻松解锁数据价值，并通过使用 ERC20 将其代币化。通过 Ocean Protocol，发布者可以在保护隐私和控制权的同时通过其代币获利，而用户则可以访问以前不可用或很难调用的数据集。该项目已经开始与知名汽车制造商戴姆勒（Daimler）以及顶尖的加密项目（例如 Filecoin）等公司合作。

Covalent 是一个由数据库和分布式系统领域的资深人士所研发的提供区块链数据查询与索引服务的开发者平台。它利用大数据技术从数以千亿计的数据点中创造意义，并通过提供统一的 Covalent API，让开发者能更加快捷地调用链上数据，让整个区块链网络呈现更高的数据透明和可见性。

The Graph 的愿景是基于 Web3 生态做一套高效且去中心化的区块链数据索引及查询 API 服务层，让 DApp 数据的访问变的更容易和更迅速。它的原理是将区块链数据进行索引，生成子图， 进而允许快速的查询接口。 它使用的查询语言是 Facebook 发明的 GraphQL 语言，该语言的特性是可以在一次查询请求中尽可能高效地返回结果。

稳定币

Reflexer 是一个旨在打造第一个只支持 ETH 的去中心化，非挂钩稳定资产 RAI 的平台。RAI 可以作为其他 DeFi 协议的更「稳定」的抵押品（与 ETH 或 BTC 相比），或作为内含利率的稳定资产。值得注意的是，官方指出，RAI 并不属于稳定币 ，其背后的系统也只关心市场价格尽可能地接近赎回价格。

Terra 是一种支持稳定的可编程支付和开放金融基础设施的区块链协议，由一篮子法定挂钩的稳定币提供支持，这些稳定币由其原生代币 Luna 在算法上维持稳定。Terra 稳定币解决方案的核心是通过调整供需算法的智能合约以自动增加或紧缩货币供应量来维持价格稳定，同时其锚定一篮子法币的稳定币可在链上可以实现互换。

DeFi

MDEX 是基于自动化做市机制的去中心化交易协议，旨在融合多种基础公链的差异化优势，打造高性能复合型 DEX 生态，以流动性挖矿与交易挖矿的“双重挖矿激励”给予参与者最大化回馈，并通过手续费回购销毁机制实现了自驱式价值捕获生态闭环。MDEX 致力于打造集 DEX、IMO、DAO 为一体的 DeFi 平台，为更多优质资产提供一站式流动性服务，为用户提供更加安全可信、资产多样、高性价比的交易体验。同时，也是首个支持手续费回购销毁平台 TOKEN 的去中心化交易协议。

Alpha Finance Lab 是一个由跨链 DeFi 产品组成的杠杆生态系统，这些产品将相互配合，为用户带来最佳的 Alpha 收益。Alpha 产品专注于以创新和用户友好的方式捕捉 DeFi 中未被解决的需求。

Acala 是全球首个去中心化开放式金融联盟、波卡 Polkadot 生态金融中心，旨在联合波卡 Polkadot 生态系统创建开放式金融框架，为全球加密资产带来金融稳定性，流动性与可及性，已推出了跨链多资产抵押的稳定币 Honzon 协议、释放 Staking 资产流动性 Homora 协议以及去中心化交易所等核心 Layer 1 模块。

Moonbeam 是 Polkadot 上面向开发人员的区块链中间件，致力于提供与现有的以太坊兼容的开发工具和网络。Moonbeam 提供完整的 EVM 兼容的环境与 Web3 兼容的 API 来构建与以太坊间的桥梁。

Kava 可以说是现阶段币安生态中最璀璨的明珠。Kava 基于 Cosmos SDK 开发，采用权益证明（PoS）作为共识机制，Kava 最大的亮点就是将陆续支持「主流跨链资产」的抵押借贷，包括 BNB、BTC、XRP 和 ATOM 等。用户可以将自己的加密资产作为抵押品，来换取 Kava 的稳定币 USDX，持有 USDX 将获得储蓄奖励，储蓄率是可变的，目前的年化率大约为 4-5%。

Mirror 是基于 Terra 生态的合成资产协议，用户通过可以铸造、交易股票、期货、交易所基金等资产的合成代币，其中主打产品是美股合成资产，帮助 DeFi 用户可以低门槛地投资各类美股资产，Mirror 用户可以通过 Terra 稳定币 UST 或 mAsset 超额抵押以铸造美股合成资产代币，UST 最小抵押比率为 150%，mAsset 最小抵押比率为 200%，目前支持铸造几乎所有主流的美股资产同时质押铸造，同时这些代币可以在 Uniswap 和 Terraswap 上进行交易。

Persistence 是基于 Cosmos 跨链协议之上开发的互操作协议, 旨在创造一个包括债务融资、借贷、DEX、固定收益等在内的去中心化金融生态系统，一方面利用 NFT 技术和协议将汇票、信用证和提单等传统的商业票据代币化，成为加密世界里通行的质押品和交易标的，在解决广大中小企业融资难的过程中，将传统金融引入了 DeFi 的星辰大海，另一方面通过 Cosmos 跨链技术，让 PoS 原生代币持有者能够进入分享以太坊丰富 DeFi 产品和收益。

TomoChain 是越南最大的公链项目，是一个针对目前所有区块链网络 (特别是以太坊) 可扩展性问题的创新解决方案，TomoChain 计划透过添加更多与 Ethereum 网络集成的高性能第二层区块链来支持横向扩展，使 TomoChain 可以支持备份与原子跨链交易。

DAOMaker 项目始于于2018年，初衷是将风险投资带向大众。DAOMaker 为初创公司提供了成长的技术平台和融资框架，同时为投资者降低了风险。这种方法已经通过DYCO + SHO模型证明了其需求和成功：DYCO，动态代币发行，可以理解为DAOMaker的私募平台，托管平台可以为投资人提供退款和代币领取。SHO，可以理解为DAOMaker的公募平台。DAOMaker 平台已经为ORN、DeFiner、PlotX等多个知名海外初创项目合规募资超过3500万美元。

KardiaChain 是一个旨在实现完全可互操作的且“非侵入性（non-invasive）”的区块链平台，这意味着区块链可以无需对自己的协议进行技术修改便可以连接到网络。KardiaChain 使用双主节点技术来促进其自己的分布式账本与其他区块链之间进行连接，它通过分片以实现可伸缩性，并为开发者提供智能合约。

Ramp 是 Solana 公链上为 SOL 代币持有者提供抵押资产的流动性和跨链流动性挖矿收益的协议。Ramp 致力于从抵押的数字资产中释放流动性。Ramp 的解决方案使 SOL 用户能够获得额外的流动性，而不必放弃抵押贷款回报或增值潜力。通过 RAMP 智能合约，用户可以抵押 SOL 在 Solana 网络上获取原生稳定币 solUSD，然后在以太坊网络上将 solUSD 1:1 兑换成等值的稳定货币 rUSD。持有 rUSD 的用户可以借入或兑换扽同于法定货币的稳定货币，例如 USDT 或 USDC。这些稳定的货币可用于投资新的 DeFi 项目、交易或兑换成法定货币。

AllianceBlock 正在建立全球首个合规的去中心化资本市场。AllianceBlock 是一个去中心化，隐式区块链的 2 层 (layer 2) 协议，致力于推出新一代的加密货币产品。AllianceBlock 由欧洲最负盛名的三家孵化器：苏黎世的 Station F，L39 和 KickstartInnovation 孵化，并由经验丰富的前摩根大通，巴克莱，法国巴黎银行，高盛投资银行家和量化团队带领前进。AllianceBlock 力图打造最先进的全球合规的去中心化资本市场，实现安全高效的未来金融市场的愿景。

Frontier 是链外的 DeFi 聚合层。用户透过其应用可以参与协议跟踪和管理、质押、最优惠的资产交换、提供流动性提供、创建 CDP 流动性等等，目前 Frontier 正在为 Solana 与 Serum 生态做出巨大的努力。

UniLend 是一个无需许可的去中心化协议，该协议将现货交易功能与同一平台内的借贷服务结合在一起，使任何用户都能够在 UniLend 上发布 ERC20 通证，此举为行业所带来的改变彻底打开了以太坊链上资产的去中心化交易和借贷的大门。UniLend 的使命是“使今日所有的通证都能运行在 DeFi 的世界中”，团队目前还在探索对更多区块链的支持。

PlasmaPay 是一款基于 Plasma 分布式账本，兼有加密货币 / 法币账户的电子支付去中心化应用（DApp），帮助用户打通法币出入金通道的同时，让用户更友好、便捷地享受 DeFi 服务。作为全球电子商务平台的支付网关，PlasmaPay 允许用户和商家购买，出售，存储和支付加密货币。与传统中心化支付系统不同，用户可以完全自由地控制其数字资产，并完成支付。截至目前，PlasmaPay 已经在 165 个国家运营，并和众多银行、电子货币机构合作来为全球用户提供法币网关服务，超 200 个国家的个人和企业用户已经开始享受 PlasmaPay 提供的多币种钱包服务。

StaFi 协议分为底层、合约层、应用层，底层是使用和波卡相同的开发框架 Substrate 开发的一条独立链，目前已上线主网。合约层支持创建多种 Staking 合约（Staking Contract），通过 Staking 合约 Stake 的资产将会生成 rToken，rToken 可流通、可交易。 应用层支持创建基于 rToken 构建的各种市场，如交易、借贷等。Staking 合约简称 SC，用户使用 SC 合约进行 Staking，即可获得对应的 rToken。rToken 是用户取回 Staking 资产的唯一凭证，且能够捕获 Staking 产生的收益。简单说 rToken 是一种 代表了其公链原生资产的生息资产，持有 rToken 能够自动获得 Staking 收益。

EasyFi 是基于 Matic Network 的数字资产 Layer 2 层 DeFi 借贷协议，旨在解决第一代 DeFi 解决方案在交易速度、成本方面，面临的一些固有挑战，突破较大规模 DeFi 操作的瓶颈。从 EasyFi 协议的自身职能上看，EasyFi 大体提供了四个去中心化金融板块包括小额贷款、抵押贷款、信用委托以及违约债权转让四个板块。

Tidal 是一个跨链的去中心化互助保险协议，其核心功能是 TIDAL 相互保险池和相应的储备智能合约。每个设置的池为加密货币合约或资产的组合提供保障。这使得储备金可以部分准备金的形式，根据准备金率和被保险资产之间的相关性，为流动性提供者（LP）提供各种级别的杠杆、收益和风险。正如 Tidal CEO Chad Liu 所述，若想最大程度地利用资本和回报，必须通过提供低相关性的捆绑保险协议来加杠杆扩大保额，同时通过多层次的准备金来降低控制破产和其他风险。

CoinWind是一个DeFi智能挖矿金融平台，通过合约自动将质押的币种进行撮合配对，配合对冲无偿损失策略，将用户收益最大化，有效解决了用户单币种挖矿收益低、LP挖矿无偿损失大等风险问题。CoinWind旨在打造一个DeFi数字资产银行，推出高收益与安全可靠并存的产品，将用户存入的不同代币汇集到不同的池中，然后根据策略调动某代币池的资金参与不同的流动性挖矿，从而为用户实现更高的收益。CoinWind产品会通过组合策略有效降低流动性挖矿中的无常损失。CoinWind首发火币生态链Heco，之后将跨多条主链如ETH、BSC等运行。

Formation Fi 是一种跨链的风险平价 Farming 协议，该协议可创建基于预计资产风险回报率的算法重新平衡投资组合。例如，规避风险用户的资金将被部署到更安全的策略中，这些策略涉及的风险较小，但提供的 APY 也较少。

公链

Injective 和众多公链项目有着合作关系，除了以太坊和币安智能链之外，还有火币Heco，波卡，Cosmos，Polygon，Klaytn，Avalanche，Elrond，Near，Stacks, Harmony，Fantom，Conflux。

Klaytn是韩国社交媒体巨头Kakao旗下区块链子公司Ground X推出的区块链平台，核心业务是国民即时通讯软件Kakao Talk的电子钱包“Klip”。用户可通过Klip管理或传输加密货币，并且能够体验NFT。而在今后还将对各类数据、版权、债券、衍生产品、优惠券、周边等进行数字资产化，通过Klaytn平台对上述资产进行交易。用户可以在自己的KakaoTalk虚拟资产钱包Klip对其进行整合管理。

Avalanche 是一个高性能，可扩展，可自定义和安全的区块链平台。 它针对三个广泛的用例：

构建特定应用程序的区块链，跨越许可（隐私）和非许可（公共）部署，构建和启动高效可扩展和分散的应用程序（Dapps），使用自定义规则，约定和附加条款（智能资产）构建任意复杂的数字资产。Avalanche 的总体目标是为创建，转让和交易数字资产提供一个统一的平台。AVAX 与同类型协议的核心优势在于可扩展性和去中心化程度。

Elrond 是一个自适应状态分片的高吞吐量公链 。 他们提出了一种改良的权益证明机制，称之为安全权益证明（SPoS）的共识机制，引入随机选择的共识组，抵押加评级的方式。与 Harmony 类似，Elrond 也有一条主链，称之为公证链或元链（Notarization (Meta) chain），负责所有网络和全局数据操作，比如节点加入和离开网络，验证人列表，节点分配等。

Harmony 是一个基于状态分片和 PoS 的高性能公链项目，它的分片架构由一条信标链和多条分片链组成，信标链提供包括去中心化的随机数，分片链 Header 的验证，接受验证节点的权益抵押等服务。 在共识算法方面，Harmony 在每个分片内部使用 FBFT 达成共识，FBFT 是 PBFT 深度优化的共识，通过 BLS 多重签名，使传统的 PBFT 的效率提升了一个数量级，通过这种方式在分片内快速达成共识。目前 Harmony 主网 v0 版本，每个分片内有 250 个节点，出块时间在 8 秒以内，官方说优化后预计达到 5 秒以内。

Near 是基于全状态分片、对开发者友好的可拓展性公链，他们提出了一种新的协议和解决方案，称之为夜影协议（Nightshade）。Near 与其他分片公链不同的是，它的技术架构并不是由一个信标链和多个分片链组成的，而是把系统建模成一个单独的区块链，在区块级别进行了分片，每个分片中有很多“段”。具体来说，Near 将区块分为一个个物理组成块，因为每一个区块都包含这些物理组成块，经过对多个区块的验证，物理组成块均相同的时候就可以验证这些区块是正确的。每个区块逻辑上包含所有分片的所有交易，并切分分片的状态，验证者只负责验证交易所针对分片相对应的状态即可。

Stacks 是一个公链项目，成立于 2015 年，目前已有两个版本的迭代：v1 版本旨在建立一个分布式计算网络，全栈式替代传统云计算，v2 转向了比特币的链上智能合约开发以及生态应用。团队共募集了 7,560 万美元，资金量较大，目前并资金量充足。其 v2 （Stack 2.0）版本在 2021 年 1 月 14 日已经上线，推出新的共识机制 PoX，让 Stack 2.0 的安全性可以受益于比特币网络，同时，Clarity 智能合约语言可以帮助开发者根据比特币链上交易状态编写和触发智能合约。

Fantom 是首个基于 DAG 技术并智能合约研发商，致力于解决现存区块链技术的可拓展性和处理时间的问题。Fantom 与传统的基于分块分类账的存储方式不同，通过 DAG 改进版本来搭建基础框架，并采用 Lachesis 协议。在 Fantom 上构建的应用，OPERA Chain 享有即时处理和近乎零的交易成本。

Conflux 巧妙融合了 DAG 技术和 PoW 共识机制，扩展了交易并发度和系统吞吐量。与 Byteball 此类典型的 DAG 技术项目相比，在共识层面提高了系统安全性。

工具类

做为去中心化金融的基础衍生品组件，Injective 可与 DeFi 世界的各种 Dapp 组合使用。目前已与众多知名项目达成合作，未来会互相集成以提高 DeFi 的普适性。

WOOTRADE 是由 Kronos Research 孵化的暗池交易平台，全天候为交易平台、钱包等用户以零手续费的价格提供交易流动性，并鼓励做市团队随时提供最佳的市场报价。据官方介绍，目前，归功于其背后的 Kronos Research 团队，WOOTRADE 可以在 0.2% 的价差下提供 100 BTC 以上的交易深度，不收取客户任何手续费，或服务费。交易平台和 DEXs 可以以零成本从 WOOTRADE 获取订单，从而增加流动性。主要节点还享有实现流需求的收入分成。对于交易平台来说，无手续费提供流动性可以降低平台交易成本，同时让平台拥有给交易用户降低手续以吸引新用户和提升新老用户交易频率的条件。

Marlin 旨在通过优化区块链之下的网络通信架构，将 Layer 0 作为开放式协议之一，以此来给区块链带来更好的可扩展性、灵活性以及去中心化特性。Marlin 作为 Layer 0 试图要解决的问题。它试图在目前的物理条件限制情况下找到最快的区块传播速度方案，从而提高可扩展性，缓解 Layer 1 公链的通信延迟烦恼。在 Marlin 协议的解决方案中，有两个核心，一个是中继网络，一个是经济激励机制。在 Marlin 网络中，有专门用于中继的节点，叫做 Marlin Node，节点相互之间传递信息，可以加速传播速度。Marlin 借鉴了 CDN 加速技术，对网络进行了划分，因此也有很多中继网络。

Orion Protocol，中文名猎户座协议。Orion Protocol 是一个提供流动性聚合方案的 DeFi 平台，旨在通过将整个加密市场的流动性汇总到一个去中心化平台中来解决 DeFi 中的痛点：从每个主要的中心化交易所，去中心化交易所和流动性池中将流动性进行聚合。Orion 核心的流动性聚合器支持基于该协议构建的每个解决方案，包括革命性的交易解决方案 Orion Terminal，在一个去中心化平台上聚合整个加密货币市场的流动性，以及针对区块链，交易所和加密货币项目的一套企业解决方案。
