摘要：

近期，新兴 NFT 项目 Be VEE 逐渐展露头角，其创作者 RDR 尝试探寻一条完整的艺术流量变现道路。该项目源自艺术家的艺术创作，再借助艺术创作者本身的流量转化为 NFT，以形成一个相对完整的 IP 链条。作为一个实验性的项目，其目前取得了不错的结果，但其最大意义的还是在于对艺术创作者变现道路的探寻。 后续，艺术创作者可以以此作为借鉴，尝试探索一条去中介化的艺术变现道路。

作为一款 freemint 项目，在短短的一个多月内，Be VEE 的推特粉丝数量就已上涨至9.5万人次，14日时更是登上了 OpenSea 24小时销量排行榜的首位，单日交易量高达300多枚以太坊。虽然其目前的地板价仍处于下跌趋势，但基本其本身的特性，笔者认为其也许具有一定的挖掘价值。

当看到 Be Vee 这个项目画风的第一眼起，笔者就不由得想起今年8月份横空出世的爆款 NFT DigiDaigaku。作为 freemint 项目,其地板价一度飙升至15.78枚 ETH，实现了熊市下的万倍跃升，着实是让人瞠目结舌。要知道，这个项目最初面世时其实并没有受到太多的关注，反而因其过于幼态化的二次元画风和过少的曝光信息而不被看好，但后来的结果却是大大出乎意料。时至今日，DigiDaigaku 的地板价仍有13.3枚 ETH，且价格较为坚挺。一定程度上，其已经具备了成为蓝筹项目的潜力。而且它的团队是传统 Web2 世界里一家非常优秀的游戏开发公司，后续或许会继续为 DigiDaigaku 赋能，使其价格得到进一步的攀升。

而如今，呈现在我们眼前的 Be VEE 不论在画风方面还是曝光程度方面都与当时的 DigiDaigaku 具有一定的相似性，因此错过了 DigiDaigaku 的笔者不禁提起了警惕：Be VEE 是否有潜力成为10月份的 DigiDaigaku 呢？所以，今天就让我们来共同探究一下 Be VEE 这个项目的前世今生。

项目简介

Be VEE 是由韩国艺术家 Ramdaram 创作的1万枚复古风 NFT 的集合。由于其创作者 Ramdaram 在YouTube 和推特上活跃已久，且拥有相当稳固的一批粉丝，所以 Be VEE 在面世之时就已经收获了一定的关注。

Be VEE 的官方推特从2022年8月底开始投入运营，时至今日也不过两个月的时间，而其将近十万的粉丝数，从客观上来说已经是一个相当出众的数字了，这些粉丝很大程度上都是被艺术家本身的流量吸引而来的，但也不排除有买粉的嫌疑。

Be VEE 在2022年10月13日正式开始 freemint ，14日时一万枚 NFT 就已经售罄，在当日其热度也达到顶峰，日销售量高达300多枚 ETH。但自此以后，其地板价不断探底，截至目前仅有0.06枚ETH。笔者认为，之所以有这样的价格表现还是由于其实际权益和曝光信息的限制。换言之，某种程度上，Be VEE 赋予持有者的实际权益，以及 NFT 本身的曝光信息可能无法支撑起太高的价格。

不过，即便 Be VEE 的价格目前仍处于下跌状态，但我们也不能全盘否认其可能具有的潜力。毕竟，DigiDaigaku 在8月时就已经为我们上了生动的一课，所以为了避免错过良机，我们需要更加全面地审视这个项目。

就 Be VEE NFT 本身而言，其画风还是相当讨喜的，复古又充满了个性，同时它没有 DigiDaigaku 的那种幼态，这就意味着它的接受面可能会更广一些。而且 Be VEE 的价格之所以下跌，很大程度上也是受到目前熊市的整体走向影响。要知道，绝大多数项目在当前的市场下其实都是存在着巨大的抛压的，由于大家都希望能够快速见到利益的回升，因此一旦没有看到即时的效益就难免会产生想要抛售的想法。

另一方面，就像上文提到的，目前 Be VEE 这个项目所披露的信息确实比较少，当下已知的就只有其艺术创作者本人的信息，而关于运营团队和项目的其他情况都是未知的。因此根据其反馈的信息也就导致价格只能表现到当前这个程度。不过正是因为其反馈信息较少，或许在后续信息进一步曝光的时候，可能会给这个项目带来进一步的价格爬升，因此并不排除其具有一定的上升潜力。

项目优势分析

在如今的市场环境之下，应用端项目的生存普遍比较困难，而作为我们今天主角的 Be VEE 却在熊市同时收获了可观的关注度（推特粉丝数）以及不错的销量，因此，Be VEE 自身必然也存在一些可取之处。

1、创作者的IP效应

Be VEE 的创作者为 Ramdaram（람다람），其粉丝通常称其为 RDR，他是一名韩国动画师和艺术家，主要活跃于 YouTube 和推特。

目前，RDR 在 YouTube 和推特上分别有86.1万和12.7万订阅者，其代表作品《苏打城市放克》播放量高达4000多万。除此之外，RDR 4年前的作品《两次》也有高达2000多万的播放量。由此我们可以看出，RDR 是一名在业界颇具声誉和才华的创作者，其自身本就凝聚了相当一批粉丝群体，而作为其亲自操刀发行的NFT项目，Be VEE 大概率会被其粉丝爱屋及乌（事实也是如此）。

以往，RDR 的作品形象以一个粉红松鼠女孩为核心，因其形象酷似松鼠，所以名为 Dharam。她有着相同颜色的眼睛和松鼠尾巴，常常把头发盘成耳朵一般的发髻，有时还会露出锯齿状的牙齿。Dharam 最具标志性的穿搭是：蓝绿色的运动服、裤子、白鞋和耳环，当然她也不会只有一种搭配，其服装通常会随着视频内容的不同而发生相应的变化。

根据以上描述，我们不难发现，Be VEE 本身具有较为丰富的背景设定，这也是其魅力所在。而这与作者 RDR 丰富的创作经验与深厚的艺术造诣也是不可分割的。

2、Free mint加成

所谓 Free Mint 就是指交易成本只包括最基本的 Gas 费，而无需支付任何其他费用。在整体走熊的大环境下，这种几乎零成本的方式，对于加密用户来说无疑是极具吸引力的。因为其意味着，用户可以以低风险的方式博取高收益的可能性，而在相对缺乏稳定性的投机市场，这种形式无疑是性价比极高的。

在项目发展的前期，正是需要流量和热度的时候，而这种方式可能会吸引大批用户涌入，以助力项目的长期发展。

3、CC0的采用

CCO NFT 所代表的其实是一种版权形式，在这种模式之下，NFT 的创建者可以允许 NFT 为他人完全所有，其形式类似于一种知识共享。而创作者本身在转移 NFT 之后没有保留任何合法权益，这意味着任何人都可以将自己所有的 NFT 用于商业用途，而无需注明其团队或艺术创作者。

这就使得 CC0 NFT 具有更为广泛的用例和拓展空间，因此具有商业意图的用户将被吸引参与其中。这使得这类项目能在较短时间内快速涨粉、引发热度，并收获关注与流量。同时，这种高度自由的模式也能在一定程度上调动更多创作者的参与积极性，从而借助其他创作者的力量促进项目的不断更新发展。所以，就总体而言，CC0 对于创作者、持有者和项目本身都将产生巨大的助力。

4、颇具人文关怀的艺术内核

区块链作为一种突破主流的文化和技术形式，本身就具有一种反建制的精神色彩。而 Be VEE 所希望表达的也恰恰如此。对于 Be VEE 的主题，项目方给出的描述是“VEE yourself BE yourself”——你自己要做你自己。

即便不愿接受，我们还是不得不承认当今的世界仍充斥着各种虚伪、傲慢、不公与歧视。这种境况之下，人们不得不委曲求全，戴着无形的面具隐藏起真实的情绪和状态，惶惶终日，可这真的是我们想要的吗？我想必然不是。

因此，在 Be VEE 的设定中便出现了机械降神一般的药丸——Vee，而吃下药丸的人开始能够露出真正幸福的微笑，逐渐回归最本真的自我。这也是 RDR 想通过 Be VEE 这个作品表达的情绪——摘掉面具，做回自己，找回真正的笑容和更加真实的情绪。

因此，Be VEE 这个 NFT 系列最大的艺术性表达就在于其所包含的人文主义关怀，就如文艺复兴时期的作品一般，RDR 开始着眼于人进行创作。

同时就 Be VEE 的绘画风格来说，其以强烈的色彩对比以及以可爱又富有个性的女孩子作为主题，会给观众强烈的视觉冲击和亲和力。这使得 Be VEE 本身具有更为广泛的接受度。

项目劣势

诚然，如同上文的分析一般，Be VEE 存在着许多无法忽视的优势。但就目前其价格和热度的表现也证明了它并不是完美无缺的，相反，相对平淡的市场表现也从多个角度反映出其当下面临的问题。

首先，就项目本身来说，Be VEE 目前的信息曝光还是太少，除了艺术家本人的基本情况，其他可谓一概不知，这种不透明也致使其价格表现只能止步于现状。

其次，权益方面，除了 CC0 之外，该项目目前并没有明确其具体的赋能和用例，在以投资为王的加密市场，仅从艺术收藏角度出发，难免会略显单薄。

再者，目前的市场环境也是不得不考虑的因素。熊市之下，投资者的耐心明显大不如前，如果短时间内其价格无法呈现出较好的态势，投资者的抛售情绪就会极大的放大，从而给二级市场带来巨大的抛压。这对项目的危害不可谓不大。

最后，艺术家本人的号召力的确可以在短时间内为项目带来一定的热度，但与此同时人们的期待值也会被相应拉高。因此，如果后续项目表现不佳，其将存在一定的反噬风险。

艺术家的新道路

在运营团队未知的情况下，我们姑且可以将 Be Vee 视为由艺术家主导的偏艺术性 NFT。其成功与否很难说，也不好定义，但 Be Vee 的确为艺术家提供了一个相对完整的变现范例，即从社交媒体，到粉丝群体，再到 NFT 项目的完整 IP 链条。

其他新时代的互联网艺术家或许也可以从这条从创作者本身的作品衍生出相关 NFT 的道路获得启发，毕竟艺术家的利益问题一直都是相当的尖锐话题。NFT 本身所具有的特性无疑可以使得艺术的商业化更加纯粹。在没有了中间商盘剥的环境里，艺术家和其追随者也可以更为直观的接触，以满足彼此的经济和精神需求。而艺术或许也会因为加密开辟全新纪元。
