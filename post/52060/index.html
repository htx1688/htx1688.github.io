随着 DeFi、NFT、GameFi、SocialFi 等赛道的兴起，各类区块链应用不断增加。不难看出，加密行业正在经历从早期去中心化货币价值的发现，到去中心化应用价值实现的重要转变。然而，若要广泛地采用这项技术，目前亟需解决一个关键问题，即**“区块链三难困境”** 。

在行业发展的早期，以太坊是业内最早、具备图灵完备特性的应用链。其为了符合 Web3 最重要的价值观，选择了舍弃可扩展性。随着加密行业的持续发展，去中心化应用功能越来越复杂、智能合约执行的操作数量不断攀升。由扩展性不足带来的网络吞吐量和 Gas 费用问题凸显，成为网络持续发展的桎梏。区块链网络能否同时满足最大化“去中心化”、“安全性”和“可扩展性”？

以太坊二层扩展解决方案 Polygon 以技术创新，给出了它的解决方案。Polygon 是一个企业级区块链平台，为以太坊网络提供更快、更便宜的交易和更高的可扩展性。而凭借其可扩展的架构、快速且低成本的交易以及不断发展的生态系统，已成为越来越多的企业和开发者的首选解决方案。

据 Messari 发布的 2023 年 Polygon 第一季度研报显示，该网络平均每天约有 400, 000 个活跃地址，且本季度增加了 2200 万个新地址，日均交易约 300 万笔。下面将从技术和生态两方面展开分析，一探 Polygon 何以收获万千开发者与用户的青睐，而又能从中发现哪些发展的机遇。

特快列车：Polygon 技术创新

Polygon 最初名为 MATIC Network，主要利用侧链技术为以太坊提供链下扩容方案。在 2021 年，项目团队考虑到单一扩容方案的局限性，因此将其更名为 Polygon（多边形），致力于整合不同的扩容方案，为以太坊建设一个模块化、通用、灵活的扩容框架。下面将从 Polygon 产品出发，概述其在技术层面的创新。

侧链扩展方案 Polygon POS

Polygon PoS 是目前 Polygon 上最成熟的扩展方案。通过建立侧链，将主链上的资产跨链到 Polygon 链上进行处理。为了在保障安全的前提下，实现提升交易速度并节约成本，Polygon PoS 网络巧妙地设计了三层架构：

  • 以太坊层： 以太坊主网上的一组合约，主要用于保管资金以及保存 Checkpoint，保障安全性；

  • Heimdall 层： Heimdall 是权益验证层的证明，它将 Bor 生成的块聚合到 Merkle 树中，并定期将 Merkle 根发布到根链。Bor 侧链快照的定期发布称为检查点（checkpoints）；

  • Bor 层： Bor 是 Polygon 的侧链区块生产者层，负责将交易聚合到区块中的实体。目前，它由基本的 Geth 实现，对共识算法进行了自定义更改。它是一个 basic Geth 实现，对共识算法进行了自定义更改；区块在 Bor 节点产生，侧链与 EVM 兼容。在 Bor 上生成的块也由 Heimdall 节点定期验证，并且由 Bor 上一组块的 Merkle 树哈希组成的检查点定期提交给以太坊。

值得注意的是，受益于 Polygon 的混合结构，Polygon POS 能够支持 Plasma 桥接框架和去中心化权益证明（POS）验证者网络，以确保资产的安全性。其中 Plasma Bridge 从以太坊主链继承安全性；而 PoS Bridge 由同一组验证者和质押的 MATIC 来保护链本身的安全。

此外，今年 1 月 Polygon PoS 进行了德里硬分叉（ Delhi hard fork） ，旨在减少 Gas 峰值、解决链重组（Reorgs）以缩短最终确定时间，以进一步提高网络性能和用户体验。不难看出，在以太坊 Gas 费用高昂、网络性能不足的情况下，Polygon PoS 以扎实的技术方案，解决网络扩展性问题，成为 Web3 领域中不容忽视的黑马，被成为扩展方案的“特快列车”。目前，据官方数据显示，Polygon PoS 生态中已部署数万去中心化应用程序、超过 2.07 亿地址，以及已处理 23 亿笔交易，Polygon PoS 链已成为 DApp 的首选目的地。

ZK 扩展方案 Polygon zkEVM

如上所述，Polygon 致力于提供全面的以太坊扩展方案，除了当家产品 Polygon POS，Polygon 还积极布局零知识证明（ZK）产品。如 Miden 是基于 STARK 技术的与以太坊 EVM 兼容的虚拟机 Miden VM，其使用新颖的状态模型帮助开发者能够构建带有客户端验证的高级 DApp；Zero 则是基于 Plonky 2 技术，支持高效的递归证明生成，证明的速度比任何其他现有技术都要快。

尤为瞩目的是在今年 3 月上线主网测试网的 Polygon zkEVM 。其将 ZK 技术优势充分发挥，使 Polygon zkEVM 在继承以太坊 L1 安全性基础上，实现吞吐量大幅提升。其优势可总结如下：

  • 高性能 ：使用 Polygon Zero 技术，实现最快的 ZK 证明，并利用递归 STARK 以实现极致的可扩展性，开发者可以根据不同需求创建不同类型的 DApp；

  • 兼容 EVM ：绝大多数现有的智能合约、开发人员工具和钱包都可以无缝运行，允许开发人员专注于改进代码而不是重写代码；

  • 低成本 ：利用 ZK 证明的力量来降低交易成本，最终降低用户总使用成本以提供更好的用户体验；

  • 安全 ：继承主网安全性，并且由 ZK 证明确保交易有效性并保护用户资金；

此次测试网协议涉及到全新且复杂的技术，尽管此前该协议经过了安全审计，但 Polygon 仍实施了多项安防措施以保障 Polygon zkEVM 用户安全。如在第一阶段，将有一个可以迅速升级协议的安全理事会；第二阶段将诞生增强的安全措施。目前正在进行 Polygon zkEVM 漏洞赏金活动，感兴趣参与其中的开发者可至下方官网了解详情。

https://polygon.technology/blog/bug-bounty-for-polygon-zkevm

缓慢的区块确认、高昂的交易费用、低可扩展性和糟糕的用户体验是区块链应用程序大规模采用的一些主要障碍，为了解决上述问题，Polygon 以多元的技术方案持续探索不同的路径。此外，其还开发了 ZK 身份证明 Polygon ID，使用户能够创建一个数字身份来确认他们的访问权限，而不会泄露他们的个人信息。总之，Polygon 以先进的技术推进，正吸引大批开发者与用户进入生态，共同建设。

开发沃土：Polygon 优越生态

Polygon 为支持基础设施开发和帮助以太坊扩展的平台设计。其在技术开发上重视为开发者营造高度友好的开发条件，而在生态上，又能为开发者项目实践提供丰富多元的资源支持。

高度开发者友好

Polygon Edge 是区块链 SDK 开发框架。下面将重点介绍官方推出的加强版的 Edge——Polygon Supernets，专注于解决 Edge 所面临的安全性和复杂性等难题。

Polygon Supernets 是一个模块化的区块链开发框架，能够实现以快速无缝的方式创建高性能、可定制的应用链，旨在支持各种扩展和基础设施解决方案。这与 Substrate 或 Cosmos SDK 类似。开发者可以根据实际项目需求，高效开发特定应用链。具有以下特点：

  • 可定制 ：开发者可以根据需求进行定制，创建快速且低成本的高性能区块链网络；

  • 模块化 ：模块化框架简化了区块链开发成本，为开发者提供了可扩展、安全和可互操作的定制区块链网络所需的工具；

  • 链上治理机制 ：实现社区驱动的区块链决策和管理。此机制支持混合治理模型；

  • 互操作性 ：原生桥接解决方案可以将资产从各种 EVM 兼容区块链无缝转移回 Polygon 生态系统，促进区块链和不同层之间的互操作性；

  • Layer 3 高性能 ：使项目能够创建高度可扩展的第 3 层网络，同时继承 Polygon PoS 和以太坊主网的安全性和完整性。

对此开发感兴趣开发者可参见下方官方文档：

https://wiki.polygon.technology/docs/supernets/get-started/what-are-supernets/

得益于高度友好的开发环境，Electric Capital 发布 2022 开发者报告将 Polygon 评为顶级开发者生态系统。报告显示，自 2018 年以来，Polygon 的开发者数量从 200 名增长到 1000 多名。开发者总数同比增长 40% ，到 2022 年平均每月活跃开发者超过 500 名。

繁荣活跃的链上生态

Polygon 具有低费用和高交易速度等优势，以太坊生态内的知名项目如 Aave、Curve、Sushiswap、Uniswap 等已经迁移至该网络。此外，Polygon 更成为 GameFi、NFT 等 Web3 项目的部署的首选，据 Messari 研报显示：

  • 在一众 EVM 链中，Polygon 拥有最大份额的游戏用户。其中最受欢迎的游戏是 Planet IX、Sunflower Land、Hunters On-Chain 和 Benji Bananas。

  • Polygon 的 NFT 是 2022 年熊市期间的一大亮点。其与 Meta、星巴克、迪士尼和 Reddit 等 Web2 公司以及 Rarible 和 Magic Eden 等 Web3 市场达成了系列合作。

目前 Polygon 已经拥有非常庞大的生态系统，已部署内项目多达近两万个。而 Polygon 日活跃用户最高达 355, 000 左右，享有繁荣活跃的链上用户群体。大量项目的部署、活跃的用户，结合优越的开发条件，Polygon 正成为 Web3 开发者耕耘的乐园。

Polygon 以技术实力与生态繁荣，有望在未来的公链竞争中继续保持领先位置。Polygon 生态对开发人员的参与亦十分重视，“未来，Polygon 希望为区块链网络提供一个框架，允许创建一个互连的区块链互联网。”  开发者何不把握 Polygon 生态沃土的开发机会？
