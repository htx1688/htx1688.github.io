一览 ETHGlobal 墨西哥 12 个获胜项目。

编译：Karen

8 月 19 日至 21 日，ETHGlobal 在墨西哥举办 ETHMexico 黑客松，在公布的 12 个入围项目中，有 9 个项目基于 Polygon 或部分使用 Polygon 构建；有 5 个项目与 NFT 相关，包括游戏和元宇宙 NFT 市场 Kiwik、动态 NFT 创建和管理平台 Spiritus，还有利用 NFT 解决现实世界中如支付或者寻找失踪人口问题的项目。此外，基于 Lens 的 Web3 社交平台 Staxx、面向实体封闭式社区的支付和可追溯性 DApp BlockCommunity 也入围本次黑客松决胜名单。

Datamorpho：使文件或 NFT 随时间进行数据变形

Datamorpho 使用密码学和 IPFS 的动态文件标准和协议，可让任何文件或者 NFT 随时间进行数据「变形」，即这些文件保持相同的数据内容，但在输出后却是不同的动态。

演示链接：https://ethglobal.com/showcase/datamorpho-omvp5

Keyppi：零知识私钥恢复平台

Keyppi 将通过实施零知识证明、去中心化存储和使用身份证明验证身份来保护用户密钥。

Keyppi 使用一个简单的基于浏览器的前端，允许用户注册一组新的私钥、恢复以前注册的一组私钥，也可以赠送钱包。如果用户尝试注册一组新密钥，在选择钱包类型类型后，需要填写钱包地址、私钥和电子邮件或 SMS 来连接钱包。

若要恢复私钥，用户需要用他们的 World ID 登录，这将触发通过电子邮件或 SMS 验证其身份证明的操作，身份证明验证成功后，平台会将接收到的 nullifier 发送到 zkApp 并启动与此 nullifier 关联的信息的解密，然后，用户将能够查看和恢复私钥。

Keyppi 也将允许用户将加密私钥共享给家人或朋友，提示他们将其 World ID nullifier 替换为新所有者之一，以保留恢复功能。

演示链接：https://ethglobal.com/showcase/keyppi-k7ur9

Kiwik：游戏和元宇宙 NFT 市场

Kiwik 允许创作者和游戏开发者买卖代币化 3D 资产，以创建更高质量和数量的游戏和元宇宙。

对于创作者而言，Kiwik 允许在一个地方以 Polygon NFT 的形式以低成本出售他们的服务、3D 和元宇宙资产，并且可以创建 Lens 社交资料来展示他们的作品，也会用 KIWIK 代币给予奖励；

对于游戏开发者和元宇宙建设者：通过优化在一个中心对所有 3D 资产的搜索、将现有或未使用资产货币化的能力以及支付选项（借助 SuperFluid 实现先买后付）来加速游戏发展流程。

演示链接：https://ethglobal.com/showcase/kiwik-9x206

Sell Out：Web3 支付组件库

Sell Out 旨在为开发人员提供高效的支付体验，允许轻松集成到他们的 DApp 中。受 RainbowKit 的启发，Sell Out 使用单个 React 组件来处理整个支付入账环节。

演示链接：https://ethglobal.com/showcase/sell-out-5y2xi

Emergence：链上视频平台

Emergence 是一个链上视频平台，主要是为了激励 DAO 的健康发展和拉近 DAO 管理者和社区之间的距离，可在通话期间分发代币，为 DAO 文档生成脚本。

Emergence 通过使用 Semaphore 协议为 DAO 提供匿名性，使用 100ms SDK 用于视频会议，Wallet Connect 用于前端钱包集成，Superfluid 为 DAO 成员提供收入。

演示链接：https://ethglobal.com/showcase/emergence-o3tns

Staxx：基于 Lens 的 Web3 社交平台

Staxx 基于 Lens 构建，旨在为社交带来更个性化的体验，让用户能够以独特的方式管理自己的社交信息。在 Staxx 中，每个 Staxx 都是一个帖子，而 Staxx 中的每个 Pin 都是一个评论。

Staxx 利用 Lens GraphQL API 向 Polygon 区块链发布和获取数据。Staxx 还可能会在平台中添加费用收集模块和投票模块。

BlockCommunity：面向实体封闭式社区的支付和可追溯性 DApp

BlockCommunity 基于以太坊构建，只有社区负责人可以注册新房地产（properties），平台用户（业主）能够指定你的房地产与某个钱包地址相关，查找每月对应的费用，并且可以每月进行付款并收到 NFT 作为付款证明。

在未来，治理代币将自动交付给提前付款的用户，而延迟付款的用户将被收取一定比例的利息，并且该利息必须用治理代币进行支付。

演示链接：https://ethglobal.com/showcase/blockcommunity-17ujy

ETHPoint：Web3 移动销售点应用

ETHPoint 为以太坊和 Polygon 生态系统创建了一个移动销售点应用，将 TradFi 与 Web3 结合在一起，以提高拉丁美洲地区的采用和金融包容性。

演示链接：https://ethglobal.com/showcase/ethpoint-c87s1

Spiritus：动态 NFT 创建和管理平台

Spiritus 是一个 Polygon 上的动态 NFT 创建和管理平台，专注于促进元数据驱动的开发，允许只需快速兑换特征、更新相应的元数据并将新资产上传到去中心化网络。Spiritus 利用 Polygon Mumbai 网络上的 Tableland，Tableland 提供了极其丰富的元数据体验并促进了持续的元数据开发。

演示链接：https://ethglobal.com/showcase/spiritus-34owk

Ethereum Uncensored：保护以太坊免受审查的公共产品工具

Ethereum Uncensored 是以太坊上的公共产品工具，会公开通知区块链拒绝的有效区块（因为网络延迟允许首先接受另一个区块），跟踪交易类型、金额或验证器之间是否存在一种模式以了解是否存在涉及恶意行为。

演示链接：https://ethglobal.com/showcase/ethereum-uncensored-h2i19

Reunited Reunited：链上失踪人口寻找

Reunited Reunited 帮助在链上寻找失踪人口，允许在智能合约上发布失踪人口信息，通过电子邮件和密码简化 Polygon 区块链上失踪人口的互动和发布。其中，数据和照片被存储到 NFT 中，并托管在 IPFS 上和通过 AI 模型进行比较。

演示链接：https://ethglobal.com/showcase/reunited-mgc5f

Woosh：加密现金应用

Woosh 旨在通过提供对用户友好且直观的现金应用将 Web2 用户带入加密世界。Woosh 基于 Polygon 网络，建立在 Magic Link (magic.link) 钱包之上，这使用户能够使用他们的 Google 帐户或电子邮件登录和注册，降低了使用门槛。

借助 Woosh，用户可以关注朋友、请求 / 发送加密货币、自定义个人资料，并且每笔交易都是与经过验证的人完成。另外，虽然用户可以创建多个电子邮件帐户，但只能通过 Worldcoin 验证一次。

演示链接：https://ethglobal.com/showcase/woosh-pu1fn
