作者：曼昆区块链法律

随着区块链技术的快速发展，从事区块链自媒体与KOL行业的朋友也越来越多，且影响力日益增大。在这个充满机遇与挑战的当下，区块链自媒体和KOL们肩负着引领行业健康发展的责任。有鉴于此，本文将探讨区块链自媒体与KOL如何合理使用区块链技术、数据和信息，以及在利用的过程中如何避免侵犯他人的知识产权、隐私权和名誉权。

0 1  区块链KOL

曼昆自专注于Web3.0以来，通过我们周三的线上直播、周五的下午茶以及其他的方式，与区块链行业内的非常多自媒体与KOL认识，我们一直认为区块链自媒体与KOL是区块链行业中的重要力量。

首先KOL对于区块链这个行业具有一定的了解，主要是因为区块链行业涉及到数学、金融学、货币学、密码学等等专业领域的知识，对于刚入圈或者想要入圈的普通用户、玩家来说，实在是相当生涩，这就需要KOL这样的角色带领更多的用户（玩家）去认知这个行业，这也是为什么KOL在区块链行业有着一席之地的重要原因。

总之，在区块链行业中，KOL通常在某些特定领域内具有专长和影响力，例如投资、技术、加密货币等领域。他们通过分享自己的见解、分析和评论，帮助公众了解和掌握 区块链技术 和应用，同时也为品牌或产品推广带来更多的曝光和影响力 。区块链KOL的影响力最直观的体现可能是粉丝数量，所以区块链项目方往往会邀请业内的KOL为其站台、喊单、授课、发文、直播来进行助推。

由 KOL 衍生而来的几个相关群体 还有如下一些：

如KOS（Key Opinion Spreader 关键意见传播者），KOS更多地隐藏在某些私域之中，比如朋友圈、社群、贴吧、论坛等。某一垂直领域达人，粉丝基数可能不如KOL，但却有能力引领某个圈层的风潮。比如汽车发烧友，热衷于使用各种新款手机评测，帮助品牌精准触达目标用户。

如KOC:（Key Opinion Consumer 关键意见消费者）。KOC就存在于我们身边，是一个热衷于共享各类好物的群体，但他们不局限在某一产品或范畴。相较于商业化较高的KOL，KOC展现出的营销性较弱，只是简单影响同类集体的消费决策计划。

KOL\KOS\KOC都是粉丝经济发展的产物，原本可能集中出现在传统行业和领域，粉丝经济的蔓延把这些群体推到了web3.0行业中，就目前的情况来看，其实同样适用。他们的最大的区别可能是面向的用户数量或者范畴不一致，但是都能够达到为品牌或者产品进行推广带来更多曝光和影响力的效果，今天就来统计一下区块链个人自媒体在日常商业模式下可能出现的一些民事法律风险。

0 2  侵犯他人肖像权、知识产权的风险

举个例子：在小红书平台，周律师关注到一个专门介绍NFT的KOL，他发布的短视频中曾出现介绍姚明开始涉猎红酒NFT，以及周杰伦和他的数字藏品，介绍前面的事件时同步贴出了姚明、周杰伦的照片，这种情况构成侵权吗？

经过检索公开的知识产权侵权案例，如果要说明确是区块链行业KOL侵权的，的确不多，不过我们可以从传统行业的裁判中得到一定的启发。

陈乔恩曾因为在微信公众号的服务软文中使用了她的一张图片，江阴某商场被陈乔恩告上了法庭，江阴法院判决商场赔偿经济损失维权费用。

法院认为：公民都享有肖像权，是否侵犯肖像权应当满足三个构成要件。

第一就是要有使用肖像的行为。

第二，是否经过本人的同意，第三就是是否以营利为目的。这个案件中，商场是构成侵犯陈乔恩的肖像权的，首先商场的微信公众号发布的文章中使用了陈乔恩的照片，那么商场应当是照片的使用人；第二商场使用陈乔恩的照片并没有经过陈乔恩本人的同意；

第三，商场发布该文章的目的是为宣传其商场内所销售的百丽女鞋，想以此借助陈乔恩的知名度来提高百丽女鞋的关注度，招揽顾客，显然系以营利为目的。所以最终法院认定商场侵犯了陈乔恩的肖像权。

区块链KOL在使用他人的背景音乐、美术作品、专有字体、短视频模板等，如果没有获得授权或未经合法使用，可能会被视为侵犯他人的知识产权，从而面临侵权责任的风险。

避雷指南

微信公众号、视频号、小红书、抖音等平台已经成为发布信息的重要平台，KOL作为自媒体行业的一员，应当对发布信息的真实性负责，当发布的信息违反真实性原则，侵害他人私法权益时，就有可能构成侵权。

因此，在发布视频、图片和文章的时候，周律师提醒：如果是以自己名义发布的信息，要确保文章不要有抄袭的行为，图片的来源是合法的，如果是转载他人的文章，也应当要经过他人的同意，并注明“转载”的字样，不能把别人的文章直接以自己的名义进行发布。

建议区块链KOL要尊重他人的知识产权和肖像权，避免未经授权使用他人的原创文章、图片、视频等。如果需要使用他人的作品或者照片，应当先获得合法授权或注明作品来源 。

0 3  侵犯他人名誉权的风险

举个例子：某区块链自媒体（或KOL）发表了一篇涉及某公司及其产品的评论文章，其中使用了过于激烈和不实的言辞，对该公司的声誉和形象造成了负面影响。该文章在自媒体上广泛传播，引起了公众的关注和讨论。该区块链自媒体（或KOL）侵犯了该公司的名誉权，可能会承担相应的民事法律责任

法律依据：《中华人民共和国民法典》第一千一百六十五条第一款规定，行为人因过错侵害他人民事权益造成损害的，应当承担侵权责任 。

避雷指南

建议区块链自媒体作者（或KOL）在发表文章时要客观、真实、准确，不得歪曲事实、捏造谎言，不得侵犯他人的名誉权 。同时，建议区块链自媒体平台建立事实审核机制，对发表的文章进行严格的事实核查 ，防止出现不实言论和侵犯他人名誉权的行为。

0 4  曼昆律师为你总结

（1）侵犯他人知识产权的风险防范：建议区块链自媒体作者要尊重他人的知识产权，避免未经授权使用他人的原创文章、图片、视频等。如果需要使用他人的知识产权作品，应当先获得合法授权或注明作品来源。

（2）侵犯他人隐私权（违规采集用户个人信息）的风险防范：建议区块链自媒体平台在收集用户个人信息前，充分告知用户收集、使用、保护个人信息的范围和方式，并获得用户的明确同意；除此，防止个人信息被泄露或滥用

（3）侵犯他人名誉权的风险防范：建议区块链自媒体和KOL在发表文章时要客观、真实、准确，切忌歪曲事实。建议区块链自媒体平台建立事实审核机制，对发表的文章进行严格的事实核查。

区块链自媒体与KOL的法律风险其实隐藏在日常工作内容的方方面面，有鉴于此，曼昆律师已经开始有针对性地对行业内的有关法律风险进行系列分享，有关刑事、民事及行政方面，借此希望区块链自媒体和KOL朋友们能够有所获益，为各位朋友后续赢得读者、玩家的信任提供支持，也希望以此能够促进Web3.0及相关自媒体和KOL行业的健康发展。
