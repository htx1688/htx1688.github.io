在此次 Devcon 活动期间，ETH Global同样举办了自己的黑客松活动 ETHBogota，并于今天公布了黑客松最终项目名单。本文对该名单中的项目进行了梳理和介绍：

frens

Frens 是一个与朋友一起质押 ETH 的精简 dApp，通过邀请朋友汇集 32 个 ETH 即可启动分布式验证器，这降低了用户单独质押的风险。并且 Frens 建立在 SSV 之上，确保了一定容错性和性能，还提高了与朋友一起在矿池中质押的社交性和回报性。

一般来看，质押 ETH 有一定的门槛，用户需要大量的技术知识和一些基础设施，并且也十分昂贵（32 个 ETH 对于单个质押者来说成本不低）。当用户在错误配置验证节点时，就可能会失去自己的 ETH。而由于这些挑战，大多数人在选择使用中心化的非托管质押服务如 Coinbase 或 Kraken 等进行质押，则又会带来中心化的风险。

工作原理：运营商通过 Frens 网站上的一键式按钮创建「质押池」智能合约，该网站为用户提供了一个指向简单的 UI 链接，因此朋友可以为完整的验证者存入必要的 ETH，而只有拥有此链接的朋友才能将 ETH 存入质押池中。并且 Frens 提供的 UI 十分简捷易于操作，收集到 32 个 ETH 后，操作员就可在 Frens 网站上启动质押存款机制。

Blobscan Explorer

EIP-4844 向以太坊引入了一种新的交易类型，它接受在信标节点中短时间保存的数据区块，这有利于存储汇总所需的数据，例如 Optimism、Aztec 和 Scroll。但用于处理这些新型事务的开发工具是新的，却一直还没有 Blob Explorer。

使用 Blobscan Explorer，可以导航和可视化这些 blob 数据，为扩展以太坊提供了必要的基础设施。其中搜索框还可以找到区块号和哈希值、交易哈希、地址、blob 版本化哈希和 blob KZG 等。

Bet Streams

Bet Streams 是一个按次付费的直播平台，使观众能够对实时流式传输的结果进行投注。

Bet Streams 允许组织准备、发送通知并提供各种活动的按次付费直播，包括关键公告、体育比赛、旅行挑战甚至 Web3 黑客马拉松。

用户可以通过 Web2 用户的流行钱包和集成轻松登录平台，并且还可以在直播之前或期间投注将在直播中显示的结果，包括游戏的最终得分、滑板技巧的结果或黑客松的获胜者。一旦结果公布，获胜者将按比例从彩池中获得奖品，而失败者则必须再次下注试试运气。

ETHernal Reviews

ETHernal Reviews 是一个 Dapps 评级平台，用户使用 ETH 地址登录，只有与某个 Dapp 交互过的用户才能对其进行评分，保证了用户的真实性。

用户现在可以查看一个项目的评论，数据库往往是集中的，ETHernal Reviews 的评论将作为 NFT 存储在 IPFS 上。每条评论都被写入用户地址，没有人可以隐藏它们。

具体做法上，ETHernal Reviews 使用 Web3Modal 连接到评论网站，在前端使用 Fast API 和 Vue，后端用 Python 构建。作为示例解决方案，还将选择 ENS 来演示验证有效地址。

PULP

PULP 是一个无需担保的借贷协议。

目前，申请贷款是一个缓慢、耗时的过程，并且借款人往往还需要提供自己过多的个人信息。PULP 通过 Polygon ID 技术集成的数字自我主权身份解决方案，允许用户验证他们的信誉并对他们的声誉进行评分。

Superseed

Superseed 是一个 DeFi DApp，它简化了用户不仅可以自动执行加密货币购买的美元成本平均 (DCA) 策略的过程，而且可以最大限度地提高他们的加密收益的利息，同时鼓励他们为公共利益分享利润。

此外，Superseed 还针对持有和交换加密货币但不精通 DeFi 更高级策略或没有时间监管自己 DCA 的用户，Superseed 将帮助他们获取 DeFi 中的一些价值，并委托给 Web 3 生态系统及其他生态系统的进一步发展提供资金。

Coedia Network

Coedia Network 是一个专为开发者打造的去中心化社交网络，旨在通过搭建一套开源的 help-and-earn 框架帮助解决高优先级代码问题，建立一个更广泛、更强大和团结的开发社区。

Coedia 上提供问题请求与可定制的赏金和制裁、异步奖励问题的解决和全球声誉分数等功能，每个提案都由社区本身给出和更新，以创建一个参与性和合作的 web3 建设者社区。团队构建该应用的是为了使 Web3 开发者们从企业倦怠中恢复过来，帮助因早期高级代码开发被搁置的项目等。团队的最终目标是减少开发人员采用新技能所花费的精力和时间，同时帮助搭建和发展更广泛、更强大和团结的 web3 社区。

Zipline

Zipline 是一个借助可证明执行的无信任以太坊 EVM 跨链桥，该项目使用 Optimism Cannon 的欺诈证明协议，在 EVM 公链上可证明地执行以太坊轻客户端协议，可以使目标 EVM 链与以太坊最终块保持同步。

Zipline 的链下中继器用于提交最终区块以及同步委员会更新，更新包含来自当前同步委员会的签名以及派生下一个委员会的证明。它们不是在链上执行签名验证，而是在链下执行并在链上提交执行证明。如果中继者进行欺诈，任何诚实的观察者都可以通过在挑战期间提交挑战来证明这一点。一旦此期限过去，则假定更新有效。此过程在每个同步周期（27 小时）重复一次，以使目标链与以太坊状态保持同步。

Anonymous Vickrey Auctions On Chain

Anonymous Vickrey Auctions On Chain 是一个私人拍卖协议。通过利用基于 CREATE2 的智能合约创建，Vickrey auctions 可以让用户将 ETH 转移到未来的合约地址来隐藏出价，让链上提交投标看起来像是单独、简单的 ETH 转账。在揭示阶段，投标人揭示他们的投标，使用 ETH 将合同部署到预先注资的地址。

与典型的基于提交和揭示方案的 Vickrey 拍卖不同，如果不依赖额外的链下数据，就不可能知道哪个地址在给定拍卖中真正出价。为确保用户在披露阶段无法调整出价，对区块链状态进行快照，用户提交 Merkle Patricia Trie 账户余额证明，以及他们的披露表明他们的出价在开始之前存在揭示阶段。

diZKreet

diZKreet 是使用 Aztec 的隐私 zkRollup 技术的 NFT 隐私层基建。

Aztec 提供了一套工具来确保个人在以太坊区块链上的隐私，diZKreet 团队借此创建了一个 ZK Rollup 桥，以便以匿名和私密的方式铸造 NFT 集合，其中以太坊主网用于记录 Aztec 上购买了 NFT 的用户，但没有人知道是谁做的，直到它在主网上被赎回。虽然 NFT 保留在 Rollup Bridge 合约中，但 NFT 的所有者将拥有一个虚拟的 zkNFT。这个 zkNFT 可以私下交换和交易，也可以从桥接合约中赎回 ETH 主网换取 NFT。我们的 zkNFT 屏蔽桥与以太坊上的任何 ERC-721 NFT 完全兼容。任何 NFT 现在都可以通过 diZKreet 进行私人购买。

ClubSpace

ClubSpace 是一个为音乐创作者打造的现场聆听派对，创作者可以在 Lens 上与他们的粉丝和朋友分享自己策展的音乐 NFT。

该项目允许创作者通过导入他们的 Spinamp 播放列表、设置他们推广的 Decent NFT、设置 Lens 帖子以及最后设置派对参与者收到的免费 NFT 来创建一个俱乐部空间派对。在直播俱乐部空间页面，已连接的 Lens 帐户可以查看彼此的头像、收听播放列表并通过 p2p pub/sub 网络发送反应

txn. xyz

txn.xyz 是针对以太坊交易的专用网页应用，此交易页面允许通过任何钱包进行连接，并允许向用户请求任何交易。应用的所有功能都可以通过一个 URL 编码系统实现，没有服务器，也没有状态。
