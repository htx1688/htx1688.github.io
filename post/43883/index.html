原文作者：William M. Peaster，来源：Bankless

目前 NFT 游戏领域正在发生大量的开发活动。 

但是开发工作呢？进行得很困难。这是因为设计一款常规游戏已经很困难了。当你把 NFT 加入其中时，就会出现一些挑战，这些挑战可能会对游戏造成伤害而非帮助。

那么，我们应该对 NFT 游戏的未来抱有什么期待？本文对此提出了一些想法!

NFT 游戏设计很困难

NFT 游戏可以有各种不同的形式和规模。

基于简易的设计，它们可以是简单的链上策略游戏。而另一方面，它们也可以是高级和扩展的大型多人在线（MMO）游戏项目，使用 NFT 资源来跟踪玩家在中心化服务器以外的所有权。

到目前为止，我们已经看到了很多关于简单方面的实验。例如，andy8052（Tessera 联合创始人）的《Battle Royale Onchain》。非常简单但很有趣！

然而，目前我们还没有看到很多更广泛的 NFT 友好型 MMO 项目。公平地说，现在有许多这样的项目正在开发中，其中一些让我感到兴奋。

因为完整的 NFT 游戏领域仍在建立自己的基础，这不足为奇：因为游戏设计非常困难！

NFT 游戏的主要挑战 

从战斗动画到关卡设计再到游戏中的经济水龙头（Faucet）和水槽（Sink，指将一种资产的供应从经济体中抽离），常规的游戏设计有大量的活动部件，它们本身就很复杂。因此，当你的项目漫不经心地将 NFTs 添加到方程式中，并将其游戏内资产过度代币化时，这只会增加游戏复杂性和它们所面临的挑战。 

事实上，如果操作不当，NFT 可能会成为游戏杀手。因此，0xKepler 在 2022 年 7 月的一篇题为“ Web3 游戏的前进之路”的文章中指出，“将大部分游戏内部资产代币化”的游戏最终如何面临两大挑战：经济外流 和过度投机 。 

关于经济外流，如果所有东西都被代币化，那么赚钱就成了玩家的主要目标，这可能导致所有人同时套现而几乎没有需求方的兴趣，也就是经济崩溃。

0xKepler 写道：“由于没有人在货币和资产的需求方，看到货币和资产以外的价值，因此价格下跌，这意味着玩家的收入减少。游戏变得无趣——现有玩家离开，新玩家加入的可能性变得更小。P2E 游戏通常过于依赖玩家增长，而不是经常性的代币移除，这导致经济迅速下滑。”

在过度猜测的点上，0xKepler 使用《暗黑破坏神 3》（Diablo 3）中拍卖行系统的情节来展示如何让所有东西都易于交易，以致于驱逐真实玩家并扼杀游戏内经济：“易于交易的资产带走了一些玩家寻求的成就感。因此，具有游戏内市场的游戏接触到的此类玩家（他们通常是经济的增值者）较少，并且更有可能吸引被金钱奖励所吸引的玩家（通常是价值提取者）。随着时间的推移，投机者推高了价格，使非投机者无法负担玩游戏所需的资产。最后，只剩下投机者了。”

如何制作更智能的 NFT 游戏

Sal.xyz 在推特上表示，“我们为什么要玩游戏，这可以归结为 3 个主要因素：1. 测试技能；2. 独立行动；3. 与他人交流……”

当谈到“是什么造就了一款优秀的 NFT 游戏”时，我们必须回到游戏的基本原理上。为此，我们首先必须了解人们玩游戏的原因，我认为 Sal.xyz 在上面的推文中完美地描述了这一点：测试我们的技能，拥有独特的独立体验，并与他人建立联系。  

Sal.xyz 还表示，事实证明，每个人都有玩游戏的理由，但他们往往分为 4 类（由 Bartle 在 1996 年提出的）——成就者～10%，探索者～10%，社交者～80%，杀戮者～1%。

考虑到这些支柱，有一些方法可以更智能地构建 NFT 游戏，以优化这些基本要素，并减轻经济外流和过度投机的挑战。对此，0xKepler 曾提出了以下原则：

  • 稀疏代币化（Tokenize sparsely） ——限制可交易资产的数量（而不是让所有东西都可交易），并进行税收交易以限制投机，以允许有机参与者建立可持续的经济。

交易税是 Eve Online《星战前夜》经济（现存最古老的虚拟经济）的巨大吸收器/稳定器 ——通过 0xKepler

  • 隔离系统 ——考虑使一些资产可交易，但避免使所有资源都可以互换。

通过 0xKepler

  • NFT > 可替代代币（FTs） —通过专注于 NFT 奖励而不是 FT 奖励，游戏可以强调乐趣和游戏内的实用性而非金融化。

  • 土地价值税 ——避免 NFT 土地变得太有价值和过度投机，通过使用元宇宙土地税（我之前写过！）以优化促进可持续经济的建设者。

为什么 NFT 游戏具有吸引力

如果实施得当，NFTs 可以带来新型的游戏体验。 

例如，考虑到开放性所带来的可定制性和社区赋权。假设你有一个团队将一个链上游戏的核心作为一个超结构，可以永久免费使用和构建。这可能会释放出 modding 社区前所未有的力量。

除了字面上的开放性之外，与大型游戏发行商相比，在 Web3 领域处于领先地位的 Web3 原生游戏项目也有一个开放的机会之窗。我们应该通过它们来为优秀的 NFT 游戏定下基调。 

Sal.xyz 认为，主要的游戏发行商对 Web3 持谨慎态度。这类似于 08 年传统游戏开发商在拥抱移动互联网时的迟钝。聪明的 Web3 游戏开发者将利用这种缺乏行业巨头竞争的优势。明智地利用这个机会窗口。

最后，游戏 NFTs 最酷的一个方面是，它可以作为一种让玩家的成就在不同的游戏之间传播的方式。在目前的主流游戏中，用户的游戏数据被孤立于每一款游戏中，这便排除了许多可能性。

Civitas

作为一个例子，即将推出的 NFT 游戏 Civitas 在平衡上文描述的挑战和机遇方面做得很有趣。

据了解，Civitas 将于明年推出，是一款基于以太坊 + L2 构建的战略 MMO 游戏，任何曾经玩过 Sid Meier 的 Civilization 系列的人都会隐约感到熟悉。Civitas 的最大区别在于，它的每个城市都是由公民玩家共同拥有和组织的 subDAO。

从 Free to Play Nomad 模式到资源交易的 CITI 代币交易税等机制，Civitas 或许有多种元素可以稳定地防止过度投机，并仍然可以营造活跃的游戏内经济。不过现在只有时间会告诉我们答案！

NFT 游戏的下一步是什么？

一个容易实现的目标是，围绕第二层（L2）扩展解决方案的游戏开发工作将继续蓬勃发展。凭借快速和超低价的交易，这些 L2s 代表着更好的 Web3 游戏的下一个链上“前沿”。
