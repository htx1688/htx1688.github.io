撰文：SnapFingers DAO

过去一年传统公司尤其是互联网企业纷纷开始 Web3 之路的尝试，对于巨头们来说，兼具流量与品牌效应的 NFT 成为了首选。以下梳理几家中外传统企业在 NFT 上的探索之路。

Reddit 社区：CryptoSnoos NFT 系列

Reddit 是一个综合性的社交媒体平台，用户可以在上面发帖参与任何感兴趣的内容，相当于美版天涯与贴吧，目前拥有 15 亿注册用户，月活 4.3 亿，日活 5200 万，是全美排名前五的社交网站。

早在 2013 年，Reddit 与加密世界就已结缘。彼时其联合创始人 Alexis Ohanian 参与了 Coinbase 的天使轮投资，网站的付费服务也率先支持比特币付款，2014 年时还曾计划推出自己的加密货币 Reddit Notes。除了 Token，Reddit 在 NFT 上也有动作。Reddit NFT 的理念同样来自 Alexis Ohanian，他在一次大学讲堂中表达了自己对自家 NFT 的理念——发现与探索人类。首批 NFT CryptoSnoos 于 2021 年 6 月发行，只有 4 个。今年 8 月，Reddit 与 Polygon 合作，推出了 CryptoSnoos NFT 的升级版，受到空前关注。

Reddit 的早期 NFT 主要由艺术家设计，为了利用 NFT 更好的实现去中心化与社区自治，后期则慢慢变为由用户根据自己的喜好进行个性化定制。免费 NFT 获取非常简单，用户满足活跃度等条件即可进入商店自由搭配，生成自己喜欢的风格。此举也成功为加密世界带来了大量新用户。根据链上 DUNE @dunetester 统计，截至今天，Reddit 的免费 NFT 共有约 375 万个，由大约 373 万个独立地址持有，而付费系列共有 44595 个，由 17511 个独立地址持有。

Reddit NFT 火爆原因可以归为以下三点：

  1. 空投范围广，地板便宜，获取简单；

  2. 降低新人进入加密世界门槛。Reddit 设计了非常好的新人使用流程，mint/ 购买 NFT 犹如 Web2 操作一样简单。例如在钱包创建中，Reddit 使用了基于 Polygon 的托管钱包 Vault，用户不必切换第三方注册，不必记忆助记词，甚至 GAS FEE 也由 Reddit 支付。而 Vault 使用也很简单，只有移动端，与 Reddit 互通。因此整套注册使用流程跟 Web2 无异。

  3. Reddit 社区活跃，创始人主动带领平台向去中心化社群发展。用户地位来自于平台互动与成员认可，引入 NFT 则加深了对用户个性化、稀缺性的追求。

百度：DuDu NFT 系列

百度是中国最大的提供搜索引擎服务的互联网公司。早先与阿里巴巴、腾讯并称 BAT——中国互联网三巨头。但在市值破千亿后，百度服务趋于闭环变现、均质化布局，伴随一系列发展问题被同行大幅超越。目前百度 AI 平台与自动驾驶逐渐起步，在内容上的积累也显示出护城河优势，综合实力来看，仍然是国内排名前七的互联网公司。

数藏

不同于国外公链环境，国内公司多选择数字藏品，或是开发在宣传中强调面向海外用户的公链 NFT 来规避可能的政策风险。但是百度也没有在数藏领域有大的动作，事实上无论是数藏还是公链 NFT，百度基本只负责销售，鲜少参与作品创作。数藏方面，百度曾经推出过三个平台，各个平台所发售的藏品各不相同，但整体与其他平台藏品无异，目前剩两个平台还在运营。

发行

NFT 方面，今年 8 月百度国际化事业部旗下的海外输入法 Simeji 与日本游戏开发商 ITI 合作共同发售 CryptoSimeji NFT。

Simeji 作为日本下载量最高的手机输入法，在日本拥有超高人气，同时也是世界最大的颜文字社区。CryptoSimeji NFT 画风以 Simeji 品牌 Logo 为主，整体为小蘑菇像素风，有趣呆萌，加上之前有传言与 ITI 的合作除了 NFT，还会有 NFT 衍生游戏，吸引了众多粉丝参与，Opensea 日榜冲到第四。小蘑菇总发行量 10000 个，团队自留 300。目前地板 0.075 ETH，最高为 0.4ETH，总成交量 2671 ETH。

小蘑菇路线图并未公布，官网给出 type to earn 玩法，但是目前只有第一阶段钻石手奖励，根据持有人的 NFT 持有数量及时间计算积分，瓜分总量 20ETH 的奖池。小蘑菇整体属于小而美的项目，本身也是 Freemint，所以目前这个成绩表现良好。但是项目开图后，基本再无后续动作，所以地板止步 0.4ETH，比较可惜。

DuDuLab Project

不同于 CryptoSimeji NFT ，百度在其第二个 NFT 项目 DuDuLab Project 中亲力亲为，项目设计、开发、运营计划等全由百度团队独立完成。NFT 主体形象也用上了百度第一大 IP——小度熊。

百度对 DuDu 熊期望非常高，除了 IP 使用，更是投入 AI 技术作为项目核心。项目方将 DuDu 熊的经历设计为一部百万字的长篇小说，NFT 持有人参与共创，整部小说中的插画设计、宣发文案等也有 AIGC 身影。甚至在第二批空投中决定完全由 AI 生成 NFT，实现 AI 与 NFT 相结合。这种社群参与互动的方式其实更接近 WEB3 社群，百度希望以共创的方式与持有人及社群成员慢慢建立共识，从而讲出自己的故事。

DuDu Lab 第一期发行总量 10000 个，地板最高 0.09ETH，目前 0.042ETH，交易量 478ETH。后续持有者权益还将包括官方文创商品、品牌合作资源、二期 NFT 空投等。DuDu 熊本身资源配置不错，但是叙事对于熊市有些审美疲劳，地板价在 mint 初期遭纸手大量出货后，基本再无变化。

哔哩哔哩：数藏平台和 Cheers UP NFT 系列

哔哩哔哩简称 B 站，是位于中国上海的一家以 ACG 内容起家的弹幕视频分享网站。经过 13 年发展，已成长为中国年轻世代高度聚集的综合性视频社区。目前用户数 3.1 亿，月活达 2.93 亿，日活 8350 万，其中 35 岁以下用户占比超过 86%，24 岁以下超过 75%。日均使用时长更是达到了 82 分钟。

围绕用户、创作者和内容，B 站构建了一个源源不断产生优质内容的生态系统，涵盖生活、游戏、时尚、知识、音乐等数千个品类和圈层，引领着流行文化的风潮，成为中文互联网极其独特的存在。目前，B 站 94% 的视频播放量都来自于专业用户创作的视频。在此基础之上，B 站提供了游戏、直播、付费内容、广告、漫画、电商等商业化产品服务，并对电竞、虚拟偶像等前沿领域展开战略布局。数字藏品也是其中之一。

可能是熟悉年轻文化与创意市场，B 站的数藏跟 NFT 之路都走的非常顺畅。今年 1 月，B 站首款数藏头像「鸽德」上线，全网限量 2233 个，仅面向每日登陆的 6 级用户，其中 233 个由平台分给了部分知名 UP 主。4 月国内数藏行业合规提上日程后，其他平台纷纷遇冷，但 B 站仍然较为激进，在开放了 30 天免费转赠后，鸽德藏品 价格一路走高，部分稀有属性更是达到万元以上。到了 7 月 B 站开始大量上新。根据 B 站数藏发售公告显示，8 月之前只有 15 款数藏作品，而截止 12 月 8 日，B 站总共发行数字藏品 42 款。频繁上新之下报名申购依然火爆。

总结 B 站数藏成功原因有以下几点：

  1. 差异化内容。各大平台数藏作品多集中在复刻古玩、名画品类，B 站数藏内容与其他平台有较为明显区别，自身二次元文化出发的数藏作品更加契合 B 站年轻用户。

  2. 联动平台原生内容与 IP。例如鸽德藏品支持衍生品制作与线上二次创作，之后的绝大部分藏品除了数字头像本体外，还附带空间页交互视频与高清版权视频，同样支持二创与授权服务，全面助力优质内容推广。

  3. 高使用场景。B 站数藏入口内置于 APP 中，自带的社交属性更易发挥数藏的稀缺性。除了头像本身的观赏属性，B 站还为持有者增加了双击特效、直播礼物、特殊弹幕样式等个性化场景，增加趣味性的同时还增加了使用价值。

  4. 30 天转赠时间。相比其他平台，较短的转赠与冷却时间无疑成为热度推动的助力一击。

总体来看，虽然目前国内还存在版权、交易管理等问题，但 B 站数藏在内容、场景、价值上都走出了一条不一样的健康发展道路。

NFT 方面，B 站在 4 月授权 Base Labs 向海外用户发行了“Cheers UP”系列 NFT，原定总量 10000 个，发行后实际 Mint 4526 个，后经 Holders 投票决定销毁 5000 个，无公售及空投。cheers UP 国库预留 241 个，Base Labs 团队预留 223 个。整体以验资抽奖 形式发白，售价 0.12ETH。Cheers UP 采用双合约机制，根据是否开图分为 Cheers UP Period 及 Cheers UP Official，是否开图由持有者决定。NFT 形象来自 B 站社区文化「Bilibili 干杯」。

除了上述提到的开盲盒，Cheers UP 还有一些别的有趣玩法。例如酿造 / 质押系统。持有人可以在官网通过酿造或质押获得累积积分 Z（开盒后为酿造功能，未开的盲盒提供对应质押功能），目前酿造每天 1 分，积分可以在商城中换取奖励。值得一提的是除了手机壳、礼盒等实物奖品外，还有 B 站大会员兑换卡。大会员是 B 站推出的付费会员机制，开通后可以解锁付费内容、4K 清晰度等。也可以选择兑换 Cheers UP NFT 盲盒。

目前除了 Cheers UP NFT，B 站还发行了 Cheers up 表情包系列 NFT、干杯节 NFT 纪念徽章。Cheers UP 系列地板最高 1.15ETH，盲盒目前 0.78ETH，交易量 2025ETH；已开图系列地板 0.529ETH，交易量 1040ETH。

  • 总体来看，Cheers UP 系列属于比较有诚意的，团队表现中规中矩，在初期 Mint 危机时表现出色，销毁使地板大涨 3 倍，但后期宣发较为停滞，未能走出国人圈子。期待下一个牛市可以晋级国产 NFT 之光。
