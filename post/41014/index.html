原文作者：Jake Pahor

原文编译：TechFlow intern

我研究了所有与以太坊第二层解决方案相关的项目，包括广义的 L2（Polygon）和特定应用的 L2（dYdX），并从这些伟大的项目中选出了我心目中的前五名。

1. 是否产生了收入？

深入研究 Token Terminal，发现了一些非常有趣的统计数据。在过去的 180 天里，获得的总收入（费用）为：

dYdX 2.122 亿美元

Arbitrum 1370 万美元

Polygon 880 万美元

Optimism 850 万美元

Loopring 250 万美元

正如你在图表中看到的，与其他项目相比，dYdX 具有压倒性的优势。Arbitrum 排在第二位，但您可以看到，在过去的一个月里，Polygon 一直处于领先地位，并且势头强劲。

2. 国库资源的分配

研究此问题的要素分为：

他们是否有不同的资产，或者只是他们的原生 Token（更多的风险）

国库有多大，给项目带来多大的发展空间？

Polygon：很难找到关于 Polygon 国库规模和内容的任何信息。然而，我们知道最近有 10 亿美元被承诺用于开发 ZK 项目，我们可以假设其国库的总规模远大于此。

Optimism：30% 的初始 Token 供应（12.9 亿 OP）将分配给社区基金（生态系统基金+ RetroPGF）=价值约 18.5 亿美元的 OP Token（基于当前价格 1.43 美元）。

dYdX：初始 Token 供应的 5%（5000 万 DYDX）被分配给社区国库。按照今天的价格（2.27 美元），这大约是 1.13 亿美元。

Loopring：20% 的 Token 供应（2.75 亿 LRC）被分配给 LEAF，价值约 1.1 亿美元（基于当前 0.40 美元的价格）。

3. 锁定的资金

如果大部分 Token 仍被锁定在归属时间表中，这可能会在未来造成巨大的抛压。

Polygon：大约 92% 的供应已经被解锁。最后的 8 亿个 Token 将在 2025 年 5 月前解锁。目前的流动性供应=918 万，而总供应量为 100 亿。

Optimism：总供应的约 5% 已经通过空投解锁。目前流通的供应量=2.15 亿，总供应量为 42.9 亿。

dYdX：所有供应量的约 11% 已经被解锁。目前的流动性=1.12 亿，总供应量为 10 亿。

Loopring：所有供应量的约 90% 已经被解锁。目前流通的供应量=12.45 亿美元，而总供应量为 13.74 亿美元。

4. 作用与案例

Polygon：MATIC Token 用于支付网络中的交易费用。你也可以用你的 MATIC 进行质押，作为保障网络安全的奖励。

Arbtirum：还没有 Token，但网络本身有巨大的效用，并提供更便宜的交易费用和可扩展性，同时保持以太坊的安全性。

Optimism：区块空间的权利是推动 OP 经济模式的可持续收入来源，并与网络本身一起成长。

dYdX：本质上是一种治理 Token，持有者还可以根据其头寸的大小获得交易费的折扣。

Loopring：协议费用分配给流动性提供者、保险人和 DAO 双方，激励 Token 持有人的良好行为。

5. 路线图

Polygon 刚刚宣布，他们将实施第一个 EVM ZK L2。Sandeep 还计划在 2-3 年内实现区块链的完全去中心化，并同时开发一些强大的产品，包括去中心化的数字身份证等。

Arbitrum 还没有公布完整的路线图，但有人猜测他们也会在某个时候通过空投的方式发布一个原生 Token。

Optimism 在下面的图表中列出了其路线图。

dYdX 正计划在 2022 年底之前，通过他们的 V4 版本成为完全去中心化的项目。这似乎是他们的主要焦点，也将是项目的一个巨大的里程碑。

Loopring 计划在 2022 年进行一些大的升级，如 Loopring Earn 和 zkEVM。

6. 板块类别

这个名单上的 5 个项目在本质上都非常相似。Polygon、Optimism 和 Arbitrum 都在争夺同一块蛋糕，竞相成为最好的 L2 扩展解决方案。

在这个阶段，从市值和 “护城河 “的角度来看，Polygon 显然是第一。然而，Arbitrum 和 Optimism 的使用率却更高，并且正在快速增长。

7. 团队和资金

Polygon 拥有一个不匿名的团队（Anurag Arjun、Jayant Kanani、Mihailo Bjelic 和 Sandeep Nailwal），并得到了该领域一些最杰出投资者的支持（Alameda、Binance Labs、Coinbase Ventures），总融资额为 4.515 亿美元。他们最近还宣布了一系列合作关系（迪斯尼、Reddit、Meta、Stripe、Draftkings），显示出他们强大的业务发展能力。

Arbitrum 也有一个知名团队和强大的靠山（Pantera, Alameda, Mark Cuban, Coinbase）。Offchain Labs 的母公司总共获得 1.237 亿美元的投资。

Vitalik 本人是 Optimism 的忠实粉丝，并表示支持他们的新治理结构。他们得到了 A16z、Paradigm 和 IDEO 的大力支持，总共获得了 1.785 亿美元的投资。

dYdX 获得了 8700 万美元的投资，来自一些著名的投资者，包括 Hashkey、Starkware 和 A16z，他们的团队也是不匿名的。

Loopring 从 Kosmos ventures、G2H2、Neo、Fundamental Labs 和 Obsidian 获得了 4500 万美元的投资。

8. 总结

根据我的研究和分析，我给每个项目的每个类别的评分都在 1-10 之间。正如你所看到的，明显最终的赢家是 Polygon。

该项目是最成熟的，拥有最大的市值，并且由于即将到来的以太坊合并，最近确实获得了一些关注。其他项目同样也非常有前途，从长远来看，都有很大的成功潜力。

这些并不是对这些项目的购买建议，这只是我的客观意见，也是我为进一步了解 L2 而进行的研究。在你投资任何项目之前，请做你自己的研究。
