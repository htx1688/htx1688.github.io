7 月 23 日，Uniswap NFT 产品负责人在社交媒体上发文表示，Uniswap 将通过集成 sudoswap 实现 NFT 交易，同时访问 sudoAMM 提供的链上流动性。

消息一出，市场上对于sudoswap 的讨论蜂拥而上。

NFT流动性提供者

今年3月，创始团队成员之一0xmons发文详细公布了sudoAMM的具体内容。

sudoAMM 是专为 NFT 设计的集中流动性 AMM 协议。交易者可以享受 NFT 和代币之间的低滑点互换，流动性提供者可以更好地控制他们 LP 的价格范围，并设置动态买卖墙。

与当今大多数其他 NFT 市场不同，sudoAMM 使用链上流动性池而不是链下订单簿。虽然这确实增加了一些 gas 费开销，但团队一直注意将这些成本降到最低。

作为回报，任何人都可以运行自己的前端和索引器，从而实现以去中心化为中心的方法。这也允许其他链上实体（如 DAO）更轻松地参与 NFT 市场，开放互动，例如在链上透明地设置 NFT 购买墙。

7月，sudoswap 公开发布 NFT 市场协议 sudoAMM。

官方称，目前，NFT 市场依赖于中心化的订单簿，存在停机和中心化风险。sudoAMM 通过完全在链上改变了这一点。任何人都可以仅使用以太坊在其应用程序中获得 sudoswap 市场使用的相同流动性。

用户可以：创建流动性池并根据价格曲线逐步买卖 NFT、为买卖 NFT 的流动性池提供流动性以赚取费用、直接以固定价格上架他们的 NFT。此外 LP 还可以完全控制流动性池的定价，并可以随时进行调整。

新的定价模型有无限可能：LBP 风格曲线、基于预言机的价格更新、自适应交易费用等等。并且由于 sudoAMM 的模块化架构，用户可以添加新的价格曲线，而不会影响现有的池或曲线。

简言之，该协议将向任何人开放，以便在发布时为任何 NFT 创建和交易池。

交易者将能够：

  • 用 NFT 交换 ETH

  • 将 ETH 换成 NFT

流动性提供者将能够：

  • 当交易者与流动性池进行交换时，存入 ETH 并接收 NFT

  • 当交易者与流动性池进行交换时，存入 NFT 并接收 ETH

  • 存入 ETH 和 NFT 并在交易者与流动性池交换时收取交易费

作为交易者，体验与其他交易所类似。

作为流动性提供者，体验与 Uniswap v3 最为相似。用户选择一个价格范围为其提供流动性，然后存入资产。

从技术上讲，sudoAMM 合约支持创建任意 ERC721/ERC20 池。但是，目前将仅支持 ETH ，以改善跨池的路由和流动性。

另外，sudoswap 官方称，由于收费，NFT 的市场结构效率低下。买家通常需要涨价 10% 才能达到收支平衡。但在 sudoswap 上进行交易意味着买家只需支付 0.5% 的费用（sudoswap 取消了交易版税），而在其他平台上通常需要支付 7.5%（2.5% + 5% 版税）费用，从而更好地发现价格。

简单一句话，Sudoswap 是一种 NFT DEX，它允许用户通过构建自己的 NFT 流动性池来分散购买和出售 NFT。此外，该平台针对 gas 费进行了优化，仅收取 0.5% 的费用，比 OpenSea 收取的 2.5% 的费用低五倍。

治理

sudoAMM 协议将由 $SUDO 治理代币管理。

该代币将是不可转让的，发行后的可转让性投票也将推迟到链上治理。

团队

sudoAMM 的初始开发团队包括：

  • 0xmons（智能合约/前端）

  • BoredGenius（智能合约负责人）

  • 0xhamachi（后端）

特色

在团队的发布的《深入挖掘：sudoAMM 对比其他 AMM》文章中，创始团队指出，sudoAMM 提供了效率和灵活性的改进。

第一，效率。任何首先使用 Uni v3（或任何其他可替代代币 AMM）进行价格发现的协议在进行定价计算之前首先将 NFT 转换为代币时都会支付额外的 gas 开销。在 sudoAMM 中，原生 NFT 与 ETH 配对。不需要额外的转换开销。

第二，灵活性。随着市场条件的变化，LP可能希望调整其资金池的参数。sudoAMM 为 LP 提供了许多选项。在任何时候，他们都可以：

  • 调整他们 LP 的价格范围（例如从 [0.667 ETH, 1.5 ETH] 到 [0.5 ETH, 2.0 ETH]）

  • 调整当前矿池价格（例如从 1 ETH 到 0.75 ETH，买/卖报价自动适应）

  • 调整买入/卖出的费用百分比

sudoAMM 还通过单面流动性池为 DCA 风格的策略提供原生支持。与 Uni v3 一样，LP 可以存放一对的一侧。例如，只用 NFT 交换 ETH（随着 NFT 价格上涨）或只用 ETH 交换 NFT（随着 NFT 价格下跌）。与 Uni v3 不同，这些池实际上只允许在所需方向进行掉期，这意味着一旦价格发生变化，就无需在不利方向“回购”。

总结

相较于当前市场，sudoAMM主要有以下优势，

1，对比于OpenSea等平台，sudoAMM更去中心化，符合Web3精神。

2，在一定程度上增加了NFT流动性。可能将更熟悉DeFi玩法的玩家带入NFT市场。

3，取消交易版税，对于NFT交易者来说是一个利好。

4，背靠Uniswap，流量不是问题。

但，sudoAMM也存在一定的风险：

1，协议安全问题，可能会被黑客组织利用。

2，取消交易版税，在一定程度上打击了项目方的积极性。
