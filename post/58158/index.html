作者：Zain Jaffer 翻译：善欧巴，金色财经

近日，贝莱德资产管理基金首席执行官拉里·芬克（Larry Fink）  在接受 CNBC 采访时谈到了加密货币。他表示，加密货币“与其他资产类别相比具有独特的价值，但更重要的是，因为它是如此国际化，它将超越任何一种货币。”

贝莱德目前是全球最大的资产管理基金，资产超过9万亿美元。芬克本人在 2017 年还是加密货币和比特币怀疑论者，当时有人引用他的话说 ，比特币只是“洗钱的指数”。他暗示想要洗钱的人越多（他们会使用比特币），价格就会上涨得越多。

但快进到 2023 年中期，他和 TradFi 的其他人现在唱起了不同的曲调。不仅如此，他还看到了加密货币使投资民主化的潜力。目前，在这个人工智能时代，我们仍然有规定，只有拥有数十万美元的合格投资者才可以投资某些资产类别，以保护他们的利益。

2017 年芬克的观点与沃伦·巴菲特、杰米·戴蒙、伊丽莎白·沃伦甚至比尔·盖茨的声明几乎如出一辙。这些人的共同点是他们主要来自婴儿潮一代或 X 一代。这很重要，因为我们许多人在二战后的岁月里长大，经常将我们可以持有并以更高价格出售的资产或有价值的东西与我们可以握在手中或实际触摸的东西联系起来。我们的资产包括汽车、房屋、艺术品、珠宝、贵金属以及手表、邮票、硬币等收藏品。

尽管债券和股票现在已经数字化，但许多老年人仍然将它们与随之而来的纸质证书联系在一起，包括我们可以欣赏的精心设计的复杂艺术品、字体和纸张。从这个意义上说，即使它们已经以数字形式存在了一段时间，但它们仍然属于我们大脑中的“实物资产”领域。

在音乐和电影方面，我们曾经拥有LP、8轨、盒式磁带、CD和DVD，它们通常有漂亮的专辑封面设计，我们可以收集、欣赏并与昂贵的Hi-Fi一起展示在客厅的架子上系统。MP3 和 MP4 音频和视频格式的出现在某种程度上让我们轻松进入了数字资产领域，但这些格式要么以微不足道的价格出售（例如每首歌曲 0.99 美元），要么在 Spotify、Netflix、HBO 等上流媒体播放。但同样，我们也有了这种娱乐内容的模拟版本。因此，我们中的许多老年人已经逐渐接受了这种进步。

但像加密货币和比特币这样的数字资产从推出之日起就完全是数字化的，因此没有模拟的对应物。这让婴儿潮一代和 X 世代的许多人望而却步。他们只是无法接受资产可以纯粹是数字化的。

但这是我们最终都意识到的。甚至芬克也可能意识到了这一点。那些在纯数字世界中长大的人正在步入成年并进入劳动力市场。他们的生活经历不包括旋转拨号付费电话、LP、CD 和磁带。相反，他们是伴随着 Spotify、Netflix、AirBNB、Uber 和其他数字服务一起成长的。那么为什么资产应该有所不同呢？例如，为什么货币仅限于山姆大叔作为货币发行的法定货币？这些是年轻一代可能会问的问题，但没有模拟同行的包袱。

因此，当芬克和贝莱德表示加密货币最终可能超越任何一种货币时，他们并不是想引起争议。他们只是认识到新技术的出现、被更年轻、更具冒险精神的阶层采用、然后传播到社会其他阶层的古老循环。

如果作为婴儿潮一代或 X 世代的成员，您很难掌握数字资产的概念，请考虑硬盘上甚至流媒体上的数字格式的歌曲或电影。您不会认为这些没有价值，因为它们是数字的，对吧？

另一种思考方式是一张有价值的棒球卡。熟练的人可能会伪造它，但你会知道它的价值在于真品，而不是赝品。如果 NFT 在数字上保证只有一份，其所有权转移给买家，那么其他非艺术家发行的类似 NFT 也都是假货，对吗？事实上，艺术品的数字所有权是无可争议的，因为它位于区块链上。

我们已经在飞机、汽车、个人电脑和互联网上看到了这一点。TradFi 中的一些人现在也意识到了这一点，如果我们不注意，我们可能会成为那些仍然想在转盘上听音乐或通过公用电话打电话的人之一。
