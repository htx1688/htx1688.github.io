如果你想购买或出售 NFT，你今天有近十几种主流选择。 在外人看来，它们看起来都一样，主页墙壁都是方形卡通人物。

然而，从标准的买卖市场（OpenSea、X2Y2、LooksRare）到试图通过创新解决方案降低 NFT 交易成本的模型正在演变。 在本文中，我们将解释三个主要的新兴技术：聚合、抵押和代币化。

什么是 General 和 Art Marketplace？

General Marketplace 是销售各种 NFT 的平台。 Art Marketplace 拥有精选的经过审查的艺术家和收藏品。 与任何事情都会发生的一般市场不同，艺术市场使用策展来为他们的平台提供所需的美感。

许多 NFT 交易市场让交易者 A 以特定的价格出售其资产。当交易者 B 愿意支付这个价格时，交易行为就会发生。这种行为在市场上一般被称为 “一口价” 的方式。OpenSea、LooksRare、X2Y2 和 SuperRare 就以这种固定价格的方式进行交易。

另一种方法是与其他买家一起竞标，就像传统的拍卖市场一样，出价最高的人获得 NFT 资产。 OpenSea 就是这种市场的一个例子。

在这两种情况下，买家都必须为交易支付 Gas 费，而缺乏流动性使得 NFT 交易效率低下。

什么是 NFT 聚合器？

NFT 市场中一个长期存在的问题是花费太多时间和 gas 购买多个 NFT。

顶级的交易市场 OpenSea 也不甘落后 NFT 市场快速发展的趋势。在 10 月 5 日宣布正式支持批量挂单和购买的功能，用户现在可以在 OpenSea 上单笔交易最多挂单和购买多达 30 件商品。

从数据上显示，尽管 OpenSea 已经整合了单个集合的批量购买，让不太活跃的交易者的生活更轻松，但认真的 NFT 交易者仍然会发现在交易时在所有这些不同的市场上进行交易很麻烦（而且成本高昂）。

GEM 和 Genie 市场是聚合多个市场的新平台。 用户无需比较不同网站上的价格，而是将所有数据集中在一个地方，并且也可以在同一个地方购买。促使 NFT 爱好者在购买多种商品时节省时间和金钱（gas 费）。

此外，在支付方面，Genie 目前支持 ETH 支付。而 GEM 接受所有 ERC-20 token 作为付款。 它允许用户使用他们选择的各种相应 token 进行一次付款。

什么是 AMM NFT 市场？

Sudoswap 是一个使用 AMM 模型的去中心化的链上 NFT 交易所。

DeFi 中的 AMM（自动做市商）是指一个分散的资产交易池，允许用户通过其流动性来进行加密货币的无缝交易。

AMM 的运作模式不同于传统的订单簿交易模式，AMM 的交易双方都是和链上流动性资产池在进行交互。流动性池允许用户以完全去中心化和非托管的方式在链上的 token 之间无缝切换。而流动性提供者（LP），则通过交易费用赚取被动收入，而交易费用则基于其对资产池贡献的百分比。

  • 用户可以从 NFT 集合的不同池子里买入或卖出

  • 池子可以买或卖给不同的用户

  • 用户能够在同一集合内用 ETH 创建 NFT 的流动性池 

什么是 NFTX？

NFTX ，用户可以抵押和代币化他们的 NFT，该平台希望这能解决 NFT 市场的流动性问题。创建 ERC-20 token 进行交易的平台。一个 NFT 持有者可以将他的数字资产转移到一个保险库中，并铸造代表 NFT 价值的可替换 token（vToken）。例如，一个 CryptoPunk NFT 铸造一个相应可替换的 CryptoPunk vToken。

用户在 NFTX 上出售 NFT 相当于铸造一个 ERC-20 token 和支付 10% 的铸造费，他们就可以在 SushiSwap 上将其兑换成 ETH。 token 是可替代的，这意味着 NFTX 的池子里拥有足够的流动性。要从保险库中移除 NFT，用户必须存入可替换的 vToken。 

至关重要的是，用户可以在 DEX 上轻松地将他们的 NFTX token换成另一个，基本上绕过了在市场上用一个 NFT 换另一个 NFT 的更长、更昂贵的过程。

小结

NFT 是区块链 token，但并非如此——缺乏流动性且交易效率低下。 已经出现了几种新的 NFT 市场模型来解决这个问题，并使交易 NFT 的过程更加顺畅。
