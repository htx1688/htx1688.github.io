TG bot 的故事还在继续，近几天大家谈论较多的就是一个名为「香蕉枪」（Banana Gun）的 bot 项目，该项目近日发布了 token 经济、空投和 token 预售等信息而受大家关注。此外，Dune 上的数据显示，香蕉枪也有着很庞大的用户群体，大部分用户熟悉的 bot 类项目只有 Uinbot 和 Maestro。但其实当大家还沉浸在 Unibot 的余温之下时，香蕉枪早已成为很多大户/KOL 的首选 bot 工具，而香蕉枪也是目前 TG 市场上公认的最快狙击机器人。

很多人看好该项目是因为 Unibot 的铺垫和 Banana 的狙击开盘，让用户手动做科学家。对比 Unibot 来看看 Banana Gun 的一些数据：

-Unibot：市值：约 8500 万美元，FDV：约 8500 万美元， 5% 的交易税；

-Banana Gun：约 130 万美元（800 ETH，占比 20% ），FDV：约 650 万美元， 4% 的交易税（其中 2% BANANA 持有者， 1% 团队， 1% 财库）；

目前 Banana Gun 市值不如 Unibot、但用户数量已经趋近于 Unibot 了，近期预售炒作之下超越 Unibot 也只是时间问题。Banana 简介、token 经济和预售相关内容可以参考：「赶超 Unibot，即将发售的黑马 Banana Gun 有多大潜力？」

与其它交易类机器人（跟单）相比的优势是，Banana Gun 专精狙击开盘，可让用户狙击即将上线的 token，或交易已发行的 token，接下来为大家介绍一下该项目的具体玩法和细则。

如何使用香蕉枪

进入 TG，搜索 BananaGunSniper_bot（注意区分真假）。

点击设置，根据自己的判断选取适合的钱包，用户配置如下：

接下来为大家解释一些重要配置的意思，帮助大家更好体验 Banana Gun。

Auto Snipe tip（对于未上线的 token）

设定用户愿意向区块构建者支付多少额外的 ETH 作为贿赂，以便比其它狙击用户更快，当然这只适用于计划狙击未上线的 token。注意：Banana 机器人使用 ETH 为标准作为贿赂，而不是 GWEI。此外，Banana 将使用 100% 的输入金额，权衡好你愿意花费的 ETH 金额。

Manual Buyer GWEI（对于已发行的 token）

该设置表示拥护愿意支付额外多少 GWEI 的数量，使用更多的 GWEI 可以让用户在区块中处于更靠前的位置抢先交易，用于计划购买的已发行 token。

Slippage

正常滑点在 0-100% 范围内，对于机器人项目，重要的是要考虑交易的规模、市场波动和流动性池深度。一定要设定一个适合你风险范围内的滑点数据。

First Bundle Backup Tip

如果 token 在 deadblock 上，或者是 MEV 发行的，就会导致「First Bundle or Fail」失效，这时 First Bundle Backup 就会发挥作用，First Bundle Backup 类似一个 First Bundle or Fail 被禁用后的预备狙击方式，同样是向区块构建者支付多少额外的 ETH 作为贿赂，以便比其他狙击手更早进入。注意：此选项仅适用于 MEV 启动和 deadblock 上的 token。

Approve GWEI

设置使用多少额外的 GWEI 来获得购买时的 token approve，更多的 GWEI 可以使交易处于领先位置。

Sell GWEI

设置使用多少额外的 GWEI 来获得出售时 token approve，更多的 GWEI 可以使交易处于领先位置。

Max Buy/Sell Tax

只有当所选 token 的购买税/卖出税等于或低于设置的数值时，狙击手/手动买家才会触发交易，香蕉枪将在税收变化的同一区块内运作。

Min/Max Liquidity Value

当 token 的流动性低于/等/高于用户设置的金额时，狙击手/手动买家才会触发。

钱包设置

用户最多可以在 Banana 机器人中连接 5 个钱包，用户可以选择导入钱包或使用 Banana Bot 生成的钱包，Banana 不会收集 bot 钱包的 Pkey。

狙击 token

设置完信息后，用户就可以狙击未上线的 token。首先粘贴该 token 的合约地址在 Banana bot 上，bot 会询问用户想要花费多少 ETH，然后设置钱包数量，狙击 token 功能收取 1% 的手续费。此外，香蕉枪还会设置每个钱包的最大购买量。

设置完成后，如下图所示，该图中的 0.03/0.02/1 分别代表：用户想要花费多少 ETH /贿赂或狙击费/钱包数量。

用户想要手动购买已发行的 token 时，同样需要粘贴 CA，Banana 将发送 token 的详细信息，用户将有根据选择来决定他们想要花费多少 ETH。如果最大钱包小于用户可以使用所选 ETH 购买的数量，它将使用所需的 ETH 数量购买最大钱包，剩余的 ETH 会转回钱包，购买 token 收取 0.5% 的手续费。

当你成功购买到 token 时会收到 Banana 的通知，也会有该 token 的一些信息，比如更改 tax、rug、市值、流动性等信息。

Banana 生态

Banana 的机制可以很好地用于激励和维持 Banana 生态的发展和平衡，其内部逻辑是把 TG 机器人变为「用户」，把用户变为该项目的投资者。

Token 持有者的收益分成

BANANA 持有者可以在 Banana dApp 上获得收入份额，收入分成将来自机器人交易量收入的 40% 和税收的 50% ，用以激励 Token 持有者，保持 BANANA 价格稳定。

注意：只有解锁的 token 才会计入收入分成，但 50% 的团队代币除外，需在快照之前持有才算合格。

用户奖励

用户通过 Banana 机器人完成交易，除了获得收入分成还将获得一些 BANANA，BANANA 持有者获得奖励的标准是：

BANANA 奖励 = 支付的费用 x 乘数（0.05 到 1 之间），可以去其官方 dApp 上领取奖励。

但随着时间的推移和用户数量的增多，这些 BANANA 会变得通胀，Banana 早就考虑到这一点，并采取了一些应对措施：

通过 1% 的财库回购，可以抵消大部分通货膨胀；

调节乘数（0.05-1）来平衡 token 释放；

销毁 BANANA 对抗通胀；

token 的高收益份额也会受到持有者的喜欢；

Ape to earn 制造的一个机器人飞轮（通过 bot 增加交易量-收益增加-token 交易量增加-收取更多费用-就会有更多的税收用于回购，排放-回购会保持一个相对轻通胀的状态），如下图

此外，Banana 也表示了，通缩/通胀将在-30% /+ 30% 的范围内，通过三个主要的因素来决定：Token 价格、Bot 交易量、总交易量。

空投细则

虽然香蕉枪空投占比不多，但还是比较受大家关注，本次空投分为两个部分：一是分给 Banana NFT 持有者的空投；二是分给做过该项目官方社交任务的空投。

根据每个 NFT 获取的难度，Banana NFT 分为五种，分别是：The Sniper、The Cannoneers、The Turreter、The Dual Wielder、The Tank Operator。每个 NFT 都附带不同的积分，按照积分来结算空投份额。计算方式是将每种 NFT 积分乘以总数量然后相加得出所有 NFT 的总积分， 100000 个 BANANA 除以总积分然后与自己持有的积分相乘就可以获得自己空投 BANANA 的具体数量；或者等 Gas 低时去其 dApp 领取。

而目前 Banana 的社交任务空投具体细则暂未公布。

结语

虽然很多用户没有资格参加预售，但有了 Unibot 的前车之鉴，大多都预测该 token 开盘后会暴涨。更有意思的是，很多人表示会用 Banana Gun 狙击 BANANA，「我冲我自己」，既然大家看好 BANANA，赶快学习一下如何使用和具体规则。DYOR！
