11月5日，马斯克在社交媒体披露，Twitter每天亏损超过400万美元。 面对如此棘手的问题，自收购以来，马斯克就已经搞了许多“大动作”，如大规模裁员、计划削减基础设施费用、计划改革内容审核……

《华尔街日报》称，马斯克希望对Twitter进行实质性改革。《纽约时报》还提到，马斯克的偏好是以他使用Twitter的方式运作Twitter：没有任何限制。

目前来看，马斯克似乎更专注于Twitter这个板块，并对其寄予厚望，在马斯克的带领下，Twitter会走向何方?

收购Twitter只为言论自由？

2021年1月8日，Twitter官方宣布，由于担心特朗普进一步煽动暴力，决定永久性关闭特朗普Twitter的8900万粉丝的账号，这一举动不仅让西方的言论自由收到了质疑，更是让Twitter的信誉一落千丈。

一直以来，马斯克都认为Twitter在限制用户的言论，并曾多次公开指责Twitter。马斯克还表示，封禁特朗普的账号是“错误的决定”。

今年5月，马斯克称，如果收购Twitter成功，那他将取消对特朗普的禁令。

而在收购Twitter之后，马斯克却表示，“Twitter不会让任何因违反Twitter规则而遭平台封禁的人回来，除非我们有这么做的明确程序，但这至少要花上几周时间”。

据外媒11月8日报道，马斯克目前的净资产跌破2000亿美元，其中很大一部分来自他在特斯拉近15%的股份。

自从今年4月他宣布即将收购Twitter以来，特斯拉的市值已经蒸发了近一半，他的净资产也减少了700亿美元。

其中主要原因是投资者担心马斯克会因为收购Twitter而出售特斯拉的股票。据悉，为了资助将Twitter私有化的交易，马斯克上周又出售了至少39.5亿美元的特斯拉股票。

根据美国证券交易委员会周二公布的文件，他又卖出了1950万股特斯拉股票。

在此之前，他还在今年4月卖出了价值超过80亿美元的特斯拉股票，在8月又卖出了价值约70亿美元特斯拉股票。

似乎马斯克收购Twitter事件带来了不小的损失，难道马斯克真的是为言论自由而战？在裁掉大部分审核人员后，Twitter用户增长创历史新高，但广告商却大量撤离。

在所谓的言论自由理念下，Twitter上的仇恨言论激增，一些具有种族歧视的敏感词汇的使用量上升了500%。

马斯克或将加密货币带入Twitter

作为加密货币的支持者，马斯在收购Twitter之后，狗狗币毫无意外的迎来了暴涨，之所以狗狗币能迎来暴涨，少不了马斯克此前的操作。

1月14日，马斯克宣布特斯拉商品支持狗狗币支付，特斯拉北美官网的“Cyberwhistle哨子”“Cyberquad for Kids儿童摩托”“Giga Texas Belt Buckle”皮带扣”可选择狗狗币支付。在宣布当天，狗狗币的价格上涨了20%以上，飙升至0.2美元。

6月18日，马斯克再次在Twitter表示，特斯拉和SpaceX的商品“也许有更多将会支持狗狗币”。

不过，这并不是唯一引起马斯克注意的加密货币。

在2021年2月提交给美国证券交易委员会（SEC）的一份文件中，特斯拉透露，该公司购买了价值15亿美元的比特币，未来可能会投资更多比特币或其他加密货币，并计划接受比特币支付。

消息一出，比特币当日上涨16%，创下了2020年3月新冠疫情引发金融市场波动以来的最大单日涨幅。

2021年5月，马斯克发推文称：“我们对比特币开采和交易中使用化石燃料的迅速增加感到担忧，尤其是煤，它的排放量在所有燃料中是最严重的。从很多层面上来说，加密货币都是一个好主意，我们相信它有一个光明的未来，但这不能以牺牲环境为代价。”

并表示，该公司也不会出售比特币，并仍计划“在挖矿过渡到更可持续的能源时”将其用于交易。马斯克还表示，特斯拉正在寻找“使用比特币能量/交易小于1%”的其它加密货币。

这距离特斯拉开始接受比特币支付还不到两个月，消息出来后的几分之内，比特币价格瞬间下跌了5%。截至当天北京时间上午8点05分，即马斯克发布消息的两个小时后，比特币下跌了17%，至47256.504美元。

目前为止，狗狗币是特斯拉唯一可支付的加密货币，所以在马斯克收购Twitter之后，狗狗币毫无意外的引来的暴涨。

近日，币安CEO赵长鹏对外表示其已经向马斯克汇出了5亿美元，而币安也因此成为了Twitter的股东。并且，币安正在组建一个团队，研究区块链和加密技术能如何帮助马斯克。

此后，不仅是狗狗币可以从Twitter 上的加密支付中受益，比特币、ETH等加密货币很有可能加入Twitter。

正在开发的Twitter NFT市场

在马斯克收购Twitter的同时，Twitter开发团队宣布，开始测试Twitter磁铁功能NFT Tweet Tiles，目前该功能已和Magic Eden、Rarible等NFT市场整合，此后，NFT链接会展示更大的NFT图片，以及标题和创作者等细节信息。

Tweet Tiles 是一种将可自定义格式附加到 Tweet 的方法，iOS 和 Web 上的用户可能会看到 Tweet Tiles 并与之交互，这些 Tweet Tiles 包含文本、图像、视频或其他元素，允许轻松与内容互动，并使时间线更加动态和可视化，会在用户粘贴 URL 时自动呈现。

值得注意的是，此前就合作过的OpenSea并未在Twitter宣布的首波测试名单之中。

据知情人士透露，OpenSea和Twitter讨论过项目上的合作，但在条款上没有达成一致。

一年多以来，OpenSea一直占据着NFT市场的主导地位，公司估值已超过130亿美元。此前Twitter推出的数字资产集成体验和NFT六边形头像正是与OpenSea合作的项目。

不过，在几个月前，Magic Eden曾超越 OpenSea 成为月交易量最大的 NFT 市场，加上Twitter与OpenSea的合作没有谈拢，这或许给了Magic Eden很好的契机。

对于NFT群体的用户来说，Twitter的这次调整无疑是给NFT群体用户带来了更好的体验，而对于广告为主要收入的Twitter而言，允许NFT平台投放广告和开放更多功能，可以使Twitter获得更多的收益。

马斯克全面拥抱Web3？

在未收购特推之前，马斯克就对web3领域颇感兴趣，此前对BTC与狗狗币的炒作更赚了不少钱。

并且，马斯克对外宣传自己为狗狗币教父，正如他所说，他的一举一动都影响着狗狗币的涨跌，马斯克在收购了Twitter之后，狗狗币更是从0.06最高涨到了0.159，涨幅高达259%。

不仅如此，在马斯克的设想中可以发现，代币机制、内容上链、赞成给与头部内容创作者分成激励计划等等，都是web3行为。

目前，其他社交媒体，如Facebook、Instagram、Reddit 等都实施了新的数字资产计划，各个企业都想抢先占领这个赛道。Twitter亦是如此。结合Twitter此前的一些动作，马斯克带领的Twitter很有可能要在web3投入大量的的精力。

结语

在马斯克的接手下，Twitter会离web3更进一步吗？

答案似乎是肯定的，一直活跃在web3领域的马斯克在追求着自己的精神价值，改变Twitter现有模式就是他“伟大征程”的一部分。

这位全球科技领域的传奇人物或许能给大家带来更多意想不到的惊喜。
