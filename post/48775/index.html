本文来自Bankless，由 Odaily星球日报译者 Katie 辜编译。

随着 ARB 空投在即，人们开始对 Arbitrum 生态系统密切关注。今天，我们来看看 Arbitrum 的一个主要产品——专注于游戏和社交的 Arbitrum Nova 链。

很多人听过 Arbitrum One，但 Arbitrum Nova 的解读不多。这个衍生 L2 的目的是简化包含大量交易的 Web3 游戏和社交应用程序。 本文我们将带新人们感受一下 Arbitrum Nova。

上周，Arbitrum 正式发布了期待已久的治理代币 ARB，该代币将于 3 月 23 日上线。值得注意的是，获得空投的资格标准之一是在 Arbitrum Nova 上进行了一些交易。

ARB 将用于管理 Arbitrum 生态系统，包括 Arbitrum One 和 Arbitrum Nova 以及其他 Arbitrum 链。考虑到 Arbitrrum 生态系统才刚刚开始发展，因此其 multi-rollup 核心的第一个基石——Arbitrum Nova 将蓬勃发展，并催生大量的机会。

什么是 Arbitrum Nova？

2022 年 6 月，Arbitrum 推出了 Arbitrum Nova，这是一款基于其 AnyTrust 技术的新的 L2 rollup，该技术针对超低成本交易和强大的安全保障进行了优化。自 2022 年 8 月以来，Arbitrrum Nova 专门为促进游戏和社交应用提供服务，而热门的 Arbitrrum One L2 继续广泛支持 DeFi 和 NFT 项目。

Arbitrum Nova 和 Arbitrum One 之间的关键区别在于如何处理交易数据。 虽然 Arbitrum One 一直将其所有交易数据批量处理至以太坊之上，但 Nova 使用数据可用性联合组织（DAC）来大幅降低成本，仅在 DAC 未能履行其职能时才将数据放在链上。

这种 DAC 方法使 Arbitrum Nova 成为交易量期望值高的项目的理想选择，例如经常制作新道具的游戏或具有多个链上交互的社交平台。此外，L2 的创造者计划实现更多的改进，以进一步降低交易成本。

Arbitrum Nova 当前链上活动

根据扩展解决方案分析服务 L 2B EAT.io 的数据，Arbitrum Nova 目前是加密经济中第 15 大 L2，锁定总价值（TVL）为 1050 万美元。当然，与 Arbitrum One 相比，这个规模相形见绌。Arbitrum One 是目前最大的L2，拥有 38.5 亿美元的 TVL，但 Arbitrum Nova 在过去一个月的 100 万笔交易也不容小觑。

目前 Arbitrum Nova 上运行的最大应用程序是 DEX Sushiswap。 因此，如果你需要在 L2 上进行任何 Swap，那么 Sushiswap 是你现在的主要起点。然而，如今 Nova 最有趣的叙事是它的游戏和社交应用新贵的不断崛起。例如，社交新闻聚合巨头 Reddit 正在将其社区积分系统过渡到 Nova。

使用 Arbitrum Nova Portal 门户网站查找 APP

如果你有兴趣发现和搜索目前正在 Arbitrum Nova 上构建的一系列应用程序，那么 Arbitrum Nova Portal 门户网站是一个很好的生态系统资源。它显示了从跨链桥到钱包等所有主要加密货币类别中目前可用或正在开发的潜力项目。

通过这个门户网站，我们可以很容易地识别已经或正在 Nova 上建立商店的早期游戏和社交应用。

其他 Arbitrum Nova 应用程序

来自 Treasure 的 Trove 是 Arbitrum One 上领先的 NFT 市场，它目前已经支持 Arbitrum Nova 的活动。 Arbitrum Nova 应用程序场景仍然相当原始，但如果你想在使用最大的 Arbitrum NFT 市场时“冲浪”，只需使用 Trove 上的网络按钮即可从以太坊或 Arbitrum One 切换到 Nova。

此外，目前在 Arbitrum Nova 上需要考虑的其他一些 NFT 项目包括 Zonic，一个涵盖 Optimism 和 Arbitrum 链的 NFT 市场，以及 NiftyKit，一个无代码 NFT 铸造解决方案，它刚刚增加了对在 Arbitrum Nova 上铸造 NFT 系列的支持。

使用 Arbitrum Nova 需要什么

要开始使用 Arbitrum Nova，请首先选择以太坊钱包，如 MetaMask，Coinbase wallet 或 WalletConnect。由于 Arbitrrum Nova 与以太坊虚拟机（EVM）兼容，基本上任何以太坊钱包都可以成为 Arbitrum Nova 钱包。

接下来，你需要使用 Chainlist 工具将 Arbitrum Nova 网络添加到你的钱包中，你要查找的网络 ID 为 42170 。

最后，你需要一些 ETH 来支付你在 Arbitrum Nova 上的交易费用。如果你在以太坊网络上有 ETH，你可以通过跨链桥将资金转到 Arbitrum Nova 上使用。

将资产通过跨链方式转移到 Arbitrum Nova

如果你想将基于以太坊的资产（如 ETH 和 ERC 20 代币）转移到 Arbitrum Nova 或从 Arbitrum Nova 转移出去，你可以考虑使用 Arbitrum 原生跨链桥系统。 只需输入你想要转账的金额，选择所需的交易链，使用你的钱包确认交易，然后你将可立即在 Arbitrum Nova 上进行交易。

总结

Arbitrum Nova 是一款新的 L2 扩展解决方案，专为处理大量 Web3 游戏和社交应用程序交易而定制。Nova 源自于 Arbitrum One，但 Arbitrum Nova 带来了一些新功能，如数据可用性联合组织（DAC）可降低交易成本。

尽管 Arbitrum Nova 仍处于“站稳脚跟”的阶段，但随着其生态系统开始入驻更多的游戏、社交和 NFT 项目，Arbitrum Nova 正在获得一些早期的势头。随着技术的不断进步，探索 Arbitrum Nova 目前为用户提供的体验 Arbitrum 生态系统前沿的机会，有可能抓住未来的增长商机。
