来源：Forbes；编译：比推BitpushNews Yanan

9月22日，加密货币交易所Bybit 宣布将主动暂停在英国的服务，以应对英国金融行为监管局（FCA）于2023年10月8日开始实施的加密货币推广新规定。

Bybit在声明中表示：“鉴于英国金融行为监管局在2023年6月发布的政策声明（PS23/6）中关于加密业务宣传推广的新规定， Bybit选择积极接受监管，并暂停在英国市场的服务。”

上个月，英国通过了《2023金融服务与市场法》（2023法案）。 该法案通过修改之前的的《2000金融服务与市场法》（FSMA）及其关于金融活动推广的规定，将加密资产纳入英国更广泛的金融监管体系中。

9月21日， FCA警告加密公司未积极参与金融推广新规。监管机构进一步表示，它最担心的是向英国消费者提供服务的海外加密公司。

为使加密产品的营销活动更加透明和精确，新规定引入了针对“首次投资者”的冷静期。根据拟议的规定，若企业未取得监管许可或未获得相应豁免，则不得邀请或劝诱他人参加投资活动。依据新的监管规定，可以开展此类推广活动的“已取得监管许可的主体”包括FCA授权公司、已注册的加密资产公司或已通过相关监管法规的审核，并获得相应授权的公司（译者注：此处”相关监管法规”指目前已提交议会审议，但尚未正式出台生效的专项法规）。总而言之，加密公司与消费者之间的沟通方式与沟通内容都将受到法规的约束，且所适用的法规也会是异常复杂。而鉴于违规可能导致罚款或入狱监禁，加密企业必须严格遵守相关规定，将合规问题摆在首位。

适用于新规定的“符合条件的加密资产（qualifying cryptoassets）”所涵盖的内容非常广泛，包括去中心化项目（如比特币和以太坊）及有明确项目方的中心化项目（如ICO）等。此外，通过将市面上的各种加密货币纳入现有的金融推广体系中，FCA也进一步拓展了监管范围，使其不再局限于传统的金融推广渠道，如投资招股说明书、电视广播广告和路演提案等。例如，在加密货币行业，常见的项目推广方式包括：赞助聚会、编程马拉松等线下活动，以及企业或者项目团队作为嘉宾出席某个会议、参与播客节目等。简而言之，只有持牌机构才能开展涉及到加密资产的宣传推广活动，且宣传内容也需要符合新规定的要求。未来，加密企业在开展涉及到英国消费者的宣传推广活动时，需要极度谨慎，并确保自身严格遵守新规定的要求。

虽然到目前为止，英国尚未采取美国的做法，强制加密公司将其代币注册为证券，但通过制订上述新规则，英国正在积极创建一套信息披露制度，以监管证券推广的方式，管理涉及到英国消费者的加密资产宣传活动。

关键性因素

·          英国金融行为监管局（FCA）

·          Sheldon Mills，消费者和竞争事务执行董事

·          Bybit,英国

历史上，英国FCA一直无权将比特币和以太坊等加密资产作为投资进行监管（或者说以监管证券等传统金融活动的方式予以介入）。而在美国，证券交易委员会（SEC）长期以来一直主张对加密行业进行监管，同时通过“豪威测试”（Howey Test）来确定哪些加密货币属于证券，从而将其纳入到SEC的监管范围。

《2023金融服务与市场法案》赋予英国FCA对特定活动的监管权限，例如涉及到以加密资产作为基础产品的交易安排或投资管理等。该法案着重关注了涉及这些活动的宣传推广行为。与SEC主要关注执法行动不同，FCA可能会通过这种“曲线救国”的监管方式，在不限制金融创新的前提下更有效地保护消费者。

“购买加密资产是个人的决定。但研究表明，许多人对仓促做出决定感到后悔。我们的规定为社会公众争取了时间和正确的风险警示，以便他们做出明智的选择。”  FCA消费者与竞争事务执行董事 Sheldon Mills 在谈到新规定时表示。

英国《2000金融服务与市场法案》第二部分“受监管活动与禁止的活动”第21条对金融推广行为进行了限制（即”金融推广限制”）。该条规定，除非获得FCA的授权或者特定推广活动得到监管认可的授权人的批准，任何人在业务经营过程中均不得邀请或劝诱他人参加投资活动。

违反上述规定的宣传活动主体将构成刑事犯罪（可能面临无限额罚款和/或入狱监禁），而涉及到的协议也可能会丧失执行力。值得注意的是，新要求具有了“属人管辖”特性，即无论加密企业或者推广方的所在地是哪里，针对英国消费者的所有金融推广活动都会受到该规定的约束。也就是说，如果一家公司注册、运营在英国境外，但其宣传活动涉及到了英国消费者，那新法规也将适用于该家公司。

如前述提到的，金融推广限制规定所适用的对象不仅局限于传统金融行业的“常规”营销推广活动，像是电视广告、投资备忘录等，还包括带有加密公司烙印的行业独属宣传活动，如播客、编程马拉松、展会、行业聚会，以及网络在线广告和推文等。此外，新规定也适用于加密企业与高净值和成熟投资者群体之间的宣传与推广活动。

这些法规变化可能对英国以外的加密企业影响最大。如我们前文所解释的，只要某一加密公司的宣传推广活动对英国市场产生了影响，不论其在何处，都会受到该规定的约束。因此，如果加密企业开展涉及到英国消费者的宣传活动，一定要格外小心，确保业务活动的合规性。

新规定将影响哪些加密资产？

适用于新规定的“符合条件的加密资产（qualifying cryptoasset）” 是指任何以加密安全为基础，具有价值或合同权利属性，且可用于转让与互换的数字表示。但这一概念不包括电子货币（e-money），也不包括已受到现有金融监管规定或控制的投资项目。具体来说，不适用该法规的资产有：1. 金融推广限制规定中已明确为“受控投资（controlled investments）”的资产（例如股票、投资信托、期权和期货）； 2.电子货币； 3. 法定货币； 4. 数字形式的法定货币； 4. 除发行人赎回外，无法转让或出售以换取金钱或其他加密资产的加密货币； 5. 由专业发行人发行的加密资产，用以发行人从有合作关系的特定服务商处购买商品和服务。

加密公司可以通过以下四种做法向英国消费者合法推广”符合条件的加密资产”：

  1.        由FCA授权的公司进行推广

  2.        由已符合专项法规要求的公司进行推广（专项法规目前已提交议会审议，但尚未正式出台生效）；

  3.        由或委托在FCA注册（依据2017年《反洗钱、打击恐怖融资和转移资金（付款人信息）条例》）但未经FCA授权的加密企业进行推广；或

  4.        推广宣传活动符合《2000金融服务与市场法案》（金融推广）2005年法令（Financial Promotion Order, FPO）中的豁免规定。需注意，现有的FPO中针对高净值和成熟投资者群体的豁免规定不适用于加密资产，英国政府将为这部分人群制订单独的豁免规定。

新规定还包括对投资风险警示的规范性要求，其中涉及到文字措辞、突出显示以及风险汇总链接。具体来说，警示内容的突出显示程度将取决于营销形式。加密企业在向首次投资者发送推广信息之前，必须先向其发送个性化的风险警示信息。根据新规定，推广加密产品或服务的公司将需要展现明确的风险警示内容，如：“除非您已做好血本无归的准备，否则不要投资。这是一项高风险投资，如果出现问题，您的权益将无法得到保护。花2分钟了解更多信息。” 同时，加密企业还需要提供一个链接，以方便投资者点击获取更多信息。

首次投资者在参与投资前，必须经过至少24小时的冷静期的“考验”。冷静期从投资者要求查看加密企业所发送的推广信息时起算，而后者在英国监管体系中有一专用名词，即“金融推广直接邀约（Direct Offer Financial Promotion，DOPF）”。可以说，几乎所有的金融推广信息都会被视作DOPF。 除非消费者在冷静期结束后再次确认要继续进行投资活动，否则加密企业将被禁止向该消费者发送推广信息。

因此，加密公司现在还需要对其营销对象开展充分的尽职调查活动，并确保其推广活动公平、清晰且不具有误导性。

同时，加密企业常用的“推荐朋友”和“新人奖励”等投资宣传措施现在也被禁止。 可以说，英国的监管措施试图在允许加密创新与加强消费者保护之间找到平衡。而在某些方面，英国的新规定与美国加密行业要求SEC所做的事情是一致的，即：通过要求信息披露来更好地保护投资者，而非试图通过执法行动直接打击、限制加密行业的发展。

如前文提到的，英国监管新规适用于去中心化项目及非去中心化项目，且两类项目没有被区别对待，均被一视同仁。尽管涉及到法规的适用性问题时，我们需要具体案例具体分析，但非同质化产品，如艺术NFT，可能不会受到该法规的约束。

Bybit表示，从10月1日起，平台将停止接受来自英国的新用户账户申请。从新规定生效的10月8日起，现有英国用户将无法存入资金、创建新的合约或增加所有产品和服务的现有仓位。同时，现有用户应减仓或平仓，并从平台上撤出资金。而Bybit 似乎也希望最终能够重新进入英国市场。该加密货币交易所指出：“暂停交易将使公司能够集中精力和资源，在未来最好地满足英国当局的规定。”

随着10月8日的截止日期临近，除了Bybit之外，其他加密企业也在评估其在英国的市场营销和运营策略。无论这些针对英国消费者的加密企业总部是否在英国，它们都需要在2023年10月8日之前确保自己的运营活动符合法规要求。这些要求将涉及到公司的营销合规、网站合规、社媒及渠道营销管理、以及为符合24小时冷静期要求而可能牵扯到的新用户开户流程更改等。同时，加密企业需要做好后台记录，以展示自身如何对客户进行分类管理，并确保与客户沟通的合规性。这些工作将需要加密企业投入一定的时间和资源。
