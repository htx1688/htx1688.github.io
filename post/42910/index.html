原文标题：《一文速览 101 个 Y Combinator 孵化的 Crypto 项目》（Mapping Out Y Combinator’s Crypto Alumni）

撰文：Edvinas Rupkus，The block 研究分析师

编译：隔夜的粥

要点：

  • Y Combinator 是一个受欢迎的创业孵化器，其每年都会举办冬季和夏季培训活动；

  • Y Combinator 在 crypto 和区块链领域的校友项目已经筹集了超过 10 亿美金的资金；

  • 一些知名的 Y Combinator 孵化项目，包括 OpenSea、CoinTracker 和 Stacks 等；

  • 总的来说，The Block 统计了 8 个类别的 101 个 Y Combinator crypto 校友项目。

Y Combinator 是一个流行的创业孵化器，其每年会举办两次为期三个月的培训活动。该孵化器会提供全面的服务，从与合作伙伴的个人一对一咨询会议开始，到演示日（Demo Day）结束。然而，合作关系并没有就此结束，因为 Y Combinator 参与者可以利用校友网络，并从网络中寻求额外的资金。

该加速器分两期向每家公司投资 50 万美元，其中 12.5 万美元投资于公司投后估值（post-money safe）的 7% ，另外 37.5 万美元投资于 「uncapped SAFE」阶段。

从 Y Combinator 毕业的一些最著名的 crypto/ 区块链项目，包括 NFT 交易市场 OpenSea、投资组合跟踪和会计软件 CoinTracker，以及比特币智能合约层 Stacks 等等。

几乎一半的 Y Combinator 加密校友项目属于金融服务或基础设施类别，而 Web3 以及 NFT/ 游戏则紧随其后。

Y Combinator 加密校友项目们已取得了相当大的成功，截至目前已筹集了超过 10 亿美元的资金。但值得指出的是，其中近一半资金都归功于 OpenSea 的筹资活动。OpenSea、Axoni、TRM Labs 以及 Proxy 是唯一一批获得 B 轮融资的加密项目，有超过一半的加密项目在毕业后没能筹集到后续的融资。

注：右下角的公司包括 Coinbooks、Aesthetic、Mysterious、Algofi、Coinrule、Coinfeeds、Hedgehog、Bitaccess、wOS、Bitstack、Bunkyr、Payourse、Quantstamp 以及 Flux，按募集金额排序。

The Block 总共分析并统计了 8 个类别的 101 个 Y Combinator 加密校友项目。

一、加密金融服务

  1. Apollo First 是 BizFirst 产品的开发商。该平台使商家能够创建和发送稳定币发票，接受稳定币付款，并使用 NFT 优惠券设计和运行基于 NFT 的忠诚计划。

  2. Arda 正在以合规的方式连接传统金融和 DeFi，这将使客户能够在单个 API 中访问高收益 DeFi 产品。

  3. Blocknom 是一家印尼平台，其为散户投资者提供加密投资服务。

  4. Coinbooks 是一种会计软件，旨在改变 DAO 执行会计的方式。

  5. CoinTracker 是一款完全支持美国、英国、加拿大和澳大利亚市场的税务合规应用程序。

  6. Derisk 开发密钥托管软件，使用 MPC 加密保护企业的数字资产。

  7. DolarApp 是一个全球美元账户的开发者，用户可以在该账户中以比索或美元接收、发送和支付，并节省美元以赚取收益。

  8. Excheqr 是一个加密安全平台，用于管理跨多个区块链网络的数字资产，并提供 DeFi 访问和风险监控等功能。

  9. Finnt 是采用区块链技术的金融应用，用于提供全球联合和储蓄账户。

  10. Flux Payments 是一个金融服务平台，为全球用户提供资产投资和交易服务。

  11. GoSats 是一家比特币奖励公司，它可以让消费者在网上购物时轻松累积比特币。这家初创公司在其平台上拥有 120 多个品牌，包括 Flipkart、亚马逊、Swiggy、Big Basket 以及 Myntra。

  12. Instamenu 正在为餐厅业务和食品卡车建立一个支付平台，平台的所有权将分配给参与者。

  13. Jasmine 是可再生能源等气候资产的去中心化市场。

  14. Kapstar 为基于加密稳定币的美元支票账户的跨境支付提供了一个 UX。

  15. LiquiFi 帮助公司和 DAO 自动化其代币归属、管理和分配给员工、投资者、合作伙伴以及社区成员。

  16. Lyra Pay 允许用户通过购买预付卡，在在线商家使用加密货币进行支付。

  17. Multis 是由 Thibaut Sahaghian 和 Théophile Villard 共同创立的企业 crypto 银行接口。该公司的界面可帮助企业存储、管理、简化支付，并通过加密货币赚取利息。

  18. Paymobil 打算通过使用稳定币成为 Web3 的 CashApp 或 Venmo。

  19. Payourse 是一个平台，通过提供基础设施帮助开发人员构建用户友好的加密产品，帮助加快对 crypto 的访问。

  20. Ping 是一家面向国际承包商和自由职业者的数字银行，为他们设置一个加密账户，以便他们可以访问个人美元账户（无论他们居住在哪里）。

  21. Rampay 是用于 Web3 应用程序的 crypto 出入金支付网关。

  22. Realty Bits 是一家提供私募股权产品的数字基础设施提供商，允许用户获得商业房地产、能源基金、固定收益证券、股票等现实世界资产的部分所有权。

  23. Request Network 是一种数字资产支付技术，它使公司和个人的加密发票能够以 crypto 的方式获得支付。

  24. Rio 是 LATAM 的加密支付提供商，用户可以使用现金、银行卡或银行转账为任何钱包购买和出售加密货币。

  25. Soon 是一个投资平台，它可以自动将闲置资金投资于用户的银行账户，涵盖了 crypto 和更传统的资产。

  26. Striga 是一种 crypto 原生银行服务，用于发卡、IBAN、托管加密钱包以及欧洲合规解决方案。

  27. Terrace 为希望接触 DeFi 的机构提供了一个符合监管要求的平台。该该平台处理资产托管、资金流动、执行、合规监控和交易会计。

  28. Valiu 是一家基于区块链的国际汇款公司。

  29. ZeFi 是一个个人金融平台，它提供一个储蓄账户，为用户的资金自动复合收益。

二、基础设施

  1. Cashmere 是 Solana 上的一个多重签名钱包，供创作者和开发人员管理他们的资金和工作流程。

  2. Alterya 为去中心化生态系统提供数据基础设施，其具有用于访问任何 DeFi 协议的统一 API 层。

  3. Blockscope 是一个 Web3 数据平台，供开发人员从区块链网络收集链上数据，并将其插入到他们的应用程序中。该平台的一些服务包括合约分析、钱包分析、事件和交易数据库即服务、推送通知、webhook 以及开发工具。

  4. Bunkyr 使公司能够使用一个简单、硬件安全的 API 来保护用户的数据。当用户忘记密码时，使用 Bunkyr 的应用程序可以为他们提供熟悉的、强大的恢复方法，以恢复对加密数据的访问。

  5. Chainsight 是一种加密和 NFT 诈骗检测解决方案，用于保护互联网相关公司和用户免受 crypto 欺诈。这些解决方案有助于防止用户与恶意应用、钱包、假代币 /NFT 和网络钓鱼网站进行交易。

  6. Immuna 协助加密货币投资者管理投资该资产类别的风险。该平台监控并保护用户免受意外攻击和不良经济状况的影响。

  7. Internet Friends 是一个以太坊钱包，用户通过它可以和朋友一起购买和使用数字资产和 NFT。

  8. NuCypher 支持在公共区块链和去中心化存储解决方案上进行端到端加密数据共享。使用 NuCypher，开发人员可以向一定数量的接收者授予和撤销对敏感数据的访问权限。该公司由 MacLane Wilkison 和 Michael Egorov 创立。

  9. Poko 正在开发 DAO 工具，这将帮助企业家和实体创建和管理他们自己的 DAO。

  10. Maybe Nothing Labs 是一家提供索引器即服务的数据基础设施公司，以帮助 Web3 团队为其应用的数据后端提供支持。

  11. Quantstamp 是一家进行智能合约、链下、网络和前端审计的公司。

  12. QuickNode 为团队提供区块链基础设施和服务，包括用于以太坊和比特币的全球 API 和 Web3 开发工具。

  13. Rainbow Wallet 声称是创建以太坊钱包、收集 NFT 和探索 Web3 新世界的一种有趣、简单、安全的方式。

  14. Satsuma 为 Web3 开发人员提供了用于索引和查询实时区块链数据的工具。

  15. SimpleHash 是一个 NFT 多链 API，使 web3 开发人员能够轻松查询 NFT 数据。

  16. SlashAuth 是用于 web3 应用程序的开发者平台。该平台提供登录功能、基于角色的访问控制、数据存储等。

  17. Stacks 是一个开源网络，支持在比特币区块链上开发应用、智能合约和数字资产。Stacks 连接到比特币并使其可编程，充当区块链的可编程基础层。

  18. Stackup 是一款移动加密货币钱包，旨在让初学者能够轻松购买数字资产和 NFT。

  19. Yatima 是一个 Substrate 区块链，它使用形式证明和 zkSNARKs 来增强智能合约的安全性和可扩展性。

  20. Zoltar Labs 是一个 Web3 开发人员执行智能合约测试的受控测试环境。

三、NFTs/ 游戏

  1. Aesthetic 是一个为 NFT 社区设计的无代码平台，创作者可以在其中创建免费或付费的 NFT 投放。

  2. Decent.xyz 是数字内容分发区块链平台 DCore 的开发商。该公司由 Matej Boda、Matej Michalko 和 Wayman Kwan 于 2015 年创立。

  3. Filta 正在建立一个美颜（face filter）NFT 市场，创造者可以从他们为粉丝创建的限量版美颜中获利。

  4. Hyper Online 可以创建独特的化身，这些化身可以在他们的社交网络上使用来录制 TikTok 和其他直播。

  5. LootRush 是区块链游戏的一站式商店，其提供快速入门流程和跨多个游戏的 NFT 租赁。

  6. Metafi 通过其 SDK 为 Web3 游戏开发者提供区块链基础设施。该公司的 SDK 可帮助开发人员在游戏中嵌入非托管钱包，访问区块链数据的 API，并集成 L2 解决方案 StarkNet。

  7. Mintplex 为 NFT 项目提供无代码工具。该平台为新发行的 NFT 集合处理 IPFS、智能合约和铸造页面 / 流程。

  8. OpenSea 是 一个 NFT 交易市场。用户可以购买、出售和拍卖他们的物品。该市场由 Devin Finzer 和 Alexander Atallah 共同创立。

  9. Players’ Lounge 是一个平台，旨在通过提供 token 将粉丝与他们最喜欢的运动员联系起来，从而增强体育社区内的关系。

  10. Rosebud A.I. 提供了一个基于订阅的应用，该应用使用人工智能来帮助创作者制作动画图像和 NFT。

  11. Sonar 是一家平台提供商，使用户能够跟踪他们所有的 Solana DeFi 资产。

  12. Supercool 是一个 NFT 铸造市场，供用户发现新的 NFT 项目。该公司还提供其它的功能，例如对 NFT 铸造地址合约的安全检查，以提供和使用 Flashbots 等高级功能来避免抢先交易。

  13. Winter 提供了一个可嵌入的小部件，使客户能够使用信用卡购买 NFT。

  14. wOS 是一个 Web3 健身平台，健身游戏的玩家可以通过完成日常健身和活动目标来获得奖励。

四、Web3

  1. 2 Cents 是一个社交应用，用户可以在其中向基于论坛的社区集合贡献内容并获得代币奖励。

  2. Argo（Movie）是一个面向电影业和电视发行的 NFT 平台，使创作者能够通过其作品获利并与观众建立关系。

  3. Democracy Earth 是一个开源平台，致力于通过向平台上验证身份的用户提供普遍基本收入（UBI）来使治理透明化。

  4. Mysterious 是一个供加密社区以开放式论坛方式连接的平台。该团队将自己与 crypto 原生的 Discord 进行比较。

  5. NewShades 正在开发 Discord 的替代方案，其将专门构建为 web3 原生和去中心化自治组织 DAO）。

  6. Notebook Labs 利用零知识证明为区块链应用提供匿名和负责任的身份。

  7. Openpod 旨在让开源项目更容易获得筹款并留住开发人员。该平台使开源项目能够启动一种激励开发人员贡献的代币。

  8. Protocol Labs 是一个开源研发实验室，它是去中心化存储解决方案和平台 Filecoin 背后的主要开发商。该项目由 Juan Benet 于 2014 年创立。

  9. Proxy 是一个 web3 身份钱包，旨在将其用于基于区块链的产品以及现实生活中的用例，例如使用蓝牙技术打开上锁的门。

  10. Questbook 运营着一个同名平台，允许公司向开发人员提供赠款，并以高效和更透明的方式对他们进行投资。

  11. SolStar 是一个创建和组织代币社区的平台，由 Solana 提供支持。

  12. Spruce Systems 是一个数据隐私平台，它通过为用户提供便携式声誉所需的必要身份验证、认证和存储，提供对数据的分散访问控制，以及 Web2 API 和 Web3 之间的互操作性，使用户能够跨网络控制其数据。

  13. the402 正在构建一个去中心化的直播平台，该平台将使艺术家和内容创作者能够进行私人直播，只有购买了 NFT 的粉丝才能观看。

  14. Trendex 是一个社交代币平台，使人们可以像投资股票和公司业绩一样投资运动员、艺术家和创作者的业绩。

五、交易 / 经纪

  1. Bitstack 是一个允许用户设置比特币定投的应用程序。

  2. Coinrule 是用于加密货币平台的智能交易机器人，允许用户完全控制他们的加密交易，同时使用机器人进行自动化交易。

  3. Hedgehog 是一个加密货币聚合器和投资组合管理器，使用户可以在交易所进行交易、跟踪钱包和了解 crypto。

  4. HyperTrader（HyperLinq）是一家为加密货币交易者构建机构级交易工具的金融科技公司。通过使用其桌面交易终端 HyperTrader，用户能够跟踪所有主要加密交易所的投资组合和市场。

  5. Mudrex 是一个提供投资工具的平台，例如自定义加密货币篮子或符合投资者风险水平的精选投资组合。

  6. SFOX 是专业交易者和机构投资者的加密货币优质交易商。该公司由 Akbar Thobhani 和 George Melika 于 2014 年创立。

  7. Tienda Crypto 是阿根廷的一家数字资产交易所，用于买卖加密货币。该交易所还支持在以太坊和 BNB 链上进行交易。

  8. Tinker 是一家加密货币电子交易和做市公司，成立于 2014 年。

  9. Weltio 提供一个美元经纪账户，其具有现金通道、机器人顾问以及为拉丁美洲用户量身定制的投资组合等功能。

六、数据 / 分析 / 信息

  1. Canopi 提供了一个排放计算器和 API，允许用户计算其 BTC 钱包的二氧化碳排放量，并投资于碳去除解决方案。

  2. Coinfeeds 提供 crypto 相关推文、播客和其他新闻。

  3. DanteHQ 提供了一套针对每个特定 crypto 项目定制的基础数据指标和数字资产研究。

  4. Invezo 提供了一种 crypto 分析工具和 API，旨在成为 crypto 领域的彭博终端。

  5. Notabene 协助企业进行合规的 crypto 相关交易。该公司总部位于纽约，由 Alice Nawfal、Andres Junge、Ania Lipinska 和 Pelle Braendgaard 于 2020 年创立。

  6. roverX 正在为 NFT 开发一个区块链分析平台，该平台监控区块链网络并索引各种事件，以创建可操作的用户洞察力。

  7. TRM Labs 是一家区块链情报公司，提供风险评估并帮助防止使用数字货币的欺诈和金融犯罪。

七、DeFi

  1. Algofi 是一个基于 Algorand 区块链的借贷市场和稳定币协议。

  2. Dialect 是一种链上消息传递协议，支持 dapp 到用户以及用户到用户的消息传递，将推送通知直接发送到用户的手机。

  3. EthosX 是一个用于 crypto 和传统衍生品的去中心化金融平台。

  4. Meson 是一个平台，能够在所有领先的区块链和 L2 rollup 中执行低成本、零滑点的稳定币跨链交易。

  5. Sylva 是一种加密货币钱包和投资协议，可实现跨链收益机会。用户能够持有和质押一种数字资产，同时也能够将其投资于非用户基础资产固有的 DeFi 应用。

八、企业

  1. Axoni 是一家资本市场技术公司，提供分布式账本技术，可实现机构之间的数据共享和数据同步，同时充当客户在其上构建金融平台的基础设施框架。

  2. Bitaccess 是加拿大的加密货币 ATM 生产商。该公司还提供企业解决方案、KYC/ 合规和智能合约开发。该公司由 Haseeb Awan、Moe Adham 和 Vignesh Sundaresan 于 2013 年创立。
