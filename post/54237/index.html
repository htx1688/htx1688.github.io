作者：Cookie

当以太坊 NFT 圈子被 Azuki Elementals 平地一声雷地掀起波澜之时，比特币 NFT 圈子却从 BRC-20 与递归小图片的热潮中逐渐退烧并再一次进入相对沉寂的时期。

正如 @0xjingle 说的那样，比特币 NFT 的「黄金时代」已经过去了。很难说究竟是什么时候，是 $OXBT 爆火之后一堆 100% 照搬其运营思路的 BRC-20 仿盘出现的时候？是最「成功」的那个采用了白名单机制的小图片项目 Bitcoin Bears 顺利地「吃下」接近 30 个比特币的时候？是大量的匿名项目方开始出现，买一堆假的 Twitter 粉丝人造 Hype 的时候？还是各种 BRC-XX 协议满天飞带起许多「BRC-XX 协议」的第一个项目炒作然后又一地鸡毛的时候？

免费、公平分发、First is first… 这些几个月前的比特币 NFT 圈流行词，不知不觉中已经被埋在了比特币 NFT 还不长的历史中。比特币 NFT 的「黄金时代」是如此短暂，在那短短的 2-3 个月中，开发者们无私地为这个生态做着技术维护与创新，项目方们更多都是因为兴趣和各自不同的理想选择在一个热度不高且看不到盈利路径的生态中启动了自己的项目，玩家们与项目方们之间没有高高在上的某一方而是重新回到平等的朋友关系…

本来，这篇文章的主题将是「比特币 NFT 圈白名单机制的流行体现了市场的什么变化」，但很快，我意识到这极有可能又是对过去的「黄金时代」的一种怀念。过去的已经是过去，更重要的还是现在。面对重新陷入沉寂的比特币 NFT 市场乃至整个 Crypto 熊市，没有收入的比特币 NFT 项目方们现在是一个什么样的心态？他们又打算做些什么呢？

「不吃草」又要「奔跑」，有多难？又为什么？

过去的半个月，失眠已经成为了我的习惯。我习惯抱着我的妻子，在她睡着以后悄悄地走出房间，坐在一片黑暗当中。我不知道该做什么，往往是打开电视与 Switch，调成静音，脑袋一团杂乱地玩上 10 来分钟。然后，又瘫坐在沙发上看着电视机里的林克在原地发呆，关掉电视，走去阳台抽掉很多的烟。

在律动的工作得以让我非常早地接触到了 Ordinals 协议，也让我第一次从对着各个项目指点江山的「键盘侠」迈出了自己做项目的第一步。在 3 月份，我启动了自己的比特币 NFT 项目 Bitcoin Pizzas。虽然，我打从心底里不用「项目方」来称呼自己。

我的朋友们都叫我甜饼，他们和我说，甜饼，你已经成功了，不要想太多。某种意义上说，我也许确实是成功了。

因为「做这件事情很酷」以及对我的信任，我成功聚集起了 pizzab0y、FyFk 以及 Ssyan 一起将 Bitcoin Pizzas 从一个脑洞变为了一个真实存在的项目。那时还是 Ordinals 刚诞生不久的 2 月份，我非常着急地想找人把「披萨」这个想法落地，但现成的资源要么对 Ordinals 协议完全不熟悉，要么是觉得收益与投入完全不成正比。大家都是想赚钱的，我能找到另外 3 个人一起做事情而不计回报，真的很不容易，也许这算一种成功。

这个项目的成交量接近 11 枚比特币。我们在开始的时候就抱着「相信大家会看见」的心态去做的，但 FOMO 来临的那一天，大家的第一反应并不是兴高采烈，而是感觉到压力扑面而来。无论市场再怎么 FOMO，我们也无法从快速增长的成交额中获取到一毛钱收益。水涨船高的价格，势必会推高某些持有者对我们的预期，然而我们并没有任何的收益去做更多的事情。但是，社区从来没有骂过我们，反而是给了我们相当多的精神支持，甚至是拿出自己的小图片支援我们去做营销。考虑到我们没有任何白名单，而且在发射的时候只有区区 300 Twitter 粉丝，也许这也算一种成功。

但我一直认为我从来没有成功过，因为成功在我看来是用于形容一件事情的结束的。

市场瞬息万变，热点切换极快，我非常担心「披萨」的注意力消失，而这也是事实上正在发生的。对此，我不知道可以做什么，因为我们，没有钱。

我已经花了数万元的资金来做这个事情，但我依然没有办法给信任我的小伙伴任何回报。即使他们不止一次和我说，钱真的不重要，又不是要饿死了，但当我看到我们的画师在朋友圈给她自己的另一个卡通形象众筹都成功筹集到了数千元的资金时，我真的很焦虑。因为焦虑，我与 FyFk 有过一次争吵，他觉得我开始「像一个老板一样发号施令」，而他只是因为兴趣才做这件事情。

我们否掉了包括发行 BRC-20 Token、后续项目 Paid Mint 等等这个圈内能够赚钱的可能路径，因为觉得这个事情的一开始或者都不是一个 NFT 项目，而是一场用实际行动对 PUA 、拿了钱不认真做事情等等我们讨厌的现象抗议的行为艺术。

没有钱的情况下还能做什么呢？除了每天在 Twitter 用「披萨」形象更新一张「GM」图片，偶尔有温暖的社区成员帮我们制作一些梗图还有视频，我们实在无法做到更多。这其实就引出了一个悖论——像 Azuki 拿了这么多钱的项目，他们可以且应当做更多能凝聚社区乃至发挥商业价值的内容，但这些内容对他们赚钱而言显得是如此的不重要。但对于我们这样的项目，我们唯一能够有效活跃社区乃至持续保持注意力的方法，恰恰就是内容。因为一张小图片就是一张小图片，如果没有可以让社区讨论起来的内容，社区除了这个涨了那个跌了，还能指望大家聊些什么呢？

我在社区乃至我个人的 Twitter 高强度地用「抽象」的手法娱乐式地分享我的生活，我们有了一些梗，这让我们能够摆脱「开局一张图」言之无物的困境，但这还远远不够。

经过半个月的思考，我决定「抽象」下去，运营起个人 Twitter 账号的流量并做一些周边贩卖的尝试，这应该是我们能想到的唯一「站着赚钱」的路子。对于在此方面毫无经验的我们而言，这是艰难的。我们要去学习和担心的包括但不限于样式算不算好看、质量能不能过关、如何控制成本… 其实最后卖得多与少，对我们而言并不重要，因为我们又一次把这个举动视为「实验性的反抗行为艺术」。如果我们在没有任何收入且资源如此有限的情况下尚且做出了大家能够满意的东西，那么那些拿了很多钱却不做事情的项目应该感到羞愧。 他们甚至都不会知道有「披萨」这么一个「弟中之弟」的存在，但无所谓，我们只是厌倦了用键盘来表示自己的不满，我们要用实际行动去反击。

这是「不吃草」又要「奔跑」的原因。当然，还有另一个原因，我们必须做点什么来让自己感到这个项目还活着。 我们当然也可以安慰自己，最开始想做的已经都完成了，躺平也不会有人说什么，尤其是在这样一个熊市，下班以后多放松放松比整一堆不会有人看见的事情或许是更好的。但是… 还是做吧。

艰难探索着发展之路的「草根」们

启动了自己的比特币 NFT 项目后，我幸运地以「项目方」的身份接触到了许多在比特币上做小图片项目的朋友们。

蓝莓（@aowu2048）是 Pixelbirds Sparrow 的创始人，这是中文圈的第一个比特币小图片项目。蓝莓是开发出身，纯粹因为对 Ordinals 协议的兴趣而做了这个项目。在这个项目外，他有一个成熟的团队，之前在以太坊上做 NFT 相关的尝试。他曾和我说，Pixelbirds Sparrow 对他而言像是一个「DLC（游戏扩展包）」，并不是指这个项目成功启动完就拉倒了，而是这个项目赚不赚钱对他来说没有那么重要。因为兴趣，Pixelbirds Sparrow 的启动也是公平且免费的，就是比特币 NFT 老玩家们曾经熟悉的那个模式——所有喜欢项目的人都可以在官网上下载自己喜欢的图，自己去刻录。

蓝莓一直想在技术层面为 Ordinals 生态做更多的事情。他在技术层面帮助 Bitcoin Knight、Pixel Hares 等项目完成了发射，并希望把这种帮助转化为一个 Launchpad 产品，再将这个产品转化成 Pixelbirds Sparrow 为持有者赋能的一部分。在今年的披萨节，我也与他做了 BRC-1155 协议的「Open Edition」尝试。蓝莓、Ivan（BRC-1155 的开发者）以及我在私下讨论过很多如何去发展 BRC-1155 的事情，但对于都有本职工作的我们来说，发展是十分艰难的，不仅要完善 BRC-1155 的索引，还要在市场侧推进 Magic Eden 等交易市场以及 UniSat 等钱包应用来适配这个协议，更有许多琐碎的事情需要讨论和细化，比如 Burn 功能的具体实现等等。我们不是以一个严格意义上的团队在工作，对效率更是无法提出硬性的要求。

ACE（@CKN_ACEE）是 Bitcoin Knight 的创始人，他曾为 UniSat 以及 $XEN 创始人 Jack Levin 的 BRC-20 项目 $VMPX 设计了 Logo。因为对 Ordinals 的热爱，他为他曾经发在 Cardano 上的项目制作了像素版本并空投给原来的持有者们，尝试将更多的玩家带入到比特币 NFT 的圈子中。在上周，Bitcoin Knight V2 顺利完成了发射，并且在公售环节筹集到了大约 3 枚比特币的发展资金。开玩笑地说，比起我和蓝莓，ACE 显得不那么「草根」。

ACE 是这么理解「收费」这个事情的——NFT 是一种娱乐型的消费，是主菜吃饱的饭后甜点。从本质上来说，钱到图片的转换完成后，交付就结束了。Bitcoin Knight 从最开始的中心思想也是降低用户的期待值，不要想太多，他相信这是一种良性的成长方式，超额交付会满足持有者的情绪价值。在给原 Bitcoin Knight 持有者的第一代的比特币像素版本空投时，ACE 也强调这只是一份礼物，这礼物可能一文不值，但至少值 1 聪，Just take it。

对于公售筹集到的资金，ACE 说已经足以落地他们想要做的协议 BRC-721K。这是一个针对比特币小图片的质押协议，除了改进 BRC-20 Token 存在的一些问题外，BRC-721K 还能够部署「彩票」来对小图片的流动性进行一些改善。简单来说，就是可以将自己想要出售的 NFT 当作「奖品」，吸引有兴趣的玩家购买「彩票」参与抽奖，比起卷地板还需要等待成交来说，这可能会是一条快速出售自己小图片的路径。

CXEEO（@CXEEO_）是 Neo Yauto 的艺术家，也是该项目的 COO。与蓝莓、ACE 以及我的项目都不同，Neo Yauto 是先发了 BRC-20 Token，再发行小图片的。与 Bitcoin Knight 一样，CXEEO 和他的团队先前在其它链上也有一个成功的项目，他之前的项目是在以太坊上的 NUO2069。

CXEEO 说，Neo Yauto 转到比特币生态发展以后也是不赚钱的。他们想做的事情和原来在以太坊上的差别不大，还是想打造一个广府文化、粤语文化的城市 IP—— 「新妖都」。他们已经勾勒出了链上平行宇宙的一个城市版图，接下来还会为自己的持有者以及其它合作项目的持有者发放一个「市民卡」。同时，这也是新妖都开启下一个阶段的钥匙，逐步构建出一个 Ordinals 多元宇宙并丰富这个虚拟城市的故事，让 Ordinals 生态可以在这里形成交互。他们正在策划一场大型线下活动，希望通过活动连接 Web2，看看链上的虚拟广州落地到现实的广州这样的虚实结合会产生什么化学反应。

这是一个很大的目标，CXEEO 坦言，这需要钱，但是现阶段想做的事情是难以变现的。因此，他们创立了一个名为 Spark 的 Launchpad，希望能够通过帮助项目发射乃至孵化的收入来反哺 Neo Yauto 的发展，目前已有包括日本 IP ACG WORLDS 及其它多个项目与 Spark 进行了合作。这个思路其实和 ACE 挺相似，ACE 也认为 Ordinals 未来的发展破局应该是 To B 而不是 To C 的，他们都想利用自己项目的影响力去帮助想进入比特币生态的实体孵化新项目。现阶段，CXEEO 和他的团队对于赚钱这个事情还是看得比较淡，因为熊市里面强求也没有用，只管做好自己手上的事情。

在三位小图片的项目方外，我还采访到了一位小图片项目方这一身份之外的开发者，他叫大大。

大大（@ordinalsda）在早期就深度参与 BRC-20 的建设，在 BRC-20 刚诞生的时候就与 BRC-20 开发者 domo 保持着交流并且提出过不少建议。大大正在筹划的是一个更便于开发者在比特币生态上落地类似 BRC-20 协议的工具，通过这个工具，开发者可以很轻松地将自己理想中的协议落地，即使是一个从来不了解 BRC-20 协议的开发者也可以很快地上手并且无需花费精力去开发索引等配套设施。

大大有这个想法是因为 BRC-20 至今还没有一个完善的治理体系，很多能够完善 BRC-20 的想法无法通过提案的方式来实现，一切还得看 domo 的想法。与其坐在原地等待 BRC-20 的治理体系完善，更应该做的是为更多好想法的落地进一步铺平道路。如果 BRC-20 一直维持现状，共识不会停留在 BRC-20 上而是会随着更好更完善的技术出现而发生迁移。

结语

Azuki 的事情，让很多原来忠实的 Azuki 持有者发出了「末世论」——对 NFT 的信仰崩塌了。可以理解，毕竟他们受到了「我最深爱的人伤我却是最深」般的伤害。但是，作为一个「纯屌丝」项目方，我还是想站出来说说心里话。

Azuki 的「破灭」，并不是整个 NFT 市场信任的彻底磨灭。实际上，还有很多人在认真做事情，但他们可能甚至都没有机会像我们这些小项目方一样足够幸运被一小部分的玩家看见。在不少群里，我看见许多这样的声音——币圈就是谈钱的，谈个鸡毛情怀。我确定这样的声音相当一部分并不来自 Azuki 的持有者，被一竿子打死，心里多少还是有点难受。

来到币圈，肯定都是想赚钱的，但去到哪个圈子又不想赚钱呢？无非是坚持「站着赚钱」或是抛开底线「收割」而已。通过左脚踩右脚的旁氏掏玩家们的口袋之外，一定有「站着赚钱」的路径。最近引起讨论的 Pudgy Pengunis，已经开始脱下「NFT」的「长衫」，认真做内容并卖着玩具。华语圈内「小幽灵」Weirdo Ghost Gang 也被更多地提起，在通过音乐为自己的「IP」打上记忆点。我们甚至可以说回 Azuki，它的破灭并不是其提出的愿景的破灭，只是它压根没有做好罢了。

如果 Azuki 的凋零是让玩家们对项目方有着更严格的要求，有没有可能这也是一场「良币驱逐劣币」的 NFT 圈大净化的开始？项目方也许从此以后将以更严苛的道德标准被要求，或者所筹集到的资金将被更严格地管理， 比如说被打低初始销售的收入所得、收入所得会被通过合约来控制使用等等。这让我想起许多直播间里会出现的一个梗，「老板们，不要把主播喂得太饱，不然又停播了」。

理想主义不会凋亡，NFT 会越来越好，好到有一天，我的文章中甚至不会再出现「NFT」这个字眼。这一次的结尾，我可以不用再说「我期待着」，因为我也在身体力行地做一些微不足道的小事。想到这里，我仿佛吃了一头牛一般地感到满足。
