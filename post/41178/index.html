原文标题：《Why NFT Creators Are Going cc0》

原文作者：Flashrekt & Scott Duke Kominers，a16z 

原文编译：老雅痞

本文来自微信公众号：老雅痞

每年元旦 1 月 1 日，也是「公共领域日」，成千上万的创造性作品首次自动进入公共领域。这意味着原创作者或版权持有人失去了他们的专属权利（例如，复制、改编或出版），有关作品可以免费供所有人使用。这种情况发生在电影、诗歌、音乐、艺术品、书籍上，这些作品的保护通常持续到作者去世后 70 年，在某些情况下甚至发生在源代码上。

向公共领域开放创意作品也为各种新用途打开了大门。今年早些时候，估计有 40 万张 1923 年以前的录音，以及著名的《小熊维尼》开始公开。(那是连字符形式的小熊维尼，而不是仍由迪斯尼拥有的 1961 年的较新的、穿衬衫的版本）。随着 A.A.Milne 在 1926 年出版的《小熊维尼》一书中的大部分角色现在都已公开，我们开始看到创造性的改编和表达，Milne 可能从未预期或打算这样做。事实上，这只爱吃蜂蜜的熊的旧版本已经被改编为一部恐怖电影 Winnie-the-Pooh: Blood and Honey……维尼和小猪是反派。

相对于许多经典的知识产权（IP）战略，实验和重组有时可以增加知识产权的价值，这是反直觉的。这是开放源码运动的一个核心动力，它明确允许公众在现有技术的基础上建立（或分叉和复制）。安卓、Linux 和其他成功的开源软件项目之所以如此有竞争力，很大一部分原因是它们对这种无许可创新的拥抱。Crypto 在吸引公众开发方面的成功，同样是由于其对开源的普遍认可，以及「合成文化」，这对一些 NFT 社区来说尤其如此。

抓住生产的 meme

通过知识产权建立品牌、社区和内容的策略在 NFT 项目中差异很大。一些项目或多或少地保持着标准的知识产权保护；另一些项目仅赋予 NFT 所有者对相关知识产权进行创新的权利；还有一些项目走得更远，它们选择完全取消版权和其他知识产权保护。

通过「Creative Commons Zero」（」cc0」）许可，由知识共享非营利组织于 2009 年发布的权利放弃工具发布数字作品，创作者可以知情地选择「不保留权利」。这个选项允许任何人制作衍生作品，并从这些努力中获利而不必担心法律后果。目前公众在适用于 NFT 的版权方面仍有很多困惑，因此这里所说的一切都不应被视为法律、财务、税务或投资建议（但请查阅这篇文章，了解 NFT 的版权漏洞，以及创作者如何采取措施确保所有者的权利：

https://medium.com/initc3org/copyright-vulnerabilities-in-nfts-317e02d8ae26）不过，本文的重点只是 cc0。

在 NFT 中使用 cc0 是由 2021 年夏天启动的 Nouns 项目推广的。许多其他项目很快就跟进了，例如：A Common Place, Anonymice, Blitmap, Chain Runners, Cryptoadz, CryptoTeddies, Goblintown, Gradis, Loot, mfers, Mirakai, Shields, and Terrarium Club 都是 cc0 项目，而且有很多很多衍生品都是由它们创造出来的。

同时，流行的化名加密艺术家 X_COPY，在今年一月份将他们标志性的 1-of-1 NFT 艺术品「Right-click and Save As Guy 右键单击并另存为家伙」置于 cc0 许可之下，而这只是在其出售该作品一个月后。这种 cc0 的指定已经导致了大量的衍生品的出现。

X_COPY 的「Right-click Save As Guy 右键另存为家伙」

基于 X_COPY 原创的衍生作品

接下来的一个周一，XCOPY 决定走得更远，他宣布打算「全身心投入」，将 cc0 应用于「自己所有的现有艺术」。这位艺术家说：」我们还没有真正看到一个 cc0 的夏天，但我相信它即将到来……」，他暗指一个可能的增长期，类似于 2020 年的「DeFi 之夏」，当时去中心化的金融吸引了更多的追随者。

为什么这么多的 NFT 创作者走上了「无版权「的道路？

其中一个原因可以简单地说是为了「文化」促进原始项目的扩展，以带来一个更有活力和参与的社区。这在加密货币的背景下特别有意义，在那里，开放共享，寻找和建立社区，是许多人的核心理念的一部分。

创意作品因其文化相关性而生存和死亡。虽然 NFT 可能允许任何数字项目的可证明的所有权，而不考虑许可问题，但 cc0 也通过主动而不是被动地邀请创造衍生作品来启动「meme 能力」。当新的衍生品被创造和分享时，人们的注意力就会流向原作，加强它在集体意识中的地位。这反过来又会激发更多的解释，产生飞轮效应，即每一个额外的衍生品都能增加原作的价值，类似于平台网络效应，即随着更多用户的加入，平台对用户来说变得更有价值。

换句话说，cc0 许可让创作者更容易「抓住制作的 meme」。

「抓住生产季节 1 – 卡片 2 的 meme」

不过，cc0 在整个数字领域的扩散只是一个开始，现实世界的实物产品也在利用 cc0 NFT 资产。在每个新的 NounsDAO NFT 上看到的标志性方框眼镜（标语：」每天一副，永远不变」）已经被 Nouns Vision 项目制作成了真正的戴在脸上的豪华太阳镜。Blitmap 已经看到他们的像素艺术被自由地描绘在鞋子、衣服和帽子上，所有这些都来自不同的实体。这与更传统的知识产权模式形成了鲜明的对比，在这种模式下，单一的所有者通常会控制创作、许可和生产。

Blitmap 标志帽实际上有多个 cc0 级别：实体「blitcap」（第三级）是 cc0 Chain Runners 系列（第二级）中的特征的衍生品，它使用了 cc0 Blitmap（第一级）的「标志「原件，这个标志实际上是 Blitmap 的第 84 号标记，是 Blitmap 集合中被用作其他独立集合中的特征的几个之一。(「Dom Rose」, token #1, 是另一个受欢迎的选择。) 这些敬意暗指 Blitmap 作为 cc0 领导者的影响力，它是最早宣布其公共领域意图的主要 NFT 项目之一。而且这种引用还在继续扩散，例如，上周推出的新系列「Citizens of Tajigen」就包括了 Blitcap 特征的一个版本。

这类衍生品对所有人来说都是双赢的，不仅仅是对原创作者，尤其是对那些利用 NFT 资产建立新品牌的项目。衍生品从源项目中借用了一些品牌知名度；然后，当人们独立意识到衍生品时，就会对原项目产生新的兴趣。如果你在街上看到有人戴着 Nouns 眼镜（或在超级碗广告中），你可能会想买一副自己的眼镜，但你也可能会对购买原始的 NounsDAO NFT 或其他相关衍生产品感兴趣。事实上，这篇文章的第二作者是通过 Chain Runners 第一次发现 Blitmap 的，虽然 Blits 远远超出了他的范围，但他最后还是买了几个「Flipmap「的衍生品。

作为共同创造的开放源码

NFT 的力量部分来自于技术的内在可组合性，因为它们是建立在智能合约技术之上的。许多智能合约被明确设计为积木，可以相互组合或堆叠，以创造更丰富的应用。

术语「money Legos」也是这样创造的，用来描述去中心化金融（」DeFi」）智能合约的组合，相互连接形成新的金融用例。(例如，收益聚合器 Yearn 与 MakerDAO 的稳定币$DAI 和交易平台流动性供应商 Curve 等互动，只需调用他们的智能合约上的公共功能。）从同样的可组合性角度来看，NFT 及其底层智能合约可以作为文化和创意可以重新组合和相互连接的基础层。

而 cc0 允许所有这一切在原创作者的明确许可下发生——从而给 NFT 的爱好者社区一个字面授权，让他们随时随地、随心所欲地建立新的价值层。

来自 HyperLoot（Loot Project 衍生产品）的游戏概念，将多个 cc0 项目显示为虚构游戏中的可玩角色

可以与更广泛的开放源码相提并论，特别是与 Linux 的崛起相提并论。当互联网还很新的时候，微软以其封闭的操作系统 Windows 控制了大部分的操作系统市场。但 Linux（及其创造者 Linus Torvalds）倡导社区第一的精神，开放源代码供所有人使用、修改，并无限制地分发。这（除其他外）导致世界各地的开发者合作并为 Linux 创造新的软件——从网络服务器到数据库以及两者之间的一切。随着人们（和公司）不断创造世界一流的开源软件，Linux 的价值主张得到加强，最终推动了整个行业的爆炸性增长和进一步创新。根据市场分析机构 Truelist 的数据显示，今天 Linux 在前 100 万台网络服务器中占 96.3% 以上的市场份额，在智能手机中占 85% 市场份额。

随着 cc0 许可开始以类似的方式授权 NFT 社区建设者，人们可能也希望这里有一个长期的创新轨迹。以 punk4156 提供的逻辑乐高为基础，NounsDAO 的匿名联合创始人：将 cc0 与 NFT 相结合「将对抗性游戏转变为合作性游戏」。这在几个层面上很重要：首先，从开源到加密的去中心化系统关乎陌生人之间的信任和协调，因此提供合作机会至关重要。其次，这种合作的动力在 NFT 的背景下特别有效，因为赋予人们对其数字资产的所有权，使他们能够通过其资产和贡献所产生的价值将共同创造的结果内化——进而激励他们首先要参与共同创造。

创造的许可

如果 cc0 项目类似于个人的开源「应用程序「或「平台」，那么 NFT 艺术品、元宇宙和智能合约提供了「用户界面」，而底层区块链（例如以太坊）是「操作系统」。但是，为了使这些应用达到类似于 Linux 的潜力，需要创建更多的支持性基础设施服务，并随时提供，以便人们能够最大限度地利用 cc0 创造的合成机会。

这些服务已经开始成形。例如，」超结构「Zora 协议和 OpenSea 的开源 Seaport 协议使开放的、无许可的市场能够被建立起来，用于交易 NFT。最近，一个像素画渲染引擎被完全在链上发布到以太坊区块链上，它已经被整合到 OKPC 和 ICE64 等项目中。每一个连续的应用都会促进「开箱即用「的区块链能力，推动新的应用从这些现在更加丰富、增强的构建块中建立起来。

虽然 web3 开发者的增长处于历史高位并迅速扩大，但总量仍是全球活跃软件开发者总数的一小部分。但是，随着越来越多的开发者进入这一领域，有抱负的 NFT 项目可能会找到更多的创意和基础设施乐高积木，为 cc0 项目和其他项目提供基础。

可组合性是增长的关键。由于这些数字资产建立在可互操作的基础设施之上的公共标准，用户很容易将他们的资产插入各种不同的平台。Loot 项目是这种可扩展性在实践中的一个例子，它是最早展示 NFT 中去中心化的共同创造、世界建设等方面的演变的项目之一。我们分享这个例子也是因为它在美学上明显地低调甚至「不完整」，为社区想象力共同创造留下了更多空间。

就背景而言，Loot 以一系列战利品包 NFT 开始，每个 NFT 只包括一个简单的黑底白字的八种「冒险物品「的清单（如 5726 号战利品包的「Katana, Divine Robe, Great Helm, Wool Sash, Divine Slippers, Chain Gloves, Amulet, Golden Ring」）。这些战利品包由最初的创造者 Dom Hofmann 免费发布，作为社区建设的一个起点。

有几个项目确实已经开始在短时间内充实了从隐喻的世界建设（传说）到物理的世界建设（游戏开发）的所有内容，来自各方的创作者为集体的「Lootverse「贡献了许多衍生品。他们已经制作了游戏（Realms & The Crypt）；角色（Genesis Project, Hyperloot, and Loot Explorers）；讲故事的项目（Banners and OpenQuill）；甚至还有关卡基础设施（The Rift）。

cc0 和可组合性在这里如何应用？因为用户拥有并控制基础性的战利品包（一种在许多不同的游戏和讲故事环境中都有意义的原始物），他们可以在任何地方使用这些核心资产，只需连接他们的加密货币钱包。这使他们能够参与众多的衍生项目，包括像 Genesis Adventurers 这样的项目，其特殊角色在许多其他项目中都有特色，它基本上实现了不属于任何一个实体的去中心化的特许经营。

何时采用 cc0

如前所述，NFT 项目在开发和建立其 IP 时可以采取许多策略。当涉及到 cc0 时，重要的是要现实。仅仅申请许可证并不能神奇地将任何项目变成轰动效应，你不能指望公有领域能突然使某些东西成为受人喜爱的、失控的成功。就像开放源码软件一样，cc0 对那些为丰富的、扩展的生态系统创造潜力的 NFT 项目来说效果最好。

迄今为止，许多最成功的 cc0 项目都是通过引入可在一系列不同背景下灵活使用的知识产权而成功的。Nouns 品牌对于啤酒广告和实体眼镜一样直观；战利品包是基本的基元，对各种冒险环境都有意义；Goblintown 的艺术风格在矮人、僵尸和暴躁的猫头鹰身上看起来和在 Val Kilmer 身上一样好。

我们相信，理想的 cc0 NFT 项目为建设者创造了增值的机会，包括：

纵向上，通过在原始 cc0 资产上直接堆叠新的内容和功能（例如，在 Loot 生态系统上建立的游戏等），以及

横向上，通过引入不同但相关的知识产权，帮助传播原始 cc0 项目的品牌（如各种 Goblintown 衍生品等）。

围绕 cc0 NFT 项目的商业模式可以直接从这些活动中获益。因为 cc0 NFT 项目通常从二次销售中获得持续的版税，第三方扩展和衍生品可以通过推动对原始 cc0 资产的需求增加而成为收入来源。

此外，使用 cc0 授权可以减少摩擦，否则可能会阻止品牌强化扩展的产生，或者更糟糕的是，导致它们以绕过原版的方式产生。正如 Robbie Broome 最近解释的那样（在他的 cc0 项目 A Common Place 的背景下）：」通过将我的知识产权交给 cc0 而不是』保护』它，它可以避免下一步的糟糕重演。例如，如果 UrbanOutfitters 想把我的设计放在 T 恤上，与其在他们的团队中雇人设计一些看起来像它的东西，他们就可以直接使用实际的作品」。有时，采用 cc0 可以有效地将竞争变成合作。

此外，cc0 项目可以从社区对核心资产的价值和贡献的认同中大大受益。社区的凝聚力和参与在这里是至关重要的。在上面已经提到的例子的基础上。虽然开发者原则上可以围绕他们想要的任何主题和物品概念创造冒险游戏，但许多人选择围绕 Loot bag 开发的事实反映了 Lootverse 中的社区凝聚力。同时，Blitmap 衍生项目 Flipmap 与原 Blitmap 艺术家分享了他们的部分收入，以承认该项目在社区中的核心地位——此举可以促进 cc0 项目生态系统中的健康文化。正如 cc0 项目评论员 NiftyPins 所指出的：」这是一个明智之举，向为他们的宇宙建立基础的人致敬。它还培养了一个环境，许多 OG Blitmap 的艺术家都会突然出现在 Flipmap 的对话中，提供信息和对话。」

但是，cc0 并不是一个放之四海而皆准的解决方案，例如，围绕已经建立起来的品牌建立的 NFT，可能更愿意选择限制性更强的许可，作为保护其现有知识产权和可能加强排他性的方式。此外，虽然 cc0 与允许 NFT 所有者专门将与其持有的 NFT 相关的知识产权商业化的策略有一些表面上的相似之处，如 Bored Ape Yacht Club，但有一个关键的区别：cc0 持有者没有权利排除其他人使用相同的知识产权。这可能使持有者更难在 cc0 资产的基础上建立商业品牌，或授予合作伙伴特定的权利，当然，持有者仍然可以引入他们完全控制的扩展知识产权（如背景故事或衍生品）。

去中心化和开放开发是区块链技术和更广泛的加密精神的核心要素。这使得加密货币项目非常自然地围绕 cc0 内容模型进行建设，该模型建立在知识共享基金会和几个开源先锋的工作之上，并且可能代表了迄今为止该开源理念的最纯粹的正式体现。

与开源软件项目的发起人一样，选择 cc0 的 NFT 创作者必须决定他们在组建周围的生态系统中发挥多大的作用。一些 cc0 项目的领导者，如 Chain Runners 的创建者，持续在最初的 cc0 资产上进行建设，积极建立一个环境，使衍生项目可以插入并建立在其上。相比之下，Dom Hofmann 从 Loot 中退了出来，让社区来负责。(也就是说，Dom 正在开发其他的 cc0 NFT 项目，作为他成立的开发 Blitmap 等公司的一部分）。其他创作者则完全选择了退出，比如最近化名为 sartoshi 的人宣布退出他开发的 cc0 项目 mfers，并完全退出了 NFT 市场，他发布了一个最终版本，恰当地命名为「sartoshi 的结束」，然后删除了他的 Twitter 账户。mfers 项目的智能合约现在由七个 mfer 社区成员的多签名钱包控制。

无论原创作者的持续参与程度如何，cc0 授权可以使一个强大的社区以为所有成员提供价值的方式进行共同创造。随着 NFT 市场的不断发展和成熟，我们希望看到更多有组织的基础设施和设计模式支持这些努力。围绕价值获取的框架也可能会有创新，就像开放源码软件一样。(例如，我们可能会设想一个「Sleepycat 许可证「的版本，它要求专有软件产品在嵌入某些开源组件时支付许可费）。随着创作者持续推进这个市场，我们希望看到他们开发和试验新的权利和许可模式，远远超过今天的使用。但无论如何，cc0 为 NFT 创作者提供了一种启动项目的方式，这些项目有一天可能会有自己的生命。
