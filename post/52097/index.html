RockTree OG Talk 系列圆桌会谈节目由石木资本与星球日报共同举办。每期 OG Talk，石木资本都会聚集 Web 3.0 领域的先锋人物，为观众带来全球最有深度的 OG 观点及各赛道顶尖企业的最新资讯。

5 月 24 日 20 点，由 YouTube Influencer CH Egan 主持的本期圆桌会谈以《香港 Web 3.0 的监管与机遇》为主题展开讨论，四位嘉宾分别是：石木资本董事长兼真格基金国际合伙人 Omer Ozden 博士、HashKey Group 资深法律顾问 Jonathan Gil、L&Y 律所首席合伙人 Henry Yu 以及 Gate.io 全球营销负责人 Asher Wu。

Omer Ozden 认为，香港作为天然的金融中心，拥有得天独厚的优势去推进 Web 3.0 技术的发展，但同时也需要内地的开发者和创业者参与。香港作为试点地区来推进监管制度的发展，世界各地的大型资本会将香港视为一个具有活力的司法辖区，这不仅能够振兴香港经济，也会推进整个亚太地区的经济发展。未来在香港将会有很多创新，企业家与资金将共同助力创新，这对于石木资本来说也是绝佳的机会。

Jonathan Gill 认为，香港在 Web 3.0 监管方面具备竞争优势。监管的确定性对于吸引投资非常重要，而投资进一步推动产业创新。香港作为有着完善法律系统的国际金融中心，有绝佳的机会能够在本轮加密寒冬中突出重围。

Henry Yu 认为，香港很有潜力成为 Web 3.0 之都，但监管的思维模式也十分重要。加密货币作为创新产业，商业模式在不断变化，每个机构的运作模式不同。如果香港证监会采用传统金融领域的监管模式来看待加密行业，将形成新的监管风险。

Asher Wu 认为，香港 Web 3.0 监管的指导方针能够指导加密企业应该在哪些方面进行创新，包括交易类型、项目类型等。Web 3.0 行业在香港受到监管意味着加密行业被政府和大众所认可，加密行业的价值被承认。香港 Web 3.0 监管将有助于提高加密货币在全球范围的大规模采用。

以下为圆桌对话实录，由Odaily星球日报精简整理。

CH Egan：首先介绍一下主办方石木资本，石木资本是一家国际投资基金及商人银行，专注于北美与大中华地区之间的开拓性前沿交易。石木资本团队专注于早期投资，为领先的Web3和移动电子商务项目提供全球生态系统支持。其网络拥有全球基金、交易所、媒体和Web3行业合作伙伴，使其成为全世界最高质量的早期Web3投资基金，助力顶级创始人及其团队实现加速的全球增长。

Omer Ozden（欧阳默）作为石木资本的董事长，也是真格基金的国际合伙人。他所在的团队发明了 VIE 结构，促使中国互联网产业向外资开放。他曾参与过很多公司的早期投资和上市，包括百度、阿里巴巴、新东方、网易和 Facebook。Omer 因其独到的见解，被称为 “十角兽猎人”，专注于市值超过 100 亿美元的公司，在全球加密货币投资行业名声显赫。他成功参与了如今最知名 Web 3.0 公司的早期投资，包括 ChainLink、The Graph、dYdX、REN、Fantom、Maple、Pocket Network 等公司。在 Omer 的领导下，石木资本成为Web3浪潮下的行业开创者之一，连接了西方与东方加密货币市场的顶尖公司与业务。石木团队成员分布于北京、上海、南京、新加坡、纽约及多伦多，涵盖了加密货币本地化金融、资本市场、数字营销及法律合规领域的专家，提供区块链项目的全球解决方案，优质项目可以连接进入到石木资本的国际生态系统。

很荣幸今天能与四位嘉宾进行交流。首先请各位嘉宾进行自我介绍。

Omer Ozden： 大家好，我是 Omer Ozden, 石木资本的创始人。非常期待今天与各位嘉宾交流，我们彼此都熟识很多年，共同经历过牛熊。

Jonathan Gill： 大家好，我是 Jonathan，负责为 HashKey 构建实用型和证券型代币，同时也为在香港的虚拟资产监管提供建议。HashKey 是一个亚洲的端对端数字金融服务集团。我们是香港仅有的两家虚拟资产交易牌照持有者之一，获准提供证券交易及自动化交易服务。

Henry Yu： 大家好，我是 Henry Yu，很高兴能在这里分享一些有关香港监管环境的观点。作为一名 20 多年的执业律师，我此前专注于投资银行业务，并于 7 年前成立律师事务所 L&Y，目前专注于加密货币领域，为交易所、基金和项目方提供法律咨询。

Asher Wu： 大家好，我是 Asher Wu，来自老牌交易所 Gate.io。Gate.io 成立于 2013 年，目前已成为全球最大的交易所之一，提供多种数字资产交易服务，用户超过 1000 万。Gate.io 一直位列加密交易所行业前十。除数字资产交易服务外，Gate.io 还提供如去中心化金融、研究、分析、风险投资实验室等服务。

CH Egan：近期，香港迎来了新的叙事、新的风口，开始欢迎 Web 3.0、接纳 DLT，跨链基础设施、拥抱元宇宙、对接 NFT、牵手稳定币及央行数字货币，给这个行业的创业者带了无限的机会。那么香港对加密货币的政策可以从哪些方面解读？对于机构及散户投资者分别又有哪些机会与挑战？我们应该如何正确看待“Web 3.0 香港”这个新世界？

Henry Yu： 在过去两三年中，有很多人尤其是中国从业者们都迁往新加坡或者其他司法管辖区。大部分人离开香港是因为加密货币在国内被完全禁止，所以他们认为香港也会在将来某个时点采取同样的政策。但现在香港却采取了不同的立场。在过去的几个月，所有人（甚至包括一些政府官员）以及中国的加密货币公司都非常看好 Web 3.0 的发展。接下来两年，香港会作为试点，如果一切顺利，那么也许能够有机会慢慢打开这个市场。目前，已有成千上万的从业者迁移到了香港。

Omer Ozden： 香港作为天然的金融中心，有完善的基础设施、充足的人才、优良的传统来开展金融活动，而 Web 3.0 的第一个主要应用就是在金融领域。在香港，去中心化金融、中心化金融和其他领域都正在扩张。我认为把此类业务转到香港，使它成为一个门户，对香港来说是水到渠成的事。香港历来缺少“创始人型”创业者，同时也需要内地的开发者和创业者，这两者如果能在未来有机结合将助力 Web 3.0 技术发展壮大。

Jonathan Gill： 我认为香港最近为 Web 3.0 提供了更明确的政策信息。香港现有的方法论是“同样的业务、同样的风险、同样的规则”，允许从业者借鉴传统证券法规经验。就机会而言，我专注于代币化，对促进高价值非流动性资产的流动性也充满热情，如房地产、艺术品或葡萄酒。说到挑战，我认为当我们在衡量资产价值时，很重要的一点是不要被那些关于加密货币短期波动的劣质消息分散注意力。我认为把资产代币化不会降低其价值的可预测性，也不会使其受到更多变化的影响，如果有什么影响的话，可能也是抬高了它的价格。

Asher Wu： 我认为出台有关加密货币的政策以及把香港打造成一个有利于加密货币发展的金融中心，对于世界经济整体以及加密货币环境来说都大有益处。我们需要逐渐完善政策并慢慢渗透给公众，才能让加密行业成为主流。公平、多元环境能够给予加密货币社区一个表达自我的渠道，也能让更多人加入加密领域，并让外界能够更多的接受加密货币和 Web 3.0 ，使其得到更广泛的应用。此外，这还将吸引到更多人才与新的流动性。最重要的是，这也提升了加密社区与普通大众之间的信任度。我认为这对目前在这个行业工作的每个人来说都是件好事，监管确定性的确非常重要，这将让整个加密领域快速发展。

CH Egan：Omer，您曾参与百度、网易和阿里巴巴等首批互联网公司的风险投资和首次公开募股，经历中国数字产业发展的各个阶段。那么您对香港最近的发展有什么判断?

Omer Ozden： 香港一直作为一个门户，让中国能够拥有不同类型的基础设施和进入全球市场的机会，现在又一次作为门户在积极推进 Web 3.0 技术。我认为中国的长期趋势是开放的，这次对香港的放开是大趋势的一部分，这也是石木资本依然在中国发展的原因。

CH Egan：HashKey 集团现已成为世界上第一家在香港和日本分别获得运营虚拟资产交易平台牌照的公司，并且根据新加坡金融管理局(MAS)的《支付服务法》获得在新加坡运营的豁免资格。作为香港仅有的两家证监会第 7 类持牌虚拟资产公司之一，您能告诉我们香港证监会多年来是如何监管虚拟资产的，以及他们的指引和总体框架与新加坡和美国等其他“金融中心”有何不同吗?  

Jonathan Gill： 我会分别解读一下这三个司法管辖区。首先是香港，香港有几个关键的监管机构：香港证监会，监管证券和期货的机构；香港金融管理局，确保货币的稳定，从更广的范围来说，确保银行和金融系统的稳定。至少从 2018 年开始，香港证监会就一直在努力了解虚拟资产行业，投资者及像 HashKey 一样受监管的实体能够从中得到更多确定性和舒适性。目前政府正在修改反洗钱条例，这将把加密货币纳入香港证监会的监管范围内。

而新加坡的监管则更加直接，受新加坡金融管理局全权监管。其规章制度更关注反洗钱和技术风险，逐渐增加对消费者的保护，而香港证监会的信条就是要提供最好的保护。我认为新加坡和香港最大的不同之处在于，新加坡金融管理局确实限制了数字资产或数字支付代币服务面向大众的营销。

接下来说下美国，由于美国是联邦国家，因此根据虚拟资产的法律分类，可能会同时受到州和联邦法律的约束，可能会有汇款许可证、证券法、商品法（都有各自的监管者和法规）。另一个区别是，美国采取了更为强硬以及更加具有干预性的方法来监管虚拟资产，他们会采取一些备受瞩目的法律措施，比如监管质押并将加密货币归类为证券。

CH Egan： HashKey 作为资产代币化的大力支持者，您认为监管以何种方式释放了资产代币化的潜力， 香港是否提供足够有利的环境，让实体资产代币化在企业层面得以采用，并充分发挥其潜力?

Jonathan Gill： 我认为监管会给资产发行者和持有者带来一种确定性和舒适性。如果代币化能够与传统金融竞争成为筹集主流资本的一种可行方法，那么它的确是需要监管的。从这个意义上来看，这对于持牌机构来说是一种实质性的飞跃，能够有一个明确的监管制度，这在加密货币寒冬期间尤为重要，所以我认为法规会从“负担”变成竞争优势。

关于第二个问题，我认为香港有着很好的监管制度，但是或许可以做更多事情来促进这个行业的发展。最近，香港证监会表示只有专业投资者才能持有证券型代币，因为他们觉得这是更复杂的产品。我认为这是一种相当保守的立场，因为虚拟资产本身并不是复杂的产品，证券也不是复杂的产品。但是当你把证券代币化时，突然间就成为了复杂性产品。这种制度会将散户投资者们排除在外，而散户投资者占据了我们客户群体的很大比例。香港证监会近期放宽了对金融香草产品的限制，但其表示对代币来说，如果没有以金融工具的方式进行法律上的完善，仍然会被认为是一种复杂性产品，前路仍旧道阻且长。

香港没有将加密货币计入专业投资者的财富当中，这对于像我们这样的加密资产交易所来说不是件好事。因为我们有很多潜在客户都是持有大量加密货币，并且达到了专业投资者的门槛，但他们只持有很少的现金。所以这影响了我们潜在客户群的规模。因此，我觉得这两件事情就是香港能够做的，来提高其作为加密货币中心的声誉。希望监管机构能够让其变成现实。

CH Egan：L&Y Law Office 为虚拟资产业务提供哪些类型的服务？香港虚拟资产市场和监管环境近期有哪些值得注意的进展？能否请您详细介绍一下以下方面的最新进展？如新的虚拟资产服务提供者牌照制度，第 1 类，证券交易，第 9 类，资产管理及其受监管活动，还有稳定币的监管？此外，香港如何看待央行数字货币的应用？

Henry Yu： 作为一家律所，我们帮助客户处理法律和监管问题，包括证券、上市、并购、合规、许可，甚至针对证监会的调查进行诉讼仲裁。现在我们已经有了虚拟资产服务提供者牌照制度，正在帮助交易所、项目及一些加密货币基金，使得他们能够遵守日渐清晰的法律法规。实际上我们也在加强与监管机构的联系，尝试帮助推进法律法规的制定与完善。

现在，有了新的虚拟资产服务提供者制度，每个加密货币交易所如果想要在香港市场上运营，就需要取得牌照，这使得所有交易所都处在同一起跑线上。我建议客户从简单的地方开始，比如第 1 类和第 9 类牌照。目前，香港允许一些第 1 类牌照升级为经纪人交易牌照，来处理加密货币虚拟资产，但前提是他们必须通过香港的持牌平台进行交易。对于第 9 类，目前有相当多的加密资产管理公司已经申请了牌照，因此他们能够 100% 地处理加密货币。此外，随着新的虚拟资产服务提供者机制的推行，很多传统资产管理公司正在计划申请牌照来进行加密货币交易。

关于稳定币，香港金融管理局已经宣布如果稳定币以法币为支撑的话，将会监管稳定币。因此，我们期待新的法律法规。实际上香港有一条法律用于发行稳定币，叫做储值设施条例（Stored Value Facilities Ordinance），但是到目前为止，香港金融管理局并没有给任何 SVF 持牌者授予发行稳定币的权利。因此，我们很快会看到一旦新的法律法规实施，监管机构将会授予大家发行稳定币的牌照。此外，我相信肯定会有企业或机构在发行以离岸人民币为支撑的稳定币方面发挥作用。 

CH Egan：Gate.io 最近有哪些最新进展？您如何看待伴随着日渐友好的香港 Web 3.0 环境 ，Gate.io 及全球交易所格局将会发生哪些变革？这对零售业务意味着什么？对于现在拥有更大市场来发展社区的新兴项目又意味着什么？香港崛起为新的“Web 3.0 之都”，甚至可能从欧洲和北美等其他地区攫取部分业务？

Asher Wu： Gate.io 在过去十年中一直处于行业最前沿，在全球范围内推动区块链和链上技术的发展，同样也成为第一个支持 BRC-20 的加密货币交易平台。关于 Gate 和其他交易所如何在全球范围内更好地拥抱香港友好的 Web 3.0 环境，我认为这对于 ToB 和 ToC 来说都是很好的机会。允许香港开放加密货币，就意味着对整个华人社群开放。随着香港监管政策的放开，将呈现一个全新的公平竞争的环境，对于交易所或即将推出的项目而言，需要关注什么才能真正吸引中文用户。我们可以清晰地看到加密中心正在向亚太地区转移，就香港如何成为 Web 3.0 之都而言，吸引的不仅是交易所也是项目，这也将提高市场的流动性。

香港的开放和香港政策的转变，将成为一座新的桥梁，促进加密货币的运用发展和渗透到全球大环境当中。而发行稳定币将作为所有人努力的一个方向，也树立对香港及内地加密行业的信心。政府需要采取措施来推动香港承担起亚洲加密货币行业复苏的角色与使命。

目前，包括 Gate.io 在内的所有加密企业也都在争取虚拟资产服务提供者牌照，同时 Gate.io 也一直在法律合规文件和许可方面做准备工作。

CH Egan：香港在成为 Web 3.0  中心的过程中面临挑战。一是针对传统资产设计的现有监管框架，可能不适用于快速发展的数字资产和科技。仅仅是获得牌照的高难度门槛，已让其成为更有利于成熟机构的赛场。但创新往往来自不可预知的“草根”阶层。那么面对监管与创新这样的冲突，大家有什么看法？香港应该如何平衡这两者之间的冲突？香港是否有机会推动加密货币产业在亚洲复苏？

Omer Ozden： 香港被选中来推进监管制度的发展，来自世界各地的大型资本以及外国公司都将香港视为一个充满活力的司法辖区，这不仅能够振兴香港经济，也将推进整个亚太地区的经济发展。未来在香港将会有很多创新，投资机构与资金将共同助力创新，这对于石木资本来说也是一个绝佳的机会。

Jonathan Gill： 我认为法规并不是一种负担，相反是一种竞争优势。监管的确定性对于吸引投资非常重要。香港作为有着完善法律系统的国际金融中心有绝佳的机会能够在这个加密寒冬中突出重围。我认为监管框架是合适的，但是其应用方式可以稍作调整。

Asher Wu： 香港 Web 3.0 监管的指导方针能够指导加密企业应该在哪些方面进行创新，包括交易类型、项目类型、区块链技术等方面。Web 3.0 行业在香港受到监管意味着加密行业被政府和大众所认可，加密行业的价值被承认。我认为香港 Web 3.0 监管将有助于提高加密货币在全球的大规模应用，以及向世界展示加密社区能够为世界带来什么。

Henry Yu： 香港很有潜力成为 Web 3.0 之都，但是最重要的是监管的思维模式。香港证监会香港证监会作为监管机构在传统金融领域有着很高的声誉。而加密货币作为创新产业，商业模式在不断变化，每个机构运作模式不同。如果香港证监会采用传统金融领域的监管模式，面对同样的风险，采用同样的方法，可能形成新的监管风险。

CH Egan：美国政府对加密货币的监管越来越严苛，这些举措使得美国的加密货币行业面临着巨大的不确定性和压力，也阻碍了许多合规和优质的加密货币企业和项目的发展。一些美国的加密货币公司不得不迁移到其他更友好的国家或地区，或者放弃部分业务或客户。美国监管政策会不会倒逼加密行业的“重心”快速转移到香港，加速香港成为全球的 Web 3.0 中心枢纽地位？如何看待全球格局的发展？这对像石木资本这样的国际投资基金意味着什么？

Omer Ozden： 加密货币的一个伟大之处在于它与金融密切相关，因此，很多加密领域的组织资金充足，它们可以进行反击，美国体制的一个伟大之处在于某些方面的分权。即使是联邦政府想要“进攻”加密货币，但是法院系统不是中心化的，这样就形成一个相对公平的环境。美国应该学习英国和欧盟的模式，欧盟出台了新的监管制度——《欧盟加密资产市场法规》。美国对于创新的扼杀，已经使得一些投资者和项目走向了海外。如果美国在加密领域闭门造车，关闭一年到 18 个月的时间，优秀的从业者都将会选择离开。
