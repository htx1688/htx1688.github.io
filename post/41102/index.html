去中心化科学 (DeSci) 是一项旨在建立公共基础设施的运动，用于使用 Web3 工具公平和平等地资助、创造、审查、信用、存储和传播科学知识。

DeSci 旨在创建一个生态系统，激励科学家公开分享他们的研究及工作成果，同时允许任何人轻松访问并为研究做出贡献。DeSci 的理念是科学知识应该为每个人所接受，科学研究的过程应该是透明的。DeSci 正在创建一个更加去中心化和分布式的科学研究模型，使其更能抵抗中央当局的审查和控制。DeSci 希望通过分散资金、科学工具和沟通渠道，创造一个新的和非常规的想法可以蓬勃发展的环境。

去中心化科学允许更多样化的资金来源（从 DAO、二次方融资到众筹等）、更易于访问的数据和方法，提供可重复性激励措施等。

DeSci 如何改进科学

科学中关键问题的不完整清单以及去中心化科学如何解决这些问题

资金分配：

DeSci：资金分配由公众使用二次方资助或 DAO 等机制确定。

CeSci：小型、封闭、集中的团体控制着资金的分配。

合作：

DeSci：在充满活力的团队中与来自世界各地的同行合作。

CeSci：资助组织和官僚机构限制合作。

决策机制：

DeSci：资金决定是在线和透明的。探索新的筹资机制。

CeSci：资金决策的周转时间长且透明度有限。很少有筹资机制存在。

共享研究资源：

DeSci：使用 Web3 原语共享实验室服务变得更容易、更透明。

CeSci：共享实验室资源通常缓慢且不透明。

访问权限：

DeSci：可以开发新的发布模型，使用 Web3 原语来实现信任、透明和普遍访问。

CeSci：通常通过效率低下、有偏见和剥削性的既定途径发布。

同行评审：

DeSci：可以通过同行评审工作获得代币和声誉。

CeSci：同行评审工作是无偿的，营利性出版商受益。

IP：

DeSci：拥有生成的知识产权 (IP)，并根据透明条款进行分发。

CeSci：所在的机构拥有研究者的 IP，且对 IP 的访问不是透明的。

出版：

DeSci：通过将所有步骤都上链来共享所有研究，包括不成功的数据。

CeSci：发表偏倚意味着研究人员更倾向于分享取得成功结果的实验。

以太坊和 DeSci

去中心化的科学系统将需要强大的安全性、最低的货币和交易成本以及丰富的应用程序开发生态系统。以太坊提供了构建去中心化科学堆栈所需的一切。

DeSci 用例

DeSci 正在构建科学工具集，以将 Web2 学术界带入数字世界。以下是 Web3 可以为科学界提供的用例示例。

出版

众所周知，科学出版是有问题的，因为它由出版社管理，依靠科学家、审稿人和编辑的免费劳动力来生成论文，然后收取高昂的出版费。公众通常通过税收间接支付作品和出版费用，如果不再次向出版商支付费用，他们通常无法访问相同的作品。发表个人科学论文的总费用通常是五位数（美元），破坏了科学知识作为公共产品的整个概念，同时为一小部分出版商带来了巨额利润。

免费和开放访问平台以预印本服务器的形式存在，例如 ArXiv。然而，这些平台缺乏质量控制和反女巫机制，并且通常不跟踪文章级别的指标，这意味着它们通常仅用于在提交给传统出版商之前宣传作品。SciHub 还允许免费访问已发表的论文，但不是合法的，而且只有在出版商已经收取费用并将作品纳入严格的版权立法之后。这为具有嵌入式合法性机制和激励模型的可访问科学论文和数据留下了关键空白。构建这样一个系统的工具存在于 Web3 中。

再现性和可复制性

可重复性和可复现性是高质量科学发现的基础。同一团队使用相同的方法可以连续多次获得可重复的结果。使用相同的实验设置的不同组可以实现可复制的结果。新的 Web3 原生工具可以确保再现性和可复制性是发现的基础。我们可以将质量科学融入学术界的技术结构中。Web3 提供了为每个分析组件创建证明的能力：原始数据、计算引擎和应用程序结果。共识系统的美妙之处在于，当创建一个可信网络来维护这些组件时，每个网络参与者都可以负责复制计算并验证每个结果。

资金

当前资助科学的标准模式是个人或科学家团体向资助机构提出书面申请。一小部分受信任的个人对申请进行评分，然后对候选人进行面试并将资金分配给一小部分申请人。除了造成有时会导致申请和获得资助之间等待数年的瓶颈之外，这种模式极易受到审查小组的偏见、自身利益和政治的影响。

研究表明，拨款审查小组在选择高质量提案方面做得很差，因为提交给不同小组的相同提案会产生截然不同的结果。随着资金变得越来越稀缺，它已经集中到一个更小的、更高级的研究人员池中，这些研究人员倾向于在智力上更保守的项目。这种影响创造了一个竞争激烈的融资环境，巩固了不正当的激励措施并扼杀了创新。

Web3 有可能通过广泛试验由 DAO 和 Web3 开发的不同激励模型来破坏这种破碎的融资模式。可追溯性公共物品资助、二次方资助、DAO 治理和代币化激励结构是一些可以彻底改变科学资金的 Web3 工具。

知识产权所有权和发展

知识产权 (IP) 是传统科学中的一个大问题：从被困在大学或未在生物技术中使用，到众所周知的难以估价。然而，数字资产（如科学数据或文章）的所有权是 Web3 通过NFT做得非常好的事情。

就像 NFT 可以将未来交易的收入回馈给原始创建者一样，您可以建立透明的价值归属链来奖励研究人员、管理机构（如 DAO），甚至是收集数据的主体。

IP-NFT 还可以作为正在进行的研究实验的去中心化数据存储库的关键，并插入 NFT 和 DeFi 金融化（从碎片化到贷款池和价值评估）。它还允许像 VitaDAO 这样的 DAO 等原生链上实体直接在链上进行研究。不可转让的“灵魂绑定”代币的出现也可能在 DeSci 中发挥重要作用，它允许个人证明他们的经验和与其以太坊地址相关的证书。

数据存储、访问和架构

使用 Web3 模式可以使科学数据更容易访问，分布式存储将使研究能够在灾难性事件中幸存下来。

起点必须是任何持有适当可验证凭证的去中心化身份都可以访问的系统。这允许受信任方安全地复制敏感数据，从而实现冗余和审查阻力、结果再现，甚至多方协作和向数据集添加新数据的能力。计算到数据等机密计算方法为原始数据复制提供了替代访问机制，为最敏感的数据创建了可信的研究环境。

灵活的 Web3 数据解决方案支持上述场景，并为真正的开放科学奠定了基础，研究人员可以在没有访问权限或费用的情况下创建公共物品。IPFS、Arweave 和 Filecoin 等 Web3 公共数据解决方案针对去中心化进行了优化。例如，dClimate 提供对气候和天气数据的普遍访问，包括来自气象站和预测气候模型的数据。
