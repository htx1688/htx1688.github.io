关于 BAYC，作为一个超过两年的持有者，我有许多想法和观点：

许多人可能并未见证其早期，也有许多人可能不记得当猿猴们在 NFT Twitter 上如野火般燎原的情形。那时的时间线上充满了无尽的 #apefollowape（猿猴持有者互关）。一个人一旦购买了一只猿猴，立即就会增加 1000 名 Twitter 的粉丝，甚至因为关注太多人而被「推特监狱」围困。

由于全商业权 IP 授权（总是想提醒 BAYC 并非首创，据我回忆 Avastars 是最早的），艺术家们用艺术衍生品充斥着时间线。人们在各种商业点子上变得越来越有创意：啤酒，霓虹灯标牌，商品，咖啡，酒吧，汉堡，故事，漫画。

社区正在蓬勃发展。老实说，我买猿类的原因是因为「害怕错过」(FOMO)。我在外面往里看，看着他们玩得很开心，我想成为这一切的一部分。

不过，让我们退后一步，回看 BAYC 为何会如此受欢迎。他们不是第一个 10k 项目，也不是第一个 ape 项目，他们在很多事情上都不是第一个（尽管他们确实有许多的首创）。那么他们是什么，他们创造了什么如此特别的东西，瓶子里的闪电是如何发生的？

在我看来是文化。更具体地说，是反文化。

从局外人到局内人，猴子带来的归属感

我相信我们中的大多数人都觉得自己在所有的「酷」项目中都被价格挤出去了。我们中的许多人，尤其是 2021 年的新手们，感觉自己来得太晚了。Punks 是为精英们准备的；什么样的「普通人」能够负担得起在头像上花费超过 15 个 ETH？我个人真的非常非常想要一个 Hashmask，但那时候，甚至这些也超出了我的价位范围。然后 Meebits 上线，我（或者说我们）再次感到被忽视，被遗忘，被价格挤出，像一个局外人在外面观望。

那时还有一个趋势，就是假冒的绑定曲线铸币价格，也就是我们说的「恐慌斜坡」，「FOMO 斜坡」。我认为我的第一个主题实际上是关于这个主题的。 基本上，铸造的人越多，铸造价格就会上涨。在最快的时间内发生令人难以置信的庞氏骗局：在前 1k 中铸币，你能以 0.02 eth 的价格得到，下一个 1k 将以 0.04 的价格得到，依此类推，直到那些「迟到」的人不得不支付 0.2 或者更多。与此同时，NFT 的关注度一直在飙升。我们进入了对加密货币的狂热，而 NFT 是每个人都在谈论的话题。Beeple 6900 万美元的拍卖成为头条新闻。越来越多的人涌入这个领域，寻找可买的东西，希望成为这一切的一部分。每个人都在想，如果它不贵就好了。

这时候，Yuga 出现了；他们决定以固定的 0.08 铸币价格发布 10k 的系列（对于那个时候，这被认为是非常公平和合理的，相比其他的选择）。

虽然在 Mint 的过程中，它的表现并不算好，花了一个星期才铸完，而且大部分的 Mint 都在最后一天。不过我们的游戏比赛也正式开始了。数以千计的人在 Twitter 上使用他们的 PFP 玩得开心。感觉我们同属一个圈子，我们拥有了前所未有的归属感。

事情失控地疯狂转动，2021 年夏天的疯狂狂热让每个人都大吃一惊，而猿猴们有了所有的冲击力。

唯一不变的只有变化本身

基于 PFP 的 NFT 社区的有趣之处在于，社区总是在变化。在某个时刻，最初几周和几个月的社区不再是原来的社区了。当然，BAYC 从一开始就拥有许多持有者，也有许多杰出的社区成员支持社区，也有许多新成员也做出了巨大贡献。

但是它改变了。变化并不总是坏事，而且总是不可避免的，所以这并不奇怪。不过，变化的规模才是最疯狂的部分。 Apes 从反主流文化、每个人都能负担得起的 PFP，变成了 NFT 领域历史上最昂贵的 PFP。突然间，买这些东西的不再是普通的「张三」和「李四」，而是名人和鲸鱼。在这一点上，我想再强调一遍，这不一定是坏事。社区只是发生了变化。

尽管经历了这一切，Yuga 还是提供了一些出色的体验，总的来说社区一直很强大。但当你变得像猿一样大时，你就会成为目标，并处于显微镜下。你变得很容易被人们讨厌，因为现在所有新的反文化项目都在反对你的文化。

猿类从街头乞丐走向了精英阶层；从泥泞沼泽走到了月球之上。

因此，在经历了一年可怕的熊市之后，人们的情绪发生了变化，开始对猿猴有些反感。但我并不感到特别惊讶，毕竟这样的情况已经有一段时间了。 它不再是一个由 4 位 degen 创始人（可能）兴奋并提出滑稽想法、在 Discord 里闲逛并与社区产生共鸣的项目。

它变成了一家在 10 位数估值上筹集了 9 位数资金的公司，正在左右应对各种诉讼，试图创造一个全新的游戏类型（MetaRPG）。 它有一个 DAO、一个 ERC-20 代币、无穷无尽的 NFT、复杂性、深度、混乱、传说、混乱。

毫无疑问，他们是十年后最有可能繁荣的社区之一。99.99% 的 NFT 项目都快没钱了，社区也死了。99.99% 的 NFT 社区无人谈论； Yuga 让每个人都在谈论它们，即使并不总是有利的，但 BAYC 总是出现在 NFT 的每次谈话中。

BAYC 也会被与一些「较新」风格的 PFP 项目进行比较：DeGods、Pudgies、Azukis。所有这些都是令人惊叹的社区，我认为它们也都会成功和繁荣。但它们是不同的野兽，仅仅因为它们都以动物 jpeg 作为核心产品而将它们进行比较并不总是公平的。 在我脑海中还有一件事：我经常想起早期的日子，和我在 Twitter 上、在 Discord 上花费大量时间待在一起的那些猿猴们。

他们中的许多人仍然在社区中，但我们中的许多人已经开始了自己的事业。当拥有自己的社区时，很难继续作为 BAYC 活跃的社区成员。 对于我来说，BAYC 以及它为我、我的生活以及我在 web3 的职业生涯所做的一切，我将永远在我心中占据最特别的位置。我知道很多人也有同样的感觉。 从某个角度上说我是幸运的，我并没有真正被价格波动所困扰——当然，当数字下降时，永远不会觉得舒服，但我并不认为这是世界的末日。

我们在 NFT 文化中有一种倾向，那就是当价格高的时候，我们很爱这个项目，当价格低的时候，我们对项目的喜欢程度就降低了。这太疯狂了，但这就是我们大部分人的想法。我认为是时候承认自从早期以来，事情发生了多大的变化了。

BAYC 不再是两年前的 BAYC，而这是可以接受的。它有战斗的疤痕，新的成员，以及生态系统中的一百万个新事物。

BAYC 是不是更加分散了？

我不知道从这里开始，价格会发生什么——它可能会继续下跌，可能会稳定，也可能会回升。然而，我非常确定的一件事是，如果它要恢复到 2021 年底/2022 年初的疯狂估值，那么这很可能不会依赖于 BAYC 所交付的产品。 没有人愿意花 50 万美元来拥有一个化身、获得一些土地、参加虚拟宇宙游戏或去寻宝。

人们花大价钱是因为他们想成为专属数字乡村俱乐部的一部分。与名人和鲸鱼在同一个俱乐部，成员们互相照顾、互相帮助、建立联系、举办精彩的活动，最重要的是大家一起玩得很开心。 这就是为什么我们看到 Azuki 和 DeGods 最近蓬勃发展：因为他们的社区在一起度过了一段愉快的时光。当然 BAYC 也很有趣，别误会我的意思。世界各地都有 BAYC 群落蓬勃发展，但现在和以前不一样了，我无法不提出疑问：BAYC 是不是更加分散了？

这个疯狂的 PFP 游戏我们所有人都在玩的（并不那么）肮脏的小秘密是，这其中实际上几乎没有什么实质性的价值。绝大多数都是基于感知的投机。我一年前说过这些，今天依然如此：我们市场的一个问题是其中有多少是由纯粹的炒作和投机推动的。

大多数 NFT 的「价值」很简单就是「其他人愿意为它们付出多少」，而人们决定他们愿意付多少主要基于情绪，通常没有任何实质性的东西。当决定是否购买 NFT（为了经济利益）时，重要的不是我对这个项目的看法，而是我认为其他人对这个项目的看法。当每个人都在决定是否购买 NFT 主要基于他们认为其他人的看法时，这很快就会变得有问题。

我认为所有这些也很好地展示了 PFP / NFT 与几乎没有外部依赖性（即 Punks）和有很多外部依赖性的 NFT 之间的区别。有一条路线图是一把双刃剑，而这把剑最终会刺穿大多数项目。

无论如何。我已经絮絮叨叨的说了足够多，我只是想分享一些我作为长期持有者对 BAYC 的看法。不可否认的是，这一年以来我是一个相当糟糕的持有者和社区成员，只专注于自己的项目。但我并没有对 BAYC 正在建设的东西，以及社区可能实现的东西，失去信心。

以及写在最后的是，所有关于「如果 Yuga 失败了，整个 NFT 领域就会失败」的言论都是无稽之谈。如果 Yuga 失败，这个行业肯定会很糟糕，并给一些无知的人提供弹药来试图摧毁我们，但这对 NFT 技术的长期采用几乎不会有任何改变。NFT 并不是"要么成功，要么失败"。它们是必然的，也是不可避免的。

在未来的数字世界乃至整个世界中，NFT 都会起着基础性的作用。问题不在于我们是否会看到这种情况，而在于何时会看到。
