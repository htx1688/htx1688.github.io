作者 | Loopy

Web3⼿机市场是⼀个加密用户熟悉又陌生的领域。一方面，Web3和我们息息相关，但另一方面，一款专门的Web3手机对我们而言又远未普及。

早在 2022 年，著名的奢侈手机品牌 Vertu 就已推出了第一代Web3手机 METAVERTU。有趣的是，这个“圈外”品牌却成为了全球最早一批的Web3手机，早于多个“圈内”Native 品牌。

在 Token 2049 的展会上，Vertu 带来了自己的二代 METAVERTU 产品。“土豪”或许是人们对这个小众品牌最熟悉的标签了。但 Vertu 远不止于此。

对手机用户来说，METAVERTU 拥有 METASPACE、Keybox 等安全功能。而 Vertu 在Web3领域的开发，并不止手机这么简单。除手机内置功能外，还有更多面相开发者、加密生态的深入探索，如 MPC 钱包、I-DID、AI，甚至还可充当 ZK 验证和预言机的节点。

近日，Odaily星球日报有幸采访到了 Vertu 亚太区 CEO Molly。带领大家走近这个小众又神秘的品牌。

以下为采访实录，内容有所删改。

**Odaily星球日报：我对 Vertu 的品牌有一定了解，它的产品定位可以说是独一无二的。能否简短的介绍一下 Vertu 的起源？
**

Molly： Vertu 品牌源于诺基亚的高端产品线， 1998 年成立于英国。

当时团队的初衷是，“为什么手机品类不可以有奢侈品？”所以在诺基亚如日中天的时候打造了这个高端品牌。之所以设立在英国，是因为英国有这方面的文化有传统。比如英伦文化、奢侈品、手工艺等等。

Odaily星球日报：Vertu 的Web3手机是什么意思？手机里有哪些Web3功能？

Molly： 很多加密用户为了钱包安全，会去购买硬件钱包。硬件钱包的原理是通过 SE 安全芯片对其进行加密。而 Vertu 从硬件级别设计了安全方案，我们的安全芯片在传统行业是用在银行 IC 卡这种层级的。包括红外、声波等物理手段都是无法破解这种安全芯片的。

Vertu 的手机可以理解成硬件钱包。手机拥有两个系统，在使用指纹或者密码解锁之后，可以进入到第二个私有的系统——METASPACE。用户在这里可以使用Web3应用。METASPACE 里的 App 受到安全芯片的保护，提供数据销毁和端到端加密功能。

此外还拥有 Keybox 应用。Keybox 与 Android 系统隔离，可以用来存放私钥，这里的数据受到 SE 安全芯片的保护，任何人都无法破解。

Odaily星球日报：除此之外，还有其他的加密功能吗？

Molly： 还有其他一系列的加密应用，例如浏览器等等。但这方面大家的差异性不大。

Vertu 有一个有意思的“一键拍”功能，拍照片之后可以快速生成 NFT。在展会上V神刚刚体验了这个照片变 NFT 的功能。

TOKEN 2049 的 Vertu 展台

Odaily星球日报：除了安全方面的设计，Vertu 在加密领域还有哪些布局？

Molly： 用户可以在安全环境里使用自己习惯的钱包 App，Vertu 也自己开发了一款 MPC 钱包。

Vertu 还开发名为 I-DID 的 DID 产品。目前Web3已经拥有很多 DID 方案了，但 I-DID 比较不一样，采用手机的 IMEI 作为身份识别的依据。

每个手机都有一个全球唯一的 IMEI 号码，而且 IMEI 不可更改。我们以此来作为 DID 的身份依据。Web3世界广泛存在着工作室和羊毛党，这样可以有效防止机器人的行为。

对于开发者来说，只要使用了 I-DID，就意味着它一定不是一个机器人。我们基于 I-DID 设计了一个 VIP 平台。开发者可以以此获取高质量真实用户。

我们的二代Web3手机还可以做节点。

在硬件层面，我们的芯片支持的不只是安全功能。Vertu 手机还可以进行 ZK 验证的计算，它的运算效率优化的很好。

你可以把 Vertu 手机理解为一个节点，未来我们会去和一些有需求的项目方进行合作。让用户可以把 Vertu 手机变成硬件 ZK 的节点。

Vertu 手机也可以充当预言机。Vertu 推出了 TOP Oracle Protocol，确保数据流中的数据真实性和可信度，包括对 L1 和 L2 解决方案的 API 支持。我们希望可以把基础设施做好，链接用户和开发者。

Odaily星球日报：所以 Vertu 手机其实是一个节点？

Molly： 是的。Vertu 的硬件可以充当超轻节点的角色。我们希望可以创造一个生态。Vertu 的系统也向项目方开放了 API，一些功能是可以通过智能合约直接调用的。

Web3项目方缺乏流量，而我们希望项目方可以通过我们获取到高净值客户的流量，而不只是纯粹的羊毛党。

未来和项目方的合作，本质上还是要服务好我们的用户。将我们的流量开放出来，手机的 C 端用户可以获得更多的服务，其实这就是我们想要的。

Odaily星球日报：Vertu 最知名的可能就是管家服务了，Web3手机有继承这个功能吗？

Molly： 当然。这个服务我们早在 2002 年就已经开始了。为每个用户配一个真人管家，比如安排就医、照顾病人等等。

而且，管家还可以帮你做到更多：比如，如果想去一个不接受预定的网红餐厅，你需要排队很久才可以吃到。但我们的管家可以帮你搞定位置。

在Web3手机上，Vertu 还开发了一款基于 AI 的智能助理应用。助手结合了大语言模型和手机端的小模型。通过一系列自然工具和框架，使用户能够系统地构建和开发他们的个人数据，同时有效地利用大型模型的功能。这让用户可训练独属于自己的 AI，它可以成为你的“第二大脑”。

“第二大脑”会记录你的喜好、替你做一些事情。例如，你可以让它帮它总结你的微信聊天，提醒你都有哪些事情需要跟进。AI 若想越智能、所需要的个人数据越多。但这个 AI 的数据掌握在用户自己手里，因此是安全的。

Vertu 并不是 Native 的加密品牌。如何说服加密用户使用 Vertu 的服务？它会取代其他的钱包提供方吗？

Vertu 并不希望做一个钱包 App 去取代其他的钱包。Vertu 是硬件、是基础设施的提供方。

Vertu 更多是基于硬件为切入点去思考加密世界，希望通过硬件级的安全防护为用户提供服务。用户完全可以使用自己习惯的主流钱包，我们提供安全保护。

Odaily星球日报：Vertu 为什么要进军Web3领域？作为一个普通的旁观者，我感觉这二者的结合听上去颇为有趣。

Molly： 是的。其实很多人对 Vertu 存在认知偏见。我们最贵的手机售价数百万元，因为 Vertu 高昂的售价，所以人们只关注到了 Vertu 的奢侈属性。但其实从诞生第一天开始，Vertu 就是一个非常注重创新的科技品牌。

Vertu 做过非常多不一样的尝试，因为品牌太小众了，所以很多人并不了解。

在若干年以前，Vertu 就在冰岛建立了一个巨大的数据库存放用户数据，兼顾环保和用户的数据安全。

最近，苹果发布会刚刚结束，你有关注到新 iPhone 的亮点吗？它使用了钛合金材料。最早钛被用于火星车上，Vertu 一直在关注新工艺和新材料，在很多年前就已经在 Vertu 手机上使用了钛合金。

再比如蓝宝石，因为成本高昂，最早只有手表可以使用一小块。Vertu 是第一个使用蓝宝石作为屏幕材料的手机。若想做到屏幕这么大的蓝宝石，成本非常高。到现在蓝宝石屏幕仍然没有普及。

Vertu 一直在尝试新事物，虽然品牌很小众，但我们始终在不断尝试。Vertu 拥有很强的创新基因，包括公司的很多高层也都是非常坚定的加密的拥护者。我们相信——加密是可以改变世界的，所以 Vertu 一定要做Web3手机，这对我们非常重要。
