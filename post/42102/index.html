你是否准备正在准备进入 Web3 的世界？你是否在找寻新的市场战略想法？

在这篇文章中，你将知道为什么在 Web3 里做市场营销，你需要一个全新的思维路径，你也将学习相关的案例巩固你的认知。

01 为什么大家应该考虑进入 Web3？

为什么大家应该开始采用 Web3 的概念和思路？这个问题的答案需要从多个维度来进行思考：

区块链是一个客户关系管理系统（CRM）

区块链不是一个普通的客户关系管理系统。它是现存的最强的客户管理系统和客户再营销系统。毕竟每个人的购买行为都会记录并且公开在了区块链上。

Facebook，Instagram 和 Twitter 是让你根据用户的身份来预测他们可能的购买行为，而区块链正好相反—你不用知道客户究竟是谁有着怎样的身份，你只知道最重要的一点—他们实际的购买行为。

商家如何从中获益呢？比如你的企业发行出售了 NFT，一些人购买了它并且将其存放在了自己的钱包中。当你发行下一个系列的 NFT 并想进行空投奖励时，你可以率先只奖励那些已经持有你发行的 NFT 的钱包。

NFTs 可以开辟一个无侵蚀效应的新产品线

从 2009 年开始，我们看到金融产品开始从法币（Web2 世界的货币）逐渐向加密货币（Web3 世界的货币）转移。现在，我们看到一个新的资产类别正在从 Web2 朝着 Web3 迈进—一个建立在权利与 IP 而非金钱之上的资产类别。一个企业可以通过将 IP 铸造成 NFT 并在 Web3 世界中进行售卖而从这个资产里获益。

通过开发并且售卖 NFT，企业可以创造一个新的带来正向收益流的产品线。这个新的产品不仅不会影响到现有产品线的收益，而且还会呈现出以下优势：

  • NFT 不需要供应链，从一开始，它就是一个全球性的业务

  • 不需要再投入新的成本建造实体产品

  • 产品形态没有任何限制，毕竟 NFT 不用遵循重力规则

这里有一个将 NFT 同贴纸进行类比的例子。想象一下你生活在一个贴纸还没有被发明的世界。你的手中握有迪士尼公主的 IP。当贴纸技术发明后后，你可以迅速开辟一个贴纸生产线用来生产销售迪士尼公主形象贴纸，这个产品线并不会其他周边产品带来负面影响。

02 如果我的客户不懂 Web3、加密货币、NFT，我该怎么办？

想想互联网。直到今天，也只有非常少的人知道互联网到底是怎么运作的，但是我们所有人依然还在使用它。我们所有人都知道如何打开开关开灯，但是我们绝大多数人并不知道如何制作一个灯泡。同理，人们最终会开始使用 Web3 技术和区块链，哪怕他们并不知道区块链本身究竟是如何运行的。现在的关键是要优化用户体验，利用合理精心的 UX 设计，逐步将去中心化的体验在每一步都满足用户的需求，从而让用户能够无痛进入 Web3 世界的怀抱。

Mojioto（一个用来帮忙推出和迭代 NFT 交易平台的 API 平台和看板产品）通过半托管型钱包的设计，让非 Web3 用户可以更容易地参与到 NFT 世界中。

Mojito 是这么做的：

当用户从你的平台买了一个 NFT 后

  • 如果用户已经有了自己的钱包，那么这个 NFT 会被转移到他们自己的加密货币钱包

  • 相反，如果用户是个没有钱包的 Web3 新人，Mojito 会在用户下订单的同时为他们开通一个钱包。这个钱包是一个半托管钱包，用户并不需要自己处理私钥等问题。

当用户的 Web3 知识逐渐增加，希望对自己的资产有更多的控制权后，他们可以逐渐将钱包设置转移成非托管型钱包。

03 在 Web3 里做营销，和在 Web2 有什么不同？

现阶段，商业模式正在发生着范式转移。在 Web2 和传统的商业模型中，有三方参与其中：制作产品的公司，消费产品的消费者，以及从产品销售中获利的投资人。在 Web 里，这三者经济上融为一体，形成了一个新的身份—社区。

如果一个 Web3 项目设计准确，社区里的每一个人都应该有动力聚集在一起研发迭代项目、进行市场推广、扩增社区规模。事实上，绝大多数 NFT 项目的 Discord 社区都是由社区的“热心群众”而非最初的概念发起者来管理的。

如果这些概念听起来很熟悉的话，那是因为这些事情已经开始在 Web2 里发生了。人们在开源社区里合作共建，通过推荐营销项目向他人推广产品，一些品牌的忠实客户还会在社交网站上自发组成社群维护品牌和产品的声誉。Web3 只不过加速了这一进程，并让这些事情可以同时发生。

社区奖励是项目成功的关键

当人们发现、使用、参与、宣传一个新的 Web3 项目时，他们期待会得到一些奖励。回溯贡献值、分配 POA 代币，并设置一些只有代币持有者才能解锁的额外福利会是不错的方法。

1. 回溯贡献值

回溯贡献值是一个绝妙的机制。它奖励了那些从最开始就相信你的项目的人。

以太坊名称服务（ENS）和域名服务类似，售卖.eth 域名，比如 amanda.eth, lisa.eth 等。ENS 根据每个用户对产品的使用情况，给其空投了相应价值的代币。

2. POA 代币和代币持有福利

Lyrical Lemonade 是一个已经在 YouTube 上有一定粉丝的乐队。他们为参加过线下活动的乐迷发放了免费领取的 NFT。这些 NFT 不仅能让乐迷获得了未来几年相关活动的入场券，也允许他们可以解锁其他的一些附加福利获得额外价值。这就让乐迷可以因为他们对乐队的支持、对活动的支持获得奖励。

NBA 球队密尔沃基雄鹿队也为其球队会员发放了 NFT。这些 NFT 允许球迷可以进入 VIP 座位并且获得球队周边的额外折扣。

从孤立营销到自我营销系统

自我营销系统类似于 Web2 里的大使或者推荐营销项目。不同的是，在 Web3 里，这些大使不再触不可及，他们成为了社区的一部分。

以太坊进入市场后，它并没有一个中心化的市场部门来处理所有的信息。其通过支持身处不同地方的组织者，让世界各地的社区成员都能在同样的图标的召集下，举办一个个充满凝聚力的分部聚会，同步讨论着生态里的项目名册。

通过项目社区组织网络的搭建，社区对于以太坊品牌的信任得以建立，管理权也回归到了社区本身。加上社区获得了布道所需要的一切支持，社区自发组织起的草根布道所达成的效果，远远超出了一个团队自行进行市场营销所能达到的水准。

记住，想要在 Web3 里成功，你必须让社区参与进来。

同时服务 Web2 和 Web3 群体

如果在 Web2 里你的品牌已经有了一定的知名度甚至是铁杆粉丝，当你开始拥抱 Web3 并售卖 NFT 时，你的项目很有可能会同时吸引到 Web2 和 Web3 的人群。你需要用两个群体都能感到舒服的平台和方式将两个群体进行连接并进行交流。

对于 Web3 原住民来说，他们习惯于使用 Discord 和 Twitter。Twitter 是他们获取基础信息的地方，而 Discord 是他们深度参与项目讨论的平台。你需要合理安排你的内容，让用户得以在合适的地点找到他们需要的信息。

而你的 Web2 用户可能会更习惯使用 Instagram 或者 Facebook 来获得一手信息。如果你在尝试将用户从这些平台（甚至是你的邮件列表）迁移到 Discord，你需要提前做好准备，让他们知道 Discord 能为他们带来什么。你的 Discord 设置会很大程度上影响 Web2 用户对待 Discord 的态度。很多旨在吸引更多女性进入 Web3 领域的 NFT 社区在这点上做得不错。他们的 Discord 分区非常合理，新成员可以清晰地知道社区规则是什么，他们在每一个讨论区里都可以说些什么，为什么需要有不同的聊天分区等。

和 Web3 博主合作

很多真正的 Web3 社区都认为向社区或平台索取费用的博主是不真实的宣传。这里的重点是，你需要找到一个真正热爱你的产品的发言人。这的确可以做到。例如，阿迪达斯和 Prada 就跟 G Money 以及 BAYC 达成了很良好的合作。但如果你只是找一个博主，付给他钱，让他聊聊你的 Web3 项目，这就跟 Web3 营销的核心理念背道而驰了。

如果你对和 Web3 原生的博主和品牌合作感兴趣，这里有一些播客可能会帮助你找到合适的人:

  • Cami Russo 的 The Defiant

  • Matt Medved 和 Alejandro Navia 的 NFT Now

  • David Hoffman 的 Bankless

  • Carly Riley 的 Overpriced JPEGs

当你在 Twitter 上越多关注正确的人，推荐算法就会更准确地给你推荐相关的内容和博主。你还可以加入一些 NFT 社群和一些 discord 社区，去认识圈子里的朋友。

当你发现一个合适的合作对象，一个个地联系他们。给他们在 Twitter 上发私信：“这个项目你感兴趣吗？”如果他们愿意成为项目和社群的一部分，再问他们有没有可能公开讨论项目为其站台。

04 苏富比是如何进入 Web3 的

Mojito 旨在帮助像苏富比这样的公司在它们自己的网站而非第三方网站上出售 NFT。苏富比，作为全世界最早的拍卖行，成立于 1744 年。它曾经和第三方 NFT 平台合作，但他们现在希望能够在自己的网站上出售 NFT—就跟他们如何出售苏富比手袋和红酒一样。

如果你谷歌苏富比元宇宙，你可以看到一个由 Mojito 支持的网站，苏富比可以在上面售卖 NFT，并得到诸如税务、客户再营销等其他支持。

苏富比的尝试得到了不俗的反响。仅仅几个月，苏富比在平台上就完成了逾 1 亿美元的营业额。苏富比的名字也出现在了 Forbes Blockchain 50 和时代杂志最具影响力的 100 家企业名单中。

我们也看到，在 Decentraland 元宇宙里，苏富比也有一个和其纽约拍卖行相似的“建筑”。人们可以进入这个虚拟拍卖行并参与拍卖活动。事实上，最近的几次藏品售卖，在元宇宙里的参与人数已经超过了纽约实地拍卖行的参与人数。这也充分证明了，元宇宙已经成为了一个重要的商业中心。
