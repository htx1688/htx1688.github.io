作者：小農 | littlefarmer.eth；来源作者推特：@0xfarmer_

时间转瞬即逝，在台湾结束了半年的替代役，顺利退伍了，好一阵子没关注圈子裡发生的事，所以想说总结与反思一下这两年入圈的成长。

大概只有鲜少朋友知道，我透过 ARB 的空投，噜了接近100万美金的收益，在这个本该找工作的节骨眼，让我能放慢一下脚步，思考人生下一个阶段该怎麽走。

出身在一个父母亲都是公务员的传统家庭，很感谢 crypto 让我有机会看到井底外的世界之大与体会自身的渺小，选对了赛道、拚了命的努力，最后加上幸运女神的眷顾，其实感觉我这辈子应该是要平庸的度过，但现在一切却看起来都不平庸。

21年刚入圈也是跟大多数人一样，搏杀土狗，买乐透币拚百倍爆击，但却赚少赔多，直到听到了 dydx、ENS的暴富神话，才意识到原来空投的产业利润是如此可观，同时也意识到我在圈内没有任何认识的人，所以开始在推特上分享乾货，当时推特上币圈本身受众少，你得非常努力输出，绞尽脑汁的写，才有可能累积到粉丝。

当时噜毛的人还很少的，不像现在烂大街的教学，我开始捲起袖子噜， $psp 成为我第一个赚钱的空投，那时有一个号可就开心地不得了，还写了篇小资族空投教学，近千次的收藏跟点讚给我满满地成就感，也是那时开始累积了我的影响力，让大家开始注意到我。

刚好银河 ( @Galxe ) 这种新兴的链上凭证平台崛起，带动了新一波噜毛潮，而肝 mod 风气当时非常盛 lol，我于是毛遂自荐地向@0xbobchien (Bob，银河当时的 BD) 提出我想做 Mod 的想法， 也天天在 dc 群裡回答问题，很幸运地在茫茫数十万社区人海中，成为了第一个社区出身的 Mod ，也就是当时大家都亲切叫的 “大黄”。

开始过著每天从中午起床就是在DC回答问题到凌晨的生活，可能因为从社区出身，我特别地懂用户的心情，当时一有活动，几乎回不完的私讯，每天各式各样的问题，能解决地先解决，不能解决地看要找谁，看要找前后端的、资料处理的、智能合约的，还是项目合作的同事。

相比当时听过 Mod 给白操粉、抑或自我感觉过于良好的 ，没啥工作经验的我，已几乎很少差评地度过了每次的社区危机直到大半年后空投发币，最后几位一起努力的 Galaxy Brains 包括我在内都领了数万美金的空投，这次的工作经历对于我非常宝贵，除了近距离的学习一个web3团队如何运筹帷幄，也认识到超级牛逼的银河老闆们、同事们，还有在台湾线下聚会有一面之缘的伙伴。

离开银河后，也刚好是我大学毕业，在家等兵单，这段接近一年的日子，我开始全职噜毛，六月份我收到 @KuiGas

丰密老师的邀请，加入了 @33daoweb3，命运的齿轮开始转动，22年也是币圈的至暗时刻，LUNA 归零、三箭破产、FTX 倒闭，但是跟著群内的大神们一起学习，我对于投研、交互、有了全新的理解。

之后就是各种花式噜毛，时间根本不够用，就暂时不怎麽更新推特了，所有的精力都花在噜毛上，我形容给你看当时的生活，现在想想真不是人过的，我也不太会写程式，就拉了张 excel 表，日复一复从起床点滑鼠点点点到睡觉，你们当时噜 NFT 白单赚钱了、我每天都在亏 gas、衝土狗迷因币赚钱了，我还在亏， 也不知道怎麽熬过来的，当时就一个信念，我这样坚持下去，一定有一个项目会给我赚到钱吧 ?

印象很深，当时行情很差，有次 33DAO 的週会，@hebi555 何老师给我们打了个香醇浓的鸡血，就那句话:【只要噜不死的项目，往死裡噜就对了】，也就这麽一句话，我听进去了，正巧 OP 当时发币，想说同样对标二层，我就选了 ARB 当目标，往死裡噜，没啥甚麽诀窍，资金量、交互频率、次数，把500套ARB号女巫隔离做到完美。

还在等ARB奥德赛来，兵单先来了，在进入成功岭的两周新生训练，一天晚上接到了大学同学的电话，ARB发币了，我心裡慌得要死，但因为非常信任他，请他帮我领，隔天晚上匆忙领了手机马上打给我朋友，我只印象当下听到数字人都傻了，我知道我成功了，那些噜到手腕发炎，半夜痛醒，一天吃三颗止痛药继续上号的日子，一切都值得了。

后续送了我朋友一辆档车、外加一大包的红包，倒也不是感谢他不 rug 我，而是经过长时间的相处，我知道他不是这种人，还有我的父母以及一路帮助我的群友 (有位科学家额外帮我跑了一些odos号，数十万美金收益、一分都没少)，也算是幸运过了头，都遇到人品非常好，非常善良的人。

一直到几个月后，把所有事情处理完，才轮到我自己，买了辆车犒赏自己，这也大概是我唯一的大额花费在自己身上，因为我知道赚钱容易守财难，赚了钱后更要保持谦逊，我把我的成功，有部分得归功于运气，而我不保证这些运气，是否能持续保持，肆意挥霍，到头来可能会失去所有。

上述大概就是简单叙述一下我入圈的经历，可以当作故事看就好，我相信很多人都有精彩的故事，但我觉得更重要的事，在币圈我学习到了很多学校绝对不会教的事情，整理几点，大概是我最受用的部分也是给小白的一些建议:

1. 经营自媒体是最有效的人脉槓杆 : 如果你本身像我一样，没有人脉没有背景，推荐你开始经营，这是让陌生人认识你最快的方式，一开始肯定辛苦，但是坚持下去，它给你带来的收穫，远远超过金钱所能衡量，更重要是能认识在现实社交圈中很难认识的那些优秀的人。

2. 尽可能地与一个行业最优秀的人为伍，吸收大佬成功的经验 : 成功的经验无法複制，但可以学习，我觉得我自己非常幸运，在入圈的时候就能够认识很多大神，他们给了我向上的动力与目标，让我不会自满，圈子决定了你的认知、你的认知最后影响了你赚钱的能力，我可以很明确地说，我就是个井底之蛙，一直到现在，越往上爬，才越发现自己是多麽的笨与不足，有机会遇见大佬，哪怕他的一句话或是给你的一个机会，都可能是给你抛下逆天改命的绳索。

3. 不需要羡慕别人，善于总结自己的进步，也懂得为别人喝采 : 以前我对于别人赚钱，总会有那种酸葡萄心态，但后来发现这样我始终不会快乐，因为你赚的再多，始终有人会比你多，所以注意自己的进步就好，别人成功，替他鼓掌，也看看有没有自己可以学习的地方，一个好的心态才能让你做出正确的投资判断。

4. 市场上1%的财富，是留给那些愿意做99%的人都不愿意做的事情的人 : 那些大佬，很多都是付出常人数倍的努力，即使我知道我就是个普通人，头脑不够聪明，学习也不好，那就花比正常人更多的时间嘛，别顾虑一堆，认知不够就努力提升、知识不够就努力充实、技术不够就努力学，很多人说选择大于努力，确实千真万确，但是没有努力，你哪裡知道要选择甚麽，知道自己哪裡不足就补甚麽，不要让懒跟笨成为你的藉口。

这篇文除了写给大家看，也献给过去那个国中休学重读过、上高中时在生日卡片上被斑导写上对我说她非常失望的字语、还有大学成绩倒数毕业的我，不要让任何人评价你，唯有你才能定义自己。
