1.2 什么是 Web3 营销?

在具有颠覆性技术的市场进行运营和营销（例如 Web3），意味着需要面对许多未知的挑战。创始人需要坚定的焦距其愿景，价值和未来发展规划路线。其次，创始人也需注重的一方面——营销。现在市场上已有少许关于 web3 市场营销的内容和资源，然而鲜少有提供关于如何推广产品的具体步骤。本指南剔除无用信息，为你提供市场营销的方法论和垂直营销领域的清单。本节将简要介绍 Web3 和区块链为何标志着营销范式的转变。

2. 营销概念：安索夫矩阵

营销的终极目的是为公司带来增长，在追逐破圈的 web3 行业中尤其如此。安索夫矩阵则是营销增长最常提起的基础概念。

安索夫矩阵是什么？

安索夫矩阵（英语：Ansoff matrix）是一种分析产品和市场的工具，能有效帮助企业制定增长策略。安索夫矩阵可用于探索不同增长策略的风险和收益。

如何使用安索夫矩阵？

安索夫矩阵有两种实现增长的营销策略:调整产品或调整市场。根据市场和产品本身的条件，增长策略落入四个象限: 市场渗透、产品开发、市场开发或产品差异化。

3.2 5 个搭建 web3 品牌的秘诀

1. 为目标用户建立客户画像

建立用户群体的全方面数据库以备不时之需，同时也有助于确保营销团队秉持统一的品牌声音和形象

2. 鼓励创始团队打造个人品牌

超过 58% 的公司通过其创始团队提高知名度。作为营销人员，在营销产品时利用创始团队的私域流量是低成本且高效的。在资源允许的情况下，为团队雇佣文案稿手，通过团队的社交媒体建立社区。在谈论产品之前，让创始团队成为行业的领军人物，有助于产品的推广和公司形象的曝光。

3. 鼓励社区创作 UGC

甄别社区中最活跃的成员，并通过激励鼓励社区成员创作 UGC 内容 ( 如视频教程、产品指引、推荐评论等 )。

4. 创建并定期更新产品介绍文档

文档（Docsend, Gitbook）是新用户认识产品的第一步。为社区开发人员提供简洁明了的指南有助于他们使用和开发产品，尤其是开源项目。

5. 创建社区专属的交流渠道

创建专属频道或服务器 ( 如 Twitter 社区、Discord 频道 )，向社区成员提供聚合空间、鼓励提问和建立深度关系。更进一步可策划核心贡献者和用户的社区聊天，以探索潜在营销活动和大使计划。

4. 案例分析: Liquid Death

Liquid Death 是什么?

  • 「硬核」饮用水品牌，以规避一次性塑料和督促喝水为宗旨

  • 有明确的品牌和定位，和具有针对性的营销策略。Liquid Death 没有谈论饮用水品牌经常提及的“自然”、“纯净”和“味道”，而是唤起了更黑暗、更前卫的主题——对塑料的谋杀、口渴和死亡。

  • 通过品牌形象，在 web3 找那个获得众多狂热的追随者

  • 成功经营名为「乡村俱乐部」的 Web3 忠诚计划 — 成千上万的粉丝“出卖灵魂”，以提前获得限量版周边、私人活动的权限

WEB3 策略

  • 入驻 web3 电商平台, Novel

  • 在 Novel 平台发布新 NFT 系列 (Murder Head Death Club MHDC)

  • 部分商品只供 NFT 持有者购买

  • 持有者可享受独家折扣，与明星见面的机会，线下 VIP 活动等

  • 通过各种活动建立 Web3 社区，例如持有者可以在 Liquid Death 的电台上 DJ，在 Discord 上放映恐怖电影，举办扑克之夜等

  • NFT 持有者的长期福利计划，例如与 CEO 会面的机会，周边的获取权等

成果

  • MHDC 发布 6666 个独特的 NFT

  • 由著名艺术家 Will Carsola 创作，每个 NFT 是一个死去的干渴者被砍下的头

  • MHDC 系列在发布后的三小时内售罄

  • 首发价为 0.0666 ETH, MHDC 的发行为 Liquid Death 带来约 445 ETH 的销售额

亮点

  • 从 Web2 到 Web3 的转换保持了一致的品牌形象和宗旨

  • 践行公司可持续发展的品牌价值 ( 例如，将 MHDC 收入的 10% 版税捐赠给慈善机构，并抵消 110% 的初始铸币碳排放量 )

  • MHDC 系列秉持 Liquid Death 品牌宗旨和形象

  • 积极举办社区建设活动和线下活动，维护社区凝聚力

5.3 元宇宙中的广告

元宇宙投放广告包含在虚拟世界或在线游戏场景中创建和放置广告。投放形式通常多元化，如虚拟广告牌，虚拟环境中的产品植入，或者作为元宇宙活动，游戏道具的赞助商等方式。

在元宇宙投放广告的主要优势之一是其能够瞄准特定用户群体并且实时跟进交互数据。广告商可以通过交互数据优化投放策略。

广告商需注意虚拟世界的基调，确保所投放的广告应景，不会破坏用户体验。

虚拟现实中的广告仍处于起步阶段，而 VR 技术也仍在发展中。

如果元宇宙行业发展成主流，广告投放可能有巨大的潜力。

5.4 NFT 的终极营销策略

营销 NFT 项目涉及各种策略，提高品牌知名度、使社区产生兴趣和热点，并最终推动 NFT 的销售。值得注意的是，加密货币和 NFT 领域发展迅速，了解最新趋势和成功案例对营销的成功至关重要。

NFT 通证门控策略贴士

NFT 通证门控（Token gating）涉及 NFT 持有者对某些内容、访问或利益的权限。 通证门控通过使用智能合约创建数字“门”来实现，该“门”可以根据特定 NFT 的权限打开或关闭。

NFT 通证门控允许创建者奖励 NFT 持有者独家权限，这有助于提高对 NFT 的需求并增加其内在价值。该策略能为品牌和创作者提供营销和盈利的机会。

5.5 成功的电子邮件营销

1. 建立邮件列表

在内容页面、社交网络简介和电子邮件签名中加入注册或订阅按钮。从网站访问者、社交媒体关注者和其他渠道收集电子邮件地址。

2. 区分订阅用户

根据兴趣、行为和点击率等因素，将邮件列表分组，针对分组创建定制化的内容。

3. 引人入胜的邮件主题

突出主题。邮件主题应当简单明了且风趣幽默。引用问题吸引读者的注意。

4. 引人注目的预览文本

引人注目的邮件内容预览，需简明扼要地传达核心信息

5. 优化用户行动号召

行动号召 (CTA) 有助于吸引用户浏览网页。需考虑按钮的内容、颜色和位置。使用触发动作词，确保文本按钮准确地描述该按钮的功能。

6. 自动化和量化邮件广告

使用营销工具, 例如 MailChimp 和 HubSpot, 将重复性任务自动化。跟踪关键指标，如打开率和点击率。

7. 宣发渠道的集合

在邮件内容中加入其他社交媒体平台账号的链接，有效增加其他宣发渠道的订阅者。

6. 营销概念: 消费者购买决策过程

在不了解潜在用户需求和偏好的情况下，销售类似区块链应用等颠覆性产品或服务可谓寸步难行。“消费者购买决策过程”有助于卖家了解用户行为，包括在挑选产品时面临的障碍和推动因素。此类信息能有效优化营销，提升产品或服务的销售。

如何在 web3 应用该模型？

  1. 引起需求: 消费者意识到可以通过区块链技术解决需求或痛点

  2. 收集信息: 消费者搜索有关区块链产品的信息，这些信息可能会解决消费者所面临的问题或满足他们的需求

  3. 评价方案: 消费者评估搜索过程中发现的不同区块链产品，以确定哪一个最适合他们的需求

  4. 决定购买: 消费者决定购买，选定的区块链产品

  5. 购后行为: 在使用产品后，消费者评估他们对产品的满意度和购买决定

7. 内容营销 7.1 内容营销六部曲

无论目的是教育用户或投资者，还是建立信誉权威，Web3 项目均需某种形式的内容营销。而高质量，可信的内容是与社区建立信任的关键之一。

然而，“内容营销”不仅仅是在在 Medium 上发布文章。「内容营销」需要依照长期策略，进行创作和发布才能为业务创造真正的价值。

以下六步可以帮助你朝着明确的目标构建「内容营销」策略。

7.2 五个内容营销策略框架

盲目的发布文章并不能有效的提升曝光或增加销售额。一个有效的内容营销策略能将高质量内容转化为流量并实现业务目标。内容营销策略焦距于内容的宣发渠道，以及宣发的最终效益。以下五个内容营销策略框架涵盖了最常见的绩效指标。

1. 潜在客户生成

适用对象

适用于需要在消费者购买决策过程的每一环节与潜在客户建立更多触点的大型项目

运作方式

与其直接向客户销售产品，该框架注重于将消费者转化为客户的每个环节

  1. 为销售的每个阶段 ( 意识、兴趣、决定、行动 ) 创作内容

  2. 将付费或自然流量引流至以提高品牌意识为目标（非强制性销售）的内容

  3. 将浏览内容的用户通过兴趣和偏好进行分类和标签

  4. 将发布的系列文章定推荐至分类用户群体，尽量迎合每个用户群体的偏好

例如，针对那些已经访问过以引起兴趣为目的内容的消费者，提供辅助购买决定的内容，如此循环下去。

2. 常青内容

适用对象

适用于想要提高其内容在搜索引擎上的权重和可见度的项目

运作方式

这种低成本的方法侧重于让自然搜索自动完成大部分工作，同时也意味着结果需要时间的沉淀。网站的曝光度和权重会随着时间逐渐增长，意味所发布的内容会慢慢从搜索引擎的第 100 位上升到第 3 位

  • 在博客或者网上发布常青内容。 常青内容本身是搜索引擎优化的，并长期与行业保持高相关性

  • 自然搜索并不能快速提升常青内容的点击率，需要结合域名权重才能达到曝光率的提升。 域名权重的提升可以通过构建链接达成。如需深入了解链接构建策略，请访问工具和资源页面。

  • 随着链接构建的内容增加，当发布更多搜索优化的常青内容时将开始出现指数级的复合增长。 理想情况下，每个内容都应该在搜索排名中拥有更高的权重。

3. 竞争性搜索引擎优化

适用对象

适用于想要提高网站权重和曝光度的项目，发布与目标用户群体搜索热点相关的内容。例，SocialFi 平台应在“SocialFi”搜索中排名第一。

运作方式

该策略与常青策略类同。唯一不同是产出内容主题不是“常青”，而是环绕与产品或业务独特竞争力相关的关键词。

4. 社区建设

适用对象

适用于侧重社区建设而非销售产品或者服务的项目

运作方式

该策略目的在于增强回访率，推动社区成员之间的交流和社区积极性。通过创作适合分享转发的内容，推动读者将内容散播到更广阔的目标市场

5. 集合策略

适用对象

适用于拥有充足资源的公司，该策略能满足几乎所有业务目标，同时涉及多重维度同时运行

运作方式

有两种方法可以实现这一策略。在内容营销投资更多资源，以处理所有不同类型内容的需求；或者通过每篇内容的发布积攒短期效益和长期效益

  1. 建立一个与目标受众关键字密切相关的常青内容库；

  2. 使用链接构建内容来提升常青内容的曝光，从而推动有机流量；

  3. 利用热搜话题，通过社交媒体和传统宣发渠道推动流量和用户粘性，获得短期效益。例如图表或长推。

8. 营销概念: PESO 模型

Paid（ 付费 ）, Earned （ 自发 ）, Shared （ 共享 ）, Owned （ 自有 ） (PESO) 模型是一个将品牌与消费者之间沟通类型进行分类的模型. 现在市场上与消费者沟通的方式多元化，该模型有助于各频道间彼此呼应，形成发挥最大化效果且协调的集合营销策略。

10. 案例分析: Reddit

Reddit 是什么?

  • 流行新闻论坛，拥有活跃的 Web3 社区

  • 在 Polygon 链上发布虚拟人偶 NFT 数字藏品

  • Reddit 虚拟玩偶限量收藏品基于平台的吉祥物，Snoo

  • 系列由 Reddit 创作者和独立艺术家创作

WEB3 策略

  • 在内容营销中避免使用 web3 相关术语，如 NFT 和钱包。根据之前的市场调查，35% 的 Reddit 社区认为品牌会毁掉 NFT 的未来

  • 通过社交媒体平台构造无摩擦的购买体验，买家只需在 Reddit 上创建一个充当数字钱包的“库”

  • 通过剔除买家购买和管理加密货币的要求，从而降低门槛

  • 持有者在 Reddit 的个人资料上可以获得额外福利，例如更换装备和定制配件等

  • 发布一个月后，Reddit 上的顶级贡献者和社区建设者免费获得空投以奖励其贡献，并担任品牌大使推广该系列

成果

超过 300 万个加密钱包因人偶系列 NFT 被创建，其中 99% 是独特账户，超过 OpenSea 的 230 万个活跃钱包。

亮点

  • 避免使用「钱包」等术语，以吸引非加密用户使用产品

  • 针对潜在买家进行市场调查

  • 为社区提供解决痛点的产品，在包装美观的情况下，使用简单清楚的界面引导用户

  • 虽然区块链是产品的技术底层，但并非产品的亮点。在产品前端设计的过程中秉持这点有助于提升新型产品的可接受度
