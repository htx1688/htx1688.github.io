底层-Layer 1 

Worldcoin 推出打击 KYC 服务商买卖虹膜信息的新措施

据Odaily星球日报近日报道，由于某些地区的加密用户无法在 Worldcoin 代币推出时注册获得份额，一些 KYC 服务商正在低收入国家和地区购买一些普通人群的虹膜信息，目前一个虹膜 KYC 已经飙升至数十美元。
Worldcoin 发言人承认了这一现象，但强调该问题仅限于“几百个实例”。发言人称：“通过持续的威胁和意识监测措施，Worldcoin 团队发现了可疑和潜在的欺诈活动，这些活动促使个人注册经过验证的 World ID，然后将其发送到第三方的 World App，而不是他们自己的。”
发言人表示，Worldcoin 已经采取了几个步骤来试图控制这个问题，其中包括“调整初始的亲自注册流程，以及实施动态与静态二维码”；此外，如果用户账号被盗，用户可以通过第二次扫描虹膜来重新获得他们的 World ID 以他们的验证账户，这将有助于阻止非法出售身份证明证书。
发言人补充道：“尽管有这些预防措施，但重要的是要承认，它们并不能完全防止串通或其他绕过一人一证原则的企图。为了应对这些挑战，机制设计和社会关系归属的创新理念将是必要的。”（The Block）

Odaily星球日报讯 BNB Greenfield 测试网将于北京时间 5 月 25 日 14 点至 18 点进行重置升级，并将 Greenfield 版本升级到 v 0.2.0 。新升级版本命名为 Mekong Testnet。
新版本将提供 SP GC 服务、升级到 cosmos-sdk v 0.47.2 以及清理陈旧权限等。

Osmosis、Axelar 等资助开发 Cosmos 生态跨链安全模型“mesh security”

Odaily星球日报讯 由 Osmosis 领导，Axelar、Akash Network、ATOM Accelerator DAO 等多个 Cosmos 项目正在资助新型跨链安全模型“mesh security”的开发。
Mesh security 允许两个或多个 Cosmos 链巩固其验证者质押代币的价值，以抵御安全攻击。在 Mesh security 框架中，特定区块链的验证者可以将其绑定的代币重新质押（re-stake）给自己选择的合作链上的验证者。
Mesh security 计划预计将分三个阶段完成，每个阶段预计持续三个月左右。在 Cosmos 生态系统各个团队的贡献下，开发和资助将继续进行。最终发布预计将在 2024 年第一季度进行。（The Block）

NEAR 选择 Wormhole 作为官方跨链桥，将共同开发 ZK 轻客户端

Odaily星球日报讯 据官方推特，跨链桥 Wormhole 宣布 NEAR Protocol 选择其作为新的官方跨链桥，Wormhole 将使所有的 NEAR 资产能够无缝桥接到 20 多个连接的链，并将数百个 Wormhole 支持的应用程序和社区带入 NEAR 生态系统。
此外，Wormhole 的核心贡献者将加入 Zpoken 和 NEAR 一起开发 ZK 轻客户端，该客户端可证明和验证以太坊和 NEAR 状态，从而使向更加安全的桥接体验。

Evmos：已离任联创试图出售 50 万枚 EVMOS，正与之合作以进行回收

Odaily星球日报讯 Cosmos 生态 EVM 兼容链 Evmos 发推称，团队注意到已离任的联合创始人一直试图在 Osmosis 上出售 50 万枚 EVMOS 代币，该地址持有 3400 万枚 EVMOS 并已质押 1150 万枚 EVMOS。该地址中代币的初始锁定期为一年，并分为四年线性释放。 
Evmos 表示，该名联创对项目有实质性的贡献，离开是因为在一些问题上存在分歧。目前 Evmos 正在与之合作将代币转回 Evmos 基金会。Evmos 基金会和核心贡献团队完全拒绝这种只会损害对项目信任的行为。

硬件钱包 Ledger 推出基于 ID 的密钥恢复服务，遭部分用户反对

Odaily星球日报讯 硬件钱包制造商 Ledger 发布 Ledger Nano X 固件（2.2.1 ）版本更新，该更新引入一项名为“LedgerRecover”的服务/功能。
据介绍，Ledger Recover 是一种基于 ID 的密钥恢复服务，可为用户的私钥恢复短语提供备份。Ledger Recover 目前与 Ledger Nano X 兼容，可在运行最新 Ledger Live 版本的 Android 和 iOS 上使用。目前，需要欧盟、英国、加拿大或美国签发的护照/国民身份证（ID）才能订阅该服务。在接下来的几个月中，该服务将覆盖更多国家并增加对更多文件的支持。
据悉，Ledger Recover 服务将钱包恢复短语（基本上是一种人类可读形式的私钥）分成三个加密碎片，并将它们分发给三个托管人：Ledger、加密货币托管公司 Coincover 和代码托管公司 EscrowTech。如果有人丢失了他们的助记词，三个分片中的两个可以结合起来（等待身份检查），以重新获得对锁定资金的访问权。从本质上讲，Ledger Recover 是一个额外的安全网；以每月 9.99 美元的价格，它消除了加密货币密钥丢失的危险。
然而，Ledger 用户对该功能表示强烈反对，因为它需要在线存储秘密助记词并与护照或身份证相关联。该功能因要求 KYC 注册而面临批评。部分用户称这违背了加密社区中许多人持有的隐私价值观，这破坏了硬件钱包的用途。

Polygon 上 Aave V2中约 1.1 亿美元资产因更新 Bug 无法提款，修复补丁将在投票后部署

Odaily星球日报讯 Aave V2的最新更新使得 Polygon 上 Aave V2中约 1.1 亿美元的 WETH、USDT、WBTC、WMATIC 无法提款，也无法进行借款和偿还操作。
目前 Aave 已提交了修复该问题的补丁，将在投票通过后部署。出现问题的原因为 Aave V2的 Polygon 实现为其利率策略合约使用了一个略有不同的函数定义列表，使得更新后存在 Bug。
需要注意的是：
1. 只有 Aave v2 Polygon 受到影响。所有其他 Aave v2实例（包括v2以太坊和 Avalanche）都正常工作。Aave v3完全没问题，因为它是一个完全不同的协议。
2. 池中的所有资金都是绝对安全的，无论是否为受漏洞影响的资产。在实践中，这些用户无法提供更多的这些资产，也无法借款、偿还或提款。
3. 目前，我们不建议通过 Aave Guardian（唯一可用的机制）暂停池，原因如下：
- 为了提高 HF，用户仍然可以向他们的头寸提供 DAI、USDC 和 AAVE 等资产。使用 pause()，这是不可能实现的。
- HF/平仓阈值动态仍然适用，因此 1 ）头寸借入的最大金额与往常完全相同；2 ）如果头寸在漏洞修复时低于 HF 1 ，则将被平仓，这意味着平仓奖金的损失。
