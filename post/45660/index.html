The Beacon 游戏在当下出现给大家较大的惊喜，即便在 web2 游戏中算不上顶级，但在 web3 游戏里面算是不错的。这也说明了 Web3 游戏还在早期，Web3 游戏会有更多各种探索。

The Beacon 游戏跟之前 GameFi 不同

把好玩放在首位

上个周期的游戏大多是 GameFi，核心是 Fi，而不是 Game，Game 只是手段，Fi 是目的。所以，GameFi 往往很难实现正向的外部性收入，从而陷入旁氏经济的困境。如果团队首先是为自己创建游戏，那么好玩就会到来。

多种游戏模式

The Beacon 是奇幻动作类 RPG 游戏。The Beacon 还融入了 Roguelite 的元素，让它变得更具挑战性。地下城的关卡是程序生成，每次通关都会面临一组独特的挑战，也带来不同的体验。

The Beacon 支持多种游戏模式，包括单人 PvE,多人合作 PvE，社交等。单人 PvE 适合玩家独立通关，在过程中获得 NFT 的奖励。多人 PvE 通过团队作战增加团队合作的体验乐趣。社交方面，则是通过房屋系统、酒馆等来进行社会化交友，可以向其他玩家展示自己的房屋，在酒馆交流等。

游戏挑战性

The Beacon 是基于技能的游戏。如果游戏过于简单，玩家很快探索完了，那就没有什么可持续性。The Beacon 的玩家需要摸索，需要攻略，需要提升技能。有部分武器学习的曲线较低，有部分武器学习的曲线较高，通过满足不同技能水平的用户，从而让玩家获得满足感，也有机会获得不同价值的 NFT。

低进入门槛

The Beacon 是免费玩模式，之前的加密游戏大多是玩赚模式，需要玩家先付费才能玩。这会将早期很多玩家拒之门外，或者通过公会系统才有机会加入。The Beacon 是让任何人都可以玩，即便用户没有创世角色，也可以玩。如果玩家没有付费角色，系统会给予用户一个默认的角色以及默认装饰的房屋。

免费玩家可以体验游戏的基本玩法，进入地下城跟怪物作战通关等，可以获得非 NFT 的角色装饰品和房屋装饰品。与此同时，免费玩家还有机会在未来获得游戏代币和其他的奖励。这让免费用户也有机会通过玩游戏获得收入，从而迈入付费玩家的角色。

付费角色玩家机会更多

付费角色玩家相对于免费玩家来说，付费玩家拥有的创世角色会有初始的角色装饰 NFT、房屋装饰 NFT，以及其他的一些福利。这些 NFT 属于玩家个人所有。

免费玩家在游戏过程中只能获得非 NFT 装饰品，而付费玩家可以获得 NFT 角色装饰品和房屋装饰品。此外，跟免费玩家不同，付费玩家有资格在排行榜上亮相，并获得相关奖励。

角色 NFT 是 The Beacon 游戏中的核心资产。创世角色不会有稀有度，在这一点上所有角色都相同。除了形象在视觉上的差别之外，创世角色拥有的装饰品 NFT 和房屋装饰品 NFT 是具有稀有性的。玩家通过拥有不同的角色装饰 NFT 和房屋装饰 NFT 来展示其独特性，这种与众不同可以显出玩家角色的差异化。

虽然任何玩家都可以参与游戏，但设立创世角色（付费）玩家，并推出一些差异化，可以为游戏的可玩性和经济机制设计带来一定的空间，兼顾了游戏的可玩门槛，也兼顾了游戏的经济机制设计。

具有稀缺性的宠物

在 The Beacon 游戏中，创世宠物一共 4096 个，它们跟主人绑定，是玩家在游戏中的伙伴。创世宠物是第 0 代宠物。宠物具有不同的视觉特征，每个特征的稀有度都不同。

宠物可以养育，可以传递其特性。宠物养殖会有显性和隐性特征。显性特征更有可能传给后代，还会有随机的突变特性。宠物会有繁殖次数的限制。宠物的作用是让持有者获益，比如可以获得更高的 NFT 掉落率等。

上述的设计在不同的加密游戏中也比较常见。这 4096 只创世宠物能否成为游戏亮眼的部分，主要取决于这些宠物跟游戏玩法的融合程度，同时也需要做好经济机制的设计。目前还不清楚具体的情况。

可互操作性

TheBeacon 是 TreasureDAO 生态的游戏之一。随着生态内部不同游戏的发展，The Beacon 可以联合生态内其他游戏创造出新的玩法。当然，具体怎么做，目前还不清楚。（推荐阅读：《一文详解打造现象级游戏The Beacon的TreasureDAO》）

The Beacon 的经济机制部分

目前 The Beacon 游戏还没有推出其经济机制设计，只是简单说明其使用 MAGIC 和 BCN 作为游戏内代币。The Beacon 跟 TreasureDAO 高度集成，TreasureDAO 的生态代币 MAGIC 计划融入到 The Beacon 游戏中。

BCN 则是其游戏的原生代币，MAGIC 和 BCN 在游戏中用于购买不同类型的商品。The Beacon DAO 将 BCN 作为其 DAO 的治理代币，玩家通过在 The Beacon 游戏内的行为来获得 BCN 代币。按照 MAGIC 生态的惯例，BCN 代币会通过 MAGIC 获得流动性。

在 The Beacon 游戏的发展中，除了可玩性之外，其经济机制设计是否合理将决定其可以抵达的高度。具体如何设计，目前还不清楚。

The Beacon 游戏所处的阶段

目前 The Beacon 游戏完成了可玩原型的启动，主要是单人 PvE 模式，在这个时期，参与用户地址超过了 2 万。

The Beacon 接下来会进入宠物生成阶段；明年第一阶段推出 ALPHA 发布，单人 PvE 游戏为主；第二阶段推出多人合作的 PvE。一切都还在演化中。

The Beacon 游戏距离真正成功还有很远距离

The Beacon 游戏并不是说有多惊艳，它在 Web3 领域中引起大量关注，重要原因之一是在 web3 游戏中，真正好玩的没有几个。而 The Beacon 游戏在当下出现给大家较大的惊喜，即便在 web2 游戏中算不上顶级，但在 web3 游戏里面算是不错的。这也说明了 Web3 游戏还在早期，Web3 游戏会有更多各种探索。

The Beacon 游戏距离真正的成功还有很远的距离，只有它获得较大规模的用户群体，比如至少几十万上百万用户，且这些用户确实是为了可玩性而进行消费，不是为了赚钱而投入。在这个时候，才有机会产生正向的游戏收入，才算是进入初步的成功阶段。
