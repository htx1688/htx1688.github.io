在 2023 年蚂蚁集团数字技术开发者大会上，蚂蚁链宣布了以其行业领先的跨链技术为特色的开源项目，以及致力于构建安全、透明和可靠的 Web 3.0 基础架构的新研究中心——蚂蚁链开放实验室

此外，蚂蚁链还与上海数据交易所、浙江文投集团合作，共同建立Web3 OpenChain，一套Web3开发者工具包和服务。

为了推动 Web3 的发展，蚂蚁链最近还开放了其开发的跨链协议的源代码，这意味着任何其他开发者都可以访问和使用该代码。跨链技术旨在改善不同区块链之间的数据交换。

据悉，开源的跨链协议AntChain Bridge，基于IEEE全球首个跨链通用国际标准打造，开源后，开发者可以使用蚂蚁链的跨链代码，共建Web3开放生态； 而Web3开放联盟链，覆盖Web3的区块链网络层、中间件层、应用层、访问层等四层架构的技术能力，并兼容以太坊等生态协议接入，进一步延伸Web3生态。

“跨链技术连接不同的区块链，促进跨链价值流通。新标准带来了统一的通信协议，显着降低了区块链互联的成本，”IEEE计算机学会区块链和分布式账本技术标准委员会主席李明表示。

“跨链技术对于释放 Web3 的全部潜力至关重要。通过将不同的区块链和跨行业的用例与技术连接起来，我们将创建一个更加互联和开放的生态系统，允许大规模的数字资产转移，”蚂蚁集团技术总监严颖表示。

互联网或 Web 1.0 的起源允许人们阅读和连接信息。Web 2.0 是社交媒体的迭代，它促进了人们与他人联系的新浪潮。Web 3.0 被广泛认为是由区块链技术驱动的去中心化互联网，加密货币、不可替代代币 (NFT) 作为底层基础设施和应用程序。

蚂蚁集团资深副总裁、数字科技事业群总裁蒋国飞也表示，基于全球标准的跨链协议，区块链将走向万链互联互通，实现价值像信息一样自由流转，加速迈向产业web3未来。

蚂蚁集团隶属于阿里巴巴，拥有全球最大的数字支付系统支付宝。 蚂蚁集团在 2020 年推出了蚂蚁链，作为其区块链技术解决方案以及人工智能和其他技术工作的标志性品牌。

蚂蚁集团自2015年开始投入Web3相关技术研发，在共识协议、智能合约、网络传输、存储引擎、跨链技术等领域处于全球领先水平。 蚂蚁集团在区块链与隐私计算领域的专利数连续多年位居全球前列，并牵头制定10余项核心技术类国际标准。

与此同时，蚂蚁链不断探索Web3的产业落地，已经在电子票据、供应链金融、司法存取证、版权保护、跨境贸易等80多个场景实现应用。 据IDC数据，蚂蚁链连续两年位居中国BaaS（区块链即服务）市场份额第一。

除蚂蚁集团外，国内在Web3研究方面，阿里达摩院、腾讯研究院等机构也长期保持对Web3技术前沿与应用的关注。
