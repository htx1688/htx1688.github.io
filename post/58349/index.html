DeFi 项目因其高收益依旧值得关注，加密 KOL Fungi Alpha 以链为标准挑选了能赚取高收益的 DeFi 项目，编译如下：

加密叙事主导了 DeFi 的发展创新，但这并不影响 DeFi 的高收益项目，以下是我挑选的值得关注的 DeFi 项目：

Ponzi 炒作很有意思，但也是高风险，所以我们要注重安全、炒作和叙事，先从 BNB 链开始介绍，无论你有没有注意到，它都拥有着庞大的用户群和其 opBNB 的广阔前景！

BNB 链

KTX.Finance

KTX.Finance 是 BNB 链上的去中心化衍生品和社交交易协议，可开高达 50 倍杠杆。该平台正在按照规划运行发展，并且该团队最近宣布在 Mantle 上进行部署, 一个安全的选择是在 KLP 池（类 GLP 模型）中提供流动性。该项目提供 143% 的APR，其中 27% APR 以 BNB 形式支付。

Arbitrum 链

HMX

Arbitrum 是相当受真实收益和 perps dex 项目欢迎，因此 Arbitrum 上有很多赚取高收益的机会，相对来说比较安全（但与 DeFi 相关的风险始终存在）。HMX 是 Arb 上的去中心化永续协议，提供跨链保证金和多资产抵押支持。

HMX 的 TVL 数据很不错，从逻辑上讲，这反映在 HLP（HMX 的流动性 token）收益率上，68% 的总 APR，其中约 31% APR 属于 USDC.e，也是 HMX 受到广泛的采用的原因。

Hyperliquid 的 HLP 金库也非常不错，TVL 达 500 万美元，金库利润 60 万美元以上，当前 APR 为 162%，存入 USDC 并让金库从交易员的 PnL（损益）中累积费用。

Good Entry

Good Entry 是 Arbitrum 上的 perp Dex，目前关注的不多，用户在开杠杆交易时就可以免受清算的影响，另一个亮点是 ezVaults，该项目大多数奖励都是原生 token，尽早参与。

Stella

Stella 是 Arbitrum 上的零成本借贷杠杆策略协议，通过零成本借贷提供杠杆流动性挖矿策略以及其独特的现收现付费用模式。下面显示了其年利率表明了最大杠杆下的年利率数据，但基本年利率也同样值得关注，该项目目前还没有 token。

Aura Finance

Aura Finance 是建立在 Balancer 系统之上的协议。它通过 BAL 存款和 Aura 的原生 token 的社交聚合，为 Balancer 流动性提供者和 BAL 利益相关者提供最大的激励。操作很简单，首先用户在 Balancer 上提供流动性，然后使用 BPT 在 Aura 上获得一些额外的收益，该池目前提供约 38% 的 APR。

Umami DAO

Umami DAO 是一个非托管的 DeFi 收益协议，其目标是创建并向市场推出一系列 DeFi 收益产品，将费用返还给 UMAMI 持有人。用户可以通过向该协议存入比如 BTC、ETH、USDC、UNI、LINK 等 token 赚取不错的 APR，USDC 或 ETH 的 APR 较高，约为 15.4%-19.9% 左右。

Buffer Finance

Buffer Finance 是一个非托管的期权交易平台，也是 Arbitrum 上的老牌项目，用户在该平台上可以选择四个时间范围：5m/15m/1hr/4hr，质押 BFR 和/或 BLP 以从产生的费用中赚取真实收益。Buffer 的 USDC Vault APR 为 25.59%，也是 Arbitrum 上不错的收益项目。

Trader Joe

Trader Joe 大家都很熟悉了，Arbitrum 上的一站式 DeFi 平台，业务涵盖广泛，包括现货 Dex、去中化借贷、NFT 交易平台（Joepeg）和 launchpad（Rocket Joe），其 APR 数据一直不错。

Gamma

Gamma 是一种专为集中流动性的非托管、自动化和主动管理而设计的协议。用户可以将资金部署到流动性池中以赚取高收益，该协议 APR 数据也非常高，因其更优秀的流动性管理机制，用户可以放心将资产添加池子。

MUX Protocol

MUX Protocol 是一个去中心化的杠杆交易协议，可实现零价格影响和高达 100 倍杠杆的交易，交易者没有交易对手风险，并提供优化的链上交易体验。此外，MUX 是一种多链本机协议，可统一跨多个已部署链的汇集流动性，以最大限度提高资本效率。目前，该协议交易量达到了 145 亿美元，ETH APR 达到了 21%。

Optimism 链

UniDex Exchange

UniDex Exchange 是一个类似于纳斯达克的 DeFi 平台，交易者可以通过该平台为任何类型的金融工具下订单，UniDex 将根据数百个来源和匹配订单将订单发送到最佳可用价格，目前该平台提供期权、合约、跨链交易等多种交易工具，UniDex 近期改良了其 UI，并提高了质押者可获得的收入份额。另外，UniDex 还在开发 UniDex 链，该平台 DAI 的存款 APR 约为 20%-40%。

The Granary

The Granary 是一个跨链借贷协议，Granary 上 DAI 存款的 APR 达到了 27%。

Kyber Network

Kyber Network 是一个多链加密货币交易和流动性中心，将来自不同来源的流动性连接起来，以实现最佳汇率的交易，旗下产品 KyberSwap 是 DEX 聚合器。Kyber Network 也是一个老牌 DeFi 项目，它的 Elastic Pools 很受欢迎，尤其是在 Op 链上，APR 非常高。

Base 链

Aerodrome

谈到 Base 链最火的 DeFi 项目，当然要数 Aerodrome，一个 Velodrome 的分支，也是一个值得埋伏的 DeFi 项目。截止发文，据 Defillama 数据显示，Aerodrome 的 TVL 已超 2 亿美元。

Polygon 链

Metavault.Trade

Metavault.Trade 是一个去中心化现货和永续交易平台，Metavault 的 MVLP 上 MATIC 存款 约 27% 的 APR，且非常安全。

Pearl

Pearl 是一个 Solidly 分叉协议。此外，Pearl 还与 USDR（Tangible 的产生收益的稳定币）合作以带来高收益，该项目中的一些稳定币 LP APR 为 27%-37%。

Retro

Retro 是 Polygon 上的 AMM DEX，除此之外，Retro 也通过 Soligly 分叉给协议带来更高的收益，该协议的平均 APR 数据也很不错，比如 USDC/agEUR 池。

Ethereum 链

说了这么多 L2，以太坊上的 DeFi 高收益协议也不容忽视。

ambient

ambient（前身为 CrocSwap）是一个 DEX，允许在任意一对资产上组合集中和环境恒定产品流动性的双边 AMM，也有消息称该协议将在 Berachain 上部署。目前看来在 ambient 上部署流动性未来可能有益（或许会有空投）。

Silo Labs

Silo Finance 是一个去中心化借贷协议，近期在以太坊上推出 Silo Llama，Silo Llama 是 Silo 协议的一个分叉，支持用户通过 crvUSD 抵押借贷，可以获得 CRV 奖励。

Sommelier Finance

Sommelier Finance 是一种基于 Cosmos SDK 的去中心化资产管理协议，其 FRAX 和 ETH 金库提供 12% 的 APY。

Jones DAO

Jones DAO 是一个针对高收益、策略和流动性的协议，内涵 vaults，可以一键访问机构交易策略，通过生息 token 收益解锁 DeFi 的流动性和资本效率。其 jAURA vaults 收益非常不错，AURA 持有者可以通过 vaults 最大化所持 AURA 的收益。

以上并不构成财务建议，只是可以增加用户资产收益的一些不错的项目，希望可以帮到你，DYOR！
