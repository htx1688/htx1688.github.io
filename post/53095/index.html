作者：金色财经，Climber

北京时间 6 月 12 日 20 时，ConsenSys 旗下 zkEVM 项目 Linea 发布了 Alpha v0.2，项目团队官方表示这次的测试版本是通过一个实质性的架构升级为主网的启动做准备。

Linea 是一个由 Consensys 公司开发的兼容 EVM 并采用 Zk-rollup 技术的 layer2 解决方案，通过将零知识证明与以太坊虚拟机 (EVM) 等效性相结合，允许构建者创建可扩展的 dapps 或迁移现有的 dapps，而无需更改代码或重写智能合约的麻烦。

此前打造出MetaMask的ConsenSys 入局Layer2，作为明星项目的Linea 自然有着十分值得期待的前景。目前已有57个项目在其网络上构建，主要赛道包括 DeFi、NFT、Bridge等，大量的生态还在开发当中。 接下来，金色财经将带你了解到，当前已有哪些项目建设其中。

钱包:

MetaMask

网站：https://metamask.io/

MetaMask（小狐狸）是用于与以太坊区块链进行交互的软件加密货币钱包，它可以通过浏览器扩展程序或移动应用程序让用户访问其以太坊钱包，与去中心化应用进行交互。MetaMask 由ConsenSys Software 开发运营，主要专注于以太坊为基础的工具及基础设施，目前在全球拥有超过3000万用户。

Fluviwallet

网站：https://fluviwallet.xyz/

Fluvi Wallet 是一款基于账户抽象的全功能智能钱包，旨在通过解决现有基于 EOA 的钱包存在的安全性和便利性问题，成为每个人的 Web3 网关。其功能包括： 链上 2FA 实现、解决因私钥导致的单点故障风险、无需私钥的多设备同步、无助记词恢复、无GAS交易和轻松批量交易。

DeFi:

Mes Protocol

网站：https://www.mesprotocol.com/

MES Protocol 是一个基于跨rollup订单簿的现货交易 DEX，旨在解决跨rollup的流动性碎片化问题。可容纳任何来自 EVM 兼容链上的交易，而不会影响以太坊链的安全性和可组合性。当“Cross-rollup Mode”开启时，跨 rollup 的流动性被聚合到一个地方，用户可以无缝参与多个链上的流动性来进行交互，从而以更低的滑点更有效地完成订单。

FWDEX

网站：https://testnet.fwdx.xyz/

FWDEX 是一个去中心化的 P2P 交换平台，提供稳定的报价和零滑点，大部分利润由用户和代币持有者分享。

Cashmere

网站：https://testnet.cashmere.exchange/swap

Cashmere建立在 Layer0 协议之上，通过模拟 MEV 机器人攻击模式并禁用它们以确保用户获得最佳交易体验。与其他跨链互换协议相比，MEV 协议能够帮助用户以最佳汇率在任何链之间交换任何资产。

Mendi Finance

网站：https://testnet.mendi.finance/

Mendi Finance是一个与EVM兼容的借贷协议，在Linea上推出。Mendi Finance提供完全去中心化、透明和无托管的点对点贷款解决方案。与Compound Finance和AAVE等现有借贷平台类似(并基于这些平台)，用户将能够在台上借出任何支持资产，并使用他们的资金来借入支持资产。

Mendi的目标是成为Linea的主要借贷平台，为货币市场提供最具竞争力的激励，拥有最大的流动性。

zkLink

网站：https://zk.link/

zkLink 是一个统一的多链交易基础设施，由zk-SNARKS 提供保障，支持下一代去中心化交易产品，如订单簿 DEX、NFT 市场等。

ZKEX.com

网站：https://testnet.app.zkex.com/

ZKEX 采用零知识保护的无桥多链加密交易，是具有 CeFi 性能的无需信任和自我托管的订单 DEX。

IZISwap

网站：https://linea.izumi.finance/home

iZUMi Finance是一个提供一站式流动性即服务(LaaS)的多链DeFi协议，iZiSwap是其产品服务之一，最初在BNB链上推出。它是一种创新的离散化流动性amm，为Web3资产提供高效的链上流动性，其资本效率比传统的x*y=k DEX高出5000倍以上。

Symbiosis

网站：https://testnet.symbiosis.finance

Symbiosis 是一种跨链 AMM DEX，可提供跨EVM 和非 EVM 链的任意代币交换。

ghostNFT引入了 NFT 2.0 开源标准，使资产支持的 NFT 能够为 NFT 持有者和创造者提供更多功能。

NFTs2Me

网站：https://nfts2me.com/

NFTs2Me 是一个对用户友好的综合平台，完全免费开放。主要用于创建、部署和管理用户的 NFT 收藏品，同时具有社区治理及其它更高级的功能。

Apetimism Launchpad

网站：https://apetimism.com/launchpad

Apetimism是一个基于 Optimism的社区驱动的 PFP NFT 项目，目前用户只要花费几分钟就可以在 Optimism、Arbitrum One、Arbitrum Nova、Polygon、BNB Chain 或 Ethereum 主网上启动自己的 NFT 项目，并享受各种丰富的功能。

Zonic

网站：https://testnet.zonic.app/

Zonic 是一个顶级的 NFT 市场，可以实现闪电般的交易速度并支持多链买卖 NFT。

Bilinear

网站：https://www.bilinear.io/

LINEA 原生NFT开发和创建平台，具有 Bilinear.it NFT 市场。用户可以发布自己的个人数字资产，并以简单直接的方式与社区分享。

Omnisea

网站：https://www.omnisea.tech/

Bridge:

Hop

网站：https://goerli.hop.exchange

一个无需信任的跨链桥，允许用户直接在L2、侧链和L1之间快速轻松地转移代币，并且允许用户在 Goerli 和Linea 之间发送Token。

Celer Network

网站：https://celer.network/

Celer 是一种区块链互操作性协议，可实现一键式跨链，用户能够轻松体验访问代币、DeFi、GameFi、 NFT、社区治理等。

LI.FI
