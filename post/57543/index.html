原创 | Odaily星球日报

作者 | 秦晓峰

借助 Coinbase 的网络效应，L2 扩容方案 Base 网络在熊市中迅速扩张。

从 8 月 10 日上线主网至今，短短两周时间，Base 主网 TVL 已达 2.4 亿美元，目前在 L2 中排名第五，仅次于 Arbitrum、OP Mainnet、zkSync Era 以及 dYdX。21co链上数据分析师 Tom Wan 发文称，就“L2 以太坊渗透率”指标而言，Base 增长用时远少于 Arbitrum、Optimism——达到 0.25% 渗透率，Arbitrum 用时 9 个月，Optimism 用时约 8 个月，Base 用时约 1 个月。

Base 协议负责人 Jesse Pollak 表示，目前已有 100 多个 DApp 和服务提供商加入，进一步壮大了 Base 生态系统。不同于其他 L2 协议，Base 官方曾明确表示不会发币，因此「撸空投」的预期大大降低。想要获得新生态早期红利，只能寄希望于生态项目本身。

本期，Odaily星球日报甄选了 Base 网络上十个重点项目，供读者参考。项目部分筛选标准：（1）Base 网络原生项目且 TVL 较高，Uniswap、Compound 等老牌多链部署项目不在统计范围在内；（2）曾获得 Coinbase Venture 投资，潜力较大，并且已经上线 Base 网络；（3）Base 网络爆火、用户量较大的知名项目。如下所示：

项目名：friend.tech

官网：https://www.friend.tech/

简介：friend.tech 是一个去中心化社交平台，允许用户购买和出售网络中其他用户的“股份”，其本质上是对大 V 们影响力进行投注；购买大 V 们“股份”的投资者将直接获得访问该用户内容以及与该用户发送消息的权限。此前，friend.tech 宣布完成种子轮投资，Paradigm 参投。

Dune 数据显示，friend.tech 目前交易量超 4 万 ETH，协议收入达 1824 个 ETH（约 303 万美元），排名 Base 生态第一；Total Friend Value 总计 12, 692.7 ETH，约 2110 万美元。

需要注意的是，目前 friend.tech 尚未发布代币，不过其每周五会向用户空投积分。friend.tech  将在为期六个月的测试阶段分发 1 亿积分，每次空投将在周五进行，积分于链下记录，并将在测试阶段结束后有特殊用途——可能兑换为代币。

项目名：BaseSwap

官网：https://baseswap.fi/

简介：BaseSwap 是一个去中心化交易平台，并号称是 Base 团队打造的——但尚未得到证实，采用双代币模型：平台币 BSX 以及收益代币 BSWAP。该平台提供以下几个服务：（1）交易，快速兑换代币，费用为 0.25% ；（2）流动性提供，成为 LP；（3）质押服务，支持 USDC、WETH 以及 BSX 质押；（4）农场，支持 LP 质押获得 BSWAP 代币；（5）跨链桥，支持包括 Base 在内的 8 个网络跨链转账；（6）NFT 市场，交易 Base 网络 NFT。

DefiLlama 数据显示，BaseSwap 当前 TVL 为 5439 万美元，排名 Base 生态第一。

项目名：SwapBased  

官网：https://swapbased.finance/

简介：SwapBased 是一个一站式交易平台，已发行平台币 BASE（不是 Base 官方代币，勿混淆），目前提供以下几个服务：（1）代币交易，收取 0.3% 的费用；（2）流动性提供；（3）农场，支持 LP 质押获得 BASE 收益；（4）跨链桥，提供 32 条链款链转账；（5）永续合约，最高支持 30 倍杠杆，目前上线了 ETH/USD 交易对，交易量超过 170 万美元，用户 120 多人。

DefiLlama 数据显示，SwapBased 目前 TVL 为 2100 万美元，排名 Base 生态第二。

项目名：Alien Base

官网：https://www.alienbase.xyz/

简介：Alien Base 是一个基于 Base 网络的去中心化交易平台，提供 Swap、流动性提供以及质押功能。Alien Base 的流动性挖矿计划允许用户提供流动性和质押 LP 代币（目前为 10 个）来赚取 ALB 代币；质押 ALB 代币也能赚取奖励和平台收入，平台费用的最多 50% 将归 DAO 所有。DefiLlama 数据显示，Alien Base 目前 TVL 为 1660 万美元，排名 Base 生态第三。

项目名：Maverick Protocol

官网：https://app.mav.xyz/boosted-positions?chain=8453

简介：Maverick Protocol 是一个采用动态分布 AMM 模型的去中心化交易协议，它可以根据市场需求动态调整流动性池中的资产分布，它是目前市场上唯一能够实现最优流动性效率、最优价格发现、最优资本控制和最优跨链互操作的 AMM 模型，同时支持高速、低成本、安全的跨链交易。目前，Maverick Protocol 已支持 Base、以太坊、BNB Chain 以及 zkSync Era，Base 链上 TVL 为 125 万美元。

今年 6 月，Maverick 完成 900 万美元融资，Peter Thiel 旗下基金领投，Pantera Capital、Binance Labs、Coinbase Ventures 和 Apollo Crypto 等参投。

项目名：WOOFi

官网：https://fi.woo.org/

简介：WOOFi 是由 WOO Network 构建的一体化去中心化应用程序(DApp)，旨在通过提供超低滑点、有竞争力的兑换费用以及其他有用功能，优化用户对 DeFi 的体验。 8 月 15 日，WOOFi 宣布上线 Base 主网，将通过“复合主动做市”模式向交易者提供竞争性报价，并提供无缝跨链兑换服务，用户现可通过 WOOFi 跨链 6 个链资产至 Base 网络。此外，WOO 代币质押者将获得 80% 交易费用，剩余 20% 将支付给经纪商。

融资方面，WOO Network 于 2021 年 11 月筹集了 3000 万美元，BitTorrent、Avalanche、Crypto.com Capital 等参投；2022 年 2 月，WOO Network 完成 A 轮 1200 万美元融资，由 Binane Labs 领投的。

项目名：DODO X

官网：https://dodoex.io/zh

简介：DODO X 是 DODO 最新推出跨链聚合器，旨在支持用户以最优价格在不同网络之间交换任意两种代币。目前，DODO X 已经成功部署在 Base Mainnet，具体交易功能包括：（1）跨链互换，用户能够使用跨链代币桥将一条链上的资产转换为另一条链上的不同资产：（2）限价订单，用户可以创建限价订单，以特定的最低价格（即限价）来交换所需的代币。只有当市场价格达到用户设定的限价时，交易才会被执行；（3）交易小部件，使用户能够构建定制的前端界面，并将其嵌入到 DeFi 应用程序中，实现快速资产交换等多种用途；（4）交易 API，允许用户使用智能路由算法获取高效的第三方代币交换路由，从而找到链上最佳价格。

项目名：Magnate Finance

官网：https://www.magnate.finance/

简介：Magnate Finance 是一个基于 Base 网络的去中心化借贷协议，采用动态利率模型，创建了更具资本效率的风险管理池，支持多种抵押品，包括加密代币、稳定币、合成资产、NFT、以及其他各类资产——目前已上线 ETH、WETH、USDC、DAI 四种加密货币借贷。

官网数据显示，Magnate Finance TVL 目前为 1648 万，位列 Base 生态借贷协议赛道第一。该项目计划下一步拓展至 Arbitrum 生态。Magnate Finance 治理代币为 MGA，目前已经上线，流通市值 150 万美元。

项目名：Moonwell

官网：https://moonwell.fi/

简介：Moonwell 是一个基于 Moonbeam 和 Base 的开放式借贷 DeFi 协议，与 Aave、MakerDAO 和 Compound 等借贷产品类似，允许用户超额抵押并借出加密货币。上线 Base 主网后，Moonwell TVL 一度上升至 2000 万美元以上，目前回落至 1520 万美元，位列 Base 生态借贷赛道第二。

该项目联合创始人 Luke Youngblood 是前 Coinbase 高级参谋工程师和 AWS 首席工程师。2022 年 4 月，Moonwell 完成了 1000 万美元私募融资，估值 9000 万美元，由 Hypersphere 和 ArringtonXRPcap 共同领投，Coinbase Ventures 等参投。

代币方面，Moonwell 曾于 2022 年 5 月发行治理代币 WELL，今年 5 月公募全部解锁完成，目前流通市值 505 万美元。

项目名：EDE Finance

官网：https://www.ede.finance/

简介：El Dorado Exchange （EDE）是一个去中心化现货和永续交易平台，动态定价由 PYTH 预言机提供支持。

EDE 借鉴 GMX 机制，目前采用了两个包括不同资产的 ELP 池为交易做支撑，用户通过做市、提供 Swap 和杠杆交易的流动性可以赚取收益，也可以自由选择铸造不同的 ELP 代币，也可以通过销毁来赎回资产。EDE 同时为用户提供了更便捷的 ELP 交易，用户可以直接使用单一资产购买或者出售 ELP。EDE 是平台的实用代币，质押 EDE 获得治理代币 gEDE，以及 EUSD 和 EDE 奖励。用户可以质押 EDE 7 天到 4 年，质押时间越长，获得 EUSD 和 EDE 奖励越多。

DefiLlama 数据显示，EDE 目前 TVL 不到 100 万美元，排名 Base 生态衍生品协议第二。
