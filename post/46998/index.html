去年 12 月份，因为 Jack Dorsey 的一次比特币捐款，让我关注到了 nostr 这个去中心化社交协议，后来便编译了 《一文了解 nostr ：一个让 Elon Musk 感到害怕的去中心化社交协议》以及 《一文对比去中心化社交协议 Nostr 和 Farcaster》这两篇文章，然而最近 Nostr 及其生态应用 Damus 的突然爆火，确实有些出乎我的意料，因此想写写一些新的看法。

在 Nostr 爆火之前，我的想法是它只会在比特币生态内传播，原因无非是觉得它没有 token 激励，并不符合 Web3 的定义，因此无法吸引投机者，只有纯粹的比特币爱好者才会去尝试使用它，而早期糟糕的用户体验，也对普通用户没有太多吸引力。当然，比特币社区已经拥有了庞大的用户群体，即便没有 token 激励，Nostr 协议也可以获得巨大的牵引力，这并不是什么昙花一现的产品，而是具有重要意义的革命性协议。

然而 Nostr 最近的发展，明显已经超过了我的想象，身边很多并非比特币爱好者的人在尝试体验它。现在来看，一部分人可能误以为它是 Web3 协议，想通过交互获取一些空投，另一部分人，或许是因为从众心理，想去体验一下目前的热点到底是什么（当然反馈的结果很可能会是糟糕的），最后一部分参与者，应该是对去中心化社交有着真正需求的人。

这里再次明确一下，Nostr 及其生态应用，均不会发行 token，它们也与 Web3 协议毫无关系，两者的关系是对立的，通过交互赚钱的想法是不切实际的。

回归去中心化本质，还是专注所有权激励？

事情的发展，让我不得不去思考一个问题 ，即去中心化协议，到底需不需要 token 经济激励？

不需要，是比特币最大主义者阵营的想法，他们认为 token 经济激励会吸引太多投机者，最终导致网络脱离去中心化的原本目标，此外投机也会造成大量的骗局项目，并不是所有人都像中本聪那样对钱毫无兴趣，相比之下，比特币可以为闪电网络和 Nostr 这样的无 token 去中心化协议提供最强大的武器，反过来，闪电网络和 Nostr 协议的成长，又可以扩大比特币的影响力。

这一点我是非常赞同的，所以我相信比特币、闪电网络以及 Nostr 拥有巨大的价值。

另一方面，比特币网络最初也是依靠 BTC 的经济激励一步一步成长过来的，它存在所有权的概念，这也是以 a16z 为代表的以太坊阵营一直在提倡的 Web3 概念的核心。一个去中心化网络，要从无到有，投机可能也具有一定的存在意义，它可以快速吸引用户群体，至于能否让用户留存下来，则需要依靠产品本身的说服力。

最后的看法

从去中心化的角度来看，我认为没有任何一款所谓的 Web3 社交协议能够媲美 Nostr，如果你在乎抗审查，就应该去关注和使用 Nostr。

另一方面，投机也是客观存在的一种现象，就短期而言，Web3 的概念依旧是充满吸引力的，并且因为牺牲了去中心化属性，Web3 产品的用户体验可能更容易被大众所接受，类似 Farcaster 这样由 VC 主导的协议尝试，也是值得一部分人去关注的。

而从长期的角度来看，Nostr 击败以太坊阵营的 Web3 社交协议，我认为将是大概率的，这将牵扯到监管方面的问题，后者将因为监管而失去吸引力。
