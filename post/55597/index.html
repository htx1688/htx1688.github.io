作者：Paul Veradittakit，Pantera Capital 执行合伙人；编译：Leo，BlockBeats

近日，Pantera Capital 执行合伙人 Paul Veradittakit 撰写了关于「Cosmos 目前核心技术和生态项目」的文章，解读了 Cosmos 在技术上的核心优势、应用链上的创新机制和目前 Cosmos 上重要的几个项目。BlockBeats 编译如下：

现今传统区块链面临的重大挑战就是其广泛采用和可扩展性的推进。其中一项难题是不同链之间缺乏互操作性，导致通信和数据传输复杂且低效。此外，因为设计的重点是安全性而不是可扩展性，所以可扩展性一直是首要难题，也导致了通信速度变慢，网络更加拥堵致从而成本增加。

通过引入一个互联的区块链范式，Cosmos 推出了解决这些难题的解决方案，它利用链间通信（IBC）协议，实现不同链之间的无缝通信和资产安全转移。凭借链间互操作性的标准化框架，Cosmos 促进了高效地跨链交互，扩展了区块链技术的可能性和实用性。此外，Cosmos 通过其模块化架构和共识算法解决了可扩展性问题。开发人员可以使用 Cosmos SDK 创建特定于应用程序的区块链，称为「zones」。zones（独立区块链，也称区域）可以采用不同的共识机制，允许水平扩展和优化性能。通过支持跨多个 zones 的并行处理，Cosmos 显著提高了网络的交易能力和可扩展性，还利用 Comet 共识算法，提供快速的区块确认时间、安全性增加和抵御恶意攻击的能力。这类共识算法作为互连区块链网络的可靠基础，结合了效率、最终性和稳固性。

核心：Cosmos SDK、IBC 和 Comet

Cosmos SDK 为开发人员提供了创建满足特定需求和目标的定制区块链或 zones 的能力。利用 Cosmos 互操作性的特性，这些定制 zones 拓展了链间交互的边界。Comet consensus 算法增强了这一功能，确保 Cosmos 网络安全高效地运行。此外，IBC 还支持跨不同链的无缝通信。

Cosmos 亮点

互操作性： Cosmos 的优势在于它对互操作性的保证，使 Cosmos 网络中的独立区块链之间能够轻松连接。这种互操作性促进了跨链应用和协作的突破，推动创新并增强了去中心化的实用性。

可扩展性和性能： Cosmos 通过授权创建独立的区块链或 zones 来应对可扩展性问题，这些 zones 可以根据用户的需求定制其共识机制和可扩展性，这种途径支持水平扩展，通过合并额外的 zones 来容纳增加的交易量。因此，Cosmos 生态拥有出色的性能和可扩展性。

个人主权： Cosmos 生态系统为创造专注个人主权的链提供了一个理想的平台。通过利用 Cosmos 提供的基础设施，如 Cosmos SDK、IBC 和 Comet BFT 共识算法，开发人员可以灵活地定制技术堆栈的任何方面，以增强性能、吞吐量、安全性和链的其他关键因素。例如，像 Evmos 这样的 EVM 链是利用现有的技术堆栈构建的。此外，还可以通过开发专用的应用层来支持这些特定的虚拟机（VMs），从而构建在不同虚拟机上运行的链，例如 Move 或 SVM。

开发人员友好的环境： Cosmos 为区块链开发人员提供了一个友好的环境。Cosmos SDK 是一个用于构建 dApps 的功能丰富的平台。利用 SDK 的模块化框架，开发人员可以在 SDK 之外创建自己独特的区块链，从而简化开发流程。这种简化的方法允许开发人员创建从 Cosmos SDK 延伸出来的定制链，从而增加区块链生态的采用和增长。

Cosmos 应用链

应用链是 Cosmos 创新生态不可或缺的组成部分，代表着满足不同用例的专业化和特定应用区块链。与传统的单片区块链不同的是，通过 Cosmos SDK 采用模块化方法，使开发人员能够根据其 dApps 的特定要求定制共识、存储和网络机制。这赋予了应用链一定程度的自主权，让开发者能够更好地控制自己的应用，增强他们塑造用户体验的能力。此外，Cosmos 应用链模式提供了诱人的收益机会，允许 dApp 创作者探索独特的货币化策略，并促进各自链内的可持续增长。

此外，Cosmos 应用程序链的灵活性扩展到了 MEV 的处理。通过为开发者提供工具来设计适合他们所需的 MEV 策略，Cosmos 激励公平和透明的 MEV 提取，促进更公平有效的生态系统。此外，IBC 协议促进的互联性是 Cosmos 应用链模型的关键优势。支持不同应用链之间的无缝通信和资产转移，支持高级跨链 DeFi 应用的开发，并为开发者和用户等扩展可能性。总之，Cosmos 生态中的应用链为 dApps 开发提供了一个有前途的途径，提供了主权、收入机会、MEV 上的灵活性和无缝互操作性，从而培育了一个充满活力和蓬勃发展的区块链生态。

链间安全性

链间安全是旨在增强新 Cosmos 链的安全性和促进其顺利启动的一种突破性途径，它涉及从 Cosmos 生态中已经建立的安全链（Cosmos Hub）中「renting」经济安全的概念。

主要优势

经济安全租赁： 为了推出新的 Cosmos 链，开发者可以通过锁仓一定量的新链原生 token 作为抵押品来利用 Cosmos Hub 或其他安全链的安全性，该过程有助于在新链和安全链之间建立金融联系。

跨链安全： 通过链间安全获得经济安全，新的 Cosmos 链对攻击抵御能力更强，它可以依靠安全链中大量不同的验证器来保护其交易和资产，降低 51% 攻击和其他安全威胁的风险。

独立的区块链： 每个 Cosmos 链都独立运行，有自己的治理、token 和用例。然而，通过像 IBC 这样的链间通信协议，他们可以安全地交换数据和价值，促进 Cosmos 生态内的互操作性和协同作用。

通过采用链间安全并在 Cosmos 生态系统内进行合作，新链可以显著降低进入门槛，并会获得强大且经过实践的安全基础设施。促进了创新，加快了区块链的发展进度，带来了一个安全、互联的 dApps 新时代。

Cosmos SDK

Cosmos SDK 支持模块化，创建了一个包含大量可互换模块的生态。这种方法将开发人员此前的「从头构建复杂功能」任务中解放出来。相反，开发人员可以将预构建的模块无缝集成到他们的应用中，从而简化特定于应用程序的区块链创建。Cosmos SDK 在模块定制方面提供了很高的灵活性，允许开发人员根据其应用需求精确定制这些模块，为 fine-tuning 链应用开辟了一个可能的领域。随着 Cosmos Network 发展，SDK 模块的生态系统也将随之发展，为开发人员提供可扩展的选择和功能。在这个动态的环境中，可以培育复杂且高度专业化的区块链应用程序，促进创新并推动 Cosmos 生态的发展。

Cosmos SDK 还通过其基于能力的安全概念优先考虑安全性。这些功能充当了「守护者」角色，明确定义了模块之间的界限，并减轻了恶意或不可预见的交互所带来的风险。通过保护创作，Cosmos SDK 专注于安全措施。

区块链间通信协议 IBC

IBC 是 Cosmos 生态的一种协议，旨在促进整个区块链的安全和去中心化通信。它作为桥梁，促进了整个 Cosmos 网络的数据传输。通过使用加密证明和共识算法，IBC 促进链间交易的完整性和可信度，维护数据机密性，防止篡改或未经授权的访问。该协议不仅增强了链的互操作性，还通过允许在网络上利用基于项目的系统来促进协作和创新。有了 IBC，Cosmos 有潜力开创区块链互联的新时代，为 dApp、跨链资产转移和互联区块链网络的创建提供新的可能性。

追踪： 链 B 连续接收链 A 传来的数据头，反之亦然。这允许每个链跟踪另一个链的验证器集，本质上讲，每条链都运行着另一条链的轻客户端。

绑定： 当 IBC 传输启动时，token 被绑定。

证明中继： 然后，10 个 token 绑定证明从链 a 中继到链 B。

验证： 在 B 链上对照 A 链的数据头验证凭证，如果凭证有效，则在 B 链创建 10 张凭证。

Cosmos 环境

目前 Cosmos 生态环境还不错，吸引了各种各样的项目，建立了有影响力的伙伴关系。通过利用 Cosmos 的强大基础设施，这些项目进入了高互操作性的区块空间，有可能释放强大的网络效应。这种无缝协作和集成极大地增强了整体生态，促进了创新，推动了 Cosmos 作为受欢迎区块链解决方案的采用。

基于 Cosmos 的项目

Osmosis

Osmosis 是建立在 Cosmos 生态上一个创新的加密项目，提供了一个促进流动性供应的 DEX 平台。在 IBC 协议的支持下，Osmosis 可以在 Cosmos 网络内的不同链之间实现无缝 token swap。这一独特的功能使用户能够访问各种资产和池，促进高效和安全地交易体验。通过利用 Cosmos 的互操作性和可扩展性，Osmosis 创建了一个包容与互联的生态系统，增强了流动性并实现了跨链协作。凭其对以用户为中心和优化流动性管理的关注，Osmosis 为 DeFi 参与者开辟了新的机会，为其探索更广阔资产领域，并发挥他们的交易潜力。

dYdX

dYdX 是一个 DEX 平台，因其以用户为中心的加密交易方法而火爆，在用户可以完全控制自己私钥和资产的前提下，dYdX 消除了对中心化机构的依赖，促进了更安全和透明的交易体验。dYdX 在 Cosmos 链上运行，利用 Cosmos SDK 来促进快速高效交易，以及与其他基于 Cosmos 应用的无缝互操作。除了基本的加密交易，dYdX 还提供保证金交易和借贷服务，迎合寻求开杠杆的交易员和希望从闲置资产中赚取利息的借贷用户。凭借其强大的安全性，Cosmos 在 dYdX 上前景广阔，彻底改变了交易者参与加密货币的方式，同时促进了 DeFi 生态的增长。

Akash Network

Akash Network 提供了去中心化云计算市场。通过利用 Cosmos 的安全性，Akash Network 提供了一个用户可以相互发送数据并开发的安全平台。该网络通过一种 PoS 的共识机制运行，提高了交易的安全性。Akash Network 原生 token 是 AKT，作为生态的实用 token。通过与 Cosmos 的集成，Akash Network 受益于更大网络的互操作性和可扩展性，实现了与其他区块链平台的无缝通信。因此，Akash Network 使开发人员和组织能够接触到价格合理的分散云计算资源。

Sei Network

Sei Network 是一个高性能区块链平台，在 Cosmos SDK 区块链平台上作为 L1 运行，为用户提供金融服务并促进 DeFi 生态的部署。该网络是创始人意识到需要一个在为用户提供创新金融工具的同时，优先考虑交易速度和安全性的去中心化金融系统。Sei 利用其专有的 PoS 共识算法，通过标记和所有权验证提供安全的交易确认。该平台的三层架构包括客户端层、服务层和链平台层，支持高吞吐量，每秒可处理高达 20,000 笔交易。借助 Rust 和 Solidity 编程语言对智能合约的内置支持，开发人员能够在 Sei 上轻松构建定制的项目和 DeFi 应用。该网络的原生 token SEI 作为奖励、质押和项目管理的内部资产，促进了可持续和繁荣的生态。此外，Sei 与 Cosmos 的集成赋予了它互操作性和可扩展性，允许与其他区块链平台进行无缝通信，并为更广泛地采用和更方便地访问分 DeFi 服务提供支持。最终，Sei Network 将成为一个价格合理、功能强大的区块链平台，推动金融普惠，支持区块链各领域尖端应用的开发。

Neutron

Neutron 是 Cosmos 生态中的一个 DeFi 平台，引入了跨链通信的转变。Neutron 的定位是成为 Cosmos 上的首个消费者链，以开放政策运营，链间通信的能力，促进不同区块链网络之间的无缝交互。通过 CosmWasm 支持多种编程语言的智能合约支持，提供了灵活性和可扩展性。其互操作性使开发人员能够访问其他区块链的功能集，开启了一个新的可能性领域，释放了区块链技术的真正潜力。通过采用来自 Cosmos Hub 的安全性，Neutron 促进了安全可靠的基础设施，为 Economic Zones 的互利关系奠定了基础。Neutron 的特性和战略定位预示着 Cosmos 生态的变革时代，对 DeFi 和链间合作的未来产生深远的影响。

Celestia

Celestia 是 Cosmos 生态的项目，它按需提供共识和安全性。它旨在通过消除引导新共识网络的需求来推进区块链部署，从而显著降低开销。与传统的区块链不同，由于数据可用性采样的独特属性，Celestia 的吞吐量随用户数量而扩展。Celestia 与众不同的是，它的区块链不依赖 honest majority assumption 来保证状态有效性，从而确保最高的安全标准。此外，Celestia 的灵活设计允许开发人员定义自己的执行环境，赋予他们自由和创造力。Celestia 支持各种类型的 Rollups，包括 EVM 和 Cosmos SDK，为构建具有安全性和可扩展性的互联区块链网络提供了解决方案。

Stride

Stride 是一个基于 Cosmos 区块链构建的流动质押平台，它革新了 token 质押和 DeFi。通过允许用户质押任何 IBC 兼容的 token 并获得 stTokens 作为回报，Stride 释放了流动性，并使用户能够通过 DeFi 交易来增加其收益率。这种创新的方法使用户能够在获得回报的同时保持灵活性，培养更具活力和生产力的生态系统。该平台对安全性的重视，通过了多次审计，确保了用户资产的安全。Stride 的重要性在于其重塑了质押格局，为 Cosmos 生态带来增强的流动性和收益机会，并通过无缝集成和以用户为中心的治理推动更广泛的 DeFi 空间的增长。

Leap Wallet

Leap Wallet 推出了一个创新的移动浏览器扩展和应用，致力于在简化和安全的基础上优化 Web3 交互。Leap 强调用户自主权，确保所有资产都在用户控制之下，坚持去中心化理念。快速且具成本效益的资产 swap 促进了 DeFi 投资和 NFT 的探索。Leap 与 dApps、Aggregators、DEX 和 Marketplaces 的广泛合作网络创造了优质的 Web3 体验。该平台经过严格审核，为用户营造了一个安全的环境，培养了信任和信心。Leap Wallet 提供跨各种设备（包括移动设备、平板、笔记本电脑和台式机）的多平台支持，是一款多功能工具，使个人能够探索 Web3 的无限潜力，同时保护他们的财务未来。

结论

Cosmos 自身灵活的架构，支持不同链之间的安全资产转移，并释放区块链技术的全部潜力。值得注意的是，早期版本的 Cosmos 上的 Terra、Polygon 和 Cronos 等关键项目的的蓬勃发展凸显了它支持多样化生态方面的多功能性。随着 Noble 链与 Circle 的合作，将 USDC 无缝引入了 Cosmos 生态，开启了 DeFi 的新篇章，用户越来越看好 Cosmos。Tether 在 Kava 上推出其 USDT 稳定币，Kava 是一个基于 Cosmos 构建的 L1 链，进一步加强了 Cosmos 的多链存在，并为整个加密社区带来了一个活力和互联互通的未来。
