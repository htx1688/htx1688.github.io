硅谷传统的初创科技公司都会举办各种 demo day 活动。但今天我们要介绍的是一批加密项目和团队，他们参加的是 Web3 加速器及建设者社区 Alliance DAO 举办的 demo day 活动。

Alliance DAO 的核心贡献者 Qiao Wang 在活动期间表示，该活动每年举办 3 次。「各个阶段的初创公司都加入了我们的行列，从刚刚有创意的公司到价值数十亿美元的公司。」

加速器聚集了 Polygon Studios 的首席执行官 Ryan Wyatt 和 Axie Infinity 的联合创始人 Jeff「Jiho」Zirlin 等导师和演讲者来帮助指导与会项目。

有 24 支团队加入训练营，只有 16 支团队最终脱颖而出。其中四分之三的团队聚焦于 NFT 和 DeFi 基础设施，大多数团队都是基于 L1 网络以太坊、Polygon 和 Solana。

训练营的主题包括：市场驱动的 DeFi、客户驱动的社交协议、Token 即协作的机制、消费品分类定价、DAO 如何作为风险分担机制等。

以下是 16 个初创企业的简单介绍。

Lulo：去中心化订单簿式借贷协议

创始人：Daniel Garay、Jesse Brauner

融资阶段：种子轮

Lulo 是一个 DeFi 借贷基础设施，旨在利用订单簿技术以无需许可的方式创建全球流动性层。它提供「流动性即服务」，因此用户可以将 DeFi 协议与全球借贷平台集成。它通过消除集中风险，贷方和借款人可以获得具有竞争力的贷款利率，因此不会存在借入利差。

LiquiFi：Web3 的 Carta

注：Carta 是资管行业 SaaS 鼻祖，截至 2021 年，估值为 74 亿美元。

创始人：Oliver Tang、Robin Ji

融资阶段：种子轮

LiquiFi 旨在为所有主要区块链（包括以太坊、Avalanche、BNB Chain、Polygon、Solana 等）的 Web3 产品和实体实现 Token 分配和运作自动化。联合创始人 Robin Ji 表示，几乎所有 Web3 产品都提供 Token 奖励，但追踪和转移 Token 的消费者和公司可能会遇到一些困难、风险且无法扩展。

LiquiFi 旨在其智能合约（已经过数轮审计）来掌握所有 Token，并提供一个界面来帮助用户追踪其投资。「我们的战略是紧跟所有趋势，这样其他公司就不必这样做了，」Ji 说。「挑战越大，您就越需要专业的解决方案来关注这个问题。」迄今为止，该公司在种子轮融资中筹集了 500 万美元。

fam.：基于 NFT 的专属社交俱乐部

创始人：Chris Tam、Shuguo Yang、Wallace Wong

融资阶段：种子轮

联合创始人 Chris Tam 表示，NFT 市场呈现爆发式增长，围绕 NFT 的社区也在蓬勃发展。但是目前的 Web2.0 平台，如推特和 Discord 作为社区通讯工具，由于其冗余信息及垃圾邮件而饱受诟病。fam. 旨在构建一个基于 NFT 自身的社交俱乐部的应用程序，使得用户可以通过他们的加密钱包连接到社区。个人可以通过该平台聊天、投票、参加活动并向拥有相同 NFT 收藏的其他人展示他们的生活方式。该项目已获红杉资本、Alliance DAO、Xoogler 等机构的种子轮融资。

PwnedNoMore：白帽自动化代码审查组织

创始人：Wen Xu, Conan Zhou, Mike Yan, Oriol Plazas

融资阶段：种子轮

PwnedNoMore 是一个由白帽黑客创建的去中心化自治组织 (DAO)，旨在帮助其他成员为 Web3 生态系统构建工具和保护措施。它正在开发一种全自动的 bug 检测引擎，目前已可以为基于 Solana 的项目提供检测，并计划在 8 月推出与以太坊兼容的引擎。该组织中的黑客包括六名 DEF CON CTF 获胜者。PwnedNoMore 的联合创始人 Mike Yan 表示，迄今为止，其社区黑客已经报告了 1,100 多个漏洞，并保护了超过 6.5 亿美元的风险加密基金。该组织已获 Alliance DAO、dao5、IDEO CoLab 等天使投资人投资。

Leather：Watch-to-earn 项目

创始人：Niko Cunningham、Shaminda Subasingha、Jason Kilic

融资阶段：种子轮

Leather 是一项「Watch to Earn」的移动服务，旨在实时分析和奖励人们在电视上观看的所有内容。Leather 的联合创始人 Niko Cunningham 表示，Netflix 和 Apple TV 等主要流媒体服务每年都会在其内容预算中投入数十亿美元，以创建更有针对性和个性化的内容。「我们不太了解那个房间里有谁，或者谁真正收听了 [节目]，或者谁真正关心。」Leather 的用户可通过与第三方分享消费和参与数据来换取 Token 奖励和 Leather 产品。

Filmine：去中心化计算的 AWS

创始人：Vukasin Vukoje

融资阶段：种子轮

Filmine 由 Alt Labs 提供支持，旨在成为去中心化计算的 AWS。该公司以去中心化存储网络 FIL 为中心，并希望帮助那些以更有效的方法挖掘 FIL 的人等等。它旨在通过名为 Filgram 的网络（一种 FIL 存储提供商发现工具）帮助矿工提供获得 10 倍以上挖矿奖励。

Legion DAO：专业游戏玩家的 Token 化联盟

创始人：Alex Beliankin, Alex Anikin

融资阶段：种子轮

Legion DAO 旨在帮助全职游戏玩家通过 Token 化工会将他们的爱好货币化。它计划通过培训游戏玩家成为教练和规范雇佣合同来改善游戏社区。Legion DAO 首席执行官 Alex Beliankin 表示，迄今为止，它拥有 100 名专业贡献者、3,000 个 NFT 会员，预计未来 2 年内的收入约为 250 万美元。

Clique：Web2 数据预言机

创始人：Kevin Yu，Jaden Yan

融资阶段：种子轮

Clique 是 Web2 用户数据的隐私保护身份预言机。Clique 的联合创始人兼首席营销官 Jaden Yan 表示，DeFi、GameFi 和 DAO 都会受到链上可用数据稀缺的限制，但该公司通过将人们在 Twitter 或 Discord 等 Web 2.0 渠道上的在线身份引入区块链来缓解这一问题。它于 4 月进行种子轮融资，得到了 GGV Capital、启明创投、Infinity Ventures Crypto、SKy9 Capital 等的支持。

Ajna：去中心化借贷平台

创始人：Greg DiPrisco、Ed Noepel、Akash Patel、George Niculae、Joseph Quintilian、Ian Harvey、Mike Hathaway、Matt Cushman 和 Howard Malzberg

融资阶段：Pre-seed

Ajna 是一个去中心化且无需许可的借贷平台。该平台联合创始人 Greg DiPrisco 表示，迄今为止，创建一个贷款市场需要大量现有流动性和复杂的治理过程来管理风险，这导致目前市场上的大部分资产无法自由借入或借出。Ajna 的目标是让人们在一个无需许可的市场中，无需预言机、治理者或管理人就能立即借出数字资产。DiPrisco 表示，贷款人只需要提供一个他们愿意贷款的价格，平台将处理剩下的部分。该平台由 9 位联合创始人自筹资金，目前没有筹集额外资金的计划。

Tilting Point：将 Web2 游戏带入 Web3

创始人：Kevin Segalla

融资阶段：后期

Tilting Point 是一家致力于将 Web2 游戏带入 Web3 的发行商。该公司旨在通过处理他们的整个加密策略以换取未来的加密收入，从而简化开发人员向 Web3 行业的迁移工作。目前该公司已经发行了 41 款游戏，并与 Polygon 达成合作，预计在未来 24 个月内将会发行 10 款 Web3 游戏。Tilting Point 的创始人兼董事长 Kevin Segalla 表示，该公司 2021 年的营收为 2 亿美元，在过去 4 年里增长达 5 倍。2021 年 7 月，该公司从 General Atlantic 筹集 2.35 亿美元资金，用于雇佣开发人员。目前该公司正在筹集 2000 万美元以用于在 Web3 进行更大扩张。

Krypton：抗 MEV 去中心化交易协议

创始人：Michael Nowotny, Nathan Moore

融资阶段：种子轮

Krypton 是一种抗 MEV（矿工可提取价值）型去中心化交易协议。它旨在成为 DeFi 空间的下一代交易协议，并保护交易者免受抢先交易、三明治攻击和其他因素的影响。它利用 Chainlink 的链下计算和流量交易来消除 MEV，保护交易者，降低交易成本并促进跨链流动性。Krypton 赢得了 2021 年秋季 Chainlink DeFi 黑客松竞赛，目前正在筹集由 Framework Ventures 牵头的种子轮。

Parasol：NFT 钱包

创始人：Kevin Phung, Tony Chen, Allen Guan, Louis Kang, Joi Ito

融资阶段：种子轮

Parasol 是一款专注于日本市场的 NFT 钱包。Parasol 联合创始人 Kevin Phung 表示，由于日本的粉丝忠诚度受到「宅男」文化的支持，许多品牌正在寻求通过 NFT 获利。Parasol 旨在提供「加密隐形」的半托管钱包，允许用户拥有资产所有权，但具有用户友好的保护措施。自成立以来的一个月内，已有超过 3.5 万个 Parasol 钱包被创建，共计购买了约 4 万枚 NFT。

Dapi：去中心化银行 API 接口

创始人：Ahmed Agour, Hesham Ghandour

融资阶段：种子轮

Dapi 是一个去中心化的银行 API 入口，通过使用 Plaid 一类的私人银行 API，旨在弥合金融科技应用程序和消费者银行账户之间的差距。Dapi 正在构建一个分散的点对点开放网络桥。Dapi 还使用去中心化激励措施，旨在允许任何人无需许可地构建 API 以获取利润份额。该平台表示，大量贡献者使得该网络能够抵抗审查。投资方包括 Y Combinator、Alliance DAO、Alloy Labs Alliance、Reserve、Cadenza 等。

Metavoice：声音制作 PFP 类 NFT

创始人：Vatsal Aggarwal、Siddharth Sharma、Tom Etminan

融资阶段：种子轮

Metavoice 旨在为用户提供拟人的、可定制的虚拟声音。Metavoice 的联合创始人汤姆・埃特米南 (Tom Etminan) 表示，该产品允许用户在线定制自己的语音，这对匿名用户尤为重要。用户可以连接他们的加密钱包，选择他们的声音，它可以转换和输出到 800 多个不同的平台。这款测试版应用将于 2022 年 8 月在 iOS 和 Windows 上发布。

Lava：去中心化的 infura 和区块链数据市场

创始人：Yair Cleper、Gil Binder

融资阶段：种子轮

Lava Network 旨在成为区块链数据的市场。Lava 通过激励节点提供商和去中心化应用程序用户将市场区块链数据输入到它的网络上。就像 Airbnb 允许业主将闲置空间变卖一样，Lava 计划允许任何节点运营商使用他们的带宽，同时协议加强数据质量。Algorand、Fireblocks、Messari、Avalanche、Polygon、Cosmos 等许多加密实体都在其候补名单上。Lava 已从投资者处筹集了资金，包括 Jump Crypto、PayPal、Tribe Capital、North Island Ventures 等。

Yakoa：NFT 欺诈检测网络

创始人：Andrew Dworschak, Graham Robinson

融资阶段：种子轮

Yakoa 旨在帮助检测 NFT 欺诈交易。Yakoa 的联合创始人 Andrew Dworschak 表示，在今天的区块链上，三分之一的 NFT 是原创的，三分之一是直接伪造的，三分之一与现有的非常相似，这导致每天的损失达数百万美元。该平台正试图通过在所有区块链中追踪侵权资产并在平台上标记真实作品来解决该问题。它的 API 已经上线并在使用中，并且已经为超过 4000 万资产建立了索引。
