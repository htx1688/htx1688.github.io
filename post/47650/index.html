欧易OKX 注册链接 欧易是全球著名的数字资产交易平台之一，主要面向全球用户提供比特币、莱特币、以太币等数字资产的币币和衍生品交易服务。
HTX火币 注册链接 火币全球专业站，是火币集团旗下服务于全球专业交易用户的创新数字资产国际站，致力于发现优质的创新数字资产投资机会。

今日凌晨，Sui 公布了该生态「Sui Builder Hero」奖项的首批获奖名单，该奖项旨在表彰那些基于 Sui 网络构建工具和应用，并通过参与讨论、推广教育以及其他形式对 Sui 生态建设作出贡献的项目方和开发者，获奖者除了可获得一定的现金奖励外，还可获得 Sui Builder Houses 邀请函等荣誉性奖励以及一些 Sui 礼品。

根据披露，本次获奖项目共有 20 个，涉及了 DeFi、NFT、钱包、预言机等多个不同赛道。在一定程度上，这批获奖选手也可被视为 Sui 生态最值得关注的 OG 项目。

1、Aftermath

官网：https://aftermath.finance/

定位：DeFi，DEX

Aftermath 是一个基于 Sui 构建的 AMM DEX，其特点是将支持构建涵盖加权资产（如指数）的流动性池，允许流动性提供者一次性在池中存入 7 种代币。

2、BeLaunch

官网：https://belaunch.io/

定位：DeFi，Launchpad

BeLaunch 是一个基于 Sui 构建的 Launchpad 平台，可帮助新兴项目创建代币并为其发行筹集资金。该平台在 Sui 的开发者网络上较受欢迎，已吸引了数千名用户。

3、BlueMove

官网：https://sui.bluemove.net/

定位：NFT，交易市场

BlueMove 是最早于 Sui 生态构建的 NFT 交易市场之一，该市场也将部署在 Aptos 之上。BlueMove 的治理代币 MOVE 现已发行，当初发行时曾通过 IFO 吸引了逾 6280 万美元资金。

4、Capsule

官网：https://www.capsulecraft.dev/

定位：基础设施

Capsule 是一种可将元数据附加到 Sui 网络上对象的标准，其开发者会积极地参与 Sui 社区线上和线下活动，帮助社区成员解决问题。

5、Cetus

官网：https://www.cetus.zone/

定位：DeFi，DEX

Cetus 是一个基于 Sui 构建的 DEX，其特点是支持资金利用效率更高的集中流动性做市（类似 Uniswap v3）。

6、Cramium

官网：https://cramium.io/

定位：NFT，品牌服务

Cramium 是一家基于 Sui 构建的现实世界品牌解决方案，其将利用 Sui 网络以及 NFT 来创建品牌制定忠诚度计划，为会员提供限定式活动以及独家内容服务。

7、Ethos

官网：https://ethoswallet.xyz/

定位：钱包

Ethos 是一家支持 Sui 网络的非托管钱包服务商。除了基础服务之外，Ethos 还会通过一些开源项目或在 Medium 上发布关于 Move 语言的文章来帮助社区进一步认识 Sui。

8、Keepsake

官网：https://keepsake.gg/

定位：NFT，交易市场

Keepsake 是一家基于 Sui 构建的 NFT 交易市场，但主要服务为游戏资产，游戏类应用可以直接将 Keepsake 整合在游戏之内，为玩家提供便捷的交易服务。

9、Kuna Labs

官网：https://kunalabs.io/

定位：DeFi，DEX

Kuna Labs 在 Sui Move 语言环境下实现并开源了高质量的 AMM 智能合约，该团队还曾与 Sui 创始团队合作解决过一些早期问题。

10、MovEX

官网：https://app.movex.exchange/

定位：DeFi，DEX

MovEX 也是一家基于 Sui 构建的 AMM DEX。该团队曾参与推动了 Move 教育计划，并曾帮助 Sui 招募新的开发者。

11、OriginByte

官网：https://originbyte.io/

定位：NFT

OriginByte 曾发布过一些 sdk 和工具，使开发者可以更容易地将标准应用到 Sui NFT 之上。

12、OtterSec

官网：https://osec.io/

定位：安全服务

OtterSec 是一家可服务于 Sui 生态项目的第三方审计机构，该机构的审计对 Sui 网络以及 Sui Move 虚拟机的安全性起到了至关重要的作用。

13、PolyMedia

官网：https://polymedia.app/

定位：DeFi，社交

PolyMedia 在 Sui 之开发有多款产品，比如去中心化的投注协议 GotBeef，以及链上聊天应用 Polymedia Chat。

14、PySui

Github 主页：[https://github.com/FrankC 01/pysui](https://github.com/FrankC 01/pysui)

定位：基础设施

PySui 开发了首个可用于 Sui 网络的 Python 客户端 SDK。

15、Sui Gallery

官网：https://www.sui.gallery/

定位：NFT，交易市场

Sui Gallery 是一个构建于 Sui Devnet 早期的 NFT 交易市场，现已有超过 3 万名用户在该平台上铸造过 NFT。

16、Suia

官网：https://suia.io/

定位：社交

Suia.io 是一个基于 Sui 构建的社交 DApp，用户可以在其中探索各种社交场景，包括参加社区组织的活动，收集创作者创作的艺术品、纪念品、摄影作品。索取品牌创建的 NFT 贡献证明，以可组合的方式与 PFP 互动等等。

17、Suiet

官网：https://suiet.app/

定位：钱包

Suiet 是一个支持 Sui 网络的钱包服务提供商，支持浏览器插件以及移动端应用。

18、SuiNS

官网：https://suins.io/

定位：NFT，社交

SuiNS 的全称是 Sui Name Service，顾名思义，这是一款类似于 ENS 的链上域名服务，提供的尾缀为「.sui」。

19、Suiscan

官网：https://suiscan.xyz/devnet/home

定位：基础设施

Suiscan 是服务于 Sui 网络的区块浏览器，该浏览器提供了一个清晰的 dashboard，可帮助用户实时追踪 Sui 网络的链上变化。

20、Switchboard

官网：https://switchboard.xyz/

定位：预言机

Switchboard 是一个多链服务的定制化预言机协议，可帮助开发者构建通用的数据源，比如价格、体育、天气等等。
