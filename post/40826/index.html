原文作者：Crescent，律动 BlockBeats

Aptos 获得 2 亿美元融资后，在市场较为萧条的当下又完成了 1.5 亿美元融资，展现着 Aptos 非同一般的潜力。Aptos 曾表示有超过 100 个项目在网络上构建，包括 DeFi、NFT、游戏等，大量的生态还在开发当中。接下来，随律动 Blockbeats 一起了解，当前披露了哪些项目建设其中。

钱包

Fewcha Wallet

Fewcha 是 Aptos 上的 Web3.0 钱包，据其描述由一个拥有 15 年以上软件开发经验和 5 年区块链工作经验的团队开发。

Fewcha 用于管理数字资产，提供上线、购买、发送、接收、交换 Token，以及管理 NFT 的功能，具有高安全性。Fewcha 还支持 Chrome、Brave、Edge、Firefox 多个浏览器。

Twitter

Martian Wallet

Martian Wallet 是 Aptos 链上的钱包，可用于管理数字资产，以及访问 Aptos 区块链上的去中心化应用程序。Martian Wallet 强调易用性、安全性和速度，当前已推出 Chrome 上的插件钱包，之后将推出 ios 版本。除此之外，该项目还支持开发者铸造 NFT。

Martian Wallet 由 Martian DAO 开发团队构建，此外还在构建一个名为 Curiosity 的 NFT 市场。Martian DAO 本意在 Aptos 链上构建各种产品，不过当前其 Twitter 上显示 Martian DAO 项目暂时搁置。

Twitter

hive

hive 是一个多重签名钱包。

Twitter

ONTO

ONTO 是一个钱包，基于 DID 为用户在 Web3 的世界中提供管理数字身份、数据和资产的一站式移动端服务。用户可以使用 ONTO 管理自己的数字资产（包括 NFT）。

Twitter

VoltWallet

VoltWallet 是一个拥有易用性和安全性的无钥匙多链 web3 钱包。

VoltWallet 指出，所有用户可通过恢复人脸扫描转移资产和钱包。无单点故障，保障用户财产安全。若设备遗弃，用户可在新设备上用 Volt App 扫描人脸恢复资产。Volt 拥有资产继承和未来付款功能，用户可提前设置触发规则，触发后资产会转移到继承合同指定的账户，让资产无后顾之忧。用户可以在一个账簿里根据使用目的设置不同的钱包名称，支持多账户，且 Volt 允许用户共同管理账簿，灵活管理业务。

当前 Volt 已经在 Web、IOS、Android 三端上线。

Twitter

DEX（AMM）

Liquidswap

Liquidswap 作为 Aptos 的第一个 DEX，是 Pontem 以 Uniswap 和 Curve 为模型，在 Aptos 上开发第一个生产级自动做市商（AMM）。

Pontem Network 是一家产品开发工作室，团队成员分布在美国、迪拜、以色列和俄罗斯。在过去几年中一直在为 Diem 区块链使用其 Move 编程语言构建产品。随着 Diem 的停摆与 Aptos 的建立，Pontem 开始和 Aptos 合作，为其构建基础产品。据其表示，Pontem 团队至少有四年使用 Move 方面的知识经验，将为 Aptos 创建一个产品开发工作室，包括协议、开发人员工具和基础设施三个领域。

Pontem 将与现有的 dApp、基础设施提供商和 L1 合作，帮助其学习 Move 语言，孵化原生于 Move 的应用。除了 DEX 以外，Pontem 还将为 Aptos 构建浏览器代码编辑器、钱包、EVM 等其他 Dapp、开发工具与基础设施。Move IntelliJ IDE 和 Move Playground（用于 Move 的浏览器代码编辑器）都是 Pontem 构建的工具，开发人员可以通过其构建完整、成熟的 Move 程序。还有 Pontem 开发的 Move VM，可实现跨链互操作，兼容 Polkadot 和 Cosmos 等区块链。

Liquidswap 同大多数 DEX 一样，使用 X*Y = K 恒定等式定价，也可供用户添加流动性，从被交换资产的交易费用中赚取被动收入。之后该协议还有可能连接 Solana、Fantom 或 AVAX 等网络。

Saber Labs

Saber Labs 是一个为金融的去中心化未来做出贡献的建设者集体，专注于建设 DeFi 基础设施协议和开发人员工具。当前已建设完 Saber、Quarry、Tribeca、Goki 四个协议。Saber 是一个基于 AMM 的 DEX，Quarry 是用于流动性挖矿的基础设施，Tribeca 是用于 DAO 的投票托管治理协议，Goki 是用于密钥管理和安全性的协议构建工具，推出了「WalletKit」钱包连接器库与「Smart Wallet」具有 Timelock 功能的多重签名钱包两种产品。

这些都是基于 Solana 创建，现团队开始对 Aptos 进行探索。

DEX（订单簿）

Econia

Econia 是一个开源且去中心化的中央限价订单簿 (CLOB) 交易平台，用户可从中交易数字资产。Econia 提出它与其他类似的交易平台不同，例如 Solana 上的 Serum DEX，几乎无法提供网络规模的性能。具体来说，比如市场参与者们同时提出两笔订单，Econia 可以使这两笔交易同时清算，而不是串行处理。

Econia 利用 Aptos 区块链的关键技术创新，也就是称为 Block-STM 的 optimistic 并发执行方法，来实现其并行交易。

流动性质押平台

Zaptos Finance

Zaptos Finance 是 Aptos 上的流动性质押平台，由微软、Snapchat 和亚马逊等科技巨头的人才开发，据其披露，曾获 Aptos，Big Brain Holdings 与 Builders Capital 的投资。

其平台上当前可用 Martian 与 Fewcha 钱包进行链接，用户可从中质押自己的资产来获得每日质押奖励，Zaptos 支持用户质押任意数量的 Token，不设最低限度。

Zaptos 中有 3 种主要 Token：ZAPTOS（简称 ZPT）、stAptos、zAptos。

其中 ZPT 是协议治理 Token，协议 20% 的收入将归 ZPT Token 持有者所有，Token 持有者在该协议中质押他们的 Token 才能获得奖励。

stAptos 则是质押 Aptos 的交换 Token，也就是说用户若质押 Aptos Token，则会以 1:1 的比例收到 stAptos。须至少存储 1-2 天，用户赎回时，可获得原本的 Token 与产生的利息，如在 1-2 天之前取消抵押会导致用户扣除 0.3% 的取消抵押费。

zAptos 与 stAptos 相似，不同的是交换 zAptos 需长期质押 Aptos Token，为期一年，赎回时可获得更多奖励。

该协议原生有两种 Token，分别为 NAMI 和 TLP。NAMI 是其治理 Token，将用于在其 DAO 中为管理决定投票。TLP 是平台的流动性提供者 Token。用户可质押 NAMI 和 TLP 以获得奖励。

消息传递协议

Dialect

Dialect 是一个用于智能消息传递的协议。可用于 Dapp 内通知，如通知存款成功等，以及用户通过钱包到对方钱包进行聊天，亦可用于电子邮件、短信等，强调消息和通知的及时、有针对性。

曾构建于 Solana，并为其提供 15 个以上最受欢迎的 Dapp 和钱包提供通知及消息传递。现正在 Aptos 链上进行构建。

Twitter

Notifi

Notifi 是 Dapp 在 Web3 中与用户互动的消息传递基础设施，与 Dialect 相似，可将通知和消息嵌入到开发者的去中心化应用程序中。

Notifi 曾从 Hashed 和 Race Capital 等投资者处筹集了 250 万美元的种子轮融资。平台采用简单的 API 设计，易于集成，简化了 Web3 消息传递的通信问题，可定制 Dapps 和 Web3 应用程序的高级功能，使开发者通过多渠道通信更好地与用户互动。Notifi 曾建设于 Solana 和 NEAR 中，当前也开始基于 Aptos 构建。

Twitter

跨链互操作

Wormhole

Wormhole 是一种跨链消息传递协议，由 Guardian 节点网络保护。可连接到多个链，包括 Ethereum、Solana、Terra、Binance Smart Chain、Polygon、Avalanche、Oasis、Fantom、Karura、Celo、Acala 和 Aurora。

通过 Wormhole 可以使不同链上的应用程序更好地协同工作。比如，用户在 Solana 上的借贷协议可以轻松利用以太坊上的保险协议。虫洞通过一条链发出消息，这些消息由节点的守护者网络观察并验证。验证后，将此消息提交给目标链进行处理。

也就是说开发人员在 Wormhole 之上构建协议，可以使用来自其他链的数据、资产和消息。

Twitter

预言机是 DeFi 基础设施的关键组成部分，使智能合约能够与现实世界的数据进行交互。Switchboard 是一个社区驱动的去中心化预言机网络，允许任何人发布链上数据，供开发人员进行构建智能合约。Switchboard 建立在 Solana 之上，现已开始在 Aptos 上布局。

DAO 支付工具

Paymagic

Paymagic 是一种 DAO 和加密团队的支付工具。通过 API 支付赏金、赠款、工资单和奖励，可以使 DAO 的奖励和支付自动化。

Paymagic 认为许多 DAO 很难跟踪捐款、按时付款以及让用户签署多重签名，其通过 Gnosis Safe 保护用户财产安全，定制一个易于使用的应用程序发送批量转账、空投、归属时间表、流式支付等。
