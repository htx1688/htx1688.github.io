NFT 交易市场将再迎新选手，DEX 聚合器1inch近期发起了创建 NFT 交易聚合器的提案投票，不出意外或以高票通过。截至发文，1inch发起的提案支持率已高达 97.8% 的支持率，投票将于 3 月 26 日结束。

随着 Uniswap、0x、SushiSwap、1inch等主流 DEX 已经或即将入局 NFT 交易市场，DEX 新一轮攻防战已经开启？

主打免费和限价单等特色，已获高票支持

为了保持在 DeFi 领域的竞争力，1inch社区发起创建一个无平台费用的 NFT 市场聚合平台提案。Iinch 认为，成功的社区市场可为其生态系统和 DEX 聚合器产品带来流量，其希望通过构建具有丰富用户体验的1inch原生市场来降低社区成员的费用，并利用自身在 DeFi 领域的优势在 DeFi、NFT 和Web3生态来获取更为广泛的市场份额。

该提案指出，当前 NFT 的交易量已较 2021 年和 2022 年的 NFT 牛市出现了明显的下降，同时 Blur 等相关挖矿奖励的 NFT 平台已经抢占了销售额和市场份额。而 NFT 交易聚合器有效性的最佳例子则是在 2021 年和 2022 年的牛市期间，当时市场份额主要分布在 Gem 和 Genie（现在已分别被 OpenSea 和 Uniswap 收购）上。

不过，这种单纯通过聚合交易的方式并没有完全解决 NFT 流动性问题，特别是长尾资产持有者缺乏交易动力，而 Blur、X2Y2等交易挖矿的激励方式进一步赋予了资产交易的流动性。

虽然在1inch看来，现阶段的 NFT 市场极具挑战性，但其 DEX 聚合器在发布之前也面临同样问题。1inch希望通过在 NFT 聚合平台上引入大型做市商作为 NFT 做市商，并通过将限价单功能、额外产品重叠和与现有网络进行连接，来为 NFT 创造更多的流动性进并推动做市商更加去中心化。

1inch的 NFT 市场聚合器旨在为买卖与交易 NFT 提供一个综合聚合平台，试图通过聚合各种市场列表来简化购买和管理 NFT 的过程，NFT 市场在 MVP 阶段的所有交易与上市服务费将归入 DAO 财库。同时，该聚合器还提供高级搜索和过滤选项功能，可帮助用户根据特征类型、地板价、稀有度和创建者等详细信息找到特定的 NFT。此外，1inch还计划提供一个统一钱包，允许用户在多个市场存储和管理他们的 NFT。

而考虑到1inch的现有平台，NFT 聚合器还将允许用户将他们的 NFT 与其他加密货币交换，比如以太坊和任何其他 ERC-20 代币，旨在为 NFT 市场提供了更多流动性，并为投资者提供一条兼具去中心化和安全的资产交易路径。

当然，要想打造竞争力就必须与X2Y2等竞争对手的费用持平甚至更低，该 NFT 市场聚合器将不收取平台费用，且将支持1inch、OpenSea、X2Y2和 Looksrare。同时，1inch还将利用该协议的限价单协议（1inch的无 Gas 限价单功能允许用户设定特定价格目标和交易条件进行 swap，当市场价格与设定的数据匹配时便可完成交易。）为 NFT 领域带来真正的限价单，此功能同样不收取任何额外费用。

举个例子，当 Bored Ape Yacht Club NFT 持有者想出售手里的 NFT 资产时，在价格未达预期且避免因提前上市产生额外的深度和买入墙的情况下，NFT 限价单功能可允许持有者设定在任何理想价格的情况下自动上架。再例如，当用户持有的 NFT 正处于崩盘，且其无法使用互联网，就可通过创建止损限价单让其 NFT 资产在通过 TWAP 计算的地板价触及时自动上架。

1inch NFT 市场聚合器的技术支持则是由全栈式Web3.0 解决方案 Solidity.io 提供，并进行了开源，旨在保持社区的安全性、去中心化和透明度。Solidity.io 专注于支持一些最大的品牌、协议和初创公司开发Web3.0 的软件解决方案，是个活跃的 DAO 参与者，已参与了多个 DAO 的创建，其中包括知名的 ApeCoin DAO。

接连入局 NFT，多家 DEX 同台竞技

随着 NFT 市场规模与日俱增，且加速渗透至更大范围的用户群体，越来越多机构加快了入局 NFT 领域的步伐，其中就包括不少 DEX 平台。借助 DeFi 领域的优势，这些 DEX 在 NFT 领域表现究竟如何？

Uniswap

作为 DEX 龙头项目之一，Uniswap 于 2022 年 6 月宣布收购 NFT 聚合市场 Genie，并在几个月后正式上线，支持 OpenSea、X2Y2、LooksRare、Sudoswap 等平台。不过，DEX“顶流”Uniswap 在 NFT 领域的表现却不尽人意。NFTScan 数据显示，截至 3 月 23 日，Uniswap NFT 市场上线至今仅产生超 4900 ETH 的交易额，交互钱包数仅 2.8 万，NFT 交易数量约 3 万个。尤其是今年以来，日交易量大部分不足 100 ETH，近期更是仅为个位数。

不过，Uniswap 似乎意识到自身吸引力不足，开始进行功能优化。比如，近期 Uniswap 上线了“使用任意 ERC 20 代币购买 NFT”功能，未来版本还将支持把多个 ERC-20 代币捆绑到一次购买中。

0x

0x是一种开放、以太坊上支持点对点资产交换的协议，它能够在以太坊网络中实现 DeFi 应用。NFTScan 数据显示，截至 3 月 23 日，0x的总交易额已超 4200 ETH，NFT 出售数量约 5.2 万个，交互钱包数超 6.5 万。整体来看，0x的日交易量基本均为个位数。

此外，SushiSwap 此前也宣布推出去中心化品牌授权和孵化器框架 Sushi Studios 也计划在第一季度推出 NFT 市场。

整体来看，现阶段这些 DEX 尚且不能成为 NFT 的流动性释放利器，而1inch入局后是否会有所改变，一切还需等其正式上线后才能知晓。
