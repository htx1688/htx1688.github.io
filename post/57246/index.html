摘要

  • 自 Lybra V2 发布之日起，用户将有 1 个月的时间从 V1 迁移到 V2。

  • 所有在V2挖矿计划启动第一周将持仓从V1迁移到V2的用户都将获得奖励。

  • 单击此处跳转至有关如何将 LBR 代币从 Lybra V1 迁移到 V2 的说明。

  • 单击此处跳转至有关如何将 eUSD 从 Lybra V1 迁移到 V2 的说明。

  • 单击此处跳转至有关如果您在 Lybra V1 上（质押）LP 该怎么做的说明。

  • 将 esLBR（未归属或归属）从 V1 迁移到 V2 无需执行任何操作。单击此处跳转至该流程如何运作的说明。

  • 要继续成为 CR 监护人、赎回提供者或清算提供者，只需按照与 V1 完全相同的方式在 V2 上注册相同角色即可。

介绍

我们很高兴地宣布 Lybra V2 将于 8 月底推出！Lybra V2 将代表我们去中心化、全链、LST 支持的稳定币解决方案的重大变革。用户将可以使用一系列新功能，包括全链功能、扩大的 LST 选项作为抵押品、资金安全的进步以及支持eUSD 挂钩的新机制，同时支持 LBR 的价值。

对于某些用户，需要采取措施将其持有的资产从 V1 迁移到 V2。他们需要在 V2 启动后 1 个月内采取这些行动。

本文解释了LBR 持有者、esLBR 持有者、质押 LP 的持有者以及 eUSD 铸币者需要做什么，以确保从 Lybra V1 顺利迁移到 V2。本文还将解释CR 监护人、赎回提供者和清算提供者需要采取哪些行动才能继续在 V2 上发挥作用。

所有在V2挖矿计划启动第一周内迁移持仓的用户都将获得奖励。

首先，这里总结了哪些令牌和函数需要执行迁移操作，以及您可以在本文中找到它们的迁移说明：

LBR 代币：在此处查找迁移说明。

eUSD：在此处查找迁移说明。

（质押）LP：在此处查找迁移说明。

esLBR：无需采取任何措施；自动迁移过程解释在这里。

继续成为 CR Guardian：只需像在 V1 上一样在 V2 上注册成为 CR Guardian。

兑换提供者：只需按照与 V1 相同的方式在 V2 上注册为兑换提供者即可。

清算提供者：按照与 V1 相同的方式在 V2 上注册为清算提供者。

现在，让我们分别深入了解每个令牌和函数的迁移过程的详细信息：

  1. 如何将 LBR 代币从 LYBRA V1 迁移到 V2

如果您是LBR持有者，迁移过程将非常简单，与原来的V1迁移基本相同。您如何迁移取决于您的资产是在中心化交易所 (CEX) 还是在您自己的钱包或去中心化交易所 (DEX) 中。以下是它在不同场景中的工作方式：

a) 中心化交易所（CEX）上 LBR 的迁移流程

在这种情况下，无需执行任何操作。CEX 将为您完成所有工作，您的 LBR 代币将自动从 V1 迁移到 V2。

b) LBR 在您自己的钱包或去中心化交易所 (DEX) 中的迁移流程

这将是一个非常简单的过程，只需单击按钮即可完成。当 V2 启动时， Lybra Finance 网站上将会有一个迁移按钮。您所需要做的就是单击此按钮，连接您的钱包并按照出现的步骤操作。

2. 如何将 eUSD 从 V1 迁移到 V2

将您的 eUSD 持有量从 Lybra V1 迁移到 V2 的过程涉及更多步骤，但仍然非常简单。根据您是通过铸造生成 eUSD 还是通过提供 LP 收到 eUSD，有不同的选项。以下是针对不同类型 eUSD 持有者的不同说明：

a) eUSD 铸币商

对于拥有自己铸造的 eUSD 的人来说，有两种选择：

i) 手动迁移您的 eUSD

您可以通过以下方式将 eUSD 从 Lybra V1 迁移到 V2：

  • 如果您已经花费了任何 eUSD，请从 Curve 池中将其买回。*

  • 偿还你的贷款。

  • 提取您所有的 stETH 抵押品并平仓您的 V1 头寸。

  • 通过Lybra Finance 网站手动连接到 V2 平台。

  • 批准 V2 合同。

  • 存入您的抵押品。

  • 最后，在 V2 上铸造 eUSD。

*请注意，对于那些已经花费了 eUSD 并将其回购的人来说，最初可能会以溢价进行交易。因此，您可以选择等待 eUSD 挂钩汇率回升至正常汇率。溢价最终可能会回落，因为在 V2 推出后，eUSD-V1 可能会停止每天重新调整基准。因此，您可以选择等待溢价回落，然后在此时回购您的 eUSD 并按照上述步骤操作。

或者

ii) 将您的 eUSD 保留在 V1 上

您还可以选择将您的 eUSD 保留在 V1 上。来自 V1 的 eUSD 贷款仍然可以铸造 V2 esLBR，并且 V1 TVL 将被计为 V2 TVL 中的金库。如果您选择这样做，则无需执行任何操作。

b) 非铸币者

一些用户可能已经在公开市场上购买了 eUSD（而不是铸造它）并提供流动性。在这种情况下，您可以选择利用迁移阶段可能出现的 eUSD 溢价，并以临时溢价出售您的 eUSD。

否则，如果您在 Lybra V1 上拥有（质押）LP，您可以按照以下说明进行操作。

3. 如果您在 LYBRA V1 上持有（质押）LP该怎么办

如果您在 Lybra V1 上有 Staked LP，那么您只需要遵循两个步骤：

  • 在 Lybra V1 上解除您的 LP 质押。

  • 然后从 Uniswap V2 / Curve（LBR/ETH、eUSD/USDC 池）中移除流动性。

注意：如果您在秘密快照之前没有删除 LP，那么您将需要执行以下操作：

  • 对于 LBR：随时删除 V1 流动性，然后按照上面“如何将 LBR 代币从 LYBRA V1 迁移到 V2”部分中的说明将您的 V1 LBR 迁移到 V2 LBR。

  • 对于 eUSD：随时删除 V1 流动性，然后按照上面“如何将 eUSD 从 V1 迁移到 V2”部分中的说明将您的 V1 eUSD 迁移到 V2 eUSD。

4. ESLBR 将如何从 V1 迁移到 V2

无论您的 esLBR 已归属或未归属，都无需采取任何操作即可将其从 V1 迁移到 V2。自动过程的工作原理如下：

  • 如果您有未归属的V1 esLBR，则会在未指定的时间进行快照，然后V2 esLBR将自动按比例转移。

既得 esLBR 示例：例如，如果您在 V1 中未归属 100 个 esLBR，那么您将在 V2 中收到 100 个未归属 esLBR。

  • 同样，如果您有归属 LBR，同样会在未指定的时间拍摄快照，您将收到按比例的 esLBR 和 LBR。

归属 esLBR 示例：例如，如果您有 100 个 esLBR 归属（比方说，归属 33 个 esLBR，归属 67 个），您将收到 33 个 LBR 和 67 个 esLBR（归属期从第一天开始）。

  • 任何现有的提升的 esLBR（锁定 3 个月/6 个月等）都将作为 v2 中的 esLBR 空投，无论提升的持续时间如何。

提升的 esLBR 示例：例如，假设用户提升并锁定了其 esLBR 12 个月。迁移时，Boosting 总共排放了 1,000 个 esLBR，通常在一年内领取。但实际上，这个发出的esLBR将被空投，没有锁定时间。

注意：一旦宣布迁移，Boost 功能将暂停。即使有人试图直接从合同中“提升”，在现阶段，提升的排放量也不算数。他们不会在 v2 中获得任何额外的 esLBR 空投，并且会放弃他们现有的提升已经积累的任何 esLBR。

5. CR 监护人/赎回提供商/清算提供商的迁移说明

对于在V1上执行以下功能的用户，您只需在V2上再次注册这些功能，方法与在V1上完全相同：

  • 成为 CR Guardian：按照与 V1 相同的方式在 V2 上注册成为 CR Guardian。

  • 成为兑换提供者：按照与 V1 相同的方式在 V2 上注册为兑换提供者。

  • 成为清算提供者：按照与 V1 相同的方式在 V2 上注册成为清算提供者。

结论

我们迫不及待地想让您开始使用 Lybra V2 提供的所有令人兴奋的新功能。我们希望本指南将使现有 V1 资产迁移到 V2 的过程在时机到来时尽可能顺利和简单。如果您遇到任何问题，请通过Discord 联系我们寻求帮助。V2 见！
