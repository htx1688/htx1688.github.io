Polygon 在2022年取得的成就可以用一个词来概括：影响力。并且 Polygon 能取得如今的成就离不开 Polygon 社区的支持与帮助，包括创作者、建设者、合作伙伴以及粉丝。我们一起来回顾 Polygon 生态在2022的发展。

首先，Polygon 生态增长迅速：

  • Polygon 网络上部署了778,000个智能合约（同比增长152%）

  • 共有234,000名合约创建者（同比增长637%）

  • 共有200,000,000个独立地址（同比增长54%）

  • 共处理了960,000,000笔成功的交易

  • 以太坊合并后， Polygon 年度二氧化碳足迹消除了60,953吨

  • Polygon zkEVM 最终测试网

不仅如此，Polygon 团队还与一些世界上最大的公司建立了合作伙伴关系，以在 Polygon 的基础上发展。包括了 Adobe 、迪士尼、Meta、NFL、NuBank、Reddit、Robinhood和星巴克等。

一月份：

  • Prada 以及 Adidas 合作于 Polygon 上发布一系列 NFT。

四月份：

  • Stripe 通过 Polygon 支持 $USDC 支付，Stripe 是线上支付商，全球超过10万家企业都在使用 Stripe 支付服务。

五月份：

  • eBay 与 OneOfNFT 合作在Polygon 上推出 NFT。

六月份：

  • Polygon 与倩碧合作，推出第一个 Avatar 化妆 NFT 系列。

  • 英国豪车品牌宾利宣布将于九月份在 Polygon 网络上发行限量 NFT（共208份）。

  • 奢侈品牌宝格丽和 Polygon 合作推出最新高级珠宝系列 NFT。

七月份：

  • Polygon 与英国初创公司 Nothing 合作，将 Polygon 技术整合到 Nothing 的智能手机 Nothing Phone （1） 中。同时还推出基于 Polygon 的 NFT 忠诚度计划。

  • Polygon 参与迪士尼加速器计划。

  • Acentrik 是梅赛德斯奔驰的一项战略举措，正在构建一个去中心化的数据市场，使企业能够轻松地跨域共享数据并从中获利。

八月份：

  • Reddit 推出了由 Polygon 提供支持的数字收藏品。 Reddit 用户铸造了超过500万个 NFT 头像，并创造了超过1000万美元的收入。

九月份：

  • 移动端股票交易软件 Robinhood 选择 Polygon 作为启动其 Web3 钱包测试网的首选网络。

  • 星巴克在 Polygon 上推出 Starbucks Odyssey 忠诚度计划，目前测试版本已在 Polygon 上线。

十月份：

  • Polygon 与 Nubank 合作，在 Polygon Supernets 推出其 Nucoin 代币还有忠臣度计划。

十一月份：

  • Instagram （Meta）开启 NFT 功能，允许部分美国创作者 他们创建或购买的 NFT，并表示这项功能很快将会向所有 Instagram 用户开启 。

  • 摩根大通和新加坡的监管机构在 Polygon 上进行第一次实时 DeFi交易。

  • Nike 在 Polygon 上推出数字创作平台 dotSwoosh, dotSwoosh 是一种全新的社区体验，旨在让用户有机会共同创造耐克的未来。用户可以在平台上创作、交易和收集自己喜欢的虚拟鞋子、球衣或配饰。

十二月份：

  • 华纳音乐与 Polygon 合作，共同开发数字收藏平台 LGND Music。 LGND Music 允许用户可以随时随地播放音乐数字收藏品或“虚拟黑胶唱片“。

同时，Polygon 团队创新不断，推出各种方案来解决随着 NFT 和 DeFi 需求暴增所带来以太坊主网拥堵问题：

三月份：

  • Polygon ID 的推出，这是一个 Web3 的自我主权、去中心化和隐私的身份。

四月份：

  • Polygon Supernets 推出，这是一种允许创建特定于应用程序、可扩展和可定制的区块链网络的技术。

五月份：

  • Polygon Nightfall主网测试版上线。Polygon Nightfall是企业最高效的区块链解决方案，通过前沿的Optimistic-ZK混合方式，企业可以获得以下的优势：

-安全性：交易存储都在以太坊主网上

-效率性：交易费用是最低的

-保密性：交易不受第三方影响

  • Polygon 推出 Polygon NFT Minter，让用户在 Polygon 上可以更简单、更快捷地创建 NFT。

六月份：

  • Polygon Avail 数据可用性解决方案进入测试网阶段。

七月份：

  • Polygon zkEVM 发布，这是一种提供零知识 (ZK) 证明的可扩展性优势并与以太坊虚拟机 (EVM) 兼容的解决方案，并于十月份推出公共测试网，目前（十二月份）已经推出最终测试网。

八月份：

  • Ploygon 推出开源 Plonky2。 Plonky2 是一个零知识证明系统，代表了 ZK 技术的重大突破。Plonky2 可以提供难以置信的快速证明以及极其高效的递归证明。

  • Polygon Miden 推出 Miden VM v0.2。Polygon 旗下基于 STARK 的以太坊兼容解决方案 Polygon Miden 推出 VM v0.2。

而关于 Polygon 生态中功不可没的开发者，他们取得的成就有那些呢？

Polygon Guilds

目前，有超过70个 Polygon Guilds 正在积极组织聚会、线上/线下会议、研讨会、演讲和其他活动，以了解 Polygon 和 Web3 的未来。

Polygon Advocate

Polygon Advocate 计划使对协议及其开发充满热情的社区成员能够帮助其他人了解有关在 Polygon 网络上构建的所有信息——并利用地球上最好的以太坊扩展解决方案。通过突出教育和社区，倡导者在帮助所有建设者方面发挥了重要作用。

并且在2022年，有超过7000名开发人员参与了由 Polygon Advocates 组织的全球研讨会和培训计划。

未来展望

2023年将是 ZK 技术之年。Polygon 团队准备在第一季度推出 zkEVM 主网的同时， Polygon Zero 和 Polygon Miden 团队也在快速地开发他们的 ZK 解决方案，以便在2023年首次亮相。2022 年是 Polygon 的突破之年。Polygon 将一些最大的财富 500 强公司带到了 Web3，并为未来的增长奠定了技术基础。在接下来新的一年里，这一轨迹将保持不变。
