评估 NFT 当前和未来的应用场景

在你心目中，NFT 代表了什么？买 NFT 的目的是什么？

一些人可能是为了新奇或者当作收藏，另一些人可能是为了 NFT 的某种功能属性，比如用在链游中的道具。bfrenz 的家人们也曾说过：「买 NFT 就像买包，纯属消费品。」

在 Web3 社区，很多人也会称持有 BAYC、Crypto Punks 头像的用户为大佬，不管你知不知道他是干什么的，一张头像已经说明了他进入这个行业的决心（毕竟花了钱的），这种情况也代表了买 NFT 的另外一种目的。

这篇文章， Messari 的分析师根据是否具有功能性这一大因素，把现有的 NFT 分为了两类，即低功能性的图片类 （PFP ）NFT，和强功能性的 NFT ，并且探讨了应用场景和内在价值。

NFT 届 OG、推特大 V Cobie， 最近将 NFT 称为「图片山寨币」 （ altcoins with pictures ），引发了一些争议——那事实是这样吗？如果是这样，这对 NFT 来说真的是一件坏事吗？如果没有，这套 NFT 标准如何在美学之外被使用？

文章重点概述

  • 最大限度地增加投机因素可以激发飞轮，使项目有机会在长期内创造内在价值。

  • 目前大多数的 NFT 都几乎没有实用用途，但长期趋势是在 NFT 项目中增加功能和属性。

  • 图片类 NFT 项目需要依靠一个豪华的框架来评估其价值

  • 功能基础的 NFT 情况预测未来生态系统的增长将会是由这些 NFT 所支持的。

Cobie 最近批评 NFT 社区只不过是一群有着「图片山寨币」 的投机者。某种程度上来讲，他是对的。就目前的情况而言，NFT 行业大部分是 「JPEG」 的投机行为。

如果是这样的话，有两个问题需要被回答：JPEG 能否合理地保留任何价值？展望未来，ERC-721 NFT 标准会在图片之外找到更多用例吗？

ERC-721 是一个开放的标准，它描述了如何在 EVM ( 以太坊虚拟机 ) 兼容的区块链上构建非同质化代币；它是非同质化代币的标准接口；这是一套规则，使其易于与 NFT 交互。NFT 不仅是 ERC-721 代币，还可以是 ERC-1155 代币。

重视美学

想了解一系列可收藏的猴子 JPEG （aka 无聊猿）为何能够达到数十亿美元的市值的话，来自 eGirl Capital 的 DegenSpartan 给了一个最合理的估值框架。这个价格主张采用了传统的估值框架，并在顶部附加了投机因素。

对于 NFT，重要的不一定是内在价值，而是投机因素的边际定价。这似乎是一个荒谬的命题，但在加密领域并非没有先例。以比特币为例，因为其低收益率，缺乏储蓄以外的其它现实世界作用，有人可能会说，比特币的价格主要是由过去十年流动性急剧增加而放大的投机性溢价所决定的。

尽管比特币有一部分是由投机开始的，但早期支持者大力支持 BTC ，让其有时间找到真正的应用场景（比如「非官方丝绸之路」、价值储存、通货膨胀对冲等）。正如比特币的发展历程所示，最大化投机因素的长度和幅度可以最终为给定资产创造内在价值。

投机因素的最大化依赖于围绕叙述的社会共识的增长。随着社会对资产价值的共识增加，该资产的价格也会上涨。图片类是产生社会共识的完美选择，因为人们喜欢图片。

人们也喜欢分享图片，传播性很高。这样做的过程中，人们分享了关于他们所拥有的图片的重要性叙述。这些分享传播活动放大了 NFT 的叙述，就像可替代代币（ fungible tokens ）放大自己的叙述一样。

最后，所有代币，无论它们是否是 NFT，都大致相同——都可以归类为是一种社交代币。这个想法最适用于从投机中获得大部分价值的代币。

Hsaka：当你把所有的层次和抽象都去掉时，从本质上讲，每个代币只是一个社交代币。

超越美学的 NFT

考虑到注重美学的 NFT 数量奇多，未来的趋势可能是更多项目专注于基于高功能资产提供内在价值。

一个例子是域名 ENS —— 其中没有什么「审美价值」的用户名。诚然，有一些高度投机的 ENS 名称，例如 3 位和 3 个字母的 ENS。但在大多数情况下，这些域名获取成本低，且功能清晰。

其它致力于高功能身份解决方案的项目也包括 Aave 的 Lens Ptotocol（一个开放的、可组合的去中心化社交媒体底层协议.）Unstoppable Domains（是一家为用户提供购买和拥有不受审查的域名的公司。） 和 Civic（以太坊区块链上的非托管钱包，允许用户存储加密货币和身份证明。）。

对于其它新的 NFT ，还有很大的设计空间。例如，注重功能化的趋势正在扩展到房地产资产的代币化，这里也运用到了 NFT 。其中 Propy 等协议致力于将房地产契约上链。Vesta Equity 等其他协议允许房主将部分财产所有权作为 NFT 出售，与 NFT 持有者分享出售产生的任何利润。

除了房地产之外，NFT 更多功能的推动还发生在游戏、音乐等垂直领域。目前尚不清楚哪些垂直领域将为 NFT 带来新的高点。但很明显，未来几年在 NFT 的功能领域将进行历史性的大量实验。  

美学与功能

没有人知道基于美学的系列是否具有持续价值，或者 NFT 在 2020-2021 年的火热牛市是否会是第一个也是最后一个。美学 NFT 的案例将 Art Blocks 或 CryptoPunks 等收藏品视为奢侈品。在这种情况下，缺乏明显的功能与其说是错误，不如说是一种属性—— NFT 的主要功能是展示财富。也就是说， NFT 不需要额外那么多功能。

如果奢侈品框架适用于 NFT，那么这些收藏品必须适应人们随时间变化的而变化的品味。然而，如果这些项目中的走出了任何一个 NFT 项目，能在长时间内保持社会共识（至少是圈内共识），那么意义巨大，因为达到主流的类别定义艺术售价数百万，可持续的价值就产生了。

2021 年，Art Blocks、Punks 和 Bored Apes 在世界级拍卖行佳士得拍卖行以数百万美元的价格卖出了数百万美元的价格，就是这个迹象。如果这些藏品保持其地位，在 10 年、20 年或 50 年后会有什么价值？

高功能性的 NFT 就不面临品味变化而价值波动的这个问题，至少不需要直接处理。如果像 STEPN（一款 Web3 应用，致力于成为面向智能手机用户的领先的加密届健身游戏） ，这样的游戏需要球鞋 NFT 才能玩，玩家别无选择，只能购买 NFT，那么 STEPN 所提供的 NFT ，就有附加价值。这听起来很简单，但也许哟一个生态， STEPN 设法创建了一个环境，在这里，NFT 的使用对其产品的使用至关重要。产品增长了，底层 NFTs 的经济价值也在增长。

随着加密应用开始吸引更广泛的受众，高功能行的 NFT 的经济价值将大幅增加。在鼎盛时期，STEPN 拥有 100,000 名每日活跃用户 (DAU)。支持第一个具有 1000 万 DAU 的加密应用程序的 NFT 的总经济价值是多少？一亿？十亿？  

最后的想法

从长远来看，很难说什么会更有价值—— 「图片山寨币」（图片类 NFT ）还是高功能性的 NFT 。

有一个情况，大多数 NFT（功能性或其他方面）的估值都不排除价值归零的可能，就像它们的图片类 NFT 同类一样。

这并不意味着整个 NFT 的终结。山寨币（ altcoin ）运动在过去 10 年保持势头，即使 Litecoin、Namecoin、EOS 等不同的迭代来来去去。山寨币运动为以太坊、Solana，DeFi 和许多其它加密创新提供了动力。

2022 年 8 月仅标志着 NFT 有真正动力的 2 年左右。即使当前 NFT 迭代的大部分被淘汰，NFT 领域也足以在未来几年内产生重大发展。值得更多的探索。
