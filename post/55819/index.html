原文作者: Solaire, YBB Capital

引言

NFT 市场近期的低迷表现正在对长期投资者产生深远影响。一波又一波的抛售潮似乎正在席卷整个市场，NFT 市场持续低迷对长期持有者的情绪产生了负面影响，许多人纷纷抛售优质藏品。最近几周，钻石手持有的所有 Azuki 藏品中约有 4.63% 被出售，占市场上 Azuki NFT 总量的近 1.88% 。Bored Ape Yacht Club (BAYC) 的需求也出现下降，近 2% 的资产被钻石手抛售。NFT 市场自今年 3 月份的顶峰以来一直在经历一个艰难的阶段，日销量和独立买家数量大幅下降（数据来自 Cointime）。关于 PFP 叙事已经走到头的声音层出不穷，本文将回看 Azuki 历史去思考 Azuki 过去成功的因素和现阶段下 PFP 该解决的问题。

NFT，一种新的连接方式

NFT（非同质化代币）的概念最早可以追溯到 2012 年，当时在比特币区块链上出现了一种叫做彩色硬币（Colored Coins）的资产。这些彩色硬币可以代表任何非货币的东西，比如股票、债券、收藏品。

然而，真正的 NFT 技术是在以太坊区块链上发展起来的。2015 年，以太坊区块链发布，它的智能合约功能使得开发者可以创建自己的代币，包括非同质化代币。2017 年，CryptoPunks 项目发布，这是一种基于以太坊的，可以买卖的图片，每个图片都是独一无二的，这可以看作是 NFT 的早期形态，也是第一个 Profile Picture（PFP）类型的项目。

所谓 PFP 通常指的是头像类 NFT 项目，它的主要特征可以归纳成六点。

  1. 唯一性 ：PFP 项目的每一个代币都是独一无二的。这是由于它们都是 NFT，也就是说，每一个代币都有它自己的元数据和所有权记录，不能与其他的代币相互替换；

  2. 艺术性 ：大多数 PFP 项目都是艺术品。每一个代币都代表了一个独特的、数字化的头像或者角色，它们的艺术风格一致但特征各不相同；

  3. 社区驱动 ：PFP 项目常常围绕着一个活跃的社区运行。持有这些 PFP 的人不仅是投资者，他们也是社区的成员，参与各种活动，如在线讨论和线下聚会；

  4. 代表身份 ：在许多 PFP 项目中，持有 NFT 的人会把它们用作网络上的代表性图像，从而显示他们的数字身份和所属社区；

  5. 经济价值 ：由于每一个 PFP 都是独一无二且数量有限的，它们在市场上有可能被高价买卖。这就赋予了持有者一种潜在的经济利益；

  6. 可互动性和扩展性： 一些 PFP 项目可能会包含游戏元素，使得代币能够用于其他的应用或者平台上。这增加了 PFP 的可互动性和扩展性。

从文化及社交的角度看 PFP 的特征建立了一个新的连接方式，苏州大学传媒学院执行院⻓陈龙及其科研助理经羽伦在论文**《作为亚文化资本的 NFT 数字藏品及其交往实践意义》** 中提出，NFT 作为一种亚文化资本，正支配着青年群体的网络亚文化生产和交往行动。以趣味为基础货币，NFT 构筑起一个充满流动的，包罗万象的网络亚文化的圈层，区隔随之发生在不同圈层、代际以及社会内部其它共同体成员之间。围绕 NFT 数字藏品展开的交往实践亦存在新的意义。

其意义分为三点，身份、融合、连接。

  • 打破圈层的身份：NFT 打破了各种社交圈层的隔离。个体可以通过更换佩戴的 NFT 代表不同的身份，这些身份可以跨越不同的社区以及社交媒体。个体可以自由地在不同的圈层之间流动，展现他们的数字身份；

  • 融合价值与文化：人们既关注 NFT 的艺术价值（例如，作为一种投资或收藏），也关注其文化意义（例如，作为一种艺术表达或者精神传达）。这使得价值和文化的界限变得模糊，在以数字媒介与万物互联为基础的 NFT 中，价值与文化的界限消弭，二者生成互嵌模式，相融相生；

  • 弱关系连接的社区：人们因为共同的兴趣或利益而建立起联系，即使他们在现实生活中可能不会有任何交集。这些相对较弱的关系形成了一个巨大的网络，允许信息和资源的交换。换句话说，人们可能通过购买、交换或讨论 NFT 而与全球各地的其他人建立联系，这些联系本身可能很弱，但是聚在一起则形成了一个强大的社交网络。弱关系的连接被前所未有地激发，不同身份的群体在 NFT 带来的文化共鸣中完成关系的聚拢与升级，最终集结成以弱关系为核心的网络社会（社区）。

本文也将以身份、连接、融合这三个围绕 NFT 文化及社交视角展开的意义为核心回看 Azuki 的成功与崩塌。

Azuki，融合的魔力

Ready to take the red bean? ——Azuki 诞生前的推文

2022 年 1 月 2 日 Azuki 开售了，Azuki 从诞生开始就是一帆风顺的，并迅速成为与 CryptoPunks 和 BAYC 齐名的三巨头之一，不同于 CryptoPunks 的历史意义和 BAYC 对于西方市场的深刻理解。Azuki 的成功在于对艺术价值与文化的巧妙融合，作为一款具有浓厚日漫风格的 NFT，它的成功不仅在于其艺术价值（对其效益和绘画水准的认可），也在于它实现了东西方文化的成功融合。从绘画细节看，Azuki 采用了日漫的风格，加上美式的粗轮廓线以及各种东西方风格元素（比如嘻哈、刀剑、不同民族的服装）。这种混合艺术很好地体现出了其口号中所宣传的“致力于成为 Metaverse 世界中由社区建立、拥有的最大去中心化品牌”，以及价值观。

Azuki 价值观：

◼ 社区优先：一切都始于且忠于社区。当我们将其他东西优先于社区的那一刻起，一切都将荡然无存。我们不会允许这样的事情发生，因为社区永远是 Azuki 的第一价值；

◼ 成为行业顶流：Azuki 是创造性的、无畏的、坚韧的。追随潮流和逐步改进很容易，做到原创并标新立异要难得多。我们独辟蹊径；

◼ 渐进式过程：好事多磨。去中心化的行动不像中心化的行动那样能够很快地破除万难、一气呵成，而这祸福相依。去中心化并不意味着不能有领头人。核心团队将为所有的 Beans 提供护栏和资源以共同成长，但这不是一朝一夕就能够实现的。

欢迎世界上任何一个人加入，欢迎各位共创最大的去中心化品牌。何尝不吸引我们这些被去中心化愿景所吸引聚集的 Web 3 原住民。

时至今日，Garden（花园，Azuki 社区）里，已经拥有了 66 个由 Azuki 爱好者所创建的社区，这更证明了 Azuki 的融合魔力是如此神奇。

但 Azuki 后来的路却是异常坎坷，项目方的劣迹与理念的冲突导致了它们的崩塌，社区并不能拯救它们一次又一次。

Azuki 保卫战，连接的力量

一滴水即使落在热沙上也会迅速蒸发，但是千万滴水汇聚在一起就能成为大海。——马克思

在 Elementals 系列折戟之前，Azuki 其实已经经历过一次 Fud， 2022 年 5 月，Azuki 创始人 ZAGABOND（下文简称 ZAGA）在 Mirror 上发布的《A Builder’s Journey》( 建设者之旅 ) 一文中，自曝其团队在过去一年时间里曾先后运营了 CryptoPhunks、Tendies、CryptoZunks 三个项目，并以失败告终。

这次自曝在社区中引发了一场轩然大波，在 ZAGA 自己承认了是三个 Rug 项目的幕后后，Azuki 随后跳水腰斩至 10 ETH，但这场负面影响并没有持续太久，支持者认为创业失败并不是可耻的事情，反而 ZAGA 大胆承认失败以及敢于创新的精神非常重要，于是在腰斩后不久，Azuki 迅速反弹到了 16 ETH。

不过在我看来所谓创业失败并不可怕的论调其实不是这次翻盘的关键，而且这个论调偷换了概念，ZAGA 是把一滩烂泥丢给社区后 Rug 了，并不是简单的创业失败。如果失败后的态度诚恳那情有可原，但 ZAGA 只是在运营几周失败后留下了一条黑白公告就消失了。

Rug 在 Crypto 的世界里是极为敏感的词语，更何况是连续三个 Rug 项目的幕后。Azuki 真正翻盘的关键在于 Holder 连接足够强大，对 Azuki 艺术风格的喜爱以及利益关系将无数互不相识的弱关系群体牢牢连接在一起，组成一个强大无比的社区，在此刻做为一个共同体，内部不满的声音会因为连接而动摇并淹没，外部的 Fud 在社区 Holder 们真金白银拉起的地板价面前似乎也显得不值一提了。

Azuki 的第一次生死保卫战就这样在社区的捍卫下安全渡过，而将 ZAGA 多次指控和反讽为诈骗犯的 Not Larva Labs 创始人 Jeremy Cahen 也成了 Azuki 社区最大的公敌。

回看第一次 Fud 之所以能安全渡过，除了连接力量的强大，也是因为 Fud 的点还没有冲击到 Azuki 本身的价值。

Elementals，身份价值的破碎

稀缺性是所有商品的价值之源。——亚当·斯密

时间来到今年 6 月，Azuki 拉斯维加斯线下大会通过一个狂拽酷炫的西部风动画短片揭开了新系列 Elementals 的面纱，短片的播出瞬间引发社区的高潮，Holder 们几乎在社交媒体把这个消息刷屏了。

不过几天后的开图却把整个 NFT 市场拉到冰点，所谓期待有多高，失望就有多深。Elementals 的开图和预告片里展示的各种新元素几乎毫不相干，不过这不是最要命的，最要命的是新系列对于 Azuki OG 系列的元素几乎是一比一的复刻，可以说是官方推出的“盗版”系列，并且由于技术问题导致了元素完全重复和错乱的图片出现。

而项目方对于图片重复这种情况的补救是直接通过合约中的 URl 把已经发售不可篡改的 NFT 图片直接改图了，关于 Elementals 元素和 OG 系列重复的问题更是发布公告直接承认了，他们就是这么设计的，并且还要根据 Beanz（小豆，Azuki 的次级系列）再发布一个绿豆。

这波迷之操作瞬间引来了各路的群嘲，曾经坚不可摧的社区在一夜之间分崩离析，部分社区核心成员甚至组建了 Azuki DAO 和 20000 E DAO 准备对创始人提起诉讼以追回 ZAGABOND 在 Elementals 发售中获得的 20000 ETH。比较戏虐的是法律援助和证据提供可能还需要 Azuki 社区公敌 Jeremy Cahen 的帮助。

虽然 Chiru Labs（Azuki 开发团队）很懂得怎么样才够酷，但他们不明白 Azuki 带来的身份价值，正如 ZAGA 在事后采访中所说的高估了自己，低估了社群 。Elementals 系列对于 Azuki 经济价值以及身份价值的伤害是致命的，做为 PFP 项目的大蓝筹 Azuki 曾是社交媒体上最棒的头像之一，在过去你可以经常看到 Twiter 上著名的 KOL 以及 Web 3 的资深玩家戴着它们四处招摇，它象征的不仅是最强蓝筹社区的一员，也是财富的象征。但当同一个项目方发布了一个元素几乎一模一样的低价系列时，这些象征都失去意义了，我相信即便是 ZAGA 也不能辨别社交网络上 Elementals 系列头像与 Azuki 系列头像的区别，佩戴 Azuki 头像（也可能是 Elementals 头像，XD）在今天沦为了其它社区的笑柄。

ZAGA 在 Elementals 事件中几乎把所有不该做的事情都做了，不仅践踏了 NFT 的所有意义也背叛了他们价值观中对社区的许诺，Azuki 强大的连接从核心开始破碎崩塌。

博弈带来的平衡，达摩克利斯之剑永远悬在我们头上

在写这篇文章前我看过很多 Azuki 崩塌相关的思考文章，大多是从 IP 和叙事的角度展开，去探讨如何做好 IP，如何找到更棒的叙事以及应用场景，但如何做好一个 IP 以及如何扩展 PFP 叙事，以大部分 NFT 项目团队的体量以及目前 NFT 极度熊市下版税收入的骤减情况来看这个探讨毫无意义。并且唯一有能力做到这件事的 Yuga Labs 已经做了一个的示范，虽然 Yuga Labs 做为先行者的魄力以及行动力是令人敬佩的，但以旗下 BAYC 所展开的元宇宙叙事也并没有挽救 NFT，新项目新叙事不断的发行换来的也只是 BAYC 及衍生系列不断的下跌。

至于寻找新的应用场景，新的 NFT 叙事，目前看来都没有一个比 PFP 更好的答案，从文化和社交的角度看 PFP 这个叙事天然具有传播性和凝聚力，这是其它类型 NFT 做不到的。

PFP 项目现阶段极度中心化的问题，其实并不是什么秘密，我们相信项目方不会作恶，只是基于博弈的角度来说，做为利益共同体的项目方并没有理由去做改图或者恶意增发的事情，但遗憾的是这个博弈与项目方的发展相悖。NFT 的所有意义都是建立在去中心化的前提上，然而项目方对项目的命运有绝对主宰权，一个几人或者十几人的小团队怎么保证每一次重大决策都不出现失误？我购买的这张图究竟是我的还是寄存在项目方电脑里的？如果 NFT 的稀缺性和不可更改的特性从来都是中心说得算，那与其期待小团队不断创造奇迹，我更愿意购买华特·迪斯尼的股票。

解决信任问题以及权利的下放才是当前要做的事情，虽然连接起社区的是项目方所创作的文化，但任何一个项目都需要社区铸起高度，Holder 为社区贡献了一切，当社区核心出现问题时他们甚至没有知情权。
