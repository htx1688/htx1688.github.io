原文标题：《什么是 NFT 质押？》

撰文：DappRadar

NFT 不仅仅是人们在互联网上交易的数字艺术品。如今，NFT 质押也是获得真正回报的可行工具。自 2014 年 5 月第一个 NFT 在纽约市新博物馆铸造并出售以来，越来越多的实用程序被添加到区块链代币中。来了解下如何将 NFT 质押添加到您的 Web3 工具包中。

如果你参与了 DeFi，那么你可能听说过加密货币质押。这是一个锁定你的数字货币以换取奖励的过程。

由于 NFT 是代币化资产，NFT 质押的工作方式相同。NFT 附带的图像是根据 ERC-721 标准编写的代码，这使得每个 NFT 都是独立的资产。

在正确的时间加入并获得最多的奖励需要大量的研究和一点运气。区块链技术使人们能够提供自己的资产，以换取由平台决定的利益。各种项目都是专门针对 Staking 而构建的。

了解 NFT 质押

NFT 质押是将不可替换的令牌锁定到区块链协议上，通常是为了换取奖励。

NFT 质押存在的原因有很多：

  • 加剧稀缺性。如果 NFT 被锁定，则无法出售。如果它不能出售，那么供应就会下降，价格就会上涨。

  • 为 NFT 持有者赚取被动收入。如果对 NFT 持有者来说质押没有任何好处，那么他们也没有理由这样做。因此，当拥有 NFT 的人将其锁定时，项目背后的团队会给予他们一些回报。

  • 支持权益证明 (PoS) 网络。PoS 是大多数区块链用来验证网络上交易的机制。 

对于任何想要在区块链上质押 NFT 的人来说，通常都会收取相关的手续费用。这是因为验证者需要支付作为保护网络安全的奖励。

了解您的质押回报率

每个 NFT 质押计划都提供不同的奖励，通常是与项目相关的货币。

The Sandbox

以 The Sandbox 为例。该平台将于 7 月引入土地质押。元世界中的土地实际上只是一个 NFT，表示虚拟世界地图中的一个位置。

任何在 The Sandbox 中质押 NFT 的玩家都将获得游戏内数字货币 SAND 的奖励。您可以使用我们的 Token Explorer 了解 SAND 的价值。

The Sandbox 已经拨出了价值 1000 万美元的 SAND，作为对土地抵押的人的奖励。该平台已宣布其年化收益率 (APY) 将是多少。他们还设置了 SAND 用户在他们所持有的每块土地上可以赚取的最大数量。

在 The Sandbox 中进行土地质押的 SAND 奖励

每当持有人质押 NFT 时，他们应该首先找出相关的质押成本是多少。确定是否有必要的质押期也很重要，因为一些平台有用户需要锁定其 NFT 的最短时间。

最后，一旦您确定了质押的成本以及需要多长时间，请查看您将获得哪些收益。The Sandbox 的预计年利率为 90%。

Wolf Game

另一个例子是 Wolf Game。DappRadar 在 2021 年 11 月写了关于 play-to-earn 的文章。游戏中的代币是 WOOL，用户可以用他们的 Wolf 或 Sheep 质押以获得不同的奖励。

Wolf Game 中的 NFT 质押奖励

Wolf 的持有者比 Sheep 的持有者赚得更多。Sheep 质押者还可以通过冒险和暴露他们的 NFT 的危险来赚取更多的 WOOL。这是有趣的游戏化金融。

NFT 质押的最佳项目

并不是所有的 NFT 质押项目都提供相同的奖励。有些其他项目一开始的回报率很高，但随着他们的代币涌入市场，它的价值就会下降。

请始终记住，您应该将您的收益与 NFT 的初始成本进行比较。如果你为 NFT A 支付的费用是 NFT B 的三倍，但 A 只给你比 B 多两倍的奖励，那么用 NFT A 收回成本将需要更长的时间。

Mutant Cats

质押 Mutant Cats 使持有者对项目的去中心化自治组织（DAO）保险库资产拥有部分的所有权。每只被质押的 Cat 每天还能获得 10 个 FISH 的收益。

该系列由 9,999 个 NFT 组成，于 2021 年 10 月推出。它得到了 Cool Cats 公司的支持，该项目背后的想法是让人们无需以高价购买即可获得这些资产。

Mutant Cats 背后的团队也有一个名为 Mutant Gorillas 的系列。每一个质押的 Gorilla 会在 60 天后产生一个 SERUM。SERUM 是一种可交易的 NFT，具有治愈 Mutant Cats 的能力。

NFT Worlds

NFT Worlds 在 3 月份共推出了 17.5 亿 WRLD 代币作为激励。该计划将在 5 年内释放这些代币，并在此后很长一段时间内创造一个可持续的经济。

NFT Worlds 中的每一块土地都有一个稀有度等级，并且质押者会根据该等级获得 WRLD 奖励。总的来说，质押的土地 NFT 越少，参与质押的人获得的奖励就越多。

持有土地的用户也可以同时将其出租给其他用户。这增加了额外的被动收入来源，并使 NFT World 持有人进一步锁定在该平台上。

Axie Infinity

Axie Infinity 本月启动了自己的土地质押项目。游戏中有 5 种类型的土地 NFT：Savannah、Forest、Arctic、Mystic 和 Genesis。目前出售最便宜的价格是 1.275 ETH（2,000 美元）。

Axie Infinity 用户于 2019 年 1 月首次开始购买土地 NFT，因此它的实用性由来已久。该平台为质押土地 NFT 的用户提供以下奖励。

Axie Infinity 中的 NFT 质押奖励

SkuxxVerse

SkuxxVerse 与大多数的土地质押机制略有不同。通过这个项目，SkuxxVerse 团队购买了 NFT 并进行质押。

然后，用户可以购买 SkuxxVerse 的 NFT 通行证，从而进入定制的虚拟世界，更重要的是，他们还可以享受质押的收益。

迄今为止，SkuxVerse 已向其持有人分发了 100 多万 WRLD，并刚刚在 NFT 世界中推出了自己的虚拟世界。每个 SkuxxVerse 通行证持有者还将获得一个免费的 NFT，代表元宇宙中的一个家。

CyberKongz

CyberKongz 就是一个例子，它的持有者无需质押就能从 NFT 中获得收益。持有 Genesis CyberKongz NFTs 的玩家每天可以获得 10 个 BANANA，并保证持续 10 年。他们还可以燃烧赚来的代币，以赋予他们 NFT 独特的特征。

2021 年 10 月，BANANA 的价格达到峰值，CyberKongz 的持有者每天能赚 1000 美元。从那以后，价格就有所下降。

该系列中还有两个角色：Baby Kongz 和 CyberKongz VX。这两类分支都没有给持有者带来 BANANA 产量。但他们确实给了独家团体访问的机会。

NFT 质押的未来

随着 NFT 价值的下降和加密货币价格的暴跌，目前的 NFT 投资前景并不乐观。那些在 2021 年牛市高峰时期表现出色的项目，现在回报率很低。

但这并不意味着 NFT 质押没有未来。在接下来的几年中，项目将为其 NFT 找到新的、更好的效用。毫无疑问，随之而来的肯定是质押计划。

所有新项目和平台的目标都是开发不只是基于炒作的东西。一旦他们实现了这一点，那么来自 NFT 质押的可持续回报将成为真正的现实。
