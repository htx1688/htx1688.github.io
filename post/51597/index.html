创业

当应用不会再有护城河，价值该被如何捕获？

小应用是一种极具潜力的新模式，但是目前还没有行之有效的商业模式。小应用所创造的潜在价值比他们能够捕获到的价值更多，而如果小应用和协议结合便能填补这一空白。小应用程序将明亮地燃烧和爆炸，而聪明的协议将不惜一切代价确保他们是在这个过程中捕获能量的戴森球。这就像《胖协议，瘦应用》的升级版，多了一个时间维度。

小应用需要找到护城河（增加了社交功能以创造网络效应，或捕获了宝贵的专有数据来个性化和改进产品，使人们不想离开），也需要尽可能降低成本，并尽量在有利可图的时候产生尽可能多的收入。

生命周期较为短暂的小应用可以与那些经久不衰的协议合作，共同建立更大、更持久的东西。让用户在小应用中携带其身份、数据和关系的协议是强有力的候选者。

写给黑客：如何靠漏洞赏金谈判追回资产？

CertiK 研究四个不同的协议（Poly Network、Allbridge、Euler Finance 和 Sentiment Protocol）的谈判过程。

所有协议都发布了用以换取被盗资产的赏金。

Euler Finance 和 Sentiment Protocol 均向攻击者发出警告（用赏金悬赏攻击者信息）。Allbridge 和 Sentiment Protocol 还宣布称如果资金被退回，则不会对黑客采取法律行动，而 Poly Network 明确表示将联系执法部门。

在这四个协议中，其中两个的“可收回”资金被全额返还，Allbridge 仍在与第二个黑客进行谈判。Sentiment Protocol 则是在经过两天的谈判后成功收回了 90% 的资金。

在与攻击者的谈判中，赏金是一个非常有效的手段。然而其也有一定的潜在风险。例如攻击者拿到赏金后不履行承诺，而继续泄露数据或再次攻击。另外，一些国家和地区可能会对支付赏金的行为采取法律措施。

DeFi

详解 Endgame：五个连续启动阶段将如何为 Maker 带来全面改变？

Endgame 将简化并并行化 Maker 生态系统，推出新的统一品牌标识和 6 个新的 SubDAO 的旗舰发布，用户可以通过它们进行 farming，它们将引领无需许可的创新和抵押品分配。

Endgame 还将引入强大的 AI 工具，帮助不那么硬核的治理参与者掌握系统的所有细节和正在发生的事情，并提供一个激励人们通过 farming 奖励参与的治理参与奖励系统。

Endgame 的最后一步将是部署一个新的区块链，这个区块链与以太坊紧密相连，同时增强了生态系统的治理安全性，并实现了 Endgame 的全部高级特性和代币经济学。

ve( 3, 3)无处不在，有哪些值得关注的项目？

ve( 3, 3)协议需要 veNFT 持有者、合作伙伴协议和 LP 每周或更频繁地互动，以最大化收益、流动性或其他目标（例如注意力份额）。

熊市压力和高排放量对各种生态系统的 ve( 3, 3) DEX 造成了持续的价格下行压力。然而，从长远来看，参与度高的持有者基础和协议之间的一致性将导致可持续的飞轮。

作者对为 ve( 3, 3) 构建者的一些构思包括：新项目代币的 Launchpad、国库管理、由 veNFT 和 LP 代币支持的 ve( 3, 3) 稳定币、veNFT AMM 用于交换锁定头寸并进行清算、veNFT 持有人的自我偿还贷款。

作者看好的 ve( 3, 3) 有 Thena、Chronos、Velodrome、Solunea。

NFT

2023 年 NFT 市场分析：内幕观察

Blur 的可选版税和零手续费策略吸引了众多价格敏感型用户。此外，Blur 上的洗盘交易数量正在逐渐减少，这表明这个市场上真正的交易者正在沉淀下来。

受到 2022 年底熊市影响，第一季度蓝筹 NFT 项目盈利能力明显下降，但部分项目仍旧有不错表现。BAYC 在卖方市场的盈利能力最强，Azuki 的巨鲸数量增长了 100% ，是今年最具弹性的蓝筹 NFT 项目之一。

NFT 市场盈利波动仍受蓝筹巨鲸交易状态影响，同时场内不同项目间交易者资产差距显著。

通过 NFT 产生收入的方法有多种，盈利的 NFT 交易者可以被分为三类：看准时机低价购买高质量 NFT 并长期持有，从小众项目中批量铸造 NFT 高价出售，或锚定较有发展前景的 NFT 类别进行短线高频交易获利。

近四个月比特币 Ordinals 市场的交易量呈上升趋势，不过仍仅占 ETH 市场的 0.02% 。由于 Ordinals NFT 交易速度较慢，成本较高，应用范围有限，因此以太坊 NFT 仍旧在应用范围和速度等层面占据优势。

LD Capital：NFTFi 缺乏新叙事与新资金，何时困境反转？

在市场缺乏新玩家进入的背景下，NFT 场内的存量资金不断萎缩成为当前 NFT 价格不断下跌的主要原因之一；增量资金的入场、市场伪买方流动性的下降或交易成本的降低是 NFT 价格企稳的观察指标。

NFT 交易平台恶性竞争已至后半程但赛道拐点未现，累积代币赤字带来的集中抛压是 Blur 面临的难题。Blend 目前对于 Blur 币价提振的作用有限。由于借贷产品的需求会在牛市加杠杆中不断放大，所以底层抵押品价格的上涨、抵押品范围的扩大是借贷赛道的爆发的重要观察指标。

熊市下借贷产品缺乏真实需求，等待 NFT 市场的整体回暖，RWA 权益型 NFT、半同质化代币、Ai+NFT 等可能成为新方向。

以太坊与扩容

为什么 Lido V2升级对 LSD 赛道很重要？

Lido V2通过引入 Staking Router 架构和协议内 stETH（ETH 取款）来增强质押体验。每个人都从中受益，质押者将把他们的资产分布在不同的实体中，这大大提高了去中心化和安全性。现在，其他类型的节点运营商 DAO、独立质押者和小团体可以参与 Lido。用户也将能够使用不同的节点运营商组合和具有各种竞争特征（例如覆盖选项和费用结构）来提议和实施模块，并申请加入 Staking Router 的模块集。

通过改进的安全性和新的质押路由器基础设施，LSD 将变得更加安全和去中心化。此外，通过快速提款，小钱包（<1000 $ETH）几乎可以即时提款，如果你是大钱包，你需要等待更长的时间，但与直接质押相比仍然更加灵活。

关停 Aztec Connect 后，Aztec 新推出的混合式 zkRollup 是什么？

隐私与 EVM 不兼容。Rollup 技术的美妙之处不仅在于模仿以太坊，而是扩展了其实用性。

Aztec 是一个机会，可以在基础层上确保隐私，并获得以太坊的安全保证。Aztec 将公共和隐私执行结合到一个单一的 zkRollup 中，在加密和非加密的交易中提供无缝的可组合性。

如果说以太坊的创新是允许对公共状态进行读写，那么 Aztec 的创新是能够对公共和隐私状态进行读写。将公有和隐私结合成一个单一的混合式 Rollup，可以实现保密性和可审计性的用例，比如：公共 DAO 的私人投票、公共游戏的私人会议、公共 AMM 上的隐私交易。

区块链应用的最大障碍不是区块空间，而是实用性。值得庆幸的是，zkSNARKs 为 Aztec 解锁了灵活的加密和可扩展性。Aztec 解决了三个问题：隐私保障、钱包体验和账户抽象化。

BitKeep 研报：深度解读 zkSync Era，原理、应用与生态

zkSync Era 的出现将交易成本对比之前降低了 50 倍，zkSync 生态系统呈指数增长趋势；

众多生态纷纷部署、迁移至 zkSync 网络，涉及 DeFi、跨链桥、NFT 等各个领域；

zk-EVM 竞争白热化、zkSync Era 运用的零知识技术还有待提升和确认；

zk-Sync 选择了高性能，从而牺牲了可组合性。

veDAO 研究院：深度解读以太坊互操作性基础设施 Omni Network

Omni Network 旨在连通不同的 Rollup 方案，通过将以太坊再质押协议 EigenLayer 和 Cosmos SDK 融合到一个统一的平台中，聚合以太坊的模块化生态系统，使开发人员能够在以太坊上的所有 Layer 2 上构建全球应用程序，同时从 ETH 本身获得安全性。

虽然 Omni Network 目前尚未推出测试网，但其旨在通过模块化应用的核心基础设施，实现无缝、安全地跨 Rollup 用户体验，已经吸引了众多目光。

在不增加最终用户体验复杂性的情况下，开发人员将能够在整个以太坊生态系统中提供他们的应用程序，同时仍然保持 Cairo 的性能优势。

其他 L1

BlockCrunch 万字长文：为什么 2023 是 Filecoin 开局之年？

虽然 IPFS 确保了网上内容的可发现性，但它无法确保内容的永续可用性或存储的正确性。Filecoin 是 IPFS 上搭建的互补协议，使用内建加密经济学模型来激励存储提供者，从而增加了长期的数据持久机制，确保了数据以可验证的方式安全存储并持续能被检索。

Filecoin 的三大支柱是存储、检索、计算。

截至 2022 年 12 月 31 日，Filecoin 网络存储了 460 PiB 的数据，代表了去中心化存储提供商数据存量的 97% 。与一年前相比，存储的数据增加了 18 倍，这个惊人的增长归功于 Filecoin Plus 的推出，它占 Filecoin 存储交易提议（deals）的 99% 。Filecoin Plus 计划旨在为用户提供补贴，让其能以折扣价存储大型的有价值数据集。

目前，Filecoin 生态系统中的应用可以分为 5 个用例：增加开发者对去中心化存储的使用，引入大型数据集，让日常用户更容易使用 Filecoin 服务，元宇宙和游戏，音视频平台。

有了 FVM，Filecoin 网络在现有存储网络之上获得了执行智能合约的能力，潜在用例包括：去中心化数据计算、更智能的存储市场、数据 DAO、支撑专门的网络。

相较于其竞争对手中，Filecoin 是最具扩展性和综合性的去中心化服务网络，因为它拥有独立的一层区块链协议；数据在链外存储，在链上持续验证；在存储的位置进行计算。

跨链

数据解读跨链桥现状：LI.FI 和 Stargate 的留存率最高

用户需要快捷便宜的桥接服务，而安全性却难以形成卖点。

桥的网络效应不像 DeFi dApp 那样明显，并且对于普通Web3用户来说，为跨链桥提供流动性是一项复杂的操作。为了成为头部协议，跨链桥需要在以下方面做出选择：B2B合作和集成（LI.FI 是一个很好的例子，专注于集成，许多用户愿意为更好的用户体验支付额外的交易费用）；B2C活动（任务、空投、社区参与）可以形成品牌差异化（Stargate 在这方面是一流的）。

市场仍然由 Arbitrum、Optimism 和 Polygon 跨链桥主导，但是诸如 Stargate、Multichain、Synapse Protocol 和 CelerNetwork 等“品牌”跨链桥正在侵蚀老牌跨链桥份额。

Stargate 目前在活跃用户方面领先，但是 LI.FI 和 Stargate 的留存率最高。

跨链桥须衡量重要的事情：协议活动、收入、活跃用户等。关于用户获取，活动任务有严重的局限性。尝试新途径（如推荐活动）是个好方法。留存对于实现长期可持续性发展至关重要。精心设计的、数据驱动的和个性化的任务应该反复测试玩家的忠诚度。
