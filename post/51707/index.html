致力于账户抽象和交互抽象的A3S Protocol 近期在 Arbitrium 主网正式上线，并同步推出了 Mint Queue Earn 社会实验型代币激励活动，旨在让更多用户迈出持有并使用合约地址的第一步。

A3S Protocol 是什么？

A3S Protocol 指出了普通用户难以直接使用智能合约实现复杂的链上交互的困境，因此创新地提供了一种将智能合约账户抽象为NFT的服务（Account As A Service）。

更形象地说，A3S Protocol 希望将合约账户的高级功能（比如闪电贷、mev等）包装为了一个个“智能芯片”NFT，用户可以直接购买这些有不同功能的芯片 NFT，自由地进行组合，实现复杂的账户功能，更简单方便地进行链上交互。

作为一个创新的协议，A3S Protocol 在活跃在各个 grant 及 hackathon 之中，也获得了用户及开发者的认可：

  • Gitcoin GR 14 中获得超 1000 名用户的自发捐赠，排名 top 3% ；

  • Polygon BUIDL IT Hackathon 获奖项目及 AWS Track 第三名

  • BNB Grant DAO获奖项目

  • …….

A3S 希望一步步，将“账户抽象”推向更深层次的“交互抽象”，重塑目前以太坊的账户体系。

在第一阶段，A3S 首先将合约账户的控制权抽象成了一个 A3S Owner NFT，实现了智能合约地址的流动性；在目前的第二阶段，A3S Protocol 将为 NFT 化的合约地址植入一套创新的有趣机制，旨在邀请更多的用户开始体验 NFT 化合约地址的创新和便捷，从而为 A3S protocol 建立起更广大的用户基础。

Mint Queue Earn 活动

A3S 推出了 Mint Queue Earn 活动及$AA 激励代币。用户只需要 mint NFT 地址，排进一个队伍中开始累积并等待领取代币，用户加入得越早，等待时间越长，排到自己时可以领取的代币就越多。

具体来看，Mint Queue Earn 活动主要就分为活动名称中的三步：Mint、Queue、Earn。

Mint：

即用户claim 自己的 A3S 合约地址，并 mint 相应的 A3S Owner NFT。持有这个 NFT 就代表用户对这个 A3S 合约地址的控制权和所有权，而无需私钥。

当然，A3S 提供的 NFT 化的合约地址并不仅是 PFP 类的 NFT，而可以实际进行使用。用户可以使用 A3S 合约地址进行代币、NFT 的存储或转账，以及通过 walletconnect 连接，在Uniswap上进行 swap 等操作；用户也可以充分发掘地址流动性带来的更多可能，进行地址的交易、托管等。

Queue：

拥有A3S NFT 之后，用户即可开始排队，用户的地址上自动开始累积$AA 代币，并在出队时方可领取累积的代币。新进入排队的用户会排在队伍最末端，并将所有前方排队的地址向前推进，排在第一位的地址就会被挤出队列，可以领取自己的代币。

通过官方白皮书可以得知，可以获得代币量的多少主要取决于几个关键因素：

  • 排队时间

    用户呆在队列中的时间越长，用户能获得的AA 越多。这是一个比较难以掌控的因子，因为用户在队列中的时间取决于用户在队列中的位置，以及后进入排队的用户多少。如果突然有大量的地址加入队列，用户可能很快就会被挤出队伍。因此官方也在这个游戏中加入了一个“Extend your Stay”这个更具有主动性的选择，队列中用户可以通过这个按钮重新排到队伍的最后一位，从而延长其排队时间。

    总之，排队时间是一个极具博弈性的因子，用户排队时间越久，能获得的代币越多，但其能够出队领取代币的时间就越难以确定。

  • Token ID

    像其他NFT 一样，用户的 A3S Owner NFT 属于同一个 NFT Collection，按 mint 顺序，NFT 的 Token ID 从小到大。Token ID 越小，即 mint 越早的用户，其 NFT 的积累代币的效率更高。这也是一个对早期参与者给予更多激励的因子。

  • 地址总量

    mint A3S 地址的总人数越多，已有的地址产币的效率就越高。这将能促进 A3S Protocol 不断推广至更多的用户群体。

Earn：

用户出队即可在72 h 内领取自己积累的$AA 代币。除此之外，用户还通过两种方赚取$AA。

第一种方式pickup。这个玩法就是被挤出队伍之后，用户在 72 h 内忘记领取自己的代币，这部分被遗忘的代币有 50% 就会进入 pickup 池，供其他还在队列中的地址捡漏。进行 pickups，要求用户的 A3S 地址仍在队列中，且进行 pickup 时，用户的地址会重新排回到队伍末端。因此，在用户想要回到队尾延长在队伍中的时间时，不如看看有没有可以 pickup 的代币，顺手捡捡漏。

第二种方式邀请，即邀请其他用户参与这个游戏。拥有A3S 地址的用户都可以进行邀请，邀请者可以获得被邀请者出队领取的代币的额外 15% 奖励，被邀请者在出队时，也可以获得额外 5% 的奖励。需要注意的是，如果被邀请者 72 h 内忘记领取自己的代币，那么邀请者和被邀请者都无法获得这些额外奖励。所以邀请者除了邀请之外，可能还要及时提醒被邀请者在出队时领取自己的奖励。

结语

总体来说，A3S Protocol 项目新颖，提供了一种新的 NFT 资产品类，并提出了新的“交互抽象”的交互模式，是 Account Abstraction 赛道非常值得关注的一颗新星。

其推出的Mint Queue Earn 是一个偏向于社会实验性质的激励活动，其操作简单，机制新颖，用户可能会形成不同的策略，并进行相互的博弈。同时，队列挤出机制和邀请机制的存在，能够正向推动游戏飞轮的转动，不断吸引更多的玩家参与。通过该活动，A3S Protcol 很可能发动一轮新热点，促使更多的用户开始了解并体验 Account Abstraction 除了钱包、社交恢复之外更多的可能性。
