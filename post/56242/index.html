当下香港加密货币市场蓬勃发展，政府支持和利好的政策频出，使得投资者和建设者们对 Web 3.0 和区块链技术充满信心。在这个蓬勃发展的市场中，位于钢线湾的数码港正在成为香港地区新兴 Web 3.0 市场的数字科技旗舰。

由香港特区政府全资拥有的数码港目前拥有 380 多家金融科技公司，涉及区块链、网络安全、人工智能、大数据、财富管理和交易工程等不同领域，得益于港府积极拥抱虚拟资产，越来越多希望在 Web 3.0 领域里一展身手的项目方开始涌入东方之珠。

那么，大陆 Web 3.0 从业者怎样才能在寸土寸金的香港拥有一席之地呢？实际上，数码港拥有充足的资金支持、人才资源和场地，Web 3.0 项目方可以充分利用这些丰富资源进行创业创新。本指南将提供最详尽的帮助。

大陆 Web 3.0 从业者入驻香港数码港基本条件

首先需要注意的是，现阶段只有在香港注册的公司才能申请入驻香港数码港，那么大陆 Web 3.0 项目方如何在香港注册公司呢？ 流程其实并不复杂：

你最先需要的是确定公司类别，目前香港公司类别有股份有限公司和担保有限公司两类，前者有股本且公司成员的法律责任根据其组织章程细则限于各成员所分别持有的股份的未缴款额，后者无股本，公司成员的法律责任限于各成员藉其组织章程细则分别承诺在公司清盘时所支付作为公司资产的款额；然后，你需要有一个公司名称，香港对公司名称的约束要求可参见《香港公司名称注册指引》。

上面两件简单的事情敲定后，就可以访问香港公司注册处的「注册易」，并按照要求如实提交申请（当然也可以线下提交纸质材料），如申请获批准，香港公司注册处会给你发送电子形式的「公司注册证明书」及「商业登记证」（也有纸质版），这些证书到手，基本上一家公司就完成注册了。

那么接下来，如何将自己的 Web 3.0 创意变成现实呢？答案其实很简单：资金。对于想要获得资金支持的 Web 3.0 项目，放心，香港数码港有足够的资金支持。接下来，就让我们来告诉如何从数码港拿到最重要的创业资源，当然也是本文最重要的部分——大陆 Web 3.0 从业者如何向香港数码港申请资金。

初创公司通常会经历「启动-孵化-加速-飞跃」四个阶段，而数码港在这四个阶段都为 Web 3.0 从业者提供了资金支持，分别是：初始启动阶段的「数码港创意微型基金」、培育孵化阶段的「数码港培育计划」、加速发展阶段的「数码港加速器支援计划」、以及突破飞跃阶段的「数码港投资创业基金」。

1、向香港数码港申请创业的初始启动资金（10 万港元）

要把 Web 3.0 想法从 0 到 1 ，最重要的就是要有启动资金，数码港提供了「 创意微型基金 」(CCMF)，旨在支持发展具有创新、创意的数码科技点子及早期初创企业的成长 。

申请人可以个人或公司名义申请创意微型基金，若以个人名义申请, 主要申请人须持有香港居民身分证; 若以公司名义申请, 则该公司须于香港成立及注册。

目前创意微型基金包括 4 个子计划，分别为：

  • 专业计划

  • 香港青年创业计划

  • 数码港大湾区青年创业计划

  • 数码港大学合作伙伴计划

为配合不同创业目标群组的需要，每个子计划都是由创意微型基金全力支持，值得一提的是，参加创意微型基金项目除了能获得现金资助外，还能获取香港数码港在创业其他方面的支持，如: 培训、师友计划及业务咨询、业务发展及人脉加强、本地及环球业务网络支援、宣传及推广支援、同业认可及认证、计划毕业生支援等。更重要的是，如果成功申请到创意微型基金的 Web 3.0 项目，之后想要申请「数码港培育计划」也可获优先考虑。

毫不夸张地说，成功申请创意微型基金，是进入香港数码港的「敲门砖」，因此了解意微型基金的评审准则和程序对于 Web 3.0 项目申请的成功率来说是相当重要的。在评审准则方面，将会由独立评审委员会负责评核申请书，评审团成员包括风险投资者、学者、业界专才及行业协会成员，并从以下四个标准来考量：

  • 项目管理团队的质素及能力（30% ）

  • 营运模式及投入市场时间（30% ）

  • 项目创意及创新元素（30% ）

  • 社会责任（10% ）

在评审程序方面，所有创意微型基金受资助者均会由企业发展顾问组(ECAG)成员组成的独立评审委员会选出，并分为三个阶段进行，分别是初步筛选、评审、和简报演示，独立评审小组将会选出最终可取录项目使其成为创意微型基金的受资助者。

2、向香港数码港申请孵化培育计划（50 万港元）

如果你的 Web 3.0 项目已经顺利迈过“初创”阶段，但希望获得香港数码港的培育孵化支持，那么此时申请「数码港培育计划」，该计划旨在全方位支持创业，助力创业家及初创企业快速成长。同样，在加入培育计划后，香港数码港也将提供一站式创业支持平台来帮助 Web 3.0 创业者发展。

那么，Web 3.0 项目又该如何申请数码港培育计划呢？

首先，入围资格需要满足以下几个条件：

  • 产品或服务在 12 至 18 个月内准备上市的可行商业计划；

  • 申请人必须是根据香港特别行政区《公司条例》注册的股份有限公司（或正在成立中）；

  • 为申请人工作的主要申请人不能是以下任何一家公司的创始人、联合创始人或董事：目前参与由香港 CMCL、香港科技园公司或香港设计中心运营的任何孵化计划的性质相同的公司（统称“香港孵化器”）；先前服务于香港孵化器所经营的任何孵化计划性质相同的公司。

  • 申请人的创始人必须在提交申请时和整个计划期内共同持有申请人至少 51% 的公司股份或绝对控制权。

其次，对于项目的性质也是有要求的，比如必须植根于数字技术相关领域且尚未获得其他公共资助计划提供的资金。

另外要特别注意的是，Web 3.0 项目必须通过数码港创业管理系统（EMS）在 网上提交申请 并附上证明文件（如有需要），资料必须由主申请人（如为非公司申请人）或申请人的董事或股东（如为公司申请人）填写 ，否则将不予处理。

「数码港培育计划」的审核标准权重如下：

  • 具有里程碑意义的市场活力以及对数码港战略集群的贡献 ( 30% )

  • 管理团队的素质和能力（20% ）

  • 业务可扩展性 ( 20% )

  • 功能原型或通过设计确保解决实际问题的产品 ( 20% )

  • 创新性 ( 10% )

3、向香港数码港申请加速资金（30 万港元）

孵化培育后，数码港还与多个加速器建立了合作网络并提供资金，「数码港加速器支援计划」的申请者，必须为现正参与或毕业于「数码港培育计划」的受培育公司，并成功获数码港认可的加速器计划取录（截至 2023 年 6 月 16 日，香港数码港认可 89 家全球加速器计划，具体可通过官网查询）。

「数码港加速器支援计划」同样需要在 网上提交申请 ，表格信息要用英文或中文填写且必须清楚注明申请人的姓名，网上申请必须由申请人（如为个人）或申请人的董事或股东（如为公司）填写 ，否则将不予处理。需要注意的是，与其他计划不同，「数码港加速器支援计划」申请及证明文件须于加速器计划开始日期前 5 个工作日提交。

4、向香港数码港申请 A 轮或后期资金（总额 4 亿港元的投资基金）

一旦顺利完成了加速，Web 3.0 项目开始逐渐走向正规，此时渴望快速实现突破飞跃，那么可以尝试申请数码港投资创业基金(CMF) 以获得支持，数码港投资创业基金投资总额为港币 4 亿元，通常单笔投资规模在 100-2000 万元，主要用于：

  • 向数码港数码科技创业家提供种子项目投资以至 A 轮或后期的融资，助力公司业务飞跃发展；

  • 推动香港创业投资生态的发展，为数码科技创业家进一步开拓创业空间。

申请数码港投资创业基金的资格如下：

  • 数码港旗下创业计划的培育公司及毕业生

  • 数码港 Smart-Space 公司

  • 数码港办公室租户

申请数码港投资创业基金的审核标准权重如下：

  • 全面业务考核，包括但不限于合规、管治、财务状况及业务的可行性

  • 潜在的投资回报

  • 积极参与数码港数码社区活动

如何利用数码港政策顺利入境香港

当你的 Web 3.0 项目手握资金之后，接下来就要招兵买马，如何让优秀的 Web 3.0 人才来到香港呢？实际上，香港数码港已经提供了「科技人才入境计划」，旨在透过快速处理安排，为合资格科技公司／机构输入海外和内地科技人才来港从事研发工作，香港科技园公司(科技园公司)和香港数码港管理有限公司(数码港)的租户及培育公司只要从事人工智能、数据分析、金融科技等业务就符合申请资格。

具体可参见香港特别行政区创新科技署官方网站。

如何利用数码港政策获得办公场地

数码港其实是香港共享办公理念的先行者，旗下 Smart-Space 早在 2009 年就已经推出，最近又上线了 Smart-Space 8 ，Web 3.0 项目方可以在网上申请办公室、工作站和灵活空间三种场地类型（不同类型的费用不同）。

在使用时间方面，Smart-Space 有灵活期限（最少 1 个月）和固定期限（最长 12 个月）两种服务。需要注意的是，申请方必须提供商业登记号码、公司注册证书、公司网站等关键信息。

总结

随着香港加密货币市场的蓬勃发展和对 Web 3.0 和区块链技术的政策支持，越来越多的大陆 Web 3.0 从业者希望在香港数码港找到发展机会。入驻数码港的基本条件是需要在香港注册公司，注册过程并不复杂。数码港拥有丰富的创业资源，为 Web 3.0 项目提供资金支持。从初始启动阶段到突破飞跃阶段，数码港提供了不同阶段的资金支持计划，为 Web 3.0 从业者提供了良好的创业环境和机会。
