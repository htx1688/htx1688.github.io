代币是创作者引导观众和资本的强大新工具。

使用 NightCafe 创建的图像

在音乐人 Daniel Allan 在 2021 年制作他的专辑《Overstimulated》之时，他本可以与很多互联网创作者一样：花费数月甚至数年时间创作和发布作品，以期建立足够大的粉丝群来资助他的全职工作。或者，他本可以走一条更传统的路线：将他的作品转卖给传统音乐厂商，希望其中一家能给他稀有的奖品：唱片合约。

相反，Daniel 选择了另一条不同的道路：他通过 Mirror 上的代币销售为他的新专辑众筹，最终从 87 个支持者那里筹集了 50 ETH（当时价值约为 14.2 万美元）来资助他的音乐创作。作为投资回报，代币持有者获得了艺术家利润份额中的 50%，并且可以直接与 Daniel 本人对话。「第一次，我拥有了我发布的音乐，人们为我的艺术赋予了实际价值」，他在 Mirror 上这样描述 Web3 对他作品带来的影响。

Daniel 这样的 Web3 原生创作者代表了创作者经济新模式的先锋力量。我们已经写了大量关于网络和平台如何通过代币分配以获得更快速和大规模发展的文章。同时，代币也代表了创作者引导观众和资本的强大新工具。创作者现在可以通过代币获利并预先建立观众群体，而不是免费创建内容以希望逐渐累积观众并有朝一日能够将其货币化。

换句话说：Web3 颠覆了传统的在线内容创作模式。 这是一个范式转变带来了新的思考：创作者的工作如何完成、粉丝如何与创作者的工作相关联，以及更广泛的创作者生态系统如何作为一个整体发挥作用产生重大影响。

创作者剧本的演变

这不是创作者剧本的第一次更新。在互联网出现之前，创作者构思了他们的内容，然后将这些内容推销给唱片公司、图书出版商、电视工作室等有权决定制作内容的机构。然后，创作者放弃了重要的创意控制权和大量未来收入，以换取前期资金和作品的发表。

随着 Web2 用户生成内容平台的兴起，出现了一条通往创造性工作的新途径。创作者能够将他们的作品在线发布到 Instagram 和 TikTok 等平台，直接面向观众。 随着受众增长，各种商业化途径变得可能，主要包括广告、品牌赞助、订阅和电商等模式。

传统的创作者生命周期

在 Web2 模式下，创作者能够保留其作品的创造性所有权，但通往内容变现的道路仍旧漫长。 创作者在创作内容上投入了大量的时间和精力，却不知道他们何时或是否会看到这种努力得到回报。这种状态导致了创作者的倦怠，也阻碍了大量创作者的进入。

换句话说，在 Web2 模式下的创作者面临着一个冷启动的挑战：为了赚钱，他们需要观众；但为了增加观众，他们又需要有资本来资助内容创作。Web3 创作者剧本是克服冷启动问题的新方法。

Web3 创作者剧本：代币引导受众和资本

Web3 模型参考了 Web2 创作者剧本，并将其反转。Web3 创作者不是从创建免费内容开始努力建立一个可以在以后货币化的庞大粉丝群，而是通过创建代币（NFT 或可替代代币）开始货币化。 这吸引了对资产所有权感兴趣的粉丝成为初始受众。然后，创作者可以使用代币销售的初始资金和利益相关者来引导他们的创意工作。粉丝可以参与其中，看到创作者取得成功，并被激励成为创作者及其作品的传道者。这有助于促进内容创作、受众增长和货币化的良性循环。

这个剧本的关键是数字稀缺性：随着创作者变得更加知名或成功，代币也会增值。在我关于 Web3 对创作者经济的影响的文章中，谈到了通过代币化实现数字稀缺性的重要性：

在将他们的作品标记为 NFT 时，创作者创建了一个可验证的链上媒体所有权和出处记录，最终形成的是一个可以追溯到艺术家独特数字资产。对创作者的作品充满热情的粉丝愿意为这种规范的媒体支付更多的费用，这让创作者能够更好地捕获到粉丝的付费意愿。

Web3 创建者生命周期

这个 Web3 创作者模型反映了 Chris Dixon 所概述的使用代币激励引导网络的内容。在传统网络中，对新用户的效用仅限于应用程序的效用——或者，就创作者而言，仅限于他们的创意产品。代币引入了额外的效用来源：金融效用。在创作者事业生涯的早期阶段，代币为观众提供了一种补充的效用来源。

资料来源：a16z

在各种创意垂直领域已经发生了许多这样的例子。在电影界，Shibuya通过出售 NFT 制片人通行证筹集了 400 ETH（120 万美元）用于制作名为《White Rabbit》的动画系列，持有人可以决定每一章的结果并成为最终电影的所有者。

同样，动画师与电影制作人 Robbie Shilstone 通过 Formfunction 平台上的 NFT 筹集了超过 324 个 SOL，使数十名支持者能够参与到电影的制作过程中。

在真人秀节目中，Mad Realities 通过 NFT 销售为其约会节目众筹了 172 ETH，其中 NFT 所有者拥有各种管理权力，包括每周对演员进行投票、在拍摄时向参赛者发送礼物以及对获胜夫妇进行投票的权力。

在出版界，Tally Labs 即将出版的小说《Bored & Dangerous》将 2800 名 Writers Room NFT 持有者纳入了写作过程，NFT 持有者可以授权书中角色并投票决定故事的方向。在音乐方面，许多 Web3 艺术家与 Daniel Allan 进行了类似的实验，向支持者分发代币，并提供聆听音乐派对等好处。

在这些例子中，代币不仅可以帮助创作者预先为他们的工作获得资金，而且在许多情况下，代币甚至可以作为吸引早期观众的方式 。通过代币提供的激励一致性，这些观众成为积极的共同创造者和利益相关者，并有动力参与创作过程，推动作品普及。

这对创造者经济意味着什么

虽然创作者发行代币这种模式仍处于初期阶段，但特别是在三个领域，新的创作者剧本可能会影响创作者经济：作品的可访问性、同人圈的性质和创作者本身的角色。

1. 创意路径变得更容易获得

在 Web2 模式下，创作者通常会花费很长时间创作免费内容，然后才能看到回报。而 Web3 代币模式加快了获得资本和观众的速度，降低了这种财务障碍，允许更广泛的有抱负的创作者追求创意事业。

当然，还有一些障碍需要考虑。虽然新剧本推动了资本和观众的增长，但它并没有消除培养一小部分热情的追随者所需的前期努力。虽然 Daniel 艾伦在他的众筹期间只有大约 200 名 Twitter 粉丝——对于广告等传统商业化模式来说，这个受众规模是行不通的——但他确实不得不花费大量时间来培养与收藏家的关系。

Billboard 上一篇文章这样描述：「在《Overstimulated》发布前的几个月里，艾伦每天花 8 个小时在 Twitter 和 Discord 上。他向 Web3 音乐空间中的每个人发送消息。他进入了数十个 Discord 服务器，说「gm」（早上好）并与音乐 NFT 收藏家建立联系。

到竞选结束时，Overstimulated EP 有 87 名支持者——其中至少有 50 名艾伦事先与他们建立了个人联系。」

Web3 创作者剧本减少了商业化所需的受众总规模，但实际上使与更少、更真实的粉丝建立更深层次的关系变得更加重要。

另一个主要障碍是教育和访问：为了让创作者使用新的创作者剧本，他们必须意识到这条途径存在，并且能够使用使之成为可能的工具。这就是为什么对于 Web3 生态系统来说，继续投资于开发各种背景的用户都可以访问的资源和工具至关重要。Seed Club 的加速器、Forefront 的 Web3 Creator Residency、HIFI Labs 的 Web3 Artist Cohort Program 和 Music NFT Launchpad 等教育计划都是满足让更多创作者加入 Web3 的重要需求的举措示例。

2. 同人圈的性质在演变

在 Web2 创作者经济中，创作者和观众之间的关系在很大程度上是单向的。Web3 创作者剧本通过两个主要途径激励粉丝，并为他们发挥更积极的作用开辟了道路：

从粉丝到投资人

在 Web2 模式下，粉丝被创作者吸引是出于内在原因：他们喜欢创作者的作品。Web3 模型引入了一个额外的动机：所有权。一些支持者会出于投机原因被吸引，他们认为所有权是从转售中获利的必要条件。

随着代币所有权转移激励措施，这有可能改变创作者与社区之间的关系。当粉丝遭受经济损失时，它如何改变创作者和观众之间的关系？如果他们的代币贬值，粉丝会放弃支持吗？随着新的创作者剧本的成熟，创作者生态系统必须解决这些问题。

从被动的旁观者到积极的利益相关者

传统内容创作是一个单向过程：创作者创作作品，观众消费作品。新的创作者剧本将创意工作转变为参与过程，代币持有者希望对工作本身产生直接影响，因为他们最后的财务收入与作品息息相关。这方面的一个例子是Jenkins the Valet，一个单一的无聊猿 NFT，其整个存在和角色都是由其所有者精心打造的，现在已与 CAA 签约，并作为全新创意项目的基础。所有这些衍生项目都通过提高收藏品的知名度来为无聊猿生态系统增值。很难想象传统环境中的粉丝会不遗余力地帮助创作者取得成功，但 Web3 提供了财务利益一致性，从而激励社区参与者走得更远。

这种转变带来了未知的影响。观众参与创作过程是增强还是限制了创作潜力？创作者将如何平衡自己的创意与支持者的愿景？

3.创作者角色改变

并非只有粉丝随着 Web3 模式的采用而改变，创作者也会看到他们的角色发生了变化。这是因为 Web3 创作者剧本改变了创作者成功所需的技能构成。

创作者必须从一开始就擅长营销、培养关系和社区管理，而不是仅仅专注于自己创建内容 。他们必须像初创公司创始人一样发挥作用：筹集资金，宣传他们的愿景，并在此过程中平衡利益相关者的利益。这在日益流行的 CC0 模型（「无版权保留」）下被发挥到极致，创作者允许其他人以任何方式使用他们的作品，包括用于商业目的。CC0 创建者的角色更多地是社区管理人员，而不是内容创作者。

这种转变的更广泛影响仍有待观察。粉丝投资者的投入和期望会导致产出脱节，阻碍创作潜力，并引发更多的创作者焦虑和倦怠吗？或者它会帮助创意项目？

更有活力的创作者经济：为创作者提供更多选择

在过去的几十年甚至几个世纪里，创造者经济的故事一直是通过更广泛的访问、商业模式和更低的新技术进入壁垒释放潜力的故事之一。代币激励是创作者经济的一个令人兴奋的新工具，允许创作者引导支持者，利用新的商业化和价值捕获机制，并与他们的观众共同创造作品以及财富。

正如 Web2 剧本没有取代传统出版商一样，Web3 剧本也不会完全取代 Web2 创造者经济剧本。 相反，它为创作者开辟了新的潜在道路，这将导致更强大、更有活力的创作者经济。
