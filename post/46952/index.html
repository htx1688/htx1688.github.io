撰文：Miles Deutscher，加密货币分析师

编译：Babywhale，Foresight News

几日前，推特上掀起了小规模的针对 ZKR 和 OR 究竟谁是更好的以太坊扩容方案的争论（推荐阅读），虽然双方还未分出胜负，但不得不承认 ZK 依然是未来几年内值得重点关注的叙事。国外加密货币分析师 Miles Deutscher 在推特整理了值得关注的 ZK 概念和生态项目，现做整理分享，其中有关 ZK Rollup 的基础介绍不再赘述：

今年有几个催化剂事件使市场对该行业的兴趣激增：即将推出代币的 zkSync 和 StarkNet 以及即将上线的 Polygon zkEVM。这使得 MINA、LRC、IMX 和 DUCK 等与 ZK 相关的协议代币表现出众。

然而，近期的上涨可能是投机驱动的，因为关键的链上指标（活跃用户、交易、跨链桥流动性）并未随着价格提高。这种反弹短期可能是不可持续的。

然而，在 zkSync、Polygon zkEVM 和 StarkNet 发布之前（和之后）有可能出现持续反弹。这是由于届时新用户的增加和新的流动性流入生态系统。

因此，为了做好准备，我列出了已发币和未发币的 ZK 相关项目：

已发币项目

Mina

Mina 是一个使用 zk-SNARKs 和 PoS 共识机制的轻量级的区块链。Mina 大大减少了节点需要下载的数据量，ZK-Rollups 允许节点存储数据量较小的证明，而不是整个链的数据。

Immutable X

Immutable X 是 Layer 2 扩容解决方案，旨在为 NFT 交易和铸造提供近乎即时的确认和接近于 0 的 Gas 费。Immutable X 基于 StarkWare 的 ZK Rollups 以实现可扩展性，并计划针对不同的用例支持多个 L2/L3 ZK-rollups。

Immutable X 将其描述为「Web3 游戏开发者平台」，其上有包括 Illuvium、Gods Unchained、Kongregate 等游戏。

Loopring

Loopring 是一个为建立和扩展 DEX 设计的 Layer 2 网络，通过 ZK Proofs 支持建立高吞吐量、非托管的 DEX。结合两种主流的叙事：ZK 和 DEX，使 Loopring 成为未来值得关注的有趣项目。

Syscoin

Syscoin 将比特币的 PoW 架构和以太坊的 EVM 结合到一个网络中，其双层架构允许创建可扩展的智能合约同时维护网络安全。Syscoin 使用 EVM 的修改版本，称为 NEVM，NEVM 使用 ZK Proof 来支持更高的 TPS。

Dusk Network

Dusk Network 是一个以隐私为中心的 Layer 1 区块链，专注于交易可编程、隐私和合规的证券。可将其视为金融应用建立的隐私区块链。

Dusk Network 使用名为「PlonKup」的 ZKP 系统以更低的成本保护数据。

Mute.io

Mute.io 是一个建立在以太坊和 zkSync 上，基于 ZK Rollup 的 DEX、收益和债券平台，可以将其视为一个基于 ZK 的综合 DeFi 平台。

推特名为 meta_quant 的加密货币交易员通过一个表格来概括了各个项目的情况：

需要注意的是，这次反弹大概率是投机所致，仅仅因为使用了 ZK 相关的技术并不一定能让项目成功。我们需要寻找正在利用技术进行创新的项目，或者学会利用炒作情绪进行交易。

对于这些项目，与大多数山寨币一样，价格走势在很大程度上取决于比特币。但是，如果您对市场前景是看涨的，那么最有力的叙事无疑会跑赢大盘（ZK 可能是其中之一）。

如何进行交易取决于个人的策略，您可以考虑以平均成本法为导向的方法并在投资前严格定义时间范围和财务目标。当然，还有 DYOR。

未发币项目

之前我们已经介绍了一些已发币的项目。但我对未发币或即将发币的协议更感兴趣，这些协议是加密货币领域最受支持的项目之一（也有空投潜力）。

zkSync

zkSync 是一个基于 ZK Rollup 的 Layer 2 网络，代表了以太坊扩展网络的「最终形态」。zkSync 是加密货币领域最受支持的项目之一，并已从 a16z、Dragonfly、Lightspeed 等投资机构处筹集了超过 4.5 亿美元。

zkSync 完整的主网还没有启动，但生态系统已经很庞大了，它将成为 ZK 生态中的「庞然大物」。

zkSync 的下一阶段是「Fair Onboarding Alpha」阶段，协议可以在封闭环境中开始部署在 zkSync 2.0 上。

StarkWare

StarkWare 是一种 Layer 2 扩容解决方案，它利用 STARK proof 来减少发送到以太坊区块链的信息量并缓解区块链拥堵。StarkNet 与 zkSync 一样，利用 ZK Rollup 技术来实现这一点。

StarkWare 也得到了大力支持，从 Paradigm、Pantera、Sequoia、以太坊基金会等机构筹集了超 2.85 亿美元，其生态系统也令人印象深刻。

Scroll

Scroll 是一个基于 zkEVM 的 ZK Rollup，可以实现对现有以太坊应用和工具的本地兼容性。在即将推出的 Layer 2 网络中，Scroll 的估值排名第四。

Scroll 曾表示正在创建一种「鼓励参与的激励机制」，可能会有空投。

Aztec

Aztec 是以太坊上专注于隐私的 ZK Rollup，为以太坊生态带来隐私性并节约成本。Aztec 最近完成了由 a16z crypto 领投的 1 亿美元融资，未来也可能会有空投。

Polygon（zkEVM）

Polygon 通过收购 Hermez Network（一个 ZK Rollup）以开发他们自己的 zkEVM 产品。zkEVM 是一种以透明方式执行以太坊交易的虚拟机，包括具有 ZK proof 的智能合约。

Hermez 加入了一个已经非常广泛的生态中，随着 Hermez 与 Polygon 合并，MATIC 将为网络带来动力。

Axiom

Axiom 是一个以太坊 ZK 协处理器，虽然 Axiom 不是严格意义上的 ZK Rollup，但我想将 Axiom 包含在此系列中，因为它为我们提供了如何在实际意义上使用该技术的一个很好的示例。

Axiom 允许开发者查询链上数据，并通过 ZK 获得无需信任的响应，使得开发人员可以以高效的方式通过 ZK 来增强他们的应用。
