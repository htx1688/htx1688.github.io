2 月 28 日，Yuga Labs 宣布将于本周晚些时候基于 Ordinal 协议在比特币区块链上推出 NFT 系列 TwelveFold，该系列将包含 300 件限量版生成艺术作品。Yuga Labs 联合创始人 Greg Solano 特意强调，选择推出 TwelveFold 虽然与 Yuga Labs 的预期背道而驰，但 Yuga Labs 对 Ordinal 铭文以及比特币数字艺术品的未来感到兴奋。

作为以太坊 NFT 生态中最具影响力的工作室，Yuga Labs 的这一决定再度掀起了比特币网络 NFT 的热潮，为了让大家更好参与比特币 NFT 生态，BlockBeats 将目前比特币 NFT 生态主要的工具，包括钱包、铸币以及交易平台等梳理如下。

生态钱包 

Ordinals Wallet

Ordinals 协议生态圈钱包，专为 Ordinals 开发的网页钱包和铭文浏览器，除基础的钱包功能外，还支持比特币 NFT 铭文浏览、检索、Collection 上传等功能，完成度较高，目前仍在不断更新中。

此外，Ordinals Wallet 还在近日进行了名为「Pixel Pepes」的 NFT 空投，总数 1563 枚，用户只需在钱包中进行一次交易即可获得空投，但目前该空投已经结束。

Hiro Wallet

Hiro Wallet 是一个基于比特币构建的钱包应用程序，目前已经支持 Ordinals，暂不支持闪电网络。

值得注意的是，Hiro Wallet 作为比特币生态钱包，拥有很多较为方便的内置功能，用户可以在 Hiro Wallet 内直接使用信用卡、借记卡甚至银行转账购买 STX，然后在钱包内直接参与质押。

目前该钱包支持 Chrome、Firefox、Brave 的浏览器拓展版本，以及 MacOS、Windows、Linux 系统的桌面端版本。

Xverse

Xverse 是基于 Stack 的比特币非托管钱包，现已加入对 Ordinals 铭文展示的功能。同时，Xverse 在昨日的更新中已经实现对 Ordinals 的转账支持，根据官方账号信息，相关功能仍在不断改进中。

铸造工具

Gamma.io

Stacks 上 NFT 市场 Gamma 原名 STXNFT， 2022 年 4 月 27 日宣布更名为 Gamma。Gamma 是希腊字母表中的第三个字母，表示网络的第三阶段：Web 1.0、Web 2.0 和现在的 Web3。

该平台旨在将收藏家、创作者和投资者聚集在比特币生态系统中探索、交易和展示 NFT。Gamma 平台由 NFT 市场、Launchpad 以及社交平台三个核心产品组成。Gamma.io 同时支持比特币 NFT 的一级市场和二级市场。

用户可以使用 Gamma bot 铸造自己的独特数字作品，收藏或售出。用户可使用无需代码的比特币 NFT 创建工具，在几分钟内创建成功。Gamma.io 解决了在比特币网络上创建 NFT 技术含量较高、复杂且耗时的痛点。但是，二级市场依然占平台销售的大部分。每笔销售都包括艺术家的版税以及市场佣金，百分比因艺术家和收藏而异。

Ordinals Bot

Ordinals Bot 是一款在 Ordinals 协议上铸造比特币 NFT 的工具，由 @satoshibleshttps://twitter.com/satoshibles 团队创建。该工具可免去用户在铸造比特币 NFT 时运行比特币节点的操作。

铸造流程：

· 用户进入官网后上传文件【jpg、.webp、.png、.gif、.txt、.mp 3、.mp 4(h 264) 等】并单击提交

· 通过闪电网络或常规网络支付铸造费用。费用为每个文件 0.00025 BTC + 10% 的网络费用，当然网络费用本身也需要用户支付。

· 转账完成后即进行铸造

· 如果用户提供了 NFT 接收地址，NFT 将会自动发送给用户

注意：如果用户不提供 NFT 接收地址，Ordinals Bot 将为其保留铭文，直到用户设置好为止。当用户设置好钱包后，只需访问订单检查工具即可添加地址并接收 NFT。

此外，近期刚刚推出新的批量 .sats 名称题词工具： https://ordinalsbot.com/sats-names 只需每行添加一个名称，Ordinals Bot 就会为它们添加必要的属性。

交易平台

generative.xyz

比特币链上生成式艺术 NFT 平台，与以太坊上的同类产品类似，网站效果十分华丽，成立时间不长，但已经 list 了众多艺术家的 AI 作品，可用 BTC 和 ETH 两种方式进行 mint。

原生协议 Generative protocol 测试网于 1 月 31 日上线，同步推出测试网 ERC-20 Token GEN，团队称该 Token 可转为未来主网 Token，用户可以通过上传艺术品、寻找 Bug、助力推广等方式获得 GEN 奖励。

openordex.org

Ordinals 去中心化交易平台，来自社区的开发者 Oren Yomtov 打造的极客项目，UI 风格十分硬核，已经陆续开发完成 Collection 展示等基础功能，值得一提的是，openordex 的中心化订单簿基于此前红极一时的 Web3 通讯协议 Nostr 开发。

openordex 目前的版本仅支持比特币 cli 客户端的买卖操作（节点客户端命令行窗口），以及 Sparrow Wallet 客户端（Ordinals 官方推崇的钱包）。

Emblem Vault

Emblem Vault 是一种 Web3 工具，它通过对比特币 NFT 进行封装将其带到了以太坊主网，使比特币 NFT 能够直接在 OpenSea 上进行交易。

简单来说，Emblem Vault 承担了一个「保管者」的角色——将比特币 NFT 交给他们，获取可在以太坊主网上交易的保管凭证，根据保管凭证对对应的 NFT 进行认领。

卖方使用 Emblem Vault 来 Mint 一个「Vault 保险库」NFT，并往 Emblem Vault 指定的比特币网络地址转入将要出售的 NFT。当检测到该比特币网络地址有资产存入时，「Vault 保险库」NFT 在 OpenSea 上将以卖方设定好的封面图及描述展示。

目前，Emblem Vault 可以使用来自比特币、以太坊、XDai、Polygon 和 BSC 网络的资产。

Ordswap

Ordswap 是一个构建在比特币网络上的 Ordinals 市场和钱包，可以在上面铸造、买卖 ordinals 铭文。

Ordswap 的整体 UI 和使用体验相对于以太坊生态现有的成熟 NFT 市场，并没有较大的差距，鉴于 BTC 网络数字艺术品当前的活跃度有限，以及 BTC 价格过高的情况，Ordswap 市场上当前的数字艺术品很少有成交价高于 0.1 BTC 的。

Scarce.City

Scarce.City 是一个使用 BTC 出价的数字商品拍卖交易平台，由 Chris Tramount 和 Aryan Jabbari 在名誉创始人 Chiefmonkey 的帮助下发起。

Scarce.City 通过即时、匿名、低费用的闪电网络和比特币主链，出价、拍卖和支付。交易的商品种类丰富多样，包括数字艺术品、虚拟土地、虚拟物品、NFT 等等。

参与者需要在拍卖期间以加价的方式竞拍，最高出价者将赢得拍卖。平台通过质押出价的 5% ，从而使拍卖参与者承担责任。

用户流程

1. 出价人存入出价的一小部分作为抵押品。 

2. 中标者在拍卖结束的 5 分钟内支付其最终出价。如果中标者 5 分钟内未支付，他们将失去押金，下一个出价最高的人有机会支付他们的最高出价。 

3. 最终出价支付结束后，平台将押金退还给其余参与者。

Ordyswap

Ordyswap 诞生的目的同样是为了实现比特币 Ordinal NFT 的交易功能。具体来说，Ordyswap 允许任何人为特定 Ordinal NFT 提交报价。为了创建报价，提交者需要锁定足够的 STX 代币，并制定其希望购买的具有特定序号 NFT 的比特币地址。

如果该 Ordinal NFT 的持有者接受了报价，则该 Ordinal NFT 将自动发送给报价方，同时将 STX 自动支付给 NFT 的卖出者，整个过程完全基于链上交易，不涉及任何中心化的托管过程。

目前的 Ordyswap 还处于概念验证阶段，项目的交易体验还很原始，甚至只能作为一个实验项目而存在，但 Ordyswap 的出现其实预示了比特币社区的强大开发潜力。
