撰文：Lily Z.King

编译：0xKin 深潮 TechFlow

作者简介：Lily Z. King 是新加坡加密货币托管平台 Cobo 的首席运营官。

最近，香港和新加坡都发布了政策声明和试点项目，表现出对加密货币创新的强烈支持。这激起了关于这两个金融中心相互竞争的激烈讨论。然而，真正令人兴奋的是，这两个中心可以相互补充，吸引全球资源，使亚洲成为采用加密货币的领先市场。

在 2019 年监管环境变得严格之前，香港一直是世界上最重要的加密货币交易中心之一。当时，香港是 BitMEX 和 FTX 等有影响力的加密货币交易所总部的所在地 。加密社区的许多早期采用者仍然与香港有着强烈的情感联系，这就是为什么香港政府对加密货币和 Web 3.0 创新的新热情被广泛称赞为“香港回来了”。

在过去两年里，由于中国的打击行动，新加坡从中国加密货币企业家的外流中受益颇多。尽管面临熊市，上个月（10月）在新加坡举行的 Token2049 会议变成了近年来参加人数最多的全球加密货币和Web3活动之一 。在 Token2049 会议上，我们发现许多刚刚迁居到这里的中国团队仍在积极打造产品。同时，交易商、投资者和行业领袖从世界各地飞来，因为他们相信亚洲的市场潜力。事实证明，即使其他地方正在发生 “脱钩”，新加坡也是一个融合的地方。

香港和新加坡的监管框架仍在发展之中，其详细政策将继续发展。但随着 FTX 的剧情展开，我们可以预期香港和新加坡都会加强对投机性交易的监管，尽管两国政府的做法已经出现了一些差异。

新加坡有严格的措施，限制零售投资者接触数字资产。虽然这些措施旨在保护散户投资者免受投机活动的影响，但也可能造成机会不平等的问题，使散户投资者/合格投资者和机构投资者相比处于劣势。

另一方面，香港可能会对零售投资活动更加开放。在其声明中，金融服务和财政局表示，他们将就零售准入问题进行公开咨询，特别是开放数字资产交易所交易基金（ETF）的可能性。

（注：本文写于11月，12月13日，南方东英资产管理有限公司正式推出两只虚拟资产ETF，南方东英比特币期货ETF及南方东英以太币期货ETF，该批ETF计划于12月16日正式在港交所上市交易）

好消息是，加密货币和 Web3 领域的企业家和投资者不必在新加坡和香港之间做出选择。资本和人员的流动——尤其是最近香港放宽了 Covid-19 的旅行限制，很容易让机构在两地开展业务并充分利用各自的优势。香港和新加坡之间的竞争也意味着它们将相互刺激，采取更大胆、更快速的措施。

在全球市场上，美国拥有一个无与伦比的风险投资、技术初创公司和深度资本市场的生态系统。它将继续塑造加密货币和 Web3 世界的基本技术、叙事甚至是法规。

但新加坡和香港可以提供不同的东西。区块链技术要真正成为下一代金融基础设施，运行在区块链上的智能合约需要与现实世界的资产进行交互。如果没有一系列复杂的链上和链下利益相关者的协调，从协议构建者到政府部门、银行和证券交易所，这是不可能发生的。

在美国，这种协调经常受到高度意识形态政治辩论的阻碍。由于在发布智能合约是否是言论自由、去中心化金融协议是否具有与传统金融服务提供商相同的合规义务，或者哪些代币是证券，哪些是商品等问题上持续存在分歧，监管的清晰度仍然缺失。

这就是香港和新加坡的务实做法可以发挥作用的地方。在守住抵御洗钱、恐怖主义融资和技术风险的底线的同时，香港和新加坡政府高效地将公共和私营部门联合起来，探索加密货币技术的新用例。

新加坡刚刚启动了机构DeFi的行业试点。新加坡金融管理局（MAS）将星展银行、摩根大通和SBI数字资产控股公司等机构聚集在一起，在公共区块链上试验 DeFi 应用，涉及用代币化的日元和新加坡元进行外汇兑换，以及用代币化的政府债券进行交易。香港也启动了一个绿色债券代币化的试点项目。

新加坡和香港正在做的事情似乎是自上而下和集中化的，与加密货币运动的无许可原则相矛盾。然而，通过将现实世界的资产代币化，他们给了机构和人们使用数字资产的理由，并有效地加速了加密货币技术的主流采用。随着更多的主流采用，草根创业公司也将为他们自下而上的创新提供更大的市场。

新加坡和香港还有另一个重要资源可以利用——来自中国大陆的科技企业家和人才。在 Web 2.0 时代，中国的互联网公司能够创造出在规模和创新功能方面与美国头部公司同台竞技的产品。他们的很多技术将继续在加密货币和 Web 3.0 经济中发挥作用。

事实上，在港交所上市的中国 Web2 巨头，如腾讯、阿里巴巴、Bilibili 和百度，一直在以不同的形式测试他们的业务或投资加密技术，但他们没有一个市场可以大胆地做实验。现在，香港可能正是他们需要的温床。

有些人可能会怀疑，香港能否制定与中国大陆根本不同的加密货币和 Web3 政策。但是，由于地缘政治和新冠疫情，中国正面临着真正的孤立威胁，它比以往任何时候都更需要香港作为连接世界的枢纽，并保持在全球金融的最前沿。

同时，新加坡也吸引了许多来自中国的科技公司，将其作为进入全球市场的一个门户。

如果我们放眼整个亚洲市场，仍然有大量人口无法获得便利的金融服务或稳定的投资机会。香港和新加坡的金融机构在该地区都有广泛的影响力，它们处于有利地位，可以利用加密技术增强该地区的金融包容性。

不幸的是，加密资产的上一个牛市周期在很大程度上是由投机驱动的，FTX、三箭资本和 Terra（LUNA）的兴衰提醒我们，投机和空洞的叙事不能给加密技术一个真正的未来。

只有当加密货币行业找到解决现实世界问题的方法并实现大规模采用时，下一个牛市周期才会到来。对于加密企业家而言，亚洲是一个完美的市场，他们不仅可以用链上代币进行创新，也可以用现实世界的资产进行创新，不仅为投资者，也为大众用户。香港和新加坡不是在争夺一个固定的蛋糕，它们都将在亚洲数字经济的下一个增长周期中发挥重要作用。

现在是一个最好的机会：亚洲的两大枢纽——香港和新加坡——将公共和私域资源有效联合起来，为加密技术和叙事提供最具有现实经济影响力的案例。
