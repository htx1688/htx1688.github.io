3.您在Web3中有哪些比较深的感触？

在Web3的理念方面，可能对于女性群体来说有一些偏见。在我的经历中，比如当我刚开始帮黑猫、静姐和推广 NFT 时，他们经常把Web3的女性称为“女玩家”或“女持有人”，而男性则没有这样的称呼。这表明在性别平等方面，我们还存在一些壁垒。当我开始涉足Web3领域时，一些人会质疑我的真实目的，认为我只是来混个知名度。作为女性，我还经常受到恶意评论的困扰，这也是很多女性在这个领域中面临的问题。

4.您认为Web3中未来有哪些具有发展潜力的赛道？

首先，我认为未来 Web 3.0 的发展方向有两个潜力比较大的赛道。

第一个赛道是 NFT，虽然目前市场并不看好，但我认为这是一个具有潜力的方向。目前 NFT 价格起不来或者增长缓慢，可能是因为被 Blur 这个项目所限制了。Blur 的项目奖励购买 NFT，并空投一些奖励，导致 NFT 艺术家或项目方被绑架或控制。例如，当 Blur 推出 Bed 去赚积分时，很多蓝筹 NFT 价格大幅下跌，因为 Blur 将原有的 NFT 平台交易模式变成了订单的薄交易所模式。这种新兴的模式对于短期投机者或想坐庄套利的人来说可能比较利好，他们会在低价时疯狂购入筹码，然后拉盘诱惑散户或机器人跟进，等到筹码集中时就在高位一次性抛货获利，赚取差价利润。现在很多蓝筹项目都被控盘了，庄家会在地板价时疯狂吸入账户进行存货持仓，最后再进行抛货砸盘，使价格快速下跌。因此，在 NFT 这个赛道中，我们要找到符合 Web 3.0 定义的去中心化社区运营方式，摆脱平台或项目方的控制。并且，我认为 NFT 是 Web 3.0 的一个分支，可以引起用户之间的共鸣，因为我们可以因为共同的目的、情绪导向等绑定在一起，交流心得体会和对项目的看法。虽然目前市场不太好，但我相信 NFT 市场终将会回来。

另一个方面，我认为将Web3和人工智能（AI）结合起来是一个非常有前途的领域。每天都有许多 Twitter 上的 Space 在讨论如何更好地将它们应用于一起。首先，在 AI 中，机器学习是非常重要的一个方面。当与Web3相结合时，可以扩展到Web3不同的网络层级，从而产生很多有趣的结合点。举个例子，AI 可以与公链相结合。公链可以称之为智能去中心化链，有几个关键层级，例如共识机制、数据层、激励层和合约等。在共识机制层面，AI 可以增强区块链的安全性，可以用机器学习去挖掘以往的数据，然后预测未来的发展，检测欺诈行为或者避免攻击风险等。在激励层方面，我们也可以通过机器学习实现我们的价值，并使下一代 Layer 1 或 Layer 2 更加智能。此外，我们还可以将 AI 与协议或 DApp 结合起来。例如在 DeFi 中，我们可以使用积极学习来评级不同类型的钱包贷款，或者在元宇宙游戏中生成自己的 avatar，并为游戏增强玩法，从而更好地满足用户的需求。最后，我也在探索将 AI 与创作者的经济结合起来。随着 AI 技术的发展，我们不是要节省或减少人类的工作，而是要更好地驾驭这些工具。当一些新兴的 AI 工具如 Mojor 或 Get 3D变得成熟时，创作者可以通过 AI 的赋能，并加上Web3的经济模型，将自己的创意和影响力无限放大，从消费者转变为参与者或所有者。虽然现在大多数 AI 和Web3项目还处于早期发展阶段，但将它们结合起来的潜力是非常巨大的。我们需要证明这些新项目和模式是可行的，并将它们应用于链上应用中，使其更加智能化、稳定和可信。

核心内容：我认为未来 Web 3.0 的发展方向有两个潜力比较大的赛道：NFT 和人工智能结合。在 NFT 领域，我们需要找到去中心化的社区运营方式，避免平台或项目方的控制，并发掘 NFT 与 Web 3.0 相结合的潜力。在将Web3和人工智能（AI）结合起来的领域，我们可以通过机器学习扩展到Web3不同的网络层级，从而产生很多有趣的结合点，例如与公链相结合、与协议或 DApp 结合、将 AI 与创作者的经济结合起来等。虽然这些项目和模式现在还处于早期发展阶段，但将它们结合起来的潜力是非常巨大的。我们需要证明这些新项目和模式是可行的，并将它们应用于链上应用中，使其更加智能化、稳定和可信。

5.您对于未来有什么展望？

核心内容：我加入了Web3社区半年，得到了中国Web3社区的认可。我希望打破圈层障碍，连接世界各地的Web3社区，了解国外项目方的合作意愿和跟国外社区进行互动和交流。我想成为一个Web3传递者，连接并帮助中国Web3社区更好地跟世界各地进行交流和合作。我希望能坐在一起，平等地讨论Web3的趋势、新想法和如何获取用户等问题，打破语言和国界的障碍。

6.您有主动想分享的东西吗？

我觉得来到Web3对我产生了很多影响，有三个方面。首先，我学到了很多前沿的技术。平时在公司的工作可能都是按部就班的，而在Web3圈子里，我可以每天了解到最新的消息和技术趋势，并与大家一起讨论。这让我觉得很有意思，同时也让我能够突破自己，去了解一些不同的东西。

其次，Web3让我思维更加开阔了。在这个圈子里，没有什么是不敢想的，我们鼓励创新和前沿技术的应用。这种创造力无限的环境吸引着我，每天都让我感到充满活力。我的工作也变得更有价值，我的作品可以被记录和关注。

最后，Web3也让我更加勇敢和自信了。一开始我可能会受到男女玩家或 builder 不平等待遇的困扰，但是随着与其他Web3人的交流，我发现大家都非常包容和多元化。这个圈子教会了我要做真正的自己，不需要被别人评论所定义，也不需要把自己的缺点拿去跟别人的优点比。我要认可自己的优点和缺点，并不断学习成长。在Web3，我的思维和心智都得到了提升，我觉得这是一段很奇妙的经历。
