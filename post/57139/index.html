8 月 15 日晚间，Gitcoin 开启了新一轮捐赠「Gitcoin Grants 18 」，本轮捐赠将持续至 8 月 29 日，并计划通过二次匹配机制为所有核心轮次（Core Rounds）的获赠项目提供 100 万美元匹配资金。

Gitcoin 的本轮捐赠共涵盖了四个核心轮次，所有轮次均由社区投票选出，具体包括了「Web3 开源软件」、「Web3 社区及教育」、「气候解决方案」、「以太坊基础设施」等四个大类。

除此之外，本次 Beta 轮捐还提供了「Web3 社交」、「代币工程」等多个特色轮次（Featured Rounds），这些轮次涵盖了与 Gitcoin 愿景一致的合作伙伴项目，但所匹配的资金量级较核心轮次要少许多。

接下来，我将在本述轮次中筛选一些个人相对比较喜欢的项目，并带你一起完成整个捐赠流程。

核心轮次：Web3 开源软件

  • 项目一：EVM Diff

EVM 兼容链的数量正在不断增长，但由于并未实现完全的等效，各个链的开发环境仍存在着各种细微的差别，开发者们需要注意这些差异，以防止意外的程序故障（zkSync Era 上曾有项目因此意外锁死过资金）。

目前，查找这些信息的路径相当繁琐，开发者们或许可以查阅每个链的技术文档，但这些文档往往却也存在着时效性问题，op-geth 或许是个可用的代码对比网站，但对于开发者而言这一比较方式太过底层，不容易理解。

EVM Diff 希望通过提供易读的方式来对比各个 EVM 兼容链在执行级别规范上的差异，解决当前的这一开发痛点。根据捐赠界面的披露，EVM Diff 计划覆盖以太坊主网、Arbitrum、Optimism 等各条链的元数据、预部署、操作码和交易/签名类型等等，包括在代理背后预部署的 ABIs 和逻辑合约地址。

  • 项目二：FundPG

FundPG 旨在提供一个更具可持续性的公共物品资助机制，该项目允许个人通过 Gitcoin 匹配池将其链上收益耕作的部分收益剥离而出，并用于公共物品捐赠（听起来有些像类似 gtcETH）。

除此之外，FundPG 还在构建开源分析工具，以追踪用于公共物品的资金的链上流动情况。

  • 项目三：Sybil-scorer

Sybil-scorer 是一个开源的 Python 工具包，其为开发者提供一系列用于地址分析的工具，以帮助项目方查找并过滤女巫地址。

Sybil-scorer 主要分为三个主要工具：sbdata、sblegos 和 sbutils。

sbdata 允许开发者从 flipside API 中快速检索大量数据，包括地址交易数据；sblegos 是 Sybil-scorer 的核心，提供了一系列用于女巫行为监测的分析逻辑，包括交易相似性分析等等；sbutils 则提供了使用 sbdata 提取数据的简便方法，以便在 sblegos 中使用。

未来，Sybil-scorer 的工作重点将放在添加新的分析逻辑上，并计划时间特征纳入算法之中。此外，Sybil-scorer 未来还将研究使用 flipside API 中的标签来过滤相似的钱包地址。

核心轮次：Web3 社区及教育

  • 项目一：ZachXBT（或许不该叫项目）

常刷推特的朋友或许对这个名字并不陌生，ZachXBT 是一个专门揭示各种业界乱象的匿名推特账号，许多网友将其称为“链上侦探”。

ZachXBT 比较知名的调查案例有：指控过黄立成的一系列欺诈行为，后者曾试图起诉 ZachXBT，但最终选择了主动撤诉；此外，前些日子 JPEG’d 被盗逾 6000 ETH，ZachXBT 事后查出攻击或与推特用户 @MichaelRazum 有关，该调查发布后不久黑客便向 JPEG’d 归还了赃款。

在本次捐赠中，ZachXBT 表示所有收到的资金将用于设备升级、未来的法律费用，以及读者认为其所花费的时间价值。

  • 项目二：JobStash

JobStash 是一个招聘信息的数据聚合平台，提供了各大组织、协议和开源生态系统的岗位招聘信息。

JobStash 目前索引了 200 多家组织，并跟踪了数千个工作岗位的数据。JobStash 完全独立，且没有收费机制，出于安全性考虑，该平台暂时只聚合了在社区内有一定信誉的组织所提供的工作机会，从而避免潜在的恶意欺诈行为。

  • 项目三：EIPsInsight

EIPsInsight 是一个用于以太坊改进提案（EIPs）的工具，它提供了过往所有 EIPs 的详细资料，并以数据和图表的方式呈现了各种 不同类别的 EIPs 的动态变化情况，从而允许开发者和用户更清晰地捕捉、了解以太坊的升级动向。

核心轮次：气候解决方案

  • 项目一：Litter Token

Litter Token 简单来说就是一个“捡垃圾 to Earn”的项目，用户捡垃圾并在 discord 内上传证明，该社区就会给你支付 LTK 代币（可以兑现）。

目前，Litter Token 已经推广到了巴西、尼日利亚、坦桑尼亚、乌干达、哥伦比亚、美国、多米尼加共和国、南非、菲律宾和波多黎各等 10 个国家，社区成员已经收集了 7.6 万件垃圾，且每一个捡垃圾的行为都拥有开源的图片证明。

核心轮次：以太坊基础设施

  • 项目一：Ethscriptions

Ethscriptions 是一个旨在让普通用户能够以合理的价格执行去中心化计算的全新协议，通过绕过所有智能合约的存储和执行，将确定性的协议规则应用于"简单"的以太坊 calldata 来计算状态，Ethscriptions 允许用户以大幅降低的成本在以太坊主网上共享信息和执行计算。

Ethscriptions 于 6 月 17 日上线，截至今日已经有超过 100 万个 ethscriptions 被创建，交易量超过 200 万美元。

  • 项目二：Stereum

Stereum 是一个以太坊节点的安装、设置和管理工具。

设置以太坊节点的技术要求很高，需要做大量的研究和准备工作。面对这一挑战，许多人选择了使用各类质押服务，这一选择可以理解，但却不可避免地加剧了以太坊的中心化。

Stereum 希望为那些有意愿运行以太坊节点但缺乏相关知识的用户扫除障碍。Stereum 它提供了一个直观的 GUI，只需单击几下即可简化节点安装过程，从而降低入门门槛及复杂性。Stereum 希望借此吸引更多用户部署节点，促进以太坊的去中心化。

特色轮次：代币工程

  • 项目一：Dynamic Optimization Algorithm for Self-regulating Tokenomics

该项目直译过来就是“一个可自调节的代币经济模型动态优化算法”。

当前，大多数代币经济模型的设计要么是静态编码的，要么依赖于 DAO 治理进行调整。这种方法确保了去中心化，但缺乏实时响应机制却导致模型很容易受到外部冲击的影响，例如价格大幅波动和黑客攻击。

该项目提出了一种全新的动态调节模型，旨在基于两个关键参数（质押率、销毁率）来让代币经济模型可以根据具体情况进行自调节。

从捐赠主页来看，该项目的开发者应是一名就读于芝加哥大学的华人学生，这一项目也是他的硕士论文的选题，其预计该项目将耗时 4-5 个月，最多不超过 8 个月，届时其会在 Github 上开源论文内容和相关代码。

如何进行捐赠？

首先，在捐赠之间，大家需要先行检查一下 Gitcoin Passport 的状态，这是一个会根据地址历史活动来评分的链上身份凭证，其目的是为了防止日趋泛滥的女巫攻击。

Gitcoin Passport 状态会每 90 天重置一次，只有状态处于最新版本，且评分超过 20 时才会在本轮捐赠中触发二次匹配机制 —— 简单来说就是你捐一笔钱，Gitcoin 也会同步捐赠一笔（没有达到分数亦可捐赠，只是无法触发匹配）。

接下来，我们就可以进行捐赠操作了，通过点击每个轮次中具体每个项目右下方的“购物车”图标，即可将其加入“待捐赠”的行列。

接下来，就是打钱环节了。我们可以选择以 DAI 或者 ETH 进行捐赠，数额可以自定义，确认具体数额后，点击右方的 Submit your donation 即可提交交易。个人建议大家选择 ETH 进行捐赠，因为使用 DAI 会有一笔额外的授权交易，需要更多的 gas 费用。

本轮 Gitcoin 捐赠的大多数操作均可在 OP Mainnet 完成，但核心轮次「以太坊基础设施」需要在 PGN 网络完成，需要进行额外的桥接操作，不过相较于上一次 Beta 轮（仅支持以太坊主网）的情况还是优化了一定的用户体验。

交易确认后，Gitcoin 界面就会跳转，蹦出来一句感谢的话语。

至此整个捐赠流程完毕，感谢你对链上生态发展所做的贡献。
