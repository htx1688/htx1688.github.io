作者：Decrypt；编译：GaryMa 吴说区块链

采访记者：Liam J. Kelly，Decrypt 新闻编辑

受访者：Alex Jupiter，MetaMask的产品负责人

Q：你在 EtherCC 大会上除了提及 L2，另一个重要的主题是“账户抽象”，让加密货币钱包更加智能。目前加密钱包领域已经涌入很多竞争选手，希望从MetaMask那里获得更多的市场份额。有些人一直在批评 MetaMask，认为它缺乏创新，并应努力超越目前的能力，以获取更多的市场份额。对于这些批评者认为 MetaMask 还有改进空间的观点，你有何回应？

A：我们一直致力于改进 MetaMask。我们的愿景还远未完成，我们拥有一支非常专注的团队，每天都在不辞辛劳地努力完善这个钱包。竞争对我们来说是一个推动力，促使我们在账户抽象方面不断进步。以太坊有两种类型的账户：

● EOA，外部所有账户，MetaMask 就是基于这种类型，其通过私钥签署交易。如果你丢失了私钥，你将失去所有资产的访问权限；

● 智能合约钱包，这种钱包已经存在了相当长的时间。

现在，借助像 Gnosis Safe 这样的创新，更多高级功能变得可能。这些钱包被认为更智能，因为它们提供了多签解决方案和社交恢复等功能。然而，我们不能忽视 EOA 带来的好处，比如完全的非托管控制和用户友好性，只需记住这些密钥。它还确保了高度的可扩展性。尽管如此，我们正在积极探索新的解决方案，因为我们明白随着以太坊的演变，MetaMask 也必须相应发展。拥有庞大的用户群，将超过 3000 万人引入新技术相较于只有少数用户的初创公司来说，自然需要更多时间。

Q：说一说为 MetaMask 引入新功能（如智能钱包）的挑战？是不是哪里必须做出改变、为以太坊引入 eips 以改变底层技术？

A：目前以太坊社区较为广泛关注的是 ERC-4337，它作为一个钱包解决方案，还是非常有吸引力的，它也不需要很长的时间来推进。我们对其周围建立更强大的生态系统感到兴奋，不过目前还有一些不成熟的地方，它还没有准备好迎接 3000 万用户。尽管如此，我们已经决定与这些团队合作，并推进钱包的发展。

Q：我想这次对话也会涉及到一个论点，即账户抽象必然是未来的发展方向，但也许它会与外部所有账户（EOA）共存，就像它们目前存在的方式一样。在场的每个人似乎都对智能钱包的趋势感到非常乐观，但你认为它是否有持续发展的未来？你认为这真的是加密货币钱包的未来吗？

A：时间会逐渐揭示 MetaMask 的发展方向，但我们的目标是促进行业内各种创新。我们对 MetaMask 的定位非常明确，我们希望开放钱包的功能以扩展其可能性。我们正在引入一套新的 API，称为 MetaMask Snaps，它允许开发者以任何方式扩展钱包功能。例如，如果他们有想法通过使用诸如多方计算或阈值签名等高级加密技术来增强安全性，他们可以创建一个 Snap 并将其附加到 MetaMask，以实现这种功能。

同样地，对于那些致力于开发账户抽象解决方案的人，他们可以开发一个Snap，并将其附加到 MetaMask，以扩展其功能。我们拥有自己的内部想法，但我们也希望使 MetaMask 具有灵活性和适应性，以适应未来的技术发展。我们的目标是保持敏捷，让开发者可以创建属于他们自己的 MetaMask 版本，希望在共同努力下，我们可以共建最佳用户体验。

Q：可以这么理解吗：人们想要做一些小的改变，可以添加这种 Snap 模块，本质上是调整它的运作方式？

A：它们不必是小改变。这些改变可能是巨大的。例如，我们最近推出了新的用户界面，以支持在多个区块链上使用 MetaMask。通过 MetaMask Snaps，我们可以逐渐添加全新的账户，同时也能引入全新的用户界面。交易分析仍然是钱包的核心功能，但随着 MetaMask 逐渐支持更多平台，未来的几年内，它可能会发生重大变化。取决于开发者选择整合哪些 Snaps，钱包的样貌可能会与当前状态有所不同。

Q：我记得你提到了 MetaMask 拥有令人瞩目的 3000 万用户。你打算如何继续增长用户数量？你是否考虑推出代币或实施更豪华的措施来实现这一目标？扩大用户基础是你每天密切关注的一个指标吗？或者，你的主要关注点是提升用户体验的质量？

A：我们当然专注于维护和提高我们服务的质量。虽然我们拥有令人瞩目的数据，例如每月活跃用户 3000 万和去年的 1 亿用户，但我们也意识到这些数字受市场和更广泛的生态系统影响，可能会有波动。因此，我们的主要关注点仍然是提供最佳的钱包体验。

过去我们观察到，像 NFT 这样的新创新可以导致使用量激增，我们希望能处于这类趋势的中心位置。作为安全且非托管的钱包，我们对安全性非常重视，因此我们会非常谨慎地选择合作伙伴。因为一旦我们发布了错误的或不足够优秀的功能，可能会带来严重后果，损害钱包的声誉和用户体验。因此，我们对每一项开发都非常谨慎和细致，确保我们提供最好的产品。

Q：这么说，你不是在迅速行动并试错，事实上却是恰恰相反？

A：是的，在构建钱包等 Web3 项目时，迅速行动并试错绝不符合 Web3 的理念。而且我认为我们的用户也不希望如此，因为他们信任我们。

Q：MetaMask 的使用量和下载量的激增无疑是有益的，而似乎 Layer 2 解决方案可能是接下来潜在的趋势，能够进一步推动加密货币的采用，并吸引更多用户下载 MetaMask。那么，你对 Layer 2 趋势对 MetaMask 的发展和计划有何看法？

A：L2 解决方案无疑将持续存在。或许在过去的一段时间里曾对其可行性进行过争论，但近期像 Polygon 这样的项目的增长和采用明确表明，人们渴望以太坊的安全性，同时也希望获得更快速和更具成本效益的交易。L2 正满足这一需求。我个人认为，这一观点也有大量证据支持，因为新一波的加密货币用户很可能会因其更快速、更便宜而选择 L2，而不是以太坊主网。

MetaMask 致力于支持每个 EVM 兼容的区块链，同时还有可能扩展到非 EVM 链，这将进一步推动在不同市场的增长和采用。然而，我们坚决决心在扩张过程中，始终保持高质量的标准，以确保我们所做的每一件事都是高质量的。

Q：MetaMask 是否发 Token 是在 Twitter 社区上成为一个热门话题，可以聊一聊吗？

A：实际上，目前对我们来说推出 Token 的实际效用非常有限。我目前没有任何公告要发布，虽然我理解你的听众可能对 Token 感兴趣，但我可以确认目前没有计划发布这样的公告
