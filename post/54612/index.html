超级链的进化之路：从 Cosmos 到 OP Stack

从 Cosmos IBC 到 Polygon 2.0——让我们来看看超级链（Superchains） 的进化之路??。

超级链的进化之路

Cosmos IBC –> Avax Subnet –> OP Stack –> Arbitrum Orbit –> Op BNB –> Zk Stack –> Polygon 2.0

Cosmos IBC

一切都始于 Cosmos IBC，这是一个处理两个区块链之间的身份认证 和数据传输 的协议。

虽然跨链区块链通信（Inter Blockchain Communication） 在 2021 年 4 月推出，但由于 Terra 的成功，这个平台的知名度升高了。

Cosmos IBC

然而，对于依赖于 Cosmos IBC 的 Terra 区块链来说，一切并未如愿，$LUNA开始崩溃，UST 的锚定失效。

这是加密货币历史上的一次黑天鹅事件，很多投资者损失了大量资金，这反过来对 Cosmos 的品牌造成了损害。

Avax Subnets

2022 年 4 月，Avax 推出了它的子网，这是整个 Avalanche 网络的一个子集或分区。

它由一组称为验证者 的节点组成，它们协作达成链内或相互连接的链内交易的共识。

Avax Subnets

你可以将子网视为更大的 Avalanche 网络内的一个网络。

它有自己的验证者集 ，并有自己的达成共识的规则和参数。

子网为 Avalanche 平台提供了可扩展性 和灵活性 。

OP Stack

OP Stack 指的是支持 Optimism生态系统的一套软件组件，包括 OP Mainnet 和 OP Superchain。

OP Stack

OP Stack 的目标是支持创建可以在生态系统内互操作的新 L2 区块链 。

OP Stack 提供了一个共享的、高质量的、完全开源 的系统，用于创建新的 Layer 2 区块链。

通过协调共享标准并避免在单独的孤岛中进行重复工作，Optimism Collective 旨在简化开发新 L2 链的过程。

Arbitrum Orbit

2023 年 3 月 16 日，Arbitrum 基金会宣布推出 Arbitrum Orbit ，这是一个允许任何开发者使用 Arbitrum 技术构建 Layer 3 区块链的无需许可 的解决方案。

Arbitrum Orbit

Arbitrum Orbit 是一个平台，允许开发者创建他们自己的专用链，这些链连接到 Arbitrum 的 layer-2 (L2) 链，即 Arbitrum One、Arbitrum Nova 或 Arbitrum Goerli。

使用 Arbitrum Orbit 的专用 AppChains 的主要优点是，开发者可以灵活地定制他们链的各个方面 。

这包括诸如隐私设置、权限、费用代币、治理机制 等定制选项。

opBNB

opBNB 是一个高性能的 Layer2 解决方案，它在 BNB 生态系统内运行，并使用 OP Stack 构建。

通过利用 OP Stack，opBNB 可以保持稳定且负担得起的Gas费。

opBNB 通过提高缓存效率实现了显著的性能。

opBNB

它旨在解决区块链网络面临的可扩展性挑战，特别是在交易吞吐量和成本方面。

opBNB 的区块大小为 100M ，允许在一个区块内处理的交易数量显著提高。

ZK Stack

ZK Stack 的主要目的是使得创建定制的 Zero-Knowledge (ZK) 驱动的 Layer-2 (L2) 和 Layer-3 (L3) 解决方案成为可能，这些解决方案被称为 Hyperchains 。

它基于之前的实现 zkSync Era 的代码。

zkStack

ZK Stack 是一个模块化和开源的框架，作为对 ZK Credo 中描述的挑战的回应而开发。

有关 ZK Stack 的详细内容已经在另一篇文章中描述：

Polygon 2.0

6月20日，Polygon实验室的工程团队最近介绍了一项将 Polygon Proof-of-Stake（PoS） 升级为 zkEVM validium 的提议。

zkEVM validium 是一个新颖的概念，结合了零知识证明和 EVM兼容环境 的要素。

Polygon 2.0

Validium 是用来描述区块链网络的一种扩展解决方案，结合了 Rollups 和 Sidechains 的各个方面。

它旨在提供比主链更高的吞吐量 和更低的成本，同时仍然从其安全性 和去中心化 中受益。

Polygon 2.0 是一个由 ZK 驱动的 Layer 2 区块链网络，通过一个新的跨链协调协议 统一。

它围绕着每个 Polygon 链都是一个 ZK L2。

目标是在保持现有功能和低费用的同时，提高用户的安全性。
