以上这段是我抄的《猜火车》开场独白，但其实我压根记不得这段“经典对白”，我只记得画面上的年轻人不停奔跑，一段超爽的蒙太奇，好似年轻就只能不停地跑，才能活下去 。

暮年回首，人的一生不过取决于年轻时几个漫不经心的选择，比如，从 WEB2 跳到 WEB3 。

在大多数人的第一印象中，从 WEB2 跳到 WEB3 当然是因为更高的薪资，更多的机会，毕竟在媒体语境中，“WEB2 已是末路黄花”，但后来我们发现不全然是这样。

就全球的薪酬体系而言，互联网一线大厂和国际投行依然是打工人的天花板，真实的情况是，一些人从硅谷大厂和一线 PE 跳槽去国际一线 Crypto Fund ，甚至得降薪三分之一或者一半 ，他们抛弃过往的高薪，只为在一个新的世界中寻找新的生活方式，以及新的可能性。

从 WEB2 到 WEB3，是什么促使他们做出这样的选择，他们后悔了么，在 WEB3 有怎样的收获和感悟？

带着此般疑问，我们采访了 4 位从 WEB2 跳到 WEB3 的不安分的年轻人，一窥他们的心路历程。

Dov，从美元 PE 到 MASK  

大家好，我是 Dov，之前的经历基本都在国内的一级美元基金，之前天使轮、成长期和后期的 PE（私募股权基金）我都有实习过，我在其中也跟着老板们学到了很多东西，在 Web2 的世界积累了很多经验。

但最后到了真正要思考要长期从事什么行业的时候，我最终放弃了同龄人中认为闪闪发光的美元基金，放弃了前往所谓金字塔尖的机会，加入了 Web3 的浪潮，去寻找自己的热爱与信仰。

就像张小龙提到的，「Web3 可能是一个虚假的狂欢，Web2 只剩下了真实的失望 」，我觉得非常贴切。

先来讲讲真实的失望。

我最后一段经历是在一家比较头部的美元 PE，这个行业的薪水很高，也很卷，非常忙碌也比较压抑，我每天大概工作 15 个小时，而且基本没有休息日。

我在成熟且专业的投资行业学到了很多东西，也非常感谢一直教我的老板。

但后面意识到这种生活状态不适合我，而且美元 PE 的层级制度非常严明，且有比较清晰的天花板 ；这对追求稳定高薪的朋友非常适合，但是对我这种比较折腾的人来说不是一个特别好的选择，大机会是老板们的，而不是新进入的年轻人的。

第二点就是，从大环境来讲，美元基金的荣耀来自中国移动互联网的崛起 ，京东、拼多多、美团、抖音、快手这五个超级互联网公司成就了美元基金的荣光，投中一个就可以跻身一线基金。但这都是旧时代留下来的辉煌。

上个时代来自移动互联网的辉煌正在慢慢褪去，随着地缘政治变化和反垄断政策，当下能投的似乎只剩下硬科技和新能源 ，这二者本身都是制造业，和美元基金所擅长的互联网垄断相差甚远。

同时，美国资本投资中国的硬科技本身就是一个不合理的事情 ，中国怎么会让核心技术有国外资本，美国又怎么能允许中国发展高端科技？

而新能源又基本都是人民币基金的天下。美元基金现在还有什么赛道能投？我不知道。可能2年之后这个行业就不存在了。因为个人性格和对大环境的判断，我离开了传统美元基金。

再来讲讲 Web3的狂欢。

对于我这种商科（非技术类非工程类）背景而言，比如新能源、制造业这种赛道并没有我能抓到的红利，我在这些行业也没有核心竞争力。最后看完所有的机会，似乎只剩下了 Web3，非常幸运的事，这也是我很多的信仰所在。

选择 Web3 的原因在于我身边最优秀的年轻人们，无论做硬件还是做软件的人，可能都在这一两年都在进入这个领域，而且做得非常火热。

我在决心加入web3之前大概聊了20-30个我认为非常厉害的投资人、创业者和年轻人，所有人对web3都是非常看好，且将会投身其中。因此，从人才供给来看，这个领域未来一定会有蓬勃的发展 。只要整个赛道是往前发展的，你的价值就是会一直往前走。

所有的努力、所有的选择都是要遵循周期和趋势的 。能够英雄造时势的人寥寥无几，和乔布斯和马斯克一样改变世界的人，大家还能列出几个？我相信必然不是。所以我是一定在跟着趋势在走，去寻找有 Beta 的行业，去有鱼的地方捕鱼。

不抬头看天，不去选有红利的行业去奋斗是很傻的行为 。如果闷头拼命就能赚钱的话，那世界首富应该是驴。

选完了赛道之后也要在对的时间做对的事情 。我只想再次警告，全球经济马上将迎来巨大的衰退。年轻人的机会非常少，但这不是你的原因，而是整个用人市场的原因，因此请不要焦虑，去顺着周期去做该做的事情。

在熊市要去休息、学习、沉淀，去修心，去扩大自己在圈内的影响力 。要在全球经济衰退周期中找一个能过冬的地方，如果每天都要担心自己会不会失业，你的情绪都会花费在焦虑和恐惧中，你将积累不到任何东西。只有生存下来了，在下一波的经济复苏中，你依然会有一个比较好的发展。

另维 Anodime，从传统金融到 WEB3 VC 投资总监

我也没有什么传奇经历，我小时候就是写作，7 岁去中国作协的鲁迅文学院学习，后面进了作协，从小发表文章，出书，如果你去微信读书搜我也是可以搜到我的作品的。

得益于小时候的写作功底、在国外读书的经历，以及在腾讯购买了 NBA 转播权后，我成为了初代女主播和波特兰开拓者的现场记者，2009 年又靠着微博的验证身份，收获了第一批粉丝。靠着我的写作能力和摄影能力，加上旅行留学生的生活，成为了一个在社交媒体上的内容创作者。现在全网大概有 500 万粉丝这个样子。

做博主基本只有两条路，一条是 To B 端，和品牌合作，另一条是 To C 端，开店，或者是培养其他博主等等 。但都不适合我，我一直在思考如何让流量落地。之前投资过餐厅，做过服装品牌，都失败了。最后，我做了知识付费，做了自己的社群，包括现在也做了自己的 Web3 社群。

但旅游博主是一个行业积累不能随着你年龄增长进行很好地正向增长的行业，永远都会是十几岁的小姑娘在镜头前更好看。

我觉得我也算是吃到了整个 Web2 的社交媒体红利，但我却没有看到在这个角度上还有更好的职业发展，博主大多数也只是路径依赖。所以我意识到不能在这条路上继续走下来，这不是努力的事，这是方向性的问题。

然后我回归了校园，毕业后去做金融。第一份工作在普华永道，第二份工作跳槽去了 Asset Management，第三份工作又跳槽回了美国，在一家叫 Equity Research 的上市公司。最后，我选择了 Web3，在一家 VC 当投资总监。

一开始我对 Web3 是没有足够的认知的，但 Web3 世界中的人都很开放，也都乐于分享。在进入 Web3 这个过程中，我交了很多新朋友，不断地巩固和提高我的认知。我觉得只要你这个人是在学习，是在吸收，是在输出的话，Web3 对于你就没有什么弯路可言。

我觉得弯路本身就是一个悖论。我不觉得我来 Web 3 之前所有的失败创业是弯路。我也不觉得我在其他地方花了精力弯路。我们过去的一个个点都会去找一个方式连接起来，那就是我们自己的路。

对于选择前往 Web3 的原因，我想得非常清楚。Web3 是一个非常符合我价值观的地方，因为 Web3 是一个完全没有国界的地方，和从 13 年前就一直旅居世界各地的我非常契合 。

还有一个原因是，Web3 行业的特性是我认为世界未来发展应该的朝向 。大家都知道 Web1 是 Read Only，Web2 是交互：Write&Read，Web3 是 Onw+Write+Read。Web3 用 Token 代表了 Ownership（所有权），对于我来说这是一个非常伟大的金融创新。

最后我也希望能够把自己在 Web2 社交媒体的经验嫁接到 Web3 行业中，继续稳扎稳打地做一些好的输出。还有就是希望能够深度参与自己投的项目，尤其是在我最熟悉的运营部分，和媒体相关的部分，和项目方一起从 0 到 1，走向成熟。

子时，从大厂到交易平台

大家好，我是子时，最早在一个投行里实习，感受到了金融的美好。所以我就想往金融的方向靠，大概面了 120 多家公司，最后是一家中小型 VC 要了我。开始正式工作后，我一直在看游戏和文娱这块的投资，后面我就去了腾讯做游戏策略，去了光子工作室做各种各样的研究，研究游戏策略和战略。

大公司给予我的瓶颈是，我只能看到公司从 100 是什么样子，却看不到从 0 到 1，也看不到从 1 到 100 。

在腾讯中，不管你做什么业务，其实你都没那么重要，真正的大业务也不会靠你。后来我看了看深圳的房价，算了算再工作三十年可以在深圳买个厕所。所以，我希望能够选择一个赔率更高的赛道，让自己在年轻的时候拥有更多的储蓄和职业发展的更多可能性 。

另外一个放弃腾讯的原因是，我加入腾讯的时候算是它最后一波高潮，股票从 500 冲到了 700，然后从 700 一路俯冲，到了 200 多。所有人都会想到，「（腾讯的）泡沫破裂，一切回归到真实的价值 」。

这个时代给我们留下了什么？这是一个非常痛的领悟：只要跟随着时代的脚步去努力，跟随着公司的脚步去努力，就像顺水推舟一样，你能够创造无数的价值和无数的财富 。但是，如果没有增长怎么办？这是很多年轻人还没有经历过的一个问题。

我进入 Web3 的契机就是：Web3 给了我更多的钱，还有更多的公司可以一起做贡献，你可以为一个 DAO 做贡献，影响更多的人，影响更多的公司，做更多有影响力的事情。

去年我们踩中了很多好玩的项目，比如 Gala Games、Big Time、Thetan Arena，还有一个叫 CloneX 的项目，那时我们赚了很多的钱，今年却全赔了回去，甚至赔了更多，本金都没了。在这个大时代的变迁中，个人很难去抵抗和作出反应。不吃这个亏，你很难理解现在正有一双无形的手正在收割全世界的财富。

这是我们走的最大的弯路，吸取到的教训就是要「顺势而为 」。

人生中赚钱的时间很短暂，亏钱和不顺利才是最常态 。你一定要去判断，在感觉机遇来临的时候，要抓紧时间去做很多自己能力所能及的事情。但是在机遇没有到来之前，就要管住自己的手，老老实实地去学习修炼。

我最大的感受和教训就是没有做到激流勇退，没有做到深刻的实时反思，也没有跟上时代的变迁。

所以我的想法就是 ：在接下来的一段时间中戒掉所有二级操作，然后去帮助朋友做一些 Buidl 和行业贡献，做一些力所能及的事情。能帮朋友就帮朋友，能帮到项目就帮项目，能帮到公司就帮公司，能发一点光就发一点光，能就出一点力。

因为我之前经历了非常多项目的周期，也有帮助别人成功融资过，或成功上过交易所，就希望用自己的这些经验帮助更多的人，参与到这个机制当中去。并且，我希望能在这个过程中想到一些更好的模式，让更多的人参与到 Web3 当中来。

这是一个不可逆的浪潮 。比如在蒸汽机时代，有很多顽固派，身为工厂主的他们不希望蒸汽机替代劳工，所以有许多关于蒸汽机的对抗和监管合法化的问题。之后的电力时代也是类似，需要一个普及和教育的过程，它可快可慢，接受程度也有高有低。

例如互联网即属于接受程度较低，但是它的变革却非常深刻 。当时对于互联网的监管也经历了一个对抗的过程，从最早的信息泄露各国家之间的网络政策、公司之间的开源、从开源到闭源、算法对于人的压榨之类的探讨，我们发现互联网的监管也是逐步完善的。

Web3 这个行业需要做的事情，就是寻找更多真实的价值，挖掘大家离不开或者必须进入其中的理由 。先是自己想清楚，同时大家也在思考的时候，说不定能看到一个桅杆，它一开始在海平面上，只是个桅杆，是一条线。但当这个船出来之后，你会发现其是个巨轮。现在看不到，几乎连桅杆都看不到。所以我们迷茫在这，焦虑于此，但是坚信巨轮会有驶向岸的一天。

Alex，从一线投行到 WEB3 VC

我现在在 Foresight Ventures 做研究员，主要关注的两个赛道就是 NFT 和 GameFi。

我之前在 CICC（中金）做债券相关的工作，但我觉得在这里，你个人的努力不重要，成长速度和个人发展会受限很多。而在一个新兴领域，比如 Web3，我们每个人的成长速度都很快。

如果你在一个传统行业或者阶级分明的行业，个人努力真的就没那么重要，你需要做的只是去做一个螺丝钉。这也是促使我想跳出来的一个主要原因，我想要有自己的贡献，我想做一些不一样的事情。

当时我也尝试了各种各样的方向，拿了一些 offer，但最后选择了 Foresight。我比较喜欢它的点是，它本身是一家偏 Aggressive 的公司。作为一个年轻人，或者做一个朝阳行业，是需要向上的动力去推进的。

我是一个喜欢接触新鲜事物的人，我不是一个无聊的，喜欢在一个环境中呆很久的人，这也是我愿意加入 Web3 的原因。我很享受学习 Web3 这个过程。至于困难我其实没怎么遇到过，因为我从传统行业的一个比较好的投行跳了出来，会拥有一个比较大的机会成本。

除了那些上学或者有孩子的人转换成本更高外，年轻的情况下，我更倾向于做出更加冒险的选择，以博取更大的 Alpha。风险更高，回报更高 。

在上一轮叙事牛市中，很多的人是赚到了钱的，而其中很大的原因是「敢想敢做」。撑死胆大的，饿死胆小的 。

圈子里的人年龄偏小，画像偏 aggressive，这跟传统领域就不太一样。我金融院校出身，身边很多朋友都想进投行，想进一些光鲜亮丽的组织，但你会发现我们圈子里的很多人就很接地气，他不关心自己的项目有多高尚，他只关心自己的项目能不能创造价值，如果能从创造价值的角度攫取到用户，那就很有意义。

但在传统领域很少会出现这样的人，他们会说我还年轻，我应该再积累。积累就是去一家大企业，锻炼两年，再跳槽去另一家，再锻炼三年。靠着这五年的经验，我再去创业。

而在 Web3，不用有经验，我就是干，我有这个想法我就要冲，而且我也可以融到资。即使作为一个学生，我都有机会拿到几千万甚至上亿的融资。

PartyDAO 就是一个很好的例子。它就是做一个团购 NFT 的协议。我其实在今年一月份就有了这个想法，甚至找了合伙人，写了白皮书，但因为我那时还没有 all in crypto/all in Web3，我没有勇气去投出我的 Deck，我甚至怀疑我这个是一个伪需求。然后今年四月 PartyDAO 就拿到了 a16z 的投资，而且那个创始团队我觉得没有我好，它的思路不如我最初的设想。

我觉得我当时缺少的，就是一个敢想敢做的态度。

我接下来的计划就是先积累，因为现在是熊市，不是一个很好的时机。我觉得每一个做 VC 的人，最终的目标一定是拥有自己的项目。之后我也会想要寻找一些不错的机会，自己创业。这是我最终的目标，也一定会尽力而为 。

现在我要做的，就是积累和等待 。
