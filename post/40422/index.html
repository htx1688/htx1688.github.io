一个加密 Degen 的经验之谈。

原文标题：《小浣熊投资概念杂谈 50 则（极主观也许错）》

撰文：Raccoon Chan 小浣熊

综合：

1. 不要迷信 BTC.d，2020 之后进圈的人，不到两成有 BTC，他们避险也不会换成大饼，另外大饼在 DeFi 和 CeFi 的 LTV 高，隐含的槓杆更大，你拿着 Shitcoin 固然解锁会砸，但是是之后的事，本身抵押 Shitcoin 不能加槓杆，Cex, CeFi 还是 DeFi 也好也不怎么能，所以没有负循环一说。

2. 解锁砸盘不能作投资的主要 Thesis，一来很多人早已套保，二来锁仓代币 OTC 交易也很活跃，很多时已经消化，真要沽空，也限制自己的仓位在一两日内赚蚀也要 Close 掉。

3. 挑项目时看 VC 的三大作用

  • 有稍微听过名字的 VC 可以省掉核查项目方会否 Hard Rug 的功夫；

  • 融资额相对团队大小和类型足够的话，不用担心他们不够钱发薪水倒在黎明前；

  • 超过三个垃圾 VC 的项目直接不看。

其他情况下，是大 VC 还是小 VC 还是没有 VC，不是项目好坏的因素

4. Audit 来说 Trails of Bits, OpenZepplin, ConsenSys 两家以上比较好，但都是只作参考。有 CertiK, Slowmist, Peckshield 这些也不错，但 Audit 的作用更多是证明项目方自己愿意花一笔「冤枉钱」进项目和愿意花时间排队等 Audit，不会赚几十万几天就割，让用户对 Code 比较安心只是配菜，另外看到垃圾 Audit 公司可以帮你筛选垃圾项目。

5. 交易所尽量用主流的（Binance, FTX, Coinbase），其他可以开个户来交易些小币，放大钱就要谨慎。

6. 在适当的地方获得适当的资讯，下面按顺序排重要性

  • Twitter：找热点 / 新项目 / 看新闻

  • DefiLlama/Debank：新项目

  • Discord：项目方 Server 在 Community Call 的零碎更新 +Token-gated 小圈子聊 Alpha

  • Telegram：找热点 / 看新闻 / 看市场气氛（Telegram 的新项目慢于 Twitter）+ 小圈子聊 Alpha

  • 数据网站（Nansen, Dune）：Nansen 捉老鼠、Dune 看数据

  • 英文付费媒体（Delphi, Messari）：学习 Narrative 怎么说和怎么向别人 Shill bag

  • 英文 / 中文免费媒体（Coindesk, 律动）：看有没有什么漏掉的旧闻和看广告

其他：Line, Youtube, Facebook, TikTok, Instagram 这些看人吧。

7. 给项目算 PE 算 DCF 的都是扯淡。有算上 Token emission 吗？有分开给 LP 的 Fee 和给 Governance Token 的 Fee 吗？有考虑项目方法人和持币人之间的 Agency Problem 吗？最低 PE 的是 Sushi 和 Mkr 的高点，PE 最不合理的是 3u 的、接近没有 Fee 的 Solana，你还信 PE 吗？

8. 乘上关于治理代币的问题，基本分成有分钱和没有分钱两个类别就够了。有分钱的，多与少也好，表示他不怕证券法了，豁出去了；没有分钱的，表示他很怕证券法，不要信他们的虚假承诺，怕证券法的，好像已婚的、怕老婆的渣男跟第三者说早晚会和正印离婚的，这个早晚，也许是一辈子。

9. 每个国家的项目都有他们的调性，主要不是说民族性格问题，而是法规问题，好像美国项目就没那么敢分红这样，或者有的国家警察没那么积极抓人的，就常常 Rug-pull，其他不说这么多了，懂的都懂。

10. Fair Launch / Stealth Launch 的项目很好，但不要开盘 Fomo，因为起始供应很低，Curve 早期锁 $CRV 四年仓的人都赚烂了，但不包括第一天衝矿币的。

11. VC 大巴（上百个 VC 投很大份的）可以短炒，挖更好，但不能锁，更不能锁得比 VC 解锁时间更长。

12. 币圈的合约，SAFT 也好，SAFE 也好，仅作参考之用，流氓不会跟你谈合约精神的。

CeFi 篇：

1. CeFi 放贷是糖衣毒药，就算不是 Alex 这些本身人品有问题的，你那个 Yield 都是来自他向更高风险的赌仔 Counterparty 借贷套出来的， 譬如他给你 8%，他借给外面的也会是 10%+，哪来这么多年化 10%、Max drawdown 不高、策略还是 Scalable 可以到几个亿的交易员，还没算上他们需要的利润呢，很大可能那些人就是扛着 10% 资金成本去挖 100% 的矿，你安心吗？

2. 双币理财不能说是骗人，但是是由期权组合出来的，收费屈指一算，很多时候比你自己去做期权组合要高 10%-20%，看你愿不愿意。

3. 把网格包装成收息产品是骗人，你有想过「利息」是什么吗？所谓机械化高卖低买，撇除情绪的影响，就是鼓励你被动频繁交易去付 Fee 给交易所和推广人，他们不会提及 Long-tail risk，也不会告诉你这些网格公司的手续费如何比起直接用交易所高，币安 BTC 对免费，FTX 挂单也免费呢，网格不叫量化，而且不适合大多数人，你交易容易受情绪影响，就不要交易，不是去搞什么网格。

4. 不要相信云挖矿 / 矿机合约，除非你能验证他们的算力。

DeFi 篇：

1. 挖矿求简单，越複杂、越创新的项目越容易被骸，你不是投资项目，你是在挖矿赚年化。

2. 稳定币挖矿，如果怕稳定币脱鈎（如 UST），可以选择放 USDC 进借贷协议借 UST 出来拿去挖，这样你也有十几趴，脱鈎也与你无关。

3. 流动性挖矿本质上是项目方向你付出租金去借流动性，不能说一定是 Ponzi，但如果借回来的流动性没人用，那就是 Ponzi 了，所以评估项目的时候，看资金利用率、看日活，有时候比看 TVL 重要。

4. DeFi 三宝（借贷、交易所、合成资产）裡面，借贷是最多猫腻的，项目方最容易使诈 / 出包的，年化不够的话不要搞借贷矿，更不要搞循环贷 Loop Loop Loop。

5. 一定不要挖桥，比借贷更高风险。

6. 现存的所有 Fixed-income DeFi 板块已死。

7. 现存的所有 CeDeFi / RWA 板块已死。

8. 链上合约是卷爆的赛道，但哪个做成了，市值应该能前 20，所以才卷爆。

9. 牛回之前，玩法简单暴力的 Ponzi 可能有，但叠娃太多层的 Ponzi 应该没了。

NFT 篇：

1. NFT 不是卖 Jpeg，单单因为图片好看而成功的寥寥可数，觉得 NFT 只是卖 Jpeg 的，最后都会亏钱。

2. 中港台明星站台的项目真要买的话，加倍谨慎。

3. Roadmap 大多都是抄回来的，甚至很多是随便找个人写的。

4. NFT 基本只有在 Ethereum L1 和 Solana，发在 Polygon、L2、Flow、Avalanche 的不是说不好，但开局先天难度更高。

5. 大多数白单是不值钱的。

6. 挑对人比挑对项目更重要，我买 Cryptoadz 是挑 Gremplin，Loot 是挑 Dom，IT, WT 是挑 Gaba。

7. 短期保持地板价的是社区感情和 Vibe，长期保持地板价的是贩卖期望，发币的期望、空投子项目的期望，比起实际发币和空投更有用。

8. OpenSea 市场佔比会下降，至于是其他竞品（Looksrare/Magic Eden 这些），还是垂直板块的（Foundation 这些），还是代币化的（NFTX, Sudoswap），还是横向发展的（Uniswap 这些），还是中心化的（Binance, FTX, Reddit, GameStop），很难说，但早晚会有缺口的。

GameFi / Web3 / SocialFi / DAO / Infra / Others

1. GameFi 我还在学习，不妄加评论。

2. 相信 Web3 的不能对中国项目有偏见，不然你视线内的项目应该挺少的。

3. Web3 核心在于 Own your data，只要底层不违背这个，项目方们，前端可以做得比较 Web2 吗？

4. 小团队又想搞 Web3 Facebook / Twitter / Telegram/Reddit 这些的，加倍谨慎，一来太卷了，二来改变用户习惯需要数倍好于本来的产品加上铺天盖地的品牌宣传加上炒作一个 Web2 Competitor 作恶的契机，不是小团队做得了。

5. SocialFi（那些下一个 Robinhood 下一个 eToro）也是依赖 Marketing 的，项目方们，烧点钱吧，浣熊欢迎投币。

6. DAO 不是 Web 3 时代的公司制度，只是 Web 3 时代的董事委员会。

7. DAO Tool 分开用 Snapshot+Gnosis+Forum+Llamapay 这样，比起现存的一站式 DAO Tool（如 SyndicateDAO, DAOhaus）好用多了。

8. Something should DAO but not everything should DAO.

9. 钱包是最重要的基建，尤其期待一个 Newbie-friendly, NFT-friendly, Mobile-friendly 的钱包。

10. 几家 RPC Provider 好是好，但估值一来贵，二来散户也投不了，不用看。

11. 去中心化 VPN 很贵啊，可以便宜点吗？

12. 桥需要这么多条吗？这条赛道是卷之馀，会是大家都输，没有赢家的卷，桥的用户太 Price-sentive 了，你一加价用户立即跑，没有 Uniswap 那种忠诚度。

唠叨一轮，都只是一个 Degen 的经验之谈，也许很快就今天的我打倒昨天的我，也许有些新项目新赛道我也认知不够，就靠着一直玩，和继续挑逗各个赛道的 Founder 聊天，去修正自己的认知吧，继续学习，欢迎指正。
