Layer 2 的大规模采用

在区块链领域，实现扩展和增加交易吞吐量是明确的需求。在本次牛市的最后阶段，Solana 和其他 L1 项目引起广泛关注，这很大程度上是因为以太坊的扩展速度和吞吐量无法满足牛市中对其区块空间的需求。

虽然市场上有许多区块链标榜着自己具有极快的速度和较高的吞吐量，但持续的停机和需求高时交易的缓慢使得用户体验非常不好。这些区块链是单一且模块化的，并试图在单个产品中提供安全性、分散性和吞吐量，但是他们无法适用于大规模的采用。那么提供高吞吐量、高安全性和高度去中心化的解决方案是什么？

L2 解决方案也许是这个问题的答案。L2 提供了增加吞吐量并以此拓展区块链的能力，同时继承了主链的安全性。例如 StarkNet 和 zkSync 支持批量处理，即将数千个交易捆绑到一个交易中，并将其发布到主链（以太坊）上。这种方法可以将每笔交易成本降低几个数量级，同时提高区块链的速度和吞吐量。

为了提高区块链网络的采用率，我们需要它更快、更安全和更加去中心化。L2 提高了交易速度，L1 提供安全性，二者结合可以让区块链网络在全世界大范围采用，并成为下一代互联网的支柱。例如每秒处理数千次延迟的电子游戏、即时共享的流媒体和以秒计的转账交易。

这也是 L2 为什么在下一次牛市中尤为重要的原因。目前，zkSync、Starknet、Polygon Hermez、Optimism 和 Arbitrum 等 L2 解决方案都在积极开发中，并取得了大量进展，未来会很快达到大范围采用所需的基本标准。

DeFi 和 NFT 的成熟

上一轮牛市很大程度上是由两件事驱动：DeFi 和 NFT，但二者都还处于起步阶段。在未来 NFT 会比 DeFi 更重要。

为什么 DeFi 会促成下一个牛市？

DeFi 是一个开放的金融系统，可以让任何人、任何地点、任何金额的资金参与其中。它不仅可以为没有银行账号的人提供金融服务，而且使金融科技得以进一步发展。由于 DeFi 还处于早期阶段，它目前无法达到中心化金融带来的服务体验要求，但它可以弥补传统金融的效率低下等问题。在 Web3 时代，DeFi 也会有更多的应用场景。

现有的中心化金融体系 CeFi 是一个由银行和政府控制的巨大围墙，他们制定规则来决定谁可以进行借贷，借贷多少金额。DeFi 借贷协议和服务可以提供与银行相同的金融服务。DeFi 协议是完全透明且无需信任的，每个人都可以通过公开地址查看资产金额，并根据自己的标准来评估债权人和债务人的财务状况。DeFi 引入了新的效率、新的机会，开辟了一个崭新的金融世界。

那些被认为是 jpeg 的 NFT 如何引发下一轮牛市呢？

非同质化代币 NFT 是真正数字所有权世界的特洛伊木马。在牛市的最后阶段，无聊猿和像素朋克火爆出圈，这些天价的 NFT 丝毫无法浇灭人们的购买热情，而这只是一个简单展示数字所有权基本使用的开始。

当我们在实体店里购买一件衬衫时，这件衬衫在任何意义上都是不可替代的。但在 NFT 出现之前，这在数字领域是不可能的。当你线上收藏一些虚拟商品时，你购买的只是存储在公司租用的服务器上的一些数据信息。而使用 NFT，资产则可以存放在由你控制的钱包里，这项技术很可能会帮助游戏、音乐，一直到房地产和金融等行业的商业模式实现变革。

Web3 单点登录流行

Web3 单点登录身份验证允许使用单个独立的密码和账户登录网站，而无需将个人详细信息附加到新网站和现有的主要平台上。

Meta、亚马逊和谷歌等公司几乎拥有你所有的隐私和数据，包括账户密码、出生日期和网上浏览记录等。数据收集市场价值达到数十亿美元，通过用户数据和隐私赚钱已经成为互联网公司的常态。

Web3 单点登录可以解决这个问题，它允许公司收集数据，了解用户的偏好，并为用户提供个性化服务，但前提是在用户允许的情况下，他们才能拥有和使用数据。

Web3 时代网上冲浪在未来可能是这样的：创建一个数字钱包，并可以通过它链接任何一个网站。当需要访问其中任何一个时，只需要通过钱包登录就可完成。

GameFi 兴起

在传统游戏行业中，玩家通常需要付费来获得游戏账户或者购买游戏道具，然而在游戏中，获得的每一个物品和角色都不是属于用户本身的。用户数据和虚拟道具被游戏公司掌握在自己的服务器中，如果用户停止玩游戏或者停止付款，他们就会失去账户和账户资产。

这意味着游戏玩家沉浸在游戏中的所有时间以及为游戏本身增加的所有价值都被游戏公司剥夺了，而目前游戏行业的利润比所有其他媒体行业的总和还要高。颠覆这个市场的时机已经成熟，允许玩家将时间和金钱投入到可验证拥有 NFT 的区块链游戏中，将会为游戏和玩家创造新的运营模式。

不幸的是，目前非常缺乏可玩性强的 Web3 游戏，但在这个熊市中，可以注意这些变得可玩且拥有强大社区的游戏

  • Influence：太空战略 MMO

  • Parallel ：科幻卡牌游戏

  • Loot Realms ：大型多人链上骑士游戏

  • Guild of Guardians：移动角色扮演游戏

音乐 NFT

如今，艺术创作者谋生比以往任何时候都更加困难， 尽管音乐的传播能力非常简单，但是流媒体服务和唱片公司占据了大部分利润，而艺术家只能获得极少的一部分。

作为一名艺术创作者，首先要极其幸运才能被唱片公司或媒体选中，只有非常少的顶端艺术创作者才能获得较高的收入。

在 Web3 中，艺术创作者可以通过新的经济模式，与他们的社区建立更加紧密的联系，同时对他们的作品拥有更加充分的所有权，在此基础上获得更多财富收入。

  • 销售独家会员 NFT：让粉丝可以与艺人进行特别互动，例如 VIP 访问节目、限量版商品和艺人面对面交流等。

  • 销售收入分红 NFT：分享未来销售收入的一定比例，让艺术家可以自由工作，而不必受到来自合作公司的干扰。

借助 NFT 和 Web3，艺术家可以将音乐以 NFT 的形式发布，无论是单张专辑还是整张专辑，可以让粉丝直接购买音乐，而不用通过唱片公司。

通过 Web3 身份验证和 NFT，艺术家可以通过小粉丝群实现可持续生活。当艺术创作者没有经济压力之后，它们能够以自己喜欢的方式去创作，更加能释放潜能。他们不再受限于平台或仅靠广告来维持生计。创作者可以为他们的粉丝提供免费的 NFT，使粉丝享有访问社区、Discord 等权益。通过这种方式，创作者和粉丝之间可以保持更加紧密的关系，艺术家获得更多获利的机会，粉丝也可以获得更佳的服务体验。

总结

Web3 目前仍处于早期阶段，在下一个阶段到来之前需要大量的开发和用户体验提升，同样需要一些重要的应用来支持整个生态系统的不断发展。例如：

  • L2 解决方案充分可操作，并为基础层提供支持足够多交易数量的能力，以支持网络和应用程序的大规模采用；

  • 以太坊成功从工作证明 PoW 转为权益证明 PoS；

  • 将 Web3 产品体验提升到与 Web2 产品体验的水平上来等。

我们无法确定下一个牛市何时到来，但我们正在走出熊市，并期待下一个牛市能够提供更加实用的应用和大规模采用的网络。
