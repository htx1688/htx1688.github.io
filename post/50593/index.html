Odaily 星球日报译者 | Moni

虽然因为“没有准备好”而推迟发布，但不出意外的话，科技巨头亚马逊公司旗下 NFT 平台“Amazon Digital Marketplace” 将会在本月 15 日正式亮相。

亚马逊公司从 NFT 入场对整个 Web3 行业意义重大，本文将从与之有关的 5 个要点入手，让你了解为什么这个新晋 NFT 市场值得更多关注，但需要提前说明的是，由于目前官方信息并不多，因此部分内容可能尚未得到 100% 的确认。

好了，让我们进入正题。

1、用户体验

亚马逊公司希望简化 NFT 的购买交易，这意味着用户将能够使用信用卡进行法币支付购买 NFT，就像在电商平台购买牛仔裤一样，用户也完全不需要任何关于区块链或自托管钱包的技术知识，只需要一个亚马逊平台账户即可。

2、虚实结合

毫无疑问，亚马逊公司不会将业务局限在虚拟世界中，而是会整合实体体验，三月初，有消息称亚马逊计划发行的 NFT 将与现实资产挂钩，这种 NFT 更像是一种数字所有权，例如用户可直接通过信用卡购买一条牛仔裤相关的 NFT，然后亚马逊将会把这条牛仔裤直接派送到你家门口。。亚马逊计划在其数字收藏品计划上线前或上线时通过邮件订阅通知每一位亚马逊 Prime 客户，而亚马逊目前在美国拥有大约 1.67 亿 Prime 会员。

3、游戏

亚马逊公司似乎已经在押注区块链游戏的未来，对于数字资产领域的许多行业领导者而言，游戏比任何其他垂直领域都具有最大的释放收入的潜力。亚马逊公司可以很好地利用 NFT 游戏热潮，该公司拥有 Twitch，这是一个在视频游戏爱好者中广受欢迎的流媒体平台。考虑到视频游戏市场的规模，购买和销售游戏时可以使用的数字资产价值每年或将达到数十亿美元。

4、私有链

亚马逊公司可能会推出自己的私有区块链，但目前尚不清楚这是否可以通过现有协议的分叉来实现，其后端区块链技术也尚不明确。据悉，自从启动 NFT 平台项目之后，亚马逊公司就开始接触 Layer 1   区块链、区块链游戏公司等多种数字资产项目，而且还与 AVAX 和 Chainlink 达成合作，未来也有可能会整合更多区块链。

5、额外信息

亚马逊公司推出的 NFT 市场名称为“Amazon Digital Marketplace”，其中并没有涉及“NFT”这三个字母，或许是考虑到公众对 NFT 的理解还不够深入，所以这个名称或许是个不错的选择。据悉，该平台最初只会在美国发布，但未来将扩展到全球范围。

亚马逊 vs. Coinbase、OpenSea、Blur

自从亚马逊公司推出 NFT 市场的消息传出后，有些人就联想到此前的 Coinbase NFT 市场，后者在发布后很快陷入低迷，让人怀疑有炒作之嫌。不过，这次亚马逊公司入场还是有很多不一样的地方。

Coinbase 的月活用户量大约只有 900 万左右，而亚马逊则高达 1.97 亿。不仅如此，Coinbase 平台对于非加密用户而言并不易于访问，相反亚马逊平台早已被大众熟悉。另一方面，亚马逊不仅拥有更多的用户，而且还拥有完全不同的受众群（Coinbase 的现有用户基本上都是已饱和的加密用户）。更重要的是，亚马逊公司提供了一种解决方案，使用户体验更好（大多数人不想要自我保管）。

如果说亚马逊 NFT 市场与 Coinbase NFT 市场相比有一定优势，但能否超越 OpenSea、Blur 等现有平台还有待观察。OpenSea 目前是第二大 NFT 市场，Blur 紧随其后，自 2017 年推出以来，这个“OG”NFT 市场的销售额已超过 300 亿美元，亚马逊的 NFT Marketplace 旨在简化 NFT 购买流程并为用户提供无缝体验。然而在功能、用户体验和销量方面能否与 OpenSea 等现有平台对抗，目前还不得而知。

重建信心

自从 FTX 倒闭引发市场暴跌之后，加密行业声誉大受影响，撇开用例不谈，像亚马逊这样的科技和电商巨头入场，势必能给处于低谷中的加密市场注入一剂强心针，也将会让更多消费者和用户重拾对加密行业的信心。

如今，越来越多来自不同行业的巨头开始转向 NFT 来促进他们的销售，从 Adidas、Puma 和 Nike 等运动服装公司到 Dolce & Gabbana 和 Gucci 等奢侈时尚公司都推出了 NFT 系列并在Web3领域里掀起波澜。

毋庸置疑，亚马逊入场 NFT，对于Web3世界而言，绝对是值得开心的好事。
