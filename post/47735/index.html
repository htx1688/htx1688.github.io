文/金色财经0xnaitive

2023年2月23日，Coinbase和Optimism之前推特上联合发布的小蓝标终于揭晓。

原来是Coinbase和Optimism Collective（管理Optimism的DAO）合作的推出基于OP Stack的Base测试网。

Base测试网推出的同时，Optimism也宣布了一个大动作，其目标将不仅仅只是条链了，而是链的平台——Optimism超级链。

金色财经用7个问题，读懂Coinbase Base和Optimism超级链。

1、什么是Base？

Base由Coinbase孵化，是部署在OP Stack上的、继Optimism主网外的第二个L2，并打算成为Optimism超级链的一部分。Coinbase Base成为OP Stack第二个核心开发团队。

2、什么是OP Stack？

OP Stack是支持Optimism的标准化、共享和开源的开发堆栈，目前由Optimism Collective维护。OP Stack是一组为Optimism提供支持的软件，目前以Optimism主网背后的软件形式出现，最终以Optimism超级链及其治理的形式出现。

OP Stack主要专注于创建一个共享的、高质量的、完全开源的系统来创建新的L2区块链。通过协调共享标准，Optimism Collective可以避免在孤岛中重复建设。

3、什么是Optimism超级链？

据Optimism Collective，多链世界有其自身挑战，尤其是以太坊L2的破碎化，为了让以太坊系统达到互联网级别的用户规模，需要建立一个统一的L2网络。因此，Optimism想成为链的平台——超级链。

Optimism超级链寻求将原本孤立的L2集成到一个可互操作和可组合的系统中。比如Coinbase Base将与Optimism Collective合作设计和构建一个协议，可以对多个L2进行排序，并通过OP Stack在它们之间提供互操作性和可组合性。

Optimism超级链，短期目标是将Optimism主网、Base和其他L2升级到一个初始的超级链结构，具有共享的桥和排序 。长期目标是可以发展成为一个庞大的网络，最大限度地提高互操作性、共享去中心化协议并标准化其核心原语 。

4、Coinbase为什么推出Base？

Base的推出在Coinbase长期规划之内。其目标是通过一个强大的、可互操作的web3生态系统扩展以太坊，这样任何人在任何地方都可以基于Base提供安全、低成本、对开发人员友好的方式来构建去中心化应用程序或链上“dapps”，从而将下一个十亿用户带入web3。

同时，Base也是Coinbase现有产品、用户和资产的链上之家。Base上的dapp可以无缝集成Coinbase 、轻松的法币入口以及对Coinbase生态系统的访问。

5、Coinbase的规划

正如上文，Base早在Coinbase长期规划之内。早在2016年，Coinbase的规划就很明确且一致：创建一个开放的金融体系，通过有意识地经历四个阶段来增加全球经济自由度。

第一阶段：制定协议（100 万人）

新协议（比特币、以太坊等）被发明出来，人们开始对它们进行修补。围绕每个协议开发的开源社区和早期采用者开始试验如何使用它们。

第二阶段：建立数字货币交易所（1000万人）

Coinbase 从这里开始。投资或投机是一种支付网络的引导机制。创建安全且易于使用的交易所成为人们将本地货币转入和转出数字货币的桥梁。

阶段 3：为数字货币应用构建大众市场界面（1 亿人）

为了超越交易和投资，消费者和企业需要一个简单的界面来开始使用数字货币及其应用程序。已经构建了无数应用程序来让任何人在任何地方访问新生的加密经济，包括 Coinbase、Coinbase Wallet、Metamask、Rainbow、Trust Wallet 等。

阶段 4：构建开放金融系统的“dapps”（10亿人）

现有的金融系统现在需要在开放网络上重新创建，并使其在全球范围内均可访问。这种情况正在发生：USDC带来了数十亿美元的链上收益，cbETH使持有者能够在任何地方保护金融系统并获得奖励，数百个dapp使无数个人和机构能够以更低的费用进行交易，更高的安全性和无与伦比的访问权限。

Coinbase与内部团队和外部合作者就如何加速该计划的第4阶段进行了无数次对话。要点是，为了释放最后阶段所需的规模、可用性和安全性，实际上需要回到起点：开发支持最终用户应用程序的底层协议。首先在以太坊上这样做，用 Optimism驱动EIP4844，使L2 rollups能够扩展。但这还不够，所以推出Base。

6、Base和Optimism Collective会有哪些合作

  • 加速超级链上的活动：  Base是Optimism主网之后部署在OP Stack之上的第二个L2，并打算成为Optimism超级链的一部分。Base将把 Coinbase的链上产品和用户群带入Optimism超级链生态系统，增加Optimism超级链的活跃度，同时使Coinbase用户能够与dapp无缝交互并在整个生态系统中移动资产。

  • 为超级链增值： 将向Optimism Collective提供一定比例的交易费用，使之成为Optimism超级链的一部分，为Optimism超级链和更广泛的加密经济的核心公共产品基础设施提供资金。

  • 推进开发者生态系统： 将继续与OP Labs和Optimism Collective密切合作，使开发者更容易构建新的L2和Rollup，并在Optimism超级链上部署他们的应用程序。例如，我们将与Optimism Collective合作，共同创建可用于加入Optimism超级链的L2的原语（例如用于ID），包括 Base和 Optimism主网。

  • 构建可互操作的加密经济：  Base和Optimism主网短期内都将作为独立的L2运行，为加密经济提供一套产品，例如swap、借贷等。从长远来看，预计会有用例在Optimism主网或Base上部署更有意义，让开发人员可以灵活地以最佳方式部署在适合其用例的L2上。

7、2023年路线图

2023年，Base和Optimism主网从Stage 0 rollup推进到Stage 1 rollup，具有显著的去中心化和去信任化水平。为实现这一目标：

  • 为OP Stack启动至少一个故障（fault）证明器 ，它将通过利用一组验证者来识别和挑战故障来保护Base和Optimism主网。

  • 将Base和Optimism主网智能合约的可升级性去中心化 ，首先交给一个安全委员会，该委员会具有不受任何单一实体控制的去中心化法定人数。

  • 启动超级链的初始版本 ，可以对Base、Optimism主网和任何OP Stack rollup的交易进行排序，保护任何人参与这些生态系统的权利

  • 努力在2024年** 将Base和Optimism主网推进到第2阶段rollup** ，其去中心化程度和安全性可与以太坊媲美。
