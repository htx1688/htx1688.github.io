撰写：Eric Hu

互联网已经从根本上改变了社会、个体和经济。作为开启知识转移的工具，它迎来了信息时代，成为实现应用、网络和服务的基本平台。

随着时间的推移不断发展，今天，我们正在从 “Web2 “转向 “Web3”，即互联网的下一次发展。 如果让 10 个人来定义 Web3，你会得到 10 个不同的答案，但互联网正在发生变化，这是多种技术融合达到成熟、全球化和社会转变的结果。

Web3 体现了诸多原则，有望成为一个更加去中心化和协作性的数据和价值互联网。区块链和加密货币是其核心，但它们也只会是支持这个新网络的众多技术的其中之一。

区块链和加密货币的爆炸性实验已经证明了新的结构和商业模式：强大的网络效应，为了一个共同的目标可以更加去信任化地协调利益相关者，调整激励机制，转变治理模式，定义商定的所有权，也许最重要的是——将数据与价值联系起来，所有这些都以数字原生的方式完成。

尽管区块链和加密货币有助于创造新的应用程序和业务一样，但重要的是要强调 Web3 是前端应用程序/用户体验以及后端技术的演变。Web3 突出了技术堆栈中更大的可组合性和更智能的网络。无论是通过智能合约还是机器学习驱动的决策，升级后的计算网络更适合自动化和管理高吞吐量分析或监测技术（NLP、计算机视觉、物联网）与数据密集型用例（AR/VR、自动驾驶、智能城市）。与 2000 年支持不断增长的互联网的带宽爆炸相类似，5G 的推出和硬件方面的边缘计算将形成网状，因为处理能力（无论是通过扩展的区块链驱动的计算和/或下一代云）为应用程序和服务提供了新的可能性。

另一点需要强调的是，文化和社会的转变是新技术成功不可或缺的因素（即时机）。过去几年，在 COVID 大流行的加速下，产生了许多趋势。我们已经看到了数字化转型、无现金支付和向云计算转变的重大进展。作为一个社会，我们越来越习惯于以数字方式进行交易和体验生活。加密货币已经进入主流文化，并逐渐被接受为一个真正的（尽管仍然是新生的）资产类别。在说了多年的 “机构何时会来 “之后，机构现在已经来了。

今天的 “Web3 “是笨拙的，充满了摩擦。 下一步是入门，更好的用户体验/用户界面，适当的监管，以及卓越或新颖的用例。但是，根据今天的情况对一项技术下赌注是愚蠢的；有大量的基础工作正在进行，而这一切都要从适当的构建开始。

在过去的两年里，Web3 已经在创新方面展现出寒武纪式的爆炸增长——从建立第一层或扩展协议的团队，新的应用程序和项目，以及新的治理或代币模型。而人才和创新走到哪里，资本就跟到哪里。2021年，250亿美元的资金已经流入基于区块链的项目。而在这些项目中，那些被归类为 “基础设施 “的项目是最关键的一些。

那么，为什么重要的是基础设施，而不是 App？

正如 Signal 的创始人 Moxie 所指出的那样——没有人愿意运行自己的服务器。随着 Web3 的成熟，就像网络的其他迭代一样，抽象化的复杂性是增长的一个关键驱动力。管理服务和工具使团队能够发挥他们的优势并进行创新，而不需要为后端（安全性、可靠性、可扩展性）投入大量资源。当我们把下一个 10 亿用户送上 Web3 时，大多数人不应该关心协议使用什么区块链或代币标准，或怎么样跳过障碍（和恐惧！）与应用程序互动。这就是为什么适当的基础设施、架构和工具是关键。在 Web3 中，这方面的前景是巨大的，所以放大来看，我们可以使用 Web3 基础架构的框架作为正在发生的事情的心智模型。

Web 和 Web3 原则的演变

在我们深入研究基础设施之前，多谈一点 Web3 的原则可能是有用的——毕竟，这就是所有项目的建设目的。

数字化： 今天，我们正处于从物理信息和资产到数字的过渡状态，Web3 存在于一个数据流动的时代。物理数据可以被测量，与数字信息结合并执行。后端计算将处理不同来源的信息——AR/VR、物联网传感器和自主设备等等。

新兴技术的融合和成熟： 技术不是无根之水，加密货币和区块链技术是架构和价值观的一个关键部分。但其他技术也在不断成熟——5G、AI、边缘计算、物联网。结合更高效、更智能地处理更大的数据集、灵活的基础设施和新的商业模式（例如代币）将为下一代应用提供基础。

  • 数据和资产的所有权： Web3 将有助于提供数据、资产和工作的真正所有权。区块链是革命性的，因为它可以追踪和证明数字所有权。数字资产——无论是法币、加密货币，还是 NFT，都可以通过钱包由用户控制。可追踪性允许对资产或个人的贡献进行测量和控制，与数字身份相结合，这是很强大的。

  • 增加控制和治理的去中心化： 加密货币和区块链从根本上说是基于去中心化、信任和协调的问题。一个代币代表的不仅仅是一种资产，还往往是一种投票权，或会员证。中心化有其自身的好处，但随着 DAO 的迭代，公司治理、网络控制和分布式决策的新模式正在形成。更加去中心化的控制将允许平台的建设者、运营商和用户能够拥有并对平台的发展有发言权。

  • 可组合性： 在互联网之前，是有围墙的局域网。越来越多的区块链似乎也是如此，互操作性是一个关键焦点。通过开源代码，Web3 的构建考虑到了可组合性。这意味着团队可以采取现有的项目或程序，并在其基础上构建。这允许更快的开发和跨应用的沟通（互操作性）。

更成熟的 Web3 将为改变网络上的数据、价值和所有权关系奠定基础。企业、用户和网络可以更自由地分享或互动数据或转移价值。因此，Web3 有很多事情要做，而今天我们正在观察一个处于发展的空间，这就是人们关心 Web3 基础设施的原因。

那么，基础设施的重要性是什么？

基础设施是发展的基石。 在过去的几年里，数字基础设施已经从手机塔或光纤网络等实物发展到 “上层 “技术，如云、数据中心和管理网络软件。

对于 Web3 来说，我们现在将基础设施定义为允许 Web3 应用程序被大规模部署、构建和使用的工具、服务和架构。毕竟，机构和散户都必须以一种更加无摩擦和可扩展的方式利用 Web3。Web3 基础设施有很多种类——特别是在去中心化的范围上。很大一部分 Web3 项目是天然去中心化的，也就是说，使用了代币或基于社区的治理。然而，去中心化只是一个范围——在资金、用户体验和运营效率方面存在权衡。在整个 Web3 基础设施中，许多人采取中心化的方法，通过传统的股权筹集资金。

Web3 基础设施有一些特点，对建设者和投资者具有吸引力。比如可预测的收入、与现有平台/软件的相似性、更有经验的管理团队以及对生态系统的任务关键性，这让它们得到了 Web3 原生/ Crypto 资本的资金，以及来自传统金融参与者的参与。因此，从 2021 年开始，这些 Web3 基础设施的赛道已经开始形成。

框架的指导性问题

当我们研究 Web3 的基础设施时，有几个指导性的问题支撑着这个框架，作为思考不断发展的前景的一种方式：

  1. 项目团队通常使用哪些 Web3 服务和工具（“架构”）以及哪些数据、存储或计算（“核心基础架构”）系统支持这些？

  2. Web3 中的建设者或机构存在哪些运营挑战？ 已经开发了哪些“关键任务”的解决方案？

  3. 对于某些用例（例如 DeFi、NFT）已经并且需要建立哪些基础服务才能大规模成功？

  4. 现有公司和项目在哪些能力上投入资源——无论是通过并购还是有机建设？

  5. Web3“基础设施”的组件如何相互交互？ 每个组成部分的建立程度如何（即该领域中的项目数量、通用标准、解决方案的成熟度）？

  6. 在 Web3 中，人们可以考虑在基础计算（区块链）层之上的三类服务和产品中的数字基础设施——数据、价值/流动性和参与者/区块链支持服务。

计算层（区块链和区块链启发的技术）：Web3 的基础是计算层。这包括区块链和受区块链启发的网络（第 1 层，子网/侧链和扩展解决方案），其他一切都建立在其上。 因此出现了跨链/全链协议以及链上消息传递项目，以允许价值和信息从一条链交换到另一条链。这一点很重要，区块链网络以一种更无信任的方式，在不同的利益相关者之间交织着数据和价值的转移。关于这个正在开发的计算/区块链层含有大量信息，但可以说这是花费了大量时间并将继续投资的地方。

在基础区块链层之上，我们可以将服务、工具、架构和基础设施分为三个部分：

  1. 数据参与者： 为团队建设和运营 Web3 项目提供核心基础设施、架构和工具 ；

  2. 价值和流动性参与者： 为资本在 Web3 应用程序和参与者之间的流动建立区块，减少摩擦；

  3. “互动者 “或区块链支持基础设施： 支持和维护区块链/计算网络的利益相关者。

数据基础设施：Web3 是语义学的一个演变。这意味着机器和计算机将需要分析、传递和计算大量的数据。

对于数据基础设施，有更多的 “核心 “或既定的数据基础设施和 “新兴 “的服务和工具，形成开发团队的架构。这里的数据基础设施包括各种工具、服务和组件，使区块链数据易于利用或建立在其之上。

简要介绍一下这些以数据为中心的构建块：

  • 核心数据基础设施： 管理与区块链互动的节点和 API 供应商，去中心化的数据存储，跟踪交易的区块探索者，以及有效访问数据的索引/查询服务；

  • 新兴/相邻数据基础设施： 为开发应用程序而互动的工具和管理服务，这包括协助代码运输的开发者环境、代码审计员、SDK（例如，隐私/零知识工具包）和数字身份服务；

  • 协助 Web3 使用案例的工具和管理服务： 与 NFT 相关的和 DAO /社区运营服务。例如，NFT 分析或 NFT 创建、组织财务管理、投票平台和社区入驻工具。

价值和流动性基础设施：Web3 的一个关键创新是将信息与最基本的价值联系在一起。

对于 Web3 来说，价值与运营网络的核心功能交织在一起，无论是通过治理、安全，还是实用代币激励，但没有流动性的价值是不值一提的。对于所有类型的资本流入 Web3，价值需要得到保障、开启/关闭、分配、转换、访问，并允许在整个生态系统中流动，而没有过多的摩擦。

因此，“价值或流动性 “是由一些参与者构建的。

今天的 “核心 “价值基础设施包括：

  • 钱包。 钱包是通往 Web3 的门户，是用户与区块链、代币和应用程序互动的主要界面。我们看到钱包最近发展得更像 “浏览器”，这是一个很好的步骤。

  • （机构或牛散）资本的托管人， 交易所在整个生态系统和代币中转换价值，以及现有支付基础设施的法币出入通道。

新兴的价值和流动性基础设施包括：

  • 用于投资组合管理的仪表板和分析； 用于增强流动性的 Defi 或交易所聚合器，以及用于高效市场的机构交易基础设施。

  • 协助资本运营方面的相邻服务 ——主要是税务、合规、监管和网络安全服务。

交互者和支持区块链的基础设施：我们已经强调，Web3 的核心是区块链技术所允许的价值和数据的结合。 因此，支持区块链计算层的基础设施，推动了价值和数据服务成功的可能。交互者和支持区块链的基础设施包括：

  • 网络维护： 加密货币矿工（用于 PoW 网络）、质押服务和验证器服务允许更轻松的网络支持或维护；这些有助于交易处理、投票（通过授权）和网络激励分享（例如区块奖励）。

  • 链上分析： 这些服务，如今在合规性或交易用例中很普遍，为计算层的活动提供仪表盘和跟踪。这对今天加密货币应用中的众多黑客或安全漏洞非常有帮助。

  • 新兴的基础设施： 正如今年夏天 EthCC 5 所强调的那样，链上通信和消息传输协议的关注度越来越大，我们可以在这里举例全链协议或跨链桥。

结论性想法

在过去的几年里，我们已经接受了一个全新的资产类别，一些最聪明的头脑、最大的公司和大量的资本都加入到创新中来。但大规模采用需要在监管、入门、新的应用（有实际收入）、商业模式、社会结构和技术基础设施方面进行改进。

我们正在实时体验一个新行业的成长，并作为一个社会学习在数字时代的互动。批评是肯定需要的，但随着建设者们解决每一个迭代的问题，从隐私到可扩展性，我们已经看到了新的治理模式、数字或物理混合业务以及激励性网络的力量的实验成果，现在是建设的最好时机。
