原文编译：Leo，BlockBeats

随着 Blur 的空投落地，NFTfi 叙事也在持续升温，有个很合逻辑的推测是今年或将迎来 NFTfi Summer ，DeFi 研究员 Minty 撰写了在 NFTfi 上的一些项目，BlockBeats 基于该文整合编译如下：

ParaSpace

ParaSpace 是近期比较火爆的去中心化借贷协议，用户可以通过该协议借入/借出 NFT 或 FT，ParaSpace 允许用户利用未被充分使用的资金来进一步的投资，并从中赚取收益。

ParaSpace 引入「全仓杠杆」、「打包抵押」等多方面机制上的创新，并借助 Ape 质押挖矿活动，仅用了两个月便从主网上线到 TVL 排名第二，在 BendDAO 的笼罩下逆势崛起，是一个非常具潜力的项目。更多解读点击参考

BendDAO

BendDAO 是最大的 NFT 流动性提供市场之一，用户可以通过抵押其蓝筹 NFT 借入 ETH，或者通过抵押 NFT 获得 5.1% 的 APR。此外，BendDAO 还提供 Ape 质押，最大限度的提高 Yuga Labs 系列藏品的收益，且在质押期间，用户仍可以出售其 NFT 并在平台使用质押借入的 ETH，并不会损失用户收益。

据 Dune 数据显示，BendDAO 近期在交易量和日活数据上排名还不错，若是 NFTfi 到来，BendDAO 绝对可以顺势上涨。

sudoswap

sudoswap 是用于 NFT 的 AMM 协议，类似于 Uni，sudoswap 支持 ERC 721 NFT、ETH 和其他 ERC 20 token。意味着用户可以从流动性池中进行买卖而不是直接交易：

-流动性提供者将 NFT 或 ETH（其他 ERC 20 token）存入池中，可以选择进行购买/出售 NFT（或者两者都有），并明确其初始价格和相关曲线参数；

-用户可以从这些池中购买/出售 NFT，每次一个项目被购买/出售，购买/出售另一个项目的价格根据它的结合曲线而改变；

-流动性提供者可以随时改变池参数或撤回资产。

sudoswap 更以用户为主的机制设计使其未来在 NFT 结合 DeFi 上更具优势。

hadeswap

hadeswap 是一个 NFT AMM 协议，提高了 NFT 流动性和交易体验，用户可以通过流动性池交易和交换 NFT。

其特点是：

-零平台费用，并由其治理 token HADES 决定此后是否更改；

-默认零版税，但基于用户意愿是可以支付相应费用的；

-渐进式购买/出售 NFT；

-赚取 SOL；

hadeswap 的零平台费用和基于用户的 token 治理是其未来转向更优机制的优势。

JPEG’d

JPEG’d 作为最初的 NFT 借贷协议之一，其背后的产品也是非常具有吸引力，随着 Blur 带动了 NFT 热潮，很多人预计 NFT 总量会增加，此举可能会推动 NFT 借贷的增长。作为顶级的 NFT 借贷协议之一，JPEG 定位于利用低借贷利率及其大量 CRV 来提高 pUSD 和 pETH 的流动性。

此外，还有一些令人心动的产品正在研发中，可以持续推动 TVL 增长，如果你想接触 NFTfi，JPEG’d 绝对不容错过。

LinoSwap

LinoSwap 可以说是 NFT AMM 市场上比较具创新性的协议，因为 NFT 的非同质化属性，导致某一类稀有度较高的 NFT 与稀有度较低的 NFT 价格没有什么区别，而传统的 NFT AMM 的对高稀有度的 NFT 并没有进行重视和价值判定。

为此，LinoSwap 引入了 Rarity Function（稀有度函数）和 Rarity-Value Curve（稀有-价值曲线），对高稀有度 NFT 进行估值和定价，解决了高稀有 NFT 在 AMM NFT 交易市场中的不被重视的问题。对于大多数真正热爱 NFT 的用户来收，LinoSwap 可以将珍稀的艺术收藏更大化地转为价值，未来或许会是 NFT AMM 市场具竞争力的协议。

X2Y2

X2Y2 仍是我迄今为止看到的最具价值的 NFT 市场之一，不仅质押者可以获得 100% 的协议费，团队还一直在努力为项目和 token 创造更多的交易量和利率。

x2y2 pro 等产品已接近完成，该团队正准备在今年改进其 tokenomic，随着定期的发展和价值主张，x2y2 在未来可能成为主要 NFT 市场的强有力竞争对手。

LooksRare

虽然 LooksRare 在最近的热度不高，但 Looks v2 是他们最大的更新之一，或许可以为该平台带来新的关注和用户。

与此同时，奖励排放时间表最近已降至最低，并将在全年保持不变。如果 Blur 的暴涨给 NFT 市场带来 FOMO，LooksRare 绝对会引发一波关注。

nftperp.xyz

nftperp.xyz 建立在 Arbitrum 上的 NFT 永续交易平台，可以追踪 NFT 地板价，并在此基础上进行做多、做空。nftperp.xyz 亮点是推出了 vAMM 交易机制，也是未来很值得关注的项目，点击参考

Pine Protocol

除了上述一些 NFT 市场的领头项目外，Pine Protocol 也是一个值得关注的项目，它是一种针对 NFT 的 P2P 借贷协议，可以提供一些有意思的功能，例如 PNPL，如果用户需要即时流动性，可以在其中抵押 NFT，然后再偿还。

Pine 还添加了一些有趣的改进，例如如果您的账户中持有足够的 Pine token，则自动展期贷款。该路线图还指出，接下来将推出多链支持和买卖贷款信贷市场。

Pine 的总成交量和借贷量都在上升，但这个协议仍然是新的且存在风险。如果你想要低风险敞口，这可能是值得关注的协议。

Botto

Botto 是一个 AI 生成艺术平台，通常被视为 NFT 和 AI 的结合产物，它可以在以往艺术史上的优秀作品中汲取灵感，通过 AI 生成新的艺术作品，然后由社区投票。

艺术品拍卖收入由 Botto token 持有者获得，Botto 艺术品每周在 SuperRare 上以相当可观的价格售出，加之 AI 近期火爆，我很期待 Botto 的未来会如何。

FloorDAO

虽然 FloorDAO 并不是非常出名的协议，甚至可能会失败，但还是做了一些深入研究。首先，FLOOR 的市值目前低于其国库价值，国库价值主要由 NFTX 产生的 swap 费的 NFT LP token 组成。

虽然该价值可能会产生波动，但如果项目失败，国库被分配给 FLOOR 持有者，这可能会进去一个安全缓冲阶段。另一个有趣的进程是目前正在讨论和重申的 Floor v2。

提出的一个潜在想法是在 RFV 大于 Mcap 时允许通过 ETH 进行国库赎回，但这尚未确定，我需要做更多的研究，但 Floor 上 NFT 藏品的收益指数方面可能值得关注。
