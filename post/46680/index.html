科技巨头微软周一宣布与ChatGPT 背后公司 OpenAI 继续深化长期合作伙伴关系，通过一项多年、数十亿美元的投资来加速在人工智能(AI) 领域的突破。微软表示，最新的合作伙伴关系建立在其 2019 年和 2021 年对 OpenAI 的投资之上，两家公司没有透露合作的具体财务条款。

OpenAI 打造的聊天机器人 ChatGPT 自发布后就火遍全网，面市仅一周就吸引了超 100 万用户，成为 AI 领域的“顶流”。

微软也想趁热打铁抓住这波人工智能的热潮，《华尔街日报》援引知情人士的话报道称，微软一直在讨论向 OpenAI 投资高达 100 亿美元。此外，OpenAI 本月正在谈判以要约收购的方式出售现有股份，该公司估值约为 290 亿美元，尽管盈利很少，但它仍是纸面上最有价值的美国初创公司之一。

这项新的投资表明，微软正在投入大量资源，将人工智能软件整合到其全部产品套件中，例如设计应用程序 Microsoft Design 、搜索应用程序 Bing，它还将资助 OpenAI 在微软的 Azure 云平台上运行其各种产品所需的计算能力。

微软在公告中描述了投资的战略重点：

  • 大规模超级计算——微软将加大对专业超级计算系统开发和部署的投资，以加速 OpenAI 开创性的独立人工智能研究。继续构建 Azure 领先的 AI 基础设施，以帮助客户在全球范围内构建和部署他们的 AI 应用程序。

  • 新的人工智能体验——微软将在消费者和企业产品中部署 OpenAI 的模型，并引入基于 OpenAI 技术的新型数字体验。这包括微软的Azure OpenAI 服务，它使开发人员能够通过直接访问由 Azure 可信的企业级功能和 AI 优化的基础设施和工具支持的 OpenAI 模型来构建尖端的 AI 应用程序。

  • 独家云提供商——作为 OpenAI 的独家云提供商，Azure 将为研究、产品和 API 服务中的所有 OpenAI 工作负载提供支持。

自 2016 年以来，微软一直致力于将 Azure 打造成面向全世界的 AI 超级计算机，微软董事长兼首席执行官 Satya Nadella 表示：“我们与 OpenAI 建立了合作伙伴关系，我们的共同目标是负责任地推进尖端人工智能研究，并将人工智能民主化为一个新技术平台。在我们合作伙伴关系的下一阶段，各行各业的开发人员和组织将能够使用 Azure 访问最好的 AI 基础设施、模型和工具链，以构建和运行他们的应用程序。”

OpenAI 由技术投资者 Sam Altman 领导，2015 年开始是一家非营利组织，特斯拉公司首席执行官埃隆·马斯克、LinkedIn 联合创始人Reid Hoffman 和其他支持者认捐了 10 亿美元。长期以来，它的目标一直是开发人工智能技术：通用人工智能，机器能够学习和理解人类可以学习和理解的任何东西。

OpenAI 今年发布了一套新产品，业内人士称这是朝着该目标迈出的重要一步，并可能为一系列新的人工智能驱动的消费者应用程序铺平道路。

去年秋天，它推出了 Dall-E 2，一个允许用户从文本字符串中生成艺术的项目，然后在 11 月 30 日公开了 ChatGPT，ChatGPT 因其精准的交付能力而在技术社区中引起了轰动，输入“谁是乔治·华盛顿·卡佛？”、 “写一个玉米饼与热狗在海滩上搏斗的电影剧本”等问题，ChatGPT都能给你惊喜。

这项技术也引起了谷歌高管的注意，他们在最近的一次全体会议上表示，虽然谷歌拥有类似的 AI 能力，但如果它在 AI 聊天技术上动作太快，其声誉可能会受到影响。

2019 年，OpenAI 创建了一个营利性部门，并从微软筹集了 10 亿美元，部分用于资助训练其人工智能算法所需的密集计算能力。 根据合同，OpenAI 同意使用 Azure 作为其独家云合作伙伴，并表示在将新产品推向市场时将优先考虑微软。

这一决定招致了人工智能界一些人的批评，他们表示，这代表着 OpenAI 背离了作为研究实验室的根源，即寻求造福于人类而不是股东。 OpenAI 表示将限制公司的利润，将剩余利润转移给非营利组织。

微软在云技术等领域面临着激烈的竞争，ChatGPT 火出圈被很多人认为是微软云业务的“胜利”，将 OpenAI 的人工智能功能引入这些平台可以让微软在与亚马逊和 Alphabet 等竞争对手的竞争中占据优势。

ChatGPT 还可能会改变搜索领域的游戏规则，通过为用户提供易于理解和对话式的搜索查询答案，使该公司的Bing 比 Alphabet 的谷歌搜索更具优势。

日前在瑞士举办的2023世界经济论坛期间，微软首席执行官 Satya Nadella 讨论了公司扩大对 OpenAI 工具的访问以及不断增长的 ChatGPT 功能。Satya Nadella 表示，微软计划将人工智能工具整合到其所有产品中，并将它们作为其他业务的平台提供，以及致力于迅速将OpenAI 的产品商业化，ChatGPT 付费版服务已经开放了等候名单，AI 图像生成器 Dall-E 2 也在持续改进中。

微软定于下周发布收益报告，可能会放出有关其未来计划的更多细节，包括对其硬件和云战略的调整。
