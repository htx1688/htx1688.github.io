原文：《潜在空投机会？一览lens协议上值得关注的新项目》

作者：Jaleel

距 BlockBeats 发布《Web3.0 创始人经济报告：CreatorFi 的发展现状和想象空间》已有数月，随着最近 Space ID 和 Arbitrum 的空投热潮，Les 协议的社区也愈加热络起来。不少人认为在 ID 和 arb 空投结束之后，下一个大空投就将是来自 AAVE 的 Les。lens 正在 Web3 上构建一个去中心化的完整社交网络图谱，有传言说将在 2023 年第二季度进行 Airdrop。除了社区比较熟知的 Lenster 、Lenstube、Phaver、Orb 以外，BlockBeats 还整理了一些在 Lens 协议上的新项目。

社交 App&创作者经济

LensTags

LensTags 是一个标签平台，旨在成为寻找由生态系统有机策划的详细、准确和高质量信息的首选。通过标签，用户将能够发现新资源并将它们组织到个人阅读列表中。LensTags 可帮助用户保存、创建、组织和分享在互联网上找到的所有精彩内容。在 Lenstags 上用户可以分享找到的所有最好的文章、视频和网站。

快速熟悉 LensTags 以及简化生活、从 Youtube 和 Twitter 等网站迁移现有书签并设置自动保存、通过使用 LensTags 的应用程序、扩展程序、公共页面或您网站上的嵌入式书签，且可以从任何地方访问。使用 LensTags，您可以提高个人组织能力并减少探索主题的时间。

Diverse

Diverse 是一个用于构建、管理和发展社区的 web3 社交平台，专注于微型创作者经济并帮助创作者找到他们的受众。Diverse 旨在构建下一个 Reddit，用户和创作者可以轻松地增加他们的受众和粉丝。使用 Diverse 时，将基于 lens 协议的去中心化社会结构，用户的社交关系将保留在区块链上。可以出售任何东西，包括电子书、博客、矢量艺术、图形、3D 内容、电影和图像。

与 OnlyFans 类似，Diverse 将提供与社区互动的工具，例如投票、锦标赛、活动和会员专区。同时还可以将社区设置为可发现的或不可检测的，以及私有的或公共的，以及使用 NFT 创建的专属社区。

此外，Diverse 的 Head of Growth @usediversehq 称正在建立一种方法在 Lens 上做视觉化社交图谱，将在 Lens 上使用 BigQuery 数据库，实现交互圈和可视化的社交图谱。

Soclly

SOCLLY 以创作者为中心，通过为创作者提供各种工具、空间和机会展现技能并赚取收益，同时用户还可以通过 SOCLLY 的链上声誉建立可信赖的数字声誉。

SOCLLY 强调将确保创作者的辛勤工作得到认可和公平的回报，通过链上评论告别虚假评论的混乱局面，创作者将得到诚实的反馈和真实的名声，通过触手可及的去中心化日历和链上可用性在 SOCLLY 上分享技能，同时还能与志趣相投的人建立联系并围绕创作者建立热情的社区。

Dumpling TV

Dumpling TV 将 Lens 的精选内容用一种新的方式呈现：怪异酷炫，包裹着怀旧的氛围的电视机风格。用户可以在 Dumpling 上以新颖的视觉效果浏览频道和帖子、镜像、喜欢、收集和享受氛围，同时 Dumpling TV 现在支持用键盘的上下左右键换频道，用自己独特的风格和方式创建了一个很棒的应用程序来管理视频内容。

NFT

Lens Garden NFT

本文档概述了该系列将于 2023 年 3 月 23 日推出，价格为 20 MATIC，用户将能够参与独特且身临其境的 NFT 体验，将他们在 Lens Protocol 生态系统上的活动与其 NFT 的价值联系起来。

Lens Garden NFT 系列是整个生态系统中的第一个动态 NFT 集合，即将在 Polygon 区块链上推出的 Lens Garden NFT 系列。Lens Garden NFT 系列是一种数字资产，旨在奖励在 Lens Protocol 生态系统中表现出高水平活动的用户。用户可以利用它来增强他们在 Lens Protocol 生态系统中的体验。铸造 NFT 后，用户可能会注意到它从高于级别 1 的级别开始。这是因为 NFT 与用户的 Lens 统计数据相关联，例如关注者和参与度。此外，Lens Garden NFT 系列不能在公开市场上交易。

Lens Garden NFT 系列的独特之处在于它试图激励用户为 Lens Protocol 生态系统的增长和可持续性做出贡献，而不是仅仅专注于 NFT 的创建和交易。该系列旨在鼓励和奖励 Lens Protocol 生态系统的积极参与，并将有助于在平台上建立一个更强大、更有活力的创作者和用户社区。

工具

目前，Lens 已经有了索引器，可以对数据进行快照并将其转换为关系型、超级易懂的数据库。这将加快查询速度以及如何使用 API 为所有 LENS 应用程序提供支持，数据保存到集中式 postgres 数据库中，并允许使用 graphQL API 查询它。

Lens 已将整个 Lens 社交图发布到公共 BigQuery 数据集。这意味着用户可以根据需要以任何方式批量查询所有数据。公共 BigQuery 数据集每月为用户提供 1TB 的配额。任何人都应该可以访问此信息，而无需编写复杂的索引器任务并设置所有基础设施。这是允许任何人在 LENS 之上构建自定义功能而无需 API 的第一步。而对于数据开发人员，使用公共 Lens Big Query 数据集可以在开放的 web3 社交图上构建算法。因此基于 BigQuery 数据集，lens 上新增了一些工具。

Lentics

Lentics 是服务 Lens 创作者的分析工具，可分析创作者的粉丝数据、互动、收入等。Lentics 完全集成 Lens，提供了比较完整的数据，帮助创作者分析自己的账号增长。

Lens Share

由 @apoorvlathey 发布的 Lens Share 浏览器插件，可以直接搬运 tweet 到 Lens，Lens Share 添加了一个按钮，当用户打开推文时，推文中的文本和图像会在后台自动上传到 IPFS，用户只需在钱包中签署消息并批准 txn 即可将其发布到 Lens Protocol，另外还可以在发布前设置自定义收藏设置，实现将已经发布的推文并将其交叉发布到镜头。

其他工具

另外，edgeandnode 的数据科学家 rickydata 也基于 Big Query 做了一个预测未来数据的工具，可以根据用户的 Lens handle 过去数据预测粉丝增长量等各个数据的未来情况，目前仅有测试版。

其他

DeSchool&Booth

DeSchool 是一个 Web3 原生教育工具。通过整合社交协议、推荐协议、安全协议和关于 X2E 的智能合约、灵魂绑定的证书、自动分配收入，重塑元空间教育系统。DeSchool 各方面都很均衡且出色，且目前正在加速集成更多的 Lens 功能。

DeSchool 基于 Lens Protocol 的链上简历工具 Booth 目前已经上线，支持一键链接 DeSchool 账号 + Lens 协议，数据互通；找到与共学同一课程的好友和同学，一键关注；基于结课凭证 SBT&Token、学习历程生成链上简历 + 个人能力图谱；根据兴趣，一键匹配好友组队共学。同时，Booth 将会根据当前地址和在 Booth 平台上所有已连接地址的可信链上教育和工作证明生成雷达图。

linkDOT

linkDOT 将自己定位于 web3 人才的终极在线投资组合工具，现已上线 LensProtocol。通过 linkDOT，用户可以在一个地方记录所有经过验证的证书和工作证明，并让社区通过链上同行证明为用户的技能担保。现在通过集成 Lens 协议，用户可以通过创建尖端的产品组合来展示自己的工作证明和同行验证，从而将用户的 Web3 职业生涯提升到一个新的水平。

linkDOT 认为对于希望以真正突出的方式展示自己技能的用户来说，这种集成将改变游戏规则。用户可以使用 Lens 铸造自己的工作证明人才档案，并展示技能和成就，建立信誉并与潜在的合作者和组织建立联以获得下一份机会。linkDOT 目前正在进行测试版，之后将在将在 Lens Mainnet 上线，目前用户可以填写白名单表单申请进行测试。
