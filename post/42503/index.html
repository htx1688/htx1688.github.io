2021年以来，元宇宙、NFT、数字藏品等新概念、新技术的不断破圈，让 Web3 得到了前所未有的关注，即使是相对保守的国内市场，也有不少互联网大厂和传统品牌相继入局。

对于参与的各大互联网大厂来说，无论是全力押注也好，还是小试牛刀也罢，总体上反映出的是各家都不愿错失下一代互联网浪潮的普遍心态，比如即使是强如谷歌这样的超级互联网巨头，也在今年5月正式成立了其第一个 Web3 部门。

本期内容我们将盘点梳理全球范围内进行 Web3 布局的互联网大厂，根据它们各自的参与方式和产品特点，我们或许能够拼凑出当前 Web3 的发展方向和未来形态。

国内布局：

阿里巴巴

2022年3月，阿里巴巴旗下《南华早报》成立了 NFT 公司“Artifact Labs”，该项目将基于Flow区块链铸造，用户可以对某一历史事件的特定 NFT 进行竞拍，或者购买一个既定事件的头版。

2022年6月，阿里巴巴宣布为海外用户提供 NFT 解决方案。

腾讯

2022年3月，腾讯首次参投澳大利亚 NFT 游戏公司 Immutable 高达2亿美元的融资，该公司旗下拥有知名链游《Gods Unchained》和《Guild of Guardians》，并同时兼具交易平台 Immutable X 。

字节跳动

2021年年10月，TikTok 宣布推出首个 NFT 系列 TikTok Top Moments，精选了来自平台上一些最受欢迎的创作者六个具有文化意义的 TikTok 视频。其中，一对一的NFT将在以太坊上提供，限量版 NFT 将由 Immutable X 提供支持。

百度

2021年4月至10月，百度相继上线了图腾（区块链原创图片服务平台）、百度百科上链和百度超级链。图腾采用自研区块链版权登记网络，配合可信时间戳、链戳双重认证，为每张原创图片生成版权DNA，可真正实现原创作品可溯源、可转载、可监控，保障原作者的权益。

2022年7月消息，百度日本以Simeji输入法为主题，正在准备发行其以太坊上的NFT，目前进行社区建设中。Simeji日语输入法在日本拥有千万级日活用户规模，项目将会采取 Freemint 的方式发售。

哔哩哔哩

2022年4月，B站在以太坊上为海外用户发布 Cheers UP NFT 系列，包含10000个头像类NFT，该NFT是一系列具有不同特征的自动生成的角色，即在面部、穿着、肤色、头饰等方面存在差异。

映客

2022年6月，映客互娱集团正式更名为映宇宙（英文名称Inkeverse），业务全面向元宇宙进军，并已经在以太坊上面向海外用户发布了第一代 Pass NFT。

陌陌

2022年7月，陌陌、探探的母公司挚文集团授权 IP 予 WanderingLabs 探索 Web3 项目，在海外发行 NFT 产品 Wandering Planet。

海外布局：

谷歌

2022年3月，Google Venture 参与了无聊猿母公司 Yuga labs 估值40亿美金的种子轮融资。

2022年5月，谷歌正式成立了其第一个 Web3 部门，目的是为运行区块链应用程序的开发人员提供后端服务。

Meta

2022年6月21日，Meta 创始人扎克伯格宣布扩大 NFT 的测试范围，让创作者不仅可以在 Instagram 上展示他们的NFT，还允许用户就在 Instagram 和 Facebook 上交叉发帖

推特

2021年7月，推特发布 NFT 产品，让推特的用户头像可以显示为 NFT，并被认证。

2021年9月，推特宣布开头加密货币打赏通道，允许用户使用加密货币，并将推出NFT身份验证。

eBay

2021年，eBay 宣布允许在平台上买卖 NFT。

2022年6月，eBay 完成对 NFT 交易平台 KnowsOrigin 的收购。

亚马逊

2022年4月，亚马逊新任首席执行官 Andy Jassy 在采访中称，对加密货币和 NFT 的未来持乐观态度。并表示亚马逊近期可能不会将加密货币作为零售业务的支付方式，但很有可能在未来销售 NFT。

另外，在2022年之前，大部分 Crypto 和 Web3 公司都选择使用亚马逊的云服务，目前已经占据了50%以上的市场，三大加密交易所等加密巨头都用亚马逊的云服务。

奈飞

2022年5月，奈飞在新一季《爱、死亡和机器人》等影视作品中中加入NFT的使用乃至彩蛋。
