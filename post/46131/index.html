原文标题：《 Bankless：关于 2023 年的 8 个 Web3 游戏主题愿望清单 》

原文作者：William M. Peaster

原文编译：DeFi 之道

毫无疑问，当我们展望未来时，包括我在内的许多加密领袖都认为 NFT 游戏生态系统将在 2023 年大放异彩。

但是，为实现这重要的一年，我们应该希望在这个生态系统中看到什么呢？

为此，我整理了一个愿望清单，也就是我（也许很快你也会）希望在 2023 年看到的更多 NFT 游戏领域的 8 个新兴主题。以下。

1) 更深入、更丰富的玩法

人们如此热爱游戏的原因之一是什么？它们允许我们测试和磨练我们的技能。

这一点在具有深度的游戏中表现得最为明显，即它们提供了丰富的战术和战略游戏的可能性。从国际象棋到《炉石传说》都是这方面的例子。

NFT 游戏生态系统的发展速度比以往任何时候都要快，但迄今为止，许多早期游戏都让人感觉「平淡无奇」，因为没有太多可做的事情，给玩家提供技能测试的机会也很少。目前，有一个项目正在朝着正确的方向努力，那就是 Axie Infinity。

2022 年 4 月，Axie 的创造者 Sky Mavis 发布了 Axie Infinity: Origins，这是其原创战术卡牌对战游戏的优化版本，引入了卡牌更新、符咒和符文制作系统，以及实时战斗。正如你在上面的推文中所看到的，这些新元素让玩家能够通过独特的定制游戏风格享受到许多深度。

当然，Axie Infinity: Origins 本身仍在进行中，但我希望在 2023 年看到更多项目跟随它「深入」的步伐。如果你有兴趣了解这款游戏，可以在 Sky Mavis 这篇指南中查看基础知识。

2) 新的移动版本

一部像样的智能手机比电脑或游戏机更便宜，所以手机游戏的可及性很高。

我一直很喜欢 Axie Infinity，因为你可以在安卓设备上玩它（即将支持 iOS）。毫无疑问，随着移动游戏活动不断上升到新的高度，我们将看到更多的 NFT 游戏模仿这种方式前进。

我现在正在关注的一个项目是 Treeverse。

Treeverse 是一款基于以太坊的移动 MMORPG，由 Endless Clouds 和 Loopify 开发，目前处于有限的 alpha 前测试阶段。从早期的测试视频来看，我认为这款游戏不仅看起来很不错，而且非常有趣。

为更多的人带来乐趣，让更多的人进入加密应用程序，是移动 NFT 游戏的方向。我真的很期待 Treeverse 和其他类似的手机游戏会如何发展。

3) 无 Gas 游戏

具有大量链上元素的游戏可能带有大量交易和钱包签名。

大量的交易和钱包签名 = 糟糕的用户体验。

因此，一款新的 NFT 游戏展示了如何提供无 Gas 的用户体验，它就是海盗国度（Pirate Nation），一款建立在 Polygon 上的链上冒险游戏。该项目使用了一个游戏钱包系统，允许 Pirate Nation 在幕后覆盖交易，所以玩家不必在游戏中支付 gas 或签署任何交易。

希望其他 web3 游戏也开始尝试这种卓越的用户体验方法！

4）Arbitrum 游戏的入侵

一段时间以来，Arbitrum 一直是最大且最受欢迎的 L2 以太坊扩展解决方案。

当然，廉价且快速的基础设施是 NFT 游戏的理想选择。这一现实，再加上 Arbitrum 即将取得的所有进步，使 L2 成为 web3 游戏热潮的沃土。

此外，像 Treasure DAO 这样的项目已经为 Arbitrum 带来了大量 NFT 场景的基础，所以好戏可能才刚刚开始。对于像 The Beacon 这样的项目来说也是如此，这是一款免费的 RPG 游戏，刚刚获得了爆炸性的发布，并将为 Arbitrrum 带来巨大的活跃度。我认为，这些都只是 L2 游戏的开始。

5）更多的比赛

竞争的快感，技巧性胜利的荣耀，成就和奖励的收获，以及对一个充满激情的同行社区的那种归属感。这些都是让玩家觉得比赛如此令人兴奋的原因。

因此，当涉及到 NFT 游戏时，比赛绝对是为玩家提供有趣和引人注目的滚动体验的好方法，我希望看到更多这样的比赛。

目前正在专注比赛这种方法的项目是 Sorare：NBA，它允许玩家建立幻想卡组，然后在他们愿意的时候注册各种不同的比赛。

6) 灵活的加密原生设计

有时，当 NFT 游戏推送更新时，游戏创作者将暂时暂停他们的合约，玩家将不得不解除质押或迁移他们的游戏资产。这不是很好的用户体验。

目前，有一个游戏正在解决这个问题，即 Shattered EON，它将于 2023 年 4 月推出。它通过灵活和模块化的 Diamond Standard 智能合约方法来实现这一点，允许 NFT 游戏（以及其他用例）随时扩展和更新他们的项目。我完全赞成在这个领域看到更多这种实用的方法。

7）更多的链上游戏引擎实验

我所说的「链上游戏引擎」是指在以太坊这样的区块链上创建的一系列智能合约，以无限期地驱动游戏的后端。

链上游戏引擎很有趣，因为它们允许第三方创作者围绕 NFT 游戏并在其上公开构建新的体验。这种动态为修改铺平了道路，反过来又允许玩家主动选择如何享受和体验他们选择的游戏。

最近，在这方面有很多有趣的成果。例如，近几个月来，我们看到 Curio、Lattice 和 Adam Cochran 推出了新的链上游戏引擎。这些都是一个新领域的萌芽，这里的可能性目前很大。

8) 无 Token 游戏

迄今为止，几乎所有的 NFT 游戏都以某种形式或方式接受了 ERC20 Token 。

到目前为止，我们看到的最流行的方法都是围绕着单 Token 或双 Token 模式。后者将治理和实用性问题分割到不同的 ERC20 中，而前者则试图只用一个 Token 来处理所有的需求。

这两种方法都可能面临重大问题，而 0xKepler，我认为是 web3 游戏领域的伟大人物之一，在这个 Thread 中出色地阐述了这些问题：

作为避免这些问题的方法，0xKepler 建议创建一个无 Token 模型，该模型将依赖于「链下货币 + NFT 的价值增值」。

这个想法是，玩家可以「用链外货币铸造他们的 NFT」，这将减轻直接的法币交换，从而减少在不受约束的情况下破坏 NFT 游戏的金融动机。我认为这是一个不错的方法，可以引致更有趣且更平衡的游戏。如果我们开始看到更多的项目将这种策略付诸行动，我会很兴奋，因为我们肯定需要在这方面进行更多的实验。

有趣的未来在等着我们

如果你对 NFT 感兴趣，而且你也是一个游戏玩家，那么在 NFT 生态系统中，这会是一个令人难以置信的时刻。

是的，NFT 游戏场景仍然处于相当原始的状态，但仔细观察后，人们会感觉到它正在转弯，正在取得重要的进展。曾经只有几个简单的 NFT 游戏，现在有许多处于不同开发阶段的游戏，它们的复杂程度各不相同。

事实上，各种可能性和技术正在不断涌现。游戏玩法越来越丰富，用户体验越来越好，我们的集体基础设施和集体知识在各个方向上都在改进。我们还远远没有解决这个领域的所有问题，这是肯定的，但作为一个狂热的玩家和 NFTer，我的希望和预测是，我们将在 2023 年看到 NFT 游戏项目继续取得重大进展！
