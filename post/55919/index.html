编译：Peng SUN，Foresight News

元宇宙的声音，沉寂了太久，最近刚融资 5400 万美元的 Futureverse 引起了关注，这是一家结合 AI 与元宇宙的初创公司，新一轮融资由 10T Holdings 领投，Ripple Labs 等参投。

Futureverse 出身极为有趣，它由 11 家不同的公司合并而成，于 2022 年底开始由 8 家公司合并，之后又有 3 家公司加入。这 11 家公司分别为：

  • 游戏工作室 Altered Phoenix

  • 人工智能提供商 Altered State Machine

  • 与 MasterCard 网络相连的多链支付产品 Immersve

  • 设计工作室 Non-Fungible Labs

  • 动画工作室 Shadows Interactive

  • 数字支付系统 Centrapay

  • 去中心化数字身份协议与资产托管协议开发商 Centrapass

  • 去中心化通信系统开发商 Sylo

  • Web3 汽车社区 ATEM Car Club

  • 风险工作室 Centrality

  • 数字街头服饰社区 DRx

值得注意的是，这些参与合并的公司大多相互关联。譬如，Centrapay、Centrapass 与 Sylo 均由 Centrality 运营。而 Futureverse 联合创始人为 Aaron Mcdonald 则是 Altered State、Centralpay、Centrapass、Immersve 和 Non-Fungible Labs 的联合创始人。

通过 11 家公司的合并，Futureverse 将不同领域的基础设施整合到一个协作生态系统中，为 Futureverse 搭建了任何元宇宙应用所需的基本组件。今天，笔者梳理了 Futureverse 白皮书，与大家一起看看 Futureverse 是如何构建开放元宇宙的。

一、Futureverse 基本构成

Futureverse 在基础设施、协议、内容和应用层提供了一套工具，这些工具在构建时充分考虑了用户体验，并进行了全面集成，以确保互操作性。

  • 去中心化区块链网络 The Root Network，优化了用户体验和进入 Web3 的门槛；

  • 去中心化身份协议与资产护照（Decentralized Identity Protocol and Asset Passport），以形成互操作性和应用的基础数据层；

  • 去中心化委托协议（Decentralized Delegation Protocol），允许应用程序分层访问资产、用户和内容；

  • 去中心化通信、通知和存储协议（Decentralized Communications, Notifications & Storage），允许用户和应用程序之间相互通信；

  • 去中心化人工智能协议（Decentralized Artificial Intelligence Protocol），允许应用程序和 Avatars 以社区管理和用户自主的方式利用这项技术；

  • 旨在帮助用户进入 Web3 和连接实物商品的支付网络和钱包，充当元宇宙的金融通道。

  • 内容资产（Content assets）：丰富的用户自有数字资产，以讲故事为核心。

二、The Root Network

The Root Network 是 Futureverse 的核心区块链基础设施，由以下组件与 Runtime 构建而成，重点关注用户的 Onboarding 体验：

（一）Substrate Core

The Root Network 以 Substrate 框架为核心，其优势在于具有庞大而成熟的开发者社区，具备可以优化用户体验的定制链 runtime 的技术能力；

（二）支持智能合约的 EVM

EVM 在 The Root Network 协议中运行，能够读取并执行 Solidity 智能合约。The Root Network 开发者可以访问智能合约功能，以前为以太坊编写的任何代码现在都可以在 The Root Network 上运行。

（三）Custom Runtimes

针对非同质化资产、GAS 经济、同质化资产、DEX、预言机等 Custom Runtimes 将为创作者提供构建应用程序和内容的要素，而无需成为智能合约开发者。

1.非同质化资产（NFT） Runtime

The Root Network NFT runtime 为 NFT 提供了统一的全网络体验。

在其他网络中，NFT 存在于单个合约层。在 The Root Network 中，NFT 存在于核心网络 runtime 中。这为建立互操作性提供了一个通用标准，包括统一的铸造、版税与结构（元数据与内容格式）。NFT runtime 具有以下特征：

  • 全网版税执行

  • 为创作者提供本地多钱包分账和分级版税服务

  • 原生 NFT 与 NFT 交换

  • 本地静态和冷铸造选项

  • 构建 NFT DApp，无需开发或部署智能合约

NFT runtime 的费用将分配给网络质押者与验证者。The Root Network 还将集成 XRPL XLS-20 NFT 标准，与 XRPL NFT 生态系统和基础设施兼容。

2.多代币经济

The Root Network 提出多代币经济以优化 Gas 费并降低初始用户 Onboarding Web3 的门槛。

The Root Network 主要的网络代币是 ROOT，这是 PoS 共识中的网络安全与治理代币，不是 Gas 代币。Futureverse 与 Ripple 和 XRPL 合作，以 XRP 的形式提供基础区块奖励。XRP 也将用来支付网络的 Gas 费。

The Root Network 提供链内 Gas 费交换服务，无需用户在使用 DApp 前理解 Gas 概念并购买 Gas，亦无需担心 Gas 费的不稳定问题。用户可以加入 The Root Network 并使用应用程序的原生代币，开发者也可以委托并代表用户支付费用。开发者在其原生代币和 Gas 代币之间建立一个流动性池。节点验证者可以收到这些 Gas 代币，作为产生区块的奖励。譬如，从以太坊跨链来的用户可以使用 ETH，通过 ASM 接入的用户可以使用 ASTO，下载了 SYLO 钱包的用户可以使用 SYLO。默认情况下，如果原生代币没有流动性池，用户可以使用网络 Gas 代币 XRP。

质押者和验证者可以积累网络代币的指数，并接触到整个生态系统的活动，而用户无需了解底层网络经济或与交易所交互即可加入。

3.同质化资产

与 NFT runtime 一样，用户无需部署智能合约代码即可创建和启动标准化的同质化代币，并兼容 XRPL 同质化代币。同质化资产 runtim 创建的每种新资产的 1% 将作为费用分配给网络节点验证者。

4.DEX

网络交易所 runtime 提供链内 DEX，用户和开发者无需部署智能合约代码即可进行资产交易、建立流动性池和启动代币价格发现。交易所费用由节点验证器收取。链内 DEX 还将集成到 XRPL DEX 中，以从 XRPL DEX 中提取流动性，从而启动网络流动性。

5.去中心化状态预言机（Decentralized State Oracle）

状态预言机允许开发者将其他链上发生的事件与 The Root Network 内的行为相联系，实现链间互操作性，允许开发者将互操作性构建到应用程序和内容中。

譬如，允许 ETH 上的 DAO 投票启动 The Root Network 上的资产转移；某网络上的 NFT 可以与 The Root Network 上的实用工具进行交互。

（四）流动性与资产桥

The Root Network 通过其通用跨链桥 runtime 原生支持资产跨链，以太坊网络上的 FLUF World 资产或 XRPL 上的同质化与非同质化资产可以跨链至 The Root Network。

三、ROOT 代币经济学

ROOT 代币总供应量为 120 亿枚，可用于协议治理、PoS 网络质押、交易费、铸造费、FLUF World 游戏代币与数据市场等。其代币分配模型如下：

  • Community Rewards 占比 20%：每月设置生态系统挑战，玩家可获得的奖励总数将由 FutureScore 决定；

  • Land Mechanics 占比 20%：面向 The Third Kingdom 中的游戏机制；

  • Ecosystem Development Fund 占比 10%：鼓励开发者、艺术家与 IP 持有者进行开发；

  • Block Reward Bootstrap 占比 10%：验证者和质押者将在 260 周内获得最多 10% 的代币作为启动资金；

  • CENNZ Burn to Mine 占比 10%：CENNZ 代币持有者可在 The Root Network 中销毁 CENNZ，每销毁一枚 CENNZ 即可按 1:1 比例获得一枚 ROOT；

  • Futureverse 团队占比 15%：锁定期为 100 周，释放期 100 周；

  • LP 分配 5%；

  • DAO 财库分配 5%：锁定期为 50 周，DAO 将控制其分配；

  • 顾问分配 5%：将为核心顾问与重要 IP 买家发放，锁定期为 100 周，释放期 100 周。

四、去中心化身份协议与资产护照

Futureverse 采用 W3C 去中心化数字 ID（DID）协议为用户和资产提供与跨元宇宙应用程序交互的方式，既能保护隐私，又能实现互操作性。

该协议允许开发者和用户创建链下身份，可在为用户与资产提供服务的应用程序之间转移，还可以为内容格式指定证明，类似于护照上的印章。

用户还可以使用无状态 eKYC 引擎，就年龄等潜在敏感数据接收与其 ID 相对应的证明，该引擎不存储用户个人身份数据，允许开发者能够提供按年龄区分的内容等体验。

五、去中心化委托协议

Doughnuts 是一种正在申请专利的委托证明协议。它们存在于两个或多个密码学密钥对之间，能够证明一个地址将某些东西委托给了另一个地址。

Doughnuts 允许对资产进行细粒度访问，譬如只允许应用程序一次性访问，或允许基于时间的访问，从而使钱包交易更加安全。

六、去中心化通信、通知与存储协议

在 Futureverse 中，节点就账本的共识由 The Root Network 提供。但通信、通知等不以价值为基础的交易服务无需全网共识，但一样需要去中心化。

Futureverse 通过 The Sylo Network 为 DApp 提供去中心化链下服务，包括实时事件、异步消息传递、通知、状态管理、实时调用、实时数据交换等。这些去中心化服务由 Seeker 节点提供，使用 Tickets 支付。

The Sylo Network 是首个将节点基础设施游戏化的网络，它将节点基础设施本身（Sylo 节点）与元宇宙角色（The Seekers，也是 NFT）紧密联系在一起，并引入社交机制和游戏奖励来激励参与。

通常情况下，运行节点的奖励仅限于代币奖励，这与网络代币激励机制一致。通过将 Seekers NFT 与 Sylo 节点相结合，Futureverse 创建了游戏内功能，鼓励更多网络参与，从而提高网络的弹性。

Seekers 还在游戏中加入了 Orders、Factions 与 Clans 概念；只有通过对网络本身做出贡献才能加入。Orders、Factions 与 Clans 将拥有更高的权利，能够在更大程度上参与游戏，并被公开承认为开放元宇宙的基础设施做出了有意义的贡献。

七、去中心化人工智能协议

Futureverse 使用 Web3 人工智能协议 Altered State Machine（ASM），其产品是一个利用 NFT 实现人工智能的可拥有性、可交易性、可组合性与互操作性的平台与协议，即 ASM Protocol 与 ASM Platform。ASM 将允许任何个人（而非公司）证明拥有、构建和管理自己的人工智能。ASM 通过集成 ASM Agent NPC，可以成为世界中的自主化身、宠物、供应商、任务提供者、传说讲述者、同伴等。ASM 产品还包括 NFT 资产投放平台。

八、内容

Futureverse 通过三种方式构建内容：

  • 遵循路径：故事与传说 > 角色 > 内容 > 小空间 > 活动 > 大空间 > 游戏 [AM1]，每一个步骤，必须花时间让社区共同创造叙事和内容；

  • 与世界领先的电影、电视、社交媒体和体育 IP 持有者合作；

  • 生态系统发展基金。
