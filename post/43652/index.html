今年 3 月，LayerZero 以 10 亿美元估值完成一轮 1.35 亿美元的 A+ 融资而进入了大众的视野 ，领投方包括了 FTX Ventures、红杉资本与 a16z。其旗下跨链桥 Stargate 的 TVL 更是在上线一周后就突破 20 亿美元，并在上线三周不到的时间内突破了 40 亿美元。

在 A+ 轮融资之后仅两个月不到的时间，LayerZero 再次传出即将融资消息，而估值已是此前的 3 倍。还未官宣完成的这一轮融资，透露了 LayerZero 会发币的事实，也引发了市场进一步的关注。

本篇文章就来盘点一下当下关注度较高的基于 LayerZero 的项目（补充阅读：《SushiSwap 创始人 0xMaki 加持，LayerZero 和 Stargate 想象空间几何？》）。由于 LayerZero 的开放性，或许有很多项目笔者并未关注，欢迎补充。

DeFi

Radiant Capital

Radiant Capital 是一个基于 LayerZero 的全链借贷协议，用户可以在 A 链上存入抵押品再在 B 链上借出，以整合多链的流动性。目前该项目在 Arbitrum 上推出了 v1 版本，该版本目前仅支持 Arbitrum 上资产的借贷。项目文档显示，v2 版本将 WBTC、ETH、USDC 的全链借贷。

官网首页显示，当前 Radiant Capital 市场规模为 8762 万美元，借款总额达到了 5795 万美元。

Cashmere Labs

Cashmere Labs 旨在成为一个具有全链互操作性的抗 MEV DEX 聚合器。目前产品暂未上线，但依据官方对产品的介绍显示，Cashmere Labs 包含两种主要产品：稳定币交易以及抗 MEV 的跨链 DEX 聚合器。

其中稳定币交易通过单边流动性和针对单边流动性的激励改善滑点和流动性等问题；跨链 DEX 聚合器则是结合了 LayerZero 的跨链互操作性以及 1inch 的流动性，由于套利者无法在链间消息传递期间获取订单信息，故无法进行抢先交易、「三明治攻击」等与 MEV 相关的攻击。

OmniBTC

OmniBTC 是一个基于 LayerZero 的跨链 DEX 聚合器，支持以太坊、BNB Chain、Avalanche、Polygon，团队表示未来将专注于新公链 Aptos 和 Sui。

以使用以太坊上 USDT 兑换 BNB Chain 上 BNB 为例，OmniBTC 提供的路径为利用 Stargate 将以太坊上 USDT 兑换为 BNB Chain 上 USDT，再使用 PancakeSwap 交易为 BNB。

Rage Trade

Rage Trade 是一个全链以太坊永续合约市场，支持以太坊、Arbitrum、Avalanche 和 Fantom。Rage Trade 通过基于 LayerZero 的跨链桥 Stargate，将支持的网络上的 DEX、货币市场（如 Aave、Rari、Euler）、衍生品市场（GMX、Ribbon）中的 ETH-USD LP 直接存入 Rage Trade 中，并将 80% 的 TVL 存入产生收益的服务中（Curve、GMX、Sushi 等），20% 的 TVL 为 Rage Trade 提供流动性，支持用户进行合约交易。

InterSwap

InterSwap 是一个基于 LayerZero 的原生多链 DEX 协议，其团队在 Pitch Deck 中表示，使用 LayerZero 协议可以实现资产在单笔交易中进行跨链交易，无需传统跨链聚合器将多笔交易聚合的方式。

目前该产品处在封闭测试阶段，暂未公开。

集成 LayerZero 的 DeFi 协议

SushiXSwap

SushiXSwap 是 SushiSwap 推出的基于 LayerZero 的全链交易协议，支持的网络包括 Optimism、Arbitrum、Fantom、BNB Chain、Polygon 和 Avalanche。用户可以在支持的网络以及资产之间进行跨链交易。

Hashflow

去中心化交易平台 Hashflow 于 4 月底宣布推出基于 LayerZero 的 Hashflow 无桥跨链交换服务，采用询价（RFQ）模式，由专业做市商设定代币价格。支持以太坊、Avalanche、BNB Chain、Arbitrum 和 Polygon 之间的代币交易。

Angle Protocol

Angle Protocol 是一个去中心化稳定币协议，其于 8 月宣布与 LayerZero 集成，实现旗下欧元稳定币 agEUR 在以太坊、Polygon、Optimism 和 Arbitrum 之间进行跨链转移。

NFT

PFP

Tiny dino

Tiny dino 是基于 LayerZero 的像素风恐龙形象 NFT，采用 CC0 许可模式。该 NFT 截至发文时在 OpenSea 上的地板价为 0.059 ETH，交易额超 4000 ETH。

Gh0stly Gh0sts

Gh0stly Gh0sts 是以幽灵为主题的首个基于 LayerZero 的 NFT 项目。该项目于今年 4 月推出时，由于是首个基于 LayerZero 的 NFT 项目，其地板价一度超过了 0.5 ETH，截至发文时其在 OpenSea 上的地板价为 0.05 ETH，交易额超 5500 ETH。

NFT 跨链协议

Parakeet.dao

Parakeet.dao 是一个基于 LayerZero 的全链跨链桥，支持以太坊、Polygon、Arbitrum、Optimism、BNB Chain、Avalanche、Fantom。目前该项目仍在测试阶段，仅支持上述网络测试网上 NFT 的跨链转账，用户可以提交希望其支持跨链的 NFT 的申请。

据官网文档显示，Parakeet.dao 未来还希望成为一个 NFT 的跨链聚合器，但目前并未透漏细节。

Holograph

Holograph 是由 CXIP Labs 推出的全链 NFT 协议，Holograph 套件将包括三个初始产品：Bridge、Core 和 Mint。Bridge 允许 NFT 在以太坊虚拟机 (EVM) 区块链之间传输。Core 是面向软件开发者的多链应用程序编程接口（API），Mint 是面向创作者的 NFT 铸造产品。

以上三个产品目前都仍处于测试阶段，仅支持 Goerli 以及 Polygon 和 Avalanche 的测试网。

Omni X

Omni X 是一个基于 LayerZero 的全链 NFT 平台。依据官方信息显示，Omni X 旨在链接创作者和用户，使得用户可以在其支持的链上铸造并交易 NFT，以聚合分散在多链上的用户和社区。

Trantor

Trantor 是一个基于 LayerZero 的 NFT 跨链通信基础设施，支持 NFT 的跨链转账。目前仅支持 BNB Chain 和 Polygon 的测试网，并支持用户铸造测试用 NFT 在测试网上进行跨链操作。

Omnisea

Omnisea 是一个基于 LayerZero 和 Axelar Network 的全链 NFT Launchpad，创作者可以在平台上发行可基于 LayerZero 跨链的 NFT。同时，该项目已发行代币 OSEA，锁定代币可以获得部分平台收入作为奖励。

Omnite

Omnite 提供了基于 LayerZero 的全链 NFT 解决方案，目前其产品暂未推出，但开发者可以使用其开发的 SDK 建立全链 NFT 项目。

NFTFi

Oswap

Oswap 是由 dLab 开发的，基于 LayerZero 的全链 NFT DEX 协议，目前其 Alpha 测试已结束。其协议涉及类似于 sudoswap，但支持基于 LayerZero 的跨链交易。

游戏

Catddle

Catddle 是一款 Web3 宠物游戏，其 NFT 基于 LayerZero 发行，截至发文时其在OpenSea 上的地板价为 0.035 ETH，交易额为 11 ETH。当前游戏内容暂未推出，但官方文档显示，该游戏类似于宠物养成游戏，并且将发行用于游戏内功能的代币 MEO 和治理代币 CAT，其中 MEO 总量无限制，CAT 总量为 10 亿枚，其中 5% 将用于空投。

Beyond

Beyond 是一款 Web3 宇宙探索类游戏，游戏内 NFT（星球等）支持基于 LayerZero 的跨链功能。
