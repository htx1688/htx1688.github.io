1.观察与观点

首先是插槽拍卖情况，致力于让Crypto资产在现实世界发挥作用的Virto Network赢得Kusama第102次插槽拍卖。波卡上正在进行第49次插槽拍卖，Energy Web X目前领先66.25%的区块。

Energy Web是一个全球性的非营利组织，通过开发和部署开源的Web3技术，帮助企业从清洁和分布式的能源资源中解锁商业价值，从而加速能源转型。Energy Web X是其在波卡上的平行链，利用波卡网络的跨链互操作、共享安全性、扩展性等优势，它能为能源行业提供更高效、更可靠、更可持续的数字化转型。

今年六月份，Energy Web与波卡合作开发开创性的可再生能源解决方案，旨在为公司提供“类似Web2的体验”。Parity Technologies的首席执行官Björn Wagner表示，“能源转型垂直领域是利用区块链技术优势的完美用例，也是波卡成为市场领导者的良好定位。”而在未来，Energy Web X或许也会与波卡生态中的其他项目进行合作，加速社会环境向清洁能源的过渡。

RWA（Real World Assets，现实世界资产）是今年各机构和项目关注最多的赛道之一，花旗银行更是称其为“区块链的下一个十亿用户和十万亿价值”。而波卡生态中的Centrifuge作为RWA赛道的知名项目，于上周和多家机构联合成立了通证化资产联盟（The Tokenized Asset Coalition），该联盟旨在通过教育和宣传将传统金融资产引入区块链，实现资产通证化。

实体资产包括房地产、艺术品、贵金属、债券等，它们通常具有高价值、低流动性、高门槛和低透明度的特点。通过将实体资产转换为区块链上的通证，可以实现部分所有权、降低交易成本、提高信息可信度、增加市场参与者和创造新的价值。

除了Centrifuge之外，该联盟的创始成员还包括DeFi借贷平台Aave、Credix、Goldfinch，Layer2网络Base，稳定币发行商Circle，Crypto交易所Coinbase，以及现实世界资产数据分析平台RWA.xyz。

该联盟的成立表明了区块链和DeFi领域对于实体资产通证化的高度关注和积极探索，而它在一定程度上对波卡生态也有着积极的影响。

一方面，它可以增加波卡网络上的实体资产类型和数量，从而丰富波卡生态中的资产流通和交易；另一方面，该联盟有望使波卡生态中的项目可以更容易地接入实体资产，并与其他区块链进行数据或资产的转移。除此之外，通过提供更多的选择和便利，它还将为波卡生态中的创新和发展带来更多的可能性。
