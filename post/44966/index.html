原文作者：Leo

近日，以太坊基础设施开发商 Consensys 更新了其隐私政策，声明当用户使用 Infura 作为默认 RPC 进行交易时，将收集他们的 IP 和 eth 地址。

Consensys 这一类似于「Web 2」的举动引发了用户对加密中个人数据的担忧，用户大多表示：「在 Web 3 的世界里，我们要确保在交易时没有泄露 IP 和 以太坊地址，保护好自己的隐私，建议更换 RPC。」

Infura 这一举动当然不妥，但其他 RPC 提供商呢？

BlockBeats 整理了几个易操作的第三方 RPC 提供商，根据网站公开文件显示，我们发现行业很多基础设施开发商也会收集用户数据。收集用户，似乎成了这些 RPC 提供商的潜规则。以下为收集用户数据的 RPC 提供商：

Alchemy

Alchemy 是一款被广泛使用的 blockchain API。Alchemy 的超级节点服务被 Ethereum, Polygon, Arbitrum, Optimism, Flow 和 Crypto.org 所采用。该服务提供了广泛的 API 支持，让开发者可方便的通过一个节点获得多种功能，包括 JSON-RPC 支持，同时提供了在区块链上运行世界级应用程序所需的强大的可靠性、数据正确性和弹性。

文件显示：「当您使用我们的服务时，我们可能会自动收集某些信息，例如您的 IP 地址、用户设置、MAC 地址、cookie 标识符、移动运营商、移动广告和其他唯一标识符浏览器或设备信息、位置信息（包括 IP 地址的大概位置）以及互联网服务提供商。我们还可能会自动收集有关您使用我们服务的信息，例如您在使用我们服务之前、之后访问的页面、您点击的链接信息、您互动的内容类型、您活动的频率和持续时间，以及有关您如何使用我们服务的其他信息。」

Moralis

Moralis 是一个 Web3 开发中间件，它为开发者提供了可扩展的后端基础设施，让用户专注于应用程序的前端。与链上交互的一系列复杂过程均可通过 Moralis 简单的完成，以此来节约开发成本并提高链上交互功能的可靠性，与其他竞品类似，Moralis 也为开发者提供了链上警报推送、智能合约监控等功能。目前，Moralis 支持 Ethereum，BSC，Polygon，Solana 和 Elrond 等多条网络。开发者基于 Moralis 可快速开发跨链 DApp。

文件显示：当用户注册帐户或以任何其他方式向 Moralis 提供个人数据时，Moralis 会收集个人信息。Moralis 还根据用户的网络活动和发送给用户的电邮中的活动收集个人数据，包括用户如何使用其网站和服务以及来自第三方的信息；手机信息：姓名和联系方式，如电邮地址、电话号码、位置、职称、所代表公司的信息、税务信息、信用卡信息和用户订购的服务；帐户详细信息，包括 user ID。

「避免疑惑，我们不会将请求的 IP 地址保存到我们的链上节点，也不会根据用户对我们的链上节点的调用进行分析或捆绑。」

Flashbots  Protect

Flashbot Protect 由 API 和 RPC 组成，Flashbots Protect API 简化了创建、签名和提交捆绑交易等一系列复杂功能，让开发人员可以轻便地将其集成到应用中，让用户能将交易发送到 Flashbots 网络，Flashbots 将交易捆绑在一起有效隐藏了信息，直接发送给矿工而不是发送到公共内存池（mempool），防止交易被抢先交易或遭遇三明治攻击。

API 主要给开发者提供了便利，针对普通用户推出的 Flashbots RPC 端点能让用户自定义添加到钱包，添加后可直接从钱包向 Flashbot 发送交易，包括在 Uniswap、SushiSwap 等 DEX 上的交易以及铸造 NFT，都可以使用 Flashbots 网络保护交易不受 MEV 攻击。

文件显示：「我们只会根据我们的合法利益处理您的个人数据，如果我们认为处理是必要的，且总的来说，您的合法利益、权利或自由不会凌驾于我们的合法利益之上，即使公司已经雇佣其他第三方代表公司执行某些活动，公司仍然是数据控制者。」

QuikNode

QuickNode 为用户的应用程序和 DApps 提供弹性和专用节点的服务，也就是应用程序编程接口（API），可查询区块链数据并促进操作去中心化应用程序，与 Infura 类似，降低了访问以太坊数据的门槛。不过需要注意的是 QuickNode 提供的不是共享的公共节点，而是专用节点。只接受用户 DApp 的调用，提高了区块链访问性能。其目前支持 11 个区块链，有 BSC、Bitcoin、Solana、Optimism、Fantom、Terra、Arbitrum、Gnosis、Polygon、Ethereum 与 Celo。

文件显示：「QuickNode 可能会自动收集有关计算机硬件和软件的信息。这些信息可以包括：您的 IP 地址、浏览器类型、域名、访问时间和推荐网站地址。这些信息用于服务的运行、维护服务质量以及提供有关 QuickNode 网站使用的统计分析。」

以下 RPC 提供商并未在网站文件中说明是否收集用户数据（可能不收集或者未披露）：

Pocket Network

Pocket Network 也是一个服务于 Web3 的去中心化 API 协议。基于 Pocket DAO 指导着协议的长期发展，它提供了一个由全节点供需端所组成的双边市场，一端为应用程序，另一端为运行全节点基础设施供应商。其由数万个分布式节点共同运行网络，不存在中心化服务器和单点故障，也就是说即使为开发者提供服务的一个节点故障了，也会有其他节点继续提供服务，由此，提供了一个容错率较高、低成本且灵活 API 服务的解决方案。

Bware Labs

Bware Labs 是一个去中心化 API 基础设施初创公司，在建立一个基于多链订阅式的 API 平台，在区块链 API 消费者和节点提供商之间提供了一个接口，允许企业和开发人员执行对多个集成区块链的数据请求。Bware Labs 在 2 月 23 日披露完成了 600 美元的融资。其团队成员多由前 Ankr 开发人员构成，部分成员还曾在 Google 任职。

Nodereal

NodeReal 成立于 2021 年，总部位于新加坡。它旨在授权开发人员、创新的 Web3 应用程序和大型 Web2 应用程序以可扩展的解决方案探索区块链。NodeReal 的 MegaNode 通过 HTTPS 和 WebSocket 提供对 EVM 区块链的即时访问，如 Binance Smart Chain 和 Ethereum。MegaNode 在最新的网络升级上运行，至少保证 99.9% 的正常运行，MegaNode 具即时可用性：用一行代码连接应用程序，没有同步，没有复杂的设置。MegaNode 提供了最佳的存档数据体验，而且每一层都是免费的。

此前于 5 月 19 日，Nodereal 完成 1600 万美元 A 轮融资，本轮融资由 Sky9 Capital 领投。

Ankr

Ankr 是去中心化的 Web3 基础架构供应商，帮助开发人员、去中心化应用程序和质押者与各种区块链轻松交互。用户访问 API 和 RPC 即可轻松构建 DApp。

据悉，此前，Binance Labs 宣布对 Web 3 基础设施提供商 Ankr 进行战略投资；此前于 11 月 23 日，Web3 基础设施提供商 Ankr 宣布已成为 Sui 首批 RPC 提供商之一。该服务现在可用于 Sui 测试网，并将在上线后立即扩展到主网连接（预计 2023 年初）。

附：如何更换你的 RPC

更换 RPC 步骤：打开小狐狸钱包，点击右上角账户-设置-网络；

下拉到添加网络；

点击底部的手动添加网络；

手动添加网络，需要自己填写相应信息，在 Network name 选择自己需要的 RPC URL 链接（前文中的替代方案部分需要注册后才能获取 RPC 链接），在 New RPC URL 栏填写自己的选择的相应的 URL 地址；

你可以在该网站选择自己需要的 RPC URL 地址（copy URL）。
