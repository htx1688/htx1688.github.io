2022 来到了尾声。这是最坏的时代，我们见证了行业史无前例的大崩盘；这是最好的时代，新世界的大门开刚刚打开，这里天地广阔。 辞旧迎新之际，Foresight News 的编辑们分享了他们对来年加密世界的看法，特整理如下：

Karen ：加密寒冬远未结束，但这不妨碍我们去探索下一波浪潮。熊市中最重要的是生存和埋伏。

第二看好的是 Web3 社交尤其是社交基础设施方面，会衍生出多种社交场景，落地价值和采用潜力是很有想象空间的，目前社交类项目因为空投预期等因素驱动关注度较高，不过这类项目还处于早起发展阶段，之后势头能否得到延续也取决于这些社交项目之后正式启动后的内部经济模型、使用体验等吸引程度，当然也可能面临着传统社交巨头的转型挑战。

Lens 通过一个 Profile NFT 来打通生态下应用和用户，部分应用在项目初期就公布了激励模型，个人还比较看好 Phaver，除了能够发布帖子之外，还每天给用户发放 5 枚代币，用户可以在浏览 Feed 流时将其质押到 5 条帖子中，质押帖子的热门程度越高，质押得越早，就可以获得更多的代币奖励，也能帮助用户发现最优质的内容，有希望成为 Web3 版的今日头条。

最后，链游、DeFi（尤其是老牌 DeFi 项目、衍生品协议）、Layer2、新公链在新一年中的机会都会很大，提前埋伏很重要。

0×11： 按照刻舟求剑的经验，明年加密行业可能还会经历萧条的一年。在 Luna、Three Arrows 和 FTX 连环暴雷之后，市场需要一段较长的时间来恢复信心，然后迎接下一波浪潮。不过在此之前，加密行业还需要技术创新为新的叙事打好基础。

我认为未来一年值得关注的领域是 MEV 和 zkEVM。

在以太坊合并之后，Flashbots 推出的 MEV-Boost 已经被绝大部分（约 90%）验证者采用，这意味着以太坊上绝大部分区块是通过 MEV 打包的。MEV 已经成为事实上的以太坊关键基础设施，对以太坊网络的稳定性发挥着举足轻重的作用，但目前整个社区对此关注度还不够高。

相关阅读：《MEV 给以太坊带来了哪些中心化问题？如何改进？》

另外一个话题，zkEVM 被认为是以太坊扩容的未来和「应许之地」。虽然实现共识层面兼容的 zkEVM 还任重道远，但是这个领域的主要探索者 zkSync、Polygon 和 Scroll 都在过去一年取得了显著进展，随着生产就绪的 zkEVM 临近，我们可以乐观估计一下该领域在明年爆发出的能量，或者能改变 L2 扩容甚至整个 L1 的竞争格局。

相关阅读：《万字详解 zkEVM：以太坊可扩展性的未来》

Peng SUN ：从市场角度来看，每一轮周期都需要一个叙事。但从长时段的视角来看，每一轮周期与每一个叙事都是加密通向自由之路的曲折进程。

首先，值得关注的是 ZK 技术。通常情况下，资本更欢迎的是能够带来直接经济与市场效益的 DeFi、NFT 等去中心化应用，与区块链、 Web3 的底层核心技术密码学相关的项目长期在寒冬下生存。早期采用 ZK 技术的 Zcash 发展至今，也仅为小众加密社区所记忆。2022 年，随着行业对链上扩容的摸索，以及 DID、SBT、Desoc、Network State 等去中心化身份方案的提出，对于隐私与安全极为重要的零知识证明技术成为加密行业未来发展的自然选择。

其次，NFT 流动性问题会是 2023 年的看点之一。在过去一年，行业经历 NFT 的狂欢高潮。乍看之下，主流 NFT 项目价格居高不下，但背后则反映着 NFT 市场缺乏足够的流动性。考虑到 NFT 被赋予的收藏价值，以及通过数学对不同稀有度的 NFT 进行定价的困难，NFT 交易量较小也不难理解。但激发 NFT 流动性市场，依然是一片星辰大海。目前，Blur 是受人关注的新兴 NFT 交易市场与聚合器，它对接了 Opensea、LookRare 与 x2y2 的接口，适合「快进快出」的专业 NFT 交易。此外，主流 DEX Uniswap 也已进入 NFT 领域。今年 6 月，Uniswap Labs 收购 NFT 聚合市场 Genie，7 月表示将通过集成 SudoSwap 实现 NFT 交易，11 月在 Genie NFT 交易聚合功能基础上推出 Permit2 和 Universal Router 两个新的智能合约，11 月底宣布上线 NFT 聚合交易平台并宣布空投。

回到 NFT 作为 Token 本身，解决流动性问题或许是将 NFT 价值从艺术品的收藏功能进一步迈向 Web3 资产所有权与价值流动的重要一步。

Aididiaojp.eth ：凭借多年韭菜经验，我对 2023 年加密市场行情的看法是市场底部虽未现，逐步建仓可开始，优先龙头再山寨，杠杆合约需等待。

市场底部的出现一般会出现两种走势，一种是 2018 年熊市的长时间低位震荡收敛；一种是 2020 年 312 高深度的暴跌。从目前市场走势来看，比特币从 69000 高点下跌到目前低点 15460，跌幅约达 77%，时长已约有 360 天。对于上一轮牛市开启的震荡区间 10500-12000 来说，市场仍有下跌空间。中期来看，受 FTX 暴雷影响，市场仍处于弱势震荡向下趋势中。

综上所述，如果按照两年一个周期，那么最迟在 2023 年年底市场将筑底完成，继续下一个周期。所以现在已经可以开始逐步建仓了。知行合一：我已经拉上室友成立一个旅行基金，把每月工资的 15% 用来定投。我目前已建仓的代币有：BTC、ETH、LINK、OP、DOGE、APE、FLOW、APTOS，比例分别为 3:2:1:2:1:1:1。APTOS 目前持有的是上次空投的代币，准备用它建杠杆仓位。

具体选择什么币种，我的编辑部大佬们已经分享了很多具有潜力的代币和赛道，大家可以结合自己的看法择优建仓。我的观点是在熊市震荡期间，优先行业龙头代币入手。等待龙头代币趋势呈震荡收敛时，再开始建仓风险更高的山寨币。

友情提示：在市场震荡时期，会出现向上或向下剧烈波动，所以任何杠杆率的合约很难活到下一个周期。

Frank： 2023 年 Web3 社交（SocialFi）赛道应该会迎来值得期待的变化，我一直相信 Web3 天然适合社交，但 Web3 社交也绝不是目前 Web3 版的微信、Telegram、贴吧等映射产品。

2023 年的从业者需要认真去思考并去挖掘 Web3 世界所该有的新社交形式。Web3 世界里，地址就是人，是承载一切经济行为、生产价值数据的载体，每个人都映射在一个个地址后面，所以基于海量、即时的链上底层数据，对地址进行不同标签的划分，挖掘并形成链上的社交图谱，可能才是 Web3 世界的社交关系和需求场景。

而只要能形成原生的链上社交图谱和社交关系，就能衍生出一系列的 Web3 社交场景，譬如个人层面可以进一步实现链上聊天、通讯、交流等基础社交功能，也能为其它 DApp 提供社交扩展服务（广告投放、信息通知、联通用户、促新拉活等等）服务。

就像今年 Etherscan 推出的「View NFT」和「Chat with Owner」功能，其中「View NFT」允许用户点击查看以太坊地址下所持有的 NFT 详细信息，「Chat with Owner」则允许用户跳转至 Blockscan Chat 直接与 NFT 持有者沟通、聊天，我一直觉得这就是基于自身链上数据积累才可实现的 Web3 式链上社交产品模式，一切都是水到渠成（延伸阅读《Web3 社交赛道的新变化：向族群化、场景化转变》）。

最近一段时间以来，从最初 Etherscan 的 Blockscan Chat，再到 Push Protocol（原以太坊消息推送应用 EPNS）的 Push Chat、Nansen 的 Nansen Connect 服务，新近做链上原生加密信息传递的应用也开始肉眼可见地增多，应该就是前奏——基于链上数据形成社交图谱，向地址要应用场景。

除此之外，2023 年以太坊大概率会再度成为新一波创新与盈利效应聚集的策源地，无论是 PoS 时代的流动性质押赛道（Lido 等）还是二层扩容赛道的新动向（zkEVM 等，延伸阅读《Web3 时代的 ZKP，如何迈向主流？》），都在逐步积蓄势能，只待技术与情绪的引爆点。

凯特 ：当大家都还在犹疑行业是否还会复刻“四年一轮”周期的不成文规律时，2022 年的加密市场其实已经给出了“大同小异”的答案，大牛后的一年标准深熊、世界杯概念的火热依然在佐证着这个规律的有效性。既然如此，不如先将过去的历史轨迹继续先当做一个未来可能会发生的假设，剩下的就边走边印证了。

个人还是觉得，在即将到来的 2023 年，会有不少局部回暖行情，其中有一定成功概率的方向里，应该仍有「莱特币减半行情」的一席之地，即便从币价角度而言，可能真正的主角未必是 LTC，但至少可以跟着整体走势，有吃上一波的机会吧。

另一个猜想，便是 NFT 板块，2022 对于 NFT 大赛道来说，应该是经历的第一个深熊，依然可以看到无数 Bulider 在这领域前行，市场虽然仍在磕磕绊绊的摸索，但能看到的是在逐步脱离单纯 PFP 炒作，既然熊市都无法使其退出舞台，那就不妨大胆的看好下 NFT 对的未来，或许在 2023 ，也能看到 NFT 赛道迎来自己的突破，实现属于自己的第二浪新春天，所以布局一下也不是坏事儿，除了 BAYC、AZUKI、Doodle 这些蓝筹选择外，作为基础设施的 NFT 交易平台 / 衍生品板块可能也是值得重点关注的一块。

Yobo ：其他小伙伴都在讲赛道，那我就不负责任地提几个自己比较期待的更具体的点吧。

Doodles，二流蓝筹里 Doodles 今年的表现还是挺拉垮的，地板价也已经被 Azuki 拉开了一个身位，团队看起来有点不务正业的状况和一拖再拖的 2.0 大版本确实有点令人失望。不过扎实的团队背景和在音乐 / 潮牌等实体领域积极尝试的基本盘摆在那（延伸阅读《关于 Doodles 的所有你知道或者不知道的故事》），配合画风诡异但相当活跃的全球社区的支持，重新起风的时候，Doodles 创始团队那个打造「迪士尼级别 IP」的梦，万一真实现了呢。

Arbitrum（DeFi 新战场），这个看起来有点无聊，不过不知道大家有没有察觉到 Arbitrum 已经成为了 DeFi 创新环境最友好的温床，没有之一。在市场遇冷的大背景下，Arbitrum 上近几个月时间里冒出了很多稀奇古怪的 DeFi 项目，在越来越多的人开始相信 DeFi 龙头地位稳固难以被撼动，DeFi 创造力不再的时候，一些脱离传统金融用例，并基于可组合性开的脑洞，选择了在 Arbitrum 上落地，这里面很可能会出现下一轮百倍甚至千倍的宝石。

Magic Eden，关于谁能挑战 OpenSea 这件事儿，Magic Eden 被忽视了。在 OpenSea 进行多链支持的过程中，Solana 生态对于 OS 兼容这件事情的积极性可以说是最低的，OS 也没有撼动 ME 的 Solana NFT 成交量，Magic Eden 本身足够好的使用体验形成了护城河。在 Magic Eden 同样开始进行多链支持并开始在游戏上加大投入之后，这个曾经 Solana 生态的荣光，同时也是 OS 都抄了的 NFT Launchpad 的先行者，Magic Eden 的创造力还有很大的想象空间。

Chloe ：对于未来一年所有所期待的赛道表现来说，个人首先是看好去中心化 Social 领域的发展。全球类似 Meta（前 Facebook）这样的中心化平台对用户的数字社交体验产生了太多的负面影响，数据透明度和所有权将是去中心化 Social 的核心价值，将数据返还给用户自己，将 Web3 社交联系民主化具有一定的意义。

其次是 NFT 相关赛道，过去一年，NFT 交易平台创新层出不穷，Opensea 收购 NFT 交易聚合器 Gem，x2y2 对所有 NFT 征收版税计划不断改革，以及近期完全去中心化 NFT 交易平台 Blur 的交易量突破等，都在为 NFT 赛道的流动性持续发力中。另外 NFT 项目方面，类似 Doodles、Moonbirds 等蓝筹项目方已经摆烂很久，这反而给了其他类似 Pudgy Penguins 这样的潜力蓝筹项目更多机会，包括小幽灵等国产社群凝聚力强的潜力项目。买 NFT 除了个人爱好，也是熊市中 E 本位杠杆性的投资价值。

再者是看好未来一年链游的发展， 链游的整体质量上升以及经济模型的越来越合理性，市场上迟早会再出现 1-2 款好玩的 3a 品种的游戏带动链游进入 2.0，突破 2020 年末首创「Play to Earn」的 Axie。同时，NFT 叙事和元宇宙场景之间的交互赋能可以更好地为链游赛道做支撑。

angelilu ：今年年末加密世界在经历一系列暴雷事件后，市场行情需要时间回暖，因此我认为非金融的基础设施将会在这个阶段得到取得一些发展。首先看好 DAO 相关的工具，DAO 的发展还处于非常早期的阶段，宪法 DAO 之后再没有什么 DAO 能激起波澜， DAO 在 Web3 领域还有很多方向值得去探索，例如投资、协作开发、信息共享等，而前提是 DAO 相关基础设施的完善。目前 DAO 最常用的协作工具还是 Web2 领域的 Discord，Web3 领域需要一个 DAO 相关的杀手级应用。

除 DAO 相关的工具外，看好灵魂绑定代币（Soulbound tokens，SBT）的发展，距离以太坊创始人 Vitalik Buterin 提出这个概念不到一年，但却在该概念上滋长了很多仍在建设中的新概念的项目，某些传统巨头也在尝试使用 SBT。相比于 NFT，SBT 不可转让的特性让 Web3 发行方乃至传统品牌能更紧密地与用户和消费者绑定关系。以及在元宇宙发展的热潮中，很多应用了 NFT 的元宇宙游戏可能会进一步采用 SBT。

Babywhale ：涨跌这种事，去年牛市给我最大的教训就是不要轻易去猜测顶和底而是学会顺势而为，所以在此就不去妄加论断涨跌问题了。

赛道方面，未来我还是更加看好跨链领域，无论是例如 Cosmos、波卡这样的跨链生态，还是 LayerZero 这样的跨链互操作性协议，抑或是 Hop Protocol、Across 等跨链桥。虽然说跨链桥一直是黑客攻击的重灾区，但可以想象下个牛市来临时链上资金寻找「金矿」必然会催生大量的资金跨链需求。加之当前跨链协议已经向着直接在 A 链调用 B 链的方向发展了，相信下个牛市跨链协议会扮演一个非常重要的基础设施角色。

第二个看好的赛道仍然是 DeFi，虽然交易、衍生品等方面已经有了几乎绝对的龙头，然而 DeFi 仍然有进一步的发展空间，这种发展空间并不体现在各种极其复杂的衍生品或结构性产品，而是在于一些创新的案例。例如 Synthetix、Integral 这样的原子交换产品，在上个牛市并未释放出足够的能量，却有可能在积蓄力量之后实现爆发。

最后，个人觉得下个牛市更底层的基础设施，例如存储、计算等有可能也会有不错的行情。随着很多存储公链智能合约功能的完善，加之传统机构的采用，虽然可能不比 crypto 原生领域更加具有想象力，但实际的使用会带来更多关注。

最后的最后，正如很多投资人所说「不投看得懂的项目」，在这个熊市逐渐展露头脚的游戏、社交、身份等等新概念，也很有可能会有引发市场的追捧，我个人也会进行持续关注。但个人偏好于一些更加高确定性的机会，见仁见智。
