与今年夏天同时升温的，还有Web3的热度。

2022年，波谲云诡在于：在愁云惨淡的整顿潮、裁员潮中，互联网造富神话并非终结, Web3逆势为大厂开辟了一条新的光明路。数据显示，仅2022年一季度，Web3领域的投资额就达到了近百亿美元，是去年同期水平的两倍多。还有机构预测，2022年Web3在应用端的市场规模将超过500亿美元。

因为Web 3是一次足够颠覆的技术革命，大厂们正在为人类略显贫瘠的想像力所能企及到的前景而争执。也正因为此，任何人都无法在这场足够新的技术浪潮面前充当权威。

一、先从“3”说起

关于Web3的报道席卷而来，及其将要给行业带来的巨大变化令人兴奋，但很少有人真正了解，它为什么会产生，以及它将会给这个世界带来什么？要弄清这两个问题，我们首先需要回到过去，来认识两位我们更为熟悉“朋友”Web1.0和Web2.0。

如果将互联网发展分为三个阶段，出现在Web 1.0和2.0时代的大厂名字，都成了夸张的印钞机，这就解释了为什么Web3能在资本寒冬中，获得那么多投资人的青睐。2022年之初虽然形成了一种“言必Web3”的氛围，但却没有任何一个人可以笃定的说Web3是什么并获得大家的一致认可。目前市场上认可度较高的共识是：如果说Web1.0是“只读”的互联网，Web2.0是“读+写”的互联网，那么Web3就是“读+写+拥有”的互联网。

图表：时间轴——从Web1.0到Web3.0

具体来看只读时代 的Web1.0 ，当时的网站投喂信息给用户，用户只能被动地浏览文字和图片，比如张朝阳和它的搜狐抓住了机会成为了那时最靓的仔。

随着博客的兴起，互联网正式进入了交互时代 的web2.0，与此同时现象级产品iPhone的横空出世，又让人们迅速进入移动互联网时代。从2004年至今，用户早已从被动接受信息，转为可以在微博、Facebook、抖快等社交网站主动发表言论，真正成为了内容创作的主角。

但Web2.0的缺点是，网民对账户和数据不具备真正的拥有权 。比如我们随时可见一个头部主播的销售额，堪比一家上市公司。然而创造完销量奇迹后，他们随时会消失不见，因为从一开始平台和用户的权力都不对，平台拥有绝对的生杀大权。

图表：WEB3.0与1.0和2.0的主要区别

Web 3.0的改变在于，通过区块链等去中心化技术，把信息的使用权和流动权都交给用户 。简单而言，Web3.0能重塑互联网的“真实”，让用户拥有真正属于自己的数字资产。以太坊创始人Gavin Wood对Web3的定义，“少一点信任，多一点真实”，在他眼中，Web3将减少用户对互联网中心化组织“信任”而导致的“赋权于人”，真正利用“真实”的技术解决“垄断”问题。

举个例子，比如最近1个月以来，鹅厂连续宣布停运三款产品，包括《秦时明月世界》《掌上WeGame》和《文明大爆炸》，那么这几款游戏用户花的时间和金钱都打了水漂。

但到了Web3.0时代，传统的游戏货币化模式将被颠覆，通过区块链安全地记录网络上的每一笔交易，并允许创建NFT，如武器、服装、化身和虚拟土地。而基于NFT的游戏物品可以在不同的平台上迁移，并在二级市场上出售给其他玩家。

所以，Web3的大饼就是让用户“当家作主”。

二、从Web3版“淘宝”，到Web3版“社会”

当众人对Web3的蓝图怀抱敬畏之时，世界上最忙的马斯克态度就比较暧昧，一会儿让比特币暴涨暴跌，一会儿又和走向Web3的推特相爱相杀，但也无法不承认他早就被 Web3 拥护者视为最重要的意见领袖之一。

而马斯克说的"预测未来最好的方法就是着手搭建它 “的观点，也让巨头们不甘示弱，先后付诸行动探索Web3。大量来自苹果、亚马逊等公司的高管辞职投入Web3项目，类似的国内互联网大厂也有一批“高P”放弃了稳定的高薪和可观的期权，争相投身Web3的创业大潮。

图：Web3.0场景典型生态场景及厂商示例

要说当前Web3最火的应用是什么？答案或许都和NFT有关。

首先我们要知道，NFT作为比特币的一种，可以通过对各种数字资产加密，来确保数字资产的唯一性。放到现实中，很多物品的价值并非来自本身，而是稀缺性，因此NFT独一无二的特点吸引了众多创作者生产数字商品。

不论是一双鞋，还是一幅画，都可以变成NFT。Twitter上最为火爆的无聊猿NFT动辄百万，国内比如奈雪的茶打造的元宇宙IP的数字盲盒，72小时斩获了GMV2个亿。全球范围内，数据显示今年仅一季度，NFT交易量就达到了260亿美元，超过了去年全年的交易量，且增长幅度接近50%。

在数字藏品热和市场井喷式发展的大环境之下，继而Web3.0时代的“淘宝”——数字藏品平台 ，成为了火热赛道。而全球最大的NFT交易平台开放sea，目前估值也超过 了100 亿美元。

没有谁不希望成为下一个Opensea，在数字藏品热和市场井喷式发展的大环境之下，各行各业、大小企业都在布局这个赛道。据统计，仅国内数字藏品平台已超过500家，阿里巴巴、腾讯、网易、B站等巨头均参与进来。各家切入的维度大差不差，更多是在深度和广度上有所区别。

除了想成为Web3版的淘宝，Web3与其他新兴科技的融合发展，也开始在金融、游戏、音乐等大众生活领域，塑造了一些行业新生态。

  • Web3.0 x 金融→DeFi

DeFi意为去中心化金融，通过允许人们、商人和企业通过新兴技术进行金融交易，从而消除了第三方机构，手续费和时间成本将被大幅地削减。

  • Web3.0 x 游戏→游戏

GameFi 即“Game+DeFi”，指引入了 DeFi 模式和工具的区块链游戏。用户在玩游戏中赚取加密货币从而取得利益，已经成为Web3领域最新潮的玩法。比如StepN、Axie Infinity等通过“买NFT运动鞋+跑步”赚钱的游戏，已经在东南亚等国家制造了不少财富神话。这种革新整个游戏商业领域的模式也成了资本火热的标的。

图表：2022Q2 Web3赛道融资情况

  • Web3.0 x 组织→DAO

DAO译为去中心化自治组织，Web3.0生态的建设，诸如应用、工具、协议等，都离不开协作，使用户有序协作的组织形式就叫做DAO。代表性的项目如公司、社区、基金、资助、创作、收集、媒体等等，或将颠覆未来的工作模式。

图表：传统组织 vs DAO

在音乐圈，已经有不少欧美音乐人靠着用NFT发行数字专辑的形式，赚到了远超其粉丝基数的超额财富；在教育领域，基于区块链技术的学生就业系统，高等教育机构可以根据学业成绩和表现评估学生专业技能，结合招聘信息实现智能就业推荐;潜在雇主获得学生授权后，可验证相关信息的真实性；在医疗领域，电子病历"上链”，可以让患者本人成为个人医疗数据的真正掌控者，从此告别书写不统一、可追溯性差、难以界定责任等问题……

可以发现在 Web3 的世界里，很多脚踏实地的案例正在发生着。根据数据研究机构Apptopia的报告，2022年可供下载的Web3应用程序数量的增长速度，几乎是2021年的5倍。

如今，技术正在快速产业化，比如5G产业已经成为了千行百业的基础设施。可以预见， Web3时代或将成为产业互联网的新起点，因为随着XR、数字孪生以及NFT等等新技术的产业化，千行百业将再次提效。

不仅如此，如果说在Web2.0时代，技术是提高效率的工具，那么在Web3.0，技术还将变成制度，重塑社会和文明，所有边界将进一步融合、所有元素进一步灵活组合。

三、有好有坏，有未知

不过，木秀于林风必摧之，Web3的“高温”也引起了不少质疑声。

不说那些骗局，目前Web3存在的问题，一方面归因于技术落后。比如效率问题，去中心化网络仿佛一座黑暗森林，是零信任的世界。所以需要耗费更多的资源去完成一件事，因此去中心化网络的分布式储存计算所需的硬件资源远高于中心化网络，同时会带来能源消耗问题。

还有内容匮乏 ，内容创作门槛高大众难于进场导致内容价值无法流通。

另一方面，归因于监管缺失 。监管相对于产业的滞后性是必然的，因此出现了不少Web3项目被认为是投资机构及创业公司为吸引资金的炒作营销，缺乏实际意义。

更多的争议直指Web3本身，即对于“去中心化愿景能否实现”的讨论，有一个极度悲观的观点：打破Web2.0时代巨头中心化垄断的，只不过是新的垄断建立者。正如谚语"灯下黑"所言，所谓的Web3革命，可能只是科技新贵们的跑马圈地场，因为当权力不关在笼子里那一定是一场新的灾难。

不过随着时间的推移，技术的事情可以交给创业者，而监管层必定也会日益完善。

技术上，诸多创业公司希望搭建Web3.0 Infra，类似像DeepNatures一样的初创企业正在用技术来帮助每一个互联网用户，成为下一代互联网的基础设施建设者和自己数据主权的拥有者。

监管上，今年我国的NFT交易领域，开始完善相关规定。在目前看来，监管层面正在日益完善，在我国的NFT交易领域，已经有了相对完善的规定。例如，中国互联网金融协会、中国银行业协会、中国证券业协会联合发起倡议，防范NFT相关金融风险，坚决遏制NFT金融化证券化倾向；国家文物局也做出规定，文博单位不应直接将文物原始数据作为限量商品发售，等等。

不论如何，当技术出现新的变化时，新行业、新模式乃至新的巨头，也有可能就此诞生，更何况，Web3才刚刚开始。

是天使还是魔鬼？是未来还是谎言？还是马斯克那句话，预测未来的最好方式就是去创造他。
