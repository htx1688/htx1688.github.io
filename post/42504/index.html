围绕市场情绪、数字稀有度、经济效用和所处生态这四大维度，对相关的 10 个价格影响因素展开详细的探讨。

写在前面

尽管身处熊市，但 NFT 依然是起起落落中的热门词。据 Chainanalysis 的数据，仅在 2021 年就售出了超过 400 亿美元的 NFT。截至 2022 年 7 月，已经售出 420 亿美元的 NFT，预计年底总销售额可能达到 600 亿美元或更多。

然而，令人瞠目结舌的不仅仅是 NFT 的整体销售。许多单独的 NFT 都以破纪录的惊人价格售出，包括 Beeple 著名的 Everydays – The First 5000 Days，它在 2021 年以 6930 万美元的价格售出。2022 年 2 月，Julian Assange 和 Pak’s Clock NFT 以 5270 万美元的价格售出，同月，CryptoPunk #5822 以惊人的 2370 万美元售出。

很多人在看到一些 NFT 的疯狂标价后，都会情不自禁的发问，怎么卖的这么贵？这个价值到底从何而来？作为一种特定的资产， NFT 的价值也会因各种重要次要因素而有所不同。

评估 NFT 的价值通常基于四大变量展开，它们分别为为市场情绪、数字稀有度、经济效用和所处生态，而基于这四个变量衍生出了市场影响因素又包括市场营销，社群运营，实际应用等等，在本文中，我们将讨论这些围绕这四大维度对文中的 10 个因素展开更详细的探讨。

01 市场情绪

一、文化基因和社区核心价值

每一个 NFT 创作的发行方或个人在进行创作之前需要明确自己作品背后代表的哪一类社会群体，作品体现出了哪类社会现象和文化特征。因为这种文化特征背后代表的是一个天然群体的审美和情感共鸣，关注者在熟悉的文化特征面前更容易产生归属感跟认同，对项目而言，这类支持者可以为社区的长期运营奠定良好的文化基因跟社区核心价值。

例如，Cryptopunk NFT 体现着强烈的加密圈原教旨主义和朋克精神，无聊猿背后体现着存在主义的虚无不羁态度，Azuki 背后蕴含着强大的日漫文化。这写独特的圈层特质和文化属性为其项目的长远建设跟社区运营提供了原动力。

具有这类特性的作品一经发行，就会吸引这类圈层群体的关注，支持者们也会更积极的对作品进行宣传，有更好的社群互动。对比没有情感连接的 NFT，这类项目可以更快的形成市场共识，并不断影响市场吸引新成员的涌入，对 NFT 价格产生积极的影响。

二、 预售和底价

一般来说，围绕 NFT 的营销在单个 NFT 被铸造之前就风风火火的展开了。其中的预售会对整个 NFT 的营销产生事半功倍的效果，项目发行方通常会在预售前就在社交媒体上营销，并创建一个或多个 Discord 或 Telegram 组。较大的项目可能会为不同的国家或语言创建不同的组别。

使用这些渠道，他们会向早期参与者透露 NFT 的样本图像、模因和视频，并讨论待发布藏品的故事和背后意义。这种营销推动越成功，NFT 收藏品推出后价值飙升的可能性就越大。

在预售发布期间，NFT 创建者也可以创建独家白名单，让早期参与者可以在特定的「预售」期间铸造 NFT，预售期一般持续持续几个小时至几天不等。如果该藏品项目前期预热足，市场热度高，会坚定早期参与者的购买的信心，不出意外的话，他们可以用比公售低得多的价格购入。

▲ 由 NFT Price Floor 提供的具有最高价格下限的 NFT 藏品。

预售前，NFT 的创建者会设定一个初始铸造价格，即铸造 NFT 的最低价格。如果一个项目前期筹备的不错，随着在预售期间或者预售后竞标的热度，特定的 NFT 底价以及最终整个藏品合集的底价都会持续攀升。当现有 NFT 持有者开始在二级市场出售时，底价也将成为他们愿意出售 NFT 的最低价格。

截至 2022 年 8 月，底价最高的 NFT 藏品无疑是 BAYC，为 84.19 ETH，接近 137,000 美元。

三、合作背书和炒作

传统互联网中 IP 联名的营销策略屡见不鲜，在 Web3 世界中也是双赢的打法，项目方如果有跟知名品牌跟名人合作，可以在前期全面带动市场的期待值。例如，娱乐圈、体育圈的名人也开始为各种 NFT 代言，甚至有名人参与到一些 NFT 项目的发行制作中。

潮流艺术平台 Ezek 联合周杰伦旗下品牌 PHANTACi，发布了首个数字头像 NFT，并且限量发售 1 万只的 Phanta Bear。尽管该项目并非由周杰伦亲自推出，但他本人也换上了 Phanta Bear 头像，剩余的 NFT 在短短 40 分钟内被抢购一空，当时总价值超过 6200 万元，登上了全球 NFT 的当日交易量榜首。不得不说，此次 NFT 项目因周杰伦明星光环流量加持，自消息传出便备受关注。

▲ 截图来自 OpenSea

事实上，BAYC 于 2021 年 5 月发售价格为 0.08 个 ETH，刚开始也是无人问津，直到 NFT 大佬 PranKsy 批量购入 BAYC 后才把局面打开。在第一波热度之后，BAYC 团队也具备了一定能力，紧随其后便开始了空投发放、组织线下聚会等一系列活动，并及时赋予了无聊猿的社交属性，让其在短短几个月内火遍全网。之后，越来越多的名人开始关注和购买 BAYC，如 NBA 球星斯蒂芬库里花费 18 万美元购买了一只 BAYC，并作为自己的社交媒体头像；波场的创始人孙宇晨以 1000 万美元的价格购买了一只 BAYC，更是引爆了加密圈的舆论。

然而由于名人背书的影响力过于大，项目的热度也可以通过炒作的方式达到欺骗性的的热度。一些山寨炒作项目的营销范围广泛，从付费广告到购买虚假账号，在到支付水军吹捧，甚至还会高价请名人推广造势。例如，著名的 YouTuber 和拳击手 Jake Paul 因他最近推广的 Animoon 受到了热议，Animoon 是一个受口袋妖怪启发的 NFT 藏品，其创始人向粉丝撒谎并未能兑现诸多承诺。据估计，Paul 通过推广加密货币和 NFT 赚了大约 200 万美元，其中大约 90% 被证明是通过欺骗性的方式获得的。

▲ 信息截图来自 CryptoSlate

一些拉胯项目最终会受到市场诟病，但在很多现实情况下，因为项目前期热度炒作，一些早期持有并抛售的人仍获得了客观的回报，但因为项目缺乏长期规划以及兑现能力，NFT 本身的价值也将持续下跌。

四、传统艺术家和名人入场

与名人合作的项目类似，传统艺术家或者名人直接以自己的名义发行 NFT 自带忠实用户，对市场有着积极的号召，具备先天优势，入圈即蓝筹。Damien Hirst 是就是著名传统艺术家进入 NFT 领域的一个成功例子。

▲ 传统艺术家 Damien Hirst 的 NFT 官网

Hirst 被认为是世界上最畅销的在世艺术家之一，已售出超过 2 亿美元的艺术品，他也是是 1990 年代年轻英国艺术家 (YBA) 运动的主要参与者。与致力于出售极其昂贵的个人 NFT 的 Beeple 不同，Damien Hirst 偏爱收藏路线，他的 10,000 NFT 收藏「The Currency」的起价为每 NFT 2,000 美元，最高飙升至每 NFT 超过 20,000 美元（最近底价在 8,200 至 9,200 美元左右）。

02 数字稀缺性

一、特征稀有性

在同一个 NFT 集合中，NFT 的价格可能会有很大差异。例如，我们已经提到，截至 2022 年 8 月，最便宜的 BAYC NFT 价格接近 137,000 美元。然而，同一个系列中的 NFT 在 2021 年 10 月以超过 340 万美元的价格售出，比当前底价高出 2400%。这种价格差异大部分归结为稀有性。

发行方在供需方面增加了最终产品的稀缺性，创建者通常在设计时涵盖普遍特征、不常见的特征、一些稀有特征和一些超稀有的特征。具有多种极其稀有特征的集合中的 NFT 通常会获得最大的价值并具有最高的底价。

NFT 的稀有性对收藏者来说非常重要，市场上有多种方法跟工具可用于确定单个 NFT 的整体稀有度，其中一些包括 Rarity.Tools、Rarity Sniper、Traitsniper 和 Rarity Sniffer，每个都可以提供数千个 NFT 藏品的大量稀有度和价格数据。例如下图为用 Rarity.tools 追踪最稀有 Moonbird NFT 的数据。

二、NFT 销毁

NFT 的数字稀缺性不仅可以通过对一些具有稀有特征的 NFT 进行编程来实现，它还可以通过销毁或「燃烧」集合中的一些 NFT 来人为干扰稀缺性。NFT 就像同质化通证一样，可以通过将其发送到不存在或锁定的钱包地址来销毁，让该 NFT 永久退出流通。

NFT 销毁通常由发行方设计完成，以将稀缺性引入组合中。另外，有时鲸鱼收藏家也会执行这类动作，他们认为销毁的 NFT 的损失可以通过 NFT 上涨的价格来弥补。

例如，在 Damien Hirst 的 NFT 收藏品 The Currency 的案例中，购买该项目的收藏家有一年的时间来确定他们是否想要拥有 NFT 或随附的实物艺术品（他们不能同时拥有两者）。每当收藏家选择实物艺术品时，随附的 NFT 就会被烧毁，而每当他们选择 NFT 时，艺术品就会被销毁。在 2022 年 7 月的最终交换截止日期之后，5,149 个 NFT 被烧毁以换取实物艺术品，使 NFT 总供应量从 1 万变成了 4,851 个，该系列在此期间底价从 2,000 美元上涨至 20,000 美元，足足上涨 10 倍，NFT 销毁成了主要驱动力。

03 经济效用

在 NFT 吸引人眼球的艺术形式下，区块链为其赋予了开创性的技术进步，它的效用可以在 Web3 生态中集合产生复合效应，可直接成倍得影响 NFT 的价值。

NFT 常见的效用形式包括艺术、社交、功能、资产等，并且我们可以在游戏、音乐、域名、保险和身份等各个领域看到 NFT 的身影。NFT 可以有多种格式，通过各种标记提供不同的功能，例如在 Axie Infinity 中为用户提供淘金功能的游戏 NFT；将艺术家和版权汇集到目录中的音乐 NFT；使用 NFT 作为抵押品的抵押和贷款等等。

另外主流社区通过与独特的艺术品 NFT 相关联，通过游戏和投资社区购买 NFT，以此来实现效能的叠加，这些 NFT 产生的效益或奖励能进一步催化 NFT 价值的上涨。

04 所处生态

一、通证经济模型的设计

一个好的通证经济模型是项目可以可持续发展的主心骨，深刻影响着 NFT 持有者的长期价值。模型设计为构建者提供了一个更丰富、更有表现力的工具集，良好的系统设计可以将生态所有的东西连接起来，为生态带来持续的流动性。

例如 StepN 这款户外跑步游戏产品，它的 Token 设计是奖励人们做一些对自己有益的事情，具有长期可持续发展的基础，对其运动鞋 NFT 的长期价值有一定保证，据统计，这款应用月活高达 200 万，有庞大的核心用户群体，即使在没有新用户进来的情况下，他们的市场也具有一定的流通性，再加上项目方对通证经济模型不断改进，可以促使项目往更加可持续的方向发展，对游戏内的 NFT 有积极的影响。

二、所铸造的区块链

用于 NFT 铸造的链有很多，但以太坊仍然是最受欢迎的。其他流行的 NFT 区块链包括以太坊 L2 Polygon，以及其他的 L1 区块链 Solana 和 币安链。

不同链的生态基础以及铸造费用有所差别，也会对供需关系以及价格产生一定的影响。

▲ Solana NFT 市场上的 SolMonkeys 

目前，大多数著名和昂贵的 NFT，包括 CryptoPunks 和 Bored Ape Yacht Club 在内，都是在以太坊区块链上铸造的。例如，以太坊 NFT 单价最高的销售已达数千万美元，但迄今为止最昂贵的 Solana NFT 售价则 为 200 万美元。2022 年 1 月，Solana NFT 的销售额终于突破了 10 亿美元，这对于 Solana 来说是一个具有里程碑性的数字，但还是与以太坊的销售相差甚远。

值得说明的是，在什么链上铸造，对于像 Damien Hirst 这类名人而言便没什么影响了，在不为人知的 Palm 区块链铸造丝毫不影响他的名气跟热度。

三、加密货币市场

除了通证模型和特定区块链，加密货币价格也可能影响 NFT 价格浮动，它们并不是正相关关系，而是对供需关系跟发行方策略的影响。例如从 2022 年初到年中，NFT 整体月销量明显放缓。应该注意的是，虽然 NFT 通常以特定的加密货币计价，例如 ETH 或 SOL，但这些加密货币的价格下跌，卖家可能会提高价格来对抗加密货币通胀。

写在最后

我们从以上四个维度罗列了影响价格的 10 个常规因素，但事实不幸的是，在暗地里，还有很多洗钱、内幕交易和地毯拉力等违规价格操纵行为。

例如当有人以低价购买资产，然后以更高的价格偷偷将其卖回给自己以欺骗市场时，就会发生洗钱交易。以至于一些加密货币专家认为，超过 33% 的 NFT 销量是由洗售交易构成的，区块链数据平台 CryptoSlam 甚至估计，在流行的 NFT 市场 Looksrare 上，洗钱交易占 95%。

内幕交易是 NFT 市场的另一个问题，在某些情况下可能会影响 NFT 价值。2022 年 6 月，OpenSea 的一名前高管成为第一个在美国被指控犯有 NFT 内幕交易的人，此前据报道该高管利用有关 NFT 上市的机密信息谋取了自己的经济利益。与洗盘交易一样，内幕交易计划有时会暂时提高 NFT 的价格，但随后价格会暴跌，从而使潜在的受害者蒙受损失。

地毯拉力则是 NFT 领域杀猪盘式操作，NFT 的价格会在前期营销期间上涨，然后在发行方圈完钱卷铺盖走人后崩溃。

总之，在 NFT 的世界里，所有闪烁的并都是黄金，其价格也受各类因素影响，在收藏或购入任何 NFT 之前都需要从各个维度综合判断，do your own research。
