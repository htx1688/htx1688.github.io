今天凌晨， Sui开发团队Mysten Labs 在官方推特上举办了主题为「The Wave to Mainnet」的Twitter Space ，两位联合创始人Evan Cheng 和 Adeniyi 参与此次Twitter Space，解答大家对SUI Network 的问题。

以下为常见问题的快问快答

Q：什么是Wave to Mainnet？

A： 「The Wave to Mainnet」 是一系列教育活动和内容，旨在进一步在主网上线前，让社区了解SUI Network 的协议的潜力。

Q：Sui 主网什么时候上线？

A： Sui主网将于今年Q2上线，更多信息待分享。

Q: 如何第一时间获取更多关于SUI 的消息？

A: 所有官方信息都可以在https://sui.io 上找到，或者是SUI 的Discord 群

http://discord.gg/sui

Q: 会有空投吗？社区该如何获得SUI Token？

A: 目前官方表示没有正式的空投计划,不过SUI Network 发布了SUI Token 社区访问计划以及SUI Token 的分配细节。

SUI Token 社区访问计划

https://sui.io/resources-sui/community-access-program

Sui 社区非常多样化，SUI Foundation 考虑到社区受众对SUI Token的渴望，正在开展不同的SUI Token 分发计划：

开发人员资助计划 ：为构建原生开源应用程序的团队提供资金；

SUI委托计划 ：将 SUI Token 委托给社区运行的验证节点，以帮助引导他们的运营网络。

大使计划： 为社区成员提供工具和资源，以推广Sui项目和技术，并与当地社区分享Sui知识；

大学拓展计划: 挖掘未来的开发者，以便在Sui的生态系统中培养下一波建设者；

教育计划： 奖励那些花时间学习和研究Sui的人，与更广泛的Sui社区分享知识；

虽然目前只有开发者资助计划在运作，但其余计划正在积极开展中，并将在短时间内启动。Sui 基金会的目标是在 Sui 主网启动之前让所有这些项目上线。

SUI Token 的分配细节  

https://sui.io/resources-tokenomics/sui-token-allocation

Sui 基金会支持并赋能 Sui 生态系统中的建设者和创造者。重要的是，Sui 基金会通过在社区中传播 SUI 代币以实现最佳用途，在带头去中心化工作中发挥着关键作用。初期，由 Sui 基金会管理的社区储备拥有超过一半的 SUI 代币，它们将用于各种社区项目：

委托计划： 基金会将帮助引导社区运行的验证节点，并将通过将其质押权委托给第三方节点来促进网络验证节点权益的公平分配；

Grants 分配： 基金会将直接向开发人员、社区大使和其他在 Sui 上构建并创建教育材料的参与者分发代币，以使其更容易加入 Sui；

研发： 基金会将支持 Sui 协议的进一步发展，确保其能够一直保持优势；

验证者补贴： 基金会已经拨出大量代币来补贴网络早期阶段的stake奖励。

上面的饼图包含了主网启动时Sui token 分配的大致描述。然而，Sui 代币分配将会随着网络成熟到去中心化，发生一些根本性的变化。

对于已经在传播有关 Sui 的知识、将开发人员引入 Sui 以及测试和帮助改进 Sui 当前应用程序的产品体验的 Sui 朋友， 一旦主网启动，基金会将确保您参与 Sui 的Token分配。
