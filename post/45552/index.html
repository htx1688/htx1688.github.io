本文，我们将深入探讨传说中的NFT项目Frosties跑路过程，以及该项目的创始人在私吞超100 万美元后是如何被抓获的。

Frosties NFT跑路过程

Frosties是一个由8888幅作品组成的NFT系列，具有Doodles的轻松、有趣的风格。该系列于2022年1月9日上线，当时ETH价格超过3000美元，但新用户仍不断涌入该项目，希望在该项目上发财致富。

项目方对Frosties进行了大肆宣传，并承诺将其作为未来元宇宙游戏和质押机制的一部分。数万人关注了该项目的 Twitter 帐户并加入了他们的Discord，项目销售额约为 110 万美元。

随后，使用“Frostie”、“Jakefiftyeight”、“Jobo”和“Joboethan”等别名的创始人关闭了该项目的Discord和Twitter账户，并将110万美元转移到一系列外部钱包。项目方最后在推特留下“对不起”给他们的社区用户，当时一些用户已经在Frosties上花费了数万美元。

故事有个圆满的结局

这个故事有一个圆满的结局。

就在匿名二人组即将启动名为“Embers”的另一个可能带来150 万美元收入的诈骗项目之前，当局追踪到了他们。

洛杉矶Ethan Nguyen和Andre Llacuna被美国司法部逮捕并被指控串谋实施欺诈和洗钱。

Michael Fasnello 目前是AnChain.AI的加密货币合规官，他在此事件中发挥了关键作用，他利用自己法律、政策和合规专业知识和经验参与了调查工作。

Fasanello将该项目的社区钱包放到一个区块链分析软件中运行，该软件列出了项目创始人将社区资金挪动后的所有地址，包括使用零知识证明（例如最近被美国财政部禁止的Tornado Cash）和 Peel Chains 发送的几个地址。

Fasnaello联系了IRSCI（美国国税局的刑事调查部门），IRSCI对案件很感兴趣，并利用 Fasanello 的初步研究采取了下一步行动。

在三个月后，Ethan Nguyen和Andre Llacuna被逮捕，两人现在面临最高20年的监禁。

如何在RugPull事件中保护自己？

Rug Pull被定义为NFT或游戏项目的创建者寻求投资，然后突然放弃项目并以欺诈手段保留项目投资者资金的情形。整个Frosties案的法律程序，都是未来NFT社区如何应对Rug Pull的宝贵案例。

Frosties案件表明，在项目方跑路之后并不代表大家损失了所有。

Fasanello说：“区块链链上金融活动，例如日期、时间、交易额和合约地址都很清楚。在区块链数据之外描绘出完整的画面是一种整体方法，涉及多方协同工作，为的是拼凑不同的信息线索。追踪Frosties创始人我们依据区块链分析软件、执法部门的调查技能、受害者和证人的配合，以及私人分析师和调查人员。”

我们可以采取谨慎的步骤来追查 Rug Pull背后的罪犯，并为受骗的社区伸张正义：

1. 评估损害：

与传统金融渠道不同，区块链是公开的。当发生 Rug Pull 时，可以追踪被盗资金，因为犯罪者使用机制将资金转移出社区钱包。这些公共信息使社区能够评估从社区偷走了多少钱、去了哪里以及偷窃的幕后黑手。

2.收集证据：

只有在能够更全面地描绘诈骗者个人细节时，公开的链上数据才有价值。重要的是收集和记录社区和受害者的经历，以提供经验数据无法提供的细节，包括与 Rug Pull 组织者的虚拟互动以及与社区钱包的交易历史。

3.聘请区块链调查员：

市场上已经有一些区块链调查和分析机构，以应对Web3发展初期的阴暗行为。这些私人调查可以与官方调查同时进行，甚至直接支持官方调查。

4.获得关注：

获得媒体报道是宣传正在进行的 Rug Pull 调查的最简单的方式。信息在现代世界传播迅速，让尽可能多的人看到您社区的故事对于吸引尽可能多的证人和受害者分享他们的故事至关重要。

小结

虽然在Rug Pull事件中有可能挽回损失，但许多诈骗者在没有被绳之以法的情况下就消失了。

Web3世界充满不确定，目前的法律保护并不完善。投资者必须对市场进行深入了解，来保护自己资金安全。
