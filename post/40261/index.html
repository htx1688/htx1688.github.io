7 月 6 日中午 12 点，周杰伦在时隔六年后重磅推出的新专辑《最伟大的作品》先行曲 MV 正式首播，几分钟内，有关周董即将发行新专辑的消息也在瞬息间“刷爆”了大家的朋友圈和多个平台首页。在 MV 视频上线的短短 3 个小时内，播放量已经破亿，创造了音乐平台 MV 最快破亿播放记录。

7 月 14 日晚上九点，周杰伦《最伟大的作品》正式重磅上线各大音乐平台，一时之间大“杀”四方，霸榜多家音乐平台首页。“方文山把最好的词给了周杰伦”、“周杰伦新专辑销售额破亿”、“红颜如霜太好听了”等相关词条在微博热搜榜上整整悬挂了一夜。

周杰伦不愧是周杰伦，周董的影响力果然毋庸置疑，不减当年，在音乐市场中，杰伦出走六载，归来仍是华语乐坛天王顶流。

然而，在 NFT 市场上，杰伦的影响力却没有发挥出其应有的“大力奇效”，由周杰伦站台的 Phanta Bear NFT 系列（“幻影熊”）接连遭遇暴跌，其价格已经从今年 1 月 11 日所达的最高点 8.8 ETH 跳水至近期的 0.66 ETH，价格缩水率接近 98%，简直是“雷霆暴击”。

因而，受周董影响而追随入局“杰伦熊”NFT 的藏友们在这一波大跌中，几乎都被稳稳“套牢”。不论是在今年 1 月 1 日发行时以发行价 0.26 ETH 买入，还是在价格高点 8.8 ETH 时高位接盘，对于买了 Phanta Bear NFT 并且还持有在手上的玩家们来说，就一个字“亏”。似乎哪怕是周杰伦，在 NFT 领域也不能成为用户们百分百的“价格定心丸”。

明星流量加持 NFT 成了虚幻泡影？娱乐圈联动 NFT 也成了一场“闹剧”？

实质上，明星流量成为了 NFT 的一项重要流量池所在，娱乐圈联动 NFT 也成为了娱乐业和 NFT 业的双向发展动能，为两个圈子的融合突破提供了更加广阔的通道。

在周董新专辑《最伟大的作品》推出后，Azuki 官方在社交媒体上发文表示，Azuki 联合创始人以周杰伦新专辑《最伟大的作品》为涉及灵感为周杰伦推出了定制款 Azuki NFT 作品。Web3.0 音乐元宇宙平台 Muverse 推出的 YOLO Bunny NFT 盲盒开启白名单铸造，首批 Private-Mint 将包含部分致敬周杰伦《最伟大的作品》NFT。可见，周董的商业价值在 NFT 圈里还是毋庸置疑。当然，NFT 频频蹭上娱乐圈的热度，这也是急需实现“破圈”的 NFT 的一条最方便的路。

周杰伦布局 NFT：纵览周董的 NFT 商业盘

很多人都在好奇，周杰伦消失在乐坛的六年，杰伦在哪里？杰伦都在干什么？

事实上，在周杰伦抱着孩子“嘤嘤嘤”，捧着奶茶“吨吨吨”的同时，周杰伦还在积极指点他的 NFT 江山，布局他的 NFT 大盘。周杰伦是国内最早一批进入“NFT 游戏场”的明星玩家之一，并且他也是公认的玩 NFT 玩得比较专业的明星，在大大小小明星入局 NFT 盘点中，周杰伦三个字一定会出现在名单前头。

去年 12 月 31 日，周杰伦与昆凌夫妇分别在社交平台中晒出了 Phanta Bear 的 NFT 形象，周杰伦表示这是“哥收到的第一个特别礼物”。今年 1 月 3 日，周杰伦将 Instagram 头像换为 Phanta Bear 的 NFT 作品，并表示“换几个月头像，感受元宇宙的感觉”。有了周杰伦的宣传，Phanta Bear 的 NFT 作品影响力日扩千里，1 月 1 日发行当天，不到一小时的时间，10000 个 Phanta Bear 的 NFT 作品“全部清空”。

Phanta Bear 的 NFT 作品是由元宇宙平台 Ezek 联合周杰伦名下潮牌 PHANTACi 推出的，它和周杰伦的关系可以说是一根蛛丝线，显而不明。周杰伦在社交平台上利用自身强大的粉丝基础和社交媒体影响力为它站台，但它实际上却不是周杰伦本人所推出的自己的 NFT 作品，因而在发行 Phanta Bear 的 NFT 作品的过程中，周杰伦给自己的定位是一位喜爱者、收藏家。

而空降 NFT 市场的 Phanta Bear 的 NFT 作品就像是一个和 BOSS 有七拐八绕的亲戚关系的“关系户”，而“关系户”的那层说不清道不明的关系成功为它结成了“保护网”，因此它被众人捧起来了，“幻影熊”也有了新名字——“杰伦熊”。

周杰伦在社交媒体平台上为“杰伦熊”站台宣传，可以被看作只是他在 NFT 领域的浅试一下，接下来，周杰伦的一系列在 NFT 领域的动作都暗示着周董在 NFT 市场上的“野心”，他不仅仅要做个收藏家。在自己玩 NFT 的同时，周杰伦还带动周边好友一起玩起了 NFT，陈冠希、方文山、阿信、林俊杰，周杰伦以一人之力撬动了整个明星 NFT 圈，引起了明星 NFT 风暴。

今年 1 月底，周杰伦在“薄盒 Mints”上首次合作拍卖了从未公开的经典歌曲 Demo 音乐数字藏品。将周董的音乐作品和 NFT 结合起来，不仅更改了其音乐作品的留存形式，使得周董的音乐可以在传播扩散的广度和深度上都得到提升，而且通过音乐赋能 NFT，NFT 的价值也获得了进一步的显现，NFT 作品从 JPG 的局限中走出，收藏意义也大为增加。

4 月 1 日，周杰伦在 Instagram 上发文，称被电话告知黄立成之前送给自己的 BAYC #3738 NFT（“无聊猿”NFT）被钓鱼网站所盗。愚人节的这一则消息，将周杰伦和“无聊猿”NFT 重新炸回大众视野，也就是这件事让大众了解到了周董在没发歌的日子里还在做些什么——玩 NFT。同时，也是此事件让大家在初步了解到独一无二、不可篡改的 NFT 作品仍然面临着被盗风险，哪怕他是周杰伦，哪怕那天恰好是愚人节，周杰伦价值百万的“无聊猿”NFT 被盗仍然成为了当月行业动荡大事件。

6 月份，他和知名足球俱乐部巴黎圣日耳曼展开了合作，为了庆祝巴黎圣日尔曼的第 10 座法甲冠军奖杯，他们联合推出了一套名为“Tiger Champs”的 NFT。7 月 4 日，周杰伦与名厨江振诚联合推出 NFT 独家甜点“Afro Bear 爆炸熊”，该甜点将在 FineDayClub 举办的一场米其林元宇宙活动中推出。可见，在准备新专辑《最伟大的作品》的同时，周董也没有放弃他的 NFT 事业。

几乎在每一个月里，周董在 NFT 领域发展上都有新的突破，周杰伦的 NFT 事业线越走越清晰。唱歌、弹琴、魔术、演戏、拍电影、双截棍、NFT……周杰伦始终在不停的尝试新的领域，并且在新的领域里，周董总是能将兴趣做成专业。因此在周杰伦新专辑《最伟大的作品》发布后，尤其是《最伟大的作品》还是发行的数字专辑版本后，大家都在疯狂刷频期待着周董新专辑的 NFT 版本发布。

明星入局 NFT：NFT 领域如何借明星之力形成“破圈效应”？

周董卷动了华语圈的明星 NFT 风暴，而除了周杰伦和他的朋友们，华语明星里进入 NFT 圈的还有汪峰、伊能静、余文乐、徐静蕾、潘玮柏、胡彦斌、修杰楷、小马及小刀等众多明星艺人。在国外的明星 NFT 圈里，也不乏大咖顶流，汤姆・布雷迪、大坂直美、莱昂内尔・梅西、勒布朗・詹姆斯、麦当娜……

明星的加盟，也让 NFT 这个圈子有了更多 New Money 的入场。

自带庞大粉丝流量池的明星们在进入 NFT 圈之后，闻风而动的粉丝、歌迷、收藏家迅速抢占坑位，也随之加入 NFT 圈，买 NFT、炒 NFT，他们大多是既是粉丝，也是投机者，因而他们也有一个新名字“粉牛”。

买明星站台或者发行的 NFT 作品，一方面是追星的投机行为，另一方面拥有明星同款系列 NFT，对于他们来说也是身份的象征、炫耀的资本，仿佛拥有了明星同款系列 NFT，他们也就仿佛和明星踏入了同一个圈层。明星们的影响力实在是不容小觑，尤其是对于周杰伦这样“路人盘”也相当大的顶流巨星来说，他们在 NFT 领域的一举一动实质上都在刺激着粉丝、收藏家的敏感神经。

当然，也有很多明星追随者也在幻想着是否有一天他们也可以凭借着手中的明星 NFT 抢占演唱会的前台，毕竟加入拥有周杰伦的 NFT 就能够有机会获得周杰伦演唱会的门票的机会，谁会不心动呢？怪不得在周杰伦出新专辑《最伟大的作品》的时候，也不乏有小伙伴在各大平台下留言“ball ball 杰伦也出个《最伟大的作品》NFT 吧”。

同样如果以 NFT 的形式发行音乐唱片作品，在传播上不仅更方便，对于明星们的铁粉们来说，音乐作品在永久保存和长期流传上也变得更加方便了。另外，NFT 还以门票、明星海报、球星卡等具备一定收藏价值的载体在传播，名人明星亲自带货，也成为了将 NFT 直接推向普通大众群体的重要手段之一。

对于 NFT 圈来说，有了明星们以及他们所自带的粉丝流量群体的加入，NFT 想要“破圈”也变得容易了起来。明星总是与潮流相连接，追星族总是与年轻、接受度强等词语相匹配，而这刚好也是 NFT 受众群体的特性，所以明星 NFT 在 NFT 圈具有先天优势。明星 NFT 在作品呈现、发行形式、传播群体上都有着更为广泛的助力优势。

但是也是由于是明星发行或站台的 NFT 作品，明星在 NFT 圈不容易做到百分百专注，更多的是“玩票性质”，代为运营管理的往往是他们的经纪公司，所以导致了明星 NFT 在发行后缺乏后续运营维护，用户流失量也相当迅速。

同样也因为缺少专业的赋能机制，而让他们的 NFT 除了收藏、炒作以外，似乎别无他用，明星 NFT 的价值也犹如坐上过山车，当明星热度加持一过，明星 NFT 也很容易沦入谷底。如果再遇上现在的“熊市打击”，明星 NFT 作品显然没有其他的 NFT 作品在持久稳定上占优势了。

当然，明星 NFT 如果能够绑定更多的福利赋能机制，它们的市场前景还是不容小觑的，毕竟再次重申，没有人能够拒绝比如买了周杰伦相关 NFT 就能直接获得周杰伦演唱会前排门票的福利机会！

娱乐圈“NFT 登月计划”：是新领域的联动发展

除了明星进入 NFT 领域，整个娱乐圈行业与 NFT 行业的联动合作也是不少。小说、电影、电视剧都可以制作成 NFT 作品，各大娱乐头部公司纷纷进军元宇宙空间进行 NFT 产业布局。

国内，王家卫的电影《花样年华一刹那》未曝光片段被制作成了电影 NFT 作品，并且在苏富比拍出了 428.4 万元港币的天价。国外，《杀死比尔》的导演昆汀・塔伦蒂诺也将《低俗小说》中 7 个未剪辑的片段制作成电影 NFT 并且放在全球最大的 NFT 交易平台 OpenSea 上进行拍卖。

国内还有不少大热电视剧也正在被制作成数字藏品进行发行，《红旗渠》、《东宫》、《风起陇西》、《且试天下》……影视化数字藏品正在成为常态化。

当然，市场上出现如此多影视作品数字藏品的原因之一，还在于众多娱乐巨头公司的 NFT 计划，国内影视大哥“华策集团”就是早期“吃螃蟹”的人。国人童年记忆《喜羊羊与灰太狼》不仅被其制作公司奥飞娱乐制作成了 NFT，而且还面向海外发行了一版。即使没有影视剧作为作品素材库的艺人经纪公司，比如 SM 娱乐公司、乐华娱乐公司等专做偶像艺人的公司也正在布局 NFT 产业。

对于整个娱乐产业而言，发行 NFT 更多的在于“顺手的事情”，NFT 最重要的一点在于作品版权，而恰好各大娱乐公司不缺的就是版权，音乐作品、影视作品推行的同时发行 NFT，既是为自己造势，也是为自己圈入更多的流量，获得更大的曝光和收益。

当然，将 NFT 当成事业发展，对于各家娱乐公司是一条相当不错的产业联动之路，特别是在后疫情时代，演唱会停摆，电视剧拍摄剧组风险大，电影院随时面临因疫情而关闭的风险，娱乐业遭受重大影响情况下。娱乐圈与 NFT 产业积极联动，长期持续的发展下去，更像是一条娱乐业的“自救之路”。

虽然，娱乐圈和 NFT 的联动也会让本就浮躁的 NFT 行业变得更加焦灼，但是早已适应“赚快钱”、“抢流量”的娱乐产业能够寻求到一条娱乐产业和 NFT 产业的平衡之路，当热度过去，娱乐 NFT 不再坐上这趟“过山车”，而是越发展越稳定。
