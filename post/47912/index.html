编译：比推BitpushNews Mary Liu

最大的 NFT 公司之一正在进军比特币 NFT 浪潮。

Yuga Labs 宣布推出基于比特币的 Ordinals 协议铸造的 NFT 系列「TwelveFold」，将包含 300 件限量版生成艺术作品，每件都刻在比特币区块链上，这是迄今为止通过 Ordinals （序数）推出的最吸睛的 NFT 藏品之一。

Ordinals（序数）是一种允许在比特币区块链上创建 NFT 的协议，它已成为加密社区的最新热门叙事，截至2月28日，超过 213,000 个独特的 Ordinals被铸造，包括DeGods、OnChain Monkey等一些著名项目。

那么，Ordinal NFT 是如何工作的？Ordinal NFT 与其他 NFT 有何不同？如何铸造？

如何运作？

Ordinal NFT 使用 inscriptions（铭文）来工作，铭文由 Ordinal 理论通过由 Casey Rodarmor 开发的序数协议提供支持。Ordinal 理论旨在赋予 satoshis（比特币的最小单位“聪”，占一枚比特币的 1/100,000,000）“个人身份，使他们能够被追踪、转移并赋予意义。”

简单来说，Ordinal 协议为每个 satoshi 分配一个序号。在分配了这个数字之后，每个聪就可以通过比特币交易来刻上图片、文本或视频等数据。一旦该交易被挖掘，任意数据将永久成为比特币区块链的一部分，并且可以通过支持 Ordinal 的比特币钱包和在线Ordinal 查看器查看。

与其他 NFT 有何不同？

Ordinal NFT 和其他 NFT 之间的一个主要区别与数据存储有关。

大多数 NFT 都是通过 ERC-721 非同质代币标准使用以太坊区块链创建的。ERC 721 是一个标准，它概述了应该如何创建 NFT，以便它在整个以太坊生态系统中得到正确识别。创建 ERC-721 NFT 时，元数据文件——字面上提供有关其他数据信息的数据（因此称为“元数据”）——提供有关 NFT 的信息。最简单的理解方式是，NFT 本身就是一份证明另一项物品所有权的合约，在该元数据中有详细说明。对于最常见的 NFT，即数字艺术，实际的 JPG 或艺术文件通常存储在以太坊区块链之外，元数据包括指向该文件的链接。这意味着实际文件或艺术品可以更改，因为它没有嵌入区块链中。

比特币的Ordinal NFT 的不同之处在于，没有描述 NFT 的可参考元数据文件；相反，整个数据文件驻留在比特币交易的见证签名字段中。这意味着整个 Ordinal NFT 都在区块链上生存和呼吸。以太坊和 Solana 上的 NFT 主要指向包含 NFT 主图像的链下数据库，但比特币 NFT 或 Ordinals 利用比特币代码的最新更新将艺术品存储在区块链本身上。

这是否重要取决于用户，但Ordinal NFT 为 NFT 带来了额外的不变性。

如何铸造 Ordinal NFT？

Ordinal NFT 刚刚兴起，因此随着生态系统的建立，它们的可访问性受到限制。现在有两种主要的方法来铸造自己的 Ordinal NFT。

第一种方法是启动一个完整的比特币节点并在该节点上运行 Ord。然后你可以开始将 satoshis 写入你控制的钱包中以制作 Ordinal NFT。这种方法技术含量高，更适合懂技术的爱好者和真正热爱NFT的人。

另一种刻写 Ordinal NFT 的方法是使用无代码刻写工具。如果您信任比特币区块链，那么这是一种更加随意的体验并且效果很好。我拥有一个 Ordinal NFT，我使用名为 Gamma 的无代码工具铸造了它，对于熟悉比特币的人来说，它相对简单，但某些步骤可能有些棘手。

以下为铸造过程：

在开始之前，请确保您有一些比特币（价值约 50 美元），以便您可以支付 inscriptions（铭文）的交易费。你拥有的钱包必须能够将比特币发送到 Taproot 地址（支持 Taproot 的钱包和服务提供商列表：https://en.bitcoin.it/wiki/Bech32_adoption）。您需要查找的内容位于“发送至 Bech32m”列中，不是“发送至 Bech32”，请再三确认。Bech32m 列中弹出“是”选项表示钱包可以将比特币发送到 Taproot 地址。请注意，Coinbase 和 Binance 等热门交易所不支持 Taproot。

一旦您的钱包中有了比特币交易费，您就可以开始铸造您的 Ordinal NFT。这是使用 Gamma 铸造第一个 Ordinal 的分步步骤。也可以使用 OrdinalsBot 等工具，过程大同小异。

  • 前往：http://: https://gamma.io/ordinals

  • 选择您的inscriptions（铭文）类型，图像文件或纯文本。

  • 通过从桌面选择文件或直接在指定位置输入文本来上传所需文件。

  • 根据您愿意等待 NFT 铸造的时间来选择交易费率，请留意为各种费率选择提供的估计铸造时间。

这是该过程中棘手的部分之一。下一步是指定应将 Ordinal NFT 发送到的比特币地址。该地址需要是Taproot 地址或Ordinal兼容地址。

建议在此步骤中使用 Sparrow Wallet，这是为 Ordinal NFT 设置 Sparrow 钱包的分步指南：https://docs.ordinals.com/guides/collecting/sparrow-wallet.html。

设置 Sparrow 钱包地址后，将其放入 Gamma 网站上的“收件人比特币地址”字段。确保此地址的前四个字符是“bc1p”。此前缀表示该地址是 Taproot 地址。

  1. 通过将指定数量的比特币发送到指定地址来支付铸造 Ordinal NFT 的交易费用。请注意，此地址也将以“bc1p”开头（十分重要！！请再三确认）。建议用与接收 Ordinal NFT 的钱包不同的钱包支付费用，除非你真的知道自己在做什么并且非常熟悉这些操作。一旦不小心，可能会错误地发送一个之前刻录的聪，你将无法再访问该 NFT。

  2. 等待。这可能需要几天时间，具体取决于选择的费率，你将收到一个链接来检查铸币状态。

  3. 通过 Ordinal 查看器验收你的 NFT。

一如既往，请对你在比特币区块链上的行为保持清醒和谨慎。Ordinal是新的，不要忽视每一个细节，你在比特币上所做的一切都可以很容易地被追踪。所以请注意，无论你做什么，都能被追踪到。

快乐铸造吧！
