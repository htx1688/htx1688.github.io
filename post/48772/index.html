PFP 之后，NFT 市场的下一个新叙事是什么？

大家都在呼唤更有实用价值的 NFT，以星巴克为首在打造更多基于 Loyalty 的用例，而 Nike 在打造基于 Fashion 的虚实结合 utility

上次我在分析星巴克的时候写过，当会员数据上链后，会出现一次商业范式转移，之前各家的私域会员变成了公开数据，我的会员也是你的会员。品牌获取用户的方法从 pull 变成 push，价值回归用户，同时品牌需要学习新的与用户对话方式

但当时更多落在理论层面，让很多读者觉得不过瘾。

这周我收听了一期名为 Understanding Open Loyalty 的 Podcast, Smart Token Labs 的 CMO Brent 认为会员数据上链本质上是构造了一个 Open Loyalty（我翻译为开放会员系统），分享了宝贵的思考和案例，刚好补上了之前的遗憾，对我来说也是一次学习。

My member can get cool stuff because you can treat them like your member —Brent Annells

TL;DR

在 web2 深受「离线」之苦的行业，通过Token 和 Open Loyalty 捕获到了更多价值。

我整理了 3 个案例和价值点，欢迎阅读??

  • Karma 电动车：加速数字化，建立更小摩擦的生态联盟

  • Bogota Devcon：构建快闪 Open Marketplace, 捕获增量价值

  • 大众汽车：构建品牌获客新方式 Claim to Earn

“

一、Karma 电动车：加速数字化，建立更小摩擦的生态联盟

我们已经习惯一切被数字化的世界，但有些行业目前还在比较「离线」的状态，汽车就是其中一个

  1. 围绕车有丰富的第三方服务生态，包括注册（机动车管理局）、保险、维修、共享、租赁、抵押贷款、俱乐部…

  2. 第三方服务生态中，每一块的信息都是不完整且孤立的，缺少让所有生态相关方全面了解汽车信息的方式

以上两点造成

  • 车主体验差：需要主动找第三方服务，且反复提交自己的信息（一个个电话真的很烦）

  • 第三方服务效率低成本高：需要主动获取用户（广告、买数据…），一方面效率低成本高，另一方面常常给用户打搅而不是服务（用户买完 A 保险，又接到 B 保险电话）

这里面的矛盾点是，联网和自动驾驶，看似让汽车成为一个数字资产，但所有权（ownership）还停留在纸质年代且割裂。

2020 年，总部在南加州的豪华电动车品牌 Karma Automotive 发起一场黑客马拉松， Smart Token Labs 提供的 Karma Car Token 有机会更好解决以上问题

  • Karma Car Token 基于汽车唯一标识 VIN 号码创建，代表其所有权（ownership）

  • 该 Token 可以放在任何加密钱包中。基于 TokenScript protocols, Karma Car Token 同时也化身为数字钥匙，可以解锁和启动汽车，以及了解车所在位置

  • 该 Token 可以在以太坊上转移和出售

基于Car Token, 整个生态实现了 Open Loyalty, 用户同时成为所有第三方服务的会员，所有的服务都变的更简单

  • 出售&登记：用户直接在 marketplace 上 list Token 即可。在发现 Token 钱包转移后，机动车管理局可以直接读取新的 Token 相关信息（车主名、联系信息、地址）来更新注册信息，不需要原车主去车管所注销，新车主再去登记

  • 抵押贷款：车主通过 Token 借贷服务即可完成，BendDao 上不但可以抵押蓝筹 NFT，可以抵押车

  • 保险：保险公司可以知道这辆车的在保状态，并且按需给车主提供优惠，而不是一味的打搅

  • 车主俱乐部：基于 Token 可以更容易验证车主身份，且 token gated 也保障了社区和活动的私密性

  • 共享：NFT 共享一直没有找到很好的落地场景，共享汽车是一个很好的用例

于是我们看到，基于 Open Loyalty, 更小交易摩擦的生态联盟被建立，同时让用户和服务商受益

“

二、Bogota Devcon：构建快闪 Open Marketplace, 捕获增量价值

另一个「离线」的行业是会议，其痛点和汽车类似：

  • 参会人员的 profile 都非常精准，也有围绕其的一系列第三方服务，从线上到线下

  • 但参会人员很难被识别，同时由于会议的非持续性，构建数字身份和平台的ROI不够高（很难为一周的活动开发个电商网站），更难形成像汽车一样的生态联盟，捕获其价值

我们来看 Token 和 Open Loyalty 是怎么来解决这个问题的

Smart Token Labs 作为以太坊基金会的公共物品合作伙伴（public goods partner），为 Bogota Devcon 的超过 6000 名参会者提供了 token ticket. 也就是说，可以通过这个公开的 token, 找到这 6000 名高质量的以太坊开发者。

于是有趣的事情发生了，出现了 25 个服务方，给这些以太坊开发者提供权益，包括

  • Polygon 提供 After Party 的 VIP 席位（用来建立人脉做招聘）

  • VC 提供 Deck 的优先查看权

  • 有项目方在 Metaverse 里提供虚拟连帽衫

  • 设计师提供免费的个人主页设计

  • ……

通过 Token 和 Open Loyalty, 本身分散在各处的这些服务方、权益和以太坊开发者自然聚合为一个 Open Marketplace, 之前很难被捕获的价值可以被捕获，带给整个生态增量价值

“

三、大众汽车：打造品牌获客新方式 Claim to Earn

过去品牌主要通过广告来获取用户，有成本且精准度有限，找到 B 类用户更难。比如大众汽车想找一些 influencer, 这该如何去做呢？

基于 token, 可以创造更简单的方式。假设你是一个 Reddit 用户，在汽车subreddit 非常活跃并被奖励了一个相关的 token。现在大众汽车可以创建一个 Open Marketplace, 为持有汽车 subreddit token 的用户设置一些奖励，作为交换需要留下 email.

于是 influencer 通过Claim, 分享自己的数据和 email 获得了奖励，大众汽车找到了目标用户，双方互相捕获了价值。同时 Web2 广告平台上的投放成本被节省下来。

Claim to Earn 会成为 Web3 更高效的获客方式。

这个方式还可以解决体育行业的世纪难题：如何衡量赞助商的 ROI ?

过去赞助商在球场花巨资投放广告，但很难衡量到底带来了多少用户和收入。现在赞助商可以通过 Claim to Earn, 为持有比赛门票 token 的用户设置奖励或者折扣，精准获取到对其感兴趣的用户，以及计算 ROI. 世纪难题就这样被解决了。

“

总结&展望

通过会员 Token 构造的 Open Loyalty（开放会员系统）

1、加速「离线」行业的数字化，建立更小摩擦的生态联盟，同时让用户和服务商受益。

希望通过 token 数字化和打造开放会员系统的行业，一定是在 web2 深受「离线」之苦。汽车、体育、会议，还有什么合适的行业吗？

2、构建快闪 Open Marketplace, 捕获增量价值

会议行业由于离线+短暂的生命周期，让其中的 marketplace 价值难以被捕获。通过 Token 和 Open Loyalty, 本身分散在各处的角色被聚合在 Open Marketplace里，为整个生态创造了增量价值，用户和服务方再次受益

3、品牌获客方式，从投放广告变成让目标用户Claim to Earn

用户因为自己的数据获得奖励，品牌方用更低成本找到了目标用户，双方互相捕获了价值。在这整个 flow 里，web2 里被平台拿走的价值回归用户

PFP 之后，NFT 市场的下一个新叙事是什么？大家都在呼唤更有实用价值的 NFT，希望会员 Token和 Open Loyalty 创造的外部正向性, 可以把更多用户和价值带入 Web3.
