由 Coinbase 基于 OP Stack 推出的 L 2 网络 Base 最近引起了加密社区的热议，原因在于发行于 Base 之上的模因币 BALD 造就了“一日百倍神话”。 此后，数千万美元的热钱被吸引至 Base 链，土狗成百上千的出现。截止发文时，Base 链 TVL 已突破 8000 万美元。

然而，除了模因币以外，Base 链上有哪些 DeFi 项目可玩？本文整理自 Crypto 研究员 2 Lambro 的推特线程。

Base 链能够爆火的原因，无非是：

• Coinbase + OP 的强大知名度

• 模因币龙头 BALD 的造富神话，价格增长了 X 倍

然而，Base 链目前只有一个没有前端的单向跨链桥（以太坊 → Base），所以体验起来非常的早期。

如何看待 Base？是否有发展前景？

这个问题取决于：

• Coinbase 在未来会采用何种能够影响生态系统的增长方式？

• Coinbase 是否为会 Base 链设计激励措施？

• Base 链能否帮助 Optimism 生态系统作为成功的 L 2 ？能否为 OP 代币带来价值？

就目前来说，对 Base 的未来下结论还为时过早，因为目前大多数正在运行的 EVM 链只有一种“分叉”罢了。

加密社区喜欢新鲜事物，但是只是不同“颜色”的相同“玩具”也会很无聊。

总体而言，Base 链值得一试，因为加密行业常常会奖励那些尝试新事物的人。

如何玩转当前的 Base 链？

首先在 MetaMask 钱包中设置 Base 网络，可以通过 Chainlist（Chainlist.org）一键添加网络，也可以手动添加以下内容：

- Network: Base Mainnet 

- RPC: https://mainnet.base.org

- Chain ID: 8453 

- Currency: ETH 

- Explorer: https://basescan.org

ETH 代币跨链（以太坊 → Base）：

• 目前还没有跨链桥，必须手动将 ETH 从以太坊发送到代理合同

• 将 ETH 在以太坊上发送至 0x49048044D57E1C92A77F79988D21FA8FAF74E97E

• 注意，该合约目前只能接收以太坊主网的 ETH，请勿从其他任何 EVM 链发送除 ETH 以外的其他资产。

1. 跟随“聪明钱”和“巨鲸”的动向

目前，就像许多其他的早期用户一样，你仍然可以在 Base 链上追逐模因币。

虽然我不推荐，但是如果你真的想参与，那么跟随“聪明钱”和“巨鲸”是最好的方法。

可在此 Dune 仪表盘查询钱包价值排行榜：https://dune.com/tk-research/base

2. 参与早期的生态系统

除了追逐模因币外，目前正在 Base 上“做事”的项目也不在少数，生态项目正在快速扩张。这其中，DeFi 项目尤为之多。作为一个新链上的新项目，代币空投的预期很高。我将这些项目划分为三种：#值得一试、#观望、#高风险或许高回报。

#值得一试：

Void Exchange

建立在 Base 上的永续 Dex，目前处于测试网阶段。

GitBook 项目文档写的非常详细。

快速研究了一下，Void 看起来像是一个改进版的 GMX，有独立的金库，但是共享稳定币。

Aerodrome

一个 Solidly Dex，营销似乎还可以，正在向 $veVELO 代币持有人空投。

Pike Finance

一个去中心化借贷平台，与其他 Base 生态项目相比，Pike 似乎更注重合法性。

LeetSwap

从 Canto 迁移到 Base 的 Dex，已上线主网，用户使用较频繁。

RocketSwap

一个 Dex，已上线主网。
