近期，Solana 生态 NFT 项目 DeGods 发推称，将于明年第一季度桥接至以太坊 ，其二代项目 y00ts也将于明年第一季度桥接至 Polygon，并于下月发布更加正式的 y00ts: Season 2 路线图。受此消息影响， DeGods 和 y00ts 24 小时交易量分别增长了 207.74% 和 303.67% 。

DeGods 是 Solana NFT 的蓝筹项目，其玩法上的创新机制给它带来了超过十倍的涨幅，也让 DeGods 成为了整个 Solana NFT 生态系统中最强大的社区之一。但由于 Solana 生态与 FTX 的密切关系，整个 Solana 生态现在都遭到波及，DeGods 也未能幸免。自 FTX 事件发生以来，其地板价跌幅超 70% ，“逃离”Solana 去其他公链也成为了其规划中的重要一步。

不管是用户量、交易量还是流动性，以太坊一直都占据着主导优势，也是 NFT 开发者的首要选择。因此，DeGods 迁移至以太坊并不让人意外。

但是，其二代项目 y00ts 并没有跟随 DeGods 的脚步，而是选择了 Polygon，多少有点出人意料。因为相比以太坊，Polygon 并没有真正强大的 NFT 生态系统，安全方面也并没有以太坊成熟。y00ts 为何会做出这样的决策呢？

Polygon 的 NFT 布局之道

在布局 NFT 方面，Polygon 颇有远见。

先是成为了除以太坊外 Opensea 首批集成的公链之一，凭借着廉价的成本以及快速的交易体验，吸引了众多 NFT 用户，交易量一度接近甚至超越以太坊，逐渐成为 NFT 用户数量第二的生态。

后又成立了 Polygon Studios 决心扩张链游和 NFT 版图。目前已经投资了超过 36 个项目。这些项目也几乎都会部署在 Polygon 链上。

虽然在熊市时，Polygon 的各方面数据都有下跌，NFT 市场也没那么活跃。但 Polygon 也看到了新的突破口，加快与 Web2 及传统企业建立合作关系。 在过去的几个月里，Polygon 吸引了大量知名品牌，目前已经与可口可乐、星巴克、Draftkings、Meta 等顶级品牌合作。

其中，Reddit 于 Polygon 发行的 NFT 系列 Reddit Collectible Avatar 在短时间内迎来了交易量高峰，并一度达到整个主流市场交易量的 25% 。目前，Reddit Collectible Avatar 总量已突破 700 万，持有地址总量超 500 万个。

作为一个主流的 Web2 线上网站，Reddit 拥有超过 15 亿注册用户， 4.3 亿月度活跃用户，以及 5200 万日活跃用户。虽然 Reddit NFT 持有地址数量和其每日活跃用户相比是小巫见大巫，但是此次尝试不仅会让许多对 NFT 持观望态度的 Web2 公司效仿，打造关于 NFT 的新叙事，还能为 Polygon 扩大用户规模，提高流动性。 DeGods 创始人 Vora 也表示， Polygon 与知名品牌的积极合作是他们做出选择的原因之一。

除此之外，Polygon 还抓住了明星效应。

在上一轮 NFT 牛市中，我们见证了明星效应的威力——不仅能为 NFT 带来了实实在在的流量，还深刻影响着 NFT 的价格。由于 NFT 自带文化和娱乐属性，明星效应也多集中在娱乐媒体等领域的名人。而 Polygon 此次却推出了前美国总统特朗普的 NFT 集合。与政商界名人合作布局 NFT 也并不是没有先例，例如推特创始人 Jack Dorsey 的推文 NFT、马斯克的 ELONGATE 等。虽然这些 NFT 的销量并不全然理想。但特朗普 NFT 自发售后 6 小时就售罄，一度登上 Opensea 24 小时交易榜首位。Nansen 的数据显示，在 NFT 发售当周，有超过 332, 000 名用户在 Polygon 上铸造 NFT，或受此推动，Polygon 日均交易量已超越 BNB Chain。

Polygon 正是利用了这些优势，在早期的几个炒作周期中获得牵引力。数据显示，Polygon 链上交易数量已经超过 9.6 亿笔，并且 234, 000 名合约创建者部署 778, 000 份智能合约。截至 12 月 26 日，Polygon 区块链上独立地址数量超过 2.02 亿个 。

Solana 上的主要项目宣布支持 Polygon

在 y00ts 之前，已经有 Solana 生态的主要项目宣布支持 Polygon。如最初专用于 Solana 的单链钱包 Phantom 宣布支持以太坊和 Polygon，以将其服务范围扩大，并正式从单链钱包升级到多链钱包。

还有 Solana 生态最大的 NFT 市场 Magic Eden 也宣布将与 Polygon 集成，并将首先在 Polygon 上线 Launchpad 和 Marketplace，为 Polygon 链上游戏开发者和创作者生态系统提供支持。

这些消息都是在一个月前发布的，不免让我们觉得 y00ts至少在一个月前就已经开始计划。从接收 Polygon NFT 的钱包到集成 Polygon 的 NFT 市场，似乎都在为 y00ts铺平了迁移的道路。

Polygon 是最佳选择吗？

虽然 Polygon 近期的表现不错，但 y00ts 迁移至 Polygon 仍然面对诸多质疑声音。

首先是 Polygon 面临着技术问题，如 3 月因节点故障而遭受 11 小时的网络中断以及 Polygon PoS 创世合约中的漏洞问题造成用户资金丢失等，这些都不免让人担心 Polygon 在网络稳定性、资金安全上是否是理想的底层。不过 Polygon 于今年 10 月推出的 Polygon zkEVM 公共测试网最后阶段也已上线，据官方表示该技术能够显著提高 Polygon 的每秒交易量，主网很快就会上线，也算是 Polygon 在技术方面的突破。

其次，有传言表明 y00ts 决定迁移至 Polygon 是因为接受了 Polygon 的资金资助。Polygon 用其合作伙伴基金的赠款进行支付，预计这项赠款将持续一年。但是 Degod 创始人 Frank III 否认了这个说法。

Frank III 表示，“做出这一决定是为了“探索新的机会”并允许系列的持续增长，这是另一个实验，是将 NFT 项目放到他认为将推动下一波 NFT 采用的平台上的问题。他将做对他的公司生存最有利的事情。因此，当时机成熟并且机会一致时，他们可能会再次做出改变。”

当然，虽然种种迹象都在表明 Polygon 正在引起大家的关注，但这也并不意味着 NFT 未来不会在其他链上爆发。有时，生态爆发的开端只需要一个足够特殊的项目或产品，就能引发流量增长。从最近的风向来看，Polygon 正积极承接项目、资金与流量。
