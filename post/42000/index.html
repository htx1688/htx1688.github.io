Web3 、NFT 与实体产业

说到 Web3 或 NFT，大多数时候第一感觉是数字的、虚拟的，因为这些词源自技术并且跟数字科技有天然的匹配关系。其中 NFT 作为数字通证在整个体系中承担确权任务，拥有一个 NFT 意味着你拥有 NFT 本身，以及相关各方对 NFT 拥有者许诺的各种权益。 

纯数字和线上的业务离大部分人的生活较远或不必要，不过以 Web3 模式对实体产业的改造活动已经在世界范围内悄然进行着实验。

① 买实物商品送 NFT

这里 NFT 只是具备收藏价值的“购物小票”。

② 买 NFT 送实物商品

采用这种方式做零售的项目其实不少。如 Adidas 发布的 Into the Metaverse NFT，花钱买 NFT 可以得到两套全球限量 Adidas 运动服，前段时间已经开始发货。这里 NFT 相当于“可自由转让的预售权凭证”。

Adidas NFT 在实物发货后开始转向纯 Metaverse 体验构建，其价值远超两套运动服。

③ NFT 形式的商品订阅服务

现实生活中商品订阅服务有如城市中每日蔬菜配送、每日一份鲜花送到办公桌上等等。和 NFT 结合的点在于用 NFT 确认订阅权限，订阅信息写在区块链上，公开可查、不可篡改，并且订阅权限所有者可随时将权限转移给其他人。

实际项目大多不会单独使用这种结合方式，因为没必要，只是一个噱头，真实的项目会将这种模式同其他 Web3 特性混合使用，比如 DAO 治理模式等。见④。

④ 基于 Web3 经济模式与治理模式的项目

这套体系很复杂，讲个参与过的小项目。去年有个 NFT 项目叫冰激凌， 项目发起者称自己有生产和销售冰激凌的经验，他发行了一万个 NFT ，NFT 销售款作为冰激凌厂启动资金，他负责建厂和经营。（这是一个众筹过程，不需要在众筹平台上进行，整个过程的资金和权利转移都写在链上，不可篡改 ）

所有购买 NFT 的投资人会形成一个治理社区，冰激凌工厂生产和销售的利润将进入社区钱包，所有 NFT 持有人可发起投票决定这些钱如何处理。社区也可以对项目未来发展进行投票。（DAO治理+投资分红 ）

NFT 持有人终身免费吃他们自己的冰激凌。（商品服务订阅 ）

这个冰激凌品牌还未真正创建，就已经拥有成千上万个忠诚于品牌的用户。他们虽然吃冰激凌不买单，但他们不会糟蹋他们自己的钱，并且会不遗余力的向周围人推销自己的冰激凌。（品牌 ）

冰激凌项目后来失败了，社区信心没有真正建立起来就急于开启销售，导致 NFT 滞销，当时市场上并不缺乏可投资标的，所有潜在投资者马上放弃了这个项目转而将精力放在其他项目上。

案例「无聊早餐俱乐部」

无聊早餐俱乐部（The Bored Breakfast Club）是世界上第一个 Web3+咖啡项目，也是我目前唯一见过的使用 NFT、半中心化 DAO 等方式重塑了实体产业链，并且目前看上去在商业层面已经形成闭环的项目，项目真正做到了让全体 NFT 持有者持续喝到免费咖啡。

项目架构

早餐俱乐部项目方

这个团队的任务是发行 NFT，构建社区。他们完全遵循公链 NFT 项目玩法来做 NFT 发行，发行能力中上等。团队以“现实世界咖啡订阅服务”作为他们 NFT 项目的亮点，围绕这个点去打市场。

YES PLZ
Yes Plz 是一家传统咖啡豆制造商，早餐俱乐部项目方和 Yes Plz 是战略合作关系，早餐俱乐部需要实体咖啡商品，Yes Plz 负责制造商品。另外在社区构建过程中 Yes Plz 持续扮演者“咖啡导师”的角色，普及咖啡知识，组织线上品鉴会等。

类比 Web2，早餐俱乐部项目方属于品牌商，Yes Plz 是代工厂。不同的是，NFT 项目中光是 NFT 的生产销售就已经创造价值，而且是核心价值，实体商品生产在这里属于额外赋能，并非核心。

Roadmap

先看项目 Roadmap，NFT 项目的 Roadmap 都很粗框架，只是关于项目要点的一系列描述。像很多 NFT 项目一样，早餐俱乐部的 Roadmap 也改过 N 版，毕竟整个 Web3 都太新，没有多少成功范式可供参考，都是边做边改。

2021 年10月版本 在 NFT 售出20%时，向所有 NFT 持有者投放第一包免费咖啡豆。在 NFT 售出100%时，向所有 NFT 持有者投放第二包免费咖啡豆。显然在这一版本中早餐俱乐部只是拿实体咖啡当做买 NFT 的赠品。

2021 年11月版本 更新内容：1、所有咖啡全球包邮。

2、建立社区钱包。项目方会往社区钱包里放5ETH（当时价值2万美金），并且 NFT 二级市场交易版税(5%)会进入社区钱包，当社区钱包中的资金够给所有 NFT 持有者发一份免费咖啡时就会统一发一次货。*NFT 在二级市场销售时可设置给 NFT 创建者分配交易费用的n%（通常5%-10%）作为作者税收，对于热门项目来说这笔收入非常可观，比如上面提到的Adidas NFT半年来版税收入接近2000万美金。

3、开发新的周边产品为社区增加价值。

2022 年后 提到了品牌早餐、咖啡博物馆、其他周边商品、虚拟咖啡商店等新内容。

NFT 设计、推广与销售

无聊早餐俱乐部的 NFT 属于 Banner 赛道，这种 NFT 的美术工作量很大，且由于不能作为头像使用，在宣传推广方面限制大，可谓性价比很低。NFT 被设计为无聊猿享用早餐的场景，由环境、装饰、餐品、用餐者（双手）等组件随机拼接而成，单就美术设计来讲属于上乘之作。 

_*__由 Park St. Studio （@_parkststudio） 广告工作室提供设计。

在启动阶段早餐俱乐部以 NFT 项目惯用的方法来推市场，做 Giveaways，给OG早鸟、白名单等身份，吸引潜在投资者进入 discord。

早餐俱乐部 NFT 在2022年1月初开始销售，售价0.08ETH（当时价值300美金），总量1万枚。销售一周后售出4000多枚，这在当时并不是一个好销量，当时几乎所有 NFT 项目都能在发售后几分钟内售罄。

一级市场首发未售罄之前，二级市场交易量会受到极大限制。此时团队决定将总量削减为5000枚。两天后5000枚售罄。项目团队一级市场销售收入400ETH（当时价值160万美金）。

经济模型

项目方一级市场收入160万美金，拿钱去做web2品牌推广、采购 Yes Plz 商品、向社区钱包进行一次性捐赠，以及向全球会员发送货物。

NFT 发售完毕之后，项目方工作重心开始转向品牌构建和电商平台拓展。除了承诺免费向 NFT holder 提供的咖啡外，项目方还将会员咖啡发货剩下的商品对外出售，并且定制主题咖啡商品进行线上销售，每包单价$25。目前定制咖啡商品有乌克兰??主题咖啡、母猴子主题咖啡。他们还在积极寻找品牌合作方推出联合咖啡。

值得一提的是，项目方在项目初期采购了4只无聊猿（包括变异猿），这4只猴子到今天的价值升值接近10倍，所获得的利润甚至比早餐俱乐部整个项目的利润更多。与此同时，早餐俱乐部的品牌价值仍在不断积累。可见在虚拟世界中，品牌是最值钱的商品。

社区（成员）侧

社区钱包接收来自二级市场交易版税（截至5月opensea单平台版税26ETH约10万美金）、会员商品库存零售利润、主题商品利润等几方面的资金，这些资金让早餐俱乐部 NFT holder 在不到半年内获得了3次免费咖啡送货，足够每天一两杯。

如果你积极参加社区活动，还能至少额外获得1包，以及咖啡杯等周边赠品。
NFT holder 可以以折扣价格购买早餐俱乐部电商网站上的咖啡。

当 NFT holder 需要流动性时，可以转让自己的早餐俱乐部 NFT，地板价长期维持在 0.05ETH。
我未被允许知晓电商平台销售状况，推测销量欠佳。

长期发展方向

大多数 NFT 项目只在发售时受到追捧，短期拉高然后回落甚至破发，再然后销声匿迹。这是90% NFT 项目的生命写照。
从 NFT 项目角度看，早餐俱乐部项目还算不错，1月发售，到4月才算确定破发，而且破发不严重没有直接归零。从实体产业角度看，早餐俱乐部的咖啡生意目前还是主要由 NFT 二级市场税收来驱动商品制造，还没有在传统需求市场上站稳脚跟。

总体来说 NFT 和咖啡商品相辅相成，现实使用性托着 NFT 价格和需求，NFT 交易又使得所有人能喝到免费咖啡。让项目能够更加长久走下去的方法有 NFT 和实体产业两个途经：NFT 方面时不时推一下市场制造利好刺激价格和交易量拉升，为循环注入新的资金；实体商品方面尽快构建品牌护城河，让实体产业脱离 NFT 也能独立运营。

商品生产

假设我没有固定死忠的咖啡品牌，那我就喝早餐俱乐部的咖啡，Yes Plz 咖啡在国际上也获过奖，也是品质很好的咖啡。早餐俱乐部从 NFT 人群这个细分赛道进入，将它的品牌影响力植入某些人的生活，从而帮助它的实体咖啡产业活下来。

基于以上，Yes Plz 咖啡不再是为了满足用户喝咖啡的需求，而是在用户玩 NFT 参与 NFT 社区的同时顺带满足用户喝咖啡的需求，那么在商品设计生产上，就需要围绕 NFT 和社区来进行。

早餐俱乐部的咖啡包装都使用 NFT 猿猴作为品牌形象，强调了 NFT 趣味。商品生产批次根据社区钱包资金积累的节奏而定，不是预先生产，也不是按需生产。甚至咖啡的口味，也需要根据联名的 NFT 形象气质而做出相应适配。

商品确权

传统商业商品确权很简单，在电商下单记录或者线下一手交钱一手交货。基于 NFT 的商品确权需要执行一次区块链快照，收集到在某一个时刻哪些区块链钱包持有多少个早餐俱乐部 NFT，那么这些地址就拥有提取商品的权限。登录早餐俱乐部官网，连接区块链钱包，就能查看当前我可以领取的咖啡数量，填写邮寄地址，等待货物到达。

全球配送

全球物流是难以解决的问题。我接触过3个需要全球运送实体产品的 NFT 项目，无一例外一地鸡毛。从发售到5月，早餐俱乐部 discord 共收到550个客服工单，其中99%是关于物流问题的询问，大半客户拿着“运送物品遗失单”找过来要求补发。Web3 是分散的去中心化的，这意味着如果涉及到商品运输，这种分布式将导致噩梦。所以不要小看全球运送，一般团队完全搞不定这件事。

Discord 社区运营

项目具备现实使用性，早餐俱乐部 discord 也在处处强调 IRL（in real life），比如 #share-breakfast-irl 中几乎每天都有人分享自己的餐饮照片。

社区的常规活动是叫做 share a cup 的积分活动，对社区活跃做出贡献（参与游戏、参与活动、参与twitter raid等）可以获得积分，积分可以换取额外实体咖啡或其它周边。

咖啡研讨。喝咖啡和手冲咖啡是一部分人的爱好，项目的实体商品也是咖啡，所以也会有一些围绕咖啡话题的频道，如 #coffee-chat 用来分享烘焙咖啡、冲咖啡的知识。

项目发过4个POAP，2个咖啡商品领取徽章，2个元宇宙咖啡品鉴会签到徽章。和其他已发售 NFT 项目比早餐俱乐部无疑是勤奋的积极的。

其他媒体平台表现

早餐俱乐部在YouTube有频道但是没运营起来。

从项目Twitter数据能看出在 NFT 市场下行时项目数据表现较差，这从侧面表明早餐俱乐部项目仍处在 NFT 市场生态中，并没有发展稳固成为一家实体企业。

早餐俱乐部的 Spatial 虚拟展厅平时用于陈列项目相关的展品，也经常用于举办线上聚会。

品牌联合

早餐俱乐部目前的重点工作是寻求品牌联合，和其他 NFT IP 做联合，发布主题咖啡商品，以这种方式使目标 NFT 项目的社区成员成为 Yes Plz 咖啡的顾客。

Web2 实体商业逻辑是发现顾客的需求然后满足顾客的需求，而在 Web3 NFT 领域可能是先跟一群人做朋友，然后问他们“我有很酷的印着你头像的咖啡，你喝不喝”。

结尾

Web3 或者说玩 NFT 的人有实体商品购物需求吗？是有的。在早餐俱乐部 NFT 发售两周后，销量惨淡，没有明显拉盘，纸手已经出货离场，NFT 投机因素影响已经大大降低。

然后我对 discord Chinese 频道作了5天的聊天内容统计：20个社区成员有过交流，其中7个是真正聊咖啡的，另有2个人对项目持消极态度，但大部分人希望长期持有 NFT 用于喝咖啡。中文频道尚且产生了真实用户，更不用说本来就有咖啡文化并且物流便利的英文频道。

我看好并支持所有具有现实实用性的 NFT 项目，这对于 Web3 破圈有着重要意义，早餐俱乐部推特下有大量“这是我的第一个NFT”、“这是我的第一个POAP”等评论。

早餐俱乐部项目还在平稳运行，虽然我的3包咖啡全卡在荷兰，但不影响我观察这种商业模式的兴致。我会长期持有早餐俱乐部 NFT。
现在实体产业很卷很难，可以尝试破圈，有 Web3 的加持一切可能会变得不一样。
