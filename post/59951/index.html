EdgeMatrix Computing (EMC) 联合创始人兼CEO Zed作了主题为《AI+Web3如何塑造全新数字资产》的主题分享。

Zed表示，EdgeMatrix Computing (EMC) 创新地将 AI 与 Web3 结合，构建了一个综合性的数字资产生态系统，EMC 不仅仅是一个Web3项目，更是一个独特的供应链和生态。

EMC 生态由多种角色构成，包括普通用户、开发者、节点以及Token持有者等。

EMC协议基于区块链技术连接分布式GPU节点，将AI任务调度分配给适合的GPU并完成链上记账，这个过程与比特币矿机的PoW相似，称之为PoC（Proof of Computing）。

在EMC协议层上构建了 EMC Hub（emchub.ai），它不同于传统的DEX只交易通证， EMC Hub方便用户交易 GPU 算力，购买或出售 AI 模型，甚至发布自己的AI DApp。

对于开发者来说，无需深入了解 AI技术，只需调用 EMC 提供的 API 和 SDK，即可轻松构建 自己的AI应用，而使用GPU算力的成本只相当于传统云服务的25%。

Zed介绍称，EMC 创始团队中包括前微软云亚太总裁和前亚马逊AWS的高管，对云业务的熟悉使EMC选择了去中心化算力这个方向，由此能够大幅降低基础设施成本，同时保持较高的利润率。

个人或专业开发者都可以运行 EMC 节点，他们可以像矿工一样使用个人电脑或大型 GPU 集群来提供AI服务（挖矿），并获得奖励。

在 EMC 生态上，模型和应用都可以转化为金融资产，并在 AI 交易市场上交易。

架构方面，协议层由代币持有者控制，确定网络的整体协议。物理层由遵循协议的所有节点组成，无论是单一的还是集群的。

交易市场包括 GPU 算力交易、模型交易和应用交易的Marketplace，资产层则是 EMC 生态的最上层，代表了 EMC 的资产收益。

Zed指出，EMC 虽然定位AI的基础设施，但其上的金融衍生品丰富多样。例如，通过DeFi协议，投资者可以集资共建一个节点，也可以将算力质押借贷。

EMC的技术特点在于，它整合了AI的各个环节，从模型、数据到算力，全部上链，并且被完全映射到了Web3平台上。这种全面的集成在当前市场上是相对罕见的。

与传统的通过云服务调用不同，现在的模型和数据完全基于节点调用，前端运行于区块链上。利用EMC提供的框架，开发者能够轻松构建出完全基于区块链的去中心化AI应用，而不需要深入理解AI的专业知识。

从市场的角度来看，目前全球的主流芯片硬件设备租赁成本正在不断上涨。Zed表示，GPU现在是一个刚性、高质量的资产，购买二手设备也是可行的。他提到，EMZ Hub计划为开发者提供奖励，预计在三个月内可以覆盖所有的模型。此外，与Hugging Face相比，EMG Hub在使用时更为便捷，驱动、配置和优化参数都是自动运行在容器中，完全基于区块链运行，为用户提供了一站式解决方案。

此外，Zed还宣布，EMC 即将在日本推出全球首创的的AI算力指数（CP Index），为投资者提供了全新的投资机会。
