5 月 Bitcoin 链上 Ordinals Inscription 以及 BRC-20 的火爆，重燃起社区对 Bitcoin 链上生态的关注。据 @bcrypt5 数据面板，截至 6 月 15 日，当前已有近 1200 万个 Bitcoin Ordinals Inscriptions 被铸造，累计支付 1,714 BTC 的费用。伴随着早期巨大的财富效应，越来越多并无实用性的各类 BRC-20 Memecoins 涌现，目前已有数万个 BRC-20 Tokens 被部署，其交易大幅度占据了区块空间。Glassnode 数据显示，尽管当前 Ordinals 投机之风逐渐趋于冷静，但 Bitcoin 链上仍约有 30% 的交易采用 Taproot，而年初仅为 1.5%。区块大小增加，导致交易成本增加。随着投机泡沫减弱，市场逐渐进入关注 Ordinals 基础设施建设的阶段。

Bitcoin Ordinals Inscription 带来的重要问题之一便是 Bitcoin 区块容量难以支持更庞大的基础设施建设的问题。关于扩容方案，有如 BCH 直接修改 Bitcoin 协议以增加其交易容量的 L1 扩容方案，以及如 Stacks 在 Bitcoin 的基础上构建了一个全新网络的 L2 扩容方案等。BTC L2 Stacks 也因此顺理成章地坐上了 Bitcoin 扩容的顺风车，是被社区看好可能会获得长足发展的协议。Stacks 采取证明转移（POX）的共识机制，矿工需要花费 BTC 以进行新的 STX 挖矿，使得 Stacks 可以借用 Bitcoin 的安全性，同时也让 BTC 能够在 Stacks 链上被利用，运行智能合约和 dAPPs。值得注意的是，Stacks 是第一家通过 SEC Regulation A+ 框架进行通证发行的区块链公司，STX 作为 Stacks 网络原生 Token ，主要用于支付网络费用和矿工奖励。

本文将盘点 Stacks 上的主要生态应用，根据官方推荐、协议数据、社交数据、笔者关注等角度综合筛选。

钱包

Hiro Wallet

Hiro Wallet 是 Stacks 链上最常用的开源钱包，帮助用户存储、接受或发送 Stacks 网络上的资产，支持 Ordinals，但尚未集成闪电网络。此外，用户还可以通过 Hiro 进行 STX 质押。当前该钱包支持浏览器扩展和桌面应用程序运行，但浏览器扩展支持的功能比桌面应用程序更多。官方数据显示，Hiro Wallet 已获得超过 28 万次的下载。

Xverse

Xverse 支持用户在 Stacks 区块链存储、接受或发送资产的非托管钱包，支持 Ordinals，并添加了生物识别功能提高钱包安全性和便利性，但尚未集成闪电网络。当前用户除了可以在浏览器扩展体验 Xverse ，还可以在 Android 或 iOS 移动端使用该钱包。用户可以通过由钱包官方团队打造的 Xverse pool 质押 STX 赚取 Bitcoin，在第 61 轮质押周期中（每 12 天为一轮），有约 300 名 Stackers，平均年化为 9%。

根据官方发布的路线图，Xverse 在 Q2 的目标是支持硬件钱包、子网集成、多签集成和 xBTC 集成，并计划在 Q4 通过闪电网络和 sBTC 实现 BTC 快速支付。

GoSats

GoSats 是由印度团队开发的聚焦于印度社区的 Bitcoin 钱包，其愿景是让每个购物者、消费者和储户都使用 BTC，并推出了 GosSats Visa 卡、忠诚度计划等。用户通过 GoSats 使用 BTC 消费可以获得 Sats（聪）的返还。

DeFi

ALEX

ALEX 是由非营利性组织 ALEX Lab 基金会支持的构建在 Stacks 链上 Dex，用户可以在该平台上进行交易、质押、流动性挖矿、跨链，以及包含彩票和 IDO 的 Launchpad 等功能。目前 ALEX 支持 ETH 或 BSC 上的 USDT 桥接至 Stacks 网络成 sUSDT。当前 ALEX TVL 约 1855 万美元。此外，ALEX 还建立了链上订单薄 BRC-20 交易市场，并配备 K 线图，当前支持 B20、PIZA、SHNT、MAXI 和 LONG 等数十种资产，但近 24 小时交易量较为惨淡。

ALEX 已发行 ALEX Token，主要用于流动性挖矿激励、质押收益以及平台治理。在今年 1 月中旬，以 ALEX Token 为核心，推出了 ALEX Genesis DAO。

Stackswap

Stackswap 自称是 Bitcoin 链上第一个功能完备的 Dex ，允许用户进行资产交易、流动性挖矿、质押、跨链、Launchpad 和 NFT 等功能，并已发行 STSW Token。目前 Stackswap LP Pool 中拥有超过 45 万美元的资产以及价值超过 530 万美元的 IBTC 质押。值得注意的是，该平台 UI/UX 设计极为简洁，社交媒体也不太活跃，可能是由小团队建造，但从去年 11 月起至今坚持在 Medium 上发布双周报告。

UWU Protocol

UWU 是建立在 Stacks 链上基于 UWU Cash 稳定币的借贷协议，由 BitAcademy 的 nickole.btc 设计，当前尚处于测试阶段，可以通过加入社区填写表单获得测试资格。用户存入 STX 作为抵押品，可以最高借出其存入的 STX 价值的 66% 的 UWU Cash 稳定币，并且当前无需缴纳利息以及不设固定还款日。UWU Cash 是该协议发行的超额稳定币，协议对 UWU Cash 的锚定价值为 $1，每个 UWU Cash 的背后都有价值 $1.5 的 STX 抵押品作为挂钩支持，即用户借出 $1 的 UWU Cash，需要以至少 $1.5 的 STX 作为抵押物，协议必须保持 150% 的最低抵押率。当抵押物价值下跌，抵押率低于 150% 时，将触发清算。UWU 协议通过清算和套利来维持 UWU Cash 稳定币的稳定性。值得注意的是，UWU 协议还试图引入独立于协议之外的稳定性模块功能，允许用户以 0.5% 的名义费用将 UWU Cash 兑换成 sUSDT，sUSDT 兑换成 UWU Cash 则免费。

此外，UWU 协议还推出了 UWU Share Token（xUWU），最大供应量为 10 万枚，用户可以通过持有该 Token 获得协议的 100% 收入，但该 Token 无治理用途。

Arkadiko

Arkadiko 是 Stack 链上集交易、流动性提供和借贷的开源 DeFi 协议，用户可以抵押 STX、xBTC 和 ALEX 铸造并借出算法稳定币 USDA。USDA 将与 1 美元软挂钩，贷款出来的 USDA 将需要支付 1% 的年化稳定费用，即贷款的年利息。不同的抵押资产有不同的清算比例，STX 为 140%，xBTC 为 130%，auto-ALEX 为 180%。当抵押物价值下跌，不足以维持该比例时，将会触发拍卖，需要缴纳 10% 的清算罚款，并由第三方清算人清算，以折价的方式购买拍卖的担保物。

据官方数据，当前  Arkadiko TVL 超过 540 万美元，其中有约 395 万美元的担保物资产，LP TVL 为 150 万美元。

流动性质押

Planbetter

Planbetter 是 Stack 链上流动性质押协议，已有超过 8.8 万名 Stackers 用户质押了 2.8 亿 STX，累积获得 25.42 BTC 奖励。使用 Planbetter 进行质押，可以随时解锁 Token，没有冷却循环等。

NFT Marketplace

Gamma

Gamma 为 Bitcoin NFT 建造的 NFT 交易市场，已集成了 Stacks 和 Ordinals 。用户可以在该平台上进行 NFT 交易、创建、铸造和拍卖功能。目前该平台上交易最活跃的 Stacks NFT 是 BNS：Bitcoin Name System、Megapont Ape Club、The Guests 等。

Boom

Boom 是 Stacks 链上原生的 NFT 平台，并推出了一种新型 NFT：Boomboxes。它允许用户进行委托锁定其 STX 并收到一个 NFT 作为锁定的奖励部分的自动领取证书。100 STX 是 Boombox 的最小锁定金额。官方将在每个月发布一个新的 Boombox。

Boom 由 4 人团队建造，该团队自 2016 年便在建设 Stacks/Blockstack dapps。在未来规划中，Boom 将为用户定制的 NFT 或 Boombox 提供交易市场、以 BTC 作为 Boombox 奖励的支付选项、更注重展示 NFT 的全新 UI/UX 设计以及提供支持创作者的新方法。

TradePort

TradePort 是一个多链聚合 NFT 交易市场，当前已支持 Stacks、Near 链，并计划拓展至 Aptos 和 Sui。TradePort 平台的功能较为完善，除了常规的挂单销售外，还提供了一键立即销售，以及数据分析图标和 Portfolio 管理。

NFT

Satoshibles 采取了中本聪可能的形象，通过算法生成的基于 Ethereum 和 Bitcoin 上的 NFT 收藏系列，当前已发布了 Satoshibles、Satoshibles Ordinals、Monster Satoshibles、Special Editions、Beary Christmas 共 5 个系列 NFT，其中发布在 Ethereum 上的 Satoshibles 地板价为 0.0696 ETH，累积交易量超过 2,000 ETH；发布在 Bitcoin 上的 Satoshibles: The Ordinals 地板价为 0.035 BTC，累积交易量达 1.78 BTC。

Satoshibles 与 Stacks 合作建立第一个可以在 Ethereum 和 Stacks 之间跨链的 NFT 桥，最初的 Satoshibles NFT 是第一个通过 Stacks 跨链桥进入 BTC 的 NFT 项目。在未来规划中，Satoshibles 打算建立一个 DAO，并在合适的时候给予 Satoshibles 持有者空投。

域名

btc.us

Stack 链上域名服务提供商，并支持连接至闪电网络。Stacks 官推 Stacks.btc 正是使用该域名服务。用户可以在该平台上注册 1-48 个字符的“.BTC”域名，字符可以包含所有小写字母和一些特殊字符。注册统一需要花费 2 STX ，每个域名的有效期为 5 年。

社交

Console

Console 是构建于链上的 Web3 聊天社区，支持 Stacks，风格类似于 Discord。Console 允许用户使用包括钱包、NFT、BTC、Ordinals、DAO 工具等加密资产，并提供了一套管理 Web3 社区的工具。目前已开放了 ConsoleHQ 向所有用户公测。

其他

Sigle

Sigle 是一个为内容创作者提供的开源 Web 3 写作平台，该平台当前支持 Stack、Bitcoin Ordinals 和 Polygon 网络。Sigle 为作家或博主创建内容或笔记的提供了工具，其创作的内容因其平台具有的抗审查特性也会得到相对应的保护。完成写作后，平台会将内容存储在IPFS上或将 Bitcoin 链上。

Hirevibes

Hirevibes 是致力于为加密货币世界提供一个丰富人才库的 Web 3 人才聘用平台，招聘方可以在该平台免费发布信息招揽人才，其中有包括 Polkadot 、Immutable 、Avalanche 等多个知名项目通过该平台招聘人才，用户也可以通过该平台应聘加入心仪的团队。Hirevibes 在 Stack 链上推出奖赏原生 Token VIBES 的赏金计划， 其中一部分的赏金用于奖励一次成功招聘案例，应聘者和招聘方都可以共享赏金，促进招聘和应聘双方在积极性。
