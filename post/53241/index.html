原文作者：Karen，Foresight News

上周，Web3 游戏工作室 Argus Labs 完成由 Haun Ventures 领投的 1000 万美元种子轮融资，并同步公布专为链游设计的 Layer 2 区块链「世界引擎（World Engine）」。

鲜为人知的是，Argus Labs 的主创团队成员也曾共同创建过曾经火爆一时、一 Key 难求的链游「黑暗森林」（Dark Forest），Dark Forest 是一款基于零知识证明技术 zkSNARKS 构建的去中心化实时战略游戏。

那 Dark Forest 原团队成员创建 Argus Labs 的初衷是什么？Argus Labs 有何创新？

Argus Labs 团队背景

Argus Labs 是一家游戏开发商和发行商，由一群黑客和设计师组成，Argus 最早可追溯到 2020 年推出的基于零知识证明技术 zkSNARKS 构建的去中心化实时战略游戏黑暗森林（Dark Forest）。Argus Labs 创始人 Scott Sunarto 是 Dark Forest 的联合创始人。

Argus Labs 创始人 Scott Sunarto 同时担任该工作室的 CEO，目前也是区块链安全公司 Veridise 的顾问，还曾于 2021 年在 Uniswap 作为实习软件工程师工作过 3 个月时间。

Christopher Wilhelm 是 Argus Labs 的艺术总监，于今年 3 月份加入 Argus Labs，在此之前在区块链游戏开发和发行商 N 3 TWORK Studios 担任角色艺术总监，N 3 TWORK Studios 在去年 5 月份完成 4600 万美元 A 轮融资。

Olivier Lee 也于今年 3 月份加入 Argus Labs 担任游戏设计主管，此前在 Gameloft 和 Ubisoft 等主要游戏公司拥有 8 年的工作经验，在 2016 年联合创立了独立视频游戏工作室 Tinker Troupe。

在 Argus Labs 最新宣布的 1000 万美元种子轮融资中，Haun Ventures 作为领投方投资，其他参投方包括 Robot Ventures、Anagram、Dispersion Capital、Web3 开发公司 Alchemy、Coinbase 前首席技术官 Balaji Srinivasan 等。

World Engine 设计初衷

Argus Labs 表示，其 World Engine 是一款专为链游设计的 Layer 2 区块链，帮助游戏开发者构建和定制自己的开放和可互操作的游戏世界。通过 World Engine 的 Layer 2 分片架构，每款游戏都可以拥有自己的水平可扩展区块链。

Argus Labs 认为，在区块链上构建游戏对程序的编写方式施加了很多限制，当前的区块链并非设计用于运行游戏，许多加密游戏完全依赖 NFT，这些 NFT 被兜售为游戏内资产的替代品，但同时为最终端用户提供的利益有限。也就是说，当前的区块链基础设施不足以支持链上游戏。

为解决该痛点，Argus Labs 计划从下往上设计一个仅用于游戏的区块链，同时保留互操作性和可组合性的优势。这也就是 World Engine 的设计初衷。World Engine 是一个分片的 Layer 2 区块链 SDK，专为满足游戏开发人员和玩家的需求而设计，既能继承底层 Layer 1 的安全性和去中心化特点，又能让游戏开发者专注于创造游戏体验。

World Engine 设计和创新

分片设计是 World Engine 的一大创新，能够使游戏开发者能够将其游戏负载分布到不同的分片上，其灵感来自计算密集型大型多人在线（MMO）游戏的服务器架构。一条 World Engine 链可以根据需求调整其吞吐量，与开发商或发行商同步增长。

此外，World Engine 的分片架构可以避免互操作性和平台碎片化问题，以及允许游戏执行（游戏分片）与智能合约执行（EVM 分片）分离，在保持高吞吐量游戏服务器性能的同时，保持区块链的互操作性能力。

Argus Labs 目前已经推出其首个游戏分片实现 Cardinal，Cardinal 能够处理性能密集型游戏的工作负载，游戏逻辑是用 Go 编写的，开发人员无需学习新的编程语言。Cardinal 还可以通过客户端库与现有的游戏引擎（如 Unity 和 Unreal）无缝集成。

World Engine EVM 基础分片为玩家和开发者提供了一个中心场所来构建用户生成内容和平台，这些内容和平台可以通过 Argus Labs 分片路由器系统与游戏分片无缝互操作。EVM 分片的功能类似于 EVM Rollup，允许像使用以太坊一样使用所有开发人员工具、钱包和库，还借助 Berachain 开发的高性能模块化 EVM 框架 Polaris 通过其状态预编译和插件系统进行定制。

不可否认的是，游戏将在 Web3 大规模采用中起到非常重要的引领作用，作为游戏基础设施的 Argus Labs 和其 World Engine 能否成为走出来让我们拭目以待。
