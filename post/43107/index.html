Web3 最重要的趋势之一就是对圈子的重新构想：观众/粉丝和创作者之间的界限模糊，以及粉丝劳动如何被资助和奖励。

两大转变正在证明这一趋势：

粉丝和创作者之间的鸿沟正在消失。粉丝可以施加重大影响甚至共同创作原创作品（质疑“原创”甚至意味着什么）；

粉丝们正在采用新的商业模式和资金来源来创建衍生项目，而这些在以前是不存在的

最终结果是扩展了互联网上的创造力，使其具有更大的访问权限、财务优势、协作和成就感。

从Canon到“Fanon”

粉丝和创作者之间的区别在 web3 中的相关性正在减弱。粉丝们正在创作与原作一样被广泛认可的衍生作品，甚至创作被官方认可为经典作品。与此密不可分的是，创作者认识到粉丝可以为创作过程增添价值，从一开始就授权作品允许衍生使用（通常选择 CC0 或“无权利保留”许可）。这与创造者经济的未来是社区经济的主题之一一致，其中共同创造是通过共享所有权来驱动和奖励的。

我们来分析正在发生的一些案例：

Shibuya 是一部动画电影，通过出售 NFT Producer Pass 获得资助。NFT 持有者可以对电影后续章节的情节进行投票和影响，从而获得代表最终电影份额的代币。粉丝反馈与创作过程的紧密结合削弱了粉丝与创作者之间的区别。

另一个例子是 Jenkins the Valet，这是一种无聊的猿 NFT，由其所有者 Tally Labs 命名和构思。Jenkins 的整个角色可以被认为是一部延伸的粉丝小说——但它已经变得如此突出，以至于被 CAA 签约。共享 IP 所有权导致 BAYC 的单一衍生作品引发了一个新的创意世界。

BAYC 持有者对他们拥有的 NFT 的无限商业权利对于角色能够——并且有动力——创造衍生作品并取得巨大成功至关重要。这是传统粉丝小说世界的重大转变，在该世界中，粉丝作品与原创作品之间有明确的界限，粉丝作品在很大程度上被视为未经批准（有时是不受欢迎）的原创作品。

现在，Tally Labs 正在制作一部更长的小说，由扩展的无聊猿宇宙主演，其中猿/突变猿的持有者可以提交他们角色的故事并影响情节，从书中获得 50% 的净利润。解锁利润分享的一个关键是，NFT持有人将他们的IP授权给Tally，以便在书中出现，这实际上是将粉丝与作品绑定在一起。

在他们的路线图上还有一个名为Azurbala的元宇宙项目，它将产生更多的角色。甚至在项目启动之前，社区成员就已经在Azurbala中构建了传说和角色，影响了项目的官方路线图和故事–这是一个粉丝小说的例子，它将原创作者和粉丝的意义折叠在一起。

粉丝创作的新商业模式

由于版权法的规定，粉丝作品在货币化方面历来处于法律的灰色地带。由于他们使用的是受版权保护的作品的内容和人物，大多数粉丝小说创作者被归类为没有商业模式的业余创作者。各种举措都试图解决这个问题–比如亚马逊的Kindle Worlds–但都有其自身的局限性。

在web3的创意项目中，代币的基本所有权为粉丝提供了一个内在的商业模式，并激励了衍生品的创作。与web2的粉丝免费工作相比，在web3中，粉丝所有者通过代币接触到他们劳动的成功：随着所持有的数字稀缺事物变得更加知名，其价值也会增加。如，通过额外的媒体项目传播Jenkins the Valet，Jenkins NFT（以及BAYC宇宙中的其他NFT）的价格会增长，让粉丝和原创作者都能获得价值。

我们也看到这种情况在更广泛的社会代币空间和其他NFT类别（如音乐）中发生，代币的所有权激励粉丝从被动的消费者变成积极的参与者和传道者。

培养共享优势

在粉丝资助方面，拥有国库的项目可以资助粉丝作品，以进一步促进其 IP 的增长。Nouns 的 27K ETH 资金（超过 4500 万美元）是通过在一年多的时间里出售 400 多个 NFT 建立起来的，本质上是一个专注于 Nouns 的基金，致力于传播 Nouns 相关 IP 领域的项目。

来源：Nouns.wtf

这就好比J.K.Rowling从她的收入中拿出一部分来资助那些使用她的角色的粉丝小说作者–这在传统媒体界是闻所未闻的。该理论认为，NFT的原始集合的潜在稀缺性可以让它随着对其角色宇宙的日益认可和普及而获得价值。

结论

总而言之，新的 web3 创意项目正在颠覆媒体创作模式：所有者社区正在围绕新生创作形成，他们将自己的故事与衍生风险相结合，并将其分发给具有共同优势的社区，然后传递给更广泛的受众。粉丝成为创作者，粉丝的粉丝也会成为创作者，从而形成了一个棱柱形的创作组合，所有这些都由原生商业模式和新的融资机制提供支持。最终结果是人人都可以讲故事和创造力的民主化。
