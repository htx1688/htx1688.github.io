Pound Swap

Pound Swap 是建立在 zkSync 上的由社区驱动的、 ve(3,3) 流动性账簿 DEX，为 DeFi 流动性和 token swap 提供强大的交易平台。作为 Trader Joe Liquidity Book 的分支，Pound Swap 在设计和功能上与 Traderjoe LB 类似，为用户提供零滑点、资本效率高的 AMM 解决方案。Pound Swap 是首个将 ve(3,3) 与 Liquidity Book 相结合的 DEX，也是 LB 与 ve(3,3) 结合的首次尝试，希望为流动性管理和 token 激励提供更公平、更高效的解决方案。Pound Swap 目前还未上线，可以关注其上线进度。

Pound Swap 特点是：

-零滑点：交易者可以在平台上以零滑点 swap token。

-Surging Pricing：LP 在市场剧烈波动期间赚取额外的动态费用。

-高资本效率：流动性账簿可以支持低流动性要求的大批量交易。

-灵活的流动性：LP 可以根据自己的策略构建灵活的流动性分配。

Mori Finance

Mori Finance 是以太坊上的下一代原生稳定资产 DeFi 协议，通过抵押 ETH 生成低波动性的稳定资产，且不会造成任何损失。此外，用户可以选择创建资产衍生资产（ETHS、ETHC）作为针对 ETH 价格波动的对冲资产，从而可以免费持有 ETH 多头头寸，而无需面临清算风险。目前处于测试网阶段。

此前也有针对 ETH 为标的的稳定币，比如 LUSD，但这类稳定币在资本效率和可扩展性方面有着限制，在体量上落后于法币支持的稳定币。Mori Finance 议将 ETH 的波动性分为低波动性稳定资产和高波动性衍生资产。以 ETH 为例，用户可以根据自己的风险偏好选择持有高波动性的 ETHC（ETH Coin）资产和低波动性的 ETHS（ETH Stable）资产。

ETHS 是由 ETH 支持的稳定币 token，可以最大限度地减少资产波动性，处于低波动性 token。此外，还可以赚取额外的 LP (ETHS/USDC) 收入。而 ETHC 被设计为杠杆做多 ETH 永续 token，可以用来杠杆做多 ETH，不会强制平仓，该 token 还有着一种紧急控制模式，可以降低 ETHC 杠杆倍数。

目前该协议仅支持 stETH 作为抵押品，用户可自行决定你的 ETH 拆分为 ETHC 和 ETHS 的比例，ETHS 价格变动为 ETH 的 10%，例如 ETH 上涨 10%，则 ETHS 上涨 1%。

Zephyr Protocol

Zephyr Protocol 是结合了门罗币隐私性和 3 代币模式的稳定币协议。其 3 种 token 分别是：

-ZEPH（基础 token）

-zephUSD（算稳 token）

-zephRSV（储备 token）

ZEPH 可以通过挖矿获得，用于超额抵押 zephUSD（系数为 400%），并且有供应上限。ZEPH 的缺点是「可用性低」，只能在Seven Seas上购买。某种程度上，这也是其「隐私性」的一部分，该协议目前处于内测阶段。Zephyr 继承了 Monero 的隐私功能，其亮点就是去中心化的、针对隐私的稳定币。该协议概念创新来自「Cardano 上首个去中心化稳定币 Djed」，为了弥补隐私和稳定性之间的差距，Zephyr 利用了 Djed Protocol 的机制，同时结合了 Monero 固有的强大隐私功能。也是业内首个针对隐私性的超额抵押稳定币，或是 crypto 发展的一次重大飞跃。

Galador

Galador 是建立在 Mantle 网络上的 DEX，其目标是为基于 Mantle 网络的 DeFi 基础设施、交易者、LP、DAO 金库和开发人员提供最具流动性的市场。该协议不同于传统 DEX，专注于提供流动性种类的定制方法，优先考虑其可组合性。

Galador 主网上线后的特点包括：

-双 AMM 模型，支持波动（UniV2）和稳定资产（Curve 类）swap

-集中流动性，实现更高的资本效率

-LP 提供流动性赚取平台 token GLDR

-通过质押平台 token 获得挖矿和平台收入。

目前 Galador 支持 Mantle 测试网，也在近期准备上线 Manle 主网 Alpha 的 DEX 之一，token 还未发行，用户可以提前埋伏，在其网络上领水进行 swap 和添加流动性，未来或将有福报，

OmxLab

OmxLab 是这些项目中较新的一个，其社交媒体于 7 月开通，Omx 与 GMX 类似，属于 GMX 分叉项目，其特点是支持高达 50 倍杠杆交易、零滑点，其值得关注的理由是该 DEX 建立在 Osmosis 上，旨在将 Osmosis 转变为综合性的「通用交易中心」，提高资金效率并为 Osmo 用户提供无缝的交易体验。

其交易功能如下：

交易：零价格影响交易、高达 50 倍杠杆、蓝筹资产零利差、无交易对手风险环境；

集成： OMX 在保证金交易、支持 IBC 的 DEX 和基于 EVM 的 DEX 之间无缝路由流动性，确保资产的最佳利用；

流动性：在 Cosmos 和 EVM 生态系统中的各种多链网络中体验统一的流动性深度，包括对 Metamask 等钱包的支持；

链： OMX 最初将部署在 Osmosis 和 Polygon 上，未来或将扩展到更多链。

此外，OmxLab 还是首届 Delphi Labs 黑客马拉松中的获奖项目，目前支持 Keplr 钱包，DEX 支持 4 种 token，分别是 BTC、ETH、ATOM、OSMO。
