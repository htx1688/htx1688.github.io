作者 | jk

今年的九月，是各国区块链峰会实实在在的高峰期：从月初的韩国区块链周（KBW），到月中在新加坡的 Token 2049 ，到紧随其后的上海万向区块链周，亚洲的区块链峰会抢走了所有的风头，一时间朋友圈里似乎“所有人都在新加坡”。同时，美国也有旗鼓相当的三次峰会，分别是德克萨斯州奥斯汀的 Permissionless，以及纽约的 Mainnet 和 ETH New York——Odaily星球日报驻北美记者也参加了这几场峰会，参会感受总结如下：

Token 2049 抢走了热度：有名的公司怎么都在新加坡？

今年由 Blockworks 和 Bankless 主办的 Permissionless 位于德克萨斯的奥斯汀，和 4 月的 Consensus 位于同一举办地。然而，今年的 Permissionless 肉眼可见的参与者要比其他会议少很多。其中一个原因就是，今年的 Permissionless 会议日期是 9 月 11-13 号，正好与新加坡的 Token 2049 撞了日期，驻场记者在会场上碰到不少来自知名公司的工作人员说，“公司有不少人在新加坡，因为那边的会议更大。”从嘉宾阵容来看，Token 2049 的数量和质量也明显高于这两场，而且 Permissionless 现场的很多嘉宾甚至也只是在线会议接入，并没有实际到场，有些人甚至就是从新加坡的 Token 2049 接入的。

Permissionless 嘉宾阵容。来源：Permissionless 官网

相比之下，Messari 举办的 Mainnet 和以太坊基金会举办的 ETH New York 的举办就相对来说更好一些：一是因为城市更大，有不少的知名机构（如 Uniswap）本身就有办公室位于纽约，从而工作人员基本都参加了会议，而同时也可以用公司场地来举办场外活动。相比之下，纽约的这两场会议的风评似乎更好一些。同时，因为 ETH New York 本身是一场 Hackathon 的原因，吸引了不少开发者和学生来打比赛，进而吸引了看项目的基金 ，从而现场的热度非常活跃，听到了不少氛围相比 Permissionless 会好很多的评论。

展台和场外活动：无法匹敌 ETH Denver/Consensus 的繁盛

回顾这两场会议，Booth 和 side events 的数量明显减少，与 ETH Denver 和 Consensus 的繁荣景象相比有些失色。此前，Booth 和 side events 一直是行业会议中的一大亮点，不仅为与会者提供了更多的交流机会，还为创业公司提供了宣传的平台。

就 Booth（机构在场馆内的展台）而言，Permissionless 和 Mainnet 的数量远远小于上半年 3 月的 ETH Denver 和 4 月的 Consensus。其中，Consensus 和 Permissionless 使用的是同一场馆，均为 Austin Convention Center；然而，Consensus 的展台占满了整个大厅，中间甚至留有一条主干道通向各个不同的分区。相比之下，Permissionless 的占地面积仅有 Consensus 的三分之二，且参会者明确感到“参加的机构数量比 4 月份的时候少了很多”。

相比起 Token 2049 整周超过 400 场的场外活动数量相比，Permissionless 和 Mainnet 的场外活动并不是很多，每天约有 10-20 场，主要也是由各个知名机构举办。

从今年 Permissionless 和 Mainnet 两场行业会议可以明显地看到，Token 2049 的存在无疑为行业带走了大量的关注热度，这在某种程度上反应了亚洲区块链社区的火热 。相比于美国的经济熊市和大量的监管压力 ，大家的参会选择在某种程度上也反应了对于未来发展方向的下注。

机构从业者众多，嘉宾阵容强大

尽管上述的几个现象有些令人遗憾，但从与会的嘉宾阵容来看，这两场会议依然具有很高的行业影响力。尤其是各大机构的从业者人数众多，而且 Speaker 的质量非常高。这不仅为参会者带来了高质量的内容分享，也展示了行业内高端专家和机构的关注度。

今年的美国会议方面，Panel 的讨论主要集中在L2解决方案和机构化的问题上 。随着区块链技术的发展，L2在今年彻底成为了行业内最热门的话题，其对于提高交易效率和降低成本的潜力吸引了大量关注。同时，随着更多的传统机构开始进入此领域，如何将现有的机构结构与 DeFi 相结合，也成为了众多讨论的焦点。这两个方向的讨论，为参会者提供了深入了解行业最新动态和未来发展趋势的机会。

有关传统机构入场的 Panel 座无虚席。来源：Permissionless 官网

华人项目的隐匿：缺乏存在感

另一个明显的现象是，在 Permissionless 中，华人项目的存在感非常低。在会场中，似乎只有 OKX 拥有 Booth，也只有零星几个美国当地的华人项目举办了场外活动，现场的亚洲面孔也比较少。深究原因，不少本来会飞来美国参会的项目，因为时间的冲突都选择了参加新加坡的会议；有不少美国的本土华人项目因为会议规模和融资难度等原因也纷纷选择了飞往新加坡。这个现象倒是一次有趣的逆行：与大家一般认为的Web3汇集于美国不同，华人项目这次不约而同地选择了新加坡。相比之下，Mainnet 和 ETH New York 的华人参与度就高很多，一是因为相当一部分的项目总部本身就在纽约，其次也是因为这场会议在 Token 2049 结束之后，因此很多项目方也飞回了美国。

峰会的过饱和：优胜劣汰的必然

经过这两场会议和年初 Denver/Consensus 的热度对比，我们或许可以得出一个大胆的结论：区块链行业的峰会和会议正在经历一个过饱和的状态。和其他行业一样，随着时间的推移，会有更多的会议和活动涌现，但并不是所有的活动都能保持其初衷和活力。最终，行业会议也会走向一个优胜劣汰的过程，只有那些真正为与会者和行业带来价值的会议，才能够持续存在并获得成功。

目前，全球区块链相关的知名会议已经有十数场：除开不同地方办活动的以太坊基金会，每年在美国的知名会议有：

  • 斯坦福区块链会议

  • 哈佛区块链会议

  • MIT Bitcoin

  • ETH Denver

  • Consensus by Coindesk

  • Bitcoin Miami

  • Permissionless by Blockworks

  • Messari Mainnet

  • NFT LA

  • NFT NYC

再加上更多的小规模会议和亚洲/欧洲会议的分流，熊市的行业热度未必能够撑起十数场美国峰会的举办。而由于飞轮效应，去年觉得某会议不值得参加的人可能明年未必会来；大胆猜测，在下一个牛市周期开始之前，必然会有相当数量的美国区块链峰会淡出大家的视线。

总结一下，尽管今年的 Permissionless 和 Mainnet 会议与之前有所不同，但它们仍然为我们提供了关于区块链行业动态和未来发展的有价值的信息。期待未来，我们更希望看到更多有深度、有价值的行业交流和合作。
