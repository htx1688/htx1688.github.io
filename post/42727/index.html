9 月 14 日，NFT 租赁协议 Rentable 宣布关闭，创始人表示未能找到「产品市场契合点」。

NFT 租赁平台 Rentable 在前几天宣布停止运营。

虽然目前平台可以取款，但租赁功能现在已被禁用，用户可在 10 月 13 日前完成取款。在该日期之后，将启动紧急提款，并将代币发送给其各自的所有者。

Rentable 是让用户出租他们的 NFT 协议，其创始人宣布，该协议在未能找到「产品市场契合点」后将关闭。

Rentable 关门大吉？

Rentable 平台出现是为了让持有人出租 NFT，反正大家的 NFT 都是放在钱包里落灰，Rentable 协议向 Decentraland 用户、Meebits 持有人和 Lobster DAO 提供服务，据 Bonassi 称，让「所有者获得没有清算风险的收益」。

平台创始人 Emiliano Bonassi 说：「Rentable 就好像是一场不可知论，在不依赖于特定用例的情况下实现租赁。Rentable V2 使任何 dApp 无需集成即可为其用户提供租赁服务（仅 WalletConnect），消除了平台之间的障碍。」

然而，也是这位创始人不得不面对这个不太乐观的市场，不得已宣布了 Rentable 无法获得资金，几乎没有在市场上获得反馈，创始人说：「我们有可能融资，但我们等待。我们选择先检查产品的市场适应性，因为我们有资金用于建设和审计，因此，关闭的主要原因是，经过一些调整后，我们没有找到产品市场契合的点。」

虽然每天都蹲在 Web3 领域里的各个角落官网各种咨询，但是我一直有一个很强烈的感受，就是 NFT、DeFi、DAO，还是区块链底层功能等等，在这个千变万化的行业里唯一不变的是，你几天没看项目看资讯，很有可能就错过生态系统中一些新发展。

我认为，NFT 租赁是 NFT 生态系统中一个不太常见但却非常引人注目的部分，虽然看到 Rentable 这个消息后也是让我眉头一紧，但是关于 NFT 租赁，首先要知道的是，虽然这个想法和玩法可能会延伸出非常有趣的模式，但作为一个新生事物，仍未得到充分发展。没有得到好的市场反馈有时候也是预料之中。

开拓 NFT 租赁市场的项目方相对较少，在未知领域开展业务，大多数看好这一想法的平台尚未启动运作。然而，这个想法很有吸引力，如果它在更广泛的 NFT 社区中能获得关注，它可能会对 Web3 格局产生重大影响。虽然不能保证这一点，但断然否定它是不明智的，接下来我们展开看看 NFT 租赁到底是如何运作的。

什么是 NFT 租赁？

拥有某种 NFT 的人可以在短暂的时间内将其出租给其他需要它的人。NFT 租赁市场使用类似 Defi 的区块链技术来促进安全交易，并保证在租赁时间结束后将 NFT 归还给所有者。

如何租用 NFT？

有两种广泛的方式可以出租 NFT：无抵押租赁和有抵押租赁。

抵押租赁

在具有内置贷款和借款功能的市场上，持有人可以列出他们的 NFT。对 NFT 感兴趣的租户将开始借贷程序，这将涉及到将 NFT 放在一个智能合约中。接下来，贷方和租方将就智能合约的条件达成一致。租用程序在条款决定和商定后开始。

这种类型的租赁围绕着借款人提供抵押品以暂时使用别人的 NFT。比如 BoredJobs 就属于这种类型，每个 BAYC 交易都是由品牌询问者和 NFT 所有者专门协商的，其中 15% 的分成归平台所有，以支付运营成本。

另一个以这种模式运作的例子是 reNFT，这是一个成立于 2020 年底的 DAO，其主要目的是促进 NFT 的租赁。去年 10 月，reNFT 宣布了由 Animoca Brands 牵头的一轮融资，总额为 150 万美元。reNFT 还与 Rarible 合作，并声称该平台将在未来率先整合其租赁功能。该公司的主要产品是一个协议层，允许在以太坊主网上进行 ERC-721 和 ERC-1155 NFT 的点对点租赁。

在技术层面上，用 reNFT 借贷意味着将你的 NFT 转移到他们的智能合约租赁列表中。然后贷款人指定租赁价格（他们希望每天得到多少补偿），NFT 价格（作为抵押品，由借款人提出），以及租赁期。

抵押品模式有几个潜在的问题。

首先，借款人作为抵押品的 NFT 的价格可能与买入该 NFT 社区的价格一样令人望而却步，比如租个无聊猿的价格可能就让我等打工人看了直摇头，这可能会减少可行的这类潜在用户的数量。

第二，这是一个有趣的问题，借款人可能决定不归还你的 NFT。当然，你有抵押物作为质押，但比起我本身自己持有的心爱的 NFT 本身来说，如果真的被人彻底拿走那点经济补偿也只能说经济层面上的安慰了。

无抵押物的租赁

无抵押物的租赁与有抵押物的租赁的程序相似。唯一的区别是，租户永远不会收到原始的 NFT，这与有抵押物的租赁不同。取而代之的是，为租房者创建一个包装好的 NFT，它具有与原始资产相同的特征，并由真实资产支持。一旦合同结束，包装好的国家信托基金就会被烧毁。在这种情况下，租户和持有人都不需要提供任何抵押品。因此，双方的财务风险都降低了。

UnitBox DAO 就是提供这样服务的一家区块链初创公司，它认为 NFT 租赁可以降低门槛，让更多人加入 GameFi 世界的游戏赚钱生态系统，比如说。UnitBox 认为，游戏中的地块、角色皮肤和装备都完全可以适应 NFT 租赁的理念。

UnitBox 的租赁系统仍处于测试阶段，但 DAO 背后的想法是通过使用协议创建 NFT 的「包装」版本来实现无抵押的租赁。假设有一个投资者（NFT 所有者）和一个资产管理者（玩家）。资产管理人免费「租用」NFT，但把在游戏中获得的收入分给 NFT 所有者，后者可以在任何时候收回资产。

为什么有人会租用 NFT？

除了上文提到过的利用租来的猴子做一些 IP 的延伸，为自己的产品打广告外，租用 NFT 是一种相对廉价的方式，可以利用 NFT 提供的一些效用，而不需要去购买，就可以得到访问独特的会员专用内容的权利。

举个例子更好理解的例子，Stoner Cats，是一个由美国女演员 Mila Kunis 创建的 NFT 项目，持有者自己肯定是已经看过无数遍自己所有的那些电视节目，然后可以通过出租 Stoner Cat NFT，对借阅者来说，付费租来一张光盘，观看内容后再归还。这类似于从多年前的影像店租借 DVD。

总结

正如我开头和我上文中介绍的，在说服更广泛的 NFT 生态系统的市场反馈之前，租赁模式还有很长的路要走。但就像 Web3 世界里的一切事物一样，这个想法目前既充满潜力又需要谨慎观望。

无论如何，这类平台所推出的租赁模式与游戏赚钱机制关系不大，而与租赁知识产权关系较大，但其核心概念是相同的。随着 NFT 的焦点在数字艺术和数字资产之间来回摇摆，对 NFT 租赁市场的大部分需求都与流动性和金融用例相联系。NFT 仍在不断成熟，并从被视为一个艺术类别过渡到具有传统金融资产的属性，通过使用 NFT 的短期和中期许可交易，这将对 IP 具有价值的社区产生更大的影响。

NFT 的租赁市场仍然相对较小，或许在未来，随着 NFT 的应用场景越来越多，对 NFT 的租赁需求将上升，更多的 NFT 租赁场所将开始出现。
