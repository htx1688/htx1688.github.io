注：在 Layer 2 战争逐渐升温的背景下，作者 Jake Pahor 通过协议收入、财政库、锁仓情况、代币功能、路线图等在内的 7 种指标分析了部分以太坊 L2 项目的发展状况，并给相关项目给出了主观评分。

我查看了一些以太坊 L2 项目，包括通用的 L2（Polygon）以及特定于应用的 L2（dYdX），为了缩短这篇文章的长度，文章不得不排除一些没有进入前 5 的伟大项目（译者注：例如 starknet 和 zksync 等通用 zk rollup 二层网络还没有进入成熟阶段）

对于这个研究框架，作者使用的是 The DeFi Edge 的项目评估方法，然后，根据分析为每个项目提供一个评分，所以你需要读到最后才能看到结果。

1 、它们创造了多少收入？

Token terminal 提供了一些非常有趣的统计数据，在过去180天内，这些 L2 项目的协议总收入（费用）情况如下：

  1. dYdX 2.122亿 美元

  2. Arbitrum 1370 万美元

  3. Polygon 880 万美元

  4. Optimism 850 万美元

  5. Loopring 250 万美元

dYdX 在协议收入这一块绝对是碾压其他项目的，而去掉 dYdX ，你可以在图表中看到，在过去的一个月里，Polygon 一直处于领先的位置，并且势头强劲。

2、协议财政库里有什么？

关于财政库的一些重要因素是：

  1. 它们是否拥有不同的资产，或者大部分都是自己的原生代币（意味着风险更大）；

  2. 财政库有多大，这可以给项目带来多少发展？

Polygon：很难找到有关 Polygon 财政库大小的任何信息，然而，我们知道 Polygon 最近有一个 10 亿美元的 ZKP 发展基金计划，我们可以假设 Polygon 的财政库远大于这个金额。

Optimism：初始代币供应的 30%（12.9 亿 OP）将分配给社区基金（生态基金+RetroPGF），基于当前市场价计算（2美元），这部分代币的理论价值大约为 25 亿美元。

dYdX：初始代币供应的 5% （5000万 DYDX）分配给了社区财政库，以目前的价格（2.27 美元）计算，这大约是 1.13 亿美元。

Loopring：20% 的代币供应（2.75亿 LRC）分配给了 Loopring 生态基金，以目前市场价（0.4美元）计算，这大约是1.1亿美元。

3、 代币锁定情况

如果大多数代币仍处于锁定状态，那这可能会在未来造成巨大的抛售压力（你不想成为 VC 退出的流动性）。以下图表和信息来自 Messari 以及 coingecko。

Polygon：大约 92% 的 matic 代币供应已经解锁，最后的 8 亿代币将在2025年5月之前解锁，当前的流动供应量大约是 100 亿代币总供应量中的 91.8 亿。

Optimism：大约 5%的 OP 供应已通过空投的方式解锁，当前流通量大约是 42.9 亿 OP 总供应量中的 2.15亿。

dYdX：大约 11% 的 DYDX 供应已经解锁，当前流通量大约是 10亿 DYDX 总供应量中的 1.12 亿。

Loopring：大约 90% 的LRC供应已经解锁，当前流通量大约是 13.74 亿 LRC 总供应量中的12.45 亿。

4、代币功能和用例

Polygon ：MATIC 代币用于支付网络中的交易费用，你可以通过质押 MATIC 获取奖励，质押的作用是保护网络（译者注：这是 Polygon L1的情况，Polygon Hermez、Polygon Zero 等以太坊 L2 解决方案还没有上线主网）；

Arbtirum：还没有发行代币，但该网络本身已经获得了巨大的牵引力；

Optimism：区块空间的权利是推动 OP 经济模式并与网络本身一起增长的可持续收入来源；

dYdX：目前 DYDX本质上还是一种治理代币，持有者还可以根据其头寸规模获得交易费用折扣；

Loopring：协议费用分配给了流动性提供者（LP）、保险者以及整个DAO，这激励了代币持有者做好的行为。

5、路线图

Polygon 刚刚宣布他们将实施第一个 EVM 等效的 ZK L2。Sandeep 还计划在2-3年内实现区块链的完全去中心化，包括去中心化数字 ID 等强大产品目前都在开发当中。

Arbitrum 尚未发布完整的路线图，但有人猜测他们将在某个时候通过空投的方式发布原生代币。

Optimism 在下图中列出了自己的路线图：

dYdX 计划在2022年底通过 V4 版本实现完全去中心化，这似乎是他们的主要关注点，并将成为该项目的一个巨大里程碑。（译者注：dYdX 将从一个以太坊 L2 项目迁移到 Cosmos 生态，并转变成 L1项目，最大的关注点将是 DYDX 代币的赋能）

Loopring 计划在 2022 年进行一些重大升级，包括 Loopring Earn 和 zkEVM。

6、模仿者情况

这个列表当中的5个项目在性质上都非常相似，其中 Polygon、Optimism 和 Arbitrum 都在竞争成为最好的 layer 2 扩容解决方案。

在目前这个阶段，从市值以及“护城河”的角度来看，Polygon 无疑是处于第一的位置，然而，Arbitrum 和 Optimism 都获得了广泛的采用，并且它们都在迅速增长。

7、团队和融资情况

注：以下融资统计数据来自 crunchbase。

Polygon 团队已经完全公开了（主要成员包括 Anurag Arjun, Jayant Kanani, Mihailo Bjelic 以及Sandeep Nailwal），并且该项目方已获得了业内头部投资机构的支持，包括 Alameda、Binance Labs 以及 Coinbase Ventures，截至目前，Polygon 的融资总额为4.515 亿美元，他们最近还宣布和一些巨头公司达成了合作关系（包括迪斯尼、Reddit、Meta以及 Stripe），这表明他们的业务发展是正确的。

Arbitrum 也拥有一支完全公开的团队，他们也获得了一些知名投资者的支持（包括 Pantera、Alameda、Mark Cuban以及 Coinbase），截至目前，Arbitrum的母公司 Offchain labs 获得的融资总额为1.237 亿美元。

Vitalik 本人是 Optimism 的忠实粉丝，并表示支持他们的新治理结构，此外，Optimism 还得到了A16z、Paradigm 以及 IDEO 的大力支持，截至目前，Optimism 获得的融资总额为 1.785 亿美元。

dYdX 从 Hashkey、Starkware 以及 a16z 等知名投资机构的手中获得了8700 万美元融资，他们的团队成员也是完全公开的。

Loopring 从 Kosmos Ventures、G2H2、Neo、Fundamental labs 以及 Obsidian 的手中获得了 4500 万美元融资。

总结

根据我的研究和分析，我给每个项目的每个类别都打了一个分数（1-10分），如你所见，Polygon 是明显的赢家。

Polygon 目前的市值也处于领先位置，并且由于即将到来的以太坊合并，该项目最近确实获得了一些动力。当然，其他的 L2 项目也是非常有前景的，从长远来看也具有巨大的成功潜力。
