Web3 社区的核心理念和承诺是创造一个自我维持和自我加强的飞轮，成员通过参与来获得社会地位、访问效用和 / 或股份，这反过来为成员创造了激励和网络效应，使他们投资和贡献更多。

互惠和能动性是创造有机动力和保持飞轮运转的驱动力和有机燃料。互惠性确保成员在共生和互惠的基础上认同共同的目标和价值观，这反过来又是成员通过积极参与和切实地为社区做出贡献来承诺能动性的先决条件。

Gate&Gain 介绍

为了使社区蓬勃发展并建立吸引力，他们需要能够控制和分层对社区活动和福利的访问，并设计激励措施，使成员表现出理想的行为并为社区的更大目标做出贡献。Gate 是指社区决定谁可以访问社区的哪些部分。Gain 是指提供给表现出理想的行为并对整个社区的福祉做出积极贡献的成员的激励。

有些社区可能会设置最低限度的门槛和分层，让所有东西都对所有人开放。然而，我认为大多数社区会因某种程度的准入门槛和分层而受益。通过引入 ( 分层的 ) 成员「给予 - 获得」，社区创造了一种「包容性排他」的感觉，通过满足一定的最低标准来限制成员的体验。

web3 中的每个社区都有 Gate 和 Gain。门是社区的入口。你可以通过开放大门来重视数量，也可以通过关闭大门来重视质量。收益是成员加入你的社区的原因。除了产品的价值主张，你还可以利用两种类型的激励：社交和财务。

有门槛的社区

为了创造必要的凝聚力，社区应该在包容和排斥之间寻求最佳平衡。社区是隐式封闭的，因为它们为社交互动提供了特定和共享的环境。然而，添加明确的入门级门槛 将确保新人对社区所代表的内容有充分的了解和认同。与城市发展背景下的例子不同，门控社区主要的目的不是为了将不受欢迎的客人拒之门外，而是为了优化社区内所有成员的体验。

准入门槛并不是为了排斥。相反，它是关于通过基于共同价值观、共同目标和互惠互利来筛选成员，以确保最佳的包容性和归属感。虽然大多数初次入职应该是无缝的和渐进的，但一些摩擦有助于评估新成员是否符合社区的目标、范围和价值观。

门控也不需要是二元的。许多社区将是相当开放的，但随着成员在社区旅程中的进展、贡献、建立声誉并承担某些支持角色，这些社区将逐步释放特定的内容、体验和好处。他们给予的越多，得到的回报就越多…

访问门控和分层的示例：

  • 社区和 / 或特定社区「渠道」

  • 优质内容和体验

  • 社区活动 (IRL & URL)

  • 会员专用的效用和福利 ( 例如独家、折扣……)

  • 社区治理和投票

  • 社区国库和会员奖励

付费、证明或赚取

付费访问 无论是作为订阅还是一次性投资，都是一种经过测试和证明的方法，可以提供仅供会员使用的价值。至少这是一种基于投资意愿和感知价值去划分你的成员或粉丝社区的方法。对于社区和创作者来说，高级会员或赞助绝对是一种基于真爱粉模式实现盈利的方式。然而，虽然支付证明可以解锁一系列额外的好处，但它并不能让会员成为积极的参与者或增值贡献者。

X 证明和 engage-to-earn 机制可以更有效地确保成员的适合性和贡献。X 证明依赖于广泛而灵活的协议来验证成员资格和贡献。可以采取准入或入职检查的形式，以验证最低会员资格标准或行动，或通过筛选机制来核实所需的证书或过去的贡献。

但更普遍的是，X 证明和 engage-to-earn 将需要一套声誉和奖励 算法，以激励成员积极而有意义地做出贡献。成员贡献的质量和价值很可能是由自动算法和其他社区成员手动「赞成 / 反对」决定的。

我们获得声誉和社会资本的方式，以及如何获得认可和奖励，往往在每个新社区都非常具体。为了融入并脱颖而出，社区定义了自己独特的规范，并要求具体的归属、参与和贡献证明。

每个社区都有自己特定的等级制度，规定了成员如何建立声誉、积累社会资本、获得社区认可和奖励。为了帮助会员融入社区并有机会脱颖而出，社区通常会建立一套内部规范，并设置相关的「成员证明」和「参与证明」。在共识机制 (X 证明 ) 下，成员可以通过展示「能力证明」、「活动证明」等来持续积累社会资本。「X 证明」中的「X」可以是任何合规行为，或者对社区及其愿景的任何有价值的贡献。

基于声誉的门控

社会地位和社区声誉通常是通过成员提供某种参与、贡献和影响的证明来积累的：任何被视为规范的行为，任何对社区有价值的行为，以及它所代表的东西。

衡量一个社区的贡献和声誉 很困难。重要的是，无论使用什么输入和模型，声誉模型都是透明、公平和公正的。它反映了社区的集体共识，即去中心化。最基本的方法是使用客观、定量的输入 ( 即成员在社区内的存在和互动 )。然而，为了鼓励质量而不是数量，社区的声誉模型需要包含定性信号 ( 例如社区认知度、成员影响力和影响 )。

声誉门控的社区 实际上增加了包容性、准入性和机会。成员可以「从零开始」，并通过积极参与社区和贡献项目逐步建立自己的声誉。作为回报，在达到一定的声誉状态的基础上可以解锁门控体验和特权。

社区现在可以基于贡献、热情和参与来进行管理——而不仅仅是拥有代币或基本的 NFT……为了让 Web3 生态系统演变成一个更有活力的生态系统，使个人活动、行动和贡献具有意义和可移植性，声誉门控社区可以而且必须存在。

代币化 Gate&Gain

在最基本的层面上，代币是可编程的激励机制，如果经过深思熟虑的建模，它可以驱动社区成员表现出理想的行为，并使集体追求和实现更大的目标。代币既可以用作会员徽章和访问通行证，也可以用作社区奖励和声誉信号。

有一种合理的担忧是，通过将人与人之间的互动代币化，参与和贡献于社区的内在动机将被新的经济激励所压倒，从根本上改变了驱动有机社区活力的行为动力。然而，代币化社区的案例并不需要以金融或经济逻辑开始或结束。

通过创建双重代币系统 ，声誉「积分」可以与社区「币」共存，社会资本可以映射为金融资本或股权。因此，社区可以发行「声誉代币」来认可成员的贡献，并向更广泛的社区发出一个人的社会地位的信号。通过分发「奖励代币或硬币」，会员可以享受同质化的好处，可以将这些代币或硬币兑换成现金，也可以兑换商品或额外的特权。结合起来，它们为社区提供了一个认可和激励成员高质量贡献的机会。

为了弄清社区如何奖励和传达社区中最重要的成员，我们面临着许多设计挑战 ：

  • 代币作为经济激励的作用

  • 代币的社会资本作用

  • 信誉可移植性

  • 全球性的、可识别的徽章

  • 为社区建设者创建一个工具包，为成员创建一个信号平台

建立自我维持的高吸引力社区

高吸引力社区是指通过提供出色的成员体验和增加成员之间的联系来吸引和留住成员的社区。社区的引力取决于它不断吸引成员接近核心的能力，包括让新成员更多地参与进来，以及让积极的贡献者成为社区领导者。在最好的情况下，引力会成为一种自我维持的力量，推动社区的发展。

比起单一的排行榜或积分系统，玩家可以参与各种社区活动并获得相关的「徽章」。这些徽章或标志是一种社会信号，是一种展现特定贡献和成就的方式。它们可以与权限和特权相关联，包括只向具有特定地位或声誉的社区成员提供特定的效用和利益，以及向那些积极参与和贡献的人授予特定的管理权利和角色。

总之，gate&gain 的核心思想不是降低社区的包容性，而是引入基于实际贡献价值划分和认可成员的方法，并相应地通过增加访问、权限、效用和所有权来奖励他们。因此它能够建立自我维持、不断扩大影响力和参与度的社区，同时在追求更大愿景和使命的过程中创造长期价值。
