背景

六月底，Polkadot Decoded 2023在哥本哈根顺利举行，其中最令人印象深刻的内容莫过于是作为惊喜嘉宾的波卡创始人Gavin Wood，为波卡的下一阶段发展提供了波卡2.0的愿景。这也让关注波卡生态的众人开始纷纷讨论和研究起来，波卡2.0会如何影响波卡的发展，它又能带来怎样的景象。

而另一方面，以太坊以及Layer2的发展也如火如荼，各个Layer2纷纷推出一键发链工具，似乎对波卡等多链体系发起冲击。并且，许多开发者甚至一直坚信以太坊会是未来最主要的叙事而一直围绕着以太坊以及Layer2进行Build。

波卡的2.0似乎略显式微，但真是如此吗？为此，我们Polkadot生态研究院在7月中旬前往上海参加了Polkadot Decoded 2023上海分会场活动，一方面带来了《Layer2后时代，什么才是波卡最核心的竞争力》的主题分享，正好说说Layer2与以太坊与波卡的一些思考；另一方面，也再次了解一下波卡生态当下都在做什么事，其中是否又有一些新的趋势正在形成。

当然最主要的是能够跟一线的波卡建设者进行面对面沟通，碰撞出新的火花，这样的机会最是难得。本文是此次上海之行的见闻，也将分享我们此次参会的直观感受。

Layer2后时代，什么才是波卡最核心的竞争力

1.Layer2的现状

首先还是简单回顾一下我们的分享，OP和Arbitrum的空投引爆了市场，也让Layer2成为当下最热门的话题，从TVL来看，Optimism以及Arbitrum等知名的Layer2已经赶超了许多Layer1了，其TVL也名列第四和第六。Layer2的发展风头正盛，已经逐渐形成一个比较丰富的多链生态系统。

然而以太坊与Layer2们组合成的以太坊生态也并不是表面上其乐融融。Layer2们其实并不是真的为了以太坊而去做的，所以Layer2们都有一些自己的想法。

  • OP Stack是Optimism团队提出的一个概念，指的是将Optimism作为一个独立的区块链平台，而不仅仅是一个Layer2解决方案，要做所有Layer1的Layer2，并且连接成为一个SuperChain。
  • Arbitrum团队也在探索在Arbitrum之上构建更多的层级，称为Layer3，将形成自己的多链生态体系，以提供更多的灵活性和选择性。
  • ZK Stack是一个免费的模块化开源框架，用于基于zkSync Era的代码构建自定义的ZK驱动的Layer2和Layer3链，最终将构建了一套由ZK科技定义的HyperChain愿景。

所以，如此看来，Layer2与以太坊的关系会比我们想的复杂许多，既是共生又是互相博弈，非常像古时候皇帝与其他藩王之间的制衡了。

其实，所有DeFi的TVL从最高近1800亿美元缩水到450亿美元左右，为什么在所有Layer1的TVL都在缩水时Layer2能逆势上涨，一大部分原因在于它是天然在吸血以太坊的。

得益于更高的性能和更低的费率，Layer2天然能够吸引以太坊上的资金、用户、开发者和应用迁移到Layer2上的，并且它相对于以太坊更进步的一点，在于它能让原本因为性能和费率问题无法在以太坊上运行的应用得以实现，如GMX等。

但是，尽管如此，Layer2本身的生态依然还是以太坊上已有的应用类型，如链游、DeFi、NFT等，对行业的发展拓展并未推动太多。

Layer2上的发展还属于是存量博弈的发展，主要还是在争夺存量市场，即已有的区块链用户、资金、应用。存量市场的竞争越来越激烈，导致了同质化、泡沫化、零和化等问题。尽管Layer2上诞生了ZK、AA相关的新技术，但是这些技术革新带来的市场增长空间相对有限，难以支撑区块链整个行业的长期发展和创新。

2.可能真正推动行业大发展的叙事

行业需要找到增量发展的逻辑：新的资金、新的用户、新的使用场景，需要找到真正推动行业大发展的叙事。

一是，去中心化基础设施的拓展。

去中心化基础设施是指为区块链生态提供基础的技术支持和服务的项目，如存储、计算、传输等。现有的区块链赛道放在互联网领域来看，只相当于其中的一小部分，更多的应用场景，有赖于基础设施的发展。正如苹果手机开创了智能手机时代后，才有了今后移动互联网的飞速发展，区块链也是如此。去中心化基础设施的发展会限制应用的拓展，因此当去中心化基础设施得到拓展，能实现大规模应用时，一定会为Web3带来全新的丰富应用。

二是，从Crypto行业外部引入更多的资源。

行业的发展离不开注入其中的资源，如果把Crypto行业看作是一个容器，资金、开发者、用户这种资源看作是水，那么只有为容器中注入更多的水，才能使水面升的更快。而水最多的地方就来自于传统企业。

两种外部引入资源的方式：

  • Web2.5：传统企业利用Web3技术与自身品牌、产品、商业模式等结合，从而开拓出新的营销方式或盈利模式。比如NBA Top Shot就是将传统的实体球星卡模式用Web3的方式实现。
  • RWA：Real World Asset的缩写，指的是将现实世界的资产映射到区块链上，实现更高的流动性、效率和可信度。

3.那么在这样的新叙事下，波卡又有哪些优势呢？

去中心化基础设施项目需要两个部分：区块链与资源网络（比如存储网络、计算网络），尤其是DePIN类（去中心化物理基础设施网络）项目为代表。其中，区块链的共识机制来判定如何奖励提供资源来支持网络的参与者。区块链的资产作为激励，区块链的技术作为智能合约平台。

波卡背后的Substrate框架，相比于OP Stack、ZK Stack、Cosmos SDK提供了更多的灵活性和可定制性，尤其是支持项目方能够使用自己特有的共识机制，因此受到众多去中心化基础设施的项目青睐。波卡生态中去中心化基础设施类也远多于以太坊生态和Cosmos生态。

波卡生态中已经有许多这个方向的项目，如：

  • 去中心化存储：CESS、Subspace、Crust等
  • 去中心化计算：Phala Network等
  • 去中心化网关入口：Deeper Network等

并且，近期的波卡黑客松中基于这些DePIN的应用类项目也正在不断涌现。

而在从外部引入资源这方面。Astar就正在努力引入日本传统企业进入Web3。此外，如Mythical Games、Deloitte、Vodafone、OpenZeppelin、EnergyWeb、Heathrow Airport等一些知名公司近期都公开与波卡建立合作关系，并在本次哥本哈根的Polkadot Decoded上，分享了如何借助波卡生态探索Web3的实践。

波卡生态中也有对RWA方向进行探索的项目，比如Centrifuge，主张打造真实世界的DeFi。提供了房地产等实物资产的通证化、借贷、交易等功能；又比如Pendulum旨在弥补法币（fiat）与DeFi中间缺失的一环，允许传统金融服务与DeFi无缝集成。

4.波卡的核心竞争力是什么

那么，经过刚才的梳理，我们可以总结出波卡的一些竞争力：

Substrate框架可以帮助波卡构建最丰富的的多链生态。丰富的生态意味着无论是怎样的叙事都可以发生在波卡生态。得益于安全性高于Layerzero等跨链协议，以及一般跨链桥的波卡XCMP，波卡能将这些生态项目以相对更具安全性地方式进行互操作。这将支持波卡逐渐构建起一个更具网络效应的多链生态系统，让更多的创新和新的叙事发生在波卡生态中。

所以，总结一句话：生态多样性以及安全的互操作性将会是波卡的核心竞争力。

会场的观察

在分享之后，我们也仔细聆听了众多团队对波卡生态项目近况的分享，也与他们在台下进行了更多的沟通，在这个过程中我们也发现了一些趋势。

1.DePIN百家争鸣

正如我们的分享中提到的波卡的优势，波卡生态中确实有很多有意思的DePIN项目正在积极发展中。不同于其他生态基本都是基于智能合约平台推出的应用，波卡生态中有许多基于DePIN的功能反而是实用且让人眼前一亮的。

比如去中心化网关入口Deeper Network也带来了他们新的硬件产品，即插即用，再也不用担心ChatGPT的注册问题了。

又比如CESS也演示了他们去中心化的存储的上传下载的实际效果，Crust也推出了其EVM Storage协议，这在去中心化存储的龙头Filecoin还主要停留在只是用作Mining的层面以及FVM还未普及的情况下，已经看出波卡生态的DePIN项目已经在奋起直追了。

又比如去中心化的计算网络Phala Network，可以像传统云服务那样推出的各类业务，Phala最近推出的Phat Contract是运行在Phala Network上的创新型链下智能合约，它提供预定义的组件（Phat Bricks）和可定制的模板（Blueprints），可以帮助开发者5分钟完成一个自定义Oracle的部署。

这样的通用产品，也帮助Phala为知名社交知识图谱Lens Protocol生态推出了Lens API Oracle，也让Phala的这款产品被加入到了Lens的官方文档中。

所以，波卡上的DePIN一些正在悄悄探索如何更好地面向C端商业化，另一方面也积极地与B端合作，让自己的产品得到更广泛的应用。

而这样的故事还有很多，只是对于不关注这方面的许多人，也许很难理解到他们正在做的这些事会带来怎样的影响。但是，对于我们从整个Crypto行业的发展来看，DePIN一定是可以行业更大发展的必经之路。

去中心化基础设施的叙事其实在几年前就已经喊过几次了，但是市场对它的接受度有限，并且由于一些明星项目的发展不太好，也让许多人对这一类方向比较灰心，但好在事物的发展不会受到个别人的情绪影响。

那些脚踏实地深耕在DePIN的项目，一定会在适当的时候重出江湖，希望届时大家放下过往的成见，好好审视一下这个赛道。

2.波卡生态正在积极的走出波卡

现在多链部署、跨链协议等技术已经不再是稀缺的技术方案了，这也意味着对于项目来说，不用只在一条链上做建设，而是可以更加开放地心态，将自身的业务或者功能拓展到更多的其他链上的生态。得益于波卡是兼容EVM的，许多波卡生态可以比较方便的走出波卡，与其他不同的区块链上的应用进行链接。

比如，Manta就已经通过推出一些带有zk功能的小工具，将业务拓展到不同的链上了。一些应用如Zenlink，也已开启对Arbitrum等Layer2的部署。而基础设施层面就更容易实现了，比如Subquery为多条链提供索引服务，支持项目的开发。刚刚提到的Phala的Len API Oracle，也是Phala Network赋能Polygon生态的一大案例。

尤其是基础设施层面，它会有一个资源网络，这些资源网络是可以支持不同的链使用，天然就能让项目的影响力扩展到其他链上。所以，或许有一个场景是各种在波卡上的应用最终会赋能整个区块链行业，尤其是DePIN基础设施。

不过也不用因此担心，波卡生态的这些行为是抛弃波卡，应该这样理解，这好比是某个城市的一个企业，它的业务可以拓展到其他城市，但是它的税收可以是只在某一个城市里。

所以，更加丰富的生态，并且建立更多的链接，才像是多链的未来嘛。波卡或许在为整个行业的大发展做好更充分的准备，也说不定。

后记

其实从以太坊上的一些知名应用的发展可以看到，一些应用所需要的解决方案，已经不能只靠智能合约来解决了，于是这些项目会反过来去找一些硬件的解决方案，这就又会回到去中心化基础设施的层面。

毕竟我们以前的小屏幕的按键手机，虽然上面仍然可以浏览网页，也有一些游戏，但是当年的这种手机的普及注定是没法实现移动互联网的热潮的，只有突破底层的限制，才有可能迎来更大的辉煌。

而近期Polygon也推出了zksupernet Stack，Starknet也宣布将推出Starknet应用链，Layer2的Stack也开始卷起来了。但是这依旧只是在智能合约平台层面上的发展，我们应该把视角拉长远，才能看到更全局的机会。目前来看，波卡还没有其他竞品可以替代，对于未来的大机会，波卡也在悄然为孕育着新的希望。
