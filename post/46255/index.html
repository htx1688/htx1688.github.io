就在刚刚过去的2022年12月，区块链预言机网络Chainlink推出了质押功能，以帮助提高平台预言机服务的经济安全性。

在发给Cointelegraph的公告中，Chainlink 表示，新的质押功能是其专注于安全和可持续增长的“Chainlink Economics 2.0”工作的一个组成部分。要知道在这之前，想要获得Chainlink ( LINK ) 代币奖励的Chainlink，用户需要启动自己的节点。

“随着网络的不断扩展，Chainlink Staking将继续发展，并在我们的生态系统和整个Web3中提供增强的安全性” Chainlink联合创始Sergey Nazarov表示，此次发布是Chainlink Economics 2.0的基础，最终会得到完善。

Chainlink Staking最初作为v0.1测试版推出，包括一个旨在保护以太坊主网内的ETH/USD 数据馈送的权益池。在LINK代币推出质押功能的前夕，Chainlink开始上涨。这似乎明里暗里都表明了质押功能的推出，将提振币圈圈友们对LINK代币和平台预言机服务的需求。

Sergey Nazarov是谁？他的Chainlink又何以在币圈享誉盛名，被大家奉为当今最成功的预言机呢？在过去的几年，币圈都在寻找比特币之父中本聪，一些报道对Sergey可能是比特币的发明者中本聪的猜测也逐渐增多。在本期重构研究院推出的重构加密大佬系列中，为大家带来这位曾被猜测为可能是中本聪的Sergey Nazarov。

从乐高积木到比特币

根据目前对于Sergey Nazarov的定义，最多的是将他定义为俄罗斯企业家。在币圈，他以共同创立成功的Chainlink和 SmartContract 项目而闻名，这些项目正在改变加密货币行业。特别是LINK的预言机对于DeFi和Web3的功能至关重要。

我们找了很多资料想要更详细的把Sergey Nazarov的信息带给大家，但想要全方位了解 Sergey Nazarov的身份却并不容易。这位Chainlink的创始人并没有过多谈论自己，他从未透露过他的出生地点和日期。不过，可以知道的是，他是生活在美国的俄罗斯移民。

1990年代初苏联解体后，他跟随家人从俄罗斯搬到了纽约。他的父母是工程师，也有有意为之的因素，也有耳濡目染的原因，Sergey从小就接触了计算机技术。这样的家庭氛围也让他对战略视频游戏产生了热情。在他的童年时期，他又对阴极射线电视机的工作原理产生了兴趣，将它们拆开并像乐高积木一样将它们重新组装起来，这是他的另一个爱好。

正当大家都以为他也会在后来的学习中将童年时期的爱好一脉相承时，然而这种“技术教育”却并没有体现在他后来的大学学习中，他反而是获得了纽约大学 (NYU)的哲学和管理学位。

大学毕业后，Sergey留在学术环境中，与Lawrence Lenihan一起在斯特恩商学院担任副教授。Lawrence不仅是教授，他还有另一个重要的身份——风险投资公司的首席执行官。作为公司的实习生，Sergey参与了多轮成功的投资，包括Pinterest和Riot Games的投资。

这段早期经历让Sergey为与QED Capital的合作做好了准备。QED Capital是另一家横跨俄罗斯和美国的风险投资公司。与此同时，Sergey的俄罗斯血统和经历在某种程度上将他与中本聪联系在一起。2011年7月30美元的飙升，促使QED公司对加密货币进行研究，尤其是挖矿。Sergey也正式开始接触比特币，他也效仿并开始在云挖矿中生成比特币区块。

跨入预言机领域：Chainlink

在比特币发布后仅两年的2014年，Sergey就创立了名为“SmartContract”的平台，当时就连以太坊都还不存在。某种程度上，是Sergey的这番作为奠定了这个伟大的工具的基础，使加密货币在满足某些条件时可以执行代码。然而，Sergey很快意识到，智能合约不应局限于区块链上的信息，因为这大大降低了其可能性。这也直接促成了后来的Chainlink——集中式预言机系统的创建。

2016年，他提出了一个解决方案，试图用他称之为“预言机”的东西来解决这个问题。虽然设计预言机的想法是革命性的，向前迈出了一大步，但Sergey很快意识到当时存在的解决方案是中心化的。那时候我们要获取信息的时候，大家第一反应都是到比如数据库之类的，一个很具体的地方找。

然而，这种方法存在一个巨大的问题：信息操纵的风险。确实，如果预言机要查找错误来源的信息，或者如果它被人恶意的修改，那么在区块链上传输的信息将因此是错误的。这将导致智能合约使用错误信息运行，从而无法提供可靠的结果。

为了解决这个问题，他决定创建Chainlink项目，一个去中心化的预言机网络。2017年，该产品被重塑为我们现在所知的Chainlink Network。

由于内部机制，外部信息由多个预言机传输并相互比较，对它们进行分析后汇总在一起，以便给出尽可能接近现实的结果。好的预言机会获得更好的声誉和更大的回报，而那些想在上面动手脚想耍小聪明的人会受到惩罚，严重点还可能被踢出系统。

预言机：会成为区块链行业发展的必备工具吗？

Sergey是最早提出智能合约想法的人之一，然后是预言机。他也能意识到，出于可靠性的原因，有必要将预言机去中心化，以便提供尽可能可靠的信息。很明显，Sergey的一系列作为促进了币圈面貌的改变，使币圈对整个社会做出了更多贡献。

Sergey的创造对整个币圈行业来说都是一件很有意义的事情。如果智能合约是一场革命，允许代码行在满足特定条件时自动运行，那么信息必须已经在区块链上。

以可靠的方式在区块链之外获取信息并将其整合，将所有的可能性进行预判，这为许多应用程序开拓了新思路。举个例子，比如说一家航空公司在延误两小时后，要向客户赔偿。这时候，通过在智能合约上编程并将其与预言机链接，预言机将能够判断飞机是否延误了，并允许是否进行自动退款。

通过以上的例子，我们能看到，Chainlink的预言机系统可以将现实世界与区块链世界连接起来。当然，企业需要开始对这项技术产生兴趣，并逐步开始大规模推广。但从长远来看，与预言机相关的智能合约没有理由不被大家、公司甚至国家组织使用。

Chainlink 2.0将使创建可以访问链下资源的混合智能合约成为可能。此外，新方法将使预言机更容易“仲裁链下预言机冲突” 。

不过，这种快速的成功，加上围绕Chainlink的观望心态，也在币圈引发了一些担忧。一方面，例如Buterin，表示该系统缺乏一致的方法来惩罚有缺陷的数据供应商。另一方面，Arcane Assets的Eric Wall一直也对Chainlink持批评态度。在2021年5月，他就指出该网络并非“加密经济安全”，理由是开发人员的状态以及该模型依赖于可信系统的事实。

尽管批评者表达了对这一领域的担忧，但很难说 Chainlink 尚未建立适合市场的产品。毕竟可扩展性和安全性是Chainlink 2.0对外宣称的最显著优势。

Sergey看好Chainlink与SWIFT的跨链合作

有争议，就有话题。Chainlink是市值和担保总价值最大的预言机项目，还有许多与之相关的加密项目。

2022年9月28日，Sergey与SWIFT的战略总监Jonathan Ehrenfeld Solé在SmartCon 2022大会上宣布了项目合作。Chainlink和环球银行间金融电信协会SWIFT合作开展跨链加密转移项目，试图弥合传统银行与区块链之间的鸿沟。

Solé在会议上表示，“机构投资者对数字资产有着不可否认的兴趣”，并补充说这些传统金融参与者希望在一个平台上访问数字资产和传统资产。

SWIFT银行间消息传递系统是传统跨境法币交易中最广泛使用的平台，连接了全球11000 多家银行。要知道，早在2022年8月份，该系统平均每天就已经能记录4480万条消息。

然而，SWIFT 网络上的交易可能需要几天时间才能完成，该公司还一直在探索区块链和DLT 技术 以及中央银行数字货币 (CBDC) 以促进更快的支付。

Sergey表示, Chainlink是一个易于集成的系统。始终为希望将其应用程序与 Chainlink集成并需要定制支持的团队提供帮助。

POC利用Chainlink的跨链互操作性协议(CCIP)允许SWIFT消息在几乎每个区块链网络中指示代币转移，Sergey表示这将加速分布式账本技术 ( DLT ) 区块链在资本市场和传统金融领域的采用。另一方面，Chainlink表示，与 SWIFT的这种合作使金融机构能够获得区块链功能，而无需替换、开发和集成新的连接到旧系统中，它表示这需要以“异常高”的成本进行大量修改。

在大多数币圈圈友看来，Sergey的Chainlink 提供的服务属于特定的利基市场，是因为预言机本质上迎合了所有使用智能合约的区块链，使得像 Chainlink 这样的平台的服务对其运营具有重要性。来自传统背景和加密领域的公司都认为智能合约具有相当大的意义，而且这种意义只会在未来增长。

大家将Sergey与他可能是中本聪的猜测联系起来的第一条证据，起源于reddit：互联网域名smartcontract.com在2008年10月25日被Sergey购买，这天正好是比特币白皮书发布前6天。以至于后来，这两个名字的首字母S和N之间的对应关系也加入了对其猜测是中本聪的证据之一。以及后来Sergey使用和中本聪同样的地理位置，也成为这种猜测的佐证之一。

但仅凭这些细微的佐证来证明Sergey是中本聪显然是不够的，但正是这样的一个个巧合，也足以激起币圈对于Sergey可能是中本聪的猜测的兴趣。

Sergey本人对此也表示：“我不知道那是谁，但如果非要我猜的话，很可能是一群人。”

对于Sergey来说，加密货币最让他感到兴奋的一点，便是加密货币从根本上改变了世界的工作方式。

在他看来，如今在金融市场上的繁荣和萧条是基于信息不对称，而信息不对称使极少数能够利用它们的人受益，而社会其他人要因此买单这样是不对的，是必须要改变的。

与此同时，新兴市场上的广大用户仍然缺乏合同系统的基本经济协议，混合智能合约的应用可以将世界上其他地方的人从0带到1，这也许是他在币圈中最值得为之奋斗的目标。

到这里，Sergey是不是中本聪就已经没那么重要了。

Chainlink虽然看好的呼声很高，但价格预测并不能替代尽职调查和研究。毕竟，Chainlink并不生产区块，只是“就数百个关于价格数据的预言机网络达成的共识” 。
