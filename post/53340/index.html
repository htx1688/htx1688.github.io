原文来源：Filecoin Network

在 FVM 启动之前，作为 FVM 主网先驱创建者的一部分， 60 多个项目团队在 FVM 的 Hyperspace 测试网络上进行构建。

2023 年 3 月 14 日，兼容 EVM 的 Filecoin 虚拟机(FVM)在主网上线。自 2023 年 2 月以来， 60 多个项目团队在推出之前在 Hyperspace 测试网络上通过 FVM 进行构建，他们是 FVM 主网创建者的一部分。FVM 主网创建者是一个为期 6 周的项目，为选定团队提供技术、产品、业务指导和资源。

作为不断发展的 Protocol Labs网络的一部分，并看到了来自 Filecoin 社区和其他社区的强烈兴趣，FVM 主网先驱者致力于支持在 FVM 上创建有影响力和可持续的解决方案。本项目招募的都是具有影响力理念、技术熟练和可靠的团队。

团队成员于 2023 年 4 月 25 日毕业，最后展示了创始人以存储为重点的项目，并向 Filecoin 社区开放了一个非常成功的推介日。这些项目展示了 FVM 带来的各种机会，包括质押协议、数据 DAO、分布式计算框架等等。

最先推出的是区块浏览器项目和质押和借贷协议，这两个项目一直在稳定获取用户。此后，又有 21 个 FVM 主网先驱项目在主网上启动，一旦合约验证和安全审计完成，将会有更多项目推出，从而确保协议的透明度和信任度。

这批团队成员毕业后，有 20 多个团队被纳入加速器项目，其中以 FVM 为重点的团队由 GraphPaperCapital、LongHash、Outlier Ventures运营。更多项目信息，请查看下方列表。

Creds Protocol ——在链上启用零知识身份解决方案

Creds Protocol 是一个分布式身份和验证协议，它利用零知识证明来提供一个保护隐私的解决方案。该协议允许用户在不向第三方泄露任何敏感信息的情况下证明其身份和其他属性。Creds Protocol 可以创建可验证的凭证(cred)，代表通过 FVM 访问内容的权利，可用于按次付费的用例等。基于 Creds Protocol，信用徽章是一种不可转让的代币(SBT)，代表可验证的权利。

ShaleProtocol ——生产就绪的 LLM 推理 API

Shale-Serve API 是面向开放式 LLM(大型语言模型)的生产就绪推理 API，目前支持 Vicuna-13 b。它可以在任何语言中作为 OpenAI API(特别是 GPT 3.5-turbo)的直接替代品，包括 Python、cURL 和 HTTP 请求。此外，该 API 还与 LangChain 进行了集成。该协议的目标是，截至 2025 年，通过支持成千上万的开放 LLM，打造一个可互操作的特定领域 LLM 网络。

Cookbook.dev ——FVM 智能合约中心

Cookbook 是一个开源智能合约注册处和市场。它允许用户搜索、贡献和集成任何 Solidity 智能合约至其应用程序中。Cookbook 已经建立了一个 Filecoin 生态系统中心来展示 FEVM 生态系统内的项目，任何开发人员都可以在此熟悉并开始构建驱动 Filecoin 网络的协议。

Resolutio ——打击 NFT 骗局的一站式解决方案

Resolutio 是一个分布式司法和社交平台，作为一项服务与 IPFS、NFT.storage 以及 FVM 一同建立，旨在克服保护艺术或文学作品的法律援助和法律工具的高成本和不可及性。Resolutio 建立了一个分布式争议解决平台(DDRP)。DDRP 在 FVM 上部署了解决纠纷的智能合约，使用 IPFS 进行证据管理，并使用 IPFS-http-client 和 NFT.storage 进行记录解决结果的决策 NFT。

Filemarket ——加密 NFT 的新代币标准

Filemarket 开发了一种协议，允许用户使用附加在 Filecoin 上的任何大小的加密文件来铸造 NFT，并且只能由 NFT 所有者独家访问。该解决方案可以让创作者在保护和保障知识产权的同时，将其内容变现。作为发布的一部分，他们现已推出 filebunnies 系列，这是一款酷炫的 Filecoin NFT 设计，其中包含加密的“宝藏”，并提供有限的免费铸币。

Trustified ——发行防篡改、可验证和永久的证书

Trustified 是一个无代码平台，旨在在区块链上发布防篡改、可普遍验证和永久的证书。一个无法验证、无法防篡改的文件并不能证明其真实性。通过 Trustified，这些数字证书在区块链上发布，存储在 Filecoin 上，并可以保持安全和永久访问。

LYNC ——快速构建、快速失败、快速发布Web3游戏

LYNC 正在构建 GameFi 基础设施，使区块链游戏对每个人都适用。让游戏开发者能够快速构建、快速发布和快速失败，而无需担心Web3的复杂性。LYNC 为启动Web3游戏提供了可扩展的基础设施，并完全支持 FVM，这对于部署存储在 Filecoin 上的 NFT 集合和动态 NFT 非常有用。

Clover ——DAO 的 G-suite 生产力

Clover 是一个用于 DAO 的分布式协作工具，与 FVM、IPFS 和Web3.storage 有内置的集成，旨在减少 DAO 在信息传播和决策方面的延迟。它使用 NFT-gating 进行身份验证，通过 IPFS 进行 DAO 存储，并与 [Huddle](https://fvm.huddle 01.com/) 的 dRTC 集成用于 DAO 会议。

MomentSwap ——分布式图像分享社交媒体平台

MomentSwap 是一个基于 FVM 的社交媒体平台，让分享美好的时刻变得简单且有利可图。该项目使用 FVM 的基础设施及其 EVM 兼容性来实现、调试和调用合约。用户数据存储在 Filecoin 网络上，并使用 The Graph 进行数据分析，实现完完全全分布式。有了 MomentSwap，您可以通过无缝和分布式体验完全控制您的资产。通过在分布式网络中的分享和影响力，还能获取额外收入。

Collectif DAO ——非托管流动租赁协议

Collectif DAO 通过将 FIL 作为抵押品租赁给网络中最复杂的存储提供者来优化 FIL 的盈利潜力。利用对 Filecoin 经济和尖端金融模型的深刻理解，Collectif DAO 为 FIL 代币持有者提供了无可比拟的高收费机会。借助 FVM，该平台强化了 FIL 的分布，促进了 Filecoin 生态系统的弹性和平衡增长。

STFIL ——具有复合回报的流动租赁

STFIL 协议是 FVM 推出的首批 Filecoin 流动性和杠杆交易协议之一。它让$FIL 代币持有者能够在没有锁定期的情况下赚取稳定费用，并为存储提供者提供抵押不足的 FIL，用于杠杆头寸。作为 FVM 的先驱者，STFIL 致力于提供分布式和无需信任的 FIL 租赁，同时确保更高的盈利能力。该协议由 Outlier Ventures 孵化，并在 Filecoin 和 IPFS 生态系统综述“月度优胜项目”中得到推荐。

Jackrabbit Finance ——分布式债券交易所

Jackrabbit 是 Filecoin 虚拟机上的一个分布式债券交易所(AMM)。它利用分布式债券代币(ERC-3475)为 Filecoin 存储提供者筹集资金。这便将传统金融的借贷方式引入了 DeFi。Jackrabbit 还支持原生 FVM 代币（交换和流动性池），从而强化 Filecoin 生态系统。
