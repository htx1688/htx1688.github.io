作者：Day

随着2020年的DeFi之夏，到2022年的NFT热潮，区块链基础设施和应用越来越繁荣，据defillama显示，DeFi总锁仓量达470亿美金，而在牛市峰值时，接近1800亿美金，据NFTGo数据显示，NFT总交易量达402亿美金。

由于刚经历过牛市，目前处于熊市阶段，相信下个牛市，DeFi和NFT的市场将会比现在更加庞大，随之带来的便是链上数据的暴增 ，链上数据分析也将成为区块链行业投资越来越基础的一个技能。

虽然，区块链标榜着公开透明，不可篡改，但面对行业内的各种繁杂的数据，对于绝大多数投资者来说，都会面对虽然数据在链上，但自己不一定能看的懂的问题。如何查看数据，以及去哪里找数据，还是有着不低的门槛。

工欲善其事，必先利其器，找对工具，可以让你花最少的时间来达到自己的目的。这篇文章，我们来整理下目前市面上用的比较多的链上分析网站。

基础 设施

  • 区块链浏览器 （Etherscan）

区块链浏览器，作为行业的基础设施，可以用于用于审查和分析存储在区块链上的所有数据。浏览器允许用户通过地址（字母数字或 ENS格式）、交易哈希、区块高度、token名称等搜索和检查链上历史信息。

可以说，行业内的所有数据的源头，其实都可以在区块链浏览器上查询到。区块链浏览器体现了区块链链上数据的可溯源和公开透明的优点。

浏览器的 用途：

1.查询： 通过浏览器查询我们日常转账的一系列信息，比如转账金额，手续费，出块时间等。

2.溯源： 查询转账记录，了解资金的流向。

3.分析： 通过浏览器查询大户以及大额交易，活跃地址数，链上交易笔数，token持有人数量等等一系列数据，并结合市场信息，进行投资分析。

  • 市场大数 据工具（CoinGecko）

作为区块链行业投资者，我们会时不时需要查询项目的官网，价格、市值、交易量等一系列信息。有需求就会有解决方案，市场大数据工具就是专门为我们提供这些服务的。

20年DeFi的崛起，使很多用户有了随时需要查各种项目合约的习惯，CoinGecko因为抓取信息速度快，准确，而在这段时间获得用户的偏爱，目前算行业内主流的加密货币数据聚合网站。

一些市场大数据工具被机构投资或者收购，在一些数据的客观性上存在一些问题，而CoinGecko相对其他市场大数据工具来说，数据客观性要好一些 。

另一方面，市面上最新出来的项目以及 相关基础信息，他都能及时的抓取到 ，查询起来比较方便，信息收录数据齐全，已收录12,319种 Token 数据，衍生功能多，开设了DeFi 、NFT 专区、新手教学专区、市场资讯推送等。

CoinGecko主 要功能：

1.查询 Token 基础信息： 包括项目官网，市值排名，token价格以及价格走势，上线平台等相关基础信息。

2.小白用户指南： 提供小白新手教学，词汇库，以及加密市场新闻等。帮助用户快速上手并跟进加密市场的动态。

DeFi数据

  • Dune analytic s（数据分析类工具）

Dune Analytics是一个集查询、可视化、共享和探索DeFi 和 NFT数据于一体的一站式平台。作为链上数据分析类工具，Dune的核心是平台对海量的链上deif和NFT原始数据进行查询和分析，然后将这些数据转化为 图表和仪表盘。 挖掘链上数据价值并为社区用户提供定制化数据分析服务。

虽然区块链的信息都是开源的，但要从海量数据中找出有价值的信息却非常困难。例如想知道某个项目的用户数，dex的成交量，对于非编程人员来说，还是太复杂。Dune Analytics 可以很好的解决上述问题。他 提供平台，抓取链上的所有数据，社区用户通过一定的编程基础来提取需要的数据，并将数据制成各种分析图表。

Dune analytics主 要功能：

1.社群控制面板（搜索功能） ：用户只需要根据自己需求进行关键字搜索，查询自己需要的内容，随着Dune的运行，目前社区用户已经创建了非常丰富的Dashboard 。也可以点击 DeFi、NFT 和以太坊等标签对 Dashboard 进行过滤筛选自己感兴趣的内容。

2.用户自定义控制面板： 当搜索不到自己需要的内容时，用户可以根据自己需求制作 Dashboard，监控感兴趣的链上数据。

二、DeFi Lla ma（ DeFi TVL 数据）

DeFi Llama 是专注于跟踪统计 DeFi TVL 的数，跟踪超过150条链和2000多个defi协议总价值锁定 (TVL) 数据和交易量数据，数据來源于Uniswap和SushiSwap，在技术方面也有Coingecko团队的支持，是目前行业内最大的TVL聚合器。

对于TVL的计算，由于新合约和新协议的不断涌出，各种抵押，锁仓的计算方式不同，导致真实的TVL其实很难被准确计算出来，但DeFi Llama在TVL数据方面的提供，目前算行业内比较准确的，大家一般以他的数据为准。而查看一个defi项目是否优质，TVL可以说一个非常重要的衡量标准。

DeFi LIama 主要 功能：

1.TVL（总锁仓量）查 询： DeFi Llama 可以查看到目前整个行业的绝大多数defi项目的各个时间段TVL的变化，整体排名以及每条链内各个协议的排名，每条链的TVL市场份额，根据板块，功能进行分类等一系列信息。点击每个协议，也可查询每个协议的基础数据，TVL变化，币种占比等信息。

比如当下非常热的arbitum生态，你可以通过查询找到arbitum生态目前哪些项目的TVL排名靠前，以及个协议的TVL变化等信息来判断哪些项目相对来说更有价值一些，辅助筛选项目。

2.协议收入查询： DeFi Llama 可以查看当下各个defi协议的收入，国库收入，活跃用户数，市值等一系列变化。20年defi summer的时候，大家判断一个defi项目是否有价值，主要是通过看其TVL变化，而经过熊市的洗礼之后，现在，判断一个defi项目是否有价值，就要看这个协议是否赚钱。

  • Messari（ 数据分析研究）

Messari 是一家数据分析和研究公司，利用开源工具构建了一套区块链数据库，其中储存的数据讯息包括各个区块链项目的资金、开发历史、管理架构、以及供应计划等。主要提供项目研究、新闻、指标和实时数据，帮助用户分析、研究加密行业前沿领域进展。他们的网站每月有数十万用户浏览，提供免费和付费订阅（年费300美金）。

Messari主要 功能：

1.研报分析： 进圈一段时间后，你肯定听过或者在社群刷到过Messari这个名字，在媒体和社群之间传的最多的，便是他们的研报。Messar的研报包括各个赛道的季度，年度进展，以及各个项目的研报以及当下热点概念等多种研报，对于行业最新动态把握及时。一些报告是收费内容，当然也有一部分是免费内容。

2.数据分析： Messari 监控所有顶级项目的关键来源和沟通渠道，覆盖范围包括主要的基础协议、DeFi 资产、最新进展等。一个项目的价格、数量、市值、波动性等，所有这些信息都有自己的指标和图表表示，帮助用户可以清晰的了解一个项目的进展。

NFT和Ga meFi

  • NFTGo（综合数 据平台）

NFTGo是一站式NFT数据聚合平台。NFTGo通过聚合，分析并呈现全网NFT数据，提供NFT搜索、数据可视化、稀有度查询、巨鲸追踪等功能。

主要 功能：

1.搜索： NFTGo对NFT市场整体数据统计较为全面且详细，实时采集全网NFT交易和资产数据，以用户可以看懂的方式将数据可视化，帮助用户判断NFT市场现状。用户通过搜索，可以了解NFT项目的基础信息。

2.排行榜功能： NFTGo拥有较全面的排行榜功能，涵盖各条链，实施交易量，gas消耗等多个维度的排行。

3.巨鲸追踪： 了解kol持有哪些NFT，实时追踪kol的仓位变化。

二、DappRadar（Da pp数据聚合平台）

DappRadar拥有部署在30多条链上的12000多款去中心化应用程序（Dapp），类别涵盖金融、游戏、收藏品、社交媒体等，月活用户超100万人次。作为Dapp发现平台和市场数据分析平台，DappRadar 可以有效帮助 Dapp 用户、了解市场数据并辅助用户做出市场决策。

主要功 能：

1.排行榜： DappRadar的排行榜功能汇总了各类Dapp的活跃用户量、交易次数、成交量等信息，可以帮助用户更好的发现、跟踪、分析最新Dapp；根据NFT交易量、平均价格和其他一系列指标计算NFT实时排名，方便用户发现并选择不同区块链上受欢迎的NFT。

2.Portfolio(地址追踪)： 输入地址，就可以分析监视该钱包资产价值变动、浏览DeFi资产与负债、NFT持仓等信息。

其他

  • ultrasound  

ultrasound可以用于查看以太坊燃烧统计（包括燃烧量，燃烧速度以及各类应用的燃烧排名）、gas费率变化、流通量变化（供应与发行量），以太坊总锁仓量（TVL）以及排名。主要用来查看以太坊燃烧数据，当发现以太坊突然变的拥堵的时候，你可以清楚的观察到，原因出在哪里。

  • l2 beat  

用来查看各个layer2的TVL锁仓数据以及排名。

  • Token Terminal  

主要提供defi协议收入数据图表，以及各个协议的活跃用户数，TVL，开发者等信息。

  • Nansen  

行业内非常不错的数据分析平台，有项目的各种数据图表分析，Nansen 最为突出的是他的 Smart Money功能，钱包地址分析功能非常强大，对大鲸的持仓变化捕捉非常迅速。把他放在这里是因为他的大多数功能都是收费的，一个月最低1000美金的费用阻拦了大多数用户。

  • 恐慌贪婪指数  

恐慌贪婪指数的范围在0-100，分数越低代表越恐惧，分数越高代表越贪婪。

0-24 分代表’极度恐惧’，意味投资者正在大量抛售。

25-49 分代表’恐惧’，意味着相当数量投资者正在抛售，而市场兴趣仍不大。

50-74 分代表’贪婪’，这意味着大量购买正在发生，导致价格上涨。

75-100 分代表‘极度贪婪’，表明市场非常火爆，市场‘泡沫’可能很快就会破裂。

恐慌贪婪指数是18年推出，一般性的数值变化其实没有什么参考价值，但出现极低（低于10）或者极高分时，大多时候离熊市低点以及牛市泡沫破灭阶段不远了。

以上是目前行业内比较热门的一些数据分析工具 ，有些 同类别，同赛道没有全部列出，只选了其中具有代表性之一， 随着区块链行业的不断发展，数据分析将成为每个投资者必备的基础技能之一，而准确的行情判断，必然是因为有好的工具辅助。

大多数人看到数据分析工具类的文章，第一反应是先收藏，建议不熟悉的读者，有时间自己体验一下各个产品，了解他们的功能和差异，相信对于个人调研分析能力会有一定提升，
