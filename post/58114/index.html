Odaily星球日报讯 香港持牌虚拟资产交易所 HashKey Exchange 在 X 平台（原 Twitter）发文表示，平台发布首日有大量用户访问官网进行注册并提交 KYC 验证，导致 KYC 审核出现拥堵。团队目前正调集更多资源优先处理 KYC 流程。

加密银行 SEBA 已获香港证监会原则性批准，可提供场外衍生品等加密相关产品

Odaily星球日报讯 瑞士加密银行 SEBA Bank 已获得香港证监会的原则性批准，以继续扩大其在亚洲的业务。
据 SEBA 官方透露，证监会已向其下发原则性批准函（AIP），这将允许该机构从事证券交易业务，包括加密货币相关产品，如场外衍生品和结构性产品。SEBA 还将能够就证券和虚拟资产提供建议，并为传统证券和虚拟资产的全权委托账户提供资产管理服务。
SEBA 亚太区首席执行官 Amy Yu 表示：“我们认为在今年年底前获得（正式批准）没有任何问题。”
Yu 补充说，香港和新加坡市场是 SEBA 在亚洲的首要目标：“在香港之后，我们将继续探索新加坡等其他司法管辖区的许可程序。”（The Block）

BlockFi 向法院申请将 ALGO、DOGE 等资产换成稳定币，以便用户提现

Odaily星球日报讯 已破产的加密借贷平台 BlockFi 于 8 月 29 日向法院提交申请，要求将用户账户中仅限交易（trade-only）的资产转换为稳定币，以便用户可以提取这些资产。
BlockFi 提议将 Algorand（ALGO）、Bitcoin Cash（BCH）、Dogecoin（DOGE）兑换成 GUSD 或其他稳定币，这些仅限交易资产的数量不超过 BlockFi 用户所有美国钱包资产的 0.5% 。（Cointelegraph）

韩国加密金融公司 Delio 存款规模约 6795 万美元，预计损失率为 30-50% 

Odaily星球日报讯 韩国加密金融公司 Delio 在暂停取款 2 个多月后首次披露该平台存款规模和损失预估。Delio 平台存款规模约为 900 亿韩元（约 6795 万美元），目前情况估计损失率为 30-50% 。
Delio 表示，最终损失率需要在完成运营资产回收和清算等措施后才能确认，因此目前只是一个估计值；这个数字不是最终的，可能会根据情况而变化；如果未来确定了最终损失规模，将会在此基础上准备额外的补偿。（ETtoday）

项目要闻

以太坊基金会正式推出以太坊执行层规范

Odaily星球日报讯 据官方消息，以太坊基金会宣布经过一年多的开发后正式推出以太坊执行层规范（Ethereum Execution Layer Specification，简称 EELS）。EELS 是以太坊执行客户端核心组件的 Python 参考实现，专注于可读性和清晰度，对程序员更友好并且与合并后分叉保持同步。
EELS 在每个分叉（包括即将到来的分叉）上提供协议的完整快照，使其比 EIP（仅提出更改）和生产客户端（通常在同一代码路径中混合多个分叉）更容易遵循。
以太坊基金会希望 EELS 成为指定核心 EIP 的默认方式、EIP 作者将其提案原型化的首选以及以太坊如何工作的最佳参考。

Avalanche 推出新跨链通信协议 Teleporter，现已开放测试

Odaily星球日报讯 据官方消息，Avalanche 宣布推出新跨链通信协议 Teleporter，现已开放测试。
Teleporter 基于 Avalanche 的原生通信协议 Avalanche Warp Messaging（AWM）构建，致力于为智能合约开发人员提供一种简单、原生的方式来调用 Avalanche 内其他基于 EVM 链上的合约。
开发人员已在 Fuji 测试网上设置了 3 个 EVM 子网（Amplify、Bulletin、Conduit），每个子网都配有 Teleporter 和 AWM Relayer，这些子网旨在成为部署与 Teleporter 交互的合约和测试集成的场所。Teleporter 和 Warp 预编译目前处于实验/测试阶段，在接下来的几个月中，团队计划将其推广到其他子网和 C 链。

在流动性迁移方面，Base 将与生态系统应用程序合作，提供从 USDbC 到 USDC 的流动性平滑过渡。Base Bridge 不会立即改变，将继续正常运作。

OpenSea 发布可赎回 NFT 标准，将向创作者推出链上可赎回支持

Odaily星球日报讯 据官方消息，OpenSea 近日表示将发布两项以太坊评论请求（ERC-7496 和 ERC-7498）和两项 Seaport 改进提案（SIP-14 和 SIP-15），以推动可赎回生态系统的发展。在接下来的几周，OpenSea 将与开发者和创作者合作，并征求反馈意见，以确保上述标准符合生态系统的需求。
