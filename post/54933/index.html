一览七大主要的 L2 网络：它们各自有哪些优势和潜力？

以太坊 Layer-2（L2）扩展解决方案正在蓬勃发展，将成为下一次牛市的最大亮点，那么如何在这个赛道寻找你的 100X？为了回答这个问题，我们将深入研究未来潜力最大的 7 个 L2 项目。

本文梳理自 Crypto 研究员 MooMs、0xFinish 在 Twitter 线程中的观点。

1. Optimism

2022 年 10 月，Optimism 推出了 OP Stack，这是一个为 OP 主网提供支持的开源开发堆栈。它引入了超级链的概念，超级链是指构建在 OP Stack 上的一组 L2 区块链（OP 链）。

该堆栈由各种代码库组成，可用于创建高度可定制的 L2 区块链。本质上，它的目的是简化 L2 区块链的创建。

2023 年 6 月 6 日的 Bedrock 升级是 OP Stack 的第一个正式版本，带来了多项关键改进：

• 费用降低

• 存款时间缩短 70%

• 改进证明的模块化

• 改进节点性能

Bedrock 升级后，Optimism 的下一步是升级到超级链。Optimism 的 L2 Rollup 链是超级链的第一个成员。Coinbase 即将推出主网的 Base 链将是第二个成员。

在筹集了 2.4 亿美元后，Worldcoin 也致力于在 OP Stack 上进行构建。币安的 BNB Chain 也推出了基于 OP Stack 的 EVM 兼容 OP 链 opBNB 的测试网。

最终目标是由使用 OP Stack 构建的各种 L2 网络组成“超级链”——所有 L2s 将共享排序、证明和跨链基础设施，促进各个网络之间的无缝通信。

Optimism 目前是以 TVL 计算的第三大 L2，但凭借其 OP Stack 的愿景和合作伙伴，它有望在未来几年成为 L2 的领导者。

太多人看好 Arbitrum，却忽视了 Optimism。Coinbase 和币安为 Optimism 带来了巨大的用户群体，我相信这只是一个开始。

2. Arbitrum

Arbitrum 是一个 Optimistic Rollup，目前是以 TVL 计算最大且最具主导地位的 L2 网络，于 2021 年 8 月上线主网。

Arbitrum 生态系统目前包含：

• Arbitrum One：整个 Arbitrum 生态系统的核心 Rollup

• Arbitrum Nova：针对交易量预期较高的项目的 Rollup

• Arbitrum Nitro：为 Arbitrum L2 提供支持的开源堆栈。

这里我们重点了解一下 Arbitrum Nova，其 Gas 费用比 Arbitrum One 低 90%。尽管 Arbitrum Nova 的安全性较低，但它非常适合游戏、社交和其他高带宽应用程序。Opensea 和 TreasureDAO 最近在 Arbitrum Nova 上推出了市场。

Arbitrum 的愿景是什么？Arbitrum 与 Optimism 一样，于 2023 年 3 月推出了一个可供所有人使用的开源框架，被称为“Arbitrum Orbit”，允许任何开发者创建和部署 L3，无需许可或正式批准。

理解 L3 的最简单方法是将它们视为本就是以太坊 L2 的 Arbitrum One 的 L2。它们大大提高了 Arbitrum One 的可扩展性，使 Arbitrum One 成为高吞吐量应用程序的最理想解决方案。

OP Stack 和 Arbitrum Orbit 之间的区别在于，L3 将构建在 Arbitrum One 之上，而 OP 链将是相互共享安全性的独立网络。

L3 未来可能会成为一个很好的叙事。然而，我认为 OP Stack 有更大的增长空间，因为它通过机构实体面向更广泛的用户。

Arbitrum vs Optimism

两条链的主要技术区别在于 Optimism 使用单轮欺诈证明，而 Arbitrum 使用多轮欺诈证明。

这意味着 Optimism 的方法速度更快，但由于 Gas 费更高，因此可能更昂贵，因为它是在 L1 上执行的。Arbitrum 的发放需要更多的时间，但更具成本效益。

此外，Optimism 使用 EVM，而 Arbitrum 拥有自己的 Arbitrum 虚拟机（AVM）。这意味着 Optimism 的编程语言仅限于 Solidity。Arbitrum 支持所有 EVM 编程语言。

3. Polygon

Polygon 是一个致力于增强以太坊可扩展性的区块链，他们通过利用不同的解决方案来实现这一目标。他们的旗舰产品是 Polygon PoS 侧链，目前每天处理 30-40 万个地址的 200-300 万笔交易。

Polygon 还涉足 dApp 链叙事，发布了他们自己的解决方案 Supernets（超级网），该解决方案使开发人员能够建立可定制的 dApps 链。

此外，Polygon zkEVM（其 EVM ZK-rollup 解决方案）于 3 月底上线主网，此后每天约有 20,000-50,000 笔交易。

最近的 Polygon 2.0 升级旨在统一这些平台以创建无缝的用户体验。

Polygon 2.0 的概念是由 ZK 技术支持的各种 L2 链组成的生态系统，利用独特的跨链协调协议来实现 Polygon zkEVM、Polygon PoS 和超级网之间的无缝互操作性。

Polygon 2.0 由四个协议层组成，每个协议层都在支持生态系统内的一个重要流程：

• 质押层：利用 Polygon 的原生代币 MATIC 为用户提供重新质押功能

• 互操作层：促进了生态系统内的跨链消息传递

• 执行层：使任何 Polygon 链都能够生成批量的有序交易，即区块

• 证明层：证明每个 Polygon 链的所有交易

我认为 Polygon 是 Optimism 大规模采用的主要竞争对手。他们已经与许多大型的 Web2 公司建立了合作伙伴关系，吸引了数百万用户。

4. Base

“Base 是一个安全、低成本、开发人员友好的以太坊 L2，旨在将下一个十亿用户带入 Web3。”

Base 由 Coinbase 基于 OP Stack 构建，并计划在未来几年逐步去中心化。

由于作为加密货币交易所的 Coinbase 有着轻松的“加密货币<->法定货币”闸道，因此 Coinbase 的 1.1 亿用户和 800 亿美元的资产可以轻松采用 Base。

Base 的测试网已于 2 月 23 日发布，预计在今年上线主网。

即便是在熊市，Coinbase 每月也有 830 万交易用户（根据 2022 年 12 月的数据），而在它的竞争对手方面，Polygon、Optisium、Arbitrum 的平均费用为 304 万，平均收入为 123 万美元，平均每日活跃用户约为 20 万人。

这也就意味着 Coinbase 只需将 830 万熊市用户中的 2.50% 转化为 Base 用户，才能达到这三个竞争对手的平均每日活跃用户数。因此，Base 有可能成为 DeFi（交易）领域主要的 L2 之一。

5. Mantle

Mantle 是一种新颖的模块化 L2 网络，旨在将以太坊的可扩展性提升到一个新的水平，同时保持高度的去中心化和强大的安全性。

Mantle 于 2023 年推出，它将区块链的三个关键层（共识、执行和数据可用性）分开，以允许独立优化：

• 共识层：其工作原理与 L1 链的工作方式类似，主要侧重于对交易和区块进行排序。

• 执行层：与其他两层隔离，专门用于执行交易和智能合约。

• 数据可用性层：基于创新的 DAS 技术，确保数据随时可用且不被隐瞒，提高网络的安全性。

Mantle 的模块化设计可能会改变 L2 赛道的游戏规则。通过将三个关键层分开以允许独立优化，它可以使网络更加灵活并提高可扩展性，而不会影响安全性或去中心化。

Mantle 还因其与去中心化自治组织 BitDAO 的合作关系而受到关注。这种战略合作伙伴关系可能会给 Mantle 网络带来巨大的价值，并巩固其在区块链生态系统中的地位。

Mantle 的一个独特之处是它专注于增强用户体验，特别是对于链游或 GameFi。随着 Mantle 的高可扩展性、更低的交易成本和更快的交易速度，它可能会彻底改变 GameFi 格局。

尽管 Mantle 是 L2 领域的新来者，但已显示出巨大的前景。其创新技术、对用户体验的关注以及对去中心化的承诺使其成为日益激烈的 L2 赛道中的有力竞争者。

6. opBNB

2023 年 6 月，币安宣布推出 opBNB 测试网。opBNB 基于 OP Stack 构建，非常注重保持与币安智能链（BSC）的兼容性，这为 BSC 开发者和用户提供了显著的优势。

与 Base 类似，opBNB 将利用币安交易所的庞大用户群，为数百万已经使用币安的用户提供无缝且经济高效的 L2 解决方案。这表明 opBNB 在 DeFi 生态系统中有着巨大的增长潜力。

值得注意的是，opBNB 的成功将与整个币安生态系统的原生代币 BNB 息息相关。如果 opBNB 获得关注，可能会对 BNB 的价值产生积极影响，从而对整个币安生态系统产生连锁反应。

根据路线图，opBNB 将在 2023 年启动主网。opBNB 主网的启动值得关注，因为它将展示币安的 L2 解决方案在高频交易、游戏和社交网络等领域中的表现。

7. zkSync

zkSync 是 Matter Labs 开发的 L2 扩展解决方案。它采用 zkRollup 技术，旨在提供以太坊级别的安全性。它是推动 L2 革命的关键项目之一。

zkSync 使用零知识证明技术将多个交易捆绑到一个证明中。通过将大部分计算工作移至链外，同时通过以太坊主网确保高级别的安全性，从而提高了可扩展性。

zkSync 的一个显著优势是它允许用户使用想要转移的代币来支付 Gas 费用，而不是必须使用 ETH。这极大地改善了用户体验，并使以太坊更接近主流采用。

Matter Labs 还致力于使 zkSync 无需硬分叉即可升级。这种模块化设计确保它可以融入未来的以太坊改进，并随着不断增长的以太坊生态系统进行扩展。

通过使用 PoS 共识机制进行区块验证，zkSync 坚持去中心化，与以太坊的核心价值观保持一致。

zkSync 的主要目标是完全无需许可，允许任何项目在其网络上构建和部署智能合约。这将进一步有助于以太坊的可扩展性和对多样化 dApp 的支持。

zkSync 的愿景不仅仅只是作为一个扩展解决方案；它是迈向更具可扩展性、安全性和用户友好性的以太坊网络的垫脚石。借助 zkSync，以太坊彻底改变各个行业的可能性是无限的。

关注 zkSync 这样的项目对于了解以太坊和 DeFi 的未来至关重要。随着以太坊的不断发展，zkSync 无疑将在以太坊的增长和采用中发挥至关重要的作用。

zkSync 不仅仅是 L2 项目的竞争对手，它还为 L2 的安全性、可扩展性和高效性设定了标准。随着以太坊生态系统继续发展，zkSync 值得长期关注。
