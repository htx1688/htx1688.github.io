周末愉快，今天给大家介绍一个DB DAO，或许我说起它的另外一个名字大家会很熟悉KurateDAO。一个由社区策划的基于区块链的Web3应用程序的数据库，其使命是使用加密经济来组织和策划世界的信息。

作为一个自治的去中心化组织，其提议类似于今天的维基百科。KurateDAO的创始人Michael Fischer解释说，该平台的想法是让一群用户创建一个地方，让他们可以搜索某个领域的工作、某种类型的音乐、特定的NFT、学术文章、加密货币钱包、烹饪食谱等。

换个角度说，社区打造了一种 Curate To Earn 模式。在计划的系统中，任何人都可以创建一个主题，例如一个国家的新闻、一个城市的餐馆，甚至是Web3工具。同一个人为要包含在其主题中的数据设置标准，成为策展人。

三个角色

所以想要搞懂DB DAO是做什么的，需要先跟着我认识一下三个有意思的身份角色，DB DAO中有三种不同的角色。每个人的报酬或支付都取决于他们提供的价值量：

  • 策划人

策展人创建一个数据库并提出规则来确定是否应该接受或拒绝内容。当球探无法解决时，策展人也是决定是否应该接受或拒绝行的最终裁决者。

  • 侦察员

侦察员查找并贡献符合数据库章程的内容。如果侦察员的内容被包含在数据库中，他们将根据其被浏览的程度按比例获得收入。侦察员监视数据库并在他们认为某行将被包含或排除在数据库中时进行质押。侦察员根据他们准确预测馆长的未来裁决的能力来赚钱。

  • 观众

浏览者支付代币以查看数据库的内容。代币支付给DAO国库，并在策展人、侦察员和网络之间分配。观众可以明确付费观看内容。 此外，他们可以通过观看广告付费。广告可以显示在内容旁边，可以存入国库，而不是由用户直接支付。

在一个稍有不同的版本中，观众也可以在“付费学习”的情况下为观看内容而付费。在这里，价值的流动是相反的。侦察员或策展人付费让他们的内容进入数据库，而观众则付费与之互动。这方面的例子是支付人们学习代码的奖学金，目的是在他们完成学习后通过安排他们从事编程工作来赚钱。

  • 利益相关者的奖励：

网络：奖金的10%。

策划人：资金池的20%。

侦察员：资金池的70%。

协议运作

DB DAO的优化是为了给浏览者提供最好的信息。其提出了一种新的自下而上和以用户为中心的数据库策展方式。

这个新的协议是透明的、分布式的，并根据用户贡献的价值对其进行奖励，从而适当地调整各方的激励。该协议如下：

  • 数据库是由具有特定目的的策展人创建的。

  • 一个侦察员将他们的row和一个代币一起加入数据库。

  • 后面的玩家可以通过投注两倍于前一个玩家的存款来投票移除该row。

  • 因此，第二个玩家可以投2个代币，将提议的row从数据库中删除。

  • 第三个玩家可以用4个代币投票以将其保留在数据库中。

  • 加倍继续进行，直到赞成派或反对派在超时时间内没有进行下一次投注而输掉。

  • 失败派系的资金由获胜派系瓜分。

  • 如果某行的代币池超过 X（比如 15）个代币阈值，则该row将交给管理者。

  • 策展人决定获胜派系，策展人和网络获得报酬，代币在获胜派系中分配。

当策展人和侦察员开发了一个有用的数据库时，它会吸引查看人。数据库是任何包含实际数据的 Web 应用程序的核心部分。最重要的是用户界面，它可以帮助人们访问该数据以及通过某种方式将数据货币化。

整体来说，任何人都可以提议向数据库添加row。当第一个玩家添加他们的元素时，他们用1个代币验证该元素是集合的一部分，所以它将被默认为包括在内。随后的玩家可以通过加倍验证元素的代币数量，将该元素从包含在集合中“翻转”为不包含在集合中。如果该元素之前是用1个代币验证的，那么它可以用2个代币排除。一个元素在被包含和被排除之间继续转变，变化的方式是将之前的押注值翻倍。翻倍一直持续到赞成派或反对派在人群超时时间内没有进行下一次投注而输掉。失败派系的代币由获胜派系分割。

然而，如果一个元素的池子在人群超时期之前超过了代币阈值，它就会被提交给策展人进行最终裁决。策展人决定获胜的派别，该元素要么被纳入，要么被排除在集合之外，获胜的派别会得到奖励。对于策展人所做的工作，策划者的决定会被写入日志链中，并且策展人会从集合中抽取一定的比例。

社区策划的链上数据库的好处：

允许贡献者对内容进行原生的货币化。DB DAO从一开始就内置了货币化功能，以支付策展人和侦察员的工作。一旦参与者组建了这些链上数据库，他们可以利用一套API和应用程序来建立可扩展的面向用户的Web3应用程序。

DB DAO在利益相关者之间建立了一个不可改变的社会契约。DB DAO的数据库由章程铸造而成。这些章程类似于Web2应用程序的服务条款（ToS）。与Web2的服务条款不同，附则是不可改变的。不变性使利益相关者确信，以后的服务条款不会被改变，以有利于不同的利益相关者。

使可扩展的内容审核民主化。DB DAO不需要一个人做所有的事情，而是扩大了可以参与策划过程的各方。

浅看一下

DB DAO/KurateDAO的策展模式类似于美国政府的三个部门，每个分支都作为其他分支的制衡。美国政府有行政（执行法律）、立法（制定法律）和司法（解释法律）三个分支。每个数据库都以数据库创建者制定的一组规则或章程开始。建立数据库时会建立章程，并且可以根据数据库的目的将其设置为可更改或固定的。这些章程由侦察员执行，当潜在的违规行为受到严重争议时，将由数据库策展人进行判断，后者做出裁决并为未来的侦察员建立先例。

Web3 是通过去中心化向用户承诺更多隐私和安全的互联网，通常只与他们将再次拥有自己的数据的想法相关联，从而减少了对大型科技公司、大型科技公司的控制。然而，在可预见的未来阶段，内容制作是最受益的。

在Web3上发布的内容以代币的形式注册在公共区块链上。也就是说，没有中央机构可以从空中删除内容。这不仅打击了任何审查行为，而且加强了文本的作者身份、内容的货币化及其在网络上的复制。

例如，作家Tim Denning在他的博客上说，文章可以作为NFT出售。将手动敲字的内容转化为不可替代的代币使其独一无二。这使得“在Web3上写作赚钱的机会无穷无尽”。

DB DAO就是有助于这一领域的飞速发展，许多应用程序可以使用KurateDAO创建，每个应用程序都将依靠创建一个策划的数据库。此外，由于KurateDAO，许多Web2应用程序可以被复制到Web3空间，并且将更加民主，参与者将得到代币的奖励。更重要的是，Web3策展将向所有人开放，让更多的人参与进来。
