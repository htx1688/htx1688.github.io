Web3 钱包追踪应用 Context 宣布其推出的 NFT 铸造聚合器 mint.fun 已开放通行证铸造，用户可在其官网免费 mint !fundrop 通行证，并通过后续交互获取积分，截止发文已有超 2 万地址成功 mint 该通行证，并获取了相应积分，且现在可以看到积分排行榜数据，该排行榜将于 32 日后结束，设置邮箱地址，邀请好友均可提高用户积分。

「!fundrop 通行证、交互获取积分」，或是在为以后空投埋伏笔，接下来为大家介绍一下 mint.fun。

mint.fun 介绍

mint.fun 是一个 NFT mint 聚合平台，用户连接钱包后，可以进行相应的 NFT 系列 mint，界面简单易操作，显示信息种类全面，以下是该平台的用例介绍。

对于创作者

对于创作者来说，创作者可以在 mint.fun 上启动你自己的 NFT 项目，如果你的 NFT 合约已经部署，可以通过 mingt.fun + 合约地址（mint.fun/contract address，合约地址需通过 Etherscan 验证）在 mint.fun 上线自己的 NFT 项目。

上线后，可以点击「Customize」自定义你的 NFT 信息（主题、页面布局、相关媒介），上线 mint 期间，你可以随时更改自己的 NFT 主题，但更改会导致 NFT 界面的布局元素重新排列，也可以通过编辑器恢复默认界面。mint.fun 目前支持以太坊 ERC-721 和 ERC-1155 合约。

对于收藏者

Mint 是 NFT 项目产生价值的方式，通常指收藏者通过钱包与 NFT 创作者的智能合约交互，从而获得藏品的过程，mint 价格由创作者定义，当然也有很多 NFT 系列是 free mint 的，mint.fun 的 minting 界面分为三类：免费 mint，用户钱包关注的 NFT mint、白名单 mint。

Free mint 就不多说了，大家应该都白嫖过一些项目的 NFT；

Following——用户钱包关注的 NFT mint 是指用户关注过的钱包地址近期 mint 过的 NFT，如果该 NFT 也上线了 mint.fun，那么用户可以直接在 mint.fun 上 mint；

Allowlist 是指用户此前获得一些 NFT 的白名单，有资格的用户也可以通过 mint.fun 完成该 NFT 的 mint，此过程需要验证钱包和邮箱地址。

许多系列 NFT 创作者都会预先定义供应量或 mint 截止日期，而其它系列则是可以永久铸造的「开放版本」。对于供应有限的项目会在很短一段时间内结束 mint，另一些项目可能在数周内完成，项目完成后，从该系列中获取 NFT 的唯一方法是在二级市场上购买。

NFT 相关的收藏品通常托管在类似 IPFS 这样去中心化文件网络上，也有一些是由智能合约代码本身以编程方式生成的，token 本身（即所有权）是在区块链上发行和交易。大多数 NFT 项目都在以太坊网络上运行，因此用户需要一个与以太坊兼容的钱包，例如 MetaMask、Coinbase Wallet 等来进行铸造。

mint.fun 现有优势

作为一个 NFT mint 聚合器，mint.fun 现有优势包括：

-Mint 聚合器的方式，一个界面 mint 多个 NFT，也会在未来上线更多 NFT 系列 mint；

-安全，mint.fun 会对每个项目运行自动安全检查、标记项目，把可疑项目进行标记以便用户谨慎交互；

-汇总项目的所有信息，包括项目启动时间、到目前为止已 mint 的百分比、unique minters 的数量、top minters、创作者钱包等等，也会向用户展示其 Twitter 好友正在 mint 哪类 NFT。

对于 mint Gas，项目创建者可以设定 mint 价格，许多项目都是免费铸造的。Gas 价格依据以太坊网络状态，mint.fun 不收取任何费用，将你的钱包直接连接到项目合约，意味着通过 mint.fun mint 的成本与通过项目自己的网站或其他第三方网站 mint 的成本相同。

而对于可以 mint 项目的详细信息，用户可以点击查看，mint 时间、总量、消耗的总 Gas 等信息，一定程度上也反映了该项目的受欢迎程度。

结语

mint.fun 是由 Context 推出的 NFT 铸造聚合器，去年宣布完成了 1950 万美元种子轮融资，由近期爆火的 OpenAI CEO Sam Altman 领投，Dragonfly Capital 等参投，投资方很硬核。

目前看来，mint.fun 上线的 NFT 项目并不多，人数也不多，但作为一个新的 NFT 类基础设施，该项目主打 mint 聚合器、信息透明且态度明确，比如用户遇见任何问题都可以私信其官方社交媒体提问。期待未来 mint.fun 会进化为更完善的应用和更多的 NFT 项目系列 mint。
