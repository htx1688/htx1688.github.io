最近看了大量 Web3 应用层 - 教育赛道的项目，也和很多相关创业团队聊过，感慨应用层破圈不易的同时，也通过 Web3 教育赛道动态看到了一些新的可能性。先简单总结几点 Web3 创投圈关于应用层的共识：

1. Web3 应用层的 Mass adoption 仍然需要关注具备破圈效应的应用，即新流量。

2. 纯靠 Meme、NFT 等热点已经很难吸引大量的新用户了；项目方仍然需要关注用户刚需，通过产品提供底层价值；或者找到更多的用户场景。

3. 区块链技术本身是生产关系的变革，很难直接 To C，还是需要和其他技术相结合；比如“区块链技术+物联网技术”、“区块链技术+AI”等。

4. 过去半年里，Web3 教育赛道也涌现出一批新项目，币安 LaunchPad 更是给教育赛道带来了大量关注，最近 Animoca 也大量布局教育赛道，比如和 Hooked 合作共建生态等，通过元宇宙与教育赛道的梦幻联动尝试破圈，值得关注后续落地情况。

总的来说，破圈仍然是重点，本文将重点探讨一下 Web3 教育赛道破圈的可能性。目前 Web3教育赛道主要有两个成长方向，一是 Web3 赛道本身的教育，通过学习降低用户进入 Web3 的门槛，从而实现赛道增长；另一个是传统教育内容 + Web3激励模式。

比如 Animoca 成立了千万美元学习基金，鼓励传统教育从业者提供学习内容。币安在熊市 LaunchPad 了两个教育赛道项目，给教育赛道吸引了足够多的关注度，但也引起了大量争议。不过过去半年里，教育赛道早期布局项目 Hooked 从 Wild Cash 的 quiz to earn ，到 ToDaMoon 直播学习再到最近结合 AI Chat 的产品 Hooked Academy Sensei，证明了教育赛道本身的可扩展性和可破圈性；Open Campus 则鼓励内容创作，展现了 Web3 教育赛道的新价值；除了币安 LaunchPad 的两个项目，NFTBooks、Dev、Dequest、ReadtoEarn 等等也都在默默成长。

站在项目方的角度去思考问题，其实所有应用层都在做一样的事情，即如何破圈？其中，用户留存和打造流量闭环都有系统的方法论，而最难破圈的点就是第一批用户从哪里来？这也是为何现在创投圈都越来越关注产品的实用性，因为有价值的产品更有可能获取初始用户。

之所以 Gamefi 一直是大家期待的 Web 3 新用户 On boarding 窗口，也是因为游戏的易上手性，用户教育门槛低，自然就容易破圈；不过由于熊市缺少造富效应，导致 Gamefi 的吸引力也一般。于是大家又开始寻找其他的破圈点，而教育就是目前大家下注的赛道。

除了币安布局了教育赛道，Web3 Game 龙头 Animoca 在教育赛道也有大量的布局。参考 Yat Siu 的推特以及 Animoca 的动态，可以看到 Animoca 和多个Web3教育协议有合作。比如 Animoca 成立的千万美元学习基金，大力支持Web3教育的发展，再比如，就在昨天， Animoca 宣布与 Hooked Protocol 进行战略合作，将结合创新教育娱乐，丰富 Web3 行业生态系统并推动大规模采用。Hooked 虽然在 IEO 之后饱受争议，但 Hooked 产品矩阵的确是目前教育赛道里相对成熟的，距最初的 Wild Cash，官方已推出了更多针对不同学习阶段和需求的产品，并且用户数据表现都不错。

同时，有留意到最近 Hooked 项目深度升级，业务版图扩张，除了持续在做的产品（即官方定义的 Academy 外），新增 Infra 及 Ecosystem 业务。简单来说，Infra 是给 Web2 项目 On boarding Web3 提供工具、技术支持及为新用户提供易用 Web3 工具，Academy 是给 C 端用户使用的丰富学习产品，Ecosystem 预计将是和其他项目方合作的重要通道。其中，最有可能率先破圈的产品一定是 C 端产品，比如近期推出的一款结合了 AI 技术的 Web3 专属学习 GPT Academy Sensei，可以针对 Web3 赛道提供最新的回答（ChatGPT 目前仅能追踪 2021 年前的数据信息）。并且和 ChatGPT 等不同的是，AI Sensei 有 4 个生动的人物设定，分别对应不同的专业领域，比如 Gamefi、NFT 等，使得每一个 Sensei 都术业有专攻。

回顾这次 Hooked 与 Animoca 战略合作，虽然官方尚未公布具体信息，但想象空间很足。比如 Web3 有一个老生常谈的话题，就是我们能不能打造 一个 Web3 的原生 IP？大家一直都寄希望于 Gamefi 或者 NFT 能够出一款 IP，但落地情况一直不理想。打造 Web3 IP 最大的难点就是如何找到高频刚需用户场景，然后在场景里融入Web3 元素；开发有趣的游戏是一个思路，但抓住用户的 Game 点属实不易。Hooked 非常值得期待的就是 AI Sensei 是具备这个潜力的，ChatGPT 的出现已经帮大家验证了类似工具的需求，而 AI Sensei 针对 Web3 推出数据更新的 GPT 产品，满足高频且刚需的需求。和 Web3 游戏龙头 Animoca 合作，那 Hooked 会获得更多的游戏动漫资源，Hooked 团队是可以把 AI Sensei 做得更加有趣，融入更多 Web3 元素的，那如此一来，Hooked 能不能抗下打造 Web3 Native IP 这个大旗？提供知识产品的同时，让大家有一个更有趣的的 Web3 AI 朋友？当然，这只是笔者的一个猜想，与 Animoca 的深度合作有非常丰富的想象空间，这点确实引发期待。

总的来说，Hooked 是教育赛道比较经典的案例，即深耕 Web3 教育、持续推出新产品并打造赛道矩阵，同时先人一步布局 Web3 教育赛道的基建和生态系统，从而引入更多 Web2 和 Web3 项目合作，做 C 端与 B 端的增量，形成商业生态闭环。而与 Animoca 的合作，也可以使得 Hooked 的愿景更快落地。

除了前文提到的努力打造 Web3 原生 IP， Hooked 生态和 Animoca 生态的链接也是双赢，比如 Web3 Game 开发者能够获得双方生态的支持，用户的体验空间也更大，一边在 Hooked 学习如何上手 Web3 工具，一边体验 Web3 Game，就像一个嵌入了使用教程的大型游戏乐园。

Hooked 和 Animoca 的尝试也是整个 Web3 应用层都在做的事情。说到底，就是努力获得第一波初始流量，然后努力的把用户留下来。Gamefi、NFT 已经有诸多先例了，但 Web3 还在为更大的破圈而努力，Hooked 的道路规划也是符合行业愿景的，教育赛道在 Animoca 和币安的双重加码下，更是有希望接棒成为下一个破圈点，希望教育赛道龙头 Hooked 能够成功走通，这对 Web3 和教育赛道都是好事。
