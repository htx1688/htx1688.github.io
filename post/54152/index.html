最近一段时间的市场行情跌跌涨涨，其中涌现出了一批有意思的项目。通常想要了解这个项目运行得怎么样，数据看板是一个非常有用的工具。

这篇文章将通过 Dune 这个工具，带你一览六月非常值得关注的几个数据。

01 Bitcoin Ordinal

作者：@thb0301

https://dune.com/thb 3100/bitcoin-nft-marketplace-overview? undefined=

这个看板关注的是比特币生态的 NFT 数据，包括 NFT 市场的排名、每日数据、每周数据等等。

02 Arbitrum One

作者：@sal_carrizo

https://dune.com/salva/ag

这是 Arbitrum 生态的数据看板，尽管 Arbitrum 已经发放了空投，但是作为 L2 的龙头，仍然是被大家极力看好，那么它的生态数据自然也就成了不可忽视的一部分。

03 Raft

作者：@OnchainDataNerd

https://dune.com/the_datanerd/raft

Raft 既是一个借贷协议，也是一个稳定币项目。它通过质押 stETH 以铸造与美元挂钩的稳定币 $R。

04 LSDfi

作者：@defi_mochi

https://dune.com/defimochi/lsdfi-summer

LSDfi 是流动性质押赛道，随着 ETH 2.0 质押的开启，这个赛道逐渐火热起来。该数据看板收集了当前热门的几个 LSDfi 项目的数据，包括 Lybra Finance、Unsheth、Origin DeFi、Alchemix、Gravita、Asymetrix、Pendle、Curve、Raft。

05 以太坊用户指标

作者：@web3_joshua

https://dune.com/joshua_web3/dau-mau

该看板包括了活跃地址、用户粘性等指标，以太坊主网的数据真实地反映了当前市场的现状，因此也是非常值得研究的内容。

06 蓝筹 NFT 数据

作者：@russwestwizzard

https://dune.com/russwest/blue-chip-nft-screener

最近 Azuki 项目方的搞事，也大家导致大家对于 NFT 的购买欲望再次下降，整个 NFT 市场也被带跌不少。

该看板收集了蓝筹 NFT 的相关数据，例如 CryptoPunks、BAYC、Doodles 等，既有整个蓝筹 NFT 地板价排名与价格变化，也有针对每个 NFT 项目单独的图表分析。

07 ERC-6551 

作者：@SeaLaunch_

https://dune.com/sealaunch/erc-6551

ERC-6551 是一个比较新的 NFT 协议，它主张为每一个 NFT 创建属于自己的智能合约，使其更具有可组合性、动态性和交互性。

Biteye 也在很久之前就发表过文章专门介绍过该协议，如果您想要详细了解，可以阅读下面的文章。

ERC-6551 ：下一个 NFT 的风口

Cryptokitties 联创：ERC 6551 带来 NFT 与 DID 新范式

08 Optimism Bedrock 迁移相关数据

作者：@optimismFND & @MSilb7  

https://dune.com/oplabspbc/optimism-bedrock-migration-impact-dashboard

最近币安推出的二层解决方案 opBNB 吸引了市场的目光，它就是基于 OP Stack 的 Bedrock 版本实现。

Bedrock 作为由 OP Labs 开发的新一代去中心化 Rollup 基础架构，旨在为 Layer 2 网络提供模块化、简单性和以太坊等效性。

在该看板中，作者也是竭力突出了 Bedrock 迁移之后手续费的降低、效率的提高，前后对比非常明显。

09 Worldcoin

作者：@subinium

https://dune.com/subinium/worldcoin-dashboard

Worldcoin 作为这些项目里面可能比较冷门的一个，光说名字，很多人一时间还反应不过来。但是他背后的那个男人，你肯定了解，那就是 OpenAI 的创始人——山姆·奥特曼。

因为有了 OpenAI 的加持，该项目也曾火过一段时间，但由于其需要虹膜验证，难度太大，导致逐渐被人们所遗忘。

该看板统计了 Worldcoin 账户目前的总用户量、每周增长用户、交易量等数据，帮助用户更加清晰地了解这个有点陌生的项目。

10 EigenLayer

作者：@sankin_eth

https://dune.com/sankin/eigenlayer-stage-1

EigenLayer 是一个以太坊再质押协议，已经拿到了六千多万美元的融资，本身也是属于上面 LSDfi 的范畴，但是由于该项目被很多人极力看好，并认为是潜在的大毛，因此使用单独的一个数据看板去研究它，也是非常有必要的。
