12月25日，1inch Network宣布推出Fusion升级。本文PANews将带你解析Fusion升级后的特性及新版代币经济学。

12月25日，头部链上聚合交易平台1inch Network宣布推出Fusion升级，用户可以在指定的价格和时间范围下单，无需支付网络费用，可以提升交易的成本效率、安全性和用户体验，同时1inch也对代币经济学进行了更新。

Fusion模式融合自限价单和聚合协议

新的交易模式取名“Fusion”，中文即为融合的意思，Fusion模式也是基于1inch交易引擎已有的两个产品1inch Limit Order Protocol（1inch限价单协议） 和 1inch Aggregation Protocol（1inch聚合协议）融合而来。

结合了Fusion模式的1inch交易引擎是一个去中心化的交易和撮合系统，通过专业的做市商引入中心化和去中心化的流动性。这一做法似乎和Hashflow有相似之处。

Fusion模式下又有三个选项，分别是Fast（快速）、Fair（公平）和Auction（拍卖），成交的时间逐步增加，相应的兑换比例可能也越来越优。

l如果选择快速，表示用户愿意接受不太理想的交易价格，以便在前面几个区块执行交易。

l如果选择公平，表示用户愿意等待一定的时间，以换取更理想的交易价格。一开始的交易价格可能会比较差，在交易执行的过程中可能会逐渐变得有利。

l如果选择拍卖，表示用户愿意等待最长的到期时间（10分钟），以便获得最优的交易价格。这一选项对大额交易比较有利。

Fusion最大的更新就是引入了荷兰式拍卖模型。用户的订单由专业的做市商（Resolvers）来完成，Gas费也由Resolvers支付，Resolvers从套利活动中获利。

要成为Resolvers，必须质押足够的1INCH代币到“feebank”合约中。Resolvers根据质押的1INCH代币数量和时间进行排序，排名靠前的有优先选择执行订单的权利。后期，1inch DAO可能会向Resolvers收取费用。

按这样理解，用户通过Fusion交易的优势包括：无需Gas费、MEV保护、没有价格影响、交易失败不收取费用、快速执行。

Resolvers也能享受到：赚取质押奖励、获得Resolvers费用、参与1inch Network Gas返还计划、获得 1inch DAO 治理投票权。

Fusion模式和Legacy模式对比

如今1inch中的交易已默认切换至Fusion模式。如下图所示，Fusion模式下快速、公平、拍卖模式下的交易都不需要用户支付Gas费。

但是这并不代表Resolvers会义务帮用户支付Gas费。以下图中分别在Fusion和Legacy模式下将1 WETH兑换为USDC为例，两者显示得到的代币的市场价值是相同的，均为1198.24 USDC。左边的Fusion模式不需要花费Gas费，而原来的Legacy模式下需要花费2.52美元的Gas费。但在市场价格不发生变化的情况，右图中Legacy模式下显示的价值即为实际得到的价值；而在左边Fusion模式下，实际得到的价值需参考下方的Est.received，这个值会在完成帮助用户完成交易后扣除Gas费和其它费用。

以下图中Fusion模式下快速将一笔USDC兑换为ETH的交易为例，智能合约直接将USDC在Uniswap V2兑换成了0.11261 ETH，并将其中的0.10793 ETH支付给用户，消耗的交易Gas费为0.00359 ETH，直接在用户的基础上套利0.00109 ETH，对用户并没有好处。

Fusion模式还有一些限制，比如不接受小额资产（低于100美元）的交易，因为Gas费由Resolvers支出，这会导致Resolvers的出价和实际价格相差较大。另外，Fusion也不支持原生代币的交易，ETH需要先兑换成WETH。

代币经济学的调整

本次更新同样调整了1inch Network的代币经济学。在新的治理体系下，用户可以将他们的1INCH代币质押一个月到两年不等，以获得Unicorn Power。

Unicorn Power可用于参加网络治理、可以获得Resolvers激励计划的奖励，也可以将Unicorn Power委托给他人。从Resolvers收取的费用可以根据他们对网络和Unicorn Power在利益相关者之间分配。

1inch Resolver激励计划从12月24日已经开始，Resolvers在Fusion模式下完成用户交易所花费的Gas可以得到返还，1inch基金会将为此分配1000万1INCH代币，每个Resolver每月最多可以得到100万1INCH代币的返还。

小结

综上，1inch的Fusion交易模式并不能凭空免除用户交易的Gas费，可能只适合大额的拍卖交易。目前来看，在Fusion快速交易的模式下，Resolver只是在链上帮助用户交易的基础上进行套利，用户交易后得到的资产比Legacy模式下支付Gas费后得到的资产更少。而Resolvers在这个过程中还可以获得1inch Resolvers激励计划的1INCH返还。
