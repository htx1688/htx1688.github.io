5月18日，Game Dosi正式发布，并同步开售会员服务，包括黄金和白金级NFT通行证，这些通行证可以提供提前体验新游戏等福利。Game Dosi是NFT平台LINE NEXT推出的首款Web3游戏平台，而后者是日本通讯巨头LINE致力于开发和扩展 NFT 生态系统的合资企业。

去年7月，Line Next宣布完成1000万美元战略融资，投资方包括SoftBank、NAVER、NAVER WEBTOON、NAVER Z、LINE Games、CJ ENM、 YG PLUS、SHINSEGAE、Hashed、K Auction，这10家公司将支持Line Next的推出面向全球的NFT平台“DOSI”的和基于IP内容的NFT项目开发。通过此次投资，Line Next也扩大了与IP内容、发行、游戏和娱乐公司在NFT内容业务和营销对接方面的战略合作。

构建Web3游戏一体化平台

GAME DOSI以“Gamers First，Web3 Next”为口号，旨在成为开发者和游戏玩家可以直接交流并建立信任的媒介。在确定开发人员的需求并投入大量精力为他们提供最佳商业模式后，GAME DOSI总结了自己的平台特色。

一体化平台

GAME DOSI 为游戏开发者提供了各种系统来轻松开发 Web3 游戏。开发者可以通过使用其提供的资源显着减少开发所需的资源量。项目方计划通过在入门、启动和运营方面支持 GAME DOSI生态，为用户和开发人员提供一体化平台。

  • 入驻：提供 Gamefi 平台 SDK 和 API

  • 推出：NFT 商店（B2C、C2C）、游戏代币交易和商品、集成系统

  • 运营：GAME DOSI Biz管理后台管理、游戏商品销售、资产利润管理一站式

法律咨询和风险管理指导

目前，基于区块链技术的加密和 NFT 市场规模正在快速增长。与此同时，防止意外事件发生的法规也在不断增加。因此，为了运营安全的区块链游戏，每个国家都需要新法规的合规策略。GAME DOSI 可通过其已推出的 DOSI 和 AVA 平台上的经验，先发制人地响应区块链法规，如钱包、令牌和 NFT。通过为进入 GAME DOSI 的游戏提供持续的监管检查和风险指南，帮助开发团队减少加密货币行业的问题并专注于游戏业务。

开发套件

GAME DOSI为游戏开发者提供 SDK 和 API，借助其开发工具包，游戏开发人员可以快速轻松地开发 Web3 游戏。

营销推广支持

除了游戏，GAME DOSI还计划在 NFT 和娱乐等多个领域利用 Web3 和区块链技术打造内容生态。得益于LINE Messenger 在全球移动通讯方面的影响力，以及对LINE Web3和区块链服务相关活动的支持。GAME DOSI 认为其作为 LINE 的主要Web3游戏平台拥有强大的力量。

代币经济咨询

基于LINK（LN）生态系统，团队支持设计针对游戏内容优化的代币和商品结构。该系统将通过 GAME DOSI 作为优化功能引入，并将陆续推出更新。

社区福利

GAME DOSI 计划通过与 NFT 平台开发和服务公司“0x&”的业务合作，引入多种不同的社区功能。

下半年将陆续推出五款游戏

根据声明，GAME DOSI将在今年下半年陆续推出五款新游戏，包括 Sweet Monster Guardians、VLO（Vestria the Last Order）、KEROZ、Drawshop Kingdom Reverse 和 Project GD，后者是 Line 内部开发的游戏，将涵盖基于 GAME DOSI IP 的许多游戏。所有游戏都将基于LINE 去年12月发布第三代区块链主网“Finschia”。

  • Sweet Monster Guardians：一款战略防御类型的网页游戏，用户可以通过拖放角色卡来用独特的玩具娃娃保卫城镇，由Movisoft开发，预计6月上线。

  • VLO（Vestria the Last Order）：集体RPG（角色扮演游戏）端游，通过动态实时PVP内容提供动态战斗体验，玩家可在战争场景中进行各种冒险和战斗，由SIS Studio开发，预计第四季度上线。

  • KEROZ：Crypto 中第一个基于 Hack & Slash （砍杀游戏类型）的Roguelike（角色扮演游戏），提供壮观的动作游戏体验，由Superflex开发，将支持PC和移动端，预计9月上线。

  • Drawshop Kingdom Reverse：一款基于区块链的元宇宙网页游戏，用户可以创建自己的 2D 点状艺术画廊，可用于装饰您自己的土地，并通过头像、机器和玩偶赚取 NFT，由Nolatency Limited开发，预计第四季度上线。

  • Project GD：使用GAME DOSI旗下IP的多款游戏开发计划，首推的游戏是TCG，玩家可以使用他们的 NFT 卡组相互决斗，预计7月上线。

此外，LINE NEXT 还计划在今年下半年发布第二批新游戏的预告。

在进入Web3游戏赛道前，LINE已在游戏行业积累数年。其Line Game社交游戏平台在2013年时，累计游戏下载量就已突破1.5亿。2018年，Line将其现有的游戏部门与《Dragon Flight》、《天命之子》的韩国开发商NextFloor合并成立LINE GAMES，着手开发手游，PC，游戏机等游戏。同年，Line Games还获得了来自Anchor Equity Partners的1250亿韩元（约合1.1亿美元）融资。

今年3月，由LINE Games开发和发行的大型多人在线 RPG 游戏《大航海时代：起源》（Uncharted Waters Origin）上线Steam，玩家可免费下载国际服。靠着大航海时代系列IP的情怀加成，这款网络游戏在初期引起了许多玩家的关注和投入。不过，由于国际服的游戏充值要数倍于韩服等问题，大量玩家开始陆续在Steam上给出负面评价，目前仍有超过半数的差评。

GAME DOSI将要发布的五款游戏目前尚未披露更多信息，根据路线图，全部游戏在年底才能上线。
