曾几何时，语音助手成为手机、电脑等智能设备上必备的功能之一，一时间之间，苹果的 Siri、微软的 Cortana（小娜）、Google 的 Assistant、三星的 Bixby、小米的小爱、华为的 Ceila 等层出不穷。

现如今，在生成式 AI 如 ChatGPT、Copilot 的冲击下，这些语音助手倘若赶不上时代的潮流，那注定要被淹没在历史的长河之中，微软的 Cortana 便是活生生的一个例子。

据外媒 Windows Last 报道，微软加快了淘汰 Cortana 的计划，其最新宣布在 Windows 11 取消了独立的 Cortana 应用程序，而淘汰掉它的正是微软如今在主力推荐的 Windows Copilot。

这也让众人将 Cortana 定义为首个被 AI 淘汰掉的个人语音助理。

Windows 11 上 Cortana 已停用

Cortana 是微软在 2014 年 4 月举办的 Build 开发者大会上首次亮相的个人助理产品。同年，微软发布了“小娜”这个名字，作为 Cortana 在中国大陆使用的中文名。

彼时一经发布，便吸引了很多人的关注。这款号称“全球首款跨平台智能个人助理” Cortana，可以设置提醒，无需键盘输入即可识别语音，也能回答问题、执行简单的任务。

另外，有些类似于 iOS 上的 Siri 和 Android 系统中的 Google Now， Cortana 最初并不是搭载在台式机和笔记本电脑上，而是在微软命运多舛的 Windows Phone 8.1 操作系统上面世的。

也许很多人并不陌生，Cortana 这个代号取自《光环》系列游戏中陪伴着主角士官长的人工智能的名字。

不过，Cortana 名声虽然打得响亮，但是并未捕获多少用户的心，这迫使微软不得不重新审视 Cortana。

  • 2019 年，Cortana 与 Windows Search 脱钩（这意味着你必须专门调用 Cortana 才能使用它），那年夏天，它从 Xbox 平台上撤下。
  • 2020 年 2 月，微软宣布，随着 Cortana 被重新定位为 “Microsoft 365 中的个人生产力助手”，包括音乐和家庭控制在内的各种消费者 “技能"将从 Cortana 中移除。
  • 在随后发布的 Windows 10 中，Cortana 与 Windows Search 完全分离，作为一个独立的应用运行。
  • 2021 年 3 月，Harmon Kardon 通过固件更新将 Cortana 从其 Invoke 扬声器中移除，不仅从新扬声器中移除，还从现有扬声器中移除。
  • 同年，微软官宣其面向 iOS 和 Android 系统 Cortana 应用将在 2021 年 3 月 31 日正式停止服务。

紧接着，在今年 6 月，微软又把注意力集中到了 Windows 平台上的 Cortana，欲痛下“杀手” 。

彼时，微软宣布：

“我们正在对 Windows 进行一些更改，这将影响 Cortana 应用程序的用户。从 2023 年底开始，我们将不再支持 Windows 中的 Cortana 作为独立应用程序……”

如今才 8 月，微软便开始“迫不及待”了。随着微软向部分 Windows 11 用户推出的新 Microsoft Store 更新关闭了独立的个人助理应用程序。

这意味着，当前如果你在 Windows 11 系统上安装了这款应用程序，那么是时候卸载了，因为当你打开时，会显示一条 Cortana 已被弃用的弹窗消息。

Windows 10 虽然还在，但是也存活不了多久

值得一提的是，虽然该应用程序已停止在 Windows 11 上运行，但它仍可在 Windows 10 上运行，并计划在 2025 年 10 月之前接收到更新。

然而，WindowsLast 透露，一位接近微软的消息人士证实，微软计划在几周内关闭 Windows 10 上的该应用。

此外，微软虽已经从 Android 版的 Microsoft Launcher 和其他地方移除了 Cortana，但在 Outlook、Teams 和 Teams rooms 等移动应用中仍可使用。

什么导致了 Cortana 的“死亡”？

作为苹果 Siri 曾经强有力的竞品，Cortana 的失败，也让众人唏嘘不已，究竟是什么杀死了 Cortana？

  • 有人说，是产品问题；
  • 也有人说，是微软管理问题；
  • 还有人说，是 AI 淘汰掉的它。

其实这些年间，我们所提及 Cortana 品牌之处，微软都带着 AI 或者机器学习的影子。

所以，作为一款早在多年前已经开始 AI 探索的前沿产品，微软自 2015 年起，就相继把 Cortana 带到了 Windows 10、iOS 和 Android 设备以及 Xbox 游戏机设备上，其实具有很大的优势。

按理来说，微软倘若好好专注开发，Cortana 应该会像今天的 Siri 一样，想着怎么和 ChatGPT 等生成式 AI 产品相结合，而不是走向下坡路。

现实来看，Cortana 的失败也有多层原因。

其一，微软的野心，希望 Cortana 可以统治一切

从微软早期的战略中，我们不难发现，微软对于 Cortana 的定位远不局限于电脑、游戏机和移动设备的范畴，而是有更大的规划。

那时，微软希望 Cortana 能够做到亚马逊的 Alexa 所能做到的一切，包括控制智能家居、内置到扬声器中帮助人们管理生活，以及为个人和企业提供无数的 “技能”，让他们能够执行诸如安排会议、搜索信息、预订机票等任务。

那时，Alexa 遥遥领先于 Cortana。据 ComputerWorld 报道，为了借助 Alexa 的成功，微软在 2017 年与亚马逊达成协议，允许 Cortana 与 Alexa 协同工作，使用 Alexa 的功能。

但即便是这项协议也无法让 Cortana 取得成功。消费者和企业根本不感兴趣。尽管微软拥有强大的市场影响力，但它只能说服一家公司在智能扬声器中内置 Cortana，那就是 2017 年 10 月推出的 Harmon Kardon Invoke。

不过，由于买的人太少，以至于在发布后不久，它的价格就被砍掉了一半，从 200 美元降到了 100 美元。这也无济于事。

根据 Canalys 市场研究公司的数据，2018 年第一季度，内置谷歌技术的智能音箱售出了 320 万台，内置 Alexa 的音箱售出了 280 万台，而 Cortana 音箱的销量太小，几乎可以忽略不计。

这不仅让 Cortana 落败，也让微软感到了沮丧。

其二，移动端的失利，扼杀了 Cortana 最重要的平台

与此同时，2015 年初，微软将操作系统被重新命名为 Windows 10 Mobile，并推动与 Windows 10 的互操作性，但无论是在应用开发者还是终端用户中，微软都无法从 Android 与 iOS 的竞争中获得优势。

基于此，微软在 2017 年停止了 Windows Mobile 的开发，并在 2020 年初停止了对该操作系统的支持，宣告移动端彻底失败，从而也扼杀了最有利 Cortana 扩展的平台。

其三，以键鼠操作为主桌面平台，Cortana 作为独立的应用程序，更没有优势可言

微软在移动端失利之后，继续将 Cortana 与自家更多的业务服务和生产力应用集成，包括 Edge 浏览器、Office 365/Microsoft 365、Skype、Teams、Dynamics 365 和 Azure 等等。

同时，微软甚至为消费者和企业提供了 Cortana Skills Kit，以扩展 Cortana 的功能。

但是在以键鼠操作为主桌面平台，且 Cortana 并非系统自带，而是独立的一款应用程序存在，在明眼看出微软欲放弃之际，我们也深知语音交互在桌面端占不了太大的优势。

对此，CCS Insight 首席分析师 Angela Ashenden 曾指出，Cortana 无法与苹果 Siri 或亚马逊 Alexa 竞争的部分原因是“微软没有像苹果和亚马逊那样关注消费者，但也因为笔记本电脑中的助手概念从未像手机或智能音箱那样流行。在 Windows 中模糊 Cortana 和搜索也没有帮助。”

Cortana 谢幕，Windows Copilot 登场

除了以上的几个重要因素之外，微软于今年带来的 Windows Copilot 成为压死 Cortana 的最后一根稻草。

微软首席产品官 Panos Panay 在今年早些时候的 CES 上表示，人工智能“将重塑你在 Windows 上做一切事情的方式”。

今年 3 月，微软联合 OpenAI，使用大模型对自家的新 Bing 和 Edge 进行了升级，带来了 Bing Chat 等类似于 Cortana 的功能。

此外，今年晚些时候，Windows 11 版本 23H2 还将推出 Windows Copilot，这是一个集成在操作系统中的侧边栏工具，可以帮助用户完成各种任务，如内容摘要、重写、解释等，会比 Cortana 更有用。

使用大模型等生成式 AI 产品已然成为行业中不可避免的趋势，就在几天前，有媒体报道，Google 计划对其智能助手 Assistant 进行大修，以加入类似 ChatGPT 和 Bard 的生成式 AI 技术，作为这一计划的一部分，Google 将会开始重组其 Assistant 团队，少许人会被裁掉；据外媒 Insider 报道，亚马逊也对其 Alexa 部门进行了裁员，现如今亚马逊 CEO Andy Jassy 目前正直接领导一支新团队，全力打造大语言模型…..

不过，对于 Cortana 的退场，有网友表示，“Windows Copilot 出来，就已经预料到今天了，但遗憾还是有的，只希望亚马逊不要轻易‘放弃’，否则有数千万台 Alex 设备就直接变成砖了！”
