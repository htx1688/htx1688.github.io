主要内容

  • 以太坊代币标准 「ERC-4907」 在 6 月下旬成为 “Final”（最终版）状态，打开了可租赁 NFTs 的潜力。

  • 一旦在智能合约中实施，可租赁 NFTs 可以推动基于区块链的游戏、元宇宙土地销售等的发展。

6月下旬，一个名为 「ERC-4907」 的新以太坊代币标准成为状态为 “Final” 的标准；该标准将允许用户借出他们的非同质化代币（NFT）。

「ERC-4907」在 NFT 标准中增加了一个新的角色，将 NFT 的所有权和使用权分开——使“租赁”成为可能。 租用者可以使用 NFT 直到租借期满，NFT 也会在租借期满后自动归还其所有者。

在 「ERC-4907」出现之前，每次你转让 NFT 时，你都会失去其所有权，“Double Protocol（一家建立可租赁 NFT 基础设施的初创公司，也是新代币标准的开发者）的发展负责人 Lareina 说道。但通过分割所有权和使用权，「ERC-4907」标准的 NFT 能以无许可的方式被借出和租用。

可以肯定的是，可租赁 NFTs 此前尚未正式可用。达到 “最终 “状态意味着以太坊提案，或旨在改善以太坊区块链的项目，不能再被更新。其他构建者现在可以将该提案纳入智能合约，而不用担心开发者之后会改变它。

一旦将其构建到智能合约中，可租用的 NFT 有可能颠覆区块链游戏、元宇宙土地销售和基于区块链的媒体平台等。

如何制作可出租的 NFT

大多数知名度很高的 NFT 项目，如 Bored Ape Yacht Club、CryptoPunks 和 Azuki，都是以「ERC-721」代币标准铸造或包装成的，它代表了数字或物理资产的所有权，可以在以太坊区块链上得到验证。

这些代币标准规定，NFT 的所有者可以使用该数字资产。比如说，为了使用 play-to-earn 游戏 Axie Infinity 中的代币 Axie #5，你的钱包里必须有该  NFT。要在钱包里有 NFT，你必须铸造该 Axie 或在二级市场上购买它。

换句话说，购买该 Axie 是你能够使用它的基础——就像其他根据「ERC-721」标准铸造的NFT一样。

而根据「ERC-4907」代币标准铸造的 NFT 分割了数字资产所有权和使用权。「ERC-4907」代币标准可以包裹在根据「ERC-721」或其他以太坊标准铸造的现有 NFT 上，允许被包裹的 NFT 被可租赁 NFT 市场的智能合约读取，并被借用或租用——就像它最初是根据 「ERC-4907」标准铸造的一样。

此外，所有者可以设置时间参数，以决定租借人可以使用其租用 NFT 的时间。当时间结束时，用户不能再使用该 NFT，而租用的 NFT 又会回到原所有者手中——所有者无需手动要求归还他们的资产。

可租用的 NFT 的使用场景及未来展望

分享 NFT 的使用权并不新鲜。游戏公会购买对大多数玩家来说过于昂贵的 NFT，并将其借出以换取玩家的部分利润。这样的公会，如 Yield Guild Games（YGG），在去年夏末秋初与 Axie Infinity 相继出现，当时该游戏的人气正处于高峰。

“当所有者将（他们的NFT的）权利转让给公会时，所面临的风险很大，“Double Protocol 的构建者Shrug Newton 在采访中告诉 The Block：“你必须信任公会。如果公会消失了，那么你就会失去你的资产。” 

然而，「ERC-4907」允许用户使用他们拥有的有价值的 NFT 手动创建工会，其中借出者可以收取租赁费，而用户可以保留从游戏中获得的利润。Lareina 说，他们可以通过利用以太坊智能合约，以无信任和无需许可的方式来做到这件事。

虽然这似乎是对公会商业模式的威胁，但 Double Protocol 认为基于区块链的游戏公会可以成为可租赁NFT 的做市商——买卖证券并经常作为提供流动性的实体。而像 YGG 这样成熟的公会组织可以利用「ERC-4907」使他们的公会更加去中心化。

Lareina 补充说，Double Protocol 认为游戏公会是可租赁 NFT 市场的大利益相关者，因此也希望尽量迎合他们。这可能包括创建工具，让公会组织可以通过点击一个按钮来创建新的公会，以及控制游戏收入在借出者和租用者之间的分配。

除了游戏之外，可租赁 NFT 也可以增加元宇宙土地的使用。Lareina 指出，元宇宙的土地所有者可以将他们的财产出租用于活动。如果投资者购买了一块元宇宙土地，但缺乏在其上进行建设的能力，他们可以将这块土地出租给另一方进行建造并赚取租金收入。与此同时，土地所有者也可以通过在租赁期内移除他们元宇宙土地上所拥有的任何其他财产，并在租赁期结束后将其重新加入来保护这些资产。

Lareina 补充说，这项技术还可以促进基于Web3 的图书借阅或新产品的免费试用，以及其他领域。

总而言之，可租用的 NFTs 解锁了 NFT 使用权的新市场，因为它已经与所有权脱钩。Lareina说：“我们正在这个理念的基础上构建我们的产品。”
