原文标题：《WhatBigPublishersThinkofBlockchainGaming》

原文作者：DevinBecker

原文编译：Yvonne，MarsBit

在过去 6 个月，Web3 游戏开发持续降温，但一个值得研究的领域是 AAA 游戏发行商的立场，他们在 2021 年底看好这项技术。当 Axie Infinity 引爆区块链游戏时（尤其是游戏中的 play-to-earn 模式），大多数大型发行商采取了观望态度。当市场在 2022 年初开始回调，大多数公司停止对该领域进行任何新的开发，并将其主要留给了规模较小、更专注的开发者。

现在一段时间过去了，大多数 AAA 级发行商似乎已经转变方向。然而，一些人仍然坚定地推动前进。这些具有开创精神的发行商决心解决问题，我们预计，尽管仍然存在障碍，但在接下来的几年里，他们可能会有足够的动力推动其他人谨慎地进入该领域。虽然持续的势头主要来自东方，但一家大型西方发行商也在继续投资。

育碧仍在边缘徘徊

育碧一直以来都积极拥抱新技术和平台。例如，其涉足云游戏，为短暂存在的 Stadia 平台做出了贡献；其订阅服务 Ubisoft+ 也支持 Stadia 和 Luna；通过游戏如「Star Trek Bridge Crew」和「Werewolves Within」进入虚拟现实游戏领域；以及通过「 Starlink」进行实验性的玩具互动游戏。然而，这些冒险都没有像育碧对区块链游戏的兴趣那样引起玩家的愤怒。自 2018 年以来，该公司一直对区块链技术表现出兴趣，当时它帮助组建了区块链游戏联盟。

最初，育碧通过其「战略创新实验室」探索区块链技术。该实验室开发了 Hashcraft，这是一款基于区块链技术的类似「我的世界」的游戏。虽然十分有趣，但 Hashcraft 从未超越原型阶段。随后，该实验室公开最新进展：基于互操作性合作，推出一款昙花一现的梦幻体育游戏「One Shot League」，使用的是 Sorare 的足球卡。然而，这些举措并没有引起公众的注意，直到育碧推出了其命运多舛的 Ubisoft Quartz 计划。

Quartz 是育碧进军 NFT 领域的入口，其产品名为「Digits」。该公司试图通过选择 Tezos 区块链 ( 出于考虑所谓的环境原因 ) 来解决对该技术的批评，使 NFT 仅用于装饰以减轻对付费获胜的担忧，并将其与热门游戏「Ghost Recon」联系起来。

该公司的营销目标是收藏家和粉丝，他们对拥有可以在游戏外交易的限量版、系列装饰品感到兴奋。然而，当时公众对 NFT 的看法大多是负面的，与高价、庞氏骗局以及不受育碧普通用户欢迎有关。除此之外，当时最新的「Ghost Recon」游戏「Breakpoint」也面临销量不佳和玩家强烈反对的困境。这一糟糕的时机使得 NFT 更加雪上加霜。

在 Quartz 之后，育碧与区块链游戏的关系恶化。有报道称，员工对这项技术的推动感到不满，最终导致一款名为「q 项目」的未命名 web3 游戏被取消，育碧甚至一度公开表示，它将把在区块链领域的探索恢复到「研究模式」，以此来摆脱过于激进的立场。尽管遭遇了这些挫折，育碧还是投资了多个 web3 项目，其中包括 Cross the Ages、Nine Chronicles、Dogami、Rebel Bots 和 Skyweaver。

该公司还利用其 Rabbids IP 与 Reddit 和 The Sandbox 展开了 NFT 合作。尽管这些项目并没有获得多少回报，且最近取消了一些项目并进行了战略重新评估，但育碧还是在上个月发布了自己的 web3 游戏:Champions Tactics: Grimoria Chronicles。

关于「Champions Tactics」的消息不多，目前仅知这是一款「PC 上的 PvP 战术 RPG 实验游戏」。根据游戏网站披露消息，玩家「组建一支神话般的冠军团队，与其他玩家进行激动人心的战术战斗，并发现黑暗和神秘的格里莫里亚世界的传说」。

宣布这一消息的时机并非巧合——这是日本区块链周期间以日本和韩国为中心的 Oasys 区块链特别活动的一部分，该游戏计划在 Oasys 上发布。育碧是 Oasys 区块链的 21 个官方验证者之一，加入了亚洲的一些大公司，包括 Square Enix, Com 2 us, NEOWIZ, Netmarble, Wemade 和 NHN Corp。Oasys 本身是由日本知名游戏开发商万代南梦宫，世嘉，Thirdverse 和 Double Jump Tokyo 创建的。该活动还被用作一个新网站的发布，旨在展示它所提供的所有游戏以及多个发行商的公告。

世嘉还在「战场」

对于开发区块链游戏，世嘉一直摇摆不定。2022 年，该公司宣布与 Double Jump Tokyo 合作，在 Oasys 上开发一款基于其街机 IP「Sangokushi Taisen」的游戏。「Sangokushi Taisen」街机系列因其在实时战略场景中通过使用交易卡牌来创建和操控军队单位的独特方法，而可能适用于 Web3 模型。

鉴于「Clash Royale」等基于卡牌的策略游戏的成功，这一举动似乎是合乎逻辑的。2022 年，日本也发行了「Arsenal Base,」，这是一款成功的基于实体交易卡牌的高达游戏，其玩法类似于「Clash Royale」。虽然该项目最初于 2022 年宣布，但世嘉抓住日本区块链周的机会发布了最新消息。即，发布了一个预告视频，以及关于游戏的更多信息，这将是一个基于卡牌的三分钟比赛自动战斗，名为「Battle of Three Kingdoms」，计划于 2023 年底发布。

Web3 尽管世嘉发布了这款游戏的最新信息和计划，但最近彭博社与世嘉首席运营官 Shuji Utsumi 对话后得到的印象是，该公司正在与 web3 游戏保持距离。Utsumi 曾批评游戏很无聊，他说道:「如果游戏毫无乐趣可言，那还有什么意义?，并对区块链技术在该行业的前景表示怀疑。然而，他也强调了该领域的潜力，他说:「对于大多数游戏行业人士来说，区块链倡导者所说的可能听起来有点极端，但这就是最初的样子。我们绝不能低估。」他补充说，虽然世嘉对其最大的 IP 被第三方用于区块链游戏持谨慎态度，但它将继续与外部合作伙伴一起探索这一领域。

与这些言论似乎矛盾的是，世嘉与 Line Next 合作，将其「主要特许经营权」之一引入 Line Next 的区块链游戏平台 Game Dosi。Line Next 是 Line 公司的韩国分公司，是日本流行的同名即时通讯服务的幕后推手。Game Dosi 于 5 月中旬发布，发布了几款第三方游戏和一款名为 Project GD 的第一方交易卡牌游戏。与 Oasys 不同，Game Dosi 使用自己的基于 cosmos 的区块链 Finschia，该区块链旨在方便使用，具有简单的钱包系统，包括 Line 和 Google 等社交登录。

虽然世嘉没有透露游戏的特许经营权或任何有关游戏的细节，但很明显，尽管有些保留意见，该公司仍然对区块链技术提供的机会感兴趣。世嘉成功的 IP 之一「Virtua Fighter」于今年 3 月在 Oasys 上发行了限量版 NFT，进一步证实了该公司的兴趣。

Square Enix 继续前进

Square Enix 是另一家对区块链游戏抱有强烈信念的大型发行商。该公司总裁松田洋介 (Yosuke Matsuda) 在 2022 年和 2023 年的年度新年贺信中强调了该公司对该技术潜力的信心，并致力于探索该技术。今年 3 月，松田被桐生隆（Kiryu Takashi）取代，后者的声明表明，他也对区块链技术感兴趣。

「在娱乐业息息相关的商业环境快速变化的情况下，拟议的变革旨在重塑管理团队，目标是采用不断发展的技术创新，最大限度地发挥公司集团的创造力，以便为全球客户提供更多娱乐服务，」桐生隆写道。

这并不奇怪，因为桐生隆曾担任电通 (Dentsu）的总经理，这是一个对区块链技术有着浓厚兴趣的前瞻性研发集团。该组织甚至在 5 月份发布了一个 web3 准备工具。关键问题是 Square Enix 将如何谨慎地应对当前区块链游戏的熊市环境，特别是关于其旗舰 IP「最终幻想」。Square Enix 通过发布「Final Fantasy VII 」NFT 对这一领域进行了初步探索，该游戏最初于 2022 年被提及，直到今年 5 月才完全公布。

到目前为止，该公司最重要的「冒险活动」便是发行了「Symbiogenesis」，这是一款介于 NFT 艺术项目和成熟游戏之间的游戏。该项目专注于艺术和故事情节，游戏玩法类似于一款寻物解谜游戏。物体是完全隐藏的，只有线索引导点击。尽管该项目采用了不同寻常的方式，但它似乎吸引了一群忠实的粉丝，每天在其拥有 37, 000 名成员的 Discord 社区活动，同时还在拥有 42, 000 个关注者的 Twitter 账号发布游戏情节和线索。

韩国是 Web3 游戏的支柱

虽然日本对区块链游戏的信心显而易见，甚至连日本首相岸田文雄 (Fumio Kishida) 都在推广 NFT，但韩国多年来一直在开拓和完善区块链游戏，并已成为亚洲区块链游戏的支柱。这一进展值得注意，因为区块链游戏自 2019 年以来在韩国一直是非法的，导致开发商转而为国外市场创建 web3 游戏项目。

Wemade 一直走在该领域的最前沿，将代币和 NFT 融入其在 Steam 和移动平台上成功的「Mir 4 」和「Mir M」系列游戏中。该公司已建立自己的区块链 WeMix。迄今为止，它举办了 62 场不同质量的比赛。也就是说，Wemade 的做法并非没有争议。代币流通数量的差异导致其代币从交易所下架。

Netmarble 是另一家接受区块链技术的知名韩国开发商。其发布的区块链游戏包括经典 IP，如「格斗之王」，以及较新的游戏，如「A 3: Still Alive」、「Golden Bros」、「Ni no Kuni: Cross Worlds」、「Meta Football」和「Meta World」。该公司还计划在 2023 年第四季度推出其他游戏，包括「GrandCross: MetaWorld」。

Netmarble 曾参与为 Marvel 等大型 IP 持有者开发游戏，并将 CCP games 的「EVE Online」移植到手机平台。然而，在不久的将来，我们似乎不太可能看到该公司为大型 IP 开发区块链游戏。

Com 2 uS 在建立并重新命名自己的区块链 XPLA( 前身为 C 2 X) 之后，今年继续推进其 web3 的雄心。它已经在 XPLA 上发布了多款游戏，如旗舰游戏「Summoners War: Chronicles」，以及「Ace Fishing Crew」，「Summoners War: Lost Centuria」等。Com 2 uS 还在 Oasys 大会上宣布将把「Summoners War: Chronicles」带到该平台。该公司甚至在今年的 GDC 上谈到了它在区块链游戏方面的经验。

另一家韩国大型游戏发行商 Nexon 在 GDC 上发布了一款基于 Polygon 的「MapleStory」系列 web3 新游戏，震惊了所有人。虽然这对该公司来说还只是一个实验，但该演示展示了该技术如何增强「MapleStory」及其生态系统的主要目标。建议大家看看 Naavik 最近对 Nexon 首席执行官 Owen Mahoney 的采访。

在所有探索区块链潜力的现有 IP 中，MapleStory 是最重要的，拥有 20 年的丰富历史。在此期间，Nexon 一直在努力应对一系列二级市场问题和游戏设计挑战，这些挑战对区块链新手开发者来说很常见。这些经验提供了宝贵的教训，可以为未来其他企业提供帮助。

在其介绍中，Nexon 明确表达了对该品牌的风险的认识，反映出许多这些发行商在涉足具有知名品牌的 Web3 金融和社交领域时所经历的热情和忧虑。

如果「MapleStory」在区块链上取得成功，它可能会成为许多对进入这一领域持观望态度的开发者的催化剂。但实际上，该项目可能还需要一段时间才能公开发布。

低调参与者

尽管在主要发行商中，最积极支持区块链游戏的声音来自日本和韩国，但其他几家发行商选择保持低调，静静地从事自己的项目或在场外观察。例如，Zynga 在被 Take-Two Interactive 收购后就暗示了自己在这一领域的野心。从那以后，该公司一直保持沉默，除了在区块链游戏领域招聘高级游戏设计师和首席制作人。EA 也同样对自己的活动严加保密，直到最近宣布将耐克 NFT 整合到其体育游戏中，这似乎是谨慎的第一步。

动视暴雪 (Activision Blizzard) 似乎对此不感兴趣，但其母公司微软 (Microsoft) 公开表示对该领域的技术方面感兴趣，并投资了包括区块链游戏在内的多个项目。值得注意的是，微软在将区块链技术整合到现有资产方面也表现得非常谨慎，Mojang 在「我的世界」中禁止 NFT 就是最好的证明。

未来

就像围绕炒作周期的技术和商业发展一样，游戏开发者向区块链技术的下一个重大转变可能只会由少数成功的游戏推动。2021 年的狂热是由「Axie Infinity」的成功所推动，尽管这一冒险最终失败了，但下一款引发类似兴奋的游戏可能会引发新的热情并吸引更多的开发者。

例如，育碧已经在多个领域努力纠正错误，包括取消和推迟项目。之前提到的大多数技术赌注都没有实现预期效果，这使得育碧很难通过「Champions Tactics」改变游戏格局，获得胜利。长期以来，如果一个区块链项目难以获得收益，无疑会导致公司重新考虑项目可行性。

世嘉 (Sega) 和史克威尔艾尼克斯 (Square Enix) 似乎都不愿在这项技术上冒太多风险，这表明更传统的发行商对区块链游戏的风险偏好较低。世嘉明智地选择了以集换卡牌为中心的游戏，因为这是一种能够在该领域持续存在的游戏类型，就像我们所看到的「Sorare」和「Gods Unchained」一样。另一方面，Square Enix 正在向 Symbiogenesis 的优势倾斜，尽管 Square 在这方面的优势并不是由区块链直接实现的。Square 的新总裁在 2024 年的年度信中不太可能过多谈论这项技术，除非该发行商从目前的项目中看到了显著的进展。

从日本区块链周来看，尽管市场处于熊市，但该地区仍有相当多的人热衷于探索区块链能为游戏做些什么。韩国也一直坚持围绕一种甚至无法在当地使用的技术开发游戏。这两个国家都展现出一定的韧性和决心，其尝试使用 Web3 和区块链游戏技术，看看它是否具有可行的商业潜力。然而，市场现实也存在一定的考量，在底层方面出现问题可能会迅速使紧张的公司高层重新采取观望态度。

最有可能的结果是，我们在这里详述的一些发行商将会把注意力转移到其他领域，而没有在早期取得实质性的成功。我们可能不会很快看到支持区块链的「最终幻想」或「刺猬索尼克」游戏。同样重要的是要考虑到开发一款 AAA 级游戏需要多年时间，让高质量的区块链游戏更像是一个副项目，因为这些发行商仍然需要将大部分资源投入到主要的项目中。

相反，世嘉依靠第三方或史克威尔艾尼克斯 (Square Enix) 开发规模较小的实验性新 IP 等策略可以让游戏在不冒险的情况下取得进展。我们也不希望看到西方发行商在未来一两年内大举进军区块链游戏，除非他们取得一些成功。
