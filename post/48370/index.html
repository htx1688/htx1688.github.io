数据驱动的DeFi研究员，收集了30个优秀加密协议的最新进展，颇具研究价值。

 Lido

Lido V2 最值得期待的特性：

  • Staking Router / 质押路由器：允许任何人通过模块化的即插即用模式成为节点操作员（目标是进一步去中心化）。

  • 提款：上海升级之后，stETH持有者可以直接从信标链上提款。

Snapshot 投票里有细节：

 Velodrome

@VelodromeFi 是 Optimism 上排名第一的原生DEX项目。

Velodrome 2.0 将在第一季度推出：

  • VELO FED：veVELO持有人在$VELO货币政策中获得发言权；

  • 针对每个交易对，可以单独自定义和调整交易费；

  • 集中流动资金池的新方法；

  • 整体UI/UX改进。

 Radiant

@RDNTCapital 是一个基于 LayerZero 建立的借贷协议，支持跨多链存款与提款。

Radiant V2将于几周后推出：

  • 跨链扩展（首先扩展到BNB链）；

  • 用于借贷的新资产，包括stETH、BNB，更可能包括GLP、gDAI和其它通过治理投票的资产；

  • 通过动态流动性减少RDNT排放；

  • UI/UX 重新设计。

Fantom

Fantom 未来几个月的进展包括：

  • fUSD v2：为建设者和用户提供的链上收费系统（可选择使用fUSD充当交易费用）。

  • FVM 新进展：Solidity兼容，执行速度更快。测试显示~4500tps，数据存储减少98%。

新的主网计划于2023年底上线。

另外，AC回归，你懂的。

Rocket Pool

Rocket Pool是一个去中心化的以太坊质押协议 ?

Atlas 升级：

  • 为（上海升级）提款做准备；

  • 8 ETH 迷你池! (剩余的24个ETH由协议提供，用于操作验证器）——更低的进入门槛+更加去中心化；

  • Gas优化（减少40%）。

PlutusDAO

PlutusDAO是一个Arbitrum原生的治理聚合器，旨在最大化用户的流动性和奖励，同时聚合PLS代币背后的治理。

GMX

GMX将在今年的某个时间点推出V2版本：可能是第一季度或者第二季度，目前正在进行审计。

V2版本将引入大量新的可交易资产，而不仅仅是有限的几种加密货币。

在Discord中也有关于V2版本将进行跨链扩展的预告。

Optimism 

Optimism 正在打造 OP Stack：一个模块化的区块链框架，用于轻松构建各种可扩展的Rollup链（OP链）。

Coinbase的Base链是第一个利用这项技术的。

Trader Joe 

Trader Joe 原本是 Avalanche 网络上的一站式去中心化交易平台，现已扩展到 Arbitrum 和 BSC。

Liquidity book V2:

  • V2 pool fee switch for $sJOE stakers;

  • Limit orders and permissionless liquidity book pools.

Rage Trade

最具流动性、可组合性和全能型的全链以太坊永续合约协议。

Synapse Protocol

Synapse 是使用最广泛的跨链消息系统和资产跨链桥。

Synapse 正在建立一个 Ethereum 原生的 Rollup（Synapse链）。

  • 跨链信息传递：在一个链上调用一个合约，在另一个链上执行。

  • 为跨链 dApps 打造的Synapse桥。

Frax Protocol

作为LSD赛道的一员，Frax 因为拥有最具竞争力的质押收益率，frxETH 的增长非常迅猛。

最新进展：

  • FIP-188: $FRAX将成为完全抵押品（从92%到100%的支持）。

  • $FRAX和Frax生态系统的普遍增长（$FPI、veFPIS、FraxSwap、Fraxlend frxETH等）。

Pendle

Pendle通过将资产分为基础代币和收益代币，提供非常高的收益率。

TVL在整个2023年持续上涨。

他们最近用GLP跨链到Arbitrum，并将在未来增加 gDAI + 额外的资产。

Synthetix 

Synthetix V3:

  • 以ETH和其他资产作为抵押品；

  • LPs可以选择将流动性分配给哪些合成资产（高风险=高费用）；

  • 更多采用建立在Synthetix之上的产品（Kwenta, Lyra, Thales等）。

Conic Finance

Conic 在 Curve 上提供 omnipools。

将用户的流动资金分散到不同的资金池中，并将其押在Convex上，用户赚取CRV + CVX + CNC。

该协议在1周内出现了6500万的TVL。

dopex(?,?)

dopex 是一个去中心化期权交易所。

最新进展：

  • rDPX v2: 能够以rDPX作为部分抵押品铸造合成代币（rDPX的有望通缩）。第一个资产将是dpxETH（注意并不是LSD）。

  • Dopex OpFi 产品：在后台利用链上期权的DeFi产品。

Polygon Labs

Polygon 正在疯狂打造新的产品矩阵：

  • Polygon Supernets

  • Polygon Avail

  • Polygon Zero, Miden & zkEVM

  • Polygon Nightfall

zkEVM将于3月27日在主网上推出。

dYdX

dYdX正在推出他们自己的应用链（dYdX v4 即 dYdX链）。

基于Cosmos SDK构建，以去中心化的方式创造近似CEX的性能。

dYdX v4值得期待的特性：

  • 交易的时候不需要Gas费；

  • 由验证者运行的链外订单簿（Off-chain orderbook）；

  • $DYDX作为原生Gas代币。

Curve Finance

随着稳定币应用的增加，Curve作为最大的链上稳定币流动性中心，将大大受益。

crvUSD 稳定币接近推出，这将为Curve/veCRV创造额外的收入。

Lyra

Lyra 最近推出了名为“Newport”的升级版本（Arbitrum部署，并整合 GMX永续合约），支持$ETH和$wBTC期权。

长期来看，DeFi期权有很大的潜力，随着采用范围的扩大，Lyra可能成为从中受益的主要协议之一。

Cosmos

几天前，链间安全（ICS）升级的第一部分通过治理获得批准，并将于3月15日启动。

ICS允许应用链借用Cosmos hub验证器集来保证安全，而不需要启动自己的验证器（需要付费）。

MakerDAO

Maker有几件事情正在开展：

  • Spark协议：第一个产品是Spark Lend，它受到Aave V3的启发，有一个直接的Maker信贷额度（4月）；

  • EtherDAI：原生的 ETH LSD 背书的 $DAI。

Gains Network ?

gTrade已经成为领先的去中心化交易商之一。

在Arbitrum上推出后，交易量大增，并增加了新的资产，包括商品和指数。

没有官方的路线图，但在Discord上有很多预告。

AAVE

Aave正在推出他们的原生稳定币$GHO。

  • 通过在Aave上超额抵押资产进行铸造；

  • 总供应量由 “促进者”（由DAO选择的实体）控制；

  • AAVE质押人将获得$GHO铸造折扣。

Treasure DAO

Treasure 是通过$MAGIC连接游戏和玩家的去中心化游戏生态系统。

  • 与Arbitrum合作的游戏建设者计划（GBP）：激励游戏开发者在Treasure DAO生态系统中建设。

  • MagicSwap v2: 游戏内经济的未来，支持ERC-20代币和NFTs的池子。计划在今年第二季度推出。

Uniswap

Uniswap最近宣布了他们即将推出的移动应用程序。

Uniswap仍然是领先的DEX，截至3月，每天大约有5万用户。

移动应用程序可以在更广泛的零售业加入方面发挥重要作用。

Filecoin

Filecoin将于3月14日推出Filecoin虚拟机（FVM）和Filecoin EVM（FEVM）。

FVM将允许在Filecoin的基础上建立自定义智能合约，开辟各种新的用例，如代币化数据、数据DAO等。

Kwenta

Kwenta 自2月推出V2版以来，数量/用户大量增加。

一些即将在第一季度/第二季度推出的功能：

  • 与 Lyra 合作；

  • 交叉保证金v2

    ；

  • 费用开关；

  • UI/UX改造。
