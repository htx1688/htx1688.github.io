吴说作者：Maverick

蓝筹 NFT 项目 Bored Ape Yacht Club（BAYC）近期宣布，将于 1 月 18 日通过 BAYC Sewer 启动铸造型游戏 Dookey Dash ，玩法类似于《神庙逃亡》。在官方表示游戏即将发行之后，得益于该游戏规则对一些 Yuga Labs 生态的需求，BAYC、MAYC、BAKC 和 APE Token 短线都有不同程度的上涨，其中 BAKC NFT 地板价更是创近 9 个月币本位新高。

Sewer Pass（下水道通行证）是 Dookey Dash 的门票，共 3 万张。每位 BAYC 或 MAYC 持有者将可以在 1 月 18 日起至 2 月 8 日免费领取一张 Sewer Pass，同时持有 BAKC 则可以领取特殊级别的通行证，没有持有 BAYC 或 MAYC 二者 NFT 的玩家也可以从 Sewer Pass 持有人处购买获得。 无论获取途径如何，任何拥有 Sewer Pass 的人都可以进行游戏。官方根据持有 NFT 系列的不同将 Sewer Pass 分成四个等级，TIER1 需要持有 MAYC，TIER2 需要同时持有 MAYC 和 BAKC，TIER3 需要持有 BAYC，TIER4 则需要同时持有 BAYC 和 BAKC。Sewer Pass 等级越高，将来获得的更好奖励可能性也会相应增加。

值得注意的是，Dookey Dash 对 APE Token 的赋能。玩家在游戏开始前可以花费 2 个 APE Token 来购买能量增强包。增强包能让玩家获得持续 10 分钟的辅助效果，并可以无限购买。

玩家在三周内可无限次游玩，但尽量在积分冻结日（2 月 8 日）前获得更高的分数，官方会对玩家最高分数进行记录。游戏积分将基于玩家在“下水道“中收集的碎片类型、摧毁的障碍物以及生存的时间等多个因素考量，玩家通过跑酷的方式获得游戏积分，全网最高分数的玩家获得“钥匙”。在 2 月 15 日前持有 Sewer Pass 且积分大于 0 的玩家，将会获得转换成新物品的资格，但当前官方未说明该新物品具体情况。

作为“猿宇宙”首个游戏，Dookey Dash 在推出前后依然维持着 BAYC 社区神秘感、故事化以及挑战性的特质。

神秘感

Yuga Labs 擅长保持神秘感，借助讲故事、留彩蛋的方法，设置悬念给社区留下想象空间，激发社区好奇心进行讨论。例如在官方推特上表示 APE Token 将在新游戏中会有用途；在宣布推出游戏 Dookey Dash 之前，从去年 12 月开始 Yuga Labs 官方就已经上线短片《The Trial of Jimmy the Monkey》进行暗示预热；在近期则推出了这款新游戏 Dookey Dash，但游戏在正式运行前许多包括奖励、奖品用途等细节又是未知状；Yuga Labs 不直接公布 2023 年路线图，后续又释放出可以获得官方路线图“宝箱”剧情供社区猜测与探寻。从 NFT 到元宇宙，BAYC 并不是一蹴而就打造出宏大的元宇宙游戏，期间还通过无数的概念艺术图、动画设计去吊足胃口，进一步稳定和活跃社区，以建构完善的元宇宙图景。

故事化

Dookey Dash 在正式上线前发布短片讲述猴子 Jimmy 的故事，交代了游戏和角色背景，而在发布该短片之前又在社交媒体平台上发布一段较为模糊的文字和图片造势。单独看起来有点无厘头，但一旦把这些串联起来，又可以从一系列宣传中体会到故事连贯性。Cryptopunks 中国社区联合发起人 @tokenbrother 认为 BAYC 这种讲故事所用的是前置放大法，就是把“讲故事”本身又设定为一个故事，对其进行再次讲故事，不停地重复进行叙事。在对用户公布的时候，则采用从前到后的手法，通过逐步释放，保留悬念。除了对故事不断地强调可以强化参与者对背景故事的记忆，还有猴子 Jimmy 不小心误吞了神秘金属箱的钥匙，游戏玩家的目标拔出卡在猴子屁股里的钥匙，这样有趣的故事情节亦增强玩家在游戏体验中的沉浸感。

挑战性

与一般 NFT 盲盒、碰运气式铸造不一样，Dookey Dash 采用的是游戏铸造方式，玩家可以用自己的“硬实力＋氪金”的方式来提高自己的分数。通过实时排行榜，玩家会有一个竞争的目标，在游戏后期玩家容易 FOMO 以购买道具冲榜提高游戏分数。但相比起单一靠 BAYC 和 MAYC 决定获得生态下一个 NFT 铸造资格概率，Dookey Dash 又给非 Holder 或低等级的玩家机会通过提高自身对游戏的熟练度和理解后来居上，让 NFT 铸造的技术含量提高。

尽管当前许多 NFT 项目试图复制 BAYC 的成功道路，但却迟迟没有走出下一个 BAYC。不可否认的是，目前能支撑起这种宏大叙事的循序渐进建设的只有像 BAYC 等极少数蓝筹 NFT 项目。当其他项目做出类似的行为则往往会被社区视为项目方在画饼或者 PUA，本质是因为他们往往只是对 BAYC 某一个步骤的简单模仿与复刻，而未立足于更长远的战略蓝图，或者说项目方的实力并不被社区认可可以构建出庞大的元宇宙世界。让社区保持对 BAYC 足够的耐心除了因为早期收藏者获得巨大的回报获得大量的用户忠诚，Yuga Labs 的背书、运营、做市等各方面中心化硬实力条件优秀以及其良好的品牌效应也是重要原因。
