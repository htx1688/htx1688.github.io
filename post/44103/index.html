吴说作者： defioasis

美东时间 10 月 28 日，历尽几番波折，世界首富埃隆·马斯克最终以 440 亿美元完成对社交媒体巨头 Twitter 的收购。收购完成后，马斯克对 Twitter 的原 CEO Parag Agrawal、CFO Ned Segal 等高管进行开除，而马斯克 Twitter 简介也改成 Chief Twit，并发文表示自己会成为公司的 CEO。在 Meta（原Facebook）、Snapchat 等社交媒体今年表现不佳的背景下，对 Twitter 的收购展示了马斯克对社交媒体行业进行大变革的野心。Twitter 作为 Crypto 圈最重要的传播媒介，Binance 在本次收购案中也占据了一席之地，加之马斯克本人一向对 BTC、DOGE 等加密货币关注有加，Twitter 未来的改革方向想必对 Crypto 社媒营销、Web3 社交以及 SocialFi 有方向性参考。本文将根据先前马斯克的一些言论摸索出 Twitter 大致改革方向。

建设公共讨论广场

首先，平台应该是鼓励对话的，容纳不同声音的。与现阶段中心化社交媒体类似，Web3 社交用户创建账号（钱包地址）即可参与，每个人都可以发声，此时的平台应该成为哈贝马斯所说的“公共领域”。作为公共的舆论场所，不同用户可以提出批判性意见。而平台的鼓励言论自由是进行这些行为的重要条件，而非可以随意解释社区规则乱封禁账号。其次，平台应保持中立，不成为政治或资本的工具。正如马斯克在推文《Dear Twitter Advertisers》所说，目前社交媒体分裂为极右和极左两派回音室的危险很大，它们会产生更多仇恨，分裂社会。以往社交媒体根据用户的观点和兴趣进行推送延长用户在平台留存时间，但造成窄化用户视野的同时也助长了群体极化，就像 2017 年 Facebook 和剑桥分析公司利用算法和数据库操纵了美国选举；以及早前的 Monaco Planet 为了杜绝薅羊毛的行为屏蔽汉语言的使用，引起了华语与外语社区的对立。Web3 社交项目应该保持平台中立性。保持中立性的前提是下放治理权给社区，由社区监督平台或管理者（项目方）是否凭借管理者/创始人的身份作恶。社区也有权选择平台管理者，也让平台管理者强化自己管理责任。

成为过滤虚假消息、传播真相的平台

社交媒体具备的低门槛、匿名性特点，加上信息发布者基数大、法不责众的心理助推虚假信息野蛮生长。社交平台根据点击量、阅读量等量化数据进行推广扩大传播，也助推了虚假消息进一步传播。一直以来，不论 Web2 还是 Web3，虚假消息都是社交媒体平台的问题，其中问题主要是审核机制的不完善，比如说 AI 对信息的判定过于死板，“聪明的”用户总能通过诸如谐音、拼音等方式绕过；人工审核精力有限等。另外在源头治理上，去中心化项目具有抗审查性，如在 Mirror 上创作的消息发表不需要先验证真伪。

目前比较理想的方法仍可能是“事后解决”，由社媒平台的所有者——DAO 社区制定篇章，并在社区中形成专门小组进行审核，用户发现后举报，根据举报情况，小组再做判定后删除言论，以此降低该地址的信誉分数。在没有被举报的虚假信息还是占主流情境下，平台可以利用“算法＋人工”模式进行内容真实性检查。对于谣言的后续跟进，DAO 组织可以选一批在领域内高信誉个人用户或机构用户发表内容辟谣并置顶在该话题，防止谣言传播的进一步扩大。

拓展服务范围，不止于图文社交

马斯克在一次采访直言若收购 Twitter 会 Copy Wechat 模式，即不止满意做图文社交媒体，还会增加影视、游戏服务范围。目前许多 Web3 社交项目只是单纯复制社交媒体模式，然后加上 Read To Earn、Write To Earn 或 Comment To Earn 等激励制度，导致链上项目只是 Token 化的链下项目。虽然 X to earn 模式可以帮助这些社交项目方短期内刷高 DAU，但长期对用户来说并没有吸引力，特别是 Token 下跌后更容易引起社区 FUD。用户使用社交 DApp 主要目的不是为了金融关系，而是维系社交关系，所以头部社交媒体的用户粘性具有强者越强的“二八现象”。

功能创新，是吸引新用户长期留存的关键因素。虽然加入影视服务等可能碍于项目运营成本受限难以行动，但基于 Web3 社交项目的 NFT 发售、对艺术家 PFP 流转的抽成，可能会弥补部分的成本。另外，NFT 作为社区门控实则也在打造特定用户的私域流量，那么诸如直播、短视频或打造类似于微信朋友圈的“私域流量”的功能或许也能以更去中心化的形式呈现出来。Web3 社交项目应思考回归社交价值，即这个社交 dapp 是否足够创新好玩吸引用户留存、以老带新，解决了哪些 Web2 社交媒体的痛点，而不是想怎样把复制别人已经成功的模式然后加上一定的经济效益让用户留下来。

真实用户为主

与 Twitter 一样，Web3 社交当前机器人或虚假账号众多，这和目前钱包地址创建成本低，以及“科学家”脚本，羊毛党、空投党等有关，产生大量无意义内容，让内容区、评论区注水，如 Monaco Planet 上曾出现了一大批追求 Write to earn 经济效益的虚假帐号产生大量的无效的冗余信息，扰乱整个页面。清理虚假账号，一方面可以提高准入门槛，例如用户连接 Gitcoin passport 身份认证做多身份验证，才可以解锁多个使用功能；项目方提高真实用户的空投或“X to earn”奖励，又或减少非身份认证的奖赏权重。另一方面项目方防女巫攻击审查预期，减少羊毛党的热情的同时减少虚假账户对项目的污染。

重视名人效应引流带货

据 CoinMarketCap 数据，在 Twitter 被收购完成后，与马斯克高度相关的 MEME Dogecoin 连续两个交易日大幅拉升，一度涨至 0.1494 美元。虽然此前 Twitter 支持用户向创作者打赏 BTC 和ETH，并有未来 DOGE 加入 Twitter 打赏计划的坊间传闻，但 DOGE 的巨幅拉升实际应和马斯克自带的名人效应有关。以优质 NFT 项目 BAYC 为例，有着库里、比伯、麦当娜和内马尔一众明星带货代言，既能为平台 OpenSea 带来流量的关注，也让一众优质 NFT 项目抢先入驻 OpenSea，以获得市场关注度，这也让平台获得了良性循环。Web3 社交项目以 Twitter、INS 这些社交媒体榜样，应该重视明星对平台的引流作用，寻找与项目定位相匹配的品牌代言人，让名人常驻之余，还可以尝试平台孵化草根 KOL，如印尼青年 Ghozali Everyday 在 OpenSea 用自拍制成的 NFT 大卖上爆火出圈，吸引了其他民众尝试在 OpenSea 上的创作，草根 KOL 的成功让普通人相信在 Web3 世界里有所成就，也吸引其他路人参与 SocialFi/NFT 项目。
