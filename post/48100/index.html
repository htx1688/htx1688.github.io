最近，Polygon 的 NFT 生态系统正在蓬勃发展。随着 Terra 和 FTX 的倒台，许多项目已经迁移到 Polygon 上。此外，Polygon 已经逐渐成为 Web2 公司 NFT 的首选目的地。

在本文中，我们将探索 Polygon 的 NFT 生态系统，分为六个类别：

  • Web2 公司

  • PFP

  • 游戏

  • 效用

  • Meme

  • 市场

Web2 公司

星巴克正在利用 Polygon 网络上的 NFT 提供 Web3 体验。用户可以通过完成各种任务（例如测验、小游戏，甚至是现实生活中购买和饮用咖啡）来获得 NFT。

邮戳 NFT 可以在星巴克自己的市场上交易。为了改善用户体验，该市场提供了入口服务，并设计了后端处理了 Gas 费用。用户可以在不需要了解区块链的情况下使用它。

耐克对 Web3 表现出了很大的兴趣，包括收购 RTFKT。最近，耐克推出了一个名为 DotSwoosh 的 Web3 平台。这是一个耐克数字物品的空间。创作者有望通 DotSwoosh 获得版税。

和星巴克一样，.swoosh 也旨在让没有区块链知识的用户轻松使用。您可以看到已经有大量用户铸造了.swoosh 地址。

全球最大的社区之一 Reddit，在 Polygon 网络上推出了头像 NFT 收藏。它允许被列入白名单的创作者设计头像并以 NFT 的形式销售它们。

到目前为止，Opensea 聚集了 300 万用户，但 Reddit 在短短六个月内就已经聚集了超过 600 万持有者。当然，这些产生的钱包在 Polygon 网络上并不经常活跃。但 Reddit 已经有效地吸引了许多用户进入区块链生态系统。

PFP

CyberKongz VX 是早期流行的以太坊 NFT 之一 CyberKongz 系列之一。这是一个将现有 PFP 以 3D 形式重现的项目，在 SandboxGame 元宇宙中可用。

HellCatsNFT 已经从 Terra 迁移，并在 OnePlanet NFT 市场上拥有最高的交易量。他们专注于实用性，通过提供质押服务，让用户可以使用点数购买各种产品或参加大型抽奖。

y 00 tNFT 是 dustlabs 的 PFP 项目，dustlabs 以构建 Degods 而闻名。在强大社区领袖 Kevin 的支持下，y 00 ts 有望与已经接入 Polygon 的大型公司进行合作。

游戏

TheSandboxGame 是一个元宇宙游戏，允许创作者通过创建基于体素的地图和物品来生成收入。该项目最初基于以太坊，为了提高用户体验，已迁移到高度可扩展的 Polygon 网络。

DerbyStars 是一款赛马+元宇宙游戏，它已经从 Terra 迁移而来。他们与 OnePlanet 市场合作，为随机盒子提供免费铸造。他们最近发布了抢先体验版本！

以及 Phantom，Phantom 是 Animoca 的子公司开发的 Web3 游戏，与现有的 P2E 相比，它拥有极高的图形和游戏性。

效用

LensProtocol 是一个基于 Polygon 的 Web3 去中心化社交协议。用户的个人资料以 NFT 的形式存在，支持在市场上进行交易。

Meme

CollectTrump 是一个以特朗普为主题的卡牌 NFT，最初旨在成为一个带有奖励的实用 NFT。然而，现在它已经成为 Polygon 生态系统中的一个受人追捧的 Meme NFT，即使在奖励活动结束后，其价格仍在飙升。

市场

OpenSea 是以太坊上的代表性市场，支持多条链。目前，它在 Polygon 上的交易量最高。交易量高的流行项目包括：Trump cards、The Sandbox 和 MocaVerse。

而 OnePlanet 是 Polygon 的原生市场。该市场从 Terra 网络迁移而来，已经成功上线了 48 个 NFT 项目。

OnePlanet 通过精心筛选来精选项目。此外，他们还推出了自己的启动平台。最近，他们支持了 DerbyStars 的随机盲盒和 Hellcat 的铸造。

结语

Polygon 的另一个优势是交易费用低，这使其成为 NFT 创作者和收藏者的一个有吸引力的选择。随着人们对区块链技术和 NFT 的兴趣和采用日益增长，Polygon NFT 生态系统很可能在未来继续发展壮大。在接下来的几个月和几年里，看看该平台如何发展以及 Polygon 上会出现哪些新项目和合作将会非常有趣。
