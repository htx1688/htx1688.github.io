未来几个月，有哪些机会呢？

撰文：五火球教主

首先，这不是一篇教你撸毛的文。

只是最近 Aptos 分发的 Token 再次暴富了一批人，于是乎，几乎被人遗忘的羊毛党又再度活跃了起来！

但对于绝大多数人来说，结局大概是如下这个样子：

暴富 Token 项目出来——晒图——吸引各路人——毛圈互卷——撸了 3 个月——一个 Token 没有——被反撸——放弃——2 个月后再出现暴富空 Token 分发项目——循环。

圈内总是不断地重复着这种轮回！

毕竟大毛本就不多，真正暴富了一批人的也就最早的 Uni，dydx，后来的 Juno，ENS，最近的 OP，Aptos，真正的羊毛党除了持之以恒的毅力和撸毛的技术手段，也多多少少需要些运气。

那么未来几个月，有哪些可能，只是说可能的大毛机会呢？一起来捋一遍, 顺便盲猜下各个项目可能的规则框架。

01 ETH 生态

毫无疑问，ETH 几个 Layer2 生态项目，是最为「确定性」的大毛。

毕竟 OP 发 Token 了，要说其他几大「Layer2 天王」不发 Token 的话，我是压根不会信的。

唯一不确定的是，他们啥时候发 Token，Token 分发多少，以及最重要的：Token 分发规则是什么？

Arbitrum 

奥德赛活动基本算是明牌送你 Token，只要你有耐心做完。之前因为 Gas 和性能的问题暂停，现在 Nitro 主网上线，年前大概率重启。

只不过既然是明牌，注定只做奥德赛的话你是拿不到大毛的，毕竟有几十万人在和你一起卷着呢。笔者个人猜测就是奥德赛会拿到一点点 Token，或者说作为 Token 分发的一个参考权重而已。

Zk-sync

ZK-Rollup 的两大扛把子之一，最近 2.0 主网上线了，撸毛党扎堆的拥挤之地，然而你要是现在才去，估计会有点晚，估计 V1 交互之类的肯定也会占不少权重，V2 现在的交互人数已经多到可怕。

Starkware 

官方跨链桥上的 ETH 也是肉眼可见的在增多，应该很多也是去撸毛交互的，Starkware 和其他几个 L2 还不太一样，毕竟人家一开始就只在折腾 L3，所以估计 Immutable X，dydx 等早期用户拿到 Token 的权重会不小。

Scroll 与 Fuel 

相对前三个，这俩的名气没有那么大。Scroll 是华人团队的 ZK-Rollup，但在业界的名气和关注度也是越来越高，相对于 ZK 的两大扛把子，Scroll 主打的是更加完善的 EVM 兼容性。关于 Scroll 撸毛的教程几乎没有，能找到的信息基本也只有几个月前的一个测试网预登记，但往往越是这种人少的地方，越有大毛的机会。

Fuel 则是除了 Arb 与 OP 另一个很有实力的 OP Rollup，且定位是一个模块化的执行层，并非专属 ETH 生态，又是非常另类的 UTXO 主打并行计算型，非常值得一看。可惜也是同样没有太多可以交互或是撸的信息，只有几个月前一个面向开发者的公共测试网，且明确表示单纯面向开发者，不是激励测试网，参与该网络的交互将不会获得直接奖励。当然了，dydx 最早也是这么说的，结果有人就是不信邪，坚持不懈的交互，最终还就真的等到了项目方「变卦」，撸到大毛！

02 Cosmos 生态

Cosmos 生态一向以分发 Token 多而著称，可惜分发的 Token 大多也都是毛毛雨，唯一一个能拿出来称道的也就是 Juno 了。

不过明显有感觉到最近 Cosmos 生态的崛起之势，以及 Appchain 和模块化区块链路线方向之正确性，所以未来几个月，Cosmos 这边的大毛机会，应该不少，毕竟这里没有 ETH 那边卷的厉害。

GNO

这项目，感觉注定是大毛，但是现在去撸应该是没戏了，因为 Token 半年前就发给了 ATOM 的质押者 ( 没法交易），当然，后续会不会有别的规则，现在也说不准。

为什么说注定是大毛呢，因为这是 ATOM 创始人 Jae Kwon 自己搞得项目，也是公链，而且是用的新语言 Gnolang, 天生为了多线程而生，同时有着更快的编译速度、更简洁的语法和资源高效的并发性。

你想到了啥？是不是很有 Move 既视感？

如果 Aptos 和 Move 系是资本推动的新公链，GNO 到时候肯定就是社区推动的典范了，项目大概率明年初会看到。

Celestia 

Celestia 这半年多名气已经大的不能再大，提到模块化第一个想到的必然是他, 最近熊市几千万美金的融资也毫无疑问的彰显了未来独角兽项目的地位。

Celestia 会分发 Token 么？感觉大概率会，毕竟分发 Token 是 Cosmos 系的一惯传统，而除了 Atom 质押者，Osmosis 质押或 LP 之外，Celestia 的测试网估计也会是 Token 分发的权重之一，技术流的小伙伴不应该错过。

Sei 

之前的文章有写过《一览 Cosmos 主要生态项目与发展现状》，是一个基于 Cosmos，专门针对 DeFi 优化的 DeFi Chain，刚好今天发布了两则消息：

一是详细介绍了自己的 7 大创新，包括智能的区块广播；优化的区块处理；DeliverTx 并行；端块并行；原生价格预言机；频繁批量拍卖（FBA）；交易订单捆绑……

二是表示 1% 的 Sei Token 供应将分配至激励测试网。引起不少人的不满，1%？？少点了吧兄弟……

但其实要知道，Aptos Token 分发的数量也就 2%，所以 1%，其实也没你想象的那么少。

03 Move 系

Sui 

这个不用说，Move 系双雄，Aptos 匆忙发了 Token 之后，现在整个加密圈最卷的项目一定是 Sui，没有之一。

Aptos 当时百里挑一的节点一度让人以为没选中就凉凉，没想到单纯只是完成申请就给 300 个，NFT 的持有者只要链接上 Discord 也给了 150，于是乎大家都在纷纷猜测 Sui 也会是类似的套路，搞得 Sui 连忙澄清，目前没有 Token 分发计划！但群众显然不信，当年 dydx 也这么说，最后不也给了不少！

总之我个人也是不信 Sui 不给 Token 的，然而指望它会像 Aptos 这么「草草」的分发 Token，连个女巫检查都没有是肯定不可能了，撸毛这种事情，永远都是大家都不看好没多少人撸的时候才是大毛，一窝蜂的挤在一个地方，注定是个狼多肉少的局面。

Linera 

当初 Diem（Meta）出来的人马其实是兵分三路的，Aptos，Sui，还有 Linera，只不过 Aptos 和 Sui 因为用了 Move 语言，所以名气大很多，融资的体量也大很多。但 Linera 其实背后也有 A16Z 这种巨头，所以说不定因为这里的人少，分到的毛要比 Sui 大。

只不过这项目比 Aptos 和 Sui 都要晚，目前也没有什么测试网之类的可以交互或者申请，保持关注即可。

写在最后：总而言之，撸毛这件事，除了不能三分钟热度，要持之以恒之外，还有下面几个基本原则可以遵循。当然了，遵循了也不一定能撸到，最多能提高下胜率。至于撸不撸，这完全是个见仁见智的事儿~

  • 账号贵精不贵多，OP 的 Token 分发规则是你最应该研究明白的；

  • 账号越早开始参与越好；

  • 保持链上的活跃度，多参与几个 Dapp，尽量保证每周都有个几次的交互；

  • 参与社区。包括 DAO 的投票治理啊、多签钱包、GitCoin 捐助之类的；

  • 多用几次桥。新的公链一开始肯定会有一两个桥支持，没事儿过去把资金跨一跨，很有可能也是交互权重的一种。
