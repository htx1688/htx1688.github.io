当人人都是“司马迁”，是否能改写“历史由胜利者决定”的局面？

作者：HistoryDAO

9 月 15 日，以太坊顺利完成了合并，其 PoW 的最后一个区块被 Vanity Block 铸造成了 NFT，即整个区块只包含一个用于铸造 Vanity Block NFT 的交易，成本为 30 ETH。在最后的区块中，Vanity Blocks 引用了美国民族植物学家特伦斯·麦肯纳的名言写下寄语：「只要勇于做出承诺，世界自会助你铲除不可逾越之阻碍。去完成未竟之梦想，宇宙绝不会抑制你前进的步伐。这即是奥义所在。」

Vanity Block 通过区块，记录了以太坊生命历程中最重要的一次升级。以太坊此次合并事件也成为了加密世界的重要回忆。

回忆，是对于人类最重要的事情之一。

在远古，人类通过壁画、绳结来记录回忆。在组成小型部落、氏族后，过去的故事通过语言口口相传。后来，文字成为了回忆和历史的载体。历史和故事就被史官们记载了下来。

而到了信息时代，互联网成为了历史记录的载体。如今，用不可篡改的区块记录回忆成了加密世界中最流行的事情。

这是信息时代和区块链技术所带来的便利，但它们又为历史记录加了一层新的诅咒。

历史记录的诅咒是什么？

中心化的历史记录： 从古至今，记录历史的权力被掌握在中心化实体手中，且不可挑战。而野史和民间记录，也大多来自古代贵族。即使到了信息时代，这种局面仍未被打破，强大的中心化实体拥有记录历史的权力。而在加密世界，用区块记录的权力也被掌握在矿工手中。

信息太过庞杂： 信息时代所带来海量数据也导致真实历史难以被记录，尤其是对于没有任何声量的普通人，他们难以记录关于自己的历史。或许在未来，就像我们对自己先祖的过往一无所知一样，最后能为后人保留下来的，也只有一张照片。

去中心化历史记录解决方案 HistoryDAO 希望通过 Web3&区块链技术解决以上两个诅咒。

用区块承载历史

希罗多德在流放期间，以自己的视角写下了《Ἱστορίαι（历史）》一书，把他在旅行中的所闻所见，以及波斯阿契美尼德帝国的历史记录了下来。这是西方史上第一部完整流传下来的历史记录。罗马演说家西塞罗评价他为“历史之父”。

不过，客观的记录往往会受到记录者主观的影响，最后让历史记录失去公允性。

这在人类史上常有发生。就像那句名言所提到的：历史由胜利者书写，但事实真相只有亲历者才晓得。 在史料搜集和对因果分析方面保有严谨态度的古希腊历史学家修斯底德也指出：不要偶然听到一个故事就写下来，甚至也不单凭我自己的一般印象作为根据——我所描述的事件，不是我亲自看见的，就是我从那些亲自看见这些事情的人那里听到后，经过我仔细考核过了的。

如何让事实真相的亲历者低门槛地记录无法被篡改的历史？这个问题一直在困扰着追求真实历史的人们。

幸运的是，如今区块链技术让它成为了可能。

就像比特币将铸币权去中心化一样，HistoryDAO 去中心化了历史记录的权力。

得益于区块链技术，HistoryDAO 允许用户在以太坊和 BNB Chain 上发行、查询和交易 NFT：

  • HistoryDAO 支持各类数据格式的历史记录形式，允许用户进行多维度的信息记录；

  • HistoryDAO 拥有一个允许用户有效地发现、评估、交易和管理的 NFT 二级市场；

  • HistoryDAO 完全由社区拥有并管理，DAO 的发展完全由社区议程推动；

简而言之，HistoryDAO 是一个由 DAO，也就是由去中心化社区组织驱动的NFT平台。

HistoryDAO 在为个体提供记录历史的服务的同时，也有由 DAO 组织记录已经达成共识的历史——社区负责记录每周、每月和每年的历史人物、历史进程、历史专辑和重大历史事件。

HistoryDAO 如何架构产品和商业化？

接下来，我们将从三个方面深入探索 HistoryDAO 的功能和服务。

铸造历史 NFT 和二级交易市场

就像上文所提及的，HistoryDAO 支持用户进行多元的历史信息记录，这些内容都将以 NFT 的形式保留在用户的区块链钱包中，公开透明，别人无法强制篡改和转移该 NFT。

更有想象力一点，用户所记录的数据可以是任何信息，包括生活的点点滴滴、与好友爱人的美好回忆、游戏中的精彩瞬间等等。或许，未来的人类也会像我们通过研究清明上河图来复现古人生活一样，通过研究 HistoryDAO 的历史 NFT 来复现我们当代人的生活图景。

其中，在 NFT 铸造时，用户需要支付 0.01 ETH 的服务费用和 Gas 费用。对应的服务费用将直接转入由 DAO 控制的市场流动性储备，也就是 DAO 国库。

同时，历史和回忆是由全人类共同拥有的。因此，用户所铸造的历史 NFT 也可以在二级市场中流通和交易。HistoryDAO 市场会收取 2%的佣金，这部分费用也会被转移到 DAO 国库。

而 HistoryDAO 市场也会对各类历史 NFT 进行排序，排序将参考对应 NFT 的上市时间、浏览量、受喜爱数量等多个维度。为了确保每个历史 NFT 的曝光率尽可能的公平，每一个新的算法变化都需要在 DAO 中提案并获得社区支持。

INO

为了让历史记录变得更有效率，HistoryDAO 引入了“Initial NFT Offering”，即 INO 的概念。任何用户都可以为 NFT 集合提供一个主题，且这个通过与其他社区成员来共建该 NFT 集合。

为了方便理解，我们可以拿一些体育赛事来举例。

以 NBA 为例，在一个赛季中，我们可以为每一场比赛创立 NFT，并随着赛季的进行，通过图片、视频剪辑和球员数据等信息记录每一场比赛的详情。理所当然的是，一场精彩比赛的 NFT 要比普通比赛的 NFT 更具收藏价值，决赛 NFT 要比正常轮次的比赛更具收藏价值。或者，我们也可以发挥想象力，在赛季结束后，选取 1000 场比赛的精彩时刻铸造 NFT 发行。对于品牌方而言，INO 活动是对之前事件影响力的再利用，极具性价比。

当然，这只是一个例子。由 HistoryDAO 推出的 INO 活动拥有极强的可扩展性，也不会仅限于体育赛事。

只要品牌营销团队拥有创新式的营销思路，便可以通过 HistoryDAO 记录任何活动和事件，便进行营销——毫无疑问，这要比纯粹的撒钱营销活动要更具吸引力。在INO活动之后，品牌方也可以围绕此次事件铸造的NFT进行后续赋能。

DAO 和 Token

与其他 NFT 平台不同，HistoryDAO 更注重社区互动和社区力量。本质上，HistoryDAO 是记录历史的工具平台，而非 NFT 交易市场。因此，HistoryDAO 在产品设计上，更注重社区活跃度的算法设计，希望构建出像 Reddit 那样强大的社区产品。

增强社区力量目的在于规避监管对于社区历史记录和言论自由的压力，去中心化社区可以抵抗强大的利益集团对于历史记录的审查，并且会提升人们对于去中心化历史记录的积极性，也会提升 HistoryDAO 历史 NFT 的价值。

而 HistoryDAO 的治理将由其原生 Token $HAO 完成。持有$HAO 是 DAO 的准入门槛。进入 DAO 之后，社区成员可以进行提交提案、评论和投票等治理行为。

除了治理外，$HAO 还将被应用于 HistoryDAO 产品的方方面面，比如在市场进行 NFT 交易时，若用$HAO 进行支付，则市场只会收取 1.5%的佣金。

从见证到书写：HistoryDAO 如何赋能 Web3？

毫无疑问，历史对于人类整体和每个人类个体有着特殊的意义。就像唐太宗李世民所言，“以铜为鉴，可以正衣冠，以人为鉴，可以知得失，以史为鉴，可以知兴替”。

历史是我们向前迈步的基础。

但相关的利益集团不管是出于掩盖相关丑闻或者达成某一种目的，往往会修改历史记录。其次，历史记录往往会被主观情绪引导，失去原本的公允性。

而如今，HistoryDAO 正在通过区块链和 NFT 技术扭转历史记录的缺陷。就像比特币将铸币权去中心化一样，HistoryDAO 将历史记录的权力分散了。

去中心化且无法篡改的历史记录将从多角度记录历史，追求历史记录的公平和真实。

这正是 Web3 所倡导之精神的体现——Web3倡导个人主权，倡导个体价值，历史不再由胜利者书写，全人类平等参与。

作为与现实话题紧密连接的 NFT 在铸造后，也会增强 Web3 在外界的话题度，吸引更多人主动了解 Web3 世界。而通过真实历史记录来赋能 NFT 采用，也会推动 Web3 世界的进一步发展，变得更加成熟。

除此之外，由 HistoryDAO 主导的 INO 活动也能为各类品牌提供强有力的营销支持。

Web2 品牌通过 Web3 手段营销已经成为了趋势。

Tiffany 推出 CryptoPunks 定制 NFT 和星巴克推出 Starbucks Odyssey 计划便是很好的例子。

其实，随着 Web3 世界的影响力与日俱增，更多的 Web2 品牌将思考如何通过 Web3 进行营销活动和社区增长。对于人生地不熟的 Web2 品牌而言，一个简单直观的 Web3 营销方案是刚需。

而由 HistoryDAO 推出的 INO 活动恰好能够满足这一需求。可以认为，这类需求将会成为 HistoryDAO 未来增长的主要动力之一。

以前是石头，后来是文字，现在是区块

最后，让我们回到 HistoryDAO 的初心——让所有人都可以低门槛地记录历史。抛开我们基于产品做出的思维拓展，HistoryDAO 的本质是允许将历史记录的权力去中心化。

这无疑是开创性的，也是对 Web3 精神的一次解构——Web3 不止是去中心化金融，还包含了更多层面的思考和应用。凭借着区块链技术所赋予的去中心化历史记录的权力，这个世界的所有人，都将有机会像凯撒一样，豪气干云地说出：我来，我见，我记录！
