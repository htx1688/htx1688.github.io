从 Degen 到 Regen：如何推动 Web3 开启正和游戏 @SCaesar

纯粹的逐利行为在 Web3 中是不适应的，加密货币一直有作为用于建立协作组织和共享资源的工具的一面，笔者引用再生经济学，认为 2023 年将成为再生之年（year of the regen）。

Degens 指的是在高收益率和高风险项目上投机的 Web3 爱好者。 

Regens 指那些在再生性加密经济中工作或建设的人，他们对 Web3 如何为世界带来好处有长远的看法，不仅仅局限于金融领域。

在再生生态中，原本为了赚取利润参与到零和游戏的 Degens 会着迷于集体行动的承诺，他们的态度和动机发生了变化，加入社区，逐渐变成了 Regens，建立下一个周期的项目。 

事实上，已经有例如 Proof of Humanity、Celo、Kolectivo 和 Gitcoin 等项目在使用加密技术重建世界。当今许多最主流的加密货币项目都有积极的正外部性。而在这个下行周期，我们需要重新发现目的，Web3 需要将资本、注意力和人才转移到产生最持久积极影响的项目上。

思考

在更早的以太坊时期，加密行业的技术性质、协作性质和社会实验性质催生出了 DAO，在如今的熊市中，我们看到许多参与者们沉下心认真 BUIDL。而在赚取快钱的经济动机驱使下，交易者太容易陷入零和游戏的怪圈，于是我们看见了繁荣期的大量入场和萧条期的急剧抛售。无论 2023 年是不是再生之年，对于加密行业来说，要把共赢共荣放在制定策略的出发点上，而各方也需要表现出真诚合作的精神和勇气，共同遵守游戏的规则，才能推动 Web3 真正开启正和游戏，给世界带来改变。行业要想长期发展，共赢是必然且必要的，但这需要从自我与社区的重建开始，辐射向行业的整体，例如 DeFi、DeSic 等等。当一个又一个周期来临时，参与者们能否坚持长期主义，是 Web3 由再生到长青的决定性因素。

从 Loot 仿盘到去中心化的任天堂：社区文化+ MLP 策略 @SCaesar

Treasure 最早的 NFT 在当时和其他 Loot 仿盘没有任何区别。在发行了 NFT 后，Treasure 就发行了 MAGIC Token 和 DeFi 项目 MAGIC Farm，意图以此曲线赋能 NFT 的方法，虽然失败了，但它开辟了后来成为其核心的 MAGIC 生态。 

由于 MAGIC 最早的预期是作为 Legions 的游戏币，但该游戏短期内开发不出来，Treasure 便推出了 Life NFT 以赋能 MAGIC，但最终以失败告终。此后，尽管 Treasure 生态什么也没有，但社区共识却令 MAGIC 持续释放，Treasure 团队于是推出了 Treasure DAO，正式围绕给 MAGIC 赋能展开构建。 

2022 年开始，Treasure DAO 作为链游平台，通过自研和外部合作的方式逐渐形成一个以 MAGIC 为核心的生态。笔者认为 Treasure DAO 的成功在于其天然具有社区文化和明确的 MLP 策略。

思考

Loot 项目本身吸引了许多长期主义者，他们抱持着让社区拥有更多参与者和构建者的心态在进行着自下而上的构建。Treasure 通过空气币激励从 Loot 社区引流，积累起具有长期属性的早期贡献者，奠定了其难以复刻的社区文化和社区运营风格，这也是许多链游没有重视的部分。另外，和其他链游推大作的风格不同，Treasure 的 MLP 策略让其不需要漫长的时间和资本的长期输血，而是仅靠少量的资金不断吸引外部的新用户和资金进入，同时把已经进入生态中的玩家进行保持和沉淀，以此实现了经济价值的累积循环，推出越来越优质的链游和 NFT，实现飞轮。 

除此之外，Treasure 是以 Game 为核心，Fi 为手段的，其对可玩性的重视，在发展过程中起到了重要的作用。但是，无论是单个链游，还是整个 MAGIC 生态的 Token 经济模型，仍需要审慎地设计，才能把整个生态带到更高的高度。事实上，Treasure DAO 离真正的成功，还有一段距离，社区文化 + MLP 的策略是不是链游的正确演进，也尚未可知。

Play-to-Own: 从游戏看 Web3 应用的所有权分配 @memeswap

笔者认为 Web3 与游戏的组合将会开启 Play-to-Own 游戏的新时代。在我们看来，游戏中的经济活动的发展将大幅增加游戏行业的产品可触达市场（「TAM」），并为价值创造提供动力。与 Web2 的游戏相比，整合了 NFT 的 Web3 游戏在玩家留存率、用户增长和收入效益方面取得了更大的成功（实际上并没有，这只是一个美好的想象，P2E 模式吸引来的投机客最终只会通过挖提卖推动项目螺旋死亡）。

对于 P2E 的项目方而言，需要保持 token 供需之间的平衡，也需要限制游戏内的投资行为，延长项目寿命周期。游戏开发者们正想办法将追求收益的玩家转变为消费者（这本身也与他们用 P2E 模式吸引的用户有关，很多真正主机游戏社群的玩家是排斥 P2E 模式的），归根到底，内容和体验才是王道。

笔者认为 play-to-own 是游戏的未来，通过 P2O 模式实现激励的一致性和真正的所有权，对游戏做过贡献的玩家以 token 或者游戏资产的形式进行激励。在既往的游戏中，开发商往往不注重 UGC 创作者的个人利益，而 web3 或许可以较好的保障这一点，这点在 sandbox 中已经得到了体现。

最终，笔者得出结论，一款成功的 web3 游戏应该是什么样子：

1. 游戏世界的搭建和 IP 增长，让更多用户参与游戏的建设，生成更多的 UGC，更多的剧情和玩法。

2. 沉浸式社交和 MMO 体验，更多的强调定制化、身份建设和社会互动性，这也是 web2 游戏中行之有效的例子。

3. 经济模型的深入探索，根据玩家增长速度调整 token 的通胀水平，持续激励 UGC 的产出。

4. 充分的社区参与。也给出了游戏评估框架

同时，笔者也给出了游戏评估框架

1. 团队背景应该是游戏开发者和 crypto 人士的合作，除了游戏开发之外，需要了解如何推动 token 和 NFT 的价值积累、管理金库和激励计划、利用智能合约提供的机制（如质押）来加速游戏的增长，以及开展社区建设。

2. 需要根据不同的群体打造分销渠道，如体验玩家、硬核玩家、UGC 型玩家和交易员等

3. 游戏经济的可持续性，管理通胀并抑制经济波动。

思考

P2E 模式在经历了几轮行情之后，最终无一沦为盘子。而游戏资产的所谓所有权，也会随着项目方的倒闭或者下跌而失去价值。无论是单 Token 、双 Token 甚至三 Token 的模式，最终也只能稍微延长项目的死亡螺旋。但是本文提出了很独到的观点——UGC 创造价值，这在既往的 dota 玩家自制地图和 steam 玩家工坊都见证了这一逻辑的有效性，UGC 能够极大的增加游戏的可玩性和生命周期，而在传统游戏行业中，对于 UGC 的激励并不到位，更多是用爱发电。 

而 Web3 的本质就是在于生产资料的重新分配，是可以同 UGC 创作很好的结合的。项目方应该通过合理的激励和共创机制激发社群的创作活力，共同创造更有价值、更长生命周期的游戏。此外，目前在 steam 平台有很多独立游戏的发售均被诟病 demo 发售，圈钱后开发进度慢，开发质量低等，游戏行业的这种状况，或许也可以通过 Web3 社区金库的方式进行优化，社区金库通过开发进度向项目方支付款项，并且实现社区共创游戏内容。

The Beacon 如何建立以游戏为中心的 GameFi @SCaesar

上个周期的 GameFi 很难实现正向的外部性收入，而陷入旁氏经济的困境，而 The Beacon 游戏把好玩放在了首位，支持多种游戏模式，有一定的游戏挑战性和社交互动性，真正的以 Game 为目的。The Beacon 的免费玩模式提供了很低的进入门槛，让任何人都可以任玩，而付费角色的玩家们则有一些差异化的设计。The Beacon 的宠物机制和作为 Treasure DAO 生态游戏的可互操作性也值得期待。The Beacon 目前使用 MAGIC 和 BCN 作为游戏内 Token ，尚未推出的经济机制设计是否合理将决定其可以抵达的高度。The Beacon 完成了可玩原型的启动，主要是单人 PvE 模式，后续将会有宠物生成阶段、ALPHA 发布、多人合作 PvE 等等。

思考

The Beacon 以 Game 作为核心，设计出有游戏挑战性和社交互动性的游戏，这将会吸引到很多非经济动机的玩家。同时，The Beacon 通过降低进入门槛，让任何人都可以游玩，而让愿意付费的玩家为 NFT 和差异化付费，在保证早期游戏玩家数量和活跃度的情况下，让游戏更具有可玩性和经济机制设计，可以发展成更成熟的 GameFi 生态。The Beacon 算是提供了一种比较新颖的思路，但能否吸引到足够数量的用户群体、能否设计合理的经济机制设计，以及玩家是否愿意为了可玩性买单，都是有待观察的，让我们保持期待。

Cipholio BitMart 联合报告 | 2023 我们身在何处？从五大核心赛道演变逻辑看加密投资趋势 @JZ

1. 公链：笔者认为公链在解决扩容的核心问题上整体在沿着满足「广度用户」需求的「通用性公链」及满足「垂直用户」需求的「专用型公链」两个大方向发展。往广度走有高性能公链提供的超级计算平台满足 Web2 用户大量进入的需求，往深度走有模块公链服务于 Web3 头部应用满足更多链上原生需求与创新。

2. Defi：看好链上订单簿，衍生品协议与合成资产在未来的发展。

3. Gaming：认为没有显著特征的泛链改游戏在未来很难生存，游戏团队必须在可玩性和金融化之间做出选择。在如今整体产业模式不固定的情况下，不看游戏产业宏观的中间件，如打金工会数据服务平台等等，一体化的游戏工作室更值得青睐。沿着 P2E/X2E 的方向，我们依然看好在Token设计和应用场景上做出创新并拥有过人运营和叙事能力的团队。在 AAA/Metaverse 方向我们会更注重基础设施的发展以迎合更多资产乃至玩法上链的需求，如云游戏，AIGC，游戏钱包，游戏资产垂直 NFT 交易平台等等。

4. NFT：从提案的总数和被通过的数量来看，可以观察到的趋势是目前社区更加关注 NFT 金融方向发展的道路，与 Web2 结合更紧密的实用性 NFT 在落地层面依然任重道远。供给侧缺乏新鲜叙事，需求侧收紧投资这不单单是 NFT 存在的问题，但受制于 NFT 资产种类和不可分割的特性会将流动性的问题无限放大，综合导致了目前 NFT 市场看起来流动性极度缺乏的局面。在 NFT 资产种类得到丰富之前，任何宣称可以解决 NFT 流动性的问题的项目都是拿着锤子在找钉子。

5. 社交：社交领域是围绕数据可组合行构建的「基础设施-中间件-应用」经典三板斧框架。

Web3 数据赛道：用户数据的景观、层次和未来 @ 猫老大

Web3 数据领域将对新秩序至关重要，其具有巨大的发展潜力。本文将讨论 Web3 数据结构、现有 Web3 数据空间中的典型玩家以及未来的发展趋势和一些投资理念。核心观点如下：

1. Web3 打破数据孤岛，同时将数据权利还给个人用户。用户拥有自己的数据，允许他们在互联网上携带和使用这些数据。 

2. Web3 数据领域的结构可以分为四个层次：1）数据来源。2）数据获取。3）数据查询与索引。4）数据分析与应用。每个级别的分散性、可扩展性、速度和准确性是无法比拟的。我们倾向于通过这些主要指标来判断一个项目的潜力。 

3. 随着数据市场参与者的逐渐增多和数据本身的积累，数据的价值将大幅增加。保护用户隐私，同时利用数据产生更大价值将继续成为重要的优先事项。 

4. Web3 数据领域未来最重要的用例之一是建立去中心化的信誉系统。基于这套信誉体系，将有可能解锁信用借贷等多种金融场景。虽然已经有独角兽项目筹集了数百亿美元，但 Web3 数据领域仍处于起步阶段，还有很多数据场景需要服务。如果你选择其中之一，你将很可能能够在 Web3 的世界中发展和成长。

思考

Web3 数据赛道是比较适合创业新手尝试的方向，但这个方向有很多细分领域。选择正确的创业方向是一方面，而另一个重要的点是时机，而现在几乎可以断定我们正处于链上数据及应用爆发的前夜，在这个时间阶段进入这个赛道是最佳时机。 

回顾 Web2 数据领域的发展，从搜索引擎的崛起到如今的电商、短视频，几乎都指向一个核心「推荐算法」。这恰恰是当下 Web3 缺少的。前几天推荐阅读的文章《NFT 可发现性困境：如何触及目标用户？》也提到这关于 OpenSea 缺少对用户个性化推荐的功能。可见，随着 Web3 数据市场和应用程序的发展，市场需要越来越懂用户的「推荐算法」。
