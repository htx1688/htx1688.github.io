2023年无疑是比特币区块链扩展生态的一个重要节点。从 Ordinals、RGB协议等概念的引入，再到Bitcoin Stamps和BRC20标准的出现，支持比特币生态的发展的呼声愈发强烈。

在过去，很少有NFT玩家会想到比特币网络，大多数人使用以太坊或专门构建的 NFT 区块链，直到2023年初Casey Rodarmor 推出了Ordinals和Inscriptions （序数和铭文），为这个最原教旨主义的比特币网络引入了 NFT，这些类似 NFT 的“铭文”逐渐获得了稀有性，成为收藏品，并开始在二级市场上市交易。2023 年1 月 21 日，Ordinals 协议发布了 0.4.0 版本，BTC NFT 铸造量开始飙升。

Ordinals给比特币带来的很多创新，但 Ordinals 的故事还远未结束。6月12日，比特币协议 Ordinals 新任首席开发者 Raph 在 Github 上将比特币协议 Ordinals 创建者 Casey Rodarmor 提出的“递归铭文#2167”合并到 Ordinals 协议，最终在 0.6.2 版本正式更新上线了递归，随着Ordinals的匿名开发者之一 Leonidas.og对其进行了介绍，递归铭文随之而生。

递归铭文是比特币Ordinals 社区的创新之一，它的出现有望彻底改变比特币 NFT 生态系统，并重新定义目前基于区块链的文件存储参数。

比特币序数铭文

要了解什么是递归铭文，首先我们需要先明白是什么比特币序数和序数铭文。

序数的概念由 Casey Rodarmor于2023年提出， 它是使用“序号”来特指一种编号方案，该方案为比特币网络上的每个聪（sat）赋予一个编号，将这些数字分配给 sat 的过程称为序数理论。这些聪按照开采顺序进行编号，意味着即使 sat 最终转移到不同的钱包，归属于它的序号也保持不变。这些数据可以是任何格式，并直接记录在比特币区块链上。

铭文是指通过在 sat 上铭刻内容来创建比特币原生数字资产。铭文存储在比特币交易的见证字段中。这要归功于2017 年实施的隔离见证 (SegWit)软分叉，该软分叉通过将比特币交易分为交易和见证组件来重组比特币交易，这种创新优化了数据存储效率。在 SegWit 升级的基础上，另一个重要的比特币软分叉Taproot于 2021 年 11 月推出，进一步将铭文的数据存储容量提高到当前最大 4MB 限制。这意味着铭文可以在比特币区块链上存储最多4MB的数据，而不占用太多空间。

有了序数，比特币就不仅仅是数字黄金，它将数据直接集成到区块链上，为创建独特的数字资产打开了通道，将艺术、收藏品和知识产权领域带入去中心化金融领域。通过利用序数，艺术家和创作者能够在不可变的比特币区块链上验证和标记他们的作品。比特币 NFT 的稀缺性吸引了创作者和收藏家，为NFT市场注入了新的潜力。因此，序数很快在比特币社区流行起来。

随时间变化的铭文数量

尽管比特币进行了升级，但将数据直接写入区块链仍会占用大量空间，而且仍限制在4MB内，递归铭文的出现就解决了这个问题。

递归铭文

什么是递归铭文

递归铭文（Recursive Inscriptions）是比特币区块链的一项新功能，允许创建可以存储在区块链上的复杂数据结构。通过创建一系列铭文来工作，每个铭文都引用之前创建的铭文数据。这允许创建任意大的数据结构，因为每个铭文可以指向另一个铭文，简而言之，递归铭文会调用先前铭文中的数据，并将该数据合并到新铭文中，这使得用户可以超越每个铭文的 4MB 限制。递归铭文通过扩展比特币 NFT 网络，将序数铭文提升到一个新的水平。

递归铭文最终允许开发人员通过一系列调用链接数据来完全在链上运行软件。许多人还认为，递归铭文代表了比特币网络互操作性的又一步，以及更复杂的序数用例，不仅限于铸造比特币 NFT。例如，用户可以创建一个包含视频游戏的递归铭文。铭文可以调用其他铭文的数据来存储游戏的图形、音乐和代码。这将允许在比特币区块链上存储大型且复杂的游戏，而不必担心 4MB 块大小的限制。

以往我们打铭文时，如果我们要打1000张图片，需要将这些图片一张张的上传到比特币网络中而后生成，这个过程一是工作量非常大，二是受限于比特币网络的区块容量，每张图片的大小必须限制在很小的范围内，以控制铭刻时的成本，这就会牺牲图片的质量。递归铭文采用的方式则是先铭刻这些图片中提取出所有特征，这里每一个铭文代表图片中的每一个特征，然后再建立1000个递归铭文，每一个递归铭文利用“/-/content/:inscription_id”代码去请求“特征铭文”上的图片，最终通过编程将完整的图片进行呈现。这个思路就像是活字印刷，在活字印刷以前，印1000页的书就要刻1000个石板，但有了活字印刷后，只需要造出有限的字库，直接从字库中拿出固定的字块进行组合拼接，就可以在很短的时间内组成需要印刷的书籍。

递归铭文的创新突破

提高存储效率：递归铭文使用特殊的“/-/content/:inscription_id”语法来请求其他铭文的内容，这直接使得用户可以用更少的容量和更低的手续费在比特币链上创建铭文。Leonidas 等递归铭文爱好者一直在宣扬其潜力，递归铭文不仅可以增加序数的用例，还可以通过减少刻在每个聪上的数据量来削减交易成本。这种通过递归方式回收存储数据的能力可以很好地消除存储重复文件副本的必要性，从而显着提高存储效率。

增强互操作性：递归铭文所拥有的灵活性、可组合型和低成本优势，给铭文带来了无限的新可能，用户可以将复杂的图片视频、3D游戏等形式的内容刻在链上，同时相对于之前一个个独立的铭文，增强了其交互性，今后铭文NFT从只能用户单向查看发展成了双向互动。

丰富了铭文表现形式：除了最早的文本铭文类型，递归铭文还新增了 “JS”和“CSS”类型，这意味着递归铭文几乎可以显示任何网页中的内容。并且递归铭文可以引用其他铭文，就像本地网页升级为互联网一样，极大地丰富了铭文的表现形式。

代表项目

Recursive Playground

Recursive Playground是第一个可交互递归铭文 NFT 项目，该项目允许用户与NFT互动，只需轻轻拖动鼠标，便可改变画面的状态。让铭文 NFT 从静态、只能眼观变为动态的、可实时互动的。目前Recursive Playground并未透露背后的团队。但作为“可交互递归铭文”的首秀，为NFT带来了更多的可能性。

OnChainMonkey

OnChainMonkey 这样的项目已经意识到了递归铭文的潜力。它们使用递归将各种数据包作为序号刻在比特币上，使它们能够通过调用这些数据包来创作出小于 1KB 的3D 艺术品。OCM Dimensions 是比特币上的3D动画铭文，总供应量有300个。作为一个3D 互动艺术作品，Dimensions 以比特币网络为基础，向上构建以创建详细的 3D 动画渲染，所有这些都在比特币链上进行。每个Dimensions 使用不到 1 KB 的区块空间。 

递归铭文赋能比特币生态

递归铭文的出现为比特币生态系统带来了无限的可能性。递归铭文的出现使得直接在比特币区块链上托管大量文件成为现实，递归铭文的影响不仅仅限于娱乐和文件存储。它还开辟了技术机会，可以通过允许用户调用已有复杂代码或数据的铭文存储库来创建新型软件。例如实施由比特币持久存储强制执行的无需许可的合同，而无需新的加密措施。开发人员可以利用预先存在的铭文存储库，通过利用已存储在另一个铭文上的数据来绕过当前的 4MB 限制。这降低了在比特币上托管文件的成本。

另外递归铭文也被提议作为改进比特币智能合约功能的一种方法。智能合约是一种可用于在区块链上实现交易自动化的软件。然而由于比特币脚本的非图灵完备，所以其智能合约功能受到限制。递归铭文和序号的集成有可能促进比特币区块链上类似智能合约的功能和其他无需许可的合约，从而为一系列复杂的软件用例打开了大门。递归铭文还可以通过允许用户调用已有复杂代码或数据的铭文存储库来创建新型软件。

递归铭文的潜力与风险

潜力

递归铭文让文件获得使用指定语法索取其他铭文的能力，允许创建可以存储在区块链上的复杂数据结构，使得比特币区块链内可以实现更大的数据存储，它打破了之前的铭刻方式，新的铭文之间将可以开启自由组合的无限可能。它不仅解决了铭文容量大小的问题，还丰富了铭文的创作形式，同时还降低成本，递归铭文的引入为区块链带来了革命性影响。

在7月份引入递归铭文后，人们开始重新提起对比特币铭文的兴趣。截至发稿，以文本格式铭文的资产超过了迄今为止创建的铭文。图片位居第二，而应用程序则占比不大。

风险

递归铭文的其中一个风险是可能被用来创建诅咒铭文（Cursed Inscriptions）。诅咒铭文是指因错误使用或故意滥用操作码来创建铭文而创建的铭文，导致它们变得无效和无法识别。诅咒铭文会在循环中引用自身，从而无法兑换，因为没有办法打破循环。诅咒铭文还可能包含恶意代码，用于窃取资金或损坏比特币区块链。然而，目前来看，诅咒铭文也可以使用序号创建，所以这不是一个新问题。此外，社区似乎也在一定程度上接受了诅咒铭文。

递归铭文的另外一个风险是其可能被用来在区块链上创建垃圾邮件或其他恶意内容。长期来看，这会给比特币网络带来负担，无数垃圾铭文堆积在比特币的内存池中，比特币网络可能变得更加拥堵，手续费也会更加昂贵。

小结

总之，递归铭文标志着 Ordinals 叙事中的另一个里程碑，进一步打开了比特币网络的互操作性和可编程性，推动了比特币生态系统的创新。虽然递归铭文仍处于早期阶段，且面临开发和测试等挑战，但它们的出现改变了比特币应用的潜力，特别是通过创建更复杂的应用程序和增加主流用户的可访问性，是不可低估的。递归铭文能够克服存储限制并为 NFT 和智能合约释放新的可能性，也必将重构比特币的未来。
