一、项目要点

1、项目优势

1.1 具备强大支撑的企业背景：依仗 Coinbase 作为后盾，Base 拥有雄厚的用户资源和财务支持。此外，Base 可以轻松构建可访问 Coinbase 产品、用户和工具的去中心化应用程序，无缝的和 Coinbase 旗下产品集成。

1.2 融入超级链生态：作为 OP Stack 的使用者，Base 能够与所有同样采纳了该技术堆栈的项目无缝互联，同时享受其他 OP Stack 项目的生态支持，从而为 Base 链的生态繁荣增色不少。

1.3 优秀的业务表现： Base 还处于相当早期的阶段，于 2023 年 8 月 9 日才正式上线主网，但是其业务数据却是远超在 2023 年 7 月 18 日就已经上线了的 Linea，其 TVL 甚至超越 Starknet 成为 Layer 2 TVL 排行榜中的第五位，由此可以看出 Base 更加受市场的欢迎。

1.4 社区高热度：随着用户得以通过 Token 合约提前进入 Base 链，一次巨大的财富效应发生在了 Base 链，千倍币“$BALD”的出现激起了 Base 链的热潮，吸引了众多社区成员的瞩目。

2、潜在风险

2.1 技术方面缺乏核心竞争优势：Base 的涌现似乎并未带来显著的重要性，它似乎只是众多 Layer 2 项目中的普通一员，尽管其背后有强大的支持，但这并不足以使其脱颖而出。而且，Base 的出现对整个行业和该领域的影响微乎其微。（这是指哪方面的核心竞争优势？是技术吗？）

2.2 监管压力导致其发币前景不明朗：在监管环境仍不明朗的情况下，Base 很难推出其 Token，并且在主网发布路线中明确指出目前暂无发行 Token 的计划。与其他预期进行 Token 空投的 Layer 2 项目相比，Base 在空投猎人中的吸引力较低。

然而，其官方白皮书中亦有明确提及：“后续将逐步去中心化”。而在 web3 的去中心化进程中，常见的实现方式涵盖了发行 Token 并建立去中心化自治组织（DAO）。此外，以往也曾有众多项目宣称不会进行 Token 发行，然而后来也进行了 Token 的发放，并且向用户进行了空投。因此，另一方观点认为，作为一条公链，Base 有很大概率还是会进行治理 Token 的发行。

二、项目基本情况

1、项目简介

Base 是一个由 Coinbase 孵化的基于 OP Stack 构建的以太坊 Layer 2 ，并且 Base 将把部分交易费收入返还给 Optimism Collective 国库。Base 的目标是让下一个 10 亿用户进入加密世界。

如上图所示，公司的创始人布莱恩·阿姆斯特朗（Brian Armstrong）提出了一项总体规划，将加密货币的增长概述为四个阶段。这些包括：

  • 开发新协议，如比特币、以太坊等。（已完成）—— 100 万人

  • 建立交易所以促进贸易（已完成）—— 1000 万人

  • 为 dApp 构建大众市场界面（已完成）—— 1 亿人

  • 构建 dApp 来为开放的金融系统提供动力—— 10 亿人

而这最后一步正是希望通过 Base 来实现，Coinbase 团队将 Base 视为实现增强世界经济自由这一使命的下一步，Base 将作为 Coinbase 链上产品的所在地，以及为任何构建以太坊扩展解决方案和 dapp 的人提供的开放生态系统。Coinbase 团队希望 Base 成为 Coinbase 用户访问更广泛的加密经济的无缝网关，最终使更多人更容易访问 web3。

Base 是 OP 代码库的一个分支。Base 团队选择 Optimism，因为 EVM 提供了“阻力最小的路径”，而 OP 的代码库提供了快速启动路径。

Coinbase 和 Optimism 团队已经合作一年多，为 EIP-4844 的开发做出了巨大贡献（EIP-4844 也称为 Proto-Danksharding，将使 L2 交易费用降低 10-100 倍），Coinbase 现在是 OP Stack 的核心贡献者，作为第二个核心项目团队加入。

2、项目团队

Base 由 Coinbase 内部孵化，Coinbase 是世界上最大的加密货币交易所之一，由前 Airbnb 工程师 Brian Armstrong 于 2012 年 6 月创立。

截至今日，已有 100 多个国家/地区约 245000 个生态系统合作伙伴相信 Coinbase 可以轻松安全地投资、消费、储蓄、赚取和使用加密货币。如上图所示，Coinbase 平台资产高达 $ 128 B，本季度交易量达到 $ 9 2B，且公司拥有超过 3400 名员工。

值得一提的是，Coinbase 在 2018 年 10 月 30 日以 80 亿美元的估值完成了 3 亿美元的 E 轮融资，并在 2021 年 4 月 14 日在纳斯达克上市，股票代码为“COIN”，目前市值 $ 20.65 B。

3、项目融资情况

目前，Base 尚未进行独立的资金筹集活动，但其母公司 Coinbase 曾成功获得 5.056 亿美元的融资。鉴于 Coinbase 作为最大的中心化交易所之一，积累了雄厚的财力和现金流，而 Base  在 Coinbase 的"总体规划"的第四阶段中被赋予了重要的角色，这意味着 Base 大概率能够从 Coinbase 获得资金支持，不会面临资金短缺的问题。

4、项目进展与路线图

Base 于 2023 年 2 月 23 日推出 Base 测试网， 2023 年 3 月 4 日在 ETHDenver 为基础建设开发者提供 15000 美元赏金，此后于 2023 年 5 月 24 日 公布上线主网前需要的准备（准备都已完成）：

  • 在测试网中成功完成 Regolith 硬分叉

  • 与 OP Labs 团队成功进行基础设施审查

  • Optimism 基岩升级成功

  • 成功完成内部和外部审核，没有严重的问题

  • 展示了测试网的稳定性

在完成了以上启动标准后，于 2023 年 7 月 13 日，官方公布 Base 主网现已向构建者开放，主网的第一阶段使开发者能够将他们的产品部署到 Base 上。

后于 8 月 3 日，Base 宣布主网将于 8 月 9 日正式向所用用户开放，且 Base 还为此举办了一场活动“OnChain Summer”，这是一个为期一个月的链上活动，由 50 名最优秀的构建者、品牌、产品、艺术家和创作者举办。八月的每一天，都会为用户带来一些有趣的链上活动，具体有以下几个活动：

  • 在 Base 上构建 – 8 月 12 日至 23 日：在 Base 上部署协议、构建基础设施或新应用程序。

  • 支持加密货币 – 8 月 14 日至 24 日：制作一个一分钟的视频，分享有关链上用例的故事，或倡导合理的立法。

  • Based Accounts – 8 月 16 日至 9 月 13 日：使用帐户抽象构建一些可以在 Base 上解锁新的或改进的用户体验的东西。

除了资助之外，Base 还与其他 Superchain（OP Stack） 成员合作，于 8 月 4 日至 18 日举办虚拟黑客马拉松 Superhack，黑客马拉松的奖金为 125, 000 美元。

同时，Coinbase 作为排名前 5 的头部区块链中心化交易所之一，拥有雄厚的资金和现金流，在为 Base 提供大量用户资源的同时还能够提供雄厚的资金支持，且上文说到，Base 作为 Coinbase 第四步（10 B+）的关键角色，必然会给予足够的支持。

四、Base 自身亮点及 OP Stack 分析

1、 背靠 coinbase

1.1 资金和资源支持

Coinbase 拥有超过 1 亿的实名认证用户基数，并且还有着近千亿美元级别的资产规模，而如上文所提到的，Coinbase 是加密货币中 Top 5 的交易所之一，拥有强大的用户资源，也有着雄厚的资金，因此 Base 背靠 Coinbase 能够获得资金和资源方面的支持。

1.2 无缝集成 Coinbase 旗下产品

Coinbase 以其卓越的能力，系统性地消除了使散户从中心化金融（CeFi）向去中心化金融（DeFi）迁移时可能遇到的障碍。

  • Coinbase 钱包是 DeFi 领域集成度最高的钱包之一，拥有超过 100 万桌面用户和超过 1000 万移动用户

  • Coinbase 与 Circle 紧密合作，共同推动了 USDC 稳定币的市值扩大到超过 $ 25 B。

  • Coinbase 的 Liquid 质押 Token cbETH 是仅次于 Lido 的最高质押实体，有233 万枚 ETH 被质押在 Coinbase 上，有效保护着 Beacon 链的稳定。

此外，Base 可以轻松构建可访问 Coinbase 产品、用户和工具的去中心化应用程序，无缝的和 Coinbase 旗下产品集成，如下图所示，就像 Metamask 中自动集成了 Linea 主网一样，在 Coinbase 钱包中也自动集成了 base 主网，用户在使用 Coinbase 钱包时无需自己添加。

同时，Coinbase 不仅仅是一个交易所，其产品涵盖数字资产质押解决方案、流动性质押衍生品、机构托管解决方案、非托管钱包产品、应用内集成 DeFi 可访问性等等。Base 是该产品线的自然延伸，可以为 Coinbase 用户提供直接融入 Web3 世界的方式。

1.3 资金安全支持

当前阶段，加密货币市场规模相对有限，因此许多资金庞大的投资者和机构可能对将大量资金投入加密货币领域感到犹豫。他们可能也对将资产存放在区块链上进行投资持有一定的保留态度。

不过，值得注意的是，Base 所依赖的 Coinbase 是一家已在纳斯达克上市的公司。Coinbase 作为一个成熟的市场参与者，在法币入口方面具备可靠性，同时也受到更为严格的监管。这使得 Coinbase 在资金安全方面表现得更为可信，因而可能更能够吸引一些机构投资者的积极参与。

2、OP Stack

在文章开头处有提到，Base 是基于 OP Stack 构建的以太坊 Layer 2 ，并且 Base 将把部分交易费收入返还给 Optimism Collective 国库。那么 OP Stack 是什么呢？Base 基于 OP Stack 构建又能为 Base 带来哪些好处呢？

2.1 Optimism Collective 是什么

据官方介绍显示，Optimism Collective 由社区、公司和公民组成，并由 Citizens’ House 和 Token House 共同治理，旨在避免出现以太坊上其他治理模型所发生的治理陷阱，两大团体的任务是平衡短期激励与项目长期愿景，Optimism Collective 的目的是为 OP 生产强大的公共产品基础设施，增加 L2 的效用，并创造一个高效且完善的加密经济基础。

2.2 OP Stack 是什么

据官方介绍显示，OP Stack 是标准化、共享和开源的开发堆栈，为 Optimism 提供支持，由 Optimism Collective 维护。OP Stack 由许多不同的软件组件组成，由 Optimism Collective 管理和维护，这些组件共同构成了 Optimism 的支柱。OP Stack 是作为以太坊和 Optimism 生态系统的公共产品而构建的。

2022 年的下半年，Optimism 的团队正在将他们原先的开发工作拆解成多个层面的模块（Data Availability Layer，Derivation Layer，Execution Layer 和 Settlement Layer）并开发了 OP Stack。简言之 OP Stack 是 Optimism 开发的一套标准化的开源模块，可以组装成一个定制的链，任何开发者可以基于 OP Stack 去创新出一条新的链。

通过 OP Stack 这个模块，开发人员可以直接插入 OP Stack 的数据位，以此来创建一个 Layer 2、Layer 3 甚至 Layer 4 ；而“标准化”则意味着这个模块的标准是具有共识的，且当前市场认可的；开源则是意味着它可以免费提供给任何人进行迭代和请求。

2.3 超级链

OP Stack 的目标是把孤立的各类 L2 整合成一个单一的超级链（Superchain），将原本孤立的 L2 集成到一个具备互操作性和组合性的系统中，即启动 L2 就像今天将智能合约部署到以太坊一样简单。

本质上而言，如上图所示，超级链是一个横向可扩展的区块链网络，各链之间共享安全性、通信层和开发套件。OP Stack 将是超级链背后的模块化开发堆栈，而超级链之内是无数可相互链接、可通信的区块链。

3、 其他采用 OP Stack 的项目有哪些？

3.1 Worldcoin

今年 5 月，Worldcoin 基金会（Worldcoin Foundation）和 Tools for Humanity (TFH) 早期协议贡献者宣布支持 Optimism Collective，共同构建基于 OP Stack 的可扩展区块链生态系统。

作为进军 Optimism Collective 的第一步，Worldcoin 旗下去中心化隐私身份协议 World ID 将上线 OP 主网，其承载 Worldcoin 的原生钱包 World App 也将于后续迁移到 OP 主网上。

事实上，早在 2020 年，Worldcoin 就曾自行研发并推出过一款仅针对简单支付的 Optimistic Rollup 应用链 Hubble，并推出了 World App 测试版。不过发布后，团队发现测试结果不甚理想，用户需求不止于简单的支付功能，这远远超过了 Hubble 的承载能力，后迁移至 Polygon PoS。

如今，Worldcoin 转而采用 OP Stack 重新构建应用链。不过，此次双方合作的重点将围绕链上身份系统构建。对于 Optimism 来说，一个链上去中心化身份系统是实现其经济自增长的关键，不仅可解锁民主治理和创新，还可以使个体更好地控制自己的财务，并按照自己的方式参与全球经济。为此，Optimism 部署了身份原语 AttestationStation，这个带有实验性质的身份系统允许任何人对其他地址进行任意证明，从而可为一个特定链上地址分配一个信任分数，衡量一个该地址的真实社交关系（影响力）。

对于 Worldcoin 而言，身份系统同样重要，既然要向全球 79 亿人免费发放数字货币，实现财富的公平分配，那就需要一种方法用来确定用户的身份，防止同一人以不同身份多次领取 Token。为此，团队开发了 World ID，作为 Worldcoin 底层身份协议，它可以保证每人只能以瞳孔注册一个账户。另外，团队还开发了一套生物识别设备，通过扫描虹膜来验证一个人的独特性，同时通过零知识证明确保被验证者的隐私。

于 2023 年 7 月 13 日，在测试阶段 World ID 注册人数突破了 200 万。

此外，于 2023 年 8 月 6 日，官方声明除了经过验证的 World ID 持有者数量的增加之外，Tools for Humanity 的 World 应用程序（第一个为 Worldcoin 网络构建的钱包）在网络推出后也出现了全球活动的激增。发布后 7 天内，World App 每周活跃用户增加了 3 倍，每周创建的帐户数量增加了 10 倍以上。

3.2 BNB Chain 应用链 opBNB

除以太坊外，其他公链也可以使用 OP Stack 来实现自身的“Layer 2 ”扩容，比如基于 OP Stack 构建的二层应用链 opBNB。

6 月 19 日，BNB Chain 启动基于 Optimism OP Stack 的 Layer 2 测试网（预计 2023 Q3 启动主网），通过采用 OP Stack Rollup，opBNB 可将计算和状态存储移至链下，从而缓解拥堵并降低交易成本。

opBNB 可将处理速度进一步提升，其性能将优于 BNB Chain。根据官方描述，opBNB 区块时间为 1 秒，转账 Gas 费低至 0.005 美元，每秒处理的交易量（TPS）超过 4000 ，与之对比，目前 1 层 BNB Smart Chain 处理速度为每秒 2, 000 笔交易，每笔交易的平均成本约为 0.109 美元。opBNB 的速度几乎是 BNB Chain 主网的两倍，成本也大幅降低。

opBNB 是一个基于 OP Stack 构建的 BSC 上的二层网络（Layer 2），与 Optimism Rollup 一样，opBNB 通过链下计算交易数据并打包，统一提交给 Layer 1 来提升网络性能，最终实现高 TPS、低 Gas 费用和与 Layer 1 一致的安全性。

那么，opBNB 为何采用 OP Stack 构建自己的应用链呢？

事实上，L2 并非以太坊的独有需求，随着 GameFi 等交易量的激增，即使是 BNB Chain 这种性能更高的公链也面临网络拥堵和 gas 费高的难题，原来的设计和架构已经无法满足新的可扩展需求的情况下，opBNB 应运而生。

而早在去年 9 月，BNB Chain 就曾推出过 zkBNB 测试网（即使用零知识证明的的第 2 层网络），然而由于 zkBNB 无法兼容 EVM（不兼容 EVM 意味着无法与以太坊上的应用程序交互），这才让 BNB Chain 转而开发新的 Layer 2 方案「opBNB」。

opBNB 可为 BNB 公链生态引入了改进的数据可用性、批量交易和高达 100 M 的 Gas 限制。同时，EVM 兼容的 OP Stack，还可以简化将以太坊虚拟机 (EVM) 代码从以太坊迁移到 BNB 链的流程，让开发者更快更轻松地创建开放的生态系统，以此扩大用户群。

和 Optimism 以及 Coinbase 的二层扩容方案 Base 不同，opBNB 建立在 BSC（而非以太坊）之上，BSC 在速度方面本身就优于以太坊，而 opBNB 则会将性能进一步提升，不仅超过 BSC，甚至可能于速度方面优于 Optimism 等扩容方案。

最重要的是，BNB Chain 采用 OP Stack 构建二层应用链具有战略层面的意义。对于公链 BNB Chain 来说，采用 OP Stack 构建应用链仍可用 BNB 作为 Gas 费支付 Token，这意味着，随着未来涌现更多用例，可极大拓宽 BNB 的应用场景，这些收益都可被 BNB 所捕获，长远来说，BNB Chain 甚至能够成为以太坊不断发展的 L2 生态系统的有力竞争对手。

3.3 Zora Network

Zora Network 基于 OP Stack 发行了一个二层 NFT 铸造平台，面向 NFT 创作者、品牌方和藏家，提供一系列 NFT 创作者工具，并可将铸币成本降低 3 美金以下。

围绕创作者和藏家打造最佳的用户体验，Zora 将提供速度更快、成本更低以及体验更好的链上体验。正如 Zora Engineering 发布推文所言：“一个有趣的事实是，‘桥接至 L2’+‘铸造’两步操作的费用加起来都比在主网上单次铸造费用少。”

Zora Network 将与 Zora 市场目前已有的基础设施实现无缝集成。这意味着目前平台已有的艺术家、品牌方和藏家用户将无缝加入至新的二层网络中，包括音乐 NFT 平台 Sound.XYZ、钱包应用 Rainbow Wallet、Web 3.0 创新孵化器 Seed Club 和 PleasrDAO 等在内的 35 个 Web3 项目方实体和目前已有的 13 万用户。为了纪念此次发布，Zora 发行了名为 Energy 的开放版 NFT，粉丝可以在接下来的 数 10 天内铸造。

此外，数据显示，目前为止，Zora Network 自从 21 日启动以来，其桥接地址突破 164 k，链上 TVL 达 1706 ETH，约合 310 万美金。

并且，目前 Zora 已经部署在 Base 上。

3.4 a16z 推出的 Rollup 客户端 Magi

Magi 是今年 4 月由 a16z Crypto 推出的二层 Rollup 客户端解决方案，它采用 Rust 语言编写，基于 OP Stack 创建，在以太坊的传统执行/共识拆分中充当共识客户端，向执行客户端提供新区块以推进链上交易。Magi 执行与参考实现（op-node）具有相同的核心功能，并与执行节点（如 op-geth）一起工作以同步到任何 OP Stack 链。

Magi 的目标是成为一个独立开发的 op-node 直接替代品，以提升 Rollup 的客户端多样性。 A16z 希望构建一个基于 Rust 的新客户端将鼓励整个 OP 堆栈更加安全和活跃，并为生态系统带来更多贡献者。

作为加密领域最知名的投资者之一，A16z 拥有超过 76 亿美元（折合人民币约 500 亿元）的资金用于该领域的投资（截止于去年 5 月）。作为以太坊扩容项目 Optimism 的投资者之一，A16z Crypto 此次主导在 OP Stack 上开发 Magi 客户端，不仅是投资者也是开发者，这也是 a16z 迈进 Optimism Collective 的第一步。

3.5 链上游戏

OPCraft，Keystone 和 Loot 链上游戏通常有高频互动的场景，因此，对速度和成本的要求极高，这意味着，链上游戏需要有定制化的区块链基础设施，众多的扩容方案都推出了针对链上游戏的扩容方案。

OP Stack 是链上游戏最受欢迎的基建之一，其中，最具代表性的链上游戏实验之一 OPcraft 就建立在 Op stack 上。

OPCraft 是一个链上虚拟世界，是使用 OP Stack 构建的一个完全基于链上制作的体素游戏（全链游戏）。OPCraft 是一个自治世界，一个完全链上的虚拟空间，世界里的每一个方面包括每条河流、每片草叶和山脉顶部的雪地，都存在于链上，并且世界上的每一个动作作为以太坊交易发生。

Keystone 则是另一条用 OP Stack 制作的链，它使用自定义预编译将游戏 tick 和 ECS 内置于链中，可将链上游戏速度提高 100 倍，并提供更加强大的游戏体验。

此外，Loot 生态项目 Adventure Gold DAO 于近日宣布将采用 OP Stack 构建 Loot 社区专用的新以太坊 L2 网络 Loot Chain，以降低 Gas 费，并用自己的原生 Token AGLD 用作 Gas Token。该链还使用 Polygon 作为数据可用性层，从而显著降低了 Loot Chain 的构建、部署和运营成本。

3.6 DeBank Chain

2023 年 8 月 11 日，一站式钱包 DeBank 宣布推出基于 OP Stack 的 DeBank Chain，测试网现已上线，主网将于明年上线。DeBank Chain 旨在成为社交资产层，在链层面原生集成了类似 Account Abstraction 的系统，让用户享受近Web2的体验，同时保持与当前 EVM 标准 100% 兼容。新账户体系支持专用L2私钥签名交易，减少社交场景中L1私钥的使用，增强用户L1资产的安全性。

五、生态发展

1、Base 生态系统基金

Coinbase 在 2 月 23 日发布公告，在宣布成立 Base 的同时，还宣布了 Base 生态系统基金的成立，这是一个投资基于 Base 网络的早期项目的资金池，并结合 Base 团队的专门支持以帮助项目取得成功。

Base 生态系统基金为生态构建者提供了 4 个建议（摘自 Base）：

可跟踪通货膨胀率的稳定币（flatcoins）：**在深刻思考在去中心化稳定币的设计中，Base 对“flatcoin”很感兴趣，这种稳定币可以跟踪通货膨胀率，使用户具有稳定的购买力，同时也具有弹性，能抵御金融系统造成的经济不确定性。

Base 也欢迎其他形式的“flatcoin”，不与法币挂钩，而是填补法币挂钩 Token 和不稳定加密资产之间的空间。鉴于全球银行体系近期面临的挑战，Base 认为这些探索比以往任何时候都更为重要。

链上声誉平台：Base 认为，区块链是下一个互联网，去中心化的身份和声誉将在定义每个人的链上角色方面发挥关键作用。通过支持原生链上实体的信誉协议，有机会在链上创建更多的信任。

这可能看起来像 ENS 名称上的 FICO 或谷歌页面排名类型评分，商家的评级/评论，以及其他有助于在链上建立信任的措施。Base 乐见于团队深入思考链上声誉和信用是什么样子，以及如何利用这些系统，同时保护用户隐私和自主权。

链上限价订单簿（LOB）交易平台：虽然 Base 相信现有的交易所产品（例如 AMM）非常有用，并且是关键的 DeFi 基础设施，但 LOB 仍有用武之地，尤其对专业交易员和机构而言。

Base 的高吞吐量将为设计现货交易、限价单、期权、永续权等新机制提供了新机会。而且，建设者可以使用 OP Stack 之类的开源工具来构建L3，在带来更快的速度和更强的控制力的同时，也有可能增加流动性的深度，而且可以通过L2访问。

更安全的 DeFi：随着 DeFi 生态系统的不断发展，Base 认为需要更好的工具，让用户和开发人员能够跟上创新的步伐，同时保证资金安全。

包括可以防止智能合约代码漏洞或协议逻辑错误的工具（例如，自助安全测试工具，审计服务），帮助减轻生态系统攻击影响的工具（例如，威胁预防，断路器和事件响应系统），以及最终的链上保险和覆盖协议，或任何其他可以在智能合约失败时为用户提供关键支持的产品。

建设者正在构建系统、协议和工具，使每个用户都可以在链上存储他们所有的资金。Base 认为，这些基础设施对于吸引下一个十亿用户是绝对必要的。

Orbiter Finance 是一个去中心化的交叉汇总桥，用于传输以太坊原生资产，这是第 2 层的基础设施，它提供低成本和几乎即时的传输。Orbiter Finance 被设计为一个去中心化的交叉汇总桥，用于转移以太坊原生资产。

  • Axelar Network

Axelar 是一个通用覆盖网络，安全连接所有区块链生态系统、应用程序、资产和用户，以提供 Web3 的互操作性。

Axelar 由去中心化的验证器网络、安全网关合约、统一转译、路由架构以及一套软件开发工具包 (SDK) 和应用程序编程接口 (API) 组成，以实现区块链之间的可组合性。这支持开发人员为他们的用例构建最好的平台，同时能够访问每个其他生态系统中的用户、资产和应用程序。其可以依赖于提供统一代码库和治理结构的网络架构，而不是成对的跨链桥。

Axelar SDK 为开发 Web3 应用程序提供了丰富的套件，确保开发人员拥有建设所需的工具。借助这些工具和 API，开发人员可以使用 Axelar 网络及其 SDK 编写可轻松部署在所有 Axelar 连接的生态系统中的 dApp。

换句话说，Axelar 将跨链互操作性提炼为一组简单的 API 请求。这绝对是其将被采用的核心，由于开发人员在部署 Web3 应用程序时的体验必须与当前 Web2 开发人员的体验类似，因此必须消除底层网络和特定生态系统的部署细节。在 Web2 中，底层网络和特定于生态系统的部署细节在很大程度上得到了简化。

Axelar 于 2022 年 2 月 15 日以 10 亿美元的估值完成了 3500 万美元的 B 轮融资，加上种子轮与 A 轮融资，总融资额超 7300 万美元。

friend.tech 是一款类似 bitclout 的粉丝经济类产品，其近期宣布基于 Base 网络上线。因为其微创新的将个人影响力化为股份可买卖，于 8 月 11 日突然走红，用户大量涌入导致其服务器崩溃。其邀请码制度产生的用户社交行为也让其具有很强的话题度和传播。

该产品的形式可以理解为每个账号是一个代币，用户可以买卖，币价也会相应涨跌。如果其他用户购买了你的账号代币，你可以从交易中获得分成。如果你购买了其他用户的账号代币，可以进入其私人聊天室，从而可以和所购买的账号博主交流，也可以在所有购买持有该账号代币的人组成的聊天室里交流。

  • CyberConnect

CyberConnect 是第一个去中心化的社交图谱协议，为 Web3.0 社会网络和元宇宙服务。它的使命是将社交图谱数据的所有权和实用性还给用户，使社交联系民主化，使社交图谱具有自主权、可移植性、可组合性，让社交图谱变得不仅仅只与区块链有关，还能让用户通过这种关联发现更多的社交身份意义。

CyberConnect 是一个对开发人员友好的协议，为所有的 Web3.0 开发者提供一个基础设施用以整合。

CyberConnect 目前已经获得两轮共 2500 万美元的融资，并且在 2023 年 5 月 19 日的公开发售中筹集 540 万美元。

2.6 Infra

  • Thirdweb

Thirdweb 是一家英国 Web3 软件开发商，推出免费的 Web3 开发工具，让开发人员无需编写任何代码，即可构建、启动和管理 web3 项目，开发者只需点击几下，即可添加功能，包括 NFT、社交 Token 和货币、Token 交易市场以及 NFT 交易等。

Thirdweb 于 2022 年的 8 月 25 日以 1.6 亿美元的估值完成了 2400 万美元的战略融资，由 Haun Ventures 领投，Coinbase Ventures、Shopify、Protocol Labs、Polygon、Shrug VC 等参投，除此之外，个人投资者还包括硅谷顶级风投公司 Kleiner Perkins 的合伙人 Joseph Lacob，他也是 NBA 金州勇士队的大股东。

并且在之前 500 万美元的种子轮融资中，Thirdweb 还获得了 NBA 达拉斯独行侠队老板 Mark Cuban 和美国知名企业家 Gary Vaynerchuk 的投资，总融资额达到 2900 万美元，拥有雄厚的资金力量。

六、总结

Base 和 Linea 一样，都背靠着强大的公司，拥有雄厚的用户资源和资金支持，这为 Base 的发展扫除了很多障碍。目前 Base 还处于相当早期的阶段， 2023 年 8 月 9 日才正式上线主网，但是其业务数据却是远超在 2023 年 7 月 18 日就已经上线了的 Linea，其 TVL 甚至超越 Starknet 成为 Layer 2 TVL 排行榜中的第五位，由此可以看出 Base 十分受市场的欢迎。

另外，在用户能够通过转币到合约的方式提前进入 Base 链之后，产生了一次引人瞩目的财富效应，出现了千倍币“$BALD”，从而使 Base 链一时火爆，吸引了大量社区成员的关注。

在这次千倍币现象中，Base 获得了显著的关注度，这背后或许归因于 Coinbase 团队卓越的运营和操作能力，在主网上线前善于抓住市场情绪，为 Base 链创造了高度的热度。此外，这也可能意味着存在一些资金庞大的投资者进场。

此外，近期 Base 主网刚刚上线，其对于行业的影响和意义还需要随着时间的推移逐步显现。尽管其背后有强大的背景支持，并且是超级链的一员，但就目前来看这似乎还并不足以使其从众多竞争中脱颖而出。

因此，要在众多 Layer 2 项目中脱颖而出，还需要 Coinbase 提供更多强力且有效的支持，并且与 Optimism 展开更为紧密的合作，共同推动 OP Stack 的发展。同时，Base 自身也应该积极在 OP Stack 基础上创新，创造出更多对整个领域及行业有意义的创新成果。
