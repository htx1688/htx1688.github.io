比特币是第一个加密货币项目，它设计了一个安全、去中心化和抗审查的网络，受到大众的广泛信任，已然成为了最重要的区块链项目。然而比特币基础层非常简单，这使得它在成为最安全、去中心化和抗审查区块链的同时，也缺乏了更多的功能性，发展至今，比特币区块链网络主要功能仍然是做价值转移。相比而言，其他类型的区块链（如Ethereum solana等）有着更多样化的生态。

直到Ordinals协议的发布，使得在比特币的链上创建 NFT 成为可能，开始逐渐打破了比特币缺乏可扩展性这一现状，立即引起公众的关注，BRC20的创新也为比特币提供了更大的可能和想象空间。自该协议推出以来，已有超过 2700万个Ordinals被记录，比特币的价格也出现了上涨，因为用户需要寻找交易平台来购买比特币，以便他们可以投资于加密世界中引入的最新数字资产。8月31日，币安矿池也正式推出Ordinals铭刻服务，供用户注册“聪”，存储所有权记录等。

那么 Ordinals 是什么原理，半年以来它又有哪些发展，为什么各大机构开始纷纷布局比特币Ordinals？相关项目有没有参与价值以及如何参与？本文就将为读者全面梳理比特币Ordinals生态。

Ordinals的起源：比特币NFT

在当前区块链网络中，NFT 主要通过基于区块链协议（例如以太坊上的 ERC-721 ）构建的代币化标准来铸造和验证的，并在 OpenSea、Rarible、Nifty Gateway 等中心化市场上进行交易。满足这些标准的交易将需要支持它们的钱包，但它们会受到网络拥塞的速度和成本的影响，并且由于依赖中心化市场而遭受第三方风险的固有敏感性。但有没有一种办法可以直接在区块链的基础层上铸造、购买和销售 NFT，并将其印在其最小的货币单位上呢。

比特币网络的非图灵完备性导致其不支持智能合约，所以无法通过合约来发行NFT。Rodarmor的解决方法是将NFT刻在BTC上。由于比特币总量有限，2100万枚，其价格又很高，所以比特币有个最小单位聪（英文sat），1个比特币可以分割成1亿个聪。Rodarmor的想法是：我们能否按照一定顺序排列这些“聪”，给它们分配一个介于 0 和 2,100,000,000,000,000 之间的序数，然后，把它们连接到其他信息：图片、文字、视频甚至一串代码。从而每个“聪”都变得独一无二，不可替代。这就相当于让比特币拥有了原生的、创造 NFT 的能力。Rodarmor想创建一个系统，来为聪编号，给每个聪分配一个唯一的序列号（Ordinals），就得到了从1开始到1亿个整数，每一个序数就代表了一“聪”比特币，是不可再分割的。这个唯一标识符将允许用户跟踪区块链上的每个“聪”。Rodarmor希望能够以一种彻底的比特币原生方式来实现这一点，不使用侧链或单独的代币。

2023年初，Rodarmor在比特币主网打造的NFT协议Ordinals正式亮相。允许根据开采顺序在时间和空间上识别和跟踪每个单独的“聪”（sat）。该协议允许无与伦比的链上资产验证，确保如果任何人从一个钱包转移到另一个钱包，其序号保持不变。Ordinals还允许铭文许多不同类型的内容，可以将图像或文本、视频、音乐等刻在比特币网络上的单个“聪”上。

自推出以来，Ordinals 彻底重新引发了围绕比特币NFT的讨论，交易量呈爆炸式增长，Ordinal NFT 的新市场如雨后春笋般涌现。Dune Analytics最新数据显示，比特币NFT协议Ordinals累计铸造量已达到27,819,966枚，产生的总费用已达到1949枚BTC。

主要原因是借助铭文服务，创建和交易Ordinal NFT比通过传统的铸造方法要容易得多，序数铭文消除了对更高层智能合约的需求，降低了潜在创作者、买家和卖家的进入门槛，并且还有助于抵消因铭文而与较大区块大小相关的较高交易费用，这是吸引用户使用序数铭文的关键。

Ordinals的探索：BRC20

与加密货币领域的大多数新颖项目一样，Ordinals引起了人们的兴趣和炒作。在 Ordinals 协议发布 2 个月后，推特用户 @domodata 在 Ordinals 协议上又提出了一个比特币上的Token标准 — —BRC20标准，@domodata 利用JSON数据的 Ordinals 铭文来部署代币合约、铸币以及转账。可以把 BRC20 标准当成 Ordinals 的一种应用类 NFT，相当于一张支票，但BRC20 没有智能合约。

BRC20核心：部署代币，Mint（铸造）代币和转移代币

他在推特上说道：我基于Ordinals协议开启了一个对于BTC上同质化代币发行的实验，就叫它BRC20。与之同时，基于BRC20标准的实验性代币$ordi也横空出世，为致敬BTC，$ordi也设计为2100万个，每个人都可以免费Mint。

@domodata 部署的第一个 BRC-20 代币“Ordi”

BRC20标准由于设置了代币供给量和最大铸造量不可改变，限制了代币的发展；而且代币命名仅限于 4 位数。为了解决以上这些问题，ORC20标准应运而生，ORC20取消了代币命名限制，增加了代币的可升级功能，让项目方可以基于代币做更多的赋能。同时增加了如设置版税、设置白名单等高级功能等。但是由于ORC20的基础设施暂未发展完善，且财富效应依然聚集在BRC20上，因此ORC20目前受到的关注十分有限。

BRC20的市场正在迅速扩大。加密世界中已经出现了37554种使用 BRC20 标准制作的不同代币，其代币市值也来到2.6亿美元，而且这个数字还在不断增加。BRC20正在成为一场新的革命，最重要的是，BRC20 背后的团队正在快速发展，他们承诺完善 BRC-20 的功能，这将使开发人员更容易使用 BRC20 制作代币。随着 DeFi 领域的发展，我们应该预计会看到更多使用 BRC20 的代币和金融产品。

Ordinals协议重塑比特币的未来

Ordinals协议一经发布，就引起了成千上万比特币用户的关注和想象。与以太坊 NFT 不同，使用 Ordinals，资产可以直接在比特币基础层上表示，而不需要单独的侧链或代币，这意味着可以使用现有的比特币基础设施在比特币网络内更无缝地创建、转移和管理资产。此外，与以太坊 NFT 相比，铭文提供了更大的灵活性和定制选项。该协议允许创建更复杂和可定制的内容类型，并且能够用任意内容铭刻“聪”，这意味着铭文可用于广泛的数字资产。

Ordinals协议改变了比特币的范式，提高了比特币网络上资产创建和管理工具的多功能性。引入Ordinals协议为每个“聪”增加了独特性，“聪”计价时代正式到来。比特币也不再只是一种价值储存手段，它现在允许创建类似NFT的独特数字资产。

Ordinals协议从多种不同方式改变了比特币生态系统，为 Web3 资产市场以及整个比特币的发展做出了贡献。 

Ordinals协议为艺术家和创作者提供了一种将他们的作品存储在最安全和去中心化的区块链上的方法，同时为比特币网络的安全性和激励做出了贡献。铭文导致交易费用和区块大小增加，激励矿工保护网络安全。也有可能会有越来越多的用户加入网络并运行节点，使比特币网络在未来更加可持续。 

同时Ordinals协议有助于促进比特币的大规模采用，并为生态系统带来更多价值。可以看到，未来比特币链上的数字资产将比其他链上的 NFT 更有价值，并且正在构建的工具将支持新一代比特币 dApp 和基础设施。比特币的成功对于 RSK、Stacks 等Layer2链的成功至关重要，高需求将为扩展层的发展提供更强有力的理由，从而提高其使用效率。

Ordinals项目及平台现状

Ordinals 协议的提出让比特币不再仅限于价值储存和支付功能，打破了比特币生态系统单一化的局限性。仅仅半年，围绕 Ordinals 已经派生出了多个分支协议，这些协议和项目为比特币网络带来了更多的新玩法

Inscription （BTC NFT）

1. Sub 10K：Sub10K指的是铭文号排名前1万的铭文，具有早期价值，是老玩家们的最爱。目前是整个ordinals生态的最强共识，最受欢迎的是合集系列，比如Ordinals Punks是Sub10k里的系列之一，由 100 个像素化身组成，这些化身已通过 Ordinals 协议按铭文方式在比特币区块链上完成铸造并且可以进行交易。

2. Bitcoin Punks：是首个使用Ordinals协议，以以太坊上的CryptoPunk为灵感衍生出的项目，所有资产都已经由收藏家免费铸造。目前基于比特币主网 Ordinals 协议的 NFT 项目 Bitcoin Punks 已完成铸造，共计 1 万个。基本上都是老玩家持有，背后还有 OKX 背书，和Ordinals Punks一样也有很强的共识。

3. Taproot Wizard：Taproot Wizard 是一个在比特币链上创建的 NFT 项目，由加密资产开发者 Udi Wertheimer 发起，旨在纪念比特币的 Taproot 升级，以及比特币社区比特币巫师的经典形象。因此，社区中不少人支持 Taproot Wizard 就是认为该 meme 和比特币的崛起有着密不可分的关系，是比特币最原始、最纯正的 meme。

BRC20（同质化代币）

1. ordi：ordi是Brc20的第一个代币合约，成为BTC链上第一个meme代币，具有很强的炒作性，已经全部mint完毕。目前其市值虽然只有7000万美元，但不可改变其在Brc20中的地位和影响。

2. sats：sats是Brc-20协议上的第九个铭文代币，全称是satoshis，是bitcoin最小的单位，简称为聪，也称为比特聪。1个BTC=1亿聪。总量2100万亿，对应2100万个BTC。sats目前已经mint完成73%，持币地址超过3万个，市场热度已经远远超出了同类brc20代币，甚至超过元老级别的ordi，是目前最有潜力的BRC20代币。

针对这一点，sats官方社区给出了答案：1.sats意义非凡：是比特币最小的单位。2.持币地址足够多，社区认可度高且广。3.对矿工矿场有很强的利益驱动，2024年比特币减半后，矿场必须要找到新的盈利点，不然收入也减半了，那这个总量足够大、共识足够大的sats就成了他们选择的百倍千倍理由。4.BRC20代币总市值很小，容易爆发，而sats是BRC-20的龙头币，必然涨幅高。5.公平分发，总量大，mint完时间足够长，对所有人都公平。

代打平台

1. UniSat Wallet：UniSat Wallet于4月23日正式上线，是最早支持BRC20的钱包。UniSat 对 BRC-20 的成功有着巨大的贡献，当每个人都对比特币图像NFT充满热情时，UniSat 致力于集成 BRC-20。UniSat Wallet Ordinal Protocol 的官方钱包，支持Ordinals NFT和BRC20代币的存储、铸造和交易。

2. BitBox：BitBox是目前全网性价比最高的打铭文工具。目前BitBox已免去了平台费用，总费用主要包括铭刻费用和网络费组成，且铭刻费用会自动返回给用户，因此和Unisat相比，BitBox更加便宜。以mint sats（gas费：12聪）为例：

比特币Ordinals走向何方

比特币是目前市值最大的加密货币，而Ordinals 给比特币生态开启了新的方向，但凡有比特币资金流入，都会给Oridinals生态打开了一个很大的天花板。

Ordinals 协议给比特币引入了以前从未实现过的新功能，包括智能合约以及代币的创建。随着比特币底层技术的不断发展，我们可以期待新的功能可以使铭文更加高效、安全和通用。未来的比特币升级可能会引入新的机制，将数据嵌入到“聪”中或允许更复杂的铭文。与此同时，区块链可扩展性和数据存储的进步可以实现更大的铭文和更复杂的比特币Ordinals协议。

首先，BTC Ordinals 的价值和效用在很大程度上取决于市场采用情况。随着越来越多的艺术家、创作者、企业和消费者了解和使用 BTC Ordinals，它们的价值和效用可能会显著增加。BTC Ordinals 交易市场和平台的发展也可能对其采用发挥至关重要的作用。

其次，BTC Ordinals 与其他区块链系统和数字资产的互操作性也会带来令人兴奋的可能性。例如，未来的发展可能允许 BTC Ordinals 与以太坊智能合约交互，用作DeFi平台的抵押品或纳入dApp中。大型区块链机构也在积极入场，OKX、Element NFT Marketplace 和Binance pool 都正在筹备推出Ordinals铭刻服务和BRC20 交易市场，这为 Ordinals 协议与 BRC20 代币提供了广阔的运用场景。

BTC Ordinals 的出现开启了比特币网络和区块链技术发展的新篇章。尽管仍处于起步阶段，但该协议开启了数字所有权、创造力和数据验证新时代的巨大潜力。BTC Ordinals 未来将变得更加成熟，用例也会扩大，尽管肯定存在挑战和不确定性，但比特币社区的创新和适应性让我们有理由对 BTC Ordinals 的未来持乐观态度。
