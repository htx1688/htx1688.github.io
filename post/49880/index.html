在“明星公链” Aptos 发行 Token 半年之后，被社区称为“MOVE 双子星之一”的 Sui 也即将迈入主网阶段。

Sui 开发团队 Mysten Labs 在去年 9 月完成了 3 亿美元融资，并已完成对破产的 FTX 持有股份的回购，展现较强的资金实力。背靠 a16z、Coinbase Ventures、Jump Crypto 等多家明星资本，Sui 网络上的生态令人期待，但由于 Sui 正式网尚未正式上线，本文将根据 Sui Network 测试网发掘其生态应用。

Wallet

Sui Wallet

Sui 官方团队开发的钱包。通过使用该钱包，用户可以进行 FT 和 NFT 的发送、存储和交易。除了钱包基础功能外，用户还可以领水参与钱包内置推荐的 dAPPs 测试，以及在钱包内进行质押 Sui Testnet Token 等活动。

Ethos Wallet

Sui Builder Hero（建设者英雄）获奖项目之一，并于今年 1 月份完成了由 Gumi Cryptos Capital 和 Boldstart Ventures 领投的 420 万美元种子轮融资，其他融资方还包括 Sui 网络开发方 Mysten Labs。用户使用该钱包进行 FT 和 NFT 的发送、存储。

Suiet Wallet

Suiet Wallet 是为社区开发的开源钱包，用户可以使用该钱包进行 FT 和 NFT 的发送、存储和交易。Suiet Wallet 当前只支持网页浏览器扩展程序，未来会开发移动端。

DeFi

MovEX

MovEX 是结合了 AMM 和订单簿优点的去中心化交易所。用户可以领水参与测试，在该平台上使用基础的 DEX 功能，进行交易和流动性提供 LP。MovEX 未来还会推出 ILO（Initial Liquidity Offering）功能，该功能可以为项目发行方提供定制的 Token 发行方案，通过选择时间段，价格范围和拍卖算法调整进行初始 Token 销售，以帮助项目最大限度地筹集资金。

WispSwap

Sui 链上的 AMM DEX 之一，提供交易、流动性挖矿、预测市场功能，将来会推出质押、借贷等功能以及 WISP Token。

CetusProtocol

Cetus 是基于 Move 生态的一个 DEX 和资金流动性协议，当前该协议已集成 Aptos 和 Sui。该平台支持用户进行交易和流动性提供，此外还可以还支持 Near、Fantom、Ethereum 等多条公链桥接资产。

Turbos Finance

一个建立在 Sui、由 Jump Crypto 提供支持的 DeFi 协议，在 Sui 主网正式开通后会开放交易和流动性提供服务。除了现货交易外，用户还可以在该平台上参与期货交易。Turbos Finance此前表示会有针对测试网阶段的早期贡献者有激励计划。

Aftermath Finance

Aftermath Finance 是一个建立在 Sui 网络上的原生 DeFi 项目，同时也是 SUI Builder Hero 获奖项目之一。在测试网阶段，用户可以在该平台上领水参与交易、质押、流动性挖矿等活动，未来平台还会推出借贷功能。

Araya Finance

Sui 网络上的原生 DeFi 协议之一，提供交易和流动性挖矿等 DEX 功能。用户在该协议上进行交易时需要支付 0.3% 的费用，其中的 0.25% 用于支付给 LP 费用。Araya Finance 还构建了一种名为 Stable Swap 的功能以提供低滑点的交易，并只需要支付通常为 0.02% 的费用，远低于一般的 Araya AMM。

Suiswap

Sui 链上的 AMM DEX 之一，支持用户进行交易和流动性挖矿。Suiswap 在近日宣布参与了 Sui 测试网期间的官方社区任务的用户将会获得一定的 Suiswap Token 空投奖励。

Scallop

Scallop 是一个超额抵押借贷协议，拥有债券、AMM 交易等功能，其最先建造在 Solana 上，已初具规模，现拓展至 Sui 上，并成为了首个获得 Sui 基金会资助的 DeFi 项目。此外，Scallop 在 2021 年的 Solana Ignition 亚洲黑客马拉松中也获得了社区投票项目的第一名。当前 Scallop 正在以 2000 万美元的估值寻求种子轮融资。

NFT Marketplace

Clutchy

Clutchy 是 Sui 链上的 NFT 交易市场。用户可以在该平台进行 NFT 交易 ，项目方也可以在该平台进行 NFT 发行。此外，Clutchy 还支持用户在连接钱包后，直接在平台上参与多个游戏的游玩。

BlueMove

BlueMove 是建立在 Move 语言公链的 NFT 交易市场，当前已经集成了 Sui 和 Aptos 网络，并推出了 APP 端。除了 NFT 交易外，BlueMove 还提供了 DEX 、质押 NFT 、Launchpad 功能。使用 BlueMove 进行 NFT 交易需要缴纳 2% 的平台费用。当前该平台已经推出原生 Token MOVE，主要用于参加白名单 NFT 铸造、质押，用户可以通过在 BlueMove 上质押 MOVE 或 NFT，以及交易 NFT 获得 MOVE。

Sui Gallery

Sui 网络上的 NFT 交易市场之一，提供 NFT 交易、Launchpad 、铸造功能。该平台在测试网期间已经拥有超过 30,000 名铸造 NFT 的用户。

Launchpad

SuiPad

Sui 链上的 Launchpad 平台之一，用户可以通过该平台参加一级市场项目的 IDO。用户可以通过质押 Token SUIP 获得特定的等级，等级将会决定用户参加 IDO 时可以保证分配的额度。SuiPad 还设计了一个保险功能来保护用户的资金，用户在参加活动前参保，这笔资金会被发送到 SuiPad Shield Pool ，当项目破发时 SuiPad Shield Pool 会弥补一定的损失。

Suistart

Sui 链上的 Launchpad 平台之一，用户可以在该平台参与一级项目打新。除了提供 Launchpad 功能外，Suistart 还提供了 DeFi 功能如 Token 交易、质押等。Suistart 未来会发行 Token SUIS，未来用户可以通过质押 SUIS 保证获得一定的参与额度。

NFT

Sui Punk

Sui Punk 自称是 Sui 链上的第一个 NFT 系列，总供应量 2,992个。该 NFT 系列于 2022 年 8 月在 ETH 网络上铸造，在 Sui 网络主网上线后会空投到该网络上。

Stork

Stork 于 2022 年开始做域名服务，但随后改变方向，推出 Stork 品牌 PFP 系列。用户持有 Stork 白名单通行证即 Stork Pass，可获得 Stork 在正式上线 Sui 主网的后的 NFT 铸造权以及 Stork Club 的一系列特权。截止 4 月 17 日 ，Stork Pass 地板价为 27 MATIC。Stork 表示将在未来为持有者以及一些社区贡献者进行空投。

域名

Sui Name Service/SuiNS

Sui 链上的域名服务商之一，用户可以注册 Sui 域名构建自己的 DID。此前 Sui 开发团队 Mysten Labs 完成了对 Sui Name Service（SuiNS）部分资产的收购，因此社区也称其具备一定的正统性。当前已经有超过 60 万个域名完成了注册。用户还可以在该平台上进行拍卖竞价，以获得独特的域名。

Social

Suia

Suia.io 是一款基于 Sui 网络构建的社交 dAPP。组织方可以在该平台上进行申请品牌并发起活动，而用户方则可以参加社区组织的活动、收集创作者作品、领取贡献证明 NFT、以可组合的方式与 PFP 交互、在Suispace 上进行展示生活等。通过 Suia 上的活动，用户可以与其他陌生人建立新的社交关系。当前该平台已经拥有超过 23 万用户，发生超过 7 万个事件。

GameFi

AIDAMETA

AIDAMETA 是建立在 BSC 和 SUI 等高效区块链上的全链游戏平台，同时也是 Sui 网络上首批 GameFi 项目之一。AIDA 目前拥有三个产品：World of META：AIDAMETA 的元宇宙形式、 Battle of META：该游戏的主战场 和 Home of META ：AIDAMETA 的社交区域。

Abyss World

Abyss World 是由 Mysten Labs 出资、在 Sui 链上的旗舰开放世界 ARPG 游戏。该游戏宣传将有多种游戏模式，玩家可以在 The Grand Colosseum 进行大型 PVP 战斗、在 The Battle Arena 进行小型 PVP 战斗、在 Central City 进行 PVE 模式游玩。当前 Abyss World 仍在开发中，并列入 Steam，因此玩家可以在 Steam 中添加该游戏进入游戏愿望清单以追踪游戏最新进展。
