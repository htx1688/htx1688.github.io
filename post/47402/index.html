备受期待的空投回馈给社区和用户，申领者热情高涨导致Gas费飙升。那么，这个团队还有什么其他的花招？ Blur 怎样才能在竞争中颠覆 OpenSea？ 今天让我们进一步探讨。

对这次空投的预期在三个游戏化阶段进行统计，分别奖励交易者、上市者和投标人，这大大提高了 Blur 的知名度 ，并帮助它成为长期 NFT 市场领头羊 OpenSea 的真正竞争对手。

然而，只是空投的话并不能与 OpenSea 抗衡，空投带来的轰动和势头是暂时的，持久度不会更高。

以 NFT 市场 LooksRare 和 X2Y2 为例，它们都在去年初推出了使用 LOOKS 和 X2Y2 代币的激励措施。 正如下图价格比较图表中看到的那样，这两种代币最初在推出时名噪一时，但此后价格大幅下跌。 随着时间的推移，两者都未能从 OpenSea 手中夺取重要的市场份额。 当然，可以这么说，他们的项目仍在发展，叙事还在进行中，但显然现在已经不像以前那样令人兴奋或诱人了。

所以 Blur 推出 BLUR 是实现去中心化治理协调的策略，是从 OpenSea 和其他竞争对手手中夺取市场份额的策略。 不过，后一种战术效果的大部分可能是暂时的，并且会在 BLUR 推出时提前释放，至少如果其他早期 NFT 市场代币有任何迹象的话。

那么对 Blur 来说是个好消息吗？ 它确实拥有引人注目的基本 UX 产品，这确实使其成为 OpenSea 的合法挑战者，超越了代币经济学和口碑。

例如，Blur 聚合了来自所有主要 NFT 市场的 NFT 列表，包括来自 OpenSea 的 Seaport Protocol，这样 Blur 就不再受 OpenSea 的市场黑名单限制。 相比之下，目前 OpenSea 仅汇总在 Seaport 上构建的项目的列表，例如 ENS Vision 和 Sound。

此外，Blur 从第一天起就将速度提高了一倍，以通过实时列表和即时 NFT 元数据显示使自己在竞争中脱颖而出。 根据 Blur 的说法，它的市场大约比 Gem 快 10 倍，gas 效率高出近 20%，Gem 是 OpenSea 为资深交易员提供的 NFT 聚合器。

广泛的选择、令人印象深刻的速度和流畅的用户体验——这些是 Blur 旨在立足并获得更多市场份额的支柱。 但问题是，随着时间的推移，这些支柱与 BLUR 的推出是否足以真正超越 OpenSea？

这是万亿 gwei 的问题，现在简单的答案是否定的。

可以肯定的是，如果 Blur 想要在 OpenSea 已经占据主导地位的情况下果断领先，还需要做更多的工作。

富有远见的 OpenSea 于 2017 年 12 月推出，多年来一直处于领先地位，并已成为 NFT 生态系统的主宰者，在其发展过程中积累了无与伦比的深度经验和资源。

OpenSea 目前与之比肩的地方是 NFT 交易量水平，因为 Blur 数周以来一直与现任巨头并驾齐驱。

考虑到 OpenSea 的先发优势，这种特征非常令人印象深刻，但它在很大程度上也受到 BLUR 的预期和炒作的推动，我们已经注意到其战术效果最终将是暂时的。

如果 Blur 要真正继续并赢得 NFT 市场大战，它将需要继续有意义地推进并建立在其 UX 支柱和 BLUR 发布的基础上。 这说起来容易做起来难吧，那么后起之秀该怎么办呢？

在我看来，为了成为下一个 NFT 市场冠军，Blur 需要具备以下五点：

1.保持独家优势

Blur 的速度优势是真实存在的，它的 gas 效率非常高，它的 gas-less 投标系统非常灵活。 当然，这些是从当前来看 Blur 独特地位的“护城河”。

然而，只要它们是 Blur 独有的，这些护城河非常有用。 如果其他竞争对手在这些方面赶上来（这是可能的），那么 Blur 的形象将开始泯然众人。

Blur 要想取胜，就需要保持更快、更高效、更灵活。 同样，这说起来容易做起来难，这里不一定有即时答案，但无论如何：必须保卫“护城河”。

也就是说，大约 65% 的 Blur 早期用户来自 OpenSea 和 Gem，因此，如果 Blur 能够稳固其护城河，那么就可能永远留住这些用户。 作为 Gem 的忠实用户，我现在可以说我发现 Blur 和 Gem 的用户体验相当匹配——Blur 有一些我更喜欢的地方，Gem 有一些我更喜欢的地方，我真的很喜欢。 在这些激烈的竞争中赢得超级用户确实是最重要的底牌。

2.保留奖励

……流动，就是这样。

Blur 的游戏化空投是一种有影响力的策略，到目前为止，它一直帮助市场在 NFT 交易量水平上与 OpenSea 并驾齐驱。

那么，称之为战术胜利，但 Blur 需要不断取得更多战术胜利才能取得成功，即使每次的效果都只是暂时的。 Blur 似乎已经牢记这一教训。 自空投后，他们推出了“第 2 季”BLUR 奖励，这将使下个月市场上的所有出价和上市点数翻倍。 季节性方法在这里很有意义。

3.推进计划

Blur 要想不断进步，就必须不断发展和创新。 无论出于何种原因，在卫冕冠军 OpenSea 面前停滞不前，无疑都会让 Blur 失去机会之窗。

作为 Bankless 和 JPG 的贡献者，我亲眼目睹了 NFT 团队是如何在幕后工作的，所以我可以放心地假设 Blur 团队有自己的新想法、新集成和新推出的路线图，它的新功能正在逐渐完善。

例如，也许 Blur 最终可能会通过将优化的专业交易 UX 集成所有顶级 L2 链上来试图战胜 OpenSea。 我们拭目以待。 但最终由 Blur 团队决定，优先考虑他们认为最好的，并以熟练的方式执行和围绕这些计划进行创新。

4.保持斗志

毫无疑问，Blur 在这场比赛中处于劣势。 OpenSea 的能力、资源和活跃用户是巨大的。

如果 Blur 团队不能克服一切困难，他们将不无法击败 OpenSea。 有时是失败者获胜，有时是不屈不挠的战斗精神，“我们会赢”的渴望，随着时间的推移，与根深蒂固的现任者产生了差异。

每次我在国际象棋比赛中与评分高得多的棋手获胜时，都是因为我进入比赛时会拼命战斗，并且在每个回合都非常专注。 如果 Blur 团队能够保持类似的渴望，他们或许就能完成以弱胜强的对抗。

5. 祈祷没有 SEA 代币

或者祈祷没有 SEA 代币。

我的意思是，成功与否还要看对手。 换句话说，Blur 可以保持其 BLUR 奖励的流动性，并继续一个接一个地推出很酷的功能，但最终仍会被 OpenSea 击败。

事实上，许多人认为 OpenSea 永远不会推出自己的代币已成定局。 情况很可能就是这样。 但想象一下，如果他们确实向过去的用户空投了 SEA 呢？ 这将是一个里程碑式的举措，可能会在未来几年进一步巩固 OpenSea 的主导地位。

毕竟，OpenSea 击败 Blur 的一种方法不就是对上述四个要点进行“以彼之道还施彼身”吗？ 蚕食 Blur 的 UX 护城河，推出自己的奖励，推进自己的计划和斗志？ 即便如此，随着时间的推移，OpenSea 最终可能不需要 SEA 代币来击败 Blur。

因此，对于 Blur 团队来说，这可能不仅仅是把每件事都做好。 它也可能像国际象棋一样，能够执行他们的计划，同时也希望 OpenSea 做出错误决策或犯下无法挽回的错误。
