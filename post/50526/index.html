原文来源：Sui World

千呼万唤始出来， 5 月 3 日晚，Sui Network 如约上线主网，在之前的文章中，Sui World 总结了关于Sui Network 主网上线不得不关注的问题，感兴趣的用户可以点击查看。目前主网成功上线，主网体验是否如之前宣传的一样高效、费用低廉、体验顺滑？本文 Sui World 将与你一起初次体验 Sui Network。

Sui 信息平台

1 ）Sui 网络信息页面

这里您可以看到 Sui Network 开发网、测试网、主网的性能以及功能的区别，其中主网上共有 98 个验证节点和两个由 Mysten Labs 运行的节点。

2 ）Sui 区块浏览器

在 Sui 区块浏览器中，可以看到 SUI 的一些基础信息，包括 TPS，网络节点的分布，出块状况，价格，市值等信息，也可以查询地址交易信息。

Sui 钱包

这里 Sui World 以 Suiet 钱包为例，点击链接进入官网，选择「连接插件钱包」下载插件，如果是新用户，需要如使用常规钱包一般，新建钱包，记录助记词登陆即可。

Sui 跨链桥

Sui Token 跨链桥现已上线，用户可将 22 个 Wormhole 支持的资产直接跨链至 Sui 网络。 

最初， Wormhole Connect 支持跨以太坊、Polygon、BSC、 Avalanche、Celo、Moonbeam、Solana 和 Sui 之间跨链的资产为：ETH, WETH, USDC, MATIC, WMATIC, BNB, WBNB, AVAX, WAVAX, FTM, WFTM, CELO, GLMR, WGLRM, SOL。

下面，我们就一起来体验官方跨链桥 portalbridge。

选择「链接钱包」后，选择网络，选择资产，可以选择原生链资产 或者 SUI 作为手续费（比如，选择 BSC 链，手续费可选 BNB 或者 SUI），选择不同的链跨资产到 SUI 上，手续费都是一样的， 0.2 SUI 或者是等值的原生链资产，价格还是非常便宜的。

Sui NFT Marketplace 

目前 Sui Network 上头部 NFT 交易平台 Souff l3 已经上线主网。 

点击官网链接：https://souff l3.com/

点击「Connect 」链接钱包后，即可参与平台的 Launchpad、NFT 的买卖。

由于 Sui 主网刚上线，很多生态项目还未一起上线主网，并且 Sui 的初始筹码分发全部都在 CEX 中心化交易所内，而不是在链上，大部分中心化交易所目前还未开放提币，影响了链上的用户积极性，关于这部分，在之前 Sui World 的文章也有解释。不过就目前链上应用使用感受来说，使用效果还算顺滑。
