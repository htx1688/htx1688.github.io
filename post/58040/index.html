前言

在本期的《Layer2 report》中，我们深入探讨了Liquid上的重点项目及其团队为2023年制定的计划。

尽管整个行业的发展速度有所放缓，但Liquid在生态系统扩展方面仍取得了显著进展。这主要得益于那些正在发展中的项目逐渐走向成熟。本文将着重介绍其中的一些项目，并更新在Liquid上更为成熟的项目及其在2023年的目标。

1. Blockstream

作为Liquid的技术支持方，Blockstream在2022年基于动态联邦（DynaFed）进行了深入的开发，进一步更新了Peg-out授权密钥（PAK）和Blocksigners。DynaFed为网络带来了显著的升级，使得Liquid的功能性更加灵活，可以更简单地进行添加或移除，从而构建了一个更为稳定的侧链系统。

同时，Elements v22.0.2版本也已推出，此版本确保Liquid的代码与Bitcoin Core 22.0同步，并对DynaFed参数进行了一系列的优化和完善。目前，团队正在对Elements v22.1-rc1进行评估，并计划在下月发布完整的22.1版本（截止目前为止，Elements已更新至22.1.1版本），该版本将全面支持Liquid Testnet。此外，他们还在筹备将其迁移到Bitcoin Core 24.0，预计这一工作将在今年晚些时候完成。

在2022年10月，Blockstream编写并发布了名为“使用Elements的非托管选项”的白皮书。该文深入探讨了新的脚本和操作码，这些新技术能够在Liquid上支持各种金融产品，如非托管限价单和完全抵押的期权合约。团队计划在不久的将来发布这些新期权合约的参考实现。

另一个由Blockstream主导的Liquid项目——Build On L2 (BOL2)。BOL2是一个一站式的社区平台，将为对在Core Lightning和Liquid上构建应用感兴趣的比特币爱好者提供开发者和商业文档、在线黑客马拉松、大师课程和其他活动。BOL2计划在本季度晚些时候启动。

2. Bitcoin Reserve

Bitcoin Reserve是一个仅支持比特币的经纪公司，也是Liquid联盟的成员。去年，它在其平台上增加了一个L-BTC购买选项。客户可以通过银行转账购买最少10欧元到超过1000万欧元的L-BTC，并在他们选择的任何Liquid钱包中进行自我保管。

自从这一消息公布以来，Bitcoin Reserve已经增加了一个L-BTC出售选项，并将其服务扩展到Liquid USDT。

3. Bitmatrix

Bitmatrix是一个基于Liquid构建的AMM协议，于2022年第四季度发布了其主网测试版，支持L-BTC、稳定币和其他资产之间的无需信任的流动性提供和自定义LP费用。

据Bitmatrix首席工程师Burak介绍，2023年上半年进行多次的更新将包括为用户提供更高级的价格和分析图表，展示TVL、费用、收益和其他指标。他还表示，资产对的信息面板也在计划中，其设计将类似于“Liquid资产的CoinMarketCap”。

展望未来，Bitmatrix计划在2024年使用Simplicity优化网络应用，以减少交易大小和成本，并启用多方池。目前，每对资产都需要部署一个单独的池。例如，交易X和Y时，必须有一个XY的直接池。但在多方池设计下，你可以间接地交易X和Y；只需要有X，LBTC和Y，LBTC的池子。

4. DIGTL

DIGTL Markets（DIGTL）为资本市场提供了基础架构，使得发行者可以管理他们的数字证券，并合规地触及全球的零售投资者。

在2022年初加入Liquid联盟之后，DIGTL随后为MERJ Exchange提供了BMN投资者的技术支持，并帮助其在交易所上市和交易，为全球的BMN投资者带来了更为亲民的价格和交易单位。据DIGTL的联合创始人及首席执行官Kyle Fry透露，DIGTL计划进一步拓展其证券交易所的合作网络，并与大型资产发行方合作，其中包括一个价值超过10亿美元的碳抵消项目，该项目将选择Liquid作为其注册平台。

此外，DIGTL还是1transfer的创始成员，这是一个经SEC注册的转让代理，旨在为Liquid上的发行者提供所需的额外监管结构，作为其全面服务平台的一部分。

5.Fuji Money

Fuji Money是一个非托管的、仅支持比特币的借贷平台，允许用户借用稳定币和合成资产，并进行点对点交易。自发布以来，Fuji已向等待名单上的前100名订阅者提供了早期封闭测试的权限，并成功集成了Lightning的潜艇交换功能。

FUJI USD（FUSD）是一种与美元挂钩、由比特币支持的稳定币，将是平台上首个可用的Fuji合成资产。用户可以通过在Liquid智能合约（即约定地址）内存入比特币来借用FUSD，这可以通过Lightning潜艇交换或直接使用L-BTC来完成。

6. Raretoshi

Raretoshi是由Adam Soltys及其团队使用开源的L-NFT协议构建的实体和数字艺术市场，旨在将赞助者与他们喜欢的比特币艺术家联系起来。

在2022年，Raretoshi为平台带来了新的功能，如直接给艺术家打赏、在货币之间切换以及在特定艺术品上留下评论。其中最大的更新是一个加密消息系统，该系统利用基于加密密钥的Nostr协议。

7. SideSwap

SideSwap是一个非托管钱包和去中心化交易场所，允许用户进入和移除Liquid，并在L-BTC、USDt和EURx之间进行即时、无需信任的交换。像证券代币和凭证代币这样的资产，由于不能进行原子交换，可以在去中心化的订单簿上进行交易。SideSwap是新手获得L-BTC并迅速体验Liquid优势的最简单方法之一。

在2022年的常规UI更新中，SideSwap在次级市场上对一定数量的股权进行了代币化。五万股份被预留给首批在其注册代理PEGx上注册的用户，其中一些现在仍然可用。上个季度，SideSwap还默认在其钱包和订单簿中支持离线交换，这是对平台交易的重大改进。离线交换得益于Blockstream工程师Leonardo Comandini推出的新LiquiDEX协议。

SideSwap的创始人Scott Millar对2023年有雄心勃勃的计划，包括添加Fuji的比特币支持的FUSD稳定币和合成资产（目前在测试网上可用）以及集成硬件钱包，如Blockstream Jade，这样用户可以直接从他们的签名设备执行交换。

8. STOKR

STOKR是一个替代资产的投资市场。在欧盟获得许可后，资产管理者和创始人可以通过Liquid在平台上进行证券的首次发行。从商业企业家到资产管理者，STOKR为合规的数字证券发行提供了全套的技术部署、投资结构、支付网关和支持。

到目前为止，STOKR上已有超过6000万美元的资产被代币化。在过去，STOKR为Exordium EXO代币和Blockstream Mining Note (BMN)提供了发行服务，同时对发行者进行审查并为投资者提供尽职调查。

STOKR成为唯一一个被欧盟成员国注册为虚拟资产服务提供商(VASP)的数字证券发行平台。这一荣誉由卢森堡的金融监管机构CSSF授予，使该公司在欧洲获得了坚实的监管基础。

展望2023年，STOKR联合创始人Arnab Naskar表示，公司将继续专注于建设资本市场基础设施，并为中小企业提供筹资和代币化解决方案。

“我们将在2023年在Liquid Network上构建更多尖端的资本市场工具。我们的重点将是为那些在当今市场中面临流动性问题的成熟资产管理者和基金管理者提供解决方案。准备好迎接一个改变游戏规则的一年”

Liquid上即将推出的一项证券代币发行是Angelpreneur，这是一个瑞士实体，投资于初创公司，帮助它们扩大规模并开发成功的特许经营模式。

9. Zaprite

Zaprite是一个提供一系列自托管商家工具的平台，如开发票、会计和记账。目前处于MVP阶段的网络应用程序允许用户连接自己的节点或钱包，以比特币、闪电网络和Liquid方式获得支付。

在2022年，Zaprite正式成为Liquid联盟的成员，并在其开发票系统中增加了Liquid支付方式。进入新的一年，Zaprite的创始人兼首席执行官John解释说，团队很快会迁移其代码库，以便在新版本中将Liquid集成到其Checkouts功能中。

Checkouts是由Zaprite托管的高度可定制、有品牌标识的页面，允许用户接受带有独特流程和信息请求的支付。Checkouts可以连接到用户自己的网站，创建即时的电商解决方案。

Zaprite还将开发新的托管服务，如销售点、产品商店、捐款页面和票务销售。这些都可以连接到现有的Checkouts，为商家提供丰富的比特币和Liquid支付处理体验。
