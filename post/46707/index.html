1 月 24 日，Sui 基金会公布了首轮获得开发者资助计划资助的 12 个项目，每个项目获得 2 万美元到 7.5 万美元不等的奖金 ，总计发放了 415,000 美元。作为资助计划的首轮和试点，Sui 基金会将目光集中在了开发者工具、商业、金融、资产 Token 化等领域，这些项目展示了 Sui 生态系统的创新开源解决方案。

Sui 网络是一个开源的区块链，其设计原则包括水平可扩展性、安全可编程性和低延迟性。Sui 基金会的使命是通过社区建设和教育确保 Sui 网络的成功，其中开发者资助计划是为开发者提供资金，以建设有利于 Sui 生态系统的创新开源项目。

Sui 基金会开发者资助计划申请入口长期开放，如果你正在 Sui 链上做项目开发，可阅读《Sui 开发者资助计划 (Developer Grant Program) 申请指南》。

开发者工具

Dapp Low Code Dev Platform：Dapp 低代码开发平台

Dapp Low Code Dev Platform 将为开发者提供一个简单的 Web IDE，用于编写业务逻辑代码，“一键式”生成 Dapp 应用，并将应用部署到云端测试环境。这个低代码开发平台允许开发者以一种非常接近于开发“传统”应用程序的体验来高效地构建极其复杂的去中心化应用程序。

Dappium：去中心化的无代码平台

使用 Dappium，开发者可以无需任何区块链技术知识，即可将已有的游戏转换为 web3 游戏。Dappium 包括一个 API 和 web2 SDK，用于自动开发、创建和部署智能合约。它能够实现互操作性，提供法币友好选项、混合模型、桥接和单点登录等。

Go Sui SDK：结合多项开发功能的 web3 平台

这个用于 ComingChat 的 Sui Golang SDK 是一个结合了账户系统、支付工具和社交功能的 web3 平台，可以帮助开发者在一个功能齐全的生态系统中构建新的应用程序。Go 开发者可以利用 ComingChat 的 Sui NFT 平台和全链钱包等功能为 Sui 链上的游戏玩家提供游戏道具和 NFT 交易场所。

Movey：Move 语言包管理器

Movey 使 Move 开发者能够发现、验证和协作开发 Move 语言包。Movey 的当前版本集成了 Move CLI，其未来版本将支持链上包的重用和发现。

Remix IDE Plugin：编写 Sui 智能合约的插件

该插件允许开发者使用 Remix IDE 编写 Sui 智能合约。Remix 最初是为以太坊和 EVM 兼容的智能合约开发的，它提供了一个用于构建智能合约的图形界面。扩展 Remix 支持 Sui 生态，这将帮助更多的开发者快速上手 Sui 智能合约，而无需构建单独的开发环境。

Sui Move Analyzer：为 Move 语言提供支持的插件

这个 Visual Studio Code IDE 插件构建在 move-analyzer 之上，为 Sui Move + Move 提供语言支持。它还提供项目模板，与 Sui Move 框架集成，支持 Sui 单元测试，以及支持 Sui Move Prover 验证系统。

商业

Keepsake：NFT 市场生态系统

Keepsake 是一个面向未来的 NFT 市场生态系统，专为游戏开发者、游戏玩家、创作者及其社区打造。Keepsake 市场允许集成游戏 SDK 以实现无缝对接，并可以从游戏中直接列出资产。此外，每个 Keepsake NFT 都可以转换为实物资产，例如收藏品、小雕像和毛绒玩具等。

生产力

AirPay：将链上资产和活动与法币结合

AirPay 服务向 Sui token 持有者发行借记卡，让他们在任何具有 NFC 销售点系统的商店进行无缝支付服务或商品。用户不需要银行或中央兑换账户，因为 AirPay 可以将 token 直接转换为法币。

资产 Token 化

Suia：Sui 链上的第一个 POAP

Suia 是 Sui 链上第一个 POAP（考勤证明）应用程序，其可用于赛事门票、贡献奖章、游戏等领域，为 web3 品牌和用户之间搭建一种全新的沉浸式、游戏化的互动方式。

开发者工具

All.Art：致力于艺术和文化的元宇宙

All.Art 是一个致力于艺术和文化领域的元宇宙，是为艺术家、画廊、博物馆、收藏家、教育机构和公司提供展示的虚拟现实世界。All.Art NFT 的 开源 Unity-Sui 钱包可实现用户和艺术家等进行无缝交易 NFT。

Mamoru：安全监测方案

Mamoru 平台是一种实时检测解决方案，为构建在 Sui 区块链上的应用程序提供自动化的安全智能合约操作。它用于区块链活动的安全和操作监控，能够近乎实时地监控和响应智能合约漏洞利用。它的 alpha 版本提供了一个定义合约监控规则的接口，而 beta 版本将增加对机器学习的支持。

金融

Scallop：高级债券协议

Scallop 是高级债券协议，可在不损失本金的情况下最大化潜在收益率。用户可以在 Scallop 的资产池上存取 token，Scallop 将使用借贷、质押、稳定池流动性挖矿等功能为用户赚取收益。
