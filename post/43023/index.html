原文标题：《 Livepeer CEO：如何构建一款杀手级 Web3 社交平台？ 》

原文作者：Doug Petkanics

原文编译：白泽研究院

自 2016 年以太坊崛起以来，区块链社区一直在寻找能够超越金融用例、对现实世界产生影响的范例。

相对于其 Web2 应用程序，由 Web3 驱动的同类应用程序能否具有颠覆性，跨越鸿沟，被主流采用？到目前为止，像 Cryptokitties 和 Axie Infinity 这样有希望的例子暂时引起了全世界的关注，并激发了大规模的加密运动，但它们本身并没有可持续的持久力来为整个世界提供一个持续的成功故事。

与此同时，社交媒体应用程序在最近几年饱受诟病，这一类别显然已经成熟，可以通过 Web3 进行颠覆，但同样未能成功获得主流采用。与 Reddit 风格类似的社区 Steemit 和 TikTok 风格的应用程序 Pepo 等早期实验性协议并未获得主流关注，而 Lens、Orbis、Farcaster 和 DeSohave 等新兴社交协议的产品成熟度并不是很高，无法激发现实世界的关注，无法让主流用户信服。

到目前为止，我们看到的很多协议都是在试图克隆现有社交网站的功能：Web3 Twitter、Web3 TikTok、Web3 Reddit、Web3 Twitch。它通常是一种近乎克隆但低于标准的体验，不得不让用户每次交互都支付 Gas 费用，并且「模糊地」承诺了更好的创作者货币化或数据所有权。

但主流用户的选择已经表明，他们不可能仅仅因为他们不太关心的数据所有权这个理由来切换到 Web3 平台。除非该应用程序真正提供了更好的、可持续的创作者货币化和分发方式，有大量的优质内容，才能获得主流用户的关注。那些将主流用户与Token激励内容结合在一起的 Web3 应用程序通常逐渐消失，变得无关紧要。

突破性的杀手级社交平台不太可能以「Web3 版本的 X」的形式出现，而是会引入一种范式转换行为，真正原生在由区块链协调的去中心化网络之上。

Web3 快速增长 - 令人惊讶

让我们来看看区块链生态系统中一些具有独特颠覆性的概念：

众筹和资本形成

无需许可访问全球、数字优先的金融服务

数字所有权和可验证性

集体、全球性协调

上述列表可以被简化为 1C0、DeFi、NFT、DAO。

虽然我们都试图指出这些领域的创新可以如何改变和改善整个社会，但我并不认为这是现实世界的延伸，而是 Web3 数字世界的界限。

DeFi 协议在与可组合的链上资产、交易和数字商品进行交互时，要比在为新购房抵押贷款时有效得多。

代表游戏内物品、PFP 或收藏品等数字资产的 NFT，要比代表门票或契约的 NFT 更具吸引力和破坏性，在这种情况下，它们通常会带来比其价值更多的摩擦和法律开销。

Token 协调的 DAO 本身通过聊天室、代码库和在线工具以数字方式将陌生人聚集在一起，其速度远远超过任何公司或非营利组织能够形成、扩展和实现目标的速度。

我接受了上述信号，并认识到，已经出现了一个数字优先、以 Web3 为中心的世界，并通过上述类别中区块链的用例而独立存在。但我认为，当以 Web3 为中心的世界正在以令人难以置信的速度增长时，这会变得特别强大。

我用了 Echo Chambers（回音室）这个词，我认为值得辩论。

六年前，当我全力以赴加入区块链行业时，我最大的恐惧是它纯粹是一个回音室——一群乐观的前瞻性思想家（和一群不道德的机会主义者）彼此谈论他们对数据和金融主权的梦想，以及创作者、用户和平台之间更好的经济协调。即使这种恐惧仍然成立，但看到这个回音室的大小快速增长也是令人难以置信的。看到更多才华横溢的开发者、企业家、创作者、投资者、社会科学家如此深入地被 Web3 的缺陷所俘获，并将他们的大量时间奉献给它，这是我在 Web3 中工作时最鼓舞人心的方面之一。

在 2000 年初期，纽约市曾经举办过互联网周，像 SXSW 这样的会议曾经将整个互联网行业聚集在自己的回音室中。不久之后，互联网触及了人们生活的方方面面，互联网周的想法变得无关紧要，因为没有办法在一个触及几乎所有事物并对每个人都意味着不同含义的行业中聚集一个单一的社区。我们在区块链中看到了类似的情况，其中有许多独特的领域——核心基础设施、扩容解决方案、游戏、NFT、监管、Web3、元宇宙等等——所有这些都依赖于区块链的特性和用例，但在他们自己的领域中蓬勃发展。

那么 Web3 在成为互联网的最新版本之前，回音室必须有多大？

构建杀手级应用的几个机会

一些最好的 Web2 社交应用程序源于「为工具而来，为网络而停留」的方法。Instagram 最初允许用户将过滤器应用到他们的手机照片，并让它们看起来很棒，所以无论你是否有关注者，这都是有用的。你可以录制和分享你唱歌或对口型的片段，并以流行的短音频剪辑，最终在社交网络和推荐算法分层后，成为 TikTok 上的热门。

因为下一个颠覆性的 Web3 社交应用程序将不仅仅是 Web2 应用程序的克隆，并且很难在 Web3 中重新创建现有的社交图谱，所以很有可能，以 Web3 为中心的大型社交应用程序将以成为一个工具来作为开端，解决用户在 Web3 中的一些问题。鉴于 DAO、NFT、DeFi 等领域的成功，围绕社交和内容的一些主题有了明显的新机会。

DAOs 需要一个值得信赖的「安全家园」来组织和分发符合 Web3 价值观的内容。

Web3 社交协议中的内容管理和审核服务缺失，并且没有得到很好的激励。

以 Web3 为中心的高质量内容的发现仍然高度分散。

我前面提到了基于区块链的 DAO 具有实现快速全球协调的力量，它们在没有直接沟通的情况下激励人们采取类似的行动。这些 DAO 通过数字工具进行交流，内容分发通常采用视频的形式——社区会议、核心开发者会议、项目演示、项目概述和评论、 Token 分析和评论等。这些活动通常在传统的社交媒体平台上并没有一个安全的「家」——在内容层面可以被任意删除，甚至更糟的是，账户被封禁。

传统的社交媒体平台拥有令人难以置信的控制权，可以在未经个人创作者同意的情况下随时随意做出改变，这对于个人创作者来说极其不利。围绕无需许可的新金融系统的内容会对现状构成威胁，所以我们需要一个不受 Web2 阻碍的「安全家园」。

仍然有许多项目和 DAO 依靠 Google Meet 和 Zoom 等工具来进行通话和演示，使用 Youtube 记录和存档他们的通话以供回放，并且很少有自己的内容管理网站来在 Youtube 频道之外分发这些内容。

机会：构建 Web3 原生 Zoom

Huddle01 有了一个很好的开端。使用 webRTC 等开放标准来支持会议，使用 Livepeer 流式传输和记录内容，现在有了 Zoom + Meet 的可行替代方案。但是为了让这个 Web3 平台原生，它进行了升级——具有匿名登录钱包的功能、NFT PFP 头像以及围绕谁可以加入和查看会议的 Token 访问控制系统。它真正包含了 Web3 的原生思维，适合作为快速增长的 Web3 项目的回音室。

机会：为 DAO 构建 Web3 原生视频 CMS

许多项目使用会议和直播工具举行电话、演示和会议之后，下一步通常直接将回访上传到 Youtube。然而，这些内容有很大的机会在项目自己的「家园」内组织起来，以便于访问、搜索、评论、讨论、教育和利用。也就是说，Web3 世界缺少的部分是针对 DAO 的视频内容管理系统（CMS）。除了传统的视频 CMS 平台之外，还有哪些机会？

Token 访问控制系统可以根据链上逻辑（如用户钱包中持有的 NFT 或 Token ）确定谁可以访问内容。这比从大型技术公司获得许可并内置于浏览器中的专有 CRM 技术更易于访问。

加密原生支付可以立即为世界各地的用户开放货币化，而不需要通过有限的支付供应商和银行建立复杂的信用关系。

ENS 和其他链上身份支持让用户可以将他们的身份和声誉带到此应用程序之外，并将现有身份特征导入到更多应用程序中 - 而不是使用基于 Facebook 账号的登录。

去中心化的存储、流式传输和播放机制为这些应用程序提供了独立的基础设施，以保证内容的持久性，而不是让内容受制于亚马逊和谷歌等云平台。

这些和更多的机会都是为了满足加密生态系统的独特需求，像 StreamETH 这样的新生工具，已经被用来组织来自以太坊生态系统的会议和存档的内容。

机会：为以 Web3 为中心的内容构建一个策展/管理 DAO

在 Web3 社交应用程序的早期阶段中经常被忽视的步骤之一是内容管理。「去中心化 Youtube」的主页只是简单地显示最近上传的视频，杂乱无章的内容对用户来说并不是一个好的内容体验。

我认为应该存在一个 DAO，根据一个明确的指南来筛选内容。该机制的初步设想是，这个 DAO 由一种 Token 来协调， Token 将分发给那些成功地在有害内容出现之前识别内容、审查内容质量和为内容筛选出标签类型的人，标签包括 DeFi、NFT、 Token 分析等。这将大大提升平台为用户带来地良好体验，并留住新用户。

拼凑一款 Web3 杀手级社交应用程序

被主流用户使用和关注，是一件很难提前完美计划的事情——通常需要结合正确的时机、产品愿景和大量的运气。因此，以下阐述的建议只是我希望看到杀手级突破性应用程序如何出现的想法。

从工具开始。为 DAO 构建一个内容管理系统，并说服他们使用 Huddle01 等应用程序来托管他们的会议记录和分发他们的内容。每个使用它的 DAO 都会获得一个页面，该页面列出了他们所有的历史视频，使它们易于导航、可发现、可搜索，并包含访问控制（或者可能是面向创作者的货币化机制）。这对于单个 DAO 或孤立的项目很有用。

在此工具的用户创建的所有内容中启用一些增值管理。Web3 内容通常默认是公开的——存储在去中心化网络上并广泛分布。使用上述的策展 DAO 开始在整个生态系统中为最高质量的 Web3 信息策展内容主页，包括存档，并根据需要按类别分类——DeFi、NFT、 Token 分析等。这里的模型是一个特定于视频的 reddit——互联网的 Web3 主页。

围绕 Web3 社交网络的身份层——基于钱包、身份、灵魂绑定 NFT 作为用户的声誉。用户可以与跨 DAO 和社区的内容互动，即在多平台之间互动，因此具有可移植性的声誉是围绕社交应用程序构建网络效应的粘性功能。

考虑创作者。创作者选择 Web3 社交媒体应用程序将要有一个令人信服的理由——向目标受众的内容分发比 Web2 社交中无法控制的内容分发更有价值，而不是提及由 Web3 支持的其他货币化和控制优势。

上述建议展示了一条通往以 Web3 为中心的 Youtube/Reddit 的道路，但要真正被主流用户所关注，这将取决于项目在产品机制、用户体验和激励机制上的创新。
