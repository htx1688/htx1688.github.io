如何在 FriendTech 的分叉项目中寻求价值？

作者：IGNAS，DEFI RESEARCH；编译：深潮 TechFlow

当 Friend Tech 刚刚推出时，我分享说 FT 只是将财富从穷人转移到富人的另一种方式，因为最受欢迎的加密货币人物最终从 FT 中获得了最多的收入。

我在很大程度上仍然维持这些观点，因为在 X 上的大量追随者几乎可以保证从交易费中获得一些现金。我只是尝试这些APP就赚了超过 2 千美元。但 FT 让我大吃一惊的是，X 上那些看似默默无闻的人获得了不小的成绩。Levi 就是一个例子，其 Key 价格约为 8,000 美元。

他从友谊证明（比 3，3 更好的话术）开始，购买新用户的廉价 Key 并与他们聊天，以吸引他们也购买他的 Key。他最终成为 FT 上第一个全职内容创作者，建立了 FT 上“最有价值的”群聊。

他的策略包括购买“最有价值”账户的 Key，整合“Alpha”，并与他的 Key 持有者分享。他成为 FT 上所有最有价值账户的代理 Key。这是一个冒险的赌注，因为顶级 Key 很昂贵，所以需要先期资金才能成功。其他影响者和诈骗者有不同的策略：

  • Vombat 正在购买自己的 Key 来提高价格，并在稍后将其卖给他的 Key 持有者。

  • 一些人向用户提供区块链服务的免费访问或潜在的空投。

  • 向一位幸运的 Key 持有者送出交易费收入的抽奖。

  • 为 Key 持有者构建 FT 上的替代 UI 和服务，如 Friendex、FriendMex 等。

我有一个问题想请教你：你真的关心这些吗？你能从 FT 上获得多少 Alpha 信息是 X 或博客上无法轻松获取的？区块链服务的访问对你真的很重要吗？空投 Key 持有者可能很诱人，但很难确定空投的精确投资回报率。

这些用例都旨在合理化你购买 Key，这将为 KOL 产生交易量并赚取费用。

也许是为了与 KOL 聊天？也许是追逐 FT 空投，或者仅仅是 Key 价格上涨的投机。不管原因如何，明确你使用该应用程序的真正动机和目标非常重要。

说实话，我仍在为我的 Key 持有者开发价值主张。目前，我在实时分享投机性机会，我发现这在一个更小的朋友群体中更容易做到。有趣的是，我感到在这些 SocialFi 平台上聊天更轻松，就像在网上与一群朋友聊天一样。

但我如何能进入排行榜前列呢？我的愿望是成为一个 Meme！因为像 Dogecoin 这样的 Meme 币没有任何基本面依附于它。

Friendtech的分叉

Uniswap V2 是 DeFi 中被分叉最多的协议。至今，共有 493 个 Uni 分叉启动。随着最初的热情消退，其中大多数已经默默无闻，但像 Pancakeswap 和 Sushiswap 这样的几个仍然屹立不倒。

这是否也是 FT 分叉的命运？我相信是的。虽然许多分叉会来来去去，但少数设法找到其独特价值主张的分叉会生存下来。

当前 FT 分叉生态系统及其独特功能：

Post Tech

在我看来，Post Tech 是我首个看好的分叉。9 月 21 日，其交易量甚至超过了 FT，PT 仍保持 23%的交易份额。

以下是我喜欢和不喜欢 PT 的地方。

  • UI 更好，更流畅（因为 FT UI 太差了）。

  • 我喜欢对所有 Key 持有者开放的聊天功能，我不会感到自己在聊天时像在自言自语。其他分叉也有。

  • 构建在 Arbitrum 上，比 Base 更适合入金<>出金。

  • 馈送内容算法需要调整，因为我的时间线上只有少数人可见。

  • 这里对持有者的激励更好，因为持有我的 Key 可以使持有者受益（获得部分交易量费用）。

  • 推荐系统激励了分享，以吸引大 KOL 来试用。

Alpha——面向比特币的 DeFi 社区

我喜欢 Alpha，因为我看好比特币 DeFi。

Alpha 是 New Bitcoin City 和比特币社区试图提供 Friend tech 的替代方案。其逻辑是将比特币、Ordinals 和 BRC-20 粉丝与影响者团结在一起。

它确实很酷。有一对一与 Key 持有者聊天的 DM 功能，还有游戏部分！

Alpha 与比特币紧密相连，很像 BRC-20 模型。该系统的基础 Trustless Computer（TC）是一个与比特币集成的“Optimisticrollup”。但 Alpha 本身建立在 NOS 上（类似 L3），区块时间为 2 秒，gas 费为 1 美分。

这听起来很复杂，但有一个易于使用的 ETH<>NOS 跨链桥可以使用。Key 和费用以 TC 代币支付，您可以桥接并在 New Bitcoin DEX 上交易。

Fan Tech

你可能已经注意到每条链都会有 FT 分叉。Mantle 也不例外。

由 Bybit 支持的 Mantle L2 的 DeFi TVL 只有 3700 万美元，所以该链需要一些激动人心的东西来吸引更多用户。Fan Tech 绝对是一个有趣的选择。

与 FT 相比，主要区别如下：

  • 股份生成活动 ：为新账户引入竞价活动，以防止机器人。

  • 流畅的 UI/UX ：包括公开/私人帖子、查看你在 X 上关注的账户、统计页面、深色模式等功能…

  • 推荐系统 ：您从被邀请者进行的股份交易的税收中赚取 1%的收入，以及他们的一份股份。

StarShares

构建在 Avalanche 上，它与最初的 Friend Tech 最相似。至少乍一看是这样。它还有一个“Feed”部分，就像在 Twitter 上，创作者可以与他们的 Key 持有者分享帖子。

然而，在目前阶段，StarShares 的价值主张在于其可以访问 Avalanche 的思想领袖，如 Luigi，我与他进行了有关 Avalanche 终局愿景的 AMA，以及 Avalanche 创始人 Emin。

因此，如果您想向 Avalanche 社区推广某些内容，StarShares 可能是您的最佳选择。

Friendzy

Solana 上的 Friendzy 目前是迄今为止最令人失望的 FT 分叉之一。其绑定曲线很平整，导致当新买家购买 Key 时，价格几乎没有变动。

然而，缺少聊天功能是一个重大缺陷。所以它在 SocialFi 的两个关键特征上都不足：社交互动和金融。

SocialFi 和 FT 分叉的价值

你可能已经注意到我没有广泛根据它们的交易费或绑定曲线的差异来比较这些应用程序（除了 Friendzy，它缺少绑定曲线）。

那是因为我坚信这些应用程序应该优先考虑社交。它们的主要目标应该是吸引对彼此有强烈联系感的特定社区成员。这些社区应该有能力分享财务和社交价值。这一方面对 SocialFi 分叉的成功至关重要。

因此，不奇怪不同链上出现了分叉。然而，您是否与特定的 L1 或 L2 感觉有联系？也许分叉应该更针对目标群体，而不是它们建立的链。这就是为什么在这方面战胜 Friend tech 的先发优势将具有挑战性。

这就是为什么我的 Key 价格在 FT 上的高于其他平台。你可以在分叉上以更低的价格获得 Key，并问我同样的问题。但问题不仅仅是我。这也与整个社区、应用程序的潜力以及空投有关。

我也不是在暗示绑定曲线或交易费百分比无关紧要。它们很重要，否则 Twitter 就足够了。投机是我们大多数人加入这些应用程序的主要动机。但围绕几个交易费百分点差异、推荐计划或代币空投投机进行竞争不会使应用程序成功或失败。

投机作为新人加入和与应用程序互动的初期动机。然而，我相信（或者至少希望）这些应用程序上形成的社交互动和关系最终将是它们长期成功的关键因素。

从短期来看，越来越多的新分叉对 Key/股价产生负面影响，因为美元金额和关注正在被过度分散。

如果人们在 FT 分叉上损失了金钱，这可能导致对 SocialFi 的幻灭。这就是为什么我希望 FT 分叉能鼓励超越投机的有价值社交交往。我想这是一厢情愿的想法。

总的来说，SocialFi 仍处于初期阶段，它吸引了新观众，并在这个熊市中提供了一些令人兴奋的事情可做。就我个人而言，我将继续研究、测试 dApp，并在 Twitter 上以及 FT、PT、Alpha、Fan Tech 等应用程序上分享我的见解。
