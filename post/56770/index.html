原文作者：Thomas Issa

原文编译：深潮 TechFlow

在加密货币领域获取用户现在就像最早的人类在跨越数千年的冰河时代的深处狩猎一样：他们的工具简陋，猎物又正在休眠。早期的人类在苔原上徘徊，对他们的猎物知之甚少，最后只有两种结果：成功捕获或饥饿和冻伤而死亡。

对于消费级加密产品来说，他们开始进行客户获取的故事也是一样的：要么获取用户，要么灭亡。

2021 年和 2022 年标志着初创公司可以将筹款公告与随后的 NFT 或代币空投结合起来，以吸引和留住早期用户的时期；这一策略被许多市场、分析工具、游戏和 Web 3 基础设施成功执行，使它们能够仅基于市场上的代币项目表现来获得早期动力。无论是数字和实体市场 Americana 的 NFT 通行证 在发布后飙升至 0.25 eth，还是曾经前景光明的 NFT 市场 Looksrare，它分配了总代币供应量的 12% ，从 OpenSea 手中吸引客户。两者都在短时间内吸引了消费者的注意力，但从长远来看，其用户基础的却一直动荡。

这两个项目的失误绝不是加密货币领域最大的错误，但这些动态的核心问题在于过度依赖代币和 NFT 发行作为长期客户获取和保留机制，尤其是在宏观环境不佳的情况下，进一步抑制了这些设计的价值提取性质。

为了解决许多消费者获取的缺点，加密货币公司和基金纷纷投资于“忠诚度”最好的产品。项目纷纷涌向“参与即赚钱”平台，其中市场领导者 Galxe、Layer 3 和 Zealy 成为获取、保留和灌输客户“忠诚度”的方法。

这些平台在顶部漏斗增长指标方面很有趣，但它们只是临时解决方案和营销支撑，对协议或项目的长期增长和持续客户获取几乎没有影响。这些平台大多关注社交虚荣指标，鼓励用户在社交媒体上关注或加入社区。奖励机制微不足道（只有少量经济上的好处，几乎没有社交或声誉上的好处）。追求和消费者之间的脱节加剧了无法跟踪未来链上交易的能力，这意味着客户无法计算其潜在客户的生命周期价值。

现在，毫无疑问，我们正朝着一切都被代币化的未来迅速前进。网络参与者本质上是贪婪的，痴迷于金融化，这一动态是由区块链技术催生的。

就像我们代币化一样，我们将“社区化”。这意味着对于每一个人类的共同点，我们都有机会通过程序化资产的兑换在数字原生社区中分享这种共同点。

无论是我们听的音乐、购买的物品还是所就读的学校，我们都可以将这些经历在链上变得不可变，并在在线社区的精神中分享。我们正在看到这一趋势在奢侈品市场上蓬勃发展，因为 NFT 作为实物产品的伴侣将买家更紧密地联系在一起。LV 最近通过他们的灵魂绑定 NFT 发行巩固了这一未来，这将提供实物产品，但更重要的是为志同道合的消费者提供无与伦比的社区接入。将重点从可交易性转移到消费上，从根本上重新定义了社区，创造了与其消费者“密不可分”的独特群体。

正如 Shayon Sengupta 所说的，加密货币是一种本质上社交的技术，我想进一步提炼，加密货币是超级社交的，它是一种部落主义。作为加密货币参与者，我们都是部落，每个部落都源于成员之间的共同点。作为社区，我们听从同伴的警告和建议。通常情况下，我们不会在没有群体的同意或推荐下采取行动。

忽视群体就是忽视加密货币消费行为的精神。

像 Mirror 的 Subscribe to Mint、POAP 的 Attend to Mint 和 IYK 的 Interact to Mint 这样的加密原生用例，使我们更接近通过行动、喜好、厌恶和欲望而形成 NFT 社区。数字时尚初创公司和奢侈时尚已经确立了以兑换为基础的铸造作为他们行业的未来。多亏了像 Medallion 这样的平台，我们将能够加入我们最喜欢的艺术家的社区，或者与我们未来的新星在 Mercury 上收集经验。所有这些都为数字原生社区在我们生活中的普及奠定了重要基础。

随着 Web 3 归因的进步，将有可能提取关于用户购买和认可的产品和品牌、他们访问过的地点、他们阅读的博客，甚至他们支持的体育队的信息。Web 3 允许进行价值捕获，这在当前的分发垄断体系中是不可能的。在新的媒体模型中，用户的价值是根据他们所带来的绝对价值来衡量的，并通过收入分享返利等机制进行奖励。

这些元素与由一种新型模式引领的未来相一致，这个概念被称为“社区即服务”（CaaS）。简单来说，CaaS 可以理解为访问特定群体的价值足够高，可以被视为一种可市场化的产品。如果我们深入了解社区的理念、它们的趋势以及它们在加密货币中的独特作用，我们就能实现一个未来：在这个未来中，提高有机可见性、提供低成本的反馈和洞察、增强品牌可信度以及实现定向广告活动将变得更加廉价。

客户获取的未来围绕着通过加密货币的社区实现货币化，通过简化和加强价值创造为消费者提供动力。在 Web 2 社交消费中，社区入门过程是简单的，并且通常与品牌本身脱节。通过 Web 3 ，消费者体验从简单的“立即购买”转变为更具包容性的“获取访问”，这代表了 CaaS 原则的最基本形式，最强大的分发平台来自这些支持 NFT 的群体。加密货币实现了一种价值捕获机制，使社区能够拥有、控制和追踪他们的影响力。这些动态最终将对品牌、项目和协议产生强大的影响，将 CaaS 动态确立为每个直接面向消费者的垂直领域的客户获取的未来。

这个引人注目的论点是，Web 3.0 和区块链技术将迎来一个增强广告商控制、公平和去中心化的时代。然而，如果没有系统来促进价值的创造和在所有价值贡献者之间公平分配，我们就无法最大化这个基本前提。CaaS 货币化模型是构建社区支持的 NFT 基础设施的第一步，该基础设施已经扩展了该领域。

一旦共识被建立起来，这一趋势将由平台推动，这些平台可以聚合社区、根据链活动推断数据并确定社区需要奖励的增值领域。建设者们将受到社区聚合工具（如 De.ID）、贡献跟踪平台（如 OrangeDAO 的 Grove）或将品牌与 Discord 社区连接起来的高效广告网络（如 Wildfire）的启发。

虽然这个概念是新颖的，并且尚未直接应用于产品或协议设计，但建设者们将会将这些已经支撑了客户获取动态的趋势产品化，我们将会看到社区将他们的社交和技术影响力货币化，这最终将增强客户获取并验证 CaaS 商业案例。
