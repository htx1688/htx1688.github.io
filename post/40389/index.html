编译及整理：比推 Amy Liu

在启动 NFT 项目时，敲定一种基础性智能合约方法是成功的一半，另一半取决于使用哪种 NFT 发行方式。当然，可以在各种不同的组合中配置 NFT 合约和发行方式，也完全有可能将不同的发行方式混合在一起。总而言之，您需要在当代 NFT 生态系统中了解一些基本的发行方式，让我们来看看流行的 NFT 发行方式有哪些。

1. 先到先得 (FCFS) 铸币

一种 NFT 发行方式，以先到先得的方式向公众或 NFT 社区成员开放铸币。

种类：

  • Permissionless——任何人都可以在 FCFS 的基础上进行铸币 |例如：Bored Ape Yacht Club。

  • Token-gated——任何持有指定代币的人都可以在FCFS的基础上铸币|例如：Blitnauts，只能由Blitmap持有者铸造。

  • Raffle-based——需要使用抽奖系统为 FCFS 铸币分配有限数量的许可名单 | 例如：Boki 最终销售。

  • Price-tiered ——集合中的不同供应层有不同的铸币价格，例如NFT ID 0-999 为 0.1 ETH ，NFT ID 1000-1999为0.2 ETH，等等 |例如：The Hashmasks。

优点：

  • 构建者实施起来很简单。

  • 铸币者使用起来很简单。

缺点：

  • 低价、高需求的 FCFS 铸币可能会导致拥堵整个链上的拥堵，例如以太坊。

  • 在铸币快结束时容易出现交易失败。

  • 容易被机器人铸币者淹没。

2. 荷兰式拍卖 (DA)

一种 NFT 发行方式，集合的 NFT 以初始铸币价格开始，例如10 ETH，价格会周期性地以特定的金额下降，例如每 30 分钟下降 0.25 ETH，直到达到指定的价格下限或达到需求平衡，所有 NFT 都销售一空。

例如：

  • Art Blocks——为自 2021 年夏天起，Art Blocks 开始让艺术家能够通过荷兰拍卖推出他们的收藏品。

  • Azuki — 以动漫为主题的 PFP 系列于 2022 年 1 月通过荷兰拍卖推出，但该系列在几分钟内就铸成，其最初的 1 ETH 铸币价格从未下降。

  • Forgotten Runes Warrior’s Guild — Forgotten Runes Wizard’s Club 扩展系列的第一阶段发布开始于荷兰拍卖；最初的铸币价格设定为 2.5 ETH，但在铸币过程中下降至最终指定的 0.6 ETH 静止价格。

优点：

提高NFT投标和价格发现过程的透明度。

缺点：

通常偏爱财力最雄厚的收藏家，他们能够以或接近初始拍卖价的价格铸造。

3. 免费申领/免费铸币

一种NFT发行方式，除了与交易相关的gas成本外，集合的NFT还可以免费铸造。最近，选择这种方式的项目通常依赖高于平均水平的二次销售特许权使用费来带来收入。

例子：

  • CryptoPunks——如今似乎很难相信，但当有史以来最具标志性的 NFT 集合于 2017 年发布时，所有“Punks NFT”都可以免费申领。

  • Loot——Dom Hofman 于 2021 年 8 月免费发行了 Loot，这是一个链上冒险装备系列。该系列中的所有 NFT 在几个小时内就从智能合约中被申领。

  • goblintown — 今年早些时候风靡 NFT 领域的以狂野妖精为主题的 NFT 系列。

优点：

  • 低成本使许多用户都可以使用这些方式。

  • 激励项目继续交付，因为随着时间的推移，持续的进展将带来更多的特许权使用费收入。

缺点：

  • 低费用/零费用 NFT 市场的兴起可能会侵蚀免费铸币项目未来的收入前景。

  • 容易被机器人铸币者淹没。

4. Nounish拍卖

由 Nouns DAO 开创的一种 NFT 发行方式，需要每天无限期地生成和拍卖一个新的 NFT。然而，最近我们看到一波项目采用 Nounish 发行方式，每天发布多个 NFT 等等。

种类：

  • Nouns DAO——“一个Noun,，每天，永远”是主要Nouns项目的口号；底层协议从一组特征中生成头像，拍卖头像，并每天拍卖一次。

  • Lil Nouns DAO——永远每天每 15 分钟拍卖一个 Lil Noun NFT。

  • WizardsDAO——每天拍卖 3 个 Wizard NFT，直到总收藏供应量达到 2,000 个 NFT。

优点 ：

  • 这种发行方式是一种新的链上方式，项目可以使用它来随着时间的推移构建 NFT 社区。

  • 可以通过不同的方式自定义铸币速度和参数。

缺点：

  • 这种方式是试验性的。

  • 高需求可能导致许多人在价格方面无法进行拍卖。

  • Nouns协议设计相当复杂，因此您需要专业的 Solidity 开发人员来构建。

5. 开放版本 (OE)

一种 NFT 发行样式，创建者或项目在其中发布NFT，没有硬性供应上限。

种类：

  • 无限——无限 OE 是一种发行方式，其中集合有无限可能供应并且铸币永远不会关闭，比如“The Room of Infinite Paintings”项目。

  • 定时——定时 OE 是一种发行方式，没有设定供应上限但仅在有限的时间窗口内提供开放铸币。最近的一个例子是 Zora 的“State of Mind”发布，本月早些时候持续了三天，并在此期间筹集了超过 123 ETH。

优点：

  • 对创作者和收藏都很灵活，因为每个想要的人都可以得到一个。

  • 使用现代 web3 创建者工具轻松启动。

6. Bonding Curves

Bonding Curves是一种在过去一年中似乎不再受欢迎的 NFT 发行方式。

这种方式在 NFT 领域的先驱之一是 EulerBeats 项目，该项目按照特定数学公式确定的价格曲线出售（并允许转售）其 NFT。

7. MultiRaffle

MultiRaffle 是一种尚未获得太多关注，但我认为应该得到更多采用的 NFT 发行方式。

简而言之，MultiRaffle 是 Agnihotri 和 Hasu 研究最公平、最有效的 NFT 发行可能是什么样子的结果。因此，在 MultiRaffle 中，这两个人针对公平性进行了优化，并最大限度地提高了铸币的成本效率。

我相信一些 MultiRaffle 实验已经实现，然而自 Agnihotri 和 Hasu 早在 2021 年 10 月发表他们的相关研究帖子以来，我个人还没有看到任何实验。这非常遗憾，因为很明显这种发行方式是平衡和公平的。
