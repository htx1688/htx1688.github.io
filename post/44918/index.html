原文：《How to Double Down on DeFi》by William M. Peaster

编译：DeFi 之道

今年最大的加密货币崩盘事件背后有一个共同的主题。从 3AC 和 Celsius 到 FTX（也许很快就会有 Genesis？）——许多中心化加密实体已经倒下，让用户承担后果。

对于想要拒绝中心化金融并拥抱 DeFi 的消费者来说，这种过渡往往令人望而生畏。因为 DeFi 工具通常比 CeFi 工具更难驾驭和发现，但是这种探索的回报却是巨大的。

本文将通过一个教程回归基础知识，并展示开始使用 DeFi 的最佳场所。

如何在 DeFi 上加倍下注

图片：Logan Craig

在最近由 FTX 驱动的加密货币中心化金融领域的动荡中，以太坊上的顶级 DeFi 项目一直进展得很顺利。

这些经过实战检验的去中心化协议向我们展示了能够亲自管理个人链上资金的力量，而不必担心像 FTX 这样的腐败中介机构。

因此，这个 Bankless 策略将引导你通过使用目前最可靠的 dapps，在 DeFi 上加倍下注来控制你的加密货币的最佳方法。

  • 目标：加倍投资于 DeFi

  • 技能：初学者

  • 工作量：1 小时

  • 投资回报率：自我保护的链上资金管理

如何让 DeFi 成为你的加密基础指令

为什么 DeFi 比以往任何时候都重要？在 FTX 崩盘之后，那些将资金存放在托管交易所的交易员们以惨痛的方式了解到，这些实际上不是他们的密钥，也不再是他们的加密货币。

FTX 的领导人滥用权力，似乎从很多人那里窃取和挥霍了资金，其中许多人都是普通人，只是想管理自己的资金并取得成功。

然而，真正的 DeFi 使权力去中介化。DeFi 让普通人和机构能够管理自己的资金，同时实现自主托管（self-custodying）。例如，在使用 Uniswap 这样的去中心化交易所（DEX）时，只有用户本人可以存入或交易或提取其资金。

在链上管理资金的权力掌握在用户手中。但是，假设你是 DeFi 的新手，或者你已经有一段时间没有接触这个领域了——那么，在这个后 FTX 时代，最值得关注的可靠应用是什么呢?

不用担心，本文将介绍三种最可靠的方法来加倍使用 DeFi。⬇️⬇️⬇️⬇️⬇️

01 Maker + Oasis 借款

Maker 和 Oasis 是 DeFi 中最古老、最成熟的两个应用程序。

Maker 是一种去中心化借贷协议，你可以在其中抵押 ETH 等加密货币，然后用你的抵押品借入 DAI 美元稳定币。Oasis 是一个用于创建 Maker Vault（也称为借入头寸）的领先前端，只需点击几下即可。

因此，假设你长期持有 ETH，并且可能不想出售任何 ETH，但是你也会定期有生活开支，在这种情况下，一些额外的流动性会有很大帮助。

通过 Oasis Borrow 配置 ETH 资金库

在这里，提示 Maker + Oasis 组合，你可以从你的抵押品中提取 DAI 流动性，然后根据需要出售 DAI，并在适合你的任何时间段内偿还贷款。请查看此完整教程，了解入门流程的分步指南。

你将承担的主要责任是什么？通过确保你的资金库在整个过程中保持足够的抵押以避免清算。幸运的是，这种管理非常简单，因为 Oasis 现在提供自动买入 / 卖出和止损保护来管理你的头寸。

02 Aave 用于借贷 ETH

多年来，Bankless 理论认为 ETH 正在成为互联网本身的结算层和无风险利率。

与此类似，正如 Zerion 在上面的推文中提到的那样，Aave 的 ETH 借贷市场有点像 DeFi 的原生「无风险」利率，通常提供约 1% 的 APY 收益率。

对于不熟悉的人来说，Aave 是一种去中心化和非托管的流动性协议。这意味着 Aave 是一个借贷系统，用户可以在其中用抵押品借款或借出加密货币以赚取利息的收益。

也就是说，如果你有一些 ETH，并想要长期持有，但不介意在此基础上赚取一些被动收入，那么在 Aave 上借贷是一个值得考虑的可靠途径。例如，DeFiSafety 团队目前对 Aave 的评分为 94% 。

Aave 上的 ETH 借贷市场

如何操作？你只需前往 app.aave.com ，连接你的钱包，在 ETH 借贷市场按「供应（Supply）」，然后填写你想要的参数并完成存款交易。随着时间的推移，你会从借款人的付款中获得相应的利息！

03 用于交易加密货币的 Uniswap

Uniswap 是迄今为止最成功和经过实战检验的 DEX。

大多数加密货币是以太坊虚拟机（EVM）区块链上的 ERC20 代币，所以你可以使用 Uniswap（目前部署在 Ethereum、Polygon、Arbitrum 和 Optimism 上）来交易几乎所有的加密货币。

因此，如果你想要或需要交易东西，你绝对不需要将资金存放在 CEX 上。将 Uniswap 与你个人控制的钱包一起使用可以很好地满足你大部分（几乎是全部）的交易需求。

Uniswap 的简单交换 UI

至于如何在 Uniswap 上进行实际交易，这很容易。进入 app.uniswap.org ，连接你的钱包，选择你想要交易的两种代币，然后用你的钱包完成交换交易。

如果你在寻找所需的代币时遇到困难，请尝试在 GeckoTerminal 等服务上进行搜索，这可以使按照链和 DEX 搜索代币对变得容易，例如 Arbitrum 上的 Uniswap。

使用 GeckoTerminal 在 Arbitrum 上搜索 Uniswap V3 交易对

额外提示：使用 DeFi Saver 升级

DeFi Saver 是一个先进的 DeFi 管理平台。

还记得 Oasis 如何成为 Maker 的简单前端吗？相比之下，DeFi Saver 好比是目前多个链上的多个顶级 DeFi 项目的专业前端，即 Ethereum、Arbitrum 和 Optimism。

事实上，一旦你学会了使用 Maker、Aave 和 Uniswap 的诀窍——加强你的 DeFi 基础指令游戏的一个好方法是将你的头寸导入 DeFi Saver，这样你就可以拥有一个一站式中心来自动化你越来越高级的 DeFi 需求。

目前该平台支持 Maker、Compound、Aave、Reflexer 和 Liquity 头寸。它还提供一个由 LiFi 驱动的代币桥，一个由 0x 驱动的 DEX UI 以及其他功能。

真正的 DeFi 将继续存在

DeFi 生态系统的发展史还不到五年。虽然它仍然有一些粗糙的补丁和我们需要更多改进的地方，但更多的进步将会出现。

毫无疑问：这种进步将通过、围绕和依靠第一批 DeFi 重量级公司，如 Maker、Aave 和 Uniswap，它们已经过实战检验并获得了目前的地位。今天在这些去中心化的重量级网站上使用和建立业务，或将在未来几年安全地引导你的链上之旅。
