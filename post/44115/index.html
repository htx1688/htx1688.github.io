原文标题：A Brief History of NFT Marketplaces 

原文编译：Leo，BlockBeats

NFT 市场行业是一个竞争激烈的行业。

从开始的只有几个参与者，到现在的已经发展成为一个繁荣的行业，每段时间都有新的入场者参与竞争，争夺同一块利润丰厚的蛋糕。

MOOAR ，一个新兴的 NFT 市场，其母公司为 Find Satoshi Lab (FSL)——Stepn 的开发商。在发布了欢迎的 Web 3 生活方式 app 后，FSL 带动了 move-to-earn 行业发展，并紧接着推出了 DOOAR，后者很快成为 Solana 旗下最大的去中心化交易所 (DEX)。

目前，FSL 团队正在推出自己的 NFT 市场。但是像 MOOAR 这样的新入场者怎样才能在老牌竞争对手林立的市场上站稳脚跟呢？

激烈的竞争

对于跟不上狂热的创新步伐的入局者来说，NFT 市场之间的竞争可能是一种激烈而残酷的过程，NFT 市场上没有所谓的「安全」。

最大的 NFT 市场—以太坊的 OpenSea 占据了交易量的最大份额。但最近几个月，后入场的「潜力股」X2Y2 已经慢慢从 OpenSea 手中抢走交易量，并在最近一个季度超越了 OpenSea。

我们见证了 Solana 生态系统中相似的优势动态循环。

Magic Eden 目前占据着 Solana NFT 交易量的 70 %- 80 %, 而第二大最受欢迎的市场（Yawww）在过去的一个月里仅占 15%，但一时优势并不是持续性的。在 Magic Eden 出现之前，Solanart 拥有 Solana 的 NFT 份额，在 2021 年第三季度，大规模推出 Degenerate Ape Academy 后，8 分钟内售出 1 万只 ape，迅速跻身主流。

正是从那时起，SOL 的最高价在 11 月份飙升至 260 美元。当时在 YouTube 上有相关报道。

注：仍然觉得很有趣的是，Solana，这条渴望以其惊人的吞吐量扰乱华尔街的链，最终在牛市中被称为 nft 的终极链。

不幸的是，对 Solana 来说，这种情况没有持续太久。

Magic Eden 是如何在一个月的时间内击败 Solanart，并在短短几个月内成为 Solana NFT 市场上的领导者的？

X2Y2 是否会在以太坊上重演历史，继续对抗 OpenSea 的主导地位？

STEPN 团队的 MOOAR 能否与 Magic Eden 竞争？

为了解决这些问题，我们需要了解这些过去的里程碑式事件是为什么以及如何发生的。

谦卑初始

许多人都熟悉 NFT 热潮是如何在以太坊区块链上开始的，CryptoKitties、CryptoPunks 和 Bored Ape Yacht Club (BAYC) 推动了这一开端。

Solana NFT 始于 2021 年 3 月左右。这个市场内的一些 og 是 Kreechures 和 solarian。紧随其后的是首个基于生成艺术的 Solana Monkey Business（SMB），当然还有于 2021 年 8 月推出的 Degenerate Ape Academy（DAA），使 Solana 的 nft 成为众人瞩目的焦点，并创造了一种新的「蓝筹」nft。

在早期，Solanart 是 Solana 上的主导市场（这只是一年前的事）。它于 2021 年 6 月推出，很快就成为了该平台首页登入的荣誉象征。

Magic eden 的崛起

随后，Solanart 就成了昨日黄花。一切转变如此之快的原因就是：Magic Eden。一年后的 Solanart 从曾经市场上的王者被推到了只有 Magic Eden 日均交易量的 3%。就像我说的——很残忍。

Magic Eden 是如何做到的?

这个雄心勃勃的团队迅速展开工作。一开始只是一个非常基础的市场，很快就演变成一个非常全面的 NFT 平台，让创作者和交易者都能很容易地发展起来，容易意味着采用。

Magic Eden 还采取了:

-一个创始人参与的 AMA 支持的活跃 Twitter

-Solana 活动中大规模地营销

-持续的新功能和平台更新;

-NFT 项目的内部客户经理。

这些利好因素——加上社区的大力支持——促进了 Magic Eden 的崛起，并在随后占据主导地位，这是一个成功的方案。

实际上，这种飞速崛起并非没有争议。

我们看到了持续出现的技术问题，如每次 mint 后的平台崩溃和臭名昭著的 Balloonsville rug，最终推动 Magic Eden 强化了他们的 launchpad 规则。然后是近期正火的创作者版税战争和该平台最近推出的 MetaShield NFT tool 的崩溃。

两巨兽的空间

就像 X2Y2 在以太坊上一飞冲天，现在几乎达到了 OpenSea 月成交量的 70 %, Solana 是否一山能容二虎？或者说 Solana 对于两者来说太小了？

多个市场将永远存在，人类在熟悉的环境中成长，因此总会有用户为了舒适和方便而最终忠于某些市场，这与为什么你总是去最喜欢的杂货店无异，即使有另一家更便宜的店离你也很近。

我认为，如果两个主导的 NFT 市场价值主张不同，那么他们同时存在似乎是合理的，也就是说，他们都可以提供彼此不同的东西。

例如，OpenSea 仍然是最具流动性的 NFT 市场，这是因为它作为老牌市场的固有知名度。另一方面，我们看到相当多的用户涌向像 X2Y2 这样的市场，因为他们将整个交易体验「游戏化」——从下注原生 token 的能力，到分发交易奖励，再到为没有「建构」的 NFT 项目提供绕过版税的新颖的豁免。

这种预期和功能上的明显差异开拓了 OpenSea 和 X2Y2 甚至 LooksRare 分别占据的多个消费者利基市场。不同的市场都为用户提供了独特的身份和功能。因此，对于一个挑战 Solana 现状的新市场来说，它需要 Magic Eden 填补其稀缺的部分。

谈谈 MOOAR：挑战者从阴影中出现

很多人都熟悉 STEPN，这是一款由 Find Satoshi Lab（FSL）的天才团队开发和扩展的 M2E app。在不到一年的时间里，FSL 已经在 Solana、BNB Chain 和 Ethereum 吸引了超过 100 万用户进入 Web 3。

点击详细介绍了 STEPN 和 FSL。

现在，该团队正在推出他们自己的 nft 市场，MOOAR，但是 MOOAR 的独特卖点是什么呢?

创作者至上的公平版税

这个行业正朝着一个方向发展：公平地补偿创作者的付出可能是可有可无的。MOOAR 拒绝参与这场竞争，并渴望成为一个支持创作者付出的「灯塔」，创造者将能够将自己的版税费用设定在一个合理的程度。

这符合 MOOAR 的重点，即成为希望推出 PFP NFT 系列的首选平台，无论是独立的还是作为其路线图的一部分。PFP 收藏品本质上是艺术，这保证了艺术家的版税。这才公平。

FSL 的目标是促进良性循环，NFT 创作者在 MOOAR 下变得更好时，他们将更愿意创作更好的内容，为鼓励留下来的成员。这建立了一个正反馈循环，将 MOOAR 定位为一个以优质项目闻名的平台，拥有忠诚的用户群和忠诚的创作者群，他们可以依靠被动收入来维持他们的工作。

用户、创作者和平台之间的一种协同忠诚形式，但是，如果不再需要支付版税与其它成员竞争优质内容，成员们真的会愿意支付版税吗？很棒的问题！这激发了下一个独特的卖点。

零服务费订阅模式

MOOAR 如何平衡创作者和用户呢？前者希望自己的作品能得到公平的报酬，而后者可能不想支付任何版税！

我的意思是，我们都喜欢创作者，但时代有点艰难，对吧？MOOAR 为 NFT 市场推出了零费用订阅模式。会员将通过在 MOOAR 上交易 NFT 支付 0% 的服务费来节省资金，而不是每月支付固定的订阅费以使用该平台。这意味着无论你是在进行更便宜的 NFT 的大批量交易还是交换单个 Bored Ape（60+ ETH），你都将节省一笔不小的费用。

就像提供非常低的加密交易费用的交易所或拥有低差价的经纪公司一样，零费用订阅模式为 MOOAR 成员提供了自由，可以随心所欲地交换 NFT。

这也为 FSL 在未来提供额外奖励和有趣的游戏化计划（例如升级奖励系统）打开了大门。甚至已经有关于在未来将 MOOAR 成员整合到所有 FSL 产品的讨论，尽管我预计这还需要更长的时间。

激动人心的东西。

最后，MOOAR 的独特卖点甚至延伸到他们的 launchpad。

社区驱动的「主题风格」launchpad

launchpad 的概念类似于传统金融中的私募股权和风险投资，只是像你我这样的普通人可以参与其中，为小人物实现金融民主化！

这个想法是让 launchpad 的参与者有机会在新项目上市前买入它们——通常是以超低价。点击了解YouTube

同样，许多 NFT 市场提供一个 launchpad 服务，创作者可以在那里推出他们的新 NFT，有趣的是 MOOAR 对 launchpad 概念有着独特的见解。

他们不再等待项目来接近他们，而是在每个「时节」将项目发布限制在一个特定的主题。这可能会持续几个月。

Web 3 团队可以构思符合主题的项目，并将其提交给 MOOAR 团队进行发布。持有 GMT 的 MOOAR 成员可以投票选出他们最喜欢的项目。每个季度由社区领导的获胜者随后都有机会在 MOOAR 的专题 launchpad 上发布，获胜者可以为项目发布前后提供了一整套营销和技术支持。MOOAR 的 litepaper

结语

NFT 市场在不断发展，有力的竞争者到来与 Magic Eden 争夺利润丰厚的市场份额只是时间问题，尤其是在 Solana——NFT 的首选链。

会是 MOOAR 吗？这还有待观察。

我要说的是，建立 STEPN 的 FSL 团队拥有在短时间内建立和扩展开创性创业公司的人才、技术和记录。

除了来自 STEPN 和 DOOAR 的可观的资金支持，再加上 FSL 不断壮大的驱动开发人员、工程师、营销人员和内容作者团队，我认为 MOOAR 有潜力与 Magic Eden 正面交锋。

该团队决心通过利用其一系列独特的卖点 (创作者至上的版税政策、零费用订阅模式、主社区驱动的「主题风格」launchpad) 立即从竞争对手中脱颖而出。

NFT 市场行业即将进入白热化！
