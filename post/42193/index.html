原文标题：《你需要关注的那些 BAYC 钻石手》

撰文：NFTGo

如今，认识并交易 BAYC 的人越来越多。在 NFT 市场逐渐成熟的当下，很多人意识到了 Yuga Labs 和其背后一系列蓝筹收藏品的重要性。但早在一年多之前，BAYC 的公售整整持续了一周之久，直到 Pranksy 为项目发声才得以售罄。从一开始的「门可罗雀」到后面的「门庭若市」，究竟有谁早在那个时候就发掘了 BAYC 的潜力，并在牛熊轮换中一直持有到现在？下面，我们将重点介绍几位黄金钻石手，并分析他们的投资组合。

@machibigbrother 是推特知名的 NFT KOL 之一，并在此前多次发表了 BAYC 相关的观点，以及开展有关 BAYC 的 Twitter Space。

在该地址的持仓中，现估计价值最多的 Collection 也是 BAYC，占据了所持有的 NFT 现总估计价值的 57.15%。在买入时价值饼状图中，MAYC 居于第二，占据了总买入时价值的 20.46%，此外 BAKC 也位于其中，可见该地址是 Yuga Labs 的持仓大户。

另外，从下图可以得知，该地址不仅仅通过 Mint 得到 BAYC，在过去一年中他自己也买了不少 BAYC。

通过这两张图可以看出，除了 Art Blocks 和 KnownOrigin，该地址持有的 More Loot 的现估计价值占比最高，从下图可以看出，该地址的 More Loot 的 HV 为零，并且 Buy Volume 和 Sell Volume 都为零，说明该 NFT 地址的 More Loot NFT 只通过 Mint/transfer 得到，并且一直持有到现在。

该地址买入价值最多的是两个 CryptoPunks，该地址在 Punk 预估可盈利 $321,294.19。

此外，该地址活跃于 Art Blocks 和 KnownOrigin 两个平台，可能是一位艺术家。Art Blocks 是专注于可编程的按需生成内容的平台，KnownOrigin 展示了艺术家、插画家和设计师的稀有数字艺术。

从上图可以得知，该地址持有的 NFT 现总价值排名靠前的几个项目中，除了 More Loot，BAYC 的 HV 也为零，说明该地址获的 BAYC 是通过 Mint 得到，从 Sell volume 这个数据可以看出，该地址卖出过 BAYC，目前还留下一个。

从数据可以看出，该地址在 BAYC 上实现了一个“低买高卖”的成功投资，Buy Volume 远远小于 Sell Volume，综观其所有的投资组合，BAYC 的预估计利润（PnL）是最高的，达到 $1,334,633.33。

而且该地址盈利第二多和第三多的项目都是 BAYC 的子项目。

接着，在该图数据中可以看出，该地址的买入时价值最高的几个项目都是一些知名的蓝筹，除了刚刚讲过的 MAYC 外，其还参与了 Otherdeed、CLONE X、Azuki 等项目，其中 Otherdeed 和 CLONE X 到目前为止还处于亏损状态，高达 $80k。
