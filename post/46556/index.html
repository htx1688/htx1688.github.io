$OP 的日活跃用户（“DAUs”）正在增加，TVL 正在增加，聪明钱正在涌入。 现在是研究 $OP 生态系统和寻找隐藏 Alpha 的时候吗？让我们跳进兔子洞，剖析最新的 L2 发展。

聪明钱流入

过去 7 天，聪明钱流入 $OP 达到 420 万美元，这是 Onchain Wizard 从 ChainEdge 看到的，这可能是为了 $OP 的 Bedrock 升级和 $OP 的最后一周任务的交易。

$OP 交易量和 TVL

  • OP 费用飙升，在过去 7 天中仅次于 BTC 排名第四。

  • 总体 TVL 从 5 亿增加到 5.72 亿，过去两周增加了约 14%。

  • OP 每日交易>Arbitrum。

OP 的价格走势

  • 当前价格 ：1.72 美元

  • 当前市值 ：3.77 亿美元

  • FDV ：75 亿美元

  • 下一次重大解锁，占总供应量的 3.6%，将在 135 天后发生。在掠夺性风险投资进入游戏之前，有足够的时间让 OP 生态系统起飞。

OP 生态系统

随着 $OP 越来越受欢迎，以下是值得关注的前 5 个项目：

  • VelodromeF i: OP 上的 Solidity-Forked AMM，具有第二高的 TVL。

  • Perpetual Protocol ：Perp Dex 拥有超过 15 个交易对和真实收益代币经济学。

  • Lyra Finance ：期权交易平台。

  • Sonne Finance ：OP 上的原生借贷协议，具有最深的 OP 流动性。

  • OPX ：具有真实收益代币经济学的低市值 Perp Dex。

在上述所有项目中，我认为风险与回报最高的项目是 Perpetual Protocol ，原因如下：

  • Perpetual Protocol 为交易者提供 10 倍杠杆。

  • 其票据交换所管理交易和 LP，同时启用杠杆头寸。

  • 通过最新的 V2 交易所，交易在 UniswapV3 虚拟代币池（如 vETH-vUSDC 池）上结算。

  • 当前价格：1.68 美元。

  • 当前市值：4300 万美元。

  • FDV：6800 万美元。

  • CMC/FDV 比率为 0.63。

Perpetual Protocol 指标 ：

  • Perpetual Protocol 占据了$ OP 中发生的大部分交易。随着 OP 交易量的增长，你可以预期它的费用大部分将流入 Perpetual Protocol。

日活跃用户（DAUs）：

  • Perpetual Protocol 的 DAUs 已经超过了 GMX 和 Gains。

  • Perp dex 商业模式取决于交易量，而交易量是由用户数量驱动的。

  • 更高的 DAU → 更高的交易量 → 协议/质押者的更多费用。

  • 交易员对 GMX_IO 收取的高额费用（0.1%的可变借款费用和 0.33%的掉期费用）越来越感到失望。这些费用正在吞噬交易员的利润，如果交易员决定转而使用 Perpetual Protocol 这样更好的产品，GMX 的收入可能会崩溃。

市盈率：

  • Perp 与 Gains Network 的市盈率较低（Perp：7.96 倍，GNS：8.83 倍），而市值只有后者的一半。

  • 考虑到 DAU 的增长，Perp 的上升空间看起来非常有吸引力。

总价值锁定（TVL）：

  • Perp 拥有约 1700 万美元的美元，从 22 年底到 23 年初保持平稳，这表明资本一直保持相对粘性，这是产品与市场契合的一个好迹象。

保险基金/票据交换所：

  • 保险基金稳步增长，通过确保交易者能够获得报酬并且没有坏账来提高协议的弹性。

  • 虽然票据交换所随着整个加密货币市场的下降而下降，但交易量仍然相当稳定。

$PERP 代币经济学

$PERP 代币分配：所有投资者的代币都被发行和归属，团队代币在 30 个月内归属。而大部分则是用于生态系统和对长期持有者的奖励。

作为持有人：

• 质押$PERP 给你vePERP。 一旦保险基金达到其门槛，vePERP 将授予您 15% 的协议收入。

• vePERP 持有人还可以锁定 $PERP 赚取总交易费的 10%。

• 我认为随着早期投资者的退出以及 DAO、团队、质押者和 LPs 一致的激励措施，$PERP 的牛市预期似乎很强大。
