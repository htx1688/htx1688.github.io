原文：Creators Dilemma and NFT Trilemma

美国前总统川普（Donald Trump）于12月16日在自创的社群平台「真实社群」（Truth Social）宣布，推出个人第一款NFT系列「川普数字交易卡」（Trump Digital Trading Card ），1张售价为99美元，总共有45种款式，象征他过去曾是美国第45任总统，全球限量4万5000张，已全数售罄。社会上，对于Trump NFT 的发行褒贬不一。加密支持者，将其视为意识形态资产化的典型代表，开启新的NFT时代。保守派则更多的观察NFT的潜在法律风险。目前，Trump NFT 在OpenSea 仍然保持第一名。本文，从特朗普NFT掀起的热议，再次讨论当前创作者面临的困境和 NFT 三难困境。

自远古以来，知识产权的概念一直是一个长期存在的问题，因为人类天生渴望因其独特的创意而得到应有的承认和奖励，无论是软件程序、音乐、写作还是其他形式的艺术。多年来，专利、版权和商标等法律措施相继出台，旨在使包括应用程序开发者和艺术家在内的创作者能够为其发明获得足够的所有权、认可和报酬。从而限制不道德的活动，例如盗版或假冒原创作品。然而，不可否认的事实是，获得足够的知识产权（如商标、专利或版权）所涉及的过程本来就是繁琐且昂贵的。这些特征也是对即将到来的艺术家或创作者的重大限制。

在区块链技术出现之前，互联网的现状已给数字艺术家和创作者留下了许多弊病。尽管社交媒体等现代互联网平台为艺术家和其他创作者创造了一个数字渠道，可以轻松展示他们的作品，但是，在大多数情况下，价值会回馈给平台，而不是创作者。

大多数 Web3 平台的特点仍是中心化和内容审查，这限制了创作者的创造力和隐私保护。此外，没有系统的方法来验证发布艺术等数字内容的真实性，这使得其他人很容易复制（盗版），创作者极难得到应有的信用和重新列名。

因此，这些挑战强化了不可替代代币 (NFT) 的实际重要性。

NFT 重要性

NFT 被描述为一种独特的数字资产，代表现实世界中的对象，例如艺术、音乐、视频、游戏内物品或房地产，其所有权可在区块链上验证。通常，各种形式的资产，从存在于虚拟世界中的数字商品，到对房地产等实物资产的债权，都可以表示为 NFT。

众所周知，人才在全球范围内均匀分布，然而，机会却高度集中。因此，区块链技术为全球的人才，如艺术家，提供了一个民主化的途径，以获得可行的经济基础设施和机会，有效地向全球观众展示他们的艺术作品。核心基础设施，特别是Jpg Store等NFT市场，使创作者和消费者能够以去中心化的方式自由交换价值，而不会失去他们的隐私或主权。

例如，尼日利亚一位年轻艺术家将他的作品以数百万美元的价格卖给全球的感兴趣者，获得应有的认可和持续性的版税，即使不必在曼哈顿举办精美的艺术展览或拍卖晚会。

在社会方面，专制政权下银行账户被政府冻结的政治家可以直接发行 NFT 作为筹集竞选捐款的一种方式。这样就可以在没有中央垄断的情况下向公民提供真正的民主红利。同样，贫穷的客户可以将资金集中在一起，以获得专有资产的部分所有权，例如独特的艺术品，甚至以 NFT 表示的房地产。从而以去中心化、无边界和无许可的方式使创建和获取独特数字资产甚至实物资产的过程民主化。

NFT创建过程

如前所述，NFT是区块链上的一个独特的数据单位，可以直接与各种资产挂钩。从而提供一个不可改变的所有权证明。虽然人们认为创建NFT的过程可能很繁琐或技术性很强。然而，在实际意义上，它是相当简单的。

下面概述了标准的 NFT 创建过程：

个人必须首先选择他们想要转换成 NFT 的独特资产。

此后，资产必须在区块链上铸造。在这种情况下，铸造只是指将数字文件转换为 NFT 集合，其记录存储在区块链上。因此，NFT 代表区块链上的数字资产，所有权证明可以通过区块链的不可变记录轻松验证。有多种区块链可用于铸造 NFT，其中包括以太坊、Cardano 和 Solana 等。

Bluemove 和 OpenSea 等 NFT 市场可以使创作者的NFT分别在Cardano或Ethereum区块链上被铸造后，列出和交易。通常情况下，上市销售NFT需要使用标准的数字钱包，这些钱包与NFT被铸造的底层区块链兼容。例如，为了在Bluemove等Aptos NFT市场上列出NFT，创建者必须建立一个与Aptos兼容的数字钱包，如Fletcher Wallet，同时拥有一些加密货币，如ADA，用于交易费用。

NFTs 的应用

NFTs 的概念成为近年来最大的技术突破之一，这主要是因为 NFTs 的实际应用、用例和潜力是无穷无尽的，新的应用不断涌现，以为创作者和消费者创造更多价值。尽管如此，NFT 仍然是处于初始阶段的细微创新。下面概述了 NFT 目前的一些应用和用途。

  • PFP和Avatars：这代表了NFTs最流行的用途之一，它通常被用作一种独特的识别形式，而在某些情况下，由于NFTs的稀有性或独特的特征，它被视为社会身份的象征。此外，人们将 Crypto Coven 等 NFT 用作他们的 PFP，以此作为识别 NFT 项目所代表的文化或事业的一种方式。

  • 音乐 NFT：在这一点上，许多人都意识到音乐行业在为艺术家提供可持续的工作谋生手段方面效率低下。大多数现有的流媒体平台都是自相残杀的，因为它们占据了音乐艺术家收入的很大一部分，在某些情况下甚至会审查将要发布的内容类型。为了应对这一挑战，开发了音乐 NFT 的概念，作为音乐音频、视频或整张专辑的代币化版本。因此，艺术家可以利用区块链技术绕过唱片公司或流媒体平台等各种中间商。从而有独特的机会直接与粉丝联系并赚钱，并仍保留作品创意所有权。

  • 游戏内 NFT：大多数基于区块链的即玩即赚 (P2E) 游戏使玩家能够赚取和拥有游戏内资产，例如元宇宙中的武器、皮肤、数字配件、角色和虚拟土地，这些本质上都是 NFT 可以进行交易以获得经济回报。

  • 收藏品：正如 NBA Top Shot 等项目所证明的，最近体育交易卡的繁荣推动了 NFT 收藏品的兴起，这些收藏品可能具有不同程度的稀有性和持有者的价值，就像实物收藏品一样。

  • 数字证书：NFT 也可以用作真实性证书，作为分配给个人或团体的所有权证明。由于 NFT 是以不可变、无需许可和透明的方式存储在区块链上的数据单元，因此很容易识别区块链上作为 NFT 的学位证书或房地产所有权证书的真伪。从而限制证书欺诈或伪造的情况。此外，诸如 Mirror 之类的平台使作家和研究人员能够将他们的博客/文章铸造成 NFT 并赚取版税。

  • DAO 会员通行证：去中心化自治组织 (DAO) 和现代分配中各种形式的社会团体倾向于依靠 NFT 为其社区内的成员提供独家会员访问和特权。这与拥有 AAA 卡的精英人士或 AARP 会员的会员卡为他们提供独家航班或酒店使用权非常相似。例如，拥有 Bored Apes 或 Mutants Apes 的 BAYC 社区的成员除了获得免费的 NFT 收藏品空投

批评和解决方案

尽管有很多有前景的使用案例，但NFTs仍然被认为是一个泡沫。这与技术和整个NFT生态系统相关的一些基本缺陷和挑战相关。

  • 区块链可扩展性限制：NFT 用户正在呈指数级增长，尤其是在以太坊等主流区块链上，结果产生了可扩展性限制，包括吞吐量较低、响应时间较慢和铸造NFT 的高Gas费。这些问题往往会影响许多人，尤其是 NFT 生态系统中的散户投资者。

然而，已经出现了一些创新的解决方案，其中包括Layer2扩展解决方案，例如 Optimism 或 Arbitrum，以及替代的Layer2区块链—具有高吞吐量和低交易费用的 Cardano。

  • 互操作性差：大多数在特定区块链上铸造的 NFT 无法在其他区块链网络的 NFT 市场上访问、交换或交易。缺乏互操作性使得 NFT 的创建者很难有效地跨多个区块链触达大众用户，这有时会导致 NFT分叉。尽管如此，OpenSea 和 Magic Eden 等 NFT 市场已经取得了一些明显的进展，分别为 Solana 和以太坊网络提供跨链支持。

  • 创造者入职挑战：任何NFT项目的成功都取决于除艺术美学之外的无数因素。一些关键的成功因素包括社区的热情，网络效应和用户的激励结构。因此，文化水平不高或知名度有限的高素质艺术家，也往往会在NFT的推广方面遇到挑战。为此，Bluemove 等创作者至上的 NFT 市场提供了 Launchpad 等整体服务，以托管、推广和营销平台上推出的 NFT 项目。

  • Rug Pulls 和诈骗：最后，在 NFT 生态系统中，诈骗、rug pulls 和剽窃艺术的铸造率越来越高。因此，需要实施增强的验证系统变的至关重要，例如 Jpg Store 验证。
