该行业将超越简单可替代的“代币投票”，此风险在于治理通证的巨鲸持有者占据主导地位。因此，这一领域的创新将大放异彩。

更多的项目将开始重新考虑其通证的设计理念，不仅纳入同质化/可转让代币。关键问题是，这能否允许更灵活的治理结构设计？加入DAOrayaki社区一起探索这个话题。

治理代币

治理代币是代表区块链项目中投票权的加密货币。这有助于通过代币将权力和权限分配给该项目的用户。

治理代币持有者可以影响项目的决策: 比如提出或决定新的特性建议，甚至更改治理系统本身。

治理代币允许用户控制、真正的地方分权和民主化。

同质化的治理代币

对于大多数项目，治理代币是可替代的，这意味着1个 Gov 代币= 任何其他1个 Gov 代币。它们具有“可互换性”，意思是它们是可整除的，不是唯一的。代币的每个部分等于代币的任何其他部分。

定义代币治理的标准方法是声明每个单独的可同质化治理代币等于协议中的所有其他可同质化治理代币。如果持有者拥有更多可同质化的治理代币，他们在该协议中有更大的投票权，因此在未来的决策中有更大的发言权。

这在很大程度上是一个“扁平”治理结构，对于大多数项目都很有效。通过积累代币，恶意参与者仍然可以执行类似于51% 攻击的活动。有了足够的财力，鲸鱼可以单枪匹马地制定和批准决策。这种情况不妙。

DAO 使用的投票选项类型有很多细微的差别，可以在我们往期的文章中查看。短时间内风靡一时的是后来的一些创新，比如通过“选民托管”(Voter Escrowed’)或 veTokenomics 进行的象征性投票锁定，但即便如此，也有其自身的缺点。

我们将趋向于在生态系统中同时使用可同质化和非同质化代币的组织。因此，在治理结构中尝试使用非同质化代币会怎么样呢？

非同质化治理代币(gNFTs)

非同质化代币(NFTs)是唯一的，1 NFT 不等于任何其他 NFT (这里不包括 ERC-1155s)。这种独特性为您提供了很大的灵活性，并允许您对治理设计进行大量的创造性。

由于 gNFTs 确实是独一无二的，你甚至可以为你的 gNFTs 分配特征或特定属性的权利ーー例如，一个更独特/罕见的 NFT 比一个更常见/较少见的 NFT 在生态系统中获得更大的管理权力。每个非同质化的治理代币可以具有不同级别的治理权力。

在理想的情况下，这可能有助于为早期采用者/强大的增值成员提供公平的竞争环境，这些成员拥有超强的协议能力，而投资者的购买力较高。

资本不再是稀缺资源，有意义的贡献才是。我们能否使用非同质化治理代币来更平等地分配治理权力？也许可以用一种更周到的方式来分发？我们看到更多关于 gNFTs 的实验，一个来自带电粒子生态系统的很酷的例子是Memphis Mural4。Memphis Mural 是一个巨大的50 x 60英尺的墙壁壁画，它被改造成了一个包含10万美元的 Aave USDC，由带电粒子提供动力，利息通过捐赠区块提供给圣犹大儿童研究医院。作为一种无损的慈善机制，这确实是一种不断给予的礼物。参观者将能够要求壁画出席证明协议 NFT 或 POAP。这些 POAPs 作为一个投票机制，决定应该在哪里捐赠累积的利息，使之成为一个社区民主捐赠机制。

如果 gNFTs 是可转移的，那么您必须考虑有人在积累 gNFTs。然后，这个参与者将对协议的治理产生巨大的影响，这与您使用可互换治理代币遇到的障碍完全相同。

每个设计决策都有利有弊，创新和迭代是让我们得到最有效的代币治理模型的唯一途径。

NFT的融合与FT治理结构

我们相信，未来的治理框架将创造性地、深思熟虑地包含可同质化和非同质化代币。

由于 APE 空投，每个 BAYC NFT 都收到了10,094美元 APE 代币(发布时大于15万美元)。所以接下来，生态系统中的每个 BAYC NFT 持有者现在都有一个可替换代币的袋子。

作为社区成员，每个可替换 $APE 都有一定的价值，每个 BAYC NFT 都有一定的价值ーー无论是在金钱上还是对持有者个人(例如身份)。

对于Yuga 生态系统，难道Yuga实验室不想鼓励持有和积累 BAYC 和 $APE 吗？有价值的 BAYC NFT，价值的 $APE 令牌… 。同时拥有 NFT 和 APE 的价值。

通过使用 Snapshot 等现有工具， Yuga生态系统(或人猿基金会)已经开始推出治理投票ーー1 $APE = 1 Vote。使用快照，您可以很容易地允许1 BAYC NFT = 1 Vote，正如许多其他 NFT 项目已经允许的那样。

这种治理投票结构允许 $APE 同质化代币持有者以及 BAYC 非同质化代币持有者在组织作出的治理决策中拥有发言权。

也许在项目的早期，这些非同质化代币的重量随着更多同质化代币的释放而减少？这里有很多可能性。

使用可替换和非同质化代币设计治理结构

以 NFTs 作为基础层重新构想治理结构设计怎么样？我可以想象，随着 NFTs 项目开始向其 NFTs 持有者发行同质化代币，NFT 项目会开始扪心自问。

重新构想治理结构，鼓励持有 BOTH 代币，因为它在组织内提供了更大的影响力。

传统上，持有者会把同质化(APE)和非同质化(BAYC NFT)代币分别放在钱包里。如果它们能合二为一呢？如果您的非同质化代币可以作为一个篮子来持有你的同质化代币？

设想 NFT 拥有自己的智能钱包来存放额外的代币ーー可同质化或不可同质化。继续使用Yuga作为例子，这将意味着您的 BAYC NFT 将持有您的 $APE 可同质化代币。

我们相信，通过结合 $APE 和 BAYC，可以实现更加动态的治理设计。

通过将 $APE 嵌套在 BAYC 中，也许这意味着你的投票权重增加了？也许通过时间锁定您的 $APE 在您的 BAYC 为6个月(强制 HODL) ，这可以增加您的投票权重甚至更多。

这就是它可能的样子:

  1. BAYC — gNFT. 1 BAYC = x Votes

  2. 1 BAYC = x 票

  3. $APE — Fungible Governance token. 1 $APE = 1 Vote

  4. $APE ー可互换治理令牌.1 $APE = 1票

  5. 1 BAYC + y $APE = z votes

  6. 1 BAYC + y $APE = z 票

  7. 1 BAYC + locked y $APE = z1 votes

  8. 1 BAYC + locky $APE = z1票

这激励了 BAYC + $APE 的积累，同时也锁定了你的 $APE… … 如果你想在Yuga生态系统中增加话语权的话。

此外，通过将你的 $APE 嵌套在 BAYC 中，它就像一个大篮子或一捆，你可以很容易地将整个捆绑包转移到一个新的钱包或一起出售整个篮子-液体包装到极致。

这些锁定的 $APE 令牌是“锁定在一个完全可转让的”BAYC，但它把特定的 BAYC 船只变成了一个预测市场，什么折扣应适用于 BAYC + 锁定 $APE 相对于 BAYC + 解锁 $APE 或只是 BAYC？

可以设想出哪些治理设计来支持这个框架？

我们可能会看到 NFT 持有者获得的新权利激增，NFT 持有者也可能获得更多可替代的象征性权利。我们非常高兴地看到标记化治理设计继续发展，将两者结合在一起。

分发

代币化治理正在发展，使用 gNFTs (可转移或不可转移)只是可以使用的另一种方法。还有很多其他的，还有很多会涌现出来。我也特别兴奋，因为有连锁的声誉包括作为一个投票乘数像 Karma7正在做的工作。

目前，基于代币的治理大多是富豪统治和鲸鱼操纵，但新的治理模式将会出现，在这种模式中，贡献的价值将被适当地置于经济实力之上。

Charged Particles8 允许用户将 ERC-20、 ERC-721和 ERC-1155代币存储到他们现有的 ERC-721 NFT 中。一个稀缺的 NFT 现在可以转换成一个装有代币的篮子。

我们是 NFTs 的铁杆粉丝，多年来一直在鼓吹“ NFTs: 不仅仅是艺术”。我们很高兴继续看到创造性，以及可同质化和不可同质化治理代币的混合，允许在治理设计中实现最大的创造性和灵活性。
