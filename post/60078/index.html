由全球領先的數字資產交易平臺 HKD.com、瑞士 世界可持續數字轉型協會WA.SDT 和 香港數字資產慈善基金會 聯合主辦《達沃斯數字化轉型高峰論壇 X 世界 WEB 3 慈善週》，將於 2023 年 9 月 27 至 29 日 於 香港會議展覽中心展覽廳 5G 舉行，並舉行開幕典禮。

高峰會受到瑞士達沃斯政府，瑞士駐港領事館及香港瑞士商會支持，將匯聚政界代表、商家、專才、投資者等，以增進瑞士香港在金融科技領域的交流合作，拓展數字金融科技業界的發展。希望將香港

打造為數字金融科技的全新港口，為全球企業的數位化轉型，構建一個全新的戰略橋頭堡，並搭建傳統金融與新興金融之間的橋樑，以發掘 WEB 3.0 發展的黃金機遇。

HKD.com 創辦人兼行政總裁 楊凱文先生 表示︰「全球 Web 3.0 行業飛躍式發展，今次活動的融合，更凸顯 Web 3.0 在推動社會影響方面的重要性。

去中心化金融（DeFi）運動使大眾能夠平等地獲得金融服務，提供一個解決全球經濟差距的潛在方案。去中心化自治組織（DAO）正在重新塑造社區和組織內的決策方式，使我們能夠更好地分配資源用於慈善事業。

同時，我們對香港政府表達最深切的感謝，感謝他們在這次活動籌辦過程中給予支持。政府一直以來致力於促進創新和技術進步，為香港成為 Web 3.0 行業蓬勃發展的樞紐奠定了基礎。」

Web 3.0 產業最近面臨挑戰，他表示監管明確性和投資者保護仍然是重要的關注點。相信未來通過產業利益相關者與政府之間的持續合作，將會在創新和監管之間取得合適的平衡，確保一個安全和繁榮的生態系統。

作為瑞士方代表的 世界可持續數字轉型協會創辦人及主席 Mr. Josef Mondl 表示︰「今次引領瑞士各大企業及專才參與高峰論壇，獲取瑞士政商界的支持，希望為香港及瑞士兩地企業提供密切的交流機會，以促進及提升兩地的經貿發展。

Web 3.0 不僅僅是一場金融科技革命，它是一場人類創新和進步的運動。在未來，個人對數據和數字資產將擁有更多的控制權，透明度和信任是線上互動的基石，科技與慈善相結合，就是今次「世界 Web 3 慈善周」所展示的。」

本屆高峰論壇圍繞數字化轉型、金融科技等重大主題，呈現一場引領未來的思想盛宴，為全球數字 東省金創區塊鏈研究院理事 朱嘉明先生、Member of the Parliament of the Canton Grisons Mrs. Valerie Favre Accola、Academia Raetica Director Mrs. Barabara Haller-Rupf 等。

為期一連三日的高峰論壇將，講題將圍繞著AI、WEB3前沿技術、ESG、金融基礎設施革新、監管政策等。現場將邀請來自香港、瑞士及達沃斯市的政商界領袖、全球多個領域的權威代表、國際文化演藝界巨星參與支持。

預計將吸引過萬位投資者、商家、專才，以致媒體和慈善家等，共同探索全球數字化轉型新機遇。期望通過論壇的影響力，並促進香港經貿發展，以加強香港在數字化領域的國際樞紐地位。

化轉型描繪前瞻藍圖。同時聚焦多個高度應用科技轉型領域的專家，將以多維視角，從研究、政策、技術等多角度，共同揭示數字化轉型未來前景，引領著全球金融科技的發展新方向，並探討如何將科技力量與社會責任相結合，推動全球可持續發展進程。

活動更得到多位來自世界各地的業內人士到場支持，包括中國國際經濟合作發展局局長 張炳忠局長、駐香港瑞士領事館領事 Mr. Daniel Frerihofer、廣東省委橫琴工委委員、省政府橫琴辦副主任 葉真先生、Global Head, Blockchain, Crypto and Web3 at Egon Zehnder Mr. Benjamin Sigg、廣東省金創區塊鏈研究院理事 朱嘉明先生、Member of the Parliament of the Canton Grisons Mrs. Valerie Favre Accola、Academia Raetica Director Mrs. Barabara Haller-Rupf 等。
