从上图可以看出来游戏被排除在外，因为游戏是图中 IP 的集合体。在这张生态图中，我们着重介绍了采用 Yuga Labs IP 的品牌、创作者、项目和艺术家。它们增强了 IP 的社交属性、效用、娱乐价值，从而吸引更多的受众加入 Yuga 的 IP 生态。

一、Yuga Labs IP 生态总览

1、时尚品牌

（1）Gucci

首先是 Gucci，它与 Wagmi-san 合作推出 Gucci Grail，BAYC、BAKC 可以在10KTFshop 这里穿上由 Alessandro Michele 设计的奢华服装。

（2）TAG Heuer

TAG Heuer 让 BAYC 收藏者可以在 Taghe Uerconnected Calibre E4 上使用 BAYC 风格。手表可以通过 $APE 购买。

（3）蒂芙尼

蒂芙尼以每个 50,000 美元的价格售罄了定制的 Cryptopunk ‘NFTiff’ 吊坠。 「NFTiffs」 是一个可以用来铸造和兑换定制设计的 CryptoPunks 吊坠和匹配的 NFT 艺术品的电子通行证。

（4）adidas Originals

Adidas Originals 将与 BAYC 和 Punks 一起进入元宇宙，adidas 处于 Into the Metaverse 计划的第 2 阶段和第 3 阶段。在第 3 阶段，持有者将 「in the metaverse. In creation… 」。第一阶段实物商品已交付。

（5）Li-Ning

中国顶级运动品牌 Li-Ning 正在进军 NFT 市场，期待它加速 Web3 的发展。它举办了关于飞盘、机车等运动的一系列活动，旨在带动更多人一起参与新的运动。

2、音乐

（1）Timbaland

Timbaland 推出基于 Bored Ape NFT 的音乐 ApeSh!t，发行在了 ApeInProd，创作者是The Zoo。

（2）Marshall Mathers

D2 LBC 专辑由两个 BAYC 持有者 SnoopDogg 和 Marshall Mathers 创作。这是 Marshall Mathers 的第二大热门合辑，在 Billboard 200 中排名第 6，一共售出了 43,000 份。

（3）Snoop Dogg

Snoop Dogg 的 BAYC 正在不断支持着他在食品和音乐领域不点探索新的可能性。

（4）Ape Rave Club

Ape Rave Club 的使命是将新一代艺术家带入电子音乐界。他们致力于支持舞蹈音乐和 NFT 领域的才华横溢的创作者人和新创意。他们也很高兴能在 web3 中塑造音乐的未来。以下是来自 Blockparty Korea 的图片：

（5）0x0 Metaverse

0x0 Metaverse 是一家提供全方位服务的音乐品牌商、制作公司和管理公司，由一群热爱音乐的未来主义者经营。他们的 BAYC 有最好的舞步。他们的曲目是也我最喜欢的曲目之一。

3、玩具和雕塑

（1）SUPERPLASTIC

SUPERPLASTIC 是一个生产优质艺术玩具的企业。名为「SuperBored」的塑料艺术玩具将 BAYC 宇宙中使用的漫画主角作为原型。

（2）IsmToys

18 岁的 Ricky da Luz 是 IsmToys 的创始人，他从去年开始为 NFT 持有者制作免费的 Bored Ape 玩具，销售额达到了 100 万美元。

（3）Polly Kole

Polly Kole 是一名 NFT 雕塑家。她为 BAYC 社区创作了许多令人惊叹的雕塑作品。其中之一是 Dr. Bombay 和 SnoopDogg 的 BAYC。

（4）Sindrax Tech

Harry Lu 是儿童互动解决方案初创公司 Sindrax Tech 的创始人，他们正在儿童产品上应用 BAYC 和 MAYC IP。他正在为我们年轻的下一代培养 NFT 和 Web 3.0 文化。

4、食物

（1）Bored & Hungry

FoodFightersU 是一个 NFT 餐厅集团，他们旗下的 Bored & Hungry 将 Web3 连接到现实生活。Kevin 和 Andy 将他们对 NFT 的热情与食物相结合。 Champ Medici 和 SnoopDogg 是他们的合作伙伴。

（2）gmgn supply co.

MAGGI XU 创立了 gmgn supply co.，它是首家采用 NFT 品牌谷物的去中心化 CPG 公司。他们的愿景是打造 100 个社区研发的 CPG 品牌。他们的麦片产品即将推出。

（3）Bored Sauce

Bored Sauce 由 Bobby Bellhop 创立，他在「猿宇宙」的大部分时间里一直在为知名酒店客人提包。在他的一位常客给了慷慨的小费后，他停止了那份工作，并开始将他对香蕉和香料的热爱结合起来！

5、饮料

（1）Bored Breakfast Club

Bored Breakfast Club 是同行业内首创的 NFT 项目，它将为为会员提供现实世界的咖啡运输和独家数字体验。

（2）BoredCoffee.eth

Bored Coffee 的团队成员是 NFT 和咖啡爱好者，他们认识到该行业需要长期的社会和生态可持续性。他们的目标是提供优质咖啡，同时让产品的生产更加透明和可持续。

（3）Ape Beverages

ApeBeverages 是 Web 3.0 的第一个可持续水饮料。在元宇宙中为你解渴。这个专门的 Ape 水饮料是可以以随时购买的，水源地是加利福尼亚州沙斯塔山。该产品 2022 年 9 月发布。

（4）BoredWineCo

BoredWine 正在采购一些世界上最好的葡萄酒。他们会将它们装瓶，并用世界上最独特的猿类风格包装它们。

（5）Cafe X 

CafeX 和 Vera 合作为旧金山机场旅客提供全新的咖啡体验——机器人咖啡吧，在 SFO 1 号和 3 号航站楼可以使用。

（6）Happy Dad Hard Seltzer

Happydad Happy Dad 由一群狂热的朋友在 2021 年创立的饮料项目，他们想制作出味道最好的硬苏打水。它具有低碳酸，清爽，简单可口的味道。 BAYC 创始人最近品尝了几款。

6、实物商品

（1）Rolling Stone

滚石特别款收藏版杂志收录了前所未有的 BAYC 艺术品，这些独家设计来自于它的合作者 Claire Salvo、TimpersHD、Dario de Siena、Crypto Geisha 和 Jeneva 等创作者和艺术家。

（2）BoardPusher Skateboards

BAYC #7676 设计了 BoardPusher.。这是提供给 BAYC 订单的限量版优惠。滑板采用了专业品质的加拿大枫木甲板。

（3）Adam Bomb Squad

The Hundreds 的 Adam Bomb Squad 很久以前就与 BAYC 合作推出限量版服装。

（4）round21

2021 年 7 月，round21 空投了 BAYC x round21 官方合作版实体篮球——仅供 NFT 所有者使用。这款实体篮球由高级复合皮革制作而成，装饰有定制的 BAYC 艺术品，是一款特别限量版收藏品。

（5）richclassdecor.eth

Rich Class Decor 制作了可以定制的 NFT 地毯。 Rich Class Decor 旨在打造复古、波西米亚风、不拘一格和现代风格的地板覆盖物和装饰的最佳选择。

（6）FallingInSand.eth

James Sun 的 falling in sand 正在通过沙子实现数字世界和现实世界的联系。他希望用沙子让最具标志性的时刻永垂不朽。

7、知识库

（1）10KTF 

Wagmi-san 为 Yuga Labs 的 PFP 制作了无数的个性化商品； 10ktf 由 @wenewlabs 重新定义了 NFT 体验，收藏 NFT 意味着新的连接和新的故事。

（2）Mutant Cartel

Mutant Cartel 通过讲故事将 MAYC 资产变成具有艺术、动画和沉浸式讲故事的有影响力的主角。故事和课程将继续在公爵日记的每周摘录中展开。

（3）Jenkins The Valet

Jenkins The Valet 梦想着让 Metaverse 通过角色，IP 变成更美好的世界。他们是全球顶尖的开发者、设计师、品牌建设者和编剧团队，他们将在更多场景应用 BAYC、MAYC 和 BAKC 的 IP。

（4）Roda

Ridaz 是冲劲十足的 Otherside 创作者。从生动的 Koda 到关于无聊钓鱼日的小故事，Rida 和他的团队已准备好在元宇宙中创造引人入胜的内容。

BoredClubCanada 是加拿大 BAYC/MAYC/BAKC 所有者协会。该协会拥有超过 150 多名无聊猿的成员。我在多伦多区块链会议上遇到了一些加拿大无聊猿持有者，比如 bayc9617 和 FUDmaster，他们超级友好！

FrenchApeYC 是一群讲法语的 BAYC 所有者组织。它拥有 100 多名成员。他们有一个协调 IRL 事件的私人 Discord 群聊。

（3）韩国

BAYCKorea 是说韩语的 BAYC 和 MAYC 持有者的社区。他们在 8 月 11 日在 NFT Community of Seoul 会场相聚。

（4）台湾

Bayc Taiwan DAO 是一个拥有台湾拥有最多 BAYC 持有者的社区，成员中有众多投资者和收藏家。SQLPANDA 和 Conan 经常参加。我有点了解了去年在俱乐部的 Conan，他原来是魔术师和气球艺术家。

二、如何创造更大的 IP 价值

以上是对品牌、项目、创作者和艺术家高光时刻的深入探讨。接下来我们将讨论生态系统参与者如何携手创造更大的价值。

从 cdixon.eth 的文章《Why Decentralization Matters》中我们可以知道，在 Web 2 中，网络参与者之间的关系从正和变为零和。

而在 Web 3 中，这种零和的关系被解构了。通过 10KTFshop 和 Otherside 的第一次合作中我们也能看出，我们正在用故事来共同开发媒体的未来。通过上述这些生态的不断合作，我们将携手释放所有权经济的价值。

Gucci 选择了成为 10KTF shop 合作伙伴之一。如果我们没有购买 Gucci Grail，我们就不会体验到 Gucci 在 Yuga Lab 元宇宙中的新体验。在 Yuga 的 IP 生态地图中，我们每个人都在共同开发故事，共同塑造文化。

故事是我们所知道的最容易吸引人的形式。当一个好故事被讲述时，人们的注意力将一直被吸引。人们还可以通过故事过得新的社交和娱乐价值。

知识、实体商品、时尚品牌、游戏、娱乐参与者都在推动品牌和 IP 价值的复合增长方面发挥着重要作用。我们相信他们会不断促进所有权经济中的网络增长和并且让能讲故事的资产的实现更大的价值创造。
