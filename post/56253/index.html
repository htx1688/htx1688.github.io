当我们谈论比特币的发展时，首先想到的就是中本聪，他在2018年发布了比特币白皮书：《比特币：一种点对点电子现金系统》，比特币协议由此问世。

与大多数软件实现一样，定期维护和升级比特币协议至关重要。由于比特币不受单一实体控制，因此开发和维护必须在网络参与者之间达成共识。本质上，没有任何个人或公司来管理比特币项目的开发，而是由全球各地的个人或团队发起。

比特币资源和生态规模不断发展，各种各样的组织也不断涌现，它们为比特币协议的发展贡献了力量，这也是比特币从一个小众开源软件到数百万人可以依赖的重要数字基础设施的健康成熟的原因。

本文，我们就梳理了一些资助开源比特币协议和生态的主要组织和个人。

Bitcoin Core：比特币身后的核心贡献团队

比特币已诞生15年有余，比特币之父早已在互联网上销声匿迹，但一直有团队在维护比特币技术的更新。但其中最出名的当属Bitcoin Core团队。

Bitcoin Core一个软件客户端，参与者使用它在比特币网络上创建节点、存储硬币、进行交易和执行其他操作。将其视为决定用户如何与比特币区块链交互的接口或网关。Bitcoin Core 的功能类似于 Google Chrome 为用户提供连接和浏览互联网所需的功能。

正如除了 Chrome 之外还有多种互联网浏览器一样，除了 Bitcoin Core 之外还有多个比特币软件客户端。然而，这些其他客户端都没有像比特币核心实现那样广泛使用或有影响力，Bitcoin Core 仍然被认为是最能保留中本聪所倡导的意识形态的比特币软件客户端。此外，Bitcoin Core 继续受益于遍布全球的大型开发团队的努力。因此，它经常为其他比特币软件客户端设定标准。 

Bitcoin Core 开发团队规模庞大，因此追踪所有贡献者有些困难。请注意，成为比特币开发人员没有特殊要求。任何人都可以加入、离开或选择他们希望专注的发展领域。开发者甚至不必披露自己的身份，就有资格为比特币核心的开发做出贡献。

Bitcoin core的贡献

Bitcoin Core 作为比特币协议的原始和主要软件实现，它为其他客户端设定了标准并保证网络的持续运行。

  • 去中心化：通过运行比特币核心，任何人都可以操作完整节点，为网络的去中心化做出贡献。这种权力分配可以防止任何单一实体控制比特币。

  • 安全性：Bitcoin Core严格的交易验证和完整的区块链记录有助于网络的安全。任何更改过去交易的尝试都会被完整节点检测到，从而防止欺诈性更改。

  • 开发和维护：比特币核心的开源性质允许世界各地的开发人员不断改进。这种集体努力确保了比特币能够适应新的挑战和不断变化的环境。2017年8月，Bitcoin core主导的隔离见证方案激活。这一方案增加了比特币的区块容量，也解决了交易的可扩展性，其中一个方案就是拟提闪电网络。

Bitcoin Core 不仅仅是软件，它是比特币网络背后的驱动力。它确保系统的运行，维护其去中心化结构，并确保区块链免受欺诈活动的影响。

专注于支持比特币生态系统的机构

在最初几年，比特币的发展主要由中本聪和其他一些志愿者来维护和支持，随着比特币逐步的普及并形成市场，开发者可以获得比特币，因此也慢慢吸引了外部开发者。

比特币基金会（Bitcoin Foundation）

比特币基金会由比特币核心开发者Gavin Andresen和Jon Matonis于2012年创建，作为非营利组织，基金会主要资助一些开发者。该组织旨在促进比特币的发展和推广，并代表比特币社区与政府、企业等各方面进行沟通协调，其使命是为全球用户标准化、保护和促进比特币密码货币的使用。基金会致力于推动比特币技术的发展与创新，在比特币生态中扮演了重要的角色。

比特币基金会最主要的资金来源之一就是会员费，然而到了2015年初，基金会就几乎耗尽了其财政储备。尽管如此，基金会在推动比特币技术的发展、促进比特币在全球范围内的普及与应用上面做出了巨大的贡献。    

Chaincode Labs

2014年，Alex Morcos和Suhas Daftuar创建了Chaincode Labs，这是一家总部位于纽约的数字货币研究组织，为比特币网络的进步做出了贡献，致力于支持比特币的重要非营利组织。Chaincode专注于比特币软件的开发、审核、测试和维护，除了其永久团队中的比特币开发人员之外，该平台还向独立的比特币开发人员提供资金。该组织资助的一些著名比特币核心开发人员包括 Pieter Wuille、Alex Morcos、Suhas Daftuar、John Newbury、Russ Yanofsky、Marco Falke 和 Carl Dong。

麻省理工学院数字货币计划（MIT DCI）

麻省理工学院的数字货币计划是另一个加密货币开发研究组织，它为外部开发人员及其自己的专门的比特币核心开发团队提供资助。然而，与大多数比特币开发赞助商不同的是，麻省理工学院 DCI 依靠捐款来筹集资金。与该组织相关的比特币核心开发人员包括Wladimir van der Laan和 Cory Field。自 2015 年 4 月以来，在 DCI 支持下工作的开发者承担了 14% 的 Bitcoin Core 代码。2016 年，他们还宣布推出 90 万美元的比特币开发基金，吸引了几家大型交易所进行捐款。

Blockstream

Blockstream 是一家由风险投资支持的区块链技术公司，由Bitcoin Core开发人员在2014年创立，包括 Gregory Maxwell、Jorge Timón、Matt Corallo、Pieter Wuille 和 Mark Friedenbach。在成立初期，Blockstream 就获得 2100 万美元 A 轮融资。

作为一家公司，其主要目标是为比特币建立一条允许互操作交易的侧链，加强比特币生态系统的扩展。Blockstream对比特币做出了许多重大贡献，从大量的改进协议（BIP）和对隔离见证的研究，到保密交易、密码代码库以及闪电网络等等。目前其产品主要包括闪电网络、侧链、多签钱包等。

比特币资助机构的新发展

除了前面提到的例子，随着资金多样性的显著改善，许多其他组织在直接或间接地支持比特币的开发和维护，加密货币交易所以及其他一些企业和组织开始支持开源比特币开发商。

图片来源：BitMex research

BitMEX 是一个通过向开发者提供资源为 Bitcoin Core 的发展做出贡献的交易所。该计划于2019年7月启动，是第一个为比特币开发者提供赠款的交易平台。目前已经提供170万总资金（包括向麻省理工学院DCI提供的50万美元），为高度透明的项目提供开源无附加条件的资助合同。目前BitMEX  Grants团队成员有五位，包括Gleb Naumenko、Sjors Provoots、Rene Pickhardt、Chris Coverdale、Calvin Kim。

Square Crypto是Square创建的一个独立研究团队，旨在改进比特币的开源软件。根据团队的使命，Square Crypto 资助了几位致力于提议和实施比特币核心升级的开发人员。Square crypto 赞助的 Bitcoin Core 开发人员包括John Atack、Vasil Dimo  和Gloria Zhu。另一方面，Matt Carlo是 Square Crypto 团队的永久成员。

Brink成立于2020年，由 John Newbery and Mike Schmidt共同创立，是一家致力于改进比特币协议的非营利组织，专注于比特币协议的开发，旨在通过基础研究和开发来加强比特币协议和网络，并通过资金、教育和指导来支持比特币开发者社区。其资金100% 来自希望支持开源比特币网络和协议的个人和组织的捐款。目前已发布两轮资助。今年6月，推特前CEO Jack Dorsey及其公共基金Start Small承诺向Brink捐赠500万美元。Brink将在未来5年每年收到100万美元资金。

Opensats是专注于比特币等项目的非营利组织，旨在资助与比特币相关的免费和开源项目以及相关的教育和研究计划。OpenSats提供一个不收取任何费用且通过我们平台向项目和普通基金捐款的100％转账的平台，致力于为比特币以及这些外围技术提供资金，包括开发新的比特币和比特币相关软件、改进现有的免费和开源软件、有关比特币的教育和宣传、研究比特币相关主题等。今年5月，OpenSats宣布从Jack Dorsey的慈善计划#startsmall收到1000万美元的捐款，将用于支持以比特币、去中心化社交协议Nostr和相关技术为重点的免费开源软件和项目开发。

小结

自比特币诞生以来，各类机构、公司和组织都在或多或少地为比特币协议的发展做出技术或资金上的贡献。随着比特币市值日益增长，基于比特币这一公链的生态也将逐步扩展，尤其是闪电网络、侧链等比特币二层方向的应用，未来，各开发者、公司和组织也将围绕这一类基础设施开展更广阔的业务。
