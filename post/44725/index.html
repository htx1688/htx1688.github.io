在 NFT 这样一个不稳定和新兴的行业中，要赋予项目为 “蓝筹 " 地位是很难的。 然而，不少的 NFT 项目宣称自己是蓝筹项目，但它们是吗？

Footprint Analytics 使用自己的蓝筹股指数来衡量 NFT 行业的实力。 本文将解释我们如何在 NFT 的背景下定义蓝筹股，以及这种分类如何帮助分析师。

什么是 “蓝筹 NFT”？

在股票市场中，蓝筹股是指长期稳定增长的公司股票、大型传统工业股以及金融股。 该股票也被视为产生稳定回报的长期投资，通常被认为相对安全，Berkshire Hathaway 等公司就是这种情况。

Footprint Analytics -  BlueChip Index Since 2022

如上图的蓝筹股指数是通过跟踪 20 多个知名的、财务稳定的公开交易非金融公司的市值份额指数。它参考标准普尔 500 指数的算法，计算每天有 [X] 个蓝筹 NFT 的所有值（美元）之和，然后在除以 [X] 个 NFT 算出。

哪些因素决定了蓝筹 NFT 项目？

说起蓝筹 NFT 项目，首先想到的是 CryptoPunks 和 BAYC，它们被大家认为是行业佼佼者。主要是它们具备有以下特征：

  • 背后有强大的团队和财政的支持

  • 常常出现在 NFT 销售排行榜上

  • 得到市场的认可，并且在 NFT 领域中形成了受欢迎的品牌

  • 对比 Terraforms、Meebits 等项目的交易，CryptoPunks 和 BAYC 几乎没有清洗交易

因此，无法从单一指标判断该项目是否为蓝筹 NFT。 为了确定一个项目是否应该在 Blue Chip Index 中，Footprint Analytics 会考虑其他 4 个因素：

设计风格

设计质量对于一个项目的长期发展至关重要。例如，CryptoPunks 作为 OG NFT，曾因其朋克像素风格而受到追捧，使像素艺术风格流行起来，许多衍生品复制了它们，从而使像素艺术项目在这个领域中脱颖而出。因此，创新的设计艺术风格会使得该项目从一些较为平淡的 NFT 中突出亮眼，成为流行的趋势。

效用

持有 NFT 有哪些特权，对用户有多大用处？ 它会增加价值吗？ 这些都是用户会考虑的问题。持续效用是通过成为社区的一部分并持有 NFT 后期带来的赋能权益，例如项目方提供白名单以铸造其他合作项目、向持有者空投 NFT 等，这不仅增加了 NFT 的寿命还提高了项目的价值。持有 BAYC 者可以得到一只 BAKC 的空投奖励。

交易量和 Floor Price 指标

交易量是验证 Floor Price 的依据，不是由持有者任意挂单价格，而是通过市场的交易而定价。它总是会遵循市场的波动而上升或下降，比如 Floor Price 上升，交易量就减少，如果 Floor Price 下降，交易量就会增加。因此，Floor Price 也成为最直观的蓝筹指标，买入的项目 Floor Price 越高，意味着它的价值也越高。

社区

社区也在考量 NFT 项目是否是蓝筹 NFT 方面发挥着重要作用。大部分蓝筹 NFT 项目都有自己的专属社区。

蓝筹 NFT：3 个示例

一般来说，在充满投机和波动的 NFT 市场中，它被认为是一种高风险的投资。蓝筹 NFT 的风险相对较小——但任何事情都有可能发生。 大多数 NFT 未能建立品牌，而蓝筹 NFT 恰恰做到了这一点。

Bored Ape Yacht Club

BAYC 包含了 10,000 只 Bored Apes 且赋予持有者权益，拥有一只就可以进入一个独家的在线俱乐部。即使它推出的时间比 CryptoPunks 要晚，但它在一年多的时间内超越 CryptoPunks。

重主要的创新在于，BAYC 将头像图像的商业权利赋予持有者和社区，从而将 PFP NFT 推向主流的趋势。 底价高达 73.15 ETH（截至 11 月 10 日），是其他 NFT 底价的数倍。 它已成为名人的新宠，可以说是 NFT 市场上最大的品牌。 它还发布了包括 MAYC 和 BAKC 在内的成功 NFT 系列。

CryptoPunks

CryptoPunks 是首批成功的 NFT 项目之一。 CryptoPunks 于 2017 年推出，拥有 10,000 个独特的数字角色，其中一些 NFT 甚至被佳士得和苏富比拍卖。 它为后来 NFT 开发的 ERC-721 标准提供了灵感和基础。 该项目现已被 Yuga Labs 收购。

Azuki

Azuki 是一个由社区打造的，去中心化的元宇宙品牌，其日系漫画风和极高的美学观赏性深受人们喜爱，品牌塑造能力强，第一批 NFT 仅短短几分钟便售罄。Azuki 有不得不提的重大创新，是新的铸造标准、白名单筛选机制、和出售方式。

  • Azuki 开发了一种新算法，采用了批量铸造的方式，支付一次 gas fee 可以 mint 多个 NFT，对于用户来说它大大降低了成本。

  • 它舍弃了随机抽取白名单的方法，结合各方面标准来完成白名单筛选。

  • 出售方式是 Azuki 分三个阶段出售：荷兰拍卖（降价拍卖）、白名单铸造和公开销售，以便为社区成员提供更多利益。

总而言之，对于想要了解市场的用户来说，蓝筹 NFT 是一个很好的起点。可以总结这些项目的特点，并分析相关指标，以比较新兴的 NFT 项目，它们已经达到了什么程度，与蓝筹 NFT 项目有多大的差距。
