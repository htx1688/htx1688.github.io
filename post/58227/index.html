Friend.tech 是近期最被热议的去中心化社交应用，在仅限邀请加入的情况下，上线两天就狂揽 50 万美金，在 8 月 11 日当天就创下了单日突破 4000 ETH 交易量的记录，以及 26 万笔链上交易，引发一阵热潮。 巅峰时期，Friend.tech 的每日收入高达第二位，超过了 Lido、Uniswap 等去中心化交易所的领军者，甚至超过了比特币等 Layer1。

然而时隔不久，断崖式的大幅降温也迅速到来 。迄今，根据 DeFiLlama 的数据，该应用程序的每日收入已下降了 95%以上，活跃用户数量也大幅下降，根据 8 月 30 日 Friend.tech 官方推特公布的数据显示，DAU 已从 42k 减半到 21k，短暂炒作的声音很快消散。

Friend.tech 究竟如何运作的？ 作为一个全新的建立在 Coinbase 刚推出的 L2 网络 Base 上的 SocialFi 应用，Friend.tech 需要绑定用户的 Twitter 账号方能使用。用户可以通过 Base 链的以太坊购买和出售任意 Friend.tech 上用户的「股份（Share）」。KOL 股份的投资者，将获取与 KOL 直接对话的权利。它允许用户买卖相关 Twitter 账户的股份，拥有某 KOL 的股份将授权用户进入该 KOL 的私人聊天室。最重要的是，这些「股份」是支持交易的，整个过程就变成了一个价格随时上下波动的投资游戏。

Friend.Tech 的本质是对 KOL 们的影响力进行投注 ，也就是一个 KOL 代币化的项目。相当于 Friend.tech 替 KOL 以个人 IP 发币，将 KOL 的价值货币化，是将社交价值进行量化并交易的一种尝试。虽然热度居高，很多加密 KOL 也开始获利，但一些人对该应用的定价模式以及数据隐私的问题表示担忧。

在推出后，一些观点认为 Friend.tech 是文字版的 Onlyfans ，甚至将取代以 LaserCat 社区为代表的 Alpha 社区模式 ，因为用 ETH 购买群股份与用 ETH 购买 NFT Pass 卡在一定程度上非常相似。有分析指出，NFT Alpha 社区的需求是真实存在的，Friend.tech 发展的方向不是吸引分散的用户，而是吸引有长期纽带关系的用户，因此从这个层面上来说，Friend.tech 吸引 Discord 社区用户可能在质量上比直接从 Twitter 上获客要高。

Friend.tech 昙花一现，为 SocialFi 赛道以及 Web3 社交带来了短暂辉煌后破灭的希望。 为了进一步了解 SocialFi 以及 Web3 社交赛道的情况，本文将从以下三个方面展开分析：

  • Web3 社交的核心特点是什么？

  • Web3 社交的重要协议和项目盘点

  • Web3 社交产品能够取代 Web2 传统社交应用吗？

Web3 社交：

所有权与收益回归个体用户

在 7 月末，知名加密货币投资机构 Paradigm 发布最新十大关注的加密领域，「加密原生社交应用」 位列其中。它代表了一种 Web3 时代社交媒体的全新范式，融合社交媒体和去中心化技术 ，利用区块链为用户提供了独特的 Web3 社交体验。

「Web3 社交」是对社交相关的 Web3 协议、产品等的总称，他们运行在公链上，在数据存储和运营管理上比 Web2 社交产品具有更明显的去中心化特征。 而在 Web3 社交场景中结合「X to earn」增长激励模型的 SocialFi ，主要指向社交金融化特征明显、侧重代币机制设计 的项目，出现了诸如 Friend.tech 这样短期火爆的应用。

SocialFi 即 Social+Finance，其本质是通过将用户的社交影响力通证化，形成一个自洽的经济系统 ，以帮助更多的、拥有不同等级的社交影响力的人获得与其影响力相对应的收益，从而破除传统社交平台巨头对于用户创造内容的垄断，以及 KOL 对于普通用户的垄断。本文将以「Web3 社交」指称包含 SocialFi、去中心化社交等概念在内的 Web3 社交赛道。

整体而言，Web3 社交是一个尚处于萌芽期的赛道。与传统 Web2 社交产品不同的是，Web3 社交产品在「是否上链」以及「所有权经济的归属」上存在很大的区别，由此也产生了激励机制的差异。

链上交互与抗中心化审查

=======================

在传统的社交产品中，一切社交行为都在链下进行，社交平台之间是一种相对封闭和固定的关系，当用户在某一社交平台发布讯息时，其他平台的用户很难跨平台交互。

而 Web3 的社交方式可能会从根本上发生转变——用户的行为地点从链下转为链上，一切都变得开放透明，社交平台的不同将不再成为交互的壁垒。 当社交关系成为公共设施，用户就可以带着沉淀的社交关系体验任何新的产品，而不会被任何一个产品所捆绑。

同时，隐私和抗审查也是 Web3 社交的主要优势。与传统平台不同，Web3 平台不集中数据存储，降低了单点故障和未经授权访问的风险 。区块链技术提供的假名允许用户在保护隐私的同时维护其在线身份，并采用去中心化的审核机制 ，让社区成员得以参与内容决策，最大限度地降低中心化审查的风险。

用户拥有账户的所有权

======================

在传统的社交平台中，用户的一切行为都要基于一个高度中心化的平台才能进行。而这也意味着用户的一切互动都在平台的掌控之中。而 Web3 社交则致力于夺回用户的对自己社交资源的掌控力 ，所有权的确立 是极为重要的方式。

在 Web3 社交平台上，用户拥有内容分发的所有权 ，可以决定如何分发内容给关注你的用户，也不会被锁定在某个特定平台，无需使用特定身份登陆，用户便可以自由选择服务和算法最好的平台，从而争取自己的利益，真正实现 Web3 时代下掌控自己网络 的愿景。

多样化的激励措施

====================

通过社交代币实现获利与激励 ，则是 SocialFi 的主要特性之一。

SocialFi 类似于 GameFi，将应用层设计与 DeFi 相结合 ，通过引入社交代币和应用内实用代币的概念，实现了社交媒体体验的多样化。在 SocialFi 的生态系统下，社交代币主要负责控制平台经济，并允许创作者管理自己的社交和内容，而用户的影响力又将决定代币的价值。但这些代币不受平台本身的控制，创作者可以决定他们的代币将如何被关注者使用。这改变了传统社交应用中，平台利用用户的注意力来投放广告，公司赚到了钱，用户什么也得不到的现状，而是将社交数据也变成资产 。

Web3 社交协议与应用盘点

Web3 社交赛道至今未有一个堪称成功破圈的龙头项目跑出来，但不乏一些优秀的协议和项目做出了有效的探索。研究机构 Messari 总结了目前 Web3 社交赛道的市场全景，详见下图。

随着越来越多的用户逃离 Reddit 和 Twitter 等传统的 Web2 社交媒体，Web3 社交协议和应用产品在未来的市场也将越来越大。下面将介绍 Web3 社交赛道常见的协议类型与重要应用：

去中心化社交协议

====================

去中心化社交协议（Decentralized Social Protocols）是一种可以让开发者自由建立社交平台和服务的协议。 它拥有更高的抗审查性和隐私性，提供了可以让开发者快速建立高度自由的社交项目的基础设施，而且拥有更强大的数据安全性和可靠性。

  • Lens Protocol

Lens Protocol 目前是 Web3 社交赛道的头部协议 。Lens Protocol 是于去年由 Aave 官方发布的一个基于 Polygon 的可组合的去中心化社交图谱 ，也是加密原生最知名的社交协议之一。Lens 旨在使创作者能够拥有自己与社区的联系，形成一个完全可组合的、由用户自己享有的社区网络。

不同的是，用户在 Lens Protocol 上产生的一切行为皆以 NFT 的形式存于链上，支持用户拥有其所创作的所有内容。该协议从一开始就考虑到了模块化 ，允许添加新功能和修复，同时确保用户拥有的内容和社交关系不可改变。

在 Lens 生态中，Lenster 是重要的标杆社交应用之一 。Lenster 是一个基于 Lens 协议构建的去中心化社交媒体应用，也是 Lens 的黑客松 LFGrow 的 12 个胜出项目之一。

用户用于登陆 Lenster 的钱包里的 NFT 和解析的 ENS 域名可以展示在个人 Profiles 中 ，其他功能类似于 Twitter。在 Lenster 上发布帖子可以选择可访问对象（关注的人、被关注的人、朋友的朋友以及所有人），也可以根据需要开放是否可被其他用户收藏的选项，还可以对帖子收藏者设置收藏费用。Lenster 也上线了内容分销功能 ，如果有用户通过你的转发去付费收藏某条帖子，你也将获得分成，目前默认值为 15%。

  • Farcaster

Farcaster 同样是去中心化应用程序的协议，用户可在应用程序之间自由地迁移社交身份，开发人员可在该网络上自由地构建具有新功能的应用程序 。在 Farcaster 上，用户可以发送简短的文字信息广播，连接以太坊地址。

在 Farcaster 上，人人都可以构建客户端应用程序，来广播消息。Farcaster 协议分为两个重要部分：链上注册表 ，供用户领取唯一用户名；及链下主机 ，供用户存储社交数据。

  • CyberConnect

CyberConnect 是一种基于区块链技术的去中心化社交图谱协议，通过构建一个统一的社交图谱，用户可以将自己的身份信息、社交关系等迁移到不同的网络和平台，同时这些个人数据也完全归用户所有 ，这将为用户创造巨大的价值。

CyberConnect 协议主要由 3 个核心部分组成：

Cyber 个人资料 ——作为一种去中心化的身份标准，Cyber 个人资料储存用户在 Web3 中的所有活动资料。所有链上的凭据和资产都可以映射到用户的 Cyber 个人资料中，实现一个统一的身份标识；

CyberConnect 社交图谱 ——将用户的身份、个人数据、内容和社交关系等相互连接在一起，构建起一个统一的社交网络。这使得用户的社交关系可以跨平台，实现无缝的互通和转移；

Cyber 钱包 ——作为协议的智能合约钱包，CyberWallet 能够安全地储存用户的资产。这为用户提供了一个可信赖的存储和管理数字资产的方式。通过这些功能，CyberConnect 为用户提供了一个更加开放、自由的社交网络体验。

在经济模型上，CyberConnect 已经完成了其协议代币的公售 ，其可以用于支付 CyberProfile 铸造时付费以及 gas、治理。

  • Nostr

Nostr（Notes and Other Stuff Transmitted by Relays） 是于 2020 年启动的一个简洁、开源的去中心化社交网络协议，本质是去中心化社交场景的信息传输协议 ，Damus、Amethyst、snort.social 是根据该协议开发的第三方应用。基于客户端和中继端 Relay 的设计使 Nostr 具有高度的互操作性，相对轻量级、简单但可扩展 ，为大规模应用提供了可能。

在 Nostr 协议架构中，每个人都可以运行一个客户端程序 ，要发布什么信息时，用自己的私钥对文本签名，发送到多个中继器上；想要获取信息时，则向各个中继器问询。客户端再对得到的信息，借助其公钥验证从而判断真实性。

  • DeSo

DeSo 是一个开源的 Layer1 区块链协议 ，旨在扩展社交 DApps 。DeSo 允许用户创建社交资料和社交代币。这些代币可以从创作者铸造的 NFT 中赚取利润。利润是由每个用户与创作者铸造的 NFT 交互产生的。

著名社交 DApp Bitclout 就使用了该协议，Bitclout 是一个由其用户控制的社交平台。除此之外，社交 DApp Diamond 也使用了该协议，Diamond 允许粉丝和创作者之间进行互动，从而建立社区。

Web3 社交应用

==================

  • DeBank

在 DeFi 领域中，DeBank 已成为了投资者们常用的钱包追踪平台 。他们在去年添加了两个重要的社交功能：DeBank Hi 和 DeBank Stream。DeBank 拥有**「钱包绑定」** 这个做社交类应用的优势，能更好地追踪真实用户。

DeBank Stream 是一个很像 Twitter 的社交应用，奖励用户创造的价值（基于时间和注意力）。用户发布内容后可以获得相应金额的奖励，奖励价值是基于用户内容的曝光度和在 Stream 上传播过程中获得的一些推广，比如其它用户的关注、点击、评论和转发数量，以及这些用户的资产价值等因素。DeBank Hi 则是一个通讯 DApp，可以进行付费咨询或聊天。

  • Bluesky

Bluesky 是由 Twitter 联合创始人 Jack Dorsey 打造的去中心化社交应用，最早在 2019 年由 Twitter 孵化。Bluesky 具有类似 Twitter 的用户界面和算法选择，但与 Twitter 不同的是，Bluesky 的用户数据存储在独立的服务器上 ，而不是存储在某家公司的服务器上，从而使用户在平台的互动方式上有更多的自主权。Bluesky 目前仍在测试版中，仅限邀请加入。

  • Mask Network

Mask Network 是一个帮助用户从 Web2 无缝过渡到 Web3 的产品。它允许用户在 Twitter 上，无缝发送加密信息、加密货币，甚至是去中心化应用（比如 DeFi、NFT 和 DAO）。Mask Network 目前可用作智能手机应用程序 ，也可用作浏览器扩展功能 ，其最早的也可能是最重要的功能是私人讯息，允许用户通过公钥密码学在社交媒体上加密和解密内容 。

  • Stealcam

Stealcam 可以说是在 Friend.tech 之前火爆的 Web3 社交产品。它是一个 Arbitrum 生态的图片分享应用 ，用户可上传图片或视频文件至该平台上，但该平台上发布的图片不是免费访问，而需要支付一定费用才能获得开图权，查看图片的具体内容。同时，Stealcam 通过引入竞拍激励模型 ，将一件作品的所有参与者都计入到了利润分享模式中，从而可促进作品的价格发现。

Web3 社交产品

能够取代 Web2 传统社交应用吗？

在今年的 EthGlobal Waterloo 2023 会议上，Vitalik 的演讲表达了他对当前 Web3 社交发展的担忧。他认为现有的 Web3 社交 Dapp 可能不足以取代 Web2 的对应产品。 但他仍然建议开发者尝试利用 Web3 的独特功能迭代新的社交 Dapp ，构建一款比 Web2 同类产品好 10 倍的社交应用程序。

当下，Web2 社交平台频频发生数据泄露、隐私风波、算法偏见的大背景下，Web3 社交逐步进入大众视野，其去中心化、抗审查、用户为中心、数字资产赋权、身份互操作性 等特点广受用户和资本青睐。

Web3 社交的未来愿景固然美好，但当下仍存在不少现实问题。由于社交迁移成本高昂，中心化社交先发优势强大，因此加密社交赛道几乎没有成功的项目 。反观 Friend.tech 等当下最火的几个项目都可以看出，其受空投、激励和融资等金融属性 的影响要远大于应用作为社交属性本身。

同时，Web3 社交产品的成熟度也远远不足 ，即便是 Lens 等赛道龙头，其产品实际的用户使用体验较为生硬和不便。相比 Twitter、Facebook、Telegram、TikTok、微信等拥有海量用户的应用，要通过颠覆和超越实现用户的大规模转移，仍然道阻且长。

对于用户来说，人与人之间的社交价值以及创作和创造的价值，或许比社交行为带来的经济利益更加重要。与任何 Web3 应用创新一样，Web3 社交的未来同样取决于其适应、发展和持续为用户提供社交价值的能力。

Web3 社交产品只有比 Web2 产品更加具有创新性 ，用户才愿意放弃已有的习惯和社交圈去尝试。而目前的 Web3 社交应用最大的动力却是薅羊毛和撸空投，尚且未把精力投入在产品力上。只有通过实现技术的迭代和市场观念的转变，或许才将诞生杀手级应用，为 Web3 社交赛道带来爆发的新机会。
